Chapter 5: Credential Wallets

What is a Digital Credential Wallet?

A digital credential wallet is a secure application that stores, manages, and shares your verifiable credentials. Think of it as a digital version of your physical wallet, but specifically for educational achievements, certifications, and diplomas.

Types of Wallets

Mobile Wallets

Smartphone apps that store credentials locally on your device. Most popular option for individuals.

  • Always accessible
  • Offline verification support
  • Biometric security (fingerprint, face ID)
  • Easy sharing via QR codes

Web Wallets

Browser-based wallets accessible from any device. Good for institutions and frequent users.

  • Cross-platform access
  • Integration with web services
  • Bulk credential management
  • Cloud backup options

Hardware Wallets

Physical devices that store credentials. Highest security for valuable credentials like doctoral degrees.

  • Maximum security
  • Offline storage
  • Tamper-proof
  • Best for long-term storage

Core Wallet Features

Credential Storage

Securely store unlimited credentials with encrypted local storage and cloud backup options.

Selective Disclosure

Choose exactly which information to share. Share your degree without revealing your GPA, or prove graduation year without showing grades.

Presentation Templates

Save common sharing scenarios (job applications, visa applications, university transfers) for quick reuse.

Verification History

Track when and with whom you've shared credentials, maintaining a complete audit trail.

Security Features

🔐 End-to-End Encryption

All credentials are encrypted with your personal key. Even wallet providers cannot access your credentials.

🔑 Biometric Authentication

Face ID, fingerprint, or PIN required to access wallet. No one can use your credentials without your authorization.

🔒 Private Key Control

You control the cryptographic keys. Wallet providers cannot impersonate you or issue credentials in your name.

Popular Wallet Apps

WIA Credential Wallet

Official WIA wallet with full EDU-011 support, free for all users, available on iOS, Android, and web.

Microsoft Authenticator

Supports W3C Verifiable Credentials, integrates with Microsoft ecosystem, free.

Velocity Network Wallet

Career credential wallet supporting education and employment credentials.

Dock Wallet

Open-source wallet with strong privacy features and multi-chain support.

Sharing Credentials

QR Code Sharing

Generate a QR code containing the credential or verification link. Recipient scans to verify instantly.

Link Sharing

Create a time-limited verification link to share via email or messaging apps.

Direct Presentation

Send credential directly to employer or institution portals via API integration.

Printed Backup

Print credentials with embedded QR codes for physical submission when required.

Backup and Recovery

Seed Phrase Backup

12-24 word recovery phrase enables wallet restoration on any device. Store securely offline.

Cloud Backup

Encrypted backup to iCloud, Google Drive, or other cloud services. Automatic sync across devices.

Institution Re-issuance

Lost credentials can be re-issued by the original institution using blockchain records.

Privacy Controls

Wallets give you complete control over your data:

  • View history of all credential shares
  • Revoke access to previously shared credentials
  • Set expiration times for shared credentials
  • Control which fields are visible
  • Anonymous verification options

Integration with Services

Modern credential wallets integrate with:

  • Job application platforms (LinkedIn, Indeed)
  • Background check services
  • University admission portals
  • Professional licensing boards
  • Immigration and visa systems
  • Employer onboarding systems
