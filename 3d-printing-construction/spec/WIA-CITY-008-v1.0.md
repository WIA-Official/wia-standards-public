## 1. 개요

### 1.1 배경

3D 프린팅 건설(Construction 3D Printing, C3DP)은 적층 제조(Additive Manufacturing) 기술을 건축 분야에 적용한 혁신적 건설 방법입니다.

**글로벌 시장 규모:**
- **2024년**: $4억 달러
- **2030년 예상**: $50억 달러 (연평균 성장률 50%)
- **2050년 전망**: 전체 건설의 30-40%

**주요 프린터 제조사:**
- **ICON** (미국): Vulcan 시리즈, NASA 달 기지 프린터
- **COBOD** (덴마크): BOD2, 세계 최대 건설 프린터
- **WASP** (이탈리아): Crane WASP, 친환경 점토 프린팅
- **Apis Cor** (러시아): 이동식 프린터, 24시간 주택 건설
- **Mighty Buildings** (미국): 복합 재료 프린팅

### 1.2 문제점

현재 3D 프린팅 건설의 주요 문제점:

1. **표준 부재**: 프린터, 재료, 품질 기준 표준화 미흡
2. **건축 법규**: 기존 법규와의 충돌, 인증 절차 불명확
3. **구조 안전**: 층간 접착력(Layer Adhesion) 검증 부족
4. **내구성**: 장기 내구성 데이터 부족 (>50년)
5. **상호운용성**: 제조사별 독자 규격, 호환성 부족
6. **인력**: 전문 인력 부족 (설계, 운영, 검사)

### 1.3 WIA-CITY-008의 가치

이 표준은 다음을 제공합니다:

- ✅ **통일된 표준**: 프린터, 재료, 공정의 표준화
- ✅ **품질 보증**: 구조 안전성 검증 및 인증 체계
- ✅ **법규 준수**: 국제 건축 법규와의 호환성
- ✅ **추적성**: 건설 전 과정의 투명한 기록
- ✅ **상호운용성**: 제조사 간 호환 가능한 데이터 형식
- ✅ **지속가능성**: 친환경 재료 및 저탄소 건설

## 2. 표준의 목적

### 2.1 핵심 목표

1. **안전성 확보**: 전통 건축과 동등 이상의 구조 안전성
2. **비용 절감**: 건설 비용 30-50% 절감
3. **공기 단축**: 건설 기간 50-70% 단축
4. **환경 보호**: CO2 배출 40-60% 감소
5. **보급 확대**: 저가 주택, 긴급 대피소 신속 공급

### 2.2 기대 효과

**경제적 효과:**
- 건설 비용: $150/m² → $70/m² (전통 대비 53% 절감)
- 인건비: 90% 절감 (자동화)
- 재료 낭비: 60% 감소

**시간적 효과:**
- 소형 주택 (60m²): 24-48시간
- 중형 주택 (100m²): 3-5일
- 대형 건물 (500m²): 2-3주

**환경적 효과:**
- CO2 배출: 40-60% 감소
- 재료 낭비: 60% 감소 (전통 건축 30-40% → 3D 프린팅 5-10%)
- 재활용 재료 사용: 최대 90%

**사회적 효과:**
- 저가 주택 공급: 개발도상국, 재난 지역
- 맞춤형 주택: 개인 맞춤 설계 가능
- 고령자, 장애인 특화 주택

## 5. 3D 프린터 유형

### 5.1 Gantry (갠트리) 시스템

**구조:**
```
┌─────────────────────────┐
│   고정 레일 (X축)        │
│   ┌───────┐             │
│   │ 프린터│ ←Y축→       │
│   │ 헤드  │             │
│   └───┬───┘             │
│       ↓ Z축             │
└─────────────────────────┘
```

**특징:**
- 프린트 영역: 최대 15m × 15m × 10m
- 정밀도: ±2-5mm
- 속도: 50-200 mm/s
- 장점: 높은 정밀도, 안정성
- 단점: 고정 설치, 초기 비용 높음

**대표 모델:**
- COBOD BOD2 (덴마크): 100m² 주택 48시간
- PERI 3D Construction Printer (독일)

### 5.2 Robotic Arm (로봇 암)

**구조:**
- 6축 산업용 로봇 암
- 작업 반경: 2-4m
- 회전 범위: 360°

**특징:**
- 정밀도: ±1-3mm
- 유연성: 복잡한 형상, 곡선 가능
- 속도: 100-300 mm/s
- 장점: 자유로운 형상, 다기능
- 단점: 프린트 영역 제한

**대표 모델:**
- Apis Cor (러시아): 이동식, 24시간 주택
- CyBe Construction (네덜란드)

### 5.3 WASP (델타 프린터)

**구조:**
- 삼각 케이블 시스템
- 중앙 타워 (높이 12m)
- 프린트 지름: 최대 12m

**특징:**
- 정밀도: ±3-7mm
- 재료: 주로 점토, 흙
- 속도: 30-100 mm/s
- 장점: 친환경, 낮은 비용
- 단점: 속도 느림, 강도 낮음

**대표 모델:**
- Crane WASP (이탈리아): 친환경 주택

### 5.4 Crane (크레인) 시스템

**구조:**
- 타워 크레인 기반
- 프린트 영역: 무제한 (크레인 이동)

**특징:**
- 정밀도: ±5-10mm
- 대형 건물 가능
- 장점: 초대형 건물
- 단점: 정밀도 낮음, 설치 복잡

## 6. 재료 표준

### 6.1 콘크리트 (Concrete)

**조성:**
```
시멘트:     25-35%
골재:       40-50%
물:         10-15%
첨가제:     3-8% (지연제, 촉진제, 섬유)
```

**물성:**
- 밀도: 2,200-2,400 kg/m³
- 점도: 50-200 Pa·s
- 압축 강도: 20-60 MPa (28일)
- 인장 강도: 2-5 MPa
- 경화 시간: 초기 30-90분, 최종 6-10시간

**프린팅 파라미터:**
- 층 높이: 10-20 mm
- 압출 속도: 150-300 kg/h
- 권장 온도: 15-30°C

**용도:**
- 주거용 주택
- 상업용 건물
- 구조물 기초

### 6.2 지오폴리머 (Geopolymer)

**조성:**
```
플라이애시:   60-80%
슬래그:       10-20%
알칼리 활성제: 5-10%
골재:         10-20%
```

**환경 장점:**
- CO2 배출: 전통 시멘트 대비 80% 감소
- 에너지 소비: 70% 감소
- 산업 폐기물 재활용 (플라이애시)

**물성:**
- 밀도: 2,000-2,300 kg/m³
- 압축 강도: 30-80 MPa (매우 높음)
- 내화성: 1,000°C까지 안정
- 내화학성: 산, 염 저항성 우수

**용도:**
- 친환경 주택
- 내화 구조물
- 화학 공장

### 6.3 점토 (Clay)

**조성:**
```
점토:       70-90%
모래:       5-15%
섬유:       2-5% (짚, 대마)
물:         15-25%
```

**환경 장점:**
- 100% 천연 재료
- 100% 재활용 가능
- CO2 배출: 거의 없음 (90% 감소)
- 현지 조달 가능

**물성:**
- 밀도: 1,600-2,000 kg/m³
- 압축 강도: 2-10 MPa (낮음)
- 건조 시간: 2-4주
- 단열성: 우수

**용도:**
- 저가 주택 (개발도상국)
- 친환경 주택
- 예술 건축

### 6.4 섬유 보강 콘크리트 (Fiber-Reinforced)

**첨가 섬유:**
- **유리 섬유** (Glass Fiber): 3-5%, 인장 강도 +50%
- **탄소 섬유** (Carbon Fiber): 1-3%, 인장 강도 +100%
- **강철 섬유** (Steel Fiber): 2-4%, 인성 향상
- **폴리프로필렌 섬유**: 0.5-1%, 균열 방지

**효과:**
- 인장 강도: 2-3배 향상
- 굽힘 강도: 1.5-2배 향상
- 균열 저항: 크게 향상
- 내구성: 향상

## 7. 프린트 파라미터

### 7.1 층 설정

**층 높이 (Layer Height):**
```
소형 주택:      10-15 mm
중형 주택:      15-20 mm
대형 건물:      20-30 mm
긴급 대피소:    20-30 mm (속도 우선)
고품질 건축:    5-10 mm (품질 우선)
```

**층 너비 (Layer Width):**
- 일반적: 40-60 mm
- 벽 두께: 150-300 mm (2-5 층으로 구성)

### 7.2 속도

**프린팅 속도:**
```
외벽:           50-100 mm/s (고품질)
내벽:           100-200 mm/s (중품질)
충진재:         150-300 mm/s (저품질)
이동 속도:      200-500 mm/s (비프린팅)
```

**압출 속도:**
- 콘크리트: 150-300 kg/h
- 지오폴리머: 100-250 kg/h
- 점토: 50-150 kg/h

### 7.3 경로 계획 (Path Planning)

**벽 구조:**
```
외벽 → 내벽 → 충진재 (Infill)

예시 (200mm 벽):
┌─────────────────┐
│ 외벽 (50mm)     │
│ 충진재 (100mm)  │  ← Honeycomb 패턴 (20-30%)
│ 내벽 (50mm)     │
└─────────────────┘
```

**충진재 패턴:**
- **Honeycomb (벌집)**: 강도-무게 최적 (권장)
- **Rectilinear (직선)**: 단순, 빠름
- **Triangular (삼각)**: 높은 강도
- **None (없음)**: 이중 벽만 (경량)

### 7.4 층간 접착 (Layer Adhesion)

**중요성:**
- 3D 프린팅 건설의 **가장 중요한 품질 지표**
- 전통 건축의 일체식 타설과 달리, 층간 계면 존재

**영향 요인:**
```
1. 층간 대기 시간:
   - 최적: 1-5분
   - 너무 빠름: 변형 (층 무너짐)
   - 너무 느림: 접착력 저하

2. 재료 점도:
   - 적정 점도: 50-200 Pa·s
   - 너무 묽음: 변형, 흐름
   - 너무 진함: 압출 곤란, 기포

3. 압출 압력:
   - 적정 압력: 0.5-2.0 MPa
   - 압력 ↑ → 접착력 ↑, 변형 ↑

4. 온도:
   - 최적: 20-25°C
   - 고온 (>35°C): 급속 경화, 접착 불량
   - 저온 (<10°C): 경화 지연, 강도 저하
```

**접착력 목표:**
- 최소 요구: >3.0 MPa (인장)
- 권장: >5.0 MPa
- 우수: >7.0 MPa

### 8.2 내진 설계 (Seismic Design)

**지진 등급별 요구사항:**

```
진도 I-II (낮음):
  - 철근 불필요
  - 벽 두께 >150 mm

진도 III-IV (중간):
  - 수직 철근 (Ø10mm, @600mm)
  - 벽 두께 >200 mm

진도 V-VI (높음):
  - 수직 철근 (Ø12mm, @400mm)
  - 수평 철근 (Ø10mm, @600mm)
  - 벽 두께 >250 mm

진도 VII+ (매우 높음):
  - 3D 프린팅 미권장 (전통 철근 콘크리트 사용)
```

### 8.3 내풍 설계 (Wind Resistance)

**풍속별 요구사항:**

| 풍속 | 벽 두께 | 철근 |
|------|---------|------|
| <30 m/s | 150 mm | 불필요 |
| 30-40 m/s | 200 mm | 수직 Ø10mm @600mm |
| 40-50 m/s | 250 mm | 수직 Ø12mm @400mm |
| >50 m/s | 미권장 | 전통 공법 |

### 8.4 내화 성능 (Fire Resistance)

## 13. 품질 관리

### 13.1 QC 체크리스트

**프린팅 전:**
```
□ 프린터 캘리브레이션 완료

□ 재료 온도/점도 확인

□ 안전 장비 준비
```

**프린팅 중:**
```
□ 첫 10층 육안 검사 (매 층)
□ 이후 매 20층마다 검사
□ 온도/습도 모니터링 (1시간 간격)
□ 재료 공급 상태 확인 (30분 간격)
□ 치수 정확도 측정 (레이저 스캐너)
```

**프린팅 후:**
```
□ 전체 육안 검사
□ 치수 측정 (±5mm 이내)
□ 표면 결함 검사
□ 층간 접착 샘플 채취 (6개소)
□ 압축 강도 샘플 채취 (6개소)
```

### 13.2 결함 분류

| 결함 종류 | 심각도 | 조치 |
|----------|--------|------|
| **층간 박리** | CRITICAL | 즉시 중지, 재시공 |
| **과도한 변형 (>10mm)** | MAJOR | 중지, 보수 |
| **표면 균열 (>2mm)** | MAJOR | 보수 필요 |
| **치수 오차 (>5mm)** | MINOR | 기록, 보완 |
| **표면 거칠기** | MINOR | 마감 처리 |
