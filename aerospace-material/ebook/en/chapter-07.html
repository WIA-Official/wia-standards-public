Chapter 7: Testing and Characterization

Material testing and characterization form the foundation of aerospace material qualification. From basic tensile tests to sophisticated fatigue and fracture mechanics evaluation, rigorous testing ensures materials meet the demanding requirements of flight applications. This chapter covers testing methods and data analysis according to the WIA-AEROSPACE_MATERIAL standard.

7.1 Mechanical Testing

Primary Test Types

Aerospace Mechanical Testing Hierarchy
======================================

Static Tests:
├── Tensile (ASTM E8, E21)
│   ├── Ultimate tensile strength (Ftu)
│   ├── Yield strength (Fty)
│   ├── Elongation (e)
│   └── Reduction of area (RA)
├── Compression (ASTM E9)
├── Shear (ASTM B831)
├── Bearing (ASTM E238)
└── Hardness (ASTM E18, E92, E10)

Dynamic Tests:
├── Fatigue
│   ├── Stress-life (S-N)
│   ├── Strain-life (ε-N)
│   └── Fatigue crack growth (da/dN)
├── Impact (Charpy, Izod)
└── Dynamic modulus

Fracture Mechanics:
├── Plane strain fracture toughness (KIc)
├── J-integral (JIc)
├── Crack tip opening displacement (CTOD)
└── R-curve

Time-Dependent:
├── Creep (ASTM E139)
├── Stress rupture
└── Stress relaxation

7.2 Fatigue Testing

// Fatigue Test Configuration
interface FatigueTest {
  stress_life: {
    standard: 'ASTM E466';
    specimen: 'smooth, polished';
    loading: 'axial, bending, rotating beam';
    parameters: {
      stress_amplitude: number;
      R_ratio: number;      // min/max stress
      frequency: number;    // Hz
      environment: string;
    };
    output: 'S-N curve, endurance limit';
    statistics: 'Weibull, log-normal';
  };

  strain_life: {
    standard: 'ASTM E606';
    specimen: 'smooth, strain-controlled';
    parameters: {
      strain_amplitude: number;
      R_ratio: number;
      frequency: number;
    };
    analysis: {
      elastic: 'σ_a = E × ε_ae';
      plastic: 'Coffin-Manson';
      total: 'Basquin + Coffin-Manson';
    };
    output: 'strain-life curve, cyclic stress-strain';
  };

  crack_growth: {
    standard: 'ASTM E647';
    specimen: ['CT', 'MT', 'SEN(B)'];
    parameters: {
      delta_K: number;      // stress intensity range
      R_ratio: number;
      frequency: number;
      environment: string;
    };
    output: 'da/dN vs ΔK curve';
    model: 'Paris law: da/dN = C(ΔK)^m';
  };
}

7.4 Elevated Temperature Testing

High-Temperature Test Considerations

  • Heating: Furnace, induction, radiant, environmental chamber
  • Temperature control: ±2°C at test section
  • Extensometry: High-temp extensometers, non-contact methods
  • Atmosphere: Air, vacuum, inert gas (critical for Ti, superalloys)
  • Soak time: Thermal equilibration before loading
// Creep Test Parameters
interface CreepTest {
  standard: 'ASTM E139';

  setup: {
    specimen: 'cylindrical, flat';
    loading: 'constant load or stress';
    temperature: 'isothermal';
    measurement: 'strain vs time';
  };

  curve_regions: {
    primary: 'decreasing strain rate';
    secondary: 'constant strain rate (minimum)';
    tertiary: 'accelerating to failure';
  };

  analysis: {
    minimum_creep_rate: 'ε_min = A × σ^n × exp(-Q/RT)';
    rupture_life: 'Larson-Miller, Manson-Haferd';
    rupture_ductility: 'elongation, RA at failure';
  };

  design_data: {
    allowable_stress: '1% creep in 100,000 hr';
    rupture_stress: 'stress for failure at time';
  };
}

7.7 Microstructural Analysis

Characterization Techniques

  • Optical Microscopy: Grain size, phase distribution, inclusions
  • SEM: Fracture surfaces, fine microstructure, EDS composition
  • TEM: Dislocations, precipitates, interfaces
  • EBSD: Texture, grain orientation, misorientation
  • XRD: Phase identification, residual stress, texture
  • CT Scanning: 3D internal structure, porosity, fiber orientation
