제6장

급여 가이드라인 및 영양 계산

6.1 에너지 요구량의 기초

적절한 급여량을 결정하기 위해서는 먼저 동물의 에너지 요구량을 이해해야 합니다. 에너지는 킬로칼로리(kcal) 또는 킬로줄(kJ)로 표시됩니다.

6.1.1 휴식 시 에너지 요구량(RER)

RER은 휴식 상태에서 체온 유지, 호흡, 혈액 순환 등 기본 대사에 필요한 에너지입니다:

RER (kcal/day) = 70 × (체중kg)^0.75

또는 간소화된 공식:

RER (kcal/day) = 30 × 체중kg + 70 (체중 2-45kg 범위)

6.1.2 일일 에너지 요구량(DER)

DER은 RER에 생애주기, 활동 수준, 건강 상태 등의 요인을 반영한 실제 에너지 요구량입니다:

DER = RER × 활동 계수 
+----------------------------------+------------------+
| 생활/건강 상태                    | 활동 계수        |
+----------------------------------+------------------+
| 중성화된 성견/성묘               | 1.6              |
| 중성화되지 않은 성견/성묘         | 1.8              |
| 비활동적/비만 성향                | 1.2 - 1.4        |
| 체중 감량 필요                    | 1.0              |
| 체중 증가 필요                    | 1.2 - 1.8        |
| 가벼운 활동                       | 2.0              |
| 중간 활동                         | 3.0              |
| 고강도 활동 (작업견)              | 4.0 - 8.0        |
| 강아지 (이유 ~ 4개월)             | 3.0              |
| 강아지 (4개월 ~ 성견)             | 2.0              |
| 임신 (전반기)                     | 1.8              |
| 임신 (후반기)                     | 3.0              |
| 수유                              | 4.0 - 8.0        |
+----------------------------------+------------------+

6.1.3 에너지 계산 예시

// 예시: 중성화된 5kg 성묘의 DER 계산

// 1단계: RER 계산
RER = 70 × (5)^0.75
    = 70 × 3.34
    = 234 kcal/day

// 2단계: DER 계산 (중성화 성묘 계수 = 1.6)
DER = 234 × 1.6
    = 374 kcal/day

// 3단계: 급여량 계산 (사료 에너지 밀도 = 3,800 kcal/kg)
일일 급여량 = 374 / 3,800 × 1,000
            = 98g/day

// JSON 형식 계산 결과
{
  "petInfo": {
    "species": "고양이",
    "weight_kg": 5,
    "neutered": true,
    "activityLevel": "normal"
  },
  "energyCalculation": {
    "RER_kcal": 234,
    "activityFactor": 1.6,
    "DER_kcal": 374
  },

    "foodEnergyDensity_kcal_per_kg": 3800,
    "dailyAmount_g": 98,
    "mealsPerDay": 2,
    "perMealAmount_g": 49
  }
}

6.2 생애주기별 급여 가이드

6.2.1 강아지/새끼 고양이

성장기 동물은 높은 에너지와 영양 밀도가 필요합니다:

+------------------+------------------+------------------+------------------+
| 연령             | 급여 횟수        | DER 계수         | 특별 고려사항    |
+------------------+------------------+------------------+------------------+
| 이유 ~ 2개월     | 4-6회/일         | 3.0              | 물에 불려 급여   |
| 2 ~ 4개월        | 3-4회/일         | 2.5              | 점진적 건사료    |
| 4 ~ 6개월        | 3회/일           | 2.0              | 성장 모니터링    |
| 6 ~ 12개월       | 2회/일           | 1.8 - 2.0        | 대형견 칼슘 주의 |
| 12 ~ 24개월*     | 2회/일           | 1.6 - 1.8        | 성견 사료 전환   |
+------------------+------------------+------------------+------------------+
* 대형견/초대형견은 24개월까지 강아지 사료 급여

성장 곡선 모니터링:

  • 주 1회 체중 측정
  • 견종별 성장 곡선과 비교
  • 과체중/저체중 시 급여량 조정
  • 갑작스러운 성장 정체는 수의사 상담

6.2.2 성견/성묘

성체의 목표는 이상적인 체형 상태(BCS) 유지입니다:

{

      },
      "mediumBreed_10to25kg": {
        "baseDER": "RER × 1.6",
        "mealsPerDay": 2
      },
      "largeBreed_over25kg": {
        "baseDER": "RER × 1.4",
        "mealsPerDay": 2,
        "adjustmentNote": "대형견은 kg당 더 낮은 에너지 필요"
      }
    },
    "cats": {
      "indoor": {
        "baseDER": "RER × 1.4",
        "mealsPerDay": "2-3 또는 자유급식",
        "adjustmentNote": "비만 주의"
      },
      "outdoor": {
        "baseDER": "RER × 1.8",
        "mealsPerDay": "2-3"
      }
    }
  }
}

6.2.3 노령 동물

노령 동물은 대사율 감소로 에너지 요구량이 줄어들지만, 단백질 요구량은 유지되거나 증가합니다:

  • 에너지: 성체 대비 20-30% 감소
  • 단백질: 고품질 단백질로 근육량 유지 (DM 기준 25-30%)
  • 섬유질: 변비 예방을 위해 약간 증가
  • 관절 지원: 글루코사민, 콘드로이틴, 오메가-3
  • 인지 기능: 항산화제, MCT 오일

6.2.4 임신 및 수유

+------------------+------------------+------------------+
| 시기             | 에너지 요구량    | 급여 방법        |
+------------------+------------------+------------------+
| 임신 1-5주       | RER × 1.8        | 일반 급여        |
| 임신 6-7주       | RER × 2.0        | 10-15% 증량      |
| 임신 8-9주       | RER × 2.5-3.0    | 고에너지 사료    |
| 수유 1주         | RER × 3.0        | 자유급식         |
| 수유 피크 (3-4주)| RER × 4.0-8.0    | 자유급식         |
| 수유 이유기      | 점진적 감소      | 새끼 이유 따라   |
+------------------+------------------+------------------+
참고: 수유 중인 어미의 에너지 요구량은 새끼 수에 따라 크게 달라집니다. 새끼당 약 RER × 0.5의 추가 에너지가 필요합니다.

6.3 체형 상태 평가(BCS)

6.3.1 BCS 9점 척도

+-------+------------------+------------------------------------------+
| 점수  | 상태             | 특징                                     |
+-------+------------------+------------------------------------------+
| 1-3   | 저체중           | 갈비뼈, 척추, 골반뼈가 눈에 띄게 보임    |

+-------+------------------+------------------------------------------+

+-------+------------------+------------------------------------------+
| 5     | 이상적           | 갈비뼈가 촉진 가능, 약간의 지방층        |
|       |                  | 측면에서 복부 올라감, 허리 보임          |
+-------+------------------+------------------------------------------+
| 6     | 약간 과체중      | 갈비뼈 촉진 어려움, 지방층 증가          |

+-------+------------------+------------------------------------------+

6.4 특수 상황별 급여

6.4.1 체중 감량 프로그램

안전한 체중 감량 속도:

  • 개: 주당 체중의 1-2%
  • 고양이: 주당 체중의 0.5-1% (급격한 감량은 지방간 위험)
// 체중 감량 급여량 계산
목표 체중 = 현재 체중 × 0.85 (15% 감량 목표)
DER = RER(목표체중) × 1.0

예시: 8kg 과체중 고양이 → 목표 체중 6.8kg
RER(6.8kg) = 70 × 6.8^0.75 = 289 kcal
DER = 289 × 1.0 = 289 kcal/day

// 고섬유 체중관리 사료 (3,200 kcal/kg) 사용 시
일일 급여량 = 289 / 3,200 × 1,000 = 90g/day

6.4.2 질병 상태

+------------------+------------------+------------------+
| 상태             | 에너지 조정      | 영양 고려사항    |
+------------------+------------------+------------------+
| 수술 후 회복     | RER × 1.2-1.5    | 고단백질         |
| 감염/발열        | RER × 1.3-1.5    | 수분, 전해질     |
| 화상/외상        | RER × 1.5-2.0    | 고단백, 아연     |
| 암               | RER × 1.2-1.5    | 오메가-3, 아르기닌|
| 심부전           | RER × 1.0-1.2    | 나트륨 제한      |
| 신부전           | RER × 1.0-1.2    | 인, 단백질 조절  |
| 당뇨             | RER × 1.2        | 저탄수화물       |
+------------------+------------------+------------------+

6.4.3 작업견/활동견

  • 추적견/경찰견: RER × 2.5-3.0
  • 썰매견 (레이스): RER × 5.0-8.0
  • 목양견: RER × 2.0-3.0
  • 스포츠견 (어질리티): RER × 2.0-3.0

작업견 영양 포인트:

  • 고지방 사료로 에너지 밀도 확보
  • 운동 전후 급여 시간 조절
  • 수분 공급 철저
  • 전해질 보충 (장시간 활동)

6.5 사료 전환

6.5.1 점진적 전환 방법

급격한 사료 변경은 소화 장애를 유발할 수 있으므로 7-10일에 걸쳐 점진적으로 전환합니다:

+------------------+------------------+------------------+
| 일차             | 기존 사료        | 새 사료          |
+------------------+------------------+------------------+
| 1-2일            | 75%              | 25%              |
| 3-4일            | 50%              | 50%              |
| 5-6일            | 25%              | 75%              |
| 7일 이후         | 0%               | 100%             |
+------------------+------------------+------------------+

6.5.2 전환 시 주의사항

  • 배변 상태 모니터링 (설사, 변비)
  • 구토, 식욕 부진 관찰
  • 알레르기 증상 주의 (가려움, 피부 발적)
  • 문제 발생 시 전환 속도 늦추기

장 요약

  1. 에너지 계산: RER(휴식 시 에너지 요구량)에 생애주기와 활동 수준에 따른 계수를 적용하여 DER(일일 에너지 요구량)을 산출합니다.
  2. 생애주기 급여: 성장기(고에너지), 성체(유지), 노령기(감소된 에너지, 유지/증가된 단백질), 임신/수유(급격히 증가)에 따라 급여량이 달라집니다.
  3. BCS 평가: 9점 척도의 체형 상태 평가로 과체중/저체중을 판단하고 급여량을 조정합니다. 이상적 BCS는 5입니다.
  4. 특수 상황: 체중 관리, 질병, 작업견 등 특수 상황에서는 에너지와 영양소 요구량이 크게 달라지므로 맞춤 급여가 필요합니다.
  5. 사료 전환: 7-10일에 걸친 점진적 전환으로 소화 장애를 예방합니다.
