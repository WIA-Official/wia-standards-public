제5장

사료 라벨링 및 규제 요건

5.1 반려동물 사료 규제 체계

반려동물 사료는 각 국가 및 지역별로 다양한 규제 체계의 적용을 받습니다. 이 장에서는 주요 국가들의 규제 요건과 라벨링 규정을 상세히 다룹니다.

5.1.1 미국 규제 체계

연방 기관:

  • FDA(식품의약국): 연방식품의약품화장품법(FD&C Act) 하에 동물사료 안전성 관할
  • FSMA(식품안전현대화법): 2011년 제정, 예방적 통제 요구
  • CGMP: 현행 우수제조관리기준 준수 의무

AAFCO(미국사료검사관협회):

  • 법적 강제력은 없으나 사실상 업계 표준
  • 모델 법규(Model Regulations) 제공
  • 영양 프로파일 및 급여 시험 프로토콜 설정
  • 원료 정의(Official Publication) 발행
+------------------------+------------------+------------------+
| AAFCO 영양 적합성 기준 | 급여 시험        | 포뮬레이션 기준  |
+------------------------+------------------+------------------+
| 최소 요건              | 6개월 급여 시험  | 영양 프로파일    |
|                        | 최소 8마리 동물  | 계산값 충족      |
| 생애주기 검증          | 생리학적 검증    | 이론적 검증      |
| 비용                   | 높음             | 낮음             |
| 신뢰도                 | 높음             | 중간             |
+------------------------+------------------+------------------+

5.1.2 유럽연합 규제 체계

주요 규정:

  • Regulation (EC) 767/2009: 사료 유통 및 라벨링
  • Regulation (EC) 1831/2003: 사료 첨가제
  • Regulation (EU) 2017/625: 공식 통제
  • FEDIAF 가이드라인: 업계 자율 영양 가이드라인

EU 라벨 필수 항목:

  1. 사료 유형 (완전사료, 보조사료 등)
  2. 대상 동물 종 및 생애주기
  3. 성분 목록 (중량 내림차순)
  4. 분석 성분 (조단백질, 조지방, 조섬유, 조회분)
  5. 첨가제 (비타민, 미량원소, 방부제 등)
  6. 제조업체/수입업체 정보
  7. 로트 번호 및 최소유효기한
  8. 순중량

5.1.3 한국 규제 체계

관할 기관:

  • 농림축산식품부: 사료관리법 소관
  • 농림축산검역본부: 사료 등록 및 검사
  • 한국사료협회: 업계 자율 기준

사료관리법 주요 내용:

  • 사료 제조업/수입업 등록제
  • 성분등록제: 신규 원료 사전 등록
  • 유해물질 잔류허용기준
  • 표시사항 규정
{
  "koreanLabelRequirements": {
    "mandatory": [
      "사료의 명칭 및 형태",
      "등록번호",
      "제조업자/수입업자 명칭 및 주소",
      "제조연월일 또는 수입연월일",
      "유통기한 또는 품질유지기한",
      "사용원료의 명칭",
      "성분량 (조단백질, 조지방, 조섬유, 조회분, 칼슘, 인)",
      "주의사항",
      "순중량 및 포장단위"
    ],
    "optional": [
      "제조공정",
      "급여방법 및 급여량",
      "보관방법",
      "기능성 표시 (사전 승인 필요)"
    ]
  }
}

5.2 라벨 구성요소 상세

5.2.1 제품명 규칙

AAFCO의 제품명 규칙은 특정 원료의 함량에 따라 명명을 규제합니다:

+------------------------+------------------+------------------+------------------+
| 규칙                   | 함량 기준        | 예시             | 수분 제외 기준   |
+------------------------+------------------+------------------+------------------+
| 95% 규칙              | ≥95% (수분 제외) | "치킨 사료"      | 단일 원료       |
|                        | ≥70% (수분 포함) |                  | 복합 원료 가능   |
+------------------------+------------------+------------------+------------------+
| 25% 규칙 ("디너" 등)   | 25-95%           | "치킨 디너"      | 원료 명시       |
|                        |                  | "치킨 엔트레"    |                  |
+------------------------+------------------+------------------+------------------+
| "With" 규칙           | ≥3%              | "치킨 포함"      | 보조 원료       |
+------------------------+------------------+------------------+------------------+
| "Flavor" 규칙         | 검출 가능량      | "치킨 맛"        | 맛 목적         |
+------------------------+------------------+------------------+------------------+

5.2.2 보증 분석(Guaranteed Analysis)

라벨에 표시되는 영양 성분의 최소/최대 보증치입니다:

필수 표시 항목:

  • 조단백질 (Crude Protein): 최소 %
  • 조지방 (Crude Fat): 최소 %
  • 조섬유 (Crude Fiber): 최대 %
  • 수분 (Moisture): 최대 %

선택적 표시 항목:

  • 회분 (Ash)
  • 칼슘, 인
  • 타우린 (고양이 사료)
  • 오메가-3, 오메가-6 지방산
  • 글루코사민, 콘드로이틴
// 라벨 보증 분석 예시
{
  "guaranteedAnalysis": {
    "crudeProtein": {"min": 26, "unit": "%"},
    "crudeFat": {"min": 15, "unit": "%"},
    "crudeFiber": {"max": 4, "unit": "%"},
    "moisture": {"max": 10, "unit": "%"},
    "calcium": {"min": 1.0, "max": 1.5, "unit": "%"},
    "phosphorus": {"min": 0.8, "max": 1.2, "unit": "%"},
    "omega6": {"min": 2.5, "unit": "%"},
    "omega3": {"min": 0.5, "unit": "%"},
    "glucosamine": {"min": 400, "unit": "mg/kg"},
    "chondroitin": {"min": 300, "unit": "mg/kg"}
  }
}

5.2.3 성분 목록(Ingredient List)

성분은 중량 내림차순으로 나열됩니다. AAFCO 공식 정의에 따른 명칭을 사용해야 합니다:

참고: "육류 부산물(Meat By-Products)"은 살코기를 제외한 신선 동물 조직으로, 폐, 비장, 신장, 뇌, 간, 혈액, 뼈, 부분 탈지 저온 지방 조직, 내용물이 제거된 위와 장을 포함합니다. 털, 뿔, 이빨, 발굽은 제외됩니다.

일반 원료 용어:

  • 닭고기(Chicken): 뼈와 함께 또는 없이 살과 껍질의 깨끗한 조합
  • 닭고기분(Chicken Meal): 건조 렌더링된 닭고기. 깃털, 머리, 발, 내장 제외
  • 닭 부산물분(Chicken By-Product Meal): 목, 발, 미발달 알, 장 포함
  • 옥수수 글루텐분(Corn Gluten Meal): 전분, 배아, 섬유 제거 후 건조물

5.2.4 영양 적합성 문구

AAFCO 영양 적합성 문구는 사료가 특정 생애주기에 적합함을 나타냅니다:

// 급여 시험 기반
"[제품명]은 AAFCO 개 사료 영양 프로파일에서 설정한 영양 
수준을 충족하도록 제조되었습니다. 성견 유지용."

// 포뮬레이션 기반  
"[제품명]은 성장 및 번식기 고양이의 영양 요구를 충족하도록 
AAFCO 고양이 사료 급여 시험 절차를 통해 실질적으로 검증되었습니다."

// 모든 생애주기
"[제품명]은 AAFCO 개 사료 영양 프로파일에서 설정한 영양 
수준을 충족하도록 제조되었습니다. 모든 생애주기용 
(대형견(70파운드 이상 성견 체중) 성장기 제외)."

5.3 기능성 및 치료식 사료

5.3.1 기능성 주장

구조/기능 주장은 제품의 의학적 효능을 주장하지 않고 일반적인 건강 혜택을 설명합니다:

+------------------------+------------------+------------------+
| 허용되는 주장          | 금지되는 주장    | 근거 요건        |
+------------------------+------------------+------------------+
| "건강한 피부와 모피    | "피부염 치료"    | 오메가-6 함량    |
|  유지에 도움"          |                  | 증빙 자료        |
+------------------------+------------------+------------------+
| "건강한 관절 지원"     | "관절염 치료"    | 글루코사민 등    |
|                        |                  | 함량 증빙        |
+------------------------+------------------+------------------+
| "건강한 소화 지원"     | "IBD 치료"       | 프리바이오틱     |
|                        |                  | 프로바이오틱     |
+------------------------+------------------+------------------+
| "체중 관리에 도움"     | "비만 치료"      | 저칼로리         |
|                        |                  | 고섬유 설계      |
+------------------------+------------------+------------------+

5.3.2 처방 사료(Therapeutic/Veterinary Diet)

질병 관리 목적의 처방 사료는 추가 규제 요건이 적용됩니다:

  • 수의사 감독 하 급여 권장
  • 질병 주장에 대한 과학적 근거
  • 일부 국가에서 별도 승인 필요
  • 일반 유통 채널 판매 제한 (일부 지역)
{
  "therapeuticDietCategories": [
    {
      "condition": "신장 질환",
      "characteristics": [
        "인 제한 (0.3-0.5% DM)",
        "적당한 고품질 단백질",
        "칼륨 보충",
        "오메가-3 강화"
      ]
    },
    {
      "condition": "당뇨병",
      "characteristics": [
        "저탄수화물 (<15% DM)",
        "고단백 (>40% DM)",
        "고섬유",
        "L-카르니틴 첨가"
      ]
    },
    {
      "condition": "식이 알레르기",
      "characteristics": [
        "제한 항원 (단일 신규 단백질)",
        "가수분해 단백질",
        "오메가-3 강화"
      ]
    }
  ]
}

5.4 유기농 및 자연식 주장

5.4.1 유기농 인증

USDA Organic (미국):

  • "100% Organic": 100% 유기농 원료
  • "Organic": 95% 이상 유기농
  • "Made with Organic": 70% 이상 유기농
  • 인증 기관의 검증 필요

EU Organic:

  • Regulation (EU) 2018/848 준수
  • 최소 95% 유기농 원료
  • EU 유기농 로고 사용 가능

5.4.2 "자연(Natural)" 정의

AAFCO 정의에 따르면 "자연"은 화학적으로 합성되지 않은 원료만을 의미합니다:

주의: "자연" 주장을 하려면 비타민/미네랄을 제외한 모든 원료가 자연 유래여야 합니다. 합성 비타민이 포함된 경우 "비타민/미네랄 첨가 자연식"으로 표시해야 합니다.

5.4.3 기타 마케팅 주장

  • "인간 등급(Human-Grade)": 모든 원료가 인간 식용 가능 기준 충족
  • "홀리스틱(Holistic)": 법적 정의 없음. 마케팅 용어
  • "프리미엄/슈퍼프리미엄": 법적 정의 없음
  • "곡물 무첨가(Grain-Free)": 곡물(밀, 옥수수, 쌀 등) 미포함
  • "글루텐 무첨가": 글루텐 함유 원료 미포함

5.5 원산지 및 이력추적

5.5.1 원산지 표시

  • "Made in [국가]": 최종 가공이 이루어진 국가
  • 원료 원산지와 제조 국가가 다를 수 있음
  • 일부 국가는 특정 원료(육류 등)의 원산지 표시 요구

5.5.2 이력추적성(Traceability)

EU Regulation 178/2002는 "한 단계 전, 한 단계 후" 추적을 요구합니다:

{
  "traceabilityRecord": {
    "productId": "LOT-2024-0815-A1",
    "productionDate": "2024-08-15",
    "ingredients": [
      {
        "name": "닭고기분",
        "supplier": "ABC 렌더링",
        "lotNumber": "CKM-240801-01",
        "origin": "대한민국"
      },
      {
        "name": "현미",
        "supplier": "XYZ 곡물",
        "lotNumber": "BR-240720-05",

5.6 안전성 및 리콜

5.6.1 리콜 분류

+------------------------+------------------+------------------+
| 리콜 등급              | 건강 위험        | 예시             |
+------------------------+------------------+------------------+
| Class I               | 심각한 건강 위험 | 살모넬라 오염    |
|                        | 또는 사망 가능   | 높은 아플라톡신  |
+------------------------+------------------+------------------+
| Class II              | 일시적 건강 영향 | 비타민 과잉      |
|                        | 심각하지 않음    | 미량 금속 검출   |
+------------------------+------------------+------------------+
| Class III             | 건강 위험 없음   | 라벨 오류        |
|                        | 규정 위반        | 중량 부족        |
+------------------------+------------------+------------------+

5.6.2 리콜 절차

  1. 문제 확인 및 위험 평가
  2. 규제 기관 통보 (FDA, 식약처 등)
  3. 유통업체/소비자 통지
  4. 제품 회수 및 폐기
  5. 근본 원인 분석
  6. 시정 조치 이행
  7. 규제 기관에 종료 보고

5.7 국제 무역 규제

5.7.1 수입 요건

미국 수입:

  • FDA 시설 등록
  • 사전 통보(Prior Notice) 제출
  • FSVP(해외공급업체검증프로그램) 준수

EU 수입:

  • 승인된 제3국 목록
  • TRACES 시스템 등록
  • 국경 통제소 검사

한국 수입:

  • 사료수입업 등록
  • 검역 신고
  • 성분 및 안전성 검사

5.7.2 위생 증명서

동물성 원료가 포함된 사료는 수출국의 위생 증명서가 필요합니다:

{
  "sanitaryCertificate": {
    "type": "반려동물사료 위생증명서",
    "issuingAuthority": "농림축산검역본부",
    "certificateNumber": "KR-PET-2024-08-12345",
    "exporter": "펫푸드 코리아",
    "destination": "일본",
    "productDescription": "건조 개사료",
    "animalOriginIngredients": ["닭고기분", "연어오일"],
    "statements": [
      "BSE 위험 없는 국가 원산",
      "살모넬라 불검출",
      "열처리 완료"
    ],
    "issueDate": "2024-08-15",
    "veterinarySignature": true
  }
}

장 요약

  1. 다층적 규제 체계: 반려동물 사료는 연방/국가 기관(FDA, 농림부)과 업계 단체(AAFCO, FEDIAF)의 복합적인 규제를 받습니다.
  2. 제품명 규칙: AAFCO의 95%, 25%, "With", "Flavor" 규칙은 제품명에 사용되는 원료 함량을 엄격히 규제합니다.
  3. 영양 적합성 검증: 급여 시험 또는 포뮬레이션 기반으로 특정 생애주기에 대한 영양 적합성을 검증하고 표시합니다.
  4. 기능성 주장: 구조/기능 주장은 허용되나, 의학적 효능 주장은 금지됩니다. 처방 사료는 추가 규제 적용.
  5. 이력추적성: 원료부터 소비자까지 전 공급망 추적은 식품 안전 및 리콜 대응의 핵심입니다.
