1장: 기초 및 역사

1.1 인공 시각 소개

인공 눈 기술은 신경과학, 생체의공학, 마이크로전자공학, 인공지능의 융합을 통해 시력을 잃은 개인에게 기능적 시각을 복원합니다. 손상된 생물학적 구조를 복구하거나 대체하려는 전통적인 치료법과 달리, 인공 눈은 시각 시스템의 손상된 부분을 완전히 우회하여 나머지 신경 경로를 직접 자극하여 시각적 인식을 생성합니다.

인공 시각의 기본 원리는 매우 간단합니다: 시각 경로에 있는 뉴런의 전기적 자극은 섬광(phosphene)이라고 불리는 빛의 감각을 생성할 수 있습니다. 이러한 전기 자극의 패턴, 강도, 타이밍을 제어함으로써 인공 눈 시스템은 환자가 환경을 인식하고, 물체를 인식하고, 독립적으로 이동할 수 있게 하는 의미 있는 시각적 패턴을 생성할 수 있습니다.

정의: 인공 눈(시각 보철물)은 환경에서 시각 정보를 캡처하고 시각 신경계에 전달되는 전기 자극 패턴으로 변환하여 특정 유형의 실명을 가진 개인에게 시각적 인식을 생성하는 전자 장치입니다.

1.2 시각 시스템의 해부학

인공 눈이 어떻게 기능하고 어디에 개입할 수 있는지 이해하기 위해서는 시각 경로를 이해하는 것이 필수적입니다:

1.2.1 눈의 구조

  • 각막: 들어오는 빛을 집중시키는 투명한 전면 표면
  • 홍채와 동공: 눈에 들어오는 빛의 양을 조절
  • 수정체: 추가적인 집중력 제공, 근거리 및 원거리 시야 조절 가능
  • 유리체: 눈 내부를 채우는 투명한 젤
  • 망막: 눈 뒤쪽을 덮고 있는 빛에 민감한 신경 조직

1.2.2 망막 구조

망막에는 약 1억 3천만 개의 광수용체와 여러 층의 처리 뉴런이 포함되어 있습니다:

  • 광수용체(간상세포와 원추세포): 빛을 전기 신호로 변환
  • 양극세포: 중간 처리층
  • 신경절세포: 축삭이 시신경을 형성하는 최종 망막 출력 뉴런
  • 수평세포와 아마크린세포: 대비와 움직임 감지를 향상시키는 측면 처리 요소

1.2.3 뇌로의 시각 경로

시각 정보는 망막에서 시신경을 통해 시상의 외측슬상핵(LGN)으로, 최종적으로 후두엽의 일차 시각 피질(V1)로 이동합니다. 더 높은 시각 영역에서는 움직임, 색상, 물체, 얼굴과 같은 점점 더 복잡한 특징을 처리합니다.


// 시각 경로 모델

1.3 시력 상실의 유형

인공 눈은 망막 신경절세포에서 뇌까지의 시각 경로가 손상되지 않은 특정 유형의 실명에 효과적입니다:

1.3.1 망막색소변성증 (RP)

광수용체의 진행성 퇴화를 일으키는 유전 질환 그룹입니다. 일반적으로 야맹증과 주변 시야 손실로 시작하여 결국 완전한 실명으로 이어집니다. 전 세계적으로 약 150만 명에게 영향을 미칩니다. 내부 망막층은 종종 비교적 보존되어 있어 RP 환자가 망막 보철물의 이상적인 후보가 됩니다.

1.3.2 연령 관련 황반변성 (AMD)

선진국에서 실명의 주요 원인인 AMD는 선명하고 세부적인 시야를 담당하는 망막의 중심부인 황반을 파괴합니다. 건성 AMD는 점진적인 광수용체 손실을 포함하고, 습성 AMD는 비정상적인 혈관 성장을 특징으로 합니다. 진행된 AMD 환자는 인공 눈 기술의 혜택을 받을 수 있지만, 주변 시야 보존이 후보 적합성을 복잡하게 만듭니다.

1.3.3 기타 후보

  • 맥락막결손증: 맥락막, RPE 및 광수용체의 진행성 퇴화
  • 어셔 증후군: 청각과 시력 손실의 조합
  • 원추-간상 이영양증: 주로 중심 시력에 영향
  • 외상성 망막 손상: 부상 관련 광수용체 손실
후보 요구사항: 환자는 이전에 유용한 시력을 가졌어야 하고(시각 피질이 정상적으로 발달했음을 보장하기 위해), 망막 이식의 경우 손상되지 않은 신경절세포와 시신경 기능을 가져야 하며, 수술을 위한 심리적 및 신체적 건강 기준을 충족해야 합니다.

1.4 역사적 발전

1.4.1 초기 선구자들 (1929-1970년대)

전기적 시각 복원의 개념은 1929년 독일 연구자 Otfrid Foerster가 신경외과 수술 중 환자의 시각 피질에 대한 전기 자극이 섬광을 생성한다는 것을 발견하면서 시작되었습니다. 이 관찰은 전자적 수단을 통한 인공 시각의 가능성을 시사했습니다.

1960년대와 1970년대에 Giles Brindley와 William Dobelle은 획기적인 피질 자극 실험을 수행하여, 시각 피질 표면의 전극 배열이 섬광 패턴을 생성할 수 있으며 피험자가 간단한 모양과 문자를 인식할 수 있음을 보여주었습니다.

1.4.2 망막 접근법 (1980년대-2000년대)

망막이 피질보다 더 접근하기 쉽고 덜 침습적인 목표를 제공한다는 인식으로 망막 보철물 개발이 시작되었습니다. 주요 이정표:

  • 1988: Mark Humayun이 망막 전기 자극에 대한 체계적인 연구 시작
  • 1996: 최초의 급성 인간 망막 자극 연구
  • 2002: 16개 전극이 있는 최초의 만성 망막 이식(Argus I)
  • 2007: 60개 전극이 있는 Second Sight의 Argus II 임상 시험 시작
  • 2013: Argus II FDA 승인 및 CE 마크 획득

1.4.3 현재 세대 (2010년대-현재)

현대 인공 눈 시스템은 크게 향상된 해상도, 더 정교한 영상 처리, 더 나은 장기 생체적합성을 특징으로 합니다:

1.5 인공 눈의 작동 원리

1.5.1 외부 구성요소

대부분의 인공 눈 시스템에는 시각적 장면을 캡처하는 안경 장착 카메라가 포함되어 있습니다. 휴대용 프로세서(종종 벨트나 어깨에 착용)는 비디오 피드를 수신하고, 영상 처리 알고리즘을 적용하고, 시각 데이터를 자극 패턴으로 변환합니다. 이 정보는 무선으로 이식된 구성요소로 전송됩니다.

1.5.2 이식 구성요소

이식된 장치는 무선 신호와 전력을 수신하고, 자극 명령을 디코딩하고, 망막 위나 근처에 배치된 전극 배열을 통해 정밀하게 제어된 전기 펄스를 전달합니다. 현재 시스템은 체액으로부터 민감한 전자장치를 보호하기 위해 밀폐 밀봉된 티타늄 케이스를 사용합니다.

1.5.3 신경 자극


// 단순화된 자극 매개변수 모델
interface StimulationPulse {
    amplitude: number;      // 일반적으로 10-500 마이크로암페어
    pulseWidth: number;     // 일반적으로 0.1-1.0 밀리초
    frequency: number;      // 일반적으로 전극당 10-50 Hz
    waveform: 'biphasic' | 'triphasic';
    chargeBalance: boolean; // 안전을 위해 반드시 true여야 함
}

// 섬광 밝기 상관관계
function calculatePerceivedBrightness(pulse: StimulationPulse): number {
    // 밝기는 일반적으로 전하 전달과 상관관계가 있음
    const charge = pulse.amplitude * pulse.pulseWidth;
    return Math.min(charge / THRESHOLD_CHARGE, 1.0);
}

1.6 현재 능력 및 한계

현대 인공 눈은 정상적인 시력이 아닌 기능적 시각을 제공합니다. 일반적인 결과:

  • 빛과 움직임 감지
  • 큰 물체와 장애물 인식
  • 향상된 이동성과 탐색
  • 큰 인쇄 텍스트 읽기 (연습 후)
  • 얼굴 감지 (인식은 아님)
  • 독립적인 일상 활동

현재 한계에는 낮은 해상도(자연 시각의 수백만 광수용체에 비해), 단색 인식, 좁은 시야, 장면을 스캔하기 위해 눈 움직임 대신 머리 움직임이 필요한 점이 포함됩니다.

핵심 통찰: 현재의 한계에도 불구하고, 인공 눈 기술은 의미 있는 시각 기능을 복원함으로써 삶을 변화시킵니다. 환자들은 시각 보철물을 받은 후 삶의 질, 독립성, 심리적 웰빙이 향상되었다고 일관되게 보고합니다.

1.7 용어 및 정의

용어정의
섬광 (Phosphene)망막이나 시각 피질의 전기적 또는 기계적 자극에 의해 생성되는 빛의 시각적 인식
전극 배열신경 조직에 자극을 전달하는 전기 접점의 그리드

동적 범위역치와 최대 편안한 자극 수준 사이의 범위

1.8 요약

이 장에서는 인공 눈 기술을 이해하는 데 필요한 기초 지식을 수립했습니다: 시각 시스템의 해부학, 보철 치료가 가능한 실명을 일으키는 조건, 분야의 역사적 발전, 인공 눈이 기능적 시각을 복원하는 방법의 기본 원리. 후속 장에서는 시스템 아키텍처, 영상 처리 알고리즘, 수술 절차, 안전 요구사항 및 미래 발전에 대해 자세히 살펴볼 것입니다.
