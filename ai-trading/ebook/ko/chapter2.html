머신러닝 모델

트레이딩에서의 ML 개요

머신러닝은 전통적인 통계 방법으로는 감지할 수 없는 금융 데이터의 복잡한 패턴을 발견함으로써 퀀트 트레이딩을 혁신했습니다.

시계열 예측 모델

LSTM (Long Short-Term Memory)

  • 순차 패턴 인식에 최적
  • 장기 종속성을 학습하는 메모리 셀
  • 변동하는 길이의 시퀀스를 자연스럽게 처리
  • 다단계 앞서 예측에 효과적

GRU (Gated Recurrent Unit)

  • LSTM보다 빠른 학습
  • 매개변수가 적어 과적합 가능성 낮음
  • 실시간 트레이딩에 적합

Transformer 모델

  • 더 빠른 학습을 위한 병렬 처리
  • 주목 메커니즘이 중요한 시간 단계 식별
  • 다중 모달 데이터에 탁월
  • 여러 예측 작업에서 최첨단 결과

앙상블 방법

Random Forest

  • 부트스트래핑을 통한 과적합에 강함
  • 특성 중요도 분석
  • 수백 개의 기술 지표와 잘 작동

XGBoost

  • 최첨단 그래디언트 부스팅 구현
  • GPU 가속으로 빠른 학습
  • 과적합 방지를 위한 내장 정규화
  • 대규모 데이터셋에 탁월

강화 학습

  • 상태: 시장 특성, 포지션, 포트폴리오 가치
  • 행동: 매수, 매도, 보유
  • 보상: 수익/손실, 샤프 비율, 리스크 조정 수익
  • 정책: 상태를 행동에 매핑하는 학습된 전략

트레이딩에서 RL의 장점

  • 시장 환경과의 상호작용에서 학습
  • 장기 누적 수익 최적화
  • 순차적 의사결정을 자연스럽게 처리
  • 명시적 규칙 없이 복잡한 전략 학습 가능

피처 엔지니어링

가격 기반 특성

  • 수익률: 로그 수익률, 백분율 수익률
  • 변동성: 실현 변동성, GARCH 예측
  • 이동평균: SMA, EMA, 가중 MA

기술 지표

  • 모멘텀: RSI, MACD, Stochastic
  • 추세: ADX, Aroon, Parabolic SAR
  • 거래량: OBV, Money Flow Index
  • 변동성: Bollinger Bands, ATR

대체 데이터 특성

  • 뉴스 및 소셜 미디어의 감정 점수
  • 검색 트렌드 및 웹 트래픽
  • 소매, 농업, 에너지를 위한 위성 이미지
