제2장: 3D 스캐닝 기술

弘益人間 · 널리 인간을 이롭게 하라

고고학 유물의 디지털 복제

3D 스캐닝은 물리적 객체를 정확한 디지털 모델로 변환하는 기술입니다. 고고학에서 3D 스캐닝은 유물을 보존하고, 연구하고, 공유하는 방식을 혁신하고 있습니다. 밀리미터 단위의 정확도로 유물을 디지털화하여 영구 보존하고, 전 세계 연구자들과 공유하며, 손상 없이 분석할 수 있습니다.

현대의 3D 스캐닝 기술은 크게 세 가지 방법으로 나뉩니다: 포토그래메트리 (사진 기반), 레이저 스캐닝 (광학 기반), 구조광 스캐닝 (패턴 투사 기반). 각 방법은 고유한 장단점이 있으며, 유물의 크기, 재질, 표면 특성, 필요한 정확도에 따라 적절한 방법을 선택해야 합니다.

1. 포토그래메트리 (Photogrammetry)

포토그래메트리는 다양한 각도에서 촬영한 여러 장의 사진을 분석하여 3D 모델을 생성하는 기술입니다. 이 방법은 비용 효율적이고, 휴대가 간편하며, 높은 해상도의 텍스처를 제공합니다.

포토그래메트리의 장점

  • 비용 효율성: 일반 DSLR 카메라로도 가능 (초기 투자 50-200만원)
  • 휴대성: 카메라와 삼각대만 있으면 현장 어디서나 스캔 가능
  • 고품질 텍스처: 사진의 색상 정보를 그대로 활용하여 사실적인 질감
  • 대규모 적용: 발굴 현장 전체, 건축물 등 큰 대상도 스캔 가능
  • 낮은 진입 장벽: 비교적 쉽게 배우고 사용 가능

포토그래메트리의 단점

  • 표면 제약: 반짝이거나 투명한 표면은 처리 어려움
  • 처리 시간: 수백 장의 사진을 처리하는 데 수 시간 소요
  • 조명 의존성: 일관된 조명 조건 필요
  • 정확도 한계: 레이저 스캐닝보다 정확도 낮음 (±0.1-1mm)

포토그래메트리 워크플로우

단계 작업 내용 소요 시간 핵심 포인트
1. 촬영 준비 조명 설정, 배경 준비, 카메라 설정 30분 확산 조명, 무채색 배경, ISO 100-400
2. 사진 촬영 다양한 각도에서 80-300장 촬영 45분 60-80% 중첩, 3-5개 링 구성
3. 이미지 정렬 카메라 위치 추정, 특징점 매칭 1-3시간 고품질 설정, GPU 가속 활용
4. 점군 생성 Dense point cloud 계산 2-6시간 Ultra-high 품질 권장
5. 메쉬 생성 삼각형 메쉬 구축 1-2시간 Manifold geometry 보장
6. 텍스처 생성 사진을 3D 모델에 매핑 30분-1시간 색상 보정, Multiband blending
7. 최적화 메쉬 간소화, 구멍 메우기 30분 용도별 LOD 생성

촬영 패턴

포토그래메트리의 성공은 촬영 패턴에 크게 좌우됩니다. 다음은 권장하는 촬영 패턴입니다:

  • Ring 1 (수평): 객체 중심 높이에서 20-30장 (간격 12-18°)
  • Ring 2 (상향 30°): 위쪽에서 아래를 향해 20-30장
  • Ring 3 (하향 30°): 아래쪽에서 위를 향해 20-30장
  • Ring 4 (상부): 객체 위쪽 세부사항 15-20장
  • Ring 5 (하부): 객체 아래쪽 (받침대 위에 올려놓은 경우) 15-20장

2. 레이저 스캐닝 (Laser Scanning)

레이저 스캐닝은 레이저 광선을 사용하여 객체 표면까지의 거리를 정밀하게 측정합니다. 이 방법은 매우 높은 정확도를 제공하며, 조명 조건에 독립적입니다.

레이저 스캐닝의 장점

  • 최고 정확도: ±0.01-0.05mm의 정밀도
  • 빠른 스캔: 초당 76,000-500,000 포인트 캡처
  • 조명 무관: 어두운 환경이나 밝은 환경 모두 가능
  • 복잡 형상: 미세한 조각, 복잡한 패턴도 정확히 캡처
  • 신뢰성: 일관되고 재현 가능한 결과

레이저 스캐닝의 단점

  • 높은 비용: 장비 가격 3,000만-3억원
  • 크기와 무게: 대형 장비로 휴대 제한적
  • 색상 제한: 일부 장비는 색상 정보 제한적
  • 표면 제약: 투명하거나 강하게 반사하는 표면 어려움

레이저 스캐닝 종류

방식 원리 정확도 속도 적용
Triangulation 삼각측량 ±0.01-0.05mm 76,000 pts/s 소형 유물 (1cm-50cm)
Time-of-Flight 비행 시간 ±1-5mm 1,000,000 pts/s 대형 유적 (1m-1km)
Phase-Shift 위상 차이 ±0.5-2mm 500,000 pts/s 중형 유물/구조물

3. 구조광 스캐닝 (Structured Light Scanning)

구조광 스캐닝은 패턴화된 광선(주로 흰색과 검은색 줄무늬)을 투사하고, 이 패턴이 객체 표면에서 어떻게 왜곡되는지를 분석하여 3D 형상을 캡처합니다.

구조광 스캐닝의 특징

  • 빠른 캡처: 1-2초에 전체 표면 스캔
  • 높은 해상도: 0.05-0.2mm 정확도
  • 휴대 가능: 소형 핸드헬드 장비 가능
  • 실시간 피드백: 스캔 진행 상황 즉시 확인
  • 비교적 저렴: 100만-1,500만원

적용 제약

  • 강한 주변 조명에 민감 (실내 권장)
  • 매우 어두운 표면 (흡광율 >95%) 어려움
  • 스캔 범위 제한 (10cm-2m)
  • 움직이는 객체는 스캔 불가

실제 사례: 백제 금동대향로 3D 스캔

국립부여박물관의 백제 금동대향로 디지털화 프로젝트는 다중 스캐닝 기법을 결합한 성공 사례입니다.

프로젝트 세부사항

유물 정보:

  • 명칭: 백제 금동대향로 (국보 제287호)
  • 시대: 백제 (6세기 후반)
  • 크기: 높이 61.8cm, 무게 11.8kg
  • 재질: 청동에 금도금
  • 특징: 산수 문양, 악기를 연주하는 인물상 5명, 봉황 장식

스캐닝 방법:

  • 1차 스캔 (전체): 포토그래메트리 (Nikon D850, 250장)
  • 2차 스캔 (세부): 구조광 스캐너 (EinScan Pro 2X Plus)
  • 3차 스캔 (초정밀): 레이저 스캐너 (Hexagon 7축 arm)

결과:

  • 폴리곤 수: 2,500만 개 (아카이브 버전)
  • 텍스처: 8K (8192×8192 픽셀)
  • 정확도: ±0.03mm
  • 파일 크기: 4.2GB (glTF with Draco)
  • 처리 시간: 총 72시간
  • 온라인 뷰어: 월 15만 방문자

소프트웨어 도구

3D 스캔 처리를 위한 주요 소프트웨어:

포토그래메트리 소프트웨어

  • RealityCapture: 가장 빠른 처리 속도, 상용 라이선스
  • Agisoft Metashape: 업계 표준, 안정적 성능
  • 3DF Zephyr: 중가형, 사용자 친화적
  • Meshroom: 무료 오픈소스 (AliceVision)
  • COLMAP: 무료, 학술 연구용

메쉬 처리 도구

  • Blender: 무료, 강력한 3D 편집 기능
  • MeshLab: 무료, 메쉬 정리 및 분석
  • CloudCompare: 무료, 점군 비교 및 분석
  • ZBrush: 조각 및 세부 추가
  • Geomagic Design X: 역설계, CAD 변환
