## 문제

```
시리아 의사 → 10년 공부 → 전쟁 → 서류 없음 → 독일에서 청소부
우크라이나 교수 → 30년 경력 → 전쟁 → 서류 없음 → 폴란드에서 공장 노동자
```

**국가가 무너지면, 그 사람의 역사도 같이 사라집니다.**

3. **Level 3**: 역량 평가 (테스트/포트폴리오)
4. **Level 4**: 문서 검증 (부분 문서라도)
