6.1 궤도상 정제의 필요성

소행성에서 채굴한 광석을 그대로 지구로 운송하는 것은 비효율적입니다. 광석의 대부분은 불순물(규산염, 산화물)이며, 실제 경제적 가치가 있는 금속은 10~20%에 불과하기 때문입니다. 따라서 우주 공간에서 정제하여 순도 높은 금속만 운송하는 것이 훨씬 경제적입니다.

6.2 물 정제 및 연료 생산

6.2.1 물 전기분해 시스템

C형 소행성에서 추출한 물은 전기분해를 통해 수소와 산소로 분리됩니다. 이는 로켓 연료(LH₂/LOX)로 사용되어, 우주 정거장이나 화성 임무에 공급됩니다.

// 물 전기분해 효율 계산
class WaterElectrolysis {
    constructor(voltage = 1.48, efficiency = 0.70) {
        this.minVoltage = 1.23; // 이론적 최소 전압 (V)
        this.operatingVoltage = voltage; // 실제 작동 전압 (V)
        this.efficiency = efficiency; // 시스템 효율 (70~80%)
    }

    calculateHydrogenProduction(waterMass, power) {
        // H₂O → H₂ + 0.5 O₂
        // 1 mol H₂O (18g) → 1 mol H₂ (2g) + 0.5 mol O₂ (16g)

        const energyPerKg = 39.4e6; // J/kg (수소 생산에 필요한 에너지)
        const energyRequired = waterMass * energyPerKg / this.efficiency;
        const time = energyRequired / power; // 초

        const h2Mass = waterMass * (2 / 18); // kg
        const o2Mass = waterMass * (16 / 18); // kg

        return {
            hydrogen: h2Mass,
            oxygen: o2Mass,
            time: time / 3600, // 시간
            energy: energyRequired / 3.6e6 // kWh
        };
    }
}

// 예시: 1000 kg 물을 100 kW 전력으로 분해
const electrolyzer = new WaterElectrolysis();
const result = electrolyzer.calculateHydrogenProduction(1000, 100000);
console.log(`수소: ${result.hydrogen.toFixed(2)} kg`);
console.log(`산소: ${result.oxygen.toFixed(2)} kg`);
console.log(`소요 시간: ${result.time.toFixed(1)} 시간`);
// 출력: 수소: 111.11 kg, 산소: 888.89 kg, 소요 시간: 156.6 시간

6.2.2 액화 및 저장

생성된 수소와 산소는 극저온으로 냉각하여 액체 상태로 저장합니다. 이는 부피를 크게 줄여 저장 효율을 높입니다.

물질 끓는점 (1 atm) 액체 밀도 (kg/L) 기체 밀도 (kg/L, 1 atm, 25°C)
액체 수소 (LH₂) -253°C (20 K) 0.071 0.000082
액체 산소 (LOX) -183°C (90 K) 1.141 0.00133

우주 공간은 이미 극저온이므로, 냉각에 필요한 에너지가 지구보다 훨씬 적습니다. 단열 처리된 탱크만 있으면 장기 보관이 가능합니다.

6.3 금속 정제

6.3.1 태양광 집열 제련

대기가 없는 우주 공간에서는 태양광을 렌즈나 거울로 집중시켜 매우 높은 온도를 얻을 수 있습니다. 이를 이용하여 광석을 녹이고 금속을 분리합니다.

  • 프레넬 렌즈: 얇고 가벼운 렌즈로 태양광 집중 (온도 1000~1500°C 달성)
  • 포물선 거울: 대형 거울로 초점에 열 집중 (온도 2000~3000°C 가능)
  • 장점: 전력 소모 없음, 무제한 에너지원
  • 단점: 태양 방향으로만 작동, 회전하는 소행성에서는 추적 필요

6.3.2 전기로 (Electric Arc Furnace)

전기 아크(방전)로 금속을 녹이는 방법입니다. 온도 제어가 정밀하고, 밀폐 용기에서 작업 가능합니다.

⚡ 전기로 장점 (우주 환경):
• 진공 환경에서 산화 방지 (지구에서는 불활성 가스 필요)
• 무중력으로 용융물 관리 용이 (부유 상태 유지)
• 높은 순도 금속 생산 가능 (대기 오염 없음)

6.3.3 백금족 금속 (PGM) 전기화학적 정제

백금, 팔라듐, 로듐 등 백금족 금속은 전기분해를 통해 분리합니다. 이는 지구에서도 사용되는 방법이지만, 우주에서는 중력이 없어 더욱 효율적입니다.

  1. 용해: 철-니켈 합금을 왕수(HCl + HNO₃)에 용해
  2. 선택적 침전: pH 조절로 금속별로 순차적으로 침전
  3. 전기분해: 각 금속 이온을 전극에 석출
  4. 정제: 순도 99.9% 이상의 순수 금속 획득

6.4 3D 프린팅 및 부품 제조

6.4.1 금속 3D 프린팅

정제된 금속 분말을 이용하여 우주에서 직접 부품을 제조할 수 있습니다. 이는 지구에서 부품을 발사하는 비용을 획기적으로 절감합니다.

기술 원리 재료 장점
SLM (Selective Laser Melting) 레이저로 금속 분말 용융 티타늄, 알루미늄, 철 고강도, 복잡한 형상
EBM (Electron Beam Melting) 전자빔으로 금속 분말 용융 티타늄, 코발트 크롬 진공에서 최적, 빠름
Wire Arc Additive 용접 와이어를 녹여 적층 강철, 스테인리스 대형 부품 제작

6.4.2 무중력 환경의 장점

무중력은 3D 프린팅에 독특한 이점을 제공합니다:

  • 지지대 불필요: 중력이 없어 돌출 부분도 지지대 없이 프린팅 가능
  • 대형 구조물: 자중에 의한 변형이 없어 수십 m 크기 구조물 제작 가능
  • 균일한 조직: 대류가 없어 금속 조직이 더욱 균일
  • 고순도: 대기 오염이 없어 산화나 질화 방지

6.4.3 ISS 실험 결과

국제우주정거장(ISS)에서는 이미 여러 차례 3D 프린팅 실험이 수행되었습니다. Made In Space사의 Additive Manufacturing Facility는 플라스틱뿐만 아니라 금속 부품도 제조할 수 있습니다.

🔧 ISS 3D 프린팅 성과:
• 2014년: 최초 우주 3D 프린팅 성공 (플라스틱 렌치)
• 2016년: 금속 부품 프린팅 (스테인리스 강)
• 2018년: 바이오 프린팅 실험 (인공 조직)
• 2020년: 광섬유 제조 실험 (지구보다 10배 순도 높음)

6.5 규산염 가공 및 건설 자재

6.5.1 현무암 유사 콘크리트

S형 소행성의 규산염 광물은 현무암과 유사하여, 이를 가공하면 우주 건설 자재로 활용할 수 있습니다.

  1. 분쇄: 규산염 암석을 미세 분말로 분쇄
  2. 혼합: 물 또는 결합제와 혼합하여 반죽 상태로
  3. 성형: 3D 프린팅 또는 주형에 부어 성형
  4. 경화: 진공 환경에서 자연 경화 (대기 중보다 빠름)

6.5.2 방사선 차폐재

규산염은 밀도가 높아 우주 방사선 차폐에 효과적입니다. 특히 수소가 풍부한 광물(예: 수화 광물)은 중성자 차폐에 탁월합니다.

재료 밀도 (g/cm³) 차폐 효과 두께 (50% 감쇄)
알루미늄 2.7 낮음 10 cm
현무암 3.0 중간 7 cm
수화 광물 2.5 높음 (수소 함유) 5 cm
납 11.3 매우 높음 3 cm

6.6 품질 관리 및 분석

6.6.1 현장 분석 장비

정제 과정에서 실시간으로 금속 순도와 성분을 분석하는 것이 중요합니다.

  • XRF (X-Ray Fluorescence): X선으로 원소 조성 분석 (비파괴, 빠름)
  • 질량 분석기: 기체 샘플의 동위원소 비율 측정
  • 분광기: 가시광선~적외선 스펙트럼으로 광물 식별
  • 경도 시험기: 금속의 기계적 특성 평가

6.6.2 품질 표준

WIA-SPACE-027 표준은 우주에서 정제된 금속에 대한 다음 품질 기준을 제시합니다:

등급 순도 용도 가격 프리미엄
Space-Grade Platinum 99.99% 이상 전자기기, 촉매 +20%
Space-Grade Iron 99.5% 이상 구조재, 우주선 +10%
Space-Grade Nickel 99.8% 이상 합금, 배터리 +15%
💎 "우주 제조" 인증: WIA-SPACE-027 표준을 충족하는 금속은 "Space-Refined" 인증을 받을 수 있으며, 이는 최고 순도와 품질을 보증하는 프리미엄 브랜드로 지구 시장에서 거래됩니다.

6.7 폐기물 처리 및 재활용

6.7.1 슬래그 활용

정제 과정에서 발생하는 슬래그(slag, 부산물)는 폐기하지 않고 다음과 같이 활용할 수 있습니다:

  • 방사선 차폐재: 고밀도 슬래그를 우주 정거장 외벽에 적층
  • 레골리스 대체재: 소행성 표면에 다시 뿌려 채굴 흔적 복원
  • 3D 프린팅 충전재: 저가 구조물 제작에 활용

6.7.2 폐열 회수

정제 과정에서 발생하는 열은 다음 용도로 재활용됩니다:

  • 전력 생산: 열전 발전기(TEG)로 전기 생산
  • 물 증류: 폐열로 수증기 생산 및 정수
  • 거주 구역 난방: 우주 정거장 온도 유지

6.8 자동화 및 원격 제어

정제 시설은 완전 무인 자동화가 필수적입니다. 사람이 상주하기에는 방사선과 극한 온도가 위험하기 때문입니다.

// 자동 정제 공정 제어 시스템
class RefineryController {
    constructor() {
        this.sensors = {
            temperature: new TemperatureSensor(),
            purity: new XRFAnalyzer(),
            mass: new MassFlowMeter()
        };
        this.actuators = {
            furnace: new ElectricFurnace(),
            separator: new MagneticSeparator(),
            storage: new CryogenicTank()
        };
    }

    async processOre(oreType, mass) {
        // 1. 광석 유형 분석
        const composition = await this.sensors.purity.analyze(oreType);

        // 2. 최적 정제 온도 및 시간 결정
        const refineParams = this.optimizeParameters(composition);

        // 3. 용융 및 정제
        await this.actuators.furnace.setTemperature(refineParams.temp);
        await this.wait(refineParams.duration);

        // 4. 분리
        const metals = await this.actuators.separator.separate();

        // 5. 품질 검사
        for (const metal of metals) {
            const purity = await this.sensors.purity.analyze(metal);
            if (purity < 0.995) {
                await this.reprocess(metal); // 재정제
            } else {
                await this.actuators.storage.store(metal);
            }
        }

        return this.generateReport(metals);
    }
}
