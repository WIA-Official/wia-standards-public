제 3장

목표 소행성 선정: NEA와 델타-V

3.1 근지구 소행성 (NEA) 개요

근지구 소행성(Near-Earth Asteroids, NEA)은 지구 궤도에 접근하는 소행성으로, 태양으로부터 1.3 AU 이내의 근일점을 가집니다. 현재 약 34,000개 이상의 NEA가 확인되었으며, 매년 약 1,000개씩 새로 발견되고 있습니다.

NEA는 채굴 관점에서 매우 유리합니다. 주소행성대(2.2~3.2 AU)보다 지구에 가까워 왕복 시간과 연료 소모가 적고, 일부는 달보다 도달하기 쉬운 델타-V를 가지고 있습니다.

3.2 NEA의 궤도 분류

NEA는 궤도 특성에 따라 네 가지 그룹으로 분류됩니다:

3.2.1 아텐(Aten) 소행성

아텐 소행성은 반장축(semi-major axis)이 1.0 AU 미만이며, 원일점(aphelion)이 0.983 AU 이상입니다. 대부분 금성 궤도와 지구 궤도 사이를 공전합니다.

  • 대표 예시: 2062 Aten, 99942 Apophis
  • 특징: 짧은 공전 주기 (< 1년), 높은 근일점 속도
  • 채굴 난이도: 높음 (태양에 가까워 열 관리 어려움)

3.2.2 아폴로(Apollo) 소행성

아폴로 소행성은 반장축이 1.0 AU 이상이고, 근일점(perihelion)이 1.017 AU 미만입니다. 지구 궤도를 횡단하며, 가장 많은 수(약 16,000개)를 차지합니다.

  • 대표 예시: 1862 Apollo, 433 Eros, 25143 Itokawa
  • 특징: 지구 충돌 위험성 높음, 다양한 크기 분포
  • 채굴 난이도: 중간 (델타-V 변동폭 큼)

3.2.3 아모르(Amor) 소행성

아모르 소행성은 근일점이 1.017~1.3 AU 사이이며, 지구 궤도를 횡단하지 않지만 화성 궤도에 근접합니다.

  • 대표 예시: 1221 Amor, 433 Eros (일부 분류)
  • 특징: 지구 충돌 위험 낮음, 안정적인 궤도
  • 채굴 난이도: 낮음~중간 (비교적 안전한 접근)

3.2.4 아티라(Atira) 소행성

아티라 소행성은 반장축과 원일점이 모두 1.0 AU 미만으로, 완전히 지구 궤도 내부를 공전합니다. 매우 희귀하며, 현재 약 30개만 확인되었습니다.

  • 대표 예시: 163693 Atira, 2019 AQ3
  • 특징: 태양에 매우 가까움, 관측 어려움
  • 채굴 난이도: 매우 높음 (극심한 열 환경)

3.3 델타-V (Δv) 개념

델타-V는 우주선이 궤도를 변경하는 데 필요한 속도 변화량입니다. 델타-V가 낮을수록 연료 소모가 적고, 임무 비용이 감소합니다. 소행성 채굴에서는 델타-V가 가장 중요한 선정 기준 중 하나입니다.

🚀 델타-V 비교:
• 지구 표면 → 저궤도(LEO): 9.4 km/s
• LEO → 달 표면: 6.3 km/s
• LEO → 일부 NEA: 4.5~6.0 km/s (달보다 적음!)
• LEO → 화성: 6.5 km/s

3.3.1 호만 전이 궤도 (Hohmann Transfer)

호만 전이는 두 개의 원형 궤도 사이를 이동하는 가장 연료 효율적인 방법입니다. 타원 궤도를 이용하여 두 번의 가속만으로 목표에 도달합니다.

델타-V 계산 알고리즘
// Hohmann Transfer Delta-V 계산
function hohmannDeltaV(r1, r2, mu = 1.327e20) {
    // r1: 출발 궤도 반경 (m)
    // r2: 목표 궤도 반경 (m)
    // mu: 태양 중력 상수 (m³/s²)

    const v1 = Math.sqrt(mu / r1); // 출발 속도
    const vTransfer1 = Math.sqrt(mu * (2/r1 - 2/(r1+r2))); // 전이 시작 속도
    const deltaV1 = Math.abs(vTransfer1 - v1);

    const v2 = Math.sqrt(mu / r2); // 목표 속도
    const vTransfer2 = Math.sqrt(mu * (2/r2 - 2/(r1+r2))); // 전이 종료 속도
    const deltaV2 = Math.abs(v2 - vTransfer2);

    return {
        deltaV1: deltaV1 / 1000, // km/s
        deltaV2: deltaV2 / 1000, // km/s
        total: (deltaV1 + deltaV2) / 1000 // km/s
    };
}

// 예시: 지구(1 AU) → 소행성(1.5 AU)
const result = hohmannDeltaV(1.496e11, 1.5 * 1.496e11);
console.log(`Total Delta-V: ${result.total.toFixed(2)} km/s`);
// 출력: Total Delta-V: 2.94 km/s

3.3.2 저에너지 전이 (Low-Energy Transfer)

라그랑주 점(Lagrange Points)을 이용한 저에너지 전이는 델타-V를 더욱 절감할 수 있습니다. 특히 지구-달 L1/L2 점을 경유하면, NEA까지 델타-V를 30% 이상 줄일 수 있습니다.

3.4 목표 소행성 선정 기준

채굴 가치가 높은 소행성을 선정하기 위한 종합 평가 기준은 다음과 같습니다:

3.4.1 궤도 접근성

기준 우수 양호 보통
델타-V (왕복) < 6 km/s 6~8 km/s > 8 km/s
근일점 거리 0.9~1.1 AU 0.8~1.3 AU < 0.8 또는 > 1.3 AU
궤도 이심률 < 0.1 0.1~0.3 > 0.3
궤도 경사 < 5° 5~15° > 15°

3.4.2 물리적 특성

  • 크기: 최소 100m 이상 (너무 작으면 중력 부족, 너무 크면 채굴 어려움)
  • 회전 주기: 2~10시간 (너무 빠르면 원심력으로 채굴 어려움)
  • 표면 중력: 0.001~0.01 m/s² (로봇 고정 가능 수준)
  • 형상: 구형에 가까울수록 유리 (불규칙하면 착륙 어려움)

3.4.3 자원 가치

3.5 유력 채굴 후보 소행성

3.5.1 1999 JU3 (Ryugu) - 탐사 완료

Hayabusa2 미션을 통해 샘플이 귀환된 C형 소행성입니다. 물과 유기물이 풍부하며, 표면은 잔자갈(rubble pile)로 구성되어 있어 채굴이 비교적 용이합니다.

  • 크기: 직경 870m
  • 델타-V: 약 5.2 km/s (왕복)
  • 궤도: 아폴로형 (a = 1.19 AU, e = 0.19)
  • 자원: 수화 광물 10~15%, 유기 화합물 풍부

3.5.2 101955 Bennu - 탐사 완료

OSIRIS-REx 미션을 통해 샘플이 귀환된 C형 소행성입니다. 표면에 물 함유 광물이 풍부하며, 탄소질 콘드라이트 특성을 보입니다.

  • 크기: 직경 490m
  • 델타-V: 약 5.0 km/s (왕복)
  • 궤도: 아폴로형 (a = 1.13 AU, e = 0.20)
  • 자원: 물 0.5~5%, 유기물, 마그네타이트

3.5.3 2011 UW158 - 백금 풍부

스피처 우주 망원경 관측 결과, 백금족 금속이 풍부한 X형 소행성으로 추정됩니다. 표면 온도가 낮아 채굴 환경이 양호합니다.

  • 크기: 직경 약 300m
  • 델타-V: 약 7.4 km/s (왕복)
  • 추정 가치: $5조 이상 (백금 기준)
  • 궤도: 아폴로형

3.5.4 1986 DA - 대형 금속 소행성

레이더 관측 결과 M형으로 추정되는 대형 소행성입니다. 철-니켈 함량이 90% 이상으로 예상됩니다.

  • 크기: 직경 약 2km
  • 델타-V: 약 8.1 km/s (왕복)
  • 추정 가치: $17조 이상
  • 회전 주기: 2.12시간 (빠름, 주의 필요)

3.6 발사 창(Launch Window) 계산

소행성 미션은 지구와 소행성의 궤도 배치에 따라 특정 시기에만 발사할 수 있습니다. 발사 창은 보통 수년마다 한 번씩 열립니다.

발사 창 계산 (단순화 모델)
// 시노딕 주기 계산 (synodic period)
function synodicPeriod(T1, T2) {
    // T1: 지구 공전 주기 (년)
    // T2: 소행성 공전 주기 (년)
    return Math.abs(1 / (1/T1 - 1/T2));
}

// 예시: Bennu의 경우
const earthPeriod = 1.0; // 1년
const bennuPeriod = 1.2; // 약 1.2년
const synodic = synodicPeriod(earthPeriod, bennuPeriod);

console.log(`발사 창 간격: ${synodic.toFixed(2)}년`);
// 출력: 발사 창 간격: 6.00년

// 실제로는 궤도 이심률, 경사 등을 고려해야 함

3.7 위험 요소 평가

3.7.1 충돌 위험 소행성 (PHA)

일부 NEA는 지구 충돌 위험이 있는 잠재적 위험 소행성(Potentially Hazardous Asteroids, PHA)으로 분류됩니다. 이들은 지구 최소 궤도 교차 거리(MOID)가 0.05 AU 미만이고, 절대 등급(H)이 22.0 이하(직경 약 140m 이상)인 천체입니다.

PHA 채굴 시에는 궤도 변경에 따른 충돌 위험 증가 가능성을 반드시 평가해야 하며, 국제 협력과 투명한 정보 공유가 필수입니다.

3.7.2 우주 날씨

  • 태양 폭발(Solar Flare): 전자 장비 손상 위험
  • 우주 방사선: 태양에 가까울수록 방사선 수준 증가
  • 마이크로미티어로이드: 소행성 표면 충돌 시 먼지 발생
