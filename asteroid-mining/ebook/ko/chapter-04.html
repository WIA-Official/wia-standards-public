제 4장

소행성 탐사 기술

4.1 탐사선 설계

소행성 탐사선은 지구에서 수억 km 떨어진 곳에서 자율적으로 작동해야 합니다. 통신 지연(light delay)이 수십 분에 달하기 때문에, 실시간 제어가 불가능하며, 고도의 자율 시스템이 필수적입니다.

4.1.1 추진 시스템

소행성 미션에는 높은 비추력(specific impulse, Isp)을 가진 전기 추진 시스템이 주로 사용됩니다. 화학 로켓보다 연료 효율이 10배 이상 높아, 장거리 임무에 적합합니다.

추진 방식 비추력 (초) 추력 (N) 적용 미션
화학 로켓 (LH₂/LOX) 450 10,000+ 발사, 지구 탈출
이온 엔진 3,000~5,000 0.1~1.0 심우주 순항, Dawn, Hayabusa
홀 추력기 1,500~2,000 0.5~5.0 정거장 유지, Psyche 미션
태양 돛 (Solar Sail) 무제한 0.001~0.01 장기 미션, LightSail 2
🚀 이온 엔진 원리: 크세논(Xe) 또는 크립톤(Kr) 가스를 전리시켜 양이온을 생성하고, 전기장으로 가속하여 배출합니다. 배기 속도가 30~50 km/s에 달해 화학 로켓(4~5 km/s)보다 훨씬 효율적입니다.

4.1.2 전력 시스템

태양 전지판은 태양 거리에 따라 출력이 크게 달라집니다. 2 AU 거리에서는 지구 궤도 대비 25% 수준으로 감소하므로, 고효율 갈륨 비소(GaAs) 태양전지가 사용됩니다.

  • 근지구 소행성 (1.0~1.5 AU): 태양 전지판 (5~10 kW)
  • 주소행성대 (2.0~3.0 AU): 대형 태양 전지판 (10~20 kW) 또는 원자력 전지(RTG)
  • 외측 소행성대 (> 3.0 AU): 방사성 동위원소 열전기 발전기 (RTG) 필수

4.1.3 통신 시스템

지구와 소행성 간 통신은 Deep Space Network(DSN)를 통해 이루어집니다. 수신기 감도와 송신 출력이 매우 중요합니다.

  • X-band (8 GHz): 표준 심우주 통신, 데이터 전송률 1~10 Mbps
  • Ka-band (32 GHz): 고속 통신, 데이터 전송률 100+ Mbps
  • 레이저 통신: 미래 기술, Gbps급 전송 가능 (LCRD 실험 진행 중)

4.2 자율 항법 및 랑데부

소행성에 접근하는 것은 매우 정밀한 항법 기술을 요구합니다. 소행성은 크기가 작고(수백 m), 중력이 약해(0.001 m/s²) 정밀한 제어가 필수적입니다.

4.2.1 광학 항법 (Optical Navigation)

탐사선에 탑재된 카메라로 소행성을 촬영하고, 배경 별자리와 비교하여 상대 위치와 속도를 계산합니다. OSIRIS-REx와 Hayabusa2 모두 이 방법을 사용했습니다.

// 광학 항법 알고리즘 (단순화)
class OpticalNavigation {
    constructor(camera, starCatalog) {
        this.camera = camera;
        this.starCatalog = starCatalog;

    calculatePosition(image) {
        // 1. 이미지에서 소행성 중심 검출

4.2.2 레이저 고도계 (LIDAR)

레이저를 소행성 표면에 발사하고 반사된 시간을 측정하여 거리를 정밀하게 측정합니다. OSIRIS-REx는 OTES(OSIRIS-REx Thermal Emission Spectrometer)와 함께 LIDAR를 사용하여 Bennu 표면 3D 지도를 작성했습니다.

4.2.3 저중력 환경 랑데부

소행성의 중력은 지구의 100만 분의 1 수준입니다. 예를 들어, Bennu(직경 490m)의 표면 중력은 약 6×10⁻⁶ m/s²에 불과합니다. 이는 시속 1m로 이동해도 탈출 속도를 초과할 수 있음을 의미합니다.

4.3 표면 탐사 및 매핑

4.3.1 지형 매핑

소행성 표면의 3D 지형도를 작성하는 것은 착륙 지점 선정과 채굴 계획 수립에 필수적입니다. 다음 기술들이 복합적으로 사용됩니다:

  • Photogrammetry: 여러 각도에서 촬영한 이미지를 조합하여 3D 모델 생성
  • 레이저 고도계: 표면까지의 정밀한 거리 측정 (정확도 1 cm 이하)
  • 레이더 측심: 지하 구조 파악 (깊이 수십~수백 m)

4.3.2 분광 분석

소행성 표면의 광물 구성을 원격으로 분석하는 기술입니다. 태양빛 반사 스펙트럼을 분석하여 특정 광물의 존재를 확인합니다.

파장 범위 탐지 가능 물질 장비 예시
가시광선 (0.4~0.7 μm) 철, 티타늄, 크롬 PolyCam (OSIRIS-REx)
근적외선 (0.7~2.5 μm) 수화 광물, 감람석, 휘석 OVIRS (OSIRIS-REx)
중적외선 (5~50 μm) 규산염, 탄산염 OTES (OSIRIS-REx)
X선 (0.5~10 keV) 마그네슘, 알루미늄, 규소 REXIS (OSIRIS-REx)

4.3.3 중력장 측정

소행성 주변을 공전하면서 탐사선의 궤도 변화를 정밀하게 추적하면, 소행성의 질량 분포를 파악할 수 있습니다. 이는 내부 구조(단단한 암석 vs. 잔자갈 더미)를 추정하는 데 중요합니다.

🔍 Bennu의 놀라운 발견: OSIRIS-REx는 Bennu가 "잔자갈 더미(rubble pile)" 구조임을 확인했습니다. 표면은 수십~수백 cm 크기의 바위로 덮여 있으며, 내부는 틈이 많아 밀도가 낮습니다(1.19 g/cm³). 이는 채굴 시 표면이 쉽게 붕괴될 수 있음을 의미합니다.

4.4 샘플 채취 및 귀환

4.4.1 터치-앤-고 (Touch-and-Go, TAG)

OSIRIS-REx가 사용한 방법으로, 소행성 표면에 짧게(수 초) 접촉하여 샘플을 채취합니다. 질소 가스를 분사하여 표면 물질을 날려 올리고, 수집 헤드로 포획합니다.

  1. 하강 단계: 궤도에서 표면까지 4시간 동안 천천히 하강
  2. 접촉 단계: 3.3m 로봇 팔(TAGSAM)이 표면에 닿으면 질소 가스 분사
  3. 샘플 포획: 5~10초 동안 표면 물질 흡입 (목표: 60g 이상)
  4. 상승 단계: 역추진으로 표면에서 이탈, 안전 고도로 복귀

4.4.2 착륙 및 드릴링 (Hayabusa2)

Hayabusa2는 Ryugu 표면에 착륙하여 소형 프로젝타일(탄환)을 발사하고, 튀어오른 물질을 채취했습니다. 또한, 인공 크레이터를 만들어 지하 물질도 채취했습니다.

  • 1차 샘플: 표면 샘플 (풍화층)
  • 2차 샘플: 지하 2m 샘플 (구리 충돌체로 크레이터 생성 후 채취)
  • 채취량: 총 5.4g (목표 0.1g 대비 54배 초과 달성)

4.4.3 샘플 귀환 캡슐

채취한 샘플은 지구 대기권 재진입 시 고온(수천 도)에 견뎌야 하므로, 특수 열 차폐막이 적용된 캡슐에 보관됩니다.

미션 귀환 속도 최고 온도 착륙 정확도
Hayabusa (2010) 12.2 km/s 3,000°C 400 m (목표 대비)
Hayabusa2 (2020) 12.0 km/s 3,000°C 수십 m (개선됨)
OSIRIS-REx (2023) 12.4 km/s 2,760°C 10 km 이내 (계획)

4.5 AI 및 자율 시스템

4.5.1 실시간 의사결정

지구-소행성 간 통신 지연은 10~40분에 달합니다. 따라서 위급 상황(장애물 감지, 자세 불안정 등)에 대응하기 위해서는 온보드 AI가 필수적입니다.

// 자율 착륙 알고리즘 (개념)
class AutonomousLanding {
    async performLanding(targetSite) {
        let altitude = await this.sensors.getAltitude();

        while (altitude > 0.5) { // 0.5m까지 하강
            // 1. 실시간 지형 스캔
            const terrainMap = await this.lidar.scan360();

            // 2. 장애물 회피 판단
            const safePath = this.pathPlanning.findSafest(terrainMap);

            if (!safePath) {
                return this.abortLanding(); // 중단 및 상승
            }

            // 3. 추력 조정 (중력 보상 + 하강 속도 제어)
            const thrust = this.calculateThrust(altitude, this.velocity);
            await this.thrusters.fire(thrust);

            // 4. 자세 제어 (수평 유지)
            await this.attitudeControl.stabilize();

            altitude = await this.sensors.getAltitude();
        }

        // 최종 접촉
        return this.touchdownSequence();
    }
}

4.5.2 머신러닝을 이용한 지형 분류

탐사선은 수천 장의 이미지를 촬영합니다. 이를 실시간으로 분석하여 채굴 가능 지역, 위험 지역, 과학적 관심 지역을 자동 분류할 수 있습니다.

  • CNN (Convolutional Neural Network): 바위, 레골리스, 크레이터 분류
  • Semantic Segmentation: 이미지 픽셀 단위로 지형 특성 라벨링
  • Anomaly Detection: 비정상적인 지형(예: 얼음 노출, 금속 반사) 자동 탐지

4.6 국제 협력 및 데이터 공유

소행성 탐사는 비용이 막대하여 국제 협력이 필수적입니다. NASA, ESA, JAXA는 상호 데이터 공유 협정을 맺고 있으며, 탐사 데이터는 공개 후 12~24개월 내에 전 세계 연구자에게 공개됩니다.
