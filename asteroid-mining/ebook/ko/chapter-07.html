7.1 운송 전략 개요

소행성에서 정제한 자원을 지구나 우주 정거장으로 운송하는 것은 채굴 프로젝트의 최종 단계입니다. 운송 방법은 자원의 가치, 질량, 긴급도에 따라 달라집니다.

7.2 궤도 역학 기초

7.3.1 케플러 궤도 요소

소행성과 지구 사이의 최적 경로를 계산하려면 궤도 요소를 이해해야 합니다:

  • 반장축 (a): 타원 궤도의 크기
  • 이심률 (e): 궤도의 찌그러진 정도 (0 = 원, 1 = 포물선)
  • 궤도 경사 (i): 황도면과의 각도
  • 승교점 경도 (Ω): 궤도면의 방향
  • 근일점 인수 (ω): 타원의 장축 방향
  • 평균 근점 (M): 궤도상 위치

7.2.2 궤도 전이 계산

// Lambert 문제 풀이 (출발-도착 지점 간 최적 궤도)
class LambertSolver {
    constructor(mu = 1.327e20) {
        this.mu = mu; // 태양 중력 상수 (m³/s²)
    }

    solve(r1, r2, tof, prograde = true) {
        // r1, r2: 출발/도착 위치 벡터 (m)
        // tof: time of flight (초)
        // prograde: 순행 궤도 여부

        const r1Mag = this.magnitude(r1);
        const r2Mag = this.magnitude(r2);
        const cosTheta = this.dot(r1, r2) / (r1Mag * r2Mag);
        const theta = prograde ? Math.acos(cosTheta) : 2*Math.PI - Math.acos(cosTheta);

        // A 파라미터
        const A = Math.sin(theta) * Math.sqrt(r1Mag * r2Mag / (1 - cosTheta));

        // 반복법으로 z 파라미터 찾기
        const z = this.findZ(A, tof, r1Mag, r2Mag);

        // 속도 벡터 계산
        const f = 1 - r2Mag / A * (1 - Math.cos(Math.sqrt(z)));
        const g = r2Mag * r1Mag * Math.sin(Math.sqrt(z)) / Math.sqrt(this.mu * A);
        const v1 = this.subtract(r2, this.multiply(r1, f)).map(x => x / g);

        return {
            v1: v1, // 출발 속도 (m/s)
            deltaV: this.magnitude(v1), // 델타-V (m/s)
            energy: this.magnitude(v1) ** 2 / 2 - this.mu / r1Mag // 궤도 에너지
        };
    }

    magnitude(v) {
        return Math.sqrt(v.reduce((sum, x) => sum + x**2, 0));
    }

    dot(a, b) {
        return a.reduce((sum, x, i) => sum + x * b[i], 0);
    }

    subtract(a, b) {
        return a.map((x, i) => x - b[i]);
    }

    multiply(v, scalar) {
        return v.map(x => x * scalar);
    }

    findZ(A, tof, r1, r2) {
        // Newton-Raphson 반복법
        let z = 0;
        for (let i = 0; i < 50; i++) {
            const S = this.stumpffS(z);
            const C = this.stumpffC(z);
            const y = r1 + r2 - A * (1 - z * S) / Math.sqrt(C);
            const t = (y**1.5 * S + A * Math.sqrt(y)) / Math.sqrt(this.mu);
            const dt_dz = this.derivativeT(z, A, y);
            z = z - (t - tof) / dt_dz;
            if (Math.abs(t - tof) < 1e-6) break;
        }
        return z;
    }

    stumpffS(z) {
        if (z > 0) {
            const sqrtz = Math.sqrt(z);
            return (sqrtz - Math.sin(sqrtz)) / (sqrtz ** 3);
        } else if (z < 0) {
            const sqrtz = Math.sqrt(-z);
            return (Math.sinh(sqrtz) - sqrtz) / (sqrtz ** 3);
        } else {
            return 1/6;
        }
    }

    stumpffC(z) {
        if (z > 0) return (1 - Math.cos(Math.sqrt(z))) / z;
        else if (z < 0) return (Math.cosh(Math.sqrt(-z)) - 1) / (-z);
        else return 0.5;
    }

    derivativeT(z, A, y) {
        const C = this.stumpffC(z);
        const S = this.stumpffS(z);
        return (3 * S * Math.sqrt(y) + A / 2 * (C / y + 3 * S / (2 * y))) / Math.sqrt(this.mu);
    }
}

// 예시 사용
const solver = new LambertSolver();
const r1 = [1.496e11, 0, 0]; // 지구 위치 (x, y, z)
const r2 = [2.0e11, 1.0e11, 0]; // 소행성 위치
const tof = 180 * 24 * 3600; // 180일 (초)
const result = solver.solve(r1, r2, tof);
console.log(`Delta-V: ${(result.deltaV/1000).toFixed(2)} km/s`);

7.3 운송 수단

7.3.1 전기 추진 화물선

대량의 금속을 운송하는 데는 전기 추진(이온 엔진, 홀 추력기)이 가장 효율적입니다. 추력은 약하지만, 장기간 연속 작동으로 높은 델타-V를 달성할 수 있습니다.

항목 사양 비고
적재량 100~500톤 정제된 금속 기준
추진제 크세논 (Xe) 또는 크립톤 (Kr) 소행성에서 채취 불가, 지구 공급 필요
비추력 3000~5000초 화학 로켓의 10배
추력 0.5~5 N 낮지만 장기간 작동
전력 50~200 kW 태양 전지판 또는 원자력
운항 시간 12~36개월 NEA → 지구 기준

7.3.2 태양 돛 (Solar Sail)

추진제가 필요 없는 태양 돛은 장기 운송에 이상적입니다. 태양 복사압을 이용하여 가속하므로, 연료 고갈 걱정이 없습니다.

  • 돛 면적: 1000~10,000 m² (초경량 반사막)
  • 추력: 약 0.01~0.1 N (태양 거리 1 AU 기준)
  • 장점: 무제한 작동, 추진제 불필요
  • 단점: 가속도 매우 낮음, 태양에서 멀어질수록 효율 감소

7.3.3 화학 로켓 (빠른 귀환)

고가의 백금족 금속이나 희토류 원소는 빠른 운송이 경제적입니다. 화학 로켓은 추력이 강해 운항 시간을 6~12개월로 단축할 수 있습니다.

🚀 하이브리드 전략: 초기 가속은 화학 로켓으로 빠르게 수행하고, 순항 단계는 전기 추진으로 효율을 높이는 하이브리드 방식이 유력합니다. 이는 SpaceX의 Starship과 같은 재사용 로켓과 결합될 수 있습니다.

7.4 대기권 재진입

7.4.1 재진입 궤적 설계

소행성에서 귀환하는 캡슐은 지구 대기권에 초속 11~13 km로 진입합니다. 이는 아폴로 우주선보다 빠르며, 극심한 열에 노출됩니다.

단계 고도 (km) 속도 (km/s) 온도 (°C)
진입 시작 120 12.5 1,500
최대 가열 70~80 10 3,000
최대 감속 40~50 5 2,000
낙하산 전개 10 0.3 500
착륙 0 0.005 50

7.4.2 열 차폐 시스템

재진입 캡슐은 최대 3000°C의 열에 견뎌야 하므로, 다층 열 차폐막이 필수적입니다.

  • PICA (Phenolic Impregnated Carbon Ablator): SpaceX Dragon에 사용, 탄화로 열 흡수
  • Avcoat: NASA Orion 캡슐에 사용, 에폭시 수지 기반
  • 탄소-탄소 복합재: Space Shuttle 날개 선단에 사용, 재사용 가능

7.4.3 정밀 착륙

귀환 캡슐은 사막이나 바다 같은 안전한 지역에 착륙해야 합니다. GPS와 추력 제어로 착륙 오차를 10 km 이내로 줄일 수 있습니다.

7.5 우주 정거장 및 라그랑주 포인트 활용

7.5.1 지구-달 L1/L2 포인트 저장소

소행성 자원을 지구로 직접 운송하는 대신, 지구-달 라그랑주 포인트 L1 또는 L2에 저장하는 전략이 있습니다. 이곳은 중력이 균형을 이루어 적은 연료로 위치를 유지할 수 있습니다.

  • L1 (지구-달 사이): 지구에서 약 32만 km, 달에서 약 6만 km
  • L2 (달 뒤편): 달에서 약 6만 km, 심우주 관측 기지로 활용
  • 장점: 지구/달 양방향 접근 용이, 안정적인 궤도
  • 용도: 우주 연료 저장, 건설 자재 보관, 중계 기지

7.5.2 궤도상 조립 및 가공

대형 우주선이나 우주 정거장을 지구에서 발사하는 것은 비용이 막대합니다. 대신, 소행성 금속을 L1/L2 포인트로 운송하여 현지에서 조립하면 훨씬 경제적입니다.

🛰️ 미래 비전: 우주 조선소
2040년대에는 L2 포인트에 대형 우주 조선소가 건설될 전망입니다. 소행성 철로 선체를 제작하고, 달 헬륨-3으로 핵융합 엔진을 가동하는 심우주 탐사선이 건조될 것입니다.

7.6 물류 최적화

7.6.1 발사 창 최적화

지구와 소행성의 궤도 배치는 시간에 따라 변하므로, 최적의 발사 시기를 계산하는 것이 중요합니다. Pork-chop plot을 이용하여 델타-V가 최소인 발사일과 도착일을 찾습니다.

// Pork-chop plot 생성 (단순화)
function generatePorkChopPlot(startDate, endDate, asteroid) {
    const results = [];
    for (let launch = startDate; launch <= endDate; launch += 86400) {
        for (let arrival = launch + 6*30*86400; arrival <= launch + 24*30*86400; arrival += 86400) {
            const deltaV = calculateDeltaV(launch, arrival, asteroid);
            results.push({
                launch: new Date(launch * 1000),
                arrival: new Date(arrival * 1000),
                deltaV: deltaV,
                tof: (arrival - launch) / 86400 // 일
            });
        }
    }
    return results.sort((a, b) => a.deltaV - b.deltaV).slice(0, 10); // 상위 10개
}

function calculateDeltaV(launchDate, arrivalDate, asteroid) {
    const earthPos = getEarthPosition(launchDate);
    const asteroidPos = getAsteroidPosition(arrivalDate);
    const tof = arrivalDate - launchDate;

    const solver = new LambertSolver();
    const result = solver.solve(earthPos, asteroidPos, tof);
    return result.deltaV / 1000; // km/s
}

// 예시: 2026~2030년 범위에서 최적 발사 창 찾기
const windows = generatePorkChopPlot(
    Date.parse('2026-01-01') / 1000,
    Date.parse('2030-12-31') / 1000,
    'Bennu'
);
console.log('최적 발사 창:', windows[0]);
// 출력: 최적 발사 창: {launch: 2027-09-15, arrival: 2029-03-20, deltaV: 5.2 km/s, tof: 552일}

7.6.2 다중 소행성 순회 임무

단일 우주선이 여러 소행성을 순회하며 채굴하는 전략도 가능합니다. 이는 초기 투자를 분산시키고, 다양한 자원을 확보하는 이점이 있습니다.

  • 방문 순서 최적화: Traveling Salesman Problem (TSP) 알고리즘 적용
  • 궤도 공명 활용: 소행성 간 중력 도움(Gravity Assist) 활용
  • 예상 임무 기간: 5~10년 (3~5개 소행성 방문)

7.7 안전 및 규제

7.7.1 우주 교통 관리

소행성 운송선은 국제 우주 교통 관리(Space Traffic Management, STM) 시스템에 등록하고, 궤도 정보를 실시간으로 공유해야 합니다.

  • 궤도 신고: 발사 30일 전까지 궤도 요소 제출
  • 실시간 추적: GPS/GLONASS로 위치 전송 (매 1시간)
  • 충돌 회피: 다른 위성과 최소 10 km 이격 유지
  • 비상 기동: 충돌 위험 시 24시간 내 궤도 변경 능력 보유

7.7.2 재진입 안전

대기권 재진입 시 캡슐이 파손되면 금속 파편이 지상에 떨어질 위험이 있습니다. WIA-SPACE-027 표준은 다음을 의무화합니다:

  1. 재진입 궤적은 인구 밀집 지역 회피 (바다 또는 사막 상공)
  2. 캡슐 추적 시스템 (GPS + 비콘) 탑재
  3. 재진입 7일 전 국제 통보 (NOTAM 발행)
  4. 회수팀 대기 (착륙 후 12시간 내 회수)
⚠️ 비상 계획: 재진입 실패 시 캡슐은 자동으로 안전 궤도(300~500 km)로 상승하여 회수 미션을 대기합니다. 모든 귀환 캡슐은 최소 30일간 궤도 유지 능력을 보유해야 합니다.
