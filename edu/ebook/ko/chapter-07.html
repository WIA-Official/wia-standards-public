// EDU 프로필에서 AAC 프로필 참조

  "wia_integrations": {
    "aac_profile_id": "AAC-2025-EF56-GH78",

      "enabled": true,

7.6 장 요약

핵심 내용:

  1. 생태계: WIA AAC, BCI, Eye Gaze와 통합된 교육 접근성
  2. AAC 통합: 어휘 적응, 의사소통 보드, 응답 형식
  3. BCI 통합: 뇌파 제어, 피로 모니터링, 인지 적응
  4. Eye Gaze 통합: 시선 탐색, 보정 전송, UI 적응
  5. 통합 관리: 중앙화된 프로필, 실시간 동기화
