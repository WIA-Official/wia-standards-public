## 1. 개요

### 1.1 목적

WIA-ENE-036 석유/가스 시추 표준은 전 세계 석유 및 천연가스 시추 작업의 안전하고 환경 친화적인 운영을 위한 포괄적인 프레임워크를 제공합니다. 본 표준은 시추 운영, 저류층 관리, 생산율 모니터링, 환경 영향 완화, 메탄 배출 관리, 유출 방지, 폐쇄 절차에 대한 데이터 형식, 프로토콜, 안전 지표를 정의합니다.

### 1.2 적용 범위

본 표준은 다음 영역에 적용됩니다:

- **탐사 시추**: 지질 조사, 시험 시추
- **개발 시추**: 생산정, 주입정 개발
- **생산 운영**: 원유/가스 추출, 처리
- **해양 플랫폼**: 고정식/부유식 플랫폼 운영
- **비전통 자원**: 셰일 오일/가스, 타이트 샌드
- **환경 모니터링**: 배출, 유출 감지
- **폐쇄 및 복원**: 우물 폐쇄, 부지 복원
- **규제 준수**: 정부 및 국제 기준

### 1.3 핵심 원칙

1. **안전 최우선**: 작업자 및 지역사회 안전 보호
2. **환경 보호**: 생태계 영향 최소화
3. **메탄 배출 저감**: 온실가스 배출 관리
4. **투명성**: 운영 데이터 공개 및 보고
5. **지속 가능성**: 장기적 환경 책임

## 2. 유정 분류 체계

### 2.1 시추 목적별 분류

#### 2.1.1 탐사정 (Exploration Wells)

| 코드 | 분류 | 목적 | 시추 깊이 | 일반적 기간 |
|------|------|------|----------|------------|
| EXP-1 | 와일드캣 | 미탐사 지역 평가 | 1,000-6,000m | 30-90일 |
| EXP-2 | 평가정 | 발견된 저류층 평가 | 2,000-5,000m | 30-60일 |
| EXP-3 | 지층 시험정 | 지질 구조 확인 | 500-3,000m | 15-30일 |

**주요 활동**:
- 지진 탐사 데이터 검증
- 지층 샘플링 및 분석
- 저류층 유체 테스트
- 상업성 평가

#### 2.1.2 생산정 (Production Wells)

| 코드 | 분류 | 유형 | 생산 방식 | 예상 수명 |
|------|------|------|----------|----------|
| PROD-1 | 수직정 | 수직 시추 | 자분 또는 인공채유 | 15-30년 |
| PROD-2 | 경사정 | 경사 시추 | 다중 저류층 접촉 | 20-40년 |
| PROD-3 | 수평정 | 수평 구간 포함 | 셰일/타이트 저류층 | 5-15년 |
| PROD-4 | 다중 분지정 | 여러 저류층 접촉 | 복합 생산 | 20-35년 |

**주요 설비**:
- 정두 장치 (Wellhead)
- 크리스마스 트리 (Christmas Tree)
- 유량 제어 시스템
- 하부 안전 밸브 (SSSV)

#### 2.1.3 주입정 (Injection Wells)

| 코드 | 분류 | 주입 물질 | 목적 | 압력 범위 |
|------|------|----------|------|----------|
| INJ-1 | 수공법정 | 물 | 저류층 압력 유지 | 150-350 bar |
| INJ-2 | 가스 재주입정 | 천연가스, CO₂ | 압력 유지, 회수율 증대 | 200-400 bar |
| INJ-3 | 화학 주입정 | 폴리머, 계면활성제 | 증진 회수 (EOR) | 100-250 bar |
| INJ-4 | 폐수 처리정 | 생산수 | 환경 보호 | 50-200 bar |

### 2.2 시추 방법별 분류

#### 2.2.1 전통적 시추

```
회전식 시추 (Rotary Drilling):
- 회전 테이블 또는 탑 드라이브
- 드릴 비트: PDC, 롤러 콘, 다이아몬드
- 시추 유체: 수성, 유성, 합성
- 깊이: 지상 ~ 10,000m+
```

#### 2.2.2 방향성 시추

```
Directional Drilling:
- MWD (Measurement While Drilling)
- LWD (Logging While Drilling)
- 스티어러블 모터
- RSS (Rotary Steerable System)
- 정확도: ±3m (목표 지점)
```

#### 2.2.3 수압 파쇄 (Hydraulic Fracturing)

```
Fracking 작업:
- 다단계 파쇄 (Multi-stage Fracking)
- 지지재: 세라믹, 모래
- 유체: 물 + 화학 첨가제 (2-5%)
- 압력: 400-1,000 bar
- 유량: 10-20 m³/min
```

## 3. 저류층 특성 데이터

### 3.1 저류층 유형

| 저류층 타입 | 암석 특성 | 공극률 | 투과도 | 회수율 |
|-----------|----------|--------|--------|--------|
| 사암 (Sandstone) | 쇄설성 | 15-30% | 10-1,000 mD | 30-50% |
| 탄산염암 (Carbonate) | 석회암, 백운암 | 5-20% | 1-500 mD | 20-40% |
| 셰일 (Shale) | 세립질 | 2-10% | 0.001-0.1 mD | 5-15% |
| 타이트 샌드 (Tight Sand) | 저투과 사암 | 8-15% | 0.01-1 mD | 10-25% |

### 3.2 저류층 유체

**원유 (Crude Oil)**

```
분류 체계:
- API 비중 (API Gravity)
  - 경질유 (Light): 31.1°-50° API
  - 중질유 (Medium): 22.3°-31.1° API
  - 중질유 (Heavy): 10°-22.3° API
  - 초중질유 (Extra Heavy): < 10° API

물리적 특성:
- 점도: 1-10,000+ cP
- 온도: 50-150°C (저류층)
- 압력: 100-600 bar
- 유황 함량: 0.1-5.0%
```

**천연가스 (Natural Gas)**

```
성분:
- 메탄 (CH₄): 70-95%
- 에탄 (C₂H₆): 1-10%
- 프로판 (C₃H₈): 1-5%
- 부탄 (C₄H₁₀): 0.5-2%
- CO₂, N₂, H₂S: 미량 ~ 20%

분류:
- 건성 가스 (Dry Gas): 메탄 > 90%
- 습성 가스 (Wet Gas): 액상 탄화수소 포함
- 산성 가스 (Sour Gas): H₂S > 4 ppm
- 비산성 가스 (Sweet Gas): H₂S < 4 ppm
```

**방제 시스템**

| 시스템 | 기능 | 용량 | 점검 주기 |
|--------|------|------|----------|
| 2차 격납 (Secondary Containment) | 탱크 누출 차단 | 150 m³ | 월간 |
| 유흡착 붐 (Oil Boom) | 수면 유출 차단 | 500 m | 연간 |
| 유 스키머 (Skimmer) | 수면 유 회수 | 50 m³/hr | 반기별 |
| 유흡착재 (Sorbent) | 소규모 유출 흡수 | 1,000 kg | 연간 교체 |

**대응 절차**

```
Tier 1 (소규모 유출: < 10 bbl):
- 즉시 작업 중단
- 유출원 차단
- 격납 시스템 활용
- 현장 대응팀 처리
- 사고 보고 (24시간 이내)

Tier 2 (중규모 유출: 10-100 bbl):
- Tier 1 절차 실행
- 지역 비상 대응팀 동원
- 규제 기관 즉시 통보
- 환경 영향 평가
- 복원 계획 수립

Tier 3 (대규모 유출: > 100 bbl):
- Tier 2 절차 실행
- 국가 비상 대응 체계 활성화
- 전문 방제 업체 동원
- 주민 대피 검토
- 지속적 모니터링 및 복원
```

## 6. 시추 안전 프로토콜

### 6.1 Well Control (유정 제어)

**Kick Detection (분출 조짐 감지)**

```
감지 지표:
1. 순환 유량 증가 (Flow Rate Increase)
   - 임계값: > 5% 증가
   - 확인: 유출구 유량계

2. 피트 용적 증가 (Pit Volume Gain)
   - 임계값: > 10 bbl 증가
   - 확인: 자동 레벨 센서

3. 펌프 압력 감소 (Pump Pressure Decrease)
   - 임계값: > 50 psi 감소
   - 확인: 스탠드파이프 압력계

4. 드릴 파이프 압력 변화
   - 연속 모니터링
   - 이상 패턴 알람

대응 절차:
1. 순환 중단 (Stop Drilling)
2. 스트링 들어올리기 (Pick Up Kelly)
3. 유량 확인 (Check for Flow)
4. BOP 폐쇄 (Close BOP if confirmed)
5. Well Kill 절차 실행
```

### 6.3 H₂S 안전 관리

**황화수소 검출 및 보호**

```
H₂S 농도 수준:
- 안전 (Safe): < 10 ppm
- 경계 (Caution): 10-20 ppm
- 위험 (Danger): 20-100 ppm
- 즉시 대피 (IDLH): > 100 ppm

필수 장비:
1. 개인용:
   - H₂S 검지기 (개인)
   - 자급식 호흡기 (SCBA)
   - 비상 탈출용 호흡기 (EEBD)

2. 시설용:
   - 고정식 H₂S 검지 시스템
   - 바람 방향 지시계
   - 비상 샤워 및 세안 설비
   - 응급 의료 장비

대응 절차:
- 10 ppm 이상: 호흡 보호구 착용 의무
- 20 ppm 이상: 작업 중단, 환기
- 100 ppm 이상: 즉시 대피, 비상 대응팀 출동
```

## 7. 폐쇄 및 복원 (Decommissioning)

### 7.1 우물 폐쇄 (Well Abandonment)

**영구 폐쇄 절차**

```
Phase 1: 계획 및 준비
- 우물 상태 평가
- 폐쇄 설계 (Plug & Abandonment Design)
- 규제 승인 획득
- 장비 및 자재 준비

Phase 2: 우물 조건화
- 생산 튜빙 회수
- 유체 순환 및 청소
- 케이싱 검사 (Casing Integrity Log)
- 천공 확인

Phase 3: 시멘트 플러그 설치
플러그 위치:
1. 생산 구간 격리 (Reservoir Isolation)
   - 깊이: 2,800-3,100m
   - 시멘트 길이: 100m
   - 시멘트 강도: > 35 MPa

2. 중간 플러그 (Intermediate Plugs)
   - 각 케이싱 슈 위/아래
   - 최소 길이: 30m

3. 지표 플러그 (Surface Plug)
   - 깊이: 0-15m
   - 지표면 아래 1m 까지

Phase 4: 검증 및 승인
- 플러그 압력 시험
- 태그 시험 (Tag Test)
- 최종 검사
- 규제 기관 승인

Phase 5: 지상 시설 철거
- 정두 장치 제거
- 케이싱 절단 (지표 아래 2m)
- 시멘트로 매립
- 마커 설치
```

## 8. API 인터페이스

### 8.1 RESTful API 엔드포인트

**Base URL**: `https://api.wia.org/ene-036/v1`

#### 8.1.1 유정 관리

**POST /wells**
- **설명**: 새 유정 등록
- **권한**: 인증된 운영사
- **요청 본문**: Well 객체
- **응답**: 201 Created + wellId

**GET /wells/{wellId}**
- **설명**: 유정 상세 정보 조회
- **응답**: 200 OK + Well 객체

**GET /wells**
- **설명**: 유정 목록 조회 (필터링/페이징)
- **쿼리 파라미터**:
  - `operatorId`: 운영사 ID
  - `wellType`: 유정 타입
  - `state`: 주/도
  - `status`: 운영 상태
  - `page`, `limit`: 페이징

**PUT /wells/{wellId}/status**
- **설명**: 유정 상태 업데이트
- **요청 본문**: { "status": "producing" | "shut_in" | "abandoned" }

#### 8.1.2 생산 데이터

**POST /production/daily**
- **설명**: 일일 생산 데이터 제출
- **요청 본문**: ProductionRecord 객체

**GET /production/daily**
- **설명**: 생산 데이터 조회
- **쿼리 파라미터**:
  - `wellId`: 유정 ID
  - `startDate`, `endDate`: 기간
  - `aggregation`: 'daily' | 'monthly' | 'yearly'

**GET /production/forecast**
- **설명**: 생산 예측 (ML 기반)
- **쿼리 파라미터**:
  - `wellId`: 유정 ID
  - `forecastPeriod`: 예측 기간 (개월)

#### 8.1.3 환경 모니터링

**POST /environmental/emissions**
- **설명**: 배출 데이터 제출
- **요청 본문**: EmissionsData 객체

**GET /environmental/emissions**
- **설명**: 배출 데이터 조회 및 분석

**POST /environmental/spills**
- **설명**: 유출 사고 보고
- **긴급도**: 즉시
- **요청 본문**: SpillIncident 객체

**GET /environmental/compliance**
- **설명**: 규제 준수 현황 조회

#### 8.1.4 안전 모니터링

**POST /safety/incidents**
- **설명**: 안전 사고 보고

**GET /safety/bop-tests**
- **설명**: BOP 시험 기록 조회

**POST /safety/h2s-alerts**
- **설명**: H₂S 검출 알람

### 8.2 웹훅 이벤트

**지원 이벤트**:
- `well.created`: 새 유정 생성
- `well.status_changed`: 유정 상태 변경
- `production.threshold`: 생산량 임계값
- `emissions.alarm`: 배출 알람
- `spill.reported`: 유출 사고 보고
- `safety.incident`: 안전 사고

| 국가/지역 | 규제 기관 | 주요 규제 |

| 🇺🇸 미국 | EPA, State Commissions | Clean Air Act, Clean Water Act |

| 🇺🇸 연방 | Bureau of Safety and Environmental Enforcement (BSEE) | 해양 작업 안전 |
| 🇨🇦 캐나다 | Alberta Energy Regulator (AER) | 시추 및 생산 규제 |
| 🇳🇴 노르웨이 | Norwegian Petroleum Directorate | 해양 안전 및 환경 |
| 🇸🇦 사우디 | Saudi Aramco Standards | 기술 표준 |

### 9.2 보고 요구사항

**생산 보고**
- 빈도: 월간
- 항목: 원유, 가스, 물 생산량
- 기한: 다음 달 15일까지

**배출 보고**
- 빈도: 분기별
- 항목: 메탄, VOCs, CO₂
- 기한: 분기 종료 후 45일

**안전 사고**
- 빈도: 즉시
- 항목: 분출, 화재, 부상
- 기한: 24시간 이내
