웨어러블 센서 기술

웨어러블 센서는 환자의 움직임, 활동, 생체 신호를 실시간으로 수집합니다. 클리닉과 가정 모두에서 사용 가능하여 연속적인 모니터링이 가능합니다.

1. 관성 측정 장치(IMU)

IMU는 가속도계, 자이로스코프, 자력계를 결합하여 3D 움직임을 추적합니다.

센서 유형 측정 값 응용
가속도계 선형 가속도 (m/s²) 활동 인식, 낙상 감지
자이로스코프 각속도 (°/s) 관절 각도, 회전 속도
자력계 자기장 방향 절대 방향 추정
IMU 데이터 처리 (Python)
# IMU 센서 데이터 융합 및 자세 추정
import numpy as np
from scipy.spatial.transform import Rotation

class IMUProcessor:
    def __init__(self, sampling_rate=100):
        self.dt = 1.0 / sampling_rate
        self.orientation = Rotation.identity()
        self.position = np.zeros(3)
        self.velocity = np.zeros(3)

    def update(self, accel, gyro, mag):
        # 자이로스코프로 방향 업데이트 (적분)
        rotation_delta = Rotation.from_rotvec(gyro * self.dt)
        self.orientation = self.orientation * rotation_delta

        # 중력 보정
        gravity = self.orientation.apply([0, 0, -9.81])
        accel_world = self.orientation.apply(accel) - gravity

        # 속도 및 위치 업데이트 (이중 적분)
        self.velocity += accel_world * self.dt
        self.position += self.velocity * self.dt

        return {
            'orientation': self.orientation.as_euler('xyz', degrees=True),
            'position': self.position,
            'velocity': self.velocity
        }

    def get_joint_angle(self, proximal_imu, distal_imu):
        # 두 IMU 사이의 상대 각도 계산
        relative_rotation = proximal_imu.inv() * distal_imu
        angle = relative_rotation.magnitude() * 180 / np.pi
        return angle

2. 근전도(EMG) 센서

근전도는 근육 활동을 측정하여 근육 활성화 패턴, 피로도, 움직임 의도를 파악합니다.

  • 표면 EMG: 피부 표면에 부착, 비침습적, 큰 근육군에 적합
  • 무선 EMG: 블루투스 통신, 자유로운 움직임 가능
  • 고밀도 EMG: 다수의 전극 배열, 상세한 근육 지도
EMG 신호 분석 (Python)
# EMG 신호 처리 및 근육 활성화 분석
import numpy as np
from scipy.signal import butter, filtfilt

class EMGAnalyzer:
    def __init__(self, sampling_rate=1000):
        self.fs = sampling_rate

    def preprocess(self, emg_signal):
        # 밴드패스 필터 (20-450 Hz)
        b, a = butter(4, [20, 450], btype='bandpass', fs=self.fs)
        filtered = filtfilt(b, a, emg_signal)

        # 노치 필터 (60 Hz, 전원 잡음 제거)
        b_notch, a_notch = butter(2, [58, 62], btype='bandstop', fs=self.fs)
        filtered = filtfilt(b_notch, a_notch, filtered)

        return filtered

    def calculate_rms(self, emg_signal, window_size=200):
        # RMS (Root Mean Square) 계산
        squared = emg_signal ** 2
        rms = np.sqrt(np.convolve(squared, np.ones(window_size)/window_size,
                                   mode='same'))
        return rms

    def detect_muscle_activation(self, emg_signal, threshold=0.1):
        # 근육 활성화 감지
        filtered = self.preprocess(emg_signal)
        rms = self.calculate_rms(filtered)

        # 임계값 초과 구간 찾기
        activations = rms > threshold
        onset_times = np.where(np.diff(activations.astype(int)) == 1)[0]
        offset_times = np.where(np.diff(activations.astype(int)) == -1)[0]

        return onset_times, offset_times

    def calculate_fatigue_index(self, emg_signal):
        # 피로 지수: 중간 주파수의 시간적 감소
        window_duration = 1.0  # 1초 윈도우
        window_samples = int(window_duration * self.fs)

        median_frequencies = []
        for i in range(0, len(emg_signal) - window_samples, window_samples):
            window = emg_signal[i:i+window_samples]
            fft = np.fft.rfft(window)
            power = np.abs(fft) ** 2
            freqs = np.fft.rfftfreq(len(window), 1/self.fs)

            # 중간 주파수 계산
            cumsum = np.cumsum(power)
            median_freq = freqs[np.where(cumsum >= cumsum[-1]/2)[0][0]]
            median_frequencies.append(median_freq)

        # 선형 회귀로 피로 기울기 계산
        slope = np.polyfit(range(len(median_frequencies)),
                          median_frequencies, 1)[0]
        fatigue_index = -slope  # 음의 기울기 = 피로

        return fatigue_index

1. 활동 인식(Activity Recognition)

머신러닝을 사용하여 센서 데이터에서 환자의 활동(걷기, 서기, 앉기, 누워있기 등)을 자동 분류합니다.

활동 인식 분류기 (Python)
# 딥러닝 기반 활동 인식
import tensorflow as tf
from tensorflow.keras import layers, models

class ActivityRecognizer:
    def __init__(self, num_classes=6):
        self.model = self.build_model(num_classes)
        self.activity_labels = [
            'walking', 'standing', 'sitting',
            'lying', 'stairs_up', 'stairs_down'
        ]

    def build_model(self, num_classes):
        # 1D CNN 모델
        model = models.Sequential([
            layers.Input(shape=(200, 6)),  # 200 샘플, 6축 센서
            layers.Conv1D(64, 3, activation='relu'),
            layers.MaxPooling1D(2),
            layers.Conv1D(128, 3, activation='relu'),
            layers.MaxPooling1D(2),
            layers.Flatten(),
            layers.Dense(128, activation='relu'),
            layers.Dropout(0.5),
            layers.Dense(num_classes, activation='softmax')
        ])
        model.compile(optimizer='adam',
                     loss='categorical_crossentropy',
                     metrics=['accuracy'])
        return model

    def predict_activity(self, sensor_data):
        # 센서 데이터로 활동 예측
        prediction = self.model.predict(sensor_data)
        activity_index = np.argmax(prediction)
        confidence = prediction[0][activity_index]

        return self.activity_labels[activity_index], confidence

2. 보행 분석

보행 매개변수를 자동으로 추출하여 보행 패턴을 정량화합니다.

매개변수 정의 정상 범위
보행 속도 단위 시간당 이동 거리 1.0-1.5 m/s
보폭(Stride Length) 한 발에서 같은 발까지 거리 1.2-1.5 m
케이던스(Cadence) 분당 걸음 수 100-120 steps/min
입각기 시간 발이 지면에 닿는 시간 보행 주기의 60%
보행 주기 시간 한 보행 주기의 총 시간 1.0-1.2 초

진행 상황 모니터링 및 대시보드

치료사와 환자가 재활 진행 상황을 쉽게 이해할 수 있도록 직관적인 시각화가 필요합니다.

대시보드 핵심 요소

  1. 진행 상황 차트: 시간에 따른 주요 지표 변화
  2. 목표 달성 현황: 주간/월간 목표 대비 실제 성과
  3. 운동 준수율: 처방된 운동 수행 비율
  4. 통증 및 피로도 추적: 환자 보고 결과(PRO)
  5. 비교 분석: 건강한 측 vs 손상된 측
대시보드 데이터 API (Python Flask)
# 재활 진행 상황 API 서버
from flask import Flask, jsonify
import sqlite3

app = Flask(__name__)

@app.route('/api/patient/<patient_id>/progress')
def get_patient_progress(patient_id):
    # 데이터베이스에서 환자 진행 상황 조회
    conn = sqlite3.connect('rehab_data.db')
    cursor = conn.cursor()

    cursor.execute('''
        SELECT date, rom_shoulder, rom_elbow, grip_strength,
               walk_speed, balance_score
        FROM patient_metrics
        WHERE patient_id = ?
        ORDER BY date DESC
        LIMIT 30
    ''', (patient_id,))

    data = cursor.fetchall()
    conn.close()

    # JSON 형식으로 변환
    progress_data = {
        'dates': [row[0] for row in data],
        'rom_shoulder': [row[1] for row in data],
        'rom_elbow': [row[2] for row in data],
        'grip_strength': [row[3] for row in data],
        'walk_speed': [row[4] for row in data],
        'balance_score': [row[5] for row in data]
    }

    return jsonify(progress_data)

@app.route('/api/patient/<patient_id>/compliance')
def get_exercise_compliance(patient_id):
    # 운동 준수율 계산
    conn = sqlite3.connect('rehab_data.db')
    cursor = conn.cursor()

    cursor.execute('''
        SELECT
            COUNT(CASE WHEN completed = 1 THEN 1 END) as completed,
            COUNT(*) as total
        FROM exercise_sessions
        WHERE patient_id = ?
        AND date >= date('now', '-7 days')
    ''', (patient_id,))

    result = cursor.fetchone()
    conn.close()

    compliance_rate = (result[0] / result[1] * 100) if result[1] > 0 else 0

    return jsonify({
        'completed_sessions': result[0],
        'total_sessions': result[1],
        'compliance_rate': round(compliance_rate, 1)
    })

AI 기반 성과 예측

머신러닝 모델을 사용하여 환자의 회복 궤적을 예측하고 최적의 치료 계획을 제안합니다.

회복 예측 모델 (Python)
# 환자 회복 궤적 예측 모델
import numpy as np
from sklearn.ensemble import RandomForestRegressor
from sklearn.preprocessing import StandardScaler

class RecoveryPredictor:
    def __init__(self):
        self.model = RandomForestRegressor(n_estimators=100)
        self.scaler = StandardScaler()

    def prepare_features(self, patient_data):
        # 특징 추출
        features = [
            patient_data['age'],
            patient_data['days_since_injury'],
            patient_data['initial_rom'],
            patient_data['initial_strength'],
            patient_data['therapy_intensity'],
            patient_data['comorbidities_count'],
            patient_data['compliance_rate']
        ]
        return np.array(features).reshape(1, -1)

    def predict_6month_outcome(self, patient_data):
        # 6개월 후 결과 예측
        features = self.prepare_features(patient_data)
        features_scaled = self.scaler.transform(features)

        predicted_rom = self.model.predict(features_scaled)[0]

        # 신뢰 구간 계산
        predictions = [tree.predict(features_scaled)[0]
                      for tree in self.model.estimators_]
        confidence_interval = (
            np.percentile(predictions, 5),
            np.percentile(predictions, 95)
        )

        return {
            'predicted_rom': predicted_rom,
            'confidence_interval': confidence_interval,
            'probability_full_recovery': sum(p > 140 for p in predictions) / len(predictions)
        }
