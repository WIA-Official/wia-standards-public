운동 기능 평가

장 개요: 이 장에서는 객관적 평가 도구, 기능 측정 방법, 표준화된 테스트, 자동화된 평가 시스템, 결과 분석 및 최소 임상적 중요 차이(MCID)를 다룹니다.

운동 기능 평가의 중요성

정확하고 신뢰할 수 있는 평가는 재활 치료의 기초입니다. 평가를 통해 초기 상태를 파악하고, 치료 목표를 설정하며, 진행 상황을 모니터링하고, 치료 효과를 측정할 수 있습니다.

평가의 핵심 속성

  • 신뢰도(Reliability): 반복 측정 시 일관된 결과
  • 타당도(Validity): 실제로 측정하고자 하는 것을 정확히 측정
  • 반응도(Responsiveness): 시간에 따른 변화를 감지하는 능력
  • 실행 가능성(Feasibility): 시간, 비용, 장비 측면에서 실용적

상지 기능 평가

상지 기능은 일상 생활 활동(ADL)에 필수적이므로 포괄적인 평가가 중요합니다.

1. Fugl-Meyer Assessment (FMA-UE)

뇌졸중 후 상지 운동 기능 평가의 표준. 33개 항목, 총 66점 만점.

영역 항목 수 점수 평가 내용
어깨/팔꿈치/전완 18 0-36 굴곡, 신전, 회전 등
손목 5 0-10 배측굴곡, 안정성
손 7 0-14 집기, 잡기 패턴
협응/속도 3 0-6 떨림, 운동완만증

점수 해석: 0-22 (심각), 23-31 (중등도), 32-47 (경도), 48-66 (최소)

MCID: 5.25점 (초기), 4.25점 (중기)

자동 FMA 점수 계산 시스템 (Python)
# Fugl-Meyer Assessment 자동 채점 시스템
class FuglMeyerScorer:
    def __init__(self):
        self.sections = {
            'shoulder_elbow': {'items': 18, 'max_score': 36},
            'wrist': {'items': 5, 'max_score': 10},
            'hand': {'items': 7, 'max_score': 14},
            'coordination': {'items': 3, 'max_score': 6}
        }

    def calculate_total(self, scores):
        # 각 항목 점수: 0 (불가능), 1 (부분적), 2 (완전)
        total = sum(scores.values())

        # 심각도 분류
        if total <= 22:
            severity = 'Severe'
        elif total <= 31:
            severity = 'Moderate'
        elif total <= 47:
            severity = 'Mild'
        else:
            severity = 'Minimal'

        return {
            'total_score': total,
            'max_possible': 66,
            'percentage': (total / 66) * 100,
            'severity': severity
        }

    def check_mcid(self, initial_score, current_score, stage):
        # 최소 임상적 중요 차이 확인
        mcid = 5.25 if stage == 'early' else 4.25
        improvement = current_score - initial_score

        return {
            'improvement': improvement,
            'mcid': mcid,
            'clinically_significant': improvement >= mcid
        }

2. Wolf Motor Function Test (WMFT)

15개의 시간 기반 과제와 2개의 근력 측정으로 구성. 실제 기능적 능력을 평가합니다.

  • 과제 예시: 팔꿈치 위로 팔 들기, 물건 잡기, 카드 뒤집기, 열쇠로 자물쇠 열기
  • 측정: 수행 시간(초), 기능 능력 척도(0-5)
  • 신뢰도: 평가자 간 ICC = 0.97

3. Box and Block Test (BBT)

1분간 블록을 한쪽에서 다른 쪽으로 옮긴 개수를 측정합니다.

연령대 남성 평균 여성 평균
20-24세 88 ± 11 88 ± 9
40-44세 83 ± 10 82 ± 10
60-64세 75 ± 11 74 ± 10

MCID: 5.5 블록 (뇌졸중 환자)

하지 및 보행 기능 평가

보행 능력은 독립성과 삶의 질에 직접적인 영향을 미칩니다.

1. 10-Meter Walk Test (10MWT)

보행 속도를 측정하는 가장 보편적인 테스트입니다.

  • 프로토콜: 14m 보행 (처음과 끝 2m는 가속/감속 구간)
  • 조건: 편안한 속도, 빠른 속도
  • 정상 보행 속도: 1.2-1.4 m/s
  • 지역사회 보행 한계: > 0.8 m/s
  • MCID: 0.16 m/s (뇌졸중)
보행 속도 자동 측정 시스템 (Python)
# IMU 센서 기반 보행 속도 자동 측정
import numpy as np

class WalkSpeedAssessment:
    def __init__(self):
        self.distance = 10.0  # meters
        self.total_distance = 14.0  # with acceleration zones

    def calculate_speed(self, time_seconds):
        # 속도 계산 (m/s)
        speed = self.distance / time_seconds

        # 분류
        if speed < 0.4:
            category = 'Household ambulation only'
        elif speed < 0.8:
            category = 'Limited community ambulation'
        elif speed < 1.0:
            category = 'Community ambulation'
        else:
            category = 'Normal ambulation'

        return {
            'speed_ms': round(speed, 2),
            'speed_kmh': round(speed * 3.6, 1),
            'category': category
        }

    def detect_gait_events(self, imu_data, sampling_rate=100):
        # 가속도 데이터에서 발 닿기/떼기 감지
        accel_z = imu_data[:, 2]  # 수직 가속도

        # 발 닿기: 가속도 급증
        heel_strikes = self.find_peaks(accel_z, threshold=15)

        # 보행 주기 계산
        stride_times = np.diff(heel_strikes) / sampling_rate
        avg_stride_time = np.mean(stride_times)
        cadence = 60 / avg_stride_time  # steps/min

        return {
            'num_strides': len(stride_times),
            'avg_stride_time': avg_stride_time,
            'cadence': cadence
        }

2. Timed Up and Go (TUG)

기능적 이동성과 낙상 위험을 평가합니다.

  • 프로토콜: 의자에서 일어나 3m 걷고 돌아와 앉기
  • 정상 성인: < 10초
  • 낙상 위험: > 13.5초
  • MCID: 2.9초 (뇌졸중)

3. Berg Balance Scale (BBS)

14개 항목으로 정적 및 동적 균형을 평가합니다. 총 56점 만점.

점수 범위 낙상 위험 권장 사항
41-56 낮음 독립적 이동 가능
21-40 중간 보행 보조 장치 필요
0-20 높음 휠체어 필요

객관적 기기 기반 평가

센서와 로봇 장치를 사용한 자동화된 평가는 주관성을 제거하고 정밀도를 높입니다.

1. 등속성 근력 측정(Isokinetic Dynamometry)

일정한 속도에서 근육의 최대 토크를 측정합니다.

  • 장비: Biodex, Cybex
  • 측정: Peak torque, Total work, Power
  • 속도: 60°/s (근력), 180°/s (파워), 300°/s (지구력)

2. 3D 동작 분석(Motion Capture)

마커 기반 또는 무마커 시스템으로 정밀한 움직임을 캡처합니다.

  • 광학 시스템: Vicon, OptiTrack - 정확도 ±0.1mm
  • 관성 시스템: Xsens, Perception Neuron - 이동성 우수
  • 무마커 시스템: Kinect Azure, Orbbec - 비용 효율적

표준화된 평가 프로토콜

일관된 평가를 위해 표준화된 프로토콜을 따라야 합니다.

평가 체크리스트

  1. 환경 준비: 조용하고 방해받지 않는 공간, 적절한 온도
  2. 환자 준비: 평가 목적 설명, 동의서 획득, 편안한 복장
  3. 장비 확인: 보정 완료, 배터리 충전, 백업 준비
  4. 측정 수행: 프로토콜 정확히 따르기, 2-3회 반복
  5. 데이터 기록: 즉시 기록, 비디오 녹화 (동의 하에)
  6. 결과 해석: 규준 데이터와 비교, 이전 평가와 비교

평가 결과 해석 및 보고

평가 결과는 환자와 다른 의료진이 쉽게 이해할 수 있도록 명확하게 제시해야 합니다.

평가 보고서 생성 시스템 (Python)
# 자동 평가 보고서 생성
from datetime import datetime

class AssessmentReport:
    def __init__(self, patient_data):
        self.patient = patient_data
        self.assessment_date = datetime.now()

    def generate_report(self, assessment_scores):
        report = {
            'patient_info': {
                'name': self.patient['name'],
                'age': self.patient['age'],
                'diagnosis': self.patient['diagnosis']
            },
            'assessment_date': self.assessment_date.strftime('%Y-%m-%d'),
            'scores': assessment_scores,
            'interpretation': self.interpret_scores(assessment_scores),
            'recommendations': self.generate_recommendations(assessment_scores)
        }

        return report

    def interpret_scores(self, scores):
        interpretations = []

        # FMA 해석
        if 'fma_ue' in scores:
            fma = scores['fma_ue']
            if fma < 23:
                interpretations.append('상지 운동 장애: 심각 (FMA-UE: {}).'.format(fma))
            elif fma < 32:
                interpretations.append('상지 운동 장애: 중등도 (FMA-UE: {}).'.format(fma))
            else:
                interpretations.append('상지 운동 장애: 경도-최소 (FMA-UE: {}).'.format(fma))

        # 보행 속도 해석
        if 'walk_speed' in scores:
            speed = scores['walk_speed']
            if speed < 0.4:
                interpretations.append('보행: 가정 내 보행만 가능 ({} m/s).'.format(speed))
            elif speed < 0.8:
                interpretations.append('보행: 제한적 지역사회 보행 ({} m/s).'.format(speed))
            else:
                interpretations.append('보행: 지역사회 보행 가능 ({} m/s).'.format(speed))

        return interpretations

    def generate_recommendations(self, scores):
        recommendations = []

        # 점수 기반 치료 권장사항
        if scores.get('fma_ue', 0) < 32:
            recommendations.append('로봇 보조 상지 재활 권장')
            recommendations.append('주 3-5회, 세션당 45-60분 집중 치료')

        if scores.get('berg_balance', 56) < 45:
            recommendations.append('균형 훈련 우선순위')
            recommendations.append('낙상 예방 교육 필요')

        return recommendations
핵심 요약: 객관적이고 표준화된 운동 기능 평가는 재활 치료의 효과를 정확히 측정하고 개인 맞춤형 치료 계획을 수립하는 데 필수적입니다. 임상 평가 도구와 기기 기반 측정을 결합하여 포괄적이고 신뢰할 수 있는 평가를 수행할 수 있습니다.
