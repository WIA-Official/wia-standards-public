## 📜 Citation & Veracity Policy

All WIA standards are publishing-grade documents. Every citation, statistic, institution name, law name, and standard number must originate from a verifiable primary source. Fabricated academic citations and unverifiable clinical numbers are forbidden in body/spec text.

→ **Full policy**: [`docs/CITATION-POLICY.md`](docs/CITATION-POLICY.md)
→ Pre-commit hook + GitHub Actions automatically enforce the policy.

## 🤝 Philosophy

**"Not 'technology for disabled people' - but 'choices for all humans'"**

WIA Standards are built on the principle that **all humans deserve to access information in their own way**. There is no "default" way - all representations are equal.

### Five Equal Representations (PubScript)

- 🎨 **Visual**: Text, images, layout, colors
- 🔊 **Auditory**: Speech, music, spatial audio
- 👆 **Tactile**: Braille, haptic feedback
- 🌍 **Spatial**: 3D positioning, AR/VR
- 👋 **Gestural**: Swipe, voice, eye tracking
