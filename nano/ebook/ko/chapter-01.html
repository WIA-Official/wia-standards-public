1장: WIA 나노기술 표준 입문

장 개요: WIA‑NANO‑016은 ISO/TC 229 어휘, IUPAC 명명법, OECD WPMN 시험지침, FDA·EMA·MFDS 규제 경로, 그리고 한국 나노 R&D 생태계(KIMS·KIST·KCN‑CC·NSP‑Net)의 데이터 흐름을 단일 와이어 포맷으로 묶는 실행 가능 명세다.

1.1 왜 나노 스케일 표준이 필요한가

“나노물질”은 외부 치수 또는 내부 구조 중 하나가 1~100 nm 범위인 물질로 흔히 정의된다(ISO/TS 80004‑1, EU Recommendation 2022/C 229/01). 이 스케일에서는 표면적·반응성·전자 구조·생체 이송 특성이 벌크 사양으로 설명되지 않는 방식으로 변한다. 누가·어떤 방법으로·어떤 기기로 측정했는지를 담는 표준 와이어가 없으면 두 실험실은 서로의 결과를 재현할 수 없고, 규제기관은 판단할 수 없으며, 임상의는 안전성 자료를 신뢰할 수 없다.

WIA‑NANO‑016은 그 격차에 대한 엔지니어링 응답이다. 새 측정법이나 독성 종점을 만들지 않는다 — 그건 ISO·IUPAC·OECD·FDA·EMA·PMDA의 영역. 본 표준은 상호운용 계층이다. 대전 KIMS의 TEM 마이크로그래프, 케임브리지의 OECD TG 412 흡입독성, 실버스프링의 FDA pre‑IND 나노의약품 자료, 헬싱키의 REACH 나노 형태 등록이 같은 식별자·같은 단위·같은 추적 가능 증거 사슬을 공유하게 한다.

1.2 표준이 가능하게 만드는 다섯 가지

  1. 재현 가능한 특성평가. 입자 크기 분포·형상·표면 화학·제타 전위가 서명된 단일 JSON‑LD로 흐른다.
  2. 국가 횡단 규제 제출. 단일 자료가 FDA pre‑IND, EMA centralised, PMDA QbD, MFDS 보충 신청, REACH 나노 형태 등록으로 자동 렌더링된다.
  3. nanoEHS 감시. 작업자 노출(NIOSH NMAM 7300/0500), 환경 방출(OECD WPMN), 소비자 안전이 단일 시계열에 적재.
  4. DNA 데이터 저장 추적. 합성·시퀀싱·디코딩이 결정론적으로 검증 가능한 이벤트로 기록된다.
  5. 공급망 추적성. 전구체 합성에서 폐기까지 모든 변환이 EdDSA‑JCS‑2022로 서명된다.

1.4 4단계 구현 흐름

Phase 1 — 데이터 형식    ISO/TS 80004 어휘, IUPAC 명명, JSON‑LD + UCUM, Allotrope ADF
Phase 2 — API 인터페이스 OpenAPI 3.1 + AsyncAPI 3.0, OAuth 2.1 + DPoP, FAIR 원칙
Phase 3 — 프로토콜       OPC UA + MQTT 5, 기기 통합(TEM·AFM·DLS·ICP‑MS)
Phase 4 — 통합          FDA / EMA / PMDA / MFDS / REACH 자료 생성, 공급망 추적성

1.5 한국 맥락의 핵심 거점

1.5.1 한국재료연구원(KIMS) · 나노소재연구본부

창원 소재. 나노입자·나노박막·계산재료과학 연구. WIA 표준 도입 시 OPC UA Lab Devices CS 기반 기기 통합과 ISO/TS 80004 어휘 활용이 1순위 거점.

1.5.2 한국과학기술연구원(KIST) 나노융합기술원

서울 본원과 서울 청계광장의 나노융합기술원. 나노바이오·나노에너지 응용 R&D. 표준의 FHIR R5 독성 어휘와 결합되는 기관.

1.5.3 한국나노기술원(KANC) · NSP‑Net

수원 광교 소재. 나노 팹·MEMS·CMOS 후공정. NSP‑Net은 한국·일본·대만 공정 표준화 네트워크. WIA의 EPCIS 2.0 supply‑chain 연결점.

1.5.4 나노안전성기술센터(KCN‑CC, KIMS 산하)

나노물질의 안전성 평가·작업자 노출 모니터링. WIA의 NIOSH NMAM 7300/0500 표준 데이터 흐름과 직접 연동.

1.5.5 「나노기술발전기본법」

2002 제정, 5년 단위 종합계획 의무화. 나노 R&D 데이터 표준화·공유에 본 명세가 기술적 밑판이 된다.

1.6 본 책이 다루는 것과 다루지 않는 것

본 책은 실무 레퍼런스다. 순서대로 읽으면 “표준이 뭐냐”에서 “첫 FDA pre‑IND를 기기 단위 특성평가까지 추적 가능한 형태로 제출했다”까지 닿는다. 나노 과학 입문서가 아니다 — 그건 IUPAC 권고와 ISO/TS 80004가 정전. OECD 시험지침이나 FDA 규제 가이드 대체물도 아니다 — 그건 각 기관의 가이드가 정전. WIA‑NANO‑016은 그 프레임워크들을 실행 가능하게 만든다.

1.7 본 책 읽는 법

  1. 1~3장 — 동기·통증·아키텍처. 한 번 읽는다.
  2. 4~7장 — Phase 1~4 명세 + 코드. 구현 옆에 두고 본다.
  3. 8장 — 도입 일정·인증 등급·ROI. 내부 설득용.

장 요약

핵심 요점

  1. 나노 스케일에서는 누가·어떻게 측정했는지가 숫자만큼 중요하다 — 와이어 표준이 그 차이를 살린다.
  2. ISO/TS 80004, IUPAC, OECD WPMN, FDA NTF, EMA, MFDS, REACH 2018/1881, KIMS·KIST·KANC를 한 어휘로 묶는다.
  3. 여섯 영역(어휘·특성평가·독성·규제·작업자 안전·수명주기)이 같은 식별자/단위 시스템을 공유한다.
  4. 4단계 흐름은 바이트에서 시작해 자료로 닿는다.
  5. 표준은 상호운용 계층 — 측정법과 규제 가이드는 권위가 그대로 살아 있다.
