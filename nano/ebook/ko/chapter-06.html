6장: Phase 3 — 프로토콜

장 개요: 데이터 평면 — OPC UA Companion Specs(기기), MQTT 5(브로커), Allotrope ADF 스트림, SiLA 2 디바이스 제어, TLS 1.3 + OAuth 2.1 + DPoP, 실패 모드.

6.1 OPC UA — 기기 단위 캡처

OPC UA(IEC 62541) Companion Spec이 벤더별 API를 균일한 정보 모델로 변환한다. 본 표준이 요구하는 Companion Spec:

기기 분류OPC UA spec정렬 벤더
전자 현미경(TEM/SEM)OPC UA Microscopy CS(개발 중) + Lab Devices CSFEI/Thermo, Hitachi, JEOL
실험실 디바이스(DLS, BET, Zetasizer)OPC UA Lab Devices CS (LADS)Malvern, Anton Paar, Micromeritics
질량 분석기OPC UA Lab Devices CSThermo, Agilent, Bruker
SiLA 2 디바이스SiLA 2 + OPC UA 브리지Tecan, Hamilton 등 자동화

6.2 MQTT 5 브로커 — cross‑lab fan‑out

OPC UA 캡처를 MQTT 5 메시지로 변환해 cross‑org 분배. 토픽:

nano/{tenant}/lab/{labId}/instrument/{instrumentId}/measurement
nano/{tenant}/lab/{labId}/instrument/{instrumentId}/state
nano/{tenant}/lab/{labId}/exposure/sampler/{samplerId}/reading
nano/{tenant}/substance/{substanceId}/event

QoS: 0(하트비트), 1(routine 측정), 2(규제 제출용 증거·노출 한도 알람).

6.3 Allotrope ADF 바이너리 스트림

ADF(HDF5)가 raw 기기 출력(이미지 스택, 상관도, 스펙트럼)을 통제 어휘로 감싼다. 두 모드:

  1. Static — 측정 종료 후 완성된 ADF 업로드.
  2. Stream — 장시간 캡처(예: 24시간 세포 흡수 라이브 이미징) 도중 청크 단위 ADF 객체 commit, 부분 결과 조회 가능.

6.5 SiLA 2 디바이스 제어

SiLA 2(gRPC + Protocol Buffers)는 액체 핸들러·로봇 플랫폼·마이크로리액터의 사실상 표준 제어 프로토콜. 본 표준은 SiLA‑2‑to‑OPC‑UA 브리지를 제공해 데이터·제어가 같은 권한 표면을 공유한다.

6.6 NIOSH 사양 노출 샘플러

NIOSH NMAM 7300(원소 나노입자 ICP‑MS)·NMAM 0500(중량) 샘플러가 MQTT 5로 시계열 송출. 표준은 ISO/IEC 17025 인정 실험실의 교정 인증서를 W3C VC로 첨부 요구.

6.7 변조 불가능 앵커링

증거의 모든 Iceberg 스냅샷과 자료 publish가 머클 루트로 WIA Chain PoA + BSC 메인넷에 앵커링. 외부 감사인은 비공개 제형 화학을 보지 않고도 무결성을 검증.

6.9 프로토콜 계층 실패 모드

  • 기기 캡처 중 오프라인: 로컬 ADF 버퍼를 원본 ts로 리플레이.
  • 시계 드리프트: 6시간 NTP, |now − ts| > 600 s 거절.
  • 지연 재발행: cache/... 토픽 접두사 사용.
  • 스키마 깨짐: 시료 subject는 BACKWARD_TRANSITIVE, 기관 간 이벤트는 FULL_TRANSITIVE.
  • 리플레이 공격: RFC 9421 nonce + JTI 캐시.

장 요약

핵심 요점

  1. OPC UA Companion Specs가 벤더 무관 통일 출력을 만든다.
  2. MQTT 5 fan‑out이 자체 미들웨어 없이 실험실 간 측정을 옮긴다.
  3. Allotrope ADF는 static + stream 두 모드로 raw 증거를 보존.
  4. SiLA 2 + OPC UA 브리지로 데이터·제어가 같은 권한 표면.
  5. WIA Chain PoA + BSC 머클 앵커링이 IP 노출 없는 외부 감사 가능성.
