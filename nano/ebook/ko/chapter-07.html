7장: Phase 4 — 통합

장 개요: 시료 레코드를 실제 결과로 만드는 통합 — FDA / EMA / PMDA / MFDS / REACH 자료, EPCIS 2.0 공급망 추적성, WIA‑CLIMATE‑022 ESG 보고, DNA 데이터 저장, 소비자 공개 검증.

7.1 FDA 나노기술 태스크포스 경로

FDA NTF 가이드는 규제 대상 제품이 나노기술 적용에 해당하는지 묻는다. WIA‑NANO‑016이 SPL XML 모듈을 NTF 4가지 결정 기준에 따라 자동 생성한다.

  1. 외부 치수 또는 내부 구조 1~100 nm.
  2. 1 µm까지 영역 안에서 나노스케일 속성을 보이는 경우.
  3. 특정 속성을 위해 의도적으로 설계.
  4. 제품의 의도적 구성요소.

7.2 EU REACH 2018/1881 나노 형태 등록

REACH Annex VI(2018/1881 개정)는 나노 형태별 크기 분포·형상·표면 화학·표면 처리·비표면적·결정성 보고를 요구한다. 본 표준은 IUCLID 호환 JSON‑LD 트윈을 lead registrant 키로 서명해 생성.

전구체에서 제품까지 모든 변환을 EPCIS 2.0 TransformationEvent로:

7.5 DNA 데이터 저장 상호운용

합성 올리고에 디지털 데이터를 인코딩하는 DNA 저장. 표준은 벤더 중립 경로(Microsoft Project Silica, Twist Bioscience, Catalog DNA, Illumina BaseSpace)를 노출.

POST /v1/dna-storage/archive
{
  "archiveId":"arc-2026-Q2-genomes-9KB",
  "encoder":{"name":"GoldmanCRC","version":"1.2"},
  "synthesis":{"vendor":"twist","plate":"TWIST-2026-Q2-014","oligoCount":18432},
  "redundancy":{"raid":"reed-solomon","k":12,"n":16},
  "checkRead":{"reader":"Oxford-Nanopore-MinION","date":"2026-04-25"}
}

7.6 OECD WPMN 자료 컴파일

자발 OECD WPMN sponsorship 자료가 기관 간 컴파일된다. 표준은 시료의 TG 정렬 종점 레코드를 lead sponsor 키로 서명한 단일 WPMN 자료로 묶어낸다.

7.7 WIA‑CLIMATE‑022 ESG 결합

나노물질 생산은 IFRS S2 / ESRS E1에 보고되는 탄소·에너지 발자국이 있다. WIA‑CLIMATE‑022 결합으로 각 배치의 Scope 1/2/3 배출이 기업 공시 트윈으로 자동 흐른다. 운송 단계는 ISO 14083:2023이 다룬다.

7.8 소비자 검증

GET https://verify.wia-nano.org/zno-20nm-pegylated-2026-04-25

→ 시료 요약, ISO 19749 크기 레코드, OECD TG 318 분산 안정성,
  NIOSH NMAM 7300 작업자 노출, FDA NTF 결정, REACH 나노 형태 ID,
  발급자 DID 서명, WIA Chain 검증.

7.9 WIA 생태계 결합

  • WIA‑VC — GLP 진술서·인정 인증서·ISO 17025 보고서.
  • WIA‑IDENTITY — 스폰서·GLP 실험실·규제기관 DID.
  • WIA‑IOT — TEM·AFM·DLS·ICP‑MS·노출 샘플러 온보딩.
  • WIA‑CARBON — 생산 단계 배출.
  • WIA‑CLIMATE‑022 — IFRS S2 / ESRS E1 라우팅.
  • WIA‑BLOCKCHAIN — 시료·자료 무결성 머클 앵커링.

장 요약

핵심 요점

  1. FDA NTF 결정은 1급 시료 필드, 사후 문서가 아니다.
  2. EU REACH 2018/1881 나노 형태 등록은 IUCLID 호환 JSON‑LD 트윈으로.
  3. 전구체→제품 공급망 변환은 EPCIS 2.0 이벤트.
  4. DNA 데이터 저장은 벤더 중립 archive·redundancy·read‑back API.
  5. 공개 검증이 제품 QR에서 검증 가능 시료 요약까지 닫는다.
