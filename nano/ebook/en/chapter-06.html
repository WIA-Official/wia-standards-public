Chapter 6: Phase 3 — Protocol

Chapter overview: The data plane — OPC UA Companion Specs for instruments, MQTT 5 broker for cross-lab fan-out, Allotrope ADF binary stream for raw evidence, SiLA 2 device control, TLS 1.3 + OAuth 2.1 + DPoP, and the failure modes you must handle.

6.1 OPC UA — instrument-level capture

OPC UA (IEC 62541) Companion Specifications turn each instrument vendor's API into a uniform information model. WIA-NANO-016 mandates the following:

6.2 MQTT 5 broker for cross-lab fan-out

OPC UA captures are translated into MQTT 5 messages for cross-organisation distribution. Topics:

nano/{tenant}/lab/{labId}/instrument/{instrumentId}/measurement
nano/{tenant}/lab/{labId}/instrument/{instrumentId}/state
nano/{tenant}/lab/{labId}/exposure/sampler/{samplerId}/reading
nano/{tenant}/substance/{substanceId}/event

QoS strategy: 0 for heartbeats, 1 for routine measurements, 2 for regulator-bound evidence and exposure-limit alerts.

6.3 Allotrope ADF binary stream

Allotrope Data Format (ADF) wraps raw instrument output (image stacks, correlograms, spectra) in HDF5 with controlled vocabulary. The standard supports two modes:

  1. Static — finished ADF file uploaded at end of a measurement run.
  2. Stream — ADF objects committed in chunks during long captures (e.g., 24 h cell-uptake live imaging) so partial results are queryable mid-run.

6.5 SiLA 2 device control

SiLA 2 (Standardisation in Lab Automation, gRPC + Protocol Buffers) is the de-facto control protocol for liquid handlers, robotic platforms, and microreactors. The standard offers a SiLA-2-to-OPC-UA gateway so data + control share the same authorisation surface.

6.6 NIOSH-spec exposure samplers

For nanoEHS surveillance, NIOSH NMAM 7300 (ICP-MS for elemental nanoparticles) and NMAM 0500 (gravimetric) samplers feed time-series via MQTT 5. The standard requires a calibration certificate signed by an ISO/IEC 17025 accredited lab attached as a Verifiable Credential.

6.7 Tamper-evident anchoring

Every Iceberg snapshot of evidence and every dossier publish produces a Merkle root anchored on WIA Chain PoA + BSC mainnet. Auditors verify tamper-evidence without seeing private formulation chemistry.

6.9 Failure modes the protocol layer must handle

  • Instrument offline mid-capture — replay the local ADF buffer with original timestamps; do NOT discard.
  • Clock drift — NTP every 6 h; ingest rejects readings with |now − ts| > 600 s.
  • Late re-publish — use cache/... topic prefix so consumers can distinguish original vs. cached.
  • Schema break — registry compatibility set to BACKWARD_TRANSITIVE for substance subjects, FULL_TRANSITIVE for inter-org events.
  • Replay attacks — RFC 9421 nonces + JTI cache.

Chapter Summary

Key Takeaways

  1. OPC UA Companion Specs unify instrument output regardless of vendor.
  2. MQTT 5 fan-out moves measurements between labs without proprietary middleware.
  3. Allotrope ADF preserves raw evidence, both static and streamed.
  4. SiLA 2 + OPC UA bridge means control and data live on the same authorisation surface.
  5. Merkle-root anchoring on WIA Chain PoA + BSC gives external auditability without leaking IP.
