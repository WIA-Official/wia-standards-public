실제 근거 (RWE)

장 개요: 이 장에서는 실제 세계 데이터(RWD)와 실제 근거(RWE)의 개념, 전자건강기록(EHR) 통합, 관찰 연구, 약물 감시 및 RCT 보완을 다룹니다.

실제 근거(RWE)란?

실제 근거(Real-World Evidence, RWE)는 일상적인 임상 진료 환경에서 수집된 실제 세계 데이터(Real-World Data, RWD)의 분석을 통해 얻은 의료 제품의 사용과 잠재적 이익 또는 위험에 관한 임상적 증거입니다. RWE는 엄격하게 통제된 무작위 대조 임상시험(RCT)을 보완하여 약물의 실제 효과성, 안전성, 경제성을 평가합니다.

전통적인 RCT는 내적 타당도가 높지만 엄격한 선정 기준, 통제된 환경, 짧은 추적 기간으로 인해 실제 진료 환경을 충분히 반영하지 못할 수 있습니다. RWE는 이러한 한계를 극복하고, 다양한 환자 집단, 실제 복약 순응도, 장기 결과를 포착할 수 있습니다.

실제 세계 데이터(RWD)의 출처

RWD는 다양한 출처에서 수집됩니다. 각 출처는 고유한 장점과 한계를 가지고 있습니다.

데이터 출처 설명 장점 한계
전자건강기록(EHR) 병원/클리닉의 진료 기록 상세한 임상 정보, 종단 추적 데이터 품질 불균일, 표준화 부족
의료보험 청구 데이터 보험 청구 및 지불 정보 대규모, 장기 추적 임상 상세 정보 부족
환자 등록부(Registry) 특정 질환/치료 환자 추적 표준화, 목적 지향적 비용, 선택 편향
웨어러블/mHealth 스마트폰, 웨어러블 기기 지속적 모니터링, 객관적 데이터 품질, 대표성
환자 보고 결과(PRO) 환자 직접 보고 환자 관점, 삶의 질 주관적, 응답 편향

전자건강기록(EHR) 데이터

EHR은 가장 풍부한 RWD 출처 중 하나입니다. 진단, 처방, 검사 결과, 시술, 입원 기록 등 종합적인 임상 정보를 포함합니다.

EHR 데이터의 과제

  1. 표준화 부족: 서로 다른 EHR 시스템(Epic, Cerner, Allscripts)이 다른 데이터 모델 사용
  2. 데이터 품질: 불완전한 기록, 오타, 코딩 오류
  3. 비정형 데이터: 진료 노트의 80%가 자유 텍스트 형식
  4. 상호운용성: 시스템 간 데이터 교환의 어려움
  5. 결측값: 검사를 시행하지 않으면 데이터가 없음 (결과가 정상이어서가 아님)

EHR 데이터 표준화: 공통 데이터 모델(CDM)

서로 다른 EHR 시스템의 데이터를 통합하기 위해 공통 데이터 모델이 개발되었습니다:

CDM 운영 특징
OMOP CDM OHDSI 컨소시엄 관찰 연구 중심, 국제적 채택, 표준 용어 (SNOMED, RxNorm)
PCORnet CDM 미국 PCORI 환자 중심 결과 연구, 미국 중심
Sentinel CDM 미국 FDA 약물 안전성 감시, 청구 데이터 중심
i2b2 Partners HealthCare 임상 연구 정보학, 쿼리 도구 포함

OMOP CDM 구조

OMOP(Observational Medical Outcomes Partnership) CDM은 가장 널리 사용되는 공통 데이터 모델입니다:

  • 표준화 테이블: Person, Visit, Condition, Drug, Procedure, Measurement 등
  • 표준 용어: SNOMED CT(진단), RxNorm(약물), LOINC(검사)
  • 개념 계층: 상세 개념을 표준 개념으로 매핑
  • 분석 도구: ATLAS, ACHILLES로 코호트 정의 및 분석

RWE 연구 설계

RWD를 사용한 연구는 다양한 관찰 연구 설계를 사용합니다.

코호트 연구(Cohort Study)

특정 노출(약물, 시술)을 받은 환자 그룹을 추적하여 결과를 비교합니다.

  • 전향적 코호트: 현재부터 미래로 추적 (비용이 높고 시간 소요)
  • 후향적 코호트: 과거 데이터를 사용하여 결과 평가 (빠르고 저렴)
  • 성향 점수 매칭(PSM): 혼란 변수를 조정하여 치료군과 대조군을 유사하게 만듦

환자-대조군 연구(Case-Control Study)

결과가 발생한 환자(case)와 발생하지 않은 환자(control)를 선정하여 과거 노출을 비교합니다.

  • 희귀 질환이나 결과 연구에 효율적
  • 후향적이므로 빠르고 저렴
  • 선택 편향과 회상 편향 위험

자가 대조 연구(Self-Controlled Study)

각 환자가 자신의 대조군이 되어 노출 전후를 비교합니다.

  • Case-Crossover: 급성 결과의 촉발 요인 연구
  • Self-Controlled Case Series: 급성 이상반응의 시간적 연관성 평가
  • 시간에 따라 변하지 않는 혼란 변수를 자동으로 통제

약물 감시(Pharmacovigilance)

시판 후 약물 안전성 모니터링은 RWE의 중요한 응용 분야입니다. 임상시험에서 발견하지 못한 희귀하거나 지연된 이상반응을 탐지할 수 있습니다.

신호 탐지(Signal Detection)

대규모 데이터베이스에서 약물과 이상반응 간의 예상치 못한 연관성을 통계적으로 탐지합니다.

방법 설명
비례 보고비(PRR) Proportional Reporting Ratio, 자발적 보고 데이터베이스에서 사용
보고 승산비(ROR) Reporting Odds Ratio, WHO VigiBase에서 사용
정보 구성요소(IC) Information Component, 베이지안 접근법
LGPS Longitudinal Gamma Poisson Shrinker, 순차 분석

FDA Sentinel 시스템

Sentinel은 FDA가 운영하는 국가 전자 감시 시스템으로, 1억 명 이상의 환자 데이터를 실시간으로 모니터링합니다.

  • 분산 네트워크: 데이터를 중앙으로 이동하지 않고 각 기관에서 분석 실행
  • 표준 쿼리: 미리 정의된 쿼리로 약물 안전성 평가
  • 신속 대응: 안전성 신호 발생 시 수주 내 평가 가능
  • 프라이버시 보호: 환자 개인정보를 보호하면서 집계 결과만 공유

RWE의 규제 활용

FDA와 EMA를 포함한 주요 규제 기관들은 RWE를 약물 승인 및 라벨 변경 결정에 점차 활용하고 있습니다.

FDA의 RWE 프레임워크

2016년 21세기 치료법(21st Century Cures Act)은 FDA에게 RWE 사용 프레임워크 개발을 요구했습니다. FDA는 RWE를 다음 용도로 사용할 수 있다고 명시했습니다:

  1. 시판 후 안전성 연구: 이상반응 모니터링 및 평가
  2. 적응증 추가: 승인된 약물의 새로운 적응증 승인
  3. 시판 후 요구사항 충족: 가속 승인 후 확증 연구

RWE 기반 승인 사례

  • Ibrutinib (2017): EHR 데이터를 사용하여 만성 림프구성 백혈병의 새로운 적응증 승인
  • Palbociclib (2019): 보험 청구 데이터로 남성 유방암 적응증 확대
  • External Control Arm: 희귀 질환에서 RWD를 역사적 대조군으로 사용

RWE의 한계와 과제

RWE는 강력한 도구이지만 여러 한계와 과제가 있습니다.

편향(Bias)

  • 선택 편향: 치료 선택이 무작위가 아니므로 환자 특성에 의해 편향될 수 있음
  • 적응증에 의한 혼란(Confounding by indication): 더 아픈 환자가 특정 치료를 받을 가능성
  • 불멸 시간 편향(Immortal time bias): 치료 시작 전 기간을 잘못 분류하여 효과 과대평가
  • 정보 편향: 노출이나 결과 측정의 부정확성

데이터 품질

  • 결측 데이터: 중요한 변수가 기록되지 않았을 수 있음
  • 코딩 오류: 진단 코드의 부정확성 (예: 청구 목적의 업코딩)
  • 시간 정확도: 정확한 날짜 정보 부족
  • 검증 부족: RCT처럼 소스 문서 검증이 없음

인과성 추론의 어려움

관찰 연구는 연관성을 보여줄 수 있지만 인과성을 증명하기는 어렵습니다. 고급 통계 방법을 사용하여 이를 완화합니다:

  • 성향 점수(Propensity Score): 매칭, 가중치, 층화로 혼란 변수 조정
  • 도구 변수(Instrumental Variable): 노출과 연관되지만 결과와는 직접 연관 없는 변수 활용
  • 차이의 차이(Difference-in-Differences): 시간에 따른 변화 비교
  • 회귀 불연속(Regression Discontinuity): 임계값 전후 비교

RWE의 미래

RWE는 계속 진화하고 있으며, 여러 혁신적 발전이 예상됩니다.

합성 대조군(Synthetic Control Arm)

RWD를 사용하여 임상시험의 대조군을 생성합니다. 특히 희귀 질환에서 유용합니다:

  • 윤리적: 플라시보 투여를 피할 수 있음
  • 효율적: 등록 환자 수 감소, 비용 절감
  • 빠른: 역사적 데이터 활용으로 연구 기간 단축

디지털 바이오마커

웨어러블 기기와 스마트폰으로 수집된 지속적인 데이터:

  • 활동량: 걸음 수, 활동 강도 (심부전, 파킨슨병)
  • 심박수 변이도: 자율신경계 기능 (당뇨, 심혈관 질환)
  • 수면 패턴: 수면의 질과 양 (우울증, 수면 장애)
  • 음성/타이핑: 인지 기능 모니터링 (알츠하이머)

AI/ML 통합

머신러닝은 RWE 생성의 여러 측면을 개선하고 있습니다:

  • 자연어 처리(NLP): 진료 노트에서 정보 추출
  • 예측 모델: 질병 진행, 치료 반응 예측
  • 인과 추론: 딥러닝 기반 성향 점수 모델
  • 페노타이핑: 복잡한 표현형 자동 식별
핵심 요약:

실제 근거(RWE)는 일상 진료에서 수집된 실제 세계 데이터(RWD)를 분석하여 얻은 임상 증거로, 엄격한 RCT를 보완합니다. EHR, 청구 데이터, 레지스트리, 웨어러블 기기 등 다양한 출처의 RWD를 활용합니다. OMOP CDM 같은 공통 데이터 모델로 이질적인 데이터를 표준화하여 다기관 연구를 가능하게 합니다. 코호트 연구, 환자-대조군 연구, 자가 대조 연구 등 다양한 관찰 연구 설계를 사용합니다. FDA Sentinel 시스템은 시판 후 약물 안전성을 실시간으로 모니터링합니다. FDA와 EMA는 RWE를 적응증 추가, 안전성 연구에 점차 활용하고 있습니다. 선택 편향, 데이터 품질, 인과성 추론의 어려움 등 한계가 있지만, 성향 점수, 도구 변수 등 고급 통계 방법으로 완화할 수 있습니다. 합성 대조군, 디지털 바이오마커, AI/ML 통합 등 혁신적 발전이 RWE의 미래를 만들어가고 있습니다.
