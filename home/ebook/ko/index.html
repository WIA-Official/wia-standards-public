WIA-HOME

건물주 없는 디지털 집 - P2P 개인 웹 서버

"홍익인간: 누구나 플랫폼 종속 없이 디지털 세상에서 자신의 집을 가질 권리가 있습니다."

목차

  • 제1장: 소개 - 문제점과 WIA 해결책
  • 제2장: P2P 아키텍처 - 개인 서버와 메시 네트워크
  • 제3장: 사이트 생성 - 클론, 의도 기반, 템플릿
  • 제4장: 콘텐츠 관리 - 페이지, 블로그, 미디어
  • 제5장: 전자상거래 - 쇼핑몰과 결제 연동
  • 제6장: 예약 시스템 - 예약 및 일정 관리
  • 제7장: 보안 - TLS-LITE와 AIR-SHIELD 연동
  • 제8장: 구현 - API 및 SDK 참조

표준 정보

문서 ID: WIA-HOME
버전: 1.0.0
상태: 활성
최종 업데이트: 2025-12

범위:
  - P2P 개인 웹 서버
  - 무료 도메인 (yourname.wia.home)
  - 사이트 클론 및 커스터마이징
  - 소상공인 기능
  - 글로벌 결제 연동

관련 표준:
  - WIA-INTENT (의도 기반 생성)
  - WIA-OMNI-API (API 게이트웨이)
  - WIA-AIR-SHIELD (보안)
  - WIA-AIR-POWER (전력 관리)
  - WIA-SOCIAL (소셜 네트워크)

비용 비교

도메인 연 1-5만원 0원 (yourname.wia.home)
