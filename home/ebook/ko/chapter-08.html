8.1 설치

NPM 설치
========

npm install @anthropic/wia-home

또는 yarn:
yarn add @anthropic/wia-home

또는 pnpm:
pnpm add @anthropic/wia-home


기본 설정
=========

import { WIAHome } from '@anthropic/wia-home';

// 홈 생성
const home = new WIAHome({
  name: '내 가게',
  domain: 'myshop.wia.shop'
});

// 서버 시작
await home.start();

console.log('라이브:', home.domain);


설정 옵션
=========

const home = new WIAHome({
  // 기본
  name: '내 가게',
  domain: 'myshop.wia.shop',

  // 서버
  server: {
    port: 'auto',
    maxConnections: 100,
    sleepMode: true,
    sleepAfter: 5  // 분
  },

  // 기능
  features: {
    shop: true,
    blog: true,
    booking: true
  },

  // 보안
  security: {
    ddos: { rateLimit: 100 },
    waf: { sqlInjection: true, xss: true }
  },

  // 저사양 모드
  lowSpecMode: {
    limits: {
      maxMemory: 256,
      maxCpu: 20
    }
  }
});

8.2 코어 API

WIAHome 클래스
==============

// 정적 메서드
WIAHome.analyze(url)      // 사이트 분석
WIAHome.clone(url, opts)  // 사이트 클론
WIAHome.quickStart(opts)  // 퀵 스타트
WIAHome.fromIntent(text)  // 의도 기반
WIAHome.fromTemplate(id)  // 템플릿에서
WIAHome.migrate(config)   // 마이그레이션


인스턴스 메서드
===============

// 서버 제어
home.start()              // 서버 시작
home.stop()               // 서버 정지
home.restart()            // 재시작

// 상태
home.status               // 'running' | 'stopped'
home.domain               // 'myshop.wia.shop'
home.uptime               // 초


속성
====

// 통계
home.stats.visitors       // 방문자 수
home.stats.pageViews      // 페이지뷰 수
home.stats.orders         // 주문 수

// 자원
home.resources.cpu        // CPU 사용량 %
home.resources.memory     // 메모리 사용량 bytes
home.resources.bandwidth  // 대역폭 통계


이벤트
======

home.on('server_started', callback)
home.on('server_stopped', callback)
home.on('visitor_connected', callback)
home.on('page_viewed', callback)
home.on('order_received', callback)
home.on('booking_received', callback)
home.on('security_alert', callback)
home.on('error', callback)

결론

WIA-HOME은 호스팅 비용 없이 자신만의 웹사이트를 만들기 위한 완전한 플랫폼을 제공합니다. P2P 네트워킹부터 전자상거래, 보안까지 모든 것이 내장되어 있고 사용하기 쉽습니다.

홍익인간 철학에 따라, WIA-HOME은 소상공인부터 프리랜서, 학생까지 모든 사람이 플랫폼 종속 없이 영원히 무료로 자신만의 디지털 집을 가질 수 있도록 보장합니다.

WIA-HOME의 약속:

1. 건물주가 없어요 - 내 PC가 서버
2. 영원히 내 것이에요 - 플랫폼 독립
3. 돈 안 들어요 - 영원히 무료 호스팅
4. 쉬워요 - 5분 만에 런칭
5. 안전해요 - 가족이 집을 지켜요

WIA 대가족의 든든한 울타리,
비바람이 와도 흔들리지 않는 집.
