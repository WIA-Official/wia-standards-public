7.2 WIA-TLS-LITE

경량 TLS
========

기존 TLS:
  - 복잡한 핸드셰이크
  - CPU 집약적
  - 자원 부담

WIA-TLS-LITE:
  - 간소화된 핸드셰이크
  - 최소 오버헤드
  - 개인 서버에 완벽


기능
====

[OK] 완전한 암호화 (AES-256-GCM)
[OK] 인증서 자동 갱신
[OK] 수동 설정 불필요
[OK] NAT 뒤에서도 작동
[OK] P2P 인증서 검증


작동 방식
=========

1. 키 생성
   - 설정 시 키쌍 생성
   - 공개키를 WIA DNS에 등록

2. 연결
   - 클라이언트 연결
   - 서버가 신원 증명
   - 암호화 채널 설정

3. 갱신
   - 만료 전 자동
   - 다운타임 없음
   - 자연스러움


인증서 표시
===========

방문자가 보는 것:
  브라우저에 자물쇠 아이콘
  "연결이 안전합니다"
  yoursite.wia.home 검증됨

기존 SSL과 동일한 신뢰.

7.3 DDoS 방어

속도 제한
=========

자동 보호:
  - IP당 최대 요청 수
  - 점진적 스로틀링
  - 임시 차단


기본 제한
=========

정상 트래픽:
  IP당 분당 100 요청

의심스러운 트래픽:
  분당 10 요청으로 스로틀링

공격 감지됨:
  임시 IP 차단


P2P 분산
========

공격 받을 때:
  1. 요청이 피어들에게 분산
  2. 피어들이 트래픽 흡수
  3. 내 서버 보호
  4. 공격자는 전체 네트워크와 대면


커스텀 설정
===========

home.security.ddos.configure({
  rateLimit: 100,           // 분당 요청
  throttleThreshold: 50,    // 감속 시점
  blockThreshold: 200,      // 차단 시점
  blockDuration: 3600       // 초
});


공격 로깅
=========

home.on('ddos_detected', (event) => {
  console.log('DDoS 감지됨:', event.data.sourceIP);
  console.log('차단된 요청:', event.data.blockedCount);
});

7.4 웹 애플리케이션 방화벽

공격 방지
=========

WAF가 차단:

SQL 인젝션:
  입력: ' OR '1'='1
  결과: 차단됨

XSS (크로스사이트 스크립팅):
  입력: 결과: 차단됨

CSRF (크로스사이트 요청 위조):
  유효하지 않은 토큰 요청
  결과: 차단됨

경로 탐색:
  입력: ../../../etc/passwd
  결과: 차단됨


WAF 설정
========

home.security.waf.configure({
  sqlInjection: true,
  xss: true,
  csrf: true,
  pathTraversal: true,
  fileUpload: {
    maxSize: 10 * 1024 * 1024,  // 10MB
    allowedTypes: ['image/*', 'application/pdf']
  }
});


폼 보호
=======

모든 폼 자동으로:

  [OK] 입력 살균
  [OK] 속도 제한
  [OK] 허니팟 필드


업로드 스캔
===========

파일 업로드:
  [OK] 유형 확인
  [OK] 크기 제한
  [OK] 악성코드 스캔 (AIR-SHIELD)
  [OK] 의심스러운 파일 격리

7.5 AIR-SHIELD 연동

이모가 집을 지킴
================

AIR-SHIELD (이모)가 제공:
  - 심층 패킷 검사
  - 악성코드 탐지
  - 행동 분석
  - 위협 인텔리전스


자동 보호
=========

home.security.airShield.configure({
  enabled: true,
  scanUploads: true,
  protectForms: true,
  blockMalicious: true
});


위협 탐지
=========

AIR-SHIELD가 탐지:
  [OK] 악성코드 업로드
  [OK] 봇 공격
  [OK] 크리덴셜 스터핑
  [OK] API 남용
  [OK] 데이터 유출


알림
====

home.on('security_alert', (event) => {
  console.log('위협 감지됨:', event.data.type);
  console.log('취해진 조치:', event.data.action);
  console.log('상세:', event.data.details);
});

    autoDelete: 90  // 일

    ccpa: true,
