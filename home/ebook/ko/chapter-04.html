제4장: 콘텐츠 관리

"홍익인간: 내 콘텐츠, 내가 통제, 플랫폼 제약 없음."

4.1 페이지 관리

페이지 구조
===========

모든 페이지에는:
  - 제목 (탭에 표시)
  - 슬러그 (URL 경로)
  - 콘텐츠 (HTML/Markdown)
  - 템플릿 (선택)
  - 메타 (SEO 정보)


페이지 생성
===========

await home.pages.create({
  title: '회사 소개',
  slug: 'about',
  content: `
    # 행복한 빵집 소개

    2010년부터 신선한 빵을
    구워왔습니다...
  `,
  meta: {
    description: '행복한 빵집에 대해 알아보세요',
    keywords: ['빵집', '빵', '서울']
  }
});


페이지 작업
===========

생성:
  await home.pages.create(pageData)

읽기:
  await home.pages.get('about')

수정:
  await home.pages.update('about', { content: newContent })

삭제:
  await home.pages.delete('about')

목록:
  await home.pages.list()


페이지 템플릿
=============

사용 가능한 템플릿:
  - default (표준 페이지)
  - landing (히어로 + 섹션)
  - blog-list (블로그 인덱스)
  - product-list (쇼핑몰 카탈로그)
  - contact (폼 + 지도)
  - gallery (이미지 그리드)

4.2 블로그 시스템

블로그 포스트 구조
==================

각 포스트에는:
  - 제목
  - 슬러그 (URL)
  - 콘텐츠 (Markdown)
  - 발췌문 (요약)
  - 대표 이미지
  - 작성자
  - 태그와 카테고리
  - 발행일


포스트 생성
===========

await home.blog.createPost({
  title: '새로운 사워도우 레시피',
  content: `
    # 새 사워도우 빵

    몇 달간의 테스트 끝에
    발표하게 되어 기쁩니다...
  `,
  excerpt: '새로운 사워도우 빵 소개',
  featuredImage: sourdoughImage,
  tags: ['빵', '사워도우', '신제품'],
  categories: ['공지', '레시피']
});


블로그 기능
===========

목록:
  - 시간순 정렬
  - 카테고리 필터링
  - 태그 필터링
  - 검색

개별 포스트:
  - 전체 콘텐츠
  - 관련 포스트
  - 공유 버튼
  - 댓글 (선택)

SEO:
  - 자동 메타 태그
  - 구조화 데이터
  - RSS 피드
  - 사이트맵

4.3 미디어 관리

지원 미디어 유형
================

이미지:
  - JPEG, PNG, GIF, WebP
  - 자동 최적화
  - 반응형 버전

비디오:
  - MP4, WebM
  - 스트리밍 지원
  - 썸네일 생성

오디오:
  - MP3, WAV, OGG
  - 플레이어 위젯

문서:
  - PDF, Word, Excel
  - 미리보기 지원


미디어 업로드
=============

const imageRef = await home.media.upload(imageFile);

// 반환:
{
  id: 'media_123',
  type: 'image',
  url: '/media/image_123.webp',
  filename: 'bread.jpg',
  mimeType: 'image/webp',
  size: 45000,
  width: 1200,
  height: 800,
  optimized: true
}


자동 최적화
===========

이미지:
  [OK] WebP로 변환 (50-80% 작아짐)
  [OK] 반응형 크기 생성
  [OK] 지연 로딩 속성
  [OK] 메타데이터에서 대체 텍스트

생성되는 크기 예시:

  - Large: 800x533

4.4 테마 엔진

테마 구조
=========

interface Theme {
  name: string;
  colors: {
    primary: '#3498db',
    secondary: '#2ecc71',
    background: '#ffffff',
    text: '#333333',
    ...
  };
  fonts: {
    heading: 'Noto Sans KR',
    body: 'Noto Sans KR',
    ...
  };
  spacing: { xs, sm, md, lg, xl };
  borderRadius: { none, sm, md, lg, full };
  shadows: { none, sm, md, lg, xl };
}


테마 적용
=========

await home.theme.apply({
  name: '따뜻한 빵집',
  colors: {
    primary: '#E8B87D',
    secondary: '#D4956A',
    background: '#FFF8F0'
  },
  fonts: {
    heading: '나눔고딕',
    body: 'Noto Sans KR'
  }
});


AI 테마 생성
============

설명에서:
  const theme = await home.theme.generate(
    '따뜻하고 아늑한 빵집 느낌, 갈색 톤'
  );

이미지에서:
  const palette = await home.theme.palette.fromImage(
    breadImage
  );

브랜드에서:
  const palette = await home.theme.palette.fromBrand(
    'Starbucks'
  );

4.5 분석

프라이버시 우선 분석
====================

WIA-HOME 분석:
  - 기본적으로 쿠키 없음
  - 개인 데이터 수집 안 함
  - 집계 통계만
  - GDPR/개인정보보호법 준수


사용 가능한 지표
================

방문자:
  - 총 방문자
  - 순 방문자
  - 재방문자
  - 현재 활성

페이지뷰:
  - 총 조회수
  - 페이지별 분류
  - 평균 체류 시간

출처:
  - 직접 트래픽
  - 검색 엔진
  - SNS
  - 참조 사이트

기기:
  - 데스크톱 vs 모바일 vs 태블릿
  - 브라우저
  - 운영체제


사용 예시
=========

// 통계 가져오기
const stats = await home.analytics.get({
  period: 'last_30_days'
});

console.log('총 방문자:', stats.visitors.total);
console.log('인기 페이지:', stats.topPages);
console.log('유입 경로:', stats.referrers);

// 실시간 모니터링
home.analytics.realtime((data) => {
  console.log('현재 방문자:', data.activeVisitors);
  console.log('현재 페이지:', data.currentPages);
});
