5.1 쇼핑몰 설정

쇼핑몰 활성화
=============

const home = new WIAHome({
  name: '행복한 빵집',
  features: {
    shop: true
  }
});

// 쇼핑몰 준비 완료!


상품 구조
=========

interface Product {
  name: string;
  price: number;
  currency: string;
  description: string;
  images: MediaRef[];
  inventory: number;
  sku?: string;
  categories: string[];
  variants?: ProductVariant[];
}


상품 추가
=========

await home.shop.addProduct({

  images: [cakeImage1, cakeImage2],

  categories: ['케이크', '초콜릿']
});


상품 옵션
=========

5.2 글로벌 결제 연동

지역별 결제
===========

WIA-HOME은 전 세계 결제 지원:


한국:
  - 카카오페이
  - 네이버페이
  - 토스페이
  - 페이코

아프리카 (모바일 머니):
  - M-Pesa (케냐, 탄자니아)
  - Airtel Money
  - Orange Money
  - MTN Money
  - Telebirr (에티오피아)

인도 (UPI):
  - UPI (10억 사용자)
  - Paytm
  - PhonePe
  - Google Pay India

동남아시아:
  - GrabPay
  - GoPay (인도네시아)
  - GCash (필리핀)
  - PromptPay (태국)
  - MoMo (베트남)

중남미:
  - Pix (브라질)
  - Mercado Pago
  - OXXO (멕시코)

글로벌:
  - Stripe
  - PayPal
  - 암호화폐


결제 설정
=========

home.shop.setPayment({
  // 한국
  korean: {
    kakaoPay: true,
    naverPay: true
  },

  // 아프리카
  africa: {
    mPesa: true,
    telebirr: true
  },

  // 계좌이체
  bankTransfer: {
    enabled: true,
    accounts: [{
      bank: '신한은행',
      number: '110-xxx-xxx',
      holder: '홍길동'
    }]
  }
});


왜 이렇게 많은 옵션?
====================

홍익인간은 모든 인류를 위함:

케냐에서:
  - M-Pesa 사용자: 3천만명+
  - 은행 계좌: 훨씬 적음
  - 모바일 머니가 은행

인도에서:
  - UPI 거래: 월 100억건
  - 즉시, 무료 송금
  - 모두가 사용

브라질에서:
  - Pix: 1.5억 사용자
  - 즉시, 24/7
  - 현금 대체

WIA-HOME은 그들 모두를 섬깁니다.

5.3 주문 관리

주문 구조
=========

interface Order {
  id: string;
  items: OrderItem[];
  customer: CustomerInfo;
  shipping: ShippingInfo;
  payment: PaymentInfo;
  status: OrderStatus;
  total: number;
  createdAt: Timestamp;
}


주문 상태 흐름
==============

대기 -> 확정 -> 처리중 -> 배송됨 -> 배달완료
  |                           |
  v                           v
취소됨                      환불됨


주문 이벤트
===========

home.on('order_received', async (event) => {
  const order = event.data;

  console.log('새 주문!');
  console.log('고객:', order.customer.name);
  console.log('총액:', order.total);

  // 알림 전송
  await sendKakaoNotification(order);
});


주문 작업
=========

// 주문 목록
const orders = await home.shop.orders.list({
  status: 'pending'
});

// 상태 업데이트
await home.shop.orders.updateStatus(
  orderId,
  'shipped',
  { trackingNumber: 'ABC123' }
);

// 환불 처리
await home.shop.orders.refund(orderId, {
  reason: '고객 요청',
  amount: 35000
});

5.4 재고 관리

자동 추적
=========

WIA-HOME 자동으로:
  - 주문 시 차감
  - 재고 부족 알림
  - 초과 판매 방지


재고 작업
=========

// 재고 확인
const stock = await home.shop.inventory.get(productId);

// 재고 설정
await home.shop.inventory.set(productId, 50);

// 조정 (추가/차감)
await home.shop.inventory.adjust(productId, -5, '오프라인 판매');


재고 부족 알림
==============

home.shop.inventory.setAlert({
  threshold: 5,
  notification: ['kakao', 'email']
});

// 재고가 5 이하가 되면:
// [알림] 초콜릿 케이크: 5개만 남음!


품절 처리
=========

옵션:
  - 상품 숨기기
  - "품절" 표시
  - 예약 주문 허용
  - "알림 신청" 표시

home.shop.setOutOfStockBehavior('notify');
// "재입고 알림 받기" 버튼 표시

====

[OK] 시간 절약

  • [OK] 상태 추적이 있는 완전한 주문 관리
  • [OK] 알림이 있는 자동 재고 관리
  • [OK] 네이버, 카카오, 배달앱과 플랫폼 연동
