제1장: 소개

"홍익인간: 집은 인권입니다. 현실 세계에서도, 디지털 세계에서도."

1.1 문제점

현재 웹사이트 비용
==================

오늘날 웹사이트를 만들려면:

1. 도메인 구매: 연 1-5만원
2. 호스팅 서버: 월 5-10만원
3. SSL 인증서: 연 0-20만원
4. 개발자 또는 학습 시간
5. 지속적인 유지보수

= 최소 연간 수십만원

소상공인의 현실:
  [!] 네이버, 인스타그램 등에 종속
  [!] 플랫폼이 정책 바꾸면? 적응 아니면 퇴출
  [!] 플랫폼이 망하면? 모든 것 사라짐
  [!] 디지털 "세입자" 신세


건물주 문제
===========

기존 웹 호스팅:

  나 -> 클라우드 제공업체 -> 내 웹사이트
              |
         매달 월세 납부
         그들의 규칙 따라야 함
         문 닫지 않기를 기도

당신은 웹사이트를 진정으로 소유하지 않습니다.
남의 건물에 세 들어 사는 것입니다.


플랫폼 종속
===========

인스타그램 셀러:
  [!] 알고리즘 변경 = 매출 하락
  [!] 계정 정지 = 사업 끝
  [!] 수수료 인상 = 마진 감소
  [!] 이동 불가 = 갇힘

쿠팡/네이버 셀러:
  [!] 15-30% 수수료
  [!] 리뷰 조작 당할 수 있음
  [!] 직접적인 고객 관계 없음
  [!] 플랫폼 자체와 경쟁

1.2 WIA 해결책

WIA-HOME 접근법
===============

WIA-HOME이 있으면:

1. 도메인: yourname.wia.home (무료)
2. 호스팅: 내 PC/노트북이 서버 (0원)
3. SSL: WIA-TLS-LITE 내장 (0원)
4. 개발: 클론하거나 말로 설명 (0원)
5. 유지보수: 자동 업데이트 (0원)

= 연간 0원

내 집이니까.
건물주가 없으니까.
영원히 내 것이니까.


작동 방식
=========

기존:
+----------+     +----------+     +----------+
|   사용자  | --> |  클라우드 | --> | 웹사이트  |
|          |     |  제공업체  |     |          |
+----------+     +----------+     +----------+
                      |
                  월세 지불
                  규칙 따라야 함

WIA-HOME:
+----------+     +----------+
|  방문자   | --> | 내 PC/   | <-- 내 집!
|          |     | 노트북    |     건물주 없음
+----------+     +----------+
                      |
                  P2P 네트워크
                  내가 통제


핵심 이점
=========

1. 건물주 없음
   - 내 PC가 서버
   - 내가 소유주
   - 월 비용 없음

2. 영원히 내 것
   - 플랫폼 망해도? 괜찮음
   - 회사 문 닫아도? 여전히 운영
   - 종속 없음

3. 무료
   - 호스팅: 0원
   - 도메인: 0원
   - 모든 것: 0원

4. 쉬움
   - URL 입력 = 5분 만에 웹사이트
   - 또는 원하는 것만 설명
   - 코딩 필요 없음

5. 안전
   - 이모 (AIR-SHIELD)가 보호
   - 내장 보안
   - 프라이버시 우선 설계

1.3 WIA 철학

홍익인간
========

모든 인류를 이롭게

집은 인권입니다.
디지털 세상에서도 마찬가지입니다.

모든 사람에게:
  - 자신을 표현할 공간이 있어야 합니다
  - 플랫폼에 종속되지 않아야 합니다
  - 영원히 자신의 것이어야 합니다


대상 사용자
===========

소상공인:
  - 빵집, 카페, 식당
  - 미용실, 병원
  - 소매점

프리랜서:
  - 디자이너, 사진작가
  - 개발자, 작가
  - 컨설턴트

학생/개인:
  - 포트폴리오 사이트
  - 개인 블로그
  - 프로젝트 쇼케이스

디지털 집을 원하는 모든 사람.


약속
====

WIA-HOME의 약속:

1. 건물주가 없어요 - 내 PC가 서버
2. 영원히 내 것이에요 - 플랫폼 독립
3. 돈 안 들어요 - 호스팅 영원히 무료
4. 쉬워요 - 5분 만에 런칭
5. 안전해요 - 가족이 집을 지켜요

WIA 대가족의 든든한 울타리,
비바람이 와도 흔들리지 않는 집.

사이트 생성 모드
================

모드 1: 클론 및 커스터마이징
  - 경쟁사 URL 입력
  - 자동 분석
  - 내 브랜딩으로 클론

모드 2: 의도 기반
  - 자연어로 설명
  - 아버지 (INTENT)가 설계
  - 자동 생성

모드 3: 템플릿
  - 업종별 템플릿 선택
  - 세부사항 커스터마이징
  - 바로 시작

모드 4: 마이그레이션
  - WordPress, Wix에서 가져오기
  - 네이버 블로그에서 가져오기
  - 모든 콘텐츠 유지
