제4장: 보험료 계산

"정확한 보험료 계산은 보험 시스템의 지속 가능성을 보장합니다. 너무 높으면 가입자가 없고, 너무 낮으면 보험사가 파산합니다. 보험수리학은 이 균형을 찾는 과학입니다."

— 보험수리의 핵심 원칙

4.1 보험수리 기초

보험수리학(Actuarial Science)은 위험과 불확실성을 수학적으로 평가하는 학문입니다.

🎯 보험료 계산 기본 공식

Premium = Expected Claims + Administrative Costs + Profit Margin + Risk Margin

보험료 = 예상 청구액 + 관리 비용 + 이윤 + 위험 마진

📊 주요 보험수리 개념

용어 정의 예시
Loss Ratio 청구액 / 보험료 0.85 (85%)
MLR Medical Loss Ratio (의료비 비율) ACA: 최소 80-85%
PMPM Per Member Per Month (회원당 월 비용) $500 PMPM
Morbidity 질병률 (인구 내 질병 발생률) 당뇨병 10%
Utilization 의료 서비스 이용률 연간 방문 4.5회

💰 보험료 구성 요소

표준 보험료 구성:

  • Pure Premium (순수 보험료) — 예상 청구액
    • Frequency (빈도) × Severity (심각도)
    • 예: 입원 확률 5% × 평균 비용 $10,000 = $500
  • Loading (부가 비용) — 25-35%
    • Administrative Expenses: 15-20%
    • Profit Margin: 3-5%
    • Risk/Contingency Margin: 5-10%
    • Taxes & Fees: 2-5%

📐 예상 청구액 계산

예시: 30세 비흡연 남성, 건강 상태 양호

1. 기본 의료비 (Base Medical Cost):
   - 외래 진료: 4회/년 × $150 = $600
   - 응급실: 0.2회/년 × $1,500 = $300
   - 입원: 0.05회/년 × $10,000 = $500
   - 처방약: 12회/년 × $50 = $600
   - 예방 검진: 1회/년 × $300 = $300
   Total Base Cost = $2,300 PMPM = $192

2. 위험 조정 (Risk Adjustment):
   - 연령 계수: 1.0 (30세 기준)
   - 성별 계수: 0.95 (남성)
   - 건강 상태: 0.90 (양호)
   Adjusted = $192 × 1.0 × 0.95 × 0.90 = $164

3. 부가 비용 (Loading 30%):
   Premium = $164 / 0.70 = $234 PMPM

월 보험료: $234
연 보험료: $2,808

4.2 위험 조정 (Risk Adjustment)

위험 조정은 개인의 건강 상태와 예상 의료비를 반영하여 보험료를 조정합니다.

🔍 위험 요인

1. 인구통계 요인

요인 범위 계수
연령 0-64세 0.3 - 3.0 (최대 3:1 비율)
흡연 흡연/비흡연 1.0 - 1.5 (최대 1.5:1)
지역 ZIP code 0.8 - 1.3 (지역별 의료비)

2. 만성 질환 (Chronic Conditions)

HCC (Hierarchical Condition Category) 모델:

  • 당뇨병 합병증 없음 — HCC 19, 위험 점수 +0.318
  • 당뇨병 합병증 있음 — HCC 18, 위험 점수 +0.368
  • 심부전 — HCC 85, 위험 점수 +0.331
  • 만성 폐쇄성 폐질환 (COPD) — HCC 111, 위험 점수 +0.328
  • 암 (전이성) — HCC 8, 위험 점수 +2.659

🧮 HCC 위험 점수 계산

예시: 65세 여성, Medicare 가입자

기본 점수:
- 연령 65-69세, 여성: 0.453
- 지역별 조정: 1.05

질환 점수 (HCC):
- 당뇨병 합병증: +0.368 (HCC 18)
- 고혈압: +0.000 (무증상, HCC 없음)
- 고콜레스테롤혈증: +0.114 (HCC 22)

총 위험 점수 = (0.453 + 0.368 + 0.114) × 1.05 = 0.981

Medicare 평균 비용 = $10,000/년
예상 비용 = $10,000 × 0.981 = $9,810/년

📊 위험 조정 모델 비교

모델 용도 특징
CMS-HCC Medicare Advantage 87개 HCC 카테고리
HHS-HCC ACA Marketplace 127개 HCC 카테고리
RxHCC Medicare Part D 처방약 중심
DxCG Commercial Insurance 진단 기반 그룹화

4.3 그룹 보험 요율 계산

그룹 보험은 고용주가 직원에게 제공하는 보험으로, 개인 보험과 다른 방식으로 요율을 계산합니다.

🏢 소규모 그룹 (Small Group, 1-50명)

ACA는 소규모 그룹에 대해 커뮤니티 요율(Community Rating)을 요구합니다:

  • 허용 요인 — 연령, 지역, 흡연, 가족 구성
  • 금지 요인 — 건강 상태, 청구 이력, 성별
  • 연령 비율 — 최대 3:1 (21세 vs 64세)

계산 예시

회사: ABC Corp, 직원 30명, 캘리포니아

1. 기본 요율 (Index Rate):
   - 지역 평균 PMPM: $450

2. 인구통계 조정:
   - 평균 연령: 40세 → 계수 1.2
   - 흡연율: 10% → 계수 1.02
   - 가족 비율: 60% → 계수 2.3

3. 조정 요율:
   $450 × 1.2 × 1.02 × 2.3 = $1,268 PMPM

4. 총 월 보험료:
   30명 × $1,268 = $38,040/월

🏭 대규모 그룹 (Large Group, 51+명)

경험 요율(Experience Rating) 사용 — 그룹의 실제 청구 이력 반영:

회사: XYZ Corp, 직원 500명

1. 과거 청구 데이터 (지난 12개월):
   - 총 청구액: $6,000,000
   - 총 회원 개월: 6,000 (500명 × 12개월)
   - 실제 PMPM: $1,000

2. 예상 청구 트렌드 (+5%):
   $1,000 × 1.05 = $1,050

3. 신뢰도 계수 (Credibility):
   - 그룹 크기 500명 → 80% 신뢰도
   - Manual Rate (산업 평균): $900
   - Blended = ($1,050 × 0.8) + ($900 × 0.2) = $1,020

4. 부가 비용 (25%):
   Premium = $1,020 / 0.75 = $1,360 PMPM

월 보험료: 500명 × $1,360 = $680,000

💡 자가 보험 (Self-Insured)

대기업은 자가 보험 방식으로 직접 청구를 지불하고, TPA(Third Party Administrator)만 고용합니다:

  • 장점 — 보험료 절감 (부가 비용 제거), 현금 흐름 개선
  • 단점 — 높은 청구 위험, 준비금 필요
  • Stop-Loss 보험 — 고액 청구 보호 ($250K+ per person)

4.4 언더라이팅 규칙

언더라이팅(Underwriting)은 위험을 평가하고 보험료를 결정하는 프로세스입니다.

📋 개인 보험 언더라이팅

ACA 이전 개인 보험은 의료 언더라이팅을 사용했으나, 현재 ACA 마켓플레이스는 금지합니다:

ACA 이전 ACA 이후
- 건강 상태 심사
- 기존 질환 제외
- 가입 거부 가능
- 높은 보험료 부과 		- 건강 상태 무관
- 기존 질환 보장
- 가입 보장
- 커뮤니티 요율

🏥 메디케어 어드밴티지 언더라이팅

Medicare Advantage는 가입을 거부할 수 없지만, HCC 위험 점수로 정부 지급액이 조정됩니다:

가입자 A: 위험 점수 0.5 (건강)
→ CMS 지급: $10,000 × 0.5 = $5,000/년

가입자 B: 위험 점수 2.5 (복합 만성 질환)
→ CMS 지급: $10,000 × 2.5 = $25,000/년

보험사는 위험 점수를 정확히 문서화하여 적절한 보상을 받아야 합니다.

🔬 AI 기반 예측 언더라이팅

최신 보험사는 머신러닝을 사용하여 위험을 예측합니다:

  • 예측 모델링
    • 과거 청구 패턴 분석
    • 약물 복용 순응도 추적
    • 응급실 방문 위험 예측
    • 만성 질환 악화 조기 경고
  • 대안 데이터 소스
    • 웨어러블 건강 데이터 (동의 시)
    • 소셜 결정 요인 (SDOH)
    • 환경 요인 (대기질, 식품 접근성)

보험료 계산의 투명성과 공정성은 WIA-MED-028의 핵심 가치입니다. 표준화된 위험 조정 방법론은 보험사와 가입자 모두에게 예측 가능성과 신뢰를 제공합니다.
