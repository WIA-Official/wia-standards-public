### 3. 연습 문제 형식

```typescript
interface Exercise {
  exercise_id: string;
  content_id: string;  // 연결된 학습 콘텐츠

  type: ExerciseType;

  metadata: {
    difficulty: number;  // 0-10
    estimated_time_seconds: number;
    skills_practiced: SkillTag[];
    cognitive_load: "low" | "medium" | "high";
  };

  prompt: {
    text: string;
    audio?: AudioFile;
    image?: ImageFile;
    video?: VideoFile;
  };

  answer_format: {
    type: "multiple_choice" | "fill_in_blank" | "open_ended" |
          "speech" | "drag_drop" | "matching";
    constraints?: any;
  };

  correct_answer: any;
  acceptable_answers?: any[];  // 허용 가능한 답변들

  feedback: {
    correct: FeedbackContent;
    incorrect: FeedbackContent;
    partial: FeedbackContent;
    hints: Hint[];
  };

  scoring: {
    max_points: number;
    partial_credit_allowed: boolean;
    time_bonus: boolean;
  };
}

type ExerciseType =
  | "multiple_choice"
  | "fill_in_blank"
  | "true_false"
  | "matching"
  | "ordering"
  | "dictation"
  | "translation"
  | "conversation"
  | "pronunciation"
  | "writing"
  | "listening_comprehension"
  | "reading_comprehension";

interface FeedbackContent {
  message: string;
  explanation?: string;
  example?: string;
  related_lesson?: string;
  encouragement?: string;

  // 적응형 피드백
  difficulty_adjustment?: "easier" | "harder" | "same";
  recommended_review?: string[];
}
```

## 핵심 요약

**주요 내용:**

1. **콘텐츠 메타데이터**: 언어 정보, 난이도, 학습 목표, 교육학적 접근, 라이선스 등 포괄적 메타데이터 정의.

2. **멀티모달 구조**: 텍스트, 오디오, 비디오, 이미지, 인터랙티브 연습을 통합한 풍부한 학습 경험 제공.

3. **진도 추적**: 스킬별 숙련도, 학습 이벤트 로그, 간격 반복 상태를 정밀하게 기록.

4. **평가 형식**: CEFR 기반 숙련도 평가, 진단 평가, AI 기반 자동 채점 결과 표준화.

5. **간격 반복**: SM-2 알고리즘 기반 플래시카드 데이터 구조로 효율적 복습 관리.

6. **API 응답**: 일관된 응답 구조로 시스템 간 상호운용성 보장.

## 복습 문제

1. WIA-LANG-008 콘텐츠 메타데이터의 주요 구성 요소를 나열하고, 각각이 시스템 상호운용성에 기여하는 방식을 설명하시오.

2. 멀티모달 학습 콘텐츠의 5가지 형식(텍스트, 오디오, 비디오, 이미지, 인터랙티브)을 설명하고, 각각의 교육학적 가치를 논하시오.

3. 학습자 프로필 데이터 구조를 설명하고, 개인화된 학습 경험 제공을 위해 어떤 정보가 수집되는지 분석하시오.

4. 발음 평가 API 응답의 4가지 스코어(accuracy, fluency, completeness, prosody)를 정의하고, 각각의 계산 방법을 설명하시오.

5. SM-2 간격 반복 알고리즘의 데이터 구조(ease_factor, interval, repetitions)를 설명하고, 복습 일정이 어떻게 동적으로 조정되는지 논하시오.

6. 숙련도 평가 결과의 진단적 통찰(diagnostic insights) 섹션이 학습자에게 제공하는 가치를 설명하고, AI가 어떻게 학습 우선순위를 결정하는지 논하시오.
