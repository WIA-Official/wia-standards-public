The Unified Principle: Integrated Information (Φ)

통일 원리: 통합정보 (Φ)

Consciousness Level ∝ Φ (Integrated Information)

Φ (Big Phi) represents the irreducible amount of integrated information in a system. It measures how much a system is more than the sum of its parts. PCI (Perturbational Complexity Index) provides a practical approximation for clinical use.

Φ (빅 파이)는 시스템의 환원 불가능한 통합 정보량을 나타냅니다. 시스템이 부분의 합보다 얼마나 더 큰지를 측정합니다. PCI(교란 복잡도 지수)는 임상적 사용을 위한 실용적인 근사치를 제공합니다.

Consciousness emerges when information is broadcast globally across brain regions via prefrontal-parietal networks.

정보가 전두-두정 네트워크를 통해 뇌 영역 전체에 전역적으로 방송될 때 의식이 발생합니다.

Consciousness requires meta-cognition: being aware that one is having an experience.

Implementation Example

구현 예시

// Example: Measuring Consciousness Index
import { ConsciousnessIndex } from '@wia/consciousness';

const measurement = new ConsciousnessIndex({
  subject_id: "patient-001",
  method: "TMS-EEG"
});

// Measure PCI (Perturbational Complexity Index)
const pci = await measurement.measurePCI();
console.log(`PCI Value: ${pci.value}`);  // e.g., 0.45

// Check consciousness state
if (pci.value >= 0.31) {
  console.log("Consciousness detected: MCS or higher");
} else {
  console.log("Below consciousness threshold");
}

// Get IIT metrics estimate
const iit = measurement.estimateIIT();
console.log(`Φ estimate: ${iit.phi_estimate} bits`);

// 예시: 의식 지수 측정
import { ConsciousnessIndex } from '@wia/consciousness';

const measurement = new ConsciousnessIndex({
  subject_id: "patient-001",
  method: "TMS-EEG"
});

// PCI(교란 복잡도 지수) 측정
const pci = await measurement.measurePCI();
console.log(`PCI 값: ${pci.value}`);  // 예: 0.45

// 의식 상태 확인
if (pci.value >= 0.31) {
  console.log("의식 감지됨: MCS 이상");
} else {
  console.log("의식 임계값 미만");
}

// IIT 지표 추정
const iit = measurement.estimateIIT();
console.log(`Φ 추정값: ${iit.phi_estimate} 비트`);
