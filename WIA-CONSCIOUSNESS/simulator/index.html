TMS-EEG Measurement Protocol

Protocol Steps

  1. Preparation: Place 60-channel EEG cap on subject
  2. TMS Setup: Position TMS coil over premotor cortex
  3. Baseline: Record 5 min baseline EEG
  4. Stimulation: Deliver 200 TMS pulses (0.5-1 Hz)
  5. Recording: Record EEG response (500ms post-pulse)
  6. Analysis: Compute PCI from averaged response

Integrated Information Theory (IIT 4.0)

Consciousness = Integrated Information (Φ)

  • Intrinsicality (exists for itself)
  • Information (specific experience)
  • Integration (unified whole)
  • Exclusion (definite borders)
  • Composition (structured)
Prediction: Posterior cortex is seat of consciousness

Global Neuronal Workspace (GNW)

Consciousness = Global Information Broadcast

  • Ignition (all-or-none)
  • Global broadcast
  • Prefrontal involvement
  • Late (>300ms) activity
  • Report accessibility
Prediction: Prefrontal-parietal network essential
