5.1 The Process Control Imperative

As semiconductor manufacturing pushes into sub-2nm territory, the margin for error shrinks to atomic dimensions. A single misaligned atom, an extra angstrom of film thickness, or an invisible particle can determine whether a chip works or fails. Metrology and inspection equipment provides the eyes of the fab, enabling the real-time visibility and control essential for achieving commercially viable yields.

The Cost of Defects

5.3 Critical Dimension Metrology

Critical dimension (CD) metrology measures the width of features that define transistor performance. At 2nm nodes, CD must be controlled to within 0.5nm across billions of transistors.

CD-SEM (Scanning Electron Microscopy)

CD-SEM remains the workhorse for direct CD measurement, providing high-resolution images of patterned structures:

Specification Current Generation Next Generation
Resolution 1-2nm <1nm
Precision (3-sigma) 0.3-0.5nm <0.3nm
Throughput 20-40 WPH 40-60 WPH
Beam Energy 300-800eV 100-500eV
Sample Damage Minimal Ultra-low

Scatterometry (Optical CD)

Scatterometry measures CD and 3D profiles by analyzing diffraction patterns from periodic structures:

Scatterometry Advantages

  • Non-Destructive: No beam-induced damage or charging
  • High Throughput: 100+ wafers per hour possible
  • 3D Information: Measures height, sidewall angle, and profile
  • In-Line Capable: Suitable for process control feedback
  • Low Cost of Ownership: Compared to CD-SEM

Hybrid Metrology

Advanced nodes require combining multiple techniques to fully characterize complex structures:

  • CD-SEM + Scatterometry: Reference measurement plus high throughput
  • AFM Calibration: Atomic force microscopy for absolute reference
  • X-ray Techniques: CD-SAXS for buried structures
  • Model-Based Inference: Machine learning for virtual metrology

5.4 Overlay Metrology

Overlay metrology measures the alignment accuracy between lithography layers. At 2nm nodes, overlay budgets are below 2nm, requiring measurement precision better than 0.2nm.

Overlay Budget Analysis

Node Total Overlay Budget Measurement Precision Technology
7nm 4nm 0.4nm Standard EUV
5nm 3nm 0.3nm Standard EUV
3nm 2nm 0.2nm EUV + corrections
2nm 1.5nm 0.15nm High-NA EUV

Imaging-Based Overlay (IBO)

IBO measures overlay by imaging dedicated targets using bright-field microscopy:

  • Target Types: AIM, BBOW, and advanced scatterometry targets
  • Wavelength: Multiple wavelengths for robustness
  • Throughput: 100+ wafers per hour
  • Precision: Sub-0.2nm demonstrated

Diffraction-Based Overlay (DBO)

DBO extracts overlay from the diffraction signature of grating-based targets:

DBO Advantages for Advanced Nodes

  • Less sensitive to asymmetric process variations
  • Better correlation to device overlay
  • Smaller target sizes possible
  • Multiple measurements per target for redundancy

5.5 Film Metrology

Thin film metrology measures thickness, composition, and optical properties of deposited layers. At advanced nodes, films as thin as 1-2 atomic layers must be characterized with angstrom precision.

Spectroscopic Ellipsometry

Ellipsometry measures how polarized light changes upon reflection from thin films:

Capability Performance Application
Thickness Range 0.1nm - 100um All film types
Precision <0.1 angstrom Gate oxide control
Spectral Range 150-2500nm Multi-layer analysis
Throughput 50-100 WPH In-line monitoring

X-Ray Techniques

  • XRF (X-ray Fluorescence): Elemental composition and film thickness
  • XRR (X-ray Reflectivity): Ultra-thin film thickness and density
  • XRD (X-ray Diffraction): Crystalline structure and stress
  • CD-SAXS: 3D metrology of buried structures

5.6 Defect Inspection

Defect inspection systems identify manufacturing defects that can cause chip failures. The challenge is detecting defects smaller than the features themselves while maintaining throughput for high-volume manufacturing.

Optical Inspection

Broadband optical inspection remains the workhorse for high-throughput defect detection:

Optical Inspection Specifications

  • Light Source: Broadband plasma (BB) or DUV laser
  • Resolution: 20-30nm detection capability
  • Throughput: 50-100 wafers per hour
  • Sensitivity: Defects 1/3 of design rule
  • Mode: Bright-field, dark-field, or combined

E-beam Inspection

Electron beam inspection provides the highest resolution for detecting the smallest defects:

Parameter Single Beam Multi-beam
Resolution 5-10nm 5-10nm
Throughput 1-5 WPH 10-20 WPH
Coverage Sample areas Larger areas
Defect Types Physical + electrical Physical + electrical

Defect Classification

  • Particles: Foreign material on wafer surface
  • Pattern Defects: Opens, shorts, bridges, CD variations
  • Scratches: Mechanical damage from handling
  • Residue: Process byproducts remaining after clean
  • Voids: Missing material in filled structures
  • Electrical Defects: Leakage paths, high resistance

AI-Powered Classification

Machine Learning in Inspection

Modern inspection systems use deep learning for automatic defect classification with accuracy exceeding 95%. AI enables:

  • Automatic nuisance filtering (false positive reduction)
  • Defect source identification for root cause analysis
  • Predictive yield modeling from inspection data
  • Real-time process excursion detection

5.7 Review and Analysis Systems

After inspection identifies defect locations, review systems provide high-resolution characterization for root cause analysis.

Defect Review SEM

Dedicated review SEMs provide high-resolution imaging of flagged defects:

  • Resolution: Sub-nanometer imaging
  • Throughput: 200-500 defects per hour
  • Navigation: Automatic defect relocation from inspection coordinates
  • Analysis: EDX for elemental composition

TEM Analysis

Transmission electron microscopy provides atomic-resolution cross-section imaging:

  • Sample Preparation: FIB-based lamella extraction
  • Resolution: Atomic columns visible
  • Analysis: Crystal structure, interfaces, defects
  • Throughput: 1-5 samples per day (destructive)

5.8 WIA-SEMI-EQP-001 Metrology Standards

The WIA-SEMI-EQP-001 standard defines comprehensive data formats and interfaces for metrology equipment integration:

Data Exchange Format

// WIA-SEMI-EQP-001 Metrology Data Interface
interface MetrologyEquipment {
    // Equipment identification
    equipmentId: string;
    metrologyType: 'CD' | 'OVERLAY' | 'FILM' | 'DEFECT';
    vendor: string;
    model: string;
    
    // Measurement results
    getMeasurements(): MeasurementResult[];
    getRecipe(): MetrologyRecipe;
    getSiteMap(): SiteDefinition[];
    
    // Quality metrics
    getPrecision(): PrecisionData;
    getAccuracy(): AccuracyData;
    getTMU(): number; // Total Measurement Uncertainty
}

interface MeasurementResult {
    parameter: string;
    value: number;
    uncertainty: number;
    site: SiteLocation;
    timestamp: Date;
    unit: string;
}

interface DefectData {
    defectId: string;
    location: {x: number, y: number};
    size: number;
    classification: string;
    confidence: number;
    image?: ImageData;
}

Integration Requirements

  • KLARF Format: Standard defect data exchange format
  • STDF Compliance: Test data format compatibility
  • Yield Management: Integration with yield analysis systems
  • SPC Integration: Automatic limit monitoring and alarms
