Implementation Guide

Getting Started

This chapter provides practical guidance for implementing digital wallets using the WIA-FIN-015 standard. We'll cover SDK integration, API usage, and real-world code examples.

Installation

NPM/Yarn

npm install @wia/digital-wallet-sdk
# or
yarn add @wia/digital-wallet-sdk

CDN (Browser)

<script src="https://cdn.wia.org/digital-wallet/v1.0.0/wallet.min.js"></script>

Basic Wallet Creation

import { DigitalWallet, WalletType } from '@wia/digital-wallet-sdk';

// Create a new HD wallet
const wallet = await DigitalWallet.create({
  type: WalletType.HD,
  currencies: ['USD', 'EUR', 'BTC', 'ETH'],
  security: {
    encryption: '256-bit',
    biometric: true,
    twoFactor: true
  }
});

console.log('Wallet created:', wallet.address);
console.log('Recovery phrase:', wallet.recoveryPhrase);
// IMPORTANT: Store recovery phrase securely!

Loading Existing Wallet

// From recovery phrase
const wallet = await DigitalWallet.fromRecoveryPhrase(
  'word1 word2 word3 ... word12',
  {
    currencies: ['USD', 'BTC', 'ETH']
  }
);

// From private key
const wallet = await DigitalWallet.fromPrivateKey(
  '0x1234567890abcdef...',
  {
    type: WalletType.SIMPLE
  }
);

Checking Balance

// Single currency balance
const usdBalance = await wallet.getBalance('USD');
console.log(`USD Balance: $${usdBalance.available}`);

// All balances
const allBalances = await wallet.getAllBalances();
allBalances.forEach(balance => {
  console.log(`${balance.currency}: ${balance.total}`);
});

NFC Payment Integration

Enable NFC

// Check NFC availability
if (await wallet.nfc.isAvailable()) {
  // Enable NFC payments
  const nfc = await wallet.nfc.enable({
    merchantId: 'merchant_123',
    defaultCurrency: 'USD'
  });

  // Handle tap events
  nfc.on('tap', async (terminal) => {
    console.log('NFC tap detected');

    const payment = await wallet.processNFCPayment({
      terminal,
      amount: terminal.amount,
      currency: terminal.currency,
      authenticate: 'biometric'
    });

    if (payment.success) {
      console.log('Payment completed!');
      terminal.showSuccess();
    }
  });
}

QR Code Payments

Generate QR Code

const qr = await wallet.qr.generate({
  type: 'payment_request',
  amount: 50.00,
  currency: 'USD',
  merchant: 'Online Store',
  orderId: 'ORD-12345'
});

// Get QR code as image
const qrImage = await qr.toImage({
  format: 'png',
  size: 300
});

// Display in HTML
document.getElementById('qr-code').src = qrImage;

Scan QR Code

// Start QR scanner
const scanner = await wallet.qr.startScanner({
  camera: 'back',
  continuous: false
});

scanner.on('detected', async (qrData) => {
  // Parse QR code data
  const payment = await wallet.qr.parsePaymentRequest(qrData);

  // Show confirmation to user
  const confirmed = await showPaymentConfirmation(payment);

  if (confirmed) {
    const result = await wallet.send(payment);
    console.log('Payment sent:', result.id);
  }

  scanner.stop();
});

Security Features

Enable Biometric Authentication

// Check biometric availability
const biometric = await wallet.security.biometric;

if (biometric.isAvailable()) {
  await biometric.enable({
    type: 'face', // or 'fingerprint'
    fallback: 'pin'
  });

  console.log('Biometric auth enabled');
}

Setup Two-Factor Authentication

// Generate TOTP secret
const totp = await wallet.security.enable2FA({
  method: 'totp',
  phoneNumber: '+1234567890'
});

// Display QR code for authenticator app
console.log('Scan this QR code:', totp.qrCode);

// Verify setup
const verified = await totp.verify(userCode);
if (verified) {
  console.log('2FA enabled successfully');
}

DeFi Integration

Connect to dApp

import { WalletConnect } from '@wia/digital-wallet-sdk';

// Initialize WalletConnect
const wc = new WalletConnect({
  bridge: 'https://bridge.walletconnect.org',
  qrcodeModal: QRCodeModal
});

// Create connection
await wc.createSession();

// Handle connection
wc.on('connect', (error, payload) => {
  if (!error) {
    const { accounts, chainId } = payload.params[0];
    console.log('Connected to dApp');
  }
});

Error Handling

try {
  const tx = await wallet.send({
    to: recipient,
    amount: 1000,
    currency: 'USD'
  });
} catch (error) {
  if (error.code === 'INSUFFICIENT_BALANCE') {
    console.error('Not enough funds');
  } else if (error.code === 'INVALID_ADDRESS') {
    console.error('Invalid recipient address');
  } else if (error.code === 'AUTHENTICATION_FAILED') {
    console.error('Authentication required');
  } else {
    console.error('Transaction failed:', error.message);
  }
}
