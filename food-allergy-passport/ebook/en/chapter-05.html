Chapter 5: International Travel

Multi-language translation and global food safety

5.1 International Travel Support

Traveling internationally with food allergies presents unique challenges: language barriers, unfamiliar ingredients, different labeling standards, and varying levels of allergy awareness. The WIA Food Allergy Passport International Travel module addresses these challenges through multi-language translation, regional allergen databases, airline integration, and customs documentation support.

5.3 International Travel API Endpoints

POST/api/v1/travel/translation-card

Generate multi-language allergy translation card for specified countries.

GET/api/v1/travel/restaurants/{country}

Find allergy-aware restaurants in destination country.

POST/api/v1/travel/airline/notify

Send allergy notification to airline for upcoming flight.

GET/api/v1/travel/emergency-numbers/{country}

Get emergency service numbers and hospital information for destination.

GET/api/v1/travel/regional-allergens/{country}

Retrieve region-specific allergen information and common ingredients.

POST/api/v1/travel/customs-letter

Generate customs declaration letter for traveling with EpiPens.

GET/api/v1/travel/safe-foods/{country}

Get list of pre-verified safe foods and brands in destination country.

POST/api/v1/travel/itinerary

Create allergy-safe travel itinerary with vetted restaurants and accommodations.

GET/api/v1/travel/pharmacy-locator/{city}

Find pharmacies that stock emergency medications in destination city.

POST/api/v1/travel/hotel-notification

Send allergy notification to hotel for safe meal preparation.

Code Example: Translation Card Generation

// JavaScript - Multi-language Translation Card
const axios = require('axios');
const PDFDocument = require('pdfkit');

async function generateTravelCard(profileId, destinationCountries, languages) {
  const response = await axios.post(
    'https://api.wia-foodallergy.org/api/v1/travel/translation-card',
    {
      profileId: profileId,
      targetLanguages: languages,
      destinationCountries: destinationCountries,
      includeEmergencyPhrases: true,
      includeRestaurantCard: true,
      includeQRCode: true,
      cardFormat: 'WALLET_SIZE'
    },
    {headers: {'Authorization': `Bearer ${authToken}`}}
  );

  const cardData = response.data;

  // Generate PDF travel card
  const doc = new PDFDocument({size: [243, 153]}); // Credit card size
  doc.fontSize(16).text('FOOD ALLERGY ALERT', {align: 'center'});
  
  cardData.allergiesTranslated.forEach(allergy => {
    doc.fontSize(12).text(`${allergy.allergenNameEnglish}:`, {underline: true});
    
    languages.forEach(lang => {
      const translation = allergy.translations[lang];
      doc.fontSize(10).text(`${lang.toUpperCase()}: ${translation.name}`);
    });
  });

  doc.addPage();
  doc.fontSize(14).text('Emergency Phrases:', {underline: true});
  Object.entries(cardData.emergencyPhrases[languages[0]]).forEach(([key, phrase]) => {
    doc.fontSize(10).text(phrase);
  });

  return doc;
}

// Usage
const card = await generateTravelCard(
  'profile-uuid',
  ['JP', 'KR', 'CN'],
  ['ja', 'ko', 'zh']
);
