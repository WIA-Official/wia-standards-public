Chapter 4: Emergency Protocols

Anaphylaxis response and automated emergency systems

4.1 Emergency Response Architecture

The WIA Food Allergy Passport Standard includes comprehensive emergency protocols designed to save lives during anaphylactic reactions. The system integrates with emergency services (911/EMS), provides automated notifications to emergency contacts, and guides both bystanders and trained responders through critical intervention steps.

Anaphylaxis can progress from initial symptoms to life-threatening shock within minutes. The emergency protocol system is designed to minimize response time through automated alerts, clear instructions, and immediate access to critical medical information without authentication barriers. Every second counts, and this system ensures that information flows instantly to those who need it.

Emergency Response Severity Levels

Level Symptoms Automatic Actions Required Response
LEVEL 1 - CRITICAL Difficulty breathing, throat swelling, loss of consciousness, severe hypotension Auto-call 911, alert all emergency contacts, activate nearby AED network IMMEDIATE EpiPen administration, CPR if needed, call 911
LEVEL 2 - SEVERE Widespread hives, facial swelling, wheezing, rapid pulse Alert emergency contacts, prepare 911 call, EpiPen reminder Administer EpiPen, call 911, monitor for progression
LEVEL 3 - MODERATE Localized hives, mild throat itching, stomach discomfort Alert emergency contact, antihistamine reminder, symptom tracker Antihistamine, monitor closely, prepare for escalation
LEVEL 4 - MILD Oral tingling, mild skin reaction Log incident, symptom monitoring Antihistamine, document trigger, consult allergist

First Responder Information Access

Information Type Access Method Available Data Authentication Required
Critical Allergy Info QR code scan, emergency URL All allergies, recent reactions, EpiPen location No
Medical History QR code + emergency PIN Past reactions, treatments, hospital records 4-digit PIN (optional)
Current Medications QR code scan All medications, dosages, expiration dates No

4.3 Emergency Protocol API Endpoints

POST /api/v1/emergency/trigger

Trigger emergency alert with automatic notifications to emergency contacts and optional 911 call.

GET /api/v1/emergency/{alertId}

Retrieve emergency alert details including current status and response log.

POST /api/v1/emergency/{alertId}/epipen-administered

Log EpiPen administration with timestamp and person who administered.

POST /api/v1/emergency/{alertId}/update-symptoms

Update symptom severity and progression for real-time monitoring.

POST /api/v1/emergency/{alertId}/responder-checkin

Allow emergency contacts or first responders to check in and update status.

GET /api/v1/emergency/{alertId}/location-track

Get real-time location tracking for patient and emergency contacts.

POST /api/v1/emergency/{alertId}/resolve

Mark emergency as resolved with outcome documentation.

GET /api/v1/emergency/profile/{profileId}/protocol

Retrieve complete emergency protocol for profile without authentication (emergency access).

GET /api/v1/emergency/nearby-aed

Find nearby AED locations using current GPS coordinates.

GET /api/v1/emergency/nearby-hospitals

Find nearby hospitals with allergy specialists and emergency departments.

POST /api/v1/emergency/test-alert

Send test emergency alert to verify notification system without triggering actual emergency.

POST /api/v1/emergency/epipen-expiration-check

Check EpiPen expiration status and send reminder if expiring within 30 days.

Code Example: Emergency Alert Trigger

// JavaScript - Emergency Alert System
const axios = require('axios');

async function triggerEmergencyAlert(profileId, symptoms, location, options = {}) {
  try {
    const response = await axios.post(
      'https://api.wia-foodallergy.org/api/v1/emergency/trigger',
      {
        profileId: profileId,
        alertLevel: 'LEVEL_1_CRITICAL',
        reportedSymptoms: symptoms,
        suspectedAllergen: options.suspectedAllergen,
        location: {
          latitude: location.latitude,
          longitude: location.longitude,
          accuracy: location.accuracy,
          locationName: options.locationName
        },
        epiPenAdministered: options.epiPenAdministered || false,
        triggerMethod: 'MANUAL_USER_ACTIVATION',
        automaticActions: {
          notifyEmergencyContacts: true,
          call911Automatically: options.auto911 || false,
          shareLocation: true,
          alertNearbyAED: true,
          launchHospitalNavigation: true
        }
      },
      {
        headers: {
          'Content-Type': 'application/json',
          'X-API-Version': '1.0.0',
          'X-Emergency-Override': 'true'
        }
      }
    );

    const alert = response.data;
    console.log('EMERGENCY ALERT TRIGGERED');
    console.log('Alert ID:', alert.alertId);
    console.log('Emergency contacts notified:', alert.contactsNotified);
    console.log('Live tracking URL:', alert.trackingUrl);

    return alert;
  } catch (error) {
    console.error('Emergency alert failed:', error);
    // Even if API fails, try local emergency actions
    fallbackEmergencyActions(profileId, location);
    throw error;
  }
}

function fallbackEmergencyActions(profileId, location) {
  // Offline emergency fallback
  console.log('OFFLINE EMERGENCY MODE ACTIVATED');
  
  // Attempt to retrieve cached emergency contacts
  const cachedProfile = localStorage.getItem(`emergency_${profileId}`);
  if (cachedProfile) {
    const profile = JSON.parse(cachedProfile);
    
    // Show emergency instructions
    displayEmergencyInstructions(profile);
    
    // Attempt to call emergency contact if phone API available
    if (navigator.contacts) {
      profile.emergencyContacts.forEach(contact => {
        console.log(`Call: ${contact.name} - ${contact.phone}`);
      });
    }
  }
}

// Usage - triggered from emergency button
navigator.geolocation.getCurrentPosition(async (position) => {
  const alert = await triggerEmergencyAlert(
    'customer-profile-uuid',
    ['DIFFICULTY_BREATHING', 'THROAT_SWELLING', 'HIVES'],

      locationName: 'Restaurant Name',
      epiPenAdministered: true,
      auto911: false  // User will call manually
    }
  );
});

Code Example: First Responder Interface

// Python - First Responder Emergency Access
import requests
from datetime import datetime

class FirstResponderInterface:
    def __init__(self):
        self.base_url = 'https://api.wia-foodallergy.org'
        
    def access_emergency_profile(self, qr_code_id):
        """Access emergency profile without authentication"""
        try:
            response = requests.get(
                f'{self.base_url}/api/v1/emergency/qr/{qr_code_id}',
                headers={
                    'X-Responder-Type': 'FIRST_RESPONDER',
                    'X-Emergency-Access': 'true'
                }
            )
            response.raise_for_status()
            return response.json()
        except Exception as e:
            print(f'Emergency access failed: {e}')
            raise
    
    def display_emergency_info(self, emergency_data):
        """Display critical emergency information for paramedics"""
        print("\n" + "="*70)
        print("🚨 FOOD ALLERGY EMERGENCY INFORMATION 🚨")
        print("="*70)
        
        patient = emergency_data['patientInfo']
        print(f"\nPATIENT: {patient['name']}")
        print(f"AGE: {patient['age']}  |  BLOOD TYPE: {patient['bloodType']}")
        print(f"WEIGHT: {patient['weight']} {patient['weightUnit']}")
        
        print("\n⚠️  CRITICAL ALLERGIES:")
        for allergy in emergency_data['criticalAllergies']:
            print(f"  • {allergy['allergen']} - {allergy['severityLevel']}")
            print(f"    Typical reaction time: {allergy['typicalReactionTime']}")
            print(f"    Symptoms: {', '.join(allergy['symptoms'])}")
            if allergy['lastReaction']:
                print(f"    Last reaction: {allergy['lastReaction']['date']}")
                print(f"    Treatment: {allergy['lastReaction']['treatment']}")
        
        print("\n💉 EMERGENCY MEDICATIONS:")
        for med in emergency_data['emergencyMedications']:
            print(f"  • {med['brandName']} ({med['genericName']})")
            print(f"    Dosage: {med['dosage']} {med['route']}")
            print(f"    Location: {med['location']}")
            if 'expirationDate' in med:
                print(f"    Expires: {med['expirationDate']}")
        
        print("\n📞 EMERGENCY CONTACTS:")
        for contact in emergency_data['emergencyContacts']:
            primary = "PRIMARY" if contact['isPrimary'] else ""
            print(f"  • {contact['name']} ({contact['relationship']}) {primary}")
            print(f"    Phone: {contact['phone']}")
        
        print("\n🏥 HOSPITAL PREFERENCES:")
        for hospital in emergency_data['hospitalPreferences'][:2]:
            print(f"  {hospital['preferenceRank']}. {hospital['name']}")
            print(f"     {hospital['address']}")
            print(f"     Distance: {hospital['distance']} | Phone: {hospital['phone']}")
        
        print("\n📋 ADDITIONAL INSTRUCTIONS:")
        print(f"   {emergency_data['additionalInstructions']}")
        
        print("\n" + "="*70 + "\n")
    
    def log_responder_action(self, alert_id, action_description):
        """Log first responder actions during emergency"""
        try:
            response = requests.post(
                f'{self.base_url}/api/v1/emergency/{alert_id}/responder-checkin',
                json={
                    'timestamp': datetime.utcnow().isoformat(),
                    'responder': 'EMS Paramedics',
                    'action': action_description,
                    'responderType': 'FIRST_RESPONDER'
                }
            )
            response.raise_for_status()
            print(f'Action logged: {action_description}')
        except Exception as e:
            print(f'Failed to log action: {e}')

# Usage
responder = FirstResponderInterface()

# Access emergency profile from QR code
emergency_data = responder.access_emergency_profile('qr-code-uuid-from-scan')

# Display critical information
responder.display_emergency_info(emergency_data)

# Log arrival
responder.log_responder_action(
    emergency_data['activeAlertId'],
    'EMS arrived on scene, patient conscious, EpiPen already administered'
)

CERT-EMERGENCY-019: Voice-activated emergency trigger must be available for hands-free activation.
CERT-EMERGENCY-020: Emergency data must be printable for inclusion in go-bags and emergency kits.
CERT-EMERGENCY-021: Multi-language emergency instructions required for international travel (minimum 12 languages).
CERT-EMERGENCY-022: Integration with Apple Health and Google Fit for emergency data sharing.

CERT-EMERGENCY-024: System must log all emergency access attempts for security and legal compliance.
