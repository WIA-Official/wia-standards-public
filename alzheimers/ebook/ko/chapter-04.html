Phase 1: 데이터 형식 표준

WIA-ALZHEIMERS 표준의 첫 번째 단계는 알츠하이머병 데이터의 표준화된 형식을 정의하는 것입니다. 이 장에서는 NAD+ 항상성 지수, 병리 프로파일, 치료 반응 추적을 위한 JSON 스키마와 검증 규칙을 상세히 설명합니다.

1. 왜 데이터 형식 표준화가 필요한가?

현재 알츠하이머 연구와 치료 현장에서는 각 기관이 서로 다른 방식으로 데이터를 수집하고 저장합니다. 이는 다음과 같은 심각한 문제를 야기합니다:

  • 데이터 통합 불가: 서로 다른 형식으로 인해 여러 기관의 데이터를 통합 분석할 수 없음
  • 연구 재현성 저하: 동일한 데이터를 다르게 해석하여 연구 결과의 일관성 부족
  • 시스템 간 상호운용성 부재: 병원, 연구소, 제약회사 간 데이터 교환이 거의 불가능
  • AI/ML 적용 어려움: 비정형 데이터로 인한 기계학습 모델 개발 지연
  • 비용 증가: 데이터 변환 및 정제에 과도한 시간과 비용 소요
弘益人間 철학의 실천

WIA-ALZHEIMERS 데이터 형식 표준은 전 세계 어느 병원, 어느 연구소에서든 동일한 방식으로 데이터를 기록하고 공유할 수 있도록 함으로써, 인류 전체가 알츠하이머병과의 싸움에서 협력할 수 있는 기반을 마련합니다.

2. 루트 스키마 구조

3. NAD+ 항상성 지수 스키마

NAD+ 항상성 지수는 WIA-ALZHEIMERS 표준의 가장 혁신적인 부분으로, NAD+ 대사, 효소 활성, 미토콘드리아 기능을 통합적으로 표현합니다.

NAD+ 대사 (nad_metabolism)

NAD+ 및 관련 대사산물의 농도를 기록합니다.

"nad_metabolism": { // NAD+ 농도 (필수) "nad_plus": { "value": 22.5, "unit": "μM", "tissue": "blood", // blood | csf | brain | plasma "method": "HPLC-MS/MS", // 측정 방법 "timestamp": "2025-12-29T09:15:00Z" }, // NADH 농도 (필수) "nadh": { "value": 4.5, "unit": "μM" }, // NADH/NAD+ 비율 (필수) "nadh_nad_ratio": { "value": 0.2, "optimal_range": [0.1, 0.3] }, // 선택사항: 추가 NAD+ 관련 대사산물 "nadp_plus": { "value": 15.2, "unit": "μM" }, "nadph": { "value": 3.8, "unit": "μM" }, "nicotinamide": { "value": 8.2, "unit": "μM" }, "nmn": { "value": 0.8, "unit": "μM" }, "nr": { "value": 0.3, "unit": "μM" } }

NAD+ 생합성에 관여하는 효소들의 활성과 발현 수준을 기록합니다.

"nad_synthesizing_enzymes": { // NAMPT: 회수 경로의 속도 제한 효소 "nampt": { "activity": 0.7, // 정규화된 활성 (0-1) "expression": "normal", // low | normal | elevated | high "protein_level": 85.2 // ng/mL (선택사항) }, // NADSYN1: NAD 신테타제 "nadsyn1": { "activity": 0.65, "expression": "normal" }, // NMNAT 아형들 "nmnat1": { "activity": 0.75, "expression": "normal" }, // 핵 "nmnat2": { "activity": 0.6, "expression": "low" }, // Golgi/axon "nmnat3": { "activity": 0.55, "expression": "normal" } // 미토콘드리아 }

NAD+를 소비하는 효소들의 활성을 기록하며, 이들의 균형이 NAD+ 항상성을 결정합니다.

"nad_consuming_enzymes": { // 시르투인 (유익한 NAD+ 소비) "sirt1": { "activity": 0.75, "expression": "normal" }, "sirt2": { "activity": 0.68, "expression": "normal" }, "sirt3": { "activity": 0.72, "expression": "normal" }, // PARP (과활성화 시 유해) "parp1": { "activity": 0.4, "expression": "elevated" }, "parp4": { "activity": 0.35, "expression": "normal" }, // 기타 NAD+ 소비 효소 "nadk2": { "activity": 0.5, "expression": "normal" }, "cd38": { "activity": 0.55, "expression": "elevated" } // 염증 시 증가 }

미토콘드리아의 기능 상태를 평가하는 핵심 지표들을 기록합니다.

"mitochondrial_function": { // 미토콘드리아 막 전위 (정상: -140 to -180 mV) "membrane_potential_mv": -140, // ATP 생산량 "atp_production": { "value": 85, "unit": "μmol/min/mg", "tissue": "brain" }, // 활성산소종 (ROS) 수준 (정규화, 낮을수록 좋음) "ros_level": { "value": 0.35, "max": 1.0 }, // 산화 스트레스 (정규화, 낮을수록 좋음) "oxidative_stress": { "value": 0.4, "max": 1.0 }, // 미토콘드리아 펼쳐진 단백질 반응 (UPR^mt) 활성화 "upmt_activation": { "atf4": 0.3, // 전사 인자 "hsp60": 0.4, // 샤페론 "lonp1": 0.35 // 프로테아제 } }
