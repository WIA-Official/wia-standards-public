# 1장: 해양 생물학 데이터 표준 소개

## 해양 과학 데이터 관리의 기초

해양 생물학은 다양한 출처에서 방대한 양의 데이터를 생성합니다: 수중 센서, 유전체 시퀀서, 위성 관측, 연구선, 시민 과학 프로그램 등. 이러한 데이터 홍수를 관리하려면 전 세계 연구자들이 데이터 품질, 상호운용성, 접근성을 보장할 수 있는 강력한 표준이 필요합니다.

### 해양 생물학 데이터의 규모

해양은 지구 표면의 71%를 차지하며 약 **220만 종의 해양 생물**을 포함하고 있으며, 현재 24만 종만이 문서화되어 있습니다. 각 관측, 측정, 분석은 해양 생태계에 대한 우리의 이해에 기여하지만, 데이터가 적절하게 구조화되고 공유되며 보존될 때만 가능합니다.

**연간 해양 생물학 데이터 생성량 (2025년):**
- 해양 센서: **500페타바이트**의 환경 데이터
- DNA 시퀀싱: **100페타바이트**의 유전체 데이터
- 위성 이미지: **300페타바이트**의 해양 관측 데이터
- 연구 출판물: 관련 데이터셋을 포함한 **50,000편의 논문**
- 시민 과학: 해안 모니터링에서 **1,000만 건의 관측**

### 표준이 중요한 이유

해양 생물학 데이터에 대한 표준화된 접근 방식이 없으면 다음과 같은 중요한 문제에 직면합니다:

**데이터 분절화:** 연구 기관들이 호환되지 않는 형식을 사용하여 연구 간 비교가 어렵습니다. 일본의 종 관측은 브라질의 관측과 다른 분류 체계를 사용할 수 있습니다.

**지식 손실:** 파일 형식이 구식이 되거나 문서가 손실되면서 귀중한 역사적 데이터에 접근할 수 없게 됩니다. 1980년대의 연구를 현대 데이터셋과 통합하는 것이 불가능할 수 있습니다.

**노력 낭비:** 연구자들은 과학적 분석 대신 데이터 정리 및 형식 변환에 시간의 60%를 소비합니다.

**발견 누락:** 다른 출처의 데이터를 효과적으로 결합할 수 없을 때 중요한 생태학적 패턴이 숨겨진 채로 남아있습니다.

### WIA-OCEAN-004 표준

WIA-OCEAN-004는 해양 생물학 데이터 관리를 위한 포괄적인 지침을 수립합니다:

#### 핵심 데이터 유형

**종 관측:**
```typescript
interface SpeciesObservation {
  scientificName: string;        // 이명법
  taxonomyId: string;             // WoRMS AphiaID 또는 NCBI taxonomy
  observationDate: Date;
  location: GeographicCoordinate;
  depth: Measurement;             // 수면 아래 미터
  abundance: number | "rare" | "common" | "abundant";
  observer: string;               // 연구자 또는 기관
  identificationMethod: "visual" | "photo" | "specimen" | "eDNA" | "acoustic";
  confidence: number;             // 0-1 척도
  metadata: ObservationMetadata;
}

interface GeographicCoordinate {
  latitude: number;   // 십진 도
  longitude: number;  // 십진 도
  datum: string;      // 예: "WGS84"
  uncertainty: number; // 미터
}

interface Measurement {
  value: number;
  unit: string;
  precision: number;
  method: string;
}
```

**환경 매개변수:**
```typescript
interface OceanEnvironment {
  location: GeographicCoordinate;
  timestamp: Date;
  temperature: Measurement;       // 섭씨
  salinity: Measurement;          // PSU (실용 염분 단위)
  pH: Measurement;
  dissolvedOxygen: Measurement;   // mg/L 또는 % 포화도
  nutrients: NutrientProfile;
  chlorophyll: Measurement;       // mg/m³
  turbidity: Measurement;         // NTU
  currentVelocity?: Measurement;  // m/s
  waveHeight?: Measurement;       // 미터
}

interface NutrientProfile {
  nitrate: Measurement;    // μmol/L
  phosphate: Measurement;  // μmol/L
  silicate: Measurement;   // μmol/L
  ammonia?: Measurement;   // μmol/L
}
```

### 글로벌 해양 데이터 이니셔티브

해양 과학 커뮤니티는 여러 주요 데이터 인프라 프로젝트를 개발했습니다:

#### 해양 생물다양성 정보 시스템 (OBIS)

**규모:** 1,500개 이상의 데이터셋에서 **1억 건 이상의 종 분포 기록**
**범위:** 전 지구적 해양 생물다양성 관측
**표준:** 생물다양성 데이터를 위한 Darwin Core
**접근:** obis.org의 개방형 데이터 포털

OBIS는 연구 크루즈, 박물관, 모니터링 프로그램 및 시민 과학의 관측을 통합하는 해양 생물다양성 데이터의 주요 글로벌 저장소 역할을 합니다.

#### 세계 해양 종 등록부 (WoRMS)

**규모:** **240,000개 이상의 인정된 해양 종 이름**
**범위:** 모든 해양 생물에 대한 권위 있는 분류 체계
**업데이트:** 300명 이상의 분류학자에 의한 지속적인 전문가 큐레이션
**API:** 프로그래밍 방식 접근을 위한 RESTful API

WoRMS는 모든 데이터베이스와 출판물에서 일관된 종 이름을 보장하는 해양 생물학의 분류학적 기반을 제공합니다.

### 한국의 해양 연구 기관

한국은 세계적 수준의 해양 연구 인프라를 보유하고 있습니다:

**한국해양과학기술원 (KIOST)**
- 동해, 황해, 남해 연구
- 해양 생물다양성 모니터링
- 기후변화 영향 연구
- 심해 탐사 프로그램

**국립수산과학원 (NIFS)**
- 수산 자원 평가
- 어업 관리 데이터
- 양식 연구
- 해양 생태계 건강 모니터링

**국립해양생물자원관 (MABIK)**
- 해양 생물 표본 수집
- 종 목록 작성
- DNA 바코딩 프로그램
- 생물다양성 데이터베이스

**한국극지연구소 (KOPRI)**
- 남극 해양 생태계 연구
- 북극 생물다양성
- 기후변화 영향
- 극지 해양 데이터

이러한 기관들은 국제 데이터베이스에 기여하고 WIA-OCEAN-004와 같은 글로벌 표준을 채택하여 한국의 해양 데이터가 전 세계 연구와 통합될 수 있도록 합니다.

### 데이터 품질 원칙

고품질 해양 생물학 데이터는 FAIR 원칙을 따릅니다:

#### 발견 가능 (Findable)
- 데이터셋을 위한 영구 식별자 (DOI)
- 키워드가 포함된 풍부한 메타데이터
- 검색 가능한 저장소에 색인화
- 명확한 데이터셋 제목 및 설명

#### 접근 가능 (Accessible)
- 개방형 프로토콜 (HTTP, FTP, OPeNDAP)
- 필요한 경우 인증
- 장기 저장소 약속
- 형식 문서화

#### 상호운용 가능 (Interoperable)
- 표준화된 어휘
- 공통 데이터 형식
- 연결된 데이터 접근 방식
- API 액세스

#### 재사용 가능 (Reusable)
- 명확한 사용 라이선스 (CC-BY, CC0)
- 상세한 방법론
- 품질 관리 문서
- 출처 추적

### 실제 예시: 산호초 모니터링

표준화된 데이터가 산호초 연구를 어떻게 지원하는지 살펴보겠습니다:

```typescript
interface CoralReefSurvey {
  site: {
    name: string;
    location: GeographicCoordinate;
    habitat: "reef_crest" | "reef_slope" | "lagoon" | "patch_reef";
    depth: Measurement;
    protectionStatus: "marine_reserve" | "protected" | "unprotected";
  };

  survey: {
    date: Date;
    method: "transect" | "quadrat" | "photogrammetry";
    area: Measurement;              // 제곱미터
    duration: number;               // 분
    observers: string[];
    visibility: Measurement;        // 미터
  };

  coralCover: {
    hardCoral: number;              // 백분율 0-100
    softCoral: number;
    deadCoral: number;
    bleached: number;
    diseased: number;
    recruits: number;               // m²당
  };

  species: {
    scientificName: string;
    aphiaID: number;                // WoRMS 식별자
    growthForm: "branching" | "massive" | "encrusting" | "foliose" | "columnar";
    coverage: number;               // 전체 산호의 백분율
    healthStatus: "healthy" | "bleached" | "diseased" | "dead";
    diseaseType?: string;
  }[];

  environment: OceanEnvironment;

  threats: {
    crownOfThorns: number;          // 헥타르당 개체 수
    bleaching: "none" | "minor" | "moderate" | "severe";
    physicalDamage: "none" | "minor" | "moderate" | "severe";
    pollution: "none" | "minor" | "moderate" | "severe";
  };
}
```

이 표준화된 형식을 통해 연구자들은 다음을 수행할 수 있습니다:
- 지리적 지역에 걸쳐 산호초 건강 비교
- 모니터링 사이트에서 시간적 변화 추적
- 산호 건강과 환경 매개변수 상관 관계 파악
- 멸종 위기 종 식별 및 보전 우선순위 지정
- 기후 변화에 대한 생태계 반응 모델링

### 철학: 弘益人間 (널리 인간을 이롭게 하라)

해양 생물학 데이터 표준은 弘益人間의 원칙을 구현합니다 - 모든 인류에게 이익을 줍니다. 해양은 다음을 제공합니다:

- 식물성 플랑크톤에서 **지구 산소의 50%**
- 어업에서 **30억 명을 위한 단백질**
- 탄소 격리를 통한 **기후 조절**
- 해양 생물에서 **의약 화합물**
- 연간 **2조 5천억 달러의 경제적 가치**

해양 데이터를 표준화함으로써 우리는 다음을 가능하게 하는 글로벌 협력을 가능하게 합니다:
- 해양 생물다양성 이해 및 보호
- 어업의 지속 가능한 관리
- 기후 영향 예측 및 완화
- 새로운 의약품 및 재료 발견
- 미래 세대를 위한 해양 보존

해양은 모든 인류에게 속합니다. 그 데이터도 그래야 합니다.
