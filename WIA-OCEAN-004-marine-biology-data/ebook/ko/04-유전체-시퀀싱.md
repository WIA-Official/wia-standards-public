# 4장: 유전체 및 DNA 시퀀싱

## 해양 생물다양성을 위한 분자 도구

DNA 시퀀싱은 해양 생물학에 혁명을 일으켜 미세한 유생으로부터 종 식별, 잠재종 발견, 물 샘플로부터 생물 검출, 진화 관계 재구성을 가능하게 했습니다. 이 장에서는 유전체 데이터가 생성, 표준화 및 전통적인 관측 데이터와 통합되는 방법을 탐구합니다.

### 해양 생물학의 DNA 혁명

전통적인 해양 분류학은 형태학적 특징에 의존하지만, 이는 미세한 생물, 유생 단계 또는 잠재종에 대해 종종 어렵습니다. DNA 시퀀싱은 다음을 가능하게 하는 분자 지문을 제공합니다:

**종 식별:** 알 수 없는 표본을 참조 서열과 매칭
**잠재종 발견:** 형태학적으로 유사한 그룹에서 숨겨진 다양성 발견
**환경 DNA (eDNA):** 포획 없이 물 샘플에서 종 검출
**위 내용물 분석:** 포식자 위의 먹이 항목 식별
**개체군 유전학:** 연결성 및 이동 패턴 추적
**계통발생학:** 진화 관계 재구성

### 종 식별을 위한 DNA 바코딩

**Barcode of Life** 이니셔티브는 종 식별을 위한 표준화된 유전자 마커를 확립했습니다:

### 한국의 해양 DNA 바코딩

한국은 적극적인 해양 DNA 바코딩 프로그램을 운영합니다:

**국립해양생물자원관 (MABIK) DNA 바코딩 센터**
- 한국 해양 생물의 DNA 바코드 구축
- 5,000종 이상 바코딩 완료
- GenBank 및 BOLD 시스템에 데이터 기여
- 한국 고유종에 중점

**주요 한국 바코딩 대상:**
- 해양 어류 (1,000종 이상)
- 해조류 및 해초
- 갑각류 (새우, 게)
- 연체동물 (조개, 복족류)
- 자포동물 (해파리, 산호)

### 환경 DNA (eDNA) 메타바코딩

생물을 포획하는 대신, eDNA 방법은 물 샘플에서 DNA를 시퀀싱하여 존재하는 모든 종을 검출합니다:

```typescript
interface eDNASample {
  sampleID: string;
  project: string;

  sampling: {
    date: Date;
    location: GeographicCoordinate;
    depth: number;              // 미터
    habitat: string;
    sampleVolume: number;       // 여과된 리터
    replicates: number;

    filtration: {
      filterType: string;       // "Sterivex", "membrane_filter"
      poreSize: number;         // 마이크로미터 (일반적으로 0.22 또는 0.45)
      filtrationMethod: "vacuum" | "peristaltic_pump";
      preservative: string;     // "ethanol", "RNAlater", "frozen"
    };

    environmental: {
      temperature: number;      // 섭씨
      salinity: number;         // PSU
      pH?: number;
      turbidity?: number;       // NTU
    };
  };

  laboratory: {
    extractionMethod: string;   // "DNeasy Blood & Tissue Kit"
    extractionDate: Date;
    DNAConcentration: number;   // ng/μL

    PCR: {
      target: string;           // "COI", "16S", "18S", "12S"
      primers: {
        forward: string;
        reverse: string;
      };
      cycles: number;
      annealingTemp: number;    // 섭씨
      replicates: number;
    };

    sequencing: {
      platform: "Illumina MiSeq" | "Illumina NovaSeq" | "Ion Torrent";
      readType: "paired_end" | "single_end";
      readLength: number;       // 염기쌍
      indexing: "single" | "dual";
      runID: string;
    };
  };

  results: eDNADetection[];
}

interface eDNADetection {
  OTU_ID: string;               // 작동 분류 단위 식별자
  sequence: string;             // 대표 서열
  readCount: number;            // 서열 읽기 수
  relativeAbundance: number;    // 총 읽기의 비율

  taxonomy: {
    kingdom: string;
    phylum: string;
    class: string;
    order: string;
    family: string;
    genus: string;
    species: string;
    confidence: number;         // 할당 신뢰도
  };
}
```

### 한국 해양 유전체 연구

한국 연구 기관들은 주요 해양 종의 전체 유전체를 시퀀싱하고 있습니다:

**KIOST 해양유전체센터**
- 경제적으로 중요한 해양 종의 유전체
- 기후 변화 적응 연구
- 수산 양식 개량
- 보전 유전학

**주요 한국 해양 유전체 프로젝트:**
- 참돔 (*Pagrus major*) - 양식 개량
- 전복 (*Haliotis discus hannai*) - 내병성 연구
- 미역 (*Undaria pinnatifida*) - 품질 개선
- 대게 (*Chionoecetes opilio*) - 자원 관리

### 유전체 데이터 모범 사례

**1. 공공 저장소에 기탁**
- DNA 바코드 → BOLD 시스템 및 GenBank
- 원시 서열 읽기 → NCBI SRA, ENA 또는 DDBJ
- 유전체 어셈블리 → GenBank/RefSeq

**2. 물리적 표본에 링크**
박물관 카탈로그 번호 및 바우처 사진 포함.

**3. 완전한 메타데이터 제공**
위치, 깊이, 환경 조건, 수집 방법.

**4. 명명 규칙 따르기**
표준화된 유전자 이름 사용 (COI, COX1 또는 COXI가 아님).

**5. 품질 메트릭 포함**
Phred 점수, BUSCO 완성도, 오염 추정치.

**6. 생물정보학 문서화**
파이프라인 버전, 매개변수, 사용된 참조 데이터베이스.

**7. 개방형 데이터**
최대 재사용성을 위해 CC0 또는 CC-BY 라이선스 사용.

### 철학: 분자 공유지

유전체 데이터는 모든 인류를 위한 (弘益人間) 공유 자원을 나타냅니다. 250억 개 이상의 서열을 포함하는 전체 GenBank 데이터베이스는 전 세계적으로 자유롭게 접근 가능합니다. 이를 통해:

- 모든 국가에서 종 식별
- 질병 발생 추적
- 생물다양성 보전
- 기후 변화 연구
- 해양 생물로부터 약물 발견

서열을 기탁할 때, 이 글로벌 공유지에 기여합니다. 서열을 다운로드할 때, 전 세계 연구자들의 수백만 시간의 작업으로부터 혜택을 받습니다. 현명하게 사용하고, 자유롭게 공유하며, 되돌려 기여하세요.
