WIA Pet Health Passport Standard - Cross-Border Compliance & Documentation

3.1 Global Pet Travel Framework

International pet travel requires compliance with complex, country-specific regulations designed to prevent disease transmission, protect native species, and ensure animal welfare. The WIA Pet Health Passport Standard provides a unified framework for documenting, validating, and presenting pet health credentials that meet diverse international requirements while maintaining digital interoperability.

Pet travel regulations vary significantly across jurisdictions, with requirements ranging from simple vaccination documentation to extensive quarantine periods, blood titer testing, parasite treatments, and government-issued import permits. Digital health passports enable real-time verification, reduce paperwork burdens, and facilitate faster border crossings while maintaining regulatory compliance.

3.3 Travel API Endpoints

GET /api/v1/travel/requirements/{destinationCountry}

Retrieve complete travel requirements for specific destination country including vaccinations, permits, and documentation.

POST /api/v1/pets/{petId}/travel/validate

Validate pet's readiness for travel to specified destination, checking all requirements against current health records.

POST /api/v1/pets/{petId}/travel/certificate

Generate official international health certificate with all required documentation and digital signatures.

GET /api/v1/travel/requirements/compare

Compare travel requirements across multiple destinations for trip planning purposes.

POST /api/v1/travel/import-permit/apply

Submit import permit application to destination country's veterinary authority (where API integration available).

GET /api/v1/travel/timeline/{petId}/{destination}

Generate travel preparation timeline showing when each requirement must be completed.

POST /api/v1/travel/endorsement

Request government veterinary endorsement for health certificates (USDA, DEFRA, etc.).

GET /api/v1/travel/carriers

Retrieve airline and carrier-specific pet travel requirements and restrictions.

POST /api/v1/travel/notification

Submit advance notification to destination country customs/veterinary authorities.

GET /api/v1/pets/{petId}/travel/history

Retrieve complete travel history including all international movements and border crossings.

POST /api/v1/travel/verify

Verify authenticity of travel documents via QR code or certificate number for border control use.

GET /api/v1/travel/quarantine/{country}

Retrieve quarantine facility information, requirements, and booking options for specific country.

Example 1: Travel Compliance Checker (JavaScript)

// Comprehensive travel compliance verification system
class InternationalTravelCompliance {
  constructor(apiKey) {
    this.apiKey = apiKey;
    this.baseUrl = 'https://api.wia-pet-health.org/v1';
  }

  async checkTravelReadiness(petId, destination, departureDate) {
    const requirements = await this.getRequirements(destination);
    const petRecords = await this.getPetRecords(petId);

    const compliance = {
      destination: destination,
      departureDate: departureDate,
      compliant: true,
      requirements: [],
      timeline: []
    };

    // Check microchip requirement
    if (requirements.microchipRequired) {
      const hasValidMicrochip = this.checkMicrochip(petRecords);
      compliance.requirements.push({
        requirement: 'Microchip',
        status: hasValidMicrochip ? 'Met' : 'Missing',
        details: hasValidMicrochip ? petRecords.microchipNumber : 'Microchip required'
      });
      if (!hasValidMicrochip) compliance.compliant = false;
    }

    // Check rabies vaccination
    const rabiesVax = this.findLatestVaccination(petRecords.vaccinations, 'rabies');
    if (requirements.rabiesRequired) {
      const rabiesValid = this.validateRabiesForTravel(
        rabiesVax,
        departureDate,
        requirements.rabiesMinDays,
        requirements.rabiesMaxDays
      );

      compliance.requirements.push({
        requirement: 'Rabies Vaccination',
        status: rabiesValid.valid ? 'Met' : 'Action Required',
        details: rabiesValid.message,
        vaccineDate: rabiesVax?.administeredDate,
        validFrom: rabiesValid.validFrom,
        validUntil: rabiesValid.validUntil
      });

      if (!rabiesValid.valid) {
        compliance.compliant = false;
        compliance.timeline.push({
          action: 'Rabies Vaccination',
          deadline: this.calculateVaccinationDeadline(departureDate, requirements.rabiesMinDays),
          urgency: this.calculateUrgency(departureDate, requirements.rabiesMinDays)
        });
      }
    }

    // Check titer test requirement
    if (requirements.titerRequired) {
      const titerTest = petRecords.titerTests?.find(t => t.testType === 'rabies');
      const titerValid = this.validateTiter(titerTest, departureDate, requirements.titerWaitDays);

      compliance.requirements.push({
        requirement: 'Rabies Titer Test',
        status: titerValid.valid ? 'Met' : 'Action Required',
        details: titerValid.message
      });

      if (!titerValid.valid) {
        compliance.compliant = false;
        const titerDeadline = this.calculateTiterDeadline(
          departureDate,
          requirements.titerWaitDays
        );
        compliance.timeline.push({
          action: 'Rabies Titer Test',
          deadline: titerDeadline,
          urgency: 'CRITICAL',
          note: `Must wait ${requirements.titerWaitDays} days after test before travel`
        });
      }
    }

    // Check parasite treatments
    if (requirements.parasiteTreatments) {
      for (const treatment of requirements.parasiteTreatments) {
        const treatmentValid = this.validateTreatment(
          petRecords.treatments,
          treatment,
          departureDate
        );

        compliance.requirements.push({
          requirement: `${treatment.type} Treatment`,
          status: treatmentValid.valid ? 'Met' : 'Action Required',
          details: treatmentValid.message
        });

        if (!treatmentValid.valid) compliance.compliant = false;
      }
    }

    // Check import permit
    if (requirements.importPermitRequired) {
      const hasPermit = petRecords.importPermits?.some(
        p => p.country === destination && new Date(p.validUntil) > new Date(departureDate)
      );

      compliance.requirements.push({
        requirement: 'Import Permit',
        status: hasPermit ? 'Met' : 'Action Required',
        details: hasPermit ? 'Valid permit on file' : 'Import permit must be obtained'
      });

      if (!hasPermit) compliance.compliant = false;
    }

    return compliance;
  }

  validateRabiesForTravel(vaccination, travelDate, minDays, maxDays) {
    if (!vaccination) {
      return { valid: false, message: 'No rabies vaccination on record' };
    }

    const vaxDate = new Date(vaccination.administeredDate);
    const travel = new Date(travelDate);
    const daysSinceVax = Math.floor((travel - vaxDate) / (1000 * 60 * 60 * 24));

    if (daysSinceVax < minDays) {
      return {
        valid: false,
        message: `Rabies vaccine must be at least ${minDays} days old. Currently ${daysSinceVax} days.`,
        validFrom: new Date(vaxDate.getTime() + minDays * 24 * 60 * 60 * 1000)
      };
    }

    if (daysSinceVax > maxDays) {
      return {
        valid: false,
        message: `Rabies vaccine expired. Maximum validity ${maxDays} days.`,
        validUntil: new Date(vaxDate.getTime() + maxDays * 24 * 60 * 60 * 1000)
      };
    }

    return {
      valid: true,
      message: 'Rabies vaccination meets requirements',
      validFrom: new Date(vaxDate.getTime() + minDays * 24 * 60 * 60 * 1000),
      validUntil: new Date(vaxDate.getTime() + maxDays * 24 * 60 * 60 * 1000)
    };
  }

  async generateTravelCertificate(petId, destination) {
    const response = await fetch(
      `${this.baseUrl}/pets/${petId}/travel/certificate`,
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${this.apiKey}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          destination: destination,
          format: 'pdf',
          includeQRCode: true,
          digitalSignature: true
        })
      }
    );

    return await response.blob();
  }

  async getRequirements(country) {
    const response = await fetch(
      `${this.baseUrl}/travel/requirements/${country}`,
      {
        headers: { 'Authorization': `Bearer ${this.apiKey}` }
      }
    );
    return await response.json();
  }

  async getPetRecords(petId) {
    const response = await fetch(
      `${this.baseUrl}/pets/${petId}/health-record`,
      {
        headers: { 'Authorization': `Bearer ${this.apiKey}` }
      }
    );
    return await response.json();
  }
}

Example 2: Multi-Country Trip Planner (Python)

from datetime import datetime, timedelta
from typing import List, Dict
import requests

class MultiCountryTravelPlanner:
    """Plan pet travel across multiple countries with requirement aggregation"""

    def __init__(self, api_key: str):
        self.api_key = api_key
        self.base_url = 'https://api.wia-pet-health.org/v1'
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }

    def plan_multi_country_trip(self, pet_id: str, itinerary: List[Dict]) -> Dict:
        """
        Plan travel across multiple countries

        Args:
            pet_id: Pet UUID
            itinerary: List of {'country': 'XX', 'arrival': 'YYYY-MM-DD', 'departure': 'YYYY-MM-DD'}

        Returns:
            Comprehensive travel plan with all requirements
        """
        plan = {
            'pet_id': pet_id,
            'countries': [],
            'all_requirements': set(),
            'critical_deadlines': [],
            'estimated_costs': 0
        }

        # Analyze each destination
        for stop in itinerary:
            country_req = self.get_country_requirements(stop['country'])

            country_plan = {
                'country': stop['country'],
                'arrival': stop['arrival'],
                'departure': stop['departure'],
                'requirements': country_req,
                'transit_considerations': []
            }

            # Check transit requirements
            if stop != itinerary[0]:
                prev_country = itinerary[itinerary.index(stop) - 1]['country']
                transit_req = self.check_transit_requirements(
                    prev_country, stop['country']
                )
                country_plan['transit_considerations'] = transit_req

            plan['countries'].append(country_plan)
            plan['all_requirements'].update(country_req['required_vaccinations'])

            # Add costs
            if 'fees' in country_req:
                plan['estimated_costs'] += country_req['fees'].get('import_permit', 0)
                plan['estimated_costs'] += country_req['fees'].get('health_certificate', 0)

        # Generate master timeline
        plan['timeline'] = self.generate_master_timeline(pet_id, plan)

        # Calculate earliest departure date possible
        plan['earliest_safe_departure'] = self.calculate_earliest_departure(plan)

        return plan

    def generate_master_timeline(self, pet_id: str, plan: Dict) -> List[Dict]:
        """Generate chronological timeline of all required actions"""
        timeline = []
        first_arrival = min(c['arrival'] for c in plan['countries'])

        # Get current pet status
        pet_records = self.get_pet_records(pet_id)

        # Rabies vaccination timeline
        if 'rabies' in plan['all_requirements']:
            rabies_vax = next(
                (v for v in pet_records['vaccinations'] if v['vaccineType'] == 'rabies'),
                None
            )

            if not rabies_vax or not self.is_valid_for_date(rabies_vax, first_arrival):
                timeline.append({
                    'action': 'Rabies Vaccination',
                    'deadline': self.calculate_deadline(first_arrival, days_before=21),
                    'priority': 'CRITICAL',
                    'estimated_cost': 50
                })

        # Titer test timeline
        requires_titer = any(
            c['requirements'].get('titer_required', False)
            for c in plan['countries']
        )

        if requires_titer:
            max_wait_days = max(
                c['requirements'].get('titer_wait_days', 180)
                for c in plan['countries']
                if c['requirements'].get('titer_required')
            )

            timeline.append({
                'action': 'Rabies Titer Test',
                'deadline': self.calculate_deadline(first_arrival, days_before=max_wait_days + 30),
                'priority': 'CRITICAL',
                'note': f'Must wait {max_wait_days} days after test before travel',
                'estimated_cost': 200
            })

        # Import permits
        for country_plan in plan['countries']:
            if country_plan['requirements'].get('import_permit_required'):
                timeline.append({
                    'action': f"Import Permit - {country_plan['country']}",
                    'deadline': self.calculate_deadline(country_plan['arrival'], days_before=60),
                    'priority': 'HIGH',
                    'processing_time': '30-60 days',
                    'estimated_cost': country_plan['requirements']['fees'].get('import_permit', 0)
                })

        # Health certificate
        timeline.append({
            'action': 'International Health Certificate',
            'deadline': self.calculate_deadline(first_arrival, days_before=10),
            'priority': 'HIGH',
            'note': 'Usually valid for 10-30 days',
            'estimated_cost': 150
        })

        # USDA endorsement (if traveling from US)
        timeline.append({
            'action': 'USDA/Government Endorsement',
            'deadline': self.calculate_deadline(first_arrival, days_before=7),
            'priority': 'HIGH',
            'note': 'Schedule appointment in advance',
            'estimated_cost': 38
        })

        # Sort by deadline
        timeline.sort(key=lambda x: x['deadline'])

        return timeline

    def calculate_deadline(self, target_date: str, days_before: int) -> str:
        """Calculate deadline date"""
        target = datetime.fromisoformat(target_date)
        deadline = target - timedelta(days=days_before)
        return deadline.isoformat()

    def get_country_requirements(self, country_code: str) -> Dict:
        """Fetch requirements for specific country"""
        response = requests.get(
            f'{self.base_url}/travel/requirements/{country_code}',
            headers=self.headers
        )
        return response.json()

    def get_pet_records(self, pet_id: str) -> Dict:
        """Get current pet health records"""
        response = requests.get(
            f'{self.base_url}/pets/{pet_id}/health-record',
            headers=self.headers
        )
        return response.json()

Example 3: Border Control Verification (Ruby)

require 'net/http'
require 'json'
require 'uri'

class BorderControlVerification
  # System for border control to verify pet travel documents

  def initialize(api_key)
    @api_key = api_key
    @base_url = 'https://api.wia-pet-health.org/v1'
  end

  def verify_travel_documents(certificate_number: nil, qr_code: nil, microchip: nil)
    # Verify documents using certificate number, QR code, or microchip

    verification_data = {}

    if certificate_number
      verification_data[:certificate_number] = certificate_number
    elsif qr_code
      verification_data[:qr_code] = qr_code
    elsif microchip
      verification_data[:microchip_number] = microchip
    else
      raise ArgumentError, 'Must provide certificate number, QR code, or microchip'
    end

    uri = URI("#{@base_url}/travel/verify")
    response = make_request(uri, verification_data)

    parse_verification_response(response)
  end

  def check_import_compliance(pet_id, country_code)
    # Verify pet meets all import requirements for specific country

    uri = URI("#{@base_url}/pets/#{pet_id}/travel/validate")
    data = {
      destination: country_code,
      validation_date: Time.now.iso8601
    }

    response = make_request(uri, data)

    compliance_report = JSON.parse(response.body)

    {
      compliant: compliance_report['overallCompliant'],
      pet_info: compliance_report['petInfo'],
      requirements_met: compliance_report['requirementsMet'],
      warnings: compliance_report['warnings'] || [],
      restrictions: compliance_report['restrictions'] || []
    }
  end

  def log_border_crossing(pet_id, country_code, entry_point)
    # Log when pet crosses border for tracking purposes

    uri = URI("#{@base_url}/travel/border-crossing")
    data = {
      pet_id: pet_id,
      country: country_code,
      entry_point: entry_point,
      timestamp: Time.now.iso8601,
      officer_id: @api_key
    }

    response = make_request(uri, data)

    JSON.parse(response.body)
  end

  private

  def make_request(uri, data)
    http = Net::HTTP.new(uri.host, uri.port)
    http.use_ssl = true

    request = Net::HTTP::Post.new(uri.path)
    request['Authorization'] = "Bearer #{@api_key}"
    request['Content-Type'] = 'application/json'
    request.body = data.to_json

    http.request(request)
  end

  def parse_verification_response(response)
    result = JSON.parse(response.body)

    {
      valid: result['valid'],
      pet_name: result['petInfo']['name'],
      species: result['petInfo']['species'],
      microchip: result['petInfo']['microchipNumber'],
      vaccinations: result['vaccinations'],
      certificate_valid: result['certificateValid'],
      expiry_date: result['certificateExpiry'],
      issuing_vet: result['issuingVet'],
      warnings: result['warnings'] || []
    }
  end
end

  puts "Certificate valid until: #{result[:expiry_date]}"
else
  puts "DENIED: #{result[:warnings].join(', ')}"
end

verifier.log_border_crossing(
  result[:pet_id],
  'FR',
  'Charles de Gaulle Airport'
