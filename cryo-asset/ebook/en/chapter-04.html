4.1 Preservation Trust Structure
--------------------------------------------------------------------------------

during cryopreservation.

4.2 Trustee Responsibilities
--------------------------------------------------------------------------------

Trustees bear fiduciary responsibility for managing trust assets and
carrying out trust purposes.

[ID] Fiduciary Duties:

    FiduciaryDuties {
      dutyOfLoyalty: {
        description:   "Act solely in beneficiary's interest"
        prohibitions: [
          "Self-dealing",
          "Conflicts of interest",
          "Personal benefit from trust assets"
        ]
        disclosure:    "All potential conflicts must be disclosed"
      }

      dutyOfPrudence: {
        description:   "Manage assets as prudent investor would"
        standard:      "Prudent Investor Rule"
        considerations: [
          "Risk and return objectives",
          "Diversification",
          "Time horizon",
          "Tax efficiency"
        ]
      }

      dutyOfImpartiality: {
        description:   "Balance interests of all beneficiaries"
        application:   "Current vs. contingent beneficiaries"
        guidance:      "Trust terms and court direction"
      }

      dutyToInform: {
        description:   "Keep beneficiaries reasonably informed"
        requirements: [
          "Respond to reasonable requests",
          "Provide periodic accountings",
          "Notify of significant events"
        ]
      }

      dutyToDelegate: {
        description:   "Delegate appropriately when prudent"
        permitted:     "Investment management, tax preparation"
        retained:      "Distribution decisions, trust interpretation"
        oversight:     "Must monitor delegates"
      }
    }

[ID] Administrative Responsibilities:

    AdministrativeResponsibilities {
      assetManagement: {
        inventory:      "Maintain complete asset records"
        safekeeping:    "Secure all assets appropriately"
        insurance:      "Maintain adequate coverage"
        maintenance:    "Preserve asset value"
      }

      financialManagement: {
        accounting:     "Maintain accurate books and records"
        taxFiling:      "Prepare and file all required returns"
        distributions:  "Make authorized distributions"
        reporting:      "Provide required reports"
      }

      recordkeeping: {
        minutes:        "Document all trustee decisions"
        correspondence: "Retain all communications"
        statements:     "Keep all financial statements"
        valuations:     "Maintain valuation records"
      }

      compliance: {
        trustTerms:     "Follow trust instrument"
        courtOrders:    "Comply with court directions"
        regulations:    "Follow applicable laws and rules"
        reporting:      "File required reports on time"
      }
    }

[ID] Trustee Actions Requiring Court Approval:

    CourtApprovalRequired {
      always: [
        "Trust termination",
        "Major trust modification",
        "Trustee removal/resignation (if contested)",
        "Distribution to incapacitated beneficiary"
      ]

      situational: [
        "Sale of principal residence",
        "Business decisions outside normal course",
        "Litigation decisions above threshold",
        "Self-dealing transactions (even if permitted)"
      ]

      recommended: [
        "Unusual investments",
        "Large discretionary distributions",
        "Interpretation of ambiguous terms",
        "Actions opposed by any beneficiary"
      ]
    }

4.3 Beneficiary Management
--------------------------------------------------------------------------------

The trust must manage multiple beneficiary interests, including the preserved
individual and contingent beneficiaries.

[ID] Beneficiary Categories:

    BeneficiaryCategories {
      primaryBeneficiary: {
        identity:       "Preserved individual"
        interest:       "100% upon restoration"
        rights: [
          "Right to restoration of assets",
          "Right to accounting information",
          "Right to trust distributions for preservation"
        ]
        representation: "Guardian represents interests"
      }

      contingentBeneficiaries: {
        identity:       "Heirs designated in trust or by law"
        interest:       "Activated upon permanent death"
        rights: [
          "Right to accounting (limited)",
          "Right to notice of major actions",
          "Right to object to improper actions"
        ]
        condition:      "Interest contingent, may never vest"
      }

      remainderBeneficiaries: {
        identity:       "Those who receive after trust ends"
        interest:       "Future interest in trust assets"
        rights:         "Similar to contingent beneficiaries"

      charities: {
        identity:       "Charitable organizations if designated"
        interest:       "Per trust terms"
        rights:         "Enforcement through Attorney General"
      }
    }

4.4 Distributions and Reserves
--------------------------------------------------------------------------------

Trust distributions must balance current needs against long-term preservation
objectives.

[ID] Distribution Framework:

    DistributionFramework {
      preservationExpenses: {
        priority:       "HIGHEST"
        description:    "Costs necessary for preservation"
        examples: [
          "Cryopreservation facility fees",
          "Medical monitoring",
          "Insurance premiums",
          "Legal compliance costs"
        ]
        authorization:  "Automatic per trust terms"
      }

      trustAdministration: {
        priority:       "HIGH"
        description:    "Costs of managing the trust"
        examples: [
          "Trustee compensation",
          "Professional fees (legal, accounting)",
          "Tax preparation",
          "Asset management fees"
        ]
        authorization:  "Per trust terms or reasonable"
      }

      assetMaintenance: {
        priority:       "MEDIUM"
        description:    "Costs to maintain asset value"
        examples: [
          "Property taxes and insurance",
          "Property maintenance",
          "Business operating expenses",
          "IP maintenance fees"
        ]
        authorization:  "Trustee discretion"
      }

      discretionaryDistributions: {
        priority:       "LOWER"
        description:    "Distributions not strictly required"
        examples: [
          "Support for dependents",
          "Charitable giving",
          "Family emergencies"
        ]
        authorization:  "Trustee discretion per trust terms"
      }
    }

[ID] Reserve Requirements:

    ReserveRequirements {
      preservationReserve: {
        purpose:        "Ensure preservation can continue"
        calculation:    "Estimated preservation costs * years"
        minimum:        "50 years of projected costs"
        recommended:    "100+ years if assets permit"
        protected:      "Cannot be invaded for other purposes"
      }

      operatingReserve: {
        purpose:        "Cover near-term expenses"
        calculation:    "24 months of operating expenses"
        replenishment:  "Priority after preservation reserve"
      }

      emergencyReserve: {
        purpose:        "Unexpected costs or opportunities"
        calculation:    "5-10% of total assets"
        access:         "Trustee discretion"
      }

      distributionReserve: {
        purpose:        "Planned beneficiary distributions"
        calculation:    "Per distribution schedule"
        source:         "Income and planned principal"
      }
    }

[ID] Distribution Process:

    DistributionProcess {
      step1_request: {
        action:     "Distribution request submitted"
        requestor:  "Beneficiary, guardian, or automatic"
        documentation: "Purpose, amount, supporting info"
      }

      step2_review: {
        action:     "Trustee reviews request"
        considerations: [
          "Trust terms permit distribution",
          "Sufficient funds available",
          "Reserves maintained",
          "Tax implications"
        ]
      }

      step3_approval: {
        action:     "Trustee approves or denies"
        documentation: "Written decision with reasoning"
        notification: "Beneficiary notified"
      }

      step4_execution: {
        action:     "Distribution made"
        method:     "Check, wire, in-kind transfer"
        documentation: "Distribution receipt obtained"
      }

      step5_recording: {
        action:     "Record distribution"
        updates: [
          "Trust accounting",
          "Beneficiary records",
          "Tax records"
        ]
      }
    }

[!] Important: Distributions that would jeopardize the preservation reserve
    are prohibited without court approval, even if beneficiaries request them.
