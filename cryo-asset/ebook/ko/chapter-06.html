6.1 위험 프로파일 관리
--------------------------------------------------------------------------------

투자 전략은 보존의 고유한 상황에 맞게 위험을 적절히 이해하고
관리하는 것으로 시작합니다.

[ID] 위험 프로파일 프레임워크:

    RiskProfileFramework {
      profiles: {
        CONSERVATIVE: {
          description:    "자본 보존 중심"
          targetReturn:   "인플레이션 + 1-2%"
          volatility:     "낮음"
          drawdownTolerance: 10
          suitability: [
            "짧은 예상 보존 기간",
            "보존 시 고령인 대상자",
            "필요 대비 제한된 자산",
            "대상자의 선호 표시"
          ]
        }

        MODERATE: {
          description:    "균형 잡힌 성장과 보존"
          targetReturn:   "인플레이션 + 3-4%"
          volatility:     "중간"
          drawdownTolerance: 20
          suitability: [
            "대부분의 상황에 대한 기본값",
            "보존을 위한 적절한 자산",

          ]
        }

        AGGRESSIVE: {
          description:    "성장 지향"

            "보존 시 젊은 대상자",
            "매우 긴 예상 보존 기간",

            "대상자가 성장 선호 표시"
          ]
        }
      }

      determination: {
        factors: [

      factors: {

6.5 장기 고려사항
--------------------------------------------------------------------------------

보존 투자 전략은 고유한 도전을 가진 잠재적으로 매우 긴
시간 지평선을 고려해야 합니다.

    LongDurationChallenges {
      economicChange: {
        challenge:    "경제 시스템이 극적으로 변할 수 있음"

        mitigation:   "글로벌 분산화, 실물 자산"
