## 이론

### 수어 3D 애니메이션의 근본 난제

한국 수어(KSL)는 **공간 기반 언어**다. 3D 아바타로 표현하려면:
- 손 위치·방향 (양손 독립)
- 손가락 모양 (handshape, 5 손가락 × 3 관절)
- 팔·어깨·몸통 회전 (IK 제약)
- 표정·눈썹 (non-manual markers, 수어 문법 일부)

Euler 각도(pitch·yaw·roll)로 구현하면 **gimbal lock**에서 무너진다. 특정 자세(팔을 위로 쭉 편 상태)에서 회전 축 2개가 합쳐져 자유도 하나 상실, 손이 갑자기 튕김.

### Quaternion이 왜 답인가

Quaternion은 4차원 수(w, x, y, z)로 3D 회전 표현.
```
q = w + xi + yj + zk,  단위 길이 (|q| = 1)
```

장점:
- **Gimbal lock 없음** (4차원 공간 회전)
- **Slerp** 구면 선형 보간으로 자연스러운 전환
- **연속 회전** 누적 시 수치 안정

형님의 WIA Talk 프로젝트 **SL-07**에서 Arm IK가 Euler 기반이라 특정 단어(예: "하늘" 처럼 팔을 머리 위로)에서 튕김 발생. **SL-07b**에서 전체 Quaternion으로 전환하며 해결 ([SL-08 프롬프트 문서](../../var/www/wiabook/claude-prompts/sign-language-roadmap/SL-08-ORIENTATION-AUDIT.md) 참조).

### Slerp — 자연스러운 전환

Slerp (Spherical Linear Interpolation):
```
slerp(q1, q2, t) = (sin((1-t)θ) · q1 + sin(tθ) · q2) / sin(θ)
```

θ는 q1과 q2 사이 각도. `t ∈ [0, 1]`로 보간 진행률.

Lerp(선형 보간)과의 차이: Lerp은 중간 지점 길이가 1 아닐 수 있어 **회전 속도가 균일하지 않음**. 수어는 속도 일정한 자연스러운 전환이 필수 → Slerp 필수.

### IK 제약 3층

1. **Pole Vector**: 팔꿈치가 어느 방향을 향할지 결정. 없으면 팔꿈치가 뒤로 꺾임.
2. **Y Floor**: 손이 바닥(Y=0) 아래로 못 가게. "내 가슴"이라는 수어에서 손이 몸을 뚫지 않게.
3. **Z Floor**: 손이 몸 뒤로 안 가게. 관찰자(카메라) 쪽 공간 유지.

이 3 제약을 동시에 만족하는 IK 솔버가 필요. FABRIK (Forward And Backward Reaching IK) 알고리즘이 제약 반영이 쉽고 수렴 빠름.

```python
# Quaternion 기반 3D 수어 애니메이션 핵심 엔진
# pip install numpy scipy numpy-quaternion transforms3d

import numpy as np
import quaternion    # numpy-quaternion
from dataclasses import dataclass, field
from typing import List, Tuple, Optional


# ──────────────────────────────────────────────
# 1. Quaternion 기본
# ──────────────────────────────────────────────
def euler_to_quat(
    pitch: float, yaw: float, roll: float,
    order: str = "XYZ",
) -> np.quaternion:
    """Euler(라디안) → Quaternion"""
    from transforms3d.euler import euler2quat
    w, x, y, z = euler2quat(pitch, yaw, roll, f"r{order.lower()}")
    return np.quaternion(w, x, y, z)


def quat_to_euler(q: np.quaternion, order: str = "XYZ") -> Tuple[float, float, float]:
    from transforms3d.euler import quat2euler
    return quat2euler([q.w, q.x, q.y, q.z], f"r{order.lower()}")


def slerp(q1: np.quaternion, q2: np.quaternion, t: float) -> np.quaternion:
    """구면 선형 보간"""
    # numpy-quaternion 제공 함수 사용
    return quaternion.slerp(q1, q2, 0, 1, t)


# ──────────────────────────────────────────────
# 2. 관절 구조 정의
# ──────────────────────────────────────────────
@dataclass
class Joint:
    """단일 관절. 부모 상대 회전을 Quaternion으로."""
    name:          str
    parent:        Optional[str] = None
    offset:        np.ndarray = field(default_factory=lambda: np.zeros(3))
    rotation:      np.quaternion = field(default_factory=lambda: np.quaternion(1, 0, 0, 0))
    children:      List[str] = field(default_factory=list)

    def local_matrix(self) -> np.ndarray:
        """4×4 로컬 변환 행렬"""
        from scipy.spatial.transform import Rotation as R
        rot = R.from_quat([self.rotation.x, self.rotation.y,
                           self.rotation.z, self.rotation.w]).as_matrix()
        mat = np.eye(4)
        mat[:3, :3] = rot
        mat[:3, 3]  = self.offset
        return mat


class Skeleton:
    """스켈레톤 — 관절 트리"""
    def __init__(self):
        self.joints: dict = {}

    def add_joint(self, joint: Joint):
        self.joints[joint.name] = joint
        if joint.parent and joint.parent in self.joints:
            self.joints[joint.parent].children.append(joint.name)

    def world_transform(self, name: str) -> np.ndarray:
        """관절의 월드 좌표 4×4 행렬"""
        joint = self.joints[name]
        mat = joint.local_matrix()
        if joint.parent:
            mat = self.world_transform(joint.parent) @ mat
        return mat

    def world_position(self, name: str) -> np.ndarray:
        return self.world_transform(name)[:3, 3]

    def build_default_human(self):
        """기본 인체 스켈레톤 (수어 관련 상반신 집중)"""
        self.add_joint(Joint("root", None, np.zeros(3)))
        self.add_joint(Joint("spine", "root", np.array([0, 1.0, 0])))
        self.add_joint(Joint("chest", "spine", np.array([0, 0.3, 0])))
        self.add_joint(Joint("neck",  "chest", np.array([0, 0.3, 0])))
        self.add_joint(Joint("head",  "neck",  np.array([0, 0.2, 0])))

        # 양쪽 팔
        for side, sign in [("L", 1), ("R", -1)]:
            self.add_joint(Joint(f"shoulder_{side}", "chest",
                                 np.array([sign * 0.25, 0.2, 0])))
            self.add_joint(Joint(f"upper_arm_{side}", f"shoulder_{side}",
                                 np.array([0, -0.3, 0])))
            self.add_joint(Joint(f"forearm_{side}", f"upper_arm_{side}",
                                 np.array([0, -0.3, 0])))
            self.add_joint(Joint(f"wrist_{side}", f"forearm_{side}",
                                 np.array([0, -0.1, 0])))

            # 손가락 5개 (각 3 관절)
            for finger in ["thumb", "index", "middle", "ring", "pinky"]:
                self.add_joint(Joint(f"{finger}_1_{side}", f"wrist_{side}",
                                     np.array([0.05, 0, 0])))
                self.add_joint(Joint(f"{finger}_2_{side}", f"{finger}_1_{side}",
                                     np.array([0.03, 0, 0])))
                self.add_joint(Joint(f"{finger}_3_{side}", f"{finger}_2_{side}",
                                     np.array([0.02, 0, 0])))


# ──────────────────────────────────────────────
# 3. FABRIK IK 솔버 (Y/Z floor 제약 포함)
# ──────────────────────────────────────────────
class FABRIKSolver:
    """
    Forward And Backward Reaching IK
    """
    def __init__(
        self,
        skeleton: Skeleton,
        chain: List[str],         # 예: ["shoulder_L", "upper_arm_L", "forearm_L", "wrist_L"]
        target_name: str = "wrist_L",
    ):
        self.sk = skeleton
        self.chain = chain
        self.target_name = target_name
        self.y_floor = 0.0         # 바닥 Y 좌표
        self.z_floor = -0.15       # 몸 뒤쪽 경계
        self.pole_vector: Optional[np.ndarray] = None    # 팔꿈치 향하는 방향

    def set_target(self, target_pos: np.ndarray) -> None:
        self.target = np.array(target_pos)

    def solve(self, max_iter: int = 10, tolerance: float = 0.01):
        """
        FABRIK 알고리즘:
        1. Forward: end-effector를 target으로 당겨옴
        2. Backward: root에서부터 체인 복원
        반복.
        """
        # 초기 월드 위치
        positions = np.array([
            self.sk.world_position(j) for j in self.chain
        ])
        # 각 세그먼트 길이 (불변)
        lengths = np.linalg.norm(
            positions[1:] - positions[:-1], axis=1,
        )
        total_len = lengths.sum()

        # Target까지 거리 체크
        root_to_target = np.linalg.norm(self.target - positions[0])
        if root_to_target > total_len:
            # 닿지 못함 — 최대한 늘림
            direction = (self.target - positions[0]) / root_to_target
            for i in range(1, len(positions)):
                positions[i] = positions[i-1] + direction * lengths[i-1]
        else:
            for _ in range(max_iter):
                # ── Forward ──
                positions[-1] = self.target
                for i in range(len(positions) - 2, -1, -1):
                    dir_vec = positions[i] - positions[i+1]
                    dir_vec /= np.linalg.norm(dir_vec)
                    positions[i] = positions[i+1] + dir_vec * lengths[i]

                # ── Backward ──
                positions[0] = self.sk.world_position(self.chain[0])
                for i in range(1, len(positions)):
                    dir_vec = positions[i] - positions[i-1]
                    dir_vec /= np.linalg.norm(dir_vec)
                    positions[i] = positions[i-1] + dir_vec * lengths[i-1]

                # Y/Z floor 제약
                for i, pos in enumerate(positions):
                    if pos[1] < self.y_floor:
                        positions[i][1] = self.y_floor
                    if pos[2] < self.z_floor:
                        positions[i][2] = self.z_floor

                # Pole vector 제약 — 팔꿈치(index 1 또는 2) 방향 고정
                if self.pole_vector is not None and len(positions) > 2:
                    positions[1] = self._apply_pole(positions[0], positions[-1], positions[1])

                if np.linalg.norm(positions[-1] - self.target) < tolerance:
                    break

        # 위치 → 각 관절 Quaternion 회전으로 역산
        self._positions_to_quaternions(positions)

    def _apply_pole(self, root, end, elbow):
        """팔꿈치를 pole vector 방향으로 투영"""
        axis = end - root
        axis /= np.linalg.norm(axis)
        projected = elbow - root
        projected = projected - np.dot(projected, axis) * axis
        # pole vector 방향
        pole_dir = self.pole_vector / np.linalg.norm(self.pole_vector)
        pole_proj = pole_dir - np.dot(pole_dir, axis) * axis
        pole_proj /= np.linalg.norm(pole_proj)

        elbow_new = root + np.dot(projected, pole_proj) * pole_proj
        elbow_new += np.dot((elbow - root), axis) * axis   # 축 방향 거리 유지
        return elbow_new

    def _positions_to_quaternions(self, positions: np.ndarray):
        """새 위치를 각 관절 rotation(Quaternion)으로 변환"""
        # 세부 구현은 Two-bone IK 수식 기반. 요약:
        for i, joint_name in enumerate(self.chain[:-1]):
            # 다음 관절 방향 벡터
            desired_dir = (positions[i+1] - positions[i])
            desired_dir /= np.linalg.norm(desired_dir)
            # 현재 방향 (부모 기준)
            current = self.sk.joints[self.chain[i+1]].offset
            current = current / np.linalg.norm(current)
            # 축·각도 → Quaternion
            axis = np.cross(current, desired_dir)
            angle = np.arccos(np.clip(np.dot(current, desired_dir), -1, 1))
            if np.linalg.norm(axis) > 1e-6:
                axis /= np.linalg.norm(axis)
                q = quaternion.from_rotation_vector(axis * angle)
                self.sk.joints[joint_name].rotation = q


# ──────────────────────────────────────────────
# 4. 수어 애니메이션 클립
# ──────────────────────────────────────────────
@dataclass
class SignPose:
    """특정 시점의 전체 스켈레톤 스냅샷"""
    time:      float                     # 초 단위
    rotations: dict                      # {joint_name: Quaternion}


@dataclass
class SignClip:
    """하나의 수어 단어·문장의 애니메이션 클립"""
    sign_name: str
    poses:     List[SignPose] = field(default_factory=list)
    duration:  float = 1.5

    def sample_at(self, t: float) -> dict:
        """t 초 시점의 관절 회전 (Slerp 보간)"""
        if t <= self.poses[0].time:
            return self.poses[0].rotations
        if t >= self.poses[-1].time:
            return self.poses[-1].rotations

        # 주변 키프레임 2개 찾아 Slerp
        for i in range(len(self.poses) - 1):
            p1, p2 = self.poses[i], self.poses[i+1]
            if p1.time <= t <= p2.time:
                alpha = (t - p1.time) / (p2.time - p1.time)
                interpolated = {}
                for name in p1.rotations:
                    if name in p2.rotations:
                        interpolated[name] = slerp(
                            p1.rotations[name], p2.rotations[name], alpha,
                        )
                    else:
                        interpolated[name] = p1.rotations[name]
                return interpolated
        return self.poses[-1].rotations


# ──────────────────────────────────────────────
# 5. 수어 클립 체이닝 — 자연스러운 연결
# ──────────────────────────────────────────────
def chain_sign_clips(
    clips: List[SignClip],
    transition_duration: float = 0.3,
) -> SignClip:
    """
    여러 수어 클립을 transition 기간 Slerp로 부드럽게 이음.
    예: "안녕하세요" + "반갑습니다" → 2개 클립을 하나로.
    """
    merged_poses = []
    current_time = 0.0

    for idx, clip in enumerate(clips):
        if idx == 0:
            # 첫 클립 그대로
            for p in clip.poses:
                merged_poses.append(SignPose(
                    time=current_time + p.time,
                    rotations=p.rotations,
                ))
            current_time += clip.duration
        else:
            # transition 생성
            last_rot  = merged_poses[-1].rotations
            first_rot = clip.poses[0].rotations
            n_trans = 5
            for step in range(1, n_trans + 1):
                alpha = step / n_trans
                interp = {}
                for name in last_rot:
                    if name in first_rot:
                        interp[name] = slerp(last_rot[name], first_rot[name], alpha)
                    else:
                        interp[name] = last_rot[name]
                merged_poses.append(SignPose(
                    time=current_time + (step / n_trans) * transition_duration,
                    rotations=interp,
                ))
            current_time += transition_duration

            # 클립 본체
            for p in clip.poses:
                merged_poses.append(SignPose(
                    time=current_time + p.time,
                    rotations=p.rotations,
                ))
            current_time += clip.duration

    return SignClip(
        sign_name=" + ".join(c.sign_name for c in clips),
        poses=merged_poses,
        duration=current_time,
    )


# ──────────────────────────────────────────────
# 6. 실행 예시
# ──────────────────────────────────────────────
if __name__ == "__main__":
    sk = Skeleton()
    sk.build_default_human()

    # "안녕하세요" 수어 — 오른손을 이마 앞에서 앞으로 내민 후 돌림
    solver = FABRIKSolver(
        sk,
        chain=["shoulder_R", "upper_arm_R", "forearm_R", "wrist_R"],
    )
    solver.pole_vector = np.array([-1, 0, 0])    # 팔꿈치는 옆으로

    # Pose 1: 오른손을 이마 앞 (0.2, 1.7, 0.3)
    solver.set_target(np.array([0.2, 1.7, 0.3]))
    solver.solve()
    pose1 = SignPose(
        time=0.0,
        rotations={name: j.rotation for name, j in sk.joints.items()},
    )

    # Pose 2: 손을 앞으로 내밀기 (0.4, 1.5, 0.5)
    solver.set_target(np.array([0.4, 1.5, 0.5]))
    solver.solve()
    pose2 = SignPose(
        time=0.75,
        rotations={name: j.rotation for name, j in sk.joints.items()},
    )

    clip_hello = SignClip("안녕", [pose1, pose2], duration=1.5)

    # "반갑습니다"도 유사하게 만들어 체이닝
    # ...

    print(f"수어 클립 '{clip_hello.sign_name}' 생성됨")
    print(f"  포즈 수: {len(clip_hello.poses)}")
    print(f"  지속:   {clip_hello.duration}s")
    print(f"  0.375초 시점 손 위치:",
          sk.world_position("wrist_R"))
```

## 함정 카드

1. **"Euler 각도로 키프레임 저장"** → 저장 후 Slerp 할 때 gimbal lock 재발. 반드시 **Quaternion으로 저장·보간**. 표시용 UI에서 Euler 변환만 허용.

2. **"Slerp의 t 범위 밖 값 허용"** → `t < 0` 또는 `t > 1`이면 "extrapolation"으로 이상한 회전. 항상 `t = max(0, min(1, t))` 클램핑.

3. **"IK 솔버를 매 프레임 전체 재계산"** → 60FPS × 양팔 × 5 손가락 = CPU 폭발. **관절 dirty 플래그**로 변경 있는 체인만 재계산. WebGL 렌더러는 기본 제공(Three.js IK 플러그인 참고).

4. **"Pole vector를 안 설정"** → FABRIK가 팔꿈치 방향을 임의로 결정 → 팔이 뒤로 꺾이거나 몸통 뚫음. 수어 아바타는 **pole vector 필수** (일반적으로 옆 바깥 방향).

5. **"손가락 관절에 Slerp 없이 점프"** → "아" 발음 수어는 엄지·검지 벌림, "우" 발음은 오므림. 두 사이 전환이 즉시 바뀌면 손가락 딸깍거림. 손가락도 **0.1~0.2초 Slerp 전환** 적용. 부드러움 필수.

## 연습문제

**문제 1.** Quaternion 곱셈 순서가 `q_shoulder × q_arm`인지 `q_arm × q_shoulder`인지가 왜 중요한가?

> **해설:** Quaternion 곱셈은 **비교환적**(non-commutative). `A × B ≠ B × A`. 회전 순서가 다르면 최종 자세 달라진다. 관절 체인은 **부모 먼저, 자식 나중**: `q_final = q_shoulder × q_arm × q_forearm × q_wrist`. 이 순서를 뒤집으면 손목 회전이 어깨 기준에서 해석돼 **팔 전체가 휘청**. 코드 리뷰에서 가장 흔한 버그. 명시적 행렬 곱 또는 Quaternion library의 chain 함수로 순서 보장.

**문제 2.** Slerp 중 q2 방향이 q1과 180도 반대로 나와 회전이 "먼 쪽"으로 돈다. 해결책은?

> **해설:** Quaternion은 `q`와 `-q`가 같은 회전을 표현. Slerp가 "짧은 경로" 선택하게 하려면: `if dot(q1, q2) < 0: q2 = -q2` 로 부호 보정. numpy-quaternion의 `slerp`는 자동 처리하지만 직접 구현 시 필수. 없으면 손이 360도 빙 돌아 도달 — 비자연스럽고 멀미 유발.

**문제 3.** 한국 수어는 왼손잡이·오른손잡이 차이가 있다. 3D 아바타에서 어떻게 지원하는가?

> **해설:** ① **대칭 플립**: X축 미러(왼손↔오른손) — 클립을 복제 후 Quaternion x 성분 부호 뒤집기 `q_flipped = Quat(w, -x, y, -z)` ② **기본 클립은 오른손 기준**으로 녹화·저장, **렌더링 시 사용자 설정**에 따라 런타임 플립 ③ 주의: 일부 수어는 **양손 동시**이고 dominant hand 역할이 고정 — 이런 경우는 왼손잡이 전용 클립 별도 녹화 필요. 실측: 한국 수어 자주 쓰는 1,000 단어 중 약 15%가 완전 대칭 불가능해 이원 세트 필요.
