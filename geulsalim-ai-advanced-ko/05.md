# 챕터 05 · EPUB 3 접근성 메타 (TTS·점자 힌트 내장)

## 학습 목표
- EPUB 3 Accessibility 1.2 (2026) 스펙의 `schema:accessibilityFeature`·`schema:accessibilityHazard` 필수 메타 필드를 작성할 수 있다
- ONIX Codelist 196과 schema.org 접근성 메타의 크로스워크를 이해하고 유통 메타(서지)까지 함께 생성할 수 있다
- 복원된 고문서 JSON에서 접근성 EPUB을 자동 생성하는 파이프라인을 구축할 수 있다

---

## 이론

### 왜 HTML만으론 부족한가

HTML은 **연결된 웹** 환경을 전제한다. 오프라인·e-reader·점자 단말·TTS 전용 기기에서는 **EPUB 3**가 사실상 표준이다. 한국시각장애인연합회가 운영하는 넷드림도서관, 일본 DAISY 컨소시엄, 미국 Bookshare 모두 EPUB 3 기반.

EPUB 3의 핵심 우위:
- **메타데이터 내장**: 접근성 정보를 콘텐츠와 분리 불가
- **미디어 오버레이**: 텍스트 + 동기화된 오디오 (SMIL)
- **MathML, SSML**: 수식·음성 합성 지시어 삽입
- **점자 힌트**: 특정 구간을 점자 단말기에 어떻게 전달할지 명세

### 2026 필수 메타 필드 (W3C EPUB Accessibility 1.2)

```xml
<metadata xmlns:dc="http://purl.org/dc/elements/1.1/"
          xmlns:opf="http://www.idpf.org/2007/opf">
  <dc:title>조선시대 상소문 복원본</dc:title>
  <dc:creator>국립중앙박물관 × WIA 글살림</dc:creator>
  <dc:language>ko</dc:language>

  <!-- 접근성 필수 메타 (W3C EPUB Accessibility 1.2) -->
  <meta property="schema:accessibilityFeature">alternativeText</meta>
  <meta property="schema:accessibilityFeature">longDescription</meta>
  <meta property="schema:accessibilityFeature">readingOrder</meta>
  <meta property="schema:accessibilityFeature">structuralNavigation</meta>
  <meta property="schema:accessibilityFeature">tableOfContents</meta>
  <meta property="schema:accessibilityFeature">MathML</meta>
  <meta property="schema:accessibilityFeature">synchronizedAudioText</meta>

  <meta property="schema:accessibilityHazard">noFlashing</meta>
  <meta property="schema:accessibilityHazard">noMotionSimulation</meta>
  <meta property="schema:accessibilityHazard">noSound</meta>

  <meta property="schema:accessibilitySummary">
    조선시대 상소문을 AI가 복원한 EPUB입니다. 
    모든 이미지에 대체 텍스트와 긴 설명이 포함되어 있으며,
    AI가 예측한 글자는 별도 마크업으로 구분됩니다.
    TTS 읽기 순서와 점자 변환 힌트가 내장되어 있습니다.
  </meta>

  <meta property="schema:accessMode">textual</meta>
  <meta property="schema:accessMode">visual</meta>

  <meta property="schema:accessModeSufficient">textual</meta>
  <meta property="schema:accessModeSufficient">textual,visual</meta>

  <!-- 준수 선언 (2026-03 명확화: refines로 다중 주장 체이닝) -->
  <meta property="dcterms:conformsTo">
    https://www.w3.org/TR/epub-a11y-12/#wcag-aa
  </meta>

  <!-- 인증기관 (있으면) -->
  <link rel="dcterms:conformsTo"
        href="https://a11y.wiabook.com/certified/urn:wia:cert:xxx"/>
</metadata>
```

### schema.org vs ONIX 크로스워크

유통·서점 연동은 **ONIX Codelist 196** 을 쓴다:

| schema.org | ONIX Code | 의미 |
|---|---|---|
| `alternativeText` | 14 | 대체 텍스트 있음 |
| `longDescription` | 15 | 긴 설명 있음 |
| `readingOrder` | 13 | 명시적 읽기 순서 |
| `structuralNavigation` | 11 | 구조적 탐색 지원 |
| `tableOfContents` | 22 | 목차 있음 |
| `MathML` | 17 | MathML 사용 |
| `synchronizedAudioText` | 20 | 동기화된 음성+텍스트 |

ONIX는 별도 XML로 서지 유통에 쓰인다. 형님의 **WIA Book 출판 파이프라인**은 ONIX 자동 생성을 이미 지원한다(`wia-standards-covers` 플러그인). 이 둘을 동시에 만들어야 e-book 판매+접근성 동시 충족.

### 미디어 오버레이 (SMIL) — 동기화 음성

고문서 복원본은 **원문 한문 + 한국어 번역** 두 층이 있다. 시각장애인은 "번역문 듣고, 원문 한자 각각은 자세히 알림"을 원한다. SMIL로 텍스트 구간과 오디오 구간 동기화:

```xml
<par>
  <text src="chapter1.xhtml#para1"/>
  <audio src="audio/chapter1_para1.mp3" clipBegin="0s" clipEnd="8.5s"/>
</par>
```

TTS로 오디오를 생성한 뒤 SMIL로 매핑하면 EPUB 리더가 **현재 읽는 구절을 하이라이트**하며 재생. DAISY 도서의 실제 사용 방식이다.

## 실전 코드

```python
# EPUB 3 Accessibility 1.2 자동 생성기
# pip install ebooklib==0.18 lxml pillow

import os
import uuid
import hashlib
from pathlib import Path
from datetime import datetime, timezone
from dataclasses import dataclass, field
from typing import List, Dict, Optional
from ebooklib import epub
from lxml import etree


# ──────────────────────────────────────────────
# 1. EPUB 빌더 (접근성 메타 내장)
# ──────────────────────────────────────────────
@dataclass
class EPUBMeta:
    title:                 str
    author:                str
    language:              str = "ko"
    identifier:            str = field(default_factory=lambda: f"urn:wia:doc:{uuid.uuid4()}")
    publisher:             str = "WIA 글살림"
    description:           str = ""
    subject_keywords:      List[str] = field(default_factory=list)
    accessibility_summary: str = ""


class AccessibleEPUBBuilder:
    """
    접근성 메타 내장 EPUB 3.3 빌더.
    """

    # schema:accessibilityFeature 값
    A11Y_FEATURES = [
        "alternativeText", "longDescription", "readingOrder",
        "structuralNavigation", "tableOfContents",
        "synchronizedAudioText",    # TTS 오디오 첨부 시
    ]
    # schema:accessibilityHazard
    A11Y_HAZARDS = ["noFlashing", "noMotionSimulation", "noSound"]
    # schema:accessMode
    A11Y_MODES_SUFFICIENT = ["textual", "textual,visual"]

    def __init__(self, meta: EPUBMeta, include_audio: bool = False):
        self.meta = meta
        self.include_audio = include_audio
        self.book = epub.EpubBook()
        self._setup()

    def _setup(self):
        book = self.book
        book.set_identifier(self.meta.identifier)
        book.set_title(self.meta.title)
        book.set_language(self.meta.language)
        book.add_author(self.meta.author)

        # ──────────── 접근성 메타 ────────────
        features = self.A11Y_FEATURES[:]
        if self.include_audio:
            features.append("synchronizedAudioText")
        else:
            features.remove("synchronizedAudioText")

        for f in features:
            book.add_metadata(
                "http://schema.org/",
                "accessibilityFeature", f,
                others={"property": "schema:accessibilityFeature"},
            )
        for h in self.A11Y_HAZARDS:
            book.add_metadata(
                "http://schema.org/",
                "accessibilityHazard", h,
                others={"property": "schema:accessibilityHazard"},
            )
        for m in ["textual", "visual"]:
            book.add_metadata(
                "http://schema.org/",
                "accessMode", m,
                others={"property": "schema:accessMode"},
            )
        for m in self.A11Y_MODES_SUFFICIENT:
            book.add_metadata(
                "http://schema.org/",
                "accessModeSufficient", m,
                others={"property": "schema:accessModeSufficient"},
            )

        book.add_metadata(
            "http://schema.org/",
            "accessibilitySummary",
            self.meta.accessibility_summary or self._default_summary(),
            others={"property": "schema:accessibilitySummary"},
        )
        # WCAG AA 준수 선언
        book.add_metadata(
            None,
            "link", "",
            {"rel": "dcterms:conformsTo",
             "href": "https://www.w3.org/TR/epub-a11y-12/#wcag-aa"},
        )

    def _default_summary(self) -> str:
        return (
            "이 EPUB은 AI 복원 고문서입니다. "
            "모든 이미지에 대체 텍스트와 긴 설명이 포함되어 있으며, "
            "AI가 예측한 글자는 <span class='predicted'>로 명확히 구분됩니다. "
            "WCAG 2.2 AA 준수, NVDA·VoiceOver·점자 단말기 호환."
        )

    # ────────── 콘텐츠 추가 ──────────
    def add_cover(self, image_path: str, alt_text: str) -> None:
        with open(image_path, "rb") as f:
            img_bytes = f.read()
        self.book.set_cover("cover.jpg", img_bytes)

    def add_chapter(
        self,
        title:       str,
        xhtml_body:  str,
        file_name:   str,
        lang:        Optional[str] = None,
    ) -> epub.EpubHtml:
        """접근성 강화된 XHTML 챕터"""
        xhtml = f"""<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml"
      xmlns:epub="http://www.idpf.org/2007/ops"
      lang="{lang or self.meta.language}">
<head>
  <meta charset="utf-8"/>
  <title>{title}</title>
  <link rel="stylesheet" type="text/css" href="style/main.css"/>
</head>
<body epub:type="bodymatter">
<section aria-labelledby="h-{file_name}">
  <h1 id="h-{file_name}">{title}</h1>
  {xhtml_body}
</section>
</body>
</html>"""
        chapter = epub.EpubHtml(
            title=title, file_name=file_name,
            lang=lang or self.meta.language,
            content=xhtml,
        )
        self.book.add_item(chapter)
        return chapter

    def add_image(self, image_path: str, alt_text: str, long_desc: str = "") -> str:
        """이미지 추가. 반환: EPUB 내부 경로"""
        with open(image_path, "rb") as f:
            img_bytes = f.read()
        file_name = Path(image_path).name
        image = epub.EpubImage(
            uid=Path(image_path).stem,
            file_name=f"images/{file_name}",
            media_type=f"image/{Path(image_path).suffix[1:]}",
            content=img_bytes,
        )
        self.book.add_item(image)
        return f"images/{file_name}"

    def add_media_overlay(
        self,
        chapter_file: str,
        audio_path:   str,
        text_segments: List[Dict],    # [{"text_id": "para1", "begin": 0, "end": 8.5}, ...]
    ) -> None:
        """SMIL 미디어 오버레이 추가 (TTS 오디오 동기화)"""
        # 오디오 파일 등록
        with open(audio_path, "rb") as f:
            audio_bytes = f.read()
        audio_name = Path(audio_path).name
        audio = epub.EpubItem(
            uid=f"audio_{Path(audio_path).stem}",
            file_name=f"audio/{audio_name}",
            media_type=f"audio/{Path(audio_path).suffix[1:]}",
            content=audio_bytes,
        )
        self.book.add_item(audio)

        # SMIL XML 생성
        smil = etree.Element(
            "{http://www.w3.org/ns/SMIL}smil",
            version="3.0",
        )
        body = etree.SubElement(smil, "{http://www.w3.org/ns/SMIL}body")
        seq = etree.SubElement(
            body, "{http://www.w3.org/ns/SMIL}seq",
            {"{http://www.idpf.org/2007/ops}textref": chapter_file,
             "{http://www.idpf.org/2007/ops}type":    "chapter"},
        )
        for seg in text_segments:
            par = etree.SubElement(seq, "{http://www.w3.org/ns/SMIL}par")
            etree.SubElement(par, "{http://www.w3.org/ns/SMIL}text",
                             src=f"{chapter_file}#{seg['text_id']}")
            etree.SubElement(par, "{http://www.w3.org/ns/SMIL}audio",
                             src=f"audio/{audio_name}",
                             clipBegin=f"{seg['begin']:.3f}s",
                             clipEnd=f"{seg['end']:.3f}s")

        smil_bytes = etree.tostring(smil, xml_declaration=True, encoding="utf-8")
        smil_item = epub.EpubItem(
            uid=f"smil_{Path(chapter_file).stem}",
            file_name=f"media/{Path(chapter_file).stem}.smil",
            media_type="application/smil+xml",
            content=smil_bytes,
        )
        self.book.add_item(smil_item)

    # ────────── TOC + Spine + 저장 ──────────
    def finalize(
        self,
        chapters: List[epub.EpubHtml],
        output_path: str,
    ) -> str:
        self.book.toc = tuple(
            (epub.Section(ch.title), (ch,)) for ch in chapters
        )
        self.book.add_item(epub.EpubNcx())
        self.book.add_item(epub.EpubNav())
        self.book.spine = ["nav"] + chapters

        # 스타일
        style = """
body { font-family: 'Noto Serif KR', serif; line-height: 1.7; }
h1 { font-size: 1.8em; margin-top: 1.5em; }
.predicted {
    background: rgba(255, 107, 53, 0.15);
    border-bottom: 2px dotted #6c3ce1;
}
.predicted::after { content: "*"; color: #6c3ce1; }
.transcript { font-family: 'Noto Sans KR', sans-serif; }
blockquote { border-left: 3px solid #ccc; padding-left: 1em; margin: 1.5em 0; }
img { max-width: 100%; height: auto; }
"""
        css = epub.EpubItem(
            uid="style_main", file_name="style/main.css",
            media_type="text/css", content=style,
        )
        self.book.add_item(css)

        Path(output_path).parent.mkdir(parents=True, exist_ok=True)
        epub.write_epub(output_path, self.book)
        return output_path


# ──────────────────────────────────────────────
# 2. ONIX 메타 병행 생성 (WIA Book 플러그인 연동)
# ──────────────────────────────────────────────
def build_onix_accessibility(
    isbn: str,
    features: List[str],   # schema.org 형식, ONIX로 매핑
) -> str:
    """ONIX Codelist 196 기반 접근성 메타"""
    SCHEMA_TO_ONIX = {
        "alternativeText":       "14",
        "longDescription":       "15",
        "readingOrder":          "13",
        "structuralNavigation":  "11",
        "tableOfContents":       "22",
        "MathML":                "17",
        "synchronizedAudioText": "20",
    }

    onix_features = []
    for f in features:
        code = SCHEMA_TO_ONIX.get(f)
        if code:
            onix_features.append(f"""    <ProductFormFeature>
      <ProductFormFeatureType>09</ProductFormFeatureType>
      <ProductFormFeatureValue>{code}</ProductFormFeatureValue>
    </ProductFormFeature>""")

    # 준수 선언 (Code 02 = EPUB Accessibility 1.2 WCAG AA)
    conformance = """    <ProductFormFeature>
      <ProductFormFeatureType>09</ProductFormFeatureType>
      <ProductFormFeatureValue>02</ProductFormFeatureValue>
      <ProductFormFeatureDescription>
        EPUB Accessibility Specification 1.2 — WCAG 2.2 Level AA
      </ProductFormFeatureDescription>
    </ProductFormFeature>"""

    return f"""<Product>
  <RecordReference>{isbn}</RecordReference>
  <NotificationType>03</NotificationType>
  <ProductIdentifier>
    <ProductIDType>15</ProductIDType>
    <IDValue>{isbn}</IDValue>
  </ProductIdentifier>
  <DescriptiveDetail>
{chr(10).join(onix_features)}
{conformance}
  </DescriptiveDetail>
</Product>"""


# ──────────────────────────────────────────────
# 3. End-to-End 실행
# ──────────────────────────────────────────────
def build_restoration_epub(
    restoration_json_path: str,
    certificate_json_path: str,
    restored_image_path:   str,
    tts_audio_path:        Optional[str],    # 없으면 SMIL 생략
    output_epub_path:      str,
    isbn:                  str = "979-11-XXXX-XXX-X",
) -> dict:
    import json
    with open(restoration_json_path, encoding="utf-8") as f:
        rest = json.load(f)
    with open(certificate_json_path, encoding="utf-8") as f:
        cert = json.load(f)

    meta = EPUBMeta(
        title="조선시대 상소문 복원본",
        author="국립중앙박물관 (WIA 글살림 복원)",
        language="ko",
        description="AI 하이브리드로 복원된 조선 후기 상소문",
        subject_keywords=["조선시대", "상소문", "고문서", "복원", "접근성"],
    )

    builder = AccessibleEPUBBuilder(meta, include_audio=bool(tts_audio_path))
    builder.add_cover(restored_image_path, alt_text="상소문 표지")

    # 이미지 삽입
    img_src = builder.add_image(
        restored_image_path,
        alt_text="복원된 조선 후기 상소문 이미지",
        long_desc=rest.get("long_description", ""),
    )

    # 예측 마크업 재사용
    transcript_html = rest.get("masked_text", "")
    for pred in rest.get("final", []):
        transcript_html = transcript_html.replace(
            "□", f'<span class="predicted" aria-label="AI 예측 글자">{pred}</span>', 1,
        )

    body = f"""
<section aria-labelledby="ch-intro">
  <h2 id="ch-intro">문서 정보</h2>
  <p>소장처: 국립중앙박물관 · 제작 시기: 조선 후기 (18세기 추정)</p>
  <p>복원 일시: {cert['issued']} · 진위 증명: 
    <a href="{cert['proof']['verifyUrl']}">WIA Chain</a>
  </p>
</section>

<section aria-labelledby="ch-image">
  <h2 id="ch-image">복원 이미지</h2>
  <figure>
    <img src="{img_src}"
         alt="조선 후기 상소문. 한지에 한자 세로 쓰기, 훼손된 영역 AI 복원"
         aria-describedby="img-longdesc"/>
    <figcaption id="img-longdesc">
      조선 후기 관료가 임금에게 올린 상소문의 일부. 
      중앙 영역이 훼손되어 AI가 주변 문맥으로 복원하였습니다.
    </figcaption>
  </figure>
</section>

<section aria-labelledby="ch-transcript">
  <h2 id="ch-transcript">판독 텍스트</h2>
  <p>AI 예측 부분은 별표(*)로 구분됩니다.</p>
  <blockquote lang="ko-Hani" class="transcript" id="para1">
    {transcript_html}
  </blockquote>
</section>

<section aria-labelledby="ch-translation">
  <h2 id="ch-translation">현대어 번역</h2>
  <p id="para2">엎드려 아뢰옵건대, 신들이 삼가 국가의 지극히 중대한 일을 진술하고자 합니다...</p>
</section>
"""
    ch = builder.add_chapter("상소문 복원본", body, "chapter1.xhtml")

    # 미디어 오버레이 (TTS 오디오 있을 때)
    if tts_audio_path:
        builder.add_media_overlay(
            chapter_file="chapter1.xhtml",
            audio_path=tts_audio_path,
            text_segments=[
                {"text_id": "para1", "begin": 0,    "end": 12.5},
                {"text_id": "para2", "begin": 12.5, "end": 28.3},
            ],
        )

    epub_path = builder.finalize([ch], output_epub_path)

    # ONIX 병행 생성
    onix_xml = build_onix_accessibility(
        isbn=isbn,
        features=AccessibleEPUBBuilder.A11Y_FEATURES + (
            ["synchronizedAudioText"] if tts_audio_path else []
        ),
    )
    onix_path = output_epub_path.replace(".epub", "_onix.xml")
    with open(onix_path, "w", encoding="utf-8") as f:
        f.write(onix_xml)

    print(f">> EPUB: {epub_path}")
    print(f">> ONIX: {onix_path}")
    return {"epub": epub_path, "onix": onix_path}


if __name__ == "__main__":
    result = build_restoration_epub(
        restoration_json_path="./output/sangso_01/restoration.json",
        certificate_json_path="./output/sangso_01/certificate.json",
        restored_image_path  ="./output/sangso_01/restored_image.jpg",
        tts_audio_path       =None,     # 챕터 11에서 생성
        output_epub_path     ="./output/sangso_01/sangso.epub",
    )
```

### EPUBCheck 자동 검증

## 함정 카드

1. **"accessibilitySummary를 영어 템플릿 그대로"** → `schema:accessibilitySummary`는 **사용자가 읽는 실제 문구**다. 도서 언어(ko)와 일치시켜야 함. 영어 템플릿을 ko EPUB에 그대로 두면 한국 시각장애인이 영어 읽기 — 접근성 자체 실패.

2. **"accessModeSufficient를 `textual` 하나만"** → "텍스트만으로도 완전 접근 가능"을 의미. 복원 이미지가 서사의 핵심이면 이는 거짓. **`textual,visual`** 조합으로 "텍스트+이미지 조합이 충분"이 정직. 거짓 선언은 EU EAA 제재 대상.

3. **"ONIX 코드와 schema.org 값 불일치"** → `schema:accessibilityFeature: alternativeText`와 ONIX Code 14가 매핑. 한쪽만 넣으면 **서점 유통 메타와 EPUB 내부 메타가 어긋남**. 반드시 동시 생성하고 매핑 테이블을 단일 소스 오브 트루스로 유지.

4. **"이미지 `alt`에 파일명을 그대로 씀"** → `alt="IMG_0032.jpg"` 유형. NVDA는 그대로 읽는다 → "아이엠지 공공삼이 점 제이피지". 반드시 **AI 자동 생성 alt**(챕터 06)로 대체. 또는 수동 필수.

5. **"SMIL 오디오 구간을 대충 맞춘다"** → 텍스트 하이라이트가 음성과 **±0.2초**만 어긋나도 사용자가 멀미. Piper·Whisper Timestamps로 구간 자동 추출 권장 (챕터 11). 수동 편집은 분 단위 EPUB에서만.

## 연습문제

**문제 1.** EPUB에 `schema:accessibilityFeature: synchronizedAudioText`를 넣었지만 실제 SMIL은 없다. 어떤 문제가 생기는가?

> **해설:** **거짓 메타 선언**. DAISY Consortium의 ACE 감사기가 "feature 선언돼 있는데 SMIL 파일 없음"으로 **실패** 판정. 유통사(Bookshare, 넷드림)에서 반송되거나 품질 점수 하락. 더 심각: 시각장애인 사용자가 "오디오 있다고 해서 받았는데 없다"고 불만 → 서비스 신뢰 훼손. **feature 선언과 실제 내용은 반드시 일치**해야 한다. 빠뜨릴 건 빠뜨리고, 넣을 건 제대로 넣는 게 정직.

**문제 2.** ONIX 메타를 생성해야 하는 이유는? EPUB 내부 메타로 충분하지 않은가?

> **해설:** **다른 계층이다**. EPUB 메타는 "책 파일 안의 정보"로 리더·AT 기기가 본다. ONIX는 "유통·서지 정보"로 **서점·도서관·출판사 시스템**이 본다. 독자가 구매 전 "이 책에 대체텍스트 있나?"를 서점 페이지에서 확인하려면 ONIX가 필요. 또한 EU EAA 2025-06 발효로 **전자책 판매 사이트는 접근성 메타 표시 의무** — ONIX가 그 데이터 원천. 둘을 분리하면 판매 페이지와 실제 파일 간 불일치 발생 가능.

**문제 3.** 내 EPUB이 KOGL(공공누리) 1유형 원본인데, EPUB 내부 메타에 어떻게 표기하는가?

> **해설:** ① **dc:rights** 필드에 `"공공누리 제1유형 (출처: 국립중앙박물관)"` 명시 ② **dc:source**에 원본 URL ③ **link rel="license"**에 `https://www.kogl.or.kr/info/license.do#01-tab` ④ 추가로 **meta property="dcterms:accessRights"** 에 `open access` 선언. 이는 접근성 메타와 별개로 **저작권 메타**라 accessibilityFeature와 충돌 없음. 다만 "원본은 공공누리, 복원 결과는 AI 생성물"로 이중 라이선스 필요 — 복원 결과는 MIT 또는 CC BY 권장하여 재이용성 보장.
