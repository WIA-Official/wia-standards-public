제2장: 메모리 유형 및 기술

DRAM, NAND Flash, SRAM, NOR, 그리고 차세대 메모리 기술의 모든 것

2.1 메모리 기술 분류 체계

메모리 반도체는 데이터 보존 방식, 접근 속도, 용량, 용도에 따라 다양한 유형으로 분류됩니다. 크게 휘발성 메모리(Volatile)와 비휘발성 메모리(Non-Volatile)로 나뉘며, 각각 독특한 물리적 구조와 작동 원리를 갖습니다.

분류 메모리 유형 대표 제품 주요 특징 전형적 용도
휘발성 DRAM DDR4, DDR5, LPDDR5, HBM3 빠른 속도, 대용량, 리프레시 필요 PC/서버 메인 메모리
SRAM Cache Memory 초고속, 소용량, 리프레시 불필요 CPU/GPU 캐시
비휘발성 NAND Flash SSD, UFS, eMMC, SD 카드 고밀도, 저전력, 블록 단위 삭제 대용량 스토리지
NOR Flash BIOS, 펌웨어 칩 바이트 단위 랜덤 액세스, 고속 읽기 임베디드 코드 저장
신흥 메모리 MRAM STT-MRAM, SOT-MRAM 자기 기반, 초저전력, 무한 내구성 IoT, 우주항공
ReRAM RRAM, Memristor 저항 변화, 3D 적층 가능 뉴로모픽 AI 칩
PCM 3D XPoint (Intel Optane) 상변화, DRAM과 SSD 중간 성능 스토리지 클래스 메모리

2.2 DRAM: 현대 컴퓨팅의 주역

2.2.1 DRAM의 작동 원리

DRAM(Dynamic Random Access Memory)은 하나의 트랜지스터와 하나의 커패시터(1T1C)로 구성된 셀에 데이터를 저장합니다. 커패시터에 전하가 충전되면 '1', 방전되면 '0'으로 인식됩니다. 그러나 커패시터는 시간이 지나면 자연 방전되므로, 수 밀리초마다 리프레시(Refresh) 작업을 통해 데이터를 재충전해야 합니다.

삼성전자와 SK하이닉스의 DRAM 기술력: 두 기업은 10nm급 이하 초미세 공정으로 DRAM을 양산하며, 세계에서 가장 앞선 기술을 보유하고 있습니다. 삼성전자는 2024년 12nm급 DDR5 DRAM을, SK하이닉스는 세계 최초 1β(베타)nm 16Gb DDR5를 개발했습니다. 이는 한 칩에 200억 개 이상의 트랜지스터를 집적하는 놀라운 기술입니다.

2.2.2 DRAM 제품군

DDR5 DRAM (PC/서버용)

  • 데이터 전송 속도: 4800~8400 MT/s (DDR4 대비 1.87배 향상)
  • 대역폭: 최대 67.2 GB/s (단일 모듈)
  • 용량: 단일 모듈 최대 128GB (서버용)
  • 전압: 1.1V (DDR4 1.2V 대비 절전)
  • 주요 기업: 삼성전자 (점유율 43%), SK하이닉스 (30%), Micron (27%)

LPDDR5/5X (모바일용 저전력 DRAM)

  • 데이터 전송 속도: LPDDR5 6400 MT/s → LPDDR5X 8533 MT/s
  • 전력 효율: LPDDR4X 대비 20% 절전
  • 응용: 스마트폰, 태블릿, 노트북
  • 리더: 삼성전자는 2024년 LPDDR5X 10.7Gbps 제품을 업계 최초로 양산

HBM3/HBM3E (AI용 고대역폭 메모리)

  • 적층 구조: 8~12개 DRAM 다이를 수직으로 쌓음 (3D 패키징)
  • 대역폭: HBM3 6.4 Tbps, HBM3E 9.6 Tbps (DDR5의 약 50배)
  • 용량: 단일 패키지 24GB~36GB
  • 응용: NVIDIA H100/H200 GPU, AMD MI300, Google TPU
  • 시장 지배자: SK하이닉스는 HBM 시장 점유율 50% 이상으로 1위, 삼성전자는 HBM3E 12단 (48GB) 개발 완료

2.2.3 DRAM의 기술적 도전 과제

  • 스케일링 한계: 10nm 이하에서는 커패시터 크기가 작아져 전하 유지 시간이 감소하고, 리프레시 빈도가 증가하여 전력 소비가 늘어납니다.
  • RowHammer 취약점: 특정 행(row)을 반복 액세스하면 인접 행의 데이터가 손상될 수 있는 물리적 현상으로, 보안 위협이 됩니다.
  • 발열 문제: DDR5는 온칩 ECC와 고속 동작으로 인해 DDR4 대비 발열이 증가하여, 히트싱크가 필수입니다.

2.3 NAND Flash: 스토리지 혁명

2.3.1 NAND Flash의 원리

NAND Flash는 플로팅 게이트(Floating Gate)에 전자를 가두어 데이터를 저장합니다. 전자가 있으면 '0', 없으면 '1'입니다. 전원이 꺼져도 전자는 절연층에 갇혀 있어 데이터가 유지되므로 비휘발성 메모리입니다.

NAND는 블록(Block) 단위로 데이터를 지우고, 페이지(Page) 단위로 읽고 씁니다. 이는 높은 집적도를 가능하게 하지만, 삭제 전에는 덮어쓰기가 불가능하다는 제약이 있습니다.

2.3.2 NAND Flash 세대: SLC → MLC → TLC → QLC

유형 셀당 비트 P/E 사이클 속도 가격 주요 용도
SLC
(Single-Level Cell)		 1 bit 100,000회 가장 빠름 가장 비쌈 산업용, 서버 캐시
MLC
(Multi-Level Cell)		 2 bits 10,000회 빠름 비쌈 엔터프라이즈 SSD
TLC
(Triple-Level Cell)		 3 bits 3,000회 보통 저렴 소비자용 SSD
QLC
(Quad-Level Cell)		 4 bits 1,000회 느림 가장 저렴 대용량 스토리지
PLC
(Penta-Level Cell)		 5 bits 500회 (예상) 매우 느림 초저가 콜드 스토리지 (연구 중)

* P/E 사이클 = Program/Erase 사이클, 즉 데이터를 쓰고 지울 수 있는 횟수

2.3.3 3D NAND: 수직으로 쌓아 올린 혁신

2D(평면) NAND가 물리적 한계에 도달하자, 2013년부터 3D NAND 기술이 등장했습니다. 메모리 셀을 수직으로 층층이 쌓아 올려 집적도를 비약적으로 높이는 방식입니다.

삼성전자와 SK하이닉스의 3D NAND 경쟁:

  • 삼성전자: 세계 최초로 24단 3D NAND를 개발(2013)한 이후, 2024년 현재 286단 V-NAND를 양산 중입니다. TLC 기준 1TB 용량을 단일 칩에 구현했습니다.
  • SK하이닉스: 2024년 321단 4D NAND를 개발하며 업계 최다 적층 기록을 보유하고 있습니다. Charge Trap 방식으로 더 안정적인 데이터 저장이 가능합니다.

2.3.4 NAND Flash 제품군

  • UFS (Universal Flash Storage): 스마트폰용 고속 스토리지, UFS 4.0은 읽기 4,200 MB/s, 쓰기 2,800 MB/s
  • NVMe SSD: PC/서버용 초고속 SSD, PCIe 5.0 기준 순차 읽기 14,000 MB/s
  • eMMC: 저가형 모바일 기기용, eMMC 5.1은 최대 400 MB/s
  • SD/microSD 카드: SD Express는 최대 985 MB/s 지원

2.4 SRAM: 속도의 왕

SRAM(Static RAM)은 6개의 트랜지스터(6T)로 구성되어 있으며, 전원이 공급되는 한 데이터를 유지합니다. 리프레시가 필요 없어 DRAM보다 훨씬 빠르지만, 셀 크기가 크고 비용이 높아 CPU/GPU의 L1/L2/L3 캐시 같은 소용량 고속 메모리로만 사용됩니다.

프로세서 L1 캐시 L2 캐시 L3 캐시 액세스 시간
Intel Core i9-14900K 80KB x 24코어 2MB x 24코어 36MB (공유) L1: ~1ns, L3: ~10ns
AMD Ryzen 9 7950X 64KB x 16코어 1MB x 16코어 64MB (공유) L1: ~0.9ns, L3: ~12ns
Apple M3 Max 192KB x 16코어 24MB (공유) - L1: ~0.8ns, L2: ~8ns

2.5 NOR Flash: 코드 실행의 베이스

NOR Flash는 NAND와 달리 바이트 단위 랜덤 액세스가 가능하고 읽기 속도가 빠릅니다. 그러나 쓰기가 느리고 용량당 가격이 비싸 주로 펌웨어, BIOS, 임베디드 시스템의 부트 코드 저장에 사용됩니다.

  • 용량: 일반적으로 512KB ~ 256MB
  • 읽기 속도: 100~150 MB/s
  • 쓰기 속도: 1~5 MB/s (NAND보다 훨씬 느림)
  • 응용: 자동차 ECU, 네트워크 장비 부트롬, IoT 디바이스

2.6 신흥 메모리 (Emerging Memory)

2.6.1 MRAM (자기저항 메모리)

MRAM(Magnetoresistive RAM)은 자기 방향으로 데이터를 저장합니다. STT-MRAM(Spin-Transfer Torque)과 SOT-MRAM(Spin-Orbit Torque) 두 가지 방식이 있으며, 다음과 같은 장점이 있습니다:

  • 무한 내구성: 10^15회 이상의 쓰기 사이클 가능 (NAND의 1000배)
  • 초저전력: 쓰기 에너지가 DRAM의 1/10 수준
  • 비휘발성: 전원 차단 시에도 데이터 유지
  • 고속: 나노초 단위 액세스 (NAND보다 1000배 빠름)

삼성전자는 2022년 28nm 공정 eMRAM을 양산하기 시작했으며, 자동차 MCU와 IoT 칩에 적용하고 있습니다.

2.6.2 ReRAM / RRAM (저항 변화 메모리)

ReRAM은 금속 산화물의 저항 변화로 데이터를 저장합니다. 전압을 가하면 산화물 내부에 필라멘트(전도 경로)가 형성되어 저항이 낮아지고(0), 역전압을 가하면 필라멘트가 끊어져 저항이 높아집니다(1).

  • 장점: 초저전력, 빠른 속도, 3D 적층 용이, 뉴로모픽 AI 연산 가능
  • 단점: 상용화 초기 단계, 내구성 개선 필요 (10^6~10^9 사이클)
  • 응용: AI 가속기의 인메모리 컴퓨팅, 엣지 디바이스

2.6.3 PCM (상변화 메모리)

PCM(Phase Change Memory)은 칼코게나이드 화합물의 결정 상태(crystalline)와 비정질 상태(amorphous) 간 전환으로 데이터를 저장합니다. Intel과 Micron이 공동 개발한 3D XPoint가 대표적이며, Intel Optane 제품군으로 상용화되었습니다.

  • 성능: DRAM의 1/2~1/3 속도, SSD의 10배 이상 속도
  • 내구성: 10^8~10^9 사이클 (NAND보다 높음)
  • 용량: DRAM보다 고밀도 (Optane은 512GB 모듈 제공)
  • 단점: DRAM보다 비싸고, 시장 확대 어려움 (Intel은 2022년 Optane 사업 종료)

핵심 요점 정리 (Key Takeaways)

  1. DRAM은 휘발성 메모리의 핵심: DDR5, LPDDR5X, HBM3E로 진화하며, 삼성전자와 SK하이닉스가 10nm급 초미세 공정으로 세계를 주도합니다.
  2. HBM은 AI 시대의 필수 메모리: HBM3E는 9.6 Tbps의 초고속 대역폭을 제공하며, SK하이닉스가 50% 이상의 시장 점유율로 1위입니다.
  3. NAND Flash는 3D 적층으로 진화: 삼성 286단, SK하이닉스 321단 3D NAND는 1TB 단일 칩을 가능하게 하며, SLC → MLC → TLC → QLC로 용량 증가와 비용 절감이 이루어집니다.
  4. SRAM은 속도가 생명: CPU/GPU 캐시로 사용되며, 리프레시가 필요 없어 나노초 단위 초고속 액세스가 가능합니다.
  5. 신흥 메모리 MRAM, ReRAM, PCM은 미래 기술: 무한 내구성, 초저전력, 비휘발성을 동시에 제공하며, AI와 IoT 응용에 적합합니다.
  6. 각 메모리는 고유한 트레이드오프 존재: 속도 vs 용량, 휘발성 vs 비휘발성, 비용 vs 성능을 고려한 시스템 설계가 중요합니다.

복습 문제 (Review Questions)

Q1: DRAM의 1T1C 구조를 설명하고, 왜 리프레시(Refresh) 작업이 필요한지 서술하세요.
Q2: HBM3E가 DDR5에 비해 어떤 기술적 우위를 가지며, 주로 어떤 응용 분야에 사용되는지 설명하세요.
Q3: NAND Flash의 SLC, MLC, TLC, QLC 차이를 P/E 사이클, 속도, 가격 측면에서 비교하세요.
Q4: 삼성전자와 SK하이닉스의 3D NAND 적층 기술을 비교하고, 현재 최고 적층 단수를 제시하세요.
Q5: SRAM이 DRAM보다 빠른 이유는 무엇이며, 왜 대용량 메인 메모리로 사용되지 않는지 설명하세요.
Q6: MRAM(자기저항 메모리)의 주요 장점 3가지를 설명하고, 삼성전자의 MRAM 양산 현황을 서술하세요.
Q7: NOR Flash와 NAND Flash의 근본적인 차이점을 데이터 액세스 방식과 주요 용도 측면에서 비교하세요.
