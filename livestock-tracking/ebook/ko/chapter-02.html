제2장: RFID 기술을 이용한 가축 식별

2.1 RFID 기본 원리

RFID(전파식별, Radio Frequency Identification)는 현대 축산 추적의 핵심 기술이다. 바코드와 달리 비접촉식 자동 인식이 가능해 가축이 게이트를 통과할 때 자동으로 식별된다. ISO/IEC 15693·14443은 일반 RFID, ISO 11784/11785는 가축 전용으로 별도 규정.

2.2 ISO 11784/11785 — 가축 RFID의 국제 표준

한국 축산물이력제도 이 표준을 따른다.

2.3 한국 축산에서 사용되는 RFID 태그 종류

2.3.1 귀표형 RFID (가장 보편)

  • 형태: 플라스틱 귀표에 RFID 칩 내장 + 12자리 이력번호 인쇄
  • 장착 위치: 오른쪽 귀(한국 표준)
  • 비용: 개당 3,000~5,000원
  • 수명: 10~15년
  • 장단점: 육안 확인 가능·저렴·장착 용이 / 연 2% 탈락

2.3.2 볼러스형 RFID (루멘 삽입)

  • 형태: 세라믹 캡슐(약 10 g, 길이 8 cm)
  • 장착: 경구 투여 → 제1위(루멘)에 정착
  • 비용: 10,000~15,000원/개
  • 장단점: 평생 탈락 없음·도난 방지 / 고가·육안 확인 불가
  • 적용: 1++ 한우, 종모우

2.4 한국 축산농가 RFID 도입 절차

  1. 축산물이력관리시스템 가입 — mtrace.go.kr에서 축산농가 등록.
  2. 이력번호 발급 — 축산물품질평가원(KAPE)이 12자리 번호 부여.
  3. RFID 태그 구매 — 인증 제조사(Allflex / Datamars / Rosse) 제품.
  4. 송아지 귀표 장착 — 생후 7일 이내.
  5. 스캔 및 등록 — RFID 리더기로 스캔 → 시스템 등록.
  6. 백신 접종 이력 갱신 — 구제역·브루셀라 등.

2.6 한국 법규 및 의무 사항

축산법 제33조 (이력관리)

  • 태그 부착 의무: 생후 30일 이내(소), 출하 전(돼지)
  • 이동 신고: 가축 판매/구매 시 48시간 내
  • 도축 신고: 도축 24시간 전
  • 위반 과태료: 1,000만원 이하

2.7 실전 팁 — 귀표 탈락 방지

  • 장착 시기: 생후 7~14일.
  • 위치: 귀 중간 1/3 지점, 혈관 회피.
  • 소독: 장착 전후 귀 소독.
  • 이중 태그: 1++ 한우는 귀표 + 볼러스 병행.
  • 정기 점검: 월 1회 상태 확인, 탈락 시 즉시 재부착.

2.8 ICAR 제조사 코드와 64-bit 페이로드

ICAR(국제가축기록위원회)가 제조사별로 코드 공간을 할당한다. 한국에 흔히 들어오는 제조사는 Allflex(0001~), Datamars(0006~), Avid(0900~), Rosse(8019~). 64-bit 페이로드 구조:

비트 0       : 동물 플래그 (1 = 가축 응용)
비트 1-15    : 예약
비트 16-26   : ICAR 제조사 코드 / 국가 사무국 코드
비트 27-37   : ISO 3166-1 숫자 국가 코드 (한국 = 410)
비트 38-65   : 12자리 고유 일련번호

예: 410-000123456789  →  국가 410 · 제조사 000 · 일련 123456789

2.9 EPC Gen2 UHF — 무리 단위 일괄 스캔

LF는 개체별 정확 인식, UHF EPC Gen2 v2(GS1 EPCglobal 표준)는 무리 단위 일괄 스캔에 적합. KR866 대역, 1~10 m 인식 거리, 게이트나 도축장 계류장에서 1초에 1,000건까지 anti-collision(ISO 18000-63 Q-protocol)으로 처리. 실무는 LF로 신원 확정, UHF로 무리 통계 — 두 기술 병행이 표준.

EPC: urn:epc:id:sgtin:8801234.056789.001234
URI: https://id.gs1.org/01/08801234567896

2.10 KAPE 축평원·농관원 시스템 흐름

RFID 스캔 결과는 다음 흐름으로 흐른다.

  1. 농가 단말 → 축산물이력관리시스템(NLMS) — 12자리 이력번호 등록.
  2. NLMS → 축산물품질평가원(KAPE) — 한우 등급 판정 시 등급 결과 매칭.
  3. NLMS → 농림축산검역본부(APQA) — 가축전염병 신고 시 자동 연동.
  4. NLMS → 식품안전나라 — 회수 명령 발령 시 매장 POS까지 5분 내 전파.

요약

RFID는 한국 축산물이력제의 핵심 기술이며 ISO 11784/11785 표준을 따른다. 귀표형이 가장 보편이고, 정부 보조금 50%로 리더기 부담을 줄인다. UHF EPC Gen2를 병행하면 무리 단위 일괄 스캔이 가능하고, KAPE·APQA·식품안전나라 시스템과 자동 연동되어 회수·등급·검역 흐름이 매끄럽게 닫힌다.
