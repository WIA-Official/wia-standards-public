# WIA-PET-010: Pet Translator Specification v1.1

**Status:** Historical  
**Release Date:** 2024-Q3  
**Previous Version:** v1.0  
**Superseded By:** v1.2

## Changes from v1.0

### New Features
- Added support for ultrasonic vocalizations (up to 100 kHz)
- Improved context window (now 5 minutes vs 2 minutes)
- Added Japanese language support
- Enhanced error handling and retry logic

### Bug Fixes
- Fixed timestamp precision issues in WebSocket protocol
- Corrected audio normalization for high-frequency sounds
- Resolved race condition in concurrent translation requests

### Performance Improvements
- Reduced average latency by 15%
- Improved accuracy to 88% (from 85%)
- Optimized memory usage for video processing

## Updated Specifications

### Audio Processing
- **Sample Rate:** Now supports up to 192 kHz (previously 48 kHz)
- **Formats:** Added FLAC and OGG support

### Supported Languages
- English (en)
- Korean (ko)
- Spanish (es)
- French (fr)
- German (de)
- **Japanese (ja)** — NEW

### Performance Requirements
- **Latency:** < 400ms for audio-only (improved from 500ms)
- **Accuracy:** > 88% on standard test set (improved from 85%)
