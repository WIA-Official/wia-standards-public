# WIA-PET-010: Pet Translator Specification v1.2

**Status:** Historical  
**Release Date:** 2024-Q4  
**Previous Version:** v1.1  
**Superseded By:** v2.0

## Changes from v1.1

### New Features
- Beta emotion detection (3 basic emotions: happy, anxious, neutral)
- Added rabbit and guinea pig species support
- Context-aware translation (basic environmental context)
- Batch translation API endpoint
- Webhook notifications for translations

### Enhancements
- Improved body language detection accuracy
- Extended context window to 10 minutes
- Added Chinese (Simplified) language support
- Enhanced privacy controls (data retention settings)

### Bug Fixes
- Fixed edge cases in silence detection
- Corrected timezone handling in historical queries
- Resolved WebSocket reconnection issues

### Deprecated Features
- Legacy v0.x compatibility layer (to be removed in v2.0)

## New API Endpoints

### POST /v1/translate/batch
Translate multiple events in a single request.

```json
{
  "events": [
    {"petId": "...", "audio": "..."},
    {"petId": "...", "audio": "..."}
  ]
}
```

### POST /v1/webhooks
Configure webhook for translation notifications.

```json
{
  "url": "https://your-domain.com/webhook",
  "events": ["translation", "emotion_change"],
  "secret": "..."
}
```

## Updated Schema

### Translation Output (v1.2)
```json
{
  "translationId": "...",
  "translation": "...",
  "emotion": {
    "primary": "happy | anxious | neutral",
    "confidence": 0.85
  },
  "intent": "...",
  "confidence": 0.92
}
```

## Performance Requirements

- **Latency:** < 350ms for audio-only (improved from 400ms)
- **Latency:** < 800ms for audio+video (improved from 1000ms)
- **Accuracy:** > 90% on standard test set (improved from 88%)
- **Uptime:** 99.9% availability (improved from 99.5%)

## Migration Notes

### From v1.1 to v1.2
- No breaking changes
- New emotion field in response (optional to use)
- Webhook configuration requires new API call
- Recommended: Update client libraries to latest version
