4.1 GHG 인벤토리 기본

온실가스 인벤토리는 모든 기후완화 노력의 기반입니다. 감축 진행 상황을 측정하는 기준선을 제공합니다.

4.1.1 대상 온실가스

가스 화학식 GWP (100년) 주요 배출원
이산화탄소 CO2 1 화석연료 연소
메탄 CH4 28 농업, 천연가스
아산화질소 N2O 265 농업, 산업
수소불화탄소 HFCs 12-14,800 냉매, 에어컨
과불화탄소 PFCs 6,500-9,200 전자, 알루미늄
육불화황 SF6 23,500 전기 장비

4.3 계산 방법론

4.3.1 기본 공식

// GHG 배출 계산
배출량 (tCO2e) = 활동 데이터 × 배출계수 × GWP

// 예시: 천연가스
{
  "activityData": {
    "value": 10000,
    "unit": "therm"
  },
  "emissionFactor": {
    "value": 5.3,
    "unit": "kg CO2/therm"
  },
  "calculation": "10,000 × 5.3 = 53,000 kg CO2 = 53 tCO2"
}

4.3.2 배출계수 출처

출처 범위 용도
IPCC 가이드라인 글로벌 기본 계수 일반 추정
EPA eGRID 미국 전력 계통 미국 Scope 2
DEFRA 영국 종합 상세 계산
Ecoinvent 전과정 데이터 Scope 3 계산

4.6 장 요약

핵심 내용

  • GHG 인벤토리는 7가지 온실가스 대상
  • 배출은 Scope 1, 2, 3으로 분류
  • 계산은 활동 데이터, 배출계수, GWP 사용
  • 경계 설정이 포함 배출 결정
  • 데이터 품질 요구사항은 MAL 수준에 따라 증가
