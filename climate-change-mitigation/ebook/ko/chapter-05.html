5.1 감축 우선순위

효과적인 기후완화는 명확한 우선순위를 따릅니다: 먼저 배출을 회피하고, 불가피한 경우 감축하며, 잔여 배출만 상쇄합니다.

// 감축 우선순위 계층
{
  "priority1_avoid": {
    "description": "배출 완전 방지",
    "examples": [
      "원격근무 정책",
      "제품 재설계",
      "순환경제"
    ],
    "effectiveness": "100% 감축"
  },
  "priority2_reduce": {
    "description": "불가피한 배출 최소화",
    "examples": [
      "에너지 효율",
      "프로세스 최적화",
      "재생에너지"
    ],
    "effectiveness": "30-90% 감축"
  },
  "priority3_substitute": {
    "description": "저탄소 대안으로 전환",
    "examples": [
      "전기화",
      "지속가능 소재",
      "그린수소"

  },
  "priority4_offset": {
    "description": "잔여 배출 보상",
    "examples": [
      "탄소 크레딧",

      "탄소 제거"
    ],
    "note": "최후의 수단으로만"
  }
}

5.2 에너지 효율

부문 효율 조치 일반적 절감
건물 LED 조명, HVAC 최적화, 단열 20-40%
산업 모터 드라이브, 열회수, 공정 제어 15-30%
운송 경로 최적화, 차량 현대화 10-25%
데이터센터 냉각 최적화, 서버 가상화 30-50%

5.6 장 요약

핵심 내용

  • 회피-감축-대체-상쇄 우선순위 준수
  • 에너지 효율이 가장 비용 효과적인 첫 단계
  • 다양한 재생에너지 조달 옵션 가용
  • 전기화로 심층 탈탄소화 가능
  • 과학기반목표는 1.5°C에 연간 4.2% 감축 필요
