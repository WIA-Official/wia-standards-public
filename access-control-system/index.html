WIA-ACS

Access Control System

접근 통제 시스템

A unified standard for physical and digital access control, integrating RBAC, ABAC, biometric authentication, and Zero Trust architecture for comprehensive security.

물리적 및 디지털 접근 통제를 위한 통합 표준으로, RBAC, ABAC, 생체 인증 및 제로 트러스트 아키텍처를 통합하여 포괄적인 보안을 제공합니다.

Four Implementation Phases

4단계 구현 과정

1

Data Format

데이터 형식

Standardized credential schemas, permission models, and user attributes for consistent access control across systems.

시스템 간 일관된 접근 통제를 위한 표준화된 자격 증명 스키마, 권한 모델 및 사용자 속성 정의.

2

API Interface

API 인터페이스

RESTful and gRPC APIs for authentication, authorization, credential management, and audit logging with multi-factor authentication support.

다중 요소 인증을 지원하는 인증, 권한 부여, 자격 증명 관리 및 감사 로깅을 위한 RESTful 및 gRPC API.

3

Protocol

프로토콜

Secure communication protocols with TLS 1.3, OAuth 2.0, OpenID Connect, and SAML for enterprise integration.

기업 통합을 위한 TLS 1.3, OAuth 2.0, OpenID Connect 및 SAML을 사용하는 보안 통신 프로토콜.

4

Integration

통합

Seamless integration with LDAP, Active Directory, PACS systems, smart cards, RFID, NFC, and biometric devices.

LDAP, Active Directory, PACS 시스템, 스마트 카드, RFID, NFC 및 생체 인식 장치와의 원활한 통합.
