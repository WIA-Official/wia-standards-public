# WIA-SEMI-012: Performance Metrics

## Key Performance Indicators

### Resolution
- Definition: Smallest detectable change
- Units: bits (ADC), meters (altitude), degrees (orientation)

### Sensitivity
- Definition: Output/Input ratio
- Examples: 16384 LSB/g, 10 mV/°C, 50 nA/ppm

### Accuracy
- Absolute: Deviation from true value
- Relative: Repeatability between measurements

### Noise
- Types: Thermal, quantization, flicker (1/f)
- Measurement: RMS noise, noise density (units/√Hz)

### Power Consumption
- Active mode: Full operation
- Sleep mode: Minimal current
- Shutdown: Complete power-off

### Response Time
- Definition: Time to reach 63% of final value (τ63)
- Or: Time to reach 90% (τ90)
