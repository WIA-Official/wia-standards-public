📷 이미지 센서의 기초

이미지 센서는 빛을 전기 신호로 변환하여 디지털 이미지를 생성하는 반도체 장치입니다. 스마트폰 카메라, DSLR, 의료 영상, 자율주행차, 보안 시스템 등 현대 사회의 거의 모든 영역에서 이미지 센서가 사용됩니다.

이미지 센서 시장은 스마트폰과 자동차 산업의 성장에 힘입어 지속적으로 확대되고 있습니다. 특히 고해상도, 저조도 성능, 3D 깊이 인식에 대한 수요가 증가하면서 기술 혁신이 가속화되고 있습니다.

💡 시장 현황

글로벌 이미지 센서 시장은 2025년까지 약 350억 달러 규모로 성장할 것으로 예상됩니다. Sony와 Samsung이 시장을 주도하며, 자동차용 센서와 산업용 비전 시스템이 새로운 성장 동력으로 부상하고 있습니다.

🔦 ToF (Time-of-Flight) 센서

ToF 센서는 빛이 대상까지 이동하고 돌아오는 시간을 측정하여 깊이 정보를 획득합니다. 얼굴 인식, 증강 현실(AR), 제스처 인식, 로봇 비전 등에 사용됩니다.

작동 원리

ToF 센서는 적외선(IR) LED 또는 레이저를 사용하여 광 펄스를 방출합니다. 이 빛이 대상에서 반사되어 돌아오는 시간을 측정하여 거리를 계산합니다. 빛의 속도(약 3×10⁸ m/s)를 알고 있으므로, 왕복 시간의 절반에 광속을 곱하면 거리를 얻습니다.

📐 거리 계산 공식

d = (c × t) / 2

d: 거리, c: 광속 (3×10⁸ m/s), t: 왕복 시간

ToF 센서 유형

유형 방식 장점 단점
Direct ToF (dToF) 펄스 왕복 시간 직접 측정 장거리, 높은 정확도 높은 전력 소비, 비용
Indirect ToF (iToF) 변조광의 위상 차이 측정 저전력, 저비용 다중 경로 간섭, 거리 제한

🔴 LiDAR 시스템

LiDAR(Light Detection and Ranging)는 레이저 펄스를 사용하여 주변 환경의 3D 점군(Point Cloud) 데이터를 생성하는 센서입니다. 자율주행차, 드론, 지도 제작, 고고학 탐사 등에서 핵심적인 역할을 합니다.

// LiDAR 점군 데이터 처리 예시
interface LiDARPoint {
    x: number;      // X 좌표 (m)
    y: number;      // Y 좌표 (m)
    z: number;      // Z 좌표 (m)
    intensity: number;   // 반사 강도
    ring: number;    // 스캔 라인 번호
    timestamp: number; // 시간 스탬프 (ns)
}

function processPointCloud(points: LiDARPoint[]): void {
    // 지면 제거
    const nonGroundPoints = points.filter(p => p.z > 0.2);
    
    // 거리 기반 필터링
    const validPoints = nonGroundPoints.filter(p => {
        const distance = Math.sqrt(p.x**2 + p.y**2 + p.z**2);
        return distance > 0.5 && distance < 100;
    });
    
    // 클러스터링 및 객체 감지
    const clusters = euclideanClustering(validPoints, 0.5);
    console.log(`감지된 객체 수: ${clusters.length}`);
}
