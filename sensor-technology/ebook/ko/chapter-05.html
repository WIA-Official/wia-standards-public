💓 심박수 및 PPG 센서

PPG(Photoplethysmography, 광용적맥파)는 빛을 사용하여 혈액량의 변화를 측정하는 비침습적 광학 기술입니다. 스마트워치, 피트니스 트래커, 의료 기기 등에서 심박수 측정에 널리 사용됩니다.

PPG 작동 원리

PPG 센서는 LED에서 방출된 빛이 피부를 통과하거나 반사되는 것을 측정합니다. 심장이 박동하면 혈액량이 변하고, 이에 따라 빛의 흡수량도 변합니다. 이 변화를 분석하여 심박수, 심박 변이도(HRV), 혈중 산소 포화도(SpO2)를 측정합니다.

🟢
녹색 LED (520-570nm)

운동 중 심박수 측정에 최적화. 피부 표면의 모세혈관에서 반사. 높은 신호 대 잡음비.

🔴
적색 LED (660nm)

SpO2 측정에 사용. 산소화 헤모글로빈의 흡수 특성 이용. 깊은 조직 투과.

⚫
적외선 LED (940nm)

SpO2 및 휴식 시 심박수. 탈산소화 헤모글로빈 특성. 깊은 조직 투과.

📊
다파장 PPG

여러 파장 조합으로 정확도 향상. 혈압 추정, 스트레스 분석에 활용.

❤️

📊 ECG (심전도) 센서

ECG(Electrocardiogram) 센서는 심장의 전기적 활동을 측정합니다. 심장 근육이 수축할 때 발생하는 전기 신호를 피부 표면의 전극을 통해 감지합니다. 부정맥, 심근경색, 심방세동 등 심장 질환 진단에 필수적입니다.

ECG 파형 해석

💓 ECG 파형 구성요소

P파: 심방 탈분극 (심방 수축)

QRS 복합체: 심실 탈분극 (심실 수축)

T파: 심실 재분극 (심실 이완)

R-R 간격: 심박수 계산에 사용

ECG 측정 구성

1️⃣
단일 리드 (Lead I)

양손 또는 손목 간 측정. 스마트워치, 휴대용 ECG에 사용. 기본 심박 분석용.

3️⃣
3-리드 ECG

사지 전극 3개 사용. Lead I, II, III 측정. 응급 모니터링용.

6️⃣
12-리드 ECG

표준 임상 ECG. 10개 전극으로 12개 뷰 생성. 심장 전체 분석.

📱
웨어러블 ECG

패치형 또는 밴드형. 장기간 연속 모니터링. 부정맥 조기 발견.

// ECG 신호 처리 예시
interface ECGAnalysis {
    heartRate: number;        // BPM
    rPeaks: number[];         // R피크 시간 (ms)
    hrv: {
        sdnn: number;          // 심박 변이도 (ms)
        rmssd: number;         // 연속 차이의 RMS
    };
    abnormalities: string[];  // 감지된 이상
}

function analyzeECG(signal: number[], sampleRate: number): ECGAnalysis {
    // 대역 통과 필터 (0.5-40 Hz)
    const filtered = bandpassFilter(signal, 0.5, 40, sampleRate);
    
    // R 피크 검출 (Pan-Tompkins 알고리즘)
    const rPeaks = detectRPeaks(filtered, sampleRate);
    
    // 심박수 계산
    const rrIntervals = calculateRRIntervals(rPeaks, sampleRate);
    const heartRate = 60000 / mean(rrIntervals);
    
    // HRV 분석
    const sdnn = std(rrIntervals);
    const rmssd = calculateRMSSD(rrIntervals);
    
    return {
        heartRate: Math.round(heartRate),
        rPeaks,
        hrv: { sdnn, rmssd },
        abnormalities: detectAbnormalities(rrIntervals)
    };
}

🩸 혈중 산소 포화도 (SpO2)

SpO2(Peripheral Oxygen Saturation) 센서는 혈액 중 산소화 헤모글로빈의 비율을 측정합니다. 호흡기 질환, 수면무호흡증, 고산병, COVID-19 모니터링 등에 중요합니다.

측정 원리

SpO2 측정은 산소화 헤모글로빈(HbO2)과 탈산소화 헤모글로빈(Hb)이 빛을 다르게 흡수하는 특성을 이용합니다. 적색(660nm)과 적외선(940nm) 두 파장에서 흡수율 비를 측정하여 산소 포화도를 계산합니다.

💨
정상 SpO2 범위
95-100%
SpO2 수준 상태 조치
95-100% 정상 특별한 조치 필요 없음
90-94% 경미한 저산소증 주의 관찰, 원인 파악
85-89% 중등도 저산소증 산소 공급, 의료 상담
85% 미만 심각한 저산소증 즉시 응급 의료 조치
⚠️ SpO2 측정 시 주의사항

매니큐어, 손가락 온도, 움직임, 주변광 등이 측정 정확도에 영향을 미칠 수 있습니다. 의료용 펄스 옥시미터는 FDA 또는 의료기기 인증을 획득해야 합니다.

💉 연속 혈당 모니터링 (CGM)

CGM(Continuous Glucose Monitoring)은 당뇨병 환자가 혈당 수준을 실시간으로 모니터링할 수 있게 해주는 기술입니다. 피하에 삽입된 센서가 간질액의 포도당 농도를 지속적으로 측정합니다.

CGM 시스템 구성

🔧
센서

피하 삽입형 전기화학 센서. 포도당 산화효소가 포도당을 분해하여 전류 생성.

📡
트랜스미터

센서 데이터를 무선으로 수신기에 전송. 블루투스 또는 NFC 사용.

📱
수신기/앱

혈당 수치 표시, 추세 분석, 경보 알림. 스마트폰 앱 또는 전용 장치.

🔄
인슐린 펌프 연동

자동화된 인슐린 전달(AID) 시스템과 통합. 폐루프 제어 가능.

🏃 활동 및 운동 센서

활동 센서는 걸음 수, 소모 칼로리, 이동 거리, 수면 품질 등을 추적합니다. MEMS 가속도계, 자이로스코프, 기압계 등을 결합하여 다양한 활동을 인식합니다.
