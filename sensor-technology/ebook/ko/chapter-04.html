환경 센서 (가스, 온도, 습도)

대기질 모니터링, 산업 안전, 스마트 빌딩을 위한 환경 센서 기술을 탐구합니다.

⏱️ 읽기 시간 30분 📊 중급 수준 📅 2025년 1월 업데이트

🌬️ 가스 센서 개요

가스 센서는 공기 중의 특정 가스 농도를 측정하는 센서입니다. 산업 안전, 대기질 모니터링, 자동차 배기가스 분석, 실내 공기질 관리 등 다양한 분야에서 필수적인 역할을 합니다.

가스 센서의 선택은 감지하려는 가스의 종류, 필요한 정확도, 작동 환경, 전력 소비량 등을 고려해야 합니다. 각 센싱 방식은 고유한 장단점을 가지고 있어 응용 분야에 따라 적합한 기술을 선택해야 합니다.

가스 센서 유형

⚡
전기화학 센서

가스가 전극에서 산화/환원 반응을 일으켜 전류 발생. CO, NO2, O3 등 유독 가스 측정에 적합.

🔥
촉매 연소식

가연성 가스가 촉매 표면에서 연소하여 열 발생. 메탄, 프로판 등 가연성 가스 탐지.

💎
반도체 (MOS)

금속 산화물 표면의 저항 변화 측정. SnO2, ZnO 등. 저비용, VOC 감지에 널리 사용.

🌈
적외선 (NDIR)

가스의 적외선 흡수 특성 이용. CO2, CH4 등. 높은 선택성과 안정성.

📡
광이온화 (PID)

UV광으로 가스 분자 이온화. ppb 수준 VOC 감지 가능. 산업 안전용.

🔬
레이저 분광

가스 분자의 특정 흡수선 측정. 극고정밀 측정, 연구 및 환경 모니터링.

주요 감지 가스 및 응용

가스 센서 유형 측정 범위 응용 분야
CO (일산화탄소) 전기화학, MOS 0-1000 ppm 가정 안전, 주차장, 터널
CO2 (이산화탄소) NDIR 0-10,000 ppm 실내 공기질, HVAC, 농업
CH4 (메탄) 촉매 연소, NDIR 0-100% LEL 가스 누출 감지, 광산
VOCs PID, MOS 0-1000 ppb 실내 공기질, 산업 위생
O3 (오존) 전기화학 0-1 ppm 대기질 모니터링
NH3 (암모니아) 전기화학, MOS 0-100 ppm 냉동 시설, 축산업

🌡️ 온도 센서

온도 센서는 가장 널리 사용되는 센서 중 하나로, 산업 공정, HVAC, 의료 기기, 가전제품, 자동차 등 거의 모든 분야에서 사용됩니다. 다양한 측정 원리와 범위를 가진 센서들이 있습니다.

온도 센서 유형

🔴
열전대 (Thermocouple)

두 금속 접합부의 제벡 효과 이용. -200°C~2000°C 광범위. 산업용 표준.

🟡
RTD (저항 온도 감지기)

백금 등 금속의 저항 변화 측정. Pt100, Pt1000. 높은 정확도, -200°C~850°C.

🟢
서미스터 (NTC/PTC)

세라믹 반도체의 저항 변화. 높은 감도, 저비용. -40°C~125°C 일반적.

🔵
반도체 온도 센서

PN 접합의 온도 특성 이용. 디지털 출력, 쉬운 인터페이스. LM35, DS18B20.

🟣
적외선 온도계

물체가 방출하는 적외선 측정. 비접촉, 고온 측정, 이동 물체.

⚪
광섬유 온도 센서

광섬유의 광학적 특성 변화 이용. EMI 면역, 위험 환경용.

열전대 유형 비교

유형 재료 온도 범위 특징
K형 크로멜-알루멜 -200°C ~ 1250°C 가장 널리 사용, 범용
J형 철-콘스탄탄 -40°C ~ 750°C 환원 분위기에 적합
T형 구리-콘스탄탄 -200°C ~ 350°C 저온용, 높은 정확도
S형 백금-로듐/백금 0°C ~ 1450°C 고온 정밀 측정
R형 백금-로듐/백금 0°C ~ 1450°C S형과 유사, 더 높은 출력 
// NTC 서미스터 온도 계산 (Steinhart-Hart 방정식)
function calculateTemperature(resistance: number): number {
    // 10kΩ NTC 서미스터 (B=3950K)
    const R0 = 10000;  // 25°C에서의 저항
    const T0 = 298.15; // 25°C in Kelvin
    const B = 3950;    // B 상수
    
    // 간소화된 B-방정식
    const tempKelvin = 1 / ((1 / T0) + (1 / B) * Math.log(resistance / R0));
    const tempCelsius = tempKelvin - 273.15;
    
    return tempCelsius;
}

// DS18B20 디지털 온도 센서 읽기
async function readDS18B20(): Promise<number> {
    const rawData = await oneWire.read(DS18B20_ADDRESS);
    const temp = (rawData[1] << 8 | rawData[0]) * 0.0625;
    return temp;
}

💧 습도 센서

습도 센서는 공기 중의 수분 함량을 측정합니다. 상대 습도(RH), 절대 습도, 이슬점 등 다양한 형태로 표현됩니다. HVAC, 기상 관측, 산업 공정, 저장 시설 등에서 중요합니다.

습도 센서 유형

🔋
용량형 습도 센서

흡습성 폴리머 유전체의 정전용량 변화 측정. 넓은 측정 범위, 선형 응답.

⚡
저항형 습도 센서

흡습성 재료의 저항 변화 측정. 저비용, 빠른 응답. 고습도에서 비선형.

🌡️
이슬점 센서

거울 냉각하여 결로점 측정. 높은 정확도. 기준 센서로 사용.

📊
복합 센서

온도와 습도 동시 측정. DHT22, SHT40, BME280 등.

주요 습도 센서 제품

제품 제조사 정확도 인터페이스
SHT40 Sensirion ±1.8% RH I2C
BME280 Bosch ±3% RH I2C/SPI
HDC2080 Texas Instruments ±2% RH I2C
DHT22 Aosong ±2-5% RH 단선 디지털
HIH-6130 Honeywell ±4% RH I2C
💡 습도 측정 단위

상대 습도 (RH): 현재 수증기량 / 최대 수증기량 × 100%

절대 습도: 단위 부피당 수증기 질량 (g/m³)

이슬점: 수증기가 응결되기 시작하는 온도 (°C)

🏭 통합 환경 센서

현대 환경 센서는 여러 측정 기능을 하나의 칩에 통합합니다. Bosch BME680/688, Sensirion SGP40 등은 온도, 습도, 기압, 가스를 동시에 측정할 수 있습니다.

대표 통합 센서

🔷
Bosch BME688

온도, 습도, 기압, VOC 가스 4-in-1. AI 기능 탑재로 가스 분류 가능.

🟢
Sensirion SEN55

PM1.0/2.5/4/10, VOC, NOx, 온도, 습도 통합. 대기질 종합 측정.

🟣
ScioSense ENS160

다중 가스 센서로 TVOC, eCO2 측정. 실내 공기질 인덱스 제공.

🔴
AMS AS7341

11채널 스펙트럼 센서. 환경광 분석, 색상 측정에 활용.

// BME688 센서 통합 읽기
interface EnvironmentData {
    temperature: number;     // °C
    humidity: number;        // %RH
    pressure: number;        // hPa
    gasResistance: number;   // Ω
    iaq: number;             // 공기질 지수 (0-500)
    co2Equivalent: number;   // ppm
    breathVoc: number;       // ppm
}

async function readBME688(): Promise<EnvironmentData> {
    // 강제 측정 모드로 전환
    await i2c.write(BME688_ADDR, CTRL_MEAS, 0x25);
    
    // 측정 완료 대기
    await delay(100);
    
    // 데이터 레지스터 읽기
    const rawData = await i2c.readBlock(BME688_ADDR, DATA_REG, 17);
    
    // 보정 계수 적용 및 값 계산
    return {
        temperature: compensateTemp(rawData),
        humidity: compensateHumidity(rawData),
        pressure: compensatePressure(rawData),
        gasResistance: calculateGasResistance(rawData),
        iaq: calculateIAQ(rawData),
        co2Equivalent: calculateCO2(rawData),
        breathVoc: calculateVOC(rawData)
    };
}

📊 대기질 지수 (AQI)

대기질 지수(Air Quality Index, AQI)는 대기 오염 수준을 이해하기 쉬운 숫자로 표현합니다. 각 국가마다 기준이 다르며, 주로 PM2.5, PM10, O3, NO2, SO2, CO 등의 농도를 기반으로 계산합니다.

AQI 수준별 의미

AQI 수준 건강 영향 권장 조치
0-50 좋음 공기질이 만족스럽고 건강 위험이 낮음 정상적인 야외 활동
51-100 보통 민감한 그룹에게 약간의 영향 민감군 장시간 야외활동 자제
101-150 민감군 나쁨 민감한 그룹에게 건강 영향 민감군 야외활동 제한
151-200 나쁨 일반인도 건강 영향 시작 야외 활동 제한, 마스크 착용
201-300 매우 나쁨 모든 사람에게 건강 영향 야외 활동 자제
301+ 위험 건강 경보, 심각한 영향 실외 활동 금지
