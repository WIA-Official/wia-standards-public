MEMS 센서 기술

마이크로-전자-기계 시스템(MEMS)의 원리, 가속도계, 자이로스코프, 압력 센서 및 마이크로 제조 기술을 탐구합니다.

⏱️ 읽기 시간 35분 📊 중급 수준 📅 2025년 1월 업데이트

🔬 MEMS란 무엇인가?

MEMS(Micro-Electro-Mechanical Systems)는 마이크로미터 규모(1~100 마이크로미터)의 기계 및 전자 구성요소를 통합하는 소형화 기술입니다. 이 기술은 반도체 제조 공정을 사용하여 복잡한 3차원 구조물을 만들어 센서, 액추에이터 및 기타 장치를 단일 칩에 통합할 수 있게 합니다.

MEMS 기술은 현대 전자기기의 핵심 구성요소가 되었습니다. 스마트폰의 가속도계와 자이로스코프부터 자동차 에어백 센서, 의료 장비, 산업 자동화 시스템까지, MEMS는 우리 일상생활 곳곳에서 사용되고 있습니다.

💡 MEMS 시장 현황

글로벌 MEMS 시장 규모는 2028년까지 약 260억 달러에 달할 것으로 예상됩니다. 주요 성장 동력으로는 IoT 장치의 확산, 자율주행차 개발, 웨어러블 기기의 대중화가 있습니다.

📐 MEMS 가속도계

MEMS 가속도계는 가속도를 측정하는 센서로, 물체의 움직임, 진동, 충격을 감지합니다. 스마트폰 화면 회전, 차량 충돌 감지, 운동 추적 등 다양한 응용 분야에서 사용됩니다.

작동 원리

대부분의 MEMS 가속도계는 정전용량 원리를 기반으로 작동합니다. 스프링에 연결된 프루프 매스(proof mass)가 가속도에 의해 변위되면, 고정 전극과 이동 전극 사이의 정전용량이 변화합니다. 이 변화를 측정하여 가속도를 계산합니다.

📱
소비자 전자기기

스마트폰 화면 회전, 만보계, 게임 컨트롤러, 태블릿 방향 감지에 사용됩니다.

🚗
자동차 안전

에어백 전개, 전자 안정성 제어(ESC), 충돌 감지 시스템에 활용됩니다.

🏭
산업용 모니터링

기계 진동 분석, 예측 유지보수, 구조물 건전성 모니터링에 사용됩니다.

🩺
의료 기기

활동량 모니터링, 낙상 감지, 수면 추적, 재활 치료 지원에 활용됩니다.

// MEMS 가속도계 데이터 읽기 예제 (I2C 통신)
interface AccelerometerData {
    x: number;  // X축 가속도 (g)
    y: number;  // Y축 가속도 (g)
    z: number;  // Z축 가속도 (g)
    timestamp: number;
}

async function readAccelerometer(): Promise<AccelerometerData> {
    const rawX = await i2c.readWord(ACCEL_ADDR, REG_X);
    const rawY = await i2c.readWord(ACCEL_ADDR, REG_Y);
    const rawZ = await i2c.readWord(ACCEL_ADDR, REG_Z);
    
    // 16비트 부호 있는 정수를 g 단위로 변환
    const scale = 2.0 / 32768.0;  // ±2g 범위
    
    return {
        x: rawX * scale,
        y: rawY * scale,
        z: rawZ * scale,
        timestamp: Date.now()
    };
}

🌀 MEMS 자이로스코프

MEMS 자이로스코프는 각속도를 측정하는 센서입니다. 물체의 회전 속도와 방향을 감지하여 내비게이션, 이미지 안정화, 게임 컨트롤 등에 사용됩니다. 가속도계와 결합하여 IMU(Inertial Measurement Unit)를 구성합니다.

작동 원리: 코리올리 효과

MEMS 자이로스코프는 코리올리 효과를 이용합니다. 진동하는 질량체가 회전하면, 코리올리 힘에 의해 진동 방향과 수직인 방향으로 변위가 발생합니다. 이 변위를 정전용량 변화로 측정하여 각속도를 계산합니다.

📐 코리올리 힘 공식

Fc = -2m(Ω × v)

여기서 m은 질량, Ω는 각속도, v는 진동 속도입니다. 이 힘이 감지 방향으로 변위를 유발하여 회전을 측정합니다.

🔘 MEMS 압력 센서

MEMS 압력 센서는 가스나 액체의 압력을 측정하는 센서입니다. 다이어프램 구조를 사용하며, 압력에 의한 다이어프램의 변형을 감지합니다. 절대 압력, 게이지 압력, 차압 측정이 가능합니다.

센싱 원리

⚡
압저항형

다이어프램 변형에 따른 저항 변화 측정. 높은 선형성과 감도. 자동차 및 산업용에 주로 사용.

🔋

💎
압전형

압력에 의해 전압 생성. 동적 압력 측정에 적합. 음향 및 진동 측정에 사용.

🎤 MEMS 마이크로폰

MEMS 마이크로폰은 음향 에너지를 전기 신호로 변환하는 소형 마이크로폰입니다. 스마트폰, 스마트 스피커, 보청기, 차량 핸즈프리 시스템 등에 널리 사용됩니다. 높은 신호 대 잡음비(SNR)와 소형 크기가 특징입니다.

MEMS 마이크로폰의 장점

  • 소형화: 1mm x 1mm 이하의 크기로 제작 가능
  • 고SNR: 65-70 dB 이상의 신호 대 잡음비
  • 일관된 성능: 대량 생산 시 균일한 특성
  • 리플로우 솔더링 가능: SMT 공정과 호환
  • 디지털 출력: PDM 또는 I2S 인터페이스 지원
  • 저전력: 배터리 구동 기기에 적합
🎙️ 음성 인식 최적화

현대 MEMS 마이크로폰은 음성 인식 AI 알고리즘에 최적화되어 있습니다. 다중 마이크로폰 배열을 통한 빔포밍, 노이즈 캔슬링, 방향 탐지 기능이 가능합니다.

🏭 MEMS 제조 공정

MEMS 장치는 반도체 제조 기술을 기반으로 제작됩니다. 표면 미세가공(Surface Micromachining)과 벌크 미세가공(Bulk Micromachining)이 두 가지 주요 방식입니다.

주요 제조 단계

1️⃣
웨이퍼 준비

고순도 실리콘 웨이퍼 산화, 세척 및 표면 처리

2️⃣
박막 증착

CVD, PVD, 스퍼터링을 통한 기능성 박막 형성

3️⃣
포토리소그래피

감광제 도포, UV 노출, 패턴 현상

4️⃣
에칭

습식/건식 에칭으로 3D 구조물 형성

5️⃣
희생층 제거

가동 부품을 해제하기 위한 선택적 에칭

6️⃣
패키징

MEMS 구조물 보호 및 전기 연결

Deep Reactive Ion Etching (DRIE)

DRIE는 고종횡비(high aspect ratio) 구조물을 만드는 핵심 공정입니다. Bosch 공정이라고도 하며, 에칭과 패시베이션을 교대로 수행하여 깊고 수직인 측벽을 형성합니다.

// MEMS 제조 공정 파라미터 예시
const drieParameters = {
    etchCycle: {
        gas: 'SF6',           // 에칭 가스
        power: 2500,          // RF 파워 (W)
        pressure: 35,        // mTorr
        time: 7              // 초
    },
    passivationCycle: {
        gas: 'C4F8',          // 패시베이션 가스
        power: 1500,          // RF 파워 (W)
        pressure: 25,        // mTorr
        time: 3              // 초
    },
    etchRate: 3.5,           // μm/분
    aspectRatio: 30          // 깊이/폭 비율
};

📚 장 요약

이 장에서는 MEMS 센서 기술의 핵심 개념과 응용을 다루었습니다:

  • MEMS 기본: 마이크로 규모 기계 및 전자 구성요소 통합 기술
  • 가속도계: 정전용량 원리 기반 가속도 측정, 스마트폰부터 에어백까지
  • 자이로스코프: 코리올리 효과를 이용한 각속도 측정
  • 압력 센서: 다이어프램 변형 감지, 다양한 산업 응용
  • 마이크로폰: 음향-전기 변환, 음성 인식 시스템의 핵심
  • 제조 공정: 표면/벌크 미세가공, DRIE 기술
