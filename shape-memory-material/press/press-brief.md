# WIA-MAT-SM-001 — Shape Memory Material (Press Brief)

**WIA Standards releases canonical Shape Memory Material envelope shape**

The WIA Standards Committee has published the WIA-MAT-SM-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for shape memory material hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
