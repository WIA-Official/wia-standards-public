## 2.1 단백질 운동의 물리학

### 동적 시스템으로서의 단백질

단백질은 생리학적 온도에서 끊임없이 운동합니다. 결정 내에서도 원자들은 0.5-2 Å 진폭으로 진동합니다. 이 운동은 무작위 노이즈가 아닙니다—기능적으로 필수적입니다.

### 에너지 지형 관점

단백질의 행동은 자유 에너지 지형을 통해 이해할 수 있습니다—각 점이 가능한 형태를 나타내는 다차원 표면:

```
자유 에너지 (G)
    ↑
    │     ╱╲
    │    ╱  ╲    ╱╲
    │   ╱    ╲  ╱  ╲
    │  ╱      ╲╱    ╲
    │ ╱                ╲
    └──────────────────────→
         형태 좌표

    최소값 = 안정 상태
    장벽 = 전이 상태
    깊이 = 집단 (Boltzmann)
```

Boltzmann 분포가 집단을 지배합니다:

```
P(상태_i) = exp(-G_i / kT) / Σ exp(-G_j / kT)
```

여기서:
- G_i = 상태 i의 자유 에너지
- k = Boltzmann 상수
- T = 온도

### 운동의 시간 척도

단백질 운동은 놀라운 범위의 시간 척도를 포괄합니다:

| 시간 척도 | 운동 유형 | 예시 | 방법 |
|----------|----------|------|------|
| fs (10⁻¹⁵ s) | 결합 진동 | C-C 신축, N-H 굽힘 | IR 분광법 |
| ps (10⁻¹² s) | 측쇄 회전 | 메틸 회전, 고리 뒤집힘 | NMR T1, MD |
| ns (10⁻⁹ s) | 루프 운동 | 유연한 루프, 말단 | NMR 이완, MD |
| μs (10⁻⁶ s) | 도메인 운동 | 힌지 굽힘, 비틀림 | FRET, μs-MD |
| ms (10⁻³ s) | 형태 교환 | 접힘, 알로스테리 | NMR CPMG, HDX |
| s | 대규모 재배열 | 접힘, 조립 | 정지 흐름, 동역학 |

## 2.2 분자 동역학 시뮬레이션

### 기본 접근법

분자 동역학(MD) 시뮬레이션은 모든 원자에 대해 뉴턴의 운동 방정식을 수치적으로 풉니다:

```
F_i = m_i × a_i = m_i × (d²r_i/dt²)
```

각 원자에 대한 힘은 잠재 에너지 함수(역장)에서 옵니다:

```
V = V_결합 + V_각도 + V_이면각 + V_비결합
```

### 역장 구성 요소

**1. 결합 항**

결합 신축 (조화 근사):
```
V_bond = Σ (k_b/2)(r - r_0)²
```

각도 굽힘:
```
V_angle = Σ (k_θ/2)(θ - θ_0)²
```

이면각 회전:
```
V_dihedral = Σ k_φ[1 + cos(nφ - δ)]
```

**2. 비결합 항**

반 데르 발스 (Lennard-Jones):
```
V_LJ = Σ 4ε[(σ/r)¹² - (σ/r)⁶]
```

정전기 (Coulomb):
```
V_elec = Σ q_i × q_j / (4πε₀r_ij)
```

### 현대 역장

| 역장 | 강점 | 응용 |
|------|------|------|
| AMBER ff19SB | 단백질 백본 | 일반 단백질 |
| CHARMM36m | 막 단백질 | GPCR, 채널 |
| OPLS-AA/M | 약물 유사 분자 | 약물 결합 |
| Martini 3 | 조립곡선 | 대규모 시스템 |
| ANI-2x | 신경망 | 높은 정확도 |

## 2.3 유연성 메트릭

### B-인자와 RMSF

**B-인자 (온도 인자)**
결정학적 B-인자는 원자 변위를 측정합니다:

```
B = (8π²/3) × <u²>
```

여기서 <u²>는 평균 제곱 변위입니다.

**제곱 평균 제곱근 요동 (RMSF)**
MD 궤적에서:

```
RMSF_i = √(<(r_i - <r_i>)²>)
```

두 메트릭 모두 유연한 영역을 식별합니다:

```json
{
  "flexibility_profile": {
    "residues": [1, 2, 3, ...],
    "rmsf_values": [0.8, 0.7, 0.6, 1.2, 2.5, ...],
    "flexible_regions": [
      {"start": 45, "end": 52, "type": "루프", "mean_rmsf": 2.3},
      {"start": 120, "end": 128, "type": "말단", "mean_rmsf": 3.1}
    ]
  }
}
```

### 순서 매개변수

NMR의 경우, Lipari-Szabo 순서 매개변수 S²는 운동을 측정합니다:

```
S² = 0: 완전히 유연함 (등방성 운동)
S² = 1: 완전히 강직함 (운동 없음)
```

전형적인 값:
- 구조화된 영역: S² > 0.85
- 루프: S² = 0.6-0.8
- 무질서: S² < 0.5

## 2.4 주성분 분석

### 차원 축소

단백질은 3N 자유도(N = 원자 수)를 가지지만, 대부분의 운동은 몇 가지 집합적 좌표를 따라 발생합니다. PCA는 이러한 본질적 동역학을 식별합니다.

### 수학

1. 궤적에서 공분산 행렬 구축:
```
C_ij = <(r_i - <r_i>)(r_j - <r_j>)>
```

2. 대각화:
```
C × v_k = λ_k × v_k
```

3. 고유값 (λ)은 설명된 분산을 제공
4. 고유벡터 (v)는 집합적 운동을 정의

### 해석

```
전형적인 설명된 분산:
  PC1: 30-50%  - 가장 큰 형태 변화
  PC2: 15-25%  - 두 번째 모드
  PC3: 8-15%   - 세 번째 모드
  ...
  PC1-3: 60-80% - 보통 주요 운동에 충분함
```

## 2.5 정규 모드 분석

### 조화 근사

에너지 최소값 근처에서 잠재력은 대략적으로 조화적입니다:

```
V ≈ V₀ + (1/2) × Σ H_ij × Δx_i × Δx_j
```

여기서 H는 2차 미분의 Hessian 행렬입니다.

### 저주파 모드

가장 낮은 주파수의 비사소한 모드(병진/회전 제거 후 모드 7-12)는 종종 기능적 운동을 포착합니다:

| 모드 | 주파수 (cm⁻¹) | 운동 유형 |
|------|--------------|----------|
| 7 | 5-20 | 전체 굽힘/비틀림 |
| 8 | 10-30 | 도메인 운동 |
| 9-12 | 20-50 | 서브도메인 운동 |

## 2.6 교차 상관 분석

### 상관된 운동

기능적으로 중요한 잔기 쌍은 종종 상관된 방식으로 움직입니다. 동적 교차 상관은 이를 측정합니다:

```
C_ij = <Δr_i · Δr_j> / (√<Δr_i²> × √<Δr_j²>)
```

여기서:
- C_ij = +1: 완벽한 양의 상관 (같은 방향)
- C_ij = -1: 완벽한 음의 상관 (반대 방향)
- C_ij = 0: 비상관

### 커뮤니티 감지

상관 행렬에서 동적으로 결합된 커뮤니티를 식별할 수 있습니다.

## 2.7 알로스테릭 통신

### 장거리 결합

알로스테리—한 부위에서의 결합이 다른 부위에서의 활성에 영향을 미치는 조절—는 동역학에 의존합니다. 통신 경로가 단백질을 가로질러 형태적 신호를 전달합니다.

### 알로스테리 정량화

**상호 정보**
잔기 운동 간 공유된 정보를 측정합니다:

```
MI(i,j) = Σ P(x_i, x_j) × log[P(x_i, x_j) / (P(x_i) × P(x_j))]
```

**전달 엔트로피**
방향성 정보 흐름을 측정합니다:

```
TE(i→j) = i에서 j로 전달된 정보
```

## 2.8 실험적 검증

### NMR 방법

**이완**
- T1: 스핀-격자 이완 (ps-ns 운동)
- T2: 스핀-스핀 이완 (μs-ms 운동)
- NOE: 핵 Overhauser 효과 (공간적 근접 + 동역학)

**화학 교환**
- CPMG 이완 분산: μs-ms 교환
- R1ρ 실험: 10 μs - 10 ms 운동
- ZZ-교환: 느린 (ms-s) 교환

### 크라이오-EM 앙상블 분석

현대 크라이오-EM은 형태적 이질성을 해결할 수 있습니다:

1. **3D 분류**: 이산 상태 분리
2. **3D 변동성 분석**: 연속적 형태 지형
3. **다중체 정제**: 상대 운동이 있는 강체로 분해

## 2.9 본질적 무질서 단백질

### 프로테옴의 나머지 절반

진핵 단백질의 30-50%가 상당한 무질서 영역을 포함합니다. 이러한 IDP/IDR은:

- 안정한 3차 구조가 없음
- 확장된 형태 앙상블을 샘플링
- 종종 결합 시 접힘 (결합된 접힘-결합)
- 신호 전달 및 조절 단백질에 풍부

### 무질서 특성화

**앙상블 설명**
IDP는 앙상블 표현을 필요로 합니다.

**주요 메트릭**

| 메트릭 | 정렬된 단백질 | IDP |
|--------|-------------|-----|
| Rg | 컴팩트 | 확장 |
| RMSF | 0.5-2 Å | 5-20 Å |
| S² | > 0.8 | < 0.4 |
| 2차 구조 | 안정 | 일시적 |

## 핵심 요점

1. 단백질은 동적이며, 형태 앙상블을 샘플링함
2. 운동은 펨토초에서 초까지 범위
3. MD 시뮬레이션은 운동 방정식을 수치적으로 해결

5. 교차 상관은 결합된 운동을 드러냄

7. IDP는 앙상블 설명을 필요로 함
