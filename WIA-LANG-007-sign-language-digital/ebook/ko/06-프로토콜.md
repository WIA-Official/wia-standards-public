```javascript
// WebRTC 연결 설정
const peerConnection = new RTCPeerConnection({
  iceServers: [
    {urls: 'stun:stun.wia-lang-007.org:3478'},
    {
      urls: 'turn:turn.wia-lang-007.org:3478',
      username: 'user',
      credential: 'pass'
    }
  ]
});

// 로컬 비디오 캡처 (수어용 고품질)
const stream = await navigator.mediaDevices.getUserMedia({
  video: {
    width: {ideal: 1920},
    height: {ideal: 1080},
    frameRate: {ideal: 60},
    facingMode: 'user'
  },
  audio: false  // 수어 통화는 비디오만
});

// 스트림 추가
stream.getTracks().forEach(track => {
  peerConnection.addTrack(track, stream);
});

// 비디오 품질 모니터링
setInterval(async () => {
  const stats = await peerConnection.getStats();
  stats.forEach(stat => {
    if (stat.type === 'inbound-rtp' && stat.kind === 'video') {
      console.log('프레임률:', stat.framesPerSecond);
      console.log('패킷 손실:', stat.packetsLost);
      console.log('지터:', stat.jitter);
    }
  });
}, 1000);
```

### 철학: 弘益人間 (널리 인간을 이롭게 하라)

통신 프로토콜은 수어 기술의 혈관입니다. 弘益人間 원칙으로:

**신뢰성:** 안정적인 프로토콜로 중요한 의사소통 보장.

**보안:** 강력한 암호화로 개인정보 및 수어 데이터 보호.

**효율성:** 최적화된 프로토콜로 모든 네트워크에서 작동.

**포용성:** 낮은 대역폭에서도 기본 수어 통신 가능.

프로토콜은 보이지 않지만 수어 사용자를 연결하는 핵심입니다.
