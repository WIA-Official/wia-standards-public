제8장: 규정 준수와 인증

8.1 국내 의료기기 허가 절차 (식품의약품안전처)

한국에서 보조 로봇은 의료기기법(2020 전부 개정) 및 식품의약품안전처(MFDS) 규정에 따라 허가를 받아야 합니다.

로봇 유형의료기기 등급허가 절차근거 분류
동력 외골격 (보행 보조)2등급기술문서심사 + 임상시험의료기기법 §6, 고시 2023-4호
재활 로봇2등급기술문서심사의료기기 분류번호 A17060.01
수술 보조 로봇3등급품목허가 + PMS 5년의료기기법 §6②, 신의료기술평가
돌봄 로봇 (비의료)해당 없음KC 인증 (전기용품안전법)KS C IEC 62368-1

외골격의 경우 임상시험계획(IND) 제출 전 식약처 의료기기사전검토제도를 활용하면 허가 기간을 단축할 수 있습니다. 큐렉소 CUREXO 외골격은 2021년 최초로 국내 2등급 의료기기로 허가를 획득했습니다.

8.2 KS B ISO 13482:2019 — 개인용 돌봄 로봇

국가기술표준원(KATS)이 ISO 13482:2014를 한국 표준으로 채택한 KS B ISO 13482:2019는 개인용 돌봄 로봇의 안전 요구 사항을 규정합니다.

  • ISO 12100(위험성 평가)에 따른 위험성 추정 및 평가 의무.
  • 안정성: 3° 경사면(Type B 이동 보조)에서 전도 금지; 6° 경사(Type A 신체 지원).
  • 협착 방지: 4 mm–25 mm 사이의 신체 포획 틈새 금지.
  • 하중 용량: 외골격은 정격 체중의 125%를 1분간 지지할 수 있어야 함.

KIRO(한국로봇융합연구원)는 KS B ISO 13482 공인 시험기관으로 국내 인증 시험을 수행합니다.

8.3 ISO 10218 — 산업용 로봇 안전

WIA-ROBOT이 준수하는 ISO 10218-1:2011 핵심 요구 사항:

  • §5.4.2 — 비상 정지: Category 0 또는 Category 1 정지; WIA 목표: E-Stop 신호 전파 10 ms 이내.
  • §5.6 — 속도 및 힘 제한: 협동 운용 시 TCP 속도 250 mm/s 이하, ISO 15066 접촉력 준수.
  • §5.7 — 제한 장치: 소프트웨어 제어와 독립적인 하드웨어 과이동 보호.

8.4 IEC 61508 — 기능 안전 (SIL)

로봇 유형목표 SIL허용 위험 고장률 (PFH)
외골격 / 재활 로봇SIL 210⁻⁷ ~ 10⁻⁶ /h
수술 로봇SIL 2–310⁻⁸ ~ 10⁻⁷ /h
돌봄 로봇 / 이동 보조SIL 110⁻⁶ ~ 10⁻⁵ /h

SIL 2 준수는 소프트웨어 개발 프로세스에 체계적 능력(SC) 3을 요구합니다. WIA-ROBOT 소프트웨어는 안전 임계 경로에 ROS 2 REP-2004 품질 레벨 1 패키지만 사용합니다.

8.5 건강보험 급여 신청 절차

외골격이나 재활 로봇의 건강보험 급여 등재를 위해서는 건강보험심사평가원(HIRA) 신의료기술평가(HTA)를 통과해야 합니다:

  1. 신의료기술평가위원회 신청 → 의료기술평가 (6–12개월)
  2. 국민건강보험 요양급여 적용 기준 고시
  3. 요양급여 비용 협상 (건보공단 ↔ 제조사/수입업체)
  4. EDI(전자문서교환) 청구 코드 발급 → 급여 청구 시작

큐렉소 CUREXO 외골격은 2022년 선별급여 50% 적용을 획득하여 환자 본인부담을 절반으로 줄였습니다.

8.6 WIA-ROBOT 자체 인증 점검 목록

# WIA-ROBOT 규정 준수 자체 점검 실행
ros2 run wia_robot_compliance check_node \
  --ros-args -p report_output:=/tmp/wia_compliance_report.json

# 수행 점검 항목:
# [1] 비상 정지 응답 지연 (목표: 10 ms 미만)
# [2] 관절 한계 적용 (하드웨어 + 소프트웨어)
# [3] 힘/토크 센서 교정 일자 (90일 이내)
# [4] URDF/SRDF 스키마 유효성 검사
# [5] JWT 토큰 필수 클레임 검증
# [6] QoS 프로파일 준수 (명령 RELIABLE, 텔레메트리 BEST_EFFORT)
# [7] 안전 임계 패키지 ROS 2 REP-2004 품질 레벨 1 확인
# [8] 워치독 하트비트 주기 (100 ms 미만)
# [9] IK 솔버 성공률 (표준 테스트 포즈 세트 95% 이상)
# [10] MoveIt 2 계획 장면에 충돌 객체 채워져 있는지 확인

8.7 로봇 산업 진흥법과 지원 정책

산업통상자원부의 로봇산업진흥법(2023 전부 개정) 주요 내용:

  • 로봇 안전 인증 표준 체계 수립 (국가기술표준원 고시)
  • 서비스 로봇 실증 특례 제도 — 의료·요양 분야 규제 샌드박스
  • 로봇산업클러스터 지정 (경기 시흥, 경남 창원, 전북 군산)
  • 제4차 지능형 로봇 기본계획(2024–2028): 보조 로봇 1만 대 보급 목표
