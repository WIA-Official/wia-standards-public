제1장: 한국 보조 로봇 표준 개요

1.1 한국 로봇 산업의 현황

한국은 산업용 로봇 밀도 세계 1위 국가(IFR 2024 기준: 10만 명당 1,012대)로, 보조 로봇 분야에서도 빠른 성장을 이루고 있습니다. 현대로보틱스(구 현대중공업 로봇사업부), KITECH(한국산업기술연구원), KIRO(한국로봇융합연구원), 로보스타, 큐렉소 등이 재활·의료 보조 로봇 생태계를 이끌고 있습니다.

정책적으로는 산업통상자원부의 로봇산업진흥법(2023 전부 개정)과 제4차 지능형 로봇 기본계획(2024–2028)이 보조 로봇의 인증·보험·의료기기 승인 절차를 규정합니다.

1.3 ROS 2 국내 도입 현황

ROS 2는 국내 로봇 연구기관(KAIST, POSTECH, 한양대 로봇공학과)과 기업(카카오모빌리티 자율주행, 네이버랩스 AMBIDEX)에서 표준 미들웨어로 채택되고 있습니다. 2025년 기준 국내 WIA-ROBOT 구현체는 ROS 2 Humble Hawksbill (LTS, 2027-05 EOL) 또는 Jazzy Jalisco (LTS, 2029-05 EOL)를 사용합니다. 

# ROS 2 Jazzy 설치 (Ubuntu 24.04 Noble)
sudo apt install ros-jazzy-desktop ros-jazzy-moveit \
     ros-jazzy-nav2-bringup ros-jazzy-cartographer-ros
source /opt/ros/jazzy/setup.bash
echo "source /opt/ros/jazzy/setup.bash" >> ~/.bashrc

1.4 DDS/RTPS 미들웨어

WIA-ROBOT의 DDS 통신 레이어는 안전 임계 명령과 실시간 센서 스트림에 각각 다른 QoS 프로파일을 적용합니다:

  • RELIABLE + TRANSIENT_LOCAL — 비상 정지(/emergency_stop), 궤적 명령, 로봇 설명 토픽. 패킷 손실 재전송 보장, 지연 참여자에게도 최근 메시지 전달.
  • BEST_EFFORT + VOLATILE — 관절 상태(/joint_states, 50 Hz), IMU, 힘/토크(1 kHz). 저지연 우선, 오래된 프레임 폐기.
  • Deadline 100 ms — ISO 10218-1 §5.4.2 비상 정지 응답 예산 준수.

1.6 이 교재의 구성

본 한국어판 교재는 WIA 4단계 아키텍처를 기반으로 국내 맥락을 중심으로 구성됩니다:

  • 2장 — URDF/XACRO 로봇 모델, 관절 상태 스키마, FHIR R5 임상 데이터 연동
  • 3장 — ROS 2 게이트웨이 API, WebSocket 텔레메트리, JWT 인증
  • 4장 — DDS QoS 프로파일, MQTT 5 병원 연동, FastDDS 설정
  • 5장 — MoveIt 2 동작 계획 (OMPL, STOMP, PILZ), IK 솔버
  • 6장 — SLAM (Cartographer, ORB-SLAM3), ROS 2 Nav2 병원 내비게이션
  • 7장 — Isaac Sim 연동, 합성 훈련 데이터 생성, 시뮬레이션-실물 이전
  • 8장 — 식약처 허가, KS B ISO 13482, IEC 61508 SIL-2 인증 절차
