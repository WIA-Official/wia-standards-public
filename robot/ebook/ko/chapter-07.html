제7장: NVIDIA Isaac Sim 연동

7.1 Isaac Sim 개요

NVIDIA Isaac Sim(Omniverse Kit 및 USD Composer 기반)은 WIA-ROBOT 개발·테스트·합성 데이터 생성을 위한 물리적으로 정확한 로봇 시뮬레이션 환경을 제공합니다. Isaac Sim은 isaacsim.ros2_bridge 확장을 통해 ROS 2와 네이티브로 통합되어 실제 로봇과 동일한 토픽·서비스·액션을 노출하므로 시뮬레이션-실물 이전이 원활합니다.

국내 활용 사례: 네이버랩스 AMBIDEX 로봇팔 IsaacSim 검증, KAIST 공대 로봇설계연구실(2024), ETRI 지능로봇연구실 합성 데이터 생성 파이프라인.

7.2 ROS 2 브리지 설정

# Isaac Sim Python 스크립팅 (Omniverse Kit)
from isaacsim.ros2_bridge import ROS2Bridge

bridge = ROS2Bridge()
bridge.start(
    ros_domain_id=200,            # Isaac Sim 가상 로봇 도메인
    robot_namespace="/wia_robot_sim",
    publish_joint_states=True,
    joint_state_hz=50,
    publish_clock=True            # use_sim_time=true용 /clock 토픽
)
print("ROS 2 브리지 도메인 200에서 활성화")

7.3 URDF 임포트 및 아티큘레이션

WIA 외골격 URDF를 Isaac Sim으로 임포트합니다: 

from isaacsim.robot.import_utils import URDFImporter

importer = URDFImporter()
result = importer.import_robot(
    urdf_path="/opt/wia_robot/urdf/wia_exoskeleton.urdf.xacro",
    dest_path="/World/WIAExoskeleton",
    import_inertia_tensor=True,
    import_materials=True,
    convex_decomp=True,           # 볼록 껍질 충돌 형상
    fix_base=False,               # 외골격은 이동 가능
    merge_fixed_joints=False
)
# /World/WIAExoskeleton에 ArticulationRoot USD prim 생성

7.4 합성 훈련 데이터 생성

WIA-ROBOT은 Isaac Sim의 도메인 무작위화 파이프라인(omni.replicator)으로 보행 분석 및 장애물 감지 모델 훈련을 위한 합성 RGB-D 데이터셋을 생성합니다. ETRI AI 연구실은 이 방법론으로 실제 데이터 수집 대비 87% 비용 절감을 달성했습니다. 

import omni.replicator.core as rep

with rep.new_layer():
    robot = rep.get.prims(path_pattern="/World/WIAExoskeleton")
    lights = rep.create.light(light_type="Sphere", count=3)

    with rep.trigger.on_frame(num_frames=5000):
        rep.modify.pose(robot,
                        position=rep.distribution.uniform((-2,0,-2),(2,0,2)),
                        rotation=rep.distribution.uniform((0,-10,0),(0,10,360)))
        rep.modify.attribute(lights, "intensity",
                             rep.distribution.uniform(500, 3000))

rep.orchestrator.run()
# 출력: 관절 각도·깊이 이미지 정답 레이블이 포함된 5000 프레임

7.5 시뮬레이션-실물 이전 워크플로우

단계작업검증 기준
1. 시뮬레이션 검증Isaac Sim에서 10,000 스텝 제어기 실행비상 정지 0회; 관절 한계 준수
2. 하드웨어-루프 테스트Isaac Sim이 시뮬레이션 관절 구동; 실시간 제어기는 실제 하드웨어에서 실행지연 20 ms 미만; 발산 없음
3. 저속 실물 테스트속도 스케일링 10%로 실제 로봇에 배포궤적 추적 오차 0.02 rad 미만
4. 전속 임상 시험물리치료사 감독하 임상 테스트ISO 13482 인체 안전 평가

7.6 병원 환경 USD 씬

WIA는 이동 보조 로봇 내비게이션 테스트용 병원 병동 USD 씬을 제공합니다. 씬에는 서울 대형 병원(4인실 기준) 레이아웃이 반영되어 있으며, 침대·링거대·의료 장비 충돌 형상이 포함됩니다. 

# WIA 병원 병동 USD 씬 로드
stage.GetRootLayer().subLayerPaths.append(
    "omniverse://hospital-assets.wia.global/WIA/HospitalWard_v2.usd"
)

# 3D 씬에서 Nav2 호환 2D 점유 격자 지도 생성
from isaacsim.nav2_utils import OccupancyMapGenerator
gen = OccupancyMapGenerator(resolution=0.05)
gen.generate_from_scene(
    output_yaml="/opt/wia_robot/maps/isaac_ward.yaml",
    output_pgm="/opt/wia_robot/maps/isaac_ward.pgm",
    height=1.0,            # 바닥에서 1 m 높이 슬라이스
    free_thresh=0.25,
    occupied_thresh=0.65
)
