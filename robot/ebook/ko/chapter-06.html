제6장: SLAM과 자율 내비게이션

6.1 보조 로봇에서의 SLAM

동시 위치 추정 및 지도 작성(SLAM)은 이동 보조 로봇(스마트 휠체어, 돌봄 로봇)이 환경 지도를 구축하면서 자신의 위치를 추적할 수 있게 합니다. 한국의 병원·요양원 환경은 좁은 복도, 자동문, 환자 침대 이동 등 동적 장애물이 많아 강건한 SLAM 구성이 필요합니다.

시스템센서알고리즘ROS 2 패키지
Cartographer2D LiDAR (SICK TiM, RPLiDAR A3)서브맵 기반 그래프 SLAM (Google)cartographer_ros
ORB-SLAM3RGB-D 카메라 / 스테레오 / IMUORB 특징 시각-관성 SLAMorb_slam3_ros2_wrapper
RTAB-MapRGB-D + LiDAR 융합루프 클로저 그래프 기반rtabmap_ros

6.2 Cartographer 2D SLAM

Google Cartographer는 2D LiDAR를 사용하는 실내 이동 보조 로봇의 권장 SLAM 시스템입니다. RELIABLE + TRANSIENT_LOCAL QoS로 /map 토픽에 nav_msgs/OccupancyGrid를 발행합니다. 서울아산병원 재활로봇센터에서는 SICK TiM571과 Cartographer를 조합하여 병동 내비게이션 지도 구축에 활용하고 있습니다. 

ros2 launch cartographer_ros cartographer.launch.py \
  configuration_directory:=/opt/wia_robot/cartographer \
  configuration_basename:=wia_ward.lua

# wia_ward.lua 핵심 설정
options = {
  map_frame = "map",
  tracking_frame = "base_link",
  provide_odom_frame = true,
  use_odometry = true,
  num_laser_scans = 1,
  rangefinder_sampling_ratio = 1.0,
}

6.3 ORB-SLAM3 시각-관성 SLAM

ORB-SLAM3는 LiDAR가 비실용적인 외골격 및 재활 환경에 적합합니다. Intel RealSense D435i(RGB-D + IMU)와 결합한 시각-관성 SLAM 구성은 단안 시각만 사용할 때 발생하는 스케일 모호성을 IMU로 해소하므로, 실내 복도와 같은 구조화된 환경에서 cm 단위 위치 정확도를 달성합니다. 

ros2 run orb_slam3_ros2_wrapper rgbd_inertial_node \
  --ros-args \
  -p vocabulary_file:=/opt/ORB_SLAM3/Vocabulary/ORBvoc.txt \
  -p settings_file:=/opt/ORB_SLAM3/Examples/RGB-D-Inertial/RealSenseD435i.yaml \
  -p enable_pangolin:=false

6.4 ROS 2 Nav2 내비게이션 스택

Nav2는 전역 경로 계획, 지역 장애물 회피, 복구 행동을 포함하는 완전한 자율 내비게이션 파이프라인을 제공합니다. WIA-ROBOT 이동 보조 로봇은 병원 환경 설정으로 Nav2를 사용합니다. 

ros2 launch nav2_bringup navigation_launch.py \
  use_sim_time:=false \
  map:=/opt/wia_robot/maps/ward_2.yaml \
  params_file:=/opt/wia_robot/nav2/wia_nav2_params.yaml

6.5 병원 환경 코스트맵 설정

병원 환경은 침대·휠체어 이용 환자를 위한 보수적인 장애물 팽창(더 넓은 여유 공간)과 이동하는 사람에 대한 동적 장애물 추적이 필요합니다: 

# wia_nav2_params.yaml
local_costmap:
  local_costmap:
    ros__parameters:
      update_frequency: 5.0
      global_frame: odom
      robot_base_frame: base_link
      rolling_window: true
      width: 6
      height: 6
      resolution: 0.05          # 5 cm/셀
      robot_radius: 0.40        # 휠체어 + 외골격 발자국
      plugins: ["obstacle_layer", "inflation_layer"]
      inflation_layer:
        cost_scaling_factor: 3.0
        inflation_radius: 0.75  # 장애물에서 75 cm 여유

6.6 AMCL 위치 추정 (지도 이후)

Cartographer로 지도를 구축한 후 정상 운용 시에는 AMCL(적응형 몬테 카를로 위치 추정)을 사용합니다. AMCL은 /amcl_pose에 geometry_msgs/PoseWithCovarianceStamped를 발행하며, WIA REST API의 GET /slam/pose로 조회 가능합니다.

6.7 지도 지속성 및 버전 관리

병동 배치가 변경될 때 지도를 버전 관리하고 백업해야 합니다. WIA-ROBOT은 nav2_map_server로 지도를 저장·불러옵니다: 

# 현재 SLAM 지도 저장
ros2 service call /map_server/save_map nav2_msgs/srv/SaveMap \
  '{map_topic: "/map",
    map_url: "/opt/wia_robot/maps/ward_2_20260425",
    image_format: "pgm",
    map_mode: "trinary",
    free_thresh: 0.25,
    occupied_thresh: 0.65}'

# 출력 파일:
# ward_2_20260425.pgm  -- 흑백 점유 이미지
# ward_2_20260425.yaml -- 지도 메타데이터 (해상도, 원점, 임계값)

한국 노인장기요양보험 적용 요양원에서 이동 보조 로봇을 운용하는 경우, 분기별 지도 검증 및 안전 재인증이 요구됩니다(보건복지부 장기요양 로봇 안전 가이드라인, 2025).
