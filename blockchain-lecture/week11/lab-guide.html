실습 1: NFT 메타데이터 작성 및 민팅

소요시간: 약 15분
나만의 NFT를 직접 만듭니다. 메타데이터(JSON) 작성부터 IPFS 업로드, 스마트 컨트랙트 민팅까지 전체 과정을 경험합니다.

Step 1: 콘텐츠 준비

시뮬레이터에서 제공하는 이미지를 선택하거나, 본인의 이미지를 업로드합니다.

Step 2: 메타데이터 JSON 작성

아래 템플릿을 참고하여 본인의 NFT 메타데이터를 작성합니다.

{ "name": "여러분의 NFT 이름", "description": "NFT에 대한 설명을 작성하세요", "image": "ipfs://업로드_후_받은_CID", "attributes": [ { "trait_type": "학과", "value": "여러분의 학과" }, { "trait_type": "학번", "value": "여러분의 학번" }, { "trait_type": "주제", "value": "자유 주제" }, { "trait_type": "희귀도", "value": "Legendary" } ] }

Step 3: IPFS 업로드

이미지와 메타데이터 JSON을 IPFS에 업로드합니다. CID(Content Identifier)를 기록하세요.

Step 4: 민팅 실행

ERC-721 컨트랙트의 mint 함수를 호출하여 NFT를 발행합니다.

Step 5: 블록 탐색기 확인

트랜잭션 해시와 토큰 ID를 블록 탐색기에서 확인합니다.

실습 2: 마켓플레이스 탐색 및 거래

소요시간: 약 15분
민팅한 NFT를 마켓플레이스에 등록하고, 동료 학생들과 거래를 진행합니다. 로열티 메커니즘도 확인합니다.

Step 6: 마켓플레이스에 NFT 리스팅

민팅한 NFT를 마켓플레이스에 등록합니다. 가격을 설정하세요.

Step 7: 로열티 설정

크리에이터 로열티를 5%로 설정합니다 (EIP-2981).

Step 8: 동료 NFT 탐색

마켓플레이스에서 다른 학생들의 NFT를 탐색하고, 마음에 드는 NFT에 오퍼를 제출합니다.

Step 9: 거래 성사

받은 오퍼를 수락하거나 다른 학생의 NFT를 구매합니다. 소유권 이전을 확인하세요.

Step 10: 2차 거래 및 로열티 확인

구매한 NFT를 다시 판매합니다. 원작자에게 로열티가 지급되는지 확인합니다.

실습 3: NFT 활용 사례 체험

소요시간: 약 10분
게임 아이템 NFT, 멤버십 NFT, 동적 NFT 등 다양한 활용 사례를 직접 체험합니다.

Step 11: ERC-1155 게임 아이템 민팅

동일한 게임 아이템 NFT를 10개 배치 민팅합니다. ERC-721과의 차이점을 확인하세요.

Step 12: 토큰 게이팅 체험

멤버십 NFT를 보유한 상태에서 VIP 콘텐츠에 접근합니다. NFT가 없으면 접근이 차단되는 것을 확인하세요.

Step 13: 동적 NFT 체험

조건(시간, 외부 데이터)에 따라 메타데이터가 변하는 동적 NFT를 확인합니다.

Step 14: 소울바운드 토큰(SBT) 개념 탐색

양도 불가능한 SBT의 개념을 이해하고, 신원 증명으로서의 활용을 탐색합니다.

Step 15: 실습 보고서 작성

오늘 실습 내용을 정리하고, NFT 활용 아이디어를 기록합니다.

실습에서 사용하는 모든 토큰과 NFT는 테스트넷용이므로 실제 경제적 가치는 없습니다. 안심하고 다양한 실험을 해보세요.

과제 안내

NFT가 활용될 수 있는 실제 비즈니스 사례 3가지를 제안하는 보고서를 작성하세요.

  • 아트/수집품 이외의 분야에서 찾을 것 (인증서, 티케팅, 공급망, 부동산 등)
  • 각 사례에 대해: 문제점 → NFT 해결방안 → 기대효과 구조로 작성
  • 기술적 실현 가능성과 한계점도 함께 논의할 것
  • A4 2페이지 이내
