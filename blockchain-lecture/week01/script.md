### 1-2. 블록체인이란? (10분)

**핵심 개념 설명:**

> "블록체인을 가장 쉽게 설명하면, **모두가 같은 장부를 가지고 있는 시스템**입니다."

**비유로 설명:**
```
[기존 시스템 - 은행]
철수 → 은행 → 영희
      (중개자)
  "은행이 기록을 관리"

[블록체인 시스템]
철수 ←→ 영희
  ↕         ↕
민수 ←→ 지영
  "모두가 기록을 관리"
```

**블록의 구조:**
```
┌─────────────────────────┐
│ Block #1                │
│ ┌─────────────────────┐ │
│ │ 이전 블록 해시      │ │
│ │ 타임스탬프          │ │
│ │ 거래 데이터         │ │
│ │ 현재 블록 해시      │ │
│ └─────────────────────┘ │
└────────────┬────────────┘
             │ (해시로 연결)
┌────────────▼────────────┐
│ Block #2                │
│ ┌─────────────────────┐ │
│ │ 이전 블록 해시      │ │
│ │ 타임스탬프          │ │
│ │ 거래 데이터         │ │
│ │ 현재 블록 해시      │ │
│ └─────────────────────┘ │
└─────────────────────────┘
```

> **교수 멘트:**
> "블록이 체인처럼 연결되어 있어서 '블록체인'입니다. 한 번 기록된 데이터는 뒤의 모든 블록을 바꿔야 하기 때문에 위조가 사실상 불가능합니다."

### 1-4. 분산원장 기술(DLT) (5분)

> "DLT는 Distributed Ledger Technology, 분산원장 기술입니다. 쉽게 말해 **중앙 서버 없이 모든 참여자가 동일한 장부를 공유**하는 기술입니다."

**중앙집중 vs 분산 비교:**
```
[중앙집중형]          [분산형]
    ★                ● ─── ●
   /|\               |  ╲  |
  / | \              |   ╲ |
 ●  ●  ●            ● ─── ●
```

**DLT의 장점:**
1. **투명성** - 모든 참여자가 기록 열람 가능
2. **불변성** - 한번 기록되면 수정 불가
3. **탈중앙화** - 단일 장애점 없음
4. **보안성** - 암호화 기반 데이터 보호

**시뮬레이터 안내:**
1. `week01/simulator.html` 파일 열기 (또는 wiastandards.com 접속)
2. 데이터 입력란에 자기 이름 입력

**실습 과제:**
- [ ] 본인 이름으로 블록 3개 생성하기

- [ ] 체인이 깨지는 것 확인하고 스크린샷 제출
