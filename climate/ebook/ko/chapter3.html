API 개요

이 챕터는 WIA 기후 데이터 형식 표준의 핵심 개념을 다룹니다.

3.1 API 개요

WIA 기후 API는 기후 데이터 수집, 쿼리 및 관리를 위한 RESTful HTTP 엔드포인트를 제공합니다.

기본 URL: https://api.wia.live/climate/v1

3.2 데이터 수집 엔드포인트

POST /data - 단일 기후 데이터 메시지 제출

POST /data/batch - 여러 메시지 일괄 제출

3.3 데이터 쿼리 엔드포인트

GET /data - 기후 데이터 쿼리

GET /data/{message_id} - ID로 특정 메시지 검색

3.4 장치 관리 엔드포인트

POST /devices - 장치 등록

GET /devices - 장치 목록

3.5 실시간 스트리밍

WebSocket: wss://api.wia.live/climate/v1/stream

3.6 분석 및 집계

GET /analytics/timeseries - 시계열 집계

GET /analytics/geospatial - 지리공간 집계
