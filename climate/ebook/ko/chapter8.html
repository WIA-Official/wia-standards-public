WIA 기후 인증 프로그램

이 챕터는 WIA 기후 데이터 형식 표준의 핵심 개념을 다룹니다.

8.1 WIA 기후 인증 프로그램

WIA 기후 인증 프로그램은 장치, 센서, 플랫폼 및 소프트웨어가 WIA 기후 데이터 형식 표준을 올바르게 구현하여 글로벌 상호 운용성을 보장하는지 검증합니다.

8.2 인증 레벨

레벨요구 사항유효 기간
레벨 1: 기본메시지 형식 준수, JSON 스키마 검증1년
레벨 2: 표준레벨 1 + 프로토콜 준수 + API 통합1년
레벨 3: 고급레벨 2 + 보안 요구 사항 + 성능 벤치마크2년
레벨 4: 참조레벨 3 + 소스 코드 검토 + 오픈 소스 기여3년

8.3 적합성 테스트

JSON 스키마 검증, 데이터 품질 요구 사항, 프로토콜 준수 테스트

8.4 환경 표준 정렬

ISO 27914, GHG 프로토콜, CF 컨벤션, CMIP6, OGC SensorThings

8.5 데이터 품질 인증

품질 계층: 연구 등급, 생산 등급, 운영 등급, 지표
