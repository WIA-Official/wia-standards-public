8장: 구현 · 인증 · ROI

장 개요: 12개월 도입 로드맵, 3가지 모범 사례, 3건의 작동 케이스 스터디, WIA 인증 3등급, 그리고 주권국·다국적 기업·감사법인의 ROI를 한자리에 정렬한다.

8.1 12개월 도입 로드맵

1~3개월차 — Phase 1 데이터 형식

  • 기존 데이터 인벤토리 작성 — NetCDF 보유, GHG 인벤토리, 공시 스프레드시트, 등록부 기록.
  • 모든 내부 식별자를 LEI / CAS / UCUM / CRS84로 매핑.
  • NetCDF 아카이브의 Zarr v3 거울을 객체 저장소에 구축.
  • CI에서 cf-checker 회귀 — 비CF 데이터셋 머지 차단.

4~6개월차 — Phase 2 API

  • pygeoapi 또는 geoserver로 OGC API Coverages/EDR/Records/Features 가동.
  • STAC 카탈로그(stac‑fastapi‑pgstac) + Zarr 거울 적재.
  • OAuth 2.1 토큰 발급, JWKS 90일 회전.
  • iXBRL 부 표현 포함 공시 트윈 엔드포인트 활성화.

7~9개월차 — Phase 3 프로토콜

  • WIS 2.0 브로커(EMQX 5 + wis2box) 배포.
  • 업스트림 WMO GISC 구독, BUFR/GRIB → CF‑1.10 NetCDF 트랜스코딩.
  • Kafka 3 idempotent + transactional 가동, CEMS 게이트웨이 통합.
  • WIA Chain PoA / BSC 머클 루트 앵커링 켜기.

10~12개월차 — Phase 4 통합

  • 기업 공시 경로 — 사업장 인벤토리 → IFRS S2 → ESRS E1 → CDP.
  • 국가 단위는 사업장 롤업 CRT 2.0 → BTR 첫 제출.
  • 탄소시장 등록부 federation, corresponding adjustment 증거 강제.
  • ESG 대시보드 엔드포인트 + 기후 정의 태그 공개.

8.2 3가지 모범 사례

검증인 우선 설계

대시보드보다 감사 추적 UX를 먼저 만든다. 어떤 공시 총량에서도 1차 기록까지 3번 클릭으로 도달할 수 있다면, 나머지는 자연히 맞는다.

단일 진실원 1회 쓰기

사업장 데이터는 미터 ingest가 단 한 번 쓴다. 공시·BTR·ESG는 읽을 뿐 재입력하지 않는다. “정합 맞추기 단계”가 도입된 순간 표준의 핵심 가치가 사라진다.

기후 정의를 컴파일타임 체크로

역사적 책임·정의로운 전환·손실 피해 태그가 빠진 빌드를 거절한다. CI 체크가 사후 보고보다 싸다.

8.3 케이스 스터디

케이스 1 — 중규모 ASEAN 주권국 (BTR 프로그램)

  • 과제: 첫 BTR 12개월 안 제출, 1차 데이터가 7개 부처 18개 산업 포털에 흩어짐.
  • 해결: Phase 1+2+4 ETF 엔드포인트 9개월 만에 가동, 자동 CRT 2.0 매퍼.
  • 결과: BTR 정시 제출 + reasonable assurance, 감사 비용 −62%, IEA 추정과 0.4% 이내 차이.

케이스 2 — 한국 철강사 (IFRS S2 + K‑ETS)

  • 과제: IFRS S2 첫 해 공시, ESRS E1 EU CSRD, K‑ETS Phase 4 정합, CDP A‑list 방어 동시.
  • 해결: 단일 사업장 인벤토리(JSON‑LD) → 4개 다운스트림 자동 렌더.
  • 결과: 공시 시즌 인력 −41%, CDP 점수 B→A, K‑ETS 정합 차이 2.3%→0.1%.

케이스 3 — 자발적 탄소 등록부 (CA 신뢰 회복)

  • 과제: 2024 ICVCM CCP·VCMI Claims Code가 회수 기준 강화, 레거시 회수 기록에 corresponding adjustment 증거 부재, 거래량 절반.
  • 해결: Verra/Gold Standard 위에 federation 계층, NDA 서명 letter가 첨부된 W3C VC 회수 기록.
  • 결과: 6개월 만에 거래량 2022 대비 78% 회복, 첫 ICVCM CCP 라벨 크레딧이 평균 가격의 3배에 거래.

마무리

과학은 끝났고 회계 프레임워크도 있고 협약도 서명됐다. 우리가 아직 갚지 못한 것은 엔지니어링이다 — 포항 어느 보일러의 미터 한 줄이 본 BTR로 가는 동안 단 한 사람도 숫자를 다시 입력하지 않고, 검증인은 몇 시간 만에 그 길을 따라 서명할 수 있는 와이어 형식. WIA‑CLIMATE‑022가 그 와이어다. 신중하게, 그리고 너그럽게 만들어 가자.
