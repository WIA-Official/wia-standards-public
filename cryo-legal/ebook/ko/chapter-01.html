--------------------------------------------------------------------------------
1.1 목적과 범위
--------------------------------------------------------------------------------

CRYO-LEGAL은 냉동보존을 받는 개인의 법적 지위 관리를 위한 표준화된
프로토콜을 수립합니다. 이 사양은 사망 선언, 법적 인격, 신흥 생명
연장 기술의 맥락에서 권리 회복 간의 복잡한 교차점을 다룹니다.

[ID] 핵심 목표:

    CryoLegalObjectives {
      primaryGoals: [
        "보존 대상자에 대한 명확한 법적 지위 범주 정의",
        "지위 전환 프로토콜 수립",
        "다중 관할권 인정 가능",
        "보존 기간 중 권리 보호",
        "소생 시 법적 회복 촉진"
      ]

      scope: {
        includes: [
          "법적 지위 관리",
          "문서 처리",
          "관할권 처리",
          "후견 프레임워크",
          "회복 절차"
        ]

        excludes: [
          "의료 결정 (CRYO-CONSENT에서 다룸)",
          "신원 확인 (CRYO-IDENTITY에서 다룸)",
          "자산 관리 (CRYO-ASSET에서 다룸)",
          "기술적 보존 (CRYO-PRESERVE에서 다룸)"
        ]
      }
    }

[ID] 대상 이해관계자:

    +------------------------------------------------------------------+
    | 이해관계자             | 역할                                     |
    +------------------------------------------------------------------+
    | 법적 기관              | 지위 결정, 문서 발급                      |
    | 법원                   | 회복 절차, 후견                          |
    | 보존 시설              | 지위 보고, 준수                          |
    | 후견인                 | 보존 중 의사결정                          |
    | 법적 대리인            | 옹호, 문서 준비                          |
    | 정부 기관              | 기록 유지, 인정                          |
    +------------------------------------------------------------------+


--------------------------------------------------------------------------------
1.2 법적 프레임워크 철학
--------------------------------------------------------------------------------

CRYO-LEGAL 프레임워크는 전통적인 법적 개념과 생명 연장 기술의
현실 사이의 균형을 맞추는 근본 원칙에 기반합니다.

[ID] 기본 원칙:

    LegalPhilosophy {
      principle1: {
        name:        "인격의 연속성"
        description: "법적 정체성은 보존 기간 동안 지속됨"
        implication: "권리는 정지되지만 소멸되지 않음"
      }

      principle2: {
        name:        "가역적 사망 선언"
        description: "사망 선언은 소생 시 번복 가능"
        implication: "새로운 법적 지위 범주 필요"
      }

      principle3: {
        name:        "보호적 대리"
        description: "보존된 개인은 법적 보호를 받을 자격"
        implication: "의사결정을 위한 후견 프레임워크"
      }

      principle4: {
        name:        "관할권 조화"
        description: "국경 간 인정 필수"
        implication: "국제 조약 프레임워크"
      }

      principle5: {
        name:        "기술 중립성"
        description: "보존 방법에 관계없이 프레임워크 적용"
        implication: "미래 기술에 적응 가능"
      }
    }

[!] 중요한 구분:

    전통적인 법적 사망은 비가역적입니다. CRYO-LEGAL은 "보존 지위"
    개념을 도입합니다 - 생물학적 중단을 인정하면서도 회복 가능성을
    보존하는 법적 상태입니다.

    이것은 보존된 개인이 전통적 의미에서 "살아있다"는 주장이
    아니라, 그들의 법적 지위가 미래 소생 가능성을 반영해야 한다는
    인식입니다.


--------------------------------------------------------------------------------
1.3 역사적 맥락
--------------------------------------------------------------------------------

냉동보존을 위한 법적 프레임워크의 진화를 이해하면 현재 표준을
맥락화하는 데 도움이 됩니다.

[ID] 법적 진화 타임라인:

    LegalHistoryTimeline {
      era1: {
        period:      "2030년 이전"
        status:      "특정 법적 프레임워크 없음"
        approach:    "표준 사망 선언 적용"
        challenges:  "보존 대상자 보호 없음"
      }

      era2: {
        period:      "2030-2050"
        status:      "초기 입법 노력"
        approach:    "사례별 사법 결정"
        developments: [
          "보존 지위의 최초 법원 인정",
          "초기 후견인 선임 선례",
          "초기 양자 인정 협정"
        ]
      }

      era3: {
        period:      "2050-2070"
        status:      "프레임워크 표준화"
        approach:    "국가 입법 등장"
        developments: [
          "국제 냉동보존 법적 지위 조약 (ICLST)",
          "표준화된 문서 형식",
          "국경 간 인정 프로토콜"
        ]
      }

      era4: {
        period:      "2070-현재"
        status:      "성숙한 법적 프레임워크"
        approach:    "종합적인 국제 표준"
        developments: [
          "보편적 보존 지위 인정",
          "간소화된 회복 절차",
          "AI 지원 법적 처리"
        ]
      }
    }

[i] 참고: 냉동보존 법적 프레임워크의 발전은 실제 소생 사례에 의해
    추진되었으며, 각각 지위 회복과 권리 인정에 중요한 선례를
    수립했습니다.

        obligations: "완전한 법적 의무"
