7.1 후견인 선임
--------------------------------------------------------------------------------

후견은 스스로 행동할 수 없는 보존된 개인에게 법적 대리와 의사결정을
제공합니다.

[ID] 후견인 선임 절차:

      timing: {
        typical:  "보존 지위 선언 시"

        emergency: "법원이 즉시 임시 후견인 선임 가능"

7.2 권한 범위
--------------------------------------------------------------------------------

보존된 개인에 대한 후견인 권한은 특정 권한과 제한을 포함합니다.

    GuardianshipScope {

        ]
        limitations: [

      financialDecisions: {

7.3 의사결정 프로토콜
--------------------------------------------------------------------------------

후견인은 보존된 개인을 위해 결정을 내릴 때 특정 프로토콜을
따라야 합니다.

[ID] 의사결정 프레임워크:

    DecisionMakingProtocol {
      routineDecisions: {
        definition: "범위 내 일상적 사항"
        process: [
          "필요한 결정 식별",
          "개인의 알려진 의사 고려",
          "최선의 이익 분석 적용",
          "결정",
          "결정 및 근거 문서화"
        ]
        reporting: "정기 보고에 포함"
      }

      significantDecisions: {
        definition: "범위 내 주요 사항"
        process: [
          "필요한 결정 식별",
          "관련 정보 수집",
          "필요시 전문가 상담",
          "대안 고려",
          "최선의 이익 분석 적용",
          "결정",
          "철저히 문서화",
          "이해관계자 통지"
        ]
        examples: [
          "시설 이전",
          "상당한 지출",
          "의료 시술 동의"
        ]
      }

      courtApprovalDecisions: {
        definition: "법원 개입이 필요한 결정"
        process: [
          "필요한 결정 식별",
          "법원 청원 준비",
          "이해관계자에게 통지 제공",
          "심리 참석",
          "법원 명령 획득",
          "명령대로 시행"

        examples: [
          "보존 종료",

7.4 후견 종료
--------------------------------------------------------------------------------

후견은 특정 상황에서 종료됩니다.

[ID] 종료 사건:

    GuardianshipTermination {
      restoration: {
        trigger:  "개인이 성공적으로 회복됨"
        process: [
          "법원이 회복 명령 발급",
          "후견인이 최종 회계 제출",
          "법원이 후견인 면책",
          "권한이 개인에게 이전"
        ]
        timeline: "회복 시 자동"
      }

      permanentDeath: {
        trigger:  "개인이 영구 사망으로 선언됨"
        process: [
          "법원이 영구 사망 명령 발급",
          "후견인이 최종 회계 제출",
          "유산 관리 시작",
          "후견인 면책"
        ]
        transition: "유언집행자/관리인이 유산 인수"
      }

      guardianResignation: {
        trigger:  "후견인이 사임 희망"

      guardianRemoval: {
        trigger:  "후견인 비위 또는 무능력"
