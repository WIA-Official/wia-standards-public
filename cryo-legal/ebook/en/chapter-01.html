1.1 Purpose and Scope
--------------------------------------------------------------------------------

CRYO-LEGAL establishes standardized protocols for managing the legal status
of individuals undergoing cryopreservation. This specification addresses the
complex intersection of death declaration, legal personhood, and rights
restoration in the context of emerging life extension technologies.

[ID] Core Objectives:

    CryoLegalObjectives {
      primaryGoals: [
        "Define clear legal status categories for preserved individuals",
        "Establish protocols for status transitions",
        "Enable multi-jurisdiction recognition",
        "Protect rights during preservation period",
        "Facilitate legal restoration upon revival"
      ]

      scope: {
        includes: [
          "Legal status management",
          "Document processing",
          "Jurisdiction handling",
          "Guardianship framework",
          "Restoration procedures"
        ]

        excludes: [
          "Medical decisions (covered by CRYO-CONSENT)",
          "Identity verification (covered by CRYO-IDENTITY)",
          "Asset management (covered by CRYO-ASSET)",
          "Technical preservation (covered by CRYO-PRESERVE)"
        ]
      }
    }

[ID] Target Stakeholders:

    +------------------------------------------------------------------+
    | Stakeholder            | Role                                    |
    +------------------------------------------------------------------+
    | Legal Authorities      | Status determination, document issuance |
    | Courts                 | Restoration proceedings, guardianship   |
    | Preservation Facilities| Status reporting, compliance            |
    | Guardians              | Decision-making during preservation     |
    | Legal Representatives  | Advocacy, document preparation          |
    | Government Agencies    | Record maintenance, recognition         |
    +------------------------------------------------------------------+

1.2 Legal Framework Philosophy
--------------------------------------------------------------------------------

The CRYO-LEGAL framework is built on fundamental principles that balance
traditional legal concepts with the realities of life extension technology.

[ID] Foundational Principles:

    LegalPhilosophy {
      principle1: {
        name:        "Continuity of Personhood"
        description: "Legal identity persists through preservation"
        implication: "Rights are suspended, not extinguished"
      }

      principle2: {
        name:        "Reversible Death Declaration"
        description: "Death declarations can be reversed upon revival"
        implication: "New legal status categories required"
      }

      principle3: {
        name:        "Protective Representation"
        description: "Preserved individuals deserve legal protection"
        implication: "Guardianship framework for decision-making"
      }

      principle4: {
        name:        "Jurisdictional Harmony"
        description: "Cross-border recognition essential"
        implication: "International treaty framework"
      }

      principle5: {
        name:        "Technological Neutrality"
        description: "Framework applies regardless of preservation method"
        implication: "Adaptable to future technologies"
      }
    }

[!] Critical Distinction:

    Traditional legal death is irreversible. CRYO-LEGAL introduces the
    concept of "preservation status" - a legal state that acknowledges
    biological cessation while preserving the potential for restoration.

    This is NOT an assertion that preserved individuals are "alive" in
    the traditional sense, but rather a recognition that their legal
    status should reflect the possibility of future revival.

1.4 Key Terminology
--------------------------------------------------------------------------------

Precise terminology is essential for legal clarity.

[ID] Legal Status Terms:

    StatusTerminology {
      LIVING: {
        definition:  "Standard legal status of a living person"
        rights:      "Full legal capacity and rights"
        obligations: "Full legal obligations"
      }

      LEGALLY_DECEASED: {
        definition:  "Traditional death declaration issued"
        rights:      "Estate passes to heirs"
        note:        "Transitional state before preservation status"
      }

      PRESERVED_STATUS: {
        definition:  "Special legal status during cryopreservation"
        rights:      "Suspended, not extinguished"
        protection:  "Guardian-managed interests"
      }

      REVIVAL_PENDING: {
        definition:  "Revival procedure initiated"
        rights:      "Restoration process underway"
        timeline:    "Temporary state during revival"
      }

      LEGALLY_RESTORED: {
        definition:  "Full legal personhood restored"
        rights:      "Complete legal capacity"
        process:     "Court-ordered restoration"
      }

      PERMANENT_DEATH: {
        definition:  "Revival impossible, final death"
        rights:      "Estate disposition completed"
        finality:    "Irreversible legal status"
      }
    }

[ID] Document Terms:

    DocumentTerminology {
      deathCertificate: {
        definition:  "Official declaration of death"
        issuer:      "Authorized medical/legal authority"
        status:      "May be superseded by restoration"
      }

      preservationDeclaration: {
        definition:  "Official recognition of preservation status"
        issuer:      "Court or designated authority"
        effect:      "Suspends normal death consequences"
      }

      restorationCertificate: {
        definition:  "Official restoration of legal personhood"
        issuer:      "Court order"
        effect:      "Reverses death declaration effects"
      }

      guardianshipOrder: {
        definition:  "Court appointment of guardian"
        authority:   "Defines scope of guardian powers"
        duration:    "Until restoration or permanent death"
      }
    }
