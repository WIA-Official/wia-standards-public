7.1 Guardian Appointment
--------------------------------------------------------------------------------

Guardianship provides legal representation and decision-making for preserved
individuals who cannot act on their own behalf.

[ID] Guardian Appointment Process:

    GuardianAppointment {
      timing: {
        typical:  "At time of preservation status declaration"
        delayed:  "May be appointed later if initially overlooked"
        emergency: "Court may appoint temporary guardian immediately"
      }

      nominationSources: {
        priority: [
          "Individual's advance directive nomination",
          "Spouse or domestic partner",
          "Adult children",
          "Parents",
          "Siblings",
          "Other family members",
          "Close friends",
          "Professional guardian"
        ]
        courtDiscretion: "Court not bound by nomination if
                          concerns exist"
      }

      qualifications: {
        mandatory: [
          "Legal adult",
          "Mentally competent",
          "No conflict of interest",
          "Willing to serve",
          "Able to fulfill duties"
        ]
        preferred: [
          "Familiarity with individual",
          "Understanding of cryopreservation",
          "Geographic proximity to facility",
          "Financial stability"
        ]
        disqualifying: [
          "Criminal history (certain offenses)",
          "Prior guardian removal",
          "Active bankruptcy",
          "Severe cognitive impairment"
        ]
      }
    }

[ID] Court Appointment Process:

    AppointmentProcess {
      petition: {
        petitioner: "Interested party (family, facility, agency)"
        contents: [
          "Information about preserved individual",
          "Proposed guardian nomination",
          "Guardian qualifications",
          "Proposed scope of authority",
          "Bond recommendation"
        ]
      }

      investigation: {
        performer:  "Court investigator or social worker"
        scope: [
          "Verify guardian qualifications",
          "Check for conflicts of interest",
          "Review background (criminal, financial)",
          "Interview proposed guardian",
          "Assess suitability"
        ]
        report:     "Submitted to court before hearing"
      }

      hearing: {
        notice:     "To interested parties"
        evidence:   "Investigation report, testimony"
        objections: "Interested parties may object"
        decision:   "Court appointment order"
      }

      order: {
        contents: [
          "Guardian identity",
          "Scope of authority",
          "Reporting requirements",
          "Bond requirement",
          "Duration"
        ]
        effective: "Upon issuance"
        review:    "Periodic review scheduled"
      }
    }

    GuardianTypes {

        activation:  "Upon original guardian's incapacity/death"

      }

7.2 Scope of Authority
--------------------------------------------------------------------------------

Guardian authority for preserved individuals includes specific powers and
limitations.

[ID] Authority Categories:

    GuardianshipScope {
      personalDecisions: {
        code:        "personal_decisions"
        includes: [
          "Living situation (facility)",
          "Personal effects management",
          "Privacy decisions",
          "Visitor authorizations",
          "Posthumous reproduction decisions"
        ]
        limitations: "Cannot contradict advance directive"
      }

      medicalDecisions: {
        code:        "medical_decisions"
        includes: [
          "Preservation method decisions",
          "Facility transfer decisions",
          "Emergency medical decisions",
          "Research participation consent"
        ]
        limitations: [
          "Cannot terminate preservation without court approval",
          "Must follow advance directive",
          "Best interests standard"
        ]
      }

      financialDecisions: {
        code:        "financial_decisions"
        includes: [
          "Pay preservation fees",
          "Manage assets (unless separate trustee)",
          "Make ordinary financial decisions",
          "File taxes",
          "Handle insurance"
        ]
        limitations: [
          "Court approval for major transactions",
          "Must maintain records",
          "Prudent investor standard"
        ]
        bonded:      "Usually required"
      }

      legalRepresentation: {
        code:        "legal_representation"
        includes: [
          "Represent in legal proceedings",
          "Sign legal documents",
          "Enforce rights",
          "Defend against claims"
        ]
        limitations: "Cannot waive fundamental rights"
      }

      revivalDecisions: {
        code:        "revival_decisions"
        includes: [
          "Consent to revival procedure",
          "Choose revival timing (if options)",
          "Select revival facility",
          "Make revival-related medical decisions"
        ]
        limitations: [
          "Must follow advance directive regarding revival",
          "Best interests of individual",
          "Cannot refuse revival if medically indicated
           and individual wanted revival"
        ]
      }
    }

[ID] Authority Limitations:

    AuthorityLimitations {
      requiresCourtApproval: [
        "Termination of preservation (always)",
        "Major asset sales",
        "Real property transactions",
        "Settlement of claims over threshold",
        "Change of preservation facility",
        "Withdrawal from preservation contract",
        "Research participation with significant risks"
      ]

      prohibited: [
        "Contradict valid advance directive",
        "Make decisions for personal benefit",
        "Terminate preservation without cause",
        "Waive fundamental rights",
        "Consent to experimentation harmful to individual"
      ]

      bestInterestsStandard: {
        description: "All decisions must be in preserved
                      individual's best interests"
        factors: [
          "Individual's expressed wishes",
          "Individual's values and preferences",
          "Preservation quality and safety",
          "Financial sustainability",
          "Revival prospects"
        ]
      }
    }

7.3 Decision-Making Protocols
--------------------------------------------------------------------------------

Guardians must follow specific protocols when making decisions for
preserved individuals.

[ID] Decision-Making Framework:

    DecisionMakingProtocol {
      routineDecisions: {
        definition: "Day-to-day matters within scope"
        process: [
          "Identify decision needed",
          "Consider individual's known wishes",
          "Apply best interests analysis",
          "Make decision",
          "Document decision and rationale"
        ]
        reporting: "Include in periodic reports"
      }

      significantDecisions: {
        definition: "Major matters within scope"
        process: [
          "Identify decision needed",
          "Gather relevant information",
          "Consult experts if needed",
          "Consider alternatives",
          "Apply best interests analysis",
          "Make decision",
          "Document thoroughly",
          "Notify interested parties"
        ]
        examples: [
          "Facility transfer",
          "Significant expenditures",
          "Medical procedure consent"
        ]
      }

      courtApprovalDecisions: {
        definition: "Decisions requiring court involvement"
        process: [
          "Identify decision needed",
          "Prepare petition for court",
          "Provide notice to interested parties",
          "Attend hearing",
          "Obtain court order",
          "Implement as ordered"
        ]
        examples: [
          "Preservation termination",
          "Real property sale",
          "Major litigation settlement"
        ]
      }
    }

[ID] Conflict Resolution:

    ConflictResolution {
      familyDisagreement: {
        scenario: "Family members disagree with guardian decision"
        process: [
          "Guardian explains reasoning",
          "Attempt informal resolution",
          "If unresolved, court petition available"
        ]
        standard: "Guardian decision stands unless court orders
                   otherwise"
      }

      coGuardianDisagreement: {
        scenario: "Co-guardians cannot agree"
        process: [
          "Attempt negotiation",
          "Mediation if available",
          "Court determination if necessary"
        ]
        urgentMatters: "Court may appoint decision-maker
                        for urgent matters"
      }

      facilityConflict: {
        scenario: "Disagreement between guardian and facility"
        process: [
          "Communication and documentation",
          "Escalation to facility management",
          "Regulatory complaint if appropriate",
          "Court involvement if necessary"
        ]
      }
    }

[ID] Documentation Requirements:

    DocumentationRequirements {
      decisionLog: {
        content: [
          "Date of decision",
          "Nature of decision",
          "Information considered",
          "Alternatives evaluated",
          "Rationale for decision",
          "Outcome"
        ]
        retention: "Duration of guardianship + 10 years"
      }

      financialRecords: {
        content: [
          "All income received",
          "All expenditures",
          "Asset inventory",
          "Investment transactions",
          "Tax filings"
        ]
        standard: "Generally accepted accounting principles"
      }

      medicalRecords: {
        content: [
          "Preservation status reports",
          "Facility correspondence",
          "Medical decisions made",
          "Consent forms"
        ]
        access: "Guardian maintains copies"
      }
    }

7.4 Guardianship Termination
--------------------------------------------------------------------------------

Guardianship terminates under specific circumstances.

[ID] Termination Events:

    GuardianshipTermination {
      restoration: {
        trigger:  "Individual successfully restored"
        process: [
          "Court issues restoration order",
          "Guardian files final accounting",
          "Court discharges guardian",
          "Authority transfers to individual"
        ]
        timeline: "Automatic upon restoration"
      }

      permanentDeath: {
        trigger:  "Individual declared permanently dead"
        process: [
          "Court issues permanent death order",
          "Guardian files final accounting",
          "Estate administration begins",
          "Guardian discharged"
        ]
        transition: "Executor/administrator takes over estate"
      }

      guardianResignation: {
        trigger:  "Guardian wishes to resign"
        process: [
          "File petition to resign",
          "Propose successor guardian",
          "Court approves successor",
          "Authority transfers"
        ]
        requirement: "Cannot abandon - must have successor"
      }

      guardianRemoval: {
        trigger:  "Guardian misconduct or incapacity"
        grounds: [
          "Breach of fiduciary duty",
          "Self-dealing",
          "Neglect of duties",
          "Incapacity",
          "Conflict of interest"
        ]
        process: [
          "Petition for removal filed",
          "Court investigation",
          "Hearing",
          "Removal order if warranted",
          "Successor appointed"
        ]
        consequences: "May face civil/criminal liability"
      }

      guardianDeath: {
        trigger:  "Guardian passes away"
        process: [
          "Successor guardian notified",
          "If no successor, court appoints",
          "Emergency guardian if needed",
          "Transition of authority"
        ]
        importance: "Successor designation crucial"
      }
    }

[ID] Final Accounting:

    FinalAccounting {
      required:    "Always upon termination"
      contents: [
        "Complete financial accounting",
        "Asset inventory",
        "Explanation of major decisions",
        "Status of pending matters",
        "Recommendations (if restoration)"
      ]
      review:      "Court reviews and approves"
      discharge:   "Guardian released from liability upon approval"
    }

[ID] Successor Planning:

    SuccessorPlanning {
      importance: {
        reason:    "Preservation may last decades or centuries"
        risk:      "Guardian death, incapacity, or resignation"
      }

      designationMethods: {
        advanceDirective: "Individual names successor preferences"
        guardianNomination: "Current guardian nominates successor"
        courtDesignation: "Court identifies successor in order"
      }

      multipleSuccessors: {
        recommendation: "Name multiple successors in priority order"
        updating:       "Review and update periodically"
      }

      institutionalGuardian: {
        option:    "Professional guardian service"
        advantages: [
          "Continuity",
          "Professional management",
          "Not subject to individual death/incapacity"
        ]
        disadvantages: [
          "Less personal connection",
          "Fees",
          "Institutional limitations"
        ]
      }
    }

Chapter Summary
--------------------------------------------------------------------------------

Key Concepts:
  [OK] Guardian appointed at preservation status declaration
  [OK] Individual's advance directive nomination has priority
  [OK] Guardian types include personal, estate, combined, and co-guardians
  [OK] Authority scope covers personal, medical, financial, legal, and
       revival decisions
  [OK] Court approval required for major decisions
  [OK] Best interests standard governs all decisions
  [OK] Guardianship terminates upon restoration, death, resignation, or
       removal
  [OK] Successor planning critical for long-term preservation
