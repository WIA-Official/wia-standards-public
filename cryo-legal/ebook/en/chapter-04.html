4.1 Preservation Declaration
--------------------------------------------------------------------------------

The preservation declaration formally establishes the PRESERVED_STATUS for
a cryopreserved individual.

[ID] Declaration Structure:

    PreservationDeclaration {
      declarationId:    "PD-2075-KR-000001234"
      recordId:         "LSR-2075-KR-001234"
      jurisdiction:     "KR"

      subject: {
        fullName:       "Kim Preservation"
        identityId:     "CID-2075-KR-001234"
        deathCertificateId: "DC-2075-KR-000001234"
      }

      preservation: {
        facilityId:     "WIA-CRYO-KR-001"
        facilityName:   "WIA Cryopreservation Facility Korea"
        preservationDate: "2075-06-15"
        preservationType: "FULL_BODY"
        storageLocation: "Building A, Section 3, Unit 127"
      }

      legalEffect: {
        effectiveDate:    "2075-06-16"
        statusGranted:    "PRESERVED_STATUS"
        deathCertificateEffect: "SUSPENDED"
        estateEffect:     "HELD_IN_TRUST"
        rightsEffect:     "SUSPENDED_NOT_TERMINATED"
      }

      authority: {
        issuingCourt:     "Seoul Family Court"
        caseNumber:       "2075-PRE-12345"
        judgeName:        "Judge Lee Wisdom"
        issuanceDate:     "2075-06-16"
      }

      conditions: {
        annualReview:     true
        guardianRequired: true
        facilityReporting: "QUARTERLY"
      }
    }

    DeclarationProcess {
      step1: {

      step2: {

      step3: {
        name:     "Declaration Issuance"

          "Issue preservation declaration",
          "Appoint guardian",
          "Establish reporting requirements",
          "Notify relevant agencies"
        ]

      step4: {
        name:     "Registration"

          "Record preservation status",

    }

4.2 Legal Status Suspension
--------------------------------------------------------------------------------

When PRESERVED_STATUS is granted, certain legal consequences of death are
suspended rather than executed.

[ID] Suspended vs. Executed Effects:

    StatusSuspensionEffects {
      suspended: {
        estateDistribution: {
          normalEffect:    "Assets distributed to heirs"
          suspendedEffect: "Assets held in trust"
          management:      "Guardian or trustee manages"
          duration:        "Until restoration or permanent death"
        }

        marriageTermination: {
          normalEffect:    "Marriage ends upon death"
          suspendedEffect: "Marriage status suspended"
          spouseRights:    "May petition for termination or
                           maintain suspended status"
          duration:        "Until court determination"
        }

        parentalRights: {
          normalEffect:    "Rights terminate upon death"
          suspendedEffect: "Rights suspended"
          childGuardianship: "Separate guardian appointed for children"
          duration:        "Until restoration or permanent death"
        }

        contracts: {
          normalEffect:    "Many terminate upon death"
          suspendedEffect: "Performance suspended where possible"
          management:      "Guardian may manage or terminate"
          exceptions:      "Some contracts explicitly terminate"
        }
      }

      executed: {
        voterRegistration: {
          effect:          "Voter removed from rolls"
          restoration:     "Must re-register upon restoration"
        }

        publicOffice: {
          effect:          "Office vacated"
          restoration:     "Must be re-elected/appointed"
        }

        professionalLicenses: {
          effect:          "Suspended or lapsed"
          restoration:     "Re-qualification may be required"
        }
      }
    }

[ID] Trust Arrangements:

    PreservationTrust {
      purpose: "Hold and manage assets during preservation"

      structure: {
        trustor:    "The preserved individual (pre-mortem)"
        trustee:    "Professional trustee or guardian"

                     or designated heirs (upon permanent death)"
      }

      trusteeResponsibilities: [
        "Manage assets prudently",
        "Pay preservation costs",
        "Maintain records",
        "Report to court annually",
        "Coordinate with guardian"
      ]

      permittedExpenses: [
        "Preservation facility fees",
        "Insurance premiums",
        "Legal fees",
        "Asset maintenance costs",
        "Guardian compensation"
      ]

      investmentStandard: "Prudent investor rule"

      termination: {
        restoration: "Assets return to individual"
        permanentDeath: "Distribute per will or intestacy"
      }
    }

4.3 Rights During Preservation
--------------------------------------------------------------------------------

Preserved individuals retain certain fundamental rights even during
preservation.

[ID] Retained Rights:

    PreservationRights {
      fundamentalRights: {
        rightToRevival: {
          description: "Right to attempt revival when feasible"
          source:      "Recognized as fundamental right"
          protection:  "Cannot be waived by others"
          limitations: "Subject to technological possibility"
        }

        dignityRight: {
          description: "Right to dignified treatment"
          source:      "Human dignity principles"
          protection:  "Applies throughout preservation"
          obligations: ["Respectful handling",
                       "Privacy protection",
                       "No exploitation"]
        }

        privacyRight: {
          description: "Protection of personal information"
          source:      "Privacy laws continue to apply"
          protection:  "Information protected during preservation"
          disclosure:  "Limited to necessary parties"
        }

        bodyIntegrity: {
          description: "Right to bodily integrity"
          source:      "Extended from living rights"
          protection:  "No unauthorized procedures"
          guardian:    "Makes necessary medical decisions"
        }
      }

      protectedInterests: {
        reputation: {
          description: "Protection against defamation"
          duration:    "Continues during preservation"
          enforcement: "Guardian may take action"
        }

        intellectualProperty: {
          description: "IP rights continue"
          management:  "Trustee or guardian manages"
          duration:    "Standard IP duration applies"
        }

        advanceDirectives: {
          description: "Pre-mortem wishes honored"
          scope:       "Revival decisions, care preferences"
          enforcement: "Guardian must follow"
        }
      }
    }

4.4 Status Monitoring and Updates
--------------------------------------------------------------------------------

Preservation status requires ongoing monitoring and periodic updates.

[ID] Monitoring Requirements:

    StatusMonitoring {
      periodicReporting: {
        quarterly: {
          reporter:   "Preservation facility"
          contents: [
            "Storage condition verification",
            "Security status",
            "Any incidents or concerns"
          ]
          recipient:  "Guardian + court (annual summary)"
        }

        annual: {
          reporter:   "Guardian"
          contents: [
            "Financial accounting",
            "Preservation status review",
            "Trust management report",
            "Any legal proceedings"
          ]
          recipient:  "Supervising court"
        }

        asNeeded: {
          reporter:   "Any party"
          triggers: [
            "Material change in circumstances",
            "Facility issues",
            "Guardian concerns",
            "Third party disputes"
          ]
          recipient:  "Court"
        }
      }

      facilityInspections: {
        routine:    "Annual by regulatory authority"
        special:    "Upon complaint or incident"
        results:    "Shared with guardians and court"
      }
    }

[ID] Status Updates:

    StatusUpdateEvents {
      facilityTransfer: {
        trigger:     "Facility closure or individual transfer"
        process: [
          "Court approval required",
          "Transfer protocol executed",
          "Records updated",
          "New facility registered"
        ]
        documentation: "Transfer certificate issued"
      }

      guardianChange: {
        trigger:     "Guardian resignation, removal, or death"
        process: [
          "Court petition filed",
          "New guardian nominated",
          "Court approval",
          "Authority transferred"
        ]

      }

      jurisdictionChange: {

          "Recognition in new jurisdiction",
          "Status registration",
          "Guardian appointment recognition",
          "Document apostille"
        ]

      }

      revivalInitiation: {
        trigger:     "Medical team initiates revival"

          "Court notified immediately",
          "Guardian confirms",
          "Status changes to REVIVAL_PENDING",
          "Restoration proceedings initiated"

        documentation: "Revival initiation certificate"
      }
    }

Chapter Summary
--------------------------------------------------------------------------------

Key Concepts:
  [OK] Preservation declaration formally establishes PRESERVED_STATUS
  [OK] Court approval required with specific documentation
  [OK] Death effects are suspended, not executed, during preservation
  [OK] Trust arrangements manage assets during preservation
  [OK] Fundamental rights including revival and dignity continue
  [OK] Ongoing monitoring and periodic reporting required
  [OK] Status updates require court involvement
