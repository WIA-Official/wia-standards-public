# WIA-ROBOT-ETHICS-001 — Robot Ethics (Press Brief)

**WIA Standards releases canonical Robot Ethics envelope shape**

The WIA Standards Committee has published the WIA-ROBOT-ETHICS-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for robot ethics hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
