## Overview

The WIA Screen Reader Standard provides a unified framework for screen reader accessibility across all platforms (NVDA, VoiceOver, TalkBack, Orca) supporting 211 languages.

## Features

- **WIHP Pronunciation**: 211-language IPA-based pronunciation system
- **WIA Braille**: Universal braille conversion (Grade 1, 2, WIA)
- **Cross-Platform SDK**: TypeScript, Python, Rust
- **NVDA Plugin**: Full integration with keyboard shortcuts
- **Browser Extensions**: Chrome, Firefox, Edge
