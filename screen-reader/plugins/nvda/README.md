# WIA Screen Reader - NVDA Plugin

> Universal Accessibility Standard for NVDA Screen Reader

**Philosophy**: 弘益人間 (홍익인간) - Benefit All Humanity

## Installation

1. Copy the `wia_screen_reader` folder to your NVDA addons directory:
   ```
   %APPDATA%\nvda\addons\
   ```

2. Restart NVDA

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| NVDA+Shift+W | Convert selection to WIHP pronunciation |
| NVDA+Shift+B | Convert selection to Braille |
| NVDA+Shift+S | Speak with optimized TTS |
| NVDA+Shift+L | Change language |
| NVDA+Shift+I | Show WIA info |

## Features

- **WIHP Pronunciation**: Convert any text to Korean Hangul pronunciation
- **WIA Braille**: Multi-format braille output (Grade 1, Grade 2, WIA)
- **211 Languages**: Support for all major world languages
- **Optimized TTS**: Enhanced text-to-speech with pronunciation optimization

## Supported Languages

English, Korean, Japanese, Chinese, Spanish, French, German, and 200+ more.

## Requirements

- NVDA 2023.1 or later
- Windows 10/11
