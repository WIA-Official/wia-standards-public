CHAPTER 6

Phase 3 - Communication Protocols

6.1 Railway Signaling - ERTMS/ETCS

European Railway Traffic Management System (ERTMS) with European Train Control System (ETCS) Level 2 provides standardized signaling for inter-Korean railways:

Key Components

  • Radio Block Centre (RBC): Central control system managing train movements
  • GSM-R: Dedicated railway communication network (876-880 MHz uplink, 921-925 MHz downlink)
  • Eurobalise: Track-mounted transponders for position reporting
  • Movement Authority: Permission to proceed with speed and distance limits

6.2 Intelligent Transportation Systems (ITS)

Highway traffic management using DATEX II protocol:

  • Variable Message Signs: Real-time traffic information display
  • Traffic Control: Incident management and route optimization
  • Electronic Toll Collection: DSRC-based automated toll payment

6.3 Aviation Protocols

ICAO-compliant standards for airspace coordination:

  • ADS-B: Automatic Dependent Surveillance-Broadcast for position reporting
  • AIDC: ATS Interfacility Data Communication for handoff coordination
  • Flight Planning: Standardized formats for Seoul-Pyongyang and international routes

6.4 Maritime Communications

  • AIS: Automatic Identification System for vessel tracking
  • VTS: Vessel Traffic Service for port coordination
  • VHF Radio: Ship-to-shore and ship-to-ship communications

6.5 IoT Sensor Networks

MQTT and CoAP protocols for real-time telemetry:

  • Vehicle location, speed, and health monitoring
  • Cargo temperature, humidity, and shock detection
  • Infrastructure condition monitoring (track, bridges, tunnels)
