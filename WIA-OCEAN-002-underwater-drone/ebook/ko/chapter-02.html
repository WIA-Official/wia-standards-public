弘益人間 (홍익인간) - 널리 인간을 이롭게 하라
정교한 하드웨어로 바다의 깊은 곳까지 도달하다

2.1 수중 드론 하드웨어 개요

수중 드론은 극한의 수중 환경에서 작동해야 하므로, 일반 드론이나 로봇과는 다른 특수한 하드웨어 설계가 필요합니다. 수압, 부식, 해수의 전도성, 제한된 통신 환경 등 다양한 도전 과제를 극복해야 합니다.

WIA-OCEAN-002 표준은 수중 드론의 하드웨어 구성 요소를 다음과 같이 분류합니다:

  • 구조 및 프레임: 수압을 견디는 외부 케이스와 내부 프레임
  • 추진 시스템: 수중 이동을 담당하는 모터와 프로펠러
  • 전원 시스템: 배터리, 전력 관리, 충전 시스템
  • 센서 시스템: 환경 데이터를 수집하는 각종 센서
  • 통신 시스템: 수중-수면 간 데이터 전송
  • 제어 시스템: 마이크로컨트롤러, 프로세서, 펌웨어

2.2 구조 및 프레임

2.2.1 압력 용기 (Pressure Hull)

압력 용기는 수중 드론의 가장 중요한 구조적 요소로, 내부 전자 장치를 수압과 물로부터 보호합니다. 깊이가 증가할수록 수압도 증가하므로, 압력 용기는 다음 재질로 제작됩니다:

재질 특성 최대 수심 장점 단점
알루미늄 합금 경량, 내식성 우수 ~500m 가공 용이, 저렴 깊은 수심 부적합
티타늄 합금 고강도, 내식성 탁월 ~6,000m 경량, 강도 우수 고가, 가공 어려움
스테인리스 스틸 고강도, 저렴 ~3,000m 내구성 우수, 중간 가격 무거움
카본 파이버 초경량, 고강도 ~2,000m 매우 가벼움 고가, 충격에 약함
아크릴 (관측창) 투명, 압력 분산 ~1,000m 시각적 관찰 가능 스크래치 취약

수압 계산 공식

수심에 따른 압력은 다음 공식으로 계산됩니다:

P = P₀ + ρgh

여기서:
P  = 절대 압력 (Pa)
P₀ = 대기압 (101,325 Pa)
ρ  = 해수 밀도 (~1,025 kg/m³)
g  = 중력 가속도 (9.81 m/s²)
h  = 수심 (m)

예시: 수심 1,000m에서의 압력
P = 101,325 + (1,025 × 9.81 × 1,000)
P ≈ 10,150,000 Pa = 10.15 MPa = 100 bar

2.2.2 부력 제어

수중 드론은 중성 부력을 유지하여 에너지 효율을 극대화해야 합니다. 부력 제어 방법은 다음과 같습니다:

  • 고정 부력재: 발포 폴리우레탄, 신텍틱 폼 등 경량 부력재를 내장
  • 가변 부력 시스템 (VBS): 펌프를 사용해 탱크에 물을 채우거나 배출하여 부력 조절
  • 드롭 웨이트: 비상 상황 시 무게추를 버려 긴급 부상

2.3 추진 시스템

2.3.1 스러스터 (Thruster) 구성

수중 드론은 6자유도(DOF) 이동을 위해 여러 개의 스러스터를 장착합니다:

스러스터 타입 위치 기능 필요 개수
수직 스러스터 상단/하단 상승/하강 (Heave) 2-4개
수평 스러스터 (전후) 후방 전진/후진 (Surge) 2개
수평 스러스터 (좌우) 측면 좌우 이동 (Sway) 2개
벡터 스러스터 각 코너 회전 (Pitch, Roll, Yaw) 4-8개

2.3.2 모터 및 프로펠러

수중 환경에 적합한 모터는 다음과 같은 특성을 가져야 합니다:

  • 브러시리스 DC 모터 (BLDC): 효율 높고, 내구성 우수, 정밀 제어 가능
  • 밀폐형 설계: 해수 침투 방지를 위한 방수 등급 IP68 이상
  • 부식 방지 코팅: 스테인리스 스틸, 아노다이징 처리
// TypeScript: 스러스터 제어 인터페이스
interface Thruster {
    id: string;
    position: 'vertical-top' | 'vertical-bottom' | 'horizontal-front' |
              'horizontal-rear' | 'lateral-left' | 'lateral-right';
    maxThrust: number;  // Newton 단위
    currentThrust: number;
    rpm: number;
    powerConsumption: number;  // Watts

    setThrust(percentage: number): void;  // -100 ~ 100
    emergencyStop(): void;
    getStatus(): ThrusterStatus;
}

interface ThrusterStatus {
    operational: boolean;
    temperature: number;
    vibration: number;
    errorCode?: string;
}

// 8-스러스터 구성 예제
class OctoThrusterDrone {
    thrusters: Thruster[];

    constructor() {
        this.thrusters = [
            { id: 'VT1', position: 'vertical-top', maxThrust: 50, /* ... */ },
            { id: 'VT2', position: 'vertical-top', maxThrust: 50, /* ... */ },
            { id: 'VB1', position: 'vertical-bottom', maxThrust: 50, /* ... */ },
            { id: 'VB2', position: 'vertical-bottom', maxThrust: 50, /* ... */ },
            { id: 'HF1', position: 'horizontal-front', maxThrust: 80, /* ... */ },
            { id: 'HF2', position: 'horizontal-front', maxThrust: 80, /* ... */ },
            { id: 'LL', position: 'lateral-left', maxThrust: 60, /* ... */ },
            { id: 'LR', position: 'lateral-right', maxThrust: 60, /* ... */ },
        ];
    }

    // 상승/하강 제어
    dive(speed: number) {
        this.thrusters
            .filter(t => t.position.includes('vertical'))
            .forEach(t => t.setThrust(speed));
    }

    // 전진/후진 제어
    move(speed: number) {
        this.thrusters
            .filter(t => t.position.includes('horizontal'))
            .forEach(t => t.setThrust(speed));
    }

    // 회전 제어
    rotate(yaw: number) {
        // 좌우 스러스터를 반대 방향으로 작동
        this.thrusters.find(t => t.id === 'LL')?.setThrust(yaw);
        this.thrusters.find(t => t.id === 'LR')?.setThrust(-yaw);
    }
}

2.4 전원 시스템

2.4.1 배터리 기술

수중 드론의 작동 시간은 배터리 용량과 에너지 효율에 의해 결정됩니다. 주로 사용되는 배터리 기술:

배터리 타입 에너지 밀도 장점 단점 사용 사례
리튬-폴리머 (LiPo) 150-250 Wh/kg 고출력, 경량 화재 위험, 수명 짧음 소형 ROV
리튬-이온 (Li-ion) 200-265 Wh/kg 안정성, 긴 수명 중간 출력 중형 AUV
고체 리튬 배터리 300-400 Wh/kg 안전성 우수, 고밀도 고가, 신기술 심해 탐사
연료 전지 (수소) 600-800 Wh/kg 초장시간 작동 복잡한 시스템 장거리 AUV
알루미늄-산소 1,300 Wh/kg 해수를 사용, 초고밀도 일회용, 실험 단계 장기 임무

⚠️ 배터리 안전 지침

  • 압력 용기 내부에 배터리를 안전하게 고정하고, 과충전 보호 회로를 반드시 설치하십시오.
  • 배터리 온도를 실시간 모니터링하고, 40°C 이상 시 자동 차단되도록 설정하십시오.
  • 물에 접촉 시 즉시 전원을 차단하고, 배터리를 안전한 장소로 이동하십시오.
  • 장기 보관 시 50-60% 충전 상태를 유지하고, 서늘하고 건조한 곳에 보관하십시오.

2.4.2 전력 관리 시스템

// TypeScript: 전력 관리 시스템
interface PowerManagementSystem {
    battery: {
        voltage: number;        // Volts
        current: number;        // Amperes
        capacity: number;       // Ah
        remainingCapacity: number;
        temperature: number;    // Celsius
        cycleCount: number;
    };

    consumers: PowerConsumer[];

    getTotalPowerConsumption(): number;
    getRemainingRuntime(): number;  // minutes
    enableLowPowerMode(): void;
    emergencyShutdown(): void;
}

interface PowerConsumer {
    name: string;
    priority: 'critical' | 'high' | 'medium' | 'low';
    powerDraw: number;  // Watts
    enabled: boolean;
}

class DronePowerManager implements PowerManagementSystem {
    battery = {
        voltage: 48.0,
        current: 12.5,
        capacity: 100,
        remainingCapacity: 75,
        temperature: 25,
        cycleCount: 45
    };

    consumers: PowerConsumer[] = [
        { name: 'Main Computer', priority: 'critical', powerDraw: 50, enabled: true },
        { name: 'Navigation System', priority: 'critical', powerDraw: 20, enabled: true },
        { name: 'Thrusters', priority: 'high', powerDraw: 300, enabled: true },
        { name: '4K Camera', priority: 'high', powerDraw: 25, enabled: true },
        { name: 'LED Lights', priority: 'medium', powerDraw: 80, enabled: true },
        { name: 'Sonar System', priority: 'medium', powerDraw: 40, enabled: true },
        { name: 'Communication', priority: 'high', powerDraw: 15, enabled: true },
    ];

    getTotalPowerConsumption(): number {
        return this.consumers
            .filter(c => c.enabled)
            .reduce((sum, c) => sum + c.powerDraw, 0);
    }

    getRemainingRuntime(): number {
        const powerConsumption = this.getTotalPowerConsumption();
        const energyRemaining = this.battery.remainingCapacity * this.battery.voltage;
        return (energyRemaining / powerConsumption) * 60;  // minutes
    }

    enableLowPowerMode(): void {
        // 우선순위가 낮은 장치 비활성화
        this.consumers.forEach(c => {
            if (c.priority === 'low' || c.priority === 'medium') {
                c.enabled = false;
            }
        });
        console.log('Low power mode activated. Estimated runtime:',
                    this.getRemainingRuntime(), 'minutes');
    }

    emergencyShutdown(): void {
        // 중요 시스템만 유지
        this.consumers.forEach(c => {
            if (c.priority !== 'critical') {
                c.enabled = false;
            }
        });
    }
}

// 사용 예제
const powerMgr = new DronePowerManager();
console.log('Total power consumption:', powerMgr.getTotalPowerConsumption(), 'W');
console.log('Remaining runtime:', powerMgr.getRemainingRuntime(), 'minutes');

if (powerMgr.battery.remainingCapacity < 30) {
    powerMgr.enableLowPowerMode();
}

2.5 센서 시스템

수중 드론은 다양한 환경 데이터를 수집하기 위해 여러 센서를 통합합니다. WIA-OCEAN-002 표준은 다음 센서 카테고리를 정의합니다:

  • 항법 센서: IMU, 나침반, DVL, 압력 센서
  • 영상 센서: 카메라, 라이트
  • 음향 센서: 소나, 수중 마이크
  • 환경 센서: CTD, pH, 용존 산소, 탁도
  • 화학 센서: 오염물질 탐지, 가스 센서

주요 센서 설명

  • IMU (Inertial Measurement Unit): 가속도계, 자이로스코프, 자력계를 통합하여 드론의 자세와 가속도를 측정
  • DVL (Doppler Velocity Log): 도플러 효과를 이용해 해저면 대비 속도와 고도를 측정
  • CTD 센서: Conductivity (전도도), Temperature (온도), Depth (깊이)를 동시에 측정하여 해수의 물리적 특성 파악
  • 멀티빔 소나: 여러 빔을 발사하여 넓은 영역의 해저 지형을 3D 매핑

📚 장 요약

  • 압력 용기: 티타늄, 알루미늄, 스테인리스 스틸 등 다양한 재질로 제작되며, 수심에 따라 적절한 재질 선택이 중요합니다. 수심 1,000m에서 약 100 bar의 압력을 견뎌야 합니다.
  • 부력 제어: 중성 부력 유지를 통해 에너지 효율을 극대화하며, 고정 부력재와 가변 부력 시스템을 조합하여 사용합니다.
  • 스러스터 시스템: 6자유도 이동을 위해 최소 6-8개의 스러스터가 필요하며, 수직/수평/측면 스러스터를 조합하여 정밀한 제어를 실현합니다.
  • 배터리 기술: 리튬-이온/폴리머가 주로 사용되며, 연료 전지와 같은 첨단 기술이 장시간 임무를 위해 개발 중입니다.
  • 전력 관리: 우선순위 기반 전력 분배와 저전력 모드를 통해 작동 시간을 최대화하고, 비상 상황에 대비합니다.
  • 센서 통합: 항법, 영상, 음향, 환경, 화학 센서를 통합하여 다차원적인 해양 데이터를 수집합니다.

🤔 복습 질문

  1. 수심 2,000m에서 작동하는 수중 드론을 설계한다면 어떤 재질의 압력 용기를 선택하겠습니까? 그 이유를 설명하십시오.
  2. 수압 계산 공식을 사용하여 수심 500m에서의 절대 압력을 계산하고, 이것이 대기압의 몇 배인지 계산하십시오.
  3. 6자유도(DOF) 이동이란 무엇이며, 수중 드론에서 이를 구현하기 위해 필요한 스러스터 구성을 설명하십시오.
  4. 브러시리스 DC 모터(BLDC)가 수중 드론에서 선호되는 이유를 세 가지 이상 나열하십시오.
  5. 제공된 코드 예제에서 OctoThrusterDrone 클래스의 rotate 메서드가 어떻게 작동하는지 설명하십시오.
  6. 리튬-이온 배터리와 연료 전지를 비교하여, 각각의 장단점과 적합한 사용 사례를 설명하십시오.
  7. PowerManagementSystem의 enableLowPowerMode 함수가 배터리 수명을 연장하는 메커니즘을 설명하십시오.
  8. DVL(Doppler Velocity Log) 센서의 작동 원리와 수중 드론 항법에서의 중요성을 설명하십시오.
