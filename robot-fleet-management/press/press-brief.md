# WIA-ROBOT-FLEET-001 — Robot Fleet Management (Press Brief)

**WIA Standards releases canonical Robot Fleet Management envelope shape**

The WIA Standards Committee has published the WIA-ROBOT-FLEET-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for robot fleet management hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
