Data Catalog 완벽 가이드

데이터 카탈로그의 모든 것을 배우는 완벽한 여정

WIA-DATA-007 표준 한글 전자책

📖 책 소개

이 책은 데이터 카탈로그의 기초부터 실전 활용까지 모든 것을 다룹니다. 데이터 디스커버리, 메타데이터 관리, 데이터 계보, 비즈니스 용어집, 데이터 분류와 태깅, 그리고 Apache Atlas, DataHub, Collibra 같은 주요 플랫폼의 활용법을 배울 수 있습니다.

대상 독자: 데이터 엔지니어, 데이터 분석가, 데이터 거버넌스 담당자, 데이터 아키텍트, CDO(Chief Data Officer)

학습 시간: 약 8-10시간

📚 목차

CHAPTER 01

Data Catalog 개요

데이터 카탈로그의 정의, 필요성, 핵심 개념을 배웁니다. 왜 조직에 데이터 카탈로그가 필요한지, 어떤 문제를 해결할 수 있는지 이해합니다.

데이터 카탈로그란? 비즈니스 가치 주요 기능 도입 배경
CHAPTER 02

Data Discovery & Search

데이터 검색과 발견의 핵심 원리를 배웁니다. 효과적인 검색 전략, 메타데이터 인덱싱, 검색 알고리즘을 이해합니다.

검색 아키텍처 Elasticsearch 자동완성 관련도 순위
CHAPTER 03

Metadata Management

메타데이터의 수집, 저장, 관리 방법을 학습합니다. 기술 메타데이터, 비즈니스 메타데이터, 운영 메타데이터를 이해합니다.

메타데이터 유형 자동 추출 스키마 관리 버전 관리
CHAPTER 04

Data Lineage Tracking

데이터 계보의 추적과 시각화 방법을 배웁니다. 데이터의 흐름, 변환, 의존성을 이해하고 관리하는 방법을 학습합니다.

계보 그래프 컬럼 레벨 계보 영향도 분석 자동 추적
CHAPTER 05

Business Glossary

비즈니스 용어집의 구축과 관리 방법을 학습합니다. 조직 전체가 동일한 언어로 데이터를 이해할 수 있도록 만드는 방법을 배웁니다.

용어 정의 용어 표준화 승인 워크플로우 용어 관계
CHAPTER 06

Data Classification & Tagging

데이터 분류와 태깅 전략을 배웁니다. PII 탐지, 자동 분류, 보안 정책 적용 방법을 이해합니다.

자동 분류 PII 탐지 태그 전략 보안 정책
CHAPTER 07

플랫폼별 활용법

Apache Atlas, DataHub, Collibra 등 주요 데이터 카탈로그 플랫폼의 특징과 활용법을 배웁니다.

Apache Atlas DataHub Collibra 비교 분석
CHAPTER 08

실전 구축 가이드

실제 조직에 데이터 카탈로그를 구축하는 방법을 배웁니다. 아키텍처 설계, 구현, 운영, 거버넌스까지 전체 프로세스를 학습합니다.

아키텍처 구현 전략 베스트 프랙티스 ROI 측정
