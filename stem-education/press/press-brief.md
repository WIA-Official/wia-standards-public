# WIA-EDU-019 — STEM Education (Press Brief)

**WIA Standards releases canonical STEM Education envelope shape**

The WIA Standards Committee has published the WIA-EDU-019 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for stem education hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
