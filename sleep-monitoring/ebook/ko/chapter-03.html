제3장: 수면다원검사 (Polysomnography, PSG)

"수면다원검사는 수면 과학의 골드 스탠다드입니다. 뇌파, 안구 운동, 근육 활동, 심장 박동, 호흡, 산소 포화도를 동시에 기록하여 수면의 모든 측면을 포괄적으로 평가합니다."

— 수면 의학의 핵심 진단 도구

3.1 PSG 개요

수면다원검사 (Polysomnography)는 수면 중 여러 생리적 신호를 동시에 기록하고 분석하는 종합적인 검사입니다.

🎯 PSG의 목적

  • 수면 단계 분류 — 각성, N1, N2, N3, REM 정확한 판정
  • 수면 장애 진단 — 수면 무호흡, 주기적 사지 운동, RBD 등
  • 치료 효과 평가 — CPAP 압력 적정, 약물 반응 모니터링
  • 연구 — 수면 메커니즘, 신약 개발, 임상 시험

📊 PSG 측정 신호

신호 약어 측정 목적 전극/센서 위치
뇌전도 EEG 뇌파, 수면 단계 두피 (F3, F4, C3, C4, O1, O2)
안전도 EOG 안구 운동, REM 감지 눈 바깥쪽 (E1, E2)
근전도 EMG 근육 긴장도, REM atonia 턱 (chin), 다리 (leg)
심전도 ECG 심박수, 부정맥 가슴
호흡 기류 Airflow 무호흡, 저호흡 코/입 (nasal cannula, thermistor)
호흡 노력 Effort 흉부/복부 움직임 가슴, 배 (RIP 벨트)
산소 포화도 SpO2 저산소증 손가락 (pulse oximeter)
코골이 Snore 코골이 감지 목 (마이크)
체위 Position 자세 (앙와위, 측와위 등) 가슴/복부 (센서)
비디오 Video 행동, 발작, RBD 적외선 카메라

3.2 EEG (뇌전도) 표준

🧠 EEG 전극 배치

AASM은 국제 10-20 시스템에 따라 최소 3개 EEG 채널을 권장합니다:

  • F4-M1 — 우측 전두부 (Frontal)
  • C4-M1 — 우측 중심부 (Central)
  • O2-M1 — 우측 후두부 (Occipital)
  • 백업: F3-M2, C3-M2, O1-M2 (좌측)

M1/M2는 귀 뒤 유양돌기 (Mastoid) 기준 전극입니다.

⚙️ EEG 기술 사양

파라미터 AASM 요구사항
샘플링 레이트 ≥ 200 Hz (권장 ≥ 500 Hz)
해상도 ≥ 12 bit
Low Frequency Filter (LFF) 0.3 Hz
High Frequency Filter (HFF) 35 Hz
민감도 (Sensitivity) 5-10 μV/mm
임피던스 < 5 kΩ (권장 < 2 kΩ)
Notch Filter (60Hz) 선택적 사용 (가능한 피함)

📊 EEG 파형 특징

뇌파 주파수 진폭 수면 단계
델타파 (δ) 0.5-2 Hz ≥75 μV N3 (깊은 수면)
세타파 (θ) 4-7 Hz 중간 N1, REM
알파파 (α) 8-13 Hz 30-50 μV 각성 (눈 감음)
수면 방추파 12-14 Hz 중간 N2
베타파 (β) 14-30 Hz 낮음 각성, REM
K-복합체 — ≥75 μV N2

3.3 EOG (안전도) 표준

👁️ EOG 전극 배치

AASM 권장 배치:

  • E1 (Left EOG) — 왼쪽 눈 바깥쪽 1cm, 아래 1cm → M2 기준
  • E2 (Right EOG) — 오른쪽 눈 바깥쪽 1cm, 위 1cm → M2 기준

이 배치는 수직 및 수평 안구 운동을 모두 감지합니다.

⚙️ EOG 기술 사양

파라미터 AASM 요구사항
샘플링 레이트 ≥ 200 Hz
LFF 0.3 Hz
HFF 35 Hz
민감도 5 μV/mm

📊 안구 운동 분류

  • 느린 안구 운동 (SEM) — N1 단계, 0.5 Hz 이하
  • 빠른 안구 운동 (REM) — REM 수면, 급격한 편향
  • 눈 깜빡임 (Blinks) — 각성 상태
  • 독서 안구 운동 — 각성 중 수평 이동

3.4 EMG (근전도) 표준

💪 EMG 전극 배치

  1. 턱 (Chin) EMG — 필수
    • 3개 전극: 턱 아래 중앙 1개, 좌우 2cm 옆 2개
    • 목적: 근긴장도 측정, REM atonia 감지
  2. 다리 (Leg) EMG — 권장
    • 양쪽 앞정강이근 (anterior tibialis)
    • 목적: 주기적 사지 운동 (PLMS) 감지

⚙️ EMG 기술 사양

파라미터 AASM 요구사항
샘플링 레이트 ≥ 200 Hz
LFF 10 Hz
HFF 100 Hz
민감도 2 μV/mm

3.5 호흡 모니터링

🌬️ 호흡 측정 방법

  1. 호흡 기류 (Airflow)
    • Nasal Pressure Transducer — 코 압력 (권장, 민감함)
    • Oronasal Thermistor — 코/입 온도 (무호흡 감지)
  2. 호흡 노력 (Respiratory Effort)
    • RIP (Respiratory Inductance Plethysmography) — 흉부/복부 벨트
    • 목적: 중추성 vs 폐쇄성 무호흡 구별
  3. 산소 포화도 (SpO2)
    • Pulse Oximetry — 손가락 센서
    • 샘플링: ≥ 25 Hz (AASM 권장)
    • 평균화 시간: ≤ 3초

📊 호흡 이벤트 정의

이벤트 정의 (AASM)
무호흡 (Apnea) 기류 ≥90% 감소, ≥10초 지속
저호흡 (Hypopnea) 기류 ≥30% 감소 + SpO2 ≥3% 하락 또는 각성, ≥10초
RERA 호흡 노력 증가 + 각성, 무호흡/저호흡 기준 미달
폐쇄성 무호흡 기류 없음 + 호흡 노력 지속
중추성 무호흡 기류 없음 + 호흡 노력 없음
혼합성 무호흡 중추성으로 시작 → 폐쇄성으로 전환

📈 무호흡-저호흡 지수 (AHI)

AHI = (무호흡 수 + 저호흡 수) / 총 수면 시간(시간)

정상: AHI < 5
경증: 5 ≤ AHI < 15
중등도: 15 ≤ AHI < 30
중증: AHI ≥ 30

3.6 PSG 검사 절차

🌙 검사 프로토콜

  1. 준비 단계 (오후 6-8시)
    • 환자 도착, 설문지 작성
    • 수면 클리닉 환경 적응
    • 카페인/알코올 금지 설명
  2. 전극 부착 (오후 8-9시)
    • 피부 세정 (알코올, 연마 젤)
    • 전극 부착 (EEG, EOG, EMG, ECG)
    • 센서 부착 (nasal cannula, RIP 벨트, SpO2)
    • 임피던스 확인 (< 5 kΩ)
  3. 교정 (Calibration)
    • 눈 뜨기/감기 (알파파 확인)
    • 눈 좌우/위아래 움직이기 (EOG 확인)
    • 이 악물기 (EMG 확인)
    • 심호흡 (호흡 센서 확인)
  4. 수면 기록 (오후 10시~오전 6시)
    • 소등 시간 (Lights Off) 기록
    • 기술사 모니터링 (신호 품질, 전극 탈락)
    • 비디오 녹화 (적외선)
  5. 각성 (오전 6시)
    • 점등 시간 (Lights On) 기록
    • 전극 제거
    • 주관적 수면 설문
  6. 데이터 분석
    • 수면 단계 분류 (30초 에폭)
    • 호흡 이벤트 점수화
    • 각성, 사지 운동 점수화
    • 보고서 작성

3.8 WIA-MED-021 PSG 인증 요구사항

✅ 필수 요구사항

  1. AASM 표준 준수 — 최신 AASM Manual 기준
  2. 최소 채널 요구사항
    • EEG: ≥3 채널 (F4-M1, C4-M1, O2-M1)
    • EOG: 2 채널 (E1-M2, E2-M2)
    • EMG: 1 턱 + 2 다리
    • ECG: 1 채널
    • 호흡: nasal pressure + thermistor + RIP 벨트
    • SpO2: 맥박 산소측정기
  3. 기술 사양
    • 샘플링: ≥ 200 Hz (EEG/EOG), ≥ 25 Hz (SpO2)
    • 해상도: ≥ 12 bit
    • 임피던스: < 5 kΩ
  4. 데이터 형식 — EDF+ 또는 EDF 호환
  5. 품질 관리
    • 기술사 자격 (RPSGT 또는 동등)
    • 정기 장비 교정 (연 1회)
    • 분석 일치도 검증 (inter-rater reliability > 80%)
