제4장: 활동 측정기 및 웨어러블 센서

"웨어러블 수면 추적기는 PSG의 정확도를 제공하지 못하지만, 장기간 일상 환경에서 수면 패턴을 모니터링할 수 있는 접근성과 편의성을 제공합니다. 이는 수면 건강의 민주화입니다."

4.1 활동 측정기 (Actigraphy)

Actigraphy는 손목에 착용하는 가속도계 기반 장치로, 움직임을 측정하여 수면-각성 패턴을 추정합니다.

📱 작동 원리

  1. 가속도계 — 3축 움직임 감지 (X, Y, Z)
  2. 활동 카운트 — 1분 에폭당 움직임 강도 점수
  3. 알고리즘 — 활동 카운트 → 수면/각성 분류
  4. 추론
    • 움직임 낮음 = 수면
    • 움직임 높음 = 각성

⚙️ 기술 사양

파라미터 표준 사양
센서 3축 가속도계
샘플링 레이트 32-100 Hz
에폭 길이 30-60초 (통상 60초)
착용 위치 비우세 손목 (non-dominant wrist)
배터리 수명 7-30일
메모리 30-90일 데이터 저장
방수 IPX7-IPX8 (권장)

📊 주요 메트릭

메트릭 설명
총 수면 시간 (TST) 수면으로 분류된 에폭 합계
수면 효율 (SE) TST / 침상 시간 × 100%
입면 잠복기 (SOL) 취침 ~ 첫 수면 에폭
입면 후 각성 (WASO) 수면 중 각성 시간 합계
각성 횟수 수면 중 각성 에피소드 수
단편화 지수 수면 연속성 지표

✅ 장점

  • 비침습적 — 편안한 착용감, 수면 방해 최소화
  • 장기 모니터링 — 수주~수개월 연속 측정 가능
  • 일상 환경 — 집에서 자연스러운 수면 측정
  • 저렴함 — PSG 대비 1/10 비용 ($200-500)
  • 객관적 데이터 — 주관적 일기 보완
  • 교대 근무 평가 — 불규칙한 수면 패턴 추적

⚠️ 한계

  • 수면 단계 미분류 — REM, NREM, 깊은 수면 구별 불가
  • 낮은 특이도 — 누워서 가만히 있으면 수면으로 오분류
  • 불면증 과대평가 — 실제보다 수면 시간 길게 측정
  • 수면 무호흡 미감지 — 호흡 측정 없음

🎯 임상 적용

적합 부적합

• 일주기 리듬 장애
• 교대 근무 장애
• 치료 효과 모니터링
• 노인 수면 패턴
• 소아 수면 연구 		• 수면 무호흡 진단

4.2 웨어러블 수면 추적기

소비자 웨어러블은 actigraphy에 추가 센서(PPG, 온도 등)를 결합하여 수면 단계 추정을 시도합니다.

⌚ 주요 웨어러블 기기

기기 센서 측정 항목 가격
Fitbit (Charge/Sense) 가속도계, PPG, SpO2, 온도 수면 단계, 수면 점수, HRV, SpO2 $150-300
Apple Watch 가속도계, PPG, SpO2 수면 단계, 호흡수, HRV $400-800
Oura Ring PPG, 가속도계, 온도 수면 단계, Readiness 점수, HRV, 체온 $299
Whoop Strap PPG, 가속도계 수면 단계, Recovery 점수, HRV, 호흡수 $30/월
Garmin (Fenix/Forerunner) 가속도계, PPG, SpO2 수면 단계, Body Battery, HRV, SpO2 $350-1000

📊 웨어러블 센서 기술

  1. 광용적맥파 (PPG, Photoplethysmography)
    • 원리: LED 빛 + 광센서 → 혈류량 측정
    • 측정: 심박수, HRV, 호흡수, SpO2
    • 위치: 손목, 손가락
    • 녹색 LED (심박수) + 적색/적외선 LED (SpO2)
  2. 가속도계 (Accelerometer)
    • 3축 움직임 감지
    • 활동 카운트, 자세 감지
  3. 온도 센서 (Thermistor)
    • 피부 온도 또는 환경 온도
    • 일주기 리듬 추적
  4. 자이로스코프 (Gyroscope)
    • 회전 운동 감지
    • 자세 정밀 측정

🧠 수면 단계 알고리즘

웨어러블은 다음 신호를 결합하여 수면 단계를 추정합니다:

단계 특징
각성 (Wake) 높은 움직임 + 높은 심박수
얕은 수면 (Light = N1+N2) 낮은 움직임 + 중간 심박수 + 중간 HRV
깊은 수면 (Deep = N3) 거의 없는 움직임 + 낮은 심박수 + 높은 HRV + 느린 호흡
REM 수면 낮은 움직임 + 높은 심박수 + 낮은 HRV + 불규칙한 호흡

주의: 웨어러블 알고리즘은 독점적이며 (closed-source), 제조사마다 다릅니다. 대부분 머신러닝 기반이지만 훈련 데이터와 검증 방법이 불투명합니다.

📈 웨어러블 정확도

연구 결과 (PSG 대비):

측정 정확도 비고
수면 vs 각성 80-95% 양호
총 수면 시간 (TST) ±30분 오차 허용 가능
수면 효율 (SE) 상관계수 0.7-0.9 양호
수면 단계 (4단계) 60-70% 중간 (개선 필요)
깊은 수면 (N3) 50-65% 낮음
REM 수면 60-75% 중간

4.3 심박수 변이도 (HRV)

HRV (Heart Rate Variability)는 심박 간 시간 간격의 변동으로, 자율신경계 활동을 반영합니다.

📊 HRV와 수면 단계

수면 단계 HRV 자율신경
각성 중간 교감/부교감 균형
NREM (N1, N2) 증가 부교감 우세
NREM (N3, SWS) 최대 부교감 최대
REM 감소 교감 증가

⚙️ HRV 메트릭

  • RMSSD — Root Mean Square of Successive Differences (ms)
  • SDNN — Standard Deviation of NN intervals (ms)
  • HF Power — High Frequency Power (0.15-0.4 Hz, 부교감 활동)
  • LF Power — Low Frequency Power (0.04-0.15 Hz, 교감 활동)

4.4 WIA-MED-021 웨어러블 요구사항

✅ 필수 요구사항

  1. 센서 사양
    • 3축 가속도계 (≥32 Hz 샘플링)
    • PPG 센서 (심박수, HRV)
    • SpO2 측정 (선택, 권장)
  2. PSG 검증
    • 최소 100명 PSG 동시 측정
    • 다양한 인구 집단 (나이, 성별, BMI)
    • 수면 장애 환자 포함 (무호흡, 불면증)
  3. 정확도 기준
    • 수면-각성 분류: ≥85% 정확도
    • 총 수면 시간: 평균 오차 ≤30분
    • 수면 효율: 상관계수 ≥0.70
    • 수면 단계 (4단계): ≥70% 정확도 (권장)
  4. 알고리즘 투명성
    • 수면 단계 분류 알고리즘 문서화
    • 훈련 데이터셋 설명
    • 검증 방법론 공개
  5. 데이터 형식
    • 원시 가속도계 데이터 접근 가능 (API 또는 export)
    • 에폭별 수면-각성 분류 데이터
    • 심박수, HRV 데이터
  6. 사용자 경험
    • 착용 편의성 (≥7일 연속 착용 가능)
    • 배터리 수명 (≥5일)
    • 방수 (최소 IPX7)
