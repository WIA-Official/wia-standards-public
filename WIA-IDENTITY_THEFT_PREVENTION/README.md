## 🚀 Quick Start

### Installation

```bash
# Clone or download
git clone https://github.com/WIA-Official/wia-standards.git
cd wia-standards/standards/WIA-IDENTITY_THEFT_PREVENTION

# Install (optional)
chmod +x install.sh
./install.sh
```

### Basic Usage

```bash
# CLI tool (if available)
./cli/wia-identity_theft_prevention.sh info
./cli/wia-identity_theft_prevention.sh validate data.json
```

```typescript
# API usage (TypeScript)
import { IDENTITY_THEFT_PREVENTIONCore } from './api/typescript';

const core = new IDENTITY_THEFT_PREVENTIONCore();
await core.initialize();

const result = await core.process(data);
```
