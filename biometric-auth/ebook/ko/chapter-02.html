"널리 인간을 이롭게 하라"

생체인식 과제를 해결함으로써 신원 시스템이 진정으로 모든 인류에게 공평하고 안전하며 접근 가능하도록 보장합니다.

2.1 스푸핑 및 위조 공격

프레젠테이션 공격(위조 공격)은 생체인식 시스템을 속이기 위해 가짜 생체 샘플을 사용합니다. 이러한 공격은 가장 큰 보안 취약점 중 하나입니다.

2.1.1 방식별 공격 벡터

방식 공격 유형 정교함 수준
지문 실리콘 몰드, 젤라틴 복제, 종이 출력 낮음 ~ 중간
얼굴 인쇄 사진, 동영상 재생, 3D 마스크 낮음 ~ 높음
홍채 인쇄된 콘택트렌즈, 고해상도 이미지 중간 ~ 높음
음성 녹음 재생, 음성 합성, 딥페이크 낮음 ~ 매우 높음

⚠️ 실제 사례

2021년 연구자들이 10달러 미만으로 만든 3D 프린팅 마스크로 10대 스마트폰 중 6대의 안면인식을 우회했습니다. 이는 위조 방지(PAD) 프레젠테이션 공격 탐지의 필요성을 강조합니다.

2.3 인구통계학적 편향

생체인식 시스템은 역사적으로 인구통계학적 그룹별로 차별적인 정확도를 보여왔으며, 이는 상당한 형평성 우려를 야기합니다.

2.3.1 문서화된 편향 문제

방식 영향받는 그룹 영향
안면인식 어두운 피부톤, 여성 34배 높은 오류율 (NIST 연구)
지문 고령자, 육체 노동자 융선 저하로 인한 높은 FRR
홍채 백내장/녹내장 환자 등록 실패
음성 비원어민, 언어 장애인 높은 FRR

2.4 템플릿 보안

2.4.1 템플릿 보안이 중요한 이유

비밀번호와 달리 생체정보는 변경할 수 없습니다. 생체 템플릿이 유출되면 해당 생체 특성은 영원히 신원 확인에 사용할 수 없게 됩니다.

// 템플릿 보호 기법
{
  "cancelableBiometrics": {
    "description": "역변환 불가능한 변환 적용",
    "benefit": "유출 시 교체 가능",
    "methods": ["블록 순열", "표면 접힘", "Bloom 필터"]
  },
  "secureSketch": {
    "description": "헬퍼 데이터 기법",
    "benefit": "잡음 있는 샘플에서 키 복원",
    "methods": ["Fuzzy Vault", "Fuzzy Commitment"]
  },
  "homomorphicEncryption": {
    "description": "암호화된 상태에서 매칭",
    "benefit": "원본 템플릿 노출 없음"
  }
}

2.5 상호운용성 과제

과제 설명 WIA 해결책
독점 형식 벤더별 고유 템플릿 형식 표준화된 WIA 템플릿 형식
알고리즘 변형 벤더 간 알고리즘 차이 표준화된 API 인터페이스
품질 메트릭 일관성 없는 품질 점수 통합 품질 점수(UQS)
국경 간 인식 다른 국가 표준 ICAO/ISO 표준 정렬

2.6 장 요약

핵심 요점

  • 스푸핑 공격에는 다중 레벨 위조 방지(PAD) 필요
  • 개인정보 보호 규정은 생체 데이터에 대해 엄격한 통제 요구
  • 인구통계학적 편향은 테스트와 완화가 필요한 형평성 우려 야기
  • 템플릿 보안은 취소 가능한 생체인식 등 보호 기법 필요
  • 상호운용성에는 표준화된 형식과 API 필요

2.7 복습 문제

  1. 프레젠테이션 공격(위조 공격)이란 무엇이며 어떻게 방어할 수 있나요?
  2. 생체 데이터에 적용되는 주요 개인정보 보호 규정은 무엇인가요?
  3. 인구통계학적 편향이 생체인식 시스템에서 우려되는 이유는 무엇인가요?
  4. 생체 템플릿 보호가 비밀번호 보호보다 중요한 이유는 무엇인가요?
  5. 벤더 간 상호운용성을 방해하는 요인은 무엇인가요?
