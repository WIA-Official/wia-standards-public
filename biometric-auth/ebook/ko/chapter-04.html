홍채 및 망막 인식

눈 기반 생체인식 기술의 원리, 하드웨어 구성, 알고리즘, 그리고 뛰어난 정확도의 비밀을 탐구합니다.

홍채 인식 개요

홍채(Iris)는 눈의 동공과 공막(흰자) 사이에 위치한 색깔 있는 고리 모양의 조직입니다. 홍채의 복잡한 패턴은 생후 약 2년 안에 형성되어 평생 변하지 않으며, 일란성 쌍둥이조차 서로 다른 홍채 패턴을 가집니다. 심지어 한 사람의 왼쪽 눈과 오른쪽 눈의 홍채도 완전히 다릅니다.

홍채 인식은 모든 생체인식 기술 중 가장 높은 정확도를 제공합니다. 지문이 약 40개의 고유 특징을 가지는 반면, 홍채는 200개 이상의 고유 특징점을 포함합니다. 이론적으로 FAR(오수락률)이 10^-78에 달할 수 있어, 사실상 오인식이 불가능합니다.

홍채 패턴 구성 요소:
- 섬유주 (Trabecular Meshwork): 방사상 결합 조직 구조
- 암점 (Crypts): 홍채 표면의 작은 구멍들
- 주름 (Furrows): 동심원 및 방사상 주름
- 관상 (Collarette): 동공과 섬모 영역의 경계선
- 색소 반점 (Pigment Spots): 멜라닌 침착 부위

홍채 인식 시스템 구조

홍채 인식 시스템은 이미지 획득, 전처리, 특징 추출, 매칭의 4단계로 구성됩니다. 각 단계는 높은 정확도를 달성하기 위해 정교하게 설계되어야 합니다.

1. 이미지 획득
   - 근적외선(NIR) 카메라 사용 (700-900nm)
   - 멜라닌 색소 영향 최소화
   - 동공/홍채 경계 명확히 촬영

2. 전처리
   - 홍채 내경/외경 검출 (Daugman's Integro-differential)
   - 극좌표 변환 (Rubber Sheet Model)
   - 조명 정규화

3. 특징 추출
   - 2D Gabor 웨이블릿 필터 적용
   - 위상 정보 추출
   - IrisCode 생성 (2048비트)

4. 매칭
   - 해밍 거리 계산
   - 회전 보정 (여러 시프트 비교)
   - 임계값 비교 (일반적으로 0.32)

홍채 이미지 획득

고품질 홍채 이미지 획득은 정확한 인식의 기초입니다. 근적외선(NIR) 조명을 사용하면 홍채의 섬세한 구조가 명확히 드러나며, 어두운 색 홍채도 밝은 색 홍채와 동등하게 처리할 수 있습니다.

Daugman의 홍채 인식 알고리즘

John Daugman 교수가 1993년에 개발한 알고리즘은 현재까지 가장 널리 사용되는 홍채 인식 방법론입니다. 이 알고리즘은 전 세계 공항과 국경 통제 시스템의 표준이 되었습니다.

Daugman 알고리즘의 핵심 단계 
1. 홍채 세그멘테이션 (Integro-differential Operator)
────────────────────────────────────────────────────

             ∂
max(r,x₀,y₀) ── | G_σ(r) * ∮  I(x,y) ds |
             ∂r           r,x₀,y₀

- 원형 경계(동공, 홍채)를 검출하는 연산자
- 가우시안 스무딩으로 노이즈 완화
- 반경과 중심을 최적화

2. 정규화 (Rubber Sheet Model)
─────────────────────────────

I(x,y) → I(r,θ)  where  0 ≤ r ≤ 1, 0 ≤ θ ≤ 2π

- 홍채 영역을 직사각형 좌표로 변환
- 동공 크기 변화에 강건
- 고정 크기 템플릿 생성

3. 특징 인코딩 (2D Gabor Wavelets)
─────────────────────────────────

h(Re,Im) = sgn{Re(∫∫ I(ρ,φ) e^(-iω(θ₀-φ)) e^(-(r₀-ρ)²/α²) 
              e^(-(θ₀-φ)²/β²) ρ dρ dφ}

- 복소 Gabor 필터로 로컬 텍스처 분석
- 위상 정보만 사용 (진폭 무시)
- 2비트 양자화 (4개 사분면)
- 결과: 2048비트 IrisCode

4. 매칭 (Hamming Distance)
──────────────────────────

HD = (1/N) Σ (codeA[i] ⊕ codeB[i]) ∩ mask[i]

- XOR 연산으로 차이 비트 계산
- 가림 영역(눈꺼풀, 속눈썹) 마스킹
- 회전 보정을 위한 시프트 테스트
- 결정 임계값: 일반적으로 0.32

홍채 인식 구현

import numpy as np
from scipy import ndimage
from scipy.signal import convolve2d

class IrisRecognition:
    """Daugman 알고리즘 기반 홍채 인식 시스템"""
    
    def __init__(self):
        self.normalized_width = 512
        self.normalized_height = 64
        self.gabor_wavelength = 18
        
    def segment_iris(self, eye_image):
        """홍채 세그멘테이션 - 동공과 홍채 경계 검출"""
        # 동공 검출 (더 어두운 영역)
        pupil_center, pupil_radius = self._find_circle(
            eye_image, 
            r_range=(20, 80),
            edge_type='inner'
        )
        
        # 홍채 경계 검출
        iris_center, iris_radius = self._find_circle(
            eye_image,
            r_range=(60, 150),
            edge_type='outer',
            center_hint=pupil_center
        )
        
        return {
            'pupil': (pupil_center, pupil_radius),
            'iris': (iris_center, iris_radius)
        }
    
    def _find_circle(self, image, r_range, edge_type, center_hint=None):
        """Integro-differential operator로 원형 경계 검출"""
        max_response = 0
        best_params = None
        
        # 가우시안 블러 적용
        blurred = ndimage.gaussian_filter(image, sigma=2)
        
        for r in range(r_range[0], r_range[1]):
            for x0 in range(image.shape[1]):
                for y0 in range(image.shape[0]):
                    # 원주 상의 픽셀 합 계산
                    response = self._circular_integral(blurred, x0, y0, r)
                    if response > max_response:
                        max_response = response
                        best_params = ((x0, y0), r)
        
        return best_params
    
    def normalize(self, image, pupil, iris):
        """Rubber Sheet 정규화"""
        pupil_center, pupil_r = pupil
        iris_center, iris_r = iris
        
        normalized = np.zeros((self.normalized_height, self.normalized_width))
        
        for i in range(self.normalized_height):
            for j in range(self.normalized_width):
                theta = 2 * np.pi * j / self.normalized_width
                r = i / self.normalized_height
                
                # 동공에서 홍채 경계까지 선형 보간
                x = int(pupil_center[0] + r * (iris_r * np.cos(theta)))
                y = int(pupil_center[1] + r * (iris_r * np.sin(theta)))
                
                if 0 <= x < image.shape[1] and 0 <= y < image.shape[0]:
                    normalized[i, j] = image[y, x]
        
        return normalized
    
    def encode(self, normalized_iris):
        """2D Gabor 웨이블릿으로 IrisCode 생성"""
        code_real = []
        code_imag = []
        
        # 여러 스케일과 방향의 Gabor 필터 적용
        for scale in [1, 2]:
            wavelength = self.gabor_wavelength * scale
            for orientation in range(8):
                theta = orientation * np.pi / 8
                
                # Gabor 필터 생성
                kernel = self._create_gabor_kernel(wavelength, theta)
                
                # 컨볼루션
                response = convolve2d(normalized_iris, kernel, mode='same')
                
                # 위상 양자화 (2비트)
                code_real.extend((response.real > 0).flatten().astype(int))
                code_imag.extend((response.imag > 0).flatten().astype(int))
        
        return np.array(code_real + code_imag)
    
    def match(self, code1, code2, mask1=None, mask2=None):
        """해밍 거리 기반 매칭"""
        min_hd = 1.0
        
        # 회전 보정을 위한 시프트 테스트
        for shift in range(-10, 11):
            shifted_code2 = np.roll(code2, shift * 16)
            
            # XOR로 차이 계산
            xor_result = np.logical_xor(code1, shifted_code2)
            
            # 마스크 적용 (가림 영역 제외)
            if mask1 is not None and mask2 is not None:
                shifted_mask2 = np.roll(mask2, shift * 16)
                valid_bits = np.logical_and(mask1, shifted_mask2)
                hd = np.sum(xor_result[valid_bits]) / np.sum(valid_bits)
            else:
                hd = np.mean(xor_result)
            
            min_hd = min(min_hd, hd)
        
        # 임계값 비교 (0.32가 일반적)
        is_match = min_hd < 0.32
        
        return is_match, min_hd

망막 인식

망막 인식(Retina Recognition)은 눈 안쪽 망막의 혈관 패턴을 분석합니다. 망막의 혈관 패턴은 개인마다 고유하며 질병이나 노화에도 거의 변하지 않습니다. 홍채 인식보다도 높은 정확도를 제공하지만, 스캔 과정이 불편하여 제한적으로 사용됩니다.

홍채 vs 망막 인식 비교
특성 홍채 인식 망막 인식
스캔 대상 눈 표면 (홍채) 눈 안쪽 (망막 혈관)
정확도 매우 높음 극히 높음
스캔 거리 10cm ~ 수 미터 접안 필요
사용자 편의성 양호 불편함
비용 중간 높음
적용 분야 공항, 스마트폰, 출입통제 최고 보안 시설, 군사
