Standards and Compliance

ISO standards, FIDO protocols, and regulatory requirements

Standards Landscape

The biometric industry operates within a comprehensive framework of international standards that ensure interoperability, quality, and security. These standards, developed primarily by ISO/IEC JTC 1/SC 37, provide the foundation for reliable biometric system deployment worldwide.

Compliance with relevant standards demonstrates commitment to quality, facilitates international interoperability, and often satisfies regulatory requirements. This chapter provides an overview of the most important standards for biometric authentication implementations.

Core ISO/IEC Biometric Standards

ISO/IEC 19795 - Biometric Performance Testing ISO/IEC JTC 1/SC 37

Defines methodology for testing and reporting biometric system performance. Essential for meaningful accuracy claims and system comparison.

  • Part 1: Principles and framework for performance testing
  • Part 2: Testing methodologies for technology and scenario evaluation
  • Part 4: Interoperability performance testing
  • Part 5: Access control scenario testing
ISO/IEC 24745 - Biometric Template Protection ISO/IEC JTC 1/SC 27

Specifies requirements for protecting biometric templates during storage and transmission. Addresses the unique challenges of securing irrevocable biometric data.

  • Cancelable biometrics requirements
  • Biometric cryptosystem specifications
  • Template protection evaluation criteria
ISO/IEC 30107 - Presentation Attack Detection ISO/IEC JTC 1/SC 37

Framework for testing and reporting anti-spoofing capabilities. Critical for demonstrating system resistance to presentation attacks.

  • Part 1: Framework and definitions
  • Part 2: Data formats
  • Part 3: Testing and reporting
  • Part 4: Mobile device profile

FIDO Alliance Standards

FIDO: Fast Identity Online

FIDO Alliance standards enable passwordless authentication using biometrics while keeping biometric data local to the user's device. This privacy-preserving approach has been adopted by major technology companies and integrated into web browsers.

FIDO2 Components:
├── WebAuthn: W3C JavaScript API for browsers
├── CTAP2: Protocol for external authenticators
└── UAF: Mobile native authentication

Privacy Benefits:
├── Biometrics never leave the device
├── No shared secrets (public key only on server)
├── No correlation between services (unique key pairs)
└── User consent required for each authentication
