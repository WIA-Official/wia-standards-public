**홍익인간 (弘益人間) - 널리 인간을 이롭게 하라**

## 1. 소개

### 1.1 대상 전문가

이 가이드는 다음 전문가를 위해 작성되었습니다:

- **SLP** (Speech-Language Pathologist) - 언어재활사
- **OT** (Occupational Therapist) - 작업치료사
- **심리사** - 임상심리전문가
- **특수교사** - 특수교육 전문가

### 1.2 전문가 대시보드 기능

| 기능 | 설명 |
|------|------|
| **다중 클라이언트 관리** | 여러 클라이언트 동시 관리 |
| **평가 도구** | 표준화된 평가 수행 및 기록 |
| **치료 계획** | 목표 설정 및 진행 추적 |
| **어휘 관리** | 핵심/주변 어휘 커스터마이징 |
| **데이터 분석** | 의사소통 패턴, 오류 분석 |
| **임상 보고서** | 자동화된 보고서 생성 |

## 3. 평가 도구

### 3.1 지원 평가 유형

| 평가 | 설명 | 주기 |
|------|------|------|
| **Communication Matrix** | 의사소통 발달 평가 | 초기/6개월 |
| **AAC 평가** | AAC 접근 및 사용 평가 | 초기/필요시 |
| **어휘 평가** | 이해/표현 어휘 평가 | 3개월 |
| **이해력 평가** | 언어 이해 수준 | 6개월 |
| **사용자 정의** | 목표 기반 맞춤 평가 | 목표별 |

### 3.2 평가 실시 절차

```
1. 클라이언트 선택
2. 평가 유형 선택
3. 평가 항목 수행
4. 점수 입력
5. 소견 작성
6. 권장사항 검토
7. 저장 및 공유
```

### 3.3 평가 결과 해석

시스템이 제공하는 자동 해석:

```
┌─────────────────────────────────────────────────────────┐
│  평가 결과 요약                                         │
├─────────────────────────────────────────────────────────┤
│  의사소통 매트릭스: 레벨 4 (상징적 의사소통)            │
│                                                         │
│  강점:                                                  │
│  - 요청 기능 우수 (85%)                                 │
│  - 거부/수락 명확 (90%)                                 │
│                                                         │
│  발전 영역:                                             │
│  - 코멘트 기능 (35%) → 목표 설정 권장                   │
│  - 질문 기능 (20%) → 추가 평가 필요                     │
│                                                         │
│  권장사항:                                              │
│  1. 코멘트 기능 확장을 위한 모델링 강화                 │

## 4. 치료 계획

### 4.1 목표 설정 (SMART)

목표는 SMART 원칙에 따라 설정:

- **S**pecific (구체적)
- **M**easurable (측정 가능)
- **A**chievable (달성 가능)
- **R**elevant (관련성 있는)
- **T**ime-bound (기한 있는)

### 4.2 목표 카테고리

| 카테고리 | 예시 |
|----------|------|
| **어휘 확장** | 3개월 내 핵심 어휘 20개 추가 |
| **의사소통 빈도** | 일 평균 30회 이상 의사소통 |
| **문장 복잡도** | MLU 2.0 이상 달성 |
| **사회적 의사소통** | 자발적 인사 80% 이상 |
| **자기 옹호** | 도움 요청 독립적 수행 |
| **독립성** | 케어기버 촉구 없이 AAC 접근 |

### 4.3 목표 진행 추적

```
┌─────────────────────────────────────────────────────────┐
│  목표: 핵심 어휘 20개 습득                              │
│  기간: 2024.01 - 2024.03                               │
├─────────────────────────────────────────────────────────┤
│  진행률: ████████████░░░░ 75%                          │
│                                                         │
│  마일스톤:                                              │
│  ✅ 5개 어휘 습득 (1월 15일 달성)                       │
│  ✅ 10개 어휘 습득 (2월 5일 달성)                       │
│  ✅ 15개 어휘 습득 (2월 25일 달성)                      │
│  ⬜ 20개 어휘 습득 (3월 15일 목표)                      │
│                                                         │
│  현재: 15개 습득 / 자발적 사용 12개                     │
└─────────────────────────────────────────────────────────┘
```

## 5. 어휘 관리

### 5.1 핵심 어휘 vs 주변 어휘

| 구분 | 정의 | 예시 |
|------|------|------|
| **핵심 어휘** | 모든 상황에서 자주 사용 | 나, 원해요, 더, 싫어 |
| **주변 어휘** | 특정 상황/관심사 관련 | 공룡, 컴퓨터, 피자 |

### 5.2 어휘 분석 도구

시스템 제공 분석:

```
┌─────────────────────────────────────────────────────────┐
│  어휘 분석 결과                                         │
├─────────────────────────────────────────────────────────┤
│  총 어휘: 78개 (핵심 25개 / 주변 53개)                  │
│                                                         │
│  누락된 핵심 어휘:                                      │
│  - "더" (more) - 즉시 추가 권장                         │
│  - "다른" (different) - 추가 검토                       │
│  - "거기" (there) - 추가 검토                           │
│                                                         │
│  가장 많이 사용:                                        │
│  1. 물 (152회)                                          │
│  2. 원해요 (134회)                                      │
│  3. 엄마 (98회)                                         │
│                                                         │
│  사용되지 않는 어휘: 12개 (제거 검토)                   │
└─────────────────────────────────────────────────────────┘
```

### 5.3 개인화 어휘 추가

클라이언트 관심사 기반 어휘 추가:

```
관심사: 공룡
추가 권장 어휘:
- 티렉스, 트리케라톱스, 화석, 박물관
- "공룡 보고 싶어요" 문장 템플릿
```

### 5.4 레이아웃 커스터마이징

```

├───────────────────────────┤

## 6. 데이터 분석

### 6.1 의사소통 패턴 분석

시스템이 자동으로 분석하는 패턴:

| 패턴 유형 | 분석 내용 |
|----------|----------|
| **시간대별** | 가장 활발한 시간 |
| **기능별** | 요청, 코멘트, 질문 비율 |
| **상대별** | 대화 상대에 따른 변화 |
| **상황별** | 활동/장소에 따른 패턴 |

### 6.2 오류 분석

```
┌─────────────────────────────────────────────────────────┐
│  오류 분석 (최근 30일)                                  │
├─────────────────────────────────────────────────────────┤
│  총 오류: 23회 (오류율: 8%)                             │
│                                                         │
│  오류 유형:                                             │
│  ┌─────────────────┬──────┬───────────────────────────┐│
│  │ 선택 오류       │ 52%  │ 인접 심볼 잘못 선택       ││
│  │ 네비게이션 오류 │ 35%  │ 페이지 찾기 어려움        ││
│  │ 취소 후 재시도  │ 13%  │ 의도 변경                 ││
│  └─────────────────┴──────┴───────────────────────────┘│
│                                                         │
│  권장 조치:                                             │
│  • 심볼 크기 증가 (선택 오류 감소)                      │
│  • 카테고리 단순화 (네비게이션 개선)                    │
└─────────────────────────────────────────────────────────┘
```

### 6.3 시스템 권장사항

데이터 기반 자동 권장:

```
📊 데이터 기반 권장사항

1. [우선순위 높음] 심볼 크기 증가
   - 근거: 선택 오류 12회 (52%)
   - 예상 효과: 오류율 30% 감소

2. [우선순위 중간] "더" 심볼 첫 페이지 이동
   - 근거: 높은 사용 빈도, 현재 2번째 페이지
   - 예상 효과: 반응 시간 15% 단축

3. [검토 필요] 미사용 심볼 10개 제거
   - 근거: 30일 이상 미사용
   - 주의: 케어기버 확인 후 제거
```

## 8. 인지 장애 유형별 고려사항

### 8.1 자폐 스펙트럼

**평가 고려:**
- 감각 처리 평가 포함
- 특별 관심사 파악
- 루틴/전환 어려움 평가

**치료 계획:**
- 예측 가능한 구조
- 특별 관심사 활용
- 점진적 변화

**어휘 선정:**
- 관심 분야 어휘 우선
- 감각/감정 어휘 포함
- 사회적 스크립트

### 8.2 치매

**평가 고려:**
- 보존된 능력 파악
- 변동성 고려
- 가족 정보 활용

**치료 계획:**
- 유지/보존 목표
- 일상생활 기능
- 참여 극대화

**어휘 선정:**
- 익숙한 어휘 우선
- 가족/장소 사진
- 단순한 구조

### 8.3 지적 장애

**평가 고려:**
- 발달 수준 고려
- 강점 기반 접근
- 일반화 평가

**치료 계획:**
- 기능적 목표
- 반복과 연습
- 자연스러운 환경
