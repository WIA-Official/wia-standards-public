# WIA Cognitive AAC - Caregiver Guide

## 케어기버를 위한 사용 가이드

**홍익인간 (弘益人間) - 널리 인간을 이롭게 하라**

---

## 1. 소개

### 1.1 이 가이드의 목적

이 가이드는 인지 장애(자폐, 치매 등)가 있는 가족이나 돌봄 대상자가 AAC(보완대체의사소통) 장치를 효과적으로 사용할 수 있도록 지원하는 케어기버를 위해 작성되었습니다.

### 1.2 케어기버 대시보드 기능

| 기능 | 설명 |
|------|------|
| **실시간 모니터링** | 현재 활동, 기분 상태, 알림 확인 |
| **일간/주간 리포트** | 의사소통 통계, 진행상황 추적 |
| **알림 관리** | 중요 알림 설정 및 수신 |
| **원격 조정** | UI 설정, 메시지 전송, 기기 잠금 |

---

## 2. 시작하기

### 2.1 계정 설정

1. 앱 다운로드 (iOS/Android)
2. 케어기버 계정으로 회원가입
3. 돌봄 대상자(클라이언트)와 연결

### 2.2 권한 설정

케어기버로서 다음 권한을 설정할 수 있습니다:

```
✅ 활동 보기
✅ 리포트 보기
✅ UI 조정
✅ 메시지 전송
✅ 알림 관리
✅ 기기 잠금
```

⚠️ **주의**: 모든 권한은 사용자(또는 법적 보호자)의 동의가 필요합니다.

| 😐 보통 | 평상시 상태 | 관찰 |

| 🟠 경고 | 비활동, 기분 급변 | 빠른 확인 |
| 🔵 정보 | 일반 활동 업데이트 | 참고 |

## 4. 리포트 활용

### 4.1 일간 요약

매일 제공되는 요약 정보:

- **총 의사소통 횟수**: 하루 동안 전달한 메시지 수
- **사용 심볼 수**: 사용한 고유 심볼 개수
- **평균 반응 시간**: 심볼 선택에 걸린 평균 시간
- **기분 추이**: 하루 동안의 기분 변화
- **주요 활동 시간대**: 가장 활발한 시간

### 4.2 주간 진행상황

주간 리포트에서 확인할 수 있는 정보:

- 일별 의사소통 추이 그래프
- 어휘 확장 현황
- 목표 달성 진행률
- 권장 사항

### 4.3 리포트 해석 팁

**긍정적 신호:**
- 의사소통 횟수 증가
- 새로운 심볼 사용
- 반응 시간 단축
- 안정적인 기분

**주의 필요 신호:**
- 의사소통 감소
- 제한된 어휘 사용
- 반응 시간 증가
- 기분 변동 심함

## 5. 알림 설정

### 5.1 알림 임계값 설정

```
비활동 알림: 30분 후 알림
오류율 알림: 20% 이상 시 알림
기분 변화 알림: 급격한 변화 시 알림
```

### 5.2 조용한 시간 설정

특정 시간에는 알림을 받지 않도록 설정:

```
조용한 시간: 22:00 - 07:00
예외: 긴급 알림은 항상 수신
```

### 5.3 긴급 연락처

여러 긴급 연락처를 등록하여 중요 알림 공유:

```
1순위: 보호자 (전화, SMS, 푸시)
2순위: 담당 치료사 (이메일)
3순위: 가족 (푸시)
```

## 6. 원격 조정

### 6.1 UI 조정

상황에 따라 원격으로 UI 설정 조정:

| 설정 | 언제 조정하나요 |
|------|----------------|
| 심볼 크기 증가 | 선택 오류가 많을 때 |
| 표시 항목 감소 | 인지 부담이 클 때 |
| 대비 높임 | 조명이 어두울 때 |
| 애니메이션 끔 | 산만해질 때 |

### 6.2 메시지 전송

격려 메시지나 알림 전송:

```
"점심 먹을 시간이에요 🍚"
"잘 하고 있어요! 👏"
"5분 후에 치료 시간이에요"
```

### 6.3 기기 잠금

필요 시 원격으로 기기 잠금:

- 휴식 시간
- 치료 세션 중
- 잠자리 시간

⚠️ **중요**: 잠금 해제 버튼은 항상 표시됩니다.

## 8. 치매 사용자 지원

### 8.1 익숙함 유지

- 가족 사진 심볼 활용
- 익숙한 장소/물건 이미지
- 좋아하는 음악 연결

### 8.2 지남력 지원

- 현재 시간/날짜 표시
- 위치 리마인더
- 가족 사진과 이름

### 8.3 회상 치료 지원

- 인생 이야기 등록
- 추억 사진 공유
- 의미 있는 날짜 알림

### 8.4 반복 질문 대응

시스템이 반복 질문을 감지하면:
- 부드럽게 재안내
- 안심 메시지 표시
- 케어기버에게 알림

## 10. 문제 해결

### 10.1 일반적인 문제

| 문제 | 해결 방법 |
|------|----------|
| 연결 안 됨 | 인터넷 연결 확인, 앱 재시작 |
| 알림 안 옴 | 알림 설정 확인, 조용한 시간 확인 |
| 데이터 늦음 | 동기화 버튼 탭 |

### 10.2 긴급 상황

긴급 상황 발생 시:
1. 앱에서 "긴급 연락" 버튼 탭
2. 등록된 모든 긴급 연락처에 알림
3. 마지막 활동 위치 공유 (동의 시)

### 10.3 지원 요청

- 앱 내 "도움말" 메뉴
- 이메일: support@wia-official.org

## 12. 자주 묻는 질문

**Q: 얼마나 자주 확인해야 하나요?**
A: 상황에 따라 다르지만, 알림 설정을 활용하면 필요할 때만 확인할 수 있습니다.

**Q: 사용자가 모니터링을 불편해하면?**
A: 모니터링 범위를 조정하고, 사용자와 함께 결정하세요. 프라이버시를 존중하는 것이 중요합니다.

**Q: 여러 케어기버가 함께 사용할 수 있나요?**
A: 네, 여러 케어기버를 등록하고 각각의 권한을 다르게 설정할 수 있습니다.

**Q: 오프라인에서도 작동하나요?**
A: 사용자의 AAC 앱은 오프라인에서도 작동합니다. 케어기버 앱은 실시간 기능을 위해 인터넷이 필요합니다.
