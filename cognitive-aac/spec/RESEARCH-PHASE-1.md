# 인지 AAC 표준 리서치 보고서

---

## 철학
**홍익인간 (弘益人間)** - 널리 인간을 이롭게 하라

---

## 1. 연구 개요

### 1.1 목적
인지장애(자폐 스펙트럼, 치매, 지적장애)를 가진 사용자를 위한 AAC 시스템의 효과적 적용을 위해 기존 평가 도구와 인지 프로파일 표준을 조사하고 분석합니다.

### 1.2 조사 범위
- 자폐 스펙트럼 진단 도구 (ADOS-2)
- 적응 행동 평가 도구 (Vineland-3)
- 치매 선별 도구 (MMSE, MoCA)
- 의사소통 평가 도구 (Communication Matrix)

---

## 2. 자폐 스펙트럼 진단: ADOS-2

### 2.1 개요
**ADOS-2 (Autism Diagnostic Observation Schedule, Second Edition)**는 자폐 스펙트럼 장애(ASD) 진단을 위한 표준 도구로, "골드 스탠다드"로 인정받고 있습니다.

### 2.2 구조
| 모듈 | 대상 | 특징 |
|------|------|------|
| Toddler | 12-30개월 | 영아 전용 |
| Module 1 | 31개월+ (비언어적) | 단어 없음~단어 일부 |
| Module 2 | 구절 사용자 | 문장 미사용 |
| Module 3 | 언어적 아동/청소년 | 유창한 언어 |
| Module 4 | 언어적 성인 | 성인 전용 |

### 2.3 평가 영역
```
ADOS-2 평가 영역:
├── 사회적 상호작용 (Social Interaction)
│   ├── 사회적 미소
│   ├── 눈맞춤
│   ├── 공동 주의
│   └── 사회적 상호성
├── 의사소통 (Communication)
│   ├── 언어적 의사소통
│   ├── 비언어적 의사소통
│   └── 대화 기술
└── 제한적/반복적 행동 (RRBs)
    ├── 반복적 행동
    ├── 감각 관심사
    └── 루틴 고집
```

### 2.4 심리측정적 특성
- **민감도**: 83-91%
- **특이도**: 86-94%
- **평가자 간 신뢰도**: 78-98%

### 2.5 AAC 적용 시사점
- 비언어적/최소 언어 사용자를 위한 모듈 존재 (Module 1, 2)
- 의사소통 능력과 사회적 상호작용 동시 평가
- AAC 기기 사용자에 대한 별도 지침 필요

## 3. 적응 행동 평가: Vineland-3

### 3.1 개요
**Vineland Adaptive Behavior Scales, Third Edition (Vineland-3)**는 일상생활에 필요한 개인적, 사회적 기술을 평가하는 표준화된 도구입니다.

### 3.2 평가 영역
| 영역 | 하위 영역 | 설명 |
|------|-----------|------|
| **의사소통** | 수용 언어 | 정보 이해 및 반응 |
| | 표현 언어 | 언어적 표현 |
| | 문자 언어 | 읽기/쓰기 |
| **일상생활** | 개인 | 자기 관리 |
| | 가정 | 가사 활동 |
| | 지역사회 | 사회 활동 |
| **사회화** | 대인 관계 | 타인과의 관계 |
| | 놀이/여가 | 여가 활동 |
| | 대처 기술 | 적응 기술 |
| **운동 기술** | 대근육 | 전체적 움직임 |
| (선택) | 소근육 | 정밀한 움직임 |

### 3.3 연령 범위
- 출생~90세 이상
- 자폐, 지적장애, 발달장애 진단에 활용

### 3.4 자폐 스펙트럼 표준 점수 (10세 이상, 무발화)
| 영역 | 평균 표준점수 |
|------|---------------|
| 의사소통 | 32.82 |
| 일상생활 | 29.94 |
| 사회화 | 33.57 |

### 3.5 AAC 적용 시사점
- 표현 언어 하위영역은 주로 구어에 초점
- AAC 사용자의 의사소통 행동 점수화 시 해석 필요
- 적응 행동 전반 평가로 AAC 개입 목표 설정 가능

## 4. 치매 인지 선별: MMSE vs MoCA

### 4.1 MMSE (Mini-Mental State Examination)

#### 개발
- **1975년** Marshal F. Folstein 개발
- 역사적으로 가장 널리 사용된 선별 도구

#### 평가 영역
```
MMSE 평가 영역 (총 30점):
├── 지남력 (10점)
│   ├── 시간 지남력 (5점)
│   └── 장소 지남력 (5점)
├── 기억력 (6점)
│   ├── 즉시 회상 (3점)
│   └── 지연 회상 (3점)
├── 주의력 및 계산 (5점)
├── 언어 (8점)
│   ├── 명명 (2점)
│   ├── 따라말하기 (1점)
│   ├── 3단계 명령 (3점)
│   ├── 읽기 (1점)
│   └── 쓰기 (1점)
└── 시공간 능력 (1점)
```

#### 절단점
- **24-30점**: 정상
- **18-23점**: 경도 인지장애
- **0-17점**: 중증 인지장애

#### 제한점
- 경도 인지장애(MCI) 민감도 낮음 (18-25%)
- 천장 효과 (ceiling effect) 존재
- 실행 기능 평가 부재

### 4.2 MoCA (Montreal Cognitive Assessment)

#### 개발
- **2005년** Ziad Nasreddine 개발
- MMSE의 한계 극복 목적

#### 평가 영역
```
MoCA 평가 영역 (총 30점):
├── 시공간/실행 기능 (5점)
│   ├── Trail Making B
│   ├── 시계 그리기
│   └── 도형 복사
├── 명명 (3점)
├── 기억력 (5점 - 지연 회상)
├── 주의력 (6점)
│   ├── 숫자 폭
│   ├── 집중력
│   └── 연속 빼기
├── 언어 (3점)
│   ├── 따라말하기
│   └── 유창성
├── 추상화 (2점)
└── 지남력 (6점)
```

#### 절단점
- **26점 이상**: 정상
- **학력 12년 미만 시 +1점 보정**

#### MoCA 우수성
| 메트릭 | MoCA | MMSE |
|--------|------|------|
| MCI 민감도 | 90-100% | 18-25% |
| 경도 알츠하이머 민감도 | 100% | 78% |
| 실행 기능 평가 | 포함 | 미포함 |
| 천장 효과 | 없음 | 있음 |

### 4.3 질환별 권장 도구
| 질환 | 권장 도구 | 이유 |
|------|-----------|------|
| 경도 인지장애 (MCI) | MoCA | 높은 민감도 |
| 전두측두엽 치매 (FTD) | MoCA | AUC 0.87 vs 0.80 |
| 파킨슨병 인지장애 | MoCA | 천장 효과 없음 |
| 중등도-중증 치매 | MMSE | 충분한 민감도 |

### 4.4 AAC 적용 시사점
- 의사소통 능력에 따라 평가 도구 수정 필요
- 비언어적 과제 비중 고려
- 인지 프로파일과 AAC 복잡도 매칭

## 5. 의사소통 평가: Communication Matrix

### 5.1 개요
**Communication Matrix**는 Charity Rowland와 Philip Schweigert가 개발한 무료 평가 도구로, 초기 의사소통 단계에 있는 모든 연령대를 대상으로 합니다.

### 5.2 7단계 의사소통 발달
| 레벨 | 단계 | 설명 |
|------|------|------|
| I | 선의도적 행동 | 반사적, 상태 반영 |
| II | 의도적 행동 | 목적 있으나 의사소통 아님 |
| III | 비관습적 의사소통 | 의도적 + 비관습적 행동 |
| IV | 관습적 의사소통 | 관습적 제스처, 발성 |
| V | 구체적 상징 | 그림, 사물 상징 |
| VI | 추상적 상징 | 수어, 단어 |
| VII | 언어 | 2-3 상징 조합 |

### 5.3 의사소통 기능
```
Communication Matrix가 평가하는 4가지 기능:
├── 거부 (Refusing)
│   └── 원하지 않는 것 거부
├── 획득 (Obtaining)
│   └── 원하는 것 요청
├── 사회적 상호작용 (Social Interaction)
│   └── 인사, 관심 끌기
└── 정보 제공 (Providing Information)
    └── 사물/사건 언급, 질문 답변
```

### 5.4 지원 양식
- **구어** (Speech)
- **제스처/바디랭귀지** (Gestures)
- **소리** (Sounds)
- **눈 응시** (Eye Gaze)
- **터치/조작** (Touch/Manipulation)
- **3D 상징** (3D Symbols)
- **2D 상징** (Pictures/Photos)
- **추상적 상징** (Abstract Symbols)
- **문자** (Written Words)

### 5.5 핵심 원리
> "Communication Matrix는 의사소통을 측정하지, 인지 능력을 측정하지 않습니다. 의사소통 부재가 지능 부재를 의미하지 않습니다."

### 5.6 AAC 적용 시사점
- AAC 사용자의 모든 의사소통 형태 수용
- 발달 단계 기반 개입 계획 수립
- 무료로 접근 가능 (communicationmatrix.org)

## 6. 인지 AAC 설계 원칙 도출

### 6.1 자폐 스펙트럼을 위한 AAC 원칙
| 인지 특성 | AAC 설계 원칙 |
|-----------|---------------|
| 시각적 강점 | 시각 기반 인터페이스 우선 |
| 예측 가능성 선호 | 일관된 레이아웃, 전환 지원 |
| 감각 민감성 | 자극 조절 옵션 |
| 특별 관심사 | 관심사 기반 어휘 개인화 |
| 일반화 어려움 | 상황별 어휘 세트 |

### 6.2 치매를 위한 AAC 원칙
| 인지 특성 | AAC 설계 원칙 |
|-----------|---------------|
| 단기 기억 손상 | 최소 항목 수, 큰 아이콘 |
| 장기 기억 보존 | 친숙한 사진/얼굴 활용 |
| 절차 기억 보존 | 일관된 인터페이스 |
| 실행 기능 저하 | 자동 컨텍스트 제안 |
| 단계별 진행 | 진행 단계별 UI 적응 |

### 6.3 공통 원칙
```
인지 AAC 공통 설계 원칙:
├── 인지 부하 최소화
│   ├── 선택지 수 제한
│   ├── 점진적 공개
│   └── 시각적 일관성
├── 개인화
│   ├── 인지 프로파일 기반 적응
│   ├── 학습 속도 적응
│   └── 관심사 반영
├── 다중 양식
│   ├── 시각 + 청각 지원
│   ├── 터치/스위치 접근
│   └── 시선 추적 옵션
└── 진행 추적
    ├── 객관적 메트릭
    ├── 개입 효과 측정
    └── 적응적 난이도 조절
```

## 7. 표준화 필요 영역

### 7.1 인지 프로파일 표준
- 기억, 주의, 언어, 실행 기능, 사회 인지 영역 정의
- 자폐/치매 특화 프로파일 확장
- JSON Schema 기반 상호운용성

### 7.2 AAC 효과 메트릭 표준
- 의사소통 빈도/성공률
- 어휘 확장 속도
- 독립성 향상도
- 인지 부하 측정

### 7.3 적응형 UI 규격
- 인지 프로파일 → UI 매개변수 매핑
- 동적 복잡도 조절 알고리즘
- 접근성 가이드라인 통합

## 9. 결론

본 리서치를 통해 다음을 확인했습니다:

1. **기존 평가 도구는 인지 영역을 다각적으로 측정**하지만, AAC 사용자를 위한 통합 프로파일 표준은 부재합니다.

2. **자폐와 치매는 서로 다른 인지 특성**을 가지며, 각각에 맞는 특화된 AAC 설계 원칙이 필요합니다.

3. **Communication Matrix**는 AAC 친화적 평가 도구로, 인지 AAC 표준의 기반으로 활용 가능합니다.

4. **WIA Cognitive AAC 표준**은 이러한 기존 도구들의 장점을 통합하면서, 상호운용 가능한 디지털 표준을 제공해야 합니다.
