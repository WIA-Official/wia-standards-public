# WIA-ROBOT-SAFETY-001 — Robot Safety (Press Brief)

**WIA Standards releases canonical Robot Safety envelope shape**

The WIA Standards Committee has published the WIA-ROBOT-SAFETY-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for robot safety hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
