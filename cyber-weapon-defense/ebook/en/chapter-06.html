Type Audience Content Timeframe

6.3 MITRE ATT&CK Framework

MITRE ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge) provides a comprehensive knowledge base of adversary behaviors based on real-world observations.

6.3.1 ATT&CK Tactics

Tactic Description Example Techniques
Reconnaissance Gathering target information Active Scanning, Search Open Websites
Initial Access Gaining entry to network Phishing, Valid Accounts
Execution Running malicious code PowerShell, Command-Line Interface
Persistence Maintaining access Registry Run Keys, Scheduled Task
Defense Evasion Avoiding detection Obfuscation, Masquerading
Credential Access Stealing credentials Credential Dumping, Keylogging
Lateral Movement Moving through network Remote Services, Internal Spearphishing
Exfiltration Stealing data Exfiltration Over C2, Alternative Protocol

6.5 Information Sharing

6.5.1 Sharing Communities

  • ISACs: Sector-specific sharing (FS-ISAC, E-ISAC, etc.)
  • ISAOs: Cross-sector sharing organizations
  • Government: CISA, FBI, NSA/CYBERCOM
  • Commercial: Vendor threat intelligence
  • Peer Networks: Trusted industry relationships

6.6 Chapter Summary

Key Takeaways

  • Threat intelligence spans strategic, operational, and tactical levels
  • The intelligence lifecycle ensures continuous improvement
  • MITRE ATT&CK provides a common language for threat behavior
  • Intelligence must be operationalized into security tools
  • Sharing communities multiply collective defense capability
