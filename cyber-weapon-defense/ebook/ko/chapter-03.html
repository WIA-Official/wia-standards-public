3.1 심층 방어

효과적인 사이버 무기 방어는 여러 계층의 보안 통제를 필요로 하며, 단일 계층의 실패가 전체 시스템을 손상시키지 않도록 보장합니다. 이 전략은 어떤 단일 보안 조치도 침투 불가능하지 않다는 것을 인정합니다.

3.1.1 7계층 방어

계층 통제 목적
1. 경계 방화벽, IDS/IPS, DDoS 완화 경계에서 알려진 위협 필터링
2. 네트워크 분할, 암호화, 모니터링 횡적 이동 제한
3. 엔드포인트 EDR, 백신, 앱 화이트리스팅 워크스테이션 및 서버 보호
4. 애플리케이션 보안 코딩, WAF, 취약점 관리 애플리케이션 익스플로잇 방지
5. 데이터 암호화, DLP, 접근 통제 민감 정보 보호
6. ID MFA, 권한 관리, 제로 트러스트 모든 접근 요청 검증
7. 운영 SIEM, SOC, 사고 대응 탐지, 대응, 복구

3.2 제로 트러스트 아키텍처

전통적인 "성과 해자" 보안은 경계 내에서 신뢰를 가정합니다. 제로 트러스트는 침해를 가정하고 소스 위치와 관계없이 모든 접근 요청을 검증합니다.

3.3 네트워크 분할

전략 구현 사용 사례
VLAN 분할 레이어 2 분리 부서 격리
방화벽 분할 레이어 3/4 필터링 보안 구역 분리
마이크로 분할 호스트 기반 방화벽, SDN 워크로드 격리
에어갭 물리적 네트워크 분리 핵심 인프라

3.5 장 요약

핵심 내용

  • 심층 방어는 공격자에게 다중 장벽 제공
  • 제로 트러스트는 침해를 가정하고 모든 접근 검증
  • 네트워크 분할이 횡적 이동 제한
  • 다층 탐지가 여러 지점에서 위협 포착
  • SOAR가 자동화된 일관된 대응 가능케 함
