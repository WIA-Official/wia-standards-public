6.1 위협 정보 소개

사이버 위협 정보(CTI)는 조직을 목표로 할 수 있는 위협에 대한 증거 기반 지식입니다. 위협 정보를 운영에 통합하면 방어자는 알려진 적대자에 대해 사전 방어할 수 있습니다.

정보 주기

계획 → 수집 → 처리 → 분석 → 배포 → 피드백 → 계획...

정보 주기는 원시 데이터를 실행 가능한 정보로 변환하는 지속적인 프로세스입니다.

6.2 정보 유형

유형 설명 청중 예시
전략적 고수준 위협 동향 및 동기 경영진, 이사회 지정학적 분석, 산업 동향
전술적 공격자 TTP 및 방법론 보안 팀, 아키텍트 MITRE ATT&CK 매핑
운영적 특정 캠페인 및 공격 SOC, IR 팀 캠페인 보고서, 표적 정보
기술적 IOC, 시그니처, 규칙 분석가, 도구 해시, IP, 도메인, YARA

6.4 MITRE ATT&CK 프레임워크

ATT&CK의 가치

MITRE ATT&CK는 적대자 행동을 설명하는 공통 언어를 제공합니다. 이를 통해 조직은 방어 범위를 특정 기법에 매핑하고 격차를 식별할 수 있습니다.

전술 목적 예시 기법
초기 접근 네트워크 진입 피싱, 취약점 익스플로잇
실행 코드 실행 PowerShell, 스크립트
지속성 접근 유지 예약 작업, 레지스트리
권한 상승 더 높은 권한 획득 익스플로잇, 토큰 조작
방어 회피 탐지 회피 난독화, 로그 삭제
자격 증명 접근 계정 정보 획득 키로깅, 덤핑
횡적 이동 환경 내 이동 RDP, SMB, WMI
데이터 유출 데이터 탈취 암호화, C2 채널

6.5.2 ISAC 참여

  • 금융 ISAC: 금융 서비스 부문 정보 공유
  • 의료 ISAC: 의료 부문 위협 정보
  • 에너지 ISAC: 전력 및 에너지 부문
  • IT ISAC: IT 인프라 및 서비스
  • KrCERT: 한국인터넷진흥원 위협 정보

6.6 정보 운영화

활동 정보 입력 출력
탐지 공학 TTP, IOC 탐지 규칙, 시그니처
위협 헌팅 행위자 TTP 가설, 헌팅 쿼리
취약점 관리 익스플로잇 정보 우선순위 패칭
사고 대응 캠페인 정보 범위 결정, 귀속
보안 아키텍처 전략적 위협 방어 투자 결정

6.7 장 요약

핵심 내용

  • CTI는 알려진 위협에 대한 사전 방어 가능케 함
  • 정보 주기가 데이터를 실행 가능 정보로 변환
  • MITRE ATT&CK가 적대자 행동의 공통 언어 제공
  • STIX/TAXII가 자동화된 정보 공유 가능케 함
  • ISAC 참여로 부문별 위협 통찰 획득
