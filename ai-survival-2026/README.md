# WIA Ai Survival 2026

> Cross-vendor data and protocol layer for Ai Survival 2026.

## Overview

WIA Ai Survival 2026 defines standardized data formats and protocol semantics.
The standard exposes four PHASE documents covering data format, API
interface, protocol, and integration concerns. Conformance follows
the three-tier model (Surface / Verified / Anchored).

## Documentation

- spec/PHASE-1-DATA-FORMAT.md
- spec/PHASE-2-API-INTERFACE.md
- spec/PHASE-3-PROTOCOL.md
- spec/PHASE-4-INTEGRATION.md

## Reference Standards

This standard cites only ALLOW sources per the WIA citation policy:
ISO/IEC, IEEE, RFC, W3C. Schemas use JSON Schema 2020-12 and OpenAPI
3.1. Signatures use Sigstore (DSSE envelope, Rekor transparency log).

## Conformance Tiers

| Tier      | Scope                                              |
|-----------|----------------------------------------------------|
| Surface   | data formats accepted; no formal audit             |
| Verified  | annual third-party audit                           |
| Anchored  | continuous evidence package per Annex G            |
