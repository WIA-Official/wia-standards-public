# 제{}장: 3D 터치 기술 상세 가이드

## 포스 터치와 압력 감지 인터페이스

본 장에서는 3D 터치 기술의 핵심 개념, 구현 방법, 그리고 실제 응용 사례를 상세하게 다룹니다.

---

## 기술 개요

3D 터치는 전통적인 2차원 터치 인터페이스에 압력이라는 세 번째 차원을 추가하여, 사용자의 터치 강도를 감지하고 이를 기반으로 다양한 상호작용을 제공하는 혁신적인 기술입니다.

### 압력 레벨 분류

**경량 터치 (Light Touch)**
- 압력 범위: 0-50gf (0-0.49 N)
- 사용자 의도: 탐색, 호버, 우발적 접촉
- 시스템 응답: 동작 없음 또는 미리보기 힌트
- 구현: 표준 정전용량 터치 감지

**일반 터치/탭 (Normal Touch)**  
- 압력 범위: 50-150gf (0.49-1.47 N)
- 사용자 의도: 표준 선택, 스크롤, 타이핑
- 시스템 응답: 표준 터치 이벤트 (탭, 스와이프 등)
- 구현: 전통적인 터치 이벤트 처리

**Peek 제스처 (가볍게 누르기)**
- 압력 범위: 150-300gf (1.47-2.94 N)
- 사용자 의도: 컨텐츠 미리보기 (확정하지 않음)
- 시스템 응답: 미리보기 오버레이, 빠른 동작
- 구현: 압력 임계값 감지 + 햅틱 피드백
- 일반적인 임계값: 200gf (기기 및 사용자 선호도에 따라 다름)

**Pop 제스처 (깊게 누르기)**
- 압력 범위: 300-500gf (2.94-4.90 N)
- 사용자 의도: 전체 컨텐츠 활성화, 강한 확정
- 시스템 응답: 전체 컨텐츠 열기, 동작 실행
- 구현: 높은 압력 임계값 + 뚜렷한 햅틱
- 일반적인 임계값: 400gf

### 센서 기술

**용량성 힘 센서 (Capacitive Force Sensor)**

용량성 힘 센서는 압력에 따라 정전용량이 변화하는 원리를 이용합니다.

**계층 구조 (하단에서 상단으로):**
1. 백라이트 유닛 (LCD) 또는 OLED 패널
2. 용량성 힘 센서 배열 (에칭된 ITO 또는 금속 트레이스)
3. 커버 글라스 (화학 강화, 0.5-0.7mm 두께)
4. 용량성 터치 센서 (상호 정전용량 그리드)
5. 발유 코팅

**주요 구성 요소:**

**1. 정전용량 센서 배열**
- 설계: 디스플레이 아래 전극 그리드
- 패턴: 직교 X-Y 그리드, 일반적으로 6×10 또는 8×12
- 재료: ITO(Indium Tin Oxide) 또는 금속 메쉬 (구리/은)
- 전극 크기: 공간 해상도를 위해 5-8mm 피치

**2. 변형 가능한 유전체**
- 목적: 힘을 정전용량 변화로 변환
- 재료: 제어된 탄성을 가진 순응성 폴리머
- 두께: 일반적으로 50-150 μm
- 압축: 힘에 따라 5-20 μm
- 유전 상수: εᵣ = 2-4 일반적

**스트레인 게이지 힘 센서 (Strain Gauge Force Sensor)**

스트레인 게이지는 기계적 변형에 따라 저항이 변화하는 원리를 이용합니다.

**원리:**
압력이 가해지면 디스플레이 어셈블리가 미세하게 변형되고, 이 변형을 스트레인 게이지가 감지합니다.

**배치:**
- 4-코너 설계: 디스플레이 각 모서리에 하나씩
- 6-센서 설계: 더 나은 공간 해상도, 중앙 감지
- 엣지 마운트: 디스플레이 주변에 센서 배치

**장점:**
- 용량성 대비 더 선형적인 힘 응답
- 더 나은 온도 안정성
- 저비용 (범용 스트레인 게이지)
- 입증된 기술

**압전 힘 센서 (Piezoelectric Force Sensor)**

압전 재료는 기계적 응력을 받으면 전하를 생성합니다.

**재료:**
- 석영 (SiO₂): d = 2.3 pC/N, 매우 안정적, 고가
- PZT (Lead Zirconate Titanate): d = 200-600 pC/N, 고감도
- PVDF (Polyvinylidene Fluoride): d = 20-30 pC/N, 유연한 필름

**장점:**
- 초고속 응답 (<1 ms)
- 높은 감도
- 넓은 동적 범위
- 자체 전원 (DC 바이어스 불필요)

### 햅틱 피드백 시스템

**선형 공진 액츄에이터 (LRA)**

전자기 액츄에이터가 기계적 공진 주파수에서 작동하여 최대 효율과 진폭을 제공합니다.

**주요 사양:**
- 공진 주파수: 일반적으로 150-250 Hz (인간 촉각 인식에 최적화)
- 가속도: 1-4 G (9.81-39.2 m/s²)
- 응답 시간: 10-20 ms (최대 진폭까지의 상승 시간)
- 소비 전력: 활성화 중 100-300 mW
- 수명: 500-1000만 사이클
- 크기: 7×7×2mm (소형), 10×10×3mm (중형), 13×11×5mm (대형)

**편심 회전 질량 (ERM)**

모터 샤프트의 중심에서 벗어난 무게가 불균형 회전력을 생성합니다.

**작동 특성:**
- 주파수 범위: 50-200 Hz (전압에 따라 다름)
- 가속도: 일반적으로 0.5-2 G
- 응답 시간: 40-80 ms (모터 스핀업 시간)
- 소비 전력: 지속적으로 60-150 mW
- 수명: 100-500만 사이클 (브러시 모터 마모)

### 교정 절차

**공장 교정 (Factory Calibration)**

**필요 장비:**
- 힘 적용 장치: 공압 또는 서보 제어 액츄에이터
- 기준 힘 센서: 0.1% 이상의 정확도 (예: 로드셀)
- 환경 챔버: 온도 제어 ±1°C
- 자동 테스트 고정구: 여러 화면 위치에 힘 적용

**절차:**

**1. 기준선 측정 (Zero Force)**
힘이 가해지지 않은 상태에서 모든 힘 센서 채널 값 기록
- 주변 조건: 25°C, 50% 상대 습도
- 노이즈 감소를 위해 100개 샘플 평균

**2. 다중 지점 교정**
표준화된 지점에서 교정 힘 적용

**일반적인 교정 힘:**
- 50 gf (가벼운 터치 기준)
- 150 gf (일반 탭 기준)
- 250 gf (peek 임계값 기준)
- 400 gf (pop 임계값 기준)
- 500 gf (최대 힘 기준)

**3. 공간 교정**
여러 화면 위치에서 교정 반복

**테스트 위치:**
- 중앙
- 4개 모서리
- 각 가장자리의 중간 지점
- 고정밀 장치를 위한 추가 그리드 포인트

**4. 온도 교정**
여러 온도에서 프로세스 반복

**온도 지점:**
- 0°C (추운 환경)
- 25°C (실온 기준선)
- 40°C (따뜻한 주머니/손)
- 60°C (뜨거운 차량 대시보드 - 극한)

### 제스처 인식

**Peek 제스처 구현**

사용자가 컨텐츠를 미리 볼 수 있도록 하는 가벼운 압력 제스처입니다.

**동작:**
1. 사용자가 항목 위에서 압력 적용 시작
2. 압력이 Peek 임계값(~200gf)에 도달
3. 햅틱 피드백 제공 (가벼운 탭)
4. 미리보기 오버레이 표시
5. 사용자가 압력 유지하면 미리보기 유지
6. 압력 해제 시 미리보기 숨김

**Pop 제스처 구현**

더 강한 압력으로 전체 컨텐츠를 활성화하는 제스처입니다.

**동작:**
1. 사용자가 Peek 상태에서 압력 증가
2. 압력이 Pop 임계값(~400gf)에 도달
3. 강한 햅틱 피드백 제공 (강한 탭)
4. 전체 컨텐츠 열기 또는 동작 실행
5. 미리보기 오버레이 닫기

### 성능 메트릭

**힘 감지 정확도:**
- 절대 정확도: 전체 스케일의 ±2-5% (예: 200gf에서 ±10gf)
- 반복성: ±1-2% (동일한 힘을 여러 번 적용)
- 선형성: ±2-3% (이상적인 직선에서의 편차)

**힘 해상도:**
- 최소 감지 가능한 힘: 0.1-1 gf
- 힘 양자화: 레벨당 0.5-2 gf
- 동적 범위: 60-70 dB (최대 힘 / 최소 힘)

**지연 시간:**
- 힘 감지 지연: 힘 적용에서 측정까지 <8 ms
- 총 응답 지연: <20 ms (햅틱 피드백 포함)

### 응용 분야

**스마트폰 및 태블릿:**
- 홈 화면 아이콘에서 빠른 작업
- Peek 제스처로 컨텐츠 미리보기
- Pop 제스처로 전체 컨텐츠 열기
- 텍스트 편집에서 커서 제어
- 압력 감도를 활용한 드로잉 앱

**스마트워치 및 웨어러블:**
- 압력을 통한 컨텍스트 메뉴 접근
- 화면을 가리지 않는 상호작용
- 무음 햅틱 알림
- 접근성 기능

**노트북 및 트랙패드:**
- Force Touch 트랙패드 (MacBook)
- 압력 감지 드로잉 태블릿
- 가변 클릭 힘 사용자 정의
- 가상 클릭을 위한 햅틱 피드백

**게임:**
- 적응형 트리거 (PlayStation 5)
- 압력 감지 버튼
- 포스 피드백 스티어링 휠
- 힘 감지 기능이 있는 VR 컨트롤러

**자동차:**
- 버튼을 대체하는 압력 감지 터치스크린
- "eyes-on-road" 상호작용을 위한 햅틱 피드백
- 압력 감지 스티어링 휠 컨트롤
- 힘 기반 제스처 인식

**산업 및 의료:**
- 수술 로봇 포스 피드백
- 조립을 위한 정밀 힘 제어
- 햅틱 훈련 시뮬레이터
- 시각 장애인을 위한 접근성 장치

### 미래 기술

**초음파 햅틱 (Ultrasonic Haptics):**
초음파 위상 배열이 공중에 촉각 감각을 생성합니다.
- 물리적 접촉 불필요
- 응용: 자동차 HMI, 공공 키오스크, AR 디스플레이

**변형 가능한 디스플레이:**
폴더블 및 롤러블 디스플레이에는 유연한 힘 센서가 필요합니다.
- PVDF 압전 필름 (유연하고 신축 가능)
- 유연한 기판에 인쇄된 스트레인 게이지 배열
- 광섬유 힘 감지

**AI 강화 힘 감지:**
여러 센서 모달리티에서 힘을 예측하기 위해 신경망을 훈련합니다:
- 정전용량 터치 (위치, 영역)
- 가속도계 (충격 서명)
- 자이로스코프 (장치 기울기)
- 역사적 힘 패턴 (사용자별)

### 통합 프레임워크

**iOS 통합:**
UIFeedbackGenerator를 사용한 햅틱 피드백과 UITouch.force를 통한 힘 감지를 제공합니다.

**Android 통합:**
MotionEvent.getPressure()를 통해 압력 데이터에 접근하고 VibrationEffect로 햅틱을 생성합니다.

**웹 통합:**
Force Touch API (제한적 브라우저 지원) 또는 터치 영역 기반 압력 추정을 사용합니다.

### 센서 기술

**용량성 힘 센서 (Capacitive Force Sensor)**

용량성 힘 센서는 압력에 따라 정전용량이 변화하는 원리를 이용합니다.

**계층 구조 (하단에서 상단으로):**
1. 백라이트 유닛 (LCD) 또는 OLED 패널
2. 용량성 힘 센서 배열 (에칭된 ITO 또는 금속 트레이스)
3. 커버 글라스 (화학 강화, 0.5-0.7mm 두께)
4. 용량성 터치 센서 (상호 정전용량 그리드)
5. 발유 코팅

**주요 구성 요소:**

**1. 정전용량 센서 배열**
- 설계: 디스플레이 아래 전극 그리드
- 패턴: 직교 X-Y 그리드, 일반적으로 6×10 또는 8×12
- 재료: ITO(Indium Tin Oxide) 또는 금속 메쉬 (구리/은)
- 전극 크기: 공간 해상도를 위해 5-8mm 피치

**2. 변형 가능한 유전체**
- 목적: 힘을 정전용량 변화로 변환
- 재료: 제어된 탄성을 가진 순응성 폴리머
- 두께: 일반적으로 50-150 μm
- 압축: 힘에 따라 5-20 μm
- 유전 상수: εᵣ = 2-4 일반적

**스트레인 게이지 힘 센서 (Strain Gauge Force Sensor)**

스트레인 게이지는 기계적 변형에 따라 저항이 변화하는 원리를 이용합니다.

**원리:**
압력이 가해지면 디스플레이 어셈블리가 미세하게 변형되고, 이 변형을 스트레인 게이지가 감지합니다.

**배치:**
- 4-코너 설계: 디스플레이 각 모서리에 하나씩
- 6-센서 설계: 더 나은 공간 해상도, 중앙 감지
- 엣지 마운트: 디스플레이 주변에 센서 배치

**장점:**
- 용량성 대비 더 선형적인 힘 응답
- 더 나은 온도 안정성
- 저비용 (범용 스트레인 게이지)
- 입증된 기술

**압전 힘 센서 (Piezoelectric Force Sensor)**

압전 재료는 기계적 응력을 받으면 전하를 생성합니다.

**재료:**
- 석영 (SiO₂): d = 2.3 pC/N, 매우 안정적, 고가
- PZT (Lead Zirconate Titanate): d = 200-600 pC/N, 고감도
- PVDF (Polyvinylidene Fluoride): d = 20-30 pC/N, 유연한 필름

**장점:**
- 초고속 응답 (<1 ms)
- 높은 감도
- 넓은 동적 범위
- 자체 전원 (DC 바이어스 불필요)

### 햅틱 피드백 시스템

**선형 공진 액츄에이터 (LRA)**

전자기 액츄에이터가 기계적 공진 주파수에서 작동하여 최대 효율과 진폭을 제공합니다.

**주요 사양:**
- 공진 주파수: 일반적으로 150-250 Hz (인간 촉각 인식에 최적화)
- 가속도: 1-4 G (9.81-39.2 m/s²)
- 응답 시간: 10-20 ms (최대 진폭까지의 상승 시간)
- 소비 전력: 활성화 중 100-300 mW
- 수명: 500-1000만 사이클
- 크기: 7×7×2mm (소형), 10×10×3mm (중형), 13×11×5mm (대형)

**편심 회전 질량 (ERM)**

모터 샤프트의 중심에서 벗어난 무게가 불균형 회전력을 생성합니다.

**작동 특성:**
- 주파수 범위: 50-200 Hz (전압에 따라 다름)
- 가속도: 일반적으로 0.5-2 G
- 응답 시간: 40-80 ms (모터 스핀업 시간)
- 소비 전력: 지속적으로 60-150 mW
- 수명: 100-500만 사이클 (브러시 모터 마모)

### 교정 절차

**공장 교정 (Factory Calibration)**

**필요 장비:**
- 힘 적용 장치: 공압 또는 서보 제어 액츄에이터
- 기준 힘 센서: 0.1% 이상의 정확도 (예: 로드셀)
- 환경 챔버: 온도 제어 ±1°C
- 자동 테스트 고정구: 여러 화면 위치에 힘 적용

**절차:**

**1. 기준선 측정 (Zero Force)**
힘이 가해지지 않은 상태에서 모든 힘 센서 채널 값 기록
- 주변 조건: 25°C, 50% 상대 습도
- 노이즈 감소를 위해 100개 샘플 평균

**2. 다중 지점 교정**
표준화된 지점에서 교정 힘 적용

**일반적인 교정 힘:**
- 50 gf (가벼운 터치 기준)
- 150 gf (일반 탭 기준)
- 250 gf (peek 임계값 기준)
- 400 gf (pop 임계값 기준)
- 500 gf (최대 힘 기준)

**3. 공간 교정**
여러 화면 위치에서 교정 반복

**테스트 위치:**
- 중앙
- 4개 모서리
- 각 가장자리의 중간 지점
- 고정밀 장치를 위한 추가 그리드 포인트

**4. 온도 교정**
여러 온도에서 프로세스 반복

**온도 지점:**
- 0°C (추운 환경)
- 25°C (실온 기준선)
- 40°C (따뜻한 주머니/손)
- 60°C (뜨거운 차량 대시보드 - 극한)

### 제스처 인식

**Peek 제스처 구현**

사용자가 컨텐츠를 미리 볼 수 있도록 하는 가벼운 압력 제스처입니다.

**동작:**
1. 사용자가 항목 위에서 압력 적용 시작
2. 압력이 Peek 임계값(~200gf)에 도달
3. 햅틱 피드백 제공 (가벼운 탭)
4. 미리보기 오버레이 표시
5. 사용자가 압력 유지하면 미리보기 유지
6. 압력 해제 시 미리보기 숨김

**Pop 제스처 구현**

더 강한 압력으로 전체 컨텐츠를 활성화하는 제스처입니다.

**동작:**
1. 사용자가 Peek 상태에서 압력 증가
2. 압력이 Pop 임계값(~400gf)에 도달
3. 강한 햅틱 피드백 제공 (강한 탭)
4. 전체 컨텐츠 열기 또는 동작 실행
5. 미리보기 오버레이 닫기

### 성능 메트릭

**힘 감지 정확도:**
- 절대 정확도: 전체 스케일의 ±2-5% (예: 200gf에서 ±10gf)
- 반복성: ±1-2% (동일한 힘을 여러 번 적용)
- 선형성: ±2-3% (이상적인 직선에서의 편차)

**힘 해상도:**
- 최소 감지 가능한 힘: 0.1-1 gf
- 힘 양자화: 레벨당 0.5-2 gf
- 동적 범위: 60-70 dB (최대 힘 / 최소 힘)

**지연 시간:**
- 힘 감지 지연: 힘 적용에서 측정까지 <8 ms
- 총 응답 지연: <20 ms (햅틱 피드백 포함)

## 기술 개요

3D 터치는 전통적인 2차원 터치 인터페이스에 압력이라는 세 번째 차원을 추가하여, 사용자의 터치 강도를 감지하고 이를 기반으로 다양한 상호작용을 제공하는 혁신적인 기술입니다.

### 압력 레벨 분류

**경량 터치 (Light Touch)**
- 압력 범위: 0-50gf (0-0.49 N)
- 사용자 의도: 탐색, 호버, 우발적 접촉
- 시스템 응답: 동작 없음 또는 미리보기 힌트
- 구현: 표준 정전용량 터치 감지

**일반 터치/탭 (Normal Touch)**  
- 압력 범위: 50-150gf (0.49-1.47 N)
- 사용자 의도: 표준 선택, 스크롤, 타이핑
- 시스템 응답: 표준 터치 이벤트 (탭, 스와이프 등)
- 구현: 전통적인 터치 이벤트 처리

**Peek 제스처 (가볍게 누르기)**
- 압력 범위: 150-300gf (1.47-2.94 N)
- 사용자 의도: 컨텐츠 미리보기 (확정하지 않음)
- 시스템 응답: 미리보기 오버레이, 빠른 동작
- 구현: 압력 임계값 감지 + 햅틱 피드백
- 일반적인 임계값: 200gf (기기 및 사용자 선호도에 따라 다름)

**Pop 제스처 (깊게 누르기)**
- 압력 범위: 300-500gf (2.94-4.90 N)
- 사용자 의도: 전체 컨텐츠 활성화, 강한 확정
- 시스템 응답: 전체 컨텐츠 열기, 동작 실행
- 구현: 높은 압력 임계값 + 뚜렷한 햅틱
- 일반적인 임계값: 400gf

- 입증된 기술

**압전 힘 센서 (Piezoelectric Force Sensor)**

압전 재료는 기계적 응력을 받으면 전하를 생성합니다.

**재료:**
- 석영 (SiO₂): d = 2.3 pC/N, 매우 안정적, 고가
- PZT (Lead Zirconate Titanate): d = 200-600 pC/N, 고감도
- PVDF (Polyvinylidene Fluoride): d = 20-30 pC/N, 유연한 필름

**장점:**
- 초고속 응답 (<1 ms)
- 높은 감도
- 넓은 동적 범위
- 자체 전원 (DC 바이어스 불필요)

### 햅틱 피드백 시스템

**선형 공진 액츄에이터 (LRA)**

전자기 액츄에이터가 기계적 공진 주파수에서 작동하여 최대 효율과 진폭을 제공합니다.

**주요 사양:**
- 공진 주파수: 일반적으로 150-250 Hz (인간 촉각 인식에 최적화)
- 가속도: 1-4 G (9.81-39.2 m/s²)
- 응답 시간: 10-20 ms (최대 진폭까지의 상승 시간)
- 소비 전력: 활성화 중 100-300 mW
- 수명: 500-1000만 사이클
- 크기: 7×7×2mm (소형), 10×10×3mm (중형), 13×11×5mm (대형)

**편심 회전 질량 (ERM)**

모터 샤프트의 중심에서 벗어난 무게가 불균형 회전력을 생성합니다.

**작동 특성:**
- 주파수 범위: 50-200 Hz (전압에 따라 다름)
- 가속도: 일반적으로 0.5-2 G
- 응답 시간: 40-80 ms (모터 스핀업 시간)
- 소비 전력: 지속적으로 60-150 mW
- 수명: 100-500만 사이클 (브러시 모터 마모)

### 교정 절차

**공장 교정 (Factory Calibration)**

**필요 장비:**
- 힘 적용 장치: 공압 또는 서보 제어 액츄에이터
- 기준 힘 센서: 0.1% 이상의 정확도 (예: 로드셀)
- 환경 챔버: 온도 제어 ±1°C
- 자동 테스트 고정구: 여러 화면 위치에 힘 적용

**절차:**

**1. 기준선 측정 (Zero Force)**
힘이 가해지지 않은 상태에서 모든 힘 센서 채널 값 기록
- 주변 조건: 25°C, 50% 상대 습도
- 노이즈 감소를 위해 100개 샘플 평균

**2. 다중 지점 교정**
표준화된 지점에서 교정 힘 적용

**일반적인 교정 힘:**
- 50 gf (가벼운 터치 기준)
- 150 gf (일반 탭 기준)
- 250 gf (peek 임계값 기준)
- 400 gf (pop 임계값 기준)
- 500 gf (최대 힘 기준)

**3. 공간 교정**
여러 화면 위치에서 교정 반복

**테스트 위치:**
- 중앙
- 4개 모서리
- 각 가장자리의 중간 지점
- 고정밀 장치를 위한 추가 그리드 포인트

**4. 온도 교정**
여러 온도에서 프로세스 반복

**온도 지점:**
- 0°C (추운 환경)
- 25°C (실온 기준선)
- 40°C (따뜻한 주머니/손)
- 60°C (뜨거운 차량 대시보드 - 극한)

### 제스처 인식

**Peek 제스처 구현**

사용자가 컨텐츠를 미리 볼 수 있도록 하는 가벼운 압력 제스처입니다.

**동작:**
1. 사용자가 항목 위에서 압력 적용 시작
2. 압력이 Peek 임계값(~200gf)에 도달
3. 햅틱 피드백 제공 (가벼운 탭)
4. 미리보기 오버레이 표시
5. 사용자가 압력 유지하면 미리보기 유지
6. 압력 해제 시 미리보기 숨김

**Pop 제스처 구현**

더 강한 압력으로 전체 컨텐츠를 활성화하는 제스처입니다.

**동작:**
1. 사용자가 Peek 상태에서 압력 증가
2. 압력이 Pop 임계값(~400gf)에 도달
3. 강한 햅틱 피드백 제공 (강한 탭)
4. 전체 컨텐츠 열기 또는 동작 실행
5. 미리보기 오버레이 닫기

### 성능 메트릭

**힘 감지 정확도:**
- 절대 정확도: 전체 스케일의 ±2-5% (예: 200gf에서 ±10gf)
- 반복성: ±1-2% (동일한 힘을 여러 번 적용)
- 선형성: ±2-3% (이상적인 직선에서의 편차)

**힘 해상도:**
- 최소 감지 가능한 힘: 0.1-1 gf
- 힘 양자화: 레벨당 0.5-2 gf
- 동적 범위: 60-70 dB (최대 힘 / 최소 힘)

**지연 시간:**
- 힘 감지 지연: 힘 적용에서 측정까지 <8 ms
- 총 응답 지연: <20 ms (햅틱 피드백 포함)

### 미래 기술

**초음파 햅틱 (Ultrasonic Haptics):**
초음파 위상 배열이 공중에 촉각 감각을 생성합니다.
- 물리적 접촉 불필요
- 응용: 자동차 HMI, 공공 키오스크, AR 디스플레이

**변형 가능한 디스플레이:**
폴더블 및 롤러블 디스플레이에는 유연한 힘 센서가 필요합니다.
- PVDF 압전 필름 (유연하고 신축 가능)
- 유연한 기판에 인쇄된 스트레인 게이지 배열
- 광섬유 힘 감지

**AI 강화 힘 감지:**
여러 센서 모달리티에서 힘을 예측하기 위해 신경망을 훈련합니다:
- 정전용량 터치 (위치, 영역)
- 가속도계 (충격 서명)
- 자이로스코프 (장치 기울기)
- 역사적 힘 패턴 (사용자별)

### 통합 프레임워크

**iOS 통합:**
UIFeedbackGenerator를 사용한 햅틱 피드백과 UITouch.force를 통한 힘 감지를 제공합니다.

**Android 통합:**
MotionEvent.getPressure()를 통해 압력 데이터에 접근하고 VibrationEffect로 햅틱을 생성합니다.

**웹 통합:**
Force Touch API (제한적 브라우저 지원) 또는 터치 영역 기반 압력 추정을 사용합니다.
