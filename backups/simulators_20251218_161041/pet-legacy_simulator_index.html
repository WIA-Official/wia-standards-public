API Examples

Create Legacy

POST /api/v1/legacy
Authorization: Bearer {token}
Content-Type: application/json

{
  "petId": "pet-001",
  "petInfo": {
    "name": "Buddy",
    "species": "Dog",
    "breed": "Golden Retriever",
    "birthDate": "2015-03-15"
  }
}

Response: 201 Created
{
  "legacyId": "legacy-12345",
  "status": "PLANNING",
  "createdAt": "2024-01-01T10:00:00Z"
}

Activate Memorial

POST /api/v1/legacy/legacy-12345/activate
Authorization: Bearer {token}
Content-Type: application/json

{
  "passingDate": "2024-01-10",
  "notifyFamily": true
}

Response: 200 OK
{
  "status": "ACTIVE",
  "memorialUrl": "https://memorial.wia.org/buddy-12345",
  "activatedAt": "2024-01-10T15:00:00Z"
}
