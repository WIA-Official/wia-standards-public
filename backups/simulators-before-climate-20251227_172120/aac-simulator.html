Supported Sensor Types

WIA AAC Standard supports all major assistive input devices with unified signal format.

👁️

Eye Tracker

Gaze-based input using infrared eye tracking. Supports dwell selection and blink activation.

Tobii, EyeTech, LC Technologies
🔘

Switch Input

Single or multiple switch scanning. Supports button, sip-puff, proximity switches.

AbleNet, Enabling Devices
💪

EMG (Muscle Sensor)

Electromyography-based input detecting muscle movements. Ideal for facial muscle control.

Ottobock, NeuroSky
🧠

EEG/BCI (Brain Interface)

Brain-Computer Interface using EEG signals. For users with severe motor impairments.

Emotiv, OpenBCI, g.tec
💨

Breath Sensor

Sip-and-puff breath detection. Differentiates between inhale/exhale intensity.

Origin Instruments, QuadJoy
🎯

Head Movement

Head tracking using camera or accelerometer. Maps head position to cursor movement.

Quha Zono, HeadMouse
