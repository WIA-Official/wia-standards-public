제3장

콘텐츠 설명자 및 경고 라벨

이 장에서는: 콘텐츠 설명자의 분류체계, 시각적 경고 시스템 및 라벨링 표준을 탐구합니다. 부모에게 콘텐츠 특성을 명확하게 전달하고 정보를 제공하면서도 압도하지 않는 효과적인 경고 시스템을 구현하는 방법을 배우게 됩니다.

3.1 콘텐츠 설명자의 목적

연령 등급은 최소 적절 연령에 대한 빠른 참조를 제공하지만, 콘텐츠 설명자는 콘텐츠가 등급을 받은 이유에 대한 자세한 정보를 제공합니다. 이 세부 정보는 부모가 가족 가치, 개별 아동의 민감성 및 특정 우려 사항을 기반으로 미묘한 결정을 내릴 수 있도록 합니다.

연구에 따르면 부모는 설명자를 매우 중요하게 생각합니다. 2023년 Common Sense Media 연구에 따르면 부모의 87%가 아동이 새로운 콘텐츠에 액세스하기 전에 콘텐츠 설명자를 정기적으로 확인하고, 73%는 연령 등급이 적절해 보여도 설명자 정보를 기반으로 결정을 변경했습니다.

설명자 vs. 등급: 상호 보완 시스템

WIA-12(10세 이상) 등급을 받은 두 게임을 고려해 보세요. 게임 A는 판타지 폭력과 가벼운 언어를 포함합니다. 게임 B는 전쟁의 현실적인 묘사, 피 효과 및 상실 주제를 포함합니다. 연령 등급은 동일하지만 콘텐츠 특성은 크게 다릅니다. 설명자는 이러한 차이를 전달하여 부모가 자녀의 특정 민감성과 폭력 묘사에 대한 가족의 가치를 기반으로 선택할 수 있도록 합니다.

3.2 WIA-CHILD-003 설명자 분류체계

WIA-CHILD-003는 정확한 콘텐츠 특성화를 제공하는 여러 하위 설명자가 있는 여섯 가지 주요 범주로 구성된 포괄적인 설명자 분류체계를 정의합니다.

상세 설명자 정의

각 설명자에는 콘텐츠 평가자와 자동화 시스템 전반에서 일관된 적용을 보장하기 위한 정확한 정의 기준이 있습니다:

폭력 설명자

  • 판타지 폭력: 만화 또는 판타지 설정의 비현실적인 폭력; 최소한의 부상 세부 사항; 캐릭터가 빠르게 회복; 슈퍼히어로 전투, 만화 슬랩스틱 포함
  • 현실적 폭력: 현실적인 결과가 묘사된 폭력; 부상 표시; 현실 상황의 맥락; 범죄, 전쟁, 격투 스포츠 포함
  • 피: 보이는 피 묘사; 빨간색 또는 양식화될 수 있음; 소량에서 상당량까지 다양
  • 고어: 심각한 부상의 그래픽 묘사; 절단; 극도의 피/내장; 일반적으로 WIA-18 등급 보장
  • 무기: 무기 묘사 또는 사용; 총, 칼, 폭발물; 맥락 중요 (교육 대 미화)

언어 설명자

  • 가벼운 언어: 가벼운 욕설의 드문 사용; "damn", "hell", 종교적 감탄사; 화장실 유머
  • 강한 언어: 중간에서 빈번한 욕설 사용; 대화의 성적 언급; 경멸적 용어
  • 저속한 유머: 화장실 유머; 신체 기능 농담; 유머러스하게 제시된 성적 암시
  • 혐오 표현: 인종, 종교, 성별, 성적 지향, 장애를 표적으로 하는 비방; 맥락에 관계없이 항상 플래그 지정

성적 콘텐츠 설명자

  • 암시적 주제: 노골적인 콘텐츠 없는 로맨틱 주제; 시시덕거림; 가벼운 암시; 손 잡기, 포옹
  • 부분 노출: 속옷 또는 부분 옷을 벗은 캐릭터; 비성적(해변 장면) 또는 암시적일 수 있음
  • 성적 콘텐츠: 성행위 묘사; 키스/포옹에서 노골적인 콘텐츠까지; 맥락 및 세부 수준이 연령 등급 결정
  • 성폭력: 강압 또는 폭력을 포함하는 성적 콘텐츠; 자동으로 최대 연령 제한 및 법 집행 통지 트리거

// TypeScript: 콘텐츠 경고 라벨 컴포넌트
interface ContentWarning {
  rating: string;
  descriptors: string[];
  detailedExplanation: string;
  thumbnailUrl?: string;
}

function generateWarningLabel(warning: ContentWarning): string {
  const descriptorBadges = warning.descriptors
    .map(desc => {
      const category = categorizeDescriptor(desc);
      const colorClass = descriptorColorMap[category];
      return `${desc}`;
    })
    .join(' ');

  return `
    

      
⚠

      
콘텐츠 경고


      

        ${warning.rating}
        ${extractAge(warning.rating)}세 이상 권장
      


      

        
이 콘텐츠는 다음을 포함합니다:

        

          ${descriptorBadges}
        

      


      

        
${warning.detailedExplanation}

      


      

        
        
      


      

        부모님: 대시보드에서 콘텐츠 제한을 구성하세요
      

    

  `;
}

// 설명자 분류
function categorizeDescriptor(descriptor: string): string {
  const categories = {
    violence: ['fantasy violence', 'realistic violence', 'blood', 'gore', 'weapons'],
    language: ['mild language', 'strong language', 'crude humor', 'profanity'],
    sexual: ['suggestive themes', 'partial nudity', 'sexual content'],
    substance: ['alcohol', 'drug', 'tobacco', 'substance use'],
    fear: ['fear', 'scary', 'horror', 'disturbing'],
    theme: ['mature themes', 'death', 'mental health', 'discrimination']
  };

  const lowerDesc = descriptor.toLowerCase();
  for (const [category, keywords] of Object.entries(categories)) {
    if (keywords.some(keyword => lowerDesc.includes(keyword))) {
      return category;
    }
  }
  return 'theme'; // 기본값
}

const descriptorColorMap = {
  violence: 'desc-violence',
  language: 'desc-language',
  sexual: 'desc-sexual',
  substance: 'desc-substance',
  fear: 'desc-fear',
  theme: 'desc-theme'
};

// 사용 예제

3.4 동적 설명자 생성

사용자 생성 또는 동적으로 변경되는 콘텐츠가 있는 플랫폼의 경우 WIA-CHILD-003는 실시간으로 콘텐츠를 분석하고 적절한 설명자를 생성하는 AI 기반 설명자 생성을 제공합니다.

// AI 기반 설명자 생성
import { ContentAnalyzer } from '@wia/content-analysis';

const analyzer = new ContentAnalyzer({
  models: {
    violence: 'wia-violence-v2.1',
    language: 'wia-language-v2.0',
    nudity: 'wia-nudity-v2.3',
    themes: 'wia-themes-v1.8'
  },
  confidenceThreshold: 0.85
});

async function analyzeAndDescribe(content: ContentBlob): Promise {
  const analysis = await analyzer.analyzeMultiModal(content);

  const descriptors: string[] = [];

  // 폭력 설명자
  if (analysis.violence.detected) {
    if (analysis.violence.type === 'fantasy' && analysis.violence.intensity < 0.5) {
      descriptors.push('Fantasy Violence');
    } else if (analysis.violence.type === 'realistic') {
      descriptors.push('Realistic Violence');
    }

    if (analysis.violence.blood > 0.3) descriptors.push('Blood');
    if (analysis.violence.gore > 0.5) descriptors.push('Gore');
    if (analysis.violence.weapons) descriptors.push('Weapons');
  }

  // 언어 설명자
  if (analysis.language.profanityScore > 0.3) {
    if (analysis.language.profanityScore < 0.6) {
      descriptors.push('Mild Language');
    } else {
      descriptors.push('Strong Language');
    }
  }
  if (analysis.language.crudeHumor > 0.4) {
    descriptors.push('Crude Humor');
  }

  // 성적 콘텐츠 설명자
  if (analysis.nudity.detected) {
    if (analysis.nudity.level === 'partial') {
      descriptors.push('Partial Nudity');
    } else if (analysis.nudity.level === 'full') {
      descriptors.push('Nudity');
    }
  }
  if (analysis.sexual.suggestive > 0.4) {
    descriptors.push('Suggestive Themes');
  }
  if (analysis.sexual.explicit > 0.3) {
    descriptors.push('Sexual Content');
  }

  // 약물 설명자
  if (analysis.substances.alcohol) descriptors.push('Alcohol Reference');
  if (analysis.substances.drugs) descriptors.push('Drug Reference');
  if (analysis.substances.tobacco) descriptors.push('Tobacco Use');

  // 공포/호러 설명자
  if (analysis.fear.intensity > 0.4) {
    if (analysis.fear.intensity < 0.6) {
      descriptors.push('Mild Fear');
    } else if (analysis.fear.intensity < 0.8) {
      descriptors.push('Scary Imagery');
    } else {
      descriptors.push('Horror Themes');
    }
  }

  // 주제 설명자
  for (const theme of analysis.themes.detected) {
    if (theme.confidence > 0.7) {
      descriptors.push(formatThemeDescriptor(theme.name));
    }
  }

  return descriptors;
}

// 라이브 스트림 콘텐츠 사용
const streamAnalyzer = new LiveStreamAnalyzer();
streamAnalyzer.on('contentChange', async (segment) => {
  const descriptors = await analyzeAndDescribe(segment);
  updateContentWarning(descriptors);
});

3.5 국제화 및 현지화

콘텐츠 설명자는 문화적으로 적절해야 하며 99개 이상의 지원 언어로 정확하게 번역되어야 합니다. WIA-CHILD-003는 설명자 의미가 문화 전반에서 일관되게 유지되면서 폭력, 언어, 노출 및 종교적 콘텐츠에 대한 지역별 콘텐츠 민감도 차이를 존중하는 현지화 지침을 제공합니다.

문화적 민감도 조정

문화마다 콘텐츠 유형에 대한 민감도가 다릅니다. 한 문화에서 가벼운 것으로 간주될 수 있는 것이 다른 문화에서는 불쾌할 수 있습니다. WIA 시스템은 이러한 차이를 설명합니다:

  • 언어 욕설: 문화에 따라 크게 다름; 종교적 감탄사는 일부 문화에서는 불쾌하지만 다른 문화에서는 가벼움
  • 노출 및 성적 콘텐츠: 일부 문화는 예술적 노출을 받아들이지만 성적 콘텐츠는 금지; 다른 문화는 이를 반대로 함
  • 폭력 맥락: 만화 폭력은 전 세계적으로 수용 가능하지만 현실적인 폭력 임계값은 지역에 따라 다름
  • 약물 묘사: 알코올은 이슬람 문화와 서양 문화에서 다르게 인식; 담배는 전 세계적으로 점점 더 낙인 찍힘
  • 종교 콘텐츠: 종교 인물 또는 신성한 상징의 묘사는 일부 문화에서 매우 민감
// 현지화된 설명자 구성
const descriptorLocalization = {

    'violence': '폭력성',
    'nudity': '노출'
  },
  'ja-JP': {
    'mild-language': '軽い言語表現',
    'violence': '暴力表現',

  },

function getLocalizedDescriptor(descriptor: string, locale: string): string {
  return descriptorLocalization[locale]?.[descriptor] || descriptor;
}

// 지역 민감도 조정
const regionalSensitivity = {
  'US': { violence: 1.0, language: 0.8, nudity: 1.2, religious: 0.7 },
  'EU': { violence: 0.9, language: 0.7, nudity: 0.8, religious: 0.8 },
  'MENA': { violence: 0.9, language: 1.1, nudity: 1.5, religious: 1.5 },
  'ASIA': { violence: 1.0, language: 0.9, nudity: 1.3, religious: 1.2 }
};

function adjustForRegion(baseRating: number, category: string, region: string): number {
  const sensitivity = regionalSensitivity[region][category] || 1.0;
  return Math.ceil(baseRating * sensitivity);
}

3.6 부모 대시보드 통합

콘텐츠 설명자는 부모 통제 대시보드와 통합되어 부모가 세부적인 기본 설정을 지정할 수 있습니다. 연령 등급만으로 차단하는 대신 부모는 자녀에게 허용 가능한 설명자를 지정할 수 있습니다.

예제: 세부적인 부모 통제

// 설명자 기반 부모 통제
interface DescriptorPreferences {
  violence: {
    fantasyViolence: 'allow' | 'warn' | 'block';
    realisticViolence: 'allow' | 'warn' | 'block';
    blood: 'allow' | 'warn' | 'block';
    gore: 'allow' | 'warn' | 'block';
  };
  language: {
    mildLanguage: 'allow' | 'warn' | 'block';
    strongLanguage: 'allow' | 'warn' | 'block';
    crudeHumor: 'allow' | 'warn' | 'block';
  };
  sexual: {
    suggestiveThemes: 'allow' | 'warn' | 'block';
    partialNudity: 'allow' | 'warn' | 'block';
    sexualContent: 'allow' | 'warn' | 'block';
  };
  customRules: CustomRule[];
}

interface CustomRule {
  name: string;
  condition: (descriptors: string[]) => boolean;
  action: 'allow' | 'warn' | 'block';
  reason: string;
}

// 부모 구성 예제
const parentPrefs: DescriptorPreferences = {
  violence: {
    fantasyViolence: 'allow',    // 만화 폭력 허용
    realisticViolence: 'warn',   // 사례별 검토 원함
    blood: 'block',              // 피 묘사 금지
    gore: 'block'                // 고어 절대 금지
  },
  language: {
    mildLanguage: 'allow',       // 약간의 가벼운 언어 허용
    strongLanguage: 'block',     // 강한 욕설 금지
    crudeHumor: 'warn'           // 화장실 유머 검토
  },
  sexual: {
    suggestiveThemes: 'allow',   // 로맨스 주제 허용
    partialNudity: 'block',      // 노출 금지
    sexualContent: 'block'       // 성적 콘텐츠 금지
  },
  customRules: [
    {
      name: '교육 예외',
      condition: (descriptors) => descriptors.includes('Educational Context'),
      action: 'allow',
      reason: '교육 콘텐츠는 더 유연함'
    },
    {
      name: '호러 금지',
      condition: (descriptors) => descriptors.some(d => d.includes('Horror')),
      action: 'block',
      reason: '아이가 무서운 콘텐츠로 악몽을 꿈'
    }
  ]
};

function evaluateContent(
  descriptors: string[],
  preferences: DescriptorPreferences
): 'allow' | 'warn' | 'block' {
  // 먼저 사용자 정의 규칙 확인
  for (const rule of preferences.customRules) {
    if (rule.condition(descriptors)) {
      return rule.action;
    }
  }

  // 각 설명자 범주 확인
  let worstAction: 'allow' | 'warn' | 'block' = 'allow';

  for (const descriptor of descriptors) {
    const action = getDescriptorAction(descriptor, preferences);
    if (action === 'block') return 'block';
    if (action === 'warn') worstAction = 'warn';
  }

  return worstAction;
}

핵심 요점

  • 콘텐츠 설명자는 콘텐츠가 등급을 받은 이유에 대한 자세한 정보를 제공하여 부모가 가족 가치 및 개별 아동의 민감성을 기반으로 미묘한 결정을 내릴 수 있도록 합니다.
  • WIA-CHILD-003는 여섯 가지 주요 설명자 범주를 정의합니다: 폭력, 언어, 성적 콘텐츠, 약물, 공포 요소, 성숙 주제, 각각 정확성을 위한 여러 하위 설명자 포함.
  • 경고 라벨은 부모 및 사용자와의 효과적인 커뮤니케이션을 보장하기 위해 가시성, 배치, 명확성, 일관성, 실행 가능성 및 접근성의 디자인 원칙을 따라야 합니다.
  • AI 기반 자동 설명자 생성은 85-90% 신뢰도 임계값으로 텍스트, 이미지, 비디오, 오디오 및 인터랙티브 미디어에 걸쳐 사용자 생성 및 동적 콘텐츠의 실시간 분석을 가능하게 합니다.
  • 문화적 현지화가 필수적이며, 99개 이상의 언어에 걸쳐 폭력, 언어, 노출 및 종교적 콘텐츠에 대한 다양한 문화적 규범을 설명하는 설명자 번역 및 지역 민감도 조정이 필요합니다.
  • 설명자 기반 부모 통제는 단순한 연령 등급을 넘어 세부적인 기본 설정을 허용하여 부모가 특정 콘텐츠 유형에 대한 허용/경고/차단 작업을 지정하고 사용자 정의 규칙을 생성할 수 있도록 합니다.
  • 효과적인 경고 시스템은 압도하지 않고 정보를 제공하며, 색상으로 구분된 배지, 명확한 설명 및 실행 가능한 선택을 사용하여 사용성을 유지하면서 정보에 입각한 동의를 지원합니다.

복습 문제

  1. 연령 등급 외에 콘텐츠 설명자가 필요한 이유를 설명하십시오. 이들이 어떻게 더 미묘한 부모의 의사 결정을 가능하게 합니까?
  2. WIA-CHILD-003의 여섯 가지 주요 설명자 범주를 설명하십시오. 각 범주 내의 하위 설명자 예를 제공하십시오.
  3. 효과적인 경고 라벨을 위한 여섯 가지 디자인 원칙은 무엇입니까? 각 원칙이 사용자 이해 및 행동에 왜 중요합니까?
  4. 다양한 콘텐츠 유형(텍스트, 이미지, 비디오)에 대한 자동 설명자 생성이 어떻게 작동합니까? 어떤 신뢰도 임계값이 사용되며 왜 그렇습니까?
  5. 문화적 차이가 콘텐츠 설명자 적용에 어떻게 영향을 미치는지 설명하십시오. 지역 전반에 걸쳐 다르게 평가될 수 있는 콘텐츠의 구체적인 예를 제공하십시오.
  6. 설명자 기반 부모 통제가 어떻게 작동합니까? 부모의 설명자 기본 설정에 대한 콘텐츠 평가 예를 살펴보십시오.
  7. '판타지 폭력'과 '현실적 폭력' 설명자의 차이점은 무엇입니까? 이러한 구분이 연령 적합성에 왜 중요합니까?
  8. 동적 설명자 생성이 사용자 생성 콘텐츠가 있는 플랫폼에서 보호를 어떻게 가능하게 하는지 설명하십시오. 관련된 기술적 과제는 무엇입니까?

弘益人間 (널리 인간을 이롭게 하라)

투명한 커뮤니케이션은 정보에 입각한 의사 결정을 위한 도구를 제공하면서 부모의 권위를 존중함으로써 인류에게 봉사합니다. 콘텐츠 설명자와 경고 라벨은 이 철학을 구현합니다 - 지시하지 않고 정보를 제공하고, 압도하지 않고 권한을 부여하며, 선택을 보존하면서 보호합니다.

콘텐츠 특성을 명확하게 전달함으로써 우리는 전 세계 부모가 자신의 가치와 자녀의 고유한 요구 사항에 부합하는 결정을 내릴 수 있도록 합니다. 다양한 관점과 개별 상황에 대한 이러한 존중은 진정으로 모든 인류에게 이익이 됩니다.
