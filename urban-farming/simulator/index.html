Welcome to Urban Farming Simulator

This interactive simulator demonstrates the WIA-AGRI-031 Urban Farming Standard, which defines data formats, APIs, protocols, and integration patterns for city-based agriculture systems including rooftop gardens, community gardens, container farming, and urban food forests.

Key Features:

  • 🏢 Rooftop garden management and tracking
  • 🌱 Community garden plot allocation
  • 📦 Container farming systems
  • 🌡️ Microclimate monitoring
  • 💧 Water usage optimization
  • 🔄 Composting integration
  • 📊 Community engagement tracking

Simulation Tabs:

  • Data Input: Create urban farm configurations and input data
  • API Test: Test RESTful API endpoints for farm management
  • Protocol: Simulate real-time data streaming protocols
  • Integration: Test integrations with city systems and platforms
