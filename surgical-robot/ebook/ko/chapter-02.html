"단순함은 궁극의 정교함이다" - 레오나르도 다빈치

2.2 외과의 콘솔 (Surgeon Console)

2.2.1 인체공학적 설계

외과의 콘솔은 장시간 수술 중 최적의 편안함과 제어를 제공하도록 설계되었습니다:

좌석 시스템:

  • 조정 가능한 높이 (60-80cm)
  • 팔 받침대 (각도 조절 가능)
  • 헤드 받침대 (3D 뷰어와 정렬)
  • 발 페달 플랫폼
  • 메모리 포지셔닝 (외과의별 설정 저장)

3D 비전 시스템:

디스플레이 사양:

해상도: 1920x1080 (각 눈당)
재생률: 60 Hz
확대: 10x - 15x (조정 가능)
시야각: 90도
입체 분리: 완전한 좌/우 채널
색 재현: sRGB 100%
밝기: 300 cd/m²
대비: 1000:1

고급 기능:
  - 형광 이미징 (ICG)
  - 혈관 구조 강조
  - 실시간 거리 측정
  - 포커스 어시스트
  - 스냅샷 캡처

2.2.2 마스터 매니퓰레이터 (핸드 컨트롤러)

외과의의 손 움직임을 캡처하는 정밀 입력 장치:

물리적 특성:

  • 무게: 각 300g (균형 조정됨)
  • 작업 공간: 30cm x 30cm x 30cm
  • 자유도: 7 DOF (어깨, 팔꿈치, 손목, 그립)
  • 분해능: 0.1mm 위치 정확도
  • 힘 피드백: 선택 사양 (차세대 시스템)

입력 매핑:

외과의 동작 마스터 센서 로봇 응답 스케일링 비율
손 이동 (X/Y/Z) 홀 효과 센서 도구 끝 위치 1:3 (기본)
손목 회전 로터리 엔코더 도구 롤 1:1
손목 굽힘 포텐셔미터 도구 피치/요 1:2
집게 동작 압력 센서 도구 조 선형

2.3 환자측 카트 (Patient-Side Cart)

2.3.1 로봇 팔 설계

각 로봇 팔은 독립적이지만 조화롭게 작동하는 정밀 메커니즘입니다:

팔 사양:

단일 로봇 팔 구조:

베이스 (고정점)
  │
  ▼
관절 1: 어깨 요 (Yaw)
  │ 범위: ±170°
  │ 토크: 50 Nm
  ▼
관절 2: 어깨 피치 (Pitch)
  │ 범위: ±90°
  │ 토크: 40 Nm
  ▼
관절 3: 팔꿈치
  │ 범위: ±120°
  │ 토크: 30 Nm
  ▼
관절 4: 원격 중심 (RCM)
  │ 고정점: 복벽 삽입부
  │ 정밀도: ±0.5 mm
  ▼
엔드이펙터 (도구)
  │ 자유도: 7 DOF
  │ 분해능: 0.01 mm
  └─→ 수술 부위

총 중량: 15 kg (팔당)
최대 도달 거리: 90 cm
반복 정밀도: ±0.02 mm

2.3.2 원격 운동 중심 (Remote Center of Motion - RCM)

[i] 핵심 개념: RCM은 로봇 팔이 복벽의 작은 절개부를 중심으로 회전하도록 보장하여 조직 손상을 방지합니다.

RCM 메커니즘:

         복벽 (삽입 지점)
              ●  ← RCM 고정점
             /│\
            / │ \
           /  │  \
    팔 움직임  │  팔 움직임
         ↙   │   ↘
       도구는 이 점을 중심으로
       자유롭게 회전하지만
       절개부를 통과하여 밀거나
       당기지 않음

제약 조건:
  - X/Y 평면: 고정 (절개부)
  - Z 축: 자유 (삽입 깊이)
  - 회전: 모든 축에서 자유
  - 최대 편차: ±0.5 mm

2.3.3 팔 구성 옵션

구성 팔 수 일반적인 할당 수술 유형
3-팔 3 1 카메라 + 2 도구 단순 절차
4-팔 표준 4 1 카메라 + 3 도구 대부분의 수술
4-팔 듀얼 콘솔 4 1 카메라 + 2 도구 (각 외과의) 복잡한 절차, 교육
확장형 5+ 다중 카메라/도구 연구, 특수 절차

2.4 엔도리스트 도구 (EndoWrist Instruments)

2.4.1 7 자유도 (DOF) 설계

엔도리스트 도구는 인간 손목의 움직임을 넘어서는 7 자유도를 제공합니다:

7 자유도 분석:

DOF 1: 삽입/후퇴 (Z축)
  - 범위: 0-20 cm
  - 분해능: 0.1 mm

DOF 2: 회전 (롤)
  - 범위: ±270°
  - 분해능: 0.1°

DOF 3: 피치 (상하)
  - 범위: ±90°
  - 분해능: 0.1°

DOF 4: 요 (좌우)
  - 범위: ±90°
  - 분해능: 0.1°

DOF 5: 손목 피치
  - 범위: ±90°
  - 분해능: 0.1°

DOF 6: 손목 요
  - 범위: ±90°
  - 분해능: 0.1°

DOF 7: 조 (그립)
  - 범위: 0-60° (개방 각도)
  - 힘: 0-20 N
  - 분해능: 0.5°

인간 손목: 3 DOF (피치, 요, 롤)
엔도리스트: 7 DOF (손목을 능가)

2.4.2 도구 유형 및 응용

도구 유형 기능 크기 사용 수명
집게 조직 파지 및 조작 5mm, 8mm 10회 사용
가위 조직 절단 5mm, 8mm 10회 사용
니들 드라이버 봉합 5mm, 8mm 10회 사용
바이폴라 집게 지혈 (혈관 봉인) 5mm, 8mm 10회 사용
모노폴라 후크 절개 및 소작 5mm, 8mm 10회 사용
클립 어플라이어 혈관/조직 클리핑 8mm 1회 사용
스테이플러 조직 스테이플링 12mm 1회 사용
