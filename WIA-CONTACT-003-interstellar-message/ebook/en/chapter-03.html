Chapter 3: Message Encoding Protocols

From Abstract Concepts to Physical Signals

The transition from conceptual message content to actual transmission represents a critical engineering challenge. WIA-CONTACT-003 specifies detailed protocols for encoding information in electromagnetic carriers—whether radio waves, optical pulses, or other modalities—that can traverse interstellar distances while remaining detectable and decodable at the destination.

This chapter explores the technical foundations of how bits become photons, how structure survives across light-years, and how we maximize signal integrity against the cosmic noise background.

Modulation Schemes

Binary On-Off Keying (OOK)

The simplest modulation: carrier is either present (1) or absent (0).

ON-OFF KEYING PATTERN:
Data:     1    0    1    1    0    1    0    0    1
         ┌─┐       ┌─┐  ┌─┐       ┌─┐            ┌─┐
Signal:  │ │       │ │  │ │       │ │            │ │
        ─┘ └───────┘ └──┘ └───────┘ └────────────┘ └─

Advantages:
✓ Extremely simple detection
✓ Robust against amplitude fluctuations
✓ Minimal required bandwidth

Disadvantages:
✗ 50% duty cycle reduces average power
✗ Susceptible to interstellar scintillation
✗ No error detection inherent in modulation

Frequency Shift Keying (FSK)

Two different frequencies represent 0 and 1:

FREQUENCY SHIFT KEYING:
Data:     1    0    1    1    0    1    0    0    1

Freq f₁: ~~~       ~~~  ~~~       ~~~            ~~~
         (1)       (1)  (1)       (1)            (1)

Freq f₂:     ~~~             ~~~       ~~~  ~~~
             (0)             (0)       (0)  (0)

Advantages:
✓ Constant transmission power
✓ Robust frequency discrimination
✓ Can use narrow filters for noise reduction

Disadvantages:
✗ Requires twice the bandwidth
✗ More complex receiver design
✗ Doppler shift affects both frequencies

Error Correction Codes

Why Error Correction is Critical

Interstellar signals face multiple error sources:

  • Interstellar scintillation: Signal fading due to ionized plasma
  • Cosmic ray impacts: Random bit flips in detectors
  • Thermal noise: Receiver electronics noise
  • Doppler effects: Frequency shifts from relative motion
  • Multipath propagation: Signals bent by gravitational lensing

Reed-Solomon Error Correction

WIA-CONTACT-003 recommends Reed-Solomon coding for its ability to correct burst errors:

Reed-Solomon RS(255,223) Example:

Input: 223 bytes of data
Output: 255 bytes (223 data + 32 parity)

Can correct: up to 16 byte errors
Overhead: 14.3% (32/223)

Encoding Process:
1. Treat 223 bytes as polynomial coefficients
2. Divide by generator polynomial
3. Append remainder as parity bytes
4. Transmit all 255 bytes

Decoding Process:
1. Receive 255 bytes (some may be corrupted)
2. Calculate syndrome polynomials
3. Identify error locations and values
4. Correct errors and extract original 223 bytes

Effectiveness for Interstellar Communication:
✓ Corrects burst errors from scintillation
✓ Well-tested in deep space missions (Voyager, etc.)
✓ Decodable with mathematical knowledge only
✓ Redundancy pattern is itself informative

Synchronization and Timing

Carrier Synchronization

Receivers must lock onto the carrier frequency despite Doppler shifts and local oscillator drift:

Synchronization Method Technique Accuracy
Coarse Acquisition Fast Fourier Transform sweep ±1000 Hz
Fine Tracking Phase-locked loop (PLL) ±1 Hz
Symbol Timing Early-late gate detector ±0.1 symbol periods
Frame Alignment Correlation with known preamble Exact

Doppler Compensation

Relative motion between transmitter and receiver causes frequency shifts:

Doppler Shift Calculation:
f_received = f_transmitted × (1 + v/c)

Where:
  v = relative velocity (positive = approaching)
  c = speed of light

Example: Star system approaching at 50 km/s
  v/c = 50,000 / 299,792,458 ≈ 1.67 × 10⁻⁴

  For 1420 MHz transmission:
  Shift = 1420 MHz × 1.67 × 10⁻⁴ ≈ 237 kHz

  Receiver must search ±237 kHz around expected frequency

For orbital motion (Earth around Sun):
  v ≈ 30 km/s
  Annual frequency variation: ±142 kHz at 1420 MHz

  Message encoding must account for this by either:
  a) Long symbol periods (>1 ms) so shift is negligible
  b) Doppler-resistant modulation (FSK with wide spacing)
  c) Timing pattern that teaches receiver the drift rate

Compression and Encoding Efficiency

Information-Theoretic Limits

Shannon's Channel Capacity theorem defines maximum achievable data rate:

Shannon Capacity:
C = B × log₂(1 + SNR)

Where:
  C = Channel capacity (bits per second)
  B = Bandwidth (Hz)
  SNR = Signal-to-noise ratio (linear, not dB)

Example: Interstellar Link Budget
  Bandwidth: 100 Hz (narrow to reduce noise)
  SNR: 10 dB = 10:1 linear

  C = 100 × log₂(1 + 10)
    = 100 × log₂(11)
    = 100 × 3.46
    ≈ 346 bits per second

This is theoretical maximum; practical systems
achieve 50-80% of Shannon limit with modern codes.

Lossless Compression

While interstellar messages should avoid complex compression (receiver may not understand algorithm), simple run-length encoding can reduce redundancy:

RUN-LENGTH ENCODING EXAMPLE: Original: 000000111111000111 Encoded: 6(0) 6(1) 3(0) 3(1) Binary representation: (count, value) pairs Efficiency gain: ~40% for highly redundant data Not recommended for primary encoding due to: - Assumes receiver knows compression algorithm - Errors corrupt subsequent symbols - Better to use redundancy for error correction

Summary

Message encoding protocols form the critical bridge between abstract information content and physical transmission. The WIA-CONTACT-003 standard specifies robust encoding schemes that:

  • Use simple, universally decodable modulation (OOK, FSK, PPM)
  • Employ strong error correction (Reed-Solomon, convolutional codes)
  • Structure data hierarchically with clear synchronization markers
  • Accommodate Doppler shifts and timing uncertainties
  • Maximize information transfer within Shannon capacity limits
  • Provide redundancy through multi-channel transmission

These protocols ensure that messages can survive the hostile interstellar environment—traversing light-years through ionized plasma, gravitational fields, and cosmic radiation—to arrive intact and decodable at their destination.

Key Principle: The encoding must be robust enough to survive interstellar space yet simple enough to be decoded by a civilization that may have taken a completely different technological path. Every encoding decision balances efficiency against universality, sophistication against comprehensibility.
