제2장: 영상 감시 시스템

도시 보안의 핵심 인프라인 영상 감시 시스템(CCTV)의 기술적 구성, 카메라 유형, 저장 솔루션, 네트워크 아키텍처 및 WIA-SECURITY-002 준수 요구사항을 상세히 학습합니다.

弘益人間 (홍익인간)

"널리 인간을 이롭게 하라" - 영상 감시 시스템은 범죄로부터 시민을 보호하고, 사고 발생 시 신속한 대응을 가능하게 하여 모든 사람의 안전한 일상을 지킵니다. 감시가 아닌 보호의 관점에서 시스템을 설계하고 운영해야 합니다.

2.1 영상 감시 시스템의 이해

영상 감시 시스템(Video Surveillance System)은 카메라를 통해 영상을 수집하고, 네트워크를 통해 전송하며, 서버에 저장하고, 운영자가 모니터링할 수 있도록 하는 통합 시스템입니다. 현대의 영상 감시 시스템은 단순한 녹화 기능을 넘어 AI 기반의 지능형 분석, 실시간 알림, 타 시스템과의 연동 등 다양한 고급 기능을 제공합니다.

2.1.1 시스템 구성 요소

영상 수집부

  • IP 카메라
  • PTZ 카메라
  • 열화상 카메라
  • 특수 목적 카메라

전송부

  • 유선 네트워크 (LAN/WAN)
  • 광케이블
  • 무선 네트워크 (WiFi/5G)
  • 전용 영상 전송 장비

저장부

  • NVR (Network Video Recorder)
  • VMS (Video Management System)
  • 클라우드 스토리지
  • 하이브리드 저장소

표출/관제부

  • 비디오월
  • 모니터링 워크스테이션
  • 모바일 클라이언트
  • 웹 기반 뷰어

2.2 카메라 기술 및 유형

영상 감시용 카메라는 설치 환경과 목적에 따라 다양한 유형이 존재합니다. 각 유형의 특성을 이해하고 적절한 카메라를 선택하는 것이 효과적인 감시 시스템 구축의 핵심입니다.

2.2.3 센서 및 렌즈 기술

이미지 센서

현대 보안 카메라는 주로 CMOS(Complementary Metal-Oxide-Semiconductor) 센서를 사용합니다. CMOS 센서는 낮은 전력 소비, 빠른 읽기 속도, 고집적화 가능성 등의 장점을 가지며, 최신 센서는 저조도 환경에서도 우수한 성능을 발휘합니다.

WDR (Wide Dynamic Range) 기술

WDR은 밝은 부분과 어두운 부분의 노출을 동시에 최적화하여 역광 환경에서도 선명한 영상을 제공하는 기술입니다. WIA-SECURITY-002 Silver 등급 이상에서는 120dB 이상의 WDR 성능을 요구합니다.

렌즈 유형

  • 고정 렌즈: 단일 화각, 비용 효율적, 특정 영역 고정 감시에 적합
  • 가변 초점 렌즈: 설치 후 화각 조정 가능, 유연한 배치
  • 전동 줌 렌즈: 원격 화각 조정, PTZ 카메라에 주로 사용
  • 어안 렌즈: 180° 이상 초광각, 왜곡 보정 소프트웨어 필요

2.3 네트워크 아키텍처

대규모 영상 감시 시스템은 안정적이고 고대역폭의 네트워크 인프라를 필요로 합니다. 네트워크 설계는 시스템의 성능과 신뢰성에 직접적인 영향을 미칩니다.

2.3.3 PoE (Power over Ethernet)

PoE 기술은 이더넷 케이블을 통해 데이터와 전원을 동시에 공급하여 IP 카메라 설치를 간소화합니다. WIA-SECURITY-002에서는 다음 PoE 표준을 인정합니다:

표준 최대 전력 적용 카메라 권장 등급
IEEE 802.3af (PoE) 15.4W 일반 고정 카메라 Bronze
IEEE 802.3at (PoE+) 30W PTZ, IR 카메라 Silver
IEEE 802.3bt (PoE++) 60-100W 고성능 PTZ, 히터 내장 Gold/Platinum

2.4 저장 및 관리 시스템

2.4.1 NVR (Network Video Recorder)

NVR은 IP 카메라로부터 네트워크를 통해 영상 스트림을 수신하여 저장하는 전용 장비입니다. 독립형(Standalone) NVR과 소프트웨어 기반 NVR로 구분됩니다.

독립형 NVR

  • 플러그 앤 플레이 설치
  • 내장 스토리지
  • 독립적 운영
  • 소규모 시설 적합

소프트웨어 NVR (VMS)

  • 유연한 확장성
  • 외부 스토리지 연동
  • 고급 분석 기능
  • 대규모 시설 적합

2.4.2 저장 용량 계산

2.5 영상 압축 기술

효율적인 영상 압축은 저장 비용 절감과 네트워크 대역폭 최적화에 필수적입니다. 최신 코덱은 화질 손실을 최소화하면서 압축률을 극대화합니다.

2.5.1 비디오 코덱 비교

코덱 표준 압축 효율 CPU 부하 라이선스
H.264/AVC ITU-T/ISO 2003 기준 (1x) 낮음 유료
H.265/HEVC ITU-T/ISO 2013 2x 향상 중간 유료
H.266/VVC ITU-T/ISO 2020 4x 향상 높음 유료
AV1 AOMedia 2018 2.5x 향상 높음 무료

스마트 코덱 활용

최신 카메라들은 장면 복잡도에 따라 동적으로 비트레이트를 조절하는 스마트 코덱 기술을 지원합니다. 움직임이 적은 장면에서는 비트레이트를 낮추고, 이벤트 발생 시 높여 저장 효율과 화질을 동시에 최적화합니다. WIA-SECURITY-002 Gold 등급 이상에서는 이 기능을 필수로 요구합니다.

2.6 표준 프로토콜

2.6.1 ONVIF (Open Network Video Interface Forum)

ONVIF는 IP 기반 보안 제품의 상호운용성을 보장하는 국제 표준입니다. 서로 다른 제조사의 카메라와 NVR/VMS가 원활하게 연동될 수 있도록 공통 인터페이스를 정의합니다.

프로파일 기능 WIA 요구사항
Profile S 영상 스트리밍 기본 기능 Bronze 이상 필수
Profile G 에지 저장 및 검색 Silver 이상 권장
Profile T H.265, 메타데이터 스트리밍 Gold 이상 필수
Profile M 분석 메타데이터 Platinum 권장

2.6.2 RTSP/RTP

RTSP(Real Time Streaming Protocol)와 RTP(Real-time Transport Protocol)는 실시간 영상 스트리밍의 표준 프로토콜입니다. RTSP는 스트리밍 세션 제어를, RTP는 실제 미디어 데이터 전송을 담당합니다.

// RTSP 스트림 URL 형식 예시 rtsp://[username:password@]camera-ip[:port]/stream-path // 일반적인 URL 패턴 rtsp://admin:password@192.168.1.100:554/Streaming/Channels/101 rtsp://192.168.1.100/live/main // 메인 스트림 rtsp://192.168.1.100/live/sub // 서브 스트림

2.7 설치 및 배치 가이드라인

2.7.1 카메라 배치 원칙

  • 출입구 감시: 모든 출입구에 최소 1대, 얼굴 식별 가능 해상도 확보
  • 중복 커버리지: 사각지대 최소화를 위한 카메라 간 시야 중첩
  • 조명 고려: 역광 방지, 야간 조명 또는 IR 카메라 활용
  • 높이 설정: 2.5-3.5m 높이, 파손 방지와 화각 확보 균형
  • 환경 보호: 실외 설치 시 IP66 이상 방수등급 필수

2.7.2 PPF (Pixels Per Foot) 가이드

감시 목적 PPF 권장값 설명
모니터링 20-30 PPF 일반적인 활동 감지
탐지 40-50 PPF 사람/차량 존재 확인
인식 80-100 PPF 이전에 본 사람 재인식
식별 150+ PPF 처음 보는 사람 식별

핵심 요점 정리

  • 영상 감시 시스템은 영상 수집, 전송, 저장, 표출/관제의 4개 구성 요소로 이루어집니다.
  • 카메라 선택 시 설치 환경, 감시 목적, 해상도 요구사항을 종합적으로 고려해야 합니다.
  • 네트워크 설계는 계층형 아키텍처를 권장하며, 충분한 대역폭 확보가 필수입니다.
  • H.265/HEVC 이상의 효율적인 코덱 사용으로 저장 비용을 절감할 수 있습니다.
  • ONVIF 표준 준수로 다양한 제조사 장비 간 상호운용성을 확보합니다.
  • 카메라 배치 시 PPF 가이드라인을 참조하여 감시 목적에 맞는 화질을 확보합니다.

복습 문제

  1. 영상 감시 시스템의 4대 구성 요소와 각각의 역할을 설명하시오.
  2. PTZ 카메라와 고정형 카메라의 장단점을 비교하시오.
  3. H.264와 H.265 코덱의 차이점과 각각의 적용 시나리오를 설명하시오.
  4. 100대 카메라(4K, H.265, 30fps)를 60일간 저장하기 위한 스토리지 용량을 계산하시오.
  5. ONVIF 프로파일 S, G, T의 차이점을 설명하시오.
  6. 얼굴 식별이 필요한 출입구에 적합한 카메라 사양을 제안하시오.
