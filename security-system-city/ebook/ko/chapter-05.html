제5장: 센서 네트워크

도시 보안을 위한 IoT 센서 네트워크의 구성, 센서 유형, 통신 프로토콜, 데이터 수집 및 처리 방법, 그리고 WIA-SECURITY-002 준수 요구사항을 상세히 학습합니다.

弘益人間 (홍익인간)

"널리 인간을 이롭게 하라" - 센서 네트워크는 도시의 감각 기관과 같습니다. 화재, 침입, 환경 위험을 조기에 감지하여 시민의 생명과 재산을 보호합니다. 보이지 않는 곳에서 끊임없이 도시를 지키는 조용한 수호자입니다.

5.1 센서 네트워크 개요

센서 네트워크는 다양한 물리적, 환경적 현상을 감지하고 데이터화하여 보안 시스템에 전달하는 분산된 센서들의 집합입니다. 영상 감시만으로는 포착할 수 없는 화재, 가스 누출, 진동, 온도 변화 등을 실시간으로 모니터링하여 종합적인 상황 인식(Situational Awareness)을 가능하게 합니다.

IoT(Internet of Things) 기술의 발전으로 센서들은 더욱 소형화, 저전력화, 지능화되었습니다. 수천, 수만 개의 센서가 도시 전역에 배치되어 실시간으로 데이터를 수집하고, 엣지 컴퓨팅과 AI를 통해 현장에서 1차 분석을 수행합니다.

5.1.1 센서 네트워크의 역할

  • 침입 탐지: 보호 구역에 대한 무단 접근 감지
  • 화재/가스 감지: 화재, 연기, 유해 가스의 조기 경보
  • 환경 모니터링: 온습도, 대기질, 소음 수준 측정
  • 구조물 안전: 건물, 교량 등의 진동, 기울기, 균열 감지
  • 수자원 관리: 수위, 유량, 수질 모니터링
  • 교통 관리: 차량 감지, 교통량 측정, 주차 현황

5.3 통신 프로토콜

센서 네트워크의 효율적인 운영을 위해서는 적절한 통신 프로토콜 선택이 중요합니다. 센서의 특성, 설치 환경, 데이터 전송량, 전력 요구사항 등을 고려하여 최적의 프로토콜을 선택해야 합니다.

5.3.1 무선 통신 프로토콜 비교

프로토콜 범위 데이터율 전력 적용 분야
WiFi ~100m 높음 (Mbps) 높음 카메라, 고대역 센서
Zigbee ~100m 낮음 (250kbps) 매우 낮음 홈오토메이션, 빌딩
Z-Wave ~30m 낮음 (100kbps) 매우 낮음 스마트홈 보안

LoRaWAN ~15km 매우 낮음 (50kbps) 매우 낮음 도시 인프라, 광역

5.3.2 LPWAN 기술

LPWAN(Low Power Wide Area Network)은 저전력으로 장거리 통신이 가능한 기술로, 도시 규모의 센서 네트워크에 적합합니다. 배터리로 수년간 운영 가능하며, 수 킬로미터에서 수십 킬로미터의 통신 범위를 제공합니다.

5.4.2 엣지 컴퓨팅

엣지 컴퓨팅은 센서 데이터를 중앙 서버로 전송하기 전에 현장에서 1차 처리하는 방식입니다. 네트워크 대역폭을 절감하고, 지연 시간을 최소화하며, 개인정보 보호에도 기여합니다.

처리 위치 지연 시간 대역폭 처리 능력 적합한 작업
센서 (On-device) <1ms 최소 제한적 임계값 경보
엣지 게이트웨이 1-10ms 낮음 중간 이벤트 필터링, 집계
엣지 서버 10-50ms 중간 높음 AI 분석, 패턴 인식
클라우드 50-200ms 높음 무제한 장기 저장, 대규모 분석

5.4.3 데이터 융합

센서 데이터 융합(Sensor Fusion)은 여러 센서의 데이터를 결합하여 더 정확하고 신뢰성 있는 정보를 생성하는 기술입니다. 단일 센서의 한계를 극복하고 오탐/미탐을 줄입니다.

센서 융합 레벨

  • 데이터 레벨: 원시 센서 데이터 결합 (예: 다중 카메라 스티칭)
  • 특징 레벨: 추출된 특징 결합 (예: 영상+음향 이벤트 감지)
  • 결정 레벨: 개별 판단 결과 결합 (예: 다중 센서 투표)

5.5 센서 네트워크 설계

5.5.1 설계 고려사항

  • 커버리지: 보호 영역의 완전한 커버, 사각지대 최소화
  • 중복성: 핵심 구역 다중 센서 배치, 단일 실패점 제거
  • 확장성: 향후 센서 추가를 위한 여유 용량 확보
  • 유지보수: 접근 용이한 설치 위치, 원격 관리 기능
  • 전원 공급: 유선/배터리/태양광 등 환경에 맞는 전원 설계
  • 환경 내성: 실외 설치 시 IP 등급, 온도 범위 고려

핵심 요점 정리

  • 센서 네트워크는 영상 감시를 보완하여 화재, 침입, 환경 위험을 종합적으로 모니터링합니다.
  • 침입 탐지에는 PIR, 마이크로웨이브, 듀얼 테크 등 다양한 센서가 활용됩니다.
  • LPWAN 기술(LoRaWAN, NB-IoT)은 도시 규모 센서 배치에 적합합니다.
  • 엣지 컴퓨팅을 통해 현장에서 1차 데이터 처리를 수행합니다.
  • 센서 융합은 단일 센서의 한계를 극복하고 정확도를 향상시킵니다.
  • 다중 레이어 설계로 심층 방어(Defense in Depth)를 구현합니다.

복습 문제

  1. PIR 센서와 마이크로웨이브 센서의 작동 원리와 각각의 장단점을 비교하시오.
  2. 화재 감지기 5가지 유형과 각각의 적용 환경을 설명하시오.
  3. LoRaWAN과 NB-IoT의 차이점과 각각의 적합한 적용 시나리오를 기술하시오.
  4. 엣지 컴퓨팅의 장점과 센서 네트워크에서의 역할을 설명하시오.
  5. 센서 융합의 3가지 레벨(데이터, 특징, 결정)을 각각 설명하시오.
  6. 침입 탐지를 위한 다중 레이어 경계 보안 설계를 제안하시오.
