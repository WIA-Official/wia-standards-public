제8장: 사이버 보안

물리적 보안 시스템을 사이버 위협으로부터 보호하기 위한 사이버 보안(Cybersecurity) 전략을 학습합니다. 네트워크 보안, 암호화, 접근 제어, 취약점 관리 및 WIA-SECURITY-002 준수 요구사항을 다룹니다.

弘益人間 (홍익인간)

"널리 인간을 이롭게 하라" - 물리적 보안 시스템이 해킹당하면 시민의 프라이버시가 침해되고, 안전이 위협받습니다. 사이버 보안은 디지털 시대의 성벽이며, 기술의 힘이 오용되지 않도록 지키는 방패입니다.

8.1 사이버 보안 개요

도시 보안 시스템은 점점 더 네트워크화되고 있습니다. IP 카메라, 클라우드 기반 VMS, IoT 센서 등 연결된 장치의 수가 급증하면서 사이버 공격의 표면도 확대되었습니다. 물리적 보안 시스템의 침해는 단순한 데이터 유출을 넘어, 감시 무력화, 출입 통제 해제, 거짓 경보 등 심각한 물리적 피해로 이어질 수 있습니다.

8.2 네트워크 보안

8.2.1 네트워크 세그멘테이션

보안 시스템 네트워크를 기업 IT 네트워크 및 인터넷으로부터 분리하는 것은 가장 기본적이면서도 효과적인 보안 조치입니다. VLAN, 물리적 분리, DMZ 등을 통해 공격 표면을 최소화합니다.

8.2.3 침입 탐지/방지 시스템

IDS/IPS 배치 권장사항

  • 경계: 인터넷-DMZ 구간, 외부 공격 탐지
  • 내부: 주요 서버 앞단, 내부 이동 탐지
  • 보안망: VMS/NVR 앞단, 비정상 트래픽 탐지

WIA-SECURITY-002 Silver 이상에서는 IDS를, Gold 이상에서는 IPS를 필수로 요구합니다. 시그니처 기반과 행위 기반 탐지를 병행합니다.

8.4 접근 제어

8.4.1 관리 계정 보안

기본 비밀번호 위험

많은 보안 사고가 장치의 기본 비밀번호(admin/admin, root/root 등)를 변경하지 않아 발생합니다. 2016년 Mirai 봇넷은 기본 비밀번호를 사용하는 40만 대 이상의 IoT 장치를 감염시켰습니다.

WIA-SECURITY-002 필수 요구사항:

  • 설치 시 기본 비밀번호 강제 변경
  • 최소 12자, 복잡성 요구 (대소문자, 숫자, 특수문자)
  • 90일마다 비밀번호 변경 (또는 MFA 적용 시 면제)

8.6 모니터링 및 대응

8.6.1 보안 로그 관리

수집 대상 로그

  • 인증 로그: 로그인 성공/실패, 계정 변경
  • 접근 로그: 영상 조회, 설정 변경, 내보내기
  • 시스템 로그: 서비스 시작/중지, 오류, 경고
  • 네트워크 로그: 방화벽, IDS/IPS, 트래픽 플로우
  • 장치 로그: 카메라, 컨트롤러, 센서 이벤트

WIA-SECURITY-002는 모든 보안 관련 로그를 중앙 SIEM으로 수집하고, Gold 이상에서는 실시간 상관 분석을 요구합니다.

8.7 개인정보 보호

8.7.1 영상 정보 보호

CCTV 영상은 개인정보로서 개인정보보호법의 적용을 받습니다. 영상 수집, 저장, 열람, 제공의 모든 단계에서 법적 요구사항을 준수해야 합니다.

단계 요구사항 WIA 권장사항

8.7.2 프라이버시 강화 기술

  • 얼굴 모자이크: 무관한 인물 자동 블러링
  • 실루엣 전환: 사람 형태만 표시, 개인 식별 불가
  • 선택적 복호화: 권한자만 원본 영상 접근
  • 데이터 최소화: 필요 최소 해상도, 보관 기간
  • 익명화: 통계/분석 시 개인 식별 정보 제거

핵심 요점 정리

  • 물리보안 시스템은 영상 탈취, 시스템 무력화, 랜섬웨어 등 고유한 사이버 위협에 노출됩니다.
  • 네트워크 세그멘테이션으로 보안망을 IT/OT 네트워크와 분리해야 합니다.
  • 영상 스트림과 저장 데이터 모두 암호화가 필수입니다.
  • 기본 비밀번호 변경과 다중 인증은 가장 기본적인 보안 조치입니다.
  • 정기적인 취약점 스캔과 신속한 패치가 중요합니다.
  • 개인정보보호법 준수와 프라이버시 강화 기술 적용이 필요합니다.

복습 문제

  1. 물리보안 시스템에 대한 6가지 주요 사이버 위협을 설명하시오.
  2. 보안 네트워크 세그멘테이션의 권장 구조를 설계하시오.
  3. 영상 데이터의 전송 및 저장 시 적용해야 할 암호화 방식을 설명하시오.
  4. 취약점 관리 프로세스 5단계를 기술하시오.
  5. 사이버 공격 발생 시 대응 절차를 단계별로 설명하시오.
  6. CCTV 영상의 개인정보 보호를 위한 기술적 조치를 나열하시오.
