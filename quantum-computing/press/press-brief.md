# WIA-QC-001 — Quantum Computing (Press Brief)

**WIA Standards releases canonical Quantum Computing envelope shape**

The WIA Standards Committee has published the WIA-QC-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for quantum computing hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
