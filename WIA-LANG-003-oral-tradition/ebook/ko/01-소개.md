# 1장: 구술 전통 디지털 보존 표준 소개

## 인류 지식의 가장 오래된 형태

구술 전통은 인류 문명의 근간입니다. 문자가 발명되기 수천 년 전부터, 인간은 이야기, 노래, 의례를 통해 지식, 역사, 문화적 가치를 전승해왔습니다. 오늘날에도 전 세계 수천 개의 공동체가 신화, 전설, 민담, 역사적 서사, 의료 지식, 환경 지식을 구술 형태로 보존하고 있습니다.

### 구술 전통의 규모와 다양성

2025년 현재, 전 세계적으로:

- **7,000개 이상의 언어**가 사용되며, 그 중 많은 언어가 주로 구술 형태로 존재
- **약 40%의 언어** (약 2,800개)가 문자 체계를 갖지 않음
- **매 2주마다 1개의 언어**가 소멸하며, 함께 독특한 구술 전통도 사라짐
- **수백만 시간**의 구술 문학, 노래, 의례가 기록되지 않은 채 소멸 위기

**전 세계 구술 전통 유형:**

| 유형 | 설명 | 예시 |
|------|------|------|
| **신화** | 세계, 신, 인류의 기원을 설명하는 신성한 이야기 | 단군 신화, 그리스 신화, 아프리카 창조 신화 |
| **전설** | 역사적 인물이나 사건에 기반한 이야기 | 이순신 장군 이야기, 아서 왕 전설 |
| **민담** | 교훈을 담은 대중적 이야기 | 흥부와 놀부, 신데렐라, 아난시 이야기 |
| **서사시** | 영웅의 모험을 노래하는 긴 시 | 호메로스의 오디세이, 티베트의 게사르 |
| **지식 체계** | 의료, 농업, 환경 지식 | 약초 사용법, 계절 예측, 항해 기술 |
| **의례 및 노래** | 종교적, 사회적 의식 | 판소리, 샤먼 의례, 결혼식 노래 |

### 구술 전통이 직면한 위기

디지털 시대에 구술 전통은 전례 없는 위협에 직면하고 있습니다:

**언어 소멸:**
- 2100년까지 현존 언어의 **50-90%가 소멸**할 것으로 예측
- 언어와 함께 수천 년의 구술 지식이 영원히 사라짐
- 노인 화자의 죽음으로 전승의 끊김

**문화적 변화:**
- 젊은 세대의 도시 이주로 전통적 전승 단절
- 글로벌 미디어가 지역 구술 전통 대체
- 전통적 공동체 구조의 붕괴

**기술적 장벽:**
- 구술 전통을 디지털로 보존하는 표준화된 방법 부재
- 오디오/비디오 파일 형식의 빠른 구식화
- 메타데이터 및 맥락 정보의 손실
- 문화적으로 적절한 보존 프로토콜 결여

**법적 및 윤리적 문제:**
- 구술 전통에 대한 지적 재산권 보호 불충분
- 신성하거나 비밀스러운 지식의 부적절한 공개
- 원주민 공동체의 동의 없는 기록 및 배포

### WIA-LANG-003 표준의 필요성

WIA-LANG-003 구술 전통 디지털 보존 표준은 이러한 도전에 대응하기 위해 개발되었습니다:

#### 표준의 목표

**1. 보존 (Preservation)**
최고 품질의 오디오 및 비디오 형식으로 구술 전통을 기록하여 미래 세대를 위해 보존합니다.

**2. 접근성 (Accessibility)**
문화적으로 적절한 방식으로 구술 전통에 대한 접근을 가능하게 하며, 공동체의 통제권을 존중합니다.

**3. 맥락화 (Contextualization)**
풍부한 메타데이터로 구술 전통을 문서화하여 문화적, 역사적, 언어적 맥락을 보존합니다.

**4. 상호운용성 (Interoperability)**
전 세계 기관과 연구자가 데이터를 공유하고 통합할 수 있도록 표준화된 형식을 제공합니다.

**5. 윤리적 관리 (Ethical Stewardship)**
원주민 공동체의 권리, 문화적 민감성, 전통 지식에 대한 주권을 존중합니다.

### 핵심 데이터 구조

WIA-LANG-003은 구술 전통을 포괄적으로 기록하기 위한 데이터 구조를 정의합니다:

```typescript
interface OralTraditionRecord {
  // 기본 식별 정보
  recordID: string;                    // 고유 식별자
  title: string;                       // 작품 제목
  type: OralTraditionType;            // 유형 분류
  language: LanguageInfo;             // 언어 정보

  // 화자 정보
  performers: Performer[];            // 화자/공연자들

  // 기록 정보
  recording: RecordingMetadata;       // 오디오/비디오 메타데이터

  // 내용 정보
  content: {
    transcription?: Transcription[];  // 전사본
    translation?: Translation[];      // 번역본
    summary: string;                  // 요약
    themes: string[];                 // 주제 태그
    keywords: string[];               // 키워드
  };

  // 문화적 맥락
  culturalContext: CulturalContext;   // 문화적 배경

  // 권리 및 접근
  rights: RightsManagement;           // 권리 관리
  access: AccessControl;              // 접근 제어

  // 보존 정보
  preservation: PreservationMetadata; // 보존 메타데이터
}

type OralTraditionType =
  | "myth"              // 신화
  | "legend"            // 전설
  | "folktale"          // 민담
  | "epic"              // 서사시
  | "song"              // 노래
  | "ritual"            // 의례
  | "historical_narrative" // 역사 서사
  | "knowledge_system"  // 지식 체계
  | "other";

interface LanguageInfo {
  iso639_3: string;        // ISO 639-3 언어 코드
  name: string;            // 언어 이름
  dialect?: string;        // 방언
  script?: string;         // 문자 체계 (있는 경우)
  endangerment: "safe" | "vulnerable" | "definitely_endangered"
                | "severely_endangered" | "critically_endangered" | "extinct";
  speakers: number;        // 화자 수 추정
}

interface Performer {
  name?: string;           // 화자 이름 (공개 동의 시)
  anonymousID?: string;    // 익명 식별자
  role: "narrator" | "singer" | "musician" | "elder" | "shaman" | "other";
  age?: number;
  gender?: "male" | "female" | "other" | "prefer_not_to_say";
  community: string;       // 소속 공동체
  biography?: string;      // 화자 배경
  consent: ConsentRecord;  // 동의 기록
}
```

### 전사 및 번역 표준

구술 전통의 텍스트화는 중요한 보존 도구입니다:

```typescript
interface Transcription {
  transcriptionID: string;
  language: string;                     // ISO 639-3
  script: string;                       // 문자 체계
  type: "orthographic" | "phonetic" | "phonemic";

  // 시간 정렬된 전사
  segments: TranscriptionSegment[];

  // 전사 방법론
  methodology: {
    transcriber: string;
    date: Date;
    conventions: string;                // 전사 규칙
    tools?: string[];                   // 사용된 도구
    aiAssisted: boolean;                // AI 사용 여부
    humanVerified: boolean;             // 인간 검증 여부
  };
}

interface TranscriptionSegment {
  segmentID: string;
  startTime: number;                    // 초
  endTime: number;                      // 초
  speaker?: string;                     // 화자 ID
  text: string;                         // 전사된 텍스트

  // 추가 주석
  annotations?: {
    type: "paralinguistic" | "environmental" | "cultural";
    description: string;
    timestamp: number;
  }[];

  // 불확실성 표시
  confidence?: number;                  // 0-1
  uncertain?: boolean;
}

interface Translation {
  translationID: string;
  targetLanguage: string;               // ISO 639-3
  type: "literal" | "free" | "poetic";

  // 번역된 텍스트
  segments: {
    sourceSegmentID: string;            // 원문 세그먼트 참조
    text: string;                       // 번역문
    notes?: string;                     // 번역 주석
  }[];

  // 번역자 정보
  translator: {
    name: string;
    credentials?: string;
    nativeSpeaker: boolean;
  };

  date: Date;
}
```

### 한국의 구술 전통 사례

한국은 풍부한 구술 전통 유산을 보유하고 있습니다:

**판소리 (Pansori)**
- 유네스코 인류무형문화유산
- 북과 함께하는 서사적 노래
- 주요 작품: 춘향가, 심청가, 흥보가, 수궁가, 적벽가

**무속 의례**
- 샤먼의 구술 서사
- 지역별 독특한 전통
- 신화, 역사, 치유를 결합

**민요 및 민담**
- 지역별 독특한 노래와 이야기
- 농업, 어업, 일상생활 반영
- 사회적 가치와 교훈 전달

### 문화적 민감성 및 윤리

구술 전통 보존은 문화적으로 민감한 접근을 요구합니다:

```typescript
interface CulturalSensitivityProtocol {
  // 사전 자유 정보 동의 (FPIC)
  consent: {
    obtained: boolean;
    date: Date;
    consentForm: string;                // 동의서 문서 참조
    withdrawalOption: boolean;          // 동의 철회 가능
    communityConsent: boolean;          // 공동체 동의
  };

  // 문화적 민감도
  culturalSensitivity: "public" | "restricted" | "sacred" | "secret";

  // 접근 제한
  accessRestrictions?: {
    ageRestriction?: number;
    genderRestriction?: "male_only" | "female_only";
    communityMembersOnly?: boolean;
    elders_approval_required?: boolean;
    seasonal_restrictions?: string;     // 특정 계절에만 접근 가능
  };

  // 문화적 맥락 경고
  contentWarnings?: {
    type: "sacred" | "grief" | "violence" | "adult_content";
    description: string;
  }[];

  // 원주민 데이터 주권
  indigenousDataSovereignty: {
    communityOwnership: boolean;
    CARE_principles: boolean;           // Collective benefit, Authority to control,
                                        // Responsibility, Ethics
    repatriation_possible: boolean;     // 원본 반환 가능
  };
}
```

### 글로벌 구술 전통 이니셔티브

전 세계 기관들이 구술 전통 보존에 협력하고 있습니다:

**유네스코 세계기억유산 (Memory of the World)**
- 구술 전통 기록을 세계 유산으로 인정
- 디지털 보존 가이드라인
- 국제 협력 촉진

**위기에 처한 언어 연합 (Endangered Languages Alliance)**
- 소멸 위기 언어의 구술 전통 기록
- 공동체 기반 프로젝트
- 디지털 아카이브 구축

**세계구비문학학회 (International Society for Folk Narrative Research)**
- 연구 표준 개발
- 국제 데이터베이스 구축
- 학술 협력

### 철학: 弘益人間 (널리 인간을 이롭게 하라)

구술 전통은 인류의 공동 유산입니다. 각 이야기, 각 노래, 각 신화는 특정 공동체에서 비롯되었지만, 그 안에 담긴 지혜, 가치, 인간 경험은 모든 인류와 공명합니다.

WIA-LANG-003 표준은 弘益人間의 철학을 구현합니다:

**문화적 다양성 보존:**
- 각 구술 전통의 독특함 존중
- 소수 문화의 목소리 증폭
- 문화적 정체성 강화

**지식의 민주화:**
- 구술 전통을 디지털로 접근 가능하게
- 교육 및 연구 자원 제공
- 세대 간 지식 전승 촉진

**공동체 역량 강화:**
- 원주민 공동체에 도구 및 훈련 제공
- 자신의 전통을 스스로 기록하도록 지원
- 문화적 주권 존중

**미래 세대를 위한 유산:**
- 소멸 위기의 구술 전통 구출
- 영구적이고 접근 가능한 아카이브 생성
- 인류의 문화적 기억 보존

구술 전통을 보존함으로써, 우리는 인류 지식의 다양성을 보호하고, 각 문화의 독특한 세계관을 존중하며, 모든 사람이 인류의 풍부한 이야기 유산으로부터 배우고 영감을 받을 수 있도록 합니다.
