6.3 Posthumous Privacy Rights

The deceased retain privacy rights that must be respected and protected.

Core Privacy Principles

  • Memory Sanctity: Personal memories cannot be disclosed without explicit consent
  • Selective Revelation: Deceased controls what aspects are revealed to whom
  • Time-locked Secrets: Some information remains sealed for specified periods
  • Relationship Privacy: Interactions with specific individuals remain confidential
  • Thought Privacy: Internal reflections never disclosed without explicit consent

Privacy Zones

Zone Content Default Access Override
Public Published works, public statements Anyone None needed
Social Shared memories with groups Named individuals Group consent
Personal Private reflections, diary Executor only Court order
Sealed Time-locked content None until date None

6.5 Encryption Implementation

Python - Encryption Service
from cryptography.hazmat.primitives import hashes, serialization
from cryptography.hazmat.primitives.kdf.hkdf import HKDF
from cryptography.hazmat.primitives.ciphers.aead import AESGCM
from cryptography.hazmat.backends import default_backend
import os

class ConsciousnessEncryption:
    """
    Encryption service for consciousness data.
    Uses AES-256-GCM with key derivation from multiple sources.
    """

    def __init__(self, master_key: bytes, subject_id: str):
        self.master_key = master_key
        self.subject_id = subject_id

    def derive_key(self, purpose: str) -> bytes:
        """Derive purpose-specific key using HKDF."""
        hkdf = HKDF(
            algorithm=hashes.SHA256(),
            length=32,
            salt=self.subject_id.encode(),
            info=purpose.encode(),
            backend=default_backend()
        )
        return hkdf.derive(self.master_key)

    def encrypt_consciousness_data(
        self,
        data: bytes,
        data_type: str
    ) -> tuple[bytes, bytes, bytes]:
        """
        Encrypt consciousness data with type-specific key.
        Returns: (ciphertext, nonce, tag)
        """
        key = self.derive_key(f"consciousness_{data_type}")
        aesgcm = AESGCM(key)
        nonce = os.urandom(12)

        # Include data type in authenticated data
        aad = f"wia_afterlife:{self.subject_id}:{data_type}".encode()
        ciphertext = aesgcm.encrypt(nonce, data, aad)

        return ciphertext, nonce, aad

    def decrypt_consciousness_data(
        self,
        ciphertext: bytes,
        nonce: bytes,
        data_type: str
    ) -> bytes:
        """Decrypt consciousness data with verification."""
        key = self.derive_key(f"consciousness_{data_type}")
        aesgcm = AESGCM(key)
        aad = f"wia_afterlife:{self.subject_id}:{data_type}".encode()

        return aesgcm.decrypt(nonce, ciphertext, aad)
