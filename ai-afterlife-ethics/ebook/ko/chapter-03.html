3장: API 엔드포인트

3.1 디지털 의식 관리 API

WIA AI 사후 윤리 표준은 디지털 의식 프로필, 동의 기록, 부활 서비스 관리를 위한 RESTful API를 정의합니다.

의식 프로필 엔드포인트

POST /api/v1/consciousness/profiles

동의 검증과 함께 새 디지털 의식 프로필 생성

GET /api/v1/consciousness/profiles/{profileId}

권한 확인과 함께 의식 프로필 조회

PUT /api/v1/consciousness/profiles/{profileId}

의식 프로필 업데이트 (사전 동의 시에만)

DELETE /api/v1/consciousness/profiles/{profileId}

감사 추적과 함께 의식 프로필 영구 삭제

3.2 동의 관리 API

POST /api/v1/consent/records

암호화 증명과 함께 새 동의 기록 등록

GET /api/v1/consent/records/{subjectId}

대상자의 모든 동의 기록 조회

POST /api/v1/consent/verify

특정 부활 작업에 대한 동의 유효성 검증

POST /api/v1/consent/revoke

동의 철회 (권한 있는 당사자만)

3.3 디지털 부활 API

POST /api/v1/resurrection/initialize

디지털 부활 인스턴스 초기화

GET /api/v1/resurrection/instances/{instanceId}

부활 인스턴스 상태 조회

POST /api/v1/resurrection/instances/{instanceId}/interact

부활된 디지털 엔티티와 상호작용

DELETE /api/v1/resurrection/instances/{instanceId}

부활 인스턴스 영구 종료
