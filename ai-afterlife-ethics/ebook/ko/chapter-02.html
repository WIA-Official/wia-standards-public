2장: 데이터 스키마

2.1 의식 데이터 모델

WIA AI 사후 윤리 표준은 AI 보존에 적합한 형식으로 인간 의식, 기억, 성격 특성을 표현하기 위한 포괄적인 데이터 스키마를 정의합니다.

JSON 스키마 - 디지털 의식 프로필
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "WIA 디지털 의식 프로필",
  "type": "object",
  "properties": {
    "profile_id": {
      "type": "string",
      "format": "uuid",
      "description": "의식 프로필의 고유 식별자"
    },
    "subject": {
      "type": "object",
      "properties": {
        "legal_name": { "type": "string" },
        "birth_date": { "type": "string", "format": "date" },
        "death_date": { "type": "string", "format": "date" },
        "biometric_hash": { "type": "string" },
        "legal_jurisdiction": { "type": "string" }
      },
      "required": ["legal_name", "biometric_hash"]
    },
    "consciousness_level": {
      "type": "string",
      "enum": ["static_memorial", "interactive_bot", "digital_avatar", "consciousness_transfer"]
    },
    "consent_record": {
      "type": "object",
      "properties": {
        "consent_type": { "type": "string" },
        "consent_date": { "type": "string", "format": "date-time" },
        "consent_document_hash": { "type": "string" },
        "witnesses": { "type": "array", "items": { "type": "string" } }
      }
    }
  },
  "required": ["profile_id", "subject", "consciousness_level", "consent_record"]
}

2.2 성격 매트릭스 스키마

성격 매트릭스는 개인의 본질적인 특성, 가치관, 행동 패턴을 캡처합니다.

JSON 스키마 - 성격 매트릭스 정의
{
  "definitions": {
    "PersonalityMatrix": {
      "type": "object",
      "properties": {
        "big_five_traits": {
          "type": "object",
          "properties": {
            "openness": { "type": "number", "minimum": 0, "maximum": 1 },
            "conscientiousness": { "type": "number", "minimum": 0, "maximum": 1 },
            "extraversion": { "type": "number", "minimum": 0, "maximum": 1 },
            "agreeableness": { "type": "number", "minimum": 0, "maximum": 1 },
            "neuroticism": { "type": "number", "minimum": 0, "maximum": 1 }
          }
        },
        "core_values": {
          "type": "array",
          "items": {
            "type": "object",
            "properties": {
              "value": { "type": "string" },
              "priority": { "type": "integer", "minimum": 1, "maximum": 10 },
              "context": { "type": "string" }
            }
          }
        },
        "communication_style": {
          "type": "object",
          "properties": {
            "formality": { "type": "string", "enum": ["casual", "moderate", "formal"] },
            "verbosity": { "type": "string", "enum": ["concise", "moderate", "verbose"] },
            "emotional_expression": { "type": "string", "enum": ["reserved", "moderate", "expressive"] }
          }
        }
      }
    }
  }
}

2.4 동의 기록 스키마

포괄적인 동의 문서화는 윤리적 디지털 부활의 기본입니다.

JSON 스키마 - 동의 문서화
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "WIA 동의 기록",
  "type": "object",
  "properties": {
    "consent_id": { "type": "string", "format": "uuid" },
    "subject_id": { "type": "string" },
    "consent_type": {
      "type": "string",
      "enum": ["explicit_pre_mortem", "inferred_pre_mortem", "proxy", "next_of_kin"]
    },
    "scope": {
      "type": "object",
      "properties": {
        "allowed_levels": { "type": "array", "items": { "type": "string" } },
        "prohibited_uses": { "type": "array", "items": { "type": "string" } },
        "time_limit": { "type": "string" },
        "geographic_restrictions": { "type": "array", "items": { "type": "string" } }
      }
    },
    "attestation": {
      "type": "object",
      "properties": {
        "date": { "type": "string", "format": "date-time" },
        "witness_signatures": { "type": "array", "items": { "type": "string" } },
        "notarization": { "type": "boolean" },
        "legal_review": { "type": "boolean" },
        "document_hash": { "type": "string" }
      }
    }
  },
  "required": ["consent_id", "subject_id", "consent_type", "attestation"]
}
