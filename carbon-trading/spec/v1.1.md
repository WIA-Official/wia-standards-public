# WIA-FIN-025: Carbon Trading Standard v1.1

**Status:** Deprecated
**Release Date:** May 20, 2023
**Superseded By:** v2.0

## Changes from v1.0

### New Features
- Added support for carbon removal credits (separate from avoidance)
- Blockchain integration for credit tokenization
- WebSocket API for real-time market data
- Quality indicators for credits

### Improvements
- Enhanced API performance (50% faster)
- Better error handling and validation
- Expanded project types
- Improved documentation

### Bug Fixes
- Fixed double counting issue in multi-registry scenarios
- Corrected vintage year calculation for ongoing projects
- Resolved timezone issues in transaction timestamps

## Deprecated Features
- Legacy XML format (use JSON)
- v1.0 batch credit issuance endpoint

For full documentation, see v1.0 specification with above changes applied.
