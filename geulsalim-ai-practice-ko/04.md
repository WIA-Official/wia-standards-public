# 챕터 04 · 이미지 정규화 + 마스크 5종 자동 생성

## 학습 목표
- 1만 장 수집 이미지를 학습 가능한 1024×1024 정규화 텐서로 변환하는 파이프라인을 구축할 수 있다
- 고문서 손상을 모사하는 5가지 마스크 유형(Otsu·균열·결손·얼룩·랜덤)을 프로그램적으로 생성할 수 있다
- (이미지, 마스크, 타깃) 트리플 페어를 학습 데이터셋 구조에 맞춰 저장할 수 있다

---

## 이론

### 출발점 — 챕터 03에서 넘어온 데이터

챕터 03 라이선스 검증에서 **7,200장이 "승인(Green)"** 으로 통과했다. 목표인 1만 장에 2,800장이 부족하다. 이 갭은 두 방식으로 메운다:

- **Wikimedia Commons 재크롤링**: CC0·PD 카테고리에서 1,500장 추가 (재검증 후 ~1,200장 승인)
- **국립중앙박물관 OpenAPI 확장**: 키워드 "문인화·산수화·고지도" 추가로 1,800장 (재검증 후 ~1,600장 승인)

이렇게 총 **10,112장**으로 확정한 뒤 본 장의 정규화 파이프라인에 투입한다. 나는 처음에 "7,200장도 충분하겠지"라며 넘어갔다가, 검증 세트 분리(1,000장) 후 학습 세트가 6,200장으로 줄어 다양성 부족을 뒤늦게 깨달았다. **데이터는 목표보다 10~15% 여유를 두고 수집**하는 게 원칙이다.

### Inpainting 학습 데이터의 핵심 — 트리플 페어

텍스트→이미지 학습은 (텍스트, 이미지) 페어 하나면 충분하다. 하지만 Inpainting은 다르다. 모델이 "어디를 복원해야 하는지"를 학습하려면 **손상된 입력**과 **정답 이미지**를 함께 보여줘야 한다.

```
입력:  원본 이미지 × (1 - 마스크) + 노이즈 × 마스크
타깃:  원본 이미지
조건:  마스크 (어디를 채워야 하는지)
```

나는 첫 Inpainting 실험에서 마스크 한 종류(랜덤 사각형)만 썼다. 결과는 참혹했다 — 모델이 "사각형 안만 채우는" 편향이 생겨서, 실제 균열 패턴에는 전혀 작동하지 않았다. **마스크 다양성이 모델의 일반화 능력을 결정한다.**

### 고문서 손상 5가지 실제 패턴

실제 국립중앙박물관·규장각 소장 자료의 손상을 분석한 결과:

| 마스크 유형 | 실제 손상 원인 | 형태 특징 | 학습 비중 |
|---|---|---|---|
| **Otsu** | 먹 번짐, 퇴색 | 임계값 이진화 영역 | 25% |
| **균열(Crack)** | 종이 노화, 접힘 | 얇고 긴 선 패턴 | 20% |
| **결손(Missing)** | 찢어짐, 구멍 | 불규칙 덩어리 | 25% |
| **얼룩(Stain)** | 곰팡이, 수분 | 원형·타원 | 15% |
| **랜덤(Random)** | 데이터 증강 | 블록·스트로크 | 15% |

이 비율은 내가 1,000장 샘플을 육안 분류해서 산출한 분포다. 학습 시 `random.choices(types, weights=[0.25, 0.20, 0.25, 0.15, 0.15])`로 샘플링하면 실제 손상 분포를 재현한다.

### 정규화 원칙 — 해상도와 채널

- **해상도**: 1024×1024 (SDXL 네이티브). 512×512는 너무 작아 세부 묘사 손실, 2048×2048은 GPU 메모리 폭증.
- **채널**: RGB 3채널. 고문서 흑백 스캔도 RGB로 변환 (SDXL는 3채널만 입력).
- **값 범위**: `[-1, 1]` (SDXL 표준). 0~255 → /127.5 - 1.
- **종횡비**: 1:1 강제. 레터박스(padding)로 정사각형화 후 리사이즈. Crop은 내용 손실.

나는 초기에 Crop 정규화를 썼다가 화면 가장자리 글자가 통째로 잘려 나가는 걸 뒤늦게 발견했다. **고문서는 Padding이 정답이다.**

## 실전 코드

```python
# 1만 장 이미지 정규화 + 5종 마스크 자동 생성 파이프라인
# pip install pillow numpy opencv-python scipy tqdm

import os
import random
import hashlib
import numpy as np
import cv2
from pathlib import Path
from PIL import Image, ImageDraw, ImageFilter
from scipy import ndimage
from tqdm import tqdm
from dataclasses import dataclass
from typing import Literal


MaskType = Literal["otsu", "crack", "missing", "stain", "random"]


@dataclass
class NormalizeConfig:
    """정규화 설정"""
    target_size:  int = 1024
    pad_color:    tuple = (255, 255, 255)  # 종이 흰색
    mask_ratio_min: float = 0.05           # 마스크 최소 면적 비율
    mask_ratio_max: float = 0.35           # 마스크 최대 면적 비율


class ImageNormalizer:
    """레터박스 Padding 후 1024 정사각형 정규화"""

    def __init__(self, cfg: NormalizeConfig = NormalizeConfig()):
        self.cfg = cfg

    def normalize(self, img: Image.Image) -> Image.Image:
        """
        1) RGB 변환
        2) 종횡비 유지 최대 리사이즈
        3) 흰색 Padding으로 정사각형화
        """
        if img.mode != "RGB":
            img = img.convert("RGB")

        w, h = img.size
        side = self.cfg.target_size
        scale = side / max(w, h)
        new_w, new_h = int(w * scale), int(h * scale)
        img = img.resize((new_w, new_h), Image.LANCZOS)

        # 흰색 캔버스 중앙 배치
        canvas = Image.new("RGB", (side, side), self.cfg.pad_color)
        offset = ((side - new_w) // 2, (side - new_h) // 2)
        canvas.paste(img, offset)
        return canvas


class MaskGenerator:
    """고문서 손상 5종 마스크 생성기"""

    def __init__(self, size: int = 1024, cfg: NormalizeConfig = NormalizeConfig()):
        self.size = size
        self.cfg = cfg

    # ──────── 1. Otsu 이진화 기반 ────────
    def otsu_mask(self, img: Image.Image) -> np.ndarray:
        """먹 번짐·퇴색 영역 추정 (어두운 영역을 손상으로 간주)"""
        gray = np.array(img.convert("L"))
        _, binary = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)
        # 팽창으로 번짐 효과
        kernel = np.ones((7, 7), np.uint8)
        dilated = cv2.dilate(binary, kernel, iterations=2)
        # 면적 비율 조정
        target_ratio = random.uniform(self.cfg.mask_ratio_min, self.cfg.mask_ratio_max)
        return self._adjust_ratio(dilated, target_ratio)

    # ──────── 2. 균열(Crack) ────────
    def crack_mask(self) -> np.ndarray:
        """얇고 긴 선 패턴 — 베지어 곡선 + 분기"""
        mask = np.zeros((self.size, self.size), np.uint8)
        num_cracks = random.randint(3, 8)

        for _ in range(num_cracks):
            # 시작점, 끝점, 제어점
            p0 = (random.randint(0, self.size), random.randint(0, self.size))
            p2 = (random.randint(0, self.size), random.randint(0, self.size))
            p1 = (
                (p0[0] + p2[0]) // 2 + random.randint(-150, 150),
                (p0[1] + p2[1]) // 2 + random.randint(-150, 150),
            )
            # 베지어 곡선 점 생성
            points = []
            for t in np.linspace(0, 1, 80):
                x = int((1 - t) ** 2 * p0[0] + 2 * (1 - t) * t * p1[0] + t**2 * p2[0])
                y = int((1 - t) ** 2 * p0[1] + 2 * (1 - t) * t * p1[1] + t**2 * p2[1])
                points.append((x, y))
            # 폭 변화
            thickness = random.randint(2, 6)
            for i in range(len(points) - 1):
                cv2.line(mask, points[i], points[i + 1], 255, thickness)
            # 분기 추가
            if random.random() < 0.5:
                mid = points[len(points) // 2]
                end = (mid[0] + random.randint(-200, 200),
                       mid[1] + random.randint(-200, 200))
                cv2.line(mask, mid, end, 255, thickness // 2 + 1)

        return mask

    # ──────── 3. 결손(Missing) ────────
    def missing_mask(self) -> np.ndarray:
        """찢어짐·구멍 — 불규칙 다각형 + 노이즈 경계"""
        mask = np.zeros((self.size, self.size), np.uint8)
        num_holes = random.randint(1, 4)

        for _ in range(num_holes):
            cx = random.randint(100, self.size - 100)
            cy = random.randint(100, self.size - 100)
            r = random.randint(50, 250)
            num_pts = random.randint(8, 16)

            # 불규칙 다각형
            angles = np.sort(np.random.uniform(0, 2 * np.pi, num_pts))
            pts = []
            for a in angles:
                rr = r * random.uniform(0.5, 1.3)
                pts.append([int(cx + rr * np.cos(a)), int(cy + rr * np.sin(a))])
            cv2.fillPoly(mask, [np.array(pts)], 255)

        # 경계 거칠게
        mask = cv2.GaussianBlur(mask, (11, 11), 0)
        mask = (mask > 127).astype(np.uint8) * 255
        return mask

    # ──────── 4. 얼룩(Stain) ────────
    def stain_mask(self) -> np.ndarray:
        """곰팡이·수분 — 타원 + 가우시안 그라디언트"""
        mask = np.zeros((self.size, self.size), np.uint8)
        num_stains = random.randint(2, 6)

        for _ in range(num_stains):
            cx = random.randint(100, self.size - 100)
            cy = random.randint(100, self.size - 100)
            ax = random.randint(40, 180)
            ay = random.randint(40, 180)
            angle = random.randint(0, 180)
            cv2.ellipse(mask, (cx, cy), (ax, ay), angle, 0, 360, 255, -1)

        # 부드러운 경계
        mask = cv2.GaussianBlur(mask, (41, 41), 0)
        mask = (mask > 100).astype(np.uint8) * 255
        return mask

    # ──────── 5. 랜덤(Random) ────────
    def random_mask(self) -> np.ndarray:
        """블록 + 스트로크 혼합 — 데이터 증강용"""
        mask = np.zeros((self.size, self.size), np.uint8)

        # 랜덤 사각형 3~8개
        for _ in range(random.randint(3, 8)):
            x1 = random.randint(0, self.size - 50)
            y1 = random.randint(0, self.size - 50)
            x2 = x1 + random.randint(30, 200)
            y2 = y1 + random.randint(30, 200)
            cv2.rectangle(mask, (x1, y1), (x2, y2), 255, -1)

        # 자유 스트로크 2~5개
        for _ in range(random.randint(2, 5)):
            pts = []
            x, y = random.randint(0, self.size), random.randint(0, self.size)
            for _ in range(random.randint(10, 30)):
                pts.append((x, y))
                x += random.randint(-30, 30)
                y += random.randint(-30, 30)
                x = max(0, min(self.size - 1, x))
                y = max(0, min(self.size - 1, y))
            for i in range(len(pts) - 1):
                cv2.line(mask, pts[i], pts[i + 1], 255, random.randint(5, 30))

        return mask

    # ──────── 면적 비율 조정 ────────
    def _adjust_ratio(self, mask: np.ndarray, target: float) -> np.ndarray:
        """마스크 면적을 target 비율에 맞게 팽창/침식"""
        current = mask.sum() / 255 / (self.size * self.size)
        if current < target * 0.9:
            iterations = int((target - current) * 30) + 1
            mask = cv2.dilate(mask, np.ones((5, 5), np.uint8), iterations=iterations)
        elif current > target * 1.1:
            iterations = int((current - target) * 30) + 1
            mask = cv2.erode(mask, np.ones((5, 5), np.uint8), iterations=iterations)
        return mask

    # ──────── 통합 생성 ────────
    def generate(self, img: Image.Image, mask_type: MaskType) -> np.ndarray:
        """유형별 마스크 생성 진입점"""
        if mask_type == "otsu":    return self.otsu_mask(img)
        if mask_type == "crack":   return self.crack_mask()
        if mask_type == "missing": return self.missing_mask()
        if mask_type == "stain":   return self.stain_mask()
        if mask_type == "random":  return self.random_mask()
        raise ValueError(f"Unknown mask type: {mask_type}")


class TripletBuilder:
    """
    (이미지, 마스크, 타깃) 트리플 페어 빌더

    디렉토리 구조:
        output/
          images/    1024×1024 원본 (타깃)
          masks/     대응 마스크 (0/255)
          inputs/    이미지 × (1 - 마스크/255) [선택 사항]
          metadata.jsonl
    """

    # 실제 손상 분포
    MASK_WEIGHTS = {
        "otsu":    0.25,
        "crack":   0.20,
        "missing": 0.25,
        "stain":   0.15,
        "random":  0.15,
    }

    def __init__(self, input_dir: str, output_dir: str):
        self.input_dir  = Path(input_dir)
        self.output_dir = Path(output_dir)
        (self.output_dir / "images").mkdir(parents=True, exist_ok=True)
        (self.output_dir / "masks").mkdir(parents=True, exist_ok=True)
        self.normalizer = ImageNormalizer()
        self.mask_gen   = MaskGenerator()

    def build(self, variants_per_image: int = 3) -> int:
        """
        이미지 1장당 variants_per_image 개 마스크 생성.
        1만 장 × 3 = 3만 페어.
        """
        import json
        meta_path = self.output_dir / "metadata.jsonl"
        files = [p for p in self.input_dir.rglob("*")
                 if p.suffix.lower() in [".jpg", ".jpeg", ".png"]]

        count = 0
        mask_types = list(self.MASK_WEIGHTS.keys())
        weights    = list(self.MASK_WEIGHTS.values())

        with open(meta_path, "w", encoding="utf-8") as meta_f:
            for img_path in tqdm(files, desc="트리플 빌드"):
                try:
                    img = Image.open(img_path)
                    img = self.normalizer.normalize(img)

                    base_id = hashlib.md5(str(img_path).encode()).hexdigest()[:10]
                    img_name = f"{base_id}.jpg"
                    img.save(self.output_dir / "images" / img_name, quality=95)

                    for v in range(variants_per_image):
                        mtype = random.choices(mask_types, weights=weights)[0]
                        mask  = self.mask_gen.generate(img, mtype)

                        mask_name = f"{base_id}_{v}_{mtype}.png"
                        cv2.imwrite(str(self.output_dir / "masks" / mask_name), mask)

                        meta_f.write(json.dumps({
                            "image":    f"images/{img_name}",
                            "mask":     f"masks/{mask_name}",
                            "mask_type": mtype,
                            "mask_ratio": float(mask.sum() / 255 / mask.size),
                        }, ensure_ascii=False) + "\n")

                        count += 1
                except Exception as e:
                    print(f"Skip {img_path.name}: {e}")
                    continue

        print(f"\n트리플 페어 {count}개 생성 → {self.output_dir}")
        return count


if __name__ == "__main__":
    builder = TripletBuilder(
        input_dir  = "./korean_images_10k",
        output_dir = "./triplet_dataset",
    )
    builder.build(variants_per_image=3)

    # 샘플 시각화
    img  = Image.open("./triplet_dataset/images").iterdir().__next__()
    print(f"샘플: {img}")
```

## 함정 카드

1. **"마스크 면적 분포를 무시한다"** → 마스크 비율이 1%면 거의 손상 없음, 70%면 거의 전부 복원. 실제 고문서 손상은 5~35% 범위가 대부분. 이 범위를 벗어난 마스크는 학습에 해롭다 — `mask_ratio_min/max`로 반드시 제한.

2. **"하나의 이미지에 같은 마스크 패턴만 쓴다"** → 이미지당 3개 Variant를 만들 때 전부 Crack 또는 전부 Random이면 편향 발생. `random.choices(weights=...)`로 5종 고루 분포시키고, Variant 간에도 다양화.

3. **"마스크 경계가 너무 깔끔하다"** → 실제 손상은 경계가 거칠고 흐릿하다. `cv2.GaussianBlur` → 재이진화로 자연스러운 경계 만들기. 경계 선명한 마스크는 사각형 편향을 유발한다.

4. **"Padding 색을 검정으로 쓴다"** → 학습 시 검정 영역을 "복원 대상"으로 혼동할 수 있다. 고문서 종이 색(흰색 또는 약간 누런색 `(250, 245, 230)`)으로 설정. 색 선택도 도메인 지식.

5. **"변환 결과 검수를 스킵한다"** → 1만 × 3 = 3만 페어 중 반드시 샘플 100개를 육안 검수하라. 내 경우 이 단계에서 `crack_mask`가 0 크랙 생성하는 버그(loop 변수 오류)를 찾았다. 자동화 파이프라인일수록 샘플링 검수가 유일한 안전장치.

## 연습문제

**문제 1.** 1,024×1,024 해상도 대신 512×512로 학습할 때의 장단점 3가지씩 설명하라.

> **해설:** **장점:** ① GPU VRAM 1/4로 감소 (24GB → 6GB), 소규모 GPU 학습 가능 ② 배치 크기 4배 증가 가능, 수렴 속도 빠름 ③ 학습 시간 단축. **단점:** ① SDXL 네이티브 해상도가 1024라 다운샘플 시 사전학습 가중치와 미스매치 ② 고문서 작은 글자 세부 디테일 소실 ③ 추론 시 해상도 업샘플 과정에서 품질 저하. **권장:** 시제품은 512로 빠르게 검증, 최종은 1024로 재학습.

**문제 2.** `crack_mask`에서 베지어 곡선 대신 직선만 사용하면 어떤 문제가 생기는가?

> **해설:** 실제 종이 균열은 섬유 방향을 따라 휘어진다. 직선만 쓰면 모델이 "일직선 손상만 복원하는" 편향이 생겨, 실제 구부러진 균열에는 경계가 부자연스럽게 이어진다. 또한 두 직선의 교차점이 L자로 꺾여 보여 비현실적. 베지어 곡선 또는 Perlin noise 변위로 자연스러운 곡률을 만들어야 한다.

**문제 3.** 생성된 마스크 면적의 실제 분포가 `mask_ratio` 히스토그램에서 양극단(5% 근처와 35% 근처)에 몰려 있다면 원인과 해결책은?

> **해설:** 원인: `_adjust_ratio`의 팽창/침식 로직이 단일 Step에서 과도한 보정을 하면 경계값에 수렴한다. 또는 초기 마스크 생성이 극단적인 분포를 보일 때. **해결:** ① 목표 비율을 `random.uniform(0.05, 0.35)` 대신 `np.random.beta(2, 2) * 0.3 + 0.05`로 중앙 집중 분포로 변경 ② `_adjust_ratio` iteration 수를 점진적 상승(1→2→3)으로 세밀 조정 ③ 히스토그램 이상치 500개를 재생성.
