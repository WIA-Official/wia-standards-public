# 챕터 12 · 검증 메트릭 — SSIM · PSNR · LPIPS · CLIP Score

## 학습 목표
- SSIM·PSNR·LPIPS·CLIP Score 네 메트릭의 측정 대상과 해석 기준을 각각 설명할 수 있다
- 체크포인트별로 검증 세트에 네 메트릭을 일괄 적용하는 자동 평가 스크립트를 작성할 수 있다
- 메트릭 숫자만 보고 "이 체크포인트를 배포해도 되는가"를 판단하는 기준값을 정할 수 있다

---

## 이론

### 왜 하나로 충분하지 않은가

각 메트릭은 서로 다른 측면을 본다:

| 메트릭 | 본질 | 범위 | 강점 | 약점 |
|---|---|---|---|---|
| **PSNR** | 픽셀 MSE의 로그 | 높을수록 good (∞=동일) | 직관적, 빠름 | 지각적 유사성 반영 X |
| **SSIM** | 구조·휘도·대비 유사도 | [-1, 1] (1=동일) | 구조 변화 민감 | 전역 통계, 국소 실패 못 잡음 |
| **LPIPS** | VGG 피처 거리 | [0, ~1] (0=동일) | 사람 눈 상관 0.7+ | 느림, 네트워크 선택 편향 |
| **CLIP Score** | CLIP 텍스트-이미지 일치 | [0, ~40] | 의미적 일치 | 픽셀 품질 무관 |

나는 초반에 PSNR만 보고 "38dB 넘으면 잘 된 거지"라고 믿었다. 그런데 육안 검수하면 글자가 뭉개진 샘플이 많았다. SSIM·LPIPS를 추가한 뒤에야 "PSNR은 좋은데 LPIPS는 나쁜" 체크포인트를 걸러낼 수 있었다.

더 잊지 못할 건 첫 외부 시연 때였다. 나는 메트릭 표를 자랑스럽게 띄우며 "SSIM 0.92 달성"이라고 말했다. 한 분이 복원 이미지를 한참 들여다보시더니 조용히 물으셨다. "글자 경계가 좀 이상해요. 이거 AI가 지어낸 거죠?" 그 분의 표정은 비난이 아니라 걱정이었다. 그 날 이후 나는 **LPIPS와 사용자 육안 평가를 배포 게이트에 함께** 넣었다. 숫자는 나를 위로해 주지만, 그분의 한 마디가 모델을 바꿨다.

**배포 게이트: 4개 메트릭 전부 임계값 통과해야 배포.**

### 각 메트릭 임계값 (고문서 복원 도메인)

내가 1,000장 정답·예측 쌍으로 측정한 분포 기반 권장값:

| 메트릭 | 불합격 | 보통 | 합격 | 우수 |
|---|---|---|---|---|
| PSNR (dB) | < 24 | 24–28 | 28–33 | > 33 |
| SSIM | < 0.75 | 0.75–0.85 | 0.85–0.92 | > 0.92 |
| LPIPS | > 0.30 | 0.20–0.30 | 0.10–0.20 | < 0.10 |
| CLIP Score | < 20 | 20–25 | 25–30 | > 30 |

**주의**: 마스크 영역만 잘라서 재측정하면 SSIM·PSNR이 크게 떨어진다. "마스크 안쪽만"과 "전체 이미지"를 **둘 다** 기록하는 게 정확한 평가.

### 왜 CLIP Score도 필요한가

Inpainting은 "복원된 결과가 원본의 의미를 유지하는가"도 중요하다. 예: "조선 시대 풍속화"라고 프롬프트를 준 복원이 현대적 디지털 아트 텍스처로 나오면 CLIP Score가 떨어진다. **의미 일치 품질 지표**로 CLIP Score가 필요.

### FID는 왜 안 쓰는가

FID(Fréchet Inception Distance)는 "세트 간 분포 거리"로, 개별 이미지가 아닌 **컬렉션 통계**를 본다. 학습 중 체크포인트 비교에 유용하지만 단일 이미지 품질 평가엔 부적합. 검증 세트가 2,000장 이상일 때만 유의미. 실무에선 SSIM/LPIPS가 더 빠른 피드백을 준다.

## 실전 코드

```python
# 체크포인트 자동 평가기 — SSIM·PSNR·LPIPS·CLIP Score
# pip install torchmetrics scikit-image lpips open-clip-torch tqdm

import torch
import numpy as np
from pathlib import Path
from PIL import Image
from dataclasses import dataclass, asdict
from typing import List
import json
import pandas as pd
from tqdm import tqdm

from skimage.metrics import structural_similarity as ski_ssim
from skimage.metrics import peak_signal_noise_ratio as ski_psnr


DEVICE = "cuda" if torch.cuda.is_available() else "cpu"


# ──────────────────────────────────────────────
# 1. 개별 메트릭 계산기
# ──────────────────────────────────────────────
class MetricSet:
    def __init__(self, device: str = DEVICE):
        import lpips
        import open_clip

        # LPIPS
        self.lpips_model = lpips.LPIPS(net="vgg").to(device).eval()
        for p in self.lpips_model.parameters(): p.requires_grad = False

        # CLIP (open_clip)
        self.clip_model, _, self.clip_preprocess = open_clip.create_model_and_transforms(
            "ViT-L-14", pretrained="openai", device=device,
        )
        self.clip_tokenizer = open_clip.get_tokenizer("ViT-L-14")
        self.clip_model.eval()

        self.device = device

    # PSNR / SSIM (skimage 기반, CPU numpy)
    @staticmethod
    def psnr_ssim(pred_np: np.ndarray, target_np: np.ndarray) -> dict:
        """[0, 255] uint8 numpy 입력, (H, W, 3)"""
        psnr = ski_psnr(target_np, pred_np, data_range=255)
        ssim = ski_ssim(target_np, pred_np, data_range=255,
                        channel_axis=-1, win_size=11)
        return {"psnr": float(psnr), "ssim": float(ssim)}

    @torch.no_grad()
    def lpips(self, pred_pil: Image.Image, target_pil: Image.Image) -> float:
        """PIL → [-1, 1] tensor → VGG LPIPS"""
        def _pp(img):
            arr = np.array(img.resize((256, 256))).astype(np.float32) / 127.5 - 1.0
            return torch.from_numpy(arr).permute(2, 0, 1).unsqueeze(0).to(self.device)
        p = _pp(pred_pil); t = _pp(target_pil)
        return float(self.lpips_model(p, t).item())

    @torch.no_grad()
    def clip_score(self, image_pil: Image.Image, text: str) -> float:
        """
        CLIP Score = cos(image_embed, text_embed) × 100
        (OpenAI convention; 20 미만은 의미 미스매치)
        """
        img_t = self.clip_preprocess(image_pil).unsqueeze(0).to(self.device)
        txt_t = self.clip_tokenizer([text]).to(self.device)

        img_f = self.clip_model.encode_image(img_t)
        txt_f = self.clip_model.encode_text(txt_t)
        img_f = img_f / img_f.norm(dim=-1, keepdim=True)
        txt_f = txt_f / txt_f.norm(dim=-1, keepdim=True)
        cos = (img_f @ txt_f.T).item()
        return float(cos * 100)

    # ──────── 통합 평가 ────────
    def eval_one(
        self,
        pred:   Image.Image,
        target: Image.Image,
        mask:   Image.Image,
        prompt: str,
    ) -> dict:
        """한 쌍 (pred, target, mask)에 대해 4 메트릭 + 마스크 영역 따로"""
        # [0, 255] uint8
        pred_np   = np.array(pred.convert("RGB"))
        target_np = np.array(target.convert("RGB"))
        mask_np   = (np.array(mask.convert("L")) > 127).astype(bool)  # (H, W)

        # 전체 이미지 메트릭
        full = self.psnr_ssim(pred_np, target_np)
        lp   = self.lpips(pred, target)
        cs   = self.clip_score(pred, prompt)

        # 마스크 영역만 (최소 경계 사각형으로 crop 후 계산)
        inside_psnr = inside_ssim = 0.0
        if mask_np.any():
            ys, xs = np.where(mask_np)
            y0, y1 = ys.min(), ys.max()
            x0, x1 = xs.min(), xs.max()
            pred_crop   = pred_np[y0:y1+1, x0:x1+1]
            target_crop = target_np[y0:y1+1, x0:x1+1]
            # 최소 크기 확보 (SSIM win_size=11)
            if pred_crop.shape[0] >= 11 and pred_crop.shape[1] >= 11:
                in_m = self.psnr_ssim(pred_crop, target_crop)
                inside_psnr = in_m["psnr"]
                inside_ssim = in_m["ssim"]

        return {
            "psnr_full":    full["psnr"],
            "ssim_full":    full["ssim"],
            "lpips":        lp,
            "clip_score":   cs,
            "psnr_inside":  inside_psnr,
            "ssim_inside":  inside_ssim,
            "mask_ratio":   float(mask_np.sum()) / mask_np.size,
        }


# ──────────────────────────────────────────────
# 2. 체크포인트 자동 평가기
# ──────────────────────────────────────────────
@dataclass
class EvalConfig:
    val_image_dir:  str       # 검증 원본 이미지
    val_mask_dir:   str       # 대응 마스크
    output_dir:     str       # 평가 결과 저장
    prompt:         str = "Korean traditional document restoration"
    num_samples:    int = 100
    inference_steps: int = 30
    guidance_scale:  float = 7.5


class CheckpointEvaluator:
    """
    체크포인트 1개 또는 여러 개를 같은 검증 세트로 평가.
    결과를 CSV로 저장하고 체크포인트 간 비교 표 생성.
    """
    def __init__(self, base_model_id: str, cfg: EvalConfig):
        self.base_model_id = base_model_id
        self.cfg = cfg
        self.metrics = MetricSet()

    def _load_pipeline_with_lora(self, lora_path: str):
        from diffusers import StableDiffusionXLInpaintPipeline
        from peft import LoraConfig
        pipe = StableDiffusionXLInpaintPipeline.from_pretrained(
            self.base_model_id, torch_dtype=torch.float16, variant="fp16",
        ).to(DEVICE)

        # LoRA 어댑터 주입 후 가중치 로드
        lora_cfg = LoraConfig(
            r=16, lora_alpha=16,
            target_modules=["to_q", "to_k", "to_v", "to_out.0"],
        )
        pipe.unet.add_adapter(lora_cfg)
        from safetensors.torch import load_file
        state = load_file(lora_path, device="cpu")
        pipe.unet.load_state_dict(state, strict=False)
        pipe.unet.to(DEVICE)

        pipe.set_progress_bar_config(disable=True)
        return pipe

    def evaluate_one_checkpoint(self, lora_path: str, tag: str) -> pd.DataFrame:
        pipe = self._load_pipeline_with_lora(lora_path)
        records = []

        val_images = sorted(Path(self.cfg.val_image_dir).glob("*.jpg"))[:self.cfg.num_samples]
        for img_path in tqdm(val_images, desc=f"평가 {tag}"):
            mask_path = Path(self.cfg.val_mask_dir) / img_path.with_suffix(".png").name
            if not mask_path.exists():
                continue

            target = Image.open(img_path).convert("RGB").resize((1024, 1024))
            mask   = Image.open(mask_path).convert("L").resize((1024, 1024))

            # 입력 이미지 = target × (1 - mask) (회색 채움)
            target_np = np.array(target)
            mask_np   = np.array(mask) > 127
            input_np  = target_np.copy()
            input_np[mask_np] = 128  # gray fill
            input_img = Image.fromarray(input_np)

            with torch.autocast("cuda", dtype=torch.float16):
                pred = pipe(
                    prompt=self.cfg.prompt,
                    image=input_img,
                    mask_image=mask,
                    num_inference_steps=self.cfg.inference_steps,
                    guidance_scale=self.cfg.guidance_scale,
                    generator=torch.Generator(DEVICE).manual_seed(42),
                ).images[0]

            metrics = self.metrics.eval_one(pred, target, mask, self.cfg.prompt)
            metrics["filename"]   = img_path.name
            metrics["checkpoint"] = tag
            records.append(metrics)

        df = pd.DataFrame(records)
        out_csv = Path(self.cfg.output_dir) / f"eval_{tag}.csv"
        out_csv.parent.mkdir(parents=True, exist_ok=True)
        df.to_csv(out_csv, index=False)
        print(f"\n>> {tag} 평균:")
        print(df[["psnr_full", "ssim_full", "lpips", "clip_score",
                  "psnr_inside", "ssim_inside"]].mean())
        return df

    def compare_checkpoints(self, lora_paths: dict) -> pd.DataFrame:
        """
        lora_paths = {"step_10000": "./ckpt-10000.safetensors", ...}
        반환: 체크포인트별 평균 메트릭 비교 DataFrame
        """
        all_dfs = []
        for tag, path in lora_paths.items():
            df = self.evaluate_one_checkpoint(path, tag)
            all_dfs.append(df)

        combined = pd.concat(all_dfs, ignore_index=True)
        summary = combined.groupby("checkpoint").agg({
            "psnr_full":    "mean",
            "ssim_full":    "mean",
            "lpips":        "mean",
            "clip_score":   "mean",
            "psnr_inside":  "mean",
            "ssim_inside":  "mean",
        }).round(3)

        print("\n" + "=" * 60)
        print("체크포인트 비교")
        print("=" * 60)
        print(summary)

        out = Path(self.cfg.output_dir) / "comparison.csv"
        summary.to_csv(out)
        return summary


# ──────────────────────────────────────────────
# 3. 배포 게이트 — 임계값 검사
# ──────────────────────────────────────────────
DEPLOY_THRESHOLDS = {
    "psnr_full":   28.0,
    "ssim_full":   0.85,
    "lpips":       0.20,   # 이건 "이하"
    "clip_score":  25.0,
    "psnr_inside": 22.0,   # 마스크 안쪽은 기준 완화
    "ssim_inside": 0.70,
}


def check_deploy_gate(summary: pd.DataFrame, tag: str) -> dict:
    """특정 체크포인트가 배포 가능한지 판단"""
    row = summary.loc[tag]
    result = {"ok": True, "violations": []}

    for metric, threshold in DEPLOY_THRESHOLDS.items():
        value = float(row[metric])
        if metric == "lpips":
            if value > threshold:
                result["ok"] = False
                result["violations"].append(
                    f"{metric}: {value:.3f} > 임계 {threshold}"
                )
        else:
            if value < threshold:
                result["ok"] = False
                result["violations"].append(
                    f"{metric}: {value:.3f} < 임계 {threshold}"
                )
    return result


if __name__ == "__main__":
    cfg = EvalConfig(
        val_image_dir="./val/images",
        val_mask_dir ="./val/masks",
        output_dir   ="./eval_output",
        num_samples=100,
    )
    evaluator = CheckpointEvaluator(
        base_model_id="diffusers/stable-diffusion-xl-1.0-inpainting-0.1",
        cfg=cfg,
    )

    lora_paths = {
        "step_5000":  "./checkpoints/lora_5000.safetensors",
        "step_15000": "./checkpoints/lora_15000.safetensors",
        "step_30000": "./checkpoints/lora_30000.safetensors",
    }
    summary = evaluator.compare_checkpoints(lora_paths)

    # 배포 게이트
    for tag in summary.index:
        gate = check_deploy_gate(summary, tag)
        status = "✅ 배포 가능" if gate["ok"] else "❌ 배포 불가"
        print(f"\n{tag}: {status}")
        for v in gate["violations"]:
            print(f"  - {v}")
```

## 함정 카드

1. **"같은 시드 없이 평가한다"** → 매번 `generator=torch.Generator().manual_seed(42)` 없이 추론하면 샘플링 노이즈로 메트릭이 ±2점 흔들린다. 체크포인트 비교가 불가능. 반드시 고정 시드.

2. **"mask inside 메트릭을 안 본다"** → 전체 이미지 SSIM이 0.95여도 마스크 영역만 보면 0.5인 경우가 흔하다. 마스크 영역이 전체의 10%라면 전체 메트릭은 복원 품질을 거의 측정 못 함. **inside 메트릭이 진짜 학습 지표**.

3. **"LPIPS를 256x256으로 리사이즈 후 계산한다"** → LPIPS VGG는 224x224 전처리 기준이지만 256은 무관. 문제는 **1024 → 256 리사이즈**로 디테일이 날아가 LPIPS가 실제보다 낮게(좋게) 나온다. 실제 배포 해상도(1024 또는 512)에서 계산해야 대표성 있음.

4. **"CLIP Score 임계값을 MSCOCO 기준(0.25)에서 가져온다"** → CLIP Score는 이미지 분포에 따라 절대값이 다르다. 고문서 도메인은 일반 사진보다 CLIP 분포가 낮다(보통 20~28). **자체 검증 세트로 baseline 분포를 먼저 측정**하고 임계값을 그 기반으로 정해야 함.

5. **"첫 체크포인트부터 마지막까지 모두 평가"** → 체크포인트 10개 × 100샘플 = 1,000 추론 × 30초 = 8시간. 실용: 처음·중간·끝 3개만 비교, 최고 구간 찾은 후 그 근방만 세밀 평가.

## 연습문제

**문제 1.** `psnr_full=32, ssim_full=0.91, lpips=0.25, clip_score=28`인 체크포인트. 배포 게이트를 통과하는가? 안 된다면 어떤 조치를?

> **해설:** **불합격.** LPIPS 0.25가 임계 0.20을 초과 — "픽셀은 비슷하지만 사람 눈엔 부자연스러움"의 시그니처. 원인 후보: ① 마스크 경계 아티팩트 ② 텍스처 과도한 부드러움(LPIPS는 이걸 penalize). 조치: ① 5~10개 실제 샘플을 wandb에 업로드해 육안 확인 ② LPIPS 0.25인 워스트 10개를 분리해 어떤 유형 손상에서 나쁜지 진단 ③ loss의 λ_lpips를 0.1 → 0.3으로 올려 추가 학습.

**문제 2.** 같은 체크포인트를 서로 다른 `guidance_scale`(CFG)로 평가하면 LPIPS가 달라진다. 왜이며 어떻게 공정 비교하는가?

> **해설:** CFG가 높으면(10+) 이미지가 선명해지지만 "CFG 아티팩트"(과채도·경계 강조)가 생겨 LPIPS가 나빠진다. CFG 낮으면(5) 자연스럽지만 프롬프트 반영 약함 → CLIP Score 하락. 공정 비교: **CFG=7.5 고정**하고 체크포인트별 비교. CFG 자체를 튜닝하려면 따로 CFG sweep으로 최적 CFG를 먼저 찾고, 그 후 체크포인트 비교.

**문제 3.** 검증 세트 100장에서 `psnr_inside`의 표준편차가 8.5로 크다(평균 25). 의미와 처리법은?

> **해설:** 모델이 **일부 손상 유형은 잘, 일부는 못** 복원한다는 신호. 평균만 보면 중간 수준이지만 실제로는 bimodal 분포. 처리: ① `mask_type`별로 그룹핑(`df.groupby("mask_type")`)해서 어떤 유형이 나쁜지 식별. 예: crack=PSNR 30, missing=PSNR 18. ② 부족한 유형 데이터를 3배 증강해 추가 학습. ③ 극단 실패 샘플 5~10개를 육안 검수 — 데이터셋 자체 노이즈인지 모델 한계인지 구분.
