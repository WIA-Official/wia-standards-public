# 챕터 03 · 데이터 라이선스 검증 체크리스트

## 학습 목표
- CC0 / CC BY / 공공누리 1유형 등 주요 라이선스 조건을 구분하고 프로젝트에 적용할 수 있다
- 수집한 이미지 각각의 라이선스를 자동으로 분류하는 스크립트를 작성할 수 있다
- 라이선스 위반 시 발생하는 법적·비즈니스 리스크를 설명할 수 있다

---

## 이론

### 라이선스 무지는 프로젝트를 죽인다

AI 챔피언 대회에서 심사위원에게 "데이터 출처와 라이선스를 말해달라"는 질문이 반드시 나온다. 여기서 "인터넷에서 긁어왔습니다"라고 답하면 그 자리에서 탈락이다.

나는 초기 프로젝트에서 Flickr 이미지를 아무 생각 없이 학습에 사용했다. 나중에 알고 보니 "All Rights Reserved" 이미지가 섞여 있었고, 결국 해당 데이터로 학습한 모델은 공개할 수 없게 됐다. **라이선스 검증은 데이터 수집 직후, 전처리 전에** 반드시 해야 한다.

### 주요 라이선스 분류

| 라이선스 | 상업 이용 | 수정 허용 | 출처 표기 | AI 학습 허용 |
|---|---|---|---|---|
| **CC0 (퍼블릭 도메인)** | ✅ | ✅ | 불필요 | ✅ |
| **CC BY 4.0** | ✅ | ✅ | 필수 | ✅ |
| **CC BY-SA 4.0** | ✅ | 동일 라이선스 | 필수 | 조건부 |
| **CC BY-NC 4.0** | ❌ | ✅ | 필수 | ⚠️ 비상업만 |
| **CC BY-ND 4.0** | ✅ | ❌ | 필수 | ⚠️ 수정 불가 |
| **공공누리 1유형** | ✅ | ✅ | 필수 | ✅ |
| **공공누리 2유형** | ✅ | ❌ | 필수 | ⚠️ |
| **공공누리 3유형** | ❌ | ✅ | 필수 | ⚠️ |
| **공공누리 4유형** | ❌ | ❌ | 필수 | ❌ |
| **All Rights Reserved** | ❌ | ❌ | — | ❌ |

**AI 학습에 안전하게 쓸 수 있는 것**: CC0, CC BY, 공공누리 1유형  
**절대 사용 금지**: All Rights Reserved, CC BY-ND, CC BY-NC (상업 프로젝트)

### 국립중앙박물관 공개 정책

국립중앙박물관은 2022년부터 소장품 이미지를 **공공누리 1유형**으로 공개하고 있다. 출처만 표기하면 상업 이용, AI 학습, 수정 모두 가능하다. 우리 프로젝트에 가장 적합한 소스다.

출처 표기 형식:
```
이미지 출처: 국립중앙박물관 (museum.go.kr), 공공누리 제1유형
```

### Wikimedia Commons 라이선스 구조

Wikimedia 이미지는 각각 라이선스가 다르다. API 응답의 `imageinfo` 필드에서 `extmetadata.License.value`를 확인해야 한다. 
- `pd` → 퍼블릭 도메인 (최고)
- `cc-zero` → CC0
- `cc-by-4.0` → CC BY 4.0
- `cc-by-sa-4.0` → 동일 조건 변경 허락 (사용 가능하나 파생물도 동일 라이선스)
- `cc-by-nc-4.0` → **비상업 제한 — 사용 금지**

## 실전 코드

```python
# 라이선스 검증 체크리스트 자동화
# pip install pandas pillow requests

import json
import csv
import hashlib
from pathlib import Path
from dataclasses import dataclass, field, asdict
from enum import Enum
from typing import Optional


class LicenseGrade(Enum):
    """AI 학습 사용 가능 등급"""
    GREEN  = "green"   # CC0, CC BY, 공공누리 1유형 — 자유 사용
    YELLOW = "yellow"  # CC BY-SA, CC BY-NC (비상업) — 조건부
    RED    = "red"     # CC BY-ND, All Rights Reserved — 사용 불가
    UNKNOWN = "unknown"


# 라이선스 식별자 → 등급 매핑
LICENSE_MAP: dict[str, LicenseGrade] = {
    # 초록불
    "pd":              LicenseGrade.GREEN,
    "cc-zero":         LicenseGrade.GREEN,
    "cc0":             LicenseGrade.GREEN,
    "cc-by-4.0":       LicenseGrade.GREEN,
    "cc-by-3.0":       LicenseGrade.GREEN,
    "cc-by-2.0":       LicenseGrade.GREEN,
    "공공누리 제1유형":  LicenseGrade.GREEN,
    "공공누리1":        LicenseGrade.GREEN,
    "kogl-type1":      LicenseGrade.GREEN,

    # 노란불 (조건부)
    "cc-by-sa-4.0":    LicenseGrade.YELLOW,
    "cc-by-sa-3.0":    LicenseGrade.YELLOW,
    "cc-by-nc-4.0":    LicenseGrade.YELLOW,
    "cc-by-nc-3.0":    LicenseGrade.YELLOW,
    "공공누리 제2유형":  LicenseGrade.YELLOW,
    "공공누리 제3유형":  LicenseGrade.YELLOW,

    # 빨간불
    "cc-by-nd-4.0":    LicenseGrade.RED,
    "cc-by-nd-3.0":    LicenseGrade.RED,
    "all rights reserved": LicenseGrade.RED,
    "공공누리 제4유형":  LicenseGrade.RED,
    "kogl-type4":      LicenseGrade.RED,
}


@dataclass
class ImageRecord:
    """이미지 라이선스 검증 레코드"""
    file_path:       str
    source:          str          # museum / wikimedia / aihub
    license_raw:     str          # 원본 라이선스 문자열
    license_grade:   str          # green / yellow / red / unknown
    attribution:     str          # 출처 표기 문구
    sha256:          str          # 파일 해시 (무결성)
    approved:        bool         # 학습 사용 승인 여부
    reject_reason:   Optional[str] = None


def classify_license(license_raw: str) -> LicenseGrade:
    """라이선스 문자열 → 등급 분류"""
    normalized = license_raw.lower().strip()
    for key, grade in LICENSE_MAP.items():
        if key in normalized:
            return grade
    return LicenseGrade.UNKNOWN


def compute_sha256(file_path: str) -> str:
    """파일 SHA-256 해시 계산"""
    h = hashlib.sha256()
    with open(file_path, "rb") as f:
        for chunk in iter(lambda: f.read(8192), b""):
            h.update(chunk)
    return h.hexdigest()


def build_attribution(source: str, license_raw: str, original_url: str = "") -> str:
    """출처 표기 문구 자동 생성"""
    if source == "museum":
        return f"이미지 출처: 국립중앙박물관 (museum.go.kr), 공공누리 제1유형"
    elif source == "wikimedia":
        return f"이미지 출처: Wikimedia Commons ({original_url}), {license_raw}"
    elif source == "aihub":
        return "이미지 출처: AI-Hub (aihub.or.kr), 연구 목적 허용"
    return f"출처 미상 — {license_raw}"


class LicenseVerifier:
    """
    데이터셋 라이선스 일괄 검증기

    사용법:
        verifier = LicenseVerifier("./dataset")
        verifier.load_metadata("metadata.csv")
        report = verifier.run()
        verifier.export_report(report, "license_report.json")
    """

    def __init__(self, dataset_dir: str):
        self.dataset_dir = Path(dataset_dir)
        self.records: list[ImageRecord] = []

    def load_metadata(self, csv_path: str) -> None:
        """
        CSV 메타데이터 로드.
        필수 컬럼: file_name, source, license, url
        """
        with open(csv_path, newline="", encoding="utf-8") as f:
            reader = csv.DictReader(f)
            for row in reader:
                file_path = str(self.dataset_dir / row["file_name"])
                if not Path(file_path).exists():
                    continue

                grade = classify_license(row.get("license", ""))
                sha = compute_sha256(file_path)
                approved = (grade == LicenseGrade.GREEN)
                reject_reason = None if approved else f"라이선스 등급 {grade.value}"

                record = ImageRecord(
                    file_path    = file_path,
                    source       = row.get("source", "unknown"),
                    license_raw  = row.get("license", "unknown"),
                    license_grade= grade.value,
                    attribution  = build_attribution(
                                       row.get("source", ""),
                                       row.get("license", ""),
                                       row.get("url", ""),
                                   ),
                    sha256       = sha,
                    approved     = approved,
                    reject_reason= reject_reason,
                )
                self.records.append(record)

        print(f"메타데이터 로드: {len(self.records)}건")

    def run(self) -> dict:
        """검증 실행 후 요약 리포트 반환"""
        total   = len(self.records)
        green   = sum(1 for r in self.records if r.license_grade == "green")
        yellow  = sum(1 for r in self.records if r.license_grade == "yellow")
        red     = sum(1 for r in self.records if r.license_grade == "red")
        unknown = sum(1 for r in self.records if r.license_grade == "unknown")

        report = {
            "total":        total,
            "approved":     green,
            "conditional":  yellow,
            "rejected":     red + unknown,
            "approval_rate": f"{green / total * 100:.1f}%" if total > 0 else "0%",
            "by_source":    self._group_by_source(),
            "rejected_list": [
                {"file": r.file_path, "reason": r.reject_reason}
                for r in self.records if not r.approved
            ],
        }

        print("=" * 50)
        print("라이선스 검증 결과")
        print("=" * 50)
        print(f"전체:     {total:,}건")
        print(f"✅ 승인:  {green:,}건 ({green/total*100:.1f}%)")
        print(f"⚠️  조건부: {yellow:,}건")
        print(f"❌ 거부:  {red + unknown:,}건")
        print(f"승인률: {report['approval_rate']}")

        return report

    def _group_by_source(self) -> dict:
        groups: dict[str, dict] = {}
        for r in self.records:
            if r.source not in groups:
                groups[r.source] = {"total": 0, "approved": 0}
            groups[r.source]["total"]   += 1
            groups[r.source]["approved"] += int(r.approved)
        return groups

    def export_report(self, report: dict, output_path: str) -> None:
        """검증 리포트를 JSON으로 저장"""
        with open(output_path, "w", encoding="utf-8") as f:
            json.dump(report, f, ensure_ascii=False, indent=2)
        print(f"리포트 저장: {output_path}")

    def export_approved_list(self, output_path: str) -> None:
        """승인된 이미지 목록 CSV 저장"""
        approved = [r for r in self.records if r.approved]
        if not approved:
            print("승인된 이미지 없음")
            return

        with open(output_path, "w", newline="", encoding="utf-8") as f:
            writer = csv.DictWriter(f, fieldnames=asdict(approved[0]).keys())
            writer.writeheader()
            writer.writerows(asdict(r) for r in approved)

        print(f"승인 목록 저장: {len(approved)}건 → {output_path}")


# 실행 예시
if __name__ == "__main__":
    verifier = LicenseVerifier("./raw_dataset")

    # metadata.csv 예시:
    # file_name,source,license,url
    # img_001.jpg,museum,공공누리 제1유형,https://museum.go.kr/...
    # img_002.jpg,wikimedia,cc-by-4.0,https://commons.wikimedia.org/...
    # img_003.jpg,wikimedia,cc-by-nc-4.0,https://commons.wikimedia.org/...

    verifier.load_metadata("metadata.csv")
    report = verifier.run()
    verifier.export_report(report, "license_report.json")
    verifier.export_approved_list("approved_dataset.csv")
```

## 함정 카드 — 자주 실수하는 5가지

1. **"Wikimedia = 모두 자유 사용"이라고 착각한다"** → Wikimedia Commons에는 CC BY-NC, CC BY-ND도 상당수 있다. 소스별 신뢰가 아니라 **이미지별 라이선스** 확인이 필수. `extmetadata.License.value`를 반드시 파싱.

2. **"공공누리 = 공공 데이터 = 무조건 OK"** → 공공누리 유형이 4가지다. 2유형(변경 금지), 3유형(비상업), 4유형(변경+비상업 모두 금지)은 AI 학습에 사용할 수 없다. 유형 번호를 반드시 확인.

3. **"라이선스 검증 나중에 하면 된다"** → 전처리까지 마친 뒤 라이선스 문제를 발견하면 그 데이터는 전부 버려야 한다. 수집 → 라이선스 → 전처리 순서가 원칙. 나는 이 순서를 바꿔서 3일치 전처리 작업을 버린 경험이 있다.

4. **"AI-Hub 데이터는 학술 목적이면 상관없다"** → AI-Hub 데이터는 **연구 목적 한정** 허가다. 상업적 서비스에 학습된 모델을 배포하면 약관 위반이다. 대회 제출용은 OK, 상용 제품은 반드시 약관 재확인.

5. **"출처 표기는 이미지 뒤에 숨겨도 된다"** → CC BY의 출처 표기는 **사용자가 실제로 볼 수 있는 방식**이어야 한다. 모델 카드 README, 웹사이트 About 페이지, 앱 내 라이선스 섹션에 명시해야 법적 요건 충족.

## 연습문제

**문제 1.** 다음 이미지의 AI 학습 사용 가능 여부를 판단하라:  
① 국립중앙박물관 소장 신윤복 풍속화 (공공누리 제1유형)  
② Flickr 사진 (CC BY-NC-SA 4.0)  
③ Wikimedia Commons 조선시대 지도 (퍼블릭 도메인)

> **해설:** ① 가능 — 공공누리 1유형, 출처 표기 후 자유 사용. ② 불가 (상업 프로젝트) — NC(비상업 전용) 조건. 대회 심사 통과 후 상용화 시 법적 문제. ③ 가능 — 퍼블릭 도메인, 아무 제약 없음. 가장 선호하는 유형.

**문제 2.** `classify_license("CC BY-SA 4.0")` 코드 실행 결과가 `UNKNOWN`으로 나온다. 원인과 수정 방법을 설명하라.

> **해설:** `LICENSE_MAP`의 키가 소문자 `"cc-by-sa-4.0"` 인데, 입력 문자열 정규화 후도 공백·대시 불일치로 매핑 실패. 수정: `normalized = license_raw.lower().strip().replace(" ", "-")` 추가 후 재시도. 또는 `"cc by-sa 4.0"`, `"cc-by-sa-4.0"` 두 형식을 모두 키로 등록.

**문제 3.** 10,000장 데이터셋에서 승인률이 72%라면, 실제 학습에 사용 가능한 이미지 수와 나머지 28%에 대한 처리 방법을 제안하라.

> **해설:** 7,200장 사용 가능. 28%(2,800장) 처리 방법: ① Yellow(조건부) 이미지는 비상업 전용 모델 학습에 별도 데이터셋으로 분리 보관 ② Red 이미지는 즉시 삭제 또는 격리 폴더 이동 ③ Unknown은 원본 URL 재크롤링 후 라이선스 재확인 ④ 7,200장으로도 LoRA 파인튜닝 충분하나, 다양성 확보를 위해 CC0 소스 추가 수집 권장.
