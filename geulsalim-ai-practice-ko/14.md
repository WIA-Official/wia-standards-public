# 챕터 14 · 추론 파이프라인 — 마스크 자동 감지 (Otsu + SAM)

## 학습 목표
- Otsu 이진화 기반의 경량 마스크 자동 감지 알고리즘을 구현하고 한계를 설명할 수 있다
- SAM(Segment Anything Model) Point/Box Prompt로 사용자 최소 입력만 받는 마스크 추출을 구현할 수 있다
- 자동 마스크 → Inpainting 파이프라인을 End-to-End 스크립트 하나로 연결할 수 있다

---

## 이론

### "사용자가 마스크를 그린다"는 UX의 실패

학습된 모델에 Inpainting을 호출하려면 `mask_image`가 필요하다. 웹 서비스에서 사용자가 포토샵처럼 마스크를 그리라 하면 **95%가 포기**한다. 내가 고문서 복원 데모를 3명에게 시연했을 때 모두 "마스크를 어떻게 그려요?"에서 멈췄다.

**결론**: 마스크 입력을 최소화하거나 제로로 만들어야 한다. 두 전략:

1. **Otsu 자동**: 0 클릭 — 어두운 영역을 자동 감지
2. **SAM Point/Box**: 1~2 클릭 — 사용자가 손상 부위 하나 찍으면 영역 추출

### Otsu 이진화 — 경량·빠름

Otsu's method: 히스토그램을 두 클래스로 나눠 클래스 간 분산을 최대화하는 임계값을 자동 계산. OpenCV의 `cv2.threshold(..., cv2.THRESH_OTSU)` 한 줄.

장점:
- GPU 불필요, CPU 10ms
- 모델 파일 없음
- 간단 스캔본 문서에 효과적

단점:
- "손상=어두운 영역" 가정. 흰 종이에 검은 얼룩만 잡음. 찢어진 구멍(빛 투과로 흰색)은 못 잡음.
- 본문 텍스트(글자) 도 어둡기 때문에 **글자도 마스크로 잡아버림** → 치명적 오탐.

실무 팁: **역 Otsu** + **면적 필터**로 "크고 연속된 어두운 영역"만 남긴다.

### SAM — 사용자 의도 기반

Segment Anything (Meta, 2023). Point 1개 또는 Box를 받으면 "그 안의 객체"를 정밀하게 세그먼트. 2024~2026 기준 SAM 2가 영상·이미지 공용으로 사용.

실무 파이프라인:
1. 사용자가 손상 부위 **클릭 1회** (Point Prompt)
2. SAM이 해당 포인트 주변 마스크 생성
3. 사용자가 추가 클릭으로 보정 (긍정/부정 포인트)
4. 완성된 마스크를 Inpainting 파이프라인에 전달

### 3가지 조합 전략

| 시나리오 | 방법 | UX 부담 | 정확도 |
|---|---|---|---|
| 배치 자동 처리 (1000장) | Otsu + 면적 필터 | 0 | 70% |
| 단일 이미지 인터랙티브 | SAM 1-Click | 낮음 | 90% |
| 전문가 세밀 교정 | SAM + 수동 브러시 | 중간 | 98% |

나는 고문서 복원 MVP에 Otsu + 사용자 선택지로 SAM 토글을 뒀다. 대부분 사용자가 Otsu로 만족했고, 전문가만 SAM을 썼다.

```python
# 마스크 자동 감지 + Inpainting 추론 End-to-End
# pip install opencv-python pillow numpy torch
# pip install segment-anything-hq  # 또는 segment-anything 공식

import cv2
import numpy as np
import torch
from pathlib import Path
from PIL import Image
from typing import Literal, Optional, Tuple


DEVICE = "cuda" if torch.cuda.is_available() else "cpu"


# ──────────────────────────────────────────────
# 1. Otsu 자동 마스크
# ──────────────────────────────────────────────
class OtsuMaskDetector:
    """
    Otsu + 면적 필터 + 글자 억제.
    간단한 문서 스캔본에 효과적.
    """
    def __init__(
        self,
        min_area_ratio: float = 0.003,   # 전체의 0.3% 미만 영역 제거
        max_area_ratio: float = 0.30,    # 30% 초과 영역은 마스크로 보기 어려움
        text_suppress:  bool  = True,    # 글자 크기 영역 제거
    ):
        self.min_area_ratio = min_area_ratio
        self.max_area_ratio = max_area_ratio
        self.text_suppress  = text_suppress

    def detect(self, img: Image.Image) -> np.ndarray:
        """
        PIL 이미지 → 마스크 numpy (uint8, 0/255).
        """
        gray = np.array(img.convert("L"))
        H, W = gray.shape
        area_total = H * W

        # 역 Otsu: 어두운 영역 = 손상 후보
        _, binary = cv2.threshold(
            gray, 0, 255,
            cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU,
        )

        # 형태학적 닫힘: 인접한 손상 영역 연결
        kernel = np.ones((9, 9), np.uint8)
        closed = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, kernel, iterations=2)

        # 연결 컴포넌트 분석 (면적 필터)
        num_labels, labels, stats, _ = cv2.connectedComponentsWithStats(closed)
        mask = np.zeros_like(gray, dtype=np.uint8)

        for i in range(1, num_labels):     # 0은 배경
            area = stats[i, cv2.CC_STAT_AREA]
            ratio = area / area_total

            # 너무 작거나 크면 제외
            if ratio < self.min_area_ratio or ratio > self.max_area_ratio:
                continue

            # 글자 억제: aspect ratio 극단 or 높이 < 50px는 글자로 간주
            w_box = stats[i, cv2.CC_STAT_WIDTH]
            h_box = stats[i, cv2.CC_STAT_HEIGHT]
            if self.text_suppress:
                if h_box < 50 or w_box < 50:
                    continue
                ar = max(w_box, h_box) / max(min(w_box, h_box), 1)
                if ar > 10:   # 매우 가늘고 긴 = 글자 줄
                    continue

            mask[labels == i] = 255

        # 경계 부드럽게
        mask = cv2.dilate(mask, np.ones((5, 5), np.uint8), iterations=1)
        return mask


# ──────────────────────────────────────────────
# 2. SAM Prompt-based 마스크
# ──────────────────────────────────────────────
class SAMMaskDetector:
    """
    Segment Anything Model — Point/Box Prompt.
    - model_type: "vit_h" (정확) | "vit_b" (빠름)
    """
    def __init__(
        self,
        checkpoint: str = "./models/sam_vit_h_4b8939.pth",
        model_type: str = "vit_h",
        device:     str = DEVICE,
    ):
        from segment_anything import sam_model_registry, SamPredictor
        sam = sam_model_registry[model_type](checkpoint=checkpoint)
        sam.to(device)
        self.predictor = SamPredictor(sam)

    def detect_from_points(
        self,
        img:          Image.Image,
        points:       list,                   # [(x, y, label), ...] label: 1=포함, 0=배제
    ) -> np.ndarray:
        """
        사용자 클릭 좌표로 마스크 생성.
        """
        img_np = np.array(img.convert("RGB"))
        self.predictor.set_image(img_np)

        coords = np.array([[p[0], p[1]] for p in points])
        labels = np.array([p[2] for p in points])

        masks, scores, _ = self.predictor.predict(
            point_coords=coords,
            point_labels=labels,
            multimask_output=True,     # 3개 마스크 반환
        )

        # 스코어 가장 높은 마스크 선택
        best = masks[scores.argmax()]
        return (best.astype(np.uint8) * 255)

    def detect_from_box(
        self,
        img: Image.Image,
        box: Tuple[int, int, int, int],    # (x0, y0, x1, y1)
    ) -> np.ndarray:
        img_np = np.array(img.convert("RGB"))
        self.predictor.set_image(img_np)

        box_np = np.array(box)
        masks, scores, _ = self.predictor.predict(
            box=box_np,
            multimask_output=False,
        )
        return (masks[0].astype(np.uint8) * 255)


# ──────────────────────────────────────────────
# 3. 통합 추론 파이프라인
# ──────────────────────────────────────────────
class DocumentRestorer:
    """
    End-to-End: 이미지 → 마스크 자동 감지 → Inpainting → 결과
    """
    def __init__(
        self,
        base_model_id: str,
        lora_path:     str,
        mask_mode:     Literal["otsu", "sam"] = "otsu",
        sam_checkpoint: Optional[str] = None,
    ):
        from diffusers import StableDiffusionXLInpaintPipeline
        from peft import LoraConfig
        from safetensors.torch import load_file

        self.pipe = StableDiffusionXLInpaintPipeline.from_pretrained(
            base_model_id, torch_dtype=torch.float16, variant="fp16",
        ).to(DEVICE)

        # LoRA 적재
        lora_cfg = LoraConfig(
            r=16, lora_alpha=16,
            target_modules=["to_q", "to_k", "to_v", "to_out.0"],
        )
        self.pipe.unet.add_adapter(lora_cfg)
        state = load_file(lora_path, device="cpu")
        self.pipe.unet.load_state_dict(state, strict=False)
        self.pipe.unet.to(DEVICE)
        self.pipe.set_progress_bar_config(disable=True)

        # 마스크 감지기
        self.otsu = OtsuMaskDetector()
        self.sam  = SAMMaskDetector(sam_checkpoint) if mask_mode == "sam" else None
        self.mask_mode = mask_mode

    def restore(
        self,
        image_path:   str,
        output_path:  str,
        points:       Optional[list] = None,          # SAM 모드
        box:          Optional[Tuple] = None,         # SAM 모드
        prompt:       str = "Korean traditional document restoration",
        guidance_scale: float = 7.5,
        num_inference_steps: int = 30,
        seed:         int = 42,
    ) -> dict:
        """
        Returns: {"output_path": str, "mask_path": str, "mask_ratio": float}
        """
        img = Image.open(image_path).convert("RGB")
        original_size = img.size
        img_1024 = img.resize((1024, 1024), Image.LANCZOS)

        # (1) 마스크 감지
        if self.mask_mode == "otsu":
            mask_np = self.otsu.detect(img_1024)
        else:
            if points is not None:
                mask_np = self.sam.detect_from_points(img_1024, points)
            elif box is not None:
                mask_np = self.sam.detect_from_box(img_1024, box)
            else:
                raise ValueError("SAM 모드에는 points 또는 box 필요")

        mask_pil = Image.fromarray(mask_np).convert("L")
        mask_ratio = mask_np.sum() / 255 / mask_np.size

        # (2) 마스크 비율 안전성 체크
        if mask_ratio < 0.005:
            print(f"⚠️  마스크가 너무 작음 ({mask_ratio*100:.2f}%). 복원 불필요.")
            return {"output_path": None, "mask_ratio": mask_ratio}
        if mask_ratio > 0.5:
            print(f"⚠️  마스크가 너무 큼 ({mask_ratio*100:.1f}%). 복원 품질 낮을 수 있음.")

        # (3) Inpainting 실행
        generator = torch.Generator(DEVICE).manual_seed(seed)
        with torch.autocast("cuda", dtype=torch.float16):
            result = self.pipe(
                prompt=prompt,
                image=img_1024,
                mask_image=mask_pil,
                guidance_scale=guidance_scale,
                num_inference_steps=num_inference_steps,
                generator=generator,
            ).images[0]

        # (4) 원본 해상도로 복원
        result = result.resize(original_size, Image.LANCZOS)

        # (5) 저장
        Path(output_path).parent.mkdir(parents=True, exist_ok=True)
        result.save(output_path, quality=95)

        mask_path = Path(output_path).with_suffix(".mask.png")
        mask_pil.resize(original_size).save(mask_path)

        print(f"복원 완료: {output_path}")
        print(f"마스크:    {mask_path} (영역 {mask_ratio*100:.1f}%)")
        return {
            "output_path": str(output_path),
            "mask_path":   str(mask_path),
            "mask_ratio":  float(mask_ratio),
        }


# ──────────────────────────────────────────────
# 4. 배치 처리 (폴더 단위)
# ──────────────────────────────────────────────
def restore_batch(
    input_dir:  str,
    output_dir: str,
    restorer:   DocumentRestorer,
) -> None:
    files = list(Path(input_dir).glob("*.jpg")) + list(Path(input_dir).glob("*.png"))
    print(f">> 배치 복원: {len(files)}개")

    from tqdm import tqdm
    for f in tqdm(files):
        try:
            out_path = Path(output_dir) / f.name
            restorer.restore(str(f), str(out_path))
        except Exception as e:
            print(f"오류 {f.name}: {e}")


if __name__ == "__main__":
    restorer = DocumentRestorer(
        base_model_id="diffusers/stable-diffusion-xl-1.0-inpainting-0.1",
        lora_path="./output/lora_final.safetensors",
        mask_mode="otsu",
    )

    # 단일 복원
    result = restorer.restore(
        image_path="./test/damaged_01.jpg",
        output_path="./test/restored_01.jpg",
    )
    print(result)

    # SAM Point 모드 예시
```

## 함정 카드

1. **"Otsu로 만든 마스크를 글자 억제 없이 쓴다"** → 고문서는 본문 텍스트가 어두운 영역이다. 글자 위에 마스크가 씌워지고 Inpainting이 "글자 위에 아무거나" 그려넣는다. 내 초기 데모에서 "조선왕조실록이 식물 사진으로 바뀐" 버그. 반드시 `text_suppress=True` + 면적 필터.

2. **"SAM 모델을 CPU에 로드"** → SAM ViT-H는 GPU에서 300ms, CPU에서 8초. 실시간 UX 불가. 메모리 여유 없으면 `vit_b`(작은 버전)이 절충점 — 1GB VRAM, 500ms 추론.

3. **"자동 마스크 감지 결과를 원본 해상도에서 돌린다"** → 4K 스캔본을 Otsu 돌리면 입자 노이즈까지 잡힌다. **1024 리사이즈 후 감지 → 원본으로 확대** 순서가 안정적. 블러 필터를 먼저 적용해도 좋음.

4. **"마스크 비율 안전성 체크를 안 한다"** → 마스크가 0.1%면 Inpainting이 거의 작동 안 한다(바뀐 영역이 안 보임). 50% 이상이면 복원이 아니라 재생성 수준이라 품질 저하. 실사용에선 5~35% 범위만 정상 처리, 외 범위는 경고 + 사용자 재확인.

5. **"SAM의 multimask_output=False로 한 마스크만 받는다"** → 사용자 Point가 애매하면 SAM은 3가지 해석(문자 vs 단어 vs 문장)을 준다. `multimask_output=True`로 받아 **스코어 최고** 선택이 실무 기본값. 수동 UI면 3개 중 사용자가 고르게 하는 것도 유효.

## 연습문제

**문제 1.** 한 이미지에 "찢어진 구멍(흰색)"과 "먹 얼룩(검정)"이 동시에 있다. Otsu 자동 감지로 처리되지 않는 영역과 해결책은?

> **해설:** **구멍(흰색)이 감지 안 된다**. Otsu 역이진화는 "어두운 영역"만 잡으므로 배경보다 더 밝은 구멍은 누락. 해결: ① **dual-threshold** — `cv2.THRESH_BINARY_INV`와 `cv2.THRESH_BINARY` 둘 다 돌려 OR 결합. 단, 밝은 배경 종이 자체를 잡지 않도록 면적 필터 엄격히. ② 엣지 기반 추가 — Canny + 연결 컴포넌트로 밝은 구멍 경계 검출. ③ 현실적 대안: Otsu + SAM 2회 사용 — 자동 1차 + 사용자 클릭 보정.

**문제 2.** SAM 추론이 너무 느려 서비스 타임아웃이 걸린다. 3가지 최적화 방안은?

> **해설:** ① **모델 다운그레이드**: vit_h → vit_b 3배 빠름, 정확도 5% 감소 ② **이미지 사전 임베딩**: `predictor.set_image`가 80% 시간 차지 → 이미지 로드 시점에 비동기 임베딩 후 캐시, 사용자 클릭 시에는 즉시 마스크 반환 ③ **해상도 축소**: 1024 → 768 (SAM 내부는 1024 고정이지만 입력 해상도 자체를 줄이면 업스케일 연산 절약). 조합: vit_b + 이미지 임베딩 캐시 + 1024 유지로 200ms까지 단축 가능.

**문제 3.** 사용자가 "마스크가 이상해요"라고 피드백한다. 재작업 없이 즉석에서 복원 품질을 올리는 5초 솔루션은?

> **해설:** **seed 변경 → 재실행**. 추론 시 `manual_seed(42)` 고정했다면 `seed=123`으로 바꿔 다시 생성. Inpainting은 시드에 따라 결과가 꽤 다르다. **3~5개 시드 병렬 생성 + 사용자 선택** UX가 데모에서 효과적. 추가 솔루션: `guidance_scale` 5 → 10 (더 프롬프트 강하게), `num_inference_steps` 30 → 50 (디테일). 이 3개 슬라이더를 UI에 노출하면 사용자가 자체 조정 가능.
