## 🚀 Quick Start

### Installation

```bash
# Clone or download
git clone https://github.com/WIA-Official/wia-standards.git
cd wia-standards/standards/WIA-FOOD_TRACEABILITY

# Install (optional)
chmod +x install.sh
./install.sh
```

### Basic Usage

```bash
# CLI tool (if available)
./cli/wia-food_traceability.sh info
./cli/wia-food_traceability.sh validate data.json
```

```typescript
# API usage (TypeScript)
import { FOOD_TRACEABILITYCore } from './api/typescript';

const core = new FOOD_TRACEABILITYCore();
await core.initialize();

const result = await core.process(data);
```
