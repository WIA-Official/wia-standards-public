개요: 시작하기

이 장에서는 WIA-ART-007 표준을 실제 프로젝트에 적용하는 방법을 단계별로 안내합니다. 개인 창작자부터 대규모 엔터프라이즈까지, 누구나 쉽게 시작할 수 있도록 실용적인 가이드와 베스트 프랙티스를 제공합니다.

빠른 시작 가이드

1. 30분 만에 시작하기

가장 간단한 방법으로 WIA-ART-007을 프로젝트에 적용해봅니다.

Step 1: SDK 설치

# npm 사용
npm install @wia/art-007-sdk

# yarn 사용
yarn add @wia/art-007-sdk

# pnpm 사용
pnpm add @wia/art-007-sdk

Step 2: API 키 발급

https://developer.wia.org에서 무료 계정을 생성하고 API 키를 발급받습니다.

Step 3: 첫 프로젝트 생성

import { WIAArtClient } from '@wia/art-007-sdk';

const client = new WIAArtClient({
  apiKey: process.env.WIA_API_KEY,
  apiSecret: process.env.WIA_API_SECRET
});

// 프로젝트 생성
const project = await client.projects.create({
  title: '나의 첫 WIA 프로젝트',
  type: 'image',
  tags: ['getting-started']
});

console.log('프로젝트 ID:', project.id);

// AI 이미지 생성
const job = await client.generate.image({
  project_id: project.id,
  prompt: {
    positive: 'Beautiful mountain landscape at sunset',
    quality: 'high'
  }
});

// 완료 대기
const result = await client.generate.waitForCompletion(job.job_id);
console.log('생성된 이미지:', result.outputs[0].url);

// 성공! 이제 WIA-ART-007 표준을 사용하고 있습니다.

인증 절차

1. Bronze 인증 (기본 준수)

Phase 1 데이터 형식 표준을 완전히 준수하여 Bronze 인증을 획득합니다.

체크리스트 요구사항 검증 방법
1. JSON 스키마 모든 프로젝트가 WIA-ART-007 JSON 스키마 준수 자동 스키마 검증 도구
2. 메타데이터 필수 필드 모두 포함 (프로젝트, 창작자, 에셋, 라이선스) 필드 완전성 검사
3. 파일 포맷 내보내기/가져오기 지원 왕복 테스트 (export → import)
4. 문서화 사용자 가이드 제공 문서 검토 
// Bronze 인증 자동 검증
import { WIACertification } from '@wia/art-007-sdk/certification';

const cert = new WIACertification();

// 프로젝트 검증
const validationResult = await cert.validate({
  level: 'bronze',
  projectId: 'proj-abc123'
});

if (validationResult.passed) {
  console.log('Bronze 인증 요구사항 충족!');
  console.log('점수:', validationResult.score); // 0-100

  // 인증 신청
  const application = await cert.apply({
    level: 'bronze',
    projectId: 'proj-abc123',
    organization: '우리 회사',
    contactEmail: 'dev@example.com'
  });

  console.log('신청 번호:', application.id);
  console.log('예상 검토 기간:', application.estimatedDays, '일');
} else {
  console.log('미충족 요구사항:', validationResult.issues);
}

2. Silver 인증 (API 통합)

Phase 2 REST API를 완전히 구현하여 Silver 인증을 획득합니다.

Silver 인증 혜택

  • WIA 공식 마켓플레이스 등록
  • 엔터프라이즈 고객 소개
  • 기술 지원 우선순위
  • 연례 WIA 컨퍼런스 초대
  • 공동 마케팅 기회

3. Gold 인증 (완전 통합)

Phase 3-4를 모두 구현하여 Gold 인증을 획득합니다.

인증 레벨 구현 시간 비용 혜택
Bronze 2-4주 무료 기본 인증 마크
Silver 2-3개월 $500/년 마켓플레이스, 기술 지원
Gold 4-6개월 $2,000/년 파트너 지위, 표준 개발 참여

베스트 프랙티스

1. 보안

  • API 키 보호: 환경 변수 사용, 코드에 직접 하드코딩 금지
  • HTTPS 사용: 모든 API 호출은 HTTPS를 통해서만
  • 토큰 갱신: 액세스 토큰 만료 전 자동 갱신
  • 최소 권한 원칙: 필요한 권한만 부여
  • 감사 로그: 모든 API 호출 및 데이터 접근 기록

2. 성능 최적화

  • 캐싱: 자주 사용하는 데이터는 로컬 캐시 활용
  • 배치 처리: 여러 에셋을 한 번에 업로드
  • 압축: 대용량 파일은 압축 후 전송
  • CDN 사용: 정적 에셋은 CDN을 통해 배포
  • 연결 풀링: WebSocket 연결 재사용

3. 오류 처리

// 견고한 오류 처리 예제
import { WIAArtClient, WIAError } from '@wia/art-007-sdk';

async function createProjectSafely(title) {
  const client = new WIAArtClient({
    apiKey: process.env.WIA_API_KEY,
    apiSecret: process.env.WIA_API_SECRET,
    retry: {
      maxAttempts: 3,
      backoff: 'exponential'
    }
  });

  try {
    const project = await client.projects.create({
      title: title,
      type: 'image'
    });

    return { success: true, project };

  } catch (error) {
    if (error instanceof WIAError) {
      // WIA 특정 오류
      switch (error.code) {
        case 'RATE_LIMIT_EXCEEDED':
          console.error('API 호출 한도 초과, 1분 후 재시도');
          await sleep(60000);
          return createProjectSafely(title); // 재시도

        case 'INVALID_API_KEY':
          console.error('잘못된 API 키');
          return { success: false, error: 'Invalid credentials' };

        case 'QUOTA_EXCEEDED':
          console.error('스토리지 할당량 초과');
          return { success: false, error: 'Storage full' };

        default:
          console.error('WIA 오류:', error.message);
          return { success: false, error: error.message };
      }
    } else {
      // 일반 오류 (네트워크 등)
      console.error('예상치 못한 오류:', error);
      return { success: false, error: 'Unknown error' };
    }
  }
}

function sleep(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

마지막으로

WIA-ART-007은 단순한 기술 표준이 아닙니다. 이것은 창의 AI의 미래를 함께 만들어가는 글로벌 커뮤니티의 노력입니다. 여러분의 참여와 기여가 이 표준을 더욱 강력하고 유용하게 만듭니다.

"弘益人間(홍익인간)" 철학처럼, 이 표준은 모든 인류에게 이로움을 주기 위해 만들어졌습니다. 여러분이 만드는 창작물이 표준화되고 보존되어, 미래 세대까지 전달되기를 바랍니다.
