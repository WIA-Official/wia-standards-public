[DATA-002] WIA 데이터 레이크 표준

공식 전자책 - 완전 가이드

홍익인간 (弘益人間)

"널리 인간을 이롭게 하라"

데이터는 현대 문명의 기초입니다. WIA 데이터 레이크 표준은 데이터 저장 및 처리 인프라가 접근 가능하고 확장 가능하며 모두에게 개방되도록 보장합니다.

이 전자책에 대하여

이 공식 전자책은 WIA 데이터 레이크 표준에 대한 포괄적인 가이드를 제공합니다. 현대 데이터 레이크 아키텍처, 테이블 형식, 메타데이터 관리, 4단계에 걸친 통합 패턴을 다룹니다.

버전 1.0.0
챕터 8개
페이지 약 120페이지
라이선스 MIT (무료)
최종 업데이트 2025년

목차

Part I: 기초

제1장: 데이터 레이크 소개

  • 데이터 레이크란 무엇인가?
  • 데이터 레이크 vs 데이터 웨어하우스
  • 데이터 저장소의 진화
  • 표준의 중요성
  • WIA 철학: 홍익인간

제2장: 데이터 레이크 아키텍처

  • 3-영역 아키텍처
  • Raw, Curated, Consumption 영역
  • 스키마 온 리드 vs 스키마 온 라이트
  • 데이터 레이크하우스 개념
  • 스토리지 최적화 전략

Part II: 표준 개요

제3장: WIA 데이터 레이크 표준 개요

  • 미션과 목표
  • 4단계 아키텍처
  • 지원 테이블 형식
  • 설계 원칙
  • 개발 단계

Part III: 기술 사양

제4장: Phase 1 - 데이터 형식 표준

  • 파일 형식 (Parquet, ORC, Avro)
  • 파티셔닝 전략
  • 압축 코덱
  • 데이터 직렬화
  • 형식 선택 가이드라인

제5장: Phase 2 - API 인터페이스 표준

  • 핵심 API 설계
  • WiaDataLake 클래스
  • 데이터 수집 API
  • 쿼리 인터페이스
  • 메타데이터 작업
  • Python & TypeScript SDK

제6장: Phase 3 - 테이블 형식 프로토콜

  • Delta Lake 사양
  • Apache Iceberg 사양
  • Apache Hudi 사양
  • ACID 트랜잭션
  • 타임 트래블 & 스냅샷
  • 스키마 진화

제7장: Phase 4 - 생태계 통합

  • 컴퓨트 엔진 통합 (Spark, Presto, Hive)
  • 카탈로그 통합 (Glue, Unity, Nessie)
  • 클라우드 플랫폼 통합
  • 스트리밍 통합 (Kafka, Kinesis)
  • BI 도구 통합

Part IV: 구현

제8장: 구현 가이드 및 모범 사례

  • 구현 로드맵
  • 아키텍처 패턴
  • 성능 튜닝
  • 보안 & 거버넌스
  • 비용 최적화
  • 모니터링 & 관측성
  • 마이그레이션 전략
