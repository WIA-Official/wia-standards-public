5.1 Cardiac Rhythm Analysis
--------------------------------------------------------------------------------

The cardiovascular restart stage begins with analysis of the cardiac
rhythm and determination of the appropriate restart approach.

[ID] Initial Rhythm Assessment:

    CardiacRhythmAnalysis {
      possibleRhythms: [
        "ASYSTOLE",               // Flatline - no electrical activity
        "VENTRICULAR_FIBRILLATION", // Chaotic electrical activity
        "VENTRICULAR_TACHYCARDIA",  // Rapid ventricular rhythm
        "PEA",                    // Electrical without mechanical
        "BRADYCARDIA",            // Slow rhythm
        "SINUS_RHYTHM"            // Normal rhythm
      ]

      diagnosticTools: [
        "12-lead ECG",
        "Continuous cardiac monitoring",
        "Echocardiography",
        "Arterial line waveform"
      ]
    }

[ID] Rhythm-Based Protocol Selection:

    +------------------------------------------------------------------+
    | Rhythm                  | Primary Approach                       |
    +------------------------------------------------------------------+
    | Asystole                | Pacing + Pharmacological               |
    | Ventricular Fibrillation| Defibrillation                        |
    | Ventricular Tachycardia | Cardioversion + Anti-arrhythmics      |
    | PEA                     | Address reversible causes + Support    |
    | Bradycardia             | Pacing +/- Chronotropes               |
    +------------------------------------------------------------------+

[ID] Expected Initial State:

    After prolonged preservation, initial rhythm is typically:
    - Asystole (most common)
    - Ventricular fibrillation (if residual electrical activity)

    The heart has been in arrest for extended period - restart
    requires patience and systematic approach.


--------------------------------------------------------------------------------
5.2 Defibrillation Protocol
--------------------------------------------------------------------------------

For shockable rhythms (VF/VT), defibrillation is the primary intervention.

[ID] Defibrillation Protocol:

    DefibrillationProtocol {
      initialEnergy:     200     // Joules (biphasic)
      maxEnergy:         360     // Joules
      escalationStep:    50      // Joules per attempt
      maxAttempts:       5       // Before reassessment

      padPlacement:      "Anterolateral or anteroposterior"
      chargingTime:      "< 10 seconds"
      safetyChecks:      "Clear all contact before shock"
    }

[ID] Defibrillation Sequence:

    1. Confirm shockable rhythm (VF or pulseless VT)
    2. Apply pads in correct position
    3. Charge to initial energy (200J)
    4. Clear and verify
    5. Deliver shock
    6. Immediately assess rhythm
    7. If unsuccessful, escalate energy and repeat

[ID] Defibrillation Record:

    DefibrillationAttempt {
      attemptNumber:   1
      timestamp:       "2075-06-15T10:30:00Z"
      energyJoules:    200
      rhythmBefore:    "VENTRICULAR_FIBRILLATION"
      rhythmAfter:     "ASYSTOLE"
      success:         false
      performedBy:     "Dr. Lee, Cardiovascular"
    }

[!] After 5 unsuccessful defibrillation attempts, reassess the
    situation and consider alternative approaches or re-preservation.

5.3 Pharmacological Support
--------------------------------------------------------------------------------

Medications support cardiovascular restart and stabilization.

[ID] First-Line Medications:

    CardiacMedications {
      epinephrine: {
        dose:         "1 mg IV"
        frequency:    "Every 3-5 minutes"
        indication:   "All cardiac arrest rhythms"
        mechanism:    "Alpha: vasoconstriction; Beta: chronotropy"
      }

      amiodarone: {
        dose:         "300 mg IV (initial), 150 mg (repeat)"
        indication:   "Refractory VF/VT"
        mechanism:    "Membrane stabilization"
      }

      atropine: {
        dose:         "0.5-1 mg IV"
        indication:   "Symptomatic bradycardia"
        mechanism:    "Parasympathetic blockade"
      }
    }

[ID] Vasoactive Support:

    VasoactiveAgents {
      dopamine: {
        lowDose:      "2-5 mcg/kg/min (renal)"
        mediumDose:   "5-10 mcg/kg/min (inotropic)"
        highDose:     ">10 mcg/kg/min (vasopressor)"
      }

      norepinephrine: {
        startDose:    "0.1 mcg/kg/min"
        titrate:      "To MAP > 65 mmHg"
        maxDose:      "2 mcg/kg/min"
      }

      dobutamine: {
        dose:         "2.5-20 mcg/kg/min"
        indication:   "Cardiogenic shock, low cardiac output"
      }
    }

[ID] ECMO Consideration:

    If pharmacological support insufficient:

    ECMO Criteria {
      indication: [
        "Refractory cardiogenic shock",
        "Unable to maintain MAP > 60",
        "Persistent acidosis despite support"
      ]

      type:         "VA-ECMO (venoarterial)"
      duration:     "Until cardiac recovery or decision"
    }

5.4 Hemodynamic Optimization
--------------------------------------------------------------------------------

Once cardiac rhythm is restored, hemodynamic optimization begins.

[ID] Hemodynamic Targets:

    HemodynamicTargets {
      heartRate:        [60, 100]     // bpm
      systolicBP:       [90, 140]     // mmHg
      diastolicBP:      [60, 90]      // mmHg
      meanArterialPressure: [65, 100] // mmHg
      centralVenousPressure: [8, 12]  // mmHg
      cardiacOutput:    [4.0, 8.0]    // L/min
      cardiacIndex:     [2.5, 4.0]    // L/min/m2
    }

[ID] Optimization Algorithm:

    If MAP < 65:
    +------------------+
    | Check volume     |  -> If CVP low: Fluid bolus 250-500mL
    | status           |
    +--------+---------+
             |
             v
    +------------------+
    | If still low,    |  -> Start/increase vasopressor
    | after fluids     |
    +--------+---------+
             |
             v
    +------------------+
    | Assess cardiac   |  -> If low CO: Add inotrope
    | output           |
    +------------------+

[ID] Cardiovascular Restart Success Criteria:

    CardiovascularRestartComplete {
      rhythm:           "SINUS" or "PACED"
      heartRate:        [60, 100]
      systolicBP:       >= 90
      MAP:              >= 65
      lactate:          < 2.0 mmol/L or decreasing
      urineOutput:      > 0.5 mL/kg/hour
      peripheralPerfusion: "Warm, pink"
      echoAssessment:   "Adequate contractility"
    }
