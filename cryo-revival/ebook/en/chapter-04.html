4.1 Perfusion Washout
--------------------------------------------------------------------------------

After warming, cryoprotective agents must be carefully removed from
the body through perfusion washout.

[ID] Cryoprotectant Removal Protocol:

    CryoprotectantRemoval {
      initialConcentration:   60      // Percentage
      targetConcentration:    0.5     // Percentage (trace)
      removalRate:            5       // Percentage per hour
      estimatedDuration:      "12 hours"

      perfusionSolution: {
        base:         "Modified Ringer's solution"
        osmolarity:   "Gradually decreasing"
        temperature:  "37C"
        oxygenation:  "100% O2"
      }
    }

[ID] Washout Stages:

    +------------------------------------------------------------------+
    | Stage | Concentration | Duration | Solution Osmolarity            |
    +------------------------------------------------------------------+
    |   1   | 60% -> 40%    | 4 hours  | 1200 mOsm/L (hyperosmolar)     |
    |   2   | 40% -> 20%    | 4 hours  | 800 mOsm/L                     |
    |   3   | 20% -> 5%     | 3 hours  | 400 mOsm/L                     |
    |   4   | 5% -> 0.5%    | 1 hour   | 290 mOsm/L (isotonic)          |
    +------------------------------------------------------------------+

[!] Rapid removal of cryoprotectant causes osmotic shock and cell damage.
    Gradual concentration reduction is essential.


--------------------------------------------------------------------------------
4.2 Osmotic Balance
--------------------------------------------------------------------------------

Maintaining osmotic balance during washout prevents cellular damage.

[ID] Osmotic Pressure Management:

    OsmoticControl {
      targetOsmolarity: 290  // mOsm/L (normal plasma)

      gradientLimits: {
        maxChangeRate:     50      // mOsm/L per hour
        maxIntracellular:  20      // mOsm/L difference
      }

      monitoring: {
        plasmaOsmolarity:  "Every 30 minutes"
        cellularIntegrity: "Continuous bioimpedance"
        fluidBalance:      "Continuous"
      }
    }

[ID] Osmotic Agents:

    For gradual osmolarity reduction:

    - Mannitol:     Initial hyperosmolar support
    - Albumin:      Oncotic pressure maintenance
    - Electrolytes: Physiological balance

[ID] Warning Signs of Osmotic Injury:

    +------------------------------------------------------------------+
    | Sign                        | Indicates                          |
    +------------------------------------------------------------------+
    | Cellular swelling           | Too rapid dilution                 |
    | Hemolysis (in samples)      | Red blood cell damage              |
    | Increased bioimpedance      | Membrane disruption                |
    | Elevated potassium          | Cell lysis                         |
    +------------------------------------------------------------------+


--------------------------------------------------------------------------------
4.3 Cellular Rehydration
--------------------------------------------------------------------------------

After cryoprotectant removal, cells must be properly rehydrated.

[ID] Rehydration Process:

    CellularRehydration {
      phase: "REHYDRATION"
      duration: "2-4 hours"

      process: {
        startOsmolarity:  290
        targetOsmolarity: 290
        fluidType:        "Balanced crystalloid"
        rate:             "Based on calculated deficit"
      }

      endpoints: {
        plasmaOsmolarity:     [285, 295]
        urine-specific-gravity: [1.010, 1.025]
        cellularVolume:       "Normal on imaging"
      }
    }

[ID] Fluid Management:

    FluidManagement {
      baseSolution:    "Plasma-Lyte or Lactated Ringer's"

      additions: {
        potassium:     "As needed to maintain 4-5 mEq/L"
        calcium:       "As needed for ionized 1.1-1.3 mmol/L"
        magnesium:     "As needed for 1.5-2.5 mg/dL"
        glucose:       "Maintain 80-120 mg/dL"
      }

      rate: {
        initial:       "20 mL/kg/hour"
        maintenance:   "Based on urine output and losses"
      }
    }

[ID] Monitoring During Rehydration:

    Continuous monitoring of:

    - Central venous pressure (CVP)
    - Urine output
    - Plasma electrolytes
    - Blood gases
    - Lactate levels


--------------------------------------------------------------------------------
4.4 Monitoring Parameters
--------------------------------------------------------------------------------

Comprehensive monitoring ensures safe cryoprotectant removal.

[ID] Laboratory Monitoring Schedule:

    LabMonitoring {
      every30Minutes: [
        "Plasma osmolarity",
        "Sodium",
        "Cryoprotectant concentration"
      ]

      everyHour: [
        "Complete metabolic panel",
        "Blood gas",
        "Lactate",
        "Hemoglobin/Hematocrit"
      ]

      every4Hours: [
        "Complete blood count",
        "Coagulation panel",
        "Liver function",
        "Renal function"
      ]
    }

[ID] Target Ranges During Washout:

    +------------------------------------------------------------------+
    | Parameter            | Target Range        | Critical Limit       |
    +------------------------------------------------------------------+
    | pH                   | 7.35-7.45           | < 7.20 or > 7.55     |
    | Sodium               | 135-145 mEq/L       | < 125 or > 155       |
    | Potassium            | 3.5-5.0 mEq/L       | < 3.0 or > 6.0       |
    | Lactate              | < 2.0 mmol/L        | > 4.0                |
    | Hemoglobin           | > 7 g/dL            | < 6                  |
    | Glucose              | 80-180 mg/dL        | < 60 or > 300        |
    +------------------------------------------------------------------+

[ID] Completion Criteria:

    CryoprotectantRemovalComplete {
      cryoprotectantLevel:   < 0.5%
      plasmaOsmolarity:      [285, 295] mOsm/L
      electrolytesNormal:    true
      cellularIntegrity:     "Intact on bioimpedance"
      hemodynamicStability:  true
      lactateTrend:          "Stable or decreasing"
    }

[i] Once these criteria are met, proceed to cardiovascular restart.
