8.1 Failure Recognition
--------------------------------------------------------------------------------

When revival cannot be completed successfully, re-preservation may be
considered to allow for future revival attempts with advanced technology.

[ID] Revival Failure Indicators:

    FailureIndicators {
      cardiac: [
        "Refractory cardiac arrest despite maximum intervention",
        "Unable to achieve stable rhythm after 4+ hours",
        "Cardiogenic shock unresponsive to ECMO",
        "Cardiac tissue irreversibly damaged"
      ]

      neurological: [
        "No brain electrical activity after 72+ hours",
        "Absent brainstem reflexes after 48+ hours",
        "Diffuse cerebral edema with herniation",
        "Massive brain tissue loss on imaging"
      ]

      multiOrgan: [
        "Three or more organ systems in failure",
        "Refractory metabolic acidosis (pH < 7.0)",
        "Unable to maintain any hemodynamic support"
      ]
    }

[ID] Failure vs. Delayed Recovery:

    +------------------------------------------------------------------+
    | Finding                    | May Recover      | Likely Failure   |
    +------------------------------------------------------------------+
    | Flat EEG                   | < 48 hours       | > 72 hours       |
    | Absent pupils              | < 24 hours       | > 48 hours       |
    | No cardiac rhythm          | < 4 hours        | > 6 hours        |
    | Refractory acidosis        | Improving        | Worsening        |
    | Multi-organ failure        | 2 organs         | 3+ organs        |
    +------------------------------------------------------------------+

[!] Important: Do not declare failure prematurely. Some subjects
    may have delayed recovery. Consult the full team before
    initiating re-preservation consideration.

8.2 Decision Criteria
--------------------------------------------------------------------------------

Re-preservation decisions follow strict criteria and team consensus.

[ID] Re-Preservation Triggers:

    RePreservationTriggers {
      IRREVERSIBLE_CARDIAC_FAILURE: {
        criteria: [
          "Multiple failed cardioversion/defibrillation",
          "Maximum vasopressor and inotrope support",
          "ECMO failure or not feasible",
          "Cardiology consensus on irreversibility"
        ]
      }

      IRREVERSIBLE_BRAIN_DAMAGE: {
        criteria: [
          "No EEG activity > 72 hours",
          "Absent brainstem reflexes > 48 hours",
          "Imaging confirms extensive damage",
          "Neurology consensus on irreversibility"
        ]
      }

      MULTIPLE_ORGAN_FAILURE: {
        criteria: [
          "3+ organ systems failing",
          "Maximum support for all systems",
          "No improvement trend over 24 hours"
        ]
      }

      SUBJECT_REQUEST: {
        criteria: [
          "Subject regains capacity",
          "Subject explicitly refuses continued revival",
          "Documented refusal with witnesses"
        ]
      }

      GUARDIAN_REQUEST: {
        criteria: [
          "Per CRYO-CONSENT document provisions",
          "Guardian consensus per voting rules",
          "After informed consultation with medical team"
        ]
      }
    }

[ID] Standard Criteria Evaluation:

    CriteriaEvaluation {
      CARDIAC_IRREVERSIBILITY: {
        weight:       0.25
        description:  "Cardiac function cannot be restored"
        assessment:   "Multiple failed attempts, refractory to support"
      }

      NEUROLOGICAL_IRREVERSIBILITY: {
        weight:       0.30
        description:  "Evidence of irreversible brain damage"
        assessment:   "Absent reflexes, flat EEG, imaging findings"
      }

      MULTI_ORGAN_FAILURE: {
        weight:       0.20
        description:  "Three or more organs failing"
        assessment:   "Renal, hepatic, respiratory despite max support"
      }

      TIME_THRESHOLD: {
        weight:       0.10
        description:  "Maximum revival attempt duration exceeded"
        assessment:   "Revival attempt > 72 hours without improvement"
      }

      TISSUE_VIABILITY: {
        weight:       0.15
        description:  "Tissue remains viable for re-preservation"
        assessment:   "Perfusion maintained, limited ischemic damage"
      }
    }

[ID] Scoring:

    Criteria Score = Sum of (weight x met) for all criteria

    Score >= 0.70 AND Team Vote > 50% proceed -> REPRESERVE
    Score <  0.30 OR Team Vote < 30% proceed  -> CONTINUE
    Otherwise                                 -> REVIEW (escalate)


--------------------------------------------------------------------------------
8.3 Team Voting
--------------------------------------------------------------------------------

Re-preservation requires formal team voting.

[ID] Voting Participants:

    VotingTeam {
      required: [
        "Revival Director",
        "Cryonics Specialist",
        "Cardiovascular Surgeon",
        "Neurologist"
      ]

      optional: [
        "Anesthesiologist",
        "Ethics Consultant",
        "Guardian Representative"
      ]

      minimumVoters: 3
    }

[ID] Voting Process:

    VotingProcess {
      step1: "Director presents case summary"
      step2: "Review of all criteria evaluations"
      step3: "Open discussion (time-limited: 30 min)"
      step4: "Each voter casts vote with rationale"
      step5: "Votes tallied and documented"
      step6: "Decision announced and recorded"
    }

[ID] Vote Options:

    PROCEED:  "Agree that re-preservation should occur"
    OPPOSE:   "Believe revival should continue"
    ABSTAIN:  "Do not vote (not counted in percentage)"

[ID] Sample Voting Record:

    VoteRecord {
      votes: [
        {
          member:    "Dr. Park (Revival Director)"
          role:      "REVIVAL_DIRECTOR"
          vote:      "PROCEED"
          rationale: "No neurological recovery after 80 hours"
        },
        {
          member:    "Dr. Kim (Cryonics Specialist)"
          role:      "CRYONICS_SPECIALIST"
          vote:      "PROCEED"
          rationale: "Tissue still viable for preservation"
        },
        {
          member:    "Dr. Lee (Neurologist)"
          role:      "NEUROLOGIST"
          vote:      "PROCEED"
          rationale: "EEG flat, no brainstem reflexes"
        },
        {
          member:    "Dr. Choi (Cardiovascular)"
          role:      "CARDIOVASCULAR_SURGEON"
          vote:      "OPPOSE"
          rationale: "Cardiac function improving slightly"
        }
      ]

      summary: {
        PROCEED: 3
        OPPOSE:  1
        ABSTAIN: 0
      }

      decision: "REPRESERVE (75% proceed)"
    }

8.4 Re-Preservation Execution
--------------------------------------------------------------------------------

If re-preservation is decided, execute carefully to maximize future viability.

[ID] Guardian Notification:

    GuardianNotification {
      timing:      "Before execution begins"
      method:      "Direct call + secure message"

      content: {
        situation:    "Revival unsuccessful"
        decision:     "Re-preservation recommended"
        rationale:    "[Summary of medical findings]"
        nextSteps:    "Re-preservation and return to storage"
        futurePlans:  "Await technological advances for future attempt"
      }

      guardianResponse: {
        options:    ["Accept", "Request more time", "Refuse"]
        timeLimit:  "2 hours for response"
        default:    "If no response, proceed per consent document"
      }
    }

[ID] Re-Preservation Protocol:

    RePreservationExecution {
      preparation: {
        team:           "Same team or dedicated preservation team"
        equipment:      "Cooling, perfusion, cryoprotectant ready"
        documentation:  "Full procedure record"
      }

      cooling: {
        method:         "Controlled cooling perfusion"
        rate:           "1-2 C per minute to 0C, then slower"
        targetTemp:     "-10C for cryoprotectant loading"
      }

      cryoprotectantLoading: {
        solution:       "Standard vitrification solution"
        concentration:  "Gradual increase to 60%"
        duration:       "4-6 hours"
      }

      finalCooling: {
        method:         "Controlled rate cooling"
        rate:           "0.5 C per minute"
        targetTemp:     "-196C (liquid nitrogen)"
      }

      storage: {
        facility:       "Return to original or designated facility"
        container:      "Long-term storage dewar"
        monitoring:     "Standard preservation monitoring"
      }
    }

[ID] Documentation:

    RePreservationRecord {
      procedureId:       "REVIVAL-2075-001"
      originalRevival:   "2075-06-15"
      rePreservation:    "2075-06-18"

      revivalDuration:   "72 hours"
      failureReason:     "NEUROLOGICAL_IRREVERSIBILITY"

      criteriaScore:     0.75
      teamVote:          "3-1 proceed"

      rePreservation: {
        coolingStarted:  "2075-06-18T14:00:00Z"
        coolingComplete: "2075-06-18T22:00:00Z"
        cryoprotectant:  "VS-55 derivative"
        finalTemp:       "-196C"
        storageLocation: "WIA Seoul Cryonics Center"
      }

      notes: "Subject re-preserved successfully. Await future
              technological advances for second revival attempt."
    }
