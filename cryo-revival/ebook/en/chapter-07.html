7.1 System Stabilization
--------------------------------------------------------------------------------

The stabilization stage focuses on achieving homeostatic balance
across all organ systems.

[ID] Stabilization Goals:

    StabilizationTargets {
      cardiovascular: {
        heartRate:     [60, 100]
        MAP:           [65, 90]
        rhythm:        "Sinus or stable paced"
        vasopressors:  "Weaning"
      }

      respiratory: {
        PaO2:          [80, 100]
        PaCO2:         [35, 45]
        FiO2:          "Weaning toward room air"
        ventMode:      "Transitioning to spontaneous"
      }

      neurological: {
        GCS:           ">= 8 (emerging from coma)"
        ICP:           "< 20 mmHg"
        seizures:      "None"
        sedation:      "Weaning"
      }

      renal: {
        urineOutput:   "> 0.5 mL/kg/hour"
        creatinine:    "Stable or improving"
        dialysis:      "Discontinue if possible"
      }

      metabolic: {
        pH:            [7.35, 7.45]
        lactate:       "< 2.0 mmol/L"
        glucose:       [80, 180]
        electrolytes:  "Normal ranges"
      }
    }

[ID] Stabilization Checklist:

    +------------------------------------------------------------------+
    | System             | Criteria                 | Status           |
    +------------------------------------------------------------------+
    | Cardiovascular     | Off high-dose pressors   | [ ] Met          |
    | Respiratory        | Spontaneous breathing    | [ ] Met          |
    | Neurological       | GCS improving            | [ ] Met          |
    | Renal              | Urine output adequate    | [ ] Met          |

    | Hematologic        | No active bleeding       | [ ] Met          |

7.2 Multi-Organ Support
--------------------------------------------------------------------------------

Supporting multiple organ systems during recovery.

[ID] Respiratory Support Weaning:

    RespiratoryWeaning {
      criteria: {
        adequate oxygenation: "PaO2/FiO2 > 200"
        hemodynamically_stable: true
        minimal_sedation: true
        adequate_cough: true
      }

      weaningProtocol: {
        step1: "Reduce FiO2 to 40%"
        step2: "Reduce PEEP to 5 cmH2O"
        step3: "Spontaneous breathing trial (SBT)"
        step4: "If SBT passed, extubate"
      }

      postExtubation: {
        supplementalO2:  "As needed"
        monitoring:      "Close respiratory monitoring x 24 hours"
        reintubation:    "If respiratory distress"
      }
    }

[ID] Renal Support:

    RenalManagement {
      acuteKidneyInjury: {
        common:     true
        causes:     ["Hypoperfusion", "Nephrotoxins", "Cryoprotectant"]
        management: ["Fluid optimization", "Avoid nephrotoxins", "RRT if needed"]
      }

      renalReplacementTherapy: {
        indications: [
          "Volume overload refractory to diuretics",
          "Severe hyperkalemia (>6.5 mEq/L)",
          "Severe acidosis (pH < 7.1)",
          "Uremic complications"
        ]
        mode:        "CVVH or intermittent HD based on hemodynamics"
        goal:        "Support until native recovery"
      }
    }

[ID] Nutrition Support:

    NutritionProtocol {
      timing:      "Begin within 24-48 hours of stabilization"
      route:       "Enteral preferred if GI function intact"

      targets: {
        calories:   "25-30 kcal/kg/day"
        protein:    "1.2-1.5 g/kg/day"
      }

      monitoring: [
        "Gastric residuals",
        "Bowel function",
        "Glucose control",
        "Electrolytes"
      ]
    }
