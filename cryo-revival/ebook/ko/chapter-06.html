--------------------------------------------------------------------------------
6.1 뇌 기능 평가
--------------------------------------------------------------------------------

신경학적 회복은 장기 보존 후 뇌 기능 회복에 초점을 맞춘
가장 중요하고 복잡한 단계입니다.

[ID] 초기 신경학적 평가:

    NeurologicalBaseline {
      assessmentTime:  "혈역학적 안정성 달성 후"

      initialExpectation: {
        consciousness:   "무반응"
        gcs:            3
        pupils:         "고정"
        reflexes:       "없음"
        eeg:            "평탄 또는 BURST_SUPPRESSION"
      }

      note: "초기 소견이 최종 결과를 예측하지 않습니다.
             뇌 회복은 수 시간에서 수일에 걸쳐 점진적으로 발생합니다."
    }

[ID] 평가 구성요소:

    +------------------------------------------------------------------+
    | 평가              | 방법                     | 빈도              |
    +------------------------------------------------------------------+
    | 의식 (GCS)        | 임상 검사                | 2시간마다         |
    | 동공 반응         | 펜라이트 검사            | 2시간마다         |
    | 뇌간 반사         | 임상 검사                | 4시간마다         |
    | EEG               | 연속 모니터링            | 연속              |
    | 두개내압 모니터링 | 침습적 모니터            | 연속              |
    | 뇌 영상           | CT/MRI                   | 필요시            |
    +------------------------------------------------------------------+

[ID] 글래스고 혼수 척도 참조:

    GCS 구성요소 점수:

    눈 개방:
      4 - 자발적
      3 - 음성에
      2 - 통증에
      1 - 없음

    언어 반응:
      5 - 지남력 있음
      4 - 혼란
      3 - 부적절한 단어
      2 - 이해 불가 소리
      1 - 없음

    운동 반응:
      6 - 명령 수행
      5 - 통증 위치 파악
      4 - 통증 회피
      3 - 비정상적 굴곡
      2 - 신전
      1 - 없음

    총점: 3-15 (3 = 깊은 혼수, 15 = 완전 의식)


--------------------------------------------------------------------------------
6.2 EEG 모니터링
--------------------------------------------------------------------------------

연속 EEG 모니터링으로 뇌 전기 활동 회복을 추적합니다.

[ID] EEG 패턴 진행:

    예상 회복 패턴:

    FLAT (평탄)                 (초기 - 활동 없음)
       |
       v  (수 시간~수일)
    BURST_SUPPRESSION           (주기적 활동 버스트)
       |
       v  (수일)
    SLOW_WAVE (서파)            (연속적인 느린 활동)
       |
       v  (수일~수주)
    NORMAL (정상)               (정상 리듬 회복)

[ID] EEG 해석:

    EEGPatterns {
      FLAT: {
        description:   "식별 가능한 전기 활동 없음"
        implication:   "초기에 예상됨; 예후적이지 않음"
        action:        "모니터링 계속, 신경보호"
      }

      BURST_SUPPRESSION: {
        description:   "억제와 교대되는 주기적 버스트"
        implication:   "초기 회복 징후"
        action:        "산소화 최적화, 정상 체온 유지"
      }

      SLOW_WAVE: {
        description:   "연속적인 델타/세타 활동"
        implication:   "뇌 회복 중, 아직 정상 아님"
        action:        "지원 계속, 임상 상관관계 평가"
      }

      NORMAL: {
        description:   "알파 리듬, 적절한 반응성"
        implication:   "좋은 신경학적 회복"
        action:        "점진적 진정제 감량 시작"
      }
    }

[!] 발작 활동:

    SeizureManagement {
      detection:    "연속 EEG 모니터링"
      alert:        "즉시 통지"

      treatment: {
        firstLine:    "로라제팜 0.1 mg/kg IV"
        secondLine:   "레베티라세탐 60 mg/kg IV"
        thirdLine:    "프로포폴 또는 미다졸람 주입"
      }

      goal:         "60분 내 발작 활동 억제"
    }


--------------------------------------------------------------------------------
6.3 의식 회복
--------------------------------------------------------------------------------

의식 회복 추적은 핵심 결과 측정입니다.

[ID] 의식 수준 척도:

    ConsciousnessLevels {
      UNRESPONSIVE:      "모든 자극에 반응 없음 (GCS 3)"
      PAIN_RESPONSE:     "통증 자극에만 반응 (GCS 4-5)"
      PRESSURE_RESPONSE: "압력에 반응 (GCS 6-8)"
      VOICE_RESPONSE:    "음성에 반응 (GCS 9-12)"
      CONFUSED:          "깨어 있지만 혼란 (GCS 13-14)"
      ORIENTED:          "완전히 의식 있고 지남력 있음 (GCS 15)"
    }

[ID] 회복 이정표:

    RecoveryMilestones {
      milestone1: {
        event:       "첫 동공 반응"
        typical:     "재시작 후 6-24시간"
        significance: "뇌간 기능 회복"
      }

      milestone2: {
        event:       "통증에 대한 첫 운동 반응"
        typical:     "12-48시간"
        significance: "피질-척수 경로 기능"
      }

      milestone3: {
        event:       "음성에 눈 개방"
        typical:     "24-72시간"
        significance: "각성 시스템 기능"
      }

      milestone4: {
        event:       "명령 수행"
        typical:     "48-96시간"
        significance: "고위 피질 기능"
      }

      milestone5: {
        event:       "언어 반응"
        typical:     "72-168시간"
        significance: "언어 기능 회복"
      }
    }

[ID] 예후 요인:

    +------------------------------------------------------------------+
    | 요인                    | 좋은 예후        | 나쁜 예후          |
    +------------------------------------------------------------------+
    | 보존 품질              | 무결성 >80%      | 무결성 <60%        |
    | EEG 활동까지 시간      | <24시간          | >72시간            |
    | 동공 반응 회복         | <12시간          | >48시간            |
    | 운동 반응 회복         | <24시간          | >72시간            |
    | 발작 활동              | 없음             | 불응성             |
    +------------------------------------------------------------------+


--------------------------------------------------------------------------------
6.4 신경보호
--------------------------------------------------------------------------------

신경보호 전략은 뇌 회복을 지원합니다.

[ID] 신경보호 조치:

    NeuroprotectionProtocol {
      temperatureManagement: {
        target:       "36.0-37.0C (정상 체온)"
        avoid:        "고체온 (>38C) - 더 나쁜 결과와 연관"
        method:       "적극적 체온 관리"
      }

      oxygenation: {
        target:       "PaO2 80-100 mmHg"
        avoid:        "저산소증 (<60) 및 과산소증 (>300)"
        saturation:   "94-98%"
      }

      bloodPressure: {
        target:       "MAP 80-100 mmHg"
        rationale:    "뇌관류압 유지"
        CPP_target:   ">60 mmHg"
      }

      glucose: {
        target:       "140-180 mg/dL"
        avoid:        "저혈당 (<80) 및 심한 고혈당 (>300)"
      }

      CO2: {
        target:       "PaCO2 35-45 mmHg"
        avoid:        "저탄산증 (혈관수축) 및 과탄산증 (두개내압 증가)"
      }
    }

[ID] 약물적 신경보호:

    NeuroprotectiveAgents {
      sedation: {
        agent:        "프로포폴 또는 미다졸람"
        purpose:      "대사 요구 감소"
        titrate:      "필요시 발작 억제까지"
      }

      anticonvulsants: {
        prophylactic: "레베티라세탐 500mg BID"
        indication:   "모든 소생 환자"
      }

      osmotherapy: {
        ifICPElevated: {
          mannitol:    "0.5-1 g/kg IV"
          hypertonic:  "23.4% 식염수 30mL"
        }
      }
    }

[ID] 두개내압 관리:

    ICPManagement {
      normalICP:         "< 20 mmHg"
      treatmentThreshold: "> 22 mmHg"

      tieredApproach: {
        tier1: [
          "침대 머리 30도 상승",
          "중립 두부 위치",
          "적절한 진정"
        ]
        tier2: [
          "삼투요법 (만니톨/고장성 식염수)",
          "가능하면 뇌척수액 배액",
          "과환기로 PaCO2 30-35 (일시적)"
        ]
        tier3: [
          "바비튜레이트 혼수",
          "감압 두개골절제술"
        ]
      }
    }

장 요약

핵심 개념:
  [OK] 신경학적 회복은 수 시간에서 수주에 걸쳐 점진적으로 진행
  [OK] GCS와 EEG가 주요 모니터링 도구
  [OK] EEG 진행: 평탄 -> 버스트 억제 -> 서파 -> 정상
  [OK] 발작 활동은 즉각적인 치료 필요
  [OK] 신경보호는 체온, 산소화, 혈압, 혈당 조절 포함
  [OK] 두개내압 관리는 단계적 접근을 따름
