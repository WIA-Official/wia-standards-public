--------------------------------------------------------------------------------
5.1 심장 리듬 분석
--------------------------------------------------------------------------------

심혈관 재시작 단계는 심장 리듬 분석과 적절한 재시작 접근법
결정으로 시작됩니다.

[ID] 초기 리듬 평가:

    CardiacRhythmAnalysis {
      possibleRhythms: [
        "ASYSTOLE",                 // 무수축 - 전기 활동 없음
        "VENTRICULAR_FIBRILLATION", // 심실세동 - 혼란스러운 전기 활동
        "VENTRICULAR_TACHYCARDIA",  // 심실빈맥 - 빠른 심실 리듬
        "PEA",                      // 무맥성 전기활동
        "BRADYCARDIA",              // 서맥
        "SINUS_RHYTHM"              // 정상 리듬
      ]

      diagnosticTools: [
        "12유도 심전도",
        "연속 심장 모니터링",
        "심초음파",
        "동맥 라인 파형"
      ]
    }

[ID] 리듬 기반 프로토콜 선택:

    +------------------------------------------------------------------+
    | 리듬                    | 주요 접근법                             |
    +------------------------------------------------------------------+
    | 무수축                  | 페이싱 + 약물                           |
    | 심실세동                | 제세동                                  |
    | 심실빈맥                | 심율동전환 + 항부정맥제                 |
    | 무맥성 전기활동         | 가역적 원인 해결 + 지원                 |
    | 서맥                    | 페이싱 +/- 변시성 약물                  |
    +------------------------------------------------------------------+

[ID] 예상 초기 상태:

    장기 보존 후, 초기 리듬은 일반적으로:
    - 무수축 (가장 흔함)
    - 심실세동 (잔여 전기 활동이 있는 경우)

    심장이 장기간 정지 상태였음 - 재시작에는
    인내와 체계적인 접근이 필요합니다.


--------------------------------------------------------------------------------
5.2 제세동 프로토콜
--------------------------------------------------------------------------------

충격 가능 리듬(심실세동/심실빈맥)의 경우, 제세동이 주요 중재입니다.

[ID] 제세동 프로토콜:

    DefibrillationProtocol {
      initialEnergy:     200     // 줄 (이상성)
      maxEnergy:         360     // 줄
      escalationStep:    50      // 시도당 줄
      maxAttempts:       5       // 재평가 전

      padPlacement:      "전측방 또는 전후방"
      chargingTime:      "10초 미만"
      safetyChecks:      "쇼크 전 모든 접촉 제거"
    }

[ID] 제세동 순서:

    1. 충격 가능 리듬 확인 (심실세동 또는 무맥성 심실빈맥)
    2. 패드를 올바른 위치에 부착
    3. 초기 에너지로 충전 (200J)
    4. 클리어하고 확인
    5. 쇼크 전달
    6. 즉시 리듬 평가
    7. 실패 시, 에너지 증가하고 반복

[ID] 제세동 기록:

    DefibrillationAttempt {
      attemptNumber:   1
      timestamp:       "2075-06-15T10:30:00Z"
      energyJoules:    200
      rhythmBefore:    "VENTRICULAR_FIBRILLATION"
      rhythmAfter:     "ASYSTOLE"
      success:         false
      performedBy:     "이심장 박사"
    }

[!] 5회 제세동 시도 실패 후, 상황을 재평가하고
    대안적 접근 또는 재보존을 고려하세요.


--------------------------------------------------------------------------------
5.3 약물 지원
--------------------------------------------------------------------------------

약물은 심혈관 재시작과 안정화를 지원합니다.

[ID] 1차 약물:

    CardiacMedications {
      epinephrine: {
        dose:         "1 mg IV"
        frequency:    "3-5분마다"
        indication:   "모든 심정지 리듬"
        mechanism:    "알파: 혈관수축; 베타: 변시성"
      }

      amiodarone: {
        dose:         "300 mg IV (초기), 150 mg (반복)"
        indication:   "불응성 심실세동/심실빈맥"
        mechanism:    "막 안정화"
      }

      atropine: {
        dose:         "0.5-1 mg IV"
        indication:   "증상성 서맥"
        mechanism:    "부교감신경 차단"
      }
    }

[ID] 혈관활성 지원:

    VasoactiveAgents {
      dopamine: {
        lowDose:      "2-5 mcg/kg/min (신장)"
        mediumDose:   "5-10 mcg/kg/min (강심)"
        highDose:     ">10 mcg/kg/min (혈관수축)"
      }

      norepinephrine: {
        startDose:    "0.1 mcg/kg/min"
        titrate:      "MAP > 65 mmHg 까지"
        maxDose:      "2 mcg/kg/min"
      }

      dobutamine: {
        dose:         "2.5-20 mcg/kg/min"
        indication:   "심인성 쇼크, 저심박출량"
      }
    }

[ID] ECMO 고려:

    약물 지원이 불충분한 경우:

    ECMO Criteria {
      indication: [
        "불응성 심인성 쇼크",
        "MAP > 60 유지 불가",
        "지원에도 불구하고 지속되는 산증"
      ]

      type:         "VA-ECMO (정맥동맥)"
      duration:     "심장 회복 또는 결정까지"
    }


--------------------------------------------------------------------------------
5.4 혈역학 최적화
--------------------------------------------------------------------------------

심장 리듬이 회복되면 혈역학 최적화가 시작됩니다.

[ID] 혈역학 목표:

    HemodynamicTargets {
      heartRate:            [60, 100]     // bpm
      systolicBP:           [90, 140]     // mmHg
      diastolicBP:          [60, 90]      // mmHg
      meanArterialPressure: [65, 100]     // mmHg
      centralVenousPressure: [8, 12]      // mmHg
      cardiacOutput:        [4.0, 8.0]    // L/min
      cardiacIndex:         [2.5, 4.0]    // L/min/m2
    }

[ID] 최적화 알고리즘:

    MAP < 65인 경우:
    +------------------+
    | 용량 상태 확인   |  -> CVP가 낮으면: 수액 볼루스 250-500mL
    |                  |
    +--------+---------+
             |
             v
    +------------------+
    | 수액 후에도      |  -> 혈관수축제 시작/증량
    | 여전히 낮으면    |
    +--------+---------+
             |
             v
    +------------------+
    | 심박출량 평가    |  -> CO가 낮으면: 강심제 추가
    +------------------+

[ID] 심혈관 재시작 성공 기준:

    CardiovascularRestartComplete {
      rhythm:           "SINUS" 또는 "PACED"
      heartRate:        [60, 100]
      systolicBP:       >= 90
      MAP:              >= 65
      lactate:          < 2.0 mmol/L 또는 감소 중
      urineOutput:      > 0.5 mL/kg/hour
      peripheralPerfusion: "따뜻하고, 분홍색"
      echoAssessment:   "적절한 수축력"
    }
