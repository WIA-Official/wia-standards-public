--------------------------------------------------------------------------------
3.1 온도 단계
--------------------------------------------------------------------------------

가온 단계는 대상자를 극저온에서 정상 체온으로 신중하게 제어된
단계를 통해 전환합니다.

[ID] 가온 단계 개요:

    +------------------------------------------------------------------+
    | 단계               | 온도 범위        | 주요 관심사               |
    +------------------------------------------------------------------+
    | VITRIFIED          | -196C ~ -123C    | 유리질 상태 유지          |
    | GLASS_TRANSITION   | -123C ~ -50C     | 임계 전이 구역            |
    | ICE_RISK_ZONE      | -50C ~ -10C      | 얼음 결정 형성            |
    | SUB_FREEZING       | -10C ~ 0C        | 해동 시작                 |
    | HYPOTHERMIC        | 0C ~ 30C         | 점진적 가온               |
    | NORMOTHERMIC       | 30C ~ 37C        | 목표 온도                 |
    +------------------------------------------------------------------+

[ID] 온도 프로파일:

    온도 (C)
    |
    37 |                                                    +-------
    30 |                                               +----+
     0 |                                         +-----+
   -10 |                                   +-----+
   -50 |                          +--------+
  -123 |             +------------+
  -196 +-------------+
       |_____________|___|___|___|___|___|_____|_____|______|______
       0            15  88 168 178 193 ~200  ~210  ~220  시간(분)

[!] 유리 전이 구역(-123C ~ -50C)은 탈유리화와 얼음 결정 형성을
    방지하기 위해 가장 느린 가온 속도가 필요합니다.


--------------------------------------------------------------------------------
3.2 가온 속도 제어
--------------------------------------------------------------------------------

각 단계는 손상을 방지하기 위해 특정 가온 속도가 필요합니다.

[ID] 최적 가온 속도:

    WarmingRateControl {
      phases: [
        {
          phase:        "VITRIFIED"
          tempRange:    [-196, -123]
          maxRate:      5.0          // 분당 C
          method:       "PERFUSION"
          duration:     "~15분"
        },
        {
          phase:        "GLASS_TRANSITION"
          tempRange:    [-123, -50]
          maxRate:      1.0          // 가장 느린 속도
          method:       "PERFUSION"
          duration:     "~73분"
        },
        {
          phase:        "ICE_RISK_ZONE"
          tempRange:    [-50, -10]
          maxRate:      0.5          // 임계 구역
          method:       "PERFUSION"
          duration:     "~80분"
        },
        {
          phase:        "SUB_FREEZING"
          tempRange:    [-10, 0]
          maxRate:      1.0
          method:       "HYBRID"
          duration:     "~10분"
        },
        {
          phase:        "HYPOTHERMIC"
          tempRange:    [0, 30]
          maxRate:      2.0
          method:       "HYBRID"
          duration:     "~15분"
        },
        {
          phase:        "NORMOTHERMIC"
          tempRange:    [30, 37]
          maxRate:      2.0
          method:       "EXTERNAL"
          duration:     "~4분"
        }
      ]
    }

[ID] 가온 방법:

    PERFUSION (관류):
    - 따뜻한 용액을 혈관계를 통해 순환
    - 가장 균일한 가열
    - 심부 저체온에 필요

    EXTERNAL (외부):
    - 표면 가온 담요
    - 대류 공기 가온
    - 최종 온도 조정에 사용

    HYBRID (하이브리드):
    - 관류와 외부의 조합
    - 전환 단계에 사용


--------------------------------------------------------------------------------
3.3 상전이 관리
--------------------------------------------------------------------------------

유리 전이는 가온의 가장 중요한 단계입니다.

[ID] 유리 전이 모니터링:

    GlassTransitionControl {
      transitionTemperature: -123  // 유리화 용액의 일반적 온도

      monitoring: {
        temperaturePoints: ["CORE", "BRAIN", "PERIPHERAL"]
        sensorInterval:    "10초"
        uniformityTolerance: 2.0  // 포인트 간 최대 차이
      }

      alerts: {
        rateExceeded:      "즉시 가열 감소"
        nonUniformity:     "관류 흐름 조정"
        devitrification:   "위험 - 얼음 형성 감지됨"
      }
    }

[ID] 중요 모니터링 매개변수:

    +------------------------------------------------------------------+
    | 매개변수           | 목표               | 경고 임계값             |
    +------------------------------------------------------------------+
    | 온도 기울기        | 포인트 간 2C 미만  | 3C 차이 초과            |
    | 가온 속도          | 단계별 사양        | 사양의 20% 초과         |
    | 관류 압력          | 30-50 mmHg         | 60 mmHg 초과            |
    | 유량               | 체질량 기준        | 목표의 80% 미만         |
    +------------------------------------------------------------------+

[!] 탈유리화(얼음 결정 형성)가 감지되면 절차를 중단하고
    재보존을 고려해야 할 수 있습니다.


--------------------------------------------------------------------------------
3.4 얼음 결정 방지
--------------------------------------------------------------------------------

얼음 결정 형성 방지는 성공적인 소생에 필수적입니다.

[ID] 얼음 방지 전략:

    1. 느린 가온 속도 유지
       - 각 단계의 최대 속도를 초과하지 않음
       - 속도 초과 모니터링
       - 자동 속도 제어 권장

    2. 온도 균일성
       - 다중 온도 센서
       - 균일한 관류 분포
       - 표면 가열 방지를 위한 외부 가온

    3. 동결보호제 관리
       - 가온 중 적절한 농도 유지
       - 유리 전이 후 점진적 세척
       - 삼투압 기울기 모니터링

    4. 실시간 영상
       - 얼음 결정 감지용 초음파
       - 조직 에코 발생도 변화 모니터링
       - 결정 형성 시 즉시 경고

[ID] 얼음 결정 감지:

    IceCrystalDetection {
      methods: [
        "초음파 영상",
        "조직 임피던스 모니터링",
        "광간섭 단층촬영"
      ]

      responseToDetection: {
        mild:     "가온 속도 50% 감소"
        moderate: "가온 일시 중지, 평가"
        severe:   "재보존 고려"
      }
    }

[ID] 가온 완료 기준:

    WarmingComplete {
      coreTemperature:        37.0    // +/- 0.5C
      peripheralTemperature:  36.5    // +/- 1.0C
      brainTemperature:       37.0    // +/- 0.3C
      uniformity:             true    // 모든 포인트 허용 범위 내
      iceCrystals:            "없음"
      tissuePerfusion:        "적절함"
    }
