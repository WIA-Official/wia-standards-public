--------------------------------------------------------------------------------
4.1 관류 세척
--------------------------------------------------------------------------------

가온 후, 동결보호제는 관류 세척을 통해 신중하게 제거해야 합니다.

[ID] 동결보호제 제거 프로토콜:

    CryoprotectantRemoval {
      initialConcentration:   60      // 백분율
      targetConcentration:    0.5     // 백분율 (미량)
      removalRate:            5       // 시간당 백분율
      estimatedDuration:      "12시간"

      perfusionSolution: {
        base:         "변형 링거 용액"
        osmolarity:   "점진적 감소"
        temperature:  "37C"
        oxygenation:  "100% O2"
      }
    }

[ID] 세척 단계:

    +------------------------------------------------------------------+
    | 단계 | 농도          | 기간    | 용액 삼투압                     |
    +------------------------------------------------------------------+
    |  1   | 60% -> 40%    | 4시간   | 1200 mOsm/L (고삼투압)          |
    |  2   | 40% -> 20%    | 4시간   | 800 mOsm/L                      |
    |  3   | 20% -> 5%     | 3시간   | 400 mOsm/L                      |
    |  4   | 5% -> 0.5%    | 1시간   | 290 mOsm/L (등삼투압)           |
    +------------------------------------------------------------------+

[!] 동결보호제의 급격한 제거는 삼투압 쇼크와 세포 손상을 유발합니다.
    점진적인 농도 감소가 필수적입니다.


--------------------------------------------------------------------------------
4.2 삼투압 균형
--------------------------------------------------------------------------------

세척 중 삼투압 균형 유지는 세포 손상을 방지합니다.

[ID] 삼투압 관리:

    OsmoticControl {
      targetOsmolarity: 290  // mOsm/L (정상 혈장)

      gradientLimits: {
        maxChangeRate:     50      // 시간당 mOsm/L
        maxIntracellular:  20      // mOsm/L 차이
      }

      monitoring: {
        plasmaOsmolarity:  "30분마다"
        cellularIntegrity: "연속 생체임피던스"
        fluidBalance:      "연속"
      }
    }

[ID] 삼투압 제제:

    점진적 삼투압 감소를 위해:

    - 만니톨:     초기 고삼투압 지원
    - 알부민:     교질삼투압 유지
    - 전해질:     생리적 균형

[ID] 삼투압 손상 경고 징후:

    +------------------------------------------------------------------+
    | 징후                     | 표시                                  |
    +------------------------------------------------------------------+
    | 세포 부종                | 너무 빠른 희석                        |
    | 용혈 (샘플에서)          | 적혈구 손상                           |
    | 생체임피던스 증가        | 막 파괴                               |
    | 칼륨 상승                | 세포 용해                             |
    +------------------------------------------------------------------+


--------------------------------------------------------------------------------
4.3 세포 재수화
--------------------------------------------------------------------------------

동결보호제 제거 후, 세포는 적절하게 재수화되어야 합니다.

[ID] 재수화 과정:

    CellularRehydration {
      phase: "REHYDRATION"
      duration: "2-4시간"

      process: {
        startOsmolarity:  290
        targetOsmolarity: 290
        fluidType:        "균형 결정질액"
        rate:             "계산된 결손량 기준"
      }

      endpoints: {
        plasmaOsmolarity:      [285, 295] mOsm/L
        urineSpecificGravity:  [1.010, 1.025]
        cellularVolume:        "영상에서 정상"
      }
    }

[ID] 수액 관리:

    FluidManagement {
      baseSolution:    "플라즈마-라이트 또는 젖산 링거"

      additions: {
        potassium:     "4-5 mEq/L 유지에 필요시"
        calcium:       "이온화 칼슘 1.1-1.3 mmol/L에 필요시"
        magnesium:     "1.5-2.5 mg/dL에 필요시"
        glucose:       "80-120 mg/dL 유지"
      }

      rate: {
        initial:       "20 mL/kg/hour"
        maintenance:   "소변량과 손실 기준"
      }
    }

[ID] 재수화 중 모니터링:

    연속 모니터링 대상:

    - 중심정맥압 (CVP)
    - 소변량
    - 혈장 전해질
    - 혈액 가스
    - 젖산 수준


--------------------------------------------------------------------------------
4.4 모니터링 매개변수
--------------------------------------------------------------------------------

종합적인 모니터링으로 안전한 동결보호제 제거를 보장합니다.

[ID] 검사실 모니터링 일정:

    LabMonitoring {
      every30Minutes: [
        "혈장 삼투압",
        "나트륨",
        "동결보호제 농도"
      ]

      everyHour: [
        "종합 대사 패널",
        "혈액 가스",
        "젖산",
        "헤모글로빈/헤마토크릿"
      ]

      every4Hours: [
        "전혈구 계산",
        "응고 패널",
        "간 기능",
        "신장 기능"
      ]
    }

[ID] 세척 중 목표 범위:

    +------------------------------------------------------------------+
    | 매개변수            | 목표 범위          | 위험 한계             |
    +------------------------------------------------------------------+
    | pH                  | 7.35-7.45          | < 7.20 또는 > 7.55    |
    | 나트륨              | 135-145 mEq/L      | < 125 또는 > 155      |
    | 칼륨                | 3.5-5.0 mEq/L      | < 3.0 또는 > 6.0      |
    | 젖산                | < 2.0 mmol/L       | > 4.0                 |
    | 헤모글로빈          | > 7 g/dL           | < 6                   |
    | 포도당              | 80-180 mg/dL       | < 60 또는 > 300       |
    +------------------------------------------------------------------+

[ID] 완료 기준:

    CryoprotectantRemovalComplete {
      cryoprotectantLevel:   < 0.5%
      plasmaOsmolarity:      [285, 295] mOsm/L
      electrolytesNormal:    true
      cellularIntegrity:     "생체임피던스에서 정상"
      hemodynamicStability:  true
      lactateTrend:          "안정 또는 감소"
    }

[i] 이 기준이 충족되면 심혈관 재시작으로 진행합니다.
