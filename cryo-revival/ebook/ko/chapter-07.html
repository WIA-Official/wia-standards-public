7.1 시스템 안정화
--------------------------------------------------------------------------------

안정화 단계는 모든 장기 시스템에 걸쳐 항상성 균형 달성에
초점을 맞춥니다.

[ID] 안정화 목표:

    StabilizationTargets {
      cardiovascular: {
        heartRate:     [60, 100]
        MAP:           [65, 90]
        rhythm:        "동율동 또는 안정적 페이싱"
        vasopressors:  "감량 중"
      }

      respiratory: {
        PaO2:          [80, 100]
        PaCO2:         [35, 45]
        FiO2:          "실내 공기로 감량 중"
        ventMode:      "자발 호흡으로 전환 중"
      }

      neurological: {
        GCS:           ">= 8 (혼수에서 벗어나는 중)"
        ICP:           "< 20 mmHg"
        seizures:      "없음"
        sedation:      "감량 중"
      }

      renal: {
        urineOutput:   "> 0.5 mL/kg/hour"
        creatinine:    "안정 또는 개선 중"
        dialysis:      "가능하면 중단"
      }

      metabolic: {
        pH:            [7.35, 7.45]
        lactate:       "< 2.0 mmol/L"
        glucose:       [80, 180]
        electrolytes:  "정상 범위"
      }
    }

[ID] 안정화 체크리스트:

    +------------------------------------------------------------------+
    | 시스템            | 기준                      | 상태             |
    +------------------------------------------------------------------+
    | 심혈관            | 고용량 승압제 중단        | [ ] 충족         |
    | 호흡기            | 자발 호흡                 | [ ] 충족         |
    | 신경계            | GCS 개선 중               | [ ] 충족         |
    | 신장              | 소변량 적절               | [ ] 충족         |
    | 간                | 간 기능 정상              | [ ] 충족         |
    | 혈액학            | 활성 출혈 없음            | [ ] 충족         |
    | 감염              | 활성 감염 없음            | [ ] 충족         |
    | 체온              | 정상 체온                 | [ ] 충족         |
    +------------------------------------------------------------------+

7.2 다장기 지원
--------------------------------------------------------------------------------

회복 중 여러 장기 시스템을 지원합니다.

[ID] 호흡 지원 이탈:

    RespiratoryWeaning {
      criteria: {
        adequate_oxygenation: "PaO2/FiO2 > 200"
        hemodynamically_stable: true
        minimal_sedation: true
        adequate_cough: true
      }

      weaningProtocol: {
        step1: "FiO2를 40%로 감소"
        step2: "PEEP를 5 cmH2O로 감소"
        step3: "자발 호흡 시험 (SBT)"
        step4: "SBT 통과 시, 발관"
      }

      postExtubation: {
        supplementalO2:  "필요시"
        monitoring:      "24시간 동안 면밀한 호흡 모니터링"
        reintubation:    "호흡 곤란 시"
      }
    }

[ID] 신장 지원:

    RenalManagement {
      acuteKidneyInjury: {
        common:     true
        causes:     ["저관류", "신독성물질", "동결보호제"]
        management: ["수액 최적화", "신독성물질 회피", "필요시 RRT"]
      }

      renalReplacementTherapy: {
        indications: [
          "이뇨제에 불응하는 체액 과부하",
          "심한 고칼륨혈증 (>6.5 mEq/L)",
          "심한 산증 (pH < 7.1)",
          "요독 합병증"
        ]
        mode:        "혈역학에 따라 CVVH 또는 간헐적 HD"
        goal:        "자연 회복까지 지원"
      }
    }

[ID] 영양 지원:

    NutritionProtocol {
      timing:      "안정화 후 24-48시간 내 시작"
      route:       "위장관 기능이 정상이면 경장 선호"

      targets: {
        calories:   "25-30 kcal/kg/day"
        protein:    "1.2-1.5 g/kg/day"
      }

      monitoring: [
        "위 잔류량",
        "장 기능",
        "혈당 조절",
        "전해질"
      ]
    }
