--------------------------------------------------------------------------------
2.1 대상자 평가
--------------------------------------------------------------------------------

소생 시작 전, 보존된 대상자에 대한 종합 평가가 필요합니다.

[ID] 소생 전 평가 기록:

    PreRevivalAssessment {
      assessmentId:     "ASSESS-2075-001"
      assessmentDate:   "2075-06-01"
      assessedBy:       "김소생 박사, 소생 책임자"

      preservationStatus: {
        duration:         "50년"
        storageFacility:  "WIA 서울 냉동보존 센터"
        storageConditions: "액체질소, -196C"
        integrityScore:   92
        transferHistory:  [...]
      }
    }

[ID] 평가 구성요소:

    +------------------------------------------------------------------+
    | 구성요소             | 목적                                      |
    +------------------------------------------------------------------+
    | 보존 이력            | 기간, 저장 조건, 이전                     |
    | 조직 영상            | 구조적 무결성 평가를 위한 MRI, CT         |
    | 바이오마커 분석      | 세포 손상 지표 평가                       |
    | 유리화 품질          | 얼음 결정 형성 확인                       |
    | 의료 기록            | 원래 상태, 사망 원인                      |
    +------------------------------------------------------------------+

[ID] 동의 확인:

    모든 소생 활동 전:

    1. CRYO-CONSENT 문서가 활성 상태인지 확인
    2. 소생 조건 충족 확인
    3. 동의 조건에 따라 모든 후견인에게 통지
    4. 필요한 승인 획득
    5. 모든 확인 문서화


--------------------------------------------------------------------------------
2.2 조직 상태 평가
--------------------------------------------------------------------------------

조직 상태에 대한 상세 평가로 소생 타당성을 결정합니다.

[ID] 조직 평가 기록:

    TissueAssessment {
      tissue:           "뇌"
      condition:        "GOOD"
      integrityScore:   88
      concerns:         ["피질의 경미한 얼음 결정 흔적"]
      specialConsiderations: ["가온 시 추가 주의"]
    }

[ID] 조직 등급 척도:

    EXCELLENT (90-100):  최소 손상, 최적의 보존
    GOOD (75-89):        경미한 손상, 좋은 예후
    FAIR (60-74):        중등도 손상, 신중한 예후
    POOR (40-59):        심각한 손상, 불확실한 결과
    CRITICAL (<40):      심한 손상, 소생 권장하지 않음

[ID] 중요 조직:

    +------------------------------------------------------------------+
    | 조직              | 가중치 | 최소 점수 | 비고                     |
    +------------------------------------------------------------------+
    | 뇌                | 40%    | 60        | 주요 관심사              |
    | 심장              | 25%    | 50        | 재시작 필수              |
    | 혈관계            | 15%    | 55        | 관류 경로                |
    | 폐                | 10%    | 50        | 지원 가능                |
    | 기타 장기         | 10%    | 40        | 지원 가능                |
    +------------------------------------------------------------------+

[!] 뇌 조직 점수가 60 미만이면 소생을 권장하지 않습니다.


--------------------------------------------------------------------------------
2.3 시설 준비 상태
--------------------------------------------------------------------------------

소생 시설은 진행 전 특정 요구사항을 충족해야 합니다.

[ID] 시설 준비 체크리스트:

    FacilityReadiness {
      facilityId:    "FAC-SEOUL-001"
      facilityName:  "WIA 소생 센터 서울"

      revivalSuite: {
        available:          true
        lastCalibration:    "2075-05-15"
        temperatureControl: "OPERATIONAL"
        perfusionSystem:    "OPERATIONAL"
        monitoring:         "OPERATIONAL"
      }

      operatingRoom: {
        available:    true
        sterilized:   true
        equipment:    [모두 작동]
      }

      icuBeds: {
        available: 5
        reserved:  1
      }

      powerSystems: {
        primary:            "OPERATIONAL"
        backup:             "OPERATIONAL"
        emergencyGenerator: "OPERATIONAL"
      }
    }

[ID] 장비 요구사항:

    필수 장비:
    - 제어된 가온 시스템 (정밀도 +/- 0.1C)
    - 산소공급기가 있는 관류 회로
    - 외부 페이싱이 가능한 제세동기
    - 다중 매개변수 모니터링 시스템
    - ECMO 기능
    - 투석 장비
    - 기계 환기기
    - EEG 모니터링 시스템

[ID] 최소 준비 점수:

    소생 시작 전 전체 준비 점수는 85% 이상이어야 합니다.


--------------------------------------------------------------------------------
2.4 팀 구성
--------------------------------------------------------------------------------

소생 절차에는 전문 팀이 필요합니다.

[ID] 소생 팀 구조:

    RevivalTeam {
      teamId:       "TEAM-2075-001"

      director: {
        name:  "박정호 박사"
        role:  "REVIVAL_DIRECTOR"
        certifications: ["WIA 소생 책임자", "중환자 치료"]
      }

      members: [
        { role: "CRYONICS_SPECIALIST",      count: 2 },
        { role: "CARDIOVASCULAR_SURGEON",   count: 1 },
        { role: "NEUROLOGIST",              count: 1 },
        { role: "ANESTHESIOLOGIST",         count: 1 },
        { role: "PERFUSIONIST",             count: 2 },
        { role: "ICU_NURSE",                count: 4 },
        { role: "MONITORING_SPECIALIST",    count: 2 },
        { role: "SUPPORT_STAFF",            count: 4 }
      ]
    }

[ID] 필수 자격:

    +------------------------------------------------------------------+
    | 역할                     | 필수 자격                             |
    +------------------------------------------------------------------+
    | 소생 책임자              | WIA 소생 책임자, 중환자 치료          |
    | 냉동보존 전문의          | WIA 냉동보존 인증                     |
    | 심혈관 외과의            | 전문의 자격, ECMO 훈련                |
    | 신경과 전문의            | 전문의 자격, 신경모니터링             |
    | 관류사                   | WIA 관류 인증                         |
    +------------------------------------------------------------------+

[ID] 팀 브리핑 요구사항:

    소생 시작 전:

    1. 대상자 병력 검토
       - 원래 상태와 사망 원인
       - 보존 세부사항과 기간
       - 동의 문서 핵심 사항

    2. 평가 검토
       - 조직 상태 소견
       - 식별된 위험 요소
       - 예상되는 도전과제

    3. 프로토콜 검토
       - 단계별 계획
       - 역할 배정
       - 소통 프로토콜

    4. 비상 계획
       - 재보존 기준
       - 응급 절차
       - 결정 에스컬레이션
