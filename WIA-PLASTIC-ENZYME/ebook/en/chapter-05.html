Chapter 5

Phase 2: API Interface

RESTful APIs, SDKs, and service integration for enzymatic plastic degradation

5.1 API Architecture Overview

Phase 2 of the WIA-PLASTIC-ENZYME standard defines the API interfaces that enable software systems to interact with enzymatic plastic degradation services. The architecture follows REST principles with JSON payloads, supporting both synchronous and asynchronous operations.

Design Principles

  • RESTful: Resource-oriented URLs with standard HTTP methods
  • JSON-first: All requests and responses in JSON format
  • Versioned: API versioning in URL path (/v1/, /v2/)
  • Authenticated: API key or OAuth 2.0 authentication
  • Rate-limited: Configurable rate limits per client

5.2 Core API Endpoints

Plastic Identification

POST /api/v1/plastic/identify

Identify plastic type from spectroscopic or image data

// Request POST /api/v1/plastic/identify Content-Type: application/json { "method": "ftir_spectrum", "data": { "wavenumbers": [4000, 3999, 3998, ...], "absorbance": [0.02, 0.021, 0.019, ...] } } // Response { "plastic_type": "PET", "confidence": 0.98, "crystallinity_estimate": 28.5, "contaminants_detected": ["PP traces"], "recommended_pretreatment": "mechanical_grinding" }

Enzyme Matching

GET /api/v1/enzyme/match/{plastic_type}

Find optimal enzymes for a given plastic type and conditions

// Request GET /api/v1/enzyme/match/PET?temperature=55&ph=8.0&target_efficiency=95 // Response { "recommendations": [ { "enzyme_id": "urn:wia:enzyme:turbopetase-v2", "name": "TurboPETase-v2", "match_score": 98.5, "predicted_efficiency": 97.2, "optimal_concentration_mg_g": 2.5, "estimated_time_hours": 48 }, { "enzyme_id": "urn:wia:enzyme:diego-v1", "name": "DIEGO", "match_score": 95.2, "predicted_efficiency": 96.0, "optimal_concentration_mg_g": 3.0, "estimated_time_hours": 96 } ], "cocktail_suggestion": { "primary": "urn:wia:enzyme:turbopetase-v2", "secondary": "urn:wia:enzyme:mhetase-std", "ratio": "2:1", "synergy_bonus": "+12%" } }

Enzyme Library

GET /api/v1/enzyme/library

Search and browse the enzyme database

// Request with filters GET /api/v1/enzyme/library?classification=PETase&min_temperature=50&engineered=true // Response { "total": 47, "page": 1, "per_page": 20, "enzymes": [ { "enzyme_id": "urn:wia:enzyme:turbopetase-v2", "name": "TurboPETase-v2", "classification": "PETase", "source_organism": "Engineered E. coli", "optimal_temperature": 55, "kcat_km": 68700, "reference": "doi:10.1038/s41467-024-45662-9" }, // ... more enzymes ] }

Degradation Optimization

POST /api/v1/degradation/optimize

Calculate optimal degradation conditions for given inputs

// Request POST /api/v1/degradation/optimize { "plastic": { "type": "PET", "weight_kg": 1000, "crystallinity_percent": 30, "form": "flakes" }, "constraints": { "max_temperature_c": 60, "max_time_hours": 72, "target_efficiency": 95 }, "optimization_goal": "cost" // or "speed" or "efficiency" } // Response { "optimal_conditions": { "temperature_c": 55, "ph": 8.0, "enzyme_loading_mg_g": 2.5, "substrate_loading_percent": 12 }, "enzyme_cocktail": [ {"enzyme_id": "urn:wia:enzyme:turbopetase-v2", "ratio": 0.7}, {"enzyme_id": "urn:wia:enzyme:mhetase-std", "ratio": 0.3} ], "predicted_outcomes": { "efficiency_percent": 97.5, "time_hours": 48, "tpa_yield_kg": 820, "cost_per_kg": 0.45 } }

Degradation Prediction

POST /api/v1/degradation/predict

Predict degradation outcomes for specified conditions

// Request POST /api/v1/degradation/predict { "plastic": {"type": "PET", "weight_kg": 100, "crystallinity_percent": 25}, "enzyme": {"enzyme_id": "urn:wia:enzyme:turbopetase-v2", "concentration_mg_g": 2.0}, "conditions": {"temperature_c": 55, "ph": 8.0, "duration_hours": 48} } // Response { "predictions": { "degradation_percent": {"mean": 95.5, "std": 2.1, "ci_95": [91.3, 99.7]}, "tpa_yield_kg": {"mean": 81.2, "std": 1.8}, "eg_yield_kg": {"mean": 27.8, "std": 0.6}, "residue_kg": {"mean": 4.5, "std": 2.1} }, "time_course": [ {"hour": 0, "degradation": 0}, {"hour": 12, "degradation": 45.2}, {"hour": 24, "degradation": 78.5}, {"hour": 36, "degradation": 91.2}, {"hour": 48, "degradation": 95.5} ], "model_version": "wia-ml-v2.3" }

5.3 Authentication and Security

The WIA-PLASTIC-ENZYME API supports two authentication methods:

API Key Authentication

// Include API key in header GET /api/v1/enzyme/library Authorization: Bearer wia_key_xxxxxxxxxxxxx

OAuth 2.0

// OAuth 2.0 flow for enterprise integrations POST /oauth/token Content-Type: application/x-www-form-urlencoded grant_type=client_credentials &client_id=your_client_id &client_secret=your_client_secret &scope=enzyme:read process:write

Available Scopes

  • enzyme:read - Read enzyme library and profiles
  • enzyme:write - Create and update enzyme profiles
  • process:read - Read degradation process data
  • process:write - Create and update process records
  • quality:read - Access quality metrics
  • quality:certify - Issue quality certifications

5.4 SDK Integration

Official SDKs are available for major programming languages:

TypeScript/JavaScript

npm install @wia/plastic-enzyme-sdk import { WiaPlasticEnzyme } from '@wia/plastic-enzyme-sdk'; const client = new WiaPlasticEnzyme({ apiKey: process.env.WIA_API_KEY, environment: 'production' }); // Find optimal enzyme const matches = await client.enzymes.match('PET', { temperature: 55, targetEfficiency: 95 }); console.log(`Best match: ${matches[0].name}`); console.log(`Predicted efficiency: ${matches[0].predicted_efficiency}%`);

Python

pip install wia-plastic-enzyme from wia_plastic_enzyme import WiaClient client = WiaClient(api_key="your-api-key") # Predict degradation result = client.degradation.predict( plastic_type="PET", weight_kg=100, enzyme_id="urn:wia:enzyme:turbopetase-v2", temperature=55, duration_hours=48 ) print(f"Expected efficiency: {result.predictions.degradation_percent.mean}%")

Rust

// Cargo.toml [dependencies] wia-plastic-enzyme = "1.0" // main.rs use wia_plastic_enzyme::{Client, PlasticType}; #[tokio::main] async fn main() -> Result<(), Box> { let client = Client::new("your-api-key")?; let enzymes = client.enzymes() .match_for(PlasticType::PET) .temperature(55.0) .target_efficiency(95.0) .send() .await?; println!("Found {} matching enzymes", enzymes.len()); Ok(()) }

5.5 Webhooks and Events

The API supports webhooks for asynchronous notifications:

// Register a webhook POST /api/v1/webhooks { "url": "https://your-system.com/wia-events", "events": ["process.completed", "quality.certified"], "secret": "your-webhook-secret" } // Webhook payload example { "event": "process.completed", "timestamp": "2025-01-15T10:30:00Z", "data": { "process_id": "urn:wia:process:2025-001-abc", "status": "completed", "efficiency": 97.5 }, "signature": "sha256=xxxxxxxx" }

5.8 Chapter Summary

Phase 2 of the WIA-PLASTIC-ENZYME standard provides comprehensive API interfaces for integrating enzymatic plastic degradation into software systems:

  • RESTful endpoints for plastic identification, enzyme matching, and process optimization
  • Secure authentication via API keys or OAuth 2.0
  • Official SDKs for TypeScript, Python, and Rust
  • Webhook support for event-driven architectures
  • Comprehensive error handling with actionable messages
