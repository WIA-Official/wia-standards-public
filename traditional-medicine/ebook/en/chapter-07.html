Chapter 7: Phase 4 - Integration

Connecting Traditional Medicine with Modern Healthcare Systems

7.1 HL7 FHIR Integration

The WIA-TRADITIONAL-MEDICINE standard provides FHIR profiles for representing traditional medicine data within standard EHR systems:

FHIR Resource Mappings

  • Condition → Pattern Diagnosis (ICD-11 TM codes)
  • Observation → Tongue/Pulse findings, Constitution scores
  • MedicationRequest → Herbal prescriptions
  • Procedure → Acupuncture, Moxibustion treatments
  • DiagnosticReport → Four Examinations summary

7.2 ICD-11 Traditional Medicine Module

The ICD-11 includes a supplementary chapter for Traditional Medicine conditions, marking a historic milestone in global health classification:

ICD-11 TM Structure:
  • TM1 - Traditional Medicine Conditions
  • TM1:S - Patterns/syndromes
  • TM1:SA-SZ - Specific pattern categories
  • Coverage: 3,100+ patterns from TCM/Korean Medicine

Pattern Code Examples

  • TM1:SA00 - Yin deficiency pattern
  • TM1:SB00 - Yang deficiency pattern
  • TM1:SC00 - Qi deficiency pattern
  • TM1:SD00 - Blood deficiency pattern
  • TM1:SE00 - Qi stagnation pattern
  • TM1:SF00 - Blood stasis pattern

7.3 WHO TCIM Dashboard Integration

The WIA-TRADITIONAL-MEDICINE standard supports data exchange with the WHO Traditional, Complementary, and Integrative Medicine (TCIM) global dashboard:

Data Exchange Capabilities

Connected Databases

  • PharmGKB: Pharmacogenomic annotations
  • KEGG: Pathway and compound information
  • PubChem: Chemical structure data
