2.1 Quantitative Genetics Foundations

Genomic selection is built on quantitative genetics principles developed over the past century. The central equation describes how phenotype arises from genetics and environment:

P = G + E

Where:
P = Phenotype (observed trait value)
G = Genotype (genetic value)
E = Environment (non-genetic effects)

2.1.1 Heritability

Heritability (h²) is the proportion of phenotypic variance due to additive genetic effects:

h² = σ²A / σ²P = σ²A / (σ²A + σ²E)

Where:
σ²A = Additive genetic variance
σ²P = Phenotypic variance
σ²E = Environmental variance

2.2 BLUP: Best Linear Unbiased Prediction

BLUP is the gold standard for estimating breeding values using pedigree and phenotype data. Developed by C.R. Henderson in 1949, it revolutionized animal breeding.

2.2.1 Henderson's Mixed Model Equations

[X'X    X'Z  ] [β^] = [X'y]
 [Z'X  Z'Z+A⁻¹λ] [u^]   [Z'y]

Where:
β^ = Estimated fixed effects
u^ = Estimated breeding values (EBVs)
A = Numerator relationship matrix (pedigree)
λ = (1-h²)/h² = variance ratio

2.3 GBLUP: Genomic BLUP

GBLUP replaces the pedigree-based relationship matrix (A) with a genomic relationship matrix (G) constructed from SNP data.

2.3.1 Genomic Relationship Matrix (VanRaden Method 1)

G = ZZ' / (2Σpᵢ(1-pᵢ))

Where:
Z = Centered genotype matrix (M - 2p)
M = Genotype matrix (0, 1, 2)
p = Allele frequencies

The G matrix diagonal elements represent individual inbreeding, off-diagonals represent genomic similarity.

2.3.2 Single-Step GBLUP (ssGBLUP)

Combines pedigree (A) and genomic (G) information in a single analysis:

H = A₂₂ + [ 0      0   ]
            [ 0  G-A₂₂ ]

Where:
H = Combined relationship matrix
A₂₂ = Pedigree relationships for genotyped animals
G = Genomic relationships

2.5 Machine Learning for Genomic Prediction

2.5.1 Neural Networks

Deep learning can capture non-additive effects (epistasis, dominance):

# Simple neural network for GEBV prediction
model = Sequential([
    Dense(512, activation='relu', input_shape=(n_markers,)),
    Dropout(0.3),
    Dense(256, activation='relu'),
    Dropout(0.3),
    Dense(128, activation='relu'),
    Dense(1)  # GEBV output
])

model.compile(optimizer='adam', loss='mse')

Summary

Genomic selection combines quantitative genetics theory with modern genomic data to predict breeding values with unprecedented accuracy. The choice of method depends on trait architecture, computational resources, and desired accuracy.
