제6장: 감각 피드백 및 체화

弘益人間 (홍익인간)

감각 피드백 기술이 잃어버린 접촉 감각을 복원하여 의수를 단순한 도구에서 신체의 확장으로 변환합니다.

6.1 감각 피드백의 중요성

인간 손의 감각 능력은 놀라울 정도로 정교합니다. 손가락 끝에는 평방 센티미터당 2,500개 이상의 기계수용기가 있어 미세한 질감, 온도, 압력을 감지합니다. 이 감각 정보 없이 의수 사용자는 시각에만 의존해야 하며, 이는 인지 부하를 증가시키고 섬세한 조작을 어렵게 만듭니다.

6.1.1 감각 피드백의 이점

  • 파지력 조절: 물체 손상 없이 적절한 힘으로 잡기
  • 미끄러짐 감지: 물체가 미끄러지기 전에 파지력 조절
  • 질감 인식: 표면 특성으로 물체 식별
  • 체화 증가: 의수가 신체의 일부로 느껴짐
  • 환지통 감소: 정상적인 감각 입력 복원으로 통증 완화

6.2 피드백 양식

6.2.1 진동촉각 피드백

가장 보편적인 비침습적 피드백 방법:

  • 원리: 진동 모터가 피부에 진동 자극 전달
  • 장점: 저비용, 간단한 구현, 비침습적
  • 단점: 제한된 정보 대역폭, 자극 위치 혼동 가능
  • 적용: 파지력 피드백, 이벤트 알림
// 진동 피드백 매핑 예시
function mapGripForceToVibration(gripForce) {
    const normalizedForce = gripForce / MAX_GRIP_FORCE;
    const frequency = 50 + normalizedForce * 200;  // 50-250 Hz
    const amplitude = 0.2 + normalizedForce * 0.8; // 20-100%
    return { frequency, amplitude };
}

6.2.2 전기촉각 피드백

  • 원리: 피부 표면 전극을 통한 전기 자극
  • 장점: 빠른 응답, 다양한 감각 유도 가능
  • 단점: 불편감, 피부 상태에 민감
  • 적용: 질감, 미끄러짐 피드백

6.2.3 기계촉각 피드백

  • 압력 커프: 공압으로 잔존 사지 주위 압력 변화
  • 피부 스트레칭: 피부를 잡아당겨 방향 정보 전달
  • 힘 피드백: 기계적으로 저항력 전달

6.2.4 신경 자극 피드백

가장 자연스러운 감각을 제공하는 침습적 방법:

  • 말초신경 자극: 이식 전극으로 감각 신경 직접 자극
  • 특정 손가락 감각: 해당 신경 영역 자극으로 특정 손가락 접촉 느낌
  • 자연스러운 질감: 자극 패턴으로 다양한 질감 인식

6.3 센서 기술

6.3.1 힘/압력 센서

  • FSR (Force Sensing Resistor): 저비용, 유연, 제한된 정확도
  • 변형률 게이지: 높은 정확도, 온도 민감
  • 용량성 센서: 높은 감도, 노이즈 민감
  • 압전 센서: 동적 힘 측정, 정적 힘 불가

6.3.2 미끄러짐 센서

  • 진동 감지: 미끄러짐 시 고주파 진동 검출
  • 광학 센서: 물체 표면 이동 추적
  • 변형률 패턴: 미끄러짐 전 전단력 변화 감지

6.3.3 질감 센서

  • 마이크로마찰: 표면 스캔 시 진동 패턴 분석
  • 다점 촉각 어레이: 고해상도 접촉 이미지

6.4 신호 처리 및 매핑

6.4.1 센서-피드백 매핑

{
  "feedbackMapping": {
    "gripForce": {
      "sensorType": "FSR",
      "range": [0, 50],
      "unit": "N",
      "feedbackMode": "vibration",
      "mapping": "linear",
      "parameters": {
        "minFrequency": 50,
        "maxFrequency": 250,
        "threshold": 2
      }
    },
    "fingerContact": {
      "sensorType": "capacitive_array",
      "feedbackMode": "neural_stimulation",
      "mapping": {
        "thumb": "median_nerve_branch_1",
        "index": "median_nerve_branch_2",
        "middle": "median_nerve_branch_3"
      }
    }
  }
}

6.4.2 폐루프 제어

감각 피드백은 폐루프 파지 제어를 가능하게 합니다:

while holding_object:
    current_force = read_grip_sensor()
    slip_detected = read_slip_sensor()
    
    if slip_detected:
        target_force = current_force * 1.2
    elif current_force > max_safe_force:
        target_force = max_safe_force
    
    motor_command = pid_controller(target_force, current_force)
    apply_feedback_to_user(current_force)
    send_motor_command(motor_command)

6.5 체화(Embodiment)

6.5.1 체화의 정의

체화는 의수가 신체의 일부로 인식되는 현상입니다. 높은 체화는 다음과 관련됩니다:

  • 의수에 대한 주인 의식
  • 의수 위치/상태의 자동적 인식
  • 의수가 위협받을 때 보호 반응
  • 의수 사용의 자동화, 낮은 인지 부하

6.5.2 체화 촉진 요소

  • 감각 피드백: 특히 신경 자극 피드백
  • 운동-감각 일치: 움직임과 피드백의 동시성
  • 외형 사실성: 생체모방 외관
  • 훈련 및 사용: 장기간 사용으로 체화 증가

6.5.3 체화 측정

  • 주관적 설문: 체화 질문지 점수
  • 고무손 착각: 의수에 대한 소유 착각 테스트
  • 위협 반응: 의수 위협 시 피부전도도 변화

장 요약

  1. 감각 피드백 중요성: 파지력 조절, 미끄러짐 감지, 체화 증가, 환지통 감소에 기여합니다.
  2. 피드백 양식: 진동촉각(비침습적, 보편적), 전기촉각, 기계촉각, 신경자극(가장 자연스러움) 등이 있습니다.
  3. 센서 기술: 힘, 미끄러짐, 질감 센서가 의수 손에 통합되어 환경 정보를 수집합니다.
  4. 폐루프 제어: 감각 피드백과 자동 파지력 조절로 신뢰할 수 있는 물체 조작을 제공합니다.
  5. 체화: 감각 피드백, 운동-감각 일치, 훈련이 의수의 신체 일부로서의 인식을 촉진합니다.
