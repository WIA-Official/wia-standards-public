제4장: 제어 알고리즘 및 패턴 인식

弘益人間 (홍익인간)

지능적 알고리즘이 전기 신호에서 인간의 의도를 해독하여 각 개인의 고유한 생리와 필요에 적응하는 자연스러운 의수 제어를 가능하게 합니다.

4.1 제어 알고리즘 분류

의수 제어 알고리즘은 EMG 신호를 의수 장치 명령으로 변환합니다. WIA 표준은 단순성, 기능성, 견고성 간의 다양한 절충을 제공하는 여러 제어 패러다임을 지원합니다.

4.1.1 직접 제어(Direct Control)

원리: 하나의 근육이 하나의 의수 기능을 제어합니다. EMG 진폭이 임계값을 초과하면 기능이 활성화됩니다.

if EMG_굴근 > 임계값:
    손.닫기()
elif EMG_신근 > 임계값:
    손.열기()
else:
    손.정지()

장점: 매우 신뢰성 있음, 직관적, 빠른 응답(<50ms), 최소한의 훈련 필요

한계: 2-4개 기능으로 제한, 동시 다관절 제어 불가

4.1.2 비례 제어(Proportional Control)

원리: EMG 진폭이 이진 on/off가 아닌 연속적으로 속도나 힘을 제어합니다.

속도 = map(EMG_진폭, 임계값, 최대값, 최소속도, 최대속도)
손.속도설정(속도)

장점: 자연스러운 힘/속도 조절, 섬세한 조작 가능, 직관적

도전: 일관된 근육 수축 강도 필요, EMG 드리프트에 민감

4.1.3 패턴 인식 제어

원리: 기계 학습 분류기가 다채널 EMG 패턴을 분석하여 훈련된 제스처 집합에서 사용자의 의도된 움직임을 식별합니다.

장점: 제한된 EMG 부위에서 많은 기능(8-20개 이상) 가능, 직관적 매핑, 동시 다자유도 제어 가능

도전: 교정/훈련 필요, 전극 이동 시 성능 저하, 계산 복잡성

4.2 패턴 인식 파이프라인

패턴 인식 시스템은 표준화된 처리 파이프라인을 따릅니다:

1. 신호 획득 → 2. 전처리 → 3. 특징 추출 → 4. 분류 → 5. 후처리 → 6. 명령 출력

4.2.1 훈련 단계

  1. 데이터 수집: 사용자가 각 원하는 제스처를 반복 수행(일반적으로 3-5초씩 3-5회)
  2. 레이블 할당: 각 기록된 세그먼트에 해당 제스처 클래스 레이블 지정
  3. 특징 추출: 일관된 윈도우와 파라미터로 모든 훈련 데이터에서 특징 계산
  4. 분류기 훈련: 기계 학습 모델이 특징 패턴과 제스처 클래스 간 관계 학습
  5. 검증: 테스트 데이터로 실제 정확도 평가

4.2.2 온라인 분류 단계

  • 연속 샘플링: EMG 신호가 연속 샘플링(채널당 일반적으로 1000Hz)
  • 슬라이딩 윈도우: 중첩 윈도우에서 특징 추출(예: 50ms마다 업데이트)
  • 분류: 각 윈도우의 특징을 훈련된 분류기에 입력
  • 후처리: 다수결 투표로 오분류 감소
  • 명령 생성: 식별된 제스처가 의수 명령으로 매핑

4.3 특징 추출

4.3.1 시간 영역 특징

+------------------+--------------------+----------------------------------+
| 특징             | 공식               | 특성                             |
+------------------+--------------------+----------------------------------+
| MAV              | Σ|xi|/N            | 근육 수축 강도, 잡음에 강건      |
| RMS              | √(Σxi²/N)          | 신호 전력, EMG 진폭 표준         |
| WL               | Σ|xi+1-xi|         | 신호 복잡성, 근피로 민감         |
| ZC               | Σsgn(-xi·xi+1)     | 주파수 정보, 빠른 계산           |
| SSC              | Σf[(xi-xi-1)·      | 주파수 정보, 3샘플 비교          |
|                  |    (xi-xi+1)]      |                                  |
+------------------+--------------------+----------------------------------+

4.3.2 주파수 영역 특징

  • 평균 주파수(MNF): 스펙트럼 무게중심. 근피로 시 감소
  • 중앙 주파수(MDF): 스펙트럼을 이등분하는 주파수
  • 스펙트럼 밴드 전력: 특정 주파수 범위의 전력

4.4 분류 알고리즘

4.4.1 선형 판별 분석(LDA)

LDA는 의수 제어의 황금 표준으로, 클래스 분리를 최대화하는 선형 투영을 찾습니다:

  • 빠른 훈련 및 분류
  • 견고하고 안정적인 성능
  • 적은 훈련 데이터로도 효과적
  • 실시간 구현 용이

4.4.2 서포트 벡터 머신(SVM)

커널 함수로 비선형 결정 경계를 찾는 강력한 분류기:

  • 비선형 패턴에 LDA보다 우수
  • 과적합에 강건
  • 훈련 시간이 더 김
  • 커널 선택 및 파라미터 튜닝 필요

4.4.3 인공 신경망

딥러닝 모델(CNN, LSTM)은 최근 의수 제어에 적용되고 있습니다:

  • 자동 특징 학습
  • 복잡한 시공간 패턴 캡처
  • 더 많은 훈련 데이터 필요
  • 계산 요구량 증가

4.5 실시간 구현 고려사항

4.5.1 지연 시간 요건

사용자가 지각할 수 없는 제어를 위해 총 시스템 지연은 300ms 이하여야 합니다:

지연 구성요소:
- EMG 획득: 10-20ms
- 특징 추출 윈도우: 100-200ms
- 분류: 1-10ms
- 통신: 5-20ms
- 기계적 응답: 50-100ms
총합: 166-350ms

4.5.2 적응 및 재교정

  • 점진적 학습: 새 데이터로 분류기 지속 업데이트
  • 전이 학습: 이전 사용자 데이터로 사전 훈련
  • 도메인 적응: 세션 간 변화 보상

장 요약

  1. 제어 패러다임: 직접 제어(단순, 신뢰성), 비례 제어(연속 조절), 패턴 인식(다기능)이 다양한 사용 사례에 적용됩니다.
  2. 패턴 인식: 다채널 EMG에서 특징을 추출하고 기계 학습으로 사용자 의도를 분류하여 직관적인 다자유도 제어를 제공합니다.
  3. 특징 추출: 시간 영역(MAV, RMS, WL) 및 주파수 영역 특징이 EMG 신호에서 의미 있는 정보를 추출합니다.
  4. 분류기: LDA가 황금 표준이며, SVM과 신경망이 복잡한 패턴에 더 나은 성능을 제공할 수 있습니다.
  5. 실시간 요건: 300ms 이하의 지연 시간과 적응적 재교정이 실용적인 의수 제어에 필수적입니다.
