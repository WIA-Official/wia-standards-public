Advanced multi-degree-of-freedom prosthetics restore complex hand and arm functionality, enabling users to perform tasks requiring precision, dexterity, and natural movement coordination.

5.1 Human Hand and Arm Biomechanics

The human hand is an engineering marvel with over 20 degrees of freedom enabling thousands of distinct postures and movements. Understanding this complexity guides the design of functional prosthetic systems.

5.1.1 Joint Architecture

Shoulder Complex (3 DOF):

  • Glenohumeral joint: Ball-and-socket joint providing flexion/extension, abduction/adduction, and internal/external rotation
  • Scapulothoracic articulation: Not a true joint but enables scapula movement contributing to shoulder range
  • Range of motion: ~180° flexion, ~180° abduction, ~90° rotation

Elbow (2 DOF):

  • Humeroulnar joint: Hinge joint providing ~150° flexion from full extension
  • Radioulnar joints: Enable forearm pronation/supination (palm down/up), ~170° total range
  • Critical for positioning hand in space and orienting grasp

Wrist (2 DOF):

  • Radiocarpal and midcarpal joints: Enable flexion/extension (~70°/~80°) and radial/ulnar deviation (~20°/~30°)
  • Essential for fine positioning and adjusting grip approach angle
  • Works synergistically with finger movements for stable grasp

Hand (14+ DOF):

  • Thumb: 4 DOF (carpometacarpal, metacarpophalangeal, interphalangeal joints) enabling opposition and precision grips
  • Fingers (4×3 DOF): Each finger has 3 joints (MCP, PIP, DIP) with 2-3 DOF total
  • Intrinsic hand muscles coordinate complex patterns like arches and cupping

5.1.2 Functional Grasp Taxonomy

While the hand has 20+ DOF, daily activities use a finite set of grasp patterns. Research by Cutkosky and others identified fundamental grasps:

Power Grasps (Force-oriented):

  • Cylindrical: Fingers wrap around cylindrical objects (jar, hammer handle). Accounts for ~28% of daily grasps.
  • Spherical: Hand conforms to round objects (ball, doorknob). ~8% of grasps.
  • Hook: Fingers flexed while thumb inactive (carrying bag handles). ~12% of grasps.

Precision Grasps (Position-oriented):

  • Tripod/Tip Pinch: Thumb opposes index and middle fingertips (picking up small objects, pen grip). ~24% of grasps.
  • Lateral/Key Pinch: Thumb presses against side of index finger (holding keys, cards). ~16% of grasps.
  • Palmar Pinch: Thumb opposes multiple fingertips (holding plate, large objects). ~6% of grasps.

Specialized Patterns (~6%):

  • Extension (pointing, pushing buttons)
  • Platform (supporting flat objects on extended fingers)
  • Various tool-specific grips

5.2 Prosthetic DOF Design Principles

Replicating all human DOF is impractical due to mechanical complexity, weight, cost, and control limitations. Effective prosthetic design prioritizes the most functionally important DOF.

5.2.1 Essential vs. Optional DOF

Minimum Functional Set (3-5 DOF):

  1. Hand Open/Close: Absolutely essential for any functional grasp
  2. Wrist Rotation (Pronation/Supination): Critical for orienting hand; studies show this is the second-most-valued function by users
  3. Wrist Flexion/Extension: Important for grasping objects at angles and typing
  4. Thumb Positioning: Enables transition between power and precision grips
  5. Elbow Flexion: For transhumeral amputees, essential for hand positioning

Enhanced Functionality (6-10 DOF):

  • Individual finger control (especially index finger independence for pointing, typing)
  • Multiple grasp patterns (cylindrical vs. lateral vs. tripod)
  • Variable grip force/speed
  • Elbow pronation/supination (for transhumeral users)

Advanced Capability (11+ DOF):

  • Individual control of all fingers
  • Proportional multi-joint coordination
  • Dynamic grasp adaptation
  • Shoulder functionality (for shoulder disarticulation)

5.2.2 Mechanical Implementation Strategies

Underactuated Mechanisms: Fewer motors than DOF, using passive compliance and mechanical coupling. Example: Bebionic hand has 5 motors but 14 DOF through differential mechanisms that distribute force among fingers based on object shape.

Advantages: Lighter weight, simpler control, adaptive grasping (hand automatically conforms to object shape), more reliable (fewer motors to fail).

Limitations: Less precise control of individual joints, limited ability for non-grasp postures (pointing, gestures).

Fully Actuated Systems: Individual motors for each DOF providing complete independent control. Example: DEKA Arm has 10 powered DOF with dedicated actuators.

Advantages: Maximum dexterity, precise positioning, enables complex coordinated movements, better for specialized tasks.

Limitations: Heavier (typically 1-2 kg vs. 0.4-0.8 kg for underactuated), more expensive ($50,000-$125,000 vs. $15,000-$40,000), higher power consumption, more potential failure points.

5.3 Control Allocation Strategies

With more prosthetic DOF than available control signals, intelligent allocation determines which DOF are actively controlled at each moment.

5.3.1 Sequential Control

Mode Switching: User switches between control modes (e.g., hand mode, wrist mode, elbow mode) using specific EMG patterns or external buttons. Only one mode is active at a time.

Implementation:

Mode 1 (Hand): Flexor controls grip close, Extensor controls grip open
Mode 2 (Wrist): Flexor controls wrist flexion, Extensor controls wrist extension
Mode 3 (Rotation): Flexor controls pronation, Extensor controls supination
Switch: Co-contraction or hold signal to cycle modes

Pros: Simple, works with 2 EMG sites, users understand clearly what mode they're in.

Cons: Slow (2-5 seconds per mode switch + action), high cognitive load, frustrating for complex tasks requiring frequent switching.

5.3.2 Hierarchical Control

Automatic Mode Selection: System intelligently selects appropriate DOF based on context, user history, or task recognition.

Example: When hand is empty and moving rapidly → prioritize wrist/elbow positioning. When hand contacts object → switch to grip control. Machine learning algorithms can learn user preferences and common task sequences.

Performance: Studies show 30-40% reduction in time-to-complete tasks vs. manual mode switching, with users reporting less frustration. However, requires sophisticated sensing and algorithms.

5.3.3 Simultaneous Proportional Control

Parallel DOF Activation: Pattern recognition or regression identifies multiple intended movements simultaneously, controlling several DOF in parallel.

Example: EMG pattern indicates "close hand while rotating wrist" → both grip motors and wrist motor activate simultaneously with appropriate velocities.

Requirements:

  • Multi-output regression or multi-label classification
  • Sufficient EMG channels (typically 6-8 minimum for reliable simultaneous control)
  • Training on combined movements (not just individual gestures)

Performance: Significant improvement in task completion speed (40-60% faster than sequential) and naturalness. Users report feeling more "connected" to the prosthetic.

5.4 Grasp Synthesis and Adaptation

Beyond simple position control, intelligent prosthetics can synthesize appropriate grasps and adapt to object properties.

5.4.1 Preshaping

Principle: When approaching an object, hand assumes an appropriate grasp posture before contact based on visual or contextual information.

Implementation:

  • User-Selected: User selects desired grasp pattern (cylindrical, pinch, lateral) from menu, hand preshapes accordingly
  • Vision-Based: Camera identifies object type/size, automatically selects appropriate grasp (emerging research)
  • Sensor-Based: Proximity sensors detect object size/distance, pre-position fingers

Benefits: Faster grasping (hand ready when contact occurs), more reliable capture (better alignment), feels more natural.

5.4.2 Adaptive Grasping

Compliant Mechanisms: Passive compliance allows fingers to conform to object shape during grasp closure. Combined with underactuation, enables stable grasps of varied shapes with single "close" command.

Active Adaptation: Force sensors detect object contact; control system adjusts finger positions to optimize contact points and distribute force. More sophisticated than pure passive compliance but requires more sensing and computation.

Slip Detection and Prevention: Tactile sensors detect object slippage; grip force automatically increases until stable. Studies show slip prevention reduces drop rates by 60-80% compared to fixed-force grips.

5.5 Example Multi-DOF Systems

Understanding specific prosthetic platforms illustrates design trade-offs and capabilities.

5.5.1 i-limb Quantum (Össur)

Specifications:

  • 5 individually powered digits
  • 36 selectable grip patterns
  • Proportional speed/force control
  • Powered thumb rotation
  • Weight: 615g (size medium)

Control: Two-site myoelectric or pattern recognition (via i-limb Biosim app). Users can customize grip selection and switching methods.

Notable Features: Vari-Grip allows mid-grasp adjustments. AutoGrasp detects object contact and applies appropriate force. BioSim smartphone app for grip selection and configuration.

5.5.2 bebionic Hand (Ottobock)

Specifications:

  • 5 motors controlling 14 DOF through mechanical coupling
  • 14 grip patterns
  • Proportional control
  • Powered thumb rotation (4 positions)
  • Weight: 539-632g depending on size

Control: Two-site myoelectric standard, pattern recognition optional. Triple-pulse switching between grips.

Notable Features: High grip force (140N), very natural cosmetic appearance, individual finger position sensors for precise control.

5.5.3 DEKA Arm (Mobius Bionics)

Specifications:

  • 10 powered DOF (shoulder, humeral rotation, elbow, wrist, hand)
  • 6 grip patterns
  • Simultaneous coordinated control
  • Multiple configurations (shoulder, transhumeral, transradial)
  • Weight: 1.6-2.5 kg depending on configuration

Control: Requires 6+ EMG sites (ideal for TMR patients). Foot controls optional for shoulder configuration. Pattern recognition with simultaneous control of multiple joints.

Notable Features: Only FDA-approved full-arm prosthetic system, exceptional dexterity (can handle eggs, use keys, etc.), modular design, extensive sensing feedback to user.

5.6 Performance Metrics and Assessment

Standardized testing enables objective comparison of multi-DOF prosthetic systems and control algorithms.

5.6.1 Functional Tests

Southampton Hand Assessment Procedure (SHAP): 26 tasks assessing various grips and manipulations. Tasks include abstract object manipulation and activities of daily living (ADL). Generates an overall score (0-100) and subscores for different grasp types. Widely used clinically; normative data available.

Assessment of Capacity for Myoelectric Control (ACMC): Observational assessment rating how effectively users employ their myoelectric prosthetic during daily activities. Assesses 22 items like gripping, releasing, timing, coordination. Used to track learning and identify training needs.

Clothespin Relocation Test: Simple timed test: move clothespins from horizontal bar to vertical bar. Measures speed and precision. Good for research comparisons, less comprehensive than SHAP.

Box and Blocks Test: Transfer blocks from one box to another in 60 seconds. Measures gross manual dexterity. Simple, quick, widely used across prosthetic research.

5.6.2 User Experience Metrics

System Usability Scale (SUS): 10-item questionnaire assessing overall usability. Score 0-100; scores >68 considered above average. Quick administration (2-3 minutes), high reliability.

Trinity Amputation and Prosthesis Experience Scales (TAPES): Comprehensive assessment of psychosocial adjustment, activity restriction, and satisfaction with prosthetic. Multiple subscales provide detailed profile.

Embodiment Questionnaire: Assesses sense of ownership (prosthetic feels like part of body), agency (sense of control), and location (knowing prosthetic position without looking). Correlates with usage rates and satisfaction.

5.7 WIA Standard Multi-DOF Support

The WIA Prosthetic Control Standard provides comprehensive frameworks for multi-DOF device control.

5.7.1 Device Capability Description

Standardized format for prosthetic devices to report their capabilities:

{
  "device_model": "advanced_hand_v2",
  "degrees_of_freedom": [
    {
      "id": "grip_open_close",
      "type": "linear",
      "range": [0, 100],
      "unit": "percent",
      "speed_range": [0, 200],
      "force_range": [0, 140],
      "priority": 1
    },
    {
      "id": "wrist_rotation",
      "type": "rotational",
      "range": [-90, 90],
      "unit": "degrees",
      "positions": ["palm_up", "neutral", "palm_down"],
      "priority": 2
    },
    ...
  ],
  "grasp_patterns": [
    "cylindrical", "tripod", "lateral", "spherical",
    "hook", "precision", "power", "point"
  ],
  "simultaneous_control": true,
  "max_simultaneous_dof": 3
}

5.7.2 Control Command Structure

Standardized commands supporting various control modalities:

{
  "control_mode": "simultaneous_proportional",
  "commands": [
    {
      "dof_id": "grip_open_close",
      "target_position": 75,
      "speed": 50,
      "force_limit": 80
    },
    {
      "dof_id": "wrist_rotation",
      "target_position": -30,
      "speed": 100
    }
  ],
  "priority": "normal",
  "timeout_ms": 5000
}

This structure enables control systems to command any WIA-compliant device regardless of manufacturer, promoting interoperability and user choice.
