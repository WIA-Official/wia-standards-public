Chapter 4: Contamination Taxonomy & Measurement Protocols

A clarity index is only as credible as the measurements feeding it. This chapter defines the standard's eight-type contamination taxonomy — the shared vocabulary for what is degrading a sensor — and then specifies reproducible protocols for how the three PCI axes are measured, anchored to ISO 12233 for the contrast/MTF axis. Together these make PCI a quantity that two labs, or two vendors, can derive consistently for the same physical condition.

4.1 The Contaminant Taxonomy

The standard enumerates eight contaminant types. They name the cause of degradation, reported per sensor in the contaminants[] array, each with a coverage fraction in [0.0, 1.0] and an optional note.

enum ContaminantType {
  RAIN_FILM        = "rain_film"          // water film
  MUD_DUST         = "mud_dust"           // mud / dust
  SALT_SPRAY       = "salt_spray"         // salt haze
  INSECT_STRIKE    = "insect_strike"      // insect impact
  FROST_ICE        = "frost_ice"          // frost / ice
  CONDENSATION     = "condensation"       // internal fogging
  SUN_GLARE        = "sun_glare"          // backlight / saturation
  SCRATCH_ABRASION = "scratch_abrasion"   // scratches / wear
}

The taxonomy is extensible. An unclassified contaminant is reported as "other" with a descriptive note. Multiple types may coexist on one sensor, and their coverage values may sum to more than the measured occlusion because contaminants overlap; individual coverage values nonetheless remain bounded in [0,1].

4.2 The Reference Baseline

Every axis is defined relative to a clean reference for the same sensor on the same platform. Distance degradation is normalized against a reference detection range; MTF reduction against a reference MTF50. The reference must be captured with the sensor pristine, at known geometry, under controlled illumination, and recorded as part of the conformance dossier. Without a fixed baseline the axes are not comparable, so establishing and version-controlling the reference is a prerequisite for any reproducible PCI.

4.3 Measuring Occlusion (OCC)

Occlusion is the fraction of the effective field of view that is blocked. The reference protocol images or scans a known uniform target (a flat calibration board for cameras and thermal; a planar return surface for LiDAR/ultrasonic; a calibrated corner reflector field for radar) and compares the live field against the clean baseline:

  • Cameras / thermal: segment regions whose local signal falls below a contamination threshold relative to baseline; OCC = blocked-area / total-area of the working field.
  • LiDAR: compare returned points per angular bin against baseline; bins that return only near-field ghost points or no valid return are counted as occluded.
  • Radar: compare detections of the reference reflector set; missing reflectors indicate blocked sectors.
  • Ultrasonic: compare echo amplitude from a fixed planar target; loss of valid echo marks the face occluded.

4.4 Measuring Distance Degradation (DIST)

Distance degradation captures the shrinking of the usable sensing envelope: DIST = (reference range − current effective range) / reference range. "Effective range" is the greatest distance at which the sensor still meets a defined detection criterion — a target of fixed size and contrast detected with a fixed probability. The protocol places a standardized target at progressively greater distances (or uses a calibrated range rig) under the contamination condition and finds the breakdown distance, then normalizes against the clean-baseline breakdown distance. For radar this directly captures RF attenuation; for thermal it captures the collapse of the night-detection envelope.

4.5 Measuring MTF / Contrast Reduction (MTF) per ISO 12233

The contrast axis is anchored to the Modulation Transfer Function, the standard optical measure of how well a system preserves contrast across spatial frequencies. MTF50 — the spatial frequency at which contrast falls to 50% of its low-frequency value — is the headline figure; a clean lens has a high MTF50, and contamination lowers it.

Measurement follows ISO 12233, the international method for resolution and spatial frequency response (SFR). The slanted-edge technique images a high-contrast slanted edge on a test chart; the edge profile is differentiated to a line-spread function and Fourier-transformed to recover the SFR/MTF curve, from which MTF50 is read. The axis value is:

MTF = 1 - ( MTF50_current / MTF50_reference )

ISO 12233 slanted-edge SFR pipeline
====================================
  capture slanted-edge chart  ->  extract edge spread function (ESF)
  ESF  ->  differentiate  ->  line spread function (LSF)
  LSF  ->  Fourier transform  ->  spatial frequency response (SFR/MTF)
  read MTF50  ->  normalize against pristine reference

For modalities without optical resolution (radar, ultrasonic) the MTF axis is undefined and its class weight is near zero or zero (Chapter 3), so the protocol does not require an MTF measurement for them.

4.6 Reproducibility Requirements

To qualify as a conformant measurement, a protocol run must record: the sensor class and identifier; the clean reference values; the test target specification; illumination and environmental conditions; the contaminant type(s) and applied coverage; and the resulting axis values with the weights used. Two independent runs under matched conditions should produce PCI values within a small tolerance. Test contaminants should be applied in repeatable ways — calibrated water films, graded dust loadings, controlled frost chambers, standardized abrasion cycles — so that the relationship between physical contamination and PCI is itself reproducible.

4.7 Mapping Measurements Back to Contaminants

The taxonomy and the axes are linked but distinct. The axes quantify how much perception is lost; the taxonomy records why. A single rain_film event may register as moderate MTF loss plus mild distance degradation; a mud_dust event as strong occlusion plus distance loss. Reporting both lets a consumer not only judge current clarity (via PCI/state) but also reason about likely remedies and trends — for instance, distinguishing a transient rain film that will clear on its own from a scratch_abrasion condition that is permanent and only worsens.

Chapter Summary

Eight contaminant types (rain_film, mud_dust, salt_spray, insect_strike, frost_ice, condensation, sun_glare, scratch_abrasion, plus "other") form the shared vocabulary for the cause of degradation, each reported with a coverage fraction. The three PCI axes are measured against a fixed clean baseline: occlusion by comparing blocked field against baseline, distance degradation by finding the effective-range breakdown point, and MTF reduction via the ISO 12233 slanted-edge SFR method reading MTF50. Recording reference values, conditions, contaminant coverage, and axis results makes PCI reproducible and audit-grade.
