"딥러닝은 수백만 개의 의료 영상에서 인간이 볼 수 없는 패턴을 발견합니다. CNN은 방사선과 전문의의 눈이 되어, 더 정확하고 더 빠른 진단을 가능하게 합니다."

4.1 의료 영상 분석의 딥러닝

딥러닝(Deep Learning)은 다층 신경망을 사용하여 복잡한 패턴을 학습하는 머신러닝의 하위 분야입니다. 의료 영상에서 딥러닝은 혁명을 일으키고 있습니다.

🎯 주요 응용 분야

  • 분류(Classification) — 영상을 범주로 분류 (정상/비정상)
  • 객체 감지(Object Detection) — 병변 위치 찾기
  • 세분화(Segmentation) — 픽셀 단위로 영역 분리
  • 등록(Registration) — 영상 정렬 및 비교

📊 의료 영상 유형

영상 유형 특징 주요 질환
X-ray 2D, 흑백, 빠름 골절, 폐렴, 결핵
CT 3D, 고해상도 암, 뇌출혈, 폐색전
MRI 3D, 연조직 우수 뇌종양, 척수 질환
초음파 실시간, 안전 태아 이상, 갑상선
병리 슬라이드 현미경, 고배율 암 진단, 조직 분석

4.2 합성곱 신경망 (CNN)

🧠 CNN 아키텍처

CNN은 영상 인식에 특화된 신경망입니다:

입력 영상 (예: 512×512 픽셀 흉부 X-ray)
    ↓
합성곱 층 1 (Conv1) — 가장자리, 질감 감지
    ↓
풀링 층 1 (Pool1) — 크기 축소 (256×256)
    ↓
합성곱 층 2 (Conv2) — 형태 패턴 감지
    ↓
풀링 층 2 (Pool2) — 크기 축소 (128×128)
    ↓
합성곱 층 3-5 (Conv3-5) — 복잡한 특징
    ↓
완전 연결 층 (FC) — 분류 결정
    ↓
출력: 폐렴 확률 87%

🔍 CNN 핵심 구성 요소

구성 요소 기능 예시
합성곱 층 특징 추출 (가장자리, 질감) 3×3 필터로 종양 윤곽 감지
활성화 함수 비선형성 추가 (ReLU) 음수 값을 0으로 변환
풀링 층 크기 축소, 계산량 감소 Max Pooling 2×2
완전 연결 층 최종 분류 결정 정상/폐렴 확률 출력

🏗️ 유명한 CNN 아키텍처

  • ResNet — 잔차 연결로 매우 깊은 네트워크 (50-152층)
  • Inception — 다중 스케일 합성곱 병렬 처리
  • DenseNet — 모든 층 간 밀집 연결
  • U-Net — 의료 영상 세분화 전용 (인코더-디코더)
  • EfficientNet — 효율적인 스케일링

4.3 의료 영상 분류

🩺 사례: 흉부 X-ray 폐렴 진단

목표: X-ray 영상에서 폐렴 감지

데이터셋:

  • 정상: 10,000장
  • 폐렴: 10,000장
  • 해상도: 512×512 픽셀

모델 아키텍처: ResNet-50 (사전 훈련된 ImageNet 가중치)

💻 구현 예시 (PyTorch)

import torch
import torchvision.models as models
from torchvision import transforms

# 1. 사전 훈련된 ResNet-50 로드
model = models.resnet50(pretrained=True)
model.fc = torch.nn.Linear(2048, 2)  # 2클래스 (정상/폐렴)

# 2. 데이터 전처리
transform = transforms.Compose([
    transforms.Resize((224, 224)),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406],
                         std=[0.229, 0.224, 0.225])
])

# 3. 훈련
criterion = torch.nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)

for epoch in range(10):
    for images, labels in train_loader:
        optimizer.zero_grad()
        outputs = model(images)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

# 4. 예측
with torch.no_grad():
    prediction = model(test_image)
    pneumonia_prob = torch.softmax(prediction, dim=1)[0][1]
    print(f'폐렴 확률: {pneumonia_prob:.2%}')

4.5 의료 영상 세분화

🔬 세분화란?

픽셀 단위로 영역을 분류하여 정확한 경계를 그립니다.

🏗️ U-Net 아키텍처

의료 영상 세분화에 가장 널리 사용되는 모델:

         입력 영상 (512×512)
              ↓
      ┌───────────────────┐
      │   인코더 (축소)   │
      │  Conv → Pool     │
      │  512 → 256 → 128 │
      └─────────┬─────────┘
                ↓
          병목 (64×64)
                ↓
      ┌─────────┴─────────┐
      │   디코더 (확대)   │
      │  UpConv + Skip   │
      │  128 → 256 → 512 │
      └───────────────────┘
              ↓
      출력: 세분화 맵 (512×512)
      (각 픽셀이 종양/정상 분류됨)

🩺 의료 응용

  • 종양 세분화 — 정확한 크기 및 경계 측정
  • 장기 세분화 — 간, 폐, 심장 부피 계산
  • 병변 경계 — 수술 계획을 위한 정확한 매핑

📊 세분화 성능 지표

IoU 교집합 over 합집합 ≥ 0.75
Hausdorff 거리 경계 정확도 < 5 픽셀

4.6 전이 학습 및 데이터 증강

🔄 전이 학습

사전 훈련된 모델을 의료 데이터에 맞게 조정합니다.

장점:
• 필요한 데이터 크기 감소 (100,000 → 1,000)
• 훈련 시간 단축 (수일 → 수시간)
• 일반화 성능 향상

🎨 데이터 증강

기존 데이터를 변환하여 인위적으로 확장합니다:

  • 회전 — ±15도
  • 이동 — ±10%
  • 확대/축소 — 90-110%
  • 좌우 반전 — 50% 확률
  • 밝기 조정 — ±20%
  • 대비 조정 — ±20%
  • 노이즈 추가 — 가우시안 노이즈
