제5장: 의료 기록을 위한 자연어 처리 (NLP)

"의료 기록의 80%는 비정형 텍스트입니다. NLP는 이 방대한 지식 보고를 구조화된 통찰로 변환하여, 숨겨진 패턴과 중요한 정보를 발견합니다."

5.1 의료 NLP란?

의료 자연어 처리(Medical NLP)는 의사의 노트, 진료 기록, 검사 보고서 등 의료 텍스트를 이해하고 분석하는 AI 기술입니다.

🎯 핵심 과제

  • 정보 추출 — 증상, 진단, 약물, 검사 결과 추출
  • 개체명 인식(NER) — 질병, 약물, 신체 부위 식별
  • 관계 추출 — "환자가 고혈압으로 리시노프릴 복용" 관계 파악
  • 의료 코딩 — ICD-10, CPT 코드 자동 할당
  • 문서 분류 — 진료 기록 유형 분류

📊 의료 텍스트의 특성

특성 설명 예시
전문 용어 의학 전문어, 라틴어 dyspnea, tachycardia
약어 극도로 축약된 표현 SOB, HTN, DM
부정 표현 증상 없음 표현 "흉통 없음", "발열 부인"
시간 표현 과거/현재/미래 구분 "3일 전 발병", "현재 진행 중"
불확실성 추정, 가능성 표현 "의심됨", "가능성 있음"

5.2 NLP 핵심 기술

🔤 1. 토큰화 및 전처리

텍스트를 분석 가능한 단위로 분할합니다:

예시:
원문: "Pt c/o SOB and chest pain x3 days."
토큰화: ["Pt", "c/o", "SOB", "and", "chest", "pain", "x3", "days"]
정규화: ["Patient", "complains of", "Shortness of breath", "and", "chest", "pain", "for 3", "days"]

🏷️ 2. 개체명 인식 (NER)

텍스트에서 의료 개체를 식별하고 분류합니다:

개체 유형 예시
질병/증상 폐렴, 발열, 기침
약물 아스피린, 메트포르민
용량 500mg, 하루 2회
검사 혈압, 혈당, X-ray
신체 부위 심장, 폐, 간

🔗 3. 관계 추출

개체 간의 관계를 파악합니다:

  • 약물-질병 — "환자가 고혈압으로 리시노프릴 복용"
  • 증상-질병 — "발열과 기침으로 폐렴 진단"
  • 검사-결과 — "혈압 측정 결과 150/95"

❌ 4. 부정 감지

증상이나 질병의 부재를 올바르게 인식합니다:

예시:
"환자는 흉통을 부인합니다." → 흉통: 없음 ✓
"발열이 없습니다." → 발열: 없음 ✓
"당뇨병 병력이 있습니다." → 당뇨병: 있음 ✓

⏰ 5. 시간 정보 추출

증상 발생 시점 및 지속 기간을 파악합니다:

  • 과거: "3일 전부터 기침" → 발병일: 2025-01-15
  • 현재: "현재 발열 지속 중" → 상태: 진행 중
  • 미래: "2주 후 재진 예정" → 예약일: 2025-02-01

5.3 트랜스포머 및 BERT

🤖 트랜스포머 모델

최신 NLP의 핵심 아키텍처:

  • Attention 메커니즘 — 문맥의 중요한 부분에 집중
  • 양방향 이해 — 앞뒤 문맥 모두 고려
  • 전이 학습 — 대규모 텍스트로 사전 훈련

🏥 의료 BERT 모델

모델 훈련 데이터 특징
BioBERT PubMed, PMC 논문 생물의학 문헌 이해
ClinicalBERT MIMIC-III 진료 기록 임상 텍스트 특화
PubMedBERT PubMed abstract 과학 문헌 분석
BlueBERT PubMed + MIMIC-III 범용 의료 NLP

💻 ClinicalBERT 사용 예시

from transformers import AutoTokenizer, AutoModel
import torch

# 1. 모델 로드
tokenizer = AutoTokenizer.from_pretrained("emilyalsentzer/Bio_ClinicalBERT")
model = AutoModel.from_pretrained("emilyalsentzer/Bio_ClinicalBERT")

# 2. 텍스트 인코딩
text = "환자는 3일 전부터 발열과 기침을 호소합니다."
inputs = tokenizer(text, return_tensors="pt")

# 3. 임베딩 추출
with torch.no_grad():
    outputs = model(**inputs)
    embeddings = outputs.last_hidden_state

# 4. NER 또는 분류 작업에 사용
print(f"임베딩 형태: {embeddings.shape}")
# 출력: 임베딩 형태: torch.Size([1, 15, 768])

5.4 의료 코딩 자동화

🏷️ ICD-10 코딩

진료 기록에서 자동으로 ICD-10 질병 코드를 할당합니다:

예시:
진료 기록: "환자는 제2형 당뇨병으로 메트포르민 복용 중이며, 최근 당화혈색소 7.2%입니다."
자동 코드:
• E11.9 - 제2형 당뇨병, 합병증 없음
• Z79.84 - 경구 혈당강하제 장기 사용

💰 청구 코딩의 중요성

  • 정확한 청구 — 부적절한 코딩으로 인한 손실 방지
  • 시간 절감 — 수작업 코딩 시간 80% 감소
  • 규정 준수 — 청구 거부 및 감사 위험 감소

📊 자동 코딩 성능

지표 수작업 AI 자동화
정확도 85% 92%
처리 시간 10분/케이스 < 1분/케이스
비용 높음 낮음 (70% 절감)
