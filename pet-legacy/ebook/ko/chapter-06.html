제6장: 추억 앨범

"홍익인간: 추억 앨범은 세대를 걸쳐 위안을 주는 보물입니다."

6.1 앨범 유형

유형 초점 적합
PHOTO_BOOK 시각적 기억 사진이 풍부한 유산
STORY_BOOK 서사적 여정 풍부한 전기
TIMELINE_BOOK 시간순 사건 사건이 많은 삶
TRIBUTE_BOOK 타인의 메시지 많은 헌사

6.2 앨범 구조

추억 앨범 구조
=============

앞부분
  표지
    - 히어로 사진
    - 제목
    - 생애 날짜

  제목 페이지
    - 전체 제목
    - 부제
    - 가족 이름

  헌정 페이지
    - 개인 헌정
    - 인용 또는 추모문

본문
  전기 섹션
    - 삶의 이야기
    - 기원
    - 성격

  타임라인 섹션
    - 생애 사건
    - 이정표
    - 모험

  사진 갤러리
    - 큐레이션된 사진
    - 캡션
    - 날짜

  좋아하는 것 섹션
    - 장난감, 음식
    - 활동
    - 장소, 사람

뒷부분
  헌사 섹션
    - 선택된 메시지
    - 선물 기록

  마무리 페이지
    - 마지막 사진
    - 추모문
    - 무지개 다리 메시지

6.3 레이아웃 엔진

페이지 레이아웃

레이아웃 사진 텍스트 용도
전체 페이지 1 캡션만 히어로 샷
두 사진 2 최소 짝지은 기억
네 사진 4 없음 갤러리 페이지
텍스트 중심 1 작게 전체 페이지 전기
타임라인 여러 작게 날짜/사건 연대기

6.4 생성 프로세스

앨범 생성 파이프라인
==================

1단계: 콘텐츠 수집
  - 전기 텍스트 수집
  - 사진 선택
  - 타임라인 사건 수집
  - 승인된 헌사 포함

2단계: 레이아웃 계획
  - 페이지 수 계산
  - 콘텐츠를 페이지에 할당
  - 페이지별 레이아웃 선택
  - 시각적 흐름 균형

3단계: 렌더링
  - 테마 스타일 적용
  - 사진 크롭과 배치
  - 텍스트를 블록에 흐름
  - 페이지 번호 추가

4단계: 출력 생성
  - 인쇄용 PDF 생성
  - 미리보기 이미지 생성
  - 인쇄 비용 계산
  - 다운로드 링크 생성

5단계: 품질 확인
  - 모든 사진 배치 확인
  - 텍스트 오버플로 확인
  - 페이지 순서 검증
  - 각 펼침 미리보기

6.5 맞춤화 옵션

옵션 선택
표지 스타일 사진, 단색, 패턴
용지 유형 무광, 유광, 펄
크기 20x20cm, 20x28cm, 28x36cm
제본 소프트커버, 하드커버, 레이플랫
페이지 수 20-100 페이지
