제4장: 추모 페이지 생성

"홍익인간: 추모는 미래 세대가 우리가 나눈 사랑을 엿보는 창입니다."

4.1 추모 유형

유형 공개 범위 적합
PRIVATE 가족만 개인 추억
SHARED 초대된 손님 친구와 가족
PUBLIC 모두에게 공개 커뮤니티 헌사
LEGACY_VAULT 보관만 표시 없이 보존

4.3 테마 설정

사용 가능한 테마

테마 스타일 적합
클래식 전통적, 우아함 정식 추모
모던 깔끔, 미니멀 현대적 느낌
자연 자연적, 유기적 야외 좋아하는 반려동물
장난 밝고 쾌활 기쁨 축하
미니멀 심플, 집중 사진 중심

4.4 추모 기능

기능 설명 기본값
allowTributes 방문자 메시지 수락 예
allowCandleLighting 가상 촛불 예
allowFlowers 가상 꽃 선물 예
allowDonations 추모 기부 링크 아니오
moderateTributes 게시 전 검토 예

4.5 히어로 섹션 디자인

히어로 이미지 선택
=================

히어로 이미지 기준:
1. 고품질 (선명, 노출 좋음)
2. 반려동물이 명확히 보임
3. 얼굴과 눈이 이상적으로 보임
4. 가능하면 긍정적 감정
5. 성격을 대표

폴백 순서:
1. 사용자 지정 히어로 이미지
2. AI 선택 최상 사진
3. 가장 최근 고품질 사진
4. 기본 플레이스홀더

히어로 디스플레이:
  크기: 전체 너비, 뷰포트 높이 60%
  오버레이: 반투명 그라데이션
  텍스트: 중앙, 어두운 오버레이 위 흰색
  요소:
    - 반려동물 이름 (크게)
    - 생애 날짜
    - 추모 인용

4.6 활성화 프로세스

추모 활성화
==========

1단계: 완료 확인
  [ ] 전기 완료
  [ ] 최소 5장 사진
  [ ] 히어로 이미지 선택
  [ ] 테마 설정

2단계: 떠난 날짜 설정
  - 떠난 날짜
  - 선택: 원인 (비공개)
  - 선택: 장소

3단계: 떠남 사건 생성
  - 자동 생성 타임라인 사건
  - 무지개 다리 언급
  - 마지막 헌사

4단계: 추모 생성
  - 모든 섹션 렌더링
  - CDN에 업로드
  - 공유 가능 URL 생성

5단계: 가족 알림
  - 활성화 통지 전송
  - 추모 링크 공유
  - 검토 접근 제공

6단계: 헌사 오픈
  - 헌사 제출 활성화
  - 지정 검토자 알림
  - 선물 수락 시작

4.7 공유 옵션

=========

   맞춤형 디스플레이
