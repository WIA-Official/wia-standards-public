제5장: 헌사 및 추모 선물

"홍익인간: 공유된 슬픔은 커뮤니티의 말과 선물을 통해 공유된 치유가 됩니다."

5.1 헌사 유형

유형 설명 형식
MESSAGE 간단한 텍스트 헌사 일반 텍스트
MEMORY 특정 이야기나 추억 텍스트와 사진
POEM 시적 헌사 포맷된 시
SONG 음악 헌사 오디오 또는 가사
ARTWORK 시각 예술 헌사 이미지 업로드

5.2 헌사 제출

헌사 제출 프로세스
=================

1. 방문자 접근
   - 공개: 로그인 불필요
   - 공유: 초대 링크 통해 접근
   - 가족: 인증 필요

2. 작성자 정보
   - 이름 (필수)
   - 이메일 (선택, 알림용)
   - 반려동물/가족과의 관계

3. 콘텐츠 생성
   - 헌사 유형 선택
   - 메시지 작성
   - 미디어 첨부 (선택)
   - 가상 선물 추가

4. 제출
   - 헌사 미리보기
   - 검토를 위해 제출
   - 확인 수신

5. 검토 (활성화된 경우)
   - 대기열에 표시
   - 가족 검토
   - 승인 또는 거부
   - 알림 전송

5.4 검토 시스템

헌사 검토 워크플로
=================

상태 흐름:
  PENDING --> APPROVED --> 게시됨
         \--> REJECTED --> 숨김
         \--> FLAGGED  --> 재검토

검토 행동:
  [승인] - 추모에 게시
  [거부] - 숨김, 선택적 알림
  [플래그] - 추가 검토 표시
  [편집] - 사소한 수정 (소유자만)
  [삭제] - 영구 제거

자동 검토:
  - 욕설 필터 (설정 가능)
  - 스팸 감지
  - 링크 확인
  - 길이 제한

5.5 알림 시스템

헌사 알림
=========

가족에게:
  - 새 헌사 제출됨
  - 헌사 승인됨 (자동 검토 시)
  - 검토 조치 필요
  - 선물 수신됨

헌사 작성자에게:
  - 헌사 승인됨
  - 헌사 거부됨 (선택적 메시지)
  - 가족 응답 (선택)

알림 채널:
  - 이메일 (주요)
  - 푸시 알림 (앱 사용 시)
  - 인앱 알림
