4.4 Section Configuration

SECTION SETTINGS
================

Each section can be configured:

{
  "sectionId": "section-001",
  "sectionType": "GALLERY",
  "title": "Cherished Memories",
  "enabled": true,
  "order": 4,
  "config": {
    "maxPhotos": 20,
    "layout": "masonry",
    "showCaptions": true
  }
}

Available Section Types:
  HERO          - Main tribute image/text
  BIOGRAPHY     - Life story narrative
  TIMELINE      - Life events display
  GALLERY       - Photo/video collection
  TRIBUTES      - Messages from others
  FAVORITES     - Pet's favorite things
  FAMILY        - Family members
  CANDLES       - Virtual candle lighting
  DONATIONS     - Memorial donations (optional)

4.6 Hero Section Design

HERO IMAGE SELECTION
====================

Criteria for hero image:
1. High quality (sharp, well-exposed)
2. Pet clearly visible
3. Face and eyes ideally visible
4. Positive emotion if possible
5. Representative of personality

Fallback order:
1. User-designated hero image
2. AI-selected best photo
3. Most recent quality photo
4. Default placeholder

Hero Display:
  Size: Full width, 60% viewport height
  Overlay: Semi-transparent gradient
  Text: Centered, white on dark overlay
  Elements:
    - Pet name (large)
    - Life dates
    - Epitaph quote

4.8 Activation Process

MEMORIAL ACTIVATION
===================

Step 1: Verify Completeness
  [ ] Biography complete
  [ ] At least 5 photos
  [ ] Hero image selected
  [ ] Theme configured

Step 2: Set Passing Date
  - Date of passing
  - Optional: Cause (private)
  - Optional: Location

Step 3: Create Passing Event
  - Auto-generated timeline event
  - Rainbow bridge reference
  - Final tribute

Step 4: Generate Memorial
  - Render all sections
  - Upload to CDN
  - Create shareable URL

Step 5: Notify Family
  - Send activation notice
  - Share memorial link
  - Provide moderation access

Step 6: Open for Tributes
  - Enable tribute submission
  - Notify designated moderators
  - Begin accepting offerings
