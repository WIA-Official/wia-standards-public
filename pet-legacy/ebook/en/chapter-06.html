6.2 Book Structure

MEMORY BOOK STRUCTURE
=====================

FRONT MATTER
  Cover Page
    - Hero photo
    - Title
    - Life dates

  Title Page
    - Full title
    - Subtitle
    - Family name

  Dedication Page
    - Personal dedication
    - Quote or epitaph

CONTENT
  Biography Section
    - Life story
    - Origin
    - Personality

  Timeline Section
    - Life events
    - Milestones
    - Adventures

  Photo Gallery
    - Curated photos
    - Captions
    - Dates

  Favorites Section
    - Toys, foods
    - Activities
    - Places, people

BACK MATTER
  Tribute Section
    - Selected messages
    - Offerings record

  Closing Page
    - Final photo
    - Epitaph
    - Rainbow bridge message

Photo Selection

PHOTO SELECTION FOR BOOK
========================

1. Start with featured photos
   (AI-selected best quality)

2. Fill remaining slots by:
   - Chronological distribution
   - Event representation
   - Emotional variety

3. Maximum per page:
   - Full page: 1 photo
   - Half page: 2 photos
   - Quarter page: 4 photos

4. Quality requirements:
   - Minimum 1200px on short side
   - Print resolution: 300 DPI
   - Sharp focus required

6.4 Generation Process

BOOK GENERATION PIPELINE
========================

Step 1: Content Collection
  - Gather biography text
  - Select photos
  - Collect timeline events
  - Include approved tributes

Step 2: Layout Planning
  - Calculate page count
  - Assign content to pages
  - Select layouts per page
  - Balance visual flow

Step 3: Rendering
  - Apply theme styling
  - Place photos with crops
  - Flow text into blocks
  - Add page numbers

Step 4: Output Generation
  - Create print-ready PDF
  - Generate preview images
  - Calculate print cost
  - Create download link

Step 5: Quality Check
  - Verify all photos placed
  - Check text overflow
  - Validate page order
  - Preview each spread
