8.3 Database Schema

CORE TABLES
===========

-- Legacy profiles
CREATE TABLE legacy_profiles (
  legacy_id UUID PRIMARY KEY,
  pet_id UUID NOT NULL,
  owner_id UUID NOT NULL,
  pet_info JSONB NOT NULL,
  status VARCHAR(20) DEFAULT 'PLANNING',
  memorial_url TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  activated_at TIMESTAMPTZ
);

-- Life events
CREATE TABLE life_events (
  event_id UUID PRIMARY KEY,
  legacy_id UUID REFERENCES legacy_profiles,
  event_type VARCHAR(30) NOT NULL,
  title VARCHAR(255) NOT NULL,
  event_date DATE NOT NULL,
  significance SMALLINT DEFAULT 3
);

-- Media
CREATE TABLE media_attachments (
  media_id UUID PRIMARY KEY,
  legacy_id UUID REFERENCES legacy_profiles,
  media_type VARCHAR(20) NOT NULL,
  original_url TEXT NOT NULL,
  archive_url TEXT,
  visibility VARCHAR(20) DEFAULT 'FAMILY'
);

-- Tributes
CREATE TABLE tributes (
  tribute_id UUID PRIMARY KEY,
  legacy_id UUID REFERENCES legacy_profiles,
  author_name VARCHAR(255) NOT NULL,
  message TEXT NOT NULL,
  status VARCHAR(20) DEFAULT 'PENDING'
);

-- Archives
CREATE TABLE legacy_archives (
  archive_id UUID PRIMARY KEY,
  legacy_id UUID REFERENCES legacy_profiles,
  storage_location TEXT NOT NULL,
  checksum VARCHAR(64) NOT NULL,
  retention_years INTEGER DEFAULT 100
);

8.5 Media Processing

MEDIA PIPELINE
==============

Upload:
  1. Receive file
  2. Validate format and size
  3. Store original
  4. Queue for processing

Processing:
  1. Extract metadata
  2. Generate thumbnails
  3. Create display versions
  4. Run AI analysis
  5. Create archive copy

Storage:
  Original   -> Hot storage
  Display    -> CDN
  Thumbnail  -> CDN
  Archive    -> Cold storage

AI Analysis:
  - Pet detection
  - Emotion recognition
  - Scene classification
  - Auto-tagging

8.6 Memorial Generation

MEMORIAL BUILD PROCESS
======================

1. Fetch Data
   - Load legacy profile
   - Get biography
   - Retrieve timeline
   - Select featured photos

2. Render Sections
   For each enabled section:
   - Hero: Main tribute
   - Bio: Narrative
   - Timeline: Events
   - Gallery: Photos
   - Tributes: Messages
   - Candles: Offerings

3. Apply Theme
   - Load theme config
   - Apply colors, fonts
   - Set layout

4. Deploy
   - Generate HTML
   - Upload to CDN
   - Update DNS/routing
   - Notify family

8.7 Deployment

DOCKER COMPOSE
==============

version: '3.8'
services:
  legacy-api:
    image: wia/legacy-api:latest
    ports:
      - "8080:8080"
    environment:
      DATABASE_URL: postgres://db:5432/legacy
    depends_on:
      - db

  media-service:
    image: wia/media-service:latest
    environment:
      STORAGE_BUCKET: wia-legacy-media
      ARCHIVE_BUCKET: wia-legacy-archive
    volumes:
      - media_temp:/tmp/media

  memorial-service:
    image: wia/memorial-service:latest
    environment:
      CDN_BUCKET: wia-memorial-cdn

  archive-worker:
    image: wia/archive-worker:latest
    environment:
      GLACIER_VAULT: wia-legacy-glacier

  db:
    image: postgres:15
    volumes:
      - pgdata:/var/lib/postgresql/data
