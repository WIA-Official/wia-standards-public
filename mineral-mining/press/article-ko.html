Mineral Mining

WIA 표준은 세계인증산업협회(World Certification Industry Association)가 운영하는 오픈 표준 카탈로그의 최신 표준으로 Mineral Mining을 공개합니다.

Mineral Mining 표준은 산업·연구·공공 부문에 걸쳐 상호운용 가능한 구현을 위해 필요한 데이터 포맷, 클라이언트 API 계약, 실시간 통신 프로토콜, 생태계 통합 패턴을 명문화합니다. WIA-Official GitHub 조직 산하 참조 구현은 MIT 라이선스로 배포됩니다.

WIA는 자가 선언, 제3자 평가, 인증의 3단계 적합성 평가 체계를 운영하며, 단계별 감사 주기는 배포의 위험도에 따라 차등 적용됩니다. Tier 3 평가는 ISO/IEC 17065:2012 적합성 평가 요건과 정렬되며, 증거 자료의 보존을 전제로 합니다.

자세한 정보는 wiastandards.com를 참조하세요.

WIA 소개 — 세계인증산업협회는 弘益人間(홍익인간) 철학 아래 상호운용 가능한 오픈 표준을 발행하는 국제 표준화 기구입니다.
