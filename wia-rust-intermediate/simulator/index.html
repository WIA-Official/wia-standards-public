Memory Safety Rules

  • Box: Single ownership, heap allocation, no overhead
  • Rc: Multiple owners (single-threaded), reference counting
  • Arc: Thread-safe Rc, atomic reference counting
  • RefCell: Runtime borrow checking, interior mutability
