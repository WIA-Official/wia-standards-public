Newtype 패턴Newtype Pattern

// 타입 안전성 강화
struct UserId(u64);
struct PostId(u64);
fn get_user(id: UserId) { /* ... */ }
let user_id = UserId(123);
let post_id = PostId(456);
get_user(user_id); // ✅ OK
// get_user(post_id); // ❌ 컴파일 에러!

📦 제로 코스트

📦 Zero cost
