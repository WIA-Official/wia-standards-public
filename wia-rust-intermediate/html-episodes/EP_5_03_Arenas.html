Arena란 무엇인가? What is an Arena?

🎯 개념
큰 메모리 블록을 미리 할당하고
그 안에서 작은 할당들을 수행

⚡ 장점
• 개별 할당보다 빠름
• 메모리 파편화 감소
• 한 번에 모두 해제 가능

🎪 사용처
컴파일러, 게임 엔진, 파서 🎯 Concept
Pre-allocate large memory block
and perform small allocations within

⚡ Benefits
• Faster than individual allocations
• Reduces memory fragmentation
• Bulk deallocation

🎪 Use Cases
Compilers, game engines, parsers

간단한 Arena 구현 Simple Arena Implementation

use std::cell::Cell;
struct Arena {
buffer: Vec<u8>,
offset: Cell<usize>,
}
impl Arena {
fn new(capacity: usize) -> Self {
Self {
buffer: Vec::with_capacity(capacity),
offset: Cell::new(0),
}
}
}

bumpalo - 빠른 Arena bumpalo - Fast Arena

use bumpalo::Bump;
fn use_bumpalo() {
let bump = Bump::new();
// 단일 값 할당
let x = bump.alloc(42);
// 슬라이스 할당
let arr = bump.alloc_slice_fill_with(100, |i| i);
// String 할당
let s = bump.alloc_str("Hello, Arena!");
// 전체 리셋
bump.reset();
}

메모리 풀 패턴 Memory Pool Pattern

struct ObjectPool<T> {
objects: Vec<T>,
available: Vec<usize>,
}
impl<T: Default> ObjectPool<T> {
fn acquire(&mut self) -> Option<usize> {
self.available.pop()
}
fn release(&mut self, idx: usize) {
self.available.push(idx);
}
}
🔄 객체 재사용
할당/해제 오버헤드 제거
