Async Trait와 GAT Async Trait with GAT

GAT로 async trait 표현

Express async trait with GAT

trait AsyncIterator {
type Item;
type NextFuture<'a>: Future<Output = Option<Self::Item>>
where Self: 'a;
fn next<'a>(&'a mut self) -> Self::NextFuture<'a>;
}
// Enables async fn in traits without boxing!

GAT의 위력 Power of GAT

  • ✅ 빌림 반환: Iterator가 참조를 반환 가능
  • ✅ 제로 코스트: 복사 없이 추상화
  • ✅ Async 트레잇: Boxing 없이 async fn
  • ✅ 유연성: 라이프타임 파라미터 자유롭게
  • ✅ 타입 안전성: 컴파일 시점 검증
  • ✅ Return Borrows: Iterator can return references
  • ✅ Zero Cost: Abstraction without copies
  • ✅ Async Traits: async fn without boxing
  • ✅ Flexibility: Free lifetime parameters
  • ✅ Type Safety: Compile-time verification

핵심 정리

Key Takeaways

  • GAT = 제네릭 파라미터를 가진 연관 타입
  • 라이프타임을 연관 타입에 추가 가능
  • Lending Iterator, Async Trait에 필수
  • Rust 1.65부터 안정화 (2022년)
  • 제로 코스트 추상화의 핵심 기능
  • GAT = Associated type with generic params
  • Can add lifetimes to associated types
  • Essential for Lending Iterator, Async Trait
  • Stabilized in Rust 1.65 (2022)
  • Key feature for zero-cost abstraction
