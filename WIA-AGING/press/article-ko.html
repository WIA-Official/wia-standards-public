WIA Aging — Healthy Aging and Geriatric Care

WIA 표준 편집부 · 2026-04-25 · 弘益人間 (Benefit All Humanity)

An open standard for healthy aging biomarkers, longevity research data, and geriatric-care interoperability.

이 표준이 필요한 이유

WIA Aging — Healthy Aging and Geriatric Care 표준은 벤더·지역·운영 규모를 가로질러 상호운용되는 구현을 위해 필요한 데이터 형식, API, 프로토콜을 명문화합니다. 이 단일 계약을 따르면 도입 조직은 다음을 얻습니다:

  • 맞춤 통합 작업 없이도 어떤 호환 시스템과도 레코드를 주고받을 수 있음.
  • 안정적이고 기계가 검증 가능한 테스트 스위트를 통한 적합성 감사.
  • WIA 적합성 프로그램이 발행하는 인증 자격증명에 대한 접근.

대상 범위

표준은 4개 페이즈로 구성됩니다:

  1. 데이터 형식 — 정규 JSON Schema, YAML, 바이너리 인코딩.
  2. API 인터페이스 — X-API-Key + OAuth 2.0 인증을 갖춘 OpenAPI 3.0 REST 계약.
  3. 프로토콜 — 요청/응답 시맨틱, 멱등성, 오류 봉투, 요율 제한.
  4. 통합 — 본 표준이 다른 WIA 표준과 어떻게 결합되는지.

도입 경로

구현은 WIA-AGING/api/sdk/typescript/ 와 WIA-AGING/api/sdk/python/ 의 오픈소스 SDK 로 시작하기를 권장하며, 이어서 WIA-AGING/cli/ 의 검증 CLI 로 적합성을 확인합니다. 양쪽이 통과하면 cert.wiastandards.com 에서 인증을 신청할 수 있습니다.

열려 있고, 검증 가능하며, 기계로 점검 가능한 표준만이 표준을 만든 조직보다 오래 살아남을 만큼 견고하다. — WIA 편집위원회
