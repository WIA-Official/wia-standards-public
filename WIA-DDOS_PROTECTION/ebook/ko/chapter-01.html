DDoS 공격의 이해: 사이버 쓰나미

1. DDoS란 무엇인가?

DDoS (Distributed Denial of Service, 분산 서비스 거부)는 수많은 장치에서 동시에 공격하여 대상 서버나 네트워크를 마비시키는 사이버 공격입니다. 마치 수천 명이 동시에 상점 문을 막아서 정상 고객이 들어갈 수 없게 만드는 것과 같습니다.

🎯 DDoS의 핵심 개념

  • Distributed (분산): 단일 출처가 아닌 수백~수백만 장치에서 발생
  • Denial of Service (서비스 거부): 정당한 사용자의 접근을 차단
  • 목적: 서비스 중단, 금전 갈취, 정치적 시위, 경쟁 방해
  • 방법: 대역폭 고갈, 서버 리소스 소진, 애플리케이션 취약점 악용

1.1 DoS vs DDoS

항목 DoS (Denial of Service) DDoS (Distributed DoS)
공격 출처 단일 장치 (1개 IP) 수천~수백만 장치 (분산)
탐지 난이도 쉬움 (IP 차단) 어려움 (출처 광범위)
공격 규모 작음 (Mbps ~ Gbps) 매우 큼 (Tbps급)
방어 난이도 낮음 (방화벽으로 차단) 높음 (전문 서비스 필요)
비용 낮음 (개인 PC로 가능) 높음 (봇넷 임대 필요)
예시 Ping Flood (1대에서) Mirai 봇넷 (60만 대)

2. DDoS 공격의 역사

2.1 주요 DDoS 사건 연대표

연도 사건 규모/영향
1999 University of Minnesota 공격 최초의 대규모 DDoS (2일간 마비)
2000 Mafiaboy의 Yahoo 공격 Yahoo, eBay, Amazon 동시 마비
2007 에스토니아 사이버 공격 국가 인프라 3주간 마비
2012 Spamhaus 공격 300 Gbps (당시 최대 규모)
2016 Dyn DNS 공격 (Mirai) 1.2 Tbps, Twitter/Netflix 마비
2018 GitHub 공격 1.35 Tbps (Memcached 증폭)
2020 AWS 공격 2.3 Tbps (CLDAP 증폭)
2021 Microsoft Azure 공격 3.47 Tbps (역대 최대)
2023 Google 공격 398M rps (초당 요청, HTTP/2)

2.2 2016 Mirai 봇넷 사건

Mirai는 IoT 장치(IP 카메라, DVR)를 감염시켜 60만 대의 봇넷을 구축했습니다. 이 봇넷이 Dyn DNS 서비스를 공격하여 Twitter, Netflix, Reddit, CNN 등 주요 웹사이트가 수 시간 동안 마비되었습니다.

⚠️ Mirai의 교훈

  • IoT 보안 취약: 대부분 기본 비밀번호 사용 (admin/admin)
  • 규모의 공포: 1.2 Tbps = 인터넷 백본의 10%
  • 소스 공개: 공격자가 소스코드 공개 → 변종 등장
  • 법적 처벌: 개발자 체포 (2017년)

3. DDoS 공격의 동기

3.1 6가지 주요 동기

동기 설명 예시
금전 갈취 랜섬 요구 (비트코인) "1 BTC 안 주면 계속 공격"
경쟁 방해 경쟁 업체 서비스 중단 E-commerce 블랙프라이데이 공격
정치적 시위 Hacktivism (해킹 활동주의) Anonymous의 정부 사이트 공격
개인적 원한 불만 고객, 해고된 직원 전 직원의 복수 공격
국가 지원 사이버 전쟁의 일환 에스토니아 공격 (러시아 의심)
실력 과시 해커 커뮤니티 명성 청소년 해커 (Mafiaboy)

3.2 DDoS-for-Hire (DDoS 임대 서비스)

다크웹에서는 $10-50/시간에 DDoS 공격을 대행해주는 "Booter" 또는 "Stresser" 서비스가 존재합니다. 신용카드만 있으면 누구나 DDoS 공격을 수행할 수 있어 진입장벽이 매우 낮아졌습니다.

# DDoS-for-Hire 서비스 예시 (불법!) # 다크웹 광고 (실제 사이트 아님, 예시) [PREMIUM DDOS SERVICE] 💀 ───────────────────────────────── ⚡ 100 Gbps 공격력 ⌚ $20/hour 🎯 모든 타겟 가능 🔒 익명 보장 (Bitcoin 결제) 📊 실시간 대시보드 Packages: - Bronze: 10 Gbps, $10/hr - Silver: 50 Gbps, $15/hr - Gold: 100 Gbps, $20/hr - Platinum: 200 Gbps, $50/hr ⚠️ 주의: 이런 서비스 이용은 중범죄입니다! 미국: Computer Fraud and Abuse Act (최대 10년) 한국: 정보통신망법 (5년 이하 징역 or 5천만원 벌금)

4. 봇넷 (Botnet)의 작동 원리

봇넷은 DDoS 공격의 핵심 인프라입니다. 수천~수백만 대의 감염된 장치(Bot)를 원격 제어하여 공격을 수행합니다.

4.1 봇넷 구조

                 ┌──────────────────┐
                 │  C&C Server      │  ← Command & Control
                 │  (공격자 제어)   │
                 └────────┬─────────┘
                          │
          ┌───────────────┼───────────────┐
          │               │               │
          ▼               ▼               ▼
    ┌─────────┐     ┌─────────┐     ┌─────────┐
    │  Bot 1  │     │  Bot 2  │ ... │ Bot N   │
    │ (PC)    │     │ (IoT)   │     │ (Server)│
    └────┬────┘     └────┬────┘     └────┬────┘
         │               │               │
         └───────────────┼───────────────┘
                         │
                         ▼
                  ┌─────────────┐
                  │  Target     │
                  │  (피해자)   │
                  └─────────────┘

C&C Server → Bots: "공격 시작! IP: 1.2.3.4, Port: 80"
Bots → Target: HTTP GET / (초당 100만 요청)

4.2 봇넷 감염 경로

  1. 악성코드 유포: 이메일 첨부파일, 피싱 링크
  2. 취약점 악용: 패치되지 않은 OS, 소프트웨어
  3. 기본 비밀번호: IoT 장치 (admin/admin, root/root)
  4. 드라이브 바이 다운로드: 감염된 웹사이트 방문 시 자동 설치
  5. 소프트웨어 번들링: 무료 프로그램에 숨겨진 악성코드

4.3 주요 봇넷 종류

봇넷 이름 발견 연도 규모 특징
Mirai 2016 60만 대 IoT 장치, 소스 공개
Zeus 2007 360만 대 금융 정보 탈취 + DDoS
Conficker 2008 1500만 대 Windows 취약점 악용
Emotet 2014 100만 대 이메일 스팸 + 뱅킹 트로이목마
Qakbot 2008 50만 대 Ransomware 전달

5. DDoS 공격 통계

5.1 2024년 DDoS 현황

📊 글로벌 통계 (Cloudflare, AWS 보고서 기준)

  • 공격 횟수: 일평균 1,700만 건 (2023년 대비 30% 증가)
  • 평균 공격 시간: 25분 (짧고 강력한 공격 증가)
  • 최대 규모: 3.8 Tbps (Cloudflare 방어 성공)
  • 주요 타겟 산업: 금융(35%), 게임(25%), E-commerce(20%)
  • 공격 비용: 기업당 평균 피해액 $22,000/시간
  • 봇넷 규모: 평균 10만 대 (Mirai 변종)

6. DDoS 방어의 필요성

6.1 비즈니스 영향

영향 범위 구체적 피해 예상 손실
직접 손실 서비스 중단 시간당 매출 손실 $22,000/시간
평판 손상 고객 이탈, 브랜드 신뢰 하락 장기적 영향
대응 비용 보안 팀 초과 근무, 긴급 대응 $50,000 - $500,000
법적 책임 SLA 위반, 고객 보상 계약 금액의 10-30%
주가 하락 상장 기업 시가총액 감소 평균 3-5% (대규모 공격 시)

6.2 산업별 DDoS 위험도

⚠️ 고위험 산업

  • 금융 서비스: 1분 중단 = 수백만 달러 손실
  • E-commerce: 블랙프라이데이 등 피크 시즌 타겟
  • 온라인 게임: 경쟁사 방해, 사용자 불만 유발
  • 의료: 생명과 직결 (병원 시스템 마비)
  • 정부 기관: 정치적 목적, 국가 안보 위협
  • 미디어: 뉴스 속보 시 공격 (여론 조작)

🎯 핵심 요점

  • DDoS = 분산 + 서비스 거부: 수천 장치에서 동시 공격
  • 역사적 성장: 2000년 Mbps → 2021년 3.47 Tbps (3,470배)
  • Mirai 사건: IoT 60만 대 감염, 1.2 Tbps, Twitter/Netflix 마비
  • 6가지 동기: 금전, 경쟁, 정치, 원한, 국가, 과시
  • 봇넷 구조: C&C 서버 → Bots → Target
  • DDoS-for-Hire: $10-50/시간, 진입장벽 낮음 (불법!)
  • 2024년 트렌드: App Layer 공격 급증, 짧고 강력, AI 활용
  • 비즈니스 피해: $22K/시간 + 평판 손상 + 법적 책임
  • 고위험 산업: 금융, E-commerce, 게임, 의료, 정부
  • 弘益人間: DDoS 방어로 안정적인 서비스, 모두에게 이로움 🛡️
