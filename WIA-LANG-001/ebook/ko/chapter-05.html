WIA-LANG-001

05.1 API 설계 원칙

WIA-LANG-001 API는 RESTful 아키텍처를 기반으로 설계되며, 언어 처리 서비스를 표준화된 방식으로 제공합니다.

🎯 핵심 설계 원칙
  • 일관성: 모든 엔드포인트가 동일한 패턴 따름
  • 확장성: 새로운 언어 및 기능 쉽게 추가
  • 버전 관리: URL 기반 버전 관리 (/v1/, /v2/)
  • 오류 처리: 명확하고 일관된 오류 응답
  • 보안: OAuth 2.0, API 키 인증 지원
  • 성능: 페이지네이션, 캐싱, 비동기 처리

RESTful 리소스 구조

리소스엔드포인트HTTP 메서드설명
언어 /v1/languages GET 지원 언어 목록
ASR /v1/asr/transcribe POST 음성-텍스트 변환
TTS /v1/tts/synthesize POST 텍스트-음성 합성
MT /v1/mt/translate POST 기계 번역
NLU /v1/nlu/analyze POST 자연어 이해
작업 /v1/jobs/{id} GET 비동기 작업 상태 조회

공통 요청 헤더

Authorization: Bearer {api_key}
Content-Type: application/json
Accept: application/json
Accept-Language: en-US,ko-KR
X-Request-ID: {unique_request_id}
X-Client-Version: wia-sdk-python/1.0.0

05.2 ASR API (자동 음성 인식)

ASR API는 오디오 파일 또는 스트림을 텍스트로 변환합니다.

Content-Type: multipart/form-data

05.8 SDK 및 클라이언트 라이브러리

Python SDK 예제

# 설치
pip install wia-lang-sdk

# 사용 예제
from wia import WIAClient

# 클라이언트 초기화
client = WIAClient(api_key="wia_live_abc123...")

# ASR
transcription = client.asr.transcribe(
    audio_file="recording.wav",
    language="haw",
    return_timestamps=True
)
print(transcription.text)

# TTS
audio = client.tts.synthesize(
    text="Aloha kākou",
    language="haw",
    voice="female_elder"
)
audio.save("output.wav")

# MT
translation = client.mt.translate(
    text="Kei te haere au ki te kura",
    source_language="mri",
    target_language="en"
)
print(translation.text)

# NLU
analysis = client.nlu.analyze(
    text="ʻO Pele ka wahine ʻai honua",
    language="haw",
    features=["entities", "sentiment", "syntax"]
)
print(analysis.entities)

JavaScript/TypeScript SDK

// 설치
npm install @wia/lang-sdk

// 사용 예제
import { WIAClient } from '@wia/lang-sdk';

const client = new WIAClient({
  apiKey: 'wia_live_abc123...'
});

// ASR
const transcription = await client.asr.transcribe({
  audioFile: './recording.wav',
  language: 'haw',
  returnTimestamps: true
});

console.log(transcription.text);

// TTS
const audio = await client.tts.synthesize({
  text: 'Aloha kākou',
  language: 'haw',
  voice: 'female_elder'
});

await audio.save('output.wav');
