# WIA-LEG-008 Data Portability Standard v2.0

**Status:** Final  
**Published:** 2025-01-10  
**Category:** Digital Legacy & Legal Rights  
**License:** MIT

## Major Changes from v1.x

### Breaking Changes
- Restructured manifest.json schema (added exporter, package metadata)
- Activity schema now strictly follows Activity Streams 2.0
- Relationship types expanded (symmetric, asymmetric, hierarchical)

### New Features
- Phase 2: RESTful Export API specification
- Phase 3: Transfer Protocol for cross-platform migration
- Phase 4: WIA Ecosystem Integration
- OAuth 2.0 authentication requirements
- Rate limiting and pagination standards
- Webhook notification system

### Enhanced Schemas
- Profile schema with verification badges
- Commerce with transaction history
- Messages with conversation threads
- Media with comprehensive EXIF/metadata

### Security
- Mandatory multi-factor authentication for exports
- End-to-end encryption for transfers
- Audit logging requirements
- Time-limited access tokens

### Compliance
- Expanded GDPR Article 20 implementation guidance
- CCPA/CPRA compliance mappings
- LGPD, PIPEDA, PIPL considerations

## Migration from v1.x

v1.x packages remain valid. Importers must support backward compatibility.
Exporters should upgrade to v2.0 schema for new exports.
