# WIA-LEG-008 Data Portability Standard v1.1

**Status:** Final  
**Published:** 2024-06-20  
**Category:** Digital Legacy & Legal Rights  
**License:** MIT

## Changes from v1.0

### Additions
- Added commerce data schema (purchases, reviews, wishlist)
- Added messages schema for private communications
- Extended media metadata with EXIF support
- Added incremental export capability

### Improvements
- Enhanced relationship modeling (mutual, labels)
- Better activity stream conformance to AS 2.0
- Improved checksum algorithms (added SHA-256)

### Deprecations
- None

## All v1.0 specifications remain valid with above additions.
