# WIA-LEG-008 Data Portability Standard v1.2

**Status:** Final  
**Published:** 2024-10-15  
**Category:** Digital Legacy & Legal Rights  
**License:** MIT

## Changes from v1.1

### Additions
- Multi-language support (BCP 47 language tags)
- Extended preference schema (notifications, privacy)
- Added deletion markers for RTBF compliance
- Support for encrypted field-level data

### Security Enhancements
- Mandatory TLS 1.3 for package transfer
- Optional AES-256-GCM package encryption
- Digital signature support (RSA, ECDSA)

### Bug Fixes
- Fixed timezone handling in activity timestamps
- Corrected media path resolution for nested directories
