2.1 CORE LEGAL IDENTITY
--------------------------------------------------------------------------------

Legal identity forms the foundation of identity preservation - the official
records that establish who a person is in the eyes of governments and law.

[ID] LEGAL NAME RECORD:

    LegalName {
      givenName:   "John"          // Required
      middleName:  "William"       // Optional
      familyName:  "Smith"         // Required
      prefix:      "Dr."           // Optional
      suffix:      "Jr."           // Optional
    }

    Alternative Names:
    - MAIDEN: Previous family name
    - ALIAS: Known aliases
    - PROFESSIONAL: Stage/pen names
    - PREVIOUS: Legally changed names


--------------------------------------------------------------------------------
2.2 GOVERNMENT IDENTIFICATIONS
--------------------------------------------------------------------------------

[ID] SUPPORTED ID TYPES:

    +------------------------------------------------------------------------+
    | Type              | Priority  | Usage                                  |
    +------------------------------------------------------------------------+
    | PASSPORT          | Critical  | International identity, travel         |
    | NATIONAL_ID       | Critical  | Domestic primary identification        |
    | DRIVERS_LICENSE   | High      | Secondary ID, address verification     |
    | SSN/TAX_ID        | High      | Financial, employment records          |
    +------------------------------------------------------------------------+

[ID] GOVERNMENT ID RECORD:

    GovernmentIdRecord {
      idType:       "PASSPORT"
      country:      "USA"
      number:       "[ENCRYPTED]"        // Stored encrypted
      issueDate:    "2020-01-15"
      expiryDate:   "2030-01-15"
      issuer:       "U.S. Department of State"
      documentHash: "sha256:a1b2c3..."   // Integrity verification
    }

[!] SECURITY REQUIREMENTS:
    - ID numbers stored encrypted at rest
    - Access requires authorized credentials
    - Document images stored separately with hashes


--------------------------------------------------------------------------------
2.3 CITIZENSHIP RECORDS
--------------------------------------------------------------------------------

Citizenship documentation ensures the preserved individual's nationality
rights can be restored upon revival.

[ID] CITIZENSHIP TYPES:

    BIRTH:        Citizenship by place of birth
    NATURALIZED:  Citizenship obtained after birth
    DUAL:         Multiple citizenships held
    STATELESS:    No recognized citizenship

[ID] CITIZENSHIP RECORD:

    Citizenship {
      country:        "USA"
      type:           "BIRTH"
      documentNumber: "[ENCRYPTED]"
      issueDate:      "1970-03-15"
    }

[i] MULTI-CITIZENSHIP:
    Some individuals hold multiple citizenships. All should be documented
    to ensure complete restoration of legal status upon revival.


--------------------------------------------------------------------------------
2.4 FAMILY RELATIONSHIPS
--------------------------------------------------------------------------------

Family relationships are legally significant and essential for identity
verification and revival decision-making.

[ID] RELATIONSHIP TYPES:

    PARENT      -> Biological or legal parent
    CHILD       -> Biological or legal child
    SIBLING     -> Brother or sister
    SPOUSE      -> Current married partner
    GRANDPARENT -> Parent's parent
    GRANDCHILD  -> Child's child

[ID] RELATIONSHIP RECORD:

    FamilyRelationship {
      relationship:     "SPOUSE"
      person: {
        name:           "Jane Smith"
        cryoId:         "CRYOID-2025-001-00002"  // If also preserved
        dateOfBirth:    "1975-06-20"
        status:         "LIVING"  // LIVING | DECEASED | PRESERVED
      }
      biological:       false
      legal:            true
      verificationStatus: "VERIFIED"
    }

[i] VERIFICATION STATUS:
    - VERIFIED: Relationship confirmed with documents
    - UNVERIFIED: Claimed but not yet verified
    - CLAIMED: Self-reported, awaiting verification


--------------------------------------------------------------------------------
2.5 LEGAL CONTINUITY DURING PRESERVATION
--------------------------------------------------------------------------------

Special legal arrangements maintain identity rights during preservation.

[ID] REQUIRED LEGAL DOCUMENTS:

    1. POWER OF ATTORNEY
       - Designate legal representative
       - Scope of authority defined
       - Valid for preservation period

    2. TRUST DOCUMENTATION
       - Asset management arrangements
       - Beneficiary designations
       - Revival contingencies

    3. REVIVAL CONSENT
       - Conditions for revival
       - Decision-maker designation
       - Medical directives

[ID] LEGAL STATUS TRACKING:

    LegalStatusRecord {
      jurisdiction:      "California, USA"
      status:            "RECOGNIZED"  // RECOGNIZED | SUSPENDED | UNRECOGNIZED
      legalDocuments:    ["POA-2025-001", "TRUST-2025-001"]
      lastVerification:  "2025-03-15"
      notes:             "Preservation rights recognized under state law"
    }
