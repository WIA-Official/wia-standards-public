1.1 신원 보존이 중요한 이유
--------------------------------------------------------------------------------

냉동보존은 생물학적 몸을 보존합니다.
하지만 당신을 "당신"으로 만드는 것은 무엇일까요?

[ID] 신원은 다음을 포함합니다:

    법적 신원     -> 정부가 인정하는 당신
    개인 신원     -> 관계와 기억
    디지털 신원   -> 온라인 존재와 자산
    생체 신원     -> 고유한 물리적 마커

부활 시, 이 모든 측면이 복원되어야 합니다. 보존이 없다면,
수십 년 후 깨어나는 것은 법적으로 존재하지 않고,
모든 계정에서 차단되고, 가족으로부터 인정받지 못하는 것을 의미합니다.

--------------------------------------------------------------------------------
1.3 보존 생애주기
--------------------------------------------------------------------------------

[ID] 신원 보존 타임라인:

    보존 전               보존 중                    부활 후
    +---------+          +------------+          +-----------+
    | 수집    |    ->    | 유지보수   |    ->    | 복원      |
    | 검증    |          | 업데이트   |          | 검증      |
    | 금고화  |          | 후견인관리 |          | 재통합    |
    +---------+          +------------+          +-----------+

[ID] 보존 전 단계:

    1. 신원 수집
       - 모든 문서 수집
       - 생체 정보 캡처
       - 디지털 자산 카탈로그화

    2. 검증
       - 문서 진위 확인
       - 교차 참조 기록
       - 후견인 지정

    3. 금고 저장
       - 암호화 및 저장
       - 다중 위치 복제
       - 접근 정책 설정

[ID] 보존 중 단계:

    1. 정기 무결성 검사
    2. 법적 상태 변경 모니터링
    3. 후견인 네트워크 유지
    4. 기술 마이그레이션 (필요시)

[ID] 부활 후 단계:

    1. 생체 검증
    2. 법적 신원 복원
    3. 디지털 신원 복구
    4. 사회 재통합 지원


--------------------------------------------------------------------------------
1.4 핵심 원칙
--------------------------------------------------------------------------------

[ID] 원칙 1: 연속성

    신원은 보존 간격에 관계없이 연속적이어야 합니다.
    50년 후 깨어나는 것은 법적, 사회적으로 "사라졌다가"
    "돌아온" 것이지, 새로운 사람이 되는 것이 아닙니다.

[ID] 원칙 2: 보안

    신원 데이터는 가장 민감한 개인 정보입니다.
    암호화, 접근 제어, 감사 로깅이 필수입니다.

[ID] 원칙 3: 이중화

    단일 실패 지점은 허용되지 않습니다.
    다중 위치, 다중 형식, 다중 후견인.

[ID] 원칙 4: 적응성

    기술은 변합니다. 법률은 변합니다.
    신원 시스템은 변화에 적응할 수 있어야 합니다.

[ID] 원칙 5: 존엄성

    신원 보존은 인간의 존엄성을 존중해야 합니다.
    개인의 자율성과 프라이버시가 우선입니다.
