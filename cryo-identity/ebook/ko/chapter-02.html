--------------------------------------------------------------------------------
2.1 법적 신원 개요
--------------------------------------------------------------------------------

법적 신원은 정부 기관이 인정하는 공식 신원입니다.
부활 후 사회로 복귀하려면 법적 신원의 복원이 필수입니다.

[ID] 법적 신원 기록 구조:

    LegalIdentityRecord {
      recordId:         "LEGAL-2025-001"
      identityId:       "CRYOID-2025-001"
      type:             "LEGAL"

      fullLegalName:    "홍길동"
      dateOfBirth:      "1970-03-15"
      placeOfBirth:     "서울특별시, 대한민국"
      nationality:      ["대한민국"]
      governmentIds: {
        주민등록번호:    "[암호화됨]"
        여권번호:        "[암호화됨]"
        운전면허번호:    "[암호화됨]"
      }
    }


--------------------------------------------------------------------------------
2.2 정부 발급 문서
--------------------------------------------------------------------------------

[ID] 필수 문서:

    +-------------------------------------------------------------------+
    | 문서 유형          | 발급 기관      | 갱신 주기    | 중요도      |
    +-------------------------------------------------------------------+
    | 주민등록증         | 주민센터       | 해당없음     | 필수        |
    | 여권               | 외교부         | 10년         | 필수        |
    | 운전면허증         | 경찰청         | 10년         | 권장        |
    | 건강보험증         | 건강보험공단   | 해당없음     | 권장        |
    +-------------------------------------------------------------------+

[ID] 문서 기록 형식:

    GovernmentDocument {
      documentId:     "DOC-PASSPORT-001"
      documentType:   "PASSPORT"
      issuingCountry: "KR"
      issuingAuthority: "대한민국 외교부"
      documentNumber: "[암호화됨]"
      issueDate:      "2020-03-15"
      expirationDate: "2030-03-14"
      status:         "ACTIVE"
      digitalCopy:    "vault://legal/passport.pdf"
      verificationProof: "sha256:a1b2c3..."
    }

[!] 문서 보관 요구사항:
    - 고해상도 스캔 (최소 300 DPI)
    - PDF/A 형식 권장 (장기 보존용)
    - 원본과 번역본 모두 보관
    - 공증 사본 권장


--------------------------------------------------------------------------------
2.3 출생 및 가족 기록
--------------------------------------------------------------------------------

[ID] 출생 기록:

    BirthRecord {
      certificateNumber: "서울-1970-123456"
      fullName:          "홍길동"
      dateOfBirth:       "1970-03-15"
      placeOfBirth:      "서울특별시 종로구"
      parents: {
        father: { name: "홍판서", relationship: "아버지" }
        mother: { name: "춘섬",   relationship: "어머니" }
      }
      registrar:         "서울시 종로구청"
      registrationDate:  "1970-03-20"
    }

[ID] 가족관계증명서:

    FamilyRecord {
      certificateType:  "가족관계증명서"
      issuingAuthority: "대법원"
      issueDate:        "2025-01-15"
      members: [
        { name: "홍길동", relationship: "본인",   birthDate: "1970-03-15" },
        { name: "홍판서", relationship: "아버지", birthDate: "1940-05-20" },
        { name: "춘섬",   relationship: "어머니", birthDate: "1945-08-10" },
        { name: "홍길순", relationship: "배우자", birthDate: "1972-07-25" },
        { name: "홍명동", relationship: "자녀",   birthDate: "2000-01-01" }
      ]
    }

2.4 법적 지위 문서
--------------------------------------------------------------------------------

[ID] 혼인 기록:

    MarriageRecord {
      certificateNumber: "서울-2000-789012"
      spouse1:           "홍길동"
      spouse2:           "홍길순"
      marriageDate:      "2000-06-15"
      marriagePlace:     "서울특별시"
      registrar:         "서울시 강남구청"
      status:            "MARRIED"
    }

[ID] 법적 지위 유형:

    LegalStatus {
      category: "PRESERVATION"  // 보존 관련 법적 지위

      statuses: [
        {
          type:      "MARRIAGE"
          status:    "MARRIED"
          since:     "2000-06-15"
          documents: ["혼인관계증명서"]
        },
        {
          type:      "CITIZENSHIP"
          status:    "CITIZEN"
          country:   "대한민국"
          since:     "1970-03-15"
          documents: ["여권", "주민등록증"]
        },
        {
          type:      "PRESERVATION"
          status:    "PRESERVED"
          since:     "2025-03-15"
          documents: ["보존동의서", "신원금고증명서"]
        }
      ]
    }


--------------------------------------------------------------------------------
2.5 국제 신원 관리
--------------------------------------------------------------------------------

[ID] 다중 국적 처리:

    복수 국적을 가진 경우 모든 국가의 신원 문서를 보관해야 합니다.

    MultipleNationality {
      primaryNationality:   "대한민국"
      additionalNationalities: ["미국"]

      documentsPerCountry: {
        "KR": ["주민등록증", "여권", "가족관계증명서"],
        "US": ["Social Security Card", "US Passport"]
      }
    }

[ID] 국제 법적 고려사항:

    - 보존 기간 중 법률 변경 가능성
    - 국가별 사망 선고 규정 차이
    - 상속법 적용 관할권
    - 부활 후 시민권 복원 절차

[i] 법률 자문:
    장기 보존 전 국제법 전문 변호사와 상담을 권장합니다.
    특히 상속, 시민권, 의료 결정권 관련 사전 준비가 필요합니다.
