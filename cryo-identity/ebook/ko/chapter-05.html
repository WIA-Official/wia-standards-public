--------------------------------------------------------------------------------
5.1 DNA 프로파일링
--------------------------------------------------------------------------------

DNA는 가장 결정적인 생체 식별 형태를 제공합니다.
보존을 통해 지속되는 고유한 유전적 지문입니다.

[ID] 시퀀스 유형:

    FULL_GENOME  -> 전체 게놈 시퀀싱 (~30억 염기쌍)
    EXOME        -> 코딩 영역만 (~게놈의 2%)
    SNP_PANEL    -> 단일 뉴클레오타이드 다형성
    STR          -> 짧은 탠덤 반복 마커 (법의학 표준)

[ID] DNA 프로파일 기록:

    DNAProfile {
      profileId:      "DNA-2025-001-a1b2c3"
      sequenceType:   "STR"
      sequenceDate:   "2025-03-01"
      laboratory:     "WIA 유전체학 연구소"
      storageFormat:  "VCF"
      storageLocation: "vault://biometric/dna/DNA-2025-001"
      integrityHash:  "sha256:d4e5f6..."
      markers: [
        { marker: "D3S1358",  value: "15,16" },
        { marker: "vWA",      value: "17,18" },
        { marker: "D16S539",  value: "11,12" },
        { marker: "CSF1PO",   value: "10,12" },
        { marker: "TPOX",     value: "8,11" },
        // ... 추가 STR 유전자좌
      ]
    }


--------------------------------------------------------------------------------
5.2 지문 캡처
--------------------------------------------------------------------------------

지문은 보조 생체 검증을 제공합니다.

[ID] 손가락 식별:

    +--------+--------+--------+--------+--------+
    |  RT    |  RI    |  RM    |  RR    |  RP    |
    | 엄지   | 검지   | 중지   | 약지   | 새끼   |
    +--------+--------+--------+--------+--------+
    |  LT    |  LI    |  LM    |  LR    |  LP    |
    | 엄지   | 검지   | 중지   | 약지   | 새끼   |
    +--------+--------+--------+--------+--------+

[ID] 지문 기록:

    Fingerprint {
      digit:       "RIGHT_INDEX"
      template:    "[암호화된 생체 템플릿]"
      format:      "ISO_19794_2"
      quality:     92  // 0-100 품질 점수
      captureDate: "2025-03-01"
    }

[i] 품질 요구사항:
    - 최소 품질 점수: 70
    - 모든 10개 손가락 권장
    - 신뢰성을 위한 다중 캡처


--------------------------------------------------------------------------------
5.3 안면 기하학
--------------------------------------------------------------------------------

안면 인식은 비침습적 신원 검증을 제공합니다.

[ID] 안면 기하학 기록:

    FacialGeometry {
      templateId:   "FACE-2025-001"
      template:     "[암호화된 템플릿]"
      format:       "ISO_19794_5"
      captureDate:  "2025-03-01"
      pose:         "FRONTAL"
      lightingCondition: "CONTROLLED"
      referencePhotos: [
        "vault://biometric/facial/front.jpg",
        "vault://biometric/facial/left.jpg",
        "vault://biometric/facial/right.jpg"
      ]
    }

[i] 캡처 요구사항:
    - 무표정, 정면 포즈 기본
    - 좌우 측면 보조
    - 일관성을 위한 통제된 조명


--------------------------------------------------------------------------------
5.4 추가 생체인식
--------------------------------------------------------------------------------

[ID] 홍채 패턴:

    IrisPattern {
      eye:          "LEFT"  // LEFT | RIGHT
      template:     "[암호화됨]"
      format:       "ISO_19794_6"
      captureDate:  "2025-03-01"
      quality:      88
    }

[ID] 음성 프린트:

    VoicePrint {
      templateId:   "VOICE-2025-001"
      template:     "[암호화됨]"
      captureDate:  "2025-03-01"
      language:     "ko-KR"
      duration:     30  // 초
      sampleRate:   48000
    }

[ID] 망막 스캔:

    RetinalScan {

      template:     "[암호화됨]"
      format:       "PROPRIETARY"
      captureDate:  "2025-03-01"
      quality:      85
    }


--------------------------------------------------------------------------------
5.5 물리적 특성
--------------------------------------------------------------------------------

[ID] 물리적 기록:

    PhysicalCharacteristics {
      height:     175      // cm
      weight:     70       // 보존 시 kg
      bloodType:  "A+"
      eyeColor:   "갈색"
      hairColor:  "검정"
      skinTone:   "중간"
      distinguishingMarks: [
        {
          type:        "SCAR"
          location:    "왼쪽 팔뚝"
          description: "어린 시절 수술 흉터 3cm"
        },
        {
          type:        "BIRTHMARK"
          location:    "오른쪽 어깨"
          description: "타원형 모반 2cm"
        }
      ]
    }

[ID] 구별 특징 유형:

    SCAR       -> 흉터 (수술, 사고)
    TATTOO     -> 문신
    BIRTHMARK  -> 모반
    MOLE       -> 점
    PIERCING   -> 피어싱


--------------------------------------------------------------------------------
5.6 생체 검증
--------------------------------------------------------------------------------

[ID] 다중 모달 검증:

    검증은 높은 신뢰도를 위해 여러 생체인식을 결합합니다:

    +------------------------------------------------------------------------+
    | 생체인식       | 가중치  | 임계값    | 신뢰도 기여                     |
    +------------------------------------------------------------------------+
    | DNA            | 40%     | 0.9999    | 결정적 식별                     |
    | 지문           | 20%     | 0.85      | 강력한 보조                     |
    | 안면           | 15%     | 0.80      | 시각적 확인                     |
    | 홍채           | 15%     | 0.90      | 높은 정확도 보조                |
    | 음성           | 10%     | 0.75      | 추가 보조                       |
    +------------------------------------------------------------------------+

[ID] 검증 결정:
    - 긍정적 식별을 위해 DNA 일치 필수
    - 결합 점수는 0.80 초과해야 함
    - 최소 2개 모달리티가 일치해야 함

[ID] 검증 워크플로:

    +------------------+
    | 샘플 수집        |
    | - DNA 샘플       |
    | - 지문           |
    | - 안면 스캔      |

             |
             v
    +------------------+
    | 금고와 비교      |
    | - DNA 일치?      |
    | - 생체 일치?     |
    +--------+---------+
             |
    +--------v---------+
    | 신원             |
