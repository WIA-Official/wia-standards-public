Chapter 4: Phase 1 - Data Format

4.1 Grid Topology Schema

All power grid assets must be registered using standardized JSON-LD format:

{
  "@context": "https://wiastandards.com/contexts/uni-007/v1",
  "@type": "PowerGridNode",
  "id": "wia:grid:HVDC-WS-001",
  "nodeType": "hvdc-terminal",
  "name": "Wonsan HVDC Converter",
  "capacity": {"value": 2000, "unit": "MW"},
  "voltage": {"value": 765, "unit": "kV"},
  "coordinates": {"lat": 39.1527, "lon": 127.4436}
}

4.2 Power Flow Measurements

Real-time power flow data updated every 5 seconds includes active/reactive power, voltage/frequency, line loading, and direction of flow. All measurements use SI units with timestamp precision to milliseconds.

4.3 Renewable Energy Metrics

Track renewable generation, carbon emissions avoided, capacity factor, and forecasted output. This enables optimized renewable integration and carbon accounting across the unified grid.
