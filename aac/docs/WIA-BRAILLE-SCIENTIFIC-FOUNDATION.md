# Executive Summary (기업 법무팀용 요약)

## 핵심 질문과 답변

### Q1: WIA Braille이 기존 점자와 무엇이 다른가?
**A: IPA(국제음성기호) 기반 통합 체계입니다.** 기존 점자는 국가마다 다른 체계를 사용하지만, WIA Braille은 IPA를 매개로 211개 언어를 단일 체계로 통합합니다.

### Q2: 기존 점자를 대체하는가?
**A: 아니오.** WIA Braille은 기존 점자와 공존합니다. 한국 사용자는 한글 점자 + WIA Braille을, 영어권은 영어 점자 + WIA Braille을 함께 사용할 수 있습니다. "추가 접근성 옵션"입니다.

### Q3: 학술적 근거가 있는가?
**A: 예.** 1888년 확립된 IPA(국제음성기호)를 기반으로 하며, 1824년 루이 브라유의 6점 체계를 8점으로 확장한 것입니다. 200년간 검증된 두 체계의 융합입니다.

### Q4: 법적으로 인정되는가?
**A: 접근성 법규에 완전히 부합합니다.** 장차법, EU Accessibility Act, ADA, UN CRPD 모두 "접근성 제공"을 요구하지 "특정 점자만 제공"을 요구하지 않습니다.

# 1. 점자의 역사와 루이 브라유

## 1.1 루이 브라유 (Louis Braille, 1809-1852)

| 항목 | 내용 |
|------|------|
| 출생 | 1809년 1월 4일, 프랑스 쿠프브레 |
| 실명 | 3세 (아버지 작업장에서 송곳 사고) |
| 발명 | 15세 (1824년), 6점 점자 시스템 |
| 완성 | 20세 (1829년), 최초 출판 |
| 사망 | 1852년 1월 6일 (43세, 결핵) |
| 공인 | 1854년 프랑스 공식 채택 (사후 2년) |

## 1.2 6점 점자의 천재성

### 수학적 구조

점자 셀 (Braille Cell):
- 2열 × 3행 = 6개 점
- 각 점: 있음(1) 또는 없음(0)
- 가능한 조합: 2의6승 = 64가지

### 인체공학적 최적화

- 손가락 끝 인식 범위: 약 6mm × 9mm
- 브라유 셀 크기: 6mm × 10mm (표준)
- 손가락 하나로 전체 셀 인식 가능
- 빠른 읽기 속도 (분당 100-200 단어)

## 1.3 점자의 세계적 확산

| 연도 | 사건 |
|------|------|
| 1829 | 브라유, 최초 점자 책 출판 |
| 1854 | 프랑스 공식 채택 |
| 1878 | 세계점자통일회의 (파리) |
| 1932 | Standard English Braille 확정 |
| 2018 | UN, 세계 점자의 날 지정 (1월 4일) |

### 현재 사용 현황

| 항목 | 수치 |
|------|------|
| 전 세계 시각장애인 | 약 2억 8,500만 명 |
| 점자 사용자 | 약 1,000만 명 |
| 점자 체계 수 | 133개 이상 |

# 2. 기존 점자 체계의 한계

## 2.1 국가별 분리된 체계

### 문제: 133개 서로 다른 점자

같은 점형의 의미가 다름:
- 영어 점자: h
- 한글 점자: ㅎ (초성)
- 일본 점자: は (ha)
- 아랍 점자: ح (ha)

다른 언어 점자를 배우려면 처음부터 다시 시작해야 함

## 2.2 학습의 어려움

### 기존 점자 학습 시간

- 영어 점자 (Grade 2): 1-3년
- 한글 점자: 6개월-1년
- 일본 점자: 1-2년
- 중국어 점자: 2-3년

### 문제의 원인

1. 축약 규칙 과다 (영어: 2,000개 이상)
2. 예외 규칙 과다
3. 표준화 부재

## 2.3 장애인 간 소통 단절

### 현재 상황

- 시각장애인: 점자 사용
- 청각장애인: 수어 사용
- 서로 다른 체계로 직접 소통 불가능

### 농맹인(Deafblind)의 고립

- 전 세계 농맹인: 약 50만 명
- 기존: 촉수화, 점자 통역 (매우 느림)
- 필요: 통합된 소통 체계

# 3. IPA 기반 통합의 학술적 근거

## 3.1 IPA (국제음성기호)란?

**IPA (International Phonetic Alphabet)**: 모든 인간 언어의 소리를 표기하는 국제 표준

| 항목 | 내용 |
|------|------|
| 창설 | 1888년, 국제음성학회 |
| 목적 | 모든 언어 발음의 통일 표기 |
| 기호 수 | 107개 문자 + 52개 구별기호 |

### IPA의 보편성

어떤 언어든 IPA로 표기 가능:
- 영어 "hello" → /həˈloʊ/
- 한국어 "안녕" → /annjʌŋ/
- 일본어 "こんにちは" → /konnitɕiwa/
- 중국어 "你好" → /nǐ hǎo/

## 3.2 IPA 기반 점자의 논리

### 기존 점자의 문제

기존: 문자 → 점자 (133개 분리된 체계)

### WIA Braille의 해결

WIA: 문자 → IPA → 점자 (하나의 통합 체계)

IPA가 매개(허브) 역할

## 3.3 5대 동등 표현 원칙

> **"ALL FIVE ARE EQUAL. NO DEFAULT EXISTS."**
> (다섯 표현 모두 동등하다. 기본은 존재하지 않는다.)

| 표현 | 대상 사용자 | WIA 시스템 |
|------|------------|-----------|
| 문자 | 일반인 | 입력 |
| IPA | AI/언어학자 | 중앙 허브 |
| 점자 | 시각장애인 | WIA Braille |
| 제스처 | 청각장애인 | WIA Talk (ISP) |
| 음성 | 일반인 | TTS/STT |

# 4. 8점 점자의 수학적 완전성

## 4.1 6점에서 8점으로

### 확장의 필요성

- 6점 점자: 2의6승 = 64 조합
- IPA 기호: 107 + 52 = 159개
- 64 < 159 → 부족!

### 해결: 8점으로 확장

- 8점 점자: 2의8승 = 256 조합
- 256 > 159 → 충분!

### 8점 점자 구조

6점 (브라유 원형): 2열 × 3행
8점 (WIA 확장): 2열 × 4행

번호 재배치로 하위 호환성 유지

## 4.2 수학적 증명

### 정리 1: 충분성

- IPA 전체 기호: 159개
- 8점 점자 조합: 256개
- 256 - 159 = 97개 여유
- 모든 IPA 기호 표현 가능

### 정리 2: 효율성

기존 점자:
- 영어 "through" → 7셀 (철자 기반) 또는 1셀 (축약, 규칙 암기 필요)

WIA Braille:
- "through" /θruː/ → 3셀 (발음 기반)
- 규칙 암기 불필요

### 정리 3: 학습 용이성

- 기존 점자: 1-3년
- WIA Braille: 1-3개월
- 학습 시간 90% 단축

## 4.3 8점 점자의 역사적 선례

이미 존재하는 8점 체계:
- 컴퓨터 점자 (ASCII 256자)
- NABCC (미국)
- EuroBraille (유럽)

WIA Braille의 차별점:
- 기존 8점: 컴퓨터 코드용
- WIA Braille: IPA 표현용 (언어 통합 목적)
- 최초의 "언어 통합" 8점 체계

# 5. WIA Braille의 혁신

## 5.1 핵심 혁신 요약

| 항목 | 기존 점자 | WIA Braille |
|------|----------|-------------|
| 기반 | 문자 (철자) | IPA (발음) |
| 체계 수 | 133개 분리 | 1개 통합 |
| 점 수 | 6점 (64 조합) | 8점 (256 조합) |
| 학습 시간 | 1-3년 | 1-3개월 |
| 언어 커버 | 각각 1개 | 211개 동시 |
| 수어 연동 | 없음 | ISP 완전 연동 |

## 5.2 WIA Talk (ISP) 연동

### 시각장애인-청각장애인 소통

기존: 시각장애인 → (통역사) → 청각장애인 (직접 소통 불가)

WIA 시스템:
- 시각장애인 (점자) → IPA → 제스처 (청각장애인)
- 청각장애인 (제스처) → ISP → IPA → 점자 (시각장애인)
- 직접 소통 가능! (역사상 최초)

## 5.3 농맹인(Deafblind) 지원

기존: 촉수화, 점자 통역 (매우 느림)

WIA 방법:
- 점자 디스플레이로 WIA Braille 입력
- ISP로 변환
- 상대방에게 출력
- 실시간 양방향 소통
- 통역사 불필요

# 7. 법적 준수 논리

## 7.1 장애인차별금지법 (장차법)

### 한국 장차법 제21조

장애인이 정보에 동등하게 접근할 수 있도록 필요한 수단을 제공하여야 한다.

### WIA Braille 적용

Q: WIA Braille이 "점자"로 인정되나요?

A: 장차법은 "특정 점자 제공"이 아닌 "접근성 제공"을 요구합니다.
   WIA Braille은 IPA 기반으로 기존 점자와 호환되며,
   "추가 접근성 옵션"입니다.
   한국 사용자 → 한글 점자 + WIA Braille
   따라서 장차법 취지에 완전히 부합합니다.

## 7.2 국제 법규

### EU Accessibility Act (2025)

- 시행: 2025년 6월 28일
- 대상: EU 내 디지털 서비스
- WIA Braille: 단일 API로 27개 EU 국가 커버

### ADA (Americans with Disabilities Act)

- 현황: 웹 접근성 소송 급증 (2023년 4,000건+)
- WIA Braille: IPA 기반 국제 표준 준수

### UN CRPD

- 제2조: 점자를 의사소통 형태로 인정
- 제9조: 접근성 보장 의무
- 비준국: 189개국
- WIA Braille: 211개 언어 지원으로 글로벌 접근성

## 7.3 법적 방어 논리

핵심: 장차법/EU/ADA/CRPD 모두 "접근성 제공"을 요구하지 "특정 점자만 제공"을 요구하지 않음

WIA Braille:
- 기존 점자 대체 X
- 추가 접근성 옵션 O
- 211개 언어 확장 O
- 수어(ISP) 연동 O

오히려 접근성 "확대" → 법적 리스크 감소

# 결론

1. 200년 역사 기반: 루이 브라유(1824) + IPA(1888) 융합
2. 수학적 완전성: 8점 256조합으로 IPA 159기호 완전 표현

3. 법적 준수: 장차법, EU, ADA, CRPD 완전 부합

## 최종 선언

루이 브라유가 6점으로 문맹을 해방했듯,
WIA Braille이 IPA로 언어 장벽을 허문다.

시각장애인과 청각장애인이 처음으로 직접 소통하고,
