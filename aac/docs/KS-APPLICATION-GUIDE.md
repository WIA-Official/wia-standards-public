# KS 표준 신청 가이드
## WIA AAC Standard → 대한민국 KS 표준

---

**작성일**: 2025년 12월 14일  
**신청 예정**: 2025년 12월 15일 (1381 전화 상담)

---

## 📋 KS 표준 신청 절차

### 1단계: 사전 상담 (12월 15일 예정)
```
전화: 1381 (국가기술표준원)
문의 내용:
1. 보완대체의사소통(AAC) 관련 기존 KS 표준 유무
2. 신규 표준 제안 절차
3. 필요 서류 목록
4. 담당 부서/담당자 연결
```

### 2단계: 표준 제안서 작성

#### 제안서 구성
```
1. 표준명 (국문/영문)
   - 보완대체의사소통 센서 인터페이스 프로토콜
   - Augmentative and Alternative Communication 
     Sensor Interface Protocol (WIA AAC)

2. 제안 배경 및 필요성
   - AAC 시장 현황 (파편화 문제)
   - 국제 동향 (표준 부재)
   - 국내 장애인 복지 정책 연계

3. 표준 개요
   - Phase 1: 신호 형식 표준
   - Phase 2: API 인터페이스 표준
   - Phase 3: 통신 프로토콜 표준
   - Phase 4: 생태계 연동 표준

4. 적용 범위
   - AAC 센서 제조사
   - AAC 소프트웨어 개발사
   - 의료/복지 기기 제조사

5. 기대 효과
   - 기기 간 호환성 확보
   - 사용자 재학습 부담 감소
   - 국내 AAC 산업 경쟁력 강화

6. 국제 표준화 계획
   - ISO 국제 표준 제안 연계
   - EU EVS 표준 병행 추진
```

### 3단계: 서류 준비

#### 필수 서류
```
□ 표준 제안서 (위 내용)
□ 표준(안) 전문 (spec 문서 4종)
   - PHASE-1-SIGNAL-FORMAT.md
   - PHASE-2-API-INTERFACE.md
   - PHASE-3-PROTOCOL.md
   - PHASE-4-INTEGRATION.md
□ 단체/기업 소개서
   - SmileStory Inc. 회사 소개
   - WIA (World Certification Industry Association)
□ 사업자등록증 사본
□ 관련 특허/저작권 현황
   - MIT 라이선스 (오픈소스)
   - 특허 없음 선언
```

#### 추가 서류 (있으면 유리)
```
□ 기술 검증 결과
   - TypeScript SDK 테스트 결과
   - Python SDK 테스트 결과
□ 시제품/데모
   - https://aac.wia.live 웹사이트
   - GitHub 저장소
□ 해외 기관 협력 증빙
   - 에스토니아 wia.family 법인
□ 산업계 지지 의견
   - 관련 기업 의견서 (있으면)
```

### 4단계: 제출 및 심의

```
1. 온라인 제출
   - e-나라표준인증 (https://standard.go.kr)
   - 또는 담당 부서 직접 제출

2. 기술심의위원회 심의
   - 기술적 타당성 검토
   - 산업 적용 가능성 평가

3. 예고고시
   - 60일간 이해관계자 의견 수렴

4. 최종 심의 및 제정
   - KS 표준 번호 부여
   - 관보 고시
```
