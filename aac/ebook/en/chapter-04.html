4.2 Base Signal Structure

4.2.1 Common Fields

Every WIA AAC signal, regardless of sensor type, contains these base fields:

{
    "type": "string",           // REQUIRED: Sensor type identifier
    "version": "string",        // REQUIRED: Signal format version
    "timestamp": number,        // REQUIRED: Unix timestamp (ms)
    "data": object,             // REQUIRED: Sensor-specific data
    "confidence": number,       // REQUIRED: 0.0-1.0
    "device": {                 // OPTIONAL: Device information
        "manufacturer": "string",
        "model": "string",
        "device_id": "string",
        "firmware_version": "string"
    },
    "session": {                // OPTIONAL: Session information
        "session_id": "string",
        "user_profile": "string",
        "calibration_id": "string"
    },
    "metadata": object          // OPTIONAL: Additional data
}

4.2.2 Field Specifications

Field Type Required Description
type string Yes One of: eye_tracker, switch, emg, eeg_bci, breath, head_movement
version string Yes Semantic version (e.g., "1.0.0")
timestamp integer Yes Unix timestamp in milliseconds (UTC)
data object Yes Sensor-specific data payload
confidence number Yes Signal quality score 0.0-1.0
device object No Hardware identification
session object No Session context
metadata object No Extension point for custom data

4.3.2 Coordinate System

Screen Coordinate System:

(0,0)                                 (1,0)
    +-----------------------------------+
    |                                   |
    |                                   |
    |                                   |
    |               (0.5,0.5)           |
    |                   +               |
    |                                   |
    |                                   |
    |                                   |
    +-----------------------------------+
(0,1)                                 (1,1)

- Origin (0,0) is top-left corner
- X increases left to right
- Y increases top to bottom
- All values normalized 0.0 to 1.0
- Resolution independent

4.3.3 Fixation Detection

Fixation States:

fixation: true
  - User is looking at a stable point
  - dwell_time_ms tracks duration
  - Suitable for dwell-based selection

fixation: false
  - User is scanning/moving gaze
  - saccade_detected may be true
  - Do not trigger selections

Dwell Time Thresholds (Recommendations):
  - Quick selection: 400-600ms
  - Standard selection: 800-1000ms
  - Careful selection: 1200-1500ms
  - Configurable by user preference

4.4.2 Multi-Switch Configurations

Common Switch Configurations:

1-Switch (Scanning):
  switch_id: 1 = Select/Activate

2-Switch:
  switch_id: 1 = Move/Next
  switch_id: 2 = Select

4-Switch (Directional):
  switch_id: 1 = Up
  switch_id: 2 = Down
  switch_id: 3 = Left
  switch_id: 4 = Right

5-Switch (Directional + Select):
  switch_id: 1-4 = Directions
  switch_id: 5 = Select

8-Switch (Full):
  switch_id: 1-4 = Directions
  switch_id: 5 = Select
  switch_id: 6 = Back/Cancel
  switch_id: 7 = Menu
  switch_id: 8 = Special/Custom

4.4.3 Timing Events

Event Type Detection:

single (default):
  - Press and release within 500ms
  - Standard activation

double:
  - Two presses within 400ms
  - Alternative action

long:
  - Press held for 800ms+
  - Special function (e.g., menu)

release:
  - Explicit release event
  - For hold-and-release interfaces

4.5.2 Gesture Recognition

Standard Gesture Types:

Facial Gestures:
  - eyebrow_raise
  - eyebrow_furrow
  - smile
  - frown
  - jaw_clench
  - cheek_puff
  - nose_wrinkle

Hand/Arm Gestures (if applicable):
  - finger_tap
  - wrist_flex
  - wrist_extend
  - grip
  - release

Gesture Detection Notes:
  - gesture_detected is OPTIONAL
  - Simple threshold-based detection uses activation only
  - Advanced pattern recognition provides gesture_detected
  - Confidence score reflects gesture certainty

4.6.2 EEG Channel Positions (10-20 System)

Standard 10-20 Electrode Positions:

          Nasion
            |
    Fp1 --- Fpz --- Fp2
     |       |       |
    F7--F3--Fz--F4--F8
     |   |   |   |   |
    T3--C3--Cz--C4--T4
     |   |   |   |   |
    T5--P3--Pz--P4--T6
     |       |       |
    O1 ---- Oz ---- O2
            |
          Inion

Common Subsets:
  - Minimal (4): Fp1, Fp2, O1, O2
  - Basic (8): + F3, F4, P3, P4
  - Standard (14): + C3, C4, T3, T4, T5, T6
  - Full (21+): All positions

4.7.2 Breath Patterns

Standard Breath Patterns:

Basic Actions:
  - single_sip: One sip action
  - single_puff: One puff action
  - hard_sip: High intensity sip
  - hard_puff: High intensity puff
  - soft_sip: Low intensity sip
  - soft_puff: Low intensity puff

Combined Patterns:
  - sip_sip: Two sips (double-tap equivalent)
  - puff_puff: Two puffs
  - sip_puff: Sip followed by puff
  - puff_sip: Puff followed by sip

Intensity Mapping (Example):
  - soft: 0.0 - 0.3
  - medium: 0.3 - 0.7
  - hard: 0.7 - 1.0

4.8.2 Rotation Reference

Head Rotation Axes:

         Pitch (nodding)
              |
              |    / Yaw (shaking head)
              |   /
              |  /
              | /
    Roll -----+------- (tilting head)
    (ear to
    shoulder)

Pitch: Positive = looking up, Negative = looking down
Yaw: Positive = turning right, Negative = turning left
Roll: Positive = tilting right, Negative = tilting left

Typical Ranges:
  - Pitch: -30 to +30 degrees
  - Yaw: -45 to +45 degrees
  - Roll: -20 to +20 degrees
