弘益人間 - TOS는 자동화 항만의 두뇌로, 글로벌 상업과 인류의 연결된 필요를 섬기기 위해 매일 수천 건의 운영을 조율합니다.

항만 운영의 지휘 센터

터미널 운영 시스템(TOS)은 선박 계획부터 게이트 거래까지 터미널 운영의 모든 측면을 조율하는 현대 항만의 중추 신경계입니다. 정교한 TOS는 매일 수백만 개의 데이터 포인트를 처리하며, 생산성을 최적화하고 비용을 최소화하며 터미널을 통한 원활한 화물 흐름을 보장하는 실시간 결정을 내립니다.

한국 항만의 TOS 혁신

부산항과 인천항은 세계 최고 수준의 TOS 시스템을 운영합니다:

  • AI 기반 의사결정: 기계학습으로 선박 입출항 시간 예측 (95% 정확도)
  • 실시간 최적화: 5초마다 모든 장비 재할당
  • 통합 플랫폼: 해운사, 세관, 육상운송과의 원활한 연동
  • 디지털 트윈: 가상 환경에서 운영 시뮬레이션
  • 블록체인 통합: 투명한 공급망 추적

핵심 TOS 모듈

  • 선박 계획: 최적의 접안 위치, 적재 순서, 자원 할당
  • 야드 관리: 컨테이너 저장 위치 관리, 공간 활용 최적화
  • 장비 제어: AGV, ASC, 안벽 크레인 실시간 지시
  • 게이트 운영: 트럭 거래 처리, 문서 검증
  • 철도 운영: 해상과 철도 운송 간 복합 운송 조정
// AI 기반 TOS 결정 엔진
class TOSAIEngine {
  // 선박 체류 시간 예측
  predictVesselTurnaround(vessel: VesselInfo): number {
    const features = {
      vesselSize: vessel.capacity,
      containersToDischarge: vessel.dischargeList.length,
      containersToLoad: vessel.loadList.length,
      timeOfDay: new Date().getHours(),
      weatherConditions: this.getCurrentWeather(),
      equipmentAvailability: this.getEquipmentAvailability()
    };

    return this.mlModel.predict(features);
  }

  // 강화 학습을 사용한 선석 할당 최적화
  async optimizeBerthAllocation(vessels: VesselInfo[]): Promise<BerthAllocation> {
    const state = this.getCurrentState();
    const action = await this.rlAgent.selectAction(state);
    const reward = this.calculateReward(action);
    await this.rlAgent.learn(state, action, reward);

    return action.berthAllocation;
  }
}

주요 성능 지표

  • 선박 체류 시간: 도착부터 출발까지 평균 시간
  • 선석 생산성: 선석당 시간당 컨테이너 이동
  • 야드 활용률: 사용 중인 야드 용량 비율
  • 트럭 회전 시간: 트럭 거래의 평균 시간
  • 장비 활용률: 장비가 생산적으로 사용되는 시간 비율
  • 재정돈 비율: 비생산적인 컨테이너 이동 횟수
