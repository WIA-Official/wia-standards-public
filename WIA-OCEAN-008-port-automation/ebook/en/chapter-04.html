The Command Center of Port Operations

The Terminal Operating System (TOS) is the central nervous system of modern ports, orchestrating every aspect of terminal operations from vessel planning to gate transactions. A sophisticated TOS processes millions of data points daily, making real-time decisions that optimize productivity, minimize costs, and ensure seamless cargo flow through the terminal.

In automated ports, the TOS takes on even greater importance, directly controlling AGVs, cranes, and other equipment while coordinating with external systems including shipping lines, customs, trucking companies, and rail operators.

Core TOS Modules

1. Vessel Planning

Determines optimal berthing positions, loading sequences, and resource allocation for incoming vessels.

2. Yard Management

Manages container storage locations, optimizes space utilization, and minimizes reshuffling operations.

3. Equipment Control

Directs automated equipment including AGVs, ASCs, and quay cranes in real-time.

4. Gate Operations

Processes truck transactions, validates documentation, and manages terminal entry/exit.

5. Rail Operations

Coordinates intermodal transfers between maritime and rail transportation.

// Equipment Control Module
class EquipmentControlModule {
  private agvFleet: AGVFleet;
  private craneSystem: CraneSystem;
  private taskQueue: PriorityQueue<OperationalTask>;

  // Assign task to optimal equipment
  async assignTask(task: OperationalTask): Promise<Equipment> {
    // Determine task priority
    const priority = this.calculateTaskPriority(task);

    // Add to queue
    this.taskQueue.enqueue(task, priority);

    // Find available equipment
    const equipment = await this.findOptimalEquipment(task);

    // Assign and execute
    await equipment.executeTask(task);

    return equipment;
  }

  private calculateTaskPriority(task: OperationalTask): number {
    let priority = 50; // Base priority

    // Vessel operations have highest priority
    if (task.type === 'vessel_operation') {
      priority += 40;
    }

    // Time-sensitive cargo
    if (task.container?.perishable) {
      priority += 20;
    }

    // Hazardous materials need special handling
    if (task.container?.hazardous) {
      priority += 15;
    }

    // Consider task deadline
    if (task.deadline) {
      const timeUntilDeadline = task.deadline.getTime() - Date.now();
      const hoursRemaining = timeUntilDeadline / 3600000;

      if (hoursRemaining < 2) {
        priority += 30;
      } else if (hoursRemaining < 6) {
        priority += 15;
      }
    }

    return priority;
  }

  // Real-time optimization
  async optimizeOperations(): Promise<void> {
    setInterval(async () => {
      // Get current system state
      const state = await this.getCurrentState();

      // Run optimization algorithms
      const optimizations = await this.calculateOptimizations(state);

      // Apply non-disruptive improvements
      await this.applyOptimizations(optimizations);
    }, 5000); // Every 5 seconds
  }
}

// AI-Powered TOS Decision Engine
class TOSAIEngine {
  private historicalData: HistoricalDataStore;
  private mlModels: Map<string, MLModel>;

  // Predict vessel turnaround time
  predictVesselTurnaround(vessel: VesselInfo): number {
    const features = {
      vesselSize: vessel.capacity,
      containersToDischarge: vessel.dischargeList.length,
      containersToLoad: vessel.loadList.length,
      timeOfDay: new Date().getHours(),
      dayOfWeek: new Date().getDay(),
      weatherConditions: this.getCurrentWeather(),
      equipmentAvailability: this.getEquipmentAvailability()
    };

    const model = this.mlModels.get('vessel_turnaround');
    return model.predict(features);
  }

  // Predict container dwell time
  predictDwellTime(container: ContainerInfo): number {
    const model = this.mlModels.get('dwell_time');

    return model.predict({
      destination: container.destination,
      shipper: container.shipper,
      cargoType: container.cargoType,
      seasonality: this.getSeasonalityFactor(),
      historicalPattern: this.getHistoricalPattern(container)
    });
  }

  // Optimize berth allocation using reinforcement learning
  async optimizeBerthAllocation(vessels: VesselInfo[]): Promise<BerthAllocation> {
    // State: current berth occupancy, pending vessels, equipment status
    const state = this.getCurrentState();

    // Use RL agent to find optimal allocation
    const action = await this.rlAgent.selectAction(state);

    // Apply action and observe reward
    const reward = this.calculateReward(action);

    // Update model
    await this.rlAgent.learn(state, action, reward);

    return action.berthAllocation;
  }
}

  }

      gateNumber: appointment.gate

}

Performance Metrics

Key performance indicators tracked by modern TOS:

  • Vessel Turnaround Time: Average time from arrival to departure
  • Berth Productivity: Container moves per hour per berth
  • Yard Utilization: Percentage of yard capacity used
  • Truck Turn Time: Average time for truck transaction
  • Equipment Utilization: Percentage of time equipment is productively used
  • Rehandle Ratio: Number of unproductive container moves
