Chapter 2: Automated Guided Vehicles (AGVs)

弘益人間 - AGVs eliminate dangerous horizontal container transport, protecting human workers while enabling efficient 24/7 operations that serve global trade.

The Backbone of Horizontal Transport

Automated Guided Vehicles (AGVs) represent the circulatory system of modern automated ports. These autonomous vehicles transport containers horizontally between quay cranes at the waterside and automated stacking cranes in the storage yard. Unlike traditional terminal trucks operated by human drivers, AGVs navigate autonomously using sophisticated sensor arrays and positioning systems, operating continuously without fatigue, breaks, or shift changes.

A typical automated port deploys 50-150 AGVs working in coordination, each capable of carrying containers weighing up to 60 tons. These vehicles travel at speeds up to 6 meters per second (about 21 km/h), precisely navigating complex port environments while maintaining safety margins of just centimeters.

AGV Technology Components

1. Navigation Systems

Modern AGVs employ multiple navigation technologies for redundancy and precision:

  • Magnetic Guidance: Embedded magnetic markers in the pavement provide primary path guidance
  • LiDAR (Light Detection and Ranging): 360-degree laser scanners detect obstacles and environment features
  • GPS/GNSS: Satellite positioning for global coordinates and backup navigation
  • Inertial Measurement Units (IMUs): Gyroscopes and accelerometers track movement and orientation
  • Computer Vision: Cameras provide visual information for enhanced situational awareness

2. Power Systems

Modern AGVs use advanced battery technology that balances performance, longevity, and environmental sustainability:

  • Lithium-ion battery packs providing 8-16 hours of continuous operation
  • Automated charging stations enabling opportunity charging during idle periods
  • Battery swapping systems for 24/7 operations without downtime
  • Regenerative braking systems that recover energy during deceleration

3. Safety Systems

AGV safety is paramount in busy port environments:

  • Multiple redundant emergency stop systems
  • 360-degree obstacle detection with configurable safety zones
  • Collision avoidance algorithms using predictive path analysis
  • Speed reduction in proximity to personnel or obstacles
  • Audible and visual warning systems
  • Fail-safe protocols that bring vehicles to controlled stops in case of system failures

AGV Traffic Management

Coordinating dozens of AGVs requires sophisticated traffic management systems that prevent collisions, optimize routes, and maximize throughput: 

// Traffic Management System
class AGVTrafficController {
  private agvs: Map<string, AGV>;
  private roadNetwork: RoadNetwork;
  private reservationSystem: ZoneReservationSystem;

  // Assign optimal route to AGV
  assignRoute(agv: AGV, destination: Coordinates): Route {
    // Calculate all possible routes
    const routes = this.roadNetwork.findRoutes(
      agv.location.coordinates,
      destination
    );

    // Score routes based on multiple factors
    const scoredRoutes = routes.map(route => ({
      route,
      score: this.scoreRoute(route, agv)
    }));

    // Select best route
    const bestRoute = scoredRoutes.reduce((best, current) =>
      current.score > best.score ? current : best
    ).route;

    // Reserve road segments
    this.reservationSystem.reservePath(bestRoute, agv.vehicleId);

    return bestRoute;
  }

  private scoreRoute(route: Route, agv: AGV): number {
    let score = 100;

    // Penalize longer routes
    score -= route.distance * 0.1;

    // Penalize routes with traffic
    const congestion = this.calculateCongestion(route);
    score -= congestion * 5;

    // Prefer routes with charging stations if battery low
    if (agv.battery.chargeLevel < 30) {
      const hasCharging = route.segments.some(s =>
        s.hasChargingStation
      );
      score += hasCharging ? 20 : 0;
    }

    // Penalize routes requiring zone changes
    score -= route.zoneChanges * 3;

    return score;
  }

  // Collision avoidance
  detectPotentialCollisions(): CollisionWarning[] {
    const warnings: CollisionWarning[] = [];
    const vehicles = Array.from(this.agvs.values());

    for (let i = 0; i < vehicles.length; i++) {
      for (let j = i + 1; j < vehicles.length; j++) {
        const prediction = this.predictCollision(
          vehicles[i],
          vehicles[j],
          10 // seconds ahead
        );

        if (prediction.willCollide) {
          warnings.push({
            vehicle1: vehicles[i].vehicleId,
            vehicle2: vehicles[j].vehicleId,
            timeToCollision: prediction.timeSeconds,
            location: prediction.collisionPoint,
            severity: prediction.severity
          });
        }
      }
    }

    return warnings;
  }
}
