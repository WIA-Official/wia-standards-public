## 1. 개요

### 1.1 배경

현대인은 하루의 **90% 이상**을 실내에서 보내고 있으며, 실내 공기질(Indoor Air Quality, IAQ)은 건강, 웰빙, 생산성에 직접적인 영향을 미칩니다. 그러나 많은 실내 공간에서 공기질이 적절히 관리되지 않고 있습니다.

**주요 통계 (2024):**
- 실내 오염도: 실외보다 **2~5배** 높을 수 있음
- 건강 영향: 호흡기 질환, 알레르기, 천식의 주요 원인
- 경제적 영향: 생산성 저하로 인한 연간 **$160억** 손실
- 사망 원인: 실내 공기 오염으로 연간 **380만 명** 조기 사망

### 1.2 문제점

현재 실내 공기질 관리의 주요 문제점:

1. **측정 부재**: 대부분의 건물에서 공기질 측정 미실시
2. **표준 부족**: 측정 방법, 데이터 형식의 표준화 부족
3. **실시간 관리 미흡**: 수동 측정, 사후 대응 중심
4. **HVAC 비효율**: 공기질 고려 없는 환기/공조 운영
5. **건강 영향 무시**: 공기질과 건강의 연관성 인식 부족
6. **데이터 분절**: 센서, 시스템 간 연동 부재

### 1.3 WIA-ENE-027의 가치

이 표준은 다음을 제공합니다:

- ✅ **통합 측정 프레임워크**: PM, 가스, VOC, 라돈 등 포괄
- ✅ **표준화된 데이터 형식**: 센서 간 상호운용성
- ✅ **실시간 모니터링**: 지속적 측정 및 경보
- ✅ **스마트 HVAC 연동**: 공기질 기반 자동 제어
- ✅ **건강 영향 평가**: 과학적 위험 분석
- ✅ **규정 준수**: 국내외 기준 자동 검증

## 5. 측정 파라미터

### 5.1 미세먼지 (Particulate Matter)

#### PM2.5 (초미세먼지)
```
정의: 공기역학적 직경 ≤ 2.5 μm
측정 단위: μg/m³
측정 방법: 광산란, 베타선 흡수

WHO 기준 (2021):
  - 연평균: 5 μg/m³
  - 24시간: 15 μg/m³

건강 영향:
  - 폐 깊숙이 침투 (폐포)
  - 혈류 유입 가능
  - 심혈관계 질환, 폐암, 뇌졸중

주요 출처:
  - 외부 유입 (교통, 산업)
  - 요리 (튀김, 구이)
  - 흡연
  - 청소 활동
```

#### PM10 (미세먼지)
```
정의: 공기역학적 직경 ≤ 10 μm
측정 단위: μg/m³

WHO 기준 (2021):
  - 연평균: 15 μg/m³
  - 24시간: 45 μg/m³

건강 영향:
  - 호흡기 자극
  - 기침, 천식 악화
  - 만성 기관지염

주요 출처:
  - 외부 유입
  - 먼지 재비산
  - 건축 활동
```

#### PM1.0 (극초미세먼지) - 선택
```
정의: 공기역학적 직경 ≤ 1 μm
특징: 가장 위험, 폐포-혈류 경계 통과
측정: 고급 센서에서 지원
```

### 5.2 가스 농도

#### CO₂ (이산화탄소)
```
측정 단위: ppm (parts per million)
측정 방법: NDIR (비분산적외선)

기준:
  - 대기 중: ~420 ppm
  - 우수: < 600 ppm
  - 양호: 600-800 ppm
  - 보통: 800-1,000 ppm
  - 나쁨: 1,000-1,500 ppm
  - 매우 나쁨: > 1,500 ppm

건강 영향:
  - 800 ppm: 집중력 미세 저하
  - 1,000 ppm: 졸음, 피로감
  - 1,500 ppm: 의사결정 능력 10% 저하
  - 2,500 ppm: 두통, 불쾌감
  - 5,000 ppm: 8시간 노출 한계

주요 출처:
  - 인체 호흡 (주)
  - 환기 부족
  - 재실 밀도 높음
```

#### CO (일산화탄소)
```
측정 단위: ppm
측정 방법: 전기화학 센서

기준:
  - EPA: 9 ppm (8시간), 35 ppm (1시간)
  - WHO: 10 mg/m³ (8시간), 30 mg/m³ (1시간)

건강 영향:
  - 헤모글로빈과 결합 → 산소 운반 방해
  - 두통, 현기증, 구토
  - 고농도: 의식 불명, 사망

주요 출처:
  - 가스레인지, 보일러
  - 벽난로, 난로
  - 주차장 (차량 배기가스)
```

#### NO₂ (이산화질소)
```
측정 단위: ppb (parts per billion) 또는 μg/m³
측정 방법: 전기화학 센서

기준:
  - WHO: 40 μg/m³ (연평균), 200 μg/m³ (1시간)
  - 1 ppb = 1.88 μg/m³ (25°C, 1 atm)

건강 영향:
  - 기도 염증
  - 천식 악화
  - 감염 감수성 증가

주요 출처:
  - 가스레인지
  - 보일러, 난방기기
  - 외부 교통 유입
```

#### O₃ (오존)
```
측정 단위: ppb 또는 μg/m³
측정 방법: UV 흡수

기준:
  - WHO: 100 μg/m³ (8시간 평균)
  - 1 ppb = 1.96 μg/m³

건강 영향:
  - 폐 조직 손상
  - 호흡 곤란
  - 천식 악화

주요 출처:
  - 프린터, 복사기
  - 공기청정기 (일부)
  - 외부 유입
```

### 5.3 휘발성유기화합물 (VOCs)

#### TVOC (총휘발성유기화합물)
```
측정 단위: μg/m³ 또는 ppb
측정 방법: PID (광이온화검출기), 금속산화물 센서

등급:
  - 우수: < 220 μg/m³
  - 양호: 220-660 μg/m³
  - 보통: 660-1,200 μg/m³
  - 나쁨: 1,200-2,200 μg/m³
  - 매우 나쁨: > 2,200 μg/m³

건강 영향:
  - 눈, 코, 목 자극
  - 두통, 현기증
  - 피로, 메스꺼움
  - 장기 노출: 간, 신장, 중추신경계 손상

주요 출처:
  - 페인트, 접착제
  - 가구, 건축 자재
  - 세제, 방향제
  - 프린터, 복사기
```

#### 포름알데히드 (Formaldehyde)
```
화학식: HCHO
측정 단위: μg/m³ 또는 ppb
측정 방법: 전기화학 센서, 광도법

기준:
  - WHO: 100 μg/m³ (30분 평균)
  - EPA: 0.1 ppm (장기)

건강 영향:
  - WHO 분류: Group 1 발암물질
  - 눈, 코, 목 자극
  - 천식 악화
  - 비인두암 위험 증가

주요 출처:
  - 합판, MDF, 파티클보드
  - 가구, 캐비닛
  - 섬유제품 (커튼, 카펫)
```

#### 벤젠, 톨루엔, 자일렌 (BTX)
```
벤젠 (C₆H₆):
  - WHO: 발암물질, 안전 한계 없음
  - 출처: 페인트, 용제, 흡연

톨루엔 (C₇H₈):
  - 기준: 260 μg/m³ (주간 평균)
  - 출처: 페인트, 잉크, 세제

자일렌 (C₈H₁₀):
  - 기준: 870 μg/m³ (24시간)
  - 출처: 페인트, 코팅, 접착제
```

### 5.4 라돈 (Radon)

```
화학 기호: Rn (원자번호 86)
특성: 무색, 무취, 방사성 기체
측정 단위: Bq/m³ (베크렐/세제곱미터) 또는 pCi/L (피코퀴리/리터)
변환: 37 Bq/m³ = 1 pCi/L

측정 방법:
  - 단기 (2-7일): 활성탄 캐니스터
  - 장기 (90일~1년): 알파트랙 검출기
  - 연속: 전자식 라돈 모니터

기준:
  - EPA: 4 pCi/L (148 Bq/m³) 조치 수준
  - WHO: 100 Bq/m³ (2.7 pCi/L) 권장
  - 한국: 148 Bq/m³ (신축 건물)

건강 영향:
  - 폐암 2위 원인 (흡연 다음)
  - 연간 폐암 사망의 3-14% 기여
  - DNA 손상 (알파 입자 방출)

위험도:
  - < 100 Bq/m³: 안전
  - 100-200 Bq/m³: 저감 권장
  - 200-400 Bq/m³: 저감 필수
  - > 400 Bq/m³: 즉시 조치

주요 출처:
  - 토양, 암반 (우라늄 붕괴)
  - 건축 자재 (화강암, 콘크리트)
  - 지하수

취약 공간:
  - 지하층, 1층
  - 균열, 틈새 많은 건물
  - 환기 불량 공간
```

### 5.5 환경 조건

#### 온도
```
측정 단위: °C (섭씨)
측정 방법: 온도 센서 (열전대, RTD, 서미스터)

쾌적 범위 (ASHRAE 55):
  - 겨울 (난방): 20-24°C
  - 여름 (냉방): 23-26°C
  - 상대습도 40-60% 가정

건강 영향:
  - < 18°C: 저체온증 위험, 심혈관계 부담
  - > 28°C: 열 스트레스, 탈수

에너지:
  - 1°C 상승/하강: 에너지 사용 5-10% 증감
```

#### 상대습도
```
측정 단위: % (퍼센트)
측정 방법: 습도 센서 (정전용량식, 저항식)

최적 범위: 40-60%

건강 영향:
  - < 30%: 피부 건조, 눈 자극, 정전기
  - 30-40%: 바이러스 생존율 증가
  - 60-70%: 곰팡이, 진드기 증식
  - > 70%: 결로, 부패, 악취

주요 출처:
  - 호흡, 발한 (습도 증가)
  - 요리, 샤워 (습도 증가)
  - 에어컨, 제습기 (습도 감소)
```

#### 기압
```
측정 단위: hPa (헥토파스칼) 또는 mbar
표준 기압: 1013.25 hPa (해수면)

영향:
  - 날씨 변화 예측
  - 환기 효율
  - 라돈 유입 (기압 낮을 때 증가)
```

### 5.6 환기율

```
ACH (Air Changes per Hour):
  - 정의: 시간당 실내 공기 전체가 교체되는 횟수
  - 계산: ACH = (환기량 m³/h) / (실내 체적 m³)

권장 기준:
  - 주거: 0.35-1.0 ACH
  - 사무실: 2-4 ACH
  - 교실: 3-4 ACH
  - 병원 병실: 6 ACH
  - 수술실: 15-25 ACH

외기 공급량 (ASHRAE 62.1):
  - 사무실: 10 L/s/person
  - 교실: 8 L/s/person
  - 회의실: 10 L/s/person

측정 방법:
  - CO₂ 감쇠법
  - 추적 가스 (SF₆, N₂O)
  - 도어 블로워 테스트
```

## 6. 센서 및 측정 방법

### 6.1 센서 분류

#### Tier 1: 고정밀 참조 센서
```
용도: 규제, 인증, 연구
정확도: ±5%
가격: $5,000-$50,000

예시:
  - PM: 베타선 흡수 (BAM), TEOM
  - CO₂: NDIR (고급)
  - VOC: GC-MS, GC-FID
```

#### Tier 2: 전문가급 센서
```
용도: 건물 관리, HVAC 연동
정확도: ±10%
가격: $500-$5,000

예시:
  - PM: 레이저 산란
  - CO₂: NDIR (중급)
  - VOC: PID
```

#### Tier 3: 소비자급 센서
```
용도: 개인 모니터링, 인식 제고
정확도: ±20%
가격: $50-$500

예시:
  - PM: LED 광산란
  - CO₂: NDIR (저가)
  - VOC: 금속산화물 (MOx)
```

### 6.2 측정 방법

#### 미세먼지 (PM)

**광산란 방식 (Light Scattering)**
```
원리: 입자에 레이저 조사 → 산란광 측정
장점: 실시간, 저비용
단점: 입자 밀도/형상 가정, 습도 영향

종류:
  - LED 산란: 저가, 낮은 정확도
  - 레이저 산란: 중가, 중간 정확도
```

**베타선 흡수 (Beta Attenuation)**
```
원리: PM 포집 → 베타선 흡수량 측정
장점: 고정밀, 규제 인정
단점: 고가, 큰 크기, 1시간 단위

사용: 대기질 측정소 (참조용)
```

#### CO₂

**NDIR (Non-Dispersive Infrared)**
```
원리: CO₂의 4.26 μm 적외선 흡수 측정
장점: 정확, 안정적, 수명 길음
단점: 정기 교정 필요

등급:
  - 고급: ±30 ppm, 자동 교정
  - 중급: ±50 ppm, 수동 교정
  - 저급: ±100 ppm, 교정 불가
```

#### VOC

**PID (Photo-Ionization Detector)**
```
원리: UV 광으로 VOC 이온화 → 전류 측정
장점: 빠른 응답, ppb 수준 감도
단점: 개별 VOC 구분 불가

측정: TVOC (총량)
```

**금속산화물 센서 (MOx)**
```
원리: VOC 산화 → 저항 변화
장점: 저가, 소형
단점: 습도/온도 영향, 드리프트

사용: 소비자급 센서
```

### 6.3 센서 설치

#### 위치 선정
```
높이:
  - PM, VOC: 호흡 높이 (1.2-1.5 m)
  - CO₂: 1.5 m (약간 높게)
  - CO: 1.5 m (CO는 공기와 비슷한 밀도)
  - 라돈: 0.5-1.0 m (무거움)

위치:
  - 중앙 또는 대표 지점
  - 오염원에서 최소 1 m 떨어짐
  - 환기구 직접 영향 피함
  - 햇빛 직사 피함
```

#### 교정 및 유지보수
```
교정 주기:
  - CO₂: 6-12개월
  - PM: 12개월 (참조 센서 비교)
  - VOC: 12개월

유지보수:
  - 청소: 3-6개월
  - 필터 교체: 필요 시
  - 펌웨어 업데이트: 정기
```

### 7.2 IAQ 지수 계산

```
IAQ = max(IAQ_PM2.5, IAQ_CO2, IAQ_VOC, IAQ_other)

  0-50: 우수 (Excellent)
  51-100: 양호 (Good)
  101-150: 보통 (Moderate)
  151-200: 나쁨 (Poor)
  201-300: 매우 나쁨 (Very Poor)
```

## 9. HVAC 연동

### 9.1 제어 전략

#### CO₂ 기반 환기 제어 (DCV - Demand Controlled Ventilation)
```
목표: CO₂ < 1,000 ppm 유지

제어 로직:
  IF CO₂ > 1,200 ppm THEN
    환기량 = 최대
  ELSE IF CO₂ > 1,000 ppm THEN
    환기량 = 현재 + 20%
  ELSE IF CO₂ < 800 ppm THEN
    환기량 = 현재 - 10% (최소 유지)
  ELSE
    환기량 = 현재

에너지 절감: 20-30% (불필요한 환기 방지)
```

#### PM2.5 기반 공기청정 제어
```
목표: PM2.5 < 15 μg/m³ 유지

제어 로직:
  IF PM2.5 > 35 μg/m³ THEN
    공기청정기 = 최대
    외기 도입 = 최소 (외부 오염 높을 때)
  ELSE IF PM2.5 > 15 μg/m³ THEN
    공기청정기 = 중간
  ELSE
    공기청정기 = 자동 또는 꺼짐
```

### 9.2 필터 관리

```
교체 기준:
  1. 사용 시간: 2,000-4,000 시간
  2. 압력 차이: 초기 대비 2배 증가
  3. 효율 저하: 초기 대비 30% 감소

모니터링:
  - 실시간 압력 차이 측정
  - 상류/하류 PM 농도 비교
  - 누적 사용 시간 추적

알림:
  - 교체 30일 전: 사전 알림
  - 교체 기한: 긴급 알림
```

### 10.1 위험도 평가

```
위험 수준:
  - LOW: 건강한 사람에게 영향 없음
  - MODERATE: 민감군에 영향 가능
  - HIGH: 일반인에게 영향
  - VERY_HIGH: 모든 사람에게 심각한 영향

민감군:
  - 어린이 (12세 이하)
  - 노인 (65세 이상)
  - 임산부
  - 천식, 심혈관계 질환자
```

### 10.2 건강 권고

```
PM2.5 > 35 μg/m³:
  - 실외 활동 제한
  - 창문 닫기
  - 공기청정기 가동

CO₂ > 1,500 ppm:
  - 즉시 환기
  - 재실자 수 제한
  - 휴식 시간 증가

라돈 > 148 Bq/m³:
  - 환기 강화
  - 균열 보수
  - 라돈 저감 시스템 설치
```
