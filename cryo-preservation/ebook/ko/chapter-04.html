제4장: 냉각 프로파일과 유리화
================================================================================--------------------------------------------------------------------------------
4.1 온도 제어 기본
--------------------------------------------------------------------------------

성공적인 유리화는 여러 임계 온도 구간을 통한 정밀한 냉각 속도 제어가
필요합니다. 너무 빠르면 열응력과 파열이 발생하고, 너무 느리면 빙정
형성이 허용됩니다.

[CRYO] 임계 온도 구간:

    +37 C  |  정상 체온 - 생물학적 과정 활성
           |
     +4 C  |  저체온 - 대사 억제 (~10배 느림)
           |
      0 C  |  위험: 수용액에서 빙정 핵 형성 시작
           |
    -40 C  |  위험: 균질 빙정 핵 형성 구간
           |          최대 빙정 형성 위험
           |
   -130 C  |  유리 전이(Tg) - 용액이 유리가 됨
           |  Tg 이하: 무기한 안정, 생물학적 변화 없음
           |
   -196 C  |  액체 질소 - 저장 온도
           |

[i] 냉각 속도 딜레마:

    너무 느림 (< 0.1 C/분):
    +------------------------------------------------------------------+
    | - 빙정이 핵을 형성하고 성장할 시간이 있음                        |
    | - 미동결 액체에서 용질 농도 증가                                 |
    | - 세포에 삼투 손상                                               |
    | - 빙정으로 인한 기계적 손상                                      |
    +------------------------------------------------------------------+

    너무 빠름 (> 50 C/분, 큰 질량의 경우):
    +------------------------------------------------------------------+
    | - 온도 구배가 응력 생성                                          |
    | - 차별적 수축이 파열 유발                                        |
    | - 표면이 중심보다 빠르게 냉각                                    |
    | - 응력 파열이 조직을 통해 전파될 수 있음                         |
    +------------------------------------------------------------------+

    최적: 빙정 방지와 열응력 사이의 균형

다른 대상자 유형은 다른 프로파일이 필요합니다.

--------------------------------------------------------------------------------
4.3 유리화 vs 완속 동결
--------------------------------------------------------------------------------

냉동보존에는 두 가지 근본적으로 다른 접근법이 있으며, 각각 특정
응용과 요구사항이 있습니다.

[CRYO] 비교:

    +------------------------------------------------------------------------+
    | 측면             | 완속 동결            | 유리화                      |
    +------------------------------------------------------------------------+
    | 빙정 형성        | 제어된 외부 형성     | 없음 (유리 상태)            |
    | CPA 농도         | 낮음 (1-2 M)         | 높음 (6-10 M)               |
    | 냉각 속도        | -1 C/분              | 가변, 제어됨                |
    | 장비             | 프로그램 냉동기      | 특수 냉각 시스템            |
    | 독성 위험        | 낮음                 | 높음 (CPA 더 많음)          |
    | 빙정 손상 위험   | 높음                 | 낮음 (빙정 없음)            |
    | 최적 용도        | 세포, 단순 조직      | 복잡한 조직, 장기           |
    +------------------------------------------------------------------------+

[i] 완속 동결 원리:

    세포 외부에서 제어된 빙정 형성:

    세포  [    H2O    ]     외부에서 형성되는 빙정이
     |    |           |     물을 끌어냄
     |    | H2O  -->  |  --> [빙정]
     |    |           |
     |    [  용질    ]     세포가 수축하지만 생존

    요구사항:
    - 막 보호를 위한 저농도 CPA
    - 느린 냉각 속도 (전형적으로 -1 C/분)
    - 제어된 핵 형성 (시딩)
    - 해동도 제어되어야 함

[i] 유리화 원리:

    빙정 형성 없음 - 직접 유리 전이:

    세포  [  CPA + H2O  ]     전체 용액이
     |    |             |     유리가 됨
     |    | 유리화됨    |  --> [유리]
     |    |             |
     |    [   CPA       ]     구조 보존

    요구사항:
    - 고농도 CPA (유리화 가능 용액)
    - 빙정 핵 형성 없이 Tg를 통과하는 냉각
    - 파열 방지 (제어된 속도)
    - 가온도 동등하게 제어되어야 함


--------------------------------------------------------------------------------
4.4 유리 전이 모니터링
--------------------------------------------------------------------------------

유리 전이를 감지하고 확인하는 것은 성공적인 유리화 확인에
필수적입니다.

[CRYO] 유리 전이 지표:

    1. 온도
       - -130 C에 접근하는 중심 온도 모니터링
       - 용액별 Tg는 조성에 따라 다름
       - M22: Tg = -123 C

    2. 점도
       - Tg에서 점도가 극적으로 증가
       - Tg 이하에서 흐름 완전 정지
       - 지표: 관류 압력 급등

    3. 광학
       - 유리화된 조직은 맑음/반투명
       - 빙정 형성은 불투명/백화 유발
       - 듀어 포트를 통한 시각적 검사

    4. 음향
       - 유리 전이에서 음속 변화
       - 초음파 모니터링 가능
       - 연구 응용

[CRYO] 유리 전이 감지 알고리즘:

    class GlassTransitionDetector:
        def __init__(self):
            self.temperature_readings = []
            self.viscosity_readings = []
            self.tg_detected = False
            self.tg_temperature = None
            self.tg_time = None

        def analyze(self, temp, viscosity, timestamp):
            self.temperature_readings.append((timestamp, temp))
            self.viscosity_readings.append((timestamp, viscosity))

            # Tg를 나타내는 점도 급등 감지
            if len(self.viscosity_readings) >= 3:
                recent = [v for t, v in self.viscosity_readings[-3:]]

                # >10배 점도 증가 확인
                if recent[-1] > recent[0] * 10:
                    # 예상 Tg 범위 내 온도 확인
                    if -140 < temp < -120:
                        self.tg_detected = True
                        self.tg_temperature = temp
                        self.tg_time = timestamp
                        return self.generate_tg_event()

            return None

        def generate_tg_event(self):
            return {
                'event': 'GLASS_TRANSITION_DETECTED',
                'temperature': self.tg_temperature,
                'timestamp': self.tg_time,
                'confidence': self.calculate_confidence()
            }

[!] 실패한 유리화 징후:

    - 불투명 또는 백화 (빙정 형성)
    - 냉각 중 균열 소리
    - 불규칙한 온도 프로파일
    - 검사 시 눈에 보이는 파열


--------------------------------------------------------------------------------
4.5 열응력과 파열
--------------------------------------------------------------------------------

성공적인 유리화가 되더라도 냉각 속도를 신중하게 관리하지 않으면
열응력이 파열을 유발할 수 있습니다.

[CRYO] 열응력 요인:

    +------------------------------------------------------------------------+
    | 요인               | 효과                  | 완화                      |
    +------------------------------------------------------------------------+
    | 온도 구배          | 차별적 응력           | 느린 냉각 속도            |
    | 질량/크기          | 클수록 = 더 많은 응력 | 연장된 냉각 시간          |
    | 형상               | 불규칙한 형태         | 균일한 냉각               |
    | 열전도율           | 재료에 따라 다름      | 프로파일 조정             |
    +------------------------------------------------------------------------+

[i] 열응력 계산:

    응력 = E * alpha * deltaT

    여기서:
    - E = 유리화된 조직의 탄성 계수
    - alpha = 열팽창 계수
    - deltaT = 조직 간 온도 차이

    전신의 경우:
    - 최대 안전 구배: 조직 간 ~5 C
    - Tg 근처에서 냉각 속도 < 0.5 C/분 필요

[CRYO] 파열 방지 프로토콜:

    1. Tg에 느린 접근
       - -100 C부터 냉각 속도 감소
       - 목표: -100에서 -130 C까지 0.3 C/분
       - 열적 평형화 허용

    2. 어닐링 유지
       - -130 C에서 30-60분 선택적 유지
       - 응력 이완 허용
       - 최종 하강 시 위험 감소

    3. 제어된 최종 하강
       - -130에서 -196 C까지 점진적 냉각
       - 속도: 최대 1.0 C/분
       - 음향 방출(균열) 모니터링

    4. 위치 최적화
       - 냉각 중 대상자 방향 중요
       - 온도 구배 최소화
       - 열전도성 패딩 사용

[CRYO] 파열 모니터링:

    음향 방출 모니터링
    +------------------------------------------------------------------+
    | 센서 배치:                                                       |
    |   - 용기에 여러 압전 센서                                        |
    |   - 파열 이벤트에서 초음파 방출 감지                             |
    |                                                                  |
    | 경보 임계값:                                                     |
    |   - 단일 이벤트 > 60 dB: 경고                                    |
    |   - 여러 이벤트 > 40 dB: 냉각 속도 감소                          |
    |   - 지속적 방출: 비상 프로토콜                                   |
    +------------------------------------------------------------------+


--------------------------------------------------------------------------------
4.6 냉각 장비 및 시스템
--------------------------------------------------------------------------------

전체 온도 범위를 통한 제어된 냉각에는 특수 장비가 필요합니다.

[CRYO] 장비 범주:

    프로그램 냉동기
    +------------------------------------------------------------------+
    | 범위: +20 ~ -80 C                                                |
    | 속도 제어: 0.1 ~ 10 C/분                                         |
    | 용도: 완속 동결, 초기 유리화 냉각                                |
    | 예: Planer Kryo 시리즈, CryoMed                                  |
    +------------------------------------------------------------------+

    제어 속도 냉각기 (유리화)
    +------------------------------------------------------------------+
    | 범위: +4 ~ -130 C                                                |
    | 속도 제어: 0.1 ~ 5 C/분                                          |
    | 특징: 다중 구역 제어, 열 균일성                                  |
    | 용도: 장기 및 전신 유리화                                        |
    +------------------------------------------------------------------+

    중간 저장
    +------------------------------------------------------------------+
    | 범위: -80 ~ -150 C                                               |
    | 용도: 이송 중 유지, 어닐링                                       |
    | 유형: 기계식 냉동기, 기상 LN2                                    |
    +------------------------------------------------------------------+

    액체 질소 시스템
    +------------------------------------------------------------------+
    | 온도: -196 C                                                     |
    | 용도: 장기 저장                                                  |
    | 유형: 침지 듀어, 기상 듀어                                       |
    +------------------------------------------------------------------+

[i] 센서 요구사항:

    온도 센서:
    - 유형: PT100 RTD 또는 열전대 (Type T 또는 K)
    - 정확도: +/- 0.1 C
    - 응답 시간: < 1초
    - 배치: 중심(가능한 경우), 표면, 용기

    대상자당 최소 센서 수:
    - 세포 배양: 1 용기 센서
    - 장기: 2 센서 (표면 + 중심)
    - 전신: 6+ 센서 (두부, 흉부, 복부, 사지)
