6.1 유리화 품질 지수(VQI)
--------------------------------------------------------------------------------

유리화 품질 지수(VQI)는 냉동보존 절차의 품질을 평가하는 표준화된
지표입니다. 시설과 프로토콜 간에 객관적이고 비교 가능한 측정을
제공합니다.

[CRYO] VQI 개요:

    VQI 범위: 0.0 ~ 1.0

    +------------------------------------------------------------------------+
    | 점수        | 등급 | 해석                                             |
    +------------------------------------------------------------------------+
    | 0.90 - 1.00 | A    | 우수 - 최적의 보존 달성                          |
    | 0.75 - 0.89 | B    | 양호 - 경미한 편차, 높은 신뢰도                  |
    | 0.60 - 0.74 | C    | 허용 - 일부 우려, 모니터링 필요                  |
    | 0.40 - 0.59 | D    | 불량 - 중대한 문제, 검토 필요                    |
    | 0.00 - 0.39 | F    | 실패 - 주요 문제, 개선 필요                      |
    +------------------------------------------------------------------------+

[CRYO] VQI 구성요소:

    VQI는 4개의 가중 구성요소로 계산됩니다:

    +----------------------------------+
    |        VQI 계산                  |
    +----------------------------------+
    |                                  |
    |  빙정 분율 점수       x 0.35     |
    |  +                               |
    |  CPA 분포 점수        x 0.25     |
    |  +                               |
    |  냉각 속도 점수       x 0.20     |
    |  +                               |
    |  구조적 무결성        x 0.20     |
    |  ================================|
    |  = 최종 VQI 점수                 |
    |                                  |
    +----------------------------------+

6.2 빙정 분율 평가
--------------------------------------------------------------------------------

빙정 분율은 냉각 중 형성된 빙정의 양을 측정합니다. 성공적인 유리화를
위해 빙정 분율은 0에 접근해야 합니다.

[CRYO] 빙정 분율 측정:

    방법:
    1. DSC (시차 주사 열량계)
       - 연구를 위한 표준
       - 가온 중 열 흐름 측정
       - 결정화 이벤트 감지

    2. 시각적 평가
       - 불투명도가 빙정 형성 표시
       - 맑음 = 유리화, 흐림 = 빙정
       - 초기 스크리닝에 유용

    3. X-ray/CT 이미징
       - 비파괴 평가
       - 밀도 변화가 빙정 표시
       - 전신 평가에 사용

[CRYO] 빙정 분율 점수화:

    빙정 분율 %   |  점수   |  평가
    ----------------+---------+-------------------
         0%         |  1.00   |  완벽한 유리화
       < 1%         |  0.95   |  우수
       1-5%         |  0.80   |  양호
       5-15%        |  0.50   |  부분 유리화
      15-30%        |  0.25   |  상당한 빙정
       > 30%        |  0.00   |  유리화 실패

[CRYO] 빙정 분율 계산:

    def calculate_ice_fraction_score(ice_percentage):
        """
        측정된 빙정 분율을 VQI 구성요소 점수로 변환.

        Args:
            ice_percentage: 측정된 빙정 분율 (0-100)

        Returns:
            0.0에서 1.0 사이의 점수
        """
        if ice_percentage <= 0:
            return 1.0
        elif ice_percentage < 1:
            return 0.95 + (1 - ice_percentage) * 0.05
        elif ice_percentage < 5:
            return 0.80 + (5 - ice_percentage) / 4 * 0.15
        elif ice_percentage < 15:
            return 0.50 + (15 - ice_percentage) / 10 * 0.30
        elif ice_percentage < 30:
            return 0.25 * (30 - ice_percentage) / 15
        else:
            return 0.0

6.3 CPA 분포 평가
--------------------------------------------------------------------------------

균일한 CPA 분포는 조직 전체에 일관된 보호를 보장합니다. 분포가
불량하면 취약한 영역이 생깁니다.

[CRYO] 분포 측정:

    방법:
    1. 유출액 분석
       - 동맥 vs 정맥 CPA 농도 비교
       - 델타가 분포 효율성 표시
       - 관류 중 실시간

    2. 조영제 MRI
       - 조직 내 CPA 분포 시각화
       - 관류 후 평가
       - 연구 응용

    3. 조직 샘플링
       - 여러 위치에서 생검 샘플
       - 굴절률 측정
       - 파괴적이지만 확정적

[CRYO] 분포 점수화:

    동맥-정맥 델타     |  점수   |  평가
    -----------------------+---------+-------------------
          < 5%             |  1.00   |  우수한 평형화
         5-10%             |  0.85   |  양호한 분포
        10-20%             |  0.65   |  허용 가능
        20-35%             |  0.40   |  불량한 침투
          > 35%            |  0.10   |  분포 실패

[i] 분포 요인:

    CPA 분포에 영향을 미치는 요인:

    1. 관류 압력
       - 높은 압력 = 더 나은 침투
       - 너무 높으면 부종 위험
       - 최적: 전신의 경우 60-80 mmHg

    2. 관류 시간
       - 긴 시간 = 더 나은 평형화
       - 시간이 길면 독성 증가
       - 균형 필요

    3. 혈관 접근
       - 캐뉼레이션 품질
       - 혈관 개통성
       - 측부 순환

    4. 조직 유형
       - 뇌는 더 긴 관류 필요
       - 간은 빠르게 평형화
       - 지방 조직은 어려움

6.4 냉각 속도 평가
--------------------------------------------------------------------------------

냉각 속도 평가는 실제 냉각이 목표 프로파일을 얼마나 잘 따랐는지
평가합니다.

[CRYO] 속도 편차 분석:

    목표 프로파일:

    시간  | 목표 (C) | 실제 (C) | 편차
    ------+------------+------------+-----------
    0 분  |     0      |     0      |    0.0
   30 분  |   -15      |   -14      |   +1.0
   60 분  |   -30      |   -32      |   -2.0
   90 분  |   -45      |   -44      |   +1.0
  120 분  |   -60      |   -58      |   +2.0
  ...

    평균 제곱근 편차 (RMSD):
    RMSD = sqrt(sum(편차^2) / n)

[CRYO] 냉각 속도 점수화:

    RMSD (C)    |  점수   |  평가
    ------------+---------+-------------------
       < 0.5    |  1.00   |  우수한 제어
     0.5-1.0    |  0.90   |  매우 양호
     1.0-2.0    |  0.75   |  양호
     2.0-5.0    |  0.50   |  허용 가능
     5.0-10.0   |  0.25   |  불량한 제어
       > 10.0   |  0.00   |  제어 실패

[CRYO] 임계 구간 가중치:

    모든 온도 구간이 동등하게 중요하지 않습니다. 점수화 알고리즘은
    구간별 가중치를 적용합니다:

    구간                  | 가중치 | 이유
    ----------------------+--------+-----------------------------
    0 ~ -40 C             |  1.5x  | 빙정 핵 형성 위험
    -40 ~ -100 C          |  1.0x  | 전이 구간
    -100 ~ -130 C         |  2.0x  | 유리 전이 임계
    -130 ~ -196 C         |  1.2x  | 파열 위험 구간

    가중 RMSD = sum(가중치_i * 편차_i^2) / sum(가중치_i)

6.5 구조적 무결성 평가
--------------------------------------------------------------------------------

구조적 무결성은 파열, 찢김, 변형을 포함한 보존 과정으로 인한
물리적 손상을 평가합니다.

[CRYO] 무결성 평가 방법:

    1. 시각적 검사
       +--------------------------------------------------------------+
       | - 관찰 포트를 통한 표면 검사                                 |
       | - 여러 각도에서 사진 촬영                                    |
       | - 보존 전 이미지와 비교                                      |
       +--------------------------------------------------------------+

    2. 음향 모니터링
       +--------------------------------------------------------------+
       | - 냉각 중 이벤트 기록                                        |
       | - 파열 이벤트가 특징적인 소리 생성                           |
       | - 이벤트의 수와 심각도 정량화                                |
       +--------------------------------------------------------------+

    3. 이미징 (비파괴)
       +--------------------------------------------------------------+
       | - 극저온 온도에서 CT 스캔                                    |
       | - MRI (특수 장비 필요)                                       |
       | - 표면 평가를 위한 초음파                                    |
       +--------------------------------------------------------------+

[CRYO] 무결성 점수화:

    발견                       |  점수 영향     |  비고
    ---------------------------+----------------+------------------
    눈에 보이는 손상 없음      |  1.00 기준     |  이상적 결과
    경미한 표면 자국           | -0.05          |  미용적 문제만
    작은 파열 (< 1cm)          | -0.15 각       |  최대 3개 계산
    큰 파열 (> 1cm)            | -0.25 각       |  상당함
    깊은/관통 파열             | -0.40 각       |  심각한 손상
    구조적 변형                | -0.30          |  형태 변화

    최소 점수: 0.00

[i] 파열 분류:

    유형 A: 표면 크레이즈
    +--------------------------------------------------------------+
    | 미세한 표면 균열 네트워크                                    |
    | 깊이: < 1mm                                                  |
    | 영향: 최소, 미용적 우려                                      |
    +--------------------------------------------------------------+

    유형 B: 얕은 파열
    +--------------------------------------------------------------+
    | 조직으로 확장되는 눈에 보이는 균열                           |
    | 깊이: 1-10mm                                                 |
    | 영향: 중간, 소생 성공에 영향 가능                            |
    +--------------------------------------------------------------+

    유형 C: 깊은 파열
    +--------------------------------------------------------------+
    | 상당한 조직 깊이로 확장되는 파열                             |
    | 깊이: > 10mm 또는 구조 관통                                  |
    | 영향: 심각, 구조적 손상 우려                                 |
    +--------------------------------------------------------------+

6.6 VQI 계산 및 인증
--------------------------------------------------------------------------------

완전한 VQI 계산은 모든 구성요소 점수를 적절한 가중치로 통합합니다.

[CRYO] VQI 계산 알고리즘:

    class VQICalculator:
        WEIGHTS = {
            'ice_fraction': 0.35,
            'cpa_distribution': 0.25,
            'cooling_rate': 0.20,
            'structural_integrity': 0.20
        }

        def calculate_vqi(self, assessment_data):
            """
            구성요소 평가에서 완전한 VQI 점수 계산.

            Args:
                assessment_data: 구성요소 측정값이 있는 Dict

            Returns:
                점수, 등급, 분석이 있는 VQIResult
            """
            # 구성요소 점수 계산
            ice_score = self.score_ice_fraction(
                assessment_data['ice_percentage']
            )

            cpa_score = self.score_cpa_distribution(
                assessment_data['arterial_conc'],
                assessment_data['venous_conc']
            )

            cooling_score = self.score_cooling_rate(
                assessment_data['temperature_profile'],
                assessment_data['target_profile']
            )

            integrity_score = self.score_integrity(
                assessment_data['inspection_findings']
            )

            # 가중 VQI 계산
            vqi = (
                ice_score * self.WEIGHTS['ice_fraction'] +
                cpa_score * self.WEIGHTS['cpa_distribution'] +
                cooling_score * self.WEIGHTS['cooling_rate'] +
                integrity_score * self.WEIGHTS['structural_integrity']
            )

            return VQIResult(
                score=round(vqi, 3),
                grade=self.score_to_grade(vqi),
                components={
                    'ice_fraction': ice_score,
                    'cpa_distribution': cpa_score,
                    'cooling_rate': cooling_score,
                    'structural_integrity': integrity_score
                },
                timestamp=datetime.utcnow(),
                assessor=assessment_data['assessor_id']
            )

        def score_to_grade(self, score):
            if score >= 0.90: return 'A'
            if score >= 0.75: return 'B'
            if score >= 0.60: return 'C'
            if score >= 0.40: return 'D'
            return 'F'
