3.1 냉동보호제 소개
--------------------------------------------------------------------------------

냉동보호제(CPA)는 유리화를 가능하게 하는 화학 물질입니다. 조직 내
물을 대체하고 유리 상태 보존을 촉진하여 빙정 형성을 방지합니다.

[CRYO] 빙정 형성 문제:

    CPA 없이:
    +------------------------------------------------------------------+
    | 온도 하락 --> 물 분자 정렬 --> 빙정                              |
    |                                                                  |
    | 빙정:                                                            |
    |   - 팽창하여 세포막 파열                                         |
    |   - 기계적 응력 파열 생성                                        |
    |   - 용질이 독성 수준으로 농축                                    |
    |   - 조직 초미세 구조 파괴                                        |
    +------------------------------------------------------------------+

    CPA 사용:
    +------------------------------------------------------------------+
    | CPA가 물 대체 --> 점성 증가 --> 유리 상태                        |
    |                                                                  |
    | 유리화:                                                          |
    |   - 빙정 형성 없음                                               |
    |   - 분자 구조 보존                                               |
    |   - 세포 무결성 유지                                             |
    |   - 무기한 안정 저장 가능                                        |
    +------------------------------------------------------------------+

[i] 핵심 개념 - 유리 전이 온도(Tg):
    CPA 용액이 액체에서 유리 상태로 전이하는 온도로,
    표준 용액의 경우 약 -130 C입니다.

3.4 CPA 농도 모니터링
--------------------------------------------------------------------------------

관류액과 유출액의 CPA 농도를 실시간으로 모니터링하는 것은
프로토콜 검증에 필수적입니다.

[CRYO] 모니터링 파라미터:

    +------------------------------------------------------------------------+
    | 파라미터             | 목표           | 경보 임계값                    |
    +------------------------------------------------------------------------+
    | 동맥 농도            | 프로토콜별     | >5% 편차                       |
    | 정맥 농도            | 추적           | 동맥보다 >10% 낮음             |
    | 굴절률               | 용액 사양      | >0.001 편차                    |
    | 관류 압력            | 60-80 mmHg     | >100 mmHg (전신)               |
    | 유속                 | 프로토콜별     | >15% 편차                      |
    | 온도                 | 단계별         | >1 C 편차                      |
    +------------------------------------------------------------------------+

[i] 굴절률 방법:

    CPA 농도는 굴절률과 상관관계:

    굴절률 (20C)
    |
    1.45 |                    *  (100% M22)
    1.43 |                *
    1.41 |            *
    1.39 |        *
    1.37 |    *
    1.35 | *                      (0% CPA - 운반용액만)
         +----------------------------> CPA 농도 %

    공식: 농도 = (RI - 1.3330) / 0.00115

[CRYO] 농도 모니터링 코드:

    class CPAMonitor:
        def __init__(self, target_protocol):
            self.protocol = target_protocol
            self.readings = []

        def add_reading(self, timestamp, arterial_ri, venous_ri,
                       flow_rate, pressure, temperature):
            arterial_conc = self.ri_to_concentration(arterial_ri)
            venous_conc = self.ri_to_concentration(venous_ri)

            reading = {
                'timestamp': timestamp,
                'arterial_concentration': arterial_conc,
                'venous_concentration': venous_conc,
                'concentration_delta': arterial_conc - venous_conc,
                'flow_rate': flow_rate,
                'pressure': pressure,
                'temperature': temperature
            }

            self.readings.append(reading)
            self.check_alerts(reading)
            return reading

        def ri_to_concentration(self, ri):
            return (ri - 1.3330) / 0.00115

        def check_alerts(self, reading):
            expected = self.protocol.expected_at(reading['timestamp'])

            if abs(reading['arterial_concentration'] - expected) > 5.0:
                self.raise_alert('CONCENTRATION_DEVIATION', reading)

            if reading['pressure'] > 100:
                self.raise_alert('PRESSURE_HIGH', reading)

3.5 독성 관리
--------------------------------------------------------------------------------

CPA는 본질적으로 고농도에서 독성이 있습니다. 프로토콜 설계는
빙정 보호와 CPA 독성 사이의 균형을 맞춰야 합니다.

[!] 독성 요인:

    1. 농도
       높은 농도 = 더 많은 보호 but 더 많은 독성
       최적: 완전 유리화를 위한 최소 농도

    2. 온도
       저온에서 독성 감소
       전략: 최저 온도에서 최고 농도 도입

    3. 노출 시간
       긴 노출 = 더 많은 세포 손상
       전략: 고농도 노출 시간 최소화

    4. 조성
       특정 CPA 조합은 감소된 독성 표시
       전략: 다성분 용액 사용 (예: M22)

[CRYO] 온도-농도 프로토콜:

    온도         |  최대 안전 CPA  |  노출 시간
    ---------------------------------------------------
       +4 C      |     25%        |   60분
        0 C      |     40%        |   45분
       -3 C      |     60%        |   30분
       -7 C      |     80%        |   20분
      -10 C      |    100%        |   15분

[i] 독성 완화 전략:

    냉각 관류:
    +------------------------------------------------------------------+
    | 4 C에서 시작, CPA 도입 중 점진적 냉각                            |
    | 유리 전이 근처에서 최종 농도 달성                                |
    +------------------------------------------------------------------+

    운반용액 최적화:
    +------------------------------------------------------------------+
    | 보호 첨가제를 포함한 운반용액 사용:                              |
    | - 항산화제 (비타민 E, CoQ10)                                     |
    | - 막 안정화제 (트레할로스)                                       |
    | - pH 완충제 (HEPES, 중탄산염)                                    |
    +------------------------------------------------------------------+

    빠른 전환:
    +------------------------------------------------------------------+
    | 중간 독성 농도에서의 시간 최소화                                 |
    | S자형 램핑으로 위험 구간 빠르게 통과                             |
    +------------------------------------------------------------------+
