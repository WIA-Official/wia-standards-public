6.1 VITRIFICATION QUALITY INDEX (VQI)
--------------------------------------------------------------------------------

The Vitrification Quality Index (VQI) is a standardized metric for
assessing the quality of cryopreservation procedures. It provides an
objective, comparable measure across facilities and protocols.

[CRYO] VQI OVERVIEW:

    VQI Range: 0.0 to 1.0

    +------------------------------------------------------------------------+
    | Score       | Grade | Interpretation                                   |
    +------------------------------------------------------------------------+
    | 0.90 - 1.00 | A     | Excellent - Optimal preservation achieved        |
    | 0.75 - 0.89 | B     | Good - Minor deviations, high confidence         |
    | 0.60 - 0.74 | C     | Acceptable - Some concerns, monitoring needed    |
    | 0.40 - 0.59 | D     | Poor - Significant issues, review required       |
    | 0.00 - 0.39 | F     | Failed - Major problems, remediation needed      |
    +------------------------------------------------------------------------+

[CRYO] VQI COMPONENTS:

    The VQI is calculated from four weighted components:

    +----------------------------------+
    |        VQI CALCULATION           |
    +----------------------------------+
    |                                  |
    |  Ice Fraction Score     x 0.35   |
    |  +                               |
    |  CPA Distribution Score x 0.25   |
    |  +                               |
    |  Cooling Rate Score     x 0.20   |
    |  +                               |
    |  Structural Integrity   x 0.20   |
    |  ================================|
    |  = FINAL VQI SCORE               |
    |                                  |
    +----------------------------------+

6.2 ICE FRACTION ASSESSMENT
--------------------------------------------------------------------------------

Ice fraction measures the amount of ice formation during cooling. For
successful vitrification, ice fraction should approach zero.

[CRYO] ICE FRACTION MEASUREMENT:

    Methods:
    1. DSC (Differential Scanning Calorimetry)
       - Gold standard for research
       - Measures heat flow during warming
       - Detects crystallization events

    2. Visual Assessment
       - Opacity indicates ice formation
       - Clear = vitrified, cloudy = ice
       - Useful for initial screening

    3. X-ray/CT Imaging
       - Non-destructive assessment
       - Density changes indicate ice
       - Used for whole body assessment

[CRYO] ICE FRACTION SCORING:

    Ice Fraction %  |  Score  |  Assessment
    ----------------+---------+-------------------
         0%         |  1.00   |  Perfect vitrification
       < 1%         |  0.95   |  Excellent
       1-5%         |  0.80   |  Good
       5-15%        |  0.50   |  Partial vitrification
      15-30%        |  0.25   |  Significant ice
       > 30%        |  0.00   |  Failed vitrification

[CRYO] ICE FRACTION CALCULATION:

    def calculate_ice_fraction_score(ice_percentage):
        """
        Convert measured ice fraction to VQI component score.

        Args:
            ice_percentage: Measured ice fraction (0-100)

        Returns:
            Score from 0.0 to 1.0
        """
        if ice_percentage <= 0:
            return 1.0
        elif ice_percentage < 1:
            return 0.95 + (1 - ice_percentage) * 0.05
        elif ice_percentage < 5:
            return 0.80 + (5 - ice_percentage) / 4 * 0.15
        elif ice_percentage < 15:
            return 0.50 + (15 - ice_percentage) / 10 * 0.30
        elif ice_percentage < 30:
            return 0.25 * (30 - ice_percentage) / 15
        else:
            return 0.0

6.3 CPA DISTRIBUTION ASSESSMENT
--------------------------------------------------------------------------------

Uniform CPA distribution ensures consistent protection throughout the
tissue. Poor distribution creates vulnerable areas.

[CRYO] DISTRIBUTION MEASUREMENT:

    Methods:
    1. Effluent Analysis
       - Compare arterial vs venous CPA concentration
       - Delta indicates distribution efficiency
       - Real-time during perfusion

    2. MRI with Contrast
       - Visualize CPA distribution in tissue
       - Post-perfusion assessment
       - Research applications

    3. Tissue Sampling
       - Biopsy samples from multiple locations
       - Refractive index measurement
       - Destructive but definitive

[CRYO] DISTRIBUTION SCORING:

    Arterial-Venous Delta  |  Score  |  Assessment
    -----------------------+---------+-------------------
          < 5%             |  1.00   |  Excellent equilibration
         5-10%             |  0.85   |  Good distribution
        10-20%             |  0.65   |  Acceptable
        20-35%             |  0.40   |  Poor penetration
          > 35%            |  0.10   |  Distribution failure

[i] DISTRIBUTION FACTORS:

    Factors affecting CPA distribution:

    1. PERFUSION PRESSURE
       - Higher pressure = better penetration
       - Risk of edema if too high
       - Optimal: 60-80 mmHg for whole body

    2. PERFUSION TIME
       - Longer time = better equilibration
       - Toxicity increases with time
       - Balance required

    3. VASCULAR ACCESS
       - Quality of cannulation
       - Vessel patency
       - Collateral circulation

    4. TISSUE TYPE
       - Brain requires longer perfusion
       - Liver equilibrates quickly
       - Fat tissue is challenging

6.4 COOLING RATE ASSESSMENT
--------------------------------------------------------------------------------

Cooling rate assessment evaluates how well the actual cooling followed
the target profile.

[CRYO] RATE DEVIATION ANALYSIS:

    Target Profile:

    Time  | Target (C) | Actual (C) | Deviation
    ------+------------+------------+-----------
    0 min |     0      |     0      |    0.0
   30 min |   -15      |   -14      |   +1.0
   60 min |   -30      |   -32      |   -2.0
   90 min |   -45      |   -44      |   +1.0
  120 min |   -60      |   -58      |   +2.0
  ...

    Root Mean Square Deviation (RMSD):
    RMSD = sqrt(sum(deviations^2) / n)

[CRYO] COOLING RATE SCORING:

    RMSD (C)    |  Score  |  Assessment
    ------------+---------+-------------------
       < 0.5    |  1.00   |  Excellent control
     0.5-1.0    |  0.90   |  Very good
     1.0-2.0    |  0.75   |  Good
     2.0-5.0    |  0.50   |  Acceptable
     5.0-10.0   |  0.25   |  Poor control
       > 10.0   |  0.00   |  Control failure

[CRYO] CRITICAL ZONE WEIGHTING:

    Not all temperature zones are equally important. The scoring
    algorithm applies zone-specific weights:

    Zone                  | Weight | Reason
    ----------------------+--------+-----------------------------
    0 to -40 C            |  1.5x  | Ice nucleation danger
    -40 to -100 C         |  1.0x  | Transition zone
    -100 to -130 C        |  2.0x  | Glass transition critical
    -130 to -196 C        |  1.2x  | Fracture risk zone

    Weighted RMSD = sum(weight_i * deviation_i^2) / sum(weight_i)

6.5 STRUCTURAL INTEGRITY ASSESSMENT
--------------------------------------------------------------------------------

Structural integrity evaluates physical damage from the preservation
process, including fractures, tears, and deformation.

[CRYO] INTEGRITY ASSESSMENT METHODS:

    1. VISUAL INSPECTION
       +--------------------------------------------------------------+
       | - Surface examination through viewing ports                  |
       | - Photography at multiple angles                             |
       | - Comparison to pre-preservation images                      |
       +--------------------------------------------------------------+

    2. ACOUSTIC MONITORING
       +--------------------------------------------------------------+
       | - Record events during cooling                               |
       | - Fracture events produce characteristic sounds              |
       | - Quantify number and severity of events                     |
       +--------------------------------------------------------------+

    3. IMAGING (Non-destructive)
       +--------------------------------------------------------------+
       | - CT scanning at cryogenic temperature                       |
       | - MRI (requires specialized equipment)                       |
       | - Ultrasound for surface assessment                          |
       +--------------------------------------------------------------+

[CRYO] INTEGRITY SCORING:

    Finding                    |  Score Impact  |  Notes
    ---------------------------+----------------+------------------
    No visible damage          |  1.00 base     |  Ideal outcome
    Minor surface marks        | -0.05          |  Cosmetic only
    Small fractures (< 1cm)    | -0.15 each     |  Max 3 counted
    Large fractures (> 1cm)    | -0.25 each     |  Significant
    Deep/through fractures     | -0.40 each     |  Severe damage
    Structural deformation     | -0.30          |  Shape change

    Minimum score: 0.00

[i] FRACTURE CLASSIFICATION:

    Type A: Surface Craze
    +--------------------------------------------------------------+
    | Fine network of surface cracks                               |
    | Depth: < 1mm                                                 |
    | Impact: Minimal, cosmetic concern                            |
    +--------------------------------------------------------------+

    Type B: Shallow Fracture
    +--------------------------------------------------------------+
    | Visible crack extending into tissue                          |
    | Depth: 1-10mm                                                |
    | Impact: Moderate, may affect revival success                 |
    +--------------------------------------------------------------+

    Type C: Deep Fracture
    +--------------------------------------------------------------+
    | Fracture extending through significant tissue depth          |
    | Depth: > 10mm or through structure                           |
    | Impact: Severe, structural damage concerns                   |
    +--------------------------------------------------------------+

6.6 VQI CALCULATION AND CERTIFICATION
--------------------------------------------------------------------------------

The complete VQI calculation integrates all component scores with
appropriate weighting.

[CRYO] VQI CALCULATION ALGORITHM:

    class VQICalculator:
        WEIGHTS = {
            'ice_fraction': 0.35,
            'cpa_distribution': 0.25,
            'cooling_rate': 0.20,
            'structural_integrity': 0.20
        }

        def calculate_vqi(self, assessment_data):
            """
            Calculate complete VQI score from component assessments.

            Args:
                assessment_data: Dict with component measurements

            Returns:
                VQIResult with score, grade, and breakdown
            """
            # Calculate component scores
            ice_score = self.score_ice_fraction(
                assessment_data['ice_percentage']
            )

            cpa_score = self.score_cpa_distribution(
                assessment_data['arterial_conc'],
                assessment_data['venous_conc']
            )

            cooling_score = self.score_cooling_rate(
                assessment_data['temperature_profile'],
                assessment_data['target_profile']
            )

            integrity_score = self.score_integrity(
                assessment_data['inspection_findings']
            )

            # Calculate weighted VQI
            vqi = (
                ice_score * self.WEIGHTS['ice_fraction'] +
                cpa_score * self.WEIGHTS['cpa_distribution'] +
                cooling_score * self.WEIGHTS['cooling_rate'] +
                integrity_score * self.WEIGHTS['structural_integrity']
            )

            return VQIResult(
                score=round(vqi, 3),
                grade=self.score_to_grade(vqi),
                components={
                    'ice_fraction': ice_score,
                    'cpa_distribution': cpa_score,
                    'cooling_rate': cooling_score,
                    'structural_integrity': integrity_score
                },
                timestamp=datetime.utcnow(),
                assessor=assessment_data['assessor_id']
            )

        def score_to_grade(self, score):
            if score >= 0.90: return 'A'
            if score >= 0.75: return 'B'
            if score >= 0.60: return 'C'
            if score >= 0.40: return 'D'
            return 'F'
