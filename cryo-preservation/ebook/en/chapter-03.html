3.1 INTRODUCTION TO CRYOPROTECTANTS
--------------------------------------------------------------------------------

Cryoprotectants (CPAs) are the chemical agents that make vitrification
possible. They prevent ice crystal formation by replacing water in tissues
and promoting glass-state preservation.

[CRYO] THE ICE FORMATION PROBLEM:

    Without CPA:
    +------------------------------------------------------------------+
    | Temperature drops --> Water molecules arrange --> ICE CRYSTALS   |
    |                                                                  |
    | Ice crystals:                                                    |
    |   - Expand and rupture cell membranes                           |
    |   - Create mechanical stress fractures                          |
    |   - Concentrate solutes to toxic levels                         |
    |   - Destroy tissue ultrastructure                               |
    +------------------------------------------------------------------+

    With CPA:
    +------------------------------------------------------------------+
    | CPA replaces water --> Viscosity increases --> GLASS STATE       |
    |                                                                  |
    | Vitrification:                                                   |
    |   - No ice crystal formation                                     |
    |   - Molecular structure preserved                                |
    |   - Cellular integrity maintained                                |
    |   - Indefinite stable storage possible                          |
    +------------------------------------------------------------------+

[i] KEY CONCEPT - Glass Transition Temperature (Tg):
    The temperature at which the CPA solution transitions from liquid
    to glass state, typically around -130 C for standard solutions.

[CRYO] CONCENTRATION RAMPING PROFILES:

    LINEAR:        Concentration increases at constant rate
                   Best for: Low toxicity CPAs, simple protocols

3.4 CPA CONCENTRATION MONITORING
--------------------------------------------------------------------------------

Real-time monitoring of CPA concentration in perfusate and effluent is
critical for protocol verification.

[CRYO] MONITORING PARAMETERS:

    +------------------------------------------------------------------------+
    | Parameter              | Target         | Alert Threshold             |
    +------------------------------------------------------------------------+
    | Arterial Concentration | Per protocol   | >5% deviation               |
    | Venous Concentration   | Tracking       | >10% below arterial         |
    | Refractive Index       | Solution spec  | >0.001 deviation            |
    | Perfusion Pressure     | 60-80 mmHg     | >100 mmHg (whole body)      |
    | Flow Rate              | Per protocol   | >15% deviation              |
    | Temperature            | Per step       | >1 C deviation              |
    +------------------------------------------------------------------------+

[i] REFRACTIVE INDEX METHOD:

    CPA concentration correlates with refractive index:

    Refractive Index (20C)
    |
    1.45 |                    *  (100% M22)
    1.43 |                *
    1.41 |            *
    1.39 |        *
    1.37 |    *
    1.35 | *                      (0% CPA - carrier only)
         +----------------------------> CPA Concentration %

    Formula: Concentration = (RI - 1.3330) / 0.00115

[CRYO] CONCENTRATION MONITORING CODE:

    class CPAMonitor:
        def __init__(self, target_protocol):
            self.protocol = target_protocol
            self.readings = []

        def add_reading(self, timestamp, arterial_ri, venous_ri,
                       flow_rate, pressure, temperature):
            arterial_conc = self.ri_to_concentration(arterial_ri)
            venous_conc = self.ri_to_concentration(venous_ri)

            reading = {
                'timestamp': timestamp,
                'arterial_concentration': arterial_conc,
                'venous_concentration': venous_conc,
                'concentration_delta': arterial_conc - venous_conc,
                'flow_rate': flow_rate,
                'pressure': pressure,
                'temperature': temperature
            }

            self.readings.append(reading)
            self.check_alerts(reading)
            return reading

        def ri_to_concentration(self, ri):
            return (ri - 1.3330) / 0.00115

        def check_alerts(self, reading):
            expected = self.protocol.expected_at(reading['timestamp'])

            if abs(reading['arterial_concentration'] - expected) > 5.0:
                self.raise_alert('CONCENTRATION_DEVIATION', reading)

            if reading['pressure'] > 100:
                self.raise_alert('PRESSURE_HIGH', reading)

3.5 TOXICITY MANAGEMENT
--------------------------------------------------------------------------------

CPAs are inherently toxic at high concentrations. Protocol design must
balance protection from ice against CPA toxicity.

[!] TOXICITY FACTORS:

    1. CONCENTRATION
       Higher concentration = More protection but more toxic
       Optimal: Minimum concentration for complete vitrification

    2. TEMPERATURE
       Toxicity decreases at lower temperatures
       Strategy: Introduce highest concentrations at coldest temps

    3. EXPOSURE TIME
       Longer exposure = More cellular damage
       Strategy: Minimize time at high concentration

    4. COMPOSITION
       Certain CPA combinations show reduced toxicity
       Strategy: Use multi-component solutions (e.g., M22)

[CRYO] TEMPERATURE-CONCENTRATION PROTOCOL:

    Temperature  |  Max Safe CPA  |  Exposure Time
    ---------------------------------------------------
       +4 C      |     25%        |   60 minutes
        0 C      |     40%        |   45 minutes
       -3 C      |     60%        |   30 minutes
       -7 C      |     80%        |   20 minutes
      -10 C      |    100%        |   15 minutes

[i] TOXICITY MITIGATION STRATEGIES:

    COLD PERFUSION:
    +------------------------------------------------------------------+
    | Begin at 4 C, progressively cool during CPA introduction         |
    | Final concentration achieved near glass transition               |
    +------------------------------------------------------------------+

    CARRIER SOLUTION OPTIMIZATION:
    +------------------------------------------------------------------+
    | Use carrier solutions with protective additives:                 |
    | - Antioxidants (Vitamin E, CoQ10)                               |
    | - Membrane stabilizers (Trehalose)                              |
    | - pH buffers (HEPES, bicarbonate)                               |
    +------------------------------------------------------------------+

    RAPID TRANSITION:
    +------------------------------------------------------------------+
    | Minimize time at intermediate, toxic concentrations              |
    | Use sigmoidal ramping to speed through danger zone               |
    +------------------------------------------------------------------+
