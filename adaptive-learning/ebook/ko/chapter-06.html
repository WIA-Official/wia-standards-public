6장: 시스템 통합과 표준

6.1 학습 기술 생태계

적응형 학습 시스템은 독립적으로 운영되기보다 LMS(학습 관리 시스템), 콘텐츠 저장소, 분석 플랫폼 등 다양한 시스템과 통합되어야 합니다. 표준화된 인터페이스를 통한 상호운용성은 효과적인 학습 생태계 구축의 핵심입니다.

상호운용성 원칙: WIA-ADAPTIVE_LEARNING 준수 시스템은 IMS Global, ADL 등의 업계 표준을 지원하여 다양한 플랫폼과 원활하게 통합될 수 있어야 합니다.

6.2 LTI (Learning Tools Interoperability)

LTI는 학습 도구와 플랫폼 간의 표준 통합 방법을 제공합니다:

// LTI 1.3 통합 구현
class LTIIntegration {
    // 플랫폼에서 도구로의 런치 처리
    async handleLaunch(request: LTILaunchRequest): Promise<LaunchResponse> {
        // 1. ID 토큰 검증
        const claims = await this.verifyIdToken(request.id_token);

        // 2. 사용자 정보 추출
        const user = this.extractUserInfo(claims);

        // 3. 컨텍스트 정보 추출
        const context = {
            courseId: claims['https://purl.imsglobal.org/spec/lti/claim/context'].id,
            resourceLinkId: claims['https://purl.imsglobal.org/spec/lti/claim/resource_link'].id,
            roles: claims['https://purl.imsglobal.org/spec/lti/claim/roles']
        };

        // 4. 세션 생성 및 리다이렉트
        const session = await this.createSession(user, context);
        return { redirectUrl: `/learn?session=${session.id}` };
    }

    // 성적 반환 (Assignment and Grade Services)
    async sendGrade(
        lineItemUrl: string,
        userId: string,
        score: number
    ): Promise<void> {
        const accessToken = await this.getAccessToken(['lineitem', 'score']);

        await fetch(`${lineItemUrl}/scores`, {
            method: 'POST',
            headers: {
                'Authorization': `Bearer ${accessToken}`,
                'Content-Type': 'application/vnd.ims.lis.v1.score+json'
            },
            body: JSON.stringify({
                userId: userId,
                scoreGiven: score,
                scoreMaximum: 100,
                activityProgress: 'Completed',
                gradingProgress: 'FullyGraded'
            })
        });
    }
}

6.3 xAPI (Experience API)

xAPI는 다양한 학습 경험을 추적하고 저장하는 표준입니다:

// xAPI 문장 생성 및 전송
class xAPIClient {
    private lrsEndpoint: string;
    private authToken: string;

    // 학습 이벤트를 xAPI 문장으로 변환
    createStatement(event: LearningEvent): xAPIStatement {
        return {
            actor: {
                objectType: 'Agent',
                account: {
                    homePage: 'https://learning.example.com',
                    name: event.learnerId
                }
            },
            verb: {
                id: this.mapEventToVerb(event.type),
                display: { 'ko-KR': event.type }
            },
            object: {
                objectType: 'Activity',
                id: event.activityId,
                definition: {
                    name: { 'ko-KR': event.activityName },
                    type: this.mapActivityType(event.activityType)
                }
            },
            result: event.result ? {
                score: event.result.score ? {
                    scaled: event.result.score / 100,
                    raw: event.result.score,
                    max: 100
                } : undefined,
                success: event.result.success,
                completion: event.result.completion,
                duration: this.formatDuration(event.result.duration)
            } : undefined,
            context: {
                registration: event.sessionId,
                extensions: {
                    'https://wia.org/xapi/adaptive': {
                        masteryLevel: event.masteryLevel,
                        difficultyLevel: event.difficultyLevel,
                        adaptationType: event.adaptationType
                    }
                }
            },
            timestamp: new Date().toISOString()
        };
    }

    // LRS에 문장 전송
    async sendStatement(statement: xAPIStatement): Promise<string> {
        const response = await fetch(`${this.lrsEndpoint}/statements`, {
            method: 'POST',
            headers: {
                'Authorization': `Basic ${this.authToken}`,
                'Content-Type': 'application/json',
                'X-Experience-API-Version': '1.0.3'
            },
            body: JSON.stringify(statement)
        });

        return response.headers.get('X-Experience-API-Consistent-Through');
    }
}

6.4 SCORM (Sharable Content Object Reference Model)

버전 특징 적응형 학습 지원
SCORM 1.2 기본 추적, 단순 시퀀싱 제한적
SCORM 2004 고급 시퀀싱, 탐색 제어 중간
cmi5 (xAPI 기반) 유연한 추적, 오프라인 지원 우수 
// SCORM 런타임 API 래퍼
class SCORMAdapter {
    private api: SCORMAPI;

    initialize(): boolean {
        // SCORM 2004
        this.api = window.API_1484_11 || window.API;
        if (!this.api) {
            console.error('SCORM API를 찾을 수 없습니다');
            return false;
        }
        return this.api.Initialize('') === 'true';
    }

    // 적응형 학습 데이터 저장
    saveAdaptiveData(data: AdaptiveData): void {
        // suspend_data에 JSON으로 저장
        const suspendData = JSON.stringify({
            learnerModel: data.learnerModel,
            currentPath: data.currentPath,
            masteryLevels: data.masteryLevels
        });

        this.api.SetValue('cmi.suspend_data', suspendData);
        this.api.Commit('');
    }

    // 적응형 학습 데이터 복원
    loadAdaptiveData(): AdaptiveData | null {
        const suspendData = this.api.GetValue('cmi.suspend_data');
        if (suspendData) {
            return JSON.parse(suspendData);
        }
        return null;
    }
}

6.5 Caliper Analytics

IMS Caliper는 학습 분석을 위한 표준 데이터 모델입니다:

// Caliper 이벤트 생성
class CaliperClient {
    createAssessmentEvent(event: AssessmentEvent): CaliperEvent {
        return {
            '@context': 'http://purl.imsglobal.org/ctx/caliper/v1p2',
            type: 'AssessmentEvent',
            actor: {
                id: `urn:uuid:${event.learnerId}`,
                type: 'Person'
            },
            action: event.action,  // 'Started', 'Submitted', 'Graded'
            object: {
                id: `urn:uuid:${event.assessmentId}`,
                type: 'Assessment',
                name: event.assessmentName
            },
            generated: event.result ? {
                id: `urn:uuid:${crypto.randomUUID()}`,
                type: 'Score',
                scoreGiven: event.result.score,
                maxScore: event.result.maxScore
            } : undefined,
            eventTime: new Date().toISOString()
        };
    }
}
