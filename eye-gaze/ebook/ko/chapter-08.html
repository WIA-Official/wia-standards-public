8.1 WIA 인증 프로그램

WIA 인증은 시선 추적 장치 및 소프트웨어가 표준을 준수하는지 확인합니다.

인증 수준

수준 요구사항 대상
기본 핵심 데이터 스키마, WebSocket 소비자 장치
AAC 기본 + 응시 감지, 낮은 지연 시간 보조 기술
연구 기본 + 고주파수, 바이너리 형식 연구 장비
프리미엄 모든 기능, IPC, 인증 전문 시스템

8.2 접근성 표준

시선 추적 시스템은 접근성 표준을 준수해야 합니다.

WCAG 2.1 준수

  • 수준 A: 기본 접근성 (필수)
  • 수준 AA: 향상된 접근성 (권장)
  • 수준 AAA: 최고 접근성 (선택)

주요 WCAG 기준

기준 요구사항 구현
2.1.1 키보드 접근 시선 기반 키보드 대안
2.4.7 포커스 표시 명확한 시선 대상 강조 표시
3.2.2 입력 시 예측 가능 응시 피드백, 확인 단계
3.3.1 오류 식별 명확한 오류 메시지

8.3 장치 테스트

기능 테스트

  • 보정 정확도: <1.5° 평균 오류
  • 데이터 품질: >95% 유효 샘플
  • 지연 시간: <50ms 종단 간
  • 샘플링 속도: 선언된 Hz의 ±5% 이내

테스트 절차

# 보정 정확도 테스트
1. 9포인트 보정 수행
2. 알려진 위치에서 100개의 시선 샘플 수집
3. 실제 위치와 보고된 위치 비교
4. 평균 오류 계산 (도 단위)

결과: 평균 오류 < 1.5° = 합격

# 지연 시간 테스트
1. 알려진 시간에 시각적 자극 표시
2. 시선 반응 시간 측정
3. 시스템 지연 시간 계산
4. 100번 반복

결과: p95 지연 시간 < 50ms = 합격

8.4 소프트웨어 인증

AAC 소프트웨어 요구사항

  • 응시 클릭: 구성 가능한 임계값 (500-2000ms)
  • 시각적 피드백: 응시 진행 표시기
  • 오류 방지: 중요한 작업에 대한 확인
  • 사용자 정의: 대상 크기 및 타이밍 조정

테스트 체크리스트

인증 테스트

  • WebSocket 핸드셰이크 검증
  • 데이터 스키마 준수 확인
  • 이벤트 처리 테스트
  • 오류 처리 검증
  • 성능 벤치마크
  • 접근성 감사
  • 보안 평가

8.5 인증 프로세스

단계

  1. 신청: WIA 웹사이트에서 인증 신청
  2. 문서: 기술 사양 및 구현 세부사항 제출
  3. 테스트: 공인 실험실 또는 자체 테스트 수행
  4. 검토: WIA 기술 위원회 검토
  5. 인증: 합격 시 인증서 발급
  6. 목록: 공식 호환 제품 목록에 추가

인증 비용

유형 비용 유효 기간
장치 (기본) $1,000 2년
장치 (AAC/연구) $2,500 2년
소프트웨어 $500 1년
오픈 소스 무료 1년

8.6 지속적인 규정 준수

연간 검토

인증된 제품은 연간 규정 준수 검토를 받아야 합니다:

  • 사용자 피드백 검토
  • 성능 메트릭 확인
  • 보안 감사

8.7 호환성 테스트 도구

자동화된 테스트 스위트

# WIA 인증 테스트 도구 실행
npm install -g @wia/eye-gaze-test-suite

# 장치 테스트
wia-test device --endpoint ws://localhost:8080/stream

# 소프트웨어 테스트
wia-test software --api http://localhost:8080/api/v1

# 전체 인증 테스트
wia-test certify --all --report certification-report.pdf
