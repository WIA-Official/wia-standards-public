Client Libraries & SDKs

Python

  • influxdb-client: Official InfluxDB 2.x Python client
  • pandas: Data manipulation and time-series analysis
  • prophet: Facebook's forecasting library
  • statsmodels: Statistical modeling and time-series analysis
  • sktime: Scikit-learn compatible time-series ML library

Node.js / JavaScript

  • @influxdata/influxdb-client: InfluxDB client for Node.js
  • pg: PostgreSQL client for TimescaleDB
  • prom-client: Prometheus metrics client
  • timestream-query: AWS Timestream query library

Go

  • influxdb-client-go: Official InfluxDB Go client
  • prometheus/client_golang: Prometheus Go client
  • lib/pq: PostgreSQL driver for TimescaleDB

Java

  • influxdb-client-java: InfluxDB Java client
  • postgresql: JDBC driver for TimescaleDB
  • simpleclient: Prometheus Java client

Analysis & Forecasting Libraries

Machine Learning & Forecasting

  • Prophet: Facebook's automatic forecasting procedure (Python/R)
  • ARIMA/SARIMA: Classical statistical models (statsmodels)
  • LSTM: Deep learning for time-series (TensorFlow/PyTorch)
  • GluonTS: Amazon's probabilistic time-series modeling toolkit
  • Darts: Python library for easy time-series forecasting
  • NeuralProphet: Neural network-based Prophet successor

Anomaly Detection

  • Isolation Forest: Scikit-learn implementation
  • DBSCAN: Density-based clustering for anomalies
  • One-Class SVM: Unsupervised anomaly detection
  • Autoencoders: Neural network-based anomaly detection
  • Z-score/MAD: Statistical outlier detection
