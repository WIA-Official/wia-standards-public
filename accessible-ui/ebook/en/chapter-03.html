Chapter 03

Visual Accessibility

3.1 Understanding Visual Impairments

Approximately 2.2 billion people globally have some form of vision impairment. Low vision affects far more people than total blindness. Users may have reduced acuity, limited field of vision, or difficulty with contrast perception.

Visual Impairment Categories
  • Blindness: No functional vision; relies on screen readers
  • Low Vision: Significant vision loss; may use magnification
  • Color Vision Deficiency: Affects 8% of males
  • Photosensitivity: Sensitivity to bright lights or flashing

// Contrast Ratio Calculation
function calculateContrastRatio(foreground, background) {
  const fgLuminance = getRelativeLuminance(foreground);
  const bgLuminance = getRelativeLuminance(background);

  const lighter = Math.max(fgLuminance, bgLuminance);
  const darker = Math.min(fgLuminance, bgLuminance);

  return (lighter + 0.05) / (darker + 0.05);
}

function checkCompliance(ratio, isLargeText = false) {
  return {
    wcagAA: isLargeText ? ratio >= 3 : ratio >= 4.5,
    wcagAAA: isLargeText ? ratio >= 4.5 : ratio >= 7
  };
}

3.3 Color Independence

WCAG 1.4.1 requires that color is not the only means of conveying information. Use icons, patterns, or text labels alongside color coding.

Common Color-Only Mistakes
  • Required fields indicated only by red asterisks
  • Error states shown only with red borders
  • Links distinguished only by color
  • Charts using color alone to differentiate data

3.4 Typography

  • Clear letterforms: Distinct shapes for I, l, 1 and O, 0
  • Generous x-height: Larger lowercase letters
  • Use rem units: Allow browser/user scaling
  • Max line length: 75 characters for readability
/* Accessible Typography */
:root {
  font-size: 100%; /* Allow browser scaling */
  --font-size-base: 1rem;
  --line-height: 1.5;
  --max-line-length: 75ch;
}

body {
  font-family: 'Segoe UI', system-ui, sans-serif;
  font-size: var(--font-size-base);
  line-height: var(--line-height);
}

p { max-width: var(--max-line-length); }

/* Dark mode support */
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg: #0f172a;
    --color-text: #f1f5f9;
  }
}

3.6 Responsive Design

Responsive design and accessibility work together. Users may zoom to 200% or more, and content must remain usable. Avoid fixed-width layouts, ensure text reflows properly, and maintain touch targets at all viewport sizes.

Responsive Accessibility Guidelines
  • Support zoom up to 200% without horizontal scrolling
  • Use relative units (rem, em) instead of fixed pixels
  • Ensure content reflows in single-column layouts
  • Maintain adequate touch targets at all breakpoints

3.7 Summary

Visual accessibility encompasses color contrast, color independence, typography, alt text, and responsive design. By implementing proper contrast ratios, ensuring information is not conveyed by color alone, choosing readable fonts, and providing meaningful alternative text, you create interfaces that serve users across the full spectrum of visual abilities.
