Chapter 7: 규정 및 공역 관리

弘益人間 (홍익인간) - 널리 인간을 이롭게 하라

WIA-SPACE-017 드론 UAV 표준

7.1 드론 규제 프레임워크 개요

드론의 안전하고 책임 있는 운영을 위해 전 세계 각국은 포괄적인 규제 체계를 구축하고 있습니다. 弘益人間(홍익인간)의 철학에 따라, 이러한 규제는 드론 기술이 공공 안전, 프라이버시, 보안을 보장하면서 혁신과 경제적 혜택을 제공할 수 있도록 균형을 맞추는 것을 목표로 합니다.

드론 규제는 일반적으로 다음과 같은 영역을 다룹니다:

  • 운영자 자격: 라이선스, 인증, 교육 요구사항
  • 드론 등록: 식별, 추적, 책임 할당
  • 비행 규칙: 고도 제한, 비행 금지 구역, 시간대 제한
  • 기술 표준: 안전 기능, Remote ID, 지오펜싱
  • 공역 통합: 유인 항공기와의 안전한 공존
  • 프라이버시 및 보안: 데이터 보호, 감시 규제

7.1.1 규제 접근 방식의 진화

드론 규제는 기술 발전과 함께 지속적으로 진화하고 있습니다:

  • 1세대 (2010년대 초반): 일괄적인 금지 또는 매우 제한적인 허가
  • 2세대 (2010년대 중반): 무게 기반 분류, 간단한 운영 규칙
  • 3세대 (2010년대 후반-현재): 위험 기반(risk-based) 접근, 성능 표준
  • 4세대 (미래): UTM 통합, 완전 자율 비행, 도심 항공 모빌리티(UAM)

7.2 FAA Part 107 (미국)

미국 연방항공청(FAA)의 Part 107 규정은 2016년 8월에 발효되어 소형 UAV(55파운드/25kg 이하)의 상업적 운영을 위한 포괄적인 프레임워크를 제공합니다. 이는 전 세계 드론 규제의 모델이 되었습니다.

7.2.2 Part 107 시험 준비

Remote Pilot Certificate 취득을 위한 지식 시험은 60개의 객관식 문제로 구성되며, 70% 이상 정답으로 합격합니다. 주요 시험 영역은 다음과 같습니다:

  • 규제 (Regulations): Part 107 규칙, 등록, 라이선스, 보고 요구사항 (25-30%)
  • 국가 공역 시스템(National Airspace System): 공역 분류, ATC 통신 (15-20%)
  • 기상(Weather): METAR, TAF, 기상 현상, 마이크로버스트, 윈드시어 (10-15%)
  • 드론 성능(Loading and Performance): 무게중심, 성능 차트, 배터리 관리 (10%)
  • 운영(Operations): 비행 전 점검, 리스크 관리, 크루 자원 관리 (CRM) (20-25%)
  • 생리학(Aeromedical Factors): 스트레스, 피로, 약물/알코올 영향 (5%)

7.2.3 야간 비행 및 사람 위 비행 (2021년 개정)

야간 비행 요구사항 (Part 107.29)

  • 조종사 교육: 야간 비행에 특화된 recurrent training 이수
  • 조명: 드론에 anti-collision light 장착 (3 statute miles에서 가시)
  • 시야: 조종사가 육안으로 드론 위치, 자세, 방향 파악 가능

사람 위 비행 카테고리

  • Category 1: 0.55파운드 미만, 노출된 회전 부품 없음 - 무제한
  • Category 2: 부상 기준 충족 (<=11 ft-lbs kinetic energy) - 무제한
  • Category 3: Category 2 기준 미충족 - 폐쇄된 장소 또는 이동하지 않는 사람 위만
  • Category 4: 비행 매뉴얼 따름, 안전 계획 - 제한적 상황

7.2.4 면제(Waiver) 신청

Part 107의 특정 규정에 대한 면제를 FAA에 신청할 수 있습니다. 일반적인 면제 대상:

  • BVLOS (Beyond Visual Line of Sight): 가장 많이 신청되는 면제
  • 고도 제한: 400피트 초과 비행 (예: 높은 구조물 검사)
  • 다수 드론 운영: 한 조종사가 여러 드론 동시 제어
  • 외부 적재물: 특정 중량/크기 초과 적재물 운반

7.3 EASA 드론 규정 (유럽)

유럽항공안전청(EASA)의 드론 규정은 2021년 1월부터 EU 전역에서 통합 적용되며, 위험 기반(risk-based) 접근 방식을 채택하고 있습니다. 이는 운영의 위험도에 따라 요구사항을 차등화하는 혁신적인 프레임워크입니다.

7.3.1 EASA 카테고리 분류

카테고리 위험도 주요 요구사항 예시
Open 낮음 등록, 온라인 시험, VLOS, 사람으로부터 거리 유지 레저, 기본 촬영, 취미
Specific 중간 운영 승인, 위험 평가(SORA), 조종사 교육 BVLOS, 도심 비행, 상업 촬영
Certified 높음 드론 형식 인증, 조종사 라이선스, 운영자 인증 사람 운송, 대형 화물 운송

7.3.2 Open 카테고리 세부 분류

서브카테고리 드론 등급 최대 이륙 중량 사람 근접 최대 고도 조종사 요구사항
A1 C0, C1 <250g (C0), <900g (C1) 사람 위 비행 가능 (비참여자) 120m 온라인 교육 (A1/A3)
A2 C2 <4kg 사람 30m 이내, Low-speed 모드 시 5m 120m 온라인 교육 + 자체 실기
A3 C2, C3, C4 <25kg 사람/주거/상업 지역 150m 이상 120m 온라인 교육 (A1/A3)

7.3.3 드론 CE 등급

EASA는 드론 제조사가 특정 안전 기준을 충족하는 제품에 CE 마크를 부여하도록 요구합니다:

  • C0: <250g, 최대 속도 19 m/s, 최대 고도 120m - 장난감급
  • C1: <900g, 최대 속도 19 m/s, Remote ID, 지오펜싱 - 소비자급
  • C2: <4kg, Remote ID, 지오펜싱, Low-speed 모드, 소음 제한 - 프로슈머급
  • C3: <25kg, Remote ID, 지오펜싱, 최대 직경 3m - 전문가급
  • C4: <25kg, 조종 가능, Remote ID 없음 - DIY/자작 드론
  • C5/C6: >25kg, Certified 카테고리용

7.3.4 Specific 카테고리 및 SORA

Specific 카테고리 운영은 SORA (Specific Operations Risk Assessment) 프로세스를 통해 승인됩니다:

  1. 운영 설명(ConOps): 임무, 환경, 드론, 조종사 능력 정의
  2. 지상 위험 평가(GRC): 지상 사람/재산 피해 가능성
  3. 공중 위험 평가(ARC): 유인 항공기 충돌 가능성
  4. 위험 완화 조치: 기술적/운영적 완화책 적용
  5. SAIL (Specific Assurance and Integrity Level): 최종 위험 수준 결정
  6. OSO (Operational Safety Objectives): 필요한 안전 목표 충족

EASA vs FAA 비교

  • EASA: 위험 기반, 유연한 카테고리, 제조사 인증 중시, EU 통합 규정
  • FAA: 규정 기반, 일관된 규칙, 운영자 인증 중시, 면제 프로세스
  • 공통점: Remote ID, 등록, 교육, VLOS 원칙, 공역 통합

7.4 Remote ID (원격 식별)

Remote ID는 드론의 실시간 식별 및 위치 정보를 방송하는 시스템으로, 弘익人間(홍익인간)의 철학에 따라 공공 안전, 보안, 책임 추적을 강화합니다. FAA, EASA, 기타 주요 규제 기관들이 필수 요구사항으로 도입하고 있습니다.

7.4.1 Remote ID 요구사항 (FAA)

2023년 9월 16일부터 미국에서 운영되는 대부분의 드론은 Remote ID를 방송해야 합니다:

  • Standard Remote ID:
    • 드론에 내장된 Remote ID 모듈
    • 이륙 위치 및 현재 위치/고도/속도 방송
    • 조종사 위치 (동적 업데이트)
  • Broadcast Remote ID Module:
    • 기존 드론에 장착하는 외부 모듈
    • 이륙 위치 고정, 드론 위치 방송
    • 최소 요구사항 충족
  • 방송 데이터:
    • 드론 ID (Serial Number 또는 Session ID)
    • 위도, 경도, 고도 (지오데틱)
    • 속도, 트랙(방향)
    • 이륙 위치
    • 조종사 위치 (Standard만)
    • 타임스탬프

7.4.2 Remote ID 면제

  • FAA-Recognized Identification Area (FRIA): 승인된 레크리에이션 비행 구역
  • 0.55파운드 미만: 등록 면제 드론
  • 인터넷 연결 불가능한 구형 드론: 2023년 9월 이전 제작, FRIA에서만 운영

7.4.3 EASA Remote ID

EASA는 C1-C3 등급 드론에 Remote ID를 요구하며, 두 가지 모드를 지원합니다:

  • Direct Remote ID: WiFi/Bluetooth로 근거리 방송 (C1-C3)
  • Network Remote ID: 인터넷을 통한 중앙 서버 전송 (선택적)

프라이버시 고려사항

Remote ID는 드론 운영의 투명성을 높이지만, 프라이버시 우려도 제기됩니다:

  • 조종사 위치 노출 (Standard Remote ID)
  • 상업적 활동 추적 가능성
  • 저널리스트, 연구자 등 민감한 운영 노출
  • 해결책: Session ID 사용, FRIA 활용, Network Remote ID (조종사 위치 비공개)

7.5 공역 분류 및 제한 구역

드론은 복잡한 국가 공역 시스템(NAS) 내에서 운영되며, 다양한 공역 등급과 제한 구역을 준수해야 합니다. 弘益人間(홍익인간)의 철학에 따라, 안전한 공역 관리는 유인 항공기와 드론의 조화로운 공존을 가능하게 합니다.

7.5.1 미국 공역 분류 (FAA)

공역 설명 드론 운영 승인 필요
Class A 18,000 ft MSL 이상 사실상 불가능 (상업 항공기 전용) N/A
Class B 주요 공항 주변 (표면~10,000 ft MSL) LAANC 또는 Part 107 면제 ATC 승인 필수
Class C 중형 공항 (표면~4,000 ft AGL) LAANC 또는 Part 107 면제 ATC 승인 필수
Class D 관제탑 있는 공항 (표면~2,500 ft AGL) LAANC 또는 Part 107 면제 ATC 승인 필수
Class E 관제 공역 (일반적으로 700/1,200 ft AGL~) Part 107 준수 시 허용 불필요 (400 ft 이하)
Class G 비관제 공역 (표면~Class E 하단) Part 107 준수 시 허용 불필요

7.5.2 특별 사용 공역 (Special Use Airspace)

  • TFR (Temporary Flight Restriction): 임시 비행 금지 구역 (대통령 이동, 재난, 스포츠 이벤트)
  • MOA (Military Operations Area): 군사 훈련 구역 - 주의 필요
  • Restricted Area: 제한 구역 (군사 기지, 사격장) - 승인 없이 금지
  • Prohibited Area: 금지 구역 (백악관, 국회의사당 등) - 절대 금지
  • Alert Area: 경고 구역 - 높은 항공 활동, 주의

7.5.3 LAANC (Low Altitude Authorization and Notification Capability)

LAANC는 FAA가 개발한 시스템으로, Class B/C/D/E 표면 구역에서 드론 운영 승인을 자동화하고 간소화합니다:

  • 실시간 승인: 수분 내 자동 승인 (조건 충족 시)
  • UAS Facility Map: 공항별 최대 허용 고도 그리드
  • 승인된 앱: Aloft, AirMap, Skyward, Kittyhawk 등
  • 운영 시간: 일반적으로 일출 전 30분~일몰 후 30분
  • 제한: LAANC 불가 지역은 DroneZone을 통한 수동 승인 (7-90일 소요)

B4UFLY 및 공역 앱

  • B4UFLY (FAA 공식 앱): 비행 가능 여부 확인, TFR 알림, 공역 정보
  • Aloft: LAANC 승인, 실시간 공역 상태, 비행 계획
  • AirMap: LAANC, 디지털 스카이 플랫폼, 자동화 통합
  • Kittyhawk: 엔터프라이즈 솔루션, LAANC, 팀 관리

7.6 국제 드론 규제

전 세계 각국은 자국의 항공 인프라, 보안 우려, 기술 수용도에 따라 다양한 드론 규제를 시행하고 있습니다. 弘益人間(홍익인간)의 철학에 따라, 국제 협력과 표준화는 드론 기술이 국경을 넘어 인류에게 이익을 제공하도록 합니다.

    • 2kg 이상 7kg 미만: 4종 (20시간 교육)
    • 7kg 이상 25kg 미만: 3종 (40시간 교육)
    • 25kg 이상: 2종 (60시간 교육)
  • 비행 승인:
    • 관제권(공항 9.3km), 비행 금지 구역: 사전 승인
    • 150m 초과, 야간, 인구 밀집 지역: 승인 필요
  • 보험: 12kg 이상 의무 보험 가입
  • 실명 등록: 한국항공안전기술원(KIAST) 드론 등록 시스템

7.6.3 ICAO 및 국제 표준화

국제민간항공기구(ICAO)는 드론의 국제 표준 개발을 주도하고 있습니다:

  • Annex 7 개정: RPAS 등록 및 식별 표준
  • RPAS 패널: 규제 프레임워크, UTM, 공역 통합 연구
  • SARPS (Standards and Recommended Practices): 국제 조화된 규정 개발
  • 목표: 국경 간 드론 운영, 상호 인증, 글로벌 UTM 네트워크

7.7 UTM (UAV Traffic Management)

UTM은 저고도 공역에서 다수의 드론을 안전하고 효율적으로 관리하기 위한 디지털 시스템입니다. 弘益人間(홍익인간)의 철학에 따라, UTM은 드론 산업의 확장과 공공 안전을 동시에 실현합니다.

7.7.1 UTM 아키텍처

  • UTM Service Suppliers (USS): 민간 UTM 서비스 제공자 (비행 계획, 충돌 회피)
  • FIMS (FAA UAS Data Exchange): FAA 중앙 데이터 교환 플랫폼
  • PSU (Public Safety USS): 공공 안전 기관용 우선 채널
  • UAS Operator: 드론 운영자 (비행 계획 제출, 승인 수신)
  • Supplemental Data Providers: 기상, 지형, TFR 등 추가 데이터

7.7.2 UTM 핵심 기능

  • 비행 계획 관리: 경로 제출, 승인, 변경, 취소
  • 동적 지오펜싱: 실시간 비행 금지 구역 업데이트
  • 전략적 충돌 회피: 비행 전 경로 충돌 감지 및 조정
  • 전술적 충돌 회피: 실시간 드론 간 분리, DAA (Detect and Avoid)
  • 공역 동적 할당: 혼잡도에 따른 공역 용량 관리
  • 비상 대응: 긴급 착륙, 손실 링크, PSU 우선권

7.7.3 글로벌 UTM 이니셔티브

  • 미국 (NASA/FAA UTM): TCL (Technology Capability Level) 1-4 단계별 개발
  • 유럽 (U-Space): SESAR 프로젝트, U1-U4 서비스 레벨
  • 일본 (UTMS): NEDO/JUTM 컨소시엄, 이동통신 기반
  • 싱가포르 (OneMap UTM): 정부 주도 통합 플랫폼

UTM의 미래: 고급 공중 모빌리티 (AAM)

UTM은 단순히 드론 관리를 넘어 미래의 도심 항공 모빌리티(UAM)를 지원할 것입니다:

  • 에어 택시 (eVTOL) 통합 관제
  • 자율 비행 복잡도 증가 (Level 4-5)
  • 고밀도 운영 (수천 대 동시 비행)
  • 유인-무인 항공기 통합 공역

핵심 요점 (Key Takeaways)

  1. FAA Part 107: 미국의 소형 상업용 드론 규제로, Remote Pilot Certificate, 등록, 400피트 고도 제한, VLOS, 주간 운영을 요구하며, 면제 신청으로 예외 가능합니다.
  2. EASA 위험 기반 접근: Open, Specific, Certified 카테고리로 위험도에 따라 요구사항을 차등화하며, SORA 프로세스로 복잡한 운영을 평가합니다.
  3. Remote ID 필수화: FAA와 EASA는 드론의 실시간 식별 및 위치 방송을 요구하여 공공 안전과 책임 추적을 강화하지만, 프라이버시 우려도 존재합니다.
  4. 공역 분류 이해: Class B/C/D 공역에서는 LAANC를 통한 ATC 승인이 필요하며, TFR, 제한 구역, 금지 구역을 준수해야 합니다.
  5. 국제 규제 다양성: 각국은 무게 기준, 등록, 라이선스, 고도 제한이 상이하므로, 국제 운영 시 현지 규정을 숙지해야 합니다.
  6. UTM의 중요성: 다수 드론의 안전한 관리를 위해 UTM은 비행 계획, 충돌 회피, 동적 공역 할당을 제공하며, 미래 AAM의 기반이 됩니다.
  7. 弘益人間 철학 구현: 규제는 드론 기술이 공공 안전, 프라이버시, 보안을 보장하면서 혁신과 경제적 혜택을 제공하도록 균형을 맞춥니다.

복습 문제 (Review Questions)

  1. FAA Part 107의 주요 요구사항 5가지를 나열하고, Remote Pilot Certificate 취득 절차를 설명하시오.
  2. Part 107의 야간 비행 요구사항(Part 107.29)과 사람 위 비행 카테고리(Category 1-4)를 비교 설명하시오.
  3. EASA의 Open, Specific, Certified 카테고리의 차이점과 각각의 주요 요구사항을 설명하고, SORA 프로세스의 단계를 서술하시오.
  4. Remote ID의 Standard 모드와 Broadcast 모드의 차이를 비교하고, Remote ID가 방송하는 데이터 종류를 5가지 이상 나열하시오.
  5. 미국 공역 Class B, C, D, E, G의 특징을 설명하고, LAANC 시스템의 작동 원리와 장점을 논의하시오.
  6. 한국의 드론 규제(무게 구분, 조종자 증명, 비행 승인)를 설명하고, FAA Part 107과의 주요 차이점을 비교하시오.
  7. UTM의 핵심 구성 요소(USS, FIMS, PSU)와 주요 기능(비행 계획 관리, 충돌 회피, 동적 지오펜싱)을 설명하시오.
  8. 弘益人間(홍익인간) 철학의 관점에서, 드론 규제와 UTM이 공공 안전, 혁신, 경제적 혜택 간의 균형을 어떻게 달성하는지 논의하시오.
