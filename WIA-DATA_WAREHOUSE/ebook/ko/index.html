WIA-DATA_WAREHOUSE

데이터 웨어하우스 표준 - 완전한 가이드
弘益人間 (홍익인간) · 널리 인간을 이롭게 하라

데이터 웨어하우스는 기업의 의사결정을 지원하는 핵심 인프라입니다. 분산된 데이터 소스를 통합하여 일관되고 신뢰할 수 있는 분석 환경을 제공하며, OLAP, BI, 머신러닝의 기반이 됩니다.

이 가이드는 데이터 웨어하우스의 기초 개념부터 차원 모델링, ETL/ELT, OLAP 큐브, 클라우드 DW (Snowflake, BigQuery, Redshift), 그리고 레이크하우스, Data Mesh까지 실무에 필요한 모든 지식을 다룹니다.

Phase 1: 기초 이론

01
데이터 웨어하우스 기초
DW 정의 및 특징, OLTP vs OLAP, Inmon vs Kimball 방법론, 단일 진실 공급원(SSOT), 히스토리 관리, SCD 타입
02
차원 모델링
스타 스키마, 스노우플레이크 스키마, 갤럭시 스키마, 팩트 테이블, 차원 테이블, 그레인(Grain), 컨포멀 차원

Phase 2: 기술적 구현

03
ETL/ELT 프로세스
Extract-Transform-Load vs Extract-Load-Transform, 데이터 파이프라인, Apache Airflow, dbt, Talend, Informatica
04
OLAP 및 큐브
다차원 분석, MOLAP, ROLAP, HOLAP, 드릴다운/업, 슬라이스/다이스, 피벗, MDX 쿼리
05
데이터 마트 설계
부서별 데이터 마트, 집계 테이블, 요약 테이블, 브리지 테이블, 미니 차원, 아웃리거 차원
06
성능 최적화
인덱싱 전략, 파티셔닝, 클러스터링, 컬럼형 저장소, 압축, 쿼리 최적화, 캐싱

Phase 3: 클라우드 및 미래

07
클라우드 데이터 웨어하우스
Snowflake, Google BigQuery, Amazon Redshift, Azure Synapse, 자동 스케일링, 사용량 기반 요금제
08
미래 트렌드와 혁신
레이크하우스 (Databricks Delta Lake), Data Mesh, 실시간 DW, AI 자동 최적화, 메타데이터 관리, 데이터 카탈로그
