## Changes from v1.1

### Major Features

#### 1. Historical Linguistics Module
- Timeline view showing language evolution from 1945-present
- Historical document translation (pre-division Korean)
- Etymology deep-dive with visual family trees
- Comparison with Middle Korean and Classical Korean

#### 2. Regional Dialect Support
- Expanded beyond standard North/South to include:
  - Jeju dialect (제주어)
  - Gyeongsang dialect (경상 방언)
  - Jeolla dialect (전라 방언)
  - Hamgyong dialect (함경 방언)
- Cross-regional dialect translation matrix

#### 3. Educational Gamification
- Interactive quizzes and challenges
- Progress tracking and achievements
- Leaderboards for learning communities
- Virtual rewards for milestones

#### 4. Business Communication Module
- Professional email templates
- Contract terminology standardization
- Business etiquette guides
- Industry-specific glossaries

### Technical Updates

#### AI Model Improvements
- Training data expanded to 5 million sentence pairs
- Accuracy improved to 93% for common vocabulary
- Context window expanded to 1024 tokens
- Added sentiment analysis for appropriate register selection

#### Integration Features
- Browser extensions for Chrome, Firefox, Safari
- Microsoft Office plugin (Word, Outlook)
- Slack and Discord bot integrations
- API rate limits increased to 10,000 requests/hour

### Dictionary Expansion
- Total entries: 65,000 (up from 50,000)
- Historical terms: 5,000+
- Regional variations documented
- Pronunciation audio for 80% of entries
