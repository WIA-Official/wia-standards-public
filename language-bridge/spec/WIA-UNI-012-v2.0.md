## Breaking Changes from v1.x

1. API endpoint structure updated (migration guide available)
2. Dictionary JSON schema v2 (backward compatible parsers provided)
3. Minimum system requirements increased for AR features
4. Deprecated legacy speech recognition (replaced with neural engine)

## Migration Guide

Organizations on v1.x should plan migration over 3-6 months:
1. Review API changes and update integrations
2. Train staff on new features
3. Update mobile apps to v2.0 SDKs
4. Test AR features with compatible hardware
5. Migrate user data to new schema
