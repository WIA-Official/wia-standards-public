Exam Format

📝

Core Exam

객관식 + 단답형. 소유권 개념 집중 검증. 60분.

💻

Expert Exam

실기 시험. 코드 작성 및 디버깅. 90분.

🎯

Master Exam

종합 시험 + 프로젝트 코드 리뷰. 120분.
