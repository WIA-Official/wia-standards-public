EP 0-03

Cargo.lock

📋 Cargo.toml

원하는 버전 범위 지정
serde = "1.0"

Desired version range
serde = "1.0"

🔒 Cargo.lock

정확한 버전 고정
serde = "1.0.193"

Exact version locked
serde = "1.0.193"

규칙:
✅ 바이너리: Cargo.lock 커밋
❌ 라이브러리: Cargo.lock 제외 (.gitignore)

Rule:
✅ Binary: Commit Cargo.lock
❌ Library: Exclude Cargo.lock (.gitignore)
