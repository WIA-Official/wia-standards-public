Unsafe란?

Unsafe 슈퍼파워 5가지

The 5 Unsafe Superpowers

  • 원시 포인터 역참조
  • Dereference raw pointers
  • unsafe 함수/메서드 호출
  • Call unsafe functions/methods
  • 정적 가변 변수 접근/수정
  • Access/modify static mut variables
  • unsafe 트레이트 구현
  • Implement unsafe traits
  • union 필드 접근
  • Access union fields

unsafe는 빌림 검사를 끄지 않습니다!

여전히 대부분의 안전 검사는 적용됩니다. 단지 위의 5가지만 허용될 뿐. unsafe doesn't disable borrow checking!

Most safety checks still apply. Only the above 5 are allowed.

원시 포인터

원시 포인터 (Raw Pointers)

Raw Pointers

let x = 42;
// 원시 포인터 생성 (safe에서 가능)
let r1 = &x as *const i32; // 불변 원시 포인터
let r2 = &x as *mut i32; // 가변 원시 포인터
// 역참조는 unsafe 블록 필요!
unsafe {
println!("r1: {}", *r1);
}
// 위험한 예: 임의 주소
let address = 0x012345 as *const i32;
// unsafe { println!("{}", *address); } // 💥 크래시!

Unsafe 함수와 메서드

Unsafe Functions and Methods

// unsafe 함수 정의
unsafe fn dangerous() {
println!("위험한 작업 수행!");
}
// 호출하려면 unsafe 블록 필요
unsafe {
dangerous();
}
// 실제 예: 슬라이스 분할
let mut v = vec![1, 2, 3, 4, 5];
let (a, b) = v.split_at_mut(3);
// split_at_mut 내부는 unsafe지만, API는 safe!

FFI

FFI - 다른 언어 연동

FFI - Foreign Function Interface

// C 함수 선언
extern "C" {
fn abs(input: i32) -> i32;
fn sqrt(input: f64) -> f64;
}
fn main() {
unsafe {
println!("abs(-5) = {}", abs(-5));
println!("sqrt(16) = {}", sqrt(16.0));
}
}
// Rust 함수를 C에서 호출 가능하게
#[no_mangle]
pub extern "C" fn call_from_c() { }

정적 가변

정적 가변 변수 (Static Mut)

Static Mutable Variables

// 전역 가변 상태 - 위험!
static mut COUNTER: u32 = 0;
fn increment() {
unsafe {
COUNTER += 1;
}
}
fn main() {
increment();
increment();
unsafe {
println!("카운터: {}", COUNTER); // 2
}
}

⚠️ 멀티스레드에서 static mut는 데이터 레이스 위험! 대신 Atomic 또는 Mutex 사용 ⚠️ static mut in multi-threaded code risks data races! Use Atomic or Mutex instead

안전한 추상화

Unsafe를 Safe로 감싸기

Wrapping Unsafe in Safe

// 안전한 API로 감싸기
fn split_at_mut(slice: &mut [i32], mid: usize)
-> (&mut [i32], &mut [i32])
{
let len = slice.len();
let ptr = slice.as_mut_ptr();
assert!(mid <= len); // 안전성 보장!
unsafe {
(
std::slice::from_raw_parts_mut(ptr, mid),
std::slice::from_raw_parts_mut(ptr.add(mid), len - mid),
)
}
}
