빠른 조회 O(1)
순서 없음

Key-value pairs
Fast lookup O(1)
No order

HashSet<T>

고유한 값만 저장
중복 불허
포함 여부 O(1)

Unique values only
No duplicates
O(1) contains

이터레이터 메서드 Iterator Methods

let v = vec![1, 2, 3, 4, 5];
// map - 변환
v.iter().map(|x| x * 2)
// filter - 필터링
v.iter().filter(|x| **x > 2)
// fold - 축소
v.iter().fold(0, |acc, x| acc + x)
// 체이닝!
let result: Vec<_> = v.iter()
.filter(|x| **x > 2)
.map(|x| x * 2)
.collect();
// [6, 8, 10]

iter() vs into_iter()

let v = vec![1, 2, 3];
// iter() - 불변 참조 (&T)
for x in v.iter() { } // v 계속 사용 가능
// iter_mut() - 가변 참조 (&mut T)
for x in v.iter_mut() { *x += 1; }
// into_iter() - 소유권 이동 (T)
for x in v.into_iter() { } // v 사용 불가!
💡 for x in &v = v.iter(), for x in v = v.into_iter() 💡 for x in &v = v.iter(), for x in v = v.into_iter()
