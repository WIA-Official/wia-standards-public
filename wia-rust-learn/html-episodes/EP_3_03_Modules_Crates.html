Crate vs PackageCrate vs Package

📦 Crate

• 컴파일 단위
• Binary crate: 실행파일
• Library crate: 라이브러리

• Compilation unit
• Binary crate: executable
• Library crate: library

📁 Package

• Cargo.toml을 가진 단위
• 0-1개의 library crate
• 여러 binary crate 가능

• Has Cargo.toml
• 0-1 library crate
• Multiple binary crates
