표현식 vs 문장 Expression vs Statement

// 문장 (Statement) - 값을 반환 X
let x = 6;
// 표현식 (Expression) - 값을 반환 O
5 + 6
add(1, 2)
{ x + 1 } // 블록도 표현식!
// 블록 표현식 활용
let y = {
let x = 3;
x + 1 // 세미콜론 없음!
};
// y = 4

EP 1-02

match 표현식 match Expression

let number = 3;
match number {
1 => println!("one"),
2 => println!("two"),
3 | 4 => println!("three or four"),
5..=10 => println!("five to ten"),
_ => println!("something else"),
}
💡 match는 모든 경우를 처리해야 함 (exhaustive). _ 는 나머지 모든 경우! 💡 match must be exhaustive. _ catches everything else!

let some_value = Some(3);

None => (),
}
