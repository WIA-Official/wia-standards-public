? 연산자The ? Operator

fn read_file() -> Result<String, std::io::Error> {
let mut file = File::open("hello.txt")?;
let mut contents = String::new();
file.read_to_string(&mut contents)?;
Ok(contents)
}
💡 ? = 에러면 즉시 반환, 성공이면 값 추출 💡 ? = Return early on error, extract value on success

언제 panic? 언제 Result?When panic? When Result?

panic!

• 복구 불가능한 버그
• 프로토타입/테스트
• 예시 코드
• 불변 조건 위반

• Unrecoverable bugs
• Prototypes/tests
• Example code
• Invariant violations

Result

• 예상 가능한 실패
• 파일/네트워크 작업
• 사용자 입력 처리
• 라이브러리 API

• Expected failures
• File/network ops
• User input handling
• Library APIs
