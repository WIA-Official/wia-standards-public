Enum

enum IpAddr {
V4(u8, u8, u8, u8),
V6(String),
}
let home = IpAddr::V4(127, 0, 0, 1);
let loopback = IpAddr::V6(String::from("::1"));
💡 Enum 배리언트마다 다른 데이터 타입을 가질 수 있다! 💡 Each enum variant can have different data types!

match로 Enum 처리 Handling Enums with match

fn process(ip: IpAddr) {
match ip {
IpAddr::V4(a, b, c, d) => {
println!("IPv4: {}.{}.{}.{}", a, b, c, d);
}
IpAddr::V6(addr) => {
println!("IPv6: {}", addr);
}
}
}
