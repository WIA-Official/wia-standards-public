기본은 불변 (Immutable) Immutable by Default

let x = 5;
x = 6; // ❌ 에러! cannot assign twice
let mut y = 5;
y = 6; // ✅ OK! mut 키워드 사용
💡 mut = mutable (변경 가능). 명시적으로 선언해야 변경 가능! 💡 mut = mutable. Explicitly declare to allow changes!

let x = 42; // i32 (기본)

복합 타입 Compound Types

// 튜플 (Tuple)
let tup: (i32, f64, char) = (500, 6.4, 'a');
let (x, y, z) = tup; // 구조분해
let first = tup.0; // 인덱스 접근

let arr = [1, 2, 3, 4, 5];
let first = arr[0]; // 인덱스 접근
let arr: [i32; 5] = [1,2,3,4,5];

EP 1-01

섀도잉 (Shadowing) Shadowing

let x = 5;
let x = x + 1; // x = 6 (새로운 x)
let x = x * 2; // x = 12 (또 새로운 x)
// 타입도 바꿀 수 있다!
let spaces = " "; // &str
let spaces = spaces.len(); // usize
💡 Shadowing vs mut: 섀도잉은 새 변수를 만드는 것, mut은 같은 변수 수정 💡 Shadowing vs mut: Shadowing creates new variable, mut modifies same one

EP 1-01

상수 (Constants) Constants

// const: 컴파일 타임 상수
const MAX_POINTS: u32 = 100_000;
const PI: f64 = 3.14159;
// static: 프로그램 전체 수명
static HELLO: &str = "Hello, World!";
📌 상수는 반드시 타입을 명시해야 하고, 섀도잉 불가! 📌 Constants must have explicit type and cannot be shadowed!
