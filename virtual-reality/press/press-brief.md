# WIA-XR-VR-001 — Virtual Reality (Press Brief)

**WIA Standards releases canonical Virtual Reality envelope shape**

The WIA Standards Committee has published the WIA-XR-VR-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for virtual reality hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
