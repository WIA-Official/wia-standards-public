# DALL-E Image Prompts for WIA-COMP-013 (Software Testing Standard)

## Image 1: Main Hero - Testing Pyramid Excellence
**Style**: Modern, scientific, systematic
**Color Scheme**: Dark navy background (#0f172a), blue accents (#3B82F6), green success highlights
**Mood**: Professional, reliable, comprehensive

**Prompt**:
```
A sophisticated 3D visualization of the testing pyramid on a dark navy (#0f172a) background. In the center, a glowing blue (#3B82F6) pyramid structure with three distinct layers: a wide base labeled "UNIT TESTS" (with code icons and green checkmarks), middle layer "INTEGRATION TESTS" (API endpoints connecting), and small top layer "E2E TESTS" (browser and user interface icons). Each layer emits soft blue light with green success indicators floating around. To the right, a circular dashboard displays "98% Coverage", "10,000+ Tests", "2 min Runtime". Holographic test reports and code quality metrics orbit the pyramid. Microscope and laboratory beaker icons subtly integrated. Hyper-realistic, scientific aesthetic, professional lighting.
```

## Image 2: Problem Visualization - Testing Nightmare
**Style**: Chaotic, buggy, unreliable
**Color Scheme**: Muted grays, red error highlights, dark tones
**Mood**: Frustrating, buggy, unstable

**Prompt**:
```
A split-screen showing software testing failures. Left side: a frustrated QA engineer surrounded by multiple screens displaying red error messages, bug reports piling up in Jira, production incidents alert dashboard with "CRITICAL BUGS IN PROD" flashing red. Sticky notes everywhere saying "RETEST", "BUG #4523", "REGRESSION". Right side shows a production server on fire (digital flames) with angry users on phones complaining. Bug icons crawling across the screens like insects. Manual test spreadsheets with 80% of boxes unchecked. Timeline showing "3 weeks to release → 2 months delayed". Muted colors with aggressive red warning highlights. Realistic, stressful atmosphere.
```

## Image 3: Solution/Future - Automated Testing Paradise
**Style**: Bright, confident, automated
**Color Scheme**: Bright blues, greens (success), whites, optimistic gradients
**Mood**: Confident, reliable, efficient

**Prompt**:
```
A beautiful modern testing command center with bright blue ambient lighting. A confident developer sits relaxed at a sleek workstation watching automated tests execute on a large curved display. The screen shows a beautiful test execution dashboard: thousands of tests running in parallel across multiple browsers and devices, all showing green checkmarks flowing like a waterfall. Code coverage visualization shows 95%+ coverage glowing in green. Test pyramid rotates in 3D showing perfect balance. Left monitor displays real-time performance metrics trending upward. Right monitor shows automatic bug detection catching issues before production. Background shows CI/CD pipeline with "All Tests Passed ✓" in large green letters. Team members in background casually drinking coffee, completely relaxed. Bright, clean, minimalist aesthetic with green success indicators everywhere.
```

---

**Design Notes**:
- Image 1: Main hero - shows WIA-COMP-013 as comprehensive testing framework
- Image 2: Problem statement - emphasizes pain points of inadequate testing
- Image 3: Future vision - demonstrates confidence and reliability from proper testing
- Blue (#3B82F6) is the primary brand color for COMP category
- Green used for test success indicators
- Scientific/laboratory aesthetic for testing precision
