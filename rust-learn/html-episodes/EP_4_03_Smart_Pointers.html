Box<T>

Box<T> - 힙 할당의 기본

Box<T> - The Basics of Heap Allocation

// 스택에 저장 (기본)
let x = 5; // i32는 스택에
// 힙에 저장 (Box 사용)
let boxed = Box::new(5); // i32가 힙에!
// 재귀적 타입 정의에 필수!
enum List {
Cons(i32, Box<List>), // 자기 참조!
Nil,
}
let list = Cons(1, Box::new(Cons(2, Box::new(Nil))));

💡 Box는 단일 소유권을 가진 힙 포인터입니다. 스코프를 벗어나면 자동 해제! 💡 Box is a heap pointer with single ownership. Auto-freed when out of scope!

Rc<T>

Rc<T> - 참조 카운팅 (공유 소유권)

Rc<T> - Reference Counting (Shared Ownership)

use std::rc::Rc;
let data = Rc::new("Hello, Rust!");
// clone()은 데이터 복사 아님! 참조 카운트만 증가
let a = Rc::clone(&data); // count: 2
let b = Rc::clone(&data); // count: 3
println!("참조 수: {}", Rc::strong_count(&data)); // 3
drop(a); // count: 2
drop(b); // count: 1
// data가 마지막이면 메모리 해제!

⚠️ Rc는 단일 스레드 전용입니다. 멀티스레드에서는 Arc를 사용하세요! ⚠️ Rc is single-threaded only. Use Arc for multi-threading!

RefCell<T>

RefCell<T> - 내부 가변성

RefCell<T> - Interior Mutability

use std::cell::RefCell;
// 불변 참조로 가변 데이터 수정!
let data = RefCell::new(5);
// borrow_mut()로 가변 참조 얻기
*data.borrow_mut() += 10;
// borrow()로 불변 참조
println!("{}", *data.borrow()); // 15
// ⚠️ 런타임에 빌림 규칙 검사!
// 동시에 여러 가변 참조 → panic!

💡 RefCell은 컴파일 타임이 아닌 런타임에 빌림 규칙을 검사합니다. 💡 RefCell checks borrowing rules at runtime, not compile time.

Rc + RefCell

Rc<RefCell<T>> - 황금 조합

Rc<RefCell<T>> - The Golden Combination

use std::rc::Rc;
use std::cell::RefCell;
// 여러 소유자가 가변 데이터 공유!
let shared = Rc::new(RefCell::new(vec![1, 2, 3]));
let a = Rc::clone(&shared);
let b = Rc::clone(&shared);
// a를 통해 수정
a.borrow_mut().push(4);
// b에서도 변경 확인!
println!("{:?}", b.borrow()); // [1, 2, 3, 4]

Smart Pointer Comparison

타입 Type 소유권 Ownership 가변성 Mutability 스레드 Thread 용도 Use Case
Box<T> 단일 Single mut 필요 Requires mut Any 힙 할당, 재귀 타입 Heap alloc, recursive types
Rc<T> 공유 (카운트) Shared (counted) 불변만 Immutable only 단일 Single 공유 읽기 전용 Shared read-only

Mutex<T> 공유 Shared 내부 가변 Interior mut 멀티 Multi 스레드 안전 가변 Thread-safe mutation

Deref

Deref 트레이트 - 역참조의 마법

Deref Trait - The Magic of Dereferencing

let boxed = Box::new("Hello");
// *로 역참조 가능! (Deref 트레이트 덕분)
println!("{}", *boxed); // "Hello"
// Deref coercion: 자동 역참조
fn greet(s: &str) { println!("Hi, {}", s); }
let name = Box::new(String::from("Rust"));
greet(&name); // Box → String → &str 자동 변환!

💡 Deref coercion 덕분에 스마트 포인터를 일반 참조처럼 사용할 수 있습니다. 💡 Deref coercion lets you use smart pointers like regular references.

  • Mutable from immutable → RefCell
  • 스레드 안전 가변 → Mutex
  • Thread-safe mutation → Mutex
