match 기초

match 표현식 기초

match Expression Basics

let number = 13;
match number {
1 => println!("One!"),
2 | 3 | 5 | 7 | 11 => println!("소수!"),
13..=19 => println!("틴에이저"),
_ => println!("기타"), // 나머지 모든 케이스
}
// match는 표현식! 값을 반환
let description = match number {
1 => "하나",
_ => "여럿",
};

💡 match는 모든 케이스를 다뤄야 합니다. 컴파일러가 빠진 케이스를 알려줍니다! 💡 match must handle all cases. The compiler tells you about missing ones!

구조 분해 패턴

Destructuring Patterns

// 튜플 분해
let point = (3, 5);
match point {
(0, y) => println!("y축 위: {}", y),
(x, 0) => println!("x축 위: {}", x),
(x, y) => println!("점: ({}, {})", x, y),
}
// 구조체 분해
struct Point { x: i32, y: i32 }
let p = Point { x: 10, y: 20 };
match p {
Point { x: 0, y } => println!("y축: {}", y),
Point { x, y: 0 } => println!("x축: {}", x),
Point { x, y } => println!("({}, {})", x, y),
}

Enum 매칭

Enum과 패턴 매칭

Enum and Pattern Matching

enum Message {
Quit,
Move { x: i32, y: i32 },
Write(String),
Color(u8, u8, u8),
}
fn process(msg: Message) {
match msg {
Message::Quit => println!("종료"),
Message::Move { x, y } => {
println!("이동: ({}, {})", x, y);
}
Message::Write(text) => println!("{}", text),
Message::Color(r, g, b) => {
println!("RGB({}, {}, {})", r, g, b);
}
}
}

매치 가드

매치 가드 (Match Guards)

Match Guards

let num = Some(4);
match num {
// if 조건으로 추가 필터링!
Some(x) if x < 5 => println!("작은 수: {}", x),
Some(x) if x >= 5 => println!("큰 수: {}", x),
None => println!("없음"),
_ => (),
}
// 외부 변수와 비교
let threshold = 10;
match num {
Some(x) if x > threshold => println!("초과!"),
_ => println!("이하"),
}

if let / while let

간결한 패턴: if let & while let

Concise Patterns: if let & while let

// if let - 단일 패턴만 처리
let value = Some(5);
// match 대신
if let Some(x) = value {
println!("값: {}", x);
} else {
println!("없음");
}
// while let - 반복 패턴
let mut stack = vec![1, 2, 3];
while let Some(top) = stack.pop() {
println!("꺼냄: {}", top);
}
// 출력: 3, 2, 1

💡 한 가지 패턴만 처리할 때는 if let이 match보다 간결합니다! 💡 When handling only one pattern, if let is more concise than match!

@ 바인딩

@ 바인딩 - 패턴과 변수 동시에

@ Bindings - Pattern and Variable at Once

enum Status {
Active(u32),
Pending,
}
let status = Status::Active(5);
match status {
// @ 로 범위 체크 + 변수 바인딩 동시에!
Status::Active(id @ 1..=5) => {
println!("작은 ID: {}", id);
}
Status::Active(id @ 6..=10) => {
println!("중간 ID: {}", id);
}
Status::Active(id) => println!("큰 ID: {}", id),
Status::Pending => println!("대기중"),
}

EP 7-01 Summary

패턴 종류

  • 리터럴: 1, "text"
  • Literals: 1, "text"
  • 범위: 1..=5, 'a'..='z'
  • Ranges: 1..=5, 'a'..='z'
  • 구조분해: (x, y), Point {x, y}
  • Destructure: (x, y), Point {x, y}
  • OR 패턴: 1 | 2 | 3
  • OR pattern: 1 | 2 | 3

고급 기능

  • 매치 가드: if 조건
  • Match guards: if condition
  • @ 바인딩: id @ 1..=5
  • @ binding: id @ 1..=5
  • if let: 간결한 단일 매칭
  • if let: concise single match
  • while let: 반복 매칭
  • while let: loop matching
