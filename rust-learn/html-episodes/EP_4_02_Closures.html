클로저 기본 문법

Basic Closure Syntax

// 일반 함수
fn add_one(x: i32) -> i32 { x + 1 }
// 클로저 (동일한 기능)
let add_one = |x: i32| -> i32 { x + 1 };
// 타입 추론 활용 (더 간결하게)
let add_one = |x| x + 1;
// 호출
let result = add_one(5); // 6

💡 클로저는 |매개변수| 문법을 사용합니다. 타입 추론 덕분에 간결하게 작성 가능! 💡 Closures use |parameters| syntax. Type inference makes them concise!

Capturing Environment - The Core of Closures!

fn main() {
let x = 4;
let y = 10;
// 클로저가 x와 y를 "캡처"!
let add_to_x = |n| n + x;
let multiply_by_y = |n| n * y;
println!("{}", add_to_x(5)); // 9
println!("{}", multiply_by_y(3)); // 30
}
// 일반 함수는 이렇게 못함!

캡처 방식

세 가지 캡처 방식

Three Capture Modes

Trait 캡처 방식 Capture Mode 설명 Description
Fn &T (불변 참조) &T (immutable ref) 값을 읽기만 함 Only reads the value
FnMut &mut T (가변 참조) &mut T (mutable ref) 값을 수정함 Modifies the value
FnOnce T (소유권 이동) T (ownership move) 값의 소유권을 가져감 Takes ownership of value
let s = String::from("hello");
// move 키워드로 강제 소유권 이동
let consume = move || println!("{}", s);
// println!("{}", s); // ❌ s는 이미 move됨!
consume(); // ✅ "hello"

Iterator와 함께

Iterator와 클로저 - 환상의 조합!

Iterators & Closures - Perfect Combination!

let numbers = vec![1, 2, 3, 4, 5];
// map: 각 요소 변환
let doubled: Vec<i32> = numbers.iter()
.map(|x| x * 2)
.collect(); // [2, 4, 6, 8, 10]
// filter: 조건에 맞는 요소만
let evens: Vec<&i32> = numbers.iter()
.filter(|x| *x % 2 == 0)
.collect(); // [2, 4]
// 체이닝!
let sum: i32 = numbers.iter()
.filter(|x| *x > 2).map(|x| x * 2).sum(); // 24

함수 매개변수

클로저를 함수 인자로 받기

Accepting Closures as Function Parameters

// 제네릭 + 트레이트 바운드로 클로저 받기
fn apply_twice<F>(f: F, x: i32) -> i32
where
F: Fn(i32) -> i32
{
f(f(x))
}
fn main() {
let add_five = |x| x + 5;
let result = apply_twice(add_five, 10);
println!("{}", result); // 20
}

💡 Fn, FnMut, FnOnce 트레이트로 클로저 타입을 표현합니다. 💡 Use Fn, FnMut, FnOnce traits to express closure types.

클로저 반환

함수에서 클로저 반환하기

Returning Closures from Functions

// impl Trait으로 클로저 반환
fn make_adder(n: i32) -> impl Fn(i32) -> i32 {
move |x| x + n
}
// Box로 동적 디스패치 (다양한 클로저 반환시)
fn make_operation(op: &str) -> Box<dyn Fn(i32) -> i32> {
match op {
"double" => Box::new(|x| x * 2),
"square" => Box::new(|x| x * x),
_ => Box::new(|x| x),
}
}
let add_10 = make_adder(10);
println!("{}", add_10(5)); // 15
