# WIA-RUST-LEARN-PRIMER — Rust Learn (Press Brief)

**WIA Standards releases canonical Rust Learn envelope shape**

The WIA Standards Committee has published the WIA-RUST-LEARN-PRIMER standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for rust learn hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
