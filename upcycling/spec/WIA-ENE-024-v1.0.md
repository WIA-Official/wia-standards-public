## 1. 개요

### 1.1 목적

WIA-ENE-024 업사이클링 표준은 폐기물을 더 높은 가치의 제품으로 전환하는 프로세스를 표준화하여, 순환경제 실현과 지속가능한 자원 활용을 촉진하기 위해 제정되었습니다.

### 1.2 업사이클링의 정의

**업사이클링(Upcycling)**은 폐기물이나 버려진 제품을 단순히 재활용(Recycling)하는 것이 아니라, 창의적인 전환 과정을 통해 원래보다 더 높은 품질과 가치를 가진 새로운 제품으로 탄생시키는 것을 의미합니다.

#### 업사이클링 vs 리사이클링

| 구분 | 업사이클링 | 리사이클링 |
|------|-----------|-----------|
| **가치** | 원재료보다 높은 가치 창출 | 원재료 수준으로 회귀 |
| **품질** | 향상된 품질 | 동등 또는 저하된 품질 |
| **과정** | 창의적 전환 디자인 | 분해 후 재가공 |
| **에너지** | 상대적으로 낮은 에너지 소비 | 높은 에너지 소비 |
| **결과물** | 고유한 제품, 예술작품 | 표준화된 재생 원료 |

### 1.3 적용 범위

본 표준은 다음 분야에 적용됩니다:

- **섬유/패션**: 의류, 가방, 액세서리 업사이클링
- **가구/인테리어**: 폐가구, 팔레트, 목재 업사이클링
- **플라스틱**: 폐플라스틱을 활용한 제품 제작
- **금속**: 폐금속을 활용한 예술작품 및 기능성 제품
- **전자제품**: 전자폐기물(E-waste)의 부품 재활용
- **건축자재**: 건설 폐기물의 건축자재 전환
- **유리/도자기**: 깨진 유리, 도자기의 예술적 재탄생
- **종이/골판지**: 폐지를 활용한 디자인 제품

### 1.4 철학

업사이클링은 **弘益人間 (홍익인간)** 정신을 실천하는 구체적 방법입니다:

- **환경 보호**: 폐기물 매립 및 소각 감소
- **자원 절약**: 신규 원자재 채굴 최소화
- **창조적 가치**: 예술과 기능의 융합
- **사회적 기여**: 일자리 창출 및 사회적 기업 활성화
- **교육적 가치**: 자원순환 의식 고취

### 2.2 단계별 상세 설명

#### 2.2.1 원료 수집 (Source Collection)

**목적**: 업사이클링 가능한 폐기물 확보

**주요 활동**:
- 원료 소싱 채널 확보 (재활용센터, 기업, 개인)
- 원료 상태 평가
- 수집 물류 최적화
- 안전한 보관

**품질 기준**:
- 구조적 손상 ≤ 30%
- 오염 수준: 세척 가능 범위
- 안전성: 유해물질 미포함

#### 2.2.2 분류 및 선별 (Sorting)

**목적**: 업사이클링 적합성 판단 및 재료 분류

**분류 기준**:

| 등급 | 설명 | 적합성 | 처리 방법 |
|------|------|--------|----------|
| A급 | 거의 손상 없음 | 매우 높음 | 고가치 제품 전환 |
| B급 | 부분 손상/변색 | 높음 | 중가치 제품 전환 |
| C급 | 상당한 손상 | 보통 | 부품 추출 또는 소재 활용 |
| D급 | 심각한 손상 | 낮음 | 리사이클링으로 전환 |

**선별 프로세스**:
1. 육안 검사
2. 재질 확인
3. 크기/규격 측정
4. 오염도 평가
5. 업사이클링 가능성 판정

#### 2.2.3 세척 및 준비 (Preparation)

**목적**: 업사이클링을 위한 재료 준비

**세척 방법**:
- **섬유류**: 친환경 세제 사용, 자연 건조
- **플라스틱**: 중성세제 세척, 열풍 건조
- **금속**: 녹 제거, 표면 처리
- **목재**: 샌딩, 방부/방충 처리

**준비 작업**:
- 불필요한 부분 제거
- 해체 및 부품 분리
- 표면 처리
- 임시 보관

#### 2.2.4 디자인 (Design)

**목적**: 창의적 가치 부여 및 제품 기획

**디자인 원칙**:
1. **기능성 우선**: 실용적 가치 확보
2. **미적 가치**: 독특한 디자인 감각
3. **스토리텔링**: 재료의 이야기 보존
4. **내구성**: 장기 사용 가능
5. **안전성**: 사용자 안전 보장

**디자인 프로세스**:
```
재료 분석 → 아이디어 스케치 → 프로토타입 제작
→ 사용성 테스트 → 최종 디자인 확정
```

#### 2.2.5 제작 (Production)

**목적**: 디자인을 실제 제품으로 구현

**제작 방법**:
- **수작업**: 장인 기술, 맞춤 제작
- **세미오토**: 기계 보조, 소량 생산
- **자동화**: 표준화된 대량 생산

**품질 관리**:
- 작업 표준 준수
- 중간 검수
- 결함 최소화
- 재작업 기준 적용

#### 2.2.6 품질 검증 (Quality Assurance)

**검증 항목**:

| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| **구조 안정성** | 하중 테스트 통과 | 실물 테스트 |
| **표면 마감** | 거칠기 ≤ 기준값 | 육안/측정 |
| **안전성** | 날카로운 부분 없음 | 촉감 검사 |
| **내구성** | 사용 수명 예측 | 가속 노화 테스트 |
| **친환경성** | VOC 방출량 기준 충족 | 화학 분석 |

#### 2.2.7 판매 및 유통 (Distribution)

**유통 채널**:
- 온라인 마켓플레이스
- 팝업 스토어
- 갤러리/전시회
- 기업 제휴 (CSR 구매)
- 크라우드펀딩

## 5. 환경 영향 평가

### 5.1 계산 방법론

업사이클링의 환경 영향은 **전과정평가(LCA, Life Cycle Assessment)** 방법론을 기반으로 합니다.

#### 5.1.1 폐기물 감소량

```
폐기물 감소량 (kg) = 원료 재료 무게 (kg)
```

#### 5.1.2 CO₂ 배출 저감량

```
CO₂ 저감량 (kg) = [신제품 생산 CO₂ - 업사이클링 CO₂]
```

**재료별 신제품 생산 배출 계수**:

| 재료 | CO₂ 배출량 (kg CO₂/kg 재료) |
|------|---------------------------|
| Cotton (면) | 8.0 |
| Polyester (폴리에스터) | 6.4 |
| Denim (데님) | 10.0 |
| Plastic (플라스틱) | 6.0 |
| Steel (철강) | 2.0 |
| Aluminum (알루미늄) | 9.0 |
| Wood (목재) | 0.5 |
| Glass (유리) | 0.85 |

**업사이클링 프로세스 배출 계수**:
- 세척: 0.2 kg CO₂/kg 재료
- 제작 (수작업): 0.1 kg CO₂/kg 재료
- 제작 (기계): 0.3 kg CO₂/kg 재료

#### 5.1.3 물 절약량

```
물 절약량 (L) = 신제품 생산 물 사용량 - 업사이클링 물 사용량
```

**재료별 신제품 물 사용량**:

| 재료 | 물 사용량 (L/kg) |
|------|-----------------|
| Cotton | 10,000 |
| Denim | 11,000 |
| Polyester | 80 |
| Paper | 300 |

#### 5.1.4 에너지 절약량

```
에너지 절약량 (kWh) = 신제품 생산 에너지 - 업사이클링 에너지
```

### 5.2 환경 인증

업사이클링 제품은 다음 인증을 획득할 수 있습니다:

| 인증 | 발급 기관 | 기준 |

## 7. 경제성 평가

### 7.1 가치 배율 (Value Multiplier)

```
가치 배율 = 최종 판매가 / 원료 획득 비용
```

**벤치마크**:
- 우수: ≥ 10배
- 양호: 5-10배
- 보통: 3-5배
- 저조: < 3배

### 7.2 비용 구조

| 비용 항목 | 비중 (평균) | 최적화 방법 |
|----------|------------|------------|
| 원료 획득 | 5-10% | 기부/제휴 확대 |
| 세척/준비 | 10-15% | 프로세스 효율화 |
| 디자인 | 5-8% | 모듈화 디자인 |
| 제작 인건비 | 40-50% | 교육/숙련도 향상 |
| 부자재 | 10-15% | 공동 구매 |
| 마케팅 | 10-15% | SNS 활용 |
| 유통 | 5-10% | 온라인 플랫폼 |

### 7.3 가격 책정 전략

```
권장 판매가 = (총 원가 × 2.5) + 예술적 프리미엄
```

**예술적 프리미엄 요소**:
- 독창적 디자인: +20-50%
- 유명 디자이너: +30-100%
- 스토리텔링: +10-30%
- 한정판: +50-200%
- 사회적 가치: +10-20%

3. **글로벌 표준**: WIA-ENE-024가 국제 표준으로 채택
4. **교육 통합**: 초중고 교육과정에 업사이클링 포함
5. **문화 정착**: "새 제품보다 업사이클 제품"이 자랑
