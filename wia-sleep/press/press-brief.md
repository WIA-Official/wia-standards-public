# WIA-HEALTH-SLEEP — WIA Sleep (Press Brief)

**WIA Standards releases canonical WIA Sleep envelope shape**

The WIA Standards Committee has published the WIA-HEALTH-SLEEP standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for wia sleep hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
