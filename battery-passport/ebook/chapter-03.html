Chapter 3: Calculation Algorithms

Carbon Footprint, State of Health, and Recycled Content Verification

3.1 Carbon Footprint Calculation

The carbon footprint is calculated using the Product Environmental Footprint (PEF) methodology:

CFtotal = CFraw + CFmanufacturing + CFtransport + CFassembly
O(n) lifecycle stages

Algorithm: calculate_carbon_footprint

  1. Sum emissions from raw material extraction
  2. Add manufacturing process emissions
  3. Calculate transport emissions based on distance and mode
  4. Include assembly and packaging emissions
  5. Apply recycled content credits
  6. Normalize to kg CO2eq/kWh
carbon_calculator.rs
pub struct CarbonCalculator;

impl CarbonCalculator {
    pub fn calculate(
        materials: &MaterialComposition,
        manufacturing: &ManufacturingData,
        transport: &TransportData,
        recycled: &RecycledContent,
    ) -> CarbonFootprintResult {
        // Raw material emissions (kg CO2eq)
        let raw_emissions = Self::calculate_raw_materials(materials);

        // Manufacturing emissions based on energy source
        let mfg_emissions = Self::calculate_manufacturing(manufacturing);

        // Transport emissions (distance * emission factor)
        let transport_emissions = transport.distance_km as f32
            * Self::get_transport_factor(transport.mode);

        // Apply recycled content credit (reduces footprint)
        let recycled_credit = Self::calculate_recycled_credit(recycled);

        let total = raw_emissions + mfg_emissions + transport_emissions - recycled_credit;
        let per_kwh = total / manufacturing.capacity_kwh;

        CarbonFootprintResult {
            total_kg_co2eq: total,
            per_kwh,
            performance_class: Self::get_performance_class(per_kwh),
        }
    }

    fn get_performance_class(per_kwh: f32) -> PerformanceClass {
        match per_kwh {
            x if x < 50.0 => PerformanceClass::A,
            x if x < 65.0 => PerformanceClass::B,
            x if x < 80.0 => PerformanceClass::C,
            _ => PerformanceClass::D,
        }
    }
}

3.2 State of Health (SoH) Algorithm

State of Health indicates remaining battery capacity compared to original:

SoH (%) = (Current Capacity / Original Capacity) x 100

[KEY] SoH Thresholds

  • [OK] >= 80%: Primary use (EV application)
  • [INFO] 70-80%: Second-life candidate
  • [RECYCLE] < 70%: Recycling recommended
soh_calculator.rs
pub fn calculate_soh(
    current_capacity_ah: f32,
    original_capacity_ah: f32,
    cycle_count: u32,
    calendar_age_days: u32,
) -> SoHResult {
    // Basic SoH from capacity fade
    let capacity_soh = (current_capacity_ah / original_capacity_ah) * 100.0;

    // Cycle degradation estimate
    let cycle_degradation = (cycle_count as f32 / 1500.0) * 20.0;

    // Calendar aging estimate (0.5-1% per year)
    let calendar_degradation = (calendar_age_days as f32 / 365.0) * 0.8;

    // Combined SoH (use measured capacity as primary)
    let soh = capacity_soh;

    // Estimate remaining useful life
    let remaining_cycles = ((soh - 70.0) / 20.0 * 1500.0).max(0.0) as u32;

    SoHResult {
        soh_percent: soh,
        status: match soh {
            x if x >= 80.0 => BatteryStatus::PrimaryUse,
            x if x >= 70.0 => BatteryStatus::SecondLife,
            _ => BatteryStatus::Recycling,
        },
        remaining_cycles,
        estimated_eol_date: calculate_eol_date(remaining_cycles),
    }
}

3.3 Recycled Content Verification

[EU] 2031 Minimum Requirements

  • Cobalt: >= 16%
  • Lithium: >= 6%
  • Nickel: >= 6%
  • Lead: >= 85% (for lead-acid)
recycled_content_verifier.rs
pub fn verify_recycled_content(
    content: &RecycledContent,
    target_year: u32,
) -> VerificationResult {
    let requirements = get_requirements_for_year(target_year);

    let mut compliant = true;
    let mut violations = Vec::new();

    if content.cobalt < requirements.cobalt {
        compliant = false;
        violations.push(format!(
            "Cobalt: {}% (required: {}%)",
            content.cobalt, requirements.cobalt
        ));
    }

    if content.lithium < requirements.lithium {
        compliant = false;
        violations.push(format!(
            "Lithium: {}% (required: {}%)",
            content.lithium, requirements.lithium
        ));
    }

    VerificationResult {
        compliant,
        violations,
        margin: calculate_compliance_margin(content, &requirements),
    }
}
