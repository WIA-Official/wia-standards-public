Chapter 5

Carbon Footprint Calculation

Chapter Overview: This chapter details the methodologies for calculating and reporting the carbon footprint of batteries in compliance with EU regulations and international standards. Accurate carbon accounting is essential for regulatory compliance and enables informed consumer choices.

Regulatory Framework for Carbon Footprint

The EU Battery Regulation requires carbon footprint declaration for EV batteries and industrial batteries above 2 kWh capacity. Starting from 2027, batteries must display a carbon footprint performance class, and maximum thresholds will be enforced. This creates significant implications for battery manufacturers and importers.

The WIA-BATTERY_PASSPORT standard provides a complete implementation of carbon footprint calculation methodologies aligned with EU requirements while supporting additional international frameworks including ISO 14067 and the Greenhouse Gas Protocol.

Lifecycle Assessment Boundaries

Carbon footprint calculation follows a lifecycle assessment (LCA) approach covering all significant emission sources from raw material extraction through end-of-life processing. The system boundary must capture all relevant activities.

Phase 1: Raw Material Acquisition

Includes mining, extraction, and primary processing of all battery materials. This phase typically represents 35-50% of total carbon footprint for NMC batteries.

  • Mining operations and equipment
  • Material extraction processes (brine, hard rock)
  • Primary refining and concentration
  • Transportation to processing facilities

Phase 2: Manufacturing

Covers all production activities from precursor synthesis through battery pack assembly. Energy source mix significantly impacts this phase.

  • Precursor material synthesis
  • Cathode and anode active material production
  • Cell manufacturing (electrode coating, assembly, formation)
  • Module and pack assembly

Phase 3: Distribution

Transportation from manufacturing facility to point of sale. Typically a smaller contributor but varies with supply chain geography.

  • Factory-to-warehouse transport
  • International shipping
  • Last-mile delivery

Phase 4: End of Life

Recycling and disposal activities. Credits for recovered materials can offset some lifecycle emissions.

  • Collection and transportation
  • Recycling processes
  • Material recovery credits
  • Waste disposal for non-recoverable fractions

Calculation Methodology

The carbon footprint calculation follows the Product Environmental Footprint (PEF) methodology adapted for batteries as specified in EU delegated acts. Key calculation steps are standardized to ensure comparability across products.

Emission Factor Application

// Carbon Footprint Calculation Engine
interface CarbonCalculator {
  calculatePhaseEmissions(
    phase: LifecyclePhase,
    activityData: ActivityData,
    emissionFactors: EmissionFactorSet
  ): PhaseEmissions;

  calculateTotal(phaseEmissions: PhaseEmissions[]): TotalFootprint;
  assignPerformanceClass(footprint: number): string;
}

class WIACarbonCalculator implements CarbonCalculator {
  private readonly efDatabase: EmissionFactorDatabase;
  private readonly allocationRules: AllocationRules;

  calculatePhaseEmissions(
    phase: LifecyclePhase,
    activityData: ActivityData,
    emissionFactors: EmissionFactorSet
  ): PhaseEmissions {
    const emissions: EmissionItem[] = [];

    for (const activity of activityData.activities) {
      // Get appropriate emission factor
      const ef = this.efDatabase.getEmissionFactor(
        activity.type,
        activity.region,
        activity.year
      );

      // Calculate emissions
      const emission = {
        source: activity.name,
        quantity: activity.quantity,
        unit: activity.unit,
        emissionFactor: ef.value,
        efUnit: ef.unit,
        efSource: ef.source,
        emissions: activity.quantity * ef.value,
        uncertainty: this.calculateUncertainty(activity, ef)
      };

      emissions.push(emission);
    }

    // Apply allocation if needed (e.g., co-products)
    const allocatedEmissions = this.allocationRules.apply(
      phase,
      emissions
    );

    return {
      phase: phase,
      totalEmissions: allocatedEmissions.reduce(
        (sum, e) => sum + e.emissions, 0
      ),
      items: allocatedEmissions,
      dataQuality: this.assessDataQuality(emissions)
    };
  }

  assignPerformanceClass(footprint: number): string {
    // EU Battery Regulation performance classes
    if (footprint <= 50) return 'A';
    if (footprint <= 60) return 'B';
    if (footprint <= 70) return 'C';
    if (footprint <= 80) return 'D';
    if (footprint <= 90) return 'E';
    return 'F';
  }
}

Carbon Footprint Performance Classes

Starting in 2027, batteries must display a carbon footprint performance class using a label system similar to energy efficiency ratings. Classes range from A (best) to F (worst).

Performance Class Carbon Footprint Range Market Position
A Less than or equal to 50 kgCO2eq/kWh Best in class, premium positioning
B 51-60 kgCO2eq/kWh Above average, competitive
C 61-70 kgCO2eq/kWh Average market performance
D 71-80 kgCO2eq/kWh Below average
E 81-90 kgCO2eq/kWh At threshold limit (2027)
F Greater than 90 kgCO2eq/kWh Non-compliant from 2027

Reduction Strategies

Manufacturers can reduce carbon footprint through multiple strategies across the lifecycle. The WIA standard tracks the impact of decarbonization initiatives.

Key Reduction Levers: Renewable energy in manufacturing (up to 30% reduction), recycled content (up to 15% reduction), optimized logistics (5-8% reduction), and sustainable material sourcing (10-20% reduction depending on material).
  • Renewable Energy: Transitioning manufacturing facilities to renewable electricity
  • Recycled Content: Increasing use of recycled materials reduces upstream emissions
  • Process Efficiency: Optimizing energy-intensive processes like formation and dry rooms
  • Supply Chain Optimization: Localizing supply chains and optimizing logistics
  • Low-Carbon Materials: Sourcing materials from suppliers with lower carbon intensity
