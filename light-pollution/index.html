About this standard

The WIA Light Pollution standard codifies the data formats, APIs, and protocols required for interoperable implementations across vendors, regions, and operational scales. It is part of the WIA (World Certification Industry Association) family of open standards and is governed by the WIA Citation Policy.

Conformance tiers

  • Conformant — implementation publishes valid records against the OpenAPI contract.
  • Audited — passes the WIA conformance CLI test suite.
  • Certified — registered with WIA at cert.wiastandards.com and exposes a public credential.

Where to start

  • Spec — light-pollution/spec/ (Phase 1–4 documents)
  • OpenAPI contract — light-pollution/api/openapi-3.0.yaml
  • SDKs — light-pollution/api/sdk/typescript/ and light-pollution/api/sdk/python/
  • CLI — light-pollution/cli/light-pollution.sh
  • Long-form docs — light-pollution/ebook/

Open governance

All citations in this standard's spec and ebook conform to the WIA Citation Policy: only verifiable primary sources (ISO, IEC, IEEE, RFC, W3C, HL7, vendor official documentation, and public datasets) appear in body text. Academic placeholder references have been removed under the Phase 3 Veracity 100% effort completed 2026-04-25.
