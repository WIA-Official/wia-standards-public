💡 Light Monitoring API & Standards

Standardized data formats, RESTful APIs, Bortle scale classification, and the International Dark-Sky Association certification framework for measuring and communicating light pollution levels.

The Need for Data Standards

Light pollution monitoring generates diverse data types—illuminance measurements, sky brightness readings, spectral distributions, temporal patterns, and geographic distributions. Without standardized formats and interfaces, this data remains siloed in proprietary systems, preventing cross-comparison, meta-analysis, and integration into urban planning and environmental assessment workflows. The WIA-ENE-029 Light Monitoring API establishes common data schemas, RESTful web service specifications, and interoperability standards enabling seamless integration across measurement instruments, management platforms, and analytical tools.

WIA-ENE-029 Data Schema

The core data model structures light pollution measurements as hierarchical entities: Sites (geographic locations with metadata), Instruments (measurement devices with specifications and calibration), Observations (individual measurements with timestamps and values), and Campaigns (coordinated measurement programs across multiple sites and times). This structure supports both real-time monitoring and historical analysis while preserving measurement context essential for proper interpretation.

Site Definition

Each monitoring site includes precise geolocation (WGS84 latitude/longitude, elevation), environmental classification (urban core, suburban, rural, pristine wilderness), nearby light sources (within 1 km, 5 km, 20 km radii), vegetation and topography affecting sky view, and administrative metadata (site owner, access restrictions, installation date). This contextual information enables comparison of equivalent sites and controls for confounding variables in temporal trend analysis.

Measurement Records

Individual observations capture measured values with full metadata: timestamp (UTC, ISO 8601 format), instrument ID and calibration status, measurement type (illuminance, luminance, sky brightness, spectral), environmental conditions (temperature, humidity, cloud cover, moon phase), and data quality flags (valid, suspect, invalid, estimated). Spectral measurements include wavelength arrays and corresponding intensity values. Time-series data employs standard sampling intervals (1 minute, 5 minutes, hourly, nightly means) with statistical summaries (min, max, mean, median, standard deviation).

RESTful API Specification

The WIA-ENE-029 API follows REST architectural principles, exposing resources via standard HTTP methods (GET, POST, PUT, DELETE) with JSON payloads and OAuth 2.0 authentication. Base endpoints provide CRUD operations on sites, instruments, and measurements, while derived endpoints offer analytical queries, statistical summaries, and export capabilities.

The Bortle Dark-Sky Scale

While quantitative measurements (SQM readings in mag/arcsec²) provide precise data, the Bortle Scale translates technical values into intuitive descriptions and observable astronomical phenomena, making light pollution levels accessible to non-specialists. Developed by amateur astronomer John Bortle in 2001, this nine-class scale has become the standard qualitative assessment tool worldwide.

International Dark-Sky Association Certification

The International Dark-Sky Association (IDA), founded in 1988, administers the world's premier dark sky certification programs, recognizing communities, parks, and reserves that demonstrate exceptional commitment to preserving night skies through responsible lighting policies and public education. IDA certification requires meeting quantitative lighting standards, implementing compliant fixtures, and demonstrating long-term commitment to dark-sky preservation.

IDA Dark Sky Community Program

Communities seeking certification must demonstrate: comprehensive outdoor lighting ordinances meeting IDA standards, majority (>50%) of outdoor lighting compliant with full-cutoff requirements, documented sky quality measurements showing Bortle Class 3-4 or better within community boundaries, public education programs on light pollution, and formal commitment to maintaining standards in perpetuity. As of 2025, approximately 200 communities worldwide hold IDA Dark Sky Community certification, ranging from small towns to entire island nations.

IDA Dark Sky Park and Reserve Programs

Parks and protected areas with exceptional night sky quality and commitment to preservation can earn Gold, Silver, or Bronze tier certification. Gold tier requires sky brightness ≥21.5 mag/arcsec² (Bortle Class 2), comprehensive lighting management plans, fixture retrofitting on existing infrastructure, and regular sky quality monitoring. These designations drive astrotourism—visitors traveling specifically to experience dark skies—generating economic benefits that incentivize preservation.

European Standards and Regulations

The European Committee for Standardization (CEN) and individual nations have developed complementary standards addressing light pollution. The Italian law (Legge Regionale n. 17/2000 - Lombardy, subsequently adopted by other regions) established among the world's first comprehensive legal frameworks limiting upward light emissions, requiring shielding, and mandating energy-efficient technologies. Slovenia's 2007 national legislation (Official Gazette RS, No. 81/2007) comprehensively addresses light pollution with specific limits for different zone types.

CIE Technical Standards

The International Commission on Illumination (CIE) publishes technical reports guiding outdoor lighting design: CIE 150:2017 "Guide on the Limitation of the Effects of Obtrusive Light from Outdoor Lighting Installations" provides design criteria for controlling light trespass, sky glow, and glare. CIE Technical Report 226:2017 addresses optical safety of LEDs. These standards inform national codes and municipal ordinances worldwide.

WIA-ENE-029 API Integration Example

// Fetch current sky brightness for a site
async function getSkyBrightness(siteId: string): Promise {
  const response = await fetch(
    `https://api.wia-ene-029.org/v1/measurements?` +
    `site=${siteId}&type=sky_brightness&latest=true`,
    {
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Accept': 'application/json'
      }
    }
  );

  const data = await response.json();
  return data.measurements[0].sky_brightness.value;
}

// Submit a measurement
async function submitMeasurement(measurement: Measurement): Promise {
  const response = await fetch(
    'https://api.wia-ene-029.org/v1/measurements',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(measurement)
    }
  );

  if (!response.ok) {
    throw new Error(`Failed to submit: ${response.statusText}`);
  }
}

// Get trend analysis
async function analyzeTrends(
  siteId: string,
  years: number
): Promise {
  const response = await fetch(
    `https://api.wia-ene-029.org/v1/trends?` +
    `site=${siteId}&years=${years}&metric=sky_brightness`,
    {
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Accept': 'application/json'
      }
    }
  );

  const data = await response.json();
  return {
    slope: data.linear_trend.slope_mag_per_year,
    r_squared: data.linear_trend.r_squared,
    significance: data.statistical_significance
  };
}

💡 Key Takeaways

  • Standardized data schemas enable interoperability between instruments, platforms, and analytical tools for light pollution monitoring.
  • RESTful APIs provide programmatic access to measurement data, supporting real-time monitoring, historical analysis, and integration with urban planning tools.
  • The Bortle Dark-Sky Scale translates quantitative SQM measurements into intuitive 9-class descriptions with observable astronomical phenomena.
  • IDA certification programs recognize communities, parks, and reserves demonstrating exceptional dark-sky preservation through policy and infrastructure.
  • Gold-tier IDA Dark Sky Parks require ≥21.5 mag/arcsec² sky brightness, representing Bortle Class 2 conditions.
  • European standards (CIE 150, national laws) provide legal frameworks limiting upward emissions, requiring shielding, and mandating efficiency.
  • Comprehensive site metadata—location, environment, calibration, conditions—is essential for proper interpretation and comparison of measurements.
  • Open data standards accelerate research and policy by enabling meta-analysis across regions and time periods.
