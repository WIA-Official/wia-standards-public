Chapter 3: Sensors and Payloads

Sensors and payloads transform drones from flying platforms into powerful agricultural data collection systems. This chapter covers the spectrum of imaging sensors used in agricultural applications, from standard RGB cameras to sophisticated multispectral, thermal, and LiDAR systems. Understanding sensor capabilities, spectral characteristics, and calibration requirements is essential for extracting actionable insights from drone-collected data.

3.1 RGB Camera Systems

// RGB Camera Specifications for Agriculture
interface RGBCameraSpecs {
  // Sensor Specifications
  sensor: {
    type: 'CMOS' | 'CCD';
    size: string;                // e.g., '1-inch', 'APS-C', 'Full-frame'
    resolution: {
      megapixels: number;
      width: number;             // pixels
      height: number;            // pixels
    };
    pixelSize: number;           // micrometers
  };

  // Lens Specifications
  lens: {
    focalLength: number;         // mm
    fieldOfView: number;         // degrees diagonal
    aperture: string;            // e.g., 'f/2.8'
    distortion: string;          // e.g., 'rectilinear', 'fisheye'
    focusType: 'fixed' | 'autofocus';
  };

  // Image Quality
  quality: {
    dynamicRange: number;        // stops
    colorDepth: number;          // bits
    formats: ('JPEG' | 'RAW' | 'DNG' | 'TIFF')[];
    geotagging: boolean;
  };

  // Performance
  performance: {
    maxFrameRate: number;        // fps
    minShutterSpeed: string;     // e.g., '1/8000'
    isoRange: { min: number; max: number };
    triggerInterval: number;     // seconds
  };
}

3.3 Thermal Imaging

Thermal Imaging Applications

  • Water Stress Detection: Canopy temperature indicates transpiration
  • Irrigation Management: Identify under/over-watered zones
  • Disease Detection: Temperature anomalies from infections
  • Drainage Assessment: Detect wet areas and drainage issues
  • Frost Monitoring: Identify frost-prone microclimates
// Thermal Camera Specifications
interface ThermalCamera {
  detector: {
    type: 'uncooled_VOx' | 'uncooled_aSi' | 'cooled';
    resolution: { width: number; height: number };
    pixelPitch: number;          // micrometers
    spectralRange: { min: number; max: number }; // micrometers
  };

  performance: {
    thermalSensitivity: number;  // mK (NETD)
    temperatureRange: { min: number; max: number }; // Celsius
    accuracy: number;            // ± Celsius
    frameRate: number;           // Hz
  };

  calibration: {
    nuc: boolean;                // Non-Uniformity Correction
    shutterCalibration: boolean;
    radiometric: boolean;
    emissivityCorrection: boolean;
  };

  output: {
    formats: ('R-JPEG' | 'TIFF' | 'radiometric_raw')[];
    temperatureData: boolean;
    geotags: boolean;
  };
}

3.7 Sensor Calibration

Calibration Requirements

  • Radiometric: Convert DN to reflectance using calibration panels
  • Geometric: Lens distortion correction and camera alignment
  • Thermal: Blackbody calibration for accurate temperature
  • Spectral: Wavelength calibration for hyperspectral
  • Temporal: Time synchronization between sensors
// Calibration Workflow
interface CalibrationWorkflow {
  preFlightCalibration: {
    reflectancePanel: {
      material: 'Spectralon' | 'gray_card';
      knownReflectance: number[];  // Per band
      captureConditions: 'Consistent lighting';
    };
    thermalReference: {
      type: 'blackbody' | 'ambient';
      knownTemperature: number;
    };
    imuAlignment: {
      levelCalibration: boolean;
      magnetometerCalibration: boolean;
    };
  };

  inFlightCalibration: {
    dls: {
      purpose: 'Account for changing light conditions';
      mounting: 'Top of aircraft, level';
      synchronization: 'Time-matched to camera';
    };
    gcp: {
      purpose: 'Accurate georeferencing';
      distribution: 'Corners and center of field';
      accuracy: 'Survey-grade GNSS';
    };
  };

  postProcessingCalibration: {
    radiometricCorrection: 'Apply DLS and panel data';
    atmosphericCorrection: 'Optional for reflectance';
    bandAlignment: 'For multi-sensor systems';
  };
}

Chapter Summary

This chapter covered the spectrum of sensors and payloads used in agricultural drones, from RGB cameras to multispectral, thermal, LiDAR, and hyperspectral systems. Understanding sensor specifications, calibration requirements, and integration considerations is essential for collecting high-quality agricultural data as specified by the WIA-AGRICULTURAL_DRONE standard.
