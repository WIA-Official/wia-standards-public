4.1 실시간 EMG 스트리밍

WIA 표준은 저지연, 고처리량 EMG 데이터 전송을 위한 실시간 스트리밍 프로토콜을 정의합니다.

4.1.1 WebSocket 프로토콜

EMG 데이터 스트리밍에 권장되는 프로토콜입니다:

wss://api.myoelectric.device/v1/stream/emg
?deviceId=EMG-SENSOR-001
&channels=0,1,2,3
&sampleRate=1000

프로토콜 특징

  • 저지연: < 10ms 데이터 전송
  • 양방향: 클라이언트-서버 양방향 통신
  • 압축: 선택적 데이터 압축 (gzip, brotli)
  • 보안: TLS 1.3 암호화 필수

4.1.2 메시지 형식

{
  "type": "emg_data",
  "timestamp": 1705312800123,

4.2 신호 전처리 파이프라인

표준 EMG 신호 처리 워크플로우:

4.2.1 노치 필터

전력선 간섭 제거 (50 Hz 또는 60 Hz):

매개변수 값 설명
중심 주파수 50/60 Hz 전력선 주파수
Q 팩터 30 필터 대역폭 제어
고조파 2차, 3차 추가 노치 (100/120, 150/180 Hz)

4.2.2 대역통과 필터

EMG 주파수 대역 분리 (20-450 Hz):

  • 하한 주파수: 20 Hz (동작 아티팩트 제거)
  • 상한 주파수: 450 Hz (EMG 대역 보존)
  • 필터 유형: 4차 Butterworth
  • 제로 위상: 순방향-역방향 필터링

4.2.3 정류 및 엔벨로프 추출

# 정류
emg_rectified = abs(emg_filtered)

# 선형 엔벨로프 (저역통과 필터)
cutoff = 5 Hz  # 평활 주파수
emg_envelope = lowpass_filter(emg_rectified, cutoff)

4.2.4 정규화

MVC 기준 정규화:

emg_normalized = (emg_envelope - rest_baseline) / (mvc_value - rest_baseline)

4.3 특징 추출

패턴 인식을 위한 EMG 특징:

4.3.1 시간 영역 특징

특징 약어 공식
평균 절대값 MAV Σ|x[n]| / N
제곱 평균 제곱근 RMS √(Σx²[n] / N)
파형 길이 WL Σ|x[n+1] - x[n]|
영교차 ZC 부호 변경 횟수
기울기 부호 변경 SSC 기울기 반전 횟수

4.3.2 주파수 영역 특징

특징 약어 설명
중앙 주파수 MDF 전력 스펙트럼을 이등분하는 주파수
평균 주파수 MNF 전력 가중 평균 주파수
전력 스펙트럼 밀도 PSD 주파수별 전력 분포

4.4 패턴 인식 알고리즘

4.4.1 선형 판별 분석 (LDA)

EMG 제스처 분류에 널리 사용됨:

  • 클래스 간 분산 최대화
  • 클래스 내 분산 최소화
  • 실시간 처리에 적합한 계산 효율성
  • 일반적으로 95% 이상의 정확도

4.4.2 지원 벡터 머신 (SVM)

복잡한 결정 경계를 위한 비선형 분류:

  • 커널: RBF, 다항식, 시그모이드
  • 다중 클래스: 일대다 또는 일대일
  • 장점: 높은 정확도, 과적합에 강함
  • 단점: 학습 시간이 긴 편

4.4.3 신경망

딥러닝 기반 EMG 처리:

모델 아키텍처:
입력층 (EMG 특징) →
은닉층 1 (128 유닛, ReLU) →
드롭아웃 (0.3) →
은닉층 2 (64 유닛, ReLU) →
출력층 (제스처 클래스, Softmax)

4.5 프로토콜 성능 요구사항

지표 목표 측정 방법

처리량 > 1000 샘플/초 채널당 샘플링 속도
패킷 손실 < 0.1% 전송된 패킷 대비 손실
분류 정확도 > 95% 교차 검증 테스트
