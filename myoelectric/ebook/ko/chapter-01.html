제1장: 근전 의지(義肢)의 이해

1.1 근전도(sEMG) 제어란?

근전도 제어는 골격근 수축 시 발생하는 전기 신호인 표면 근전도(sEMG)를 이용해 동력 의지(義肢)와 외골격을 제어하는 기술입니다. 근육이 수축하면 운동 단위(motor unit)가 활동 전위를 발생시키고, 이 신호의 합이 피부 표면에서 50~5000 μV의 전압으로 감지됩니다. 잔존 지체에 부착된 전극이 이 신호를 포착하여 의지 명령으로 변환합니다.

근전 의지의 역사는 1950년대 후반 외부 동력 팔 의지의 등장에서 시작됩니다. 상업용 근전 의지는 1970년대 오토복(Ottobock) System Electric Hand의 보급으로 정착했고, 이후 Hudgins TD4 시간 영역 특징과 LDA 분류기 도입을 거쳐 단순 2-사이트 진폭 제어에서 8~16 클래스 다중 그립의 지능형 손으로 발전했습니다.

1.4 신호 생성 원리

sEMG 신호는 운동 단위 활동 전위(MUAP)에서 비롯됩니다. 표면 전극은 수백~수천 개의 MUAP가 공간·시간적으로 중첩된 간섭 패턴을 측정합니다. 신호 특성:

  • 진폭: 50~5000 μV; 최대 수의 수축(MVC)의 60%까지 힘에 비례하여 증가.
  • 주파수: 20~500 Hz 우세; 중앙 주파수(MDF)는 근피로 진행에 따라 감소 — 임상 피로 모니터링 지표로 사용.
  • 노이즈: 전원선 간섭(50 Hz), 모션 아티팩트(20 Hz 이하), 증폭기 노이즈 — 차동 증폭기(CMRR ≥ 80 dB) 필수.

1.5 WIA 근전 표준 구성

WIA-MYO 표준의 4단계 아키텍처:

  • Phase 1 — 데이터 형식: sEMG 프레임 JSON 스키마, Hudgins 특징 추출, 분류기 출력, 세션 메타데이터, ISO 22523 분류.
  • Phase 2 — API 인터페이스: OpenAPI 3.1 REST 엔드포인트, 실시간 스트리밍, 분류기 모델 배포.
  • Phase 3 — 프로토콜: ROS 2 토픽, BLE 5.4 GATT 무선 전극, USB HID 유선 시스템.
  • Phase 4 — 통합: 병원 EMR(FHIR R5), 보장구 처방 임상 워크플로우, Coapt/Ottobock SDK.

1.6 이 교재의 구성

  • 2장 — sEMG 신호 수집: 전극 기술, SENIAM 배치, Delsys/MyoWare 하드웨어, 신호 품질.
  • 3장 — 특징 추출과 패턴 인식: Hudgins TD4, AR(4), LDA, LSTM, 제스처 클래스.
  • 4장 — 비례 제어와 Coapt 연동.
  • 5장 — 하지 근전 및 동력 의지.
  • 6장 — 통신 프로토콜 (BLE GATT, USB HID, ROS 2).
  • 7장 — 임상 처방과 EMG 훈련 프로토콜.
  • 8장 — 국내외 규제 및 표준 준수 (식약처, ISO 22523, IEC 60601).
