제2장: sEMG 신호 수집

2.1 전극 기술

표면 EMG 전극은 피부 표면의 전기화학적 신호를 측정 가능한 전압으로 변환합니다. 근전 의지에서 사용되는 세 가지 전극 유형:

유형재질임피던스내구성용도
Ag/AgCl 일회용은/염화은 젤낮음 (2–10 kΩ)1회용임상 처방, 실험실 연구
스테인리스 드라이316L 스테인리스중간 (10–100 kΩ)수년일상 사용 (오토복 13E200)
건식 텍스타일은도금 섬유높음 (50–500 kΩ)세탁 가능소프트 라이너, 웨어러블 연구

WIA-MYO는 20 Hz에서 전극-피부 임피던스 ≤ 50 kΩ을 허용 가능 품질로, ≤ 10 kΩ을 최적으로 규정합니다. 미세 사포 연마 후 이소프로필 알코올 닦기로 대부분 10 kΩ 이하 달성 가능합니다.

2.2 차동 증폭 방식

모든 WIA 호환 sEMG 수집 시스템은 전용 기준(Ground) 전극이 포함된 차동 증폭 방식을 사용합니다. 차동 증폭기는 근육 복(belly)을 따라 배치된 두 활성 전극 간의 전위 차이를 측정하여 공통 모드 간섭(전원선 노이즈, 신체 전위 변동)을 CMRR ≥ 80 dB로 제거합니다. 

# WIA-MYO 신호 체인 사양
입력 임피던스:      ≥ 100 MΩ
공통 모드 제거비:    ≥ 80 dB (50 Hz)
이득:              1000× (60 dB) — ±5 mV 입력 범위에 적합

샘플 레이트:        ≥ 2 kHz (패턴 인식용; 1 kHz 콘텐츠에 대한 Nyquist)

2.4 국내 연구기관 주요 장비

Delsys Trigno IM

Delsys Trigno IM은 연구 표준 무선 16채널 sEMG 시스템(2 kHz, 20 Hz 하한차단, 센서 모듈별 IMU 내장)입니다. KAIST 바이오 및 뇌공학과, 서울대 기계항공공학부 로보틱스 연구실 등에서 표준 연구 장비로 활용됩니다.

오토복 13E200 전극

스테인리스 13E200은 전 세계 전완 의지의 임상 표준입니다. 조정 가능한 이득(3단계 스위치)을 갖춘 차동 증폭기가 통합되어 있으며, 비례 제어용 아날로그 출력과 스위치 제어용 로직 레벨 출력을 제공합니다. 국내에서는 동부팜한농 의료기기사업부를 통해 공급됩니다.

2.5 신호 품질 평가

import numpy as np
from scipy.signal import butter, filtfilt

def assess_emg_quality(raw_uv: np.ndarray, fs: float = 2000) -> dict:
    rms = np.sqrt(np.mean(raw_uv ** 2))
    saturation = np.any(np.abs(raw_uv) > 4900)
    motion_artifact = rms > 200  # 안정 시 고기저 활성

    # SNR: 400 Hz 이상 고역 통과 필터를 잡음 추정에 활용
    b, a = butter(4, 400 / (fs/2), btype='high')
    noise = filtfilt(b, a, raw_uv)
    noise_rms = np.sqrt(np.mean(noise**2))
    snr_db = 20 * np.log10(rms / noise_rms) if noise_rms > 0 else 0

    quality = "good" if snr_db >= 28 else "fair" if snr_db >= 20 else "poor"
    return {"rms_uv": rms, "snr_db": snr_db, "saturation": saturation,
            "motion_artifact": motion_artifact, "overall_quality": quality}

2.6 표적 근재지배술 (TMR)

상완 절단이나 견관절 절단(E2/E3) 환자에게는 표적 근재지배술(Targeted Muscle Reinnervation, TMR)이 적용됩니다. TMR은 잔존 신경을 가슴·어깨의 탈신경 근육으로 외과적으로 재배치하여 추가 sEMG 제어 사이트를 만드는 수술입니다. 4개 이상의 독립 제어 사이트 확보가 가능해 8~16전극 패턴 인식의 분류 정확도와 일상 사용 신뢰도를 향상시킵니다. 국내에서도 대학병원 수부·상지 전문 진료과를 중심으로 TMR 수술이 점진적으로 도입되고 있으며, WIA-MYO는 TMR 환자의 다중 제어 사이트 매핑을 위한 세션 메타데이터 필드(tmr_target_muscle, reinnervation_date)를 정의합니다.
