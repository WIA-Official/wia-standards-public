5.1 하지 절단과 동력 의지 분류

하지 동력 의지는 보행 시 관절 토크를 능동적으로 생성하는 동력 의족(Powered Prosthetic Foot/Knee)으로, 수동(탄성 에너지 저장) 의지와 대비됩니다. ISO 22523 K-코드 체계로 절단 부위를 분류하며, 한국에서는 건강보험 급여 인정 코드와 연계됩니다.

5.3 Ottobock Kenevo — 노인·K2 사용자용 전동 무릎

Kenevo는 60세 이상, K2 활동 수준 사용자를 위한 처방 무릎 관절입니다. Bluetooth 5.0 기반 Cockpit 앱과 연동하여 댐핑 계수, 좌석 높이, 재활 모드를 원격 조정할 수 있습니다. 국내에서는 연세세브란스·서울성모병원 재활의학과가 공인 피팅 센터입니다.

WIA-MYO 통합 보행 위상 감지기: 힘 감지 족저(FSR 4점), 무릎 관절 각도 인코더, 대퇴 IMU 데이터를 결합하여 5단계 보행 위상(입각 전반/중간/후반, 유각 전반/후반)을 실시간 분류합니다. 

import numpy as np
from dataclasses import dataclass
from enum import IntEnum

class GaitPhase(IntEnum):
    LOADING_RESPONSE = 0   # 입각 전반 (발뒤꿈치 착지)
    MID_STANCE       = 1   # 중간 입각
    TERMINAL_STANCE  = 2   # 입각 후반 (발가락 이지)
    PRE_SWING        = 3   # 전기 유각
    SWING            = 4   # 유각

@dataclass
class GaitSensorFrame:
    fsr_heel: float     # N
    fsr_toe: float      # N
    knee_angle_deg: float
    thigh_accel_z: float  # m/s²

def classify_gait_phase(frame: GaitSensorFrame) -> GaitPhase:
    loaded = (frame.fsr_heel + frame.fsr_toe) > 50.0  # N
    knee_flex = frame.knee_angle_deg > 15.0
    if loaded and not knee_flex:
        if frame.fsr_heel > frame.fsr_toe * 1.5:
            return GaitPhase.LOADING_RESPONSE
        elif frame.fsr_toe > frame.fsr_heel * 1.2:
            return GaitPhase.TERMINAL_STANCE
        return GaitPhase.MID_STANCE
    elif loaded and knee_flex:
        return GaitPhase.PRE_SWING
    return GaitPhase.SWING

5.4 지형 분류 CNN — 계단·경사·평지

대퇴 IMU 6축 × 50 ms 윈도우 데이터를 입력으로 하는 1D-CNN은 6가지 지형(평지/계단 오름·내림/오르막·내리막/장애물)을 실시간으로 분류 가능하며, 의지 측 모터 토크 프로파일의 사전 전환에 사용됩니다. 분류 결과는 보행 위상 감지기와 융합되어 환경 적응형 제어 모드(예: 계단 진입 시 무릎 댐핑 증가)를 트리거합니다. 

import torch
import torch.nn as nn

class TerrainCNN(nn.Module):
    def __init__(self, in_channels=6, num_classes=6):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv1d(in_channels, 32, kernel_size=5, padding=2),
            nn.ReLU(), nn.BatchNorm1d(32),
            nn.Conv1d(32, 64, kernel_size=5, padding=2),
            nn.ReLU(), nn.BatchNorm1d(64),
            nn.AdaptiveAvgPool1d(8),
        )
        self.fc = nn.Sequential(
            nn.Linear(64 * 8, 128), nn.ReLU(),
            nn.Linear(128, num_classes)
        )

    def forward(self, x):  # x: (배치, 6, 시퀀스_길이)
        return self.fc(self.conv(x).flatten(1))
