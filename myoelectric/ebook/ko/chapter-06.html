6.1 BLE 5.4 GATT 프로파일

WIA-MYO는 Bluetooth Low Energy 5.4 (LE Audio/Extended Advertising 지원) 기반의 표준 GATT 프로파일을 정의합니다. 전극 모듈과 의지 프로세서 간 무선 sEMG 전송에 사용되며, 연결 간격 7.5 ms(최소)~20 ms(일반)로 분류 지연을 15 ms 이하로 유지합니다.

서비스/특성UUID속성페이로드
WIA EMG Service0xFEA0—서비스 선언
EMG Raw Stream0xFEA1Notify8ch × 2B int16 + 2B 타임스탬프 (18B/패킷)
EMG RMS (처리됨)0xFEA2Notify8ch × 1B uint8 (0–255 = 0–5mV RMS)
Classifier Output0xFEA3Notify1B 클래스 ID + 1B 신뢰도 + 1B 플래그
Device Config0xFEA4Read/Write이득, 샘플레이트, 필터 파라미터
Battery Level0x2A19Read/Notify표준 BLE Battery Service

패킷 구조 (EMG Raw, 18B): [ch0_hi][ch0_lo][ch1_hi][ch1_lo]…[ch7_hi][ch7_lo][ts_hi][ts_lo] — big-endian int16, μV 단위, 타임스탬프는 1/32768 s 틱 카운터.

6.2 BLE Python 클라이언트 (bleak)

import asyncio, struct
from bleak import BleakClient, BleakScanner

EMG_RAW_UUID  = "0000FEA1-0000-1000-8000-00805F9B34FB"
CLASSIF_UUID  = "0000FEA3-0000-1000-8000-00805F9B34FB"

GESTURE_NAMES = ["안정", "손 펴기", "쥐기", "회내전",
                 "회외전", "손목 굴곡", "손목 신전", "세 손가락 집기"]

async def run():
    devices = await BleakScanner.discover(timeout=5.0)
    target = next(d for d in devices if "WIA-EMG" in (d.name or ""))

    async with BleakClient(target.address) as client:
        def on_raw(_, data):
            vals = struct.unpack(">8h", data[:16])
            ts   = struct.unpack(">H", data[16:])[0]
            print(f"[{ts}] 채널(μV): {vals}")

        def on_class(_, data):
            cls_id, conf, flags = data[0], data[1] / 255.0, data[2]
            rejected = bool(flags & 0x01)
            print(f"제스처: {GESTURE_NAMES[cls_id]}  "
                  f"신뢰도: {conf:.2f}  거부: {rejected}")

        await client.start_notify(EMG_RAW_UUID, on_raw)
        await client.start_notify(CLASSIF_UUID, on_class)
        await asyncio.sleep(60.0)

asyncio.run(run())

6.3 USB HID 유선 인터페이스

USB HID(Human Interface Device) 클래스를 사용하면 드라이버 설치 없이 OS에서 즉시 인식됩니다. WIA-MYO USB HID 디스크립터는 Usage Page: 0xFF00 (Vendor Defined), Usage: 0x01, 64B 인터럽트 IN 엔드포인트(인터벌 1 ms)를 정의합니다. 

import hid  # pip install hidapi

VENDOR_ID  = 0x2C7C  # WIA
PRODUCT_ID = 0x0501  # WIA-EMG-8CH

dev = hid.device()
dev.open(VENDOR_ID, PRODUCT_ID)
dev.set_nonblocking(False)

while True:
    report = dev.read(64, timeout_ms=100)
    if report:
        import struct
        # 바이트 0: 리포트 ID (1 = EMG 원시, 2 = 분류 결과)
        rid = report[0]
        if rid == 1:
            channels = struct.unpack_from("<8h", bytes(report), 1)
            print("EMG(μV):", channels)
        elif rid == 2:
            cls_id = report[1]
            conf   = report[2] / 255.0
            print(f"분류: {cls_id}  신뢰도: {conf:.2f}")
