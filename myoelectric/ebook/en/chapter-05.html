Chapter 5: Lower-Limb Myoelectric and Powered Prosthetics

5.1 Lower-Limb Prosthetic Control Overview

Lower-limb myoelectric control differs significantly from upper-limb in that muscle activity is primarily used for intent detection (distinguishing walking, stair climbing, sitting) rather than direct position or velocity control. The residual limb muscles for transtibial (K1) and transfemoral (K3) amputees provide valuable sEMG signals:

Amputation LevelISO CodeAvailable MusclesSignals Available
TranstibialK1Tibialis anterior, gastrocnemius, soleusStrong, reliable

TransfemoralK3Rectus femoris, biceps femoris, adductorsModerate
Hip disarticulationK4+Gluteus maximus, iliacus (residual)Limited; IMU supplementation

5.2 Össur Proprio Foot and Rheo Knee

The Össur Proprio Foot uses sEMG from residual calf muscles combined with accelerometers and gyroscopes to detect terrain changes (stairs vs. ramp vs. level) and automatically adjust ankle dorsiflexion. The Össur Rheo Knee uses similar sensor fusion to control magnetorheological fluid damping in real time. 

# Össur Open Prosthetics API (experimental Python bindings)
from ossur_api import RheoKneeDevice

knee = RheoKneeDevice.connect(ble_address="AA:BB:CC:DD:EE:FF")
print(f"Activity mode: {knee.get_activity_mode()}")
# → "walking" / "stair_ascent" / "stair_descent" / "sitting" / "standing"

# Stream gait events at 100 Hz
for event in knee.stream_gait_events():
    print(f"Heel strike: {event.heel_strike}  "
          f"Toe off: {event.toe_off}  "
          f"Knee angle: {event.knee_angle_deg:.1f}°")

5.3 Ottobock Kenevo and C-Brace

The Ottobock Kenevo microprocessor-controlled knee uses sEMG from residual quadriceps and hamstrings to differentiate voluntary vs. stumble responses. The C-Brace (Stance Control Orthosis) extends this to patients with residual limbs by detecting voluntary activation patterns to switch between stance-phase locking and swing-phase control.

Gait phase detection algorithm (simplified):

class GaitPhaseDetector:
    """sEMG + IMU gait phase detection for lower-limb prosthetics."""
    PHASES = ["initial_contact", "loading_response", "mid_stance",
              "terminal_stance", "pre_swing", "initial_swing",
              "mid_swing", "terminal_swing"]

    def detect_phase(self, emg_rms: dict, imu_data: dict) -> str:
        accel_z = imu_data["linear_acceleration"]["z"]
        quad_rms = emg_rms.get("rectus_femoris", 0)
        ham_rms = emg_rms.get("biceps_femoris", 0)
        ta_rms = emg_rms.get("tibialis_anterior", 0)

        if accel_z > 9.5 and quad_rms > 50:   # loading response: vertical + quad burst
            return "loading_response"
        elif ta_rms > 40 and accel_z < 8.0:   # swing: TA dorsiflexion active
            return "initial_swing"
        elif ham_rms > 30 and accel_z < 6.0:  # terminal swing: hamstring brake
            return "terminal_swing"
        else:
            return "mid_stance"

5.4 Neural Networks for Terrain Classification

Recent research has moved beyond hand-crafted features to end-to-end convolutional neural networks (CNNs) for terrain classification from sEMG signals. The WIA-MYO standard supports a tcn (Temporal Convolutional Network) classifier type: 

import torch
import torch.nn as nn

class TerrainCNN(nn.Module):
    """1D CNN for terrain classification from 8-channel sEMG."""
    def __init__(self, in_channels=8, num_classes=6):
        super().__init__()
        self.conv1 = nn.Conv1d(in_channels, 64, kernel_size=7, padding=3)
        self.conv2 = nn.Conv1d(64, 128, kernel_size=5, padding=2)
        self.conv3 = nn.Conv1d(128, 256, kernel_size=3, padding=1)
        self.pool = nn.AdaptiveAvgPool1d(1)
        self.fc = nn.Linear(256, num_classes)
        self.relu = nn.ReLU()

    def forward(self, x):  # x: (batch, 8 channels, 400 samples at 2kHz = 200ms)
        x = self.relu(self.conv1(x))
        x = self.relu(self.conv2(x))
        x = self.relu(self.conv3(x))
        x = self.pool(x).squeeze(-1)
        return self.fc(x)

# Classes: 0=level, 1=stairs_up, 2=stairs_down, 3=ramp_up, 4=ramp_down, 5=sitting
