Chapter 2: sEMG Signal Acquisition

2.1 Electrode Technologies

Surface EMG electrodes convert the electrochemical signal at the skin surface into a measurable voltage. Three electrode types are used in myoelectric prosthetics:

TypeMaterialImpedanceDurabilityClinical Use
Ag/AgCl disposableSilver/Silver Chloride gelLow (2–10 kΩ)Single useLab testing, clinical fitting
Stainless steel dryBrushed 316L SSMedium (10–100 kΩ)YearsLong-term daily use (Ottobock 13E200)
Dry textileSilver-coated fabricHigh (50–500 kΩ)WashableSoft prosthetic liners, wearable research

WIA-MYO mandates electrode-skin impedance ≤ 50 kΩ at 20 Hz for acceptable signal quality; ≤ 10 kΩ is optimal. Abrasive gel and light skin preparation (fine sandpaper or alcohol wipe) typically achieves impedance below 10 kΩ.

2.2 Differential Amplification

All WIA-compliant sEMG acquisition systems use differential amplification with a dedicated ground (reference) electrode. The differential amplifier measures the potential difference between two active electrodes placed along the muscle belly, rejecting common-mode interference (power line noise, body potential shifts) with a Common Mode Rejection Ratio (CMRR) ≥ 80 dB. 

# Signal chain specification
Input impedance:      ≥ 100 MΩ (to not load skin electrode impedance)
CMRR:                 ≥ 80 dB at 50/60 Hz
Gain:                 1000× (60 dB) — typical for ±5 mV input range
Band-pass filter:     20–500 Hz (2nd order Butterworth or higher)
Notch filter:         50 Hz (EU) or 60 Hz (US/KR) — optional if CMRR sufficient
ADC resolution:       ≥ 12 bit (16-bit recommended for Delsys Trigno)
Sample rate:          ≥ 2 kHz for pattern recognition (Nyquist for 1 kHz content)

2.3 SENIAM Electrode Placement

The SENIAM (Surface EMG for Non-Invasive Assessment of Muscles) project established standardised electrode placement guidelines for 27 muscles, ensuring reproducible recordings across sessions and laboratories. WIA-MYO-DATA-001 mandates SENIAM placement codes.

Key forearm muscle placements for transradial prosthetics:

MuscleSENIAM PlacementPrimary Action
FCR (Flexor Carpi Radialis)1/3 of distance from medial epicondyle to radial styloid, on volar forearmWrist flexion, radial deviation
ECR (Extensor Carpi Radialis)1/4 of distance from lateral epicondyle to radial styloid, dorsal forearmWrist extension, radial deviation
BB (Biceps Brachii)Anterior upper arm, 1/3 from antecubital fossaElbow flexion, supination
TB (Triceps Brachii)Posterior upper arm, midpointElbow extension, pronation
FDS (Flexor Digitorum Superficialis)Middle volar forearm, between FCR and ECUFinger flexion (grip)

2.4 Hardware Systems

Delsys Trigno IM

The Delsys Trigno IM is the research gold standard for high-density sEMG acquisition, offering 16 wireless channels at 2 kHz with a 20 Hz lower cut-off and integrated IMU per sensor pod. The Delsys Trigno SDK exposes a Python/MATLAB API and is supported by the WIA delsys_ros2_driver package. 

# delsys_ros2_driver launch
ros2 launch delsys_ros2_driver trigno_im.launch.py \
  num_channels:=16 \
  sample_rate_hz:=2000 \
  bandwidth_low_hz:=20 \
  bandwidth_high_hz:=450 \
  publish_raw:=true \
  publish_rms:=true

MyoWare 2.0 (SparkFun)

MyoWare 2.0 is an affordable, open-source single-channel sEMG module (3.3–5 V, I2C configurable gain) popular in academic prosthetics research and WIA developer kits. Chain up to 8 modules for an 8-channel array. 

# Arduino/Python read (MyoWare 2.0 analog output)
# Output: 0–VCC mV envelope (rectified, low-pass filtered at ~60 Hz)
# NOT suitable for pattern recognition (requires raw signal)
# Use MyoWare with external ADC at ≥ 1 kHz for Hudgins features

Ottobock 13E200 Electrode

The stainless steel 13E200 is the clinical standard in transradial prosthetics worldwide. It integrates a differential amplifier with adjustable gain (3-position switch: low/medium/high sensitivity) and provides both an analog output for proportional control and a logic-level output for simple switch control.

2.5 Signal Quality Assessment

# Python signal quality assessment
import numpy as np

def assess_emg_quality(raw_samples_uv: np.ndarray, fs: float = 2000) -> dict:
    """Assess sEMG frame quality per WIA-MYO-DATA-001 §7."""
    rms = np.sqrt(np.mean(raw_samples_uv ** 2))
    saturation = np.any(np.abs(raw_samples_uv) > 4900)  # within 2% of ±5000 uV rail
    motion_artifact = rms > 200  # high baseline during supposed rest

    # SNR: compare signal RMS to noise floor (high-pass > 400 Hz)
    from scipy.signal import butter, filtfilt
    b, a = butter(4, 400 / (fs/2), btype='high')
    noise = filtfilt(b, a, raw_samples_uv)
    noise_rms = np.sqrt(np.mean(noise**2))
    snr_db = 20 * np.log10(rms / noise_rms) if noise_rms > 0 else 0

    quality = "good" if snr_db >= 28 else "fair" if snr_db >= 20 else "poor"
    return {"rms_uv": rms, "snr_db": snr_db, "saturation": saturation,
            "motion_artifact": motion_artifact, "overall_quality": quality}

2.6 Targeted Muscle Reinnervation (TMR)

For high-level amputees (transhumeral, shoulder disarticulation), Targeted Muscle Reinnervation (TMR) surgically reroutes residual nerves to denervated chest or shoulder muscles, creating additional sEMG sites that represent the phantom missing limb. TMR can provide 4+ independent control sites from a single-limb amputee, enabling pattern recognition with 8–16 electrodes over the reinnervated skin areas.

WIA-MYO session metadata includes a tmr_performed boolean and tmr_target_muscles array to enable stratified analysis of TMR vs. conventional transradial classifier performance.
