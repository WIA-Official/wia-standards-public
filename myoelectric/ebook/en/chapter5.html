5.2 Rehabilitation System Integration

EMG-based rehabilitation systems use myoelectric signals for therapy monitoring, progress tracking, and biofeedback training.

5.2.1 Therapy Session Workflow

  1. Pre-Session Setup: Patient registration, electrode placement verification, baseline calibration
  2. Assessment: Measure current muscle strength, control accuracy, fatigue resistance
  3. Training Protocol: Structured exercises with real-time EMG feedback
  4. Progress Monitoring: Track improvements in signal quality, control precision, endurance
  5. Post-Session Analysis: Generate reports, update patient records, plan next session

5.2.2 Therapy Metrics

Metric Description Target Improvement
MVC Strength Maximum voluntary contraction amplitude 20-30% increase over 6 weeks
Control Accuracy Gesture classification success rate From 75% to >90%
Reaction Time Time from cue to correct gesture Reduce from 800ms to <500ms
Endurance Time until 50% MVC degradation Increase from 5 to 15+ minutes
Pattern Separation Inter-class distance in feature space Increase by 40-50%

5.3.2 Unity Integration Example

// C# script for Unity VR integration
using UnityEngine;
using WIA.Myoelectric;

public class EMGHandController : MonoBehaviour
{
    private EMGDevice emgDevice;
    private GestureClassifier classifier;
    private Animator handAnimator;

    void Start()
    {
        emgDevice = new EMGDevice("EMG-SENSOR-001");
        classifier = new GestureClassifier("classifier-78901");
        handAnimator = GetComponent<Animator>();

        emgDevice.OnGestureRecognized += HandleGesture;
        emgDevice.StartStreaming();
    }

    void HandleGesture(Gesture gesture, float confidence)
    {
        if (confidence > 0.85f)
        {
            switch (gesture.name)
            {
                case "hand_close":
                    handAnimator.SetFloat("GripStrength", gesture.intensity);
                    break;
                case "hand_open":
                    handAnimator.SetFloat("GripStrength", 0.0f);
                    break;
                case "precision_pinch":
                    handAnimator.SetTrigger("PrecisionGrip");
                    break;
            }
        }
    }

    void OnDestroy()
    {
        emgDevice.StopStreaming();
    }
}

5.4 Mobile Application Integration

5.4.1 iOS Integration (Swift)

import Foundation
import WIAMyoelectric

class EMGViewController: UIViewController {
    let emgManager = EMGDeviceManager()
    let classifier = GestureClassifier()

    override func viewDidLoad() {
        super.viewDidLoad()

        emgManager.onGestureDetected = { [weak self] gesture in
            self?.updateUI(for: gesture)
        }

        emgManager.startMonitoring()
    }

    func updateUI(for gesture: Gesture) {
        DispatchQueue.main.async {
            self.gestureLabel.text = gesture.name
            self.confidenceBar.progress = Float(gesture.confidence)
            self.intensitySlider.value = Float(gesture.intensity)
        }
    }
}

5.4.2 Android Integration (Kotlin)

import com.wia.myoelectric.*

class EMGActivity : AppCompatActivity() {
    private lateinit var emgDevice: EMGDevice
    private lateinit var classifier: GestureClassifier

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_emg)

        emgDevice = EMGDevice.getInstance()
        classifier = GestureClassifier.load("classifier-78901")

        emgDevice.onGestureDetected { gesture ->
            runOnUiThread {
                updateGestureDisplay(gesture)
            }
        }

        emgDevice.startStreaming()
    }

    private fun updateGestureDisplay(gesture: Gesture) {
        gestureText.text = gesture.name
        confidenceProgress.progress = (gesture.confidence * 100).toInt()
        intensitySeekBar.progress = (gesture.intensity * 100).toInt()
    }
}

Integration Considerations

  • Latency: Ensure end-to-end latency < 200 ms for natural control
  • Synchronization: Maintain precise timing between EMG and prosthetic actuators
  • Battery Life: Optimize wireless transmission to preserve device battery
  • Fallback Modes: Implement safe defaults if EMG signal quality degrades
  • User Feedback: Provide haptic or visual confirmation of recognized gestures
