📡 Spike Encoding Protocols

Rate Coding

Information in firing frequency: f = k·I

Simple, robust, but loses temporal precision

Temporal Coding

Information in precise spike timing

High information capacity, fast processing

Population Coding

Distributed representation across neurons

Robust to noise, error-tolerant

Phase Coding

Spike timing relative to oscillation

Used in hippocampus, high capacity
