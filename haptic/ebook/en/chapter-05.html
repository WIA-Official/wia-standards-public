🤚 Chapter 5: Phase 2 - Pattern Library

This chapter defines standardized haptic patterns for navigation, alerts, confirmations, and Braille encoding.

5.1 Pattern Categories

5.1.1 Category Overview

Category Purpose Examples
Confirmation Acknowledge user actions Tick, success, done
Notification Alert user attention Warning, error, urgent
Navigation Spatial orientation cues Direction, obstacle, boundary
Progress Indicate ongoing state Loading, counting, scrolling
Braille Tactile text encoding Letters, numbers, punctuation

5.5 Braille Encoding

5.5.1 Braille Cell Structure

Standard 6-dot Braille Cell:

  1 ● ● 4
  2 ● ● 5
  3 ● ● 6

Each dot maps to a unique haptic pulse.

5.5.2 Haptic Braille Schema

interface BrailleHaptic {
  // Braille character being encoded
  character: string;

  // Unicode Braille representation
  unicode: string;

  // Active dot positions (1-6)
  dots: number[];

  // Haptic encoding
  encoding: {
    // Duration of each dot pulse (ms)
    dotDuration: number;

    // Gap between dots (ms)
    dotGap: number;

    // Gap between cells (ms)
    cellGap: number;

    // Intensity for each dot position
    dotIntensities: number[];
  };
}

5.5.3 Example Braille Characters

const brailleA: BrailleHaptic = {
  character: 'A',
  unicode: '⠁',
  dots: [1],
  encoding: {
    dotDuration: 50,
    dotGap: 30,
    cellGap: 150,
    dotIntensities: [0.8]
  }
};

const brailleB: BrailleHaptic = {
  character: 'B',
  unicode: '⠃',
  dots: [1, 2],
  encoding: {
    dotDuration: 50,
    dotGap: 30,
    cellGap: 150,
    dotIntensities: [0.8, 0.8]
  }
};

5.5.4 Grade 2 Braille Contractions

Contraction Meaning Dots Pattern
⠮ THE 2-3-4-6 Four distinct pulses
⠯ AND 1-2-3-4-6 Five pulses
⠿ FOR 1-2-3-4-5-6 Full cell pattern

5.6 Pattern Sequences

5.6.1 Sequence Definition

interface HapticSequence {
  id: string;
  name: string;

  // Ordered list of patterns with timing
  steps: Array<{
    patternId: string;
    delayBefore: number;
    repeatCount?: number;
    repeatDelay?: number;
  }>;

  // Total sequence duration (calculated)
  totalDuration: number;
}

5.6.2 Example Sequence

const urgentAlertSequence: HapticSequence = {
  id: 'sequence.urgent_alert',
  name: 'Urgent Alert',
  steps: [
    {
      patternId: 'notification.warning',
      delayBefore: 0,
      repeatCount: 3,
      repeatDelay: 500
    }
  ],
  totalDuration: 2400
};
