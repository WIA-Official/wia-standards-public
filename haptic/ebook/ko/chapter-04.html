🤚 제4장: Phase 1 - 햅틱 프리미티브

이 장에서는 WIA Haptic 표준의 기본 빌딩 블록인 파형, ADSR 엔벨로프, 주파수 대역 및 강도 레벨을 정의합니다.

4.1 파형 유형

4.1.1 기본 파형

파형 설명 촉각 인식 용도
sine 부드러운 연속 진동 온화하고 유동적인 느낌 배경 피드백, 거리 표시
square 급격한 온/오프 전환 선명하고 확실한 펄스 확인, 알림
triangle 선형 상승 및 하강 중간 선명도 방향 신호
sawtooth 점진적 상승, 급격한 하강 래칫 느낌 진행 표시
noise 랜덤 변화 거칠고 질감 있는 느낌 경고, 장애물

4.1.2 파형 설정 스키마

interface WaveformConfig {
  // 파형 유형
  type: 'sine' | 'square' | 'triangle' | 'sawtooth' | 'noise';

  // 사각파의 듀티 사이클 (0.0-1.0)
  dutyCycle?: number;

  // 전환 스무딩 계수 (0.0-1.0)
  smoothing?: number;

  // 노이즈 특성
  noiseType?: 'white' | 'pink' | 'brown';
}

4.2 ADSR 엔벨로프

Attack-Decay-Sustain-Release (ADSR) 엔벨로프는 시간에 따른 햅틱 신호의 진폭을 형성합니다.

4.2.1 엔벨로프 시각화

강도
   ^
1.0|     /\
   |    /  \____
0.7|   /        \____
   |  /              \
0.0|_/________________\___> 시간
   |A | D |  S   | R  |

   A = Attack   (피크까지 시간)
   D = Decay    (서스테인 레벨까지 시간)
   S = Sustain  (유지 강도 레벨)
   R = Release  (페이드아웃 시간)

4.2.2 프리셋 엔벨로프

프리셋 Attack Decay Sustain Release 특성
sharp 5ms 20ms 0.0 30ms 빠른 탭
punch 10ms 50ms 0.3 50ms 임팩트 느낌
smooth 100ms 100ms 0.7 150ms 부드러운 파동
pulse 5ms 0ms 1.0 5ms 온/오프
swell 200ms 50ms 0.8 200ms 증가하는 강도
fade 50ms 200ms 0.5 300ms 점진적 감쇠

4.3 주파수 사양

4.3.1 인간 촉각 인식 범위

수용기 유형 주파수 범위 위치 민감도
마이스너 소체 10-50 Hz 손가락 끝, 손바닥 높음
파치니 소체 40-400 Hz 심부 조직 중간
머켈 원반 0.4-3 Hz 표피 저주파
루피니 종말 0.4-3 Hz 진피 신장

4.3.2 의미론적 주파수 매핑

의미 주파수 대역 근거
안전/명확 낮음 (30-80 Hz) 차분하고 안심되는
정보 중간 (80-150 Hz) 중립적이고 명확한
주의 높음 (150-250 Hz) 주의 환기
위험/긴급 매우 높음 (250-300 Hz) 경보

4.4 강도 사양

4.4.1 표준 강도 프리셋

레벨 값 설명 용도
subtle 0.2 간신히 인식 가능 배경, 주변
light 0.4 부드러운 피드백 확인
medium 0.6 표준 피드백 탐색
strong 0.8 확실한 피드백 알림
maximum 1.0 최대 강도 비상

4.5 완전한 프리미티브 정의

interface HapticPrimitive {
  // 고유 식별자
  id: string;

  // 사람이 읽을 수 있는 이름
  name: string;

  // 기본 파형 설정
  waveform: WaveformConfig;

  // 진폭 엔벨로프
  envelope: HapticEnvelope;

  // Hz 단위 진동 주파수 (1-300)
  frequency: number;

  // 기본 강도 (0.0-1.0)
  intensity: number;

  // 밀리초 단위 지속 시간
  duration: number;

  // 선택적 메타데이터
  metadata?: {
    category?: HapticCategory;
    description?: string;
    learnability?: 'easy' | 'moderate' | 'complex';
  };
}

4.6 장 요약

핵심 내용:

  1. 파형: 5가지 유형 (사인, 사각, 삼각, 톱니, 노이즈)
  2. ADSR 엔벨로프: Attack, Decay, Sustain, Release가 시간에 따른 진폭 형성
  3. 주파수 대역: 의미론적 의미를 가진 1-300 Hz의 5개 대역
  4. 강도 레벨: subtle (0.2)에서 maximum (1.0)까지 5개 프리셋
  5. 프리미티브: 모든 매개변수를 결합한 완전한 빌딩 블록
