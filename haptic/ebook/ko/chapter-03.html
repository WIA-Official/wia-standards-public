🤚 제3장: WIA Haptic 표준 개요

弘益人間 (홍익인간)

WIA Haptic 표준은 시각 장애 사용자가 모든 장치와 플랫폼에서 일관되고 학습 가능한 커뮤니케이션을 가능하게 하는 보편적인 촉각 어휘를 만듭니다.

3.1 미션과 목표

3.1.1 미션 선언문

시각 장애 사용자가 표준화된 터치 기반 피드백을 통해 디지털 장치와 탐색, 커뮤니케이션 및 상호 작용할 수 있도록 하는 개방적이고 상호 운용 가능한 햅틱 피드백 프레임워크를 구축하여 일관된 촉각 언어를 제공합니다.

3.1.2 주요 목표

  1. 보편성: 모든 플랫폼과 장치에서 공통 햅틱 어휘
  2. 학습 용이성: 빠른 학습과 장기 기억을 위해 설계된 패턴
  3. 접근성: 촉각 인식에 의존하는 사용자에게 최적화
  4. 상호운용성: iOS, Android, Web 및 웨어러블 전반에서 원활하게 작동
  5. 효율성: 모바일 및 웨어러블 장치를 위한 배터리 효율적 패턴

3.2.1 계층 관계

계층 목적 내용
프리미티브 기본 빌딩 블록 파형, 엔벨로프, 주파수, 강도
패턴 의미론적 햅틱 어휘 탐색 신호, 알림, 점자 인코딩
프로토콜 플랫폼 추상화 API 매핑, 기능 감지, 폴백
통합 WIA 생태계 연결 AAC, BCI, Eye Gaze, 내비게이션 연결

3.3 설계 원칙

3.3.1 접근성 우선

  • 모든 패턴은 시각적 참조 없이 구별 가능
  • 인간의 촉각 인식에 최적화된 주파수 범위
  • 빠른 학습과 기억을 위해 설계된 패턴
  • 다양한 민감도 수준 지원

3.3.2 하드웨어 독립적

  • 프리미티브는 장치 기능을 추상화
  • 제한된 하드웨어에 대한 우아한 저하
  • 장치 유형 전반에서 일관된 경험
  • 기능 감지 및 적응

3.3.3 에너지 효율

  • 모바일/웨어러블 배터리 제약에 최적화
  • 긴 연속 진동보다 짧고 효과적인 패턴 선호
  • 전력 인식 패턴 선택

3.4 햅틱 프리미티브 개요

3.4.1 파형 유형

파형 기호 인식 용도
사인 ∿∿∿ 부드럽고 온화한 배경, 거리
사각 ▃▃▃ 선명하고 확실한 확인, 알림

3.4.2 주파수 대역

대역 범위 의미
매우 낮음 1-30 Hz 본능적, 경고
낮음 30-80 Hz 차분한, 안심 (안전)
중간 80-150 Hz 중립적, 명확 (정보)
높음 150-250 Hz 주의 환기 (관심)
매우 높음 250-300 Hz 경보 (위험/긴급)

3.5 개발 단계

Phase 1: 햅틱 프리미티브

기본 빌딩 블록: 파형, ADSR 엔벨로프, 주파수 대역, 강도 레벨.

Phase 2: 패턴 라이브러리

탐색, 알림, 확인, 점자 인코딩을 위한 표준화된 패턴.

Phase 3: 장치 프로토콜

iOS, Android, Web 및 웨어러블 API에 매핑되는 크로스 플랫폼 통신 프로토콜.

Phase 4: WIA 통합

WIA AAC, BCI, Eye Gaze 및 내비게이션 표준과의 통합.

3.6 장 요약

핵심 내용:

  1. 미션: 보조 기술을 위한 보편적인 촉각 어휘
  2. 아키텍처: 4계층 설계 (프리미티브 → 패턴 → 프로토콜 → 통합)
  3. 원칙: 접근성 우선, 하드웨어 독립적, 에너지 효율
  4. 프리미티브: 5가지 파형, 5가지 주파수 대역, 6가지 ADSR 프리셋
  5. 단계: 프리미티브 → 패턴 → 프로토콜 → 통합
