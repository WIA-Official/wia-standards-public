🤚 제5장: Phase 2 - 패턴 라이브러리

이 장에서는 탐색, 알림, 확인 및 점자 인코딩을 위한 표준화된 햅틱 패턴을 정의합니다.

5.1 패턴 카테고리

5.1.1 카테고리 개요

카테고리 목적 예시
확인 사용자 동작 인정 틱, 성공, 완료
알림 사용자 주의 환기 경고, 오류, 긴급
탐색 공간 방향 신호 방향, 장애물, 경계
진행 진행 상태 표시 로딩, 카운팅, 스크롤
점자 촉각 텍스트 인코딩 글자, 숫자, 구두점

5.3 탐색 패턴

5.3.1 방향 신호

방향 패턴 설명
왼쪽 짧음-긺 짧은 탭 후 지속 펄스
오른쪽 긺-짧음 지속 펄스 후 짧은 탭
앞으로 상승 강도가 증가하는 패턴
뒤로 하강 강도가 감소하는 패턴
정지 강함-유지 급격한 시작과 지속 유지

5.3.2 장애물 감지

const obstaclePattern: HapticPattern = {
  id: 'navigation.obstacle',
  name: '장애물',
  category: 'navigation',

  dynamicIntensity: {

    curve: 'inverse_linear'
  }
};

5.4 점자 인코딩

5.4.1 점자 셀 구조

표준 6점 점자 셀:

  1 ● ● 4
  2 ● ● 5
  3 ● ● 6

각 점은 고유한 햅틱 펄스에 매핑됩니다.

5.4.2 햅틱 점자 스키마

interface BrailleHaptic {
  // 인코딩되는 점자 문자
  character: string;

  // 유니코드 점자 표현
  unicode: string;

  // 활성 점 위치 (1-6)
  dots: number[];

  // 햅틱 인코딩
  encoding: {
    // 각 점 펄스의 지속 시간 (ms)
    dotDuration: number;

    // 점 사이의 간격 (ms)
    dotGap: number;

    // 셀 사이의 간격 (ms)
    cellGap: number;

    // 각 점 위치의 강도
    dotIntensities: number[];
  };
}

5.4.3 점자 문자 예시

const brailleA: BrailleHaptic = {
  character: 'ㄱ',
  unicode: '⠁',
  dots: [1],
  encoding: {
    dotDuration: 50,
    dotGap: 30,
    cellGap: 150,
    dotIntensities: [0.8]
  }
};

const brailleB: BrailleHaptic = {
  character: 'ㄴ',
  unicode: '⠃',
  dots: [1, 2],
  encoding: {
    dotDuration: 50,
    dotGap: 30,
    cellGap: 150,
    dotIntensities: [0.8, 0.8]
  }
};

5.5 패턴 시퀀스

5.5.1 시퀀스 정의

interface HapticSequence {
  id: string;
  name: string;

  // 타이밍을 가진 순서화된 패턴 목록
  steps: Array<{
    patternId: string;
    delayBefore: number;
    repeatCount?: number;
    repeatDelay?: number;
  }>;

  // 총 시퀀스 지속 시간 (계산됨)
  totalDuration: number;
}

5.5.2 시퀀스 예시

const urgentAlertSequence: HapticSequence = {
  id: 'sequence.urgent_alert',
  name: '긴급 알림',
  steps: [
    {
      patternId: 'notification.warning',
      delayBefore: 0,
      repeatCount: 3,
      repeatDelay: 500
    }
  ],
  totalDuration: 2400
};

5.6 장 요약

핵심 내용:

  1. 카테고리: 확인, 알림, 탐색, 진행, 점자
  2. 확인: 피드백을 위한 틱, 성공, 완료 패턴
  3. 탐색: 방향 신호 및 장애물 감지
  4. 점자: 타이밍 매개변수를 가진 6점 셀 인코딩
  5. 시퀀스: 여러 프리미티브를 결합한 복잡한 패턴
