제5장: Phase 2 - API 인터페이스

5.1 핵심 API 설계 철학

WIA 점자 디스플레이 API는 개발자가 점자 디스플레이 하드웨어와 상호작용할 수 있는 일관되고 직관적인 인터페이스를 제공합니다. 이 API의 설계는 다음과 같은 핵심 원칙을 따릅니다:

5.1.1 설계 원칙

  • 단순성: 복잡한 하드웨어 세부사항을 추상화하여 개발자가 쉽게 사용할 수 있도록 합니다.
  • 일관성: 모든 기능에서 일관된 명명 규칙과 패턴을 사용합니다.
  • 안전성: 타입 안전성과 오류 처리를 통해 런타임 오류를 최소화합니다.
  • 확장성: 미래의 기능 추가를 고려한 유연한 구조를 제공합니다.
  • 호환성: 다양한 플랫폼과 언어에서 사용할 수 있도록 설계되었습니다.
  • 弘益人間: 모든 사용자가 접근 가능한 기술을 통해 인류에게 이로움을 제공합니다.

5.1.2 API 계층 구조

WIA 점자 디스플레이 API는 다음과 같은 계층 구조로 설계되었습니다:

계층 설명 예시
고수준 API 일반적인 사용 사례를 위한 간단한 인터페이스 display.write("안녕하세요")
중간 수준 API 세밀한 제어를 위한 인터페이스 display.setCells(0, brailleCells)
저수준 API 하드웨어 직접 제어 display.sendCommand(0x01, data)

5.1.3 비동기 프로그래밍 모델

모든 I/O 작업은 비동기적으로 수행되어 애플리케이션의 응답성을 보장합니다:

// Promise 기반
display.write("테스트").then(() => {
    console.log("쓰기 완료");
});

// async/await 사용
async function updateDisplay() {
    await display.write("안녕하세요");
    await display.scroll(1);
}

5.2 WiaBrailleDisplay 클래스

WiaBrailleDisplay는 점자 디스플레이와 상호작용하기 위한 메인 클래스입니다. 이 클래스는 기기 관리, 셀 제어, 이벤트 처리 등 모든 핵심 기능을 제공합니다.

5.2.1 클래스 정의

/**
 * WIA 점자 디스플레이를 제어하는 메인 클래스
 * @class WiaBrailleDisplay
 */
class WiaBrailleDisplay extends EventEmitter {
    /**
     * 점자 디스플레이 인스턴스 생성
     * @param {BrailleDisplayConfig} config - 설정 옵션
     */
    constructor(config?: BrailleDisplayConfig);

    /**
     * 디스플레이에 연결
     * @returns {Promise<void>}
     */
    async connect(): Promise<void>;

    /**
     * 디스플레이 연결 해제
     * @returns {Promise<void>}
     */
    async disconnect(): Promise<void>;

    /**
     * 연결 상태 확인
     * @returns {boolean}
     */
    isConnected(): boolean;

    /**
     * 디스플레이 정보 가져오기
     * @returns {BrailleDisplayInfo}
     */
    getInfo(): BrailleDisplayInfo;
}

5.2.2 설정 인터페이스

interface BrailleDisplayConfig {
    /**
     * 자동 연결 시도 여부
     * @default true
     */
    autoConnect?: boolean;

    /**
     * 연결 재시도 횟수
     * @default 3
     */
    retryAttempts?: number;

    /**
     * 통신 프로토콜 (usb, bluetooth, serial)
     * @default "usb"

    protocol?: "usb" | "bluetooth" | "serial";

    /**
     * 디바이스 ID (특정 기기 연결 시)
     */
    deviceId?: string;

    /**
     * 로깅 레벨
     * @default "info"
     */
    logLevel?: "debug" | "info" | "warn" | "error";
}

5.3 기기 검색 및 열거

WIA 점자 디스플레이 API는 연결된 모든 점자 디스플레이를 검색하고 열거할 수 있는 기능을 제공합니다.

5.3.1 기기 검색

/**
 * 사용 가능한 모든 점자 디스플레이 검색
 * @returns {Promise<BrailleDisplayInfo[]>}
 */
async function discoverDevices(): Promise<BrailleDisplayInfo[]> {
    const devices = await WiaBrailleDisplay.discover();
    return devices;
}

// 사용 예제
const devices = await WiaBrailleDisplay.discover();
console.log(`발견된 기기: ${devices.length}개`);

devices.forEach((device, index) => {
    console.log(`${index + 1}. ${device.manufacturer} ${device.model}`);
    console.log(`   셀 개수: ${device.cellCount}`);
    console.log(`   연결 방식: ${device.connectionType}`);
});

5.3.2 특정 기기 연결

/**
 * 특정 기기에 연결
 */
async function connectToDevice(deviceId: string): Promise<WiaBrailleDisplay> {
    const display = new WiaBrailleDisplay({
        deviceId: deviceId,
        autoConnect: true
    });

    await display.connect();
    return display;
}

// 사용 예제
const devices = await WiaBrailleDisplay.discover();
if (devices.length > 0) {
    const display = await connectToDevice(devices[0].serialNumber);
    console.log("연결 성공!");
}

5.3.3 자동 연결

/**
 * 첫 번째로 발견된 기기에 자동 연결

async function autoConnect(): Promise<WiaBrailleDisplay> {
    const display = new WiaBrailleDisplay({
        autoConnect: true
    });

    // 자동으로 첫 번째 기기에 연결 시도

// 사용 예제
try {

}

5.5.3 커서 라우팅 통합

/**
 * 스크린 리더와 통합된 커서 라우팅 예제
 */
class BrailleNavigator {
    private display: WiaBrailleDisplay;
    private currentText: string = "";
    private viewportStart: number = 0;

    constructor(display: WiaBrailleDisplay) {
        this.display = display;
        this.setupRoutingKeys();
    }

    private setupRoutingKeys() {
        this.display.on("routingKey", async (event) => {
            if (event.type === "press") {
                const absolutePosition = this.viewportStart + event.cellIndex;
                await this.moveCursorTo(absolutePosition);
            }
        });
    }

    async displayText(text: string) {
        this.currentText = text;
        await this.updateDisplay();
    }

    private async updateDisplay() {
        const cellCount = this.display.getInfo().cellCount;
        const visibleText = this.currentText.substring(
            this.viewportStart,
            this.viewportStart + cellCount
        );
        await this.display.write(visibleText);
    }

    private async moveCursorTo(position: number) {
        console.log(`커서를 ${position} 위치로 이동`);
        // 스크린 리더 API를 통해 커서 이동
        // screenReader.moveCursor(position);
    }
}

// 사용
const navigator = new BrailleNavigator(display);
await navigator.displayText("안녕하세요. WIA 점자 디스플레이입니다.");

5.6 키 이벤트 처리

점자 디스플레이에는 라우팅 키 외에도 다양한 물리적 버튼이 있습니다. 이러한 키 이벤트를 처리하는 방법을 설명합니다.

5.6.1 버튼 이벤트

/**
 * 버튼 이벤트 타입
 */
interface ButtonEvent {
    /**
     * 버튼 ID
     */
    buttonId: number;

    /**
     * 버튼 이름 (예: "space", "dot1", "dot2")
     */
    buttonName: string;

    /**
     * 이벤트 타입
     */
    type: "press" | "release" | "longPress";

    /**
     * 동시에 눌린 다른 버튼들
     */
    modifiers: string[];

    /**
     * 타임스탬프
     */
    timestamp: number;
}

/**
 * 버튼 이벤트 리스너

display.on("button", (event: ButtonEvent) => {

    // 조합키 처리
    if (event.modifiers.includes("space")) {
        console.log("스페이스 + " + event.buttonName);
    }
});

5.7.2 오류 처리 패턴

/**
 * Try-catch 패턴
 */
async function safeWrite(display: WiaBrailleDisplay, text: string) {
    try {
        await display.write(text);
    } catch (error) {
        if (error instanceof WiaBrailleError) {
            switch (error.code) {
                case BrailleErrorCode.NOT_CONNECTED:
                    console.error("디스플레이가 연결되지 않았습니다");
                    // 재연결 시도
                    await display.connect();
                    await display.write(text);
                    break;

                case BrailleErrorCode.TIMEOUT:
                    console.error("통신 시간 초과");
                    // 재시도
                    await display.write(text);
                    break;

                default:
                    console.error("알 수 없는 오류:", error.message);
            }
        } else {
            throw error;
        }
    }
}

/**
 * 복구 가능한 오류 처리
 */
async function handleRecoverableError(error: WiaBrailleError) {
    switch (error.code) {
        case BrailleErrorCode.CONNECTION_LOST:
            await display.reconnect();
            break;
        case BrailleErrorCode.TIMEOUT:
            await display.reset();
            break;
    }
}

/**
 * 치명적 오류 처리
 */
function handleFatalError(error: WiaBrailleError) {

    // 로그 기록
    logger.error(error);
    // 연결 종료
    display.disconnect();
}

5.7.4 재시도 메커니즘

/**
 * 재시도 옵션
 */
interface RetryOptions {
    /**
     * 최대 재시도 횟수
     */
    maxRetries: number;

    /**
     * 재시도 간격 (ms)
     */
    retryDelay: number;

    /**
     * 지수 백오프 사용 여부
     */
    exponentialBackoff: boolean;
}

/**
 * 재시도 래퍼 함수
 */
async function withRetry<T>(
    fn: () => Promise<T>,
    options: RetryOptions
): Promise<T> {
    let lastError: Error;

    for (let i = 0; i < options.maxRetries; i++) {
        try {
            return await fn();
        } catch (error) {
            lastError = error;

            const delay = options.exponentialBackoff
                ? options.retryDelay * Math.pow(2, i)
                : options.retryDelay;

            console.log(`재시도 ${i + 1}/${options.maxRetries} (${delay}ms 후)`);
            await sleep(delay);
        }
    }

    throw lastError;
}

// 사용 예제
const text = await withRetry(
    () => display.write("안녕하세요"),
    {
        maxRetries: 3,
        retryDelay: 1000,
        exponentialBackoff: true
    }
);

import { WiaBrailleDisplay } from "@wia/braille-display";

class BrailleScroller {
    private display: WiaBrailleDisplay;
    private text: string = "";
    private position: number = 0;
    private isScrolling: boolean = false;

    constructor(display: WiaBrailleDisplay) {
        this.display = display;
    }

    async start(text: string, speed: number = 500) {
        this.text = text;
        this.position = 0;
        this.isScrolling = true;

        const cellCount = this.display.getInfo().cellCount;

        while (this.isScrolling) {
            const visibleText = this.text.substring(
                this.position,
                this.position + cellCount
            );

            await this.display.write(visibleText);
            await sleep(speed);

            this.position++;
            if (this.position + cellCount > this.text.length) {
                this.position = 0;  // 처음부터 다시
            }
        }
    }

    stop() {
        this.isScrolling = false;
    }
}

import { WiaBrailleDisplay } from "@wia/braille-display";

class MultiDisplayManager {
    private displays: Map<string, WiaBrailleDisplay> = new Map();

    async discoverAndConnect() {
        const devices = await WiaBrailleDisplay.discover();

        for (const deviceInfo of devices) {
            const display = new WiaBrailleDisplay({
                deviceId: deviceInfo.serialNumber
            });

            await display.connect();
            this.displays.set(deviceInfo.serialNumber, display);

            console.log(`연결됨: ${deviceInfo.model}`);
        }
    }

    async writeToAll(text: string) {
        const promises = Array.from(this.displays.values()).map(
            display => display.write(text)
        );

        await Promise.all(promises);
    }

    async disconnectAll() {
        const promises = Array.from(this.displays.values()).map(
            display => display.disconnect()
        );

        await Promise.all(promises);
        this.displays.clear();
    }
}

// 사용
async function multiDisplayExample() {
    const manager = new MultiDisplayManager();
    await manager.discoverAndConnect();

    await manager.writeToAll("모든 디스플레이에 표시");

    await sleep(3000);
    await manager.disconnectAll();

5.8.5 실시간 알림 시스템 예제

import { WiaBrailleDisplay } from "@wia/braille-display";

class BrailleNotificationSystem {
    private display: WiaBrailleDisplay;
    private queue: string[] = [];
    private isProcessing: boolean = false;

    constructor(display: WiaBrailleDisplay) {
        this.display = display;
    }

    async notify(message: string, priority: "low" | "normal" | "high" = "normal") {
        if (priority === "high") {
            // 긴급 알림은 즉시 표시
            await this.displayMessage(message);
        } else {
            // 일반 알림은 큐에 추가
            this.queue.push(message);
            if (!this.isProcessing) {
                await this.processQueue();
            }
        }
    }

    private async processQueue() {
        this.isProcessing = true;

        while (this.queue.length > 0) {
            const message = this.queue.shift()!;
            await this.displayMessage(message);
            await sleep(3000);  // 각 메시지를 3초간 표시
        }

        this.isProcessing = false;
    }

    private async displayMessage(message: string) {
        await this.display.clear();
        await this.display.write(message);

        // 진동 피드백 (지원되는 경우)
        if (this.display.supportsVibration()) {
            await this.display.vibrate(200);
        }
    }
}

// 사용
async function notificationExample() {
    const display = new WiaBrailleDisplay();
    await display.connect();

    const notifier = new BrailleNotificationSystem(display);

    // 일반 알림
    await notifier.notify("새 메시지가 도착했습니다");
    await notifier.notify("회의가 10분 후 시작됩니다");

    // 긴급 알림
    await notifier.notify("긴급: 배터리 부족!", "high");
}
