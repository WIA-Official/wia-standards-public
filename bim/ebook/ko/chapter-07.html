7.1 통합 레이어 개요

통합 레이어는 WIA-BIM이 디지털 트윈과 IoT 시스템부터 기업 애플리케이션과 레거시 플랫폼까지 더 넓은 기술 생태계와 연결할 수 있게 합니다. 이 포괄적인 통합 기능은 BIM을 고립된 설계 도구에서 건물 지능의 중앙 허브로 변환합니다.

🏗️ 통합 목표

  • 디지털 트윈: 물리적 건물을 미러링하는 살아있는 모델 생성
  • 기업 시스템: ERP, CMMS, 비즈니스 애플리케이션과 연결
  • IoT 네트워크: 건물의 실시간 센서 데이터 통합
  • GIS 플랫폼: 지리적 컨텍스트 내에 건물 배치
  • 레거시 시스템: 기존 인프라 투자와 연결

7.3 기업 시스템 통합

7.3.1 일반적인 기업 통합

시스템 유형 통합 목적 데이터 흐름
ERP (SAP, Oracle) 재무, 조달 비용, 발주서, 송장
CMMS (Maximo, ServiceNow) 유지보수 관리 작업 주문, 자산 데이터, 일정
IWMS (Archibus, FM:Systems) 공간 관리 점유율, 이동, 공간 할당
프로젝트 관리 일정, 리소스 작업, 마일스톤, 진행
문서 관리 문서화 도면, 사양, 제출물

7.3.2 ERP 통합 패턴

// WIA-BIM에서 ERP로 커넥터
class ERPConnector {
  constructor(config) {
    this.endpoint = config.erpEndpoint;
    this.credentials = config.credentials;
    this.mapping = config.fieldMapping;
  }

  // BIM 객체를 ERP 자산으로 동기화
  async syncAssets(bimObjects) {
    const assets = bimObjects.map(obj => ({
      assetNumber: this.generateAssetNumber(obj),
      description: obj.name,
      category: this.mapCategory(obj.type),
      location: this.mapLocation(obj.containedIn),
      acquisitionDate: obj.metadata.createdAt,
      value: this.extractCost(obj),
      specifications: {
        manufacturer: obj.properties.manufacturer,
        model: obj.properties.model,
        serialNumber: obj.properties.serialNumber
      }
    }));

    return await this.erpClient.bulkCreate('assets', assets);
  }
}

7.4.2 센서 유형 및 BIM 매핑

센서 유형 측정 BIM 요소 속성
온도 공기 온도 공간, 구역 liveData.temperature
습도 상대 습도 공간, 구역 liveData.humidity
재실 인원 수 공간 liveData.occupancy
전력계 에너지 소비 장비, 구역 liveData.power
조도 센서 조명도 공간 liveData.lightLevel
CO2 센서 공기 질 공간 liveData.co2Level

7.6 레거시 시스템 브릿지

7.6.1 레거시 통합 패턴

패턴 용도 구현
파일 기반 배치 내보내기/가져오기 예약된 파일 드롭, FTP
데이터베이스 링크 직접 데이터 접근 ODBC, JDBC 커넥터
API 래퍼 레거시에 API 있음 REST/SOAP 어댑터
화면 스크래핑 API 없음 RPA, UI 자동화
메시지 큐 비동기 통합 MQ, Kafka, RabbitMQ

7.7 장 요약

핵심 내용

  • 디지털 트윈은 BIM 형상을 실시간 센서 데이터 및 분석과 결합
  • 기업 통합은 BIM을 ERP, CMMS, 비즈니스 시스템에 연결
  • IoT 통합은 MQTT, OPC-UA, BACnet 등 여러 프로토콜 지원
  • GIS 통합은 건물을 지리적 및 도시 컨텍스트에 배치
  • 레거시 어댑터는 기존 시스템에서 점진적 마이그레이션 가능
