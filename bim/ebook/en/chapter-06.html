6.1 Protocol Layer Overview

The Protocol layer defines how WIA-BIM data is exchanged, synchronized, and managed across distributed systems. This layer ensures reliable, secure, and consistent communication between all participants in a BIM project.

📡 Protocol Design Goals

  • Reliability: Guaranteed message delivery with acknowledgment
  • Consistency: All participants see the same data state
  • Security: End-to-end encryption and authentication
  • Efficiency: Minimal bandwidth with delta transfers
  • Scalability: Support for thousands of concurrent users

6.5 Conflict Resolution

6.5.1 Conflict Types

Type Description Resolution Strategy
Property Conflict Same property, different values Auto-merge or manual
Geometry Conflict Overlapping modifications Visualization + manual
Relationship Conflict Incompatible relationships Rule-based resolution
Schema Conflict Different schema versions Migration + validation
Semantic Conflict Logical inconsistency Expert review

6.5.2 Resolution Workflow

// Conflict resolution process
class ConflictResolver {
  resolve(conflict) {
    // Step 1: Classify conflict
    const classification = this.classify(conflict);

    // Step 2: Apply auto-resolution rules
    if (classification.autoResolvable) {
      return this.autoResolve(conflict, classification.rules);
    }

    // Step 3: Generate resolution options
    const options = this.generateOptions(conflict);

    // Step 4: Request human decision if needed
    if (classification.requiresHumanReview) {
      return this.createReviewTask(conflict, options);
    }

    // Step 5: Apply default strategy
    return this.applyDefaultStrategy(conflict, classification);
  }

  autoResolve(conflict, rules) {
    // Priority-based: Later timestamp wins
    if (rules.includes('LATEST_WINS')) {
      return conflict.versions.sort((a, b) =>
        new Date(b.timestamp) - new Date(a.timestamp)
      )[0];
    }

    // Authority-based: Higher authority wins
    if (rules.includes('AUTHORITY_WINS')) {
      return conflict.versions.sort((a, b) =>
        this.getAuthorityLevel(b.source) - this.getAuthorityLevel(a.source)
      )[0];
    }
  }
}
