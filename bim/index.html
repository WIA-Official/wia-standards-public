About BIM Standard

BIM 표준 소개

Building Information Modeling (BIM) is a digital representation of physical and functional characteristics of a facility. WIA-CITY-006 provides a comprehensive standard for BIM implementation, covering IFC data formats, LOD (Level of Development) specifications, clash detection algorithms, and integration with FM (Facility Management) systems.

빌딩정보모델링(BIM)은 시설물의 물리적, 기능적 특성을 디지털로 표현하는 기술입니다. WIA-CITY-006은 IFC 데이터 형식, LOD(상세 수준) 사양, 충돌 검토 알고리즘, FM(시설물 관리) 시스템 연동 등을 포괄하는 종합적인 BIM 구현 표준을 제공합니다.

This standard enables seamless interoperability between Revit, ArchiCAD, and other BIM tools through openBIM protocols, supporting the entire lifecycle from design to construction and operation.

이 표준은 openBIM 프로토콜을 통해 Revit, ArchiCAD 등 다양한 BIM 도구 간의 원활한 상호운용성을 지원하며, 설계부터 시공, 운영까지 전체 생애주기를 지원합니다.

Key Features

주요 기능

IFC Data Standards

IFC 데이터 표준

Industry Foundation Classes (IFC) format for universal BIM data exchange, supporting 3D geometry, metadata, and relationships.

범용 BIM 데이터 교환을 위한 IFC(Industry Foundation Classes) 형식으로 3D 지오메트리, 메타데이터, 관계 정보를 지원합니다.

LOD Framework

LOD 프레임워크

Level of Development specification from LOD 100 (concept) to LOD 500 (as-built), defining model detail requirements at each project phase.

LOD 100(개념)부터 LOD 500(준공)까지 상세 수준 사양으로 각 프로젝트 단계별 모델 상세도 요구사항을 정의합니다.

Clash Detection

충돌 검토

Automated algorithms for detecting conflicts between architectural, structural, and MEP elements before construction.

건축, 구조, 설비 요소 간 충돌을 시공 전에 자동으로 감지하는 알고리즘을 제공합니다.

4D/5D BIM

4D/5D BIM

Time-based construction simulation (4D) and cost estimation (5D) capabilities for comprehensive project management.

시간 기반 시공 시뮬레이션(4D)과 원가 산출(5D) 기능으로 종합적인 프로젝트 관리를 지원합니다.

OpenBIM Protocol

OpenBIM 프로토콜

Open standards for vendor-neutral collaboration, ensuring data exchange between Revit, ArchiCAD, Tekla, and other platforms.

벤더 중립적인 협업을 위한 개방형 표준으로 Revit, ArchiCAD, Tekla 등 다양한 플랫폼 간 데이터 교환을 보장합니다.

Digital Twin Integration

디지털 트윈 연동

Seamless connection to FM systems and IoT sensors for real-time building operation and maintenance.

FM 시스템 및 IoT 센서와 원활한 연동으로 실시간 건물 운영 및 유지보수를 지원합니다.
