# WIA-MEDIA-001 — Streaming (Press Brief)

**WIA Standards releases canonical Streaming envelope shape**

The WIA Standards Committee has published the WIA-MEDIA-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for streaming hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
