## 📊 현재 상태 분석

### 문제: 분산된 복잡성
```
수십 가지 의식 이론이 각각 따로 연구됨:
├── 통합정보이론 (IIT)
├── 글로벌 워크스페이스 이론 (GWT)
├── 고차 사고 이론 (HOT)
├── 예측 처리 이론
├── 양자 의식 이론
├── 신경상관자 (NCC) 연구
├── 명상/변성 의식 연구
└── 마취/수면 의식 연구
```

### 발견된 빈틈 (2025)
```
┌─────────────────────────────────────────────────────────────┐
│  핵심 발견: 의식 = 정보 통합의 질                           │
│                                                             │

│  - Φ (파이) = 통합된 정보량                                 │
│  - 의식 수준 = 정보 통합 복잡도                             │

│  글로벌 워크스페이스 (Baars, Dehaene):                      │
│  - 의식 = "방송" 메커니즘                                   │
│  - 무의식 → 의식 전환 기준                                  │
│  - 신경 시그니처 측정 가능                                  │
│                                                             │
│  2025 발전:                                                 │
│  - 마취 중 의식 측정 기술 발전                              │
│  - AI 의식 여부 판단 필요성 대두                            │
│  - 의식 장애 환자 진단 개선                                 │
└─────────────────────────────────────────────────────────────┘
```

## 📋 표준 개발 지시사항

### Phase 1: 데이터 형식

**1.1 의식 지표 스키마**
```json
{
  "consciousness_index": {
    "phi_value": 0.0-∞,
    "global_workspace_activation": 0.0-1.0,
    "neural_complexity": 0.0-1.0,
    "information_integration": 0.0-1.0,
    "recurrent_processing": 0.0-1.0
  },
  "measurement": {
    "method": "EEG|fMRI|PCI|combined",
    "perturbational_complexity_index": 0.0-1.0,
    "timestamp": "ISO8601",
    "conditions": "awake|sleep|anesthesia|meditation"
  }
}
```

**1.2 의식 상태 분류**
- 완전 각성
- 수면 단계 (N1, N2, N3, REM)
- 마취 수준
- 변성 의식 상태
- 의식 장애 (VS, MCS, EMCS)

### Phase 2: API 인터페이스

**2.1 의식 측정 API**
```
POST /api/v1/consciousness/measure
GET /api/v1/consciousness/state/{subject_id}
POST /api/v1/consciousness/compare
GET /api/v1/consciousness/history/{subject_id}
```

**2.2 분석 대시보드**
- Φ 값 실시간 모니터링
- 의식 상태 시각화
- 시계열 분석

### Phase 3: 프로토콜

**3.1 측정 프로토콜**
- PCI (Perturbational Complexity Index) 측정
- EEG 기반 의식 지표
- fMRI 연결성 분석

**3.2 해석 프로토콜**
- 의식 수준 판정 기준
- 의식 장애 진단 가이드
- 마취 깊이 모니터링

### Phase 4: 통합

**4.1 의료 시스템 연동**
- 마취과 모니터링
- 중환자실 의식 평가
- 재활의학 의식 회복 추적

**4.2 연구 플랫폼**
- 다기관 데이터 공유
- 표준화된 연구 프로토콜
- 메타 분석 지원
