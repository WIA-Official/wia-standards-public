제5장: 모니터링 및 평가

弘益人間 (홍익인간) - "널리 인간을 이롭게 하라"

"측정할 수 없는 것은 관리할 수 없다." 효과적인 모니터링과 평가는 복원 성공을 보장하고, 적응 관리를 안내하며, 투자자와 이해관계자에게 책임을 제공합니다.

5.1 모니터링 프레임워크

포괄적인 모니터링 프로그램은 생태적, 물리적, 사회경제적 지표를 추적하여 복원 진행 상황과 영향을 평가합니다.

모니터링 수준

수준 규모 빈도 방법 비용
레벨 1: 기본 플롯 수준 3, 6, 12개월, 이후 연 1회 현장 조사, 간단한 측정 $500-$1,500/ha/년
레벨 2: 표준 현장 수준 연 2-4회 상세 플롯, 일부 원격 감지 $1,500-$4,000/ha/년
레벨 3: 종합 경관 수준 연중 지속 센서, 드론, 위성, 생물다양성 $4,000-$10,000/ha/년
레벨 4: 연구 다중 현장 연중 + 실험 모든 방법 + 특수 연구 $10,000+ /ha/년

주요 지표 범주

맹그로브 복원 모니터링 지표 프레임워크:

1. 구조적 지표:
   ├── 생존율 (%)
   ├── 밀도 (줄기/ha)
   ├── 높이 (m)
   ├── 흉고직경 (cm)
   ├── 캐노피 피복 (%)
   └── 기저 면적 (m²/ha)

2. 기능적 지표:
   ├── 일차 생산성 (톤/ha/년)
   ├── 탄소 저장 (톤 C/ha)
   ├── 영양분 순환 (N, P 플럭스)
   ├── 침전물 축적 (cm/년)
   └── 수질 개선 (오염물질 감소 %)

3. 생물다양성 지표:
   ├── 맹그로브 종 풍부도
   ├── 어류 다양성 및 풍부도
   ├── 조류 종 풍부도
   ├── 갑각류 밀도
   └── Shannon 다양성 지수

4. 수문 지표:
   ├── 조수 침수 빈도
   ├── 침수 지속 시간
   ├── 염분 범위 (ppt)
   ├── 수위 변동
   └── 수류 속도

5. 사회경제적 지표:
   ├── 지역 고용 (인일)
   ├── 어획량 변화 (kg/ha/년)
   ├── 생태관광 수익 (USD/년)
   ├── 지역사회 만족도 (설문)
   └── 생계 개선 (가구 수)

5.2 현장 측정 방법

지상 기반 모니터링은 상세한 데이터와 원격 감지 검증을 위한 기준을 제공합니다.

영구 플롯 설치

표준 플롯 프로토콜:

  • 플롯 크기: 10m × 10m (100m²) 식생용, 5m × 5m 모종용
  • 플롯 수: 최소 20-30개/현장 (통계적 신뢰성)
  • 배치: 층화 무작위 (대상, 식재 연도, 지형 대표)
  • 표시: GPS, PVC 파이프 모서리 표시, 금속 태그
  • 방향: 진북에 정렬, 나침반 방위 기록

식생 측정

측정 도구 방법 빈도
생존/밀도 카운터, 데이터시트 플롯 내 모든 살아있는 줄기 카운트 3, 6, 12개월, 이후 연 1회
높이 측정봉, 하이프소미터 지면에서 최고 잎까지, 10% 표본 6, 12개월, 이후 연 1회
DBH (직경) 직경 테이프, 캘리퍼스 1.3m 높이, 모든 > 5cm 줄기 연 1회 (나무 > 2년)
캐노피 피복 구형 밀도계, 앱 플롯 중심에서 4-5 방향 연 1회
건강 상태 시각적 평가 잎 색, 질병, 손상 점수 (1-5) 연 2회

환경 매개변수

현장 환경 모니터링 프로토콜:

1. 토양 매개변수:
   ├── 염분: 토양 전도도 미터 (0-30cm 깊이)
   ├── pH: 휴대용 pH 미터 (1:2 토양:물)
   ├── 산화환원 전위: 백금 전극 (10, 20cm 깊이)
   ├── 유기물: LOI (450°C 연소 손실)
   └── 텍스처: 입자 크기 분석 (샌드, 실트, 클레이 %)

2. 수문 측정:
   ├── 수위: 압력 로거 (연속, 15분 간격)
   ├── 염분: CTD 센서 (연속) 또는 염분 측정기 (주간)
   ├── 온도: 수온 로거 (연속)
   ├── 조수 침수: 석고 블록 (월간 교체)
   └── 유속: 유속계 (분기별)

3. 대기 조건:
   ├── 온도: 기상대 (연속)
   ├── 강수량: 우량계 (일일)
   ├── 습도: 습도 센서 (연속)
   ├── 광도: PAR 센서 (연속 또는 주간)
   └── 풍속: 풍속계 (연속)

원격 감지는 넓은 지역에 대한 비용 효과적이고 반복 가능한 모니터링을 제공하며, 시간 경과에 따른 변화를 감지합니다.

위성 해상도 재방문 주기 용도 비용

위성 이미지 분석 워크플로우

Google Earth Engine을 사용한 맹그로브 모니터링:

// 1단계: 관심 지역 및 기간 정의
var aoi = ee.Geometry.Rectangle([lng1, lat1, lng2, lat2]);
var startDate = '2020-01-01';
var endDate = '2024-12-31';

// 2단계: Sentinel-2 이미지 컬렉션 로드 및 필터링
var s2 = ee.ImageCollection('COPERNICUS/S2_SR')
  .filterBounds(aoi)
  .filterDate(startDate, endDate)
  .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 20));

// 3단계: NDVI 계산 (정규 식생 지수)
var addNDVI = function(image) {
  var ndvi = image.normalizedDifference(['B8', 'B4'])
                   .rename('NDVI');
  return image.addBands(ndvi);
};
var s2WithNDVI = s2.map(addNDVI);

// 4단계: 맹그로브 지수 계산 (예: MVI)
var addMVI = function(image) {
  var mvi = image.expression(
    '(NIR - GREEN) / (SWIR - GREEN)',
    {
      'NIR': image.select('B8'),
      'GREEN': image.select('B3'),
      'SWIR': image.select('B11')
    }).rename('MVI');
  return image.addBands(mvi);
};
var s2WithMVI = s2WithNDVI.map(addMVI);

// 5단계: 중앙값 합성 생성
var composite = s2WithMVI.median().clip(aoi);

// 6단계: 맹그로브 분류
var mangrove = composite.select(['NDVI', 'MVI'])
  .expression(
    '(NDVI > 0.4) && (MVI > 0.3) ? 1 : 0',
    {
      'NDVI': composite.select('NDVI'),
      'MVI': composite.select('MVI')
    });

// 7단계: 면적 계산
var area = mangrove.multiply(ee.Image.pixelArea())
  .reduceRegion({
    reducer: ee.Reducer.sum(),
    geometry: aoi,
    scale: 10,
    maxPixels: 1e13
  });
var mangroveAreaHa = ee.Number(area.get('constant'))
                       .divide(10000);
print('맹그로브 면적 (ha):', mangroveAreaHa);

// 8단계: 변화 감지 (2년 비교)
var image2020 = s2WithMVI.filterDate('2020-01-01', '2020-12-31')
                         .median();
var image2024 = s2WithMVI.filterDate('2024-01-01', '2024-12-31')
                         .median();
var change = image2024.select('NDVI')
                      .subtract(image2020.select('NDVI'));
// NDVI 증가 > 0.2 = 복원 성공, 감소 < -0.2 = 퇴화

5.4 드론 (UAV) 모니터링

드론은 위성과 지상 조사 사이의 간격을 메우며, 고해상도 데이터를 제공하고 비용 효과적입니다.

드론 모니터링 프로토콜

UAV 조사 표준 작업 절차:

  1. 사전 비행 계획:
    • 현장 경계 매핑 (GPS, GIS)
    • 비행 고도 설정: 50-120m (해상도 대 범위)
    • 이미지 중첩: 정면 80%, 측면 70%
    • 지상 기준점 (GCP) 배치: 5-10개/50ha
  2. 센서 선택:
    • RGB 카메라: 오소모자이크, 높이 모델 (DSM)
    • 다중스펙트럼: NDVI, 건강 평가
    • 열화상: 수분 스트레스 감지
    • LiDAR: 구조 측정, 바이오매스 추정
  3. 데이터 수집:
    • 비행 시간: 정오 ±2시간 (일관된 조명)
    • 기상 조건: 풍속 < 10m/s, 구름 < 30%
    • 배터리 관리: 비행당 15-20분, 예비 배터리
  4. 처리 및 분석:
    • 포토그래메트리 소프트웨어 (Pix4D, Agisoft, OpenDroneMap)
    • 오소모자이크 생성 (지리 참조 이미지)
    • 디지털 표면 모델 (DSM, 높이 데이터)
    • 분류 및 특징 추출 (AI/머신러닝)

5.5 생물다양성 평가

생태계 회복의 핵심 지표는 생물다양성의 귀환입니다 - 종 풍부도, 풍부함, 생태계 기능.

어류 모니터링

어류 다양성 평가 방법:

  • 수중 시각 조사 (UVC): 스노클링/다이빙, 표준화된 트랜섹트 (50m × 5m), 종 ID 및 카운트
  • 길 그물: 표준 그물 (30m × 2m), 2시간 세트, 모든 어류 측정 및 방류
  • 투망: 얕은 지역, 치어 및 작은 종, 정량적 샘플링
  • 환경 DNA (eDNA): 물 샘플 (1-5L), DNA 시퀀싱, 종 존재/부재
  • 수중 비디오: 유인 원격 수중 비디오 (BRUV), 1시간 전개, 종 풍부도

   ├── 구덩이 함정: 10개/현장, 24시간

2. 연체동물 (굴, 달팽이):
   ├── 벨트 트랜섹트: 50m × 2m
   ├── 뿌리 표본 조사: 10 뿌리 × 0.5m 길이
   ├── 침전물 코어: 10cm 직경 × 20cm 깊이
   └── 지표: 다양성, 바이오매스 (g/m²), 크기

3. 곤충:
   ├── 스윕 네트: 100 스윕/현장
   ├── 말레이즈 트랩: 연속 (주간 수집)
   ├── 라이트 트랩: 야간 (월간)
   └── 지표: 종 풍부도, 수분 매개자, 해충

4. 데이터 분석:
   ├── Shannon 다양성 지수: H' = -Σ(pi × ln(pi))
   ├── Simpson 다양성: D = 1 - Σ(pi²)
   ├── 종 풍부도: 총 종 수
   └── 기준선 대비 비교 (참조 현장)

5.6 탄소 측정 및 검증

탄소 금융과 기후 완화 주장을 위해서는 정확한 탄소 측정이 필수적입니다.

탄소 풀 측정

탄소 풀 방법 샘플링 정확도
지상부 바이오매스 이상 방정식 (DBH, 높이) 모든 나무 > 5cm DBH ±15-25%
지하부 바이오매스 뿌리:싹 비율 (종 특이적) AGB의 0.5-2.0 × ±25-40%
토양 유기 탄소 코어 샘플, LOI 또는 건식 연소 20-30 코어, 0-100cm 깊이 ±10-20%
죽은 목재 체적 × 밀도 모든 조각 > 10cm 직경 ±20-30%
낙엽 낙엽 함정 (0.25m²) 10-20 함정/현장, 월간 ±10-15%

탄소 계산 예

맹그로브 탄소 저장량 계산 단계:

1. 지상부 바이오매스 (AGB):
   # 관련 분야 자료 방정식 사용
   AGB (kg) = 0.251 × ρ × (DBH²×H)^0.911

   여기서:
   - ρ = 목재 밀도 (g/cm³)
   - DBH = 흉고직경 (cm)
   - H = 나무 높이 (m)

   예: Rhizophora, ρ=0.79, DBH=15cm, H=8m
   AGB = 0.251 × 0.79 × (15²×8)^0.911
       = 0.251 × 0.79 × 1800^0.911
       = 0.251 × 0.79 × 1089
       = 216 kg/나무

2. 지하부 바이오매스 (BGB):
   BGB = AGB × 뿌리:싹 비율

   Rhizophora의 경우 R:S = 0.6-1.0 (평균 0.8 사용)
   BGB = 216 × 0.8 = 173 kg/나무

3. 총 바이오매스:
   총 = AGB + BGB = 216 + 173 = 389 kg/나무

4. 탄소 함량:
   탄소 = 바이오매스 × 0.47 (기본 47% C)
   탄소 = 389 × 0.47 = 183 kg C/나무

5. CO₂ 등가:
   CO₂e = 탄소 × 3.67 (C의 분자량/CO₂의 분자량)
   CO₂e = 183 × 3.67 = 671 kg CO₂/나무

6. 헥타르당 확장 (밀도 = 1,000 나무/ha):
   탄소 = 183 kg C/나무 × 1,000 = 183,000 kg C/ha
        = 183 톤 C/ha
   CO₂e = 671 톤 CO₂/ha

7. 토양 유기 탄소 (SOC) 추가:
   # 코어 샘플에서 측정 (0-100cm)
   가정: SOC = 250 톤 C/ha (전형적)

   총 탄소 저장 = 183 + 250 = 433 톤 C/ha
   총 CO₂e = 433 × 3.67 = 1,589 톤 CO₂/ha

5.7 사회경제적 영향 평가

복원 프로젝트는 지역 사회에 혜택을 주어야 합니다. 사회경제적 지표는 생계, 웰빙, 거버넌스의 변화를 추적합니다.

사회경제적 모니터링 방법

참여적 평가 도구:

  • 가구 조사: 반구조화 설문지, 기준선 및 주기적 (3-5년), 무작위 샘플 (30-50 가구)
  • 집중 그룹 토론 (FGD): 8-12 참가자, 주제 가이드, 다양한 이해관계자 그룹
  • 핵심 정보 제공자 인터뷰: 지도자, 장로, 자원 사용자, 반구조화
  • 참여적 농촌 평가 (PRA): 시간선, 자원 매핑, 계절 달력, 커뮤니티 워크숍
  • 모바일 데이터 수집: ODK, KoBoToolbox, 디지털 조사, 실시간 데이터

주요 사회경제적 지표

지표 측정 방법 목표
가구 소득 조사 (USD/월) > 20% 증가 (5년)
어획량 어부 일지 (kg/일) > 30% 증가 (10년)
고용 프로젝트 기록 (인일) > 100 인일/ha
생계 다양화 조사 (소득원 수) > 2 소득원/가구
식량 안보 HFIAS 점수 (0-27) 점수 감소 > 30%
지역사회 만족 리커트 척도 (1-5) > 3.5 평균
성별 형평성 여성 참여율 (%) > 40%

5.8 데이터 관리 및 품질 관리

좋은 데이터 관리는 정확성, 재현성, 장기 가치를 보장합니다.

데이터 관리 모범 사례

데이터 관리 워크플로우:

1. 데이터 수집:
   ├── 표준화된 양식 (디지털 또는 종이)
   ├── 필드 ID, 날짜, 관찰자, 조건 기록
   ├── 사진 문서화 (지오태그)
   ├── 즉각적인 품질 확인 (범위, 이상치)
   └── 백업 (복제 노트북, 클라우드)

2. 데이터 입력 및 검증:
   ├── 수집 후 24-48시간 내 입력
   ├── 이중 입력 (오류 감지)
   ├── 범위 확인 (최소/최대 임계값)
   ├── 논리 확인 (일관성 규칙)
   └── 이상치 플래그 지정 (통계 테스트)

3. 데이터 저장:
   ├── 원본 데이터 보존 (읽기 전용)
   ├── 버전 관리 (날짜, 버전 번호)
   ├── 중복 백업 (로컬 + 클라우드)
   ├── 메타데이터 문서화 (누가, 무엇을, 언제, 어디서, 어떻게)
   └── 접근 제어 (권한, 보안)

4. 데이터 분석:
   ├── 재현 가능한 스크립트 (R, Python)
   ├── 통계 소프트웨어 (R, SPSS, SAS)
   ├── 공간 분석 (QGIS, ArcGIS, Python)
   ├── 문서화된 방법 (README, 주석)
   └── 피어 리뷰 (내부 확인)

5. 데이터 공유 및 보관:
   ├── 공개 저장소 (Dryad, Figshare, Zenodo)
   ├── DOI 할당 (인용 가능)
   ├── 메타데이터 표준 (EML, ISO 19115)
   ├── 라이선스 (CC-BY, CC0)
   └── FAIR 원칙 (찾기 쉬움, 접근 가능, 상호 운용 가능, 재사용 가능)

5.10 적응 관리 트리거

모니터링 데이터는 관리 결정을 알려야 합니다. 미리 정의된 트리거는 조치가 필요한 시기를 결정합니다.

관리 트리거 및 대응

조치 트리거 및 권장 대응:

  • 생존율 < 50% (1년): 즉시 조사, 원인 식별, 보호 조치 또는 재식재
  • 성장 < 15cm/년: 스트레스 요인 평가 (염분, 영양분, 물), 토양 개량 고려
  • 자연 모집 < 5 모종/100m² (3년 후): 번식 장벽 조사, 종자 분산 보충
  • 종 다양성 < 50% 원래 (5년): 추가 종 도입, 서식지 개선
  • 어류 풍부도 < 20% 증가 (5년): 연결성 평가, 어업 압력, 수질
  • 지역사회 만족 < 2.5 (리커트 1-5): FGD 소집, 우려 사항 해결, 참여 개선
  • 질병/해충 발생 > 30% 나무: 병원체 식별, 치료 옵션, 감염된 재료 제거
  • 기후 극한 사건 (태풍, 홍수): 피해 평가, 잔해 청소, 복구 계획

장 요약

효과적인 모니터링과 평가는 복원 성공의 기초입니다. 포괄적인 프레임워크는 구조적 (생존율, 밀도, 높이), 기능적 (탄소, 영양분), 생물다양성 (어류, 조류, 무척추동물), 수문, 사회경제적 지표를 포함합니다. 모니터링 수준은 기본 ($500-1,500/ha/년)에서 종합 ($4,000-10,000/ha/년)까지 다양하며, 프로젝트 목표와 자원에 맞게 조정됩니다.

원격 감지는 비용 효과적인 대규모 모니터링을 제공합니다: Sentinel-2 (무료, 10m, 5일), Planet ($$, 3-5m, 매일), WorldView ($$$, 0.3-2m). Google Earth Engine은 시계열 분석과 변화 감지를 가능하게 합니다. 드론은 위성(대규모)과 지상(상세)의 간격을 메우며, 2-5cm 정확도로 100ha를 $2,000-4,000에 매핑합니다. 하이브리드 접근법(드론 + 지상 플롯)이 최고의 결과를 제공합니다.

탄소 측정은 이상 방정식(AGB), 뿌리:싹 비율(BGB), 코어 샘플링(SOC)을 사용하여 총 탄소 저장량을 계산합니다. 사회경제적 평가는 참여적 도구(조사, FGD, PRA)를 사용하여 생계, 소득, 웰빙의 변화를 추적합니다. 좋은 데이터 관리는 품질 관리, 버전 관리, 백업, FAIR 원칙을 따릅니다. 적응 관리 트리거는 모니터링 데이터를 시기적절한 조치로 변환하여 학습과 지속적인 개선을 보장합니다.
