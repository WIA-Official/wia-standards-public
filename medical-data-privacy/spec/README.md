# WIA-MED-017 Specifications

Technical specifications for the Medical Data Privacy Standard.

## Documents

- **WIA-MED-017-v1.0.md** - Complete technical specification

## Compliance Frameworks

- HIPAA (Health Insurance Portability and Accountability Act)
- GDPR (General Data Protection Regulation)
- HITECH Act
- ISO 27001 (Information Security Management)
- ISO 27701 (Privacy Information Management)

## Key Standards

- Encryption: AES-256-GCM, TLS 1.3
- Access Control: RBAC, ABAC, MFA
- De-identification: k-anonymity (k≥5), differential privacy (ε≤1.0)
- Logging: 6-year retention, tamper-proof
- Data Transfer: SCCs, TIA, adequacy decisions
