🆕 7장: 콜드 스타트 문제

부트스트랩 과제

콜드 스타트 문제는 추천 시스템에서 가장 지속적인 과제 중 하나입니다: 상호작용 이력이 없는 신규 사용자에게 어떻게 추천할까요? 아무도 평가하지 않은 신규 아이템을 어떻게 추천할까요?

👤 신규 사용자 콜드 스타트

전략 1: 인기 기반 추천

대부분의 사람들에게 효과가 있는 것부터 시작하세요 - 인기 아이템, 트렌딩 콘텐츠, 베스트셀러

전략 2: 온보딩 설문지

가입 중에 사용자에게 선호도를 명시적으로 물어보세요

전략 3: 인구통계 기반 추천

인구통계 정보(연령, 위치, 성별)를 사용하여 유사한 사용자 찾기

📦 신규 아이템 콜드 스타트

전략 1: 콘텐츠 기반 특징

아이템 메타데이터를 사용하여 유사한 아이템 찾기

전략 2: 능동 학습

다양한 사용자 그룹에 전략적으로 신규 아이템을 표시하여 빠르게 피드백 수집

전략 3: 전문가 큐레이션

높은 가치의 신규 아이템의 경우 도메인 전문가가 초기 배치와 추천 스팟 생성

🌟 하이브리드 콜드 스타트 전략

가장 효과적인 접근법은 여러 전략을 결합합니다

🔄 콜드에서 웜으로: 전환 전략

사용자가 상호작용을 누적하면 콜드 스타트 방법에서 협업 필터링으로 부드럽게 전환
