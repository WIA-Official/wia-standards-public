1. 가정법원의 역할

디지털 유산 집행자 지정은 가정법원의 비송사건 절차로 진행됩니다. 2025년부터 서울가정법원을 포함한 전국 18개 가정법원에서 전담재판부를 운영하고 있습니다.

📌 학습 목표

  • 디지털 유산 집행자 지정 신청 절차 이해
  • 자격 요건과 결격 사유 파악
  • 필요 서류 준비 및 작성 방법
  • 심리 과정과 법원의 판단 기준
  • 결정문 발급 및 집행 권한 행사

2. 신청 자격과 요건

2.1 신청 권한자

신청인 유형 우선순위 요건 비고
유언 지정자 1순위 유언장에 명시된 자 고인의 의사 최우선
배우자 2순위 법률혼 관계 사실혼 배제
직계비속 3순위 자녀, 손자녀 등 연장자 우선
직계존속 4순위 부모, 조부모 등 연소자 우선
전문 집행인 5순위 변호사, 공인회계사 등 법원 직권 지정 가능

2.2 디지털 유산 집행자 자격 요건

/**
 * WIA-DIGITAL_EXECUTOR 자격 요건
 * @standard WIA-DE-003
 * @legal 민법 제1091조 제2항
 */

interface ExecutorQualification {
  // 기본 요건
  basic: {
    age: number;                    // 만 19세 이상
    citizenship: 'KOREAN' | 'FOREIGN_RESIDENT';
    legalCapacity: boolean;         // 완전한 행위능력
    criminalRecord: boolean;        // 금고 이상 형 미수형
  };

  // 관계 요건
  relationship: {
    toDeceased: RelationType;
    verificationDoc: string;        // 가족관계증명서
    conflictOfInterest: boolean;    // 이해충돌 여부
  };

  // 기술 역량 (선택)
  technicalSkills?: {
    digitalLiteracy: 'BASIC' | 'INTERMEDIATE' | 'ADVANCED';
    certifications: string[];       // 관련 자격증
    experience: {
      years: number;
      domains: string[];            // 금융, IT, 법률 등
    }
  };

  // 결격 사유 확인
  disqualifications: {
    bankruptcy: boolean;            // 파산선고 미복권
    guardianship: boolean;          // 피성년후견인
    previousMisconduct: boolean;    // 과거 집행자 해임 이력
    adverseInterest: boolean;       // 상속인과 이해 상반
  };
}

3. 신청 서류 및 절차

3.1 필수 제출 서류

신원조회 동의서 신청인 작성 - 범죄경력 조회용

4. 심리 과정

4.1 단계별 절차

⚖️ 디지털 유산 집행자 지정 심리 단계

  1. 신청서 접수 (Day 0)
    • 법원 방문 또는 전자소송 시스템
    • 수수료: 20,000원
  2. 형식 심사 (Day 1-3)
    • 서류 완비 여부 확인
    • 보정 명령 (필요시)
  3. 이해관계인 통지 (Day 4-14)
    • 모든 상속인에게 통지
    • 의견 제출 기간: 14일
  4. 신원 조회 (Day 5-10)
    • 경찰청 범죄경력 조회
    • 신용정보 확인
  5. 심문 (필요시) (Day 15-21)
    • 신청인 출석 심문
    • 기술 역량 평가
  6. 결정 (Day 22-30)
    • 인용 또는 기각
    • 조건부 인용 가능
  7. 결정문 송달 (Day 31-35)
    • 신청인 및 이해관계인
    • 항고 기간: 14일
  8. 확정 (Day 49)
    • 항고 없이 기간 경과
    • 집행 권한 발생

심사 항목 평가 기준 배점 합격 기준

5. 결정문 유형

5.1 인용 결정

✅ 서울가정법원 2025느단12345 결정

주문

신청인 홍순례를 망 홍길동(1980.1.1.생, 2025.1.15. 사망)의 디지털 유산 집행자로 지정한다.

이유

  1. 신청인은 망인의 배우자로서 민법 제1091조에 따른 유언집행자 자격을 갖추었음
  2. 망인이 생전에 운영한 유튜브 채널(구독자 50만), 네이버 블로그 등 상당한 규모의 디지털 자산 존재
  3. 다른 상속인(자녀 2명)이 신청인의 집행자 지정에 서면으로 동의
  4. 신청인의 디지털 문해력 및 성실성이 인정됨

집행자 권한

  • 망인의 모든 디지털 계정 접근 및 관리
  • 디지털 자산 목록 작성 및 가치 평가
  • 플랫폼 사업자에 대한 정보 제공 요청
  • 금융기관에 대한 계좌 조회 및 거래
  • 디지털 저작물 수익 수령 및 분배
  • 불필요한 계정의 폐쇄 또는 추모 계정 전환

의무 사항

  • 매 3개월마다 법원에 집행 현황 보고
  • 상속인 전원에게 분기별 진행 상황 통지
  • 고인의 프라이버시 및 명예 보호
  • 집행 완료 후 최종 보고서 제출

5.2 조건부 인용 결정

📋 조건부 인용 사례

주문: 신청인을 디지털 유산 집행자로 지정하되, 다음 조건을 부과한다.

  • 공동 집행: 신청인과 상속인 대표 1인이 공동으로 집행
  • 감독인 지정: 변호사 A를 집행 감독인으로 지정
  • 보증 제공: 디지털 자산 가액의 30% 상당 보증금 공탁
  • 사전 승인: 1,000만원 초과 거래는 법원 사전 허가
  • 교육 이수: 디지털 유산 관리 교육 과정 수료

조건 부과 사유

신청인이 기본 자격은 갖추었으나, 디지털 자산의 규모가 크고(약 3억원), 일부 상속인이 우려를 표명하여 신중한 관리가 필요하다고 판단됨.

7. 집행 권한과 범위

7.1 권한별 행사 방법

/**
 * WIA-DIGITAL_EXECUTOR 권한 행사
 * @standard WIA-DE-004
 * @legal 민법 제1103조의2
 */

interface ExecutorAuthority {
  // 조회 권한
  inquiry: {
    scope: 'ALL_PLATFORMS' | 'SPECIFIED_ONLY';
    authentication: AuthMethod;
    validUntil: Date;

    actions: [
      'VIEW_ACCOUNT_LIST',      // 계정 목록 조회
      'ACCESS_CONTENT',         // 콘텐츠 접근
      'DOWNLOAD_DATA',          // 데이터 다운로드
      'VIEW_FINANCIALS'         // 금융정보 조회
    ]
  };

  // 관리 권한
  management: {
    accountActions: [
      'CHANGE_PASSWORD',        // 비밀번호 변경
      'UPDATE_SETTINGS',        // 설정 변경
      'ADD_AUTHORIZED_USER',    // 권한자 추가
      'TRANSFER_OWNERSHIP'      // 소유권 이전
    ];

    contentActions: [
      'EDIT_PROFILE',           // 프로필 수정
      'DELETE_POSTS',           // 게시물 삭제
      'MANAGE_PRIVACY',         // 프라이버시 설정
      'DOWNLOAD_ARCHIVE'        // 전체 아카이브
    ]
  };

  // 처분 권한
  disposition: {
    allowedActions: [
      'DELETE_ACCOUNT',         // 계정 삭제
      'MEMORIALIZE',            // 추모 전환
      'DEACTIVATE',             // 비활성화
      'TRANSFER_TO_HEIR'        // 상속인 이전
    ];

    restrictions: {
      requiresCourtApproval: boolean;    // 법원 승인 필요
      requiresHeirConsent: boolean;      // 상속인 동의 필요
      monetaryLimit: number;             // 금액 한도
    }
  };

  // 금융 권한
  financial: {
    banking: {
      viewBalance: boolean;
      makeTransfers: boolean;
      closeAccounts: boolean;
      maximumAmount: number;    // 1회 이체 한도
    };

    cryptocurrency: {
      viewWallets: boolean;
      initiateTransfers: boolean;
      requiresCustodian: boolean;    // 공동 관리 필요
    };

    securities: {
      viewPortfolio: boolean;
      executeTrades: boolean;
      liquidate: boolean;
      approvalRequired: boolean;
    }
  };
}

8. 전문 집행인 제도

8.1 전문 집행인의 필요성

다음의 경우 법원은 직권으로 전문 집행인을 지정할 수 있습니다:

  • 디지털 자산의 가액이 5억원을 초과하는 경우
  • 상속인 간 심각한 분쟁이 예상되는 경우
  • 암호화폐, NFT 등 전문 지식이 필요한 경우
  • 국제적 디지털 자산이 포함된 경우
  • 적격한 친족 집행자가 없는 경우

8.2 전문 집행인 자격

자격 유형 요구사항 보수
변호사 변호사 자격 + 디지털 유산 교육 이수 자산 가액의 3-5%
공인회계사 회계사 자격 + 금융 전문성 자산 가액의 2-4%
IT 전문가 정보보안기사 + 경력 5년 이상 시간당 20-30만원
신탁회사 금융위원회 인가 자산 가액의 1-2%/년

9. 집행 보고와 감독

9.1 정기 보고서 양식

/**
 * 디지털 유산 집행 분기 보고서
 * @frequency 분기별 (3개월마다)
 */

interface QuarterlyReport {
  reportPeriod: {
    from: Date;
    to: Date;
    quarterNumber: 1 | 2 | 3 | 4;
  };

  // 집행 진행 상황
  progress: {
    totalAccounts: number;          // 전체 계정 수
    completedAccounts: number;      // 처리 완료 계정
    pendingAccounts: number;        // 처리 중 계정
    progressPercentage: number;     // 진행률
  };

  // 계정별 처리 내역
  accountActions: Array<{
    platform: string;
    accountId: string;
    action: 'DELETED' | 'MEMORIALIZED' | 'TRANSFERRED' | 'ARCHIVED';
    actionDate: Date;
    heirApproval: boolean;
  }>;

  // 금융 거래 내역
  financialTransactions: Array<{
    date: Date;
    institution: string;
    type: 'DEPOSIT' | 'WITHDRAWAL' | 'TRANSFER';
    amount: number;
    purpose: string;
    approvedBy: string;
  }>;

  // 수익 현황
  revenues: {
    youtub AdRevenue: number;
    blogAdRevenue: number;
    affiliateIncome: number;
    otherIncome: number;
    totalRevenue: number;
  };

  // 지출 내역
  expenses: {
    platformFees: number;
    professionalFees: number;
    administrativeCosts: number;
    totalExpenses: number;
  };

  // 이슈 및 특이사항
  issues: Array<{
    platform: string;
    description: string;
    severity: 'LOW' | 'MEDIUM' | 'HIGH';
    resolution: string;
  }>;

  // 다음 분기 계획
  nextQuarterPlan: {
    targetAccounts: string[];
    anticipatedChallenges: string[];
    requiredApprovals: string[];
  };
}

10. 요약 및 핵심 개념

🎯 Chapter 03 핵심 요약

  • 관할 법원: 가정법원 비송사건 절차, 전국 18개 법원 전담재판부 운영

  • 필수 서류: 신청서, 사망진단서, 가족관계증명서, 유언장, 디지털 자산 목록
  • 심리 기간: 약 30-50일 (신청 → 형식심사 → 통지 → 심문 → 결정 → 확정)
  • 판단 기준: 법적 자격(30점), 관계 적정성(25점), 기술 역량(20점), 공정성(15점)
  • 결정 유형: 인용, 조건부 인용 (공동집행, 감독인, 보증금 등), 기각
  • 집행 권한: 조회, 관리, 처분, 금융 권한 (금액·대상별 제한)
  • 전문 집행인: 5억원 초과, 분쟁 예상, 전문성 필요 시 법원 직권 지정
  • 보고 의무: 분기별 집행 현황, 금융 거래 내역, 이슈 보고
  • 弘益人間: 공정하고 투명한 절차로 모든 이해관계인에게 이롭게 하라 🌍
