금융 디지털 자산 관리

1. 금융 디지털 자산의 범위

금융 디지털 자산은 전통적 금융자산(은행, 증권, 보험)과 신규 디지털 자산(암호화폐, NFT, 디지털 화폐)을 모두 포함합니다. 2024년 금융감독원 조사에 따르면, 한국 성인 1인당 평균 금융 디지털 자산은 약 2,340만 원에 달합니다.

📌 학습 목표

  • 금융감독원 디지털 자산 통합 조회 시스템 활용
  • 은행별 디지털 뱅킹 상속 절차
  • 증권·펀드 디지털 자산 처리
  • 암호화폐 상속 가이드 및 기술적 이해
  • 보험·연금 디지털 청구 프로세스

2. 금융감독원 통합 조회 시스템

2.1 "금융 유산 ONE-CALL" 서비스

금융감독원은 2024년 10월 "금융 유산 ONE-CALL" 서비스를 출시하여, 고인의 모든 금융자산을 한 번에 조회할 수 있게 했습니다.

조회 가능 자산 포함 기관 조회 항목 처리 시간
은행 예금 은행 18개 예금, 적금, 대출, 카드 즉시
증권 증권사 56개 주식, 채권, 펀드, CMA 3시간 이내
보험 보험사 48개 생명보험, 손해보험, 연금 24시간 이내
저축은행 저축은행 79개 예금, 적금 즉시
암호화폐 거래소 4개 업비트, 빗썸, 코인원, 코빗 48시간 이내

2.2 ONE-CALL 신청 절차

/**
 * 금융감독원 ONE-CALL API
 * @endpoint https://api.fss.or.kr/v1/digital-estate/one-call
 * @authorization Bearer {법원_결정문_기반_토큰}
 */

interface OneCallRequest {
  // 집행자 인증
  executor: {
    id: string;
    name: string;
    courtOrderId: string;        // 법원 결정문 번호
    digitalCert: string;          // 공동인증서
  };

  // 고인 정보
  deceased: {
    name: string;
    ssn: string;                  // 주민등록번호 (암호화)
    dateOfDeath: Date;
    deathCertificateId: string;
  };

  // 조회 범위
  scope: {
    assetTypes: [
      'BANK_DEPOSIT',             // 은행 예금
      'SECURITIES',               // 증권
      'INSURANCE',                // 보험
      'PENSION',                  // 연금
      'CRYPTO',                   // 암호화폐
      'SAVINGS_BANK',             // 저축은행
      'LOANS'                     // 대출 (부채)
    ];
    includeDormant: boolean;      // 휴면 계좌 포함
    includeSmallAmount: boolean;  // 소액(1만원 미만) 포함
  };
}

// ONE-CALL 응답
interface OneCallResponse {
  requestId: string;
  status: 'PROCESSING' | 'COMPLETED' | 'PARTIAL';
  generatedAt: Date;

  summary: {
    totalInstitutions: number;
    totalAccounts: number;
    totalAssets: number;            // KRW
    totalLiabilities: number;       // 부채
    netAssets: number;              // 순자산
  };

  // 기관별 상세
  byInstitution: Array<{
    institutionType: string;
    institutionName: string;
    accounts: Array<{
      accountNumber: string;        // 마스킹: "123-45-******"
      accountType: string;
      balance: number;
      lastActivity: Date;
      status: 'ACTIVE' | 'DORMANT' | 'CLOSED';
    }>;
    subtotal: number;
  }>;

  // 다운로드
  downloadLinks: {
    pdf: string;                    // PDF 보고서
    excel: string;                  // 엑셀 파일
    json: string;                   // JSON 데이터
  };
}

3. 은행 디지털 뱅킹 상속

3.1 주요 시중은행 절차

2024년 금융감독원 지침에 따라 모든 은행이 표준화된 디지털 뱅킹 상속 절차를 따릅니다.

단계 절차 소요 시간 비고
1. 신청 온라인 또는 영업점 방문 30분 KB, 신한, 우리는 온라인 가능
2. 서류 제출 법원 결정문, 가족관계증명서 즉시 전자문서 인정
3. 본인 인증 공동인증서 또는 금융인증서 5분 Level 3 이상
4. 계좌 조회 전체 계좌 목록 확인 즉시 휴면 계좌 포함
5. 자산 이전 상속인 계좌로 이체 1-3 영업일 대출 상계 후
6. 계좌 해지 모든 계좌 폐쇄 즉시 선택 사항

3.2 특수 상황 처리

⚠️ 주의: 대출 및 마이너스 통장

자동 상계: 예금과 대출이 동시에 있는 경우, 은행은 자동으로 상계합니다.

  • 예시: 예금 5,000만원, 대출 3,000만원 → 순자산 2,000만원만 상속
  • 예외: 주택담보대출 등 특정 대출은 상속인이 승계 가능
  • 절차: 상계 전 집행자에게 통지 (7일 이내 이의 제기 가능)

휴면 계좌: 최종 거래일로부터 5년 경과 시 휴면 계좌로 전환

  • 휴면 계좌도 ONE-CALL로 조회 가능
  • 휴면 해제 후 즉시 이체 가능
  • 10년 경과 시 금융결제원으로 이관 (영구 보존)

4. 증권 디지털 자산

4.1 주식 및 펀드 상속

증권 자산은 시가 변동이 크므로 신속한 처리가 중요합니다.

/**
 * 증권 계좌 상속 처리
 * @type 주식, 펀드, ETF, 채권
 */

interface SecuritiesInheritance {
  // Step 1: 계좌 조회
  accountInquiry: {
    method: '증권사 방문 또는 온라인',
    required Docs: [
      '법원 결정문',
      '가족관계증명서',
      '신분증'
    ];
    processing Time: '즉시';
  };

  // Step 2: 자산 평가
  valuation: {
    basis: '사망일 종가 기준',
    volatileAssets: {
      stocks: '실시간 시가',
      funds: 'NAV (순자산가치)',
      bonds: '채권 시장가'
    };
    freeze Period: '평가 완료까지 거래 정지';
  };

  // Step 3: 처분 방식 선택
  disposalOptions: {
    option1: {
      name: '현금화 (매도)',
      pros: '즉시 상속 분배 가능',
      cons: '매도 시점 시세 리스크',
      tax: '양도소득세 (고인 기준가 적용)'
    };
    option2: {
      name: '계좌 이전 (승계)',
      pros: '시세 변동 영향 없음',
      cons: '증권사 수수료 (자산의 0.1-0.3%)',
      tax: '증여세 아님 (상속세만)'
    };
    option3: {
      name: '분할 매도',
      pros: '시세 리스크 분산',
      cons: '처리 기간 장기화',
      tax: '거래별 양도세 계산'
    }
  };

  // Step 4: 세금 처리
  taxation: {
    inheritanceTax: {
      threshold: 200000000,         // 2억원 공제
      rate: '10-50% 누진',
      deadline: '6개월 이내 신고'
    };
    capital GainsTax: {
      applicable: '상속 후 매도 시',
      costBasis: '고인의 취득가',
      rate: '20-25%'
    }
  };
}

5. 암호화폐 상속

5.1 중앙화 거래소 (CEX)

업비트, 빗썸, 코인원, 코빗 등 국내 4대 거래소는 표준화된 상속 절차를 제공합니다.

✅ 업비트 암호화폐 상속 절차

  1. 고객센터 문의
    • 이메일: support@upbit.com
    • 제목: "디지털 자산 상속 신청"
  2. 서류 제출
    • 법원 결정문 (필수)
    • 사망진단서
    • 가족관계증명서
    • 집행자 신분증
  3. 본인 인증
    • 휴대폰 인증
    • 이메일 인증
    • OTP 해제 (고객센터 협조)
  4. 출금 계좌 등록
    • 집행자 명의 계좌
    • 1원 인증
  5. 자산 이체
    • KRW 출금: 즉시-1일
    • 코인 매도 후 출금: 2-3일
    • 수수료: 출금액의 0.05%

처리 기간: 총 14-30일 (서류 검토 기간 포함)

5.2 개인 지갑 (하드웨어 월렛, 소프트웨어 월렛)

🔐 개인 지갑 상속의 어려움

핵심 문제: 개인키 (Private Key) 없이는 접근 불가능

지갑 유형 복구 방법 성공률
하드웨어 월렛 시드 문구 (12-24단어) 시드 있으면 100%
소프트웨어 월렛 개인키 또는 keystore 파일 비밀번호 필요
종이 지갑 QR 코드 또는 개인키 종이 보존 시 100%
브레인 월렛 암기한 문구 사실상 불가능

법원도 강제할 수 없는 이유:

  • 암호학적으로 불가역적 (개인키 = 절대 권한)
  • 중앙 서버 없음 (탈중앙화)
  • 복구 기능 없음 (by design)

사전 대비책:

  1. 시드 문구를 금고에 보관
  2. 시드를 2-3개로 분할 보관 (Shamir's Secret Sharing)
  3. 다중서명 지갑 사용 (2-of-3, 3-of-5 등)
  4. 스마트 계약 기반 상속 설정 (예: Gnosis Safe Inheritance 모듈)

6. 보험 및 연금

6.1 생명보험 디지털 청구

2024년부터 모든 생명보험사가 온라인 보험금 청구를 지원합니다.

/**
 * 생명보험 디지털 청구
 * @platform 생명보험협회 통합 시스템
 */

interface LifeInsuranceClaim {
  // 보험 조회
  inquiry: {
    method: '생명보험협회 보험금 조회 서비스',
    url: 'https://www.klia.or.kr/consumer/claim',
    required: ['주민등록번호', '이름'],
    result: '전체 생명보험사 가입 내역'
  };

  // 청구 절차
  claimProcess: {
    step1: {
      action: '온라인 청구서 작성',
      duration: '10-15분',
      docs: [
        '사망진단서',
        '가족관계증명서',
        '청구인 통장사본'
      ]
    };
    step2: {
      action: '보험사 심사',
      duration: '3-7일',
      criteria: [
        '사망 원인 확인',
        '보험금 지급 사유 검토',
        '수익자 확인'
      ]
    };
    step3: {
      action: '보험금 지급',
      duration: '즉시-3일',
      method: '계좌 이체'
    }
  };

  // 특수 케이스
  specialCases: {
    suicide: {
      condition: '가입 후 2년 이내 자살',
      result: '보험금 부지급',
      exception: '정신질환 등 책임조각 사유'
    };
    accident: {
      condition: '재해 사망',
      result: '일반 사망 보험금 + 재해 특약',
      additionalDocs: '사고 경위서'
    };
    overseas: {
      condition: '해외 사망',
      result: '번역 공증 필요',
      processingTime: '30-60일'
    }
  };

7. 세금 및 법적 의무

7.1 상속세 신고 및 납부

금융 디지털 자산을 상속받으면 반드시 상속세를 신고하고 납부해야 합니다.

항목 내용 기한 비고
신고 기한 사망일로부터 6개월 이내 필수 해외 거주자는 9개월
공제액 기본 2억원 + 배우자 5억원 - 자녀 공제 별도
세율 10% ~ 50% (누진) - 과세표준에 따라
납부 방법 일시납 또는 분납 (5년) 선택 2천만원 초과 시 분납 가능
연부연납 최대 5년 분할 납부 신청 이자 2.1%/년

8. 실무 사례 연구

📋 사례 1: 다수 금융기관 자산

상황: 故 김씨(65세), 은행 8곳, 증권사 3곳, 보험 5개, 암호화폐 거래소 2곳

추정 자산: 약 8억원

문제: 어디에 얼마나 있는지 파악 어려움

해결:

  1. 금융감독원 ONE-CALL 신청 (3시간 만에 전체 파악)
  2. WIA 통합 대시보드에서 우선순위 설정
  3. 은행 → 증권사 → 보험 → 암호화폐 순으로 처리
  4. 총 처리 기간: 45일 (일반적으로는 6개월 소요)
  5. 상속세: 1억 2천만원 (15% 세율 적용)

교훈: ONE-CALL 서비스로 조회 기간 90% 단축

📋 사례 2: 암호화폐 개인 지갑 분실

상황: 故 이씨(42세), 레저 나노 하드웨어 월렛에 비트코인 5 BTC (약 5억원)

문제: 시드 문구 (24단어) 분실, 가족 아무도 모름

시도:

  • 법원에 강제 개봉 청구 → 기각 (기술적 불가능)
  • 레저 제조사에 문의 → "시드 없이는 복구 불가"
  • 해킹 업체에 의뢰 → "brute force 공격도 수백년 소요"

결과: 5억원 상당 비트코인 영구 손실

교훈: 생전에 시드 문구를 안전하게 보관하고 가족에게 위치 알려야 함

9. 요약 및 핵심 개념

🎯 Chapter 05 핵심 요약

  • 금융 유산 ONE-CALL: 금융감독원 통합 시스템으로 모든 금융자산 일괄 조회
  • 은행 자산: 온라인 신청 가능 (KB, 신한, 우리), 1-3일 처리
  • 증권 자산: 현금화 vs 계좌 이전 선택, 수수료 0.1-0.3%
  • 암호화폐 (CEX): 법원 결정문으로 14-30일 처리, 수수료 0.05%
  • 암호화폐 (개인 지갑): 시드 문구 필수, 분실 시 복구 불가능
  • 보험: 온라인 청구 가능, 3-21일 지급
  • 연금: 유족연금 (국민연금), 전액 상속 (퇴직연금, IRP)
  • 상속세: 6개월 이내 신고, 2억원 기본공제, 10-50% 누진세율
  • 실무 팁: 우선순위 설정 (은행 → 증권 → 보험 → 암호화폐)
  • 弘益人間: 금융 디지털 자산이 유족에게 온전히 전달되어 널리 이롭게 하라 🌍
