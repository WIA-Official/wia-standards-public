Chapter 3

공역 구조 및 분류

A~G 등급 공역, 관제권, 비행정보구역

이 장에서 배울 내용:

  • ICAO 공역 분류 체계 (Class A-G)
  • 각 공역 등급의 특징과 요구사항
  • 관제권, 비행정보구역, 특수공역
  • 공역 관리 및 운영

3.1 공역 분류의 필요성

공역(Airspace)은 항공기가 비행하는 3차원 공간입니다. 안전하고 효율적인 항공 교통 관리를 위해, 공역은 그 특성과 요구사항에 따라 여러 등급으로 분류됩니다.

국제민간항공기구(ICAO)는 전 세계적으로 통일된 공역 분류 체계를 제시하고 있으며, 이는 Class A부터 Class G까지 7개 등급으로 나뉩니다.

3.2 ICAO 공역 분류

ICAO는 공역을 크게 두 가지 카테고리로 구분합니다:

  • 관제 공역(Controlled Airspace): Class A, B, C, D, E
  • 비관제 공역(Uncontrolled Airspace): Class F, G

3.2.1 공역 분류 개요 표

등급 비행 규칙 ATC 서비스 속도 제한 진입 허가
Class A IFR만 모든 항공기 없음 필요
Class B IFR/VFR 모든 항공기 250kts 이하 필요
Class C IFR/VFR 모든 항공기 250kts 이하 VFR 교신 필요
Class D IFR/VFR IFR만 200kts 이하 VFR 교신 필요
Class E IFR/VFR IFR만 250kts 이하 불필요
Class F IFR/VFR 조언만 없음 불필요
Class G IFR/VFR 없음 250kts 이하 불필요

3.3 각 공역 등급의 상세 설명

🔴 Class A Airspace (A등급 공역)

고도: 통상 FL180 (18,000피트) 이상

특징:

  • IFR(계기비행규칙) 비행만 허용
  • 모든 항공기에 ATC 분리 서비스 제공
  • 트랜스폰더 Mode C 이상 필수
  • 속도 제한 없음
  • VFR(시계비행규칙) 비행 금지

용도: 대부분의 상업 제트 여객기가 순항하는 고고도 항로

🟠 Class B Airspace (B등급 공역)

고도: 지표면부터 약 10,000피트

특징:

  • IFR 및 VFR 비행 모두 허용
  • 모든 항공기에 ATC 분리 서비스 제공
  • 진입 전 명시적 허가 필수
  • 트랜스폰더 Mode C 필수
  • 속도 제한: 250kts (10,000피트 이하)
  • 조종사 자격: 최소 Private Pilot

용도: 주요 공항 주변의 혼잡한 터미널 공역 (예: 뉴욕 JFK, 로스앤젤레스 LAX)

🟡 Class C Airspace (C등급 공역)

고도: 지표면부터 약 4,000피트

특징:

  • IFR 및 VFR 비행 모두 허용
  • 모든 항공기에 ATC 서비스 제공 (IFR은 분리, VFR은 교통 정보)
  • VFR은 교신 설정 필요 (명시적 허가는 불필요)
  • 트랜스폰더 Mode C 필수
  • 속도 제한: 250kts

용도: 중간 규모의 공항 주변 (예: 지역 허브 공항)

🟢 Class D Airspace (D등급 공역)

고도: 지표면부터 약 2,500피트

특징:

  • IFR 및 VFR 비행 모두 허용
  • IFR 항공기만 분리 서비스 제공
  • VFR 항공기는 교통 정보만 제공
  • VFR은 교신 설정 필요
  • 트랜스폰더 권장 (일부 국가는 필수)
  • 속도 제한: 200kts

용도: 관제탑이 있는 소규모 공항

🔵 Class E Airspace (E등급 공역)

고도: 다양 (통상 700 또는 1,200피트 AGL부터 FL180까지)

특징:

  • IFR 및 VFR 비행 모두 허용
  • IFR 항공기만 ATC 분리 서비스 제공
  • VFR 항공기는 ATC 서비스 없음
  • 진입 허가 불필요
  • 트랜스폰더 10,000피트 이상에서 필수
  • 속도 제한: 250kts (10,000피트 이하)

용도: 관제 공역 중 가장 넓은 영역 차지, Class A 아래 대부분

🟣 Class F Airspace (F등급 공역)

고도: 국가별로 상이

특징:

  • IFR 및 VFR 비행 모두 허용
  • 조언(Advisory) 서비스만 제공
  • ATC 분리 서비스 없음
  • 특수 활동 공역 (예: 군사 훈련 구역)
  • 대부분의 국가에서 사용하지 않음

용도: 유럽 일부 국가의 특수 공역

⚪ Class G Airspace (G등급 공역)

고도: 지표면부터 관제 공역 시작까지

특징:

  • IFR 및 VFR 비행 모두 허용
  • ATC 서비스 없음 (비관제 공역)
  • 조종사 책임 하 분리 유지
  • 진입 허가 불필요
  • 기본 속도 제한: 250kts (10,000피트 이하)

용도: 농촌 지역, 저고도 비행, 경량 항공기

3.4 특수 공역

일반 공역 분류 외에도 특별한 목적이나 제한을 가진 특수 공역들이 있습니다:

3.4.1 제한 공역(Restricted Area)

군사 활동, 미사일 발사, 위험 물질 처리 등이 이루어지는 공역으로, 사전 허가 없이 진입할 수 없습니다.

3.4.2 금지 공역(Prohibited Area)

국가 안보나 공공 안전을 위해 모든 항공기 진입이 금지된 공역입니다. 예: 청와대 상공, 백악관 상공

3.4.3 경고 공역(Warning Area)

위험 활동이 이루어질 수 있으나, 진입이 금지되지는 않는 공역입니다. 조종사는 각별히 주의해야 합니다.

3.4.4 군사 작전 구역(MOA - Military Operations Area)

군사 훈련 활동이 이루어지는 공역으로, IFR 교통은 분리되지만 VFR 조종사는 주의가 필요합니다.

3.4.5 경계 구역(Alert Area)

비정상적으로 높은 조종사 훈련 활동이나 비정상적인 항공 활동이 있는 구역입니다.

3.5 비행정보구역(FIR)

비행정보구역(Flight Information Region, FIR)은 항공 정보 서비스와 경보 서비스가 제공되는 공역입니다.

3.5.1 FIR의 특징

  • 국가 또는 지역 단위로 설정
  • 항공 정보 및 수색 구조 책임 구역
  • 공역 등급과 무관하게 설정
  • 국경과 반드시 일치하지는 않음

3.5.2 한국의 FIR

한국은 인천 FIR을 운영하고 있으며, 다음과 같은 특징이 있습니다:

항목 내용
명칭 Incheon FIR (RKRR)
관할 기관 인천 ACC (Area Control Center)
면적 약 431,000 km²
일일 관제 편수 약 2,000편 이상
특징 동북아 허브, 이어도 포함

3.6 관제권(Control Zone)

관제권(Control Zone, CTR)은 공항 주변의 관제되는 공역으로, 보통 지표면부터 시작합니다.

3.6.1 관제권의 구조

관제권은 일반적으로 다음과 같은 구조를 가집니다:

  • 수평 범위: 공항 중심에서 반경 5~10 해리
  • 수직 범위: 지표면부터 약 2,000~3,000피트
  • 형태: 원형 또는 활주로 방향을 고려한 타원형

3.7 터미널 관제구역(TMA)

터미널 관제구역(Terminal Control Area, TMA 또는 TCA)은 하나 이상의 주요 공항을 포함하는 관제 공역입니다.

3.7.1 TMA의 특징

  • 관제권보다 넓은 범위
  • 여러 공항의 출발/도착 항공기 통합 관제
  • 복잡한 교통 흐름 관리
  • 계단식 고도 구조 (보통 "Wedding Cake" 형태)

3.8 공역 관리

효율적인 공역 관리는 안전과 용량 최적화를 위해 필수적입니다.

3.8.1 공역 설계 원칙

  • 안전: 충분한 분리 간격 보장
  • 효율성: 교통 흐름 최적화
  • 유연성: 수요 변화에 대응
  • 환경: 소음 및 배출 최소화
  • 형평성: 민간/군사 공역 조화

3.8.2 동적 공역 관리(FUA)

유연한 공역 사용(Flexible Use of Airspace, FUA)은 공역을 고정적으로 할당하는 대신, 실시간 수요에 따라 동적으로 관리하는 개념입니다:

  • 전략적 관리: 연간/계절별 계획
  • 사전 전술적 관리: 일일 조정
  • 실시간 관리: 즉각적 공역 재할당

3.9 미래의 공역 구조

항공 교통의 증가와 새로운 항공 이동성(UAM, 드론)의 등장으로 공역 구조는 계속 진화할 것입니다:

미래 공역의 특징

  • 4D 궤적 관리: 위치(3D) + 시간(1D) 정밀 관리
  • 동적 공역: 실시간 수요 기반 공역 재구성
  • UAM 통합: 저고도 도심 항공 교통 레이어 추가
  • 성능 기반 공역: 항공기 성능에 따른 차등 운영
  • AI 최적화: 머신러닝 기반 공역 용량 예측 및 할당
