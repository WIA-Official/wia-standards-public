Chapter 4

항공 통신 시스템

VHF, HF, CPDLC 및 차세대 통신 기술

이 장에서 배울 내용:

  • VHF 음성 통신 시스템
  • HF 장거리 통신
  • 데이터 링크 통신 (CPDLC, ACARS)
  • 차세대 통신 기술 및 위성 통신

4.1 항공 통신의 중요성

항공 통신은 조종사와 관제사 간의 정확하고 신속한 정보 교환을 가능하게 하여 안전한 비행을 보장합니다. 항공 통신 시스템은 크게 음성 통신과 데이터 통신으로 나뉩니다.

4.2 VHF 통신 시스템

초고주파(Very High Frequency, VHF) 통신은 항공에서 가장 널리 사용되는 음성 통신 방식입니다.

4.2.1 VHF의 특성

항목 내용
주파수 범위 118.000 ~ 136.975 MHz
채널 간격 8.33 kHz 또는 25 kHz
전파 방식 직선 (Line of Sight)
통신 거리 약 200 해리 (고도에 따라 변동)
음질 우수
간섭 기상 영향 적음

4.2.2 VHF 주파수 할당

📻 주요 VHF 주파수

  • 121.5 MHz: 긴급 주파수 (Emergency)
  • 122.75 MHz: 공대공 통신
  • 123.45 MHz: 비공식 공대공 통신
  • 126.7 MHz: 국제 조난 주파수 (SAR)
  • 131.8 MHz: 유럽 공통 긴급 주파수

4.2.3 VHF의 장단점

장점

  • 명확한 음질
  • 낮은 기상 간섭
  • 즉각적인 통신 가능
  • 간단한 운용

단점

  • 직선 전파로 인한 거리 제한
  • 산악 지형에서 음영 지역 발생
  • 주파수 혼잡 (특히 유럽)
  • 동시 통신 불가 (반이중 통신)

4.2.4 8.33 kHz 채널 간격

유럽에서는 주파수 혼잡을 해결하기 위해 2018년부터 FL195 이상에서 8.33 kHz 채널 간격을 의무화했습니다. 이는 기존 25 kHz 대비 3배 많은 채널을 제공합니다.

4.3 HF 통신 시스템

고주파(High Frequency, HF) 통신은 대양 상공과 원격 지역에서 장거리 통신을 위해 사용됩니다.

4.3.1 HF의 특성

항목 내용
주파수 범위 2.8 ~ 22 MHz
전파 방식 전리층 반사 (Sky Wave)
통신 거리 수천 해리
음질 보통 (잡음 多)
간섭 대기 조건, 태양 활동 영향

4.3.2 HF 통신의 활용

HF는 주로 다음 상황에서 사용됩니다:

  • 대서양, 태평양 횡단 비행
  • 극지방 비행
  • 아프리카, 남미 등 VHF 인프라가 부족한 지역
  • 긴급 상황 시 장거리 통신

4.3.3 SELCAL (Selective Calling)

SELCAL은 HF 통신의 불편함을 줄이기 위한 시스템입니다. 조종사가 계속 주파수를 모니터링하는 대신, 관제사가 특정 항공기를 호출하면 콕핏에서 신호음이 울립니다.

🔔 SELCAL 작동 방식

  1. 각 항공기에 고유한 4글자 코드 할당 (예: AB-CD)
  2. 관제사가 해당 코드를 송신
  3. 항공기의 SELCAL 시스템이 인식하고 경보음 발생
  4. 조종사가 HF 통신 개시

4.4 데이터 링크 통신

디지털 시대에 맞춰 데이터 링크 통신이 점차 음성 통신을 보완하거나 대체하고 있습니다.

4.4.1 ACARS (Aircraft Communications Addressing and Reporting System)

ACARS는 항공기와 지상 간의 자동 메시지 교환 시스템입니다.

특징 설명
개발 시기 1970년대
전송 방식 VHF, HF, 위성
주요 용도 운항 정보, 기상 정보, 정비 데이터
메시지 길이 최대 220자

ACARS가 전송하는 정보:

  • 이륙 및 착륙 시간
  • 엔진 성능 데이터
  • 연료 소비량
  • 기상 정보 요청 및 수신
  • 게이트 정보, 승객 수
  • 정비 상태 알림

4.4.2 CPDLC (Controller-Pilot Data Link Communications)

CPDLC는 조종사와 관제사 간의 텍스트 기반 통신 시스템으로, 음성 통신을 보완하거나 대체합니다.

CPDLC의 장점

  • 정확성: 오해의 소지가 있는 음성 통신 대비 명확함
  • 기록: 모든 통신 내용 자동 기록
  • 주파수 혼잡 감소: VHF 주파수 부담 완화
  • 다국어 문제 해소: 표준화된 메시지 사용
  • 업무 부담 감소: 조종사와 관제사의 음성 통신 부담 완화

CPDLC 메시지 유형

메시지 유형 예시 응답
고도 변경 "CLIMB TO FL350" WILCO / UNABLE / STANDBY
방향 변경 "TURN LEFT HDG 090" WILCO / UNABLE / STANDBY
속도 조정 "MAINTAIN MACH 0.82" WILCO / UNABLE / STANDBY
주파수 변경 "CONTACT 132.45" WILCO

4.4.3 ATN (Aeronautical Telecommunication Network)

ATN은 차세대 항공 통신을 위한 IP 기반 네트워크입니다. ICAO가 주도하여 개발 중이며, 다음 기능을 제공합니다:

  • 고속 데이터 전송
  • 다양한 애플리케이션 지원
  • 글로벌 상호 운용성
  • 보안 강화

4.5 위성 통신

위성 통신은 전 세계 어디서나 통신이 가능하도록 합니다.

4.5.1 Inmarsat

Inmarsat은 항공용 위성 통신 서비스를 제공하는 대표적인 시스템입니다:

  • Classic Aero: 음성 및 저속 데이터
  • SwiftBroadband: 고속 데이터 (최대 432 kbps)
  • GX Aviation: 초고속 광대역 (최대 50 Mbps)

4.5.2 Iridium

Iridium은 66개의 저궤도 위성으로 구성된 네트워크로, 극지방을 포함한 전 세계를 커버합니다. Iridium NEXT 세대 위성은 Aireon ADS-B 수신기를 탑재하여 글로벌 항공기 추적을 가능하게 합니다.

4.6 차세대 통신 기술

4.6.1 LDACS (L-band Digital Aeronautical Communications System)

LDACS는 유럽에서 개발 중인 차세대 지상-항공 통신 시스템입니다:

🚀 LDACS 특징

  • 주파수: 960-1164 MHz (L-band)
  • 데이터 속도: 최대 1.2 Mbps
  • 용도: CPDLC, ADS-B, 감시, 기상 정보
  • 목표: VHF 음성 통신 완전 대체

4.6.2 AeroMACS (Aeronautical Mobile Airport Communications System)

AeroMACS는 공항 지상에서 고속 무선 데이터 통신을 제공합니다:

  • 기술: WiMAX 기반 (IEEE 802.16)
  • 주파수: 5091-5150 MHz (C-band)
  • 적용: 활주로, 유도로, 램프
  • 용도: 항공기-지상 데이터 교환, 정비 데이터, 비디오 스트리밍

4.6.3 5G 항공 통신

5G 기술의 항공 적용이 연구되고 있습니다:

5G 항공 통신의 잠재력

  • 초고속 데이터 전송 (수 Gbps)
  • 초저지연 통신 (1ms 이하)
  • 대규모 기기 연결 (IoT)
  • 네트워크 슬라이싱으로 우선순위 보장

주의: 5G C-band와 항공 고도계 간섭 문제가 2022년 미국에서 이슈화되어, 현재 주파수 조정 및 보호 대책이 시행되고 있습니다.

4.7 통신 보안 및 신뢰성

4.7.1 통신 보안

항공 통신 보안은 사이버 공격으로부터 시스템을 보호하기 위해 필수적입니다:

  • 암호화: AES, RSA 등 강력한 암호화 적용
  • 인증: 송신자 신원 확인
  • 무결성 검증: 메시지 변조 방지
  • 침입 탐지: 비정상 통신 패턴 모니터링

4.7.2 통신 중복성

항공 안전을 위해 여러 통신 수단을 동시에 갖추는 것이 일반적입니다:

비행 단계 주 통신 백업 통신
공항 지상 VHF AeroMACS, ACARS
대륙 상공 VHF, CPDLC 위성 통신, HF
대양 상공 위성 통신, CPDLC HF
극지방 위성 통신 (Iridium) HF

4.8 미래 전망

항공 통신은 점차 디지털화, 자동화, 지능화되고 있습니다:

미래 항공 통신의 방향

  • 완전 데이터 링크: 음성 통신의 데이터 통신 전환
  • AI 기반 통신: 자동 메시지 생성 및 우선순위 결정
  • 양자 통신: 절대적 보안 보장
  • 통합 네트워크: 지상, 위성, 항공기 간 seamless 통신
  • 6G 통신: 테라헤르츠 대역 활용
