6.3 안전성 시험
--------------------------------------------------------------------------------

스마트 알약은 엄격한 생물학적 안전성 평가를 거쳐야 합니다.

[ID] 전임상 시험:

    PreclinicalTesting {
      iso10993: {
        name:        "ISO 10993 - 의료기기의 생물학적 평가"

        part5_cytotoxicity: {
          name:      "Part 5: 세포독성 시험"
