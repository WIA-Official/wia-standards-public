1.1 스마트 알약이란?
--------------------------------------------------------------------------------

스마트 알약(Smart Pill)은 섭취 가능한 소형 전자 장치로, 인체 내부에서
건강 데이터를 수집하고 무선으로 전송할 수 있는 혁신적인 의료 기술입니다.
전통적인 약물 캡슐과 첨단 센서 기술을 결합하여 실시간 건강 모니터링과
정밀 약물 전달을 가능하게 합니다.

[ID] 핵심 정의:

    SmartPill {
      fullName:    "섭취 가능 전자 센서 시스템"
      version:     "1.0"
      domain:      "의료 기기"

      primaryFunctions: {
        monitoring:      "체내 생리학적 파라미터 측정"
        tracking:        "약물 복용 확인 및 추적"
        delivery:        "제어된 약물 방출"
        communication:   "무선 데이터 전송"
        diagnosis:       "위장관 질환 진단 지원"
      }

      keyComponents: [
        "바이오센서 어레이",
        "마이크로프로세서",
        "무선 송신기",
        "전력 공급 시스템",
        "생체적합성 캡슐",
        "약물 저장소 (선택적)"
      ]

      relatedStandards: [
        "WIA-MED-001:  의료 데이터 표준",
        "WIA-MED-005:  원격 건강 모니터링",
        "WIA-HEALTH:   개인 건강 기록",
        "WIA-IOT:      사물 인터넷 연결"
      ]
    }

[ID] 스마트 알약의 주요 특징:

    +------------------------------------------------------------------+
    | 특징                | 설명                                       |
    +------------------------------------------------------------------+
    | 비침습성            | 수술이나 주사 없이 경구 섭취               |
    | 실시간 모니터링     | 연속적인 데이터 수집 및 전송               |
    | 정밀 측정           | 체내 환경의 직접 측정                      |
    | 환자 순응도 추적    | 객관적인 약물 복용 확인                    |
    | 일회용 설계         | 자연 배출 후 폐기                          |
    | 무선 연결           | 외부 장치와의 실시간 통신                  |
    +------------------------------------------------------------------+

[ID] 주요 혁신 이정표:

    1957: 최초 섭취 가능 센서 개념 제안 (Mackay & Jacobson)
    1972: 무선 내시경 캡슐 특허 출원
    1981: Heidelberg pH 캡슐 FDA 승인
    2001: PillCam 캡슐 내시경 유럽 승인
    2007: PillCam 미국 FDA 승인
    2012: Proteus 디지털 약물 센서 FDA 승인
    2017: Abilify MyCite (약물+센서) FDA 승인
    2020: AI 기반 진단 캡슐 임상 시험
    2025: 다기능 통합 스마트 알약 상용화
