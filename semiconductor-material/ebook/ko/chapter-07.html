제7장

CMP 슬러리 및 소모품

화학 기계적 연마 공정의 슬러리, 패드, 컨디셔너 및 후세정 소재

7.1 CMP 개요

화학 기계적 연마(Chemical Mechanical Polishing/Planarization, CMP)는 반도체 제조에서 표면 평탄화를 달성하는 핵심 공정입니다. CMP는 화학적 반응과 기계적 연마를 동시에 활용하여 웨이퍼 표면을 원자 수준으로 평탄하게 만듭니다.

CMP의 주요 응용 분야:

  • STI (Shallow Trench Isolation): 소자 분리 트렌치 절연막 평탄화
  • ILD (Interlayer Dielectric): 층간 유전체 평탄화
  • 구리 다마신: 구리 배선 형성 및 과잉 금속 제거
  • 텅스텐 플러그: W 비아 플러그 평탄화
  • 폴리실리콘: 폴리 게이트 및 메모리 구조

핵심 개념: Preston 방정식

CMP 제거 속도는 Preston 방정식으로 설명됩니다: RR = k × P × V, 여기서 RR은 제거 속도, k는 Preston 계수(소재/슬러리 의존), P는 연마 압력, V는 상대 속도입니다. 현대 CMP에서는 화학적 효과가 크게 기여하여 실제 거동은 더 복잡합니다.

7.2 CMP 슬러리

CMP 슬러리는 연마재(abrasive), 화학 첨가제, 그리고 액상 매체로 구성됩니다. 각 응용에 따라 최적화된 슬러리가 사용됩니다.

7.2.1 슬러리 구성 요소

구성 요소 역할 예시
연마재 기계적 물질 제거 SiO₂, CeO₂, Al₂O₃
산화제 금속 표면 산화 H₂O₂, Fe(NO₃)₃
착화제 산화물 용해 촉진 구연산, 아미노산
계면활성제 분산, 선택비 제어 비이온성 계면활성제
pH 조절제 화학 반응 제어 KOH, HNO₃
부식 억제제 정적 에칭 방지 BTA, 아졸 화합물

7.2.2 산화물 CMP 슬러리

SiO₂ 산화물 평탄화용 슬러리:

  • 연마재: 콜로이드 실리카(20-100nm) 또는 세리아(CeO₂)
  • pH: 알칼리성 (pH 10-11) 또는 산성 (pH 4-6, 세리아용)
  • 메커니즘: Si-O-Si 결합의 화학적 약화 + 기계적 제거
  • 제거 속도: 100-500 nm/min

7.2.3 구리 CMP 슬러리

구리 다마신 공정용 다단계 슬러리:

단계 목적 슬러리 특성 Cu:배리어 선택비
1단계 벌크 Cu 제거 높은 RR, 높은 산화제 > 50:1
2단계 배리어 제거 Cu/Ta 동시 제거 ~1:1
3단계 (버핑) 결함 제거, 마무리 저압, 저RR 높은 선택비
구리 CMP 화학 반응 메커니즘 ================================ 1단계: 산화 Cu + H₂O₂ → CuO + H₂O Cu + 2H₂O₂ → Cu(OH)₂ + H₂O 2단계: 착화 CuO + 2H⁺ + 착화제 → Cu-착화물 + H₂O 3단계: 기계적 제거 연마재에 의한 반응층 물리적 제거 표면: ┌────────────────────────────┐ │ Cu-착화물층 (연질, 쉽게 제거) │ ├────────────────────────────┤ │ CuO/Cu₂O 층 │ ├────────────────────────────┤ │ 벌크 Cu │ └────────────────────────────┘
그림 7.2: 구리 CMP의 화학적 메커니즘

7.2.4 텅스텐 CMP 슬러리

  • 산화제: Fe(NO₃)₃ 또는 H₂O₂ + Fe³⁺ 촉매
  • 연마재: 알루미나(Al₂O₃) 또는 실리카
  • pH: 산성 (pH 2-4)
  • W:SiO₂ 선택비: 1:1 ~ 3:1

세리아 슬러리의 장점

세리아(CeO₂) 슬러리는 실리카 대비 낮은 연마재 농도(0.5-2%)에서도 높은 산화물 제거 속도를 달성합니다. Ce³⁺/Ce⁴⁺ 산화환원 반응이 화학적 반응을 촉진하여 STI와 ILD CMP에 널리 사용됩니다.

7.3 연마 패드

연마 패드는 슬러리를 운반하고 웨이퍼에 기계적 작용을 전달하는 핵심 소모품입니다.

7.3.1 패드 구조

  • 상부 패드 (Top Pad): 연마 작용 수행, 폴리우레탄 소재
  • 서브 패드: 압력 분포 균일화, 불균일성 흡수
  • 그루브: 슬러리 운반 및 분포, 잔류물 배출
패드 특성 영향 일반 범위
경도 평탄화 성능, 스크래치 50-70 Shore D
두께 수명, 균일성 1.2-2.0 mm
기공 크기 슬러리 보유, RR 20-50 μm
기공 밀도 슬러리 분포 20-40%
압축률 균일성, 결함 2-8%

7.3.2 패드 유형

  • IC1000: 가장 일반적인 경질 폴리우레탄 패드, 높은 평탄화
  • SUBAIV: 연질 서브패드, 균일성 향상
  • 다층 패드: 경질 상부 + 연질 하부 조합
  • 고정 연마재 패드: 연마재가 패드에 내장, 슬러리리스

7.3.3 패드 그루브 패턴

패드 그루브 패턴 유형 ================================ 동심원형 방사형 XY 그리드 ╭───────────╮ ╭───────────╮ ┌─┬─┬─┬─┬─┐ │ ○○○○○○ │ │ \ | / │ ├─┼─┼─┼─┼─┤ │ ○○○○○○ │ │ ──●── │ ├─┼─┼─┼─┼─┤ │ ○○○○○○ │ │ / | \ │ ├─┼─┼─┼─┼─┤ │ ○○○○○○ │ │ │ └─┴─┴─┴─┴─┘ ╰───────────╯ ╰───────────╯ 그루브 기능: • 슬러리 운반 및 분포 • 연마 잔류물 배출 • 열 방출 • 웨이퍼-패드 접촉 최적화 그루브 치수: 폭 0.3-0.5mm, 깊이 0.3-0.5mm, 피치 2-5mm

7.4 패드 컨디셔너

패드 컨디셔너(다이아몬드 디스크)는 패드 표면을 재생하여 일정한 연마 성능을 유지합니다.

7.4.1 컨디셔너의 역할

  • 글레이징 제거: 폴리머 스미어링 및 연마재 막힘 제거
  • 표면 재생: 새로운 미세 요철(asperity) 생성
  • RR 안정화: 패드 수명 동안 일정한 제거 속도 유지

7.4.2 다이아몬드 디스크 특성

파라미터 범위 영향
다이아몬드 크기 100-200 μm 패드 절삭 공격성
다이아몬드 밀도 5-50 개/mm² 절삭 효율
디스크 직경 100-150 mm 커버리지
컨디셔닝 압력 2-7 lb 패드 절삭률
회전 속도 30-100 rpm 균일성

7.4.3 컨디셔닝 방식

  • In-situ: 연마 중 동시 컨디셔닝, 일정한 패드 상태 유지
  • Ex-situ: 연마 사이클 사이에 컨디셔닝
  • Combined: 두 방식 병용

컨디셔너 관리

컨디셔너의 다이아몬드 탈락은 심각한 스크래치 결함을 유발할 수 있습니다. 정기적인 검사와 교체 주기 관리가 필수적이며, 일반적으로 500-1000 패드 시간 사용 후 교체합니다.

7.5 후세정(Post-CMP Clean)

CMP 후 웨이퍼 표면에는 슬러리 잔류물, 연마재 입자, 금속 오염물이 남아 있어 철저한 세정이 필요합니다.

7.5.1 오염물 유형

  • 입자: 연마재(SiO₂, CeO₂), 패드 잔류물
  • 금속: Cu, Fe, Al 이온 및 입자
  • 유기물: 슬러리 첨가제, 부식 억제제
  • 산화물: 표면 산화층

7.5.2 세정 화학물질

세정액 조성 제거 대상
알칼리 세정 NH₄OH 또는 TMAH 기반 입자 (제타 전위 이용)
산성 세정 구연산, HF 기반 금속 이온
킬레이트제 EDTA, DTPA 금속 착화/제거
계면활성제 비이온성/양이온성 유기물, 입자 분산

7.5.3 물리적 세정 방법

  • 브러시 세정: PVA 브러시로 기계적 입자 제거
  • 메가소닉: 초고주파 음파로 미세 입자 탈착
  • 스프레이: 고압 분사로 오염물 제거
  • 버핑: 소프트 패드로 표면 마무리
후세정 공정 흐름 (예시) ================================ CMP 후 웨이퍼 ↓ ┌─────────────────────┐ │ 1. 알칼리 브러시 세정 │ ← NH₄OH + 계면활성제 │ (입자 제거) │ PVA 브러시 └─────────────────────┘ ↓ ┌─────────────────────┐ │ 2. 산성 린스 │ ← HF 희석액 또는 구연산 │ (금속 제거) │ └─────────────────────┘ ↓ ┌─────────────────────┐ │ 3. 메가소닉 세정 │ ← UPW + 메가소닉 │ (미세 입자 제거) │ └─────────────────────┘ ↓ ┌─────────────────────┐ │ 4. 건조 │ ← IPA 마란고니 건조 │ │ 또는 스핀 건조 └─────────────────────┘ ↓ 클린 웨이퍼

7.6 CMP 품질 관리

CMP 공정의 품질 관리는 제거 속도, 균일성, 결함 수준의 지속적인 모니터링을 요구합니다.

7.6.1 주요 측정 항목

항목 측정 방법 목표
제거 속도 (RR) 두께 측정 프로세스 안정성
WIWNU 다점 두께 측정 < 3% (1σ)
WTWNU 웨이퍼간 비교 < 2%
디싱 프로파일 측정 < 20nm
에로전 프로파일 측정 < 20nm
스크래치 광학 검사 0
입자 표면 검사 < 0.01/cm²

7.6.2 디싱과 에로전

디싱(Dishing)과 에로전(Erosion) ================================ 이상적 표면 실제 표면 (결함) ─────────────── ───╲___╱─── 디싱 │ Cu │ ILD │ │Cu│ │ (넓은 Cu 라인) ─────────────── ─────────── ─╲ ╱───── 에로전 │Cu│Cu│Cu│ILD│ │Cu│Cu│ (밀집 Cu 라인) ─────────────── ─────────── 원인: 디싱: Cu의 높은 연마 속도, 넓은 패턴 에로전: 산화물 동시 연마, 밀집 패턴 대책: • 선택비 최적화 • 다단계 연마 • 더미 패턴 삽입

CMP 종점 검출

정밀한 종점 검출은 과연마나 불완전 연마를 방지합니다. 광학 종점 검출(반사율 변화), 모터 전류 모니터링, 광학 간섭계 등이 사용됩니다. 첨단 노드에서는 실시간 두께 모니터링이 필수적입니다.

7.7 첨단 CMP 기술

미세 피치와 새로운 소재를 위한 첨단 CMP 기술이 개발되고 있습니다.

7.7.1 코발트/루테늄 CMP

  • Co CMP: 산성 슬러리, H₂O₂ 산화제, BTA 억제제
  • Ru CMP: 산화제 선택 중요, 주기적 산화물 형성
  • 과제: 부식 제어, 갈바닉 효과

7.7.2 GAA/나노시트용 CMP

  • SiGe 선택적 제거: Si:SiGe 높은 선택비
  • 채널 릴리스: 초저압, 초저결함
  • 3D 구조: 단차 균일성

7.7.3 하이브리드 본딩용 CMP

  • 초평탄화: 표면 거칠기 < 0.3nm RMS
  • Cu/유전체 공면: 디싱 최소화
  • 클린 표면: 입자 0, 금속 오염 최소

미래 CMP 방향

1nm 이하 노드를 위해 원자층 평탄화(Atomic Layer Planarization), 무연마재 CMP, 스마트 슬러리(pH/온도 반응성) 등이 연구되고 있습니다. 또한 AI/ML 기반 공정 제어로 실시간 최적화가 가능해지고 있습니다.
