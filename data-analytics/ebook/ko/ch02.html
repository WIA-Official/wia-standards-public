제2장: 기술 통계 및 탐색적 데이터 분석

2.1 기술적 분석 소개

기술적 분석은 모든 데이터 분석의 기초입니다. "무엇이 일어났는가?"라는 근본적인 질문에 답하여 과거 데이터를 의미 있는 패턴으로 요약합니다. 이 장에서는 데이터를 설명, 요약 및 시각화하여 초기 인사이트를 얻는 데 사용되는 기법과 도구를 살펴봅니다.

2.2 요약 통계

요약 통계는 데이터 특성에 대한 빠른 개요를 제공합니다. 이는 데이터의 분포, 중심 경향 및 산포도를 이해하는 첫 번째 단계입니다.

2.2.1 중심 경향 측정

측정정의사용 시기공식
평균 (Mean)모든 데이터 포인트의 평균값이상치 없는 대칭 분포μ = Σx / n
중앙값 (Median)정렬된 데이터의 중간값왜곡된 분포 또는 이상치 존재정렬된 데이터의 중간값
최빈값 (Mode)가장 자주 발생하는 값범주형 데이터 또는 이산 값가장 높은 빈도의 값
가중평균값에 대한 다른 가중치를 가진 평균일부 값이 더 중요할 때Σ(w·x) / Σw

2.2.2 산포도 측정

산포도 측정은 데이터가 얼마나 퍼져 있는지 알려줍니다:

측정설명해석
범위 (Range)최댓값과 최솟값의 차이간단한 산포도 측정, 이상치에 민감
분산 (Variance)평균으로부터의 평균 제곱 편차평균으로부터의 평균 거리 측정 (제곱 단위)
표준편차 (SD)분산의 제곱근평균으로부터의 평균 거리 측정 (원래 단위)
사분위수 범위 (IQR)25번째와 75번째 백분위수 사이의 범위이상치의 영향을 받지 않는 강건한 산포도 측정
변동계수표준편차 / 평균 × 100%상대적 변동 측정, 데이터셋 비교에 유용

2.2.3 분포 형태 측정

왜도 (Skewness): 분포의 비대칭성 측정
  • 음의 왜도 (좌측 왜곡): 꼬리가 왼쪽으로 확장, 평균 < 중앙값
  • 왜도 0 (대칭): 균형 잡힌 분포, 평균 ≈ 중앙값
  • 양의 왜도 (우측 왜곡): 꼬리가 오른쪽으로 확장, 평균 > 중앙값
첨도 (Kurtosis): 분포의 "꼬리 무게" 측정
  • 첨형 (kurtosis > 3): 무거운 꼬리, 더 많은 이상치
  • 중간형 (kurtosis ≈ 3): 정규 분포
  • 평평형 (kurtosis < 3): 가벼운 꼬리, 더 적은 이상치

2.3 탐색적 데이터 분석 (EDA)

EDA는 시각적 방법을 자주 사용하여 데이터셋의 주요 특성을 요약하는 데이터 분석 접근 방식입니다. 공식적인 모델링을 적용하기 전에 데이터를 이해하는 데 도움이 되는 반복적인 프로세스입니다.

2.3.1 EDA 프로세스

단계활동도구 및 기법
1. 데이터 개요• 데이터 구조 이해
• 데이터 유형 식별
• 차원 확인		head(), info(), describe(), shape
2. 데이터 품질 확인• 결측값 찾기
• 중복 감지
• 불일치 식별		isnull(), duplicated(), value_counts()
3. 단변량 분석• 각 변수를 개별적으로 분석
• 분포 확인
• 이상치 식별		히스토그램, 박스플롯, 밀도플롯
4. 이변량 분석• 변수 쌍 간의 관계 탐색
• 상관관계 계산
• 그룹 비교		산점도, 상관관계 행렬, 그룹 비교
5. 다변량 분석• 여러 변수를 함께 검사

• 차원 축소		페어플롯, 히트맵, PCA, t-SNE

2.3.2 결측 데이터 처리

결측 데이터는 실제 데이터셋에서 흔한 문제입니다. 결측 데이터의 패턴과 양에 따라 다른 전략이 적용됩니다:

전략사용 시기장점단점
삭제< 5% 결측, 무작위 패턴간단, 데이터 무결성 유지샘플 크기 감소, 편향 도입 가능
평균/중앙값 대체수치형 데이터, < 20% 결측간단, 샘플 크기 유지분산 감소, 관계 왜곡 가능
최빈값 대체범주형 데이터간단, 샘플 크기 유지가장 흔한 범주를 과대 표현 가능
앞/뒤 채우기시계열 데이터시간적 패턴 유지오류 전파 가능
보간시계열 또는 순서 데이터부드러운 추정연속적 관계 가정
모델 기반복잡한 패턴, > 20% 결측가장 정확, 모든 가용 정보 사용계산 비용 높음, 복잡함

2.3.3 이상치 탐지

이상치는 다른 관측값과 크게 다른 데이터 포인트입니다. 오류이거나 진짜 극단값일 수 있습니다:

일반적인 이상치 탐지 방법:
  1. Z-점수 방법: |z-점수| > 3인 점은 잠재적 이상치
  2. IQR 방법: Q1 - 1.5×IQR 미만 또는 Q3 + 1.5×IQR 초과 점
  3. 수정된 Z-점수: 중앙값 절대 편차(MAD) 사용, 더 강건
  4. Isolation Forest: 다변량 이상치에 대한 머신러닝 접근법
  5. DBSCAN: 밀도 기반 클러스터링이 고립된 점 식별
# Python 예제: IQR을 사용한 이상치 탐지
import pandas as pd
import numpy as np

def detect_outliers_iqr(data, column):
    Q1 = data[column].quantile(0.25)
    Q3 = data[column].quantile(0.75)
    IQR = Q3 - Q1
    
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR
    
    outliers = data[(data[column] < lower_bound) | (data[column] > upper_bound)]
    
    return outliers, lower_bound, upper_bound

# 사용
outliers, lb, ub = detect_outliers_iqr(df, 'revenue')
print(f"{len(outliers)}개의 이상치 발견")
print(f"유효 범위: [{lb:.2f}, {ub:.2f}]")

2.4 데이터 시각화

시각화는 데이터 패턴을 이해하고 인사이트를 효과적으로 전달하는 데 중요합니다.

2.4.1 올바른 시각화 선택

목적차트 유형최적 용도
분포히스토그램, 밀도 플롯, 박스 플롯, 바이올린 플롯데이터 분포 이해, 이상치 찾기
비교막대 차트, 열 차트, 그룹 막대, 레이더 차트카테고리 또는 그룹 간 값 비교
관계산점도, 버블 차트, 히트맵, 페어 플롯변수 간 상관관계 탐색
구성파이 차트, 누적 막대, 트리맵, 폭포수전체의 일부 표시
추세선 차트, 영역 차트, 스트림 그래프시간 경과에 따른 변화 표시
순위정렬된 막대 차트, 롤리팝 차트상대적 위치 또는 순위 표시

2.4.2 시각화 모범 사례

디자인 원칙:
  • 명확성: 메시지를 즉시 명확하게 만들기
  • 정확성: 왜곡 없이 데이터를 정직하게 표현
  • 단순성: 불필요한 요소 제거 (차트 쓰레기)
  • 일관성: 일관된 색상, 척도 및 형식 사용
  • 접근성: 색맹 친화적인 팔레트 고려
  • 맥락: 제목, 레이블 및 주석 제공

2.5 상관관계 분석

상관관계는 변수 간 관계의 강도와 방향을 측정합니다.

2.5.1 상관관계 유형

방법관계 유형범위사용 시기
피어슨선형 관계-1 ~ +1연속 변수, 정규 분포
스피어만단조 관계-1 ~ +1서수 데이터 또는 비정규 분포
켄달 타우단조 관계-1 ~ +1작은 샘플 크기, 많은 동점 순위
점-이분법선형 관계-1 ~ +1하나의 연속, 하나의 이진 변수

2.5.2 상관계수 해석

계수 값강도해석
±0.00 ~ ±0.19매우 약함무시할 만한 관계
±0.20 ~ ±0.39약함낮은 연관성
±0.40 ~ ±0.59보통중간 연관성
±0.60 ~ ±0.79강함강한 연관성
±0.80 ~ ±1.00매우 강함매우 강한 연관성
중요: 상관관계는 인과관계를 의미하지 않습니다! 변수 A와 B 사이의 강한 상관관계는 A가 B를 유발한다는 것을 의미하지 않습니다. 다음과 같을 수 있습니다:
  • 역 인과관계 (B가 A를 유발)
  • 혼란 변수 (C가 A와 B를 모두 유발)
  • 가짜 상관관계 (우연)

2.6 그룹 비교

그룹 비교는 세그먼트 간의 차이와 패턴을 이해하는 데 필수적입니다.

2.6.1 그룹 비교를 위한 통계 검정

검정목적가정
t-검정두 그룹의 평균 비교정규 분포, 등분산
ANOVA세 개 이상 그룹의 평균 비교정규 분포, 등분산, 독립성
Mann-Whitney U두 그룹 비교 (비모수)서수 또는 연속 데이터, 분포 가정 없음
Kruskal-Wallis세 개 이상 그룹 비교 (비모수)서수 또는 연속 데이터, 독립 그룹
카이제곱범주형 변수 비교범주형 데이터, 기대 빈도 ≥ 5

2.7 실용적 예제: 판매 데이터 분석

# Python을 사용한 완전한 EDA 예제
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 데이터 로드
df = pd.read_csv('sales_data.csv')

# 1. 데이터 개요
print("데이터셋 형태:", df.shape)
print("\n데이터 유형:")
print(df.dtypes)
print("\n처음 몇 행:")
print(df.head())

# 2. 요약 통계
print("\n요약 통계:")
print(df.describe())

# 3. 결측 데이터 분석
missing = df.isnull().sum()
missing_pct = (missing / len(df)) * 100
print("\n결측값:")
print(pd.DataFrame({'개수': missing, '비율': missing_pct}))

# 4. 분포 분석
fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 히스토그램
axes[0, 0].hist(df['revenue'], bins=50, edgecolor='black')
axes[0, 0].set_title('수익 분포')
axes[0, 0].set_xlabel('수익')
axes[0, 0].set_ylabel('빈도')

# 박스 플롯
axes[0, 1].boxplot(df['revenue'])
axes[0, 1].set_title('수익 박스 플롯')
axes[0, 1].set_ylabel('수익')

# Q-Q 플롯 (정규성 확인)
from scipy import stats
stats.probplot(df['revenue'], dist="norm", plot=axes[1, 0])
axes[1, 0].set_title('Q-Q 플롯')

# 밀도 플롯
df['revenue'].plot(kind='density', ax=axes[1, 1])
axes[1, 1].set_title('수익 밀도 플롯')

plt.tight_layout()
plt.show()

# 5. 상관관계 분석
correlation_matrix = df.select_dtypes(include=[np.number]).corr()
plt.figure(figsize=(10, 8))
sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm', center=0)
plt.title('상관관계 히트맵')
plt.show()

# 6. 그룹 비교
by_category = df.groupby('category')['revenue'].agg([
    'count', 'mean', 'median', 'std', 'min', 'max'
])
print("\n카테고리별 수익:")
print(by_category)

# 7. 시계열 분석 (해당되는 경우)
df['date'] = pd.to_datetime(df['date'])
daily_revenue = df.groupby('date')['revenue'].sum()
daily_revenue.plot(figsize=(12, 6))
plt.title('일별 수익 추세')
plt.xlabel('날짜')
plt.ylabel('수익')
plt.show()
