Chapter 4: Prescriptive Analytics and Optimization

4.1 Introduction to Prescriptive Analytics

Prescriptive analytics goes beyond predicting what will happen to recommend what should be done. It combines predictive models with business rules and optimization algorithms to suggest the best course of action. While descriptive analytics tells us what happened and predictive analytics forecasts what will happen, prescriptive analytics advises on what actions to take to achieve desired outcomes.

4.2 Optimization Fundamentals

Optimization is the process of finding the best solution from all feasible solutions. It involves:

  • Objective Function: The goal to maximize or minimize (e.g., profit, cost, time)
  • Decision Variables: Variables we can control
  • Constraints: Limitations or requirements that must be satisfied
  • Feasible Region: Set of all solutions that satisfy constraints

4.2.1 Types of Optimization Problems

4.3 Linear Programming

Linear Programming (LP) is one of the most widely used optimization techniques. It finds the optimal solution when both the objective function and constraints are linear.

Standard Form:
Maximize: c₁x₁ + c₂x₂ + ... + cₙxₙ
Subject to:
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
...
x₁, x₂, ..., xₙ ≥ 0 
# Python Linear Programming Example
from scipy.optimize import linprog

# Maximize: 3x + 4y
# Subject to: x + 2y ≤ 14, 3x + y ≤ 18, x,y ≥ 0

# Convert to minimization: -3x - 4y
c = [-3, -4]

# Inequality constraints (Ax ≤ b)
A_ub = [[1, 2], [3, 1]]
b_ub = [14, 18]

# Bounds
x_bounds = (0, None)
y_bounds = (0, None)
bounds = [x_bounds, y_bounds]

# Solve
result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')

print(f"Optimal solution: x={result.x[0]:.2f}, y={result.x[1]:.2f}")
print(f"Maximum value: {-result.fun:.2f}")

4.5 Simulation and Scenario Analysis

Simulation models the behavior of complex systems by running multiple scenarios to understand possible outcomes.

4.5.1 Monte Carlo Simulation

Uses random sampling to model uncertainty and variability in systems.

# Monte Carlo Simulation Example
import numpy as np
import matplotlib.pyplot as plt

# Simulate revenue forecast with uncertainty
n_simulations = 10000
base_revenue = 1000000
growth_rate_mean = 0.15
growth_rate_std = 0.05

simulated_revenues = []
for _ in range(n_simulations):
    growth_rate = np.random.normal(growth_rate_mean, growth_rate_std)
    future_revenue = base_revenue * (1 + growth_rate)
    simulated_revenues.append(future_revenue)

# Analyze results
mean_revenue = np.mean(simulated_revenues)
median_revenue = np.median(simulated_revenues)
percentile_5 = np.percentile(simulated_revenues, 5)
percentile_95 = np.percentile(simulated_revenues, 95)

print(f"Expected Revenue: ${mean_revenue:,.0f}")
print(f"90% Confidence Interval: [${percentile_5:,.0f}, ${percentile_95:,.0f}]")
print(f"Probability of exceeding $1.2M: {np.mean(np.array(simulated_revenues) > 1200000)*100:.1f}%")

4.7 Reinforcement Learning for Prescriptive Analytics

Reinforcement learning (RL) learns optimal actions through trial and error, receiving rewards or penalties for actions taken.

Key Concepts:
  • Agent: The decision-maker
  • Environment: The system being interacted with
  • State: Current situation
  • Action: Decision made by agent
  • Reward: Feedback from environment
  • Policy: Strategy for selecting actions

4.7.1 RL Applications in Business

  • Dynamic Pricing: Adjust prices in real-time to maximize revenue
  • Inventory Management: Optimize reorder points and quantities
  • Marketing Optimization: Determine best timing and channel for campaigns
  • Resource Allocation: Dynamically allocate resources to maximize efficiency

4.9.2 Price Optimization

# Price Optimization Example
import numpy as np
from scipy.optimize import minimize

def profit_function(price, demand_params, cost):
    # Demand curve: Q = a - b*P
    a, b = demand_params
    quantity = a - b * price[0]
    revenue = price[0] * quantity
    total_cost = cost * quantity
    profit = revenue - total_cost
    return -profit  # Minimize negative profit = maximize profit

# Parameters
demand_params = (10000, 50)  # Q = 10000 - 50*P
unit_cost = 20

# Optimize
result = minimize(
    profit_function,
    x0=[50],  # Initial guess
    args=(demand_params, unit_cost),
    method='SLSQP',
    bounds=[(unit_cost, 200)]  # Price must be > cost
)

optimal_price = result.x[0]
optimal_quantity = demand_params[0] - demand_params[1] * optimal_price
max_profit = -result.fun

print(f"Optimal Price: ${optimal_price:.2f}")
print(f"Expected Quantity: {optimal_quantity:.0f}")
print(f"Expected Profit: ${max_profit:,.2f}")
