Chapter 5: Algorithms and Matching

"The right AI for the right task - intelligent matching is the key to effective collaboration."

5.1 Core Algorithms Overview

The WIA LLM Interoperability Standard defines several algorithms:

  • Capability Matching - Find the best AI for a task
  • Message Routing - Deliver messages efficiently
  • Task Decomposition - Break complex tasks into subtasks
  • Consensus Building - Reach agreement among AIs
  • Trust Scoring - Evaluate AI reliability
  • Load Balancing - Distribute work evenly

5.2 Capability Matching Algorithm

Finds the best AI model for a given task based on requirements.

Matching Weights

FactorWeightDescription
Domain Match40%Required expertise areas
Level Match25%Minimum capability level
Language Match20%Required language support
Tool Match15%Required tool capabilities

Algorithm: CalculateCapabilityMatch

function CalculateCapabilityMatch(requirements, capability):

    # Domain Score (40% weight)
    domain_score = 0
    for required_domain in requirements.domains:
        for cap_domain in capability.domains:
            if cap_domain.code == required_domain:
                domain_score += cap_domain.proficiency
    domain_score = domain_score / len(requirements.domains)

    # Level Score (25% weight)
    if capability.level >= requirements.min_level:
        level_score = capability.level / 4.0
    else:
        level_score = 0  # Does not meet minimum

    # Language Score (20% weight)
    language_score = 0
    for lang in capability.languages:
        if lang.code == requirements.language:
            language_score = lang.proficiency
            break

    # Tool Score (15% weight)
    tool_score = 0
    if requirements.tools:
        matches = 0
        for tool in requirements.tools:
            if tool in capability.tools:
                matches += 1
        tool_score = matches / len(requirements.tools)
    else:
        tool_score = 1.0  # No tools required

    # Calculate weighted total
    total_score = (
        domain_score * 0.40 +
        level_score * 0.25 +
        language_score * 0.20 +
        tool_score * 0.15
    )

    return total_score

5.3 Capability Handshake Protocol

Before collaboration, AI models verify mutual capabilities:

Step 1: Initiator sends capability query
AI-A ---> AI-B: CAPABILITY_REQUEST

Step 2: Responder shares capability document
AI-B ---> AI-A: CAPABILITY_RESPONSE

Step 3: Initiator evaluates compatibility
AI-A: CalculateCapabilityMatch()

Step 4: If compatible, collaboration proceeds
AI-A ---> AI-B: HANDSHAKE_ACCEPT

Step 5: Session established
AI-A <---> AI-B: Ready for messages

5.4 Message Routing Algorithms

Direct Routing

Send directly to a known recipient.

function DirectRoute(message, recipient_did):
    endpoint = ResolveEndpoint(recipient_did)
    return SendToEndpoint(message, endpoint)

Capability-Based Routing

Find the best recipient based on message requirements.

function CapabilityRoute(message, requirements):
    # Query capability registry
    candidates = QueryCapabilities(requirements)

    # Sort by match score
    sorted_candidates = SortByMatchScore(candidates)

    # Try each candidate until success
    for candidate in sorted_candidates:
        result = DirectRoute(message, candidate.did)
        if result.success:
            return result

    return Error("No capable recipient found")

Load-Balanced Routing

Distribute messages across available models.

function LoadBalancedRoute(message, requirements):
    candidates = QueryCapabilities(requirements)

    # Filter by minimum match score
    qualified = [c for c in candidates if c.score >= 0.7]

    # Weight by availability and load
    for candidate in qualified:
        candidate.weight = (
            candidate.score * 0.5 +
            candidate.availability * 0.3 +
            (1 - candidate.load) * 0.2
        )

    # Select using weighted random
    selected = WeightedRandomSelect(qualified)
    return DirectRoute(message, selected.did)

Broadcast Routing

Send to all federation members.

function BroadcastRoute(message, federation):
    results = []
    for member in federation.members:
        result = DirectRoute(message, member.did)
        results.append(result)
    return AggregateResults(results)

5.5 Task Decomposition Algorithm

Break complex tasks into manageable subtasks:

function DecomposeTask(task, federation):
    # Analyze task complexity
    complexity = AnalyzeComplexity(task)

    if complexity == "LOW":
        return [task]  # No decomposition needed

    # Identify required capabilities
    required_caps = ExtractRequirements(task)

    # Generate subtasks based on capabilities
    subtasks = []

    for capability in required_caps:
        # Find best member for this capability
        best_member = FindBestMatch(federation.members, capability)

        subtask = {
            "id": GenerateId(),
            "description": GenerateSubtaskDescription(task, capability),
            "assignee": best_member.did,
            "capability": capability,
            "dependencies": []
        }
        subtasks.append(subtask)

    # Establish dependencies
    subtasks = EstablishDependencies(subtasks, task.decomposition_strategy)

    return subtasks

function EstablishDependencies(subtasks, strategy):
    if strategy == "PARALLEL":
        # No dependencies
        return subtasks

    elif strategy == "SEQUENTIAL":
        # Each depends on previous
        for i in range(1, len(subtasks)):
            subtasks[i].dependencies = [subtasks[i-1].id]
        return subtasks

    elif strategy == "HIERARCHICAL":
        # Build dependency tree
        return BuildDependencyTree(subtasks)

    elif strategy == "ADAPTIVE":
        # Dependencies determined at runtime
        return subtasks

5.6 Consensus Algorithms

Majority Consensus

function MajorityConsensus(votes, threshold=0.5):
    yes_count = count(v for v in votes if v.vote == YES)
    no_count = count(v for v in votes if v.vote == NO)
    total = yes_count + no_count

    if total == 0:
        return UNDECIDED

    yes_ratio = yes_count / total

    if yes_ratio > threshold:
        return YES
    elif yes_ratio < (1 - threshold):
        return NO
    else:
        return UNDECIDED

Weighted Consensus

function WeightedConsensus(votes, threshold=0.6):
    yes_weight = 0
    no_weight = 0
    total_weight = 0

    for vote in votes:
        if vote.vote == ABSTAIN:
            continue

        weight = vote.confidence * vote.trust_score
        total_weight += weight

        if vote.vote == YES:
            yes_weight += weight
        else:
            no_weight += weight

    if total_weight == 0:
        return UNDECIDED

    yes_ratio = yes_weight / total_weight

    if yes_ratio >= threshold:
        return YES
    elif (1 - yes_ratio) >= threshold:
        return NO
    else:
        return UNDECIDED

Debate Consensus

function DebateConsensus(question, members, max_rounds=5):
    positions = {}

    # Round 1: Initial positions
    for member in members:
        positions[member.did] = GetInitialPosition(member, question)

    for round in range(2, max_rounds + 1):
        # Share positions and counter-arguments
        for member in members:
            other_positions = {k:v for k,v in positions.items()
                            if k != member.did}

            # Member considers other positions
            refined = RefinePosition(member, positions[member.did],
                                    other_positions)
            positions[member.did] = refined

        # Check for convergence
        convergence = CalculateConvergence(positions)
        if convergence >= 0.8:
            return DetermineConsensus(positions)

    # Max rounds reached
    return DetermineConsensus(positions)

5.7 Trust Score Algorithm

function CalculateTrustScore(model):
    # Component weights
    ACCURACY_WEIGHT = 0.40
    RESPONSE_TIME_WEIGHT = 0.15
    CONSISTENCY_WEIGHT = 0.25
    PEER_REVIEW_WEIGHT = 0.20

    # Normalize response time (lower is better)
    max_acceptable_time = 30000  # 30 seconds
    time_score = max(0, 1 - (model.avg_response_time / max_acceptable_time))

    # Calculate weighted trust score
    trust_score = (
        model.historical_accuracy * ACCURACY_WEIGHT +
        time_score * RESPONSE_TIME_WEIGHT +
        model.consistency_score * CONSISTENCY_WEIGHT +
        model.peer_review_avg * PEER_REVIEW_WEIGHT
    )

    return trust_score

function UpdateTrustScore(model, interaction_result):
    # Decay factor for old data
    DECAY = 0.95

    # Weight of new interaction
    NEW_WEIGHT = 0.1

    new_score = (
        model.trust_score * DECAY * (1 - NEW_WEIGHT) +
        interaction_result.score * NEW_WEIGHT
    )

    model.trust_score = clamp(new_score, 0, 1)
    return model.trust_score

5.8 Malicious AI Detection

function DetectMaliciousAI(model, recent_interactions):
    THRESHOLD_ACCURACY = 0.3
    THRESHOLD_CONSISTENCY = 0.2
    THRESHOLD_CONFLICT = 0.5

    # Check for consistently wrong answers
    recent_accuracy = CalculateRecentAccuracy(recent_interactions)
    if recent_accuracy < THRESHOLD_ACCURACY:
        return SUSPICIOUS, "Low accuracy"

    # Check for inconsistent responses
    consistency = CalculateConsistency(recent_interactions)
    if consistency < THRESHOLD_CONSISTENCY:
        return SUSPICIOUS, "Inconsistent responses"

    # Check for conflict-inducing behavior
    conflict_rate = CalculateConflictRate(recent_interactions)
    if conflict_rate > THRESHOLD_CONFLICT:
        return SUSPICIOUS, "High conflict rate"

    return TRUSTED, "No issues detected"

5.9 Circuit Breaker Pattern

class CircuitBreaker:
    CLOSED = "CLOSED"       # Normal operation
    OPEN = "OPEN"           # Blocking requests
    HALF_OPEN = "HALF_OPEN" # Testing recovery

    def __init__(self, threshold=5, timeout=30000):
        self.state = CLOSED
        self.failure_count = 0
        self.threshold = threshold
        self.timeout = timeout
        self.last_failure_time = None

    def execute(self, operation):
        if self.state == OPEN:
            if time_since(self.last_failure_time) > self.timeout:
                self.state = HALF_OPEN
            else:
                raise CircuitOpenError()

        try:
            result = operation()
            self.on_success()
            return result
        except Exception as e:
            self.on_failure()
            raise e

    def on_success(self):
        self.failure_count = 0
        self.state = CLOSED

    def on_failure(self):
        self.failure_count += 1
        self.last_failure_time = now()

        if self.failure_count >= self.threshold:
            self.state = OPEN

5.10 Load Balancing Algorithm

function BalanceLoad(task, candidates):
    # Calculate effective capacity for each candidate
    for candidate in candidates:
        candidate.effective_capacity = (
            candidate.capacity * (1 - candidate.current_load) *
            candidate.trust_score
        )

    # Sort by effective capacity (highest first)
    sorted_candidates = sort(candidates,
                            key=lambda c: c.effective_capacity,
                            reverse=True)

    # Select candidate with most capacity
    selected = sorted_candidates[0]

    # Update load tracking
    selected.current_load += task.estimated_load

    return selected
