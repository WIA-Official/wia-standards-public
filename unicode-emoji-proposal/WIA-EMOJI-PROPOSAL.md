# WIA Accessibility Emoji Proposal to Unicode Consortium

**Submission Date**: 2025-12-17
**Proposer**: World Certification Industry Association (WIA)
**Contact**: SmileStory Inc. / WIA
**Philosophy**: нҷҚмқөмқёк°„ (ејҳзӣҠдәәй–“) - Benefit All Humanity

---

## Executive Summary

WIA proposes **25 new accessibility-focused emojis** to Unicode Consortium, designed by accessibility experts with technical standards backing (ISP, WIA-Braille, WIA-Talk).

This is the **first proposal from an accessibility certification organization** (not an IT company), providing authentic representation of:
- Assistive technology devices
- Multi-language sign languages (not just ASL)
- Braille communication tools
- Modern accessibility aids

## 1. Background & Rationale

### 1.1 Current Accessibility Emoji Gap

| Category | Unicode 12.0 (2019) | Missing |
|----------|---------------------|---------|
| Mobility | вҷҝрҹҰҪрҹҰј | Exoskeletons, Sports wheelchairs |
| Hearing | рҹҰ»рҹ§Ҹ | Cochlear implants, Multi-language sign |
| Vision | рҹҰҜрҹҰ® | Braille displays, Screen readers |
| AAC | None | Communication devices |
| Cognitive | None | Neurodiversity symbols |

### 1.2 WIA's Authority to Propose

WIA has developed international standards for:
- **WIA-CI**: Cochlear implant data format (ISO-aligned)
- **ISP**: International Sign Phonology (300+ sign languages)
- **WIA-Braille**: Universal braille system (214 languages)
- **a11y.wiabook.com**: 211-language accessibility platform

### 1.3 Global Impact

| Disability | Population | Affected |
|------------|------------|----------|
| Hearing loss | 466 million | No multi-language sign emoji |
| Visual impairment | 285 million | No braille device emoji |
| Physical disability | 75 million | Limited device emoji |
| AAC users | 4 million | Zero emoji representation |

## 3. Selection Factors (per Unicode requirements)

### 3.1 Compatibility

All proposed emojis are designed to be:
- вң… Distinguishable at small sizes (emoji scale)
- вң… Culturally neutral (not specific to one country)
- вң… Distinct from existing emoji

### 3.2 Expected Usage Level

| Evidence Type | Source |
|---------------|--------|
| Web searches | "cochlear implant emoji" - 50K+ monthly |
| Social media | #CochlearImplant - 2M+ posts |
| Community requests | NAD, RNID, WFD petitions |
| Existing hashtags | #DeafEmoji, #AccessibilityMatters |

### 3.3 Multiple Meanings Not Required

These are **specific accessibility symbols** with clear, singular meaningsвҖ”similar to вҷҝ (wheelchair symbol).

### 3.4 Breaking New Ground

| Proposed | Current Alternative | Why Insufficient |
|----------|---------------------|------------------|
| Cochlear Implant | рҹҰ» (hearing aid) | Different device, different community |
| KSL/BSL Sign | рҹ§Ҹ (ASL only) | 300+ sign languages excluded |
| Braille Display | None | No representation |
| AAC Device | None | 4M users invisible |
| Exoskeleton | рҹҰҝ (prosthetic leg) | Different technology |
