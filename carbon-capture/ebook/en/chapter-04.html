The Need for Carbon Dioxide Removal

Even with aggressive emissions reductions, climate models indicate that achieving the Paris Agreement goals will require active removal of CO2 from the atmosphere. Carbon dioxide removal (CDR) technologies complement emissions reductions by addressing historical emissions and offsetting hard-to-abate sectors.

Direct Air Capture Fundamentals

Direct air capture (DAC) technology removes CO2 directly from ambient air, which contains approximately 420 ppm CO2. While this low concentration presents thermodynamic challenges, DAC offers unique advantages including location flexibility and the ability to address diffuse emissions.

Solid Sorbent DAC Systems

Solid sorbent DAC systems use porous materials functionalized with amine groups to capture CO2 from air. Regeneration is typically achieved through temperature swing using low-grade heat.

// Solid Sorbent DAC Technology
interface SolidSorbentDAC {
    sorbent: {
        baseMatrix: "silica" | "cellulose" | "MOF" | "polymer";
        functionalGroup: string;
        CO2Capacity: number;          // mmol/g
        cycleTime: number;            // minutes
        lifetime: number;             // cycles
    };

    contactor: {
        type: "fixed bed" | "moving bed" | "fluidized" | "structured";
        airVelocity: number;          // m/s
        pressureDrop: number;         // Pa
        contactArea: number;          // m²/m³
    };

    regeneration: {
        method: "temperature" | "vacuum" | "steam" | "combined";
        temperature: number;          // °C
        duration: number;             // minutes
        energySource: string;
    };

    performance: {
        captureRate: number;          // tonnes CO2/year per unit
        energyConsumption: number;    // GJ/tonne CO2
        waterConsumption: number;     // m³/tonne CO2
    };
}

Climeworks Achievement: The Orca plant in Iceland (2021) was the world's first commercial DAC facility, capturing 4,000 tonnes CO2/year using geothermal energy. The Mammoth plant (2024) scales this to 36,000 tonnes/year.

Liquid Solvent DAC Systems

Liquid solvent DAC systems use aqueous solutions (typically potassium hydroxide) in large air contactors. The captured CO2 is released through a high-temperature calcination process.

// Liquid Solvent DAC Technology
interface LiquidSolventDAC {
    contactor: {
        type: "cooling tower" | "cross-flow" | "spray tower";
        height: number;               // meters
        airFlow: number;              // m³/s
        solutionCirculation: number;  // m³/s
    };

    chemistry: {
        captureSolution: "KOH (aq)";
        concentration: number;        // mol/L
        reaction1: "2KOH + CO2 → K2CO3 + H2O";
        pelletReactor: {
            reaction: "K2CO3 + Ca(OH)2 → 2KOH + CaCite3",
            product: "CaCO3 pellets"
        };
        calciner: {
            reaction: "CaCO3 → CaO + CO2",
            temperature: 900,         // °C
            fuel: "Natural gas with oxy-combustion"
        };
        slaker: {
            reaction: "CaO + H2O → Ca(OH)2",
            purpose: "Regenerate calcium hydroxide"
        };
    };

    performance: {
        captureRate: number;          // tonnes CO2/year
        thermalEnergy: number;        // GJ/tonne CO2
        electricalEnergy: number;     // GJ/tonne CO2
        waterBalance: string;
    };
}

Bioenergy with Carbon Capture and Storage (BECCS)

BECCS combines biomass energy production with carbon capture, achieving negative emissions. Plants absorb CO2 during growth, which is then captured during combustion and permanently stored.

// BECCS System Configuration
interface BECCSSystem {
    biomassSupply: {
        type: "forestry residues" | "agricultural" | "energy crops" | "municipal waste";
        annualSupply: number;         // tonnes/year
        carbonContent: number;        // percent dry basis
        moistureContent: number;      // percent
        lowerHeatingValue: number;    // MJ/kg (as received)
    };

    conversionTechnology: {
        type: "combustion" | "gasification" | "fermentation" | "pyrolysis";
        capacity: number;             // MW thermal
        efficiency: number;           // percent
        CO2InFlueGas: number;         // percent
    };

    carbonCapture: {
        technology: string;
        captureRate: number;          // percent
        specificEnergy: number;       // GJ/tonne CO2
    };

    carbonBalance: {
        biomassCO2: number;           // tonnes CO2/year absorbed
        capturedCO2: number;          // tonnes CO2/year captured
        emissionsCCS: number;         // tonnes CO2/year from CCS energy
        netNegative: number;          // tonnes CO2/year net removal
    };
}

BECCS Potential: IPCC scenarios suggest BECCS could remove 1-7 GtCO2/year by 2050. However, sustainable biomass availability and land-use competition are key constraints that must be carefully managed.

Biochar and Soil Carbon

Biochar is a stable form of carbon produced by heating biomass in the absence of oxygen (pyrolysis). When added to soils, biochar can sequester carbon for centuries while improving soil health.

Summary

Direct air capture and negative emissions technologies are essential components of climate change mitigation. While currently expensive, DAC costs are declining rapidly. Combined with BECCS, enhanced weathering, biochar, and ocean-based approaches, a diverse portfolio of CDR methods can help achieve the scale of carbon removal needed to meet climate goals.
