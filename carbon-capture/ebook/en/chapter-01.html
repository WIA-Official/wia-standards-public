Carbon Dioxide: Molecular Properties

Carbon dioxide (CO2) is a linear triatomic molecule consisting of one carbon atom double-bonded to two oxygen atoms. Understanding its physical and chemical properties is essential for designing effective capture systems. At standard temperature and pressure, CO2 is a colorless, odorless gas that is denser than air.

The Carbon Dioxide Phase Diagram

CO2 exhibits unique phase behavior that is critical for capture, transport, and storage applications. Unlike most substances, CO2 does not exist as a liquid at atmospheric pressure - it sublimates directly from solid to gas. Understanding the phase diagram enables optimal design of capture and transport systems.

Phase Conditions Application
Gas Below critical point, low pressure Initial capture, atmospheric conditions
Liquid High pressure, moderate temperature Pipeline transport, intermediate storage
Supercritical Above 31.1°C, 73.8 bar Geological storage, enhanced oil recovery
Solid (Dry Ice) Below -78.5°C at 1 bar Specialized transport, temporary storage

Chemical Absorption Principles

Chemical absorption is the most mature and widely deployed carbon capture technology. It involves the chemical reaction between CO2 and a liquid solvent, typically an amine solution. The reaction forms a weakly bonded compound that can be broken down by heating, releasing pure CO2 for storage.

// Amine-Based Chemical Absorption Reactions
// Primary Amine (e.g., MEA - Monoethanolamine)
// CO2 + 2RNH2 ⇌ RNHCOO⁻ + RNH3⁺

const amineCaptureChemistry = {
    primaryAmine: {
        example: "Monoethanolamine (MEA)",
        formula: "HOCH2CH2NH2",
        reaction: "CO2 + 2MEA ⇌ MEACOO⁻ + MEAH⁺",
        stoichiometry: "0.5 mol CO2 per mol MEA",
        regenerationTemp: "120-140°C",
        heatOfReaction: -85  // kJ/mol CO2
    },
    secondaryAmine: {
        example: "Diethanolamine (DEA)",
        formula: "(HOCH2CH2)2NH",
        stoichiometry: "0.5 mol CO2 per mol DEA",
        advantage: "Lower regeneration energy"
    },
    tertiaryAmine: {
        example: "Methyldiethanolamine (MDEA)",
        formula: "(HOCH2CH2)2NCH3",
        stoichiometry: "1 mol CO2 per mol MDEA",
        mechanism: "Base-catalyzed hydration",
        advantage: "Lowest regeneration energy"
    }
};
Key Principle: The choice of amine solvent involves trade-offs between reaction rate, CO2 capacity, regeneration energy, and degradation resistance. Blended amine systems often provide optimal performance by combining the fast kinetics of primary amines with the lower energy requirements of tertiary amines.

Absorption Column Design

The absorption column is the heart of a chemical absorption system. Flue gas containing CO2 enters at the bottom and flows upward through packing material, while lean amine solution flows downward. This counter-current contact maximizes mass transfer efficiency.

// Absorption Column Design Parameters
interface AbsorptionColumn {
    type: "packed" | "tray";
    height: number;        // meters
    diameter: number;      // meters
    packingType: string;
    packingHeight: number; // meters

    operatingConditions: {
        temperature: number;      // °C
        pressure: number;         // bar
        liquidFlowRate: number;   // m³/h
        gasFlowRate: number;      // m³/h
        L_G_ratio: number;        // liquid to gas ratio
    };

    performance: {
        captureEfficiency: number;  // percent
        pressureDrop: number;       // mbar/m
        floodingVelocity: number;   // m/s
    };
}

// Example Design Calculation
function calculateColumnHeight(
    flueGasFlow: number,      // mol/s
    CO2Inlet: number,         // mol fraction
    CO2Outlet: number,        // mol fraction
    solventCapacity: number,  // mol CO2/mol solvent
    HTU: number               // Height of Transfer Unit (m)
): number {
    const NTU = Math.log((CO2Inlet * (1 - CO2Outlet)) /
                         (CO2Outlet * (1 - CO2Inlet)));
    const columnHeight = NTU * HTU;
    return columnHeight;  // meters
}

Physical Adsorption Mechanisms

Physical adsorption (physisorption) relies on weak van der Waals forces to bind CO2 molecules to solid surfaces. Unlike chemical absorption, no chemical bonds are formed, resulting in lower regeneration energy requirements but typically lower selectivity.

Common Adsorbent Materials

  • Activated Carbon: High surface area (1000-3500 m²/g), low cost, moderate CO2 capacity
  • Zeolites: Microporous aluminosilicates with defined pore sizes for molecular sieving
  • Metal-Organic Frameworks (MOFs): Ultra-high surface area (up to 7000 m²/g), tunable properties
  • Silica Gels: Modified silica with amine functionalization for enhanced selectivity

            P: "Partial pressure"

// Calculate CO2 Loading
function calculateLoading(P_CO2: number, qmax: number, K: number): number {
    return (qmax * K * P_CO2) / (1 + K * P_CO2);  // mol/kg
}

Pressure Swing Adsorption (PSA)

PSA is a cyclic adsorption process that separates gases based on differences in adsorption affinity. The process operates by alternating between high-pressure adsorption and low-pressure desorption phases.

// PSA Cycle Configuration
interface PSACycle {
    steps: {
        adsorption: {
            pressure: number;      // bar
            duration: number;      // seconds
            feedFlow: number;      // mol/s
        };
        blowdown: {
            pressure: number;      // bar
            duration: number;      // seconds
        };
        purge: {
            pressure: number;      // bar
            duration: number;      // seconds
            purgeFlow: number;     // mol/s
        };
        pressurization: {
            targetPressure: number; // bar
            duration: number;       // seconds
        };
    };
    numberOfBeds: number;
    cycleTime: number;  // total seconds
}

// Example 4-Bed PSA System
const psaSystem: PSACycle = {
    steps: {
        adsorption: { pressure: 10, duration: 120, feedFlow: 100 },
        blowdown: { pressure: 1, duration: 30 },
        purge: { pressure: 1, duration: 60, purgeFlow: 20 },
        pressurization: { targetPressure: 10, duration: 30 }
    },
    numberOfBeds: 4,
    cycleTime: 240
};

Temperature Swing Adsorption (TSA)

TSA uses temperature changes to drive adsorption and desorption. During the adsorption phase, CO2 is captured at lower temperatures. Regeneration occurs by heating the adsorbent, which releases the captured CO2. TSA is particularly suited for applications where waste heat is available.

Energy Comparison: While TSA typically requires more energy than PSA due to the heat capacity of the adsorbent, it can achieve higher CO2 purity and is preferred when waste heat is available. Hybrid systems combining PSA and TSA elements can optimize overall efficiency.

Membrane Separation Technology

Membrane-based carbon capture uses selective barriers that allow CO2 to pass through more readily than other gases. This technology offers advantages in terms of modularity, simplicity, and the absence of chemical reagents.

// Membrane Performance Parameters
interface MembraneSystem {
    material: string;
    permeability: {
        CO2: number;      // Barrer
        N2: number;       // Barrer
    };
    selectivity: number;  // CO2/N2 ratio
    thickness: number;    // micrometers
    area: number;         // m²

    operatingConditions: {
        feedPressure: number;       // bar
        permeatePressure: number;   // bar
        temperature: number;        // °C
    };

    performance: {
        recovery: number;           // percent
        purity: number;             // percent
        flux: number;               // mol/m²/s
    };
}

// Flux Calculation
function calculateFlux(
    permeability: number,  // Barrer
    thickness: number,     // micrometers
    deltaPressure: number  // bar
): number {
    // Convert Barrer to SI units
    const P_SI = permeability * 3.35e-16;  // mol·m/(m²·s·Pa)
    const flux = P_SI * (deltaPressure * 1e5) / (thickness * 1e-6);
    return flux;  // mol/m²/s
}

Thermodynamics of Carbon Capture

The minimum theoretical energy required to separate CO2 from a gas mixture is determined by thermodynamic principles. Understanding these limits helps evaluate the efficiency of capture technologies and identify opportunities for improvement.

// Minimum Work of Separation
function minimumSeparationWork(
    y_CO2: number,        // CO2 mole fraction in feed
    T: number,            // Temperature (K)
    captureRate: number   // Fraction captured (0-1)
): number {
    const R = 8.314;  // J/mol/K

    // Minimum work per mole of CO2 captured
    const W_min = R * T * (
        Math.log(1 / y_CO2) +
        ((1 - captureRate) / captureRate) * Math.log(1 / (1 - y_CO2))
    );

    return W_min;  // J/mol CO2
}

// Example: Coal power plant flue gas
const y_CO2_coal = 0.13;     // 13% CO2
const temperature = 313;      // 40°C = 313K
const captureRate = 0.90;     // 90% capture

const W_min = minimumSeparationWork(y_CO2_coal, temperature, captureRate);
// Result: ~8.4 kJ/mol CO2 or ~190 kJ/kg CO2

// Actual processes require 3-5x minimum due to irreversibilities
Efficiency Benchmark: Current commercial amine systems require approximately 3-4 GJ/tonne CO2, while the theoretical minimum for coal flue gas is about 0.4 GJ/tonne. This gap represents significant opportunity for technology improvement.
