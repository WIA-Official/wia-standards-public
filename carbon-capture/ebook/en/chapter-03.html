Overview of Point Source Capture

Point source capture refers to the capture of CO2 from large stationary emission sources such as power plants, refineries, cement factories, and steel mills. These sources account for approximately 60% of global CO2 emissions, making point source capture a critical component of climate change mitigation strategies.

Post-Combustion Capture

Post-combustion capture is the most mature and widely deployed carbon capture technology. It involves separating CO2 from flue gases after fuel combustion. This approach can be retrofitted to existing power plants and industrial facilities, making it particularly attractive for near-term deployment.

Post-Combustion Process Flow

// Post-Combustion Capture Process
interface PostCombustionSystem {
    flueGasConditions: {
        temperature: number;      // °C (typically 120-180)
        pressure: number;         // bar (typically 1.0-1.1)
        CO2Concentration: number; // mol% (typically 10-15)
        flowRate: number;         // Nm³/hour
        contaminants: {
            SO2: number;          // ppm
            NOx: number;          // ppm
            particulateMatter: number;  // mg/Nm³
            O2: number;           // mol%
        };
    };

    pretreatment: {
        directContactCooler: boolean;
        desulfurization: boolean;
        particulateRemoval: boolean;
        targetConditions: {
            temperature: 40,      // °C
            SO2: "<10 ppm",
            PM: "<5 mg/Nm³"
        };
    };

    captureSection: {
        technology: "amine" | "carbonate" | "membrane";
        solvent: string;
        captureEfficiency: number;  // percent
        specificEnergy: number;     // GJ/tonne CO2
    };
}

Advanced Amine Solvents

While monoethanolamine (MEA) remains the benchmark solvent, significant research has developed advanced solvents with improved performance characteristics.

Solvent System Regeneration Energy Advantages TRL
30% MEA 3.6-4.0 GJ/t Mature, well-understood 9
CESAR1 (AMP/PZ) 2.7-3.0 GJ/t 25% lower energy 7
KS-1 (Hindered Amine) 2.5-2.8 GJ/t Lower degradation 8
Aqueous Ammonia 2.0-2.5 GJ/t Lowest energy, cheap 6
Phase-Change Solvents 2.0-2.5 GJ/t Reduced circulation 5
Solvent Selection Criteria: The choice of solvent depends on multiple factors including CO2 loading capacity, reaction kinetics, regeneration energy, thermal and oxidative stability, corrosivity, volatility, environmental impact, and cost. Blended solvents often provide the best balance of properties.

Pre-Combustion Capture

Pre-combustion capture involves converting fuel to a hydrogen-rich syngas before combustion, capturing the CO2 from this high-pressure, high-concentration stream. This approach is particularly suited for integrated gasification combined cycle (IGCC) power plants and hydrogen production facilities.

// Pre-Combustion Capture Process
/*
FUEL (Coal/Gas)
    ↓
GASIFIER/REFORMER → SYNGAS (CO + H2)
    ↓
WATER-GAS SHIFT → H2 + CO2
    ↓
CO2 CAPTURE → Pure CO2 for Storage
    ↓
HYDROGEN → Combustion in Gas Turbine → Electricity
*/

interface PreCombustionSystem {
    fuelProcessing: {
        type: "gasification" | "steamReforming" | "autothermalReforming";
        operatingPressure: number;    // bar
        operatingTemperature: number; // °C
        oxygenRequirement: number;    // kg O2/kg fuel
    };

    waterGasShift: {
        stages: number;
        catalysts: string[];
        COConversion: number;         // percent
        outletComposition: {
            H2: number;               // mol%
            CO2: number;              // mol%
            CO: number;               // mol%
            H2O: number;              // mol%
        };
    };

    CO2Capture: {
        technology: "physical" | "chemical";
        solvent: string;
        pressure: number;             // bar
        captureEfficiency: number;    // percent
        specificEnergy: number;       // GJ/tonne CO2
    };
}

Oxy-Fuel Combustion

Oxy-fuel combustion replaces air with pure oxygen for combustion, producing a flue gas that is primarily CO2 and water vapor. After water condensation, a high-purity CO2 stream is obtained without the need for a separate capture process.

// Oxy-Fuel Combustion System
interface OxyFuelSystem {
    airSeparationUnit: {
        technology: "cryogenic" | "membrane" | "adsorption";
        oxygenPurity: number;       // percent
        oxygenProduction: number;   // tonnes/hour
        specificEnergy: number;     // kWh/tonne O2
    };

    combustion: {
        oxygenExcess: number;       // percent
        recycleRatio: number;       // fraction of flue gas recycled
        flameTemperature: number;   // °C
        adiabaticTemp: number;      // °C (without recycle)
    };

    flueGasComposition: {
        CO2: number;                // mol% (dry basis)
        H2O: number;                // mol%
        O2: number;                 // mol%
        N2: number;                 // mol%
        Ar: number;                 // mol%
    };

    CO2Processing: {
        condensationTemp: number;   // °C
        dehydration: boolean;
        purification: "none" | "distillation" | "catalytic";
        finalPurity: number;        // percent
    };
}

// Example: Oxy-Fuel Coal Power Plant
const oxyFuelPlant: OxyFuelSystem = {
    airSeparationUnit: {
        technology: "cryogenic",
        oxygenPurity: 95,
        oxygenProduction: 850,
        specificEnergy: 220
    },
    combustion: {
        oxygenExcess: 3,
        recycleRatio: 0.7,
        flameTemperature: 1900,
        adiabaticTemp: 3500
    },
    flueGasComposition: {
        CO2: 80,
        H2O: 8,
        O2: 4,
        N2: 6,
        Ar: 2
    },
    CO2Processing: {
        condensationTemp: 25,
        dehydration: true,
        purification: "distillation",
        finalPurity: 99.9
    }
};
Oxy-Fuel Advantage: By eliminating nitrogen from the combustion process, oxy-fuel technology produces a flue gas with CO2 concentrations of 80-95% (dry basis), dramatically simplifying CO2 purification. The main energy penalty comes from the air separation unit.

Chemical Looping Combustion

Chemical looping combustion (CLC) is an emerging technology that uses metal oxide particles as oxygen carriers, enabling inherent CO2 separation without energy-intensive air separation or solvent regeneration.

// Chemical Looping Combustion Process
interface ChemicalLoopingSystem {
    oxygenCarrier: {
        metal: "Fe" | "Ni" | "Cu" | "Mn" | "Co";
        support: string;
        particleSize: number;        // micrometers
        density: number;             // kg/m³
        oxygenCapacity: number;      // wt%
        lifetime: number;            // hours
    };

    fuelReactor: {
        type: "fluidizedBed" | "movingBed" | "rotary";
        temperature: number;         // °C
        pressure: number;            // bar
        solidCirculationRate: number; // kg/s
        gasResidenceTime: number;    // seconds
    };

    airReactor: {
        temperature: number;         // °C
        airFlowRate: number;         // kg/s
        heatRelease: number;         // MW
    };

    performance: {
        fuelConversion: number;      // percent
        CO2Purity: number;           // percent
        thermalEfficiency: number;   // percent
        inherentCapture: true;
    };
}

Summary

Point source capture technologies offer multiple pathways for reducing industrial CO2 emissions. Post-combustion capture provides the most flexible retrofit option, pre-combustion capture offers efficiency advantages for new plants, and oxy-fuel combustion provides inherent CO2 separation. The optimal choice depends on the specific application, existing infrastructure, and project economics.
