챕터 3: 포인트 소스 포집 기술

학습 목표: 연소후, 연소전, 순산소 연소 포집 기술 습득. 산업 환경에서 각 접근 방식의 장점, 한계 및 최적 응용 분야 이해

포인트 소스 포집 개요

포인트 소스 포집은 발전소, 정유소, 시멘트 공장, 제철소와 같은 대규모 고정 배출원에서 CO2를 포집하는 것을 말합니다. 이러한 출처는 전 세계 CO2 배출량의 약 60%를 차지하며, 포인트 소스 포집은 기후 변화 완화 전략의 핵심 구성 요소입니다.

연소후 포집

연소후 포집은 가장 성숙하고 널리 배포된 탄소 포집 기술입니다. 연료 연소 후 배기가스에서 CO2를 분리하는 방식입니다. 이 접근 방식은 기존 발전소 및 산업 시설에 레트로핏이 가능하여 단기 배포에 특히 매력적입니다.

연소후 공정 흐름

// 연소후 포집 공정
interface PostCombustionSystem {
    flueGasConditions: {
        temperature: number;      // °C (일반적으로 120-180)
        pressure: number;         // bar (일반적으로 1.0-1.1)
        CO2Concentration: number; // mol% (일반적으로 10-15)
        flowRate: number;         // Nm³/시간
        contaminants: {
            SO2: number;          // ppm
            NOx: number;          // ppm
            particulateMatter: number;  // mg/Nm³
            O2: number;           // mol%
        };
    };

    pretreatment: {
        directContactCooler: boolean;
        desulfurization: boolean;
        particulateRemoval: boolean;
        targetConditions: {
            temperature: 40,      // °C
            SO2: "<10 ppm",
            PM: "<5 mg/Nm³"
        };
    };

    captureSection: {
        technology: "amine" | "carbonate" | "membrane";
        solvent: string;
        captureEfficiency: number;  // 퍼센트
        specificEnergy: number;     // GJ/톤 CO2
    };
}

// 예시: 석탄 발전소 연소후 포집
const coalPlantCapture: PostCombustionSystem = {
    flueGasConditions: {
        temperature: 150,
        pressure: 1.05,
        CO2Concentration: 13.5,
        flowRate: 2500000,
        contaminants: { SO2: 50, NOx: 200, particulateMatter: 20, O2: 5 }
    },
    pretreatment: {
        directContactCooler: true,
        desulfurization: true,
        particulateRemoval: true,
        targetConditions: { temperature: 40, SO2: "<10 ppm", PM: "<5 mg/Nm³" }
    },
    captureSection: {
        technology: "amine",
        solvent: "30 wt% MEA",
        captureEfficiency: 90,
        specificEnergy: 3.6
    }
};

고급 아민 용매

모노에탄올아민(MEA)이 벤치마크 용매로 남아 있지만, 상당한 연구를 통해 성능 특성이 개선된 고급 용매가 개발되었습니다.

용매 시스템 재생 에너지 장점 TRL
30% MEA 3.6-4.0 GJ/t 성숙, 잘 이해됨 9
CESAR1 (AMP/PZ) 2.7-3.0 GJ/t 25% 낮은 에너지 7
KS-1 (입체장애 아민) 2.5-2.8 GJ/t 낮은 열화 8
수용성 암모니아 2.0-2.5 GJ/t 최저 에너지, 저렴 6
상변화 용매 2.0-2.5 GJ/t 감소된 순환 5
용매 선택 기준: 용매 선택은 CO2 로딩 용량, 반응 속도, 재생 에너지, 열 및 산화 안정성, 부식성, 휘발성, 환경 영향 및 비용을 포함한 여러 요소에 따라 달라집니다. 혼합 용매는 종종 최적의 특성 균형을 제공합니다.

연소전 포집

연소전 포집은 연소 전에 연료를 수소가 풍부한 합성가스로 전환하고, 이 고압, 고농도 스트림에서 CO2를 포집합니다. 이 접근 방식은 석탄가스화복합발전(IGCC) 발전소와 수소 생산 시설에 특히 적합합니다.

// 연소전 포집 공정
/*
연료 (석탄/가스)
    ↓
가스화기/개질기 → 합성가스 (CO + H2)
    ↓
수성 가스 전환 → H2 + CO2
    ↓
CO2 포집 → 저장을 위한 순수 CO2
    ↓
수소 → 가스 터빈에서 연소 → 전기
*/

interface PreCombustionSystem {
    fuelProcessing: {
        type: "gasification" | "steamReforming" | "autothermalReforming";
        operatingPressure: number;    // bar
        operatingTemperature: number; // °C
        oxygenRequirement: number;    // kg O2/kg 연료
    };

    waterGasShift: {
        stages: number;
        catalysts: string[];
        COConversion: number;         // 퍼센트
        outletComposition: {
            H2: number;               // mol%
            CO2: number;              // mol%
            CO: number;               // mol%
            H2O: number;              // mol%
        };
    };

    CO2Capture: {
        technology: "physical" | "chemical";
        solvent: string;
        pressure: number;             // bar
        captureEfficiency: number;    // 퍼센트
        specificEnergy: number;       // GJ/톤 CO2
    };
}

// 예시: 연소전 포집이 있는 IGCC
const igccCapture: PreCombustionSystem = {
    fuelProcessing: {
        type: "gasification",
        operatingPressure: 40,
        operatingTemperature: 1400,
        oxygenRequirement: 0.85
    },
    waterGasShift: {
        stages: 2,
        catalysts: ["고온 Fe-Cr", "저온 Cu-Zn"],
        COConversion: 98,
        outletComposition: { H2: 55, CO2: 40, CO: 1, H2O: 4 }
    },
    CO2Capture: {
        technology: "physical",
        solvent: "Selexol (DEPG)",
        pressure: 35,
        captureEfficiency: 95,
        specificEnergy: 0.8
    }
};

순산소 연소

순산소 연소는 연소를 위해 공기 대신 순수 산소를 사용하여 주로 CO2와 수증기로 구성된 배기가스를 생산합니다. 물 응축 후 별도의 포집 공정 없이 고순도 CO2 스트림을 얻습니다.

// 순산소 연소 시스템
interface OxyFuelSystem {
    airSeparationUnit: {
        technology: "cryogenic" | "membrane" | "adsorption";
        oxygenPurity: number;       // 퍼센트
        oxygenProduction: number;   // 톤/시간
        specificEnergy: number;     // kWh/톤 O2
    };

    combustion: {
        oxygenExcess: number;       // 퍼센트
        recycleRatio: number;       // 재순환 배기가스 비율
        flameTemperature: number;   // °C
        adiabaticTemp: number;      // °C (재순환 없이)
    };

    flueGasComposition: {
        CO2: number;                // mol% (건조 기준)
        H2O: number;                // mol%
        O2: number;                 // mol%
        N2: number;                 // mol%
        Ar: number;                 // mol%
    };

    CO2Processing: {
        condensationTemp: number;   // °C
        dehydration: boolean;
        purification: "none" | "distillation" | "catalytic";
        finalPurity: number;        // 퍼센트
    };
}

// 예시: 순산소 석탄 발전소
const oxyFuelPlant: OxyFuelSystem = {
    airSeparationUnit: {
        technology: "cryogenic",
        oxygenPurity: 95,
        oxygenProduction: 850,
        specificEnergy: 220
    },
    combustion: {
        oxygenExcess: 3,
        recycleRatio: 0.7,
        flameTemperature: 1900,
        adiabaticTemp: 3500
    },
    flueGasComposition: {
        CO2: 80,
        H2O: 8,
        O2: 4,
        N2: 6,
        Ar: 2
    },
    CO2Processing: {
        condensationTemp: 25,
        dehydration: true,
        purification: "distillation",
        finalPurity: 99.9
    }
};
순산소 연소 장점: 연소 과정에서 질소를 제거함으로써 순산소 기술은 CO2 농도가 80-95%(건조 기준)인 배기가스를 생산하여 CO2 정제를 크게 단순화합니다. 주요 에너지 페널티는 공기 분리 장치에서 발생합니다.

기술 비교

매개변수 연소후 연소전 순산소
공급 CO2 3-15% 15-60% 80-95%
공급 압력 ~1 bar 15-40 bar ~1 bar
에너지 페널티 25-40% 15-25% 25-35%

성숙도 (TRL) 9 7-8 6-7

화학 루핑 연소

화학 루핑 연소(CLC)는 금속 산화물 입자를 산소 운반체로 사용하여 에너지 집약적인 공기 분리 또는 용매 재생 없이 본질적인 CO2 분리를 가능하게 하는 신흥 기술입니다.

// 화학 루핑 연소 공정
interface ChemicalLoopingSystem {
    oxygenCarrier: {
        metal: "Fe" | "Ni" | "Cu" | "Mn" | "Co";
        support: string;
        particleSize: number;        // 마이크로미터
        density: number;             // kg/m³
        oxygenCapacity: number;      // wt%
        lifetime: number;            // 시간
    };

    fuelReactor: {
        type: "fluidizedBed" | "movingBed" | "rotary";
        temperature: number;         // °C
        pressure: number;            // bar
        solidCirculationRate: number; // kg/s
        gasResidenceTime: number;    // 초
    };

    airReactor: {
        temperature: number;         // °C
        airFlowRate: number;         // kg/s
        heatRelease: number;         // MW
    };

    performance: {
        fuelConversion: number;      // 퍼센트
        CO2Purity: number;           // 퍼센트
        thermalEfficiency: number;   // 퍼센트
        inherentCapture: true;
    };
}

요약

포인트 소스 포집 기술은 산업 CO2 배출량을 줄이기 위한 여러 경로를 제공합니다. 연소후 포집은 가장 유연한 레트로핏 옵션을 제공하고, 연소전 포집은 신규 플랜트에 효율성 장점을 제공하며, 순산소 연소는 본질적인 CO2 분리를 제공합니다. 최적의 선택은 특정 응용 분야, 기존 인프라 및 프로젝트 경제성에 따라 달라집니다.
