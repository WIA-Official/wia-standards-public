챕터 4: 직접 공기 포집 및 자연 솔루션

학습 목표: 직접 공기 포집(DAC) 기술 이해, 탄소 포집 및 저장이 결합된 바이오에너지(BECCS) 탐구, 해양 기반 탄소 제거 학습, 탄소 격리를 위한 강화된 자연 솔루션 평가

이산화탄소 제거의 필요성

적극적인 배출 감소에도 불구하고 기후 모델은 파리협정 목표를 달성하려면 대기에서 CO2를 적극적으로 제거해야 한다고 나타냅니다. 이산화탄소 제거(CDR) 기술은 역사적 배출을 해결하고 감축이 어려운 부문을 상쇄함으로써 배출 감소를 보완합니다.

직접 공기 포집 기초

직접 공기 포집(DAC) 기술은 약 420 ppm의 CO2를 포함하는 주변 공기에서 직접 CO2를 제거합니다. 이 낮은 농도는 열역학적 도전을 제시하지만, DAC는 위치 유연성과 분산 배출을 해결하는 능력을 포함한 독특한 장점을 제공합니다.

// DAC 열역학적 도전
const dacThermodynamics = {
    atmosphericConditions: {
        CO2Concentration: 420,       // ppm (0.042%)
        comparedToFlueGas: "300배 더 희석",
        temperatureRange: "-40 ~ +50°C",
        humidityRange: "5-100% RH"
    },

    minimumWork: {
        theoretical: 20,             // kJ/mol CO2
        practical: "100-200",        // kJ/mol CO2
        comparison: {
            coalFlueGas: 8.4,        // kJ/mol CO2
            ratio: "2.5-5배 더 많은 에너지 필요"
        }
    },

    airProcessing: {
        airPerTonneCO2: 1400000,     // m³ (420 ppm에서)
        contactorDesign: "대용량 공기 처리량 필요",
        pressureDrop: "최소화 필수 (<100 Pa)"
    }
};

고체 흡착제 DAC 시스템

고체 흡착제 DAC 시스템은 아민 그룹으로 기능화된 다공성 소재를 사용하여 공기에서 CO2를 포집합니다. 재생은 일반적으로 저급 열을 사용한 온도 스윙을 통해 달성됩니다.

// 고체 흡착제 DAC 기술
interface SolidSorbentDAC {
    sorbent: {
        baseMatrix: "silica" | "cellulose" | "MOF" | "polymer";
        functionalGroup: string;
        CO2Capacity: number;          // mmol/g
        cycleTime: number;            // 분
        lifetime: number;             // 사이클
    };

    contactor: {
        type: "fixed bed" | "moving bed" | "fluidized" | "structured";
        airVelocity: number;          // m/s
        pressureDrop: number;         // Pa
        contactArea: number;          // m²/m³
    };

    regeneration: {
        method: "temperature" | "vacuum" | "steam" | "combined";
        temperature: number;          // °C
        duration: number;             // 분
        energySource: string;
    };

    performance: {
        captureRate: number;          // 톤 CO2/년 (유닛당)
        energyConsumption: number;    // GJ/톤 CO2
        waterConsumption: number;     // m³/톤 CO2
    };
}

// 예시: Climeworks 유형 시스템
const climeworksDAC: SolidSorbentDAC = {
    sorbent: {
        baseMatrix: "cellulose",
        functionalGroup: "아민 기능화 필터",
        CO2Capacity: 1.5,
        cycleTime: 180,
        lifetime: 10000
    },
    contactor: {
        type: "structured",
        airVelocity: 1.5,
        pressureDrop: 50,
        contactArea: 500
    },
    regeneration: {
        method: "temperature",
        temperature: 100,
        duration: 60,
        energySource: "저급 열 / 지열"
    },
    performance: {
        captureRate: 50,
        energyConsumption: 6.0,
        waterConsumption: 0
    }
};
Climeworks 성과: 아이슬란드의 Orca 플랜트(2021년)는 지열 에너지를 사용하여 연간 4,000톤 CO2를 포집하는 세계 최초의 상업용 DAC 시설이었습니다. Mammoth 플랜트(2024년)는 이를 연간 36,000톤으로 확장합니다.

액체 용매 DAC 시스템

액체 용매 DAC 시스템은 대형 공기 접촉기에서 수용액(일반적으로 수산화칼륨)을 사용합니다. 포집된 CO2는 고온 소성 공정을 통해 방출됩니다.

// 액체 용매 DAC 기술
interface LiquidSolventDAC {
    contactor: {
        type: "cooling tower" | "cross-flow" | "spray tower";
        height: number;               // 미터
        airFlow: number;              // m³/s
        solutionCirculation: number;  // m³/s
    };

    chemistry: {
        captureSolution: "KOH (aq)";
        concentration: number;        // mol/L
        reaction1: "2KOH + CO2 → K2CO3 + H2O";
        pelletReactor: {
            reaction: "K2CO3 + Ca(OH)2 → 2KOH + CaCO3",
            product: "CaCO3 펠릿"
        };
        calciner: {
            reaction: "CaCO3 → CaO + CO2",
            temperature: 900,         // °C
            fuel: "순산소 연소 천연가스"
        };
        slaker: {
            reaction: "CaO + H2O → Ca(OH)2",
            purpose: "수산화칼슘 재생"
        };
    };

    performance: {
        captureRate: number;          // 톤 CO2/년
        thermalEnergy: number;        // GJ/톤 CO2
        electricalEnergy: number;     // GJ/톤 CO2
        waterBalance: string;
    };
}

탄소 포집 및 저장이 결합된 바이오에너지 (BECCS)

BECCS는 바이오매스 에너지 생산과 탄소 포집을 결합하여 음의 배출을 달성합니다. 식물은 성장 중에 CO2를 흡수하고, 이는 연소 중에 포집되어 영구적으로 저장됩니다.

// BECCS 시스템 구성
interface BECCSSystem {
    biomassSupply: {
        type: "forestry residues" | "agricultural" | "energy crops" | "municipal waste";
        annualSupply: number;         // 톤/년
        carbonContent: number;        // 건조 기준 퍼센트
        moistureContent: number;      // 퍼센트
        lowerHeatingValue: number;    // MJ/kg (수령 기준)
    };

    conversionTechnology: {
        type: "combustion" | "gasification" | "fermentation" | "pyrolysis";
        capacity: number;             // MW 열
        efficiency: number;           // 퍼센트
        CO2InFlueGas: number;         // 퍼센트
    };

    carbonCapture: {
        technology: string;
        captureRate: number;          // 퍼센트
        specificEnergy: number;       // GJ/톤 CO2
    };

    carbonBalance: {
        biomassCO2: number;           // 톤 CO2/년 흡수
        capturedCO2: number;          // 톤 CO2/년 포집
        emissionsCCS: number;         // 톤 CO2/년 (CCS 에너지에서)
        netNegative: number;          // 톤 CO2/년 순 제거
    };
}

// 예시: Drax 발전소 BECCS
const draxBECCS: BECCSSystem = {
    biomassSupply: {
        type: "forestry residues",
        annualSupply: 7000000,
        carbonContent: 50,
        moistureContent: 10,
        lowerHeatingValue: 17
    },
    conversionTechnology: {
        type: "combustion",
        capacity: 2600,
        efficiency: 38,
        CO2InFlueGas: 14
    },
    carbonCapture: {
        technology: "연소후 아민",
        captureRate: 90,
        specificEnergy: 3.5
    },
    carbonBalance: {
        biomassCO2: 12000000,
        capturedCO2: 8000000,
        emissionsCCS: 500000,
        netNegative: 7500000
    }
};
BECCS 잠재력: IPCC 시나리오는 BECCS가 2050년까지 연간 1-7 GtCO2를 제거할 수 있다고 제안합니다. 그러나 지속 가능한 바이오매스 가용성과 토지 이용 경쟁은 신중하게 관리해야 하는 핵심 제약 사항입니다.

강화된 풍화

강화된 풍화는 CO2를 소비하는 자연적인 암석 풍화 과정을 가속화합니다. 농경지에 미세하게 분쇄된 규산염 암석을 뿌리면 토양 비옥도를 개선하면서 탄소를 격리할 수 있습니다.

// 강화된 풍화 공정
interface EnhancedWeathering {
    rockType: {
        name: string;
        mineralogy: string;
        CO2Sequestration: number;     // kg CO2/톤 암석
        weatheringRate: string;
    };

    application: {
        grindSize: number;            // 마이크로미터
        spreadingRate: number;        // 톤/헥타르/년
        landArea: number;             // 헥타르
        annualSequestration: number;  // 톤 CO2/년
    };

    energy: {
        mining: number;               // kWh/톤 암석
        grinding: number;             // kWh/톤 암석
        transport: number;            // kWh/톤 암석
        spreading: number;            // kWh/톤 암석
        total: number;                // kWh/톤 CO2
    };

    cobenefits: {
        soilpH: string;
        nutrients: string[];
        cropYield: string;
    };
}

// 예시: 현무암 적용
const basaltWeathering: EnhancedWeathering = {
    rockType: {
        name: "현무암",
        mineralogy: "Ca,Mg 규산염",
        CO2Sequestration: 300,
        weatheringRate: "수년에서 수십 년"
    },
    application: {
        grindSize: 100,
        spreadingRate: 50,
        landArea: 100000,
        annualSequestration: 150000
    },
    energy: {
        mining: 5,
        grinding: 40,
        transport: 10,
        spreading: 3,
        total: 175
    },
    cobenefits: {
        soilpH: "토양 산성도 감소",
        nutrients: ["칼슘", "마그네슘", "칼륨", "규소"],
        cropYield: "5-20% 증가 보고"
    }
};

바이오차 및 토양 탄소

바이오차는 산소 없이 바이오매스를 가열(열분해)하여 생산되는 안정적인 형태의 탄소입니다. 토양에 첨가하면 바이오차는 토양 건강을 개선하면서 수백 년 동안 탄소를 격리할 수 있습니다.

// 바이오차 탄소 격리
const biocharSystem = {
    production: {
        feedstock: ["농업 잔류물", "임업 폐기물", "분뇨"],
        pyrolysisTemp: "400-700°C",
        yield: "바이오매스의 25-35%",
        carbonContent: "70-90%",
        stability: "100-1000년 이상"
    },

    carbonBalance: {
        biomassCarbonIn: 1000,        // kg C (톤 원료당)
        biocharCarbonOut: 350,        // kg C (톤 원료당)
        biogasCarbon: 150,            // kg C (포집 가능)
        bioOilCarbon: 200,            // kg C (화석 연료 대체 가능)
        emissions: 50,                // kg C (처리)
        netSequestered: 300           // kg C (톤 원료당)
    },

    soilBenefits: {
        waterRetention: "+20-30%",
        nutrientRetention: "침출 감소",
        microbialActivity: "강화",
        cropYield: "토양 유형에 따라 0-200%"
    },

    globalPotential: {
        sustainable: "1-2 Gt CO2/년",
        landRequirement: "최소 (폐기물 사용)",
        cost: "$30-120/톤 CO2"
    }
};

요약

직접 공기 포집과 음의 배출 기술은 기후 변화 완화의 필수 구성 요소입니다. 현재 비용이 비싸지만 DAC 비용은 빠르게 하락하고 있습니다. BECCS, 강화된 풍화, 바이오차, 해양 기반 접근 방식과 결합된 다양한 CDR 방법 포트폴리오는 기후 목표를 달성하는 데 필요한 탄소 제거 규모를 달성하는 데 도움이 될 수 있습니다.
