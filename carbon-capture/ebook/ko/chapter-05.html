학습 목표: WIA-CARBON_CAPTURE 사양에 따른 압력 시스템 관리, 화학물질 취급 절차, 비상 대응 계획, 지속적인 모니터링 요구사항을 포함한 탄소 포집 운영의 중요 안전 프로토콜 습득

안전 개요

탄소 포집 시설은 고온 및 고압에서의 화학물질 취급, 압축 가스, 회전 장비를 포함합니다. 포괄적인 안전 관리 시스템은 인력, 대중 및 환경을 보호하는 데 필수적입니다. 이 챕터에서는 WIA-CARBON_CAPTURE 표준 및 국제 모범 사례에 맞춘 안전 요구사항을 설명합니다.

이산화탄소 위험

CO2는 가연성이 없거나 전통적인 의미에서 독성이 없지만, 밀폐 공간이나 축적 구역에서 심각한 질식 위험을 제시하며 심각한 부상이나 사망을 초래할 수 있습니다.

CO2 농도 영향 노출 한계
0.04% (400 ppm) 정상 대기 수준 배경
0.5% (5,000 ppm) 호흡 속도 증가 8시간 TWA 한계
1.5% (15,000 ppm) 경미한 호흡 자극 15분 STEL
4% (40,000 ppm) 두통, 어지러움, 혼란 IDLH 수준
10% (100,000 ppm) 수분 내 의식 상실 생명 위협
20%+ (200,000 ppm) 급속한 무의식, 사망 치명적
중요 안전 주의: CO2는 공기보다 무겁고 (밀도 1.98 kg/m³ 대 공기 1.2 kg/m³) 저지대, 트렌치, 밀폐 공간에 축적됩니다. CO2가 축적될 수 있는 모든 구역에서 고정 및 휴대용 CO2 모니터가 필수적입니다.

압력 시스템 안전

탄소 포집 시설에는 특히 CO2 압축 및 운송 시스템에서 고압으로 작동하는 장비가 포함됩니다. 압력 안전은 적절한 설계, 검사 및 릴리프 시스템을 통해 관리됩니다.

// 릴리프 밸브 크기 결정
function sizeReliefValve(
    scenario: string,
    relievingRate: number,      // kg/hr
    setPressure: number,        // barg
    backPressure: number,       // barg
    temperature: number         // °C
): ReliefValveSpec {
    // API 520 크기 결정 방법론
    const molecularWeight = 44.01;  // CO2
    const compressibility = 0.85;

    const requiredArea = relievingRate / (
        51.45 * setPressure * Math.sqrt(molecularWeight / (temperature + 273))
    );

    return {
        scenario,
        orificeSize: selectStandardOrifice(requiredArea),
        setPressure,
        capacity: relievingRate
    };
}

요약

안전은 탄소 포집 운영에서 가장 중요합니다. 위험 식별, 위험 평가, 엔지니어링 안전장치, 모니터링, 비상 대응 및 변경 관리를 포괄하는 포괄적인 안전 관리 시스템은 환경 목표를 달성하면서 시설이 안전하게 운영되도록 보장합니다. WIA-CARBON_CAPTURE 표준은 이러한 안전 요구사항을 탄소 포집 구현의 모든 측면에 통합합니다.
