챕터 1: 탄소 포집의 기초

학습 목표: CO2 분자 특성 이해, 흡수 및 흡착 원리 습득, 멤브레인 분리 기술 학습, 탄소 포집 프로세스의 열역학 이해

이산화탄소: 분자 특성

이산화탄소(CO2)는 하나의 탄소 원자가 두 개의 산소 원자와 이중 결합을 형성한 선형 삼원자 분자입니다. 효과적인 포집 시스템을 설계하려면 물리적, 화학적 특성을 이해하는 것이 필수적입니다. 표준 온도와 압력에서 CO2는 무색, 무취의 기체로 공기보다 밀도가 높습니다.

// CO2 물리적 특성
const CO2Properties = {
    molecularFormula: "CO2",
    molecularWeight: 44.01,      // g/mol
    bondAngle: 180,              // 도 (선형 분자)
    bondLength: 1.16,            // 옹스트롬 (C=O)
    criticalTemperature: 31.1,   // °C
    criticalPressure: 73.8,      // bar
    triplePoint: {
        temperature: -56.6,      // °C
        pressure: 5.18           // bar
    },
    density: {
        gas: 1.977,              // kg/m³ (표준 상태)
        liquid: 770,             // kg/m³ (56 bar, 20°C)
        supercritical: 469       // kg/m³ (임계점)
    },
    solubility: {
        water: 1.45,             // g/L (25°C, 1 bar)
        temperatureEffect: "온도 상승 시 감소"
    }
};

이산화탄소 상 다이어그램

CO2는 포집, 운송 및 저장 응용 분야에 중요한 독특한 상 거동을 보입니다. 대부분의 물질과 달리 CO2는 대기압에서 액체로 존재하지 않고 고체에서 기체로 직접 승화합니다. 상 다이어그램을 이해하면 포집 및 운송 시스템의 최적 설계가 가능합니다.

상 조건 응용
기체 임계점 이하, 저압 초기 포집, 대기 조건
액체 고압, 중간 온도 파이프라인 운송, 중간 저장
초임계 31.1°C 이상, 73.8 bar 이상 지질 저장, 석유회수증진
고체 (드라이아이스) 1 bar에서 -78.5°C 이하 특수 운송, 임시 저장

화학적 흡수 원리

화학적 흡수는 가장 성숙하고 널리 배포된 탄소 포집 기술입니다. CO2와 액체 용매(일반적으로 아민 용액) 간의 화학 반응을 포함합니다. 반응은 가열에 의해 분해될 수 있는 약한 결합 화합물을 형성하여 저장을 위한 순수 CO2를 방출합니다.

// 아민 기반 화학 흡수 반응
// 1차 아민 (예: MEA - 모노에탄올아민)
// CO2 + 2RNH2 ⇌ RNHCOO⁻ + RNH3⁺

const amineCaptureChemistry = {
    primaryAmine: {
        example: "모노에탄올아민 (MEA)",
        formula: "HOCH2CH2NH2",
        reaction: "CO2 + 2MEA ⇌ MEACOO⁻ + MEAH⁺",
        stoichiometry: "MEA 1몰당 CO2 0.5몰",
        regenerationTemp: "120-140°C",
        heatOfReaction: -85  // kJ/mol CO2
    },
    secondaryAmine: {
        example: "디에탄올아민 (DEA)",
        formula: "(HOCH2CH2)2NH",
        stoichiometry: "DEA 1몰당 CO2 0.5몰",
        advantage: "낮은 재생 에너지"
    },
    tertiaryAmine: {
        example: "메틸디에탄올아민 (MDEA)",
        formula: "(HOCH2CH2)2NCH3",
        stoichiometry: "MDEA 1몰당 CO2 1몰",
        mechanism: "염기 촉매 수화",
        advantage: "가장 낮은 재생 에너지"
    }
};
핵심 원리: 아민 용매 선택은 반응 속도, CO2 용량, 재생 에너지, 열화 저항성 간의 트레이드오프를 포함합니다. 혼합 아민 시스템은 종종 1차 아민의 빠른 반응 속도와 3차 아민의 낮은 에너지 요구사항을 결합하여 최적의 성능을 제공합니다.

흡수탑 설계

흡수탑은 화학 흡수 시스템의 핵심입니다. CO2를 포함한 배기 가스는 하부에서 들어가 충전재를 통해 위쪽으로 흐르고, 린 아민 용액은 아래쪽으로 흐릅니다. 이 역류 접촉은 물질 전달 효율을 극대화합니다.

// 흡수탑 설계 매개변수
interface AbsorptionColumn {
    type: "packed" | "tray";
    height: number;        // 미터
    diameter: number;      // 미터
    packingType: string;
    packingHeight: number; // 미터

    operatingConditions: {
        temperature: number;      // °C
        pressure: number;         // bar
        liquidFlowRate: number;   // m³/h
        gasFlowRate: number;      // m³/h
        L_G_ratio: number;        // 액체 대 가스 비율
    };

    performance: {
        captureEfficiency: number;  // 퍼센트
        pressureDrop: number;       // mbar/m
        floodingVelocity: number;   // m/s
    };
}

// 탑 높이 계산 예시
function calculateColumnHeight(
    flueGasFlow: number,      // mol/s
    CO2Inlet: number,         // 몰 분율
    CO2Outlet: number,        // 몰 분율
    solventCapacity: number,  // mol CO2/mol 용매
    HTU: number               // 이동 단위 높이 (m)
): number {
    const NTU = Math.log((CO2Inlet * (1 - CO2Outlet)) /
                         (CO2Outlet * (1 - CO2Inlet)));
    const columnHeight = NTU * HTU;
    return columnHeight;  // 미터
}

물리적 흡착 메커니즘

물리적 흡착(물리흡착)은 CO2 분자를 고체 표면에 결합시키기 위해 약한 반데르발스 힘에 의존합니다. 화학적 흡수와 달리 화학 결합이 형성되지 않아 재생 에너지 요구사항이 낮지만 일반적으로 선택성이 낮습니다.

일반적인 흡착제 소재

  • 활성탄: 높은 표면적 (1000-3500 m²/g), 저비용, 중간 CO2 용량
  • 제올라이트: 분자체를 위한 정의된 기공 크기를 가진 미세다공성 알루미노실리케이트
  • 금속-유기 골격체 (MOFs): 초고 표면적 (최대 7000 m²/g), 조정 가능한 특성
  • 실리카겔: 향상된 선택성을 위한 아민 기능화 실리카
// 흡착 등온선 모델
const adsorptionModels = {
    langmuir: {
        equation: "q = qmax * K * P / (1 + K * P)",
        parameters: {
            qmax: "최대 흡착 용량 (mol/kg)",
            K: "랑뮤어 상수 (1/bar)",
            P: "CO2 분압 (bar)"
        },
        assumptions: [
            "단분자층 흡착",
            "균일한 표면",
            "흡착질 간 상호작용 없음"
        ]
    },
    freundlich: {
        equation: "q = K * P^(1/n)",
        parameters: {
            K: "프로인들리히 상수",
            n: "이질성 계수",
            P: "분압"
        },
        application: "이질적 표면"
    },
    dualSiteLangmuir: {
        equation: "q = q1*K1*P/(1+K1*P) + q2*K2*P/(1+K2*P)",
        application: "여러 흡착 사이트를 가진 MOF 및 제올라이트"
    }
};

// CO2 로딩 계산
function calculateLoading(P_CO2: number, qmax: number, K: number): number {
    return (qmax * K * P_CO2) / (1 + K * P_CO2);  // mol/kg
}

압력 스윙 흡착 (PSA)

PSA는 흡착 친화도의 차이에 기반하여 가스를 분리하는 순환 흡착 프로세스입니다. 고압 흡착과 저압 탈착 단계를 번갈아 가며 운영됩니다.

// PSA 사이클 구성
interface PSACycle {
    steps: {
        adsorption: {
            pressure: number;      // bar
            duration: number;      // 초
            feedFlow: number;      // mol/s
        };
        blowdown: {
            pressure: number;      // bar
            duration: number;      // 초
        };
        purge: {
            pressure: number;      // bar
            duration: number;      // 초
            purgeFlow: number;     // mol/s
        };
        pressurization: {
            targetPressure: number; // bar
            duration: number;       // 초
        };
    };
    numberOfBeds: number;
    cycleTime: number;  // 총 초
}

// 4베드 PSA 시스템 예시
const psaSystem: PSACycle = {
    steps: {
        adsorption: { pressure: 10, duration: 120, feedFlow: 100 },
        blowdown: { pressure: 1, duration: 30 },
        purge: { pressure: 1, duration: 60, purgeFlow: 20 },
        pressurization: { targetPressure: 10, duration: 30 }
    },
    numberOfBeds: 4,
    cycleTime: 240
};

온도 스윙 흡착 (TSA)

TSA는 흡착과 탈착을 구동하기 위해 온도 변화를 사용합니다. 흡착 단계에서는 낮은 온도에서 CO2가 포집됩니다. 재생은 흡착제를 가열하여 포집된 CO2를 방출함으로써 이루어집니다. TSA는 폐열이 이용 가능한 응용 분야에 특히 적합합니다.

에너지 비교: TSA는 일반적으로 흡착제의 열용량으로 인해 PSA보다 더 많은 에너지를 필요로 하지만, 더 높은 CO2 순도를 달성할 수 있으며 폐열이 이용 가능할 때 선호됩니다. PSA와 TSA 요소를 결합한 하이브리드 시스템은 전반적인 효율성을 최적화할 수 있습니다.

멤브레인 분리 기술

멤브레인 기반 탄소 포집은 CO2가 다른 가스보다 더 쉽게 통과하도록 하는 선택적 장벽을 사용합니다. 이 기술은 모듈성, 단순성, 화학 시약 부재 측면에서 장점을 제공합니다.

// 멤브레인 성능 매개변수
interface MembraneSystem {
    material: string;
    permeability: {
        CO2: number;      // Barrer
        N2: number;       // Barrer
    };
    selectivity: number;  // CO2/N2 비율
    thickness: number;    // 마이크로미터
    area: number;         // m²

    operatingConditions: {
        feedPressure: number;       // bar
        permeatePressure: number;   // bar
        temperature: number;        // °C
    };

    performance: {
        recovery: number;           // 퍼센트
        purity: number;             // 퍼센트
        flux: number;               // mol/m²/s
    };
}

// 플럭스 계산
function calculateFlux(
    permeability: number,  // Barrer
    thickness: number,     // 마이크로미터
    deltaPressure: number  // bar
): number {
    // Barrer를 SI 단위로 변환
    const P_SI = permeability * 3.35e-16;  // mol·m/(m²·s·Pa)
    const flux = P_SI * (deltaPressure * 1e5) / (thickness * 1e-6);
    return flux;  // mol/m²/s
}

탄소 포집의 열역학

가스 혼합물에서 CO2를 분리하는 데 필요한 최소 이론적 에너지는 열역학 원리에 의해 결정됩니다. 이러한 한계를 이해하면 포집 기술의 효율성을 평가하고 개선 기회를 식별하는 데 도움이 됩니다.

// 분리의 최소 일
function minimumSeparationWork(
    y_CO2: number,        // 공급에서 CO2 몰 분율
    T: number,            // 온도 (K)
    captureRate: number   // 포집 비율 (0-1)
): number {
    const R = 8.314;  // J/mol/K

    // 포집된 CO2 1몰당 최소 일
    const W_min = R * T * (
        Math.log(1 / y_CO2) +
        ((1 - captureRate) / captureRate) * Math.log(1 / (1 - y_CO2))
    );

    return W_min;  // J/mol CO2
}

// 예시: 석탄 발전소 배기가스
const y_CO2_coal = 0.13;     // 13% CO2
const temperature = 313;      // 40°C = 313K
const captureRate = 0.90;     // 90% 포집

const W_min = minimumSeparationWork(y_CO2_coal, temperature, captureRate);
// 결과: ~8.4 kJ/mol CO2 또는 ~190 kJ/kg CO2

// 실제 프로세스는 비가역성으로 인해 최소값의 3-5배 필요
효율성 벤치마크: 현재 상용 아민 시스템은 약 3-4 GJ/톤 CO2를 필요로 하는 반면, 석탄 배기가스에 대한 이론적 최소값은 약 0.4 GJ/톤입니다. 이 격차는 기술 개선의 상당한 기회를 나타냅니다.
