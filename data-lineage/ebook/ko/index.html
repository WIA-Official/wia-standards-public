📖 데이터 계보

데이터 계보 및 출처 추적 완벽 가이드

이 eBook에 대하여

데이터 계보에 대한 포괄적인 가이드에 오신 것을 환영합니다. 이 eBook은 데이터 출처 추적, 데이터 흐름 이해, 영향 분석 수행, 조직에 계보 솔루션 구현에 필요한 모든 것을 다룹니다.

데이터 엔지니어, 분석가, 아키텍트, 거버넌스 전문가 모두에게 유용한 내용으로, 데이터 계보 모범 사례를 이해하고 구현하는 데 도움을 드립니다.

1장
데이터 계보 소개
데이터 계보가 무엇인지, 왜 중요한지, 현대 데이터 아키텍처에서의 역할
2장
데이터 계보의 유형
컬럼 레벨, 테이블 레벨, 프로세스 레벨 계보 설명
3장
데이터 출처와 추적
시스템 전반에 걸친 데이터 원본, 변환, 이동 캡처
4장
영향 분석
변경의 다운스트림 영향 이해 및 손상 방지
5장
근본 원인 분석
계보를 사용하여 데이터 품질 문제를 소스까지 추적
6장
계보 시각화
데이터 관계 탐색을 위한 직관적인 그래프와 다이어그램 생성
7장
OpenLineage 표준
계보 메타데이터 수집 및 상호 운용성을 위한 개방형 표준
8장
구현 및 모범 사례
조직에서 계보를 구현하기 위한 실용적인 가이드
