Find Friends Across Platforms
=============================

// Search by name
const results = await social.findAcrossPlatforms('Kim Minsu');

Result:
{
  found: true,
  confidence: 0.95,  // How certain we are it's same person
  identity: {
    wia_id: 'wia:kim.minsu.1234',
    displayName: 'Kim Minsu'
  },
  platforms: [
    { platform: 'instagram', handle: '@kim_minsu', following: true },
    { platform: 'twitter', handle: '@minsukim', following: true },
    { platform: 'tiktok', handle: '@minsu_dance', following: false },
    { platform: 'bluesky', handle: '@minsu.bsky.social', following: true }
  ],
  mutualFriends: 15
}


How It Works
============

Matching methods:
  1. Same email linked to accounts
  2. Same phone number
  3. Explicit WIA-SOCIAL link
  4. Bio link analysis
  5. Profile photo matching
  6. Friend network overlap
  7. Username similarity

Privacy respected:
  - Only public information used
  - No unauthorized data access
  - User consent required for linking

5.4 Follow Everywhere

One Click, All Platforms
========================

// Follow on all available platforms
await social.followEverywhere(friendId);

Result:
{
  success: [
    { platform: 'instagram', status: 'following' },
    { platform: 'twitter', status: 'following' },
    { platform: 'bluesky', status: 'following' }
  ],
  skipped: [
    { platform: 'tiktok', reason: 'already_following' }
  ],
  failed: []
}


Selective Follow
================

// Choose specific platforms
await social.follow(friendId, {
  platforms: ['instagram', 'twitter']
});


Unfollow Everywhere
===================

await social.unfollowEverywhere(friendId);

5.8 Lists and Groups

Cross-Platform Lists
====================

// Create a list
await social.createList({
  name: 'Tech Friends',
  description: 'People I know from tech',
  members: [
    'wia:kim.minsu.1234',
    'wia:lee.jiyoung.5678',
    'wia:park.jihoon.9012'
  ],
  privacy: 'private'
});

// List works across all platforms
// See their content unified
const techFeed = await social.getFeed({ list: 'tech_friends' });


Groups
======

// Create cross-platform group for messaging
const group = await social.createGroup({
  name: 'Project Team',
  members: [member1, member2, member3],
  platforms: ['auto']  // Use best available
});

// Message entire group
await social.messageGroup(group.id, 'Meeting tomorrow at 3pm');
