2.1 The Identity Problem

Without Universal Identity
==========================

Platform A: @hong_gildong
Platform B: @honggildong
Platform C: @gildong_hong
Platform D: @hgildong1992

Same person.
Different usernames.
Different passwords.
Different profiles.
Different friend lists.

No one knows they're all you.
You manage them separately.
Your friends see fragmented versions.

This is digital chaos.

The Cost of Fragmentation

Time spent:
  - Updating profile on 5 platforms: 30 min
  - Managing passwords for 10 accounts: constant
  - Telling friends your new handle: endless
  - Rebuilding network after platform dies: months

Trust issues:
  - Which @hong_gildong is the real one?
  - Impersonation is easy
  - Verification needed for each platform

Identity Structure

interface UniversalIdentity {
  // Globally unique ID
  wia_id: 'wia:hong.gildong.1234';

  // Display name
  displayName: string;

  // Profile info
  profile: {
    avatar: MediaRef;
    bio: string;
    links: Link[];
    verifications: Verification[];
  };

  // Connected platforms
  connectedPlatforms: [
    { platform: 'instagram', handle: '@hong_gildong' },
    { platform: 'twitter', handle: '@honggildong' },
    { platform: 'tiktok', handle: '@gildong_hong' }
  ];

  // Privacy settings
  privacy: PrivacySettings;
}

2.3 Creating Your Identity

Step 1: Create WIA Identity
===========================

const identity = await WIASocial.createIdentity({
  displayName: 'Hong Gildong',
  email: 'hong@example.com'
});

// Result:
{
  wia_id: 'wia:hong.gildong.7892',
  displayName: 'Hong Gildong',
  status: 'created',
  verificationLevel: 'basic'
}


Step 2: Connect Platforms
=========================

// Connect Instagram
await social.connectPlatform('instagram', {
  token: instagramAccessToken
});

// Connect Twitter
await social.connectPlatform('twitter', {
  token: twitterAccessToken
});

// Connect TikTok
await social.connectPlatform('tiktok', {
  token: tiktokAccessToken
});


Step 3: Import Existing Data (Optional)
======================================

// Import your history
await social.importFromPlatform('instagram', {
  posts: true,
  followers: true,
  following: true
});

2.4 Profile Sync

One Profile, Everywhere
=======================

Update once, reflect everywhere:

// Update your profile
await social.updateProfile({
  displayName: 'Hong Gildong (Gil)',
  bio: 'Digital explorer | Coffee lover',
  avatar: newAvatarImage
});

// Sync to all platforms
await social.syncProfile({
  platforms: ['instagram', 'twitter', 'bluesky'],
  fields: ['displayName', 'bio', 'avatar']
});

Result:
  [OK] Instagram profile updated
  [OK] Twitter profile updated
  [OK] Bluesky profile updated

No more updating 5 profiles separately!

Platform-Specific Overrides

// Different bio for LinkedIn
await social.setProfileOverride('linkedin', {

// Platform-specific settings
await social.setPlatformSettings('tiktok', {

2.7 Privacy Settings

Identity Privacy Controls
=========================

// Who can see your connected platforms?
await social.setPrivacy({
  showConnectedPlatforms: {
    visibility: 'friends', // 'public' | 'followers' | 'friends' | 'private'
    excludePlatforms: ['linkedin'] // hide specific ones
  },

  // Who can find you?
  discoverability: {
    byEmail: true,
    byPhone: true,
    byUsername: true,
    crossPlatform: true // allow finding you across platforms
  },

  // Profile information
  profile: {
    showBio: 'public',
    showAvatar: 'public',
    showStats: 'followers'
  }
});
