4.1 The Fragmented Feed Problem

Current Reality
===============

Morning routine:
  1. Open Instagram - scroll for 10 min
  2. Open Twitter - scroll for 10 min
  3. Open TikTok - scroll for 15 min
  4. Open Threads - scroll for 5 min
  5. Open Bluesky - scroll for 5 min

Total: 45 minutes
Missed posts: Many (algorithms decide)
Context switching: Exhausting
Important updates: Possibly missed

The problem:
  - Each app shows what IT wants you to see
  - Friends' posts hidden by algorithms
  - No chronological option on most platforms
  - Duplicates across platforms

4.6 Smart Features

Duplicate Detection
===================

When you cross-post, friends might see:
  - Your Instagram post
  - Your Twitter post (same content)
  - Your Threads post (same content)

WIA-SOCIAL detects this:

const feed = await social.getFeed({
  deduplication: {
    enabled: true,
    strategy: 'keep_first', // or 'keep_best_engagement'
    threshold: 0.9  // similarity threshold
  }
});

Result: Only one version shown


Highlight Important
===================

// Posts from close friends ranked higher
await social.setFeedPreferences({
  prioritize: {
    closeFriends: true,
    mutualFollows: true,
    frequentInteractions: true
  },
  deprioritize: {
    promotedContent: true,
    lowEngagement: false
  }
});


Save for Later
==============

// Bookmark across platforms
await social.bookmark(postId);

// Get all bookmarks
const saved = await social.getBookmarks();
// Unified bookmarks from all platforms

4.7 Unified Interactions

React to Posts
==============

// Like/favorite
await social.react(postId, 'like');

// The reaction goes to the original platform
// Instagram: Like
// Twitter: Like
// Bluesky: Like


Comment
=======

// Comment from unified feed
await social.comment(postId, {
  text: 'Great post!'
});

// Comment goes to original platform


Share/Repost
============

// Repost to your connected platforms
await social.repost(postId, {
  targets: ['twitter', 'bluesky'],
  comment: 'Check this out!'
});


View Unified Reactions
======================

const reactions = await social.getReactions(postId);

Result:
{
  total: 150,
  breakdown: {
    instagram: { likes: 100 },
    twitter: { likes: 30, retweets: 10 },
    bluesky: { likes: 10 }
  }
}

4.8 Feed Customization

Create Custom Feeds
===================

// Tech news feed
const techFeed = await social.createCustomFeed({
  name: 'Tech News',
  sources: {
    platforms: ['twitter', 'bluesky'],
    keywords: ['tech', 'AI', 'startup'],
    authors: [techInfluencers]
  }
});

// Close friends feed
const closeFriendsFeed = await social.createCustomFeed({
  name: 'Close Friends',
  sources: {
    lists: ['close_friends'],
    excludeReposts: true
  }
});


Lists
=====

// Create lists that work across platforms
await social.createList({
  name: 'Work Colleagues',
  members: [
    'wia:kim.minsu.1234',
    'wia:lee.jiyoung.5678'
  ]
});

// Feed from list
const workFeed = await social.getFeed({
  list: 'work_colleagues'
});
