7.1 락인 문제

오늘날 플랫폼 락인
=================

인스타그램에서 보낸 10년:
  - 5,000장의 사진
  - 10,000개의 댓글
  - 50,000개의 좋아요
  - 500개의 친구 관계
  - 셀 수 없는 추억

떠나고 싶으면?
  - ZIP 파일 다운로드 (지저분함)
  - 모든 팔로워 잃음
  - 모든 댓글 잃음
  - 다른 곳에서 처음부터
  - 10년이 사라짐

디지털 인질극.


플랫폼 전환 비용
===============

트위터에서 블루스카이로 전환:
  - 블루스카이에서 팔로워 0
  - 블루스카이에서 팔로잉 0
  - 콘텐츠 히스토리 0
  - 수년간의 구축 = 잃음
  - 네트워크 효과 = 함정

// 트위터 데이터만 내보내기

선택적 내보내기
==============

// 특정 데이터 유형 내보내기
const partial = await social.export({
  platforms: ['instagram', 'twitter'],
  include: {
    posts: true,
    media: true,
    followers: false,  // 필요 없음
    messages: false    // 비공개
  },
  dateRange: {
    from: '2024-01-01',
    to: '2024-12-31'
  }
});


표준 포맷
========

모든 내보내기는 WIA-SOCIAL-v1 포맷 사용:
  - 구조화된 데이터는 JSON
  - 미디어는 원본 품질
  - 통합 게시물 포맷
  - 플랫폼 독립적

WIA-SOCIAL 지원하는 어디든 가져오기 가능.

7.4 플랫폼 이주

새 플랫폼으로 이동
=================

// 트위터에서 블루스카이로 이주
const migration = await social.migrate({
  from: 'twitter',
  to: 'bluesky',

  include: {
    posts: true,        // 모든 게시물 이동
    following: true,    // 팔로잉 목록 이동
    profile: true       // 프로필 복사
  },

  options: {
    preserveDates: true,    // 원래 게시 날짜 유지
    findFriends: true,      // 블루스카이에서 트위터 친구 찾기
    postAnnouncement: true  // 팔로워에게 이동 알림
  }
});

결과:
{
  status: 'completed',

  migrated: {
    posts: 3247,
    following: 187,  // 블루스카이에서 찾음
    profile: true
  },

  notFound: {
    following: 333   // 아직 블루스카이에 없음
  },

  announcement: {
    posted: true,
    text: '블루스카이로 이사합니다! @hong.bsky.social에서 만나요'
  }
}


이주 도우미
==========

// "찾지 못함" 친구 나중을 위해 저장
const pending = migration.notFound.following;

// 가입하면 알림
social.on('friend_joined_platform', (event) => {
  if (pending.includes(event.friend) && event.platform === 'bluesky') {
    console.log(`${event.friend.displayName}님이 블루스카이 가입!`);
    // 자동 팔로우 옵션
  }
});

데이터 가져오기

// 내보낸 데이터를 새 플랫폼에 가져오기

결과:

    posts: 2500,
    media: 1800,
    profile: true
  },

    reason: 'video_format_unsupported'
  },

  following: {
    found: 45,      // 마스토돈에서 매칭됨
    followed: 45,   // 자동 팔로우됨
    notFound: 475   // 마스토돈에 없음
  }
}


포맷 변환
========

WIA-SOCIAL이 자동 변환:
  - 게시물 포맷 (스레드 등)
  - 미디어 포맷 (비디오 코덱)
  - 멘션 포맷 (@user)
  - 해시태그 스타일

// 가져오기 전 호환성 확인
const compatibility = await social.checkCompatibility(
  exportedData,
  'mastodon'
);

결과:
{
  compatible: 0.89,  // 89% 호환
  issues: [
    { type: 'video_codec', count: 47, solution: 'transcode' },
    { type: 'poll_format', count: 12, solution: 'convert' }
  ]
}

=====================

7.7 백업 전략

자동 백업
========

// 자동 백업 설정
await social.backup.configure({
  frequency: 'weekly',
  storage: 'local',  // 또는 'cloud', 's3' 등
  path: '/backups/social/',

  include: {
    posts: true,
    media: true,
    messages: true
  },

  retention: {
    keepLast: 12,  // 마지막 12개 백업 유지
    compress: true
  }
});


증분 백업
========

// 마지막 백업 이후 변경만 백업
const incremental = await social.backup.createIncremental();

결과:
{
  type: 'incremental',
  since: '2025-01-08',
  newItems: {
    posts: 15,
    media: 8,
    messages: 45
  },
  size: '125 MB'  // 전체 백업보다 훨씬 작음
}


백업 확인
========

// 백업 무결성 확인
const verification = await social.backup.verify('/backups/latest.zip');

결과:
