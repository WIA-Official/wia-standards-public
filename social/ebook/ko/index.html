WIA 가족:
+-- 아버지 (WIA-INTENT): "의도를 표현해"
+-- 어머니 (WIA-OMNI-API): "내가 다 품어줄게"
+-- 삼촌 (WIA-AIR-POWER): "내가 힘 나눠줄게"
+-- 이모 (WIA-AIR-SHIELD): "내가 지켜줄게"
+-- 조카 (WIA-SOCIAL): "나 통해서 다 연결돼!" <-- 이 책

목차

  1. 제1장: 소개
    • 플랫폼 문제

    • 철학: 연결은 권리
  2. 제2장: 통합 정체성
    • 하나의 ID, 모든 플랫폼
    • 크로스 플랫폼 프로필
    • 인증 레벨
  3. 제3장: 크로스 플랫폼 포스팅
    • 한 번 작성, 모든 곳에 게시
    • 플랫폼 최적화
    • 예약 및 스레드
  4. 제4장: 통합 피드
    • 모든 플랫폼, 하나의 스트림
    • 필터링과 정렬
    • 실시간 업데이트
  5. 제5장: 소셜 그래프
    • 크로스 플랫폼 친구 찾기
    • 통합 관계
    • 모든 곳에서 팔로우
  6. 제6장: 개인정보 보호
    • 이모 (AIR-SHIELD) 연동
    • 개인정보 레벨
    • 게시 전 스캔
  7. 제7장: 데이터 이동성
    • 데이터 내보내기
    • 플랫폼 이주
    • 삭제 권리
  8. 제8장: 구현
    • 코어 API
    • 플랫폼 어댑터
    • 프로토콜 브릿지
