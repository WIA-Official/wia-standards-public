제6장: 개인정보 보호

"홍익인간: 개인정보는 당신의 권리, 이모가 지켜줍니다."

6.1 소셜 미디어 개인정보 위험

일반적인 개인정보 노출
====================

자주 공유하는 것이 드러내는 것:
  - 집 주소 (사진 배경)
  - 일상 루틴 (게시 시간)
  - 직장 (태그된 위치)
  - 전화번호 (실수로 게시)
  - 여행 계획 (실시간 업데이트)
  - 다른 사람 얼굴 (동의 없이)

사진 메타데이터가 드러내는 것:
  - GPS 좌표
  - 기기 정보
  - 날짜와 시간
  - 카메라 설정

집계된 데이터가 드러내는 것:
  - 정치적 견해
  - 건강 상태
  - 재정 상황
  - 관계

6.2 이모 (AIR-SHIELD) 연동

WIA-SOCIAL 개인정보 보호
=======================

WIA-SOCIAL은 이모 (AIR-SHIELD)와 연동:

import { WIASocial } from '@anthropic/wia-social';
import { AirShield } from '@anthropic/wia-air-shield';

const social = new WIASocial({
  privacy: {
    airShield: {
      enabled: true,
      mode: 'balanced'  // 'balanced' | 'fortress' | 'paranoid'
    }
  }
});


보호 모드
========

Balanced (균형):
  - 명백한 위험 경고
  - 기본적으로 메타데이터 제거
  - 개인정보 개선 제안
  - 비방해적

Fortress (요새):
  - 모든 게시 전 능동적 스캔
  - 고위험 콘텐츠 차단
  - 민감 데이터 확인 요구
  - 개인정보 이슈 알림

Paranoid (편집증):
  - 최대 보호
  - 기본적으로 모든 얼굴 블러
  - 모든 위치 퍼징
  - 종단간 암호화 필수
  - 사용성에 영향 가능

6.3 게시 전 개인정보 스캔

자동 스캔
========

// 게시 전 위험 스캔
const scan = await social.privacy.scanContent({
  text: '새 집 보세요! 서울시 강남구 123번지!',
  media: [housePhoto]
});

결과:
{
  safe: false,

  risks: [
    {
      type: 'address_exposure',
      severity: 'high',
      detail: '물리적 주소 감지: 서울시 강남구 123번지',
      recommendation: '주소 제거 또는 일반화'
    },
    {
      type: 'location_in_media',
      severity: 'medium',
      detail: '사진 메타데이터에 GPS 좌표',
      recommendation: '게시 전 메타데이터 제거'
    },
    {
      type: 'identifiable_location',
      severity: 'medium',
      detail: '사진에 집 번호 보임',
      recommendation: '블러 또는 크롭'
    }
  ],

  autoFixes: {
    stripMetadata: true,
    addressRedacted: '[주소]'
  }
}


자동 수정 옵션
=============

// 감지된 이슈 자동 수정
await social.post({
  content: myContent,
  privacy: {
    autoFix: true,  // 권장 수정 자동 적용
    confirmHighRisk: true  // 고위험 콘텐츠 게시 전 확인
  }
});

6.4 개인정보 레벨

공개 설정
========

type PrivacyLevel =
  | 'public'      // 모두에게 공개
  | 'protected'   // 팔로워만
  | 'private'     // 맞팔로우만
  | 'secret'      // 특정인만
  | 'anonymous';  // 익명


기본 개인정보 설정
================

await social.setPrivacy({
  defaultVisibility: 'protected',

  // 플랫폼별 다른 기본값
  platformDefaults: {
    instagram: 'protected',
    twitter: 'public',
    linkedin: 'public'
  },

  // 콘텐츠 유형별 기본값
  contentDefaults: {
    text: 'public',
    photos: 'protected',
    location: 'private'
  }
});


게시물별 개인정보
================

await social.post({
  content: { text: '개인적인 업데이트...' },
  privacy: {
    visibility: 'private',
    closeFriendsOnly: true
  }
});

6.5 메타데이터 보호

자동 메타데이터 제거
==================

// 게시 전 모든 메타데이터 제거
const safeMedia = await social.privacy.stripMetadata(photo);

제거되는 것:
  [OK] GPS 좌표
  [OK] 기기 정보
  [OK] 촬영 날짜/시간
  [OK] 카메라 시리얼 번호
  [OK] 사용된 소프트웨어
  [OK] 다른 데이터를 가진 썸네일
  [OK] 파일의 사용자 코멘트/태그


위치 퍼징
========

// 정확한 위치 대신 대략적으로
const fuzzyLocation = social.privacy.fuzzyLocation(
  exactLocation,
  { radius: '5km' }  // 5km 내에서 무작위화
);

// "서울시 강남구 123번지 4층" 대신
// "강남구" 표시


선택적 제거
==========

await social.privacy.configure({
  media: {
    stripGPS: true,
    stripDevice: true,
    stripDate: false,  // 날짜 유지
    stripAll: false
  }
});

6.8 개인정보 모니터링

실시간 알림
==========

social.on('privacy_alert', (event) => {
  console.log('개인정보 이슈:', event.type);
  console.log('상세:', event.details);
  console.log('권장:', event.recommendation);
});

알림 유형:
  - 누가 당신을 게시물에 태그
  - 위치가 공유됨
  - 새 앱이 데이터 접근 요청
  - 의심스러운 로그인 시도
  - 데이터 유출 알림


개인정보 대시보드
================

const report = await social.privacy.getReport();

결과:
