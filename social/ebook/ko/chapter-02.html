제2장: 통합 정체성

"홍익인간: 하나의 정체성, 한 사람, 모든 플랫폼에서."

2.1 정체성 문제

통합 정체성 없이
===============

플랫폼 A: @hong_gildong
플랫폼 B: @honggildong
플랫폼 C: @gildong_hong
플랫폼 D: @hgildong1992

같은 사람.
다른 아이디.
다른 비밀번호.
다른 프로필.
다른 친구 목록.

아무도 이게 다 당신인지 모름.
따로따로 관리해야 함.
친구들은 파편화된 버전만 봄.

디지털 혼란.

파편화의 비용

소요 시간:
  - 5개 플랫폼 프로필 업데이트: 30분
  - 10개 계정 비밀번호 관리: 끊임없이
  - 친구들에게 새 핸들 알리기: 끝없이
  - 플랫폼 없어지면 네트워크 재구축: 수개월

신뢰 문제:
  - 어떤 @hong_gildong이 진짜?
  - 사칭 쉬움
  - 각 플랫폼마다 인증 필요

2.2 WIA 통합 정체성

하나의 정체성, 모든 플랫폼
=========================

당신의 WIA-SOCIAL 정체성:

+------------------------------------------+
|                                          |
|   WIA ID: wia:hong.gildong.1234         |
|                                          |
|   표시 이름: 홍길동                       |
|   아바타: [사진]                          |
|   바이오: 디지털 탐험가                    |
|                                          |
|   연결된 플랫폼:                          |
|     [OK] 인스타그램: @hong_gildong       |
|     [OK] 트위터: @honggildong            |
|     [OK] 틱톡: @gildong_hong             |
|     [OK] 블루스카이: @hong.bsky.social   |
|                                          |
|   인증: 완료                              |
|                                          |
+------------------------------------------+

하나의 정체성.
하나의 프로필.
업데이트할 곳 하나.
모든 플랫폼 연결.

정체성 구조

interface UniversalIdentity {
  // 전 세계 유일한 ID
  wia_id: 'wia:hong.gildong.1234';

  // 표시 이름
  displayName: string;

  // 프로필 정보
  profile: {
    avatar: MediaRef;
    bio: string;
    links: Link[];
    verifications: Verification[];
  };

  // 연결된 플랫폼
  connectedPlatforms: [
    { platform: 'instagram', handle: '@hong_gildong' },
    { platform: 'twitter', handle: '@honggildong' },
    { platform: 'tiktok', handle: '@gildong_hong' }
  ];

  // 개인정보 설정
  privacy: PrivacySettings;
}

2.3 정체성 만들기

1단계: WIA 정체성 생성
=====================

const identity = await WIASocial.createIdentity({
  displayName: '홍길동',
  email: 'hong@example.com'
});

// 결과:
{
  wia_id: 'wia:hong.gildong.7892',
  displayName: '홍길동',
  status: 'created',
  verificationLevel: 'basic'
}


2단계: 플랫폼 연결
=================

// 인스타그램 연결
await social.connectPlatform('instagram', {
  token: instagramAccessToken
});

// 트위터 연결
await social.connectPlatform('twitter', {
  token: twitterAccessToken
});

// 틱톡 연결
await social.connectPlatform('tiktok', {
  token: tiktokAccessToken
});


3단계: 기존 데이터 가져오기 (선택)
================================

// 히스토리 가져오기
await social.importFromPlatform('instagram', {
  posts: true,
  followers: true,
  following: true
});

2.4 프로필 동기화

하나의 프로필, 어디서든
======================

한 번 업데이트, 모든 곳에 반영:

// 프로필 업데이트
await social.updateProfile({
  displayName: '홍길동 (길)',
  bio: '디지털 탐험가 | 커피 애호가',
  avatar: newAvatarImage
});

// 모든 플랫폼에 동기화
await social.syncProfile({
  platforms: ['instagram', 'twitter', 'bluesky'],
  fields: ['displayName', 'bio', 'avatar']
});

결과:
  [OK] 인스타그램 프로필 업데이트됨
  [OK] 트위터 프로필 업데이트됨
  [OK] 블루스카이 프로필 업데이트됨

5개 프로필 따로 업데이트할 필요 없음!

플랫폼별 오버라이드

// 링크드인은 다른 바이오
await social.setProfileOverride('linkedin', {
  bio: '소프트웨어 엔지니어 | WIA 옹호자'
});

// 플랫폼별 설정
await social.setPlatformSettings('tiktok', {
  bioLink: 'mywebsite.com/tiktok',
  displayName: '길홍'  // 틱톡은 다른 이름
});
