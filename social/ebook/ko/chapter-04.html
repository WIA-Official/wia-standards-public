제4장: 통합 피드

"홍익인간: 모든 연결, 하나의 스트림."

4.1 파편화된 피드 문제

현재 현실
========

아침 루틴:
  1. 인스타그램 열기 - 10분 스크롤
  2. 트위터 열기 - 10분 스크롤
  3. 틱톡 열기 - 15분 스크롤
  4. 스레드 열기 - 5분 스크롤
  5. 블루스카이 열기 - 5분 스크롤

합계: 45분
놓친 게시물: 많음 (알고리즘이 결정)
컨텍스트 전환: 피곤함
중요한 업데이트: 아마 놓침

문제:
  - 각 앱이 보여주고 싶은 것만 보여줌
  - 친구 게시물이 알고리즘에 숨겨짐
  - 대부분 플랫폼에 시간순 옵션 없음
  - 플랫폼 간 중복

4.3 정렬 옵션

시간순
=====

const feed = await social.getFeed({
  sorting: 'chronological'
});

// 정확한 시간 순서로 게시물
// 최신 먼저
// 알고리즘 간섭 없음
// 모두의 모든 것 보기


알고리즘
=======

const feed = await social.getFeed({
  sorting: 'algorithmic'
});

// 기반 스마트 정렬:
// - 참여 히스토리
// - 관계 강도
// - 콘텐츠 관련성
// - 시간 감소

// 플랫폼 알고리즘과 달리:
// - 투명한 랭킹 요소
// - 가중치 제어 가능
// - 광고 콘텐츠 없음


참여도순
=======

const feed = await social.getFeed({
  sorting: 'engagement'
});

// 총 참여도로 정렬:
// - 좋아요 + 댓글 + 공유
// - 모든 플랫폼에서
// - 친구들 사이 트렌딩 보기

4.4 필터링

플랫폼 필터
==========

// 특정 플랫폼만
const feed = await social.getFeed({
  filters: {
    platforms: ['instagram', 'bluesky']
  }
});


콘텐츠 유형 필터
===============

// 사진만
const feed = await social.getFeed({
  filters: {
    contentTypes: ['image']
  }
});

// 영상만
const feed = await social.getFeed({
  filters: {
    contentTypes: ['video']
  }
});


작성자 필터
==========

// 특정 친구만
const feed = await social.getFeed({
  filters: {
    authors: ['wia:kim.minsu.1234', 'wia:lee.jiyoung.5678']
  }
});


복합 필터
========

const feed = await social.getFeed({
  sorting: 'chronological',
  filters: {
    platforms: ['instagram', 'twitter'],
    contentTypes: ['image', 'video'],
    dateRange: {
      from: '2025-01-01',
      to: '2025-01-15'
    },
    excludeReposts: true
  }
});

4.5 실시간 업데이트

피드 구독
========

// 실시간 새 게시물
const unsubscribe = social.subscribeFeed((post) => {
  console.log('새 게시물:', post.author.displayName);
  console.log('플랫폼:', post.platform);
  console.log('내용:', post.content.text);

  // UI 업데이트
  addToFeed(post);
});

// 나중에 구독 해지
unsubscribe();


알림
====

// 특정 이벤트 알림
social.on('feed_updated', (event) => {
  console.log('새 게시물:', event.data.count);
});

social.on('friend_posted', (event) => {
  if (event.data.author.isMutual) {
    showNotification(event.data);
  }
});

4.6 스마트 기능

중복 감지
========

크로스포스트하면 친구가 볼 수 있음:
  - 인스타그램 게시물
  - 트위터 게시물 (같은 내용)
  - 스레드 게시물 (같은 내용)

WIA-SOCIAL이 감지:

const feed = await social.getFeed({
  deduplication: {
    enabled: true,
    strategy: 'keep_first', // 또는 'keep_best_engagement'
    threshold: 0.9  // 유사도 임계값
  }
});

결과: 한 버전만 표시


중요한 것 하이라이트
===================

// 가까운 친구 게시물 우선
await social.setFeedPreferences({
  prioritize: {
    closeFriends: true,
    mutualFollows: true,
    frequentInteractions: true
  },
  deprioritize: {
    promotedContent: true,
    lowEngagement: false
  }
});


나중에 저장
==========

// 플랫폼 간 북마크
await social.bookmark(postId);

// 모든 북마크 가져오기
const saved = await social.getBookmarks();
// 모든 플랫폼의 통합 북마크

4.7 통합 상호작용

게시물 반응
==========

// 좋아요
await social.react(postId, 'like');

// 원본 플랫폼으로 반응 전달
// 인스타그램: 좋아요
// 트위터: 좋아요
// 블루스카이: 좋아요


댓글
====

// 통합 피드에서 댓글
await social.comment(postId, {
  text: '좋은 게시물!'
});

// 원본 플랫폼으로 댓글 전달


공유/리포스트
============

// 연결된 플랫폼에 리포스트
await social.repost(postId, {
  targets: ['twitter', 'bluesky'],
  comment: '이거 봐봐!'
});


통합 반응 보기
=============

const reactions = await social.getReactions(postId);

결과:
{
  total: 150,
  breakdown: {
    instagram: { likes: 100 },
    twitter: { likes: 30, retweets: 10 },
    bluesky: { likes: 10 }
  }
}

4.8 피드 커스터마이징

커스텀 피드 생성
===============

// 테크 뉴스 피드
const techFeed = await social.createCustomFeed({
  name: '테크 뉴스',
  sources: {
    platforms: ['twitter', 'bluesky'],
    keywords: ['tech', 'AI', 'startup'],
    authors: [techInfluencers]
  }
});

// 가까운 친구 피드
const closeFriendsFeed = await social.createCustomFeed({
  name: '가까운 친구',
  sources: {
    lists: ['close_friends'],
    excludeReposts: true
  }
});


리스트
=====

// 플랫폼 간 작동하는 리스트 생성
await social.createList({
  name: '직장 동료',
  members: [
    'wia:kim.minsu.1234',
    'wia:lee.jiyoung.5678'
  ]
});

// 리스트에서 피드
const workFeed = await social.getFeed({
