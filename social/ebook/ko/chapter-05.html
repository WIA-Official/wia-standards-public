작동 방식
========

매칭 방법:
  1. 계정에 연결된 같은 이메일
  2. 같은 전화번호
  3. 명시적 WIA-SOCIAL 링크
  4. 바이오 링크 분석
  5. 프로필 사진 매칭
  6. 친구 네트워크 겹침
  7. 아이디 유사도

개인정보 존중:
  - 공개 정보만 사용
  - 무단 데이터 접근 없음
  - 링크에 사용자 동의 필요

5.4 모든 곳에서 팔로우

한 번 클릭, 모든 플랫폼
=====================

// 가능한 모든 플랫폼에서 팔로우
await social.followEverywhere(friendId);

결과:
{
  success: [
    { platform: 'instagram', status: 'following' },
    { platform: 'twitter', status: 'following' },
    { platform: 'bluesky', status: 'following' }
  ],
  skipped: [
    { platform: 'tiktok', reason: 'already_following' }
  ],
  failed: []
}


선택적 팔로우
============

// 특정 플랫폼 선택
await social.follow(friendId, {
  platforms: ['instagram', 'twitter']
});


모든 곳에서 언팔로우
==================

await social.unfollowEverywhere(friendId);

5.8 리스트와 그룹

크로스 플랫폼 리스트
==================

// 리스트 생성
await social.createList({
  name: '테크 친구들',
  description: '테크에서 아는 사람들',
  members: [
    'wia:kim.minsu.1234',
    'wia:lee.jiyoung.5678',
    'wia:park.jihoon.9012'
  ],
  privacy: 'private'
});

// 리스트는 모든 플랫폼에서 작동
// 콘텐츠 통합 보기
const techFeed = await social.getFeed({ list: 'tech_friends' });


그룹
====

// 메시징을 위한 크로스 플랫폼 그룹 생성
const group = await social.createGroup({
  name: '프로젝트 팀',
  members: [member1, member2, member3],
  platforms: ['auto']  // 최적 플랫폼 사용
});

// 전체 그룹에 메시지
await social.messageGroup(group.id, '내일 3시 미팅');
