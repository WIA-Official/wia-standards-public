## 1. 소개

### 1.1 WIA 외골격 시스템 개요

WIA 재활 외골격은 척수손상, 뇌졸중 등 신경학적 손상 환자의 보행 재활을 지원하는 로봇 보조 장치입니다.

**주요 특징:**
- 양측 고관절, 무릎, 발목 관절 구동
- 적응형 보조 제어 시스템
- 실시간 생체역학적 피드백
- 개인 맞춤형 재활 프로그램

### 1.2 치료사의 역할

치료사는 WIA 외골격을 사용한 재활 훈련에서 핵심적인 역할을 담당합니다:

| 역할 | 설명 |
|-----|------|
| 환자 평가 | 외골격 적합성 및 안전성 평가 |
| 프로그램 설계 | 개인화된 재활 프로그램 수립 |
| 세션 진행 | 훈련 세션 감독 및 안내 |
| 진행 관리 | 진척도 모니터링 및 프로그램 조정 |
| 안전 관리 | 환자 안전 확보 및 비상 대응 |

### 1.3 대상 환자

WIA 외골격은 다음 환자들에게 적용될 수 있습니다:

**적응증:**
- 척수손상 (T4 이하)
- 뇌졸중 후 편마비
- 외상성 뇌손상
- 다발성 경화증
- 불완전 마비

**금기사항:**
- 심한 골다공증
- 조절되지 않는 경직 (Modified Ashworth Scale > 3)
- 활성 욕창 (Stage 2 이상)
- 불안정한 심혈관 상태
- 심한 인지 장애

## 2. 시작하기

### 2.1 시스템 구성

```
WIA 외골격 시스템
├── 외골격 장치 (Exoskeleton Unit)
│   ├── 고관절 구동부 (좌/우)
│   ├── 무릎 구동부 (좌/우)
│   ├── 발목 구동부 (좌/우)
│   └── 배터리 팩
├── 제어 시스템 (Control System)
│   ├── 메인 컨트롤러
│   ├── 태블릿 인터페이스
│   └── 비상 정지 버튼
├── 센서 시스템 (Sensor System)
│   ├── 관절 인코더
│   ├── 힘/토크 센서
│   └── IMU 센서
└── 보조 장비 (Support Equipment)
    ├── 보행 보조기
    ├── 트레드밀 (선택)
    └── 체중 지지 시스템 (선택)
```

### 2.2 시스템 시작

**시작 절차:**

1. **전원 점검**
   - 배터리 잔량 확인 (> 30%)
   - 전원 케이블 연결 상태 확인

2. **시스템 부팅**
   ```
   1. 메인 전원 스위치 ON
   2. 컨트롤러 부팅 대기 (약 30초)
   3. 태블릿 연결 확인
   4. 시스템 자가 진단 완료 확인
   ```

3. **안전 점검**
   - 비상 정지 버튼 기능 테스트
   - 관절 가동 범위 점검
   - 센서 캘리브레이션 확인

### 2.3 환자 착용

**착용 순서:**

```
Step 1: 환자 준비
├── 편안한 운동복 착용 확인
├── 신발 착용 (미끄럼 방지)
└── 보호대/패드 장착 (필요 시)

Step 2: 외골격 장착
├── 골반 프레임 고정
├── 대퇴부 커프 장착 (좌/우)
├── 하퇴부 커프 장착 (좌/우)
└── 발판 고정

Step 3: 정렬 확인
├── 고관절 축 정렬 확인
├── 무릎 축 정렬 확인
└── 발목 정렬 확인

Step 4: 최종 점검
├── 모든 스트랩 고정 확인
├── 압박 부위 점검
└── 환자 편안함 확인
```

## 3. 환자 평가

### 3.1 초기 평가

새로운 환자는 외골격 훈련 시작 전 종합적인 평가가 필요합니다.

**필수 평가 항목:**

| 평가 영역 | 평가 도구 | 기준 |
|----------|----------|------|
| 기립 균형 | Standing Balance Test | 최소 30초 기립 가능 |
| 체간 조절 | Trunk Impairment Scale | 최소 점수 10/23 |
| 상지 기능 | 기능 평가 | 보조 기구 파지 가능 |
| 관절 가동범위 | Goniometry | 고관절/무릎 굴곡 90° 이상 |
| 경직 | Modified Ashworth Scale | MAS ≤ 3 |
| 인지 기능 | MMSE 또는 MoCA | 기본 지시 수행 가능 |
| 심폐 기능 | 6MWT (가능 시) | 의사 승인 필요 |

### 3.2 적합성 체크리스트

```
□ 신체 조건
  □ 신장: 150-190 cm
  □ 체중: 40-100 kg
  □ BMI: 18.5-35

□ 의학적 조건
  □ 의사 승인서
  □ 금기 사항 없음
  □ 활력 징후 안정

□ 기능적 조건
  □ 지시 수행 능력
  □ 최소 기립 가능
  □ 비상 정지 이해

□ 환경적 조건
  □ 보호자/가족 동의
  □ 정기적 참여 가능
  □ 교통편 확보
```

### 3.3 기준선 평가

훈련 시작 전 다음 평가를 실시하여 기준선을 설정합니다:

**표준화된 평가:**

1. **10미터 보행 검사 (10MWT)**
   - 자가 선택 속도
   - 빠른 속도
   - 보조 기구 기록

2. **6분 보행 검사 (6MWT)**
   - 총 거리
   - 활력 징후 (전/후)
   - 휴식 횟수

3. **Timed Up and Go (TUG)**
   - 완료 시간
   - 보조 수준

4. **Berg Balance Scale**
   - 14개 항목 평가
   - 낙상 위험도 판정

### 3.4 목표 설정

환자와 함께 SMART 목표를 설정합니다:

```
SMART 목표 예시:

S (Specific - 구체적)
  "6분 보행 거리를 현재 50m에서 100m로 증가"

M (Measurable - 측정 가능)
  "10미터 보행 속도 0.5 m/s 달성"

A (Achievable - 달성 가능)
  환자의 현재 수준과 잠재력 고려

R (Relevant - 관련성)
  "집안에서 독립적으로 이동하기 위해"

T (Time-bound - 시간 제한)
  "8주 프로그램 완료 시점까지"
```

## 4. 프로그램 선택 및 설정

### 4.1 표준 프로그램

WIA 시스템은 다음과 같은 표준 프로그램을 제공합니다:

| 프로그램 | 대상 | 기간 | 주당 세션 |
|---------|-----|------|----------|
| SCI 초급 | 척수손상 초보자 | 12주 | 3회 |
| SCI 중급 | 척수손상 경험자 | 8주 | 4회 |
| 뇌졸중 초급 | 뇌졸중 초보자 | 10주 | 3회 |
| 뇌졸중 중급 | 뇌졸중 경험자 | 8주 | 4회 |
| 맞춤형 | 개별 설계 | 가변 | 가변 |

### 4.2 프로그램 구조

**표준 프로그램 구조 (SCI 초급 예시):**

```
Phase 1: 기립 적응 (1-3주)
├── 목표: 30분 기립 유지
├── 보조 수준: 90-100%
└── 주요 활동: 기립, 체중 이동

Phase 2: 체중 이동 (4-6주)
├── 목표: 60:40 체중 이동
├── 보조 수준: 80-95%
└── 주요 활동: 측방 이동, 전후 이동

Phase 3: 트레드밀 보행 (7-9주)
├── 목표: 15분 연속 보행
├── 보조 수준: 60-85%
└── 주요 활동: 트레드밀 보행

Phase 4: 지상 보행 (10-12주)
├── 목표: 100m 지상 보행
├── 보조 수준: 40-70%
└── 주요 활동: 실내/실외 보행
```

### 4.3 프로그램 개인화

환자별로 프로그램을 조정할 때 고려사항:

**조정 가능한 파라미터:**

| 파라미터 | 범위 | 조정 기준 |
|---------|-----|----------|
| 보조 수준 | 0-100% | 환자 능력에 따라 |
| 보행 속도 | 0.1-1.0 m/s | 점진적 증가 |
| 세션 시간 | 30-90분 | 체력/피로도 |
| 활동 종류 | 다양 | 목표에 따라 |
| 휴식 시간 | 30-300초 | 회복 필요 시 |

### 4.4 진행 기준

다음 단계로 진행하기 위한 기준:

```
자동 진행 조건:
├── 목표 메트릭 달성 (2회 연속 세션)
├── 피로도 < 7/10
├── 통증 < 5/10
└── 보조 수준 목표 달성

수동 진행 조건:
├── 치료사 임상 판단
├── 표준화 평가 개선
└── 환자 준비도 확인
```

## 5. 세션 진행

### 5.1 세션 전 준비

**체크리스트:**

```
환자 준비
□ 건강 상태 확인 (통증, 피로, 수면)
□ 활력 징후 측정
  - 혈압: ___/___mmHg
  - 심박수: ___bpm
  - 산소포화도: ___%
□ 금기 증상 확인
□ 피부 상태 점검

장비 준비
□ 시스템 자가 진단 통과
□ 배터리 충전 확인 (>30%)
□ 비상 정지 기능 테스트
□ 센서 캘리브레이션

환경 준비
□ 적정 온도 (18-28°C)
□ 장애물 제거
□ 보조 장비 준비
```

### 5.2 표준 세션 구성

**60분 세션 예시:**

```
시간        활동                 내용
─────────────────────────────────────────
0-10분     워밍업              수동/능동 관절 운동
                               스트레칭
                               체중 이동 연습

10-45분    주요 활동           기립 훈련
                               보행 훈련
                               균형 훈련

45-55분    쿨다운              느린 보행
                               스트레칭
                               호흡 운동

55-60분    마무리              피드백 수집
                               다음 세션 안내
                               기록 작성
```

### 5.3 활동별 가이드

#### 5.3.1 기립 훈련

**목적:** 기립 내성 향상, 심혈관 적응

**진행 방법:**
1. 앉은 자세에서 시작
2. "기립" 명령으로 천천히 일어나기
3. 정렬 확인 및 유지
4. 목표 시간까지 기립 유지
5. 어지럼증 모니터링

**주의사항:**
- 기립성 저혈압 주의 (특히 SCI 환자)
- 혈압 변화 모니터링
- 어지럼증 시 즉시 착석

#### 5.3.2 체중 이동 훈련

**목적:** 균형 향상, 보행 준비

**진행 방법:**
1. 안정된 기립 자세에서 시작
2. 좌우 체중 이동 연습
3. 전후 체중 이동 연습
4. 시각적 피드백 활용
5. 점진적으로 이동 범위 확대

**언어 지시 예시:**
- "왼쪽 다리로 체중을 옮기세요"
- "목표선까지 이동해 보세요"
- "천천히, 균형을 유지하면서"

#### 5.3.3 트레드밀 보행

**목적:** 반복적 보행 패턴 학습

**진행 방법:**
1. 트레드밀에 안전하게 위치
2. 체중 지지 시스템 연결 (필요 시)
3. 낮은 속도로 시작
4. 보행 패턴 피드백 제공
5. 점진적 속도/시간 증가

**속도 가이드라인:**
| 수준 | 속도 (m/s) | 적용 |
|-----|-----------|------|
| 매우 느림 | 0.1-0.2 | 초기 적응 |
| 느림 | 0.2-0.4 | 패턴 학습 |
| 보통 | 0.4-0.6 | 일반 훈련 |
| 빠름 | 0.6-0.8 | 속도 훈련 |
| 매우 빠름 | >0.8 | 고급 훈련 |

#### 5.3.4 지상 보행

**목적:** 실제 환경 보행 능력 향상

**진행 방법:**
1. 평평한 실내 환경에서 시작
2. 보조 기구 사용 (필요 시)
3. 직선 보행 → 방향 전환 → 장애물 통과
4. 점진적으로 환경 복잡성 증가

**환경 난이도:**
```
Level 1: 실내 평지 (병원 복도)
Level 2: 실내 카펫/타일 변화
Level 3: 실외 평지
Level 4: 경사로
Level 5: 계단 (보조 필요)
```

### 5.4 실시간 모니터링

**모니터링 항목:**

| 항목 | 정상 범위 | 경고 | 위험 |
|-----|----------|------|------|
| 심박수 | 60-100 | 100-130 | >130 |
| 혈압 (수축기) | 90-140 | 140-160 | >160 |
| 산소포화도 | >95% | 92-95% | <92% |
| 피로도 (RPE) | 11-13 | 14-16 | >16 |
| 통증 | 0-3 | 4-5 | >5 |

### 5.5 피드백 제공

**효과적인 피드백 원칙:**

1. **즉시성**: 동작 직후 피드백 제공
2. **구체성**: "좋아요" 대신 "발뒤꿈치 접촉이 잘 됩니다"
3. **긍정성**: 개선점과 함께 잘한 점 강조
4. **적절성**: 과도한 피드백 지양

**피드백 유형:**

```
언어 피드백 (Verbal)
├── 지시: "발을 조금 더 앞으로"
├── 격려: "잘하고 있어요"
└── 교정: "무릎을 조금 더 펴세요"

시각 피드백 (Visual)
├── 거울
├── 화면 디스플레이
└── 목표 마커

촉각 피드백 (Tactile)
├── 가벼운 터치 가이드
└── 진동 피드백 (장치)
```

## 6. 진척도 모니터링

### 6.1 세션별 기록

매 세션 후 다음 항목을 기록합니다:

```
세션 기록 양식

날짜: _____________
세션 번호: ___

활력 징후:
- 혈압(전): ___/___  (후): ___/___
- 심박수(전): ___    (후): ___

훈련 내용:
- 총 훈련 시간: ___ 분
- 보행 거리: ___ m
- 보행 시간: ___ 분
- 보조 수준: ___ %

환자 보고:
- 통증: ___/10
- 피로: ___/10
- 만족도: ___/10

특이사항/메모:
_______________________
_______________________
```

### 6.2 주간 평가

주 1회 다음 지표를 검토합니다:

| 지표 | 측정 방법 | 목표 |
|-----|----------|------|
| 보행 속도 | 10MWT | 주당 0.05 m/s 증가 |
| 보행 거리 | 세션 데이터 합계 | 주당 10% 증가 |
| 보조 수준 | 평균 보조 % | 주당 5% 감소 |
| 대칭성 | 보행 분석 | >0.8 유지 |
| 출석률 | 세션 참여 | >80% |

### 6.3 정기 재평가

**평가 일정:**

| 시점 | 평가 내용 |
|-----|----------|
| 매 2주 | 10MWT, 보행 대칭성 |
| 매 4주 | 10MWT, 6MWT, TUG, Berg |
| 프로그램 종료 | 전체 평가 + 기능적 목표 달성 |

### 6.4 진척도 리포트

**리포트 구성:**

```
진척도 리포트

1. 환자 정보
   - 이름, 진단, 프로그램

2. 요약
   - 전체 진행률
   - 주요 성취
   - 개선 필요 영역

3. 메트릭 상세
   - 보행 속도 변화 그래프
   - 보행 거리 트렌드
   - 보조 수준 감소율

4. 표준화 평가 결과
   - 기준선 vs 현재
   - 임상적 의미 있는 변화 여부

5. 권고사항
   - 프로그램 조정 제안
   - 다음 단계 계획
```

## 7. 안전 지침

### 7.1 비상 정지 절차

**E-Stop 사용 상황:**
- 환자가 균형을 잃었을 때
- 장비 오작동 감지 시
- 환자가 심한 통증/불편 호소 시
- 이상 활력 징후 발생 시
- 환자 요청 시

**비상 정지 버튼 위치:**
1. 외골격 허리 벨트 (환자용)
2. 컨트롤러 (치료사용)
3. 벽면 (고정식)

**비상 정지 후 절차:**
```
1. 환자 상태 확인
   └── 의식, 호흡, 맥박

2. 상황 평가
   └── 부상 여부, 원인 파악

3. 필요시 응급 조치
   └── 환자 해방, 응급 서비스 연락

4. 문서화
   └── 사고 보고서 작성
```

### 7.2 낙상 예방

**낙상 위험 요인:**
- 피로/근력 저하
- 어지럼증
- 주의력 저하
- 환경적 장애물
- 부적절한 속도

**예방 전략:**
1. 항상 가드벨트 착용
2. 보조자 근접 대기
3. 적절한 휴식 제공
4. 환경 정리
5. 점진적 난이도 증가

### 7.3 의학적 응급상황

#### 기립성 저혈압
```
증상: 어지럼증, 시야 흐림, 식은땀
조치:
1. 즉시 착석/눕힘
2. 다리 올려줌
3. 활력 징후 모니터링
4. 회복까지 대기
```

#### 자율신경 반사이상 (SCI 환자)
```
증상: 두통, 발한, 안면 홍조, 혈압 상승
조치:
1. 즉시 앉은 자세로 (머리 높게)
2. 원인 제거 (방광, 피부 압박 등)
3. 혈압 모니터링
4. 필요시 의료진 호출
```

#### 심폐 관련
```
증상: 흉통, 호흡곤란, 청색증
조치:
1. 훈련 중단
2. 환자 편한 자세
3. 응급 서비스 호출 (119)
4. 필요시 CPR
```

### 7.4 피부 관리

**압박 부위 점검:**
- 골반 크레스트
- 대퇴부 내/외측
- 슬개골 주변
- 종아리
- 복사뼈

**피부 문제 대응:**

| 상태 | 조치 |
|-----|------|
| 발적 (30분 내 소실) | 관찰, 패딩 조정 |
| 발적 (30분 이상 지속) | 휴식, 핏팅 재조정 |
| 물집/수포 | 훈련 중단, 치료 |
| 열린 상처 | 훈련 금지, 상처 관리 |

## 8. 문제 해결

### 8.1 장비 문제

| 문제 | 가능한 원인 | 해결 방법 |
|-----|------------|----------|
| 시스템 부팅 안됨 | 배터리 방전 | 충전 또는 교체 |
| 관절 움직임 안됨 | 비상정지 활성화 | E-Stop 해제 |
| 센서 오류 | 캘리브레이션 필요 | 재캘리브레이션 |
| 불규칙한 움직임 | 연결 불량 | 케이블 점검 |
| 과열 경고 | 연속 사용 | 냉각 대기 |

### 8.2 환자 관련 문제

| 문제 | 접근 방법 |
|-----|----------|
| 공포/불안 | 천천히 진행, 안심시키기, 점진적 노출 |
| 동기 저하 | 목표 재설정, 성취 강조, 가족 참여 |
| 피로 누적 | 세션 조정, 휴식 증가, 영양/수면 확인 |
| 통증 증가 | 원인 평가, 강도 조절, 의료진 상담 |
| 진전 정체 | 프로그램 수정, 새로운 목표, 평가 재실시 |

### 8.3 환경 문제

| 문제 | 해결 방법 |
|-----|----------|
| 공간 부족 | 장애물 제거, 대체 장소 사용 |
| 바닥 미끄러움 | 미끄럼 방지 매트, 청소 |
| 조명 불량 | 조명 조절, 창문 블라인드 |
| 소음 | 조용한 시간대 선택 |

### 9.1 용어집

| 용어 | 영문 | 설명 |
|-----|-----|------|
