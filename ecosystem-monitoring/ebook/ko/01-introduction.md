## 1.1 생물다양성 및 기후 위기

### 1.1.1 전 지구적 생물다양성 현황

지구는 여섯 번째 대멸종 사건을 겪고 있지만, 자연 현상으로 인한 이전 멸종과 달리 이번 멸종은 인간 활동에 의해 주도되고 있습니다:

```
전 지구적 생물다양성 추세:
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  야생동물 개체군                                                        │
│  └─ 1970년 이후 평균 69% 감소 (Living Planet Index)                    │
│  └─ 담수종: 83% 감소                                                  │
│  └─ 라틴아메리카/카리브해: 94% 감소                                      │
│                                                                     │
│  종 멸종 위험                                                          │
│  └─ 100만 종이 멸종 위험에 처함                                         │
│  └─ 포유류의 25%가 위험                                                │
│  └─ 조류의 14%가 위험                                                  │
│  └─ 양서류의 40%가 위험                                                │
│                                                                     │
│  서식지 손실                                                           │
│  └─ 육상 환경의 75%가 변화됨                                           │
│  └─ 해양 환경의 66%가 영향받음                                          │
│  └─ 1700년 이후 습지의 85%가 손실됨                                     │
│                                                                     │
│  기후 변화                                                             │
│  └─ 산업화 이전 대비 1.1°C 상승                                         │
│  └─ 종 분포가 10년당 16.9km 이동                                        │
│  └─ 2050년까지 산호초의 70-90%가 위험                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**출처:** IPBES 전 지구 평가 2019, WWF Living Planet Report 2022

### 1.1.3 모니터링의 중요성

효과적인 보전 및 환경 관리는 강력한 모니터링을 필요로 합니다:

| 보전 목표 | 모니터링 필요성 | 사례 |
|-------------------|-----------------|---------|
| **상태 평가** | 개체군 크기, 분포 | 적색목록 평가 |
| **추세 감지** | 풍부도의 시계열 | 조류 현황 보고서 |
| **위협 식별** | 서식지 손실, 오염 수준 | 수질 모니터링 |
| **개입 평가** | 전후 비교 | 보호구역 효과성 |
| **변화 예측** | 조기 경보 지표 | 기후변화 취약성 |
| **정책 정보 제공** | 의사결정을 위한 증거 | 멸종위기종법 목록 작성 |

## 1.2 생태계 모니터링이란?

### 1.2.1 정의와 범위

**생태계 모니터링**은 변화를 감지하고, 추세를 이해하며, 관리 결정에 정보를 제공하기 위해 시간 경과에 따른 환경 및 생물학적 데이터를 체계적으로 수집, 분석 및 해석하는 것입니다.

```
생태계 모니터링 구성요소:
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  생물학적 모니터링                                                      │
│  ├─ 종 관찰 (출현, 부재, 풍부도)                                        │
│  ├─ 개체군 동태 (출생, 사망, 이동)                                      │
│  ├─ 군집 구성 (다양성, 구조)                                           │
│  ├─ 생물계절 (생활사 사건의 시기)                                       │
│  └─ 종 상호작용 (포식, 수분)                                           │
│                                                                     │
│  물리적 모니터링                                                        │
│  ├─ 기후 (온도, 강수량, 습도)                                          │
│  ├─ 수문 (수위, 유량)                                                 │
│  ├─ 토양 특성 (수분, 온도, 영양염)                                      │
│  └─ 지형 (고도, 경사, 방향)                                            │
│                                                                     │
│  화학적 모니터링                                                        │
│  ├─ 수질 (영양염, pH, 용존산소)                                        │
│  ├─ 대기질 (오염물질, 온실가스)                                         │
│  ├─ 토양 화학 (pH, 영양염, 오염물질)                                    │
│  └─ 독소 수준 (살충제, 중금속)                                         │
│                                                                     │
│  원격 감지                                                             │
│  ├─ 토지 피복 분류                                                     │
│  ├─ 식생 지수 (NDVI, EVI)                                            │
│  ├─ 지표 온도                                                         │
│  └─ 변화 감지                                                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 1.3 현재 모니터링 데이터 상태

### 1.3.1 데이터 볼륨 및 증가

생태 및 환경 데이터의 양은 기하급수적으로 증가하고 있습니다:

| 데이터 소스 | 연간 기록 수 | 증가율 | 플랫폼 예시 |
|-------------|--------------|-------------|------------------|
| GBIF (생물다양성) | 1억+ | 연 10-15% | Global Biodiversity Information Facility |
| eBird (조류) | 5천만+ | 연 15-20% | Cornell Lab of Ornithology |
| iNaturalist | 3천만+ | 연 30% | 시민과학 플랫폼 |
| NEON (미국 센서) | 수십억 개의 판독값 | 지속적 | National Ecological Observatory Network |
| 위성 이미지 | 일일 페타바이트 | 지수적 | Landsat, Sentinel, Planet |
| 카메라 트랩 | 1억+ 이미지 | 연 20% | Wildlife Insights |

**사례 연구: Hubbard Brook Experimental Forest**
- 설립: 1963년 (62년간 연속 데이터)
- 주요 발견:
  - 산림에 대한 산성비 영향 문서화
  - 토양의 장기 칼슘 고갈 발견
  - 오염 감소 후 지연된 회복 확인
  - 예상치 못한 질소 포화 발견
- 영향: 대기정화법 개정에 영향

**사례 연구: Serengeti Ecosystem Monitoring**
- 설립: 1960년대
- 주요 발견:
  - 누 개체군 회복 문서화 (25만 마리 → 130만 마리)
  - 생태계에 대한 연쇄 효과 발견
  - 이동 시기의 기후 동인 식별
  - 신흥 질병 위협 감지
- 영향: 세계유산 관리 모델

## 1.4 데이터 통합 과제

### 1.4.1 파편화된 모니터링 환경

현재 생태계 모니터링은 심각한 파편화를 겪고 있습니다:

```
파편화된 모니터링 생태계:
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  정부 기관                                                             │
│  ├─ 국립공원 모니터링 (NPS 형식)                                        │
│  ├─ 야생동물 조사 (주 어업/수렵 DB)                                     │
│  ├─ 수질 (EPA STORET)                                                │
│  ├─ 대기질 (EPA AQS)                                                 │
│  └─ 기상 데이터 (NOAA 형식)                                            │
│      ⇩ (호환되지 않는 형식)                                            │
│                                                                     │
│  연구 기관                                                             │
│  ├─ 장기 생태 연구 (사용자 정의 스키마)                                  │
│  ├─ 대학 현장 연구소 (Excel 스프레드시트)                                │
│  ├─ 개별 PI 프로젝트 (다양한 형식)                                      │
│  └─ 센서 네트워크 (독점 형식)                                           │
│      ⇩ (데이터 사일로)                                                │
│                                                                     │
│  NGO 보전 단체                                                         │
│  ├─ Nature Conservancy (내부 DB)                                     │
│  ├─ Audubon (별도 조류 데이터)                                         │
│  ├─ WWF (프로젝트별 데이터)                                            │
│  └─ 지역 토지 신탁 (다양한 시스템)                                      │
│      ⇩ (제한된 공유)                                                  │
│                                                                     │
│  시민 과학                                                             │
│  ├─ iNaturalist (자체 데이터 모델)                                     │
│  ├─ eBird (조류 특화 스키마)                                           │
│  ├─ Project BudBurst (생물계절 데이터)                                │
│  └─ 수백 개의 소규모 프로젝트                                           │
│      ⇩ (일관성 없는 품질)                                              │
│                                                                     │
│  결과: 소스 간 집계, 비교 또는 종합 불가능                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.4.2 파편화의 결과

| 문제 | 영향 | 예시 |
|---------|--------|---------|
| **호환되지 않는 형식** | 데이터 결합 불가 | 조류 조사는 개체수 사용, 카메라 트랩은 감지율 사용 |
| **일관성 없는 분류체계** | 같은 종, 다른 이름 | *Puma concolor* vs. *Felis concolor* vs. "퓨마" |
| **메타데이터 누락** | 품질 평가나 해석 불가 | 조사에 "10 관찰"이라고만 - 무엇을? 어디서? 어떻게? |
| **독점 시스템** | 데이터가 공급업체 시스템에 잠김 | GIS 소프트웨어 특화 형식, 라이선스 필요 |
| **표준화되지 않은 단위** | 측정값 비교 불가 | 화씨 vs. 섭씨, 피트 vs. 미터 |
| **품질 플래그 없음** | 좋은 데이터와 나쁜 데이터 혼재 | 이상치, 센서 오작동, 오동정 |

### 1.5.1 데이터 표준의 이점

```

│  ├─ 사전 구축된 도구로 사용자 정의 개발 감소                              │
│  ├─ 데이터가 사용되고 인용될 가능성 증가                                  │

│  └─ 데이터셋 간 작동하는 도구 구축                                       │
│                                                                     │
│  보전 관리자를 위해                                                     │
│  ├─ 여러 소스를 집계하는 실시간 대시보드                                  │
│  ├─ 위협에 대한 조기 경보 시스템                                         │
│  ├─ 증거 기반 적응 관리                                                 │
│  ├─ 규제 기관에 자동 보고                                               │
│  └─ 장기 추세 분석                                                     │
│                                                                     │
│  정책 입안자를 위해                                                     │
│  ├─ 포괄적인 환경 상태 보고                                             │
│  ├─ 보전 목표에 대한 진행 상황 추적                                      │

│  ├─ 공중에 대한 책임성 입증                                             │
│  └─ 국제 조약 의무 지원                                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.5.2 기존 표준과 격차

여러 생태계 모니터링 표준이 존재하지만 상당한 격차가 있습니다:

| 표준 | 범위 | 강점 | 한계 |
|----------|-------|-----------|-------------|
| **Darwin Core** | 생물다양성 관찰 | 널리 채택됨, GBIF 백본 | 출현에 초점, 제한된 환경 데이터 |
| **EML** | 생태 메타데이터 | 포괄적, 복잡한 설계 지원 | 장황함, 가파른 학습 곡선 |
| **NetCDF-CF** | 기후/센서 데이터 | 시계열에 효율적 | 생물다양성용으로 설계되지 않음 |
| **ISO 19115** | 지리공간 메타데이터 | 철저함, 국제 표준 | 일반적, 생태학 특화되지 않음 |
| **QUDT** | 측정 단위 | 정확한 단위 정의 | 생태 표준과 통합되지 않음 |
| **ENVO** | 환경 온톨로지 | 서식지를 위한 통제 어휘 | 채택이 여전히 제한적 |

**격차:** FAIR 원칙을 따르는 생물다양성 관찰, 센서 측정, 수질/대기질을 통합하는 통합 표준이 없음.

### 1.6.2 WIA 표준의 범위

WIA 생태계 모니터링 표준은 다음을 다룹니다:

1. **데이터 형식** (1단계)
   - 관찰 및 측정을 위한 JSON 스키마
   - 통제 어휘
   - 검증 규칙

2. **API 인터페이스** (2단계)
   - 데이터 접근을 위한 RESTful 엔드포인트
   - 실시간 스트리밍 프로토콜
   - 대량 데이터 다운로드

3. **프로토콜** (3단계)
   - 품질 보증 절차
   - 보정 요구사항
   - 현장 샘플링 프로토콜

4. **시스템 통합** (4단계)
   - GIS 플랫폼 연결
   - 보전 데이터베이스 연결
   - 클라우드 플랫폼 배포

## 1.8 핵심 요점

| 주제 | 핵심 사항 |
|-------|------------|
| 생물다양성 위기 | 1970년 이후 야생동물 69% 감소; 100만 종 위협 |
| 모니터링 목적 | 상태 평가, 추세 감지, 위협 식별, 개입 평가 |
| 데이터 증가 | 연간 1억+ 생물다양성 기록; 기하급수적 센서 데이터 |
| 파편화 문제 | 호환되지 않는 형식, 누락된 메타데이터, 사일로화된 시스템 |
| 통합 가치 | 종합, 학제간 연구, 증거 기반 정책 가능 |
| WIA 표준 | 포괄적, 상호운용 가능, FAIR 준수 모니터링 프레임워크 |

### 기억해야 할 중요한 통계
- **69%**: 1970년 이후 평균 야생동물 개체군 감소
- **100만**: 멸종 위험에 처한 종
- **1억+**: GBIF에 연간 추가되는 생물다양성 관찰 기록
- **75%**: 인간에 의해 크게 변화된 육상 환경
- **85%**: 1700년 이후 손실된 습지

### 다음 장 미리보기

2장에서는 WIA 표준이 다루는 데이터 사일로, 호환되지 않는 시스템, 품질 관리 문제 및 통합 장벽을 포함하여 생태계 모니터링의 현재 과제를 자세히 살펴봅니다.
