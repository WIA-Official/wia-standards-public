7.1 Post-Quantum Cryptography

Quantum computers threaten current asymmetric encryption. NIST has selected post-quantum algorithms:

Algorithm Type FIPS Standard
ML-KEM (Kyber) Key Encapsulation FIPS 203
ML-DSA (Dilithium) Digital Signatures FIPS 204
SLH-DSA (SPHINCS+) Stateless Signatures FIPS 205

Migration Timeline

Begin planning post-quantum migration now. Standards finalized 2024, widespread adoption expected by 2030. Sensitive data encrypted today may be harvested for future quantum decryption.

7.2 Zero-Knowledge Proofs

Prove you know something without revealing what you know:

// Zero-Knowledge Proof Concept
// Alice proves she knows x such that hash(x) = H
// WITHOUT revealing x

1. Alice generates commitment: c = commit(x, r)
2. Verifier sends random challenge: e
3. Alice sends response based on e
4. Verifier checks response is valid for c

// If valid: Alice knows x
// Verifier learns: nothing about x

7.2.1 Applications

  • Privacy-preserving auth: Prove identity without revealing credentials
  • Anonymous credentials: Prove attributes without full identity
  • Blockchain privacy: Private transactions (Zcash)
  • Voting systems: Verify vote without revealing choice

7.3 Homomorphic Encryption

Compute on encrypted data without decrypting:

// Homomorphic Encryption Concept
let encrypted1 = encrypt(10);  // E(10)
let encrypted2 = encrypt(20);  // E(20)

// Add encrypted numbers
let encryptedSum = add(encrypted1, encrypted2);  // E(30)

// Decrypt to get result
let result = decrypt(encryptedSum);  // 30
// Cloud never sees the plaintext numbers!

7.3.1 Types

Type Operations Performance
Partially HE Add OR multiply Fast
Somewhat HE Limited operations Medium
Fully HE Any computation Slow (improving)

7.4 Secure Multi-Party Computation

Multiple parties compute a function without revealing their inputs:

  • Example: Calculate average salary without revealing individual salaries
  • Technique: Shamir's Secret Sharing, garbled circuits
  • Use Cases: Private auctions, secure voting, collaborative analytics

7.5 Quantum Key Distribution

Use quantum mechanics for theoretically unbreakable key exchange:

  • Principle: Measuring quantum states disturbs them
  • Detection: Eavesdropping is detectable
  • Limitation: Requires special hardware, short distances
  • Status: Commercially available, limited deployment

7.6 Chapter Summary

Key Takeaways

  • Post-quantum algorithms (Kyber, Dilithium) are standardized
  • Zero-knowledge proofs enable privacy-preserving verification
  • Homomorphic encryption allows computation on encrypted data
  • MPC enables collaborative computation without sharing inputs
  • QKD provides theoretically perfect key exchange
