7.1 포스트 양자 암호화

양자 컴퓨터는 현재의 비대칭 암호화를 위협합니다. NIST가 포스트 양자 알고리즘을 선정했습니다:

알고리즘 유형 FIPS 표준
ML-KEM (Kyber) 키 캡슐화 FIPS 203
ML-DSA (Dilithium) 디지털 서명 FIPS 204
SLH-DSA (SPHINCS+) 무상태 서명 FIPS 205

마이그레이션 일정

지금 포스트 양자 마이그레이션 계획을 시작하세요. 표준은 2024년에 확정되었으며, 2030년까지 광범위한 채택이 예상됩니다. 오늘 암호화된 민감한 데이터가 미래 양자 복호화를 위해 수집될 수 있습니다.

7.2 영지식 증명

무엇을 알고 있는지 공개하지 않고 알고 있음을 증명합니다:

// 영지식 증명 개념
// Alice가 hash(x) = H가 되는 x를 알고 있음을 증명
// x를 공개하지 않고

1. Alice가 커밋먼트 생성: c = commit(x, r)
2. 검증자가 랜덤 챌린지 전송: e
3. Alice가 e에 기반한 응답 전송
4. 검증자가 c에 대해 응답이 유효한지 확인

// 유효한 경우: Alice는 x를 알고 있음
// 검증자가 알게 되는 것: x에 대해 아무것도 없음

7.2.1 응용

  • 프라이버시 보존 인증: 자격증명 공개 없이 신원 증명
  • 익명 자격증명: 전체 신원 없이 속성 증명
  • 블록체인 프라이버시: 비공개 거래 (Zcash)
  • 투표 시스템: 선택 공개 없이 투표 검증

7.3 동형 암호화

복호화 없이 암호화된 데이터에서 계산:

// 동형 암호화 개념
let encrypted1 = encrypt(10);  // E(10)
let encrypted2 = encrypt(20);  // E(20)

// 암호화된 숫자 더하기
let encryptedSum = add(encrypted1, encrypted2);  // E(30)

// 복호화하여 결과 획득
let result = decrypt(encryptedSum);  // 30
// 클라우드는 평문 숫자를 절대 보지 못함!

7.3.1 유형

유형 연산 성능
부분 동형 (PHE) 덧셈 또는 곱셈 빠름
준 동형 (SHE) 제한된 연산 중간
완전 동형 (FHE) 모든 계산 느림 (개선 중)

7.4 안전한 다자간 계산

여러 당사자가 입력을 공개하지 않고 함수를 계산:

  • 예시: 개별 급여 공개 없이 평균 급여 계산
  • 기법: Shamir의 비밀 공유, 혼란 회로
  • 사용 사례: 비공개 경매, 안전한 투표, 협업 분석

7.5 양자 키 분배

이론적으로 깨질 수 없는 키 교환을 위해 양자 역학 사용:

  • 원리: 양자 상태 측정이 상태를 교란
  • 탐지: 도청이 탐지 가능
  • 제한: 특수 하드웨어 필요, 짧은 거리
  • 상태: 상업적으로 이용 가능, 제한된 배치

7.7 장 요약

핵심 내용

  • 포스트 양자 알고리즘 (Kyber, Dilithium)이 표준화됨
  • 영지식 증명으로 프라이버시 보존 검증 가능
  • 동형 암호화로 암호화된 데이터에서 계산 가능
  • MPC로 입력 공유 없이 협업 계산 가능
  • QKD는 이론적으로 완벽한 키 교환 제공
