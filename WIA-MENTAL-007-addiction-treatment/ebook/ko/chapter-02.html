제02장

평가 및 치료 계획 시스템

弘益人間 · 널리 인간을 이롭게 하라

소개

효과적인 중독 치료는 포괄적인 평가로 시작됩니다. 의사가 먼저 상태를 진단하지 않고 약물을 처방하지 않는 것처럼, 중독 치료는 적절한 치료 계획을 개발하기 전에 장애의 성격, 심각도 및 맥락을 이해하기 위해 철저한 평가가 필요합니다.

선별 도구

AUDIT(알코올 사용 장애 식별 테스트), DAST-10(약물 남용 선별 테스트), CAGE, NIDA 빠른 선별, CRAFFT(청소년), CUDIT-R(대마초), IGDS9-SF(게임), PGSI(도박)를 포함한 간단한 설문지입니다.

진단 평가

DSM-5는 4가지 범주에 걸친 11가지 기준에 따라 물질 사용 장애를 정의합니다: 손상된 통제(의도한 것보다 더 많이/오래 사용, 줄이기 위한 실패한 노력, 많은 시간 소비, 갈망), 사회적 손상(역할 의무 실패, 대인 관계 문제에도 불구하고 계속 사용, 중요한 활동 포기), 위험한 사용(물리적으로 위험한 상황에서 사용, 신체적/심리적 문제에도 불구하고 계속 사용), 약리학적(내성, 금단).

심각도 및 기능 평가

중독 심각도 지수(ASI), 타임라인 추적(TLFB), 금단 척도(COWS/CIWA), 의존 심각도 척도(SDS), 약물 사용 결과 목록(InDUC)을 포함한 여러 표준화된 도구가 이러한 차원을 측정합니다.

위험 평가

과다 복용 위험, 금단 위험, 자살 위험, 폭력 위험, 재발 위험, 의료 위험, 심리사회적 위험을 포함한 여러 유형의 위험을 평가해야 합니다.

개인화된 치료 계획

평가 데이터와 증거 기반 알고리즘을 통합하여 SMART 목표, 일치하는 개입, 약물 권장 사항 및 진행 측정을 생성합니다.

디지털 선별 구현

디지털 선별 도구는 전통적인 종이와 연필 방법에 비해 상당한 이점을 제공합니다. 자동화된 채점은 관리 시간을 줄이고, 적응형 질문은 응답에 따라 후속 항목을 조정하며, 전자 건강 기록과의 통합은 데이터 관리를 간소화하고, 증상 변화의 종적 추적은 진행 상황을 모니터링하며, 임상 환경뿐만 아니라 환자의 자연 환경에서 평가를 관리할 수 있는 능력은 접근성을 향상시킵니다.

일반적인 선별 도구

AUDIT(알코올 사용 장애 식별 테스트)는 위험한 음주를 식별하기 위한 10개 항목 선별 도구입니다. DAST-10(약물 남용 선별 테스트)은 약물에 대한 10개 항목 버전입니다. CAGE는 알코올 문제에 대한 4개 항목 빠른 선별입니다. NIDA 빠른 선별은 여러 물질에 대한 적응형 분기 논리를 사용합니다.

DSM-5 물질 사용 장애 기준

DSM-5는 4가지 범주에 걸친 11가지 기준에 따라 물질 사용 장애를 정의합니다. 손상된 통제 범주에는 의도한 것보다 더 많이/오래 사용, 줄이기 위한 실패한 노력, 많은 시간 소비, 갈망 또는 강한 사용 충동이 포함됩니다. 사회적 손상 범주에는 주요 역할 의무를 이행하지 못함, 사회적/대인 관계 문제에도 불구하고 계속 사용, 중요한 활동을 포기하거나 줄임이 포함됩니다.

컴퓨터화된 진단 인터뷰

컴퓨터화된 진단 인터뷰는 일관성과 효율성을 향상시킵니다. 표준화된 질문은 인터뷰 간 변동성을 줄이고, 자동화된 채점은 오류를 제거하며, 적응형 질문은 관련 항목에 집중하고, 즉각적인 결과는 임상 의사 결정을 가속화하며, 데이터 통합은 포괄적인 기록을 생성합니다.

심각도 및 기능 평가

진단을 넘어서 중독의 심각도와 기능에 미치는 영향을 이해하는 것은 치료 계획에 중요합니다. 중독 심각도 지수(ASI)는 의료, 고용, 알코올, 약물, 법적, 가족/사회적, 정신과적 영역을 평가하는 60-90분 인터뷰입니다. 타임라인 추적(TLFB)은 30-90일 동안의 일일 물질 사용 패턴을 캘린더 방법을 사용하여 15-30분 안에 평가합니다.

위험 평가 영역

포괄적인 중독 평가는 환자 안전을 보장하고 적절한 치료 수준을 안내하기 위해 여러 유형의 위험을 평가해야 합니다. 과다 복용 위험은 특히 오피오이드, 다물질 사용, 금주 기간 후 사용 재개에서 중요합니다. 금단 위험은 알코올 또는 벤조디아제핀 금단으로 인한 의료 합병증(발작, 섬망 진전)을 평가합니다. 자살 위험은 중독, 금단 및 초기 회복 중에 높으며 사고, 계획, 수단, 의도를 평가합니다.

개인화된 치료 계획 구성 요소

평가 데이터는 개인의 고유한 요구, 선호 및 상황을 해결하는 실행 가능하고 개인화된 치료 계획으로 변환되어야 합니다. 현대 치료 계획 시스템은 평가 결과를 증거 기반 치료 알고리즘과 통합하여 포괄적이고 개별화된 치료 계획을 생성합니다.

치료 계획 알고리즘

디지털 치료 계획 시스템은 진단, 심각도, 위험, 동반 장애, 환자 선호 및 장벽에 대한 정보를 추출하여 문제 목록을 생성하고, 목표를 협력적으로 개발하고, 개입을 문제에 일치시키고, 약물을 결정하고, 빈도와 기간을 할당하고, 진행 측정을 정의하고, 퇴원 기준을 설정합니다.

문화적으로 적절한 평가

평가 도구는 다양한 문화적 배경에 맞게 적응되어야 합니다. 단순한 번역을 넘어서 문화적 적응은 질문의 문화적 관련성, 응답 옵션의 적절성, 중독에 대한 문화적 신념 및 치료 추구 패턴을 고려해야 합니다. 다문화 타당성 연구는 평가 도구가 다양한 인구에서 신뢰할 수 있고 유효한지 확인합니다.

통합 치료 계획

최상의 치료 계획은 물질 사용뿐만 아니라 동반 정신 건강 장애, 의료 문제, 사회적 결정 요인(주택, 고용, 법적 문제) 및 가족 역학을 해결합니다. 통합 치료 접근법은 모든 문제 영역을 동시에 해결하여 분절된 치료보다 우수한 결과를 생성합니다. 디지털 플랫폼은 다양한 제공자와 서비스 간의 조정을 촉진할 수 있습니다.

적응형 평가 시스템

적응형 평가는 이전 응답에 따라 질문을 조정하여 평가를 더 효율적이고 정확하게 만듭니다. 예를 들어, 사용자가 알코올을 전혀 마시지 않는다고 보고하면 시스템은 알코올 관련 질문을 건너뛰고 다른 물질로 이동합니다. 사용자가 심각한 우울증 증상을 보고하면 시스템은 자살 위험에 대한 후속 질문을 추가합니다. 이 적응형 접근법은 평가 시간을 줄이고 사용자 부담을 최소화하면서 임상적으로 관련된 정보를 수집합니다.

종적 평가 및 진행 추적

디지털 시스템은 시간 경과에 따른 종적 평가를 가능하게 하여 치료에 대한 반응, 증상 변화 및 기능 개선을 추적합니다. 정기적인 재평가(주간, 월간 또는 분기별)는 치료가 효과적인지 또는 조정이 필요한지를 보여줍니다. 진행 추적 대시보드는 시간 경과에 따른 개선을 시각화하여 환자와 임상의 모두에게 동기를 제공합니다. 진행 부족 또는 악화에 대한 자동 경고는 신속한 임상 검토 및 개입 조정을 트리거합니다.

평가 데이터 보안

평가 중에 수집된 민감한 데이터는 최고 수준의 보안으로 보호되어야 합니다. 전송 중 및 저장 중 암호화, 접근 제어, 감사 로그 및 정기적인 보안 평가가 필수적입니다.

한국 치료 계획 API 예제

한국 의료 시스템과 통합된 치료 계획 생성 코드 예제:

import { WIAMental007 } from '@wia/mental-007';
import { NHISIntegration } from '@korea/nhis-api';

// 평가 실시 및 치료 계획 생성
async function createKoreanTreatmentPlan(patientData: PatientData) {
  // 1. 한국 표준화 평가 도구 실행
  const assessments = await WIAMental007.assessment.runBattery({
    patientId: patientData.id,
    tools: [
      { name: 'AUDIT-K', version: '2024' },
      { name: 'K-척도', version: '2024' },
      { name: 'CPGI-K', version: '2024' }
    ],
    language: 'ko',
    culturalNorms: 'korean'
  });

  // 2. 평가 결과 분석
  const analysis = await WIAMental007.ai.analyzeAssessment({
    assessments,
    considerFactors: {
      familyDynamics: true,
      workplacePressure: true,
      socialStigma: 'high',
      insuranceCoverage: await getNHISCoverage(patientData.id)
    }
  });

  // 3. 한국 치료 리소스 매칭
  const treatmentPlan = await WIAMental007.planning.create({
    patientId: patientData.id,
    assessment: analysis,
    preferences: {
      location: patientData.location,
      preferredLanguage: 'ko',
      familyInvolvement: 'encouraged',
      culturalSensitivity: 'required'
    },
    resources: {
      localCenters: await findNearbyKoreanCenters(patientData.location),
      insuranceCovered: true,
      governmentPrograms: await getEligiblePrograms(patientData)
    }
  });

  // 4. 국민건강보험 연동
  const nhisIntegration = new NHISIntegration();
  await nhisIntegration.submitTreatmentPlan({
    plan: treatmentPlan,
    patientNHISNumber: patientData.nhisNumber,
    expectedCoverage: await calculateNHISCoverage(treatmentPlan)
  });

  // 5. 치료 계획 문서화
  return {
    plan: treatmentPlan,
    assessmentResults: analysis,
    coveredServices: await nhisIntegration.getApprovedServices(),
    referrals: {
      primaryCenter: treatmentPlan.assignedCenter,
      supportGroups: await findKoreanAAGroups(patientData.location),
      digitalTherapy: await getKoreanDigitalPrograms(analysis)
    },
    followUpSchedule: generateKoreanFollowUpPlan(treatmentPlan)
  };
}

// 국민건강보험 보장 범위 확인
async function getNHISCoverage(patientId: string) {
  const nhis = new NHISIntegration();
  return await nhis.checkCoverage({
    patientId,
    serviceType: 'addiction-treatment',
    includeServices: [
      'outpatient-counseling',
      'medication-assisted-treatment',
      'residential-treatment',
      'family-therapy'
    ]
  });
}

한국 정부 지원 프로그램

중독 치료를 위한 주요 정부 지원 프로그램:

국민건강보험 중독 치료 보장

국민건강보험은 다음 중독 치료 서비스를 보장합니다:

  • 외래 상담 치료 (주 1-2회, 최대 6개월)
  • 입원 치료 (최대 3개월, 연장 가능)
  • 약물 보조 치료 (메타돈, 부프레노르핀 등)
  • 가족 치료 세션 (월 2회까지)
  • 재활 프로그램 (주간 재활, 사회 복귀 훈련)

무료 상담 전화 서비스

  • 중독 상담 전화: 1899-0893 (24시간, 무료)
  • 도박 문제 상담: 1336 (24시간, 무료)
  • 인터넷 중독 상담: 1599-0075 (평일 9시-18시)
  • 청소년 상담: 1388 (24시간, 무료)
  • 자살 예방 상담: 1393 (24시간, 무료)

저소득층 지원

기초생활수급자 및 차상위계층을 위한 특별 지원:

  • 치료비 100% 지원 (입원 및 외래)
  • 교통비 지원 (월 최대 10만원)
  • 재활 프로그램 무료 참여
  • 사후 관리 서비스 무료 제공
  • 가족 상담 무료 지원

핵심 요점

  • 포괄적인 평가는 효과적인 중독 치료의 기초이며, 치료 계획 전에 선별, 진단, 심각도 측정 및 위험 평가가 필요합니다
  • 디지털 선별 도구는 AUDIT, DAST 및 장애별 척도와 같은 검증된 도구의 신속하고 자동화된 관리 및 채점을 가능하게 합니다
  • DSM-5 진단 평가는 4가지 범주(손상된 통제, 사회적 손상, 위험한 사용, 약리학적)에 걸친 11가지 기준을 평가하며, 심각도는 기준 수(2-3 경증, 4-5 중등도, 6+ 중증)를 기반으로 합니다
  • 타임라인 추적(TLFB)은 30-90일 동안 물질 사용의 상세한 패턴 분석을 제공하여 개입 목표에 필수적인 유발 요인, 고위험 기간 및 추세를 식별합니다
  • 위험 평가는 환자 안전과 적절한 치료 수준을 보장하기 위해 여러 영역(과다 복용, 금단, 자살, 폭력, 재발, 의료, 심리사회적)을 평가해야 합니다

복습 질문

  1. 선별과 진단 평가의 차이점은 무엇입니까? 임상 워크플로에서 각 유형의 도구를 언제 사용하겠습니까?
  2. 물질 사용 장애에 대한 DSM-5 심각도 지정자를 설명하십시오. 경증, 중등도 및 중증 분류에 대해 충족해야 하는 기준은 몇 개입니까?
  3. 디지털 선별 도구에서 적응형/분기 논리가 효율성과 사용자 경험을 어떻게 개선할 수 있습니까? 예를 들어 제공하십시오.
  4. 타임라인 추적(TLFB)이 무엇이며, TLFB 데이터에서 어떤 유형의 패턴을 식별할 수 있습니까? 치료 계획에서 이 정보를 어떻게 사용하겠습니까?
  5. 중독 치료 인구에서 자살 위험 평가가 왜 중요합니까? C-SSRS 프레임워크를 사용한 자살 위험 평가의 주요 요소를 설명하십시오.

弘益人間 · 널리 인간을 이롭게 하라

디지털 중독 치료 기술은 중독으로 고통받는 모든 사람에게 증거 기반 치료를 접근 가능하게 함으로써 弘益人間(홍익인간)의 원칙을 구현합니다. 지리, 경제적 지위 또는 사회적 상황에 관계없이 기술을 활용하여 전통적인 장벽을 극복함으로써 우리는 중독과 절망의 순환에 갇혀 있던 수백만 명에게 치유의 손길을 확장합니다.
