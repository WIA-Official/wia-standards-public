고급 기술 고려사항

Phase 2 구현 단계는 Phase 1에서 구축한 기초 위에 고급 자율 기능을 추가하는 단계입니다. 이 단계의 핵심은 AI 기반 의사결정 시스템 구현으로, 머신러닝 모델을 통해 복잡한 해상 상황을 분석하고 최적의 항로를 결정합니다. 컴퓨터 비전 시스템을 구축하여 카메라 영상에서 다른 선박, 부표, 해안선 등을 실시간으로 감지하고 분류합니다. 딥러닝 모델(YOLO, Faster R-CNN 등)을 선박 탐지에 특화하여 학습시키며, 다양한 기상 조건과 조명 환경에서도 안정적으로 작동하도록 데이터 증강 기법을 적용합니다. 충돌 회피 알고리즘은 국제해상충돌예방규칙(COLREG)을 코드로 구현하고, 다중 선박이 관여된 복잡한 상황에서도 안전한 기동을 계산합니다. 경로 계획 시스템은 A* 또는 RRT* 알고리즘을 기반으로 하되, 해상 특유의 제약조건(조류, 풍향, 수심, 교통 분리 구역 등)을 고려하도록 확장합니다. 예측 모델은 다른 선박의 미래 위치를 예측하여 충돌 위험을 조기에 감지하며, Kalman 필터나 파티클 필터를 사용하여 불확실성을 관리합니다. 원격 제어 시스템을 고도화하여 육상 관제자가 필요시 선박을 직접 조종할 수 있도록 하며, 저지연 통신과 직관적인 인터페이스를 제공합니다. 이 단계 완료 시 선박은 ALO 2-3 수준에 도달합니다.

성능 최적화

Phase 2의 성능 최적화는 AI 모델의 실시간 처리 능력 향상에 중점을 둡니다. 첫째, GPU 가속을 활용하여 딥러닝 추론 속도를 높이고, 모델 양자화(quantization)와 프루닝(pruning)을 통해 모델 크기를 줄이면서도 정확도를 유지합니다. TensorRT나 ONNX Runtime 같은 최적화 엔진을 사용하여 추론 성능을 2-3배 향상시킵니다. 둘째, 센서 융합의 정확도를 높이기 위해 Extended Kalman Filter나 Unscented Kalman Filter를 적용하여 노이즈가 많은 센서 데이터에서도 정확한 상태 추정을 수행합니다. 셋째, 경로 계획 알고리즘의 계산 효율성을 개선하기 위해 그리드 해상도를 동적으로 조정하고, 중요한 영역(혼잡 해역, 장애물 근처)에서만 높은 해상도를 사용합니다. 넷째, 병렬 처리 아키텍처를 도입하여 객체 감지, 추적, 경로 계획 등을 동시에 실행하고, 파이프라인 병목을 제거합니다. 다섯째, 예측 모델의 정확도를 높이기 위해 실제 운항 데이터로 지속적으로 재학습하고, 온라인 학습 기법을 적용하여 새로운 패턴에 적응합니다.

품질 보증

Phase 2의 품질 보증은 AI 시스템의 신뢰성과 안전성 검증에 집중합니다. 컴퓨터 비전 모델은 다양한 해상 조건에서 촬영된 수만 장의 이미지로 테스트되며, 정밀도(precision)와 재현율(recall)이 95% 이상을 달성해야 합니다. 충돌 회피 알고리즘은 수천 개의 시뮬레이션 시나리오를 통과해야 하며, 각 시나리오는 실제 해상 사고 사례를 기반으로 설계됩니다. 경로 계획 시스템은 최적성과 안전성을 동시에 평가하며, 생성된 경로가 모든 해사 규정을 준수하는지 자동으로 검증합니다. AI 모델의 설명 가능성(explainability)을 확보하기 위해 SHAP이나 LIME 같은 기법을 적용하여 모델의 결정 근거를 시각화하고, 전문가가 검토합니다. 모든 AI 컴포넌트는 edge case와 failure mode 분석을 거치며, 예상치 못한 입력에 대한 대응 방안을 마련합니다.
