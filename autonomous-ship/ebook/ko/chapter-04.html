고급 기술 고려사항

Phase 1 기초 단계는 자율운항선박 시스템 구축의 토대를 마련하는 중요한 시기입니다. 이 단계에서는 핵심 인프라와 기본 자동화 기능을 구현하여 향후 고급 자율 기능의 기반을 확립합니다. 첫 번째 중점 영역은 센서 하드웨어 설치와 기본 데이터 수집 파이프라인 구축입니다. 레이더, GPS, AIS 수신기, 기상 센서 등 필수 센서들을 선박에 장착하고, 이들로부터 데이터를 수집하는 기본 소프트웨어 스택을 개발합니다. 데이터 수집률, 정확도, 지연시간 등 기본 성능 지표를 정의하고 모니터링 체계를 구축합니다. 두 번째는 네트워크 인프라 구축으로, 선박 내부 네트워크(CAN, Ethernet)와 외부 통신(위성, 4G/5G) 시스템을 설치합니다. 세 번째는 기본 자동화 기능 구현인데, 오토파일럿의 개선, 엔진 자동 제어, 기본 경보 시스템 등이 포함됩니다. 네 번째는 데이터 저장 및 로깅 시스템 구축으로, 모든 센서 데이터와 시스템 이벤트를 안전하게 저장하고 나중에 분석할 수 있도록 합니다. 다섯 번째는 기본 사용자 인터페이스 개발로, 선원과 육상 관제자가 시스템 상태를 모니터링하고 기본 제어를 수행할 수 있는 인터페이스를 제공합니다. Phase 1은 일반적으로 3-6개월이 소요되며, 완료 시 선박은 ALO 1 수준의 자동화를 달성합니다.

성능 최적화

Phase 1의 성능 최적화는 안정성과 신뢰성에 초점을 맞춥니다. 첫째, 센서 데이터 수집의 안정성을 확보하기 위해 센서 고장 감지 알고리즘을 구현하고, 센서가 이상 데이터를 생성하거나 응답하지 않을 때 즉시 알림을 발생시킵니다. 둘째, 네트워크 통신의 신뢰성을 높이기 위해 자동 재연결 메커니즘과 데이터 버퍼링을 구현하여, 일시적인 통신 두절에도 데이터 손실이 없도록 합니다. 셋째, 데이터 저장 시스템의 효율성을 높이기 위해 압축 알고리즘을 적용하고, 중요도에 따라 데이터 보존 기간을 차등화합니다. 넷째, 기본 자동화 기능의 응답 속도를 최적화하여 조타 명령이 1초 이내에 실행되도록 하고, 엔진 제어 명령도 즉시 반영되도록 합니다. 다섯째, 사용자 인터페이스의 성능을 개선하여 대시보드가 최소 30fps로 업데이트되고, 사용자 입력에 즉각 반응하도록 합니다. 이 단계에서는 고급 AI 기능보다는 기본 기능의 완벽한 작동에 우선순위를 둡니다.
