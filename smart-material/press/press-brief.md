# WIA-MAT-SMART-001 — Smart Material (Press Brief)

**WIA Standards releases canonical Smart Material envelope shape**

The WIA Standards Committee has published the WIA-MAT-SMART-001 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for smart material hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
