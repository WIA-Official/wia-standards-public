Chapter 3: Tracking and Trajectory Prediction

Kalman Filtering and Ballistic Trajectory Mathematics

3.1 The Extended Kalman Filter

Once a ballistic missile is detected, the tracking system must continuously estimate its position, velocity, and acceleration despite noisy sensor measurements. The Extended Kalman Filter (EKF) provides the optimal solution to this state estimation problem by recursively combining sensor measurements with predictions from kinematic models of missile motion.

3.1.1 State Vector Representation

The complete state of a ballistic missile at any instant is represented by a nine-dimensional state vector containing position, velocity, and acceleration components in three-dimensional space:

X = [x, y, z, vx, vy, vz, ax, ay, az]ᵀ
State Vector Components: Position (meters): x = East position in Earth-Centered Fixed (ECF) frame y = North position in ECF frame z = Up position (altitude above reference ellipsoid) Velocity (meters/second): vx = East velocity component vy = North velocity component vz = Vertical velocity component (positive = ascending) Acceleration (meters/second²): ax = East acceleration component ay = North acceleration component az = Vertical acceleration (includes gravity + drag) Example State Vector at T+30 seconds: Position: [45,000 m, 12,000 m, 85,000 m] Velocity: [2,500 m/s, 1,800 m/s, -3,200 m/s] Acceleration:[0 m/s², 0 m/s², -9.81 m/s²] (ballistic flight, gravity only)

3.1.2 Kalman Filter Equations

The Kalman filter operates in two steps: prediction and update. During prediction, the current state estimate is propagated forward in time using the missile's equations of motion. During update, new sensor measurements are incorporated to refine the state estimate.

Prediction Step (Time Update): State Prediction: X̂(k|k-1) = F·X(k-1|k-1) + G·u(k) Error Covariance Prediction: P(k|k-1) = F·P(k-1|k-1)·Fᵀ + Q where: X̂(k|k-1) = Predicted state at time k given measurements through k-1 F = State transition matrix (9×9) G = Control input matrix (9×3) u(k) = Known inputs (e.g., gravity) P = Error covariance matrix (9×9) Q = Process noise covariance (9×9) Update Step (Measurement Update): Kalman Gain: K(k) = P(k|k-1)·Hᵀ·[H·P(k|k-1)·Hᵀ + R]⁻¹ State Update: X̂(k|k) = X̂(k|k-1) + K(k)·[Z(k) - H·X̂(k|k-1)] Error Covariance Update: P(k|k) = [I - K(k)·H]·P(k|k-1) where: Z(k) = Sensor measurement at time k H = Observation matrix (maps state to measurement) R = Measurement noise covariance I = Identity matrix K = Kalman gain matrix Innovation (Measurement Residual): ν(k) = Z(k) - H·X̂(k|k-1) This represents the "surprise" in the measurement—how much it differs from what we predicted. Large innovations indicate: - Sensor malfunction - Maneuver by target - Incorrect kinematic model
Matrix Dimension Physical Meaning Typical Update Rate
F (State Transition) 9×9 How state evolves over time 1-100 Hz
Q (Process Noise) 9×9 Uncertainty in motion model Constant or adaptive
H (Observation) 3×9 to 6×9 Maps state to measurements Per measurement
R (Measurement Noise) 3×3 to 6×6 Sensor accuracy Per measurement
P (Error Covariance) 9×9 Estimate uncertainty 1-100 Hz

3.2 Ballistic Trajectory Physics

Understanding the physics of ballistic missile flight is essential for accurate trajectory prediction. A ballistic trajectory consists of three distinct phases, each governed by different forces and requiring different modeling approaches.

3.2.1 Boost Phase (Powered Flight)

During boost phase, the missile's rocket motor provides thrust, causing continuous acceleration. The trajectory is highly observable due to the bright infrared signature, but the acceleration complicates prediction since thrust varies with propellant consumption and staging events.

Boost Phase Characteristics: Duration: 30-300 seconds (depending on missile type) SRBM: 30-90 seconds MRBM: 90-180 seconds ICBM: 180-300 seconds (multi-stage) Thrust-to-Weight Ratio: Initial: 1.2-2.0 (just sufficient to overcome gravity) Burnout: 3.0-8.0 (as propellant is consumed, mass decreases) Acceleration Profile: a(t) = [F_thrust(t) - m(t)·g] / m(t) where: F_thrust(t) = Thrust force (time-varying) m(t) = Instantaneous mass (decreasing as propellant burns) g = Gravitational acceleration (9.81 m/s² at sea level) Velocity at Burnout (Tsiolkovsky Rocket Equation): Δv = v_exhaust · ln(m_initial / m_final) Typical values: v_exhaust ≈ 2,500-3,500 m/s (specific impulse = 250-350 seconds) Mass ratio (m_initial/m_final) ≈ 4-10 (single stage) Δv ≈ 3,500-8,000 m/s (depending on staging)

3.2.2 Midcourse Phase (Ballistic Flight)

After engine burnout, the missile follows a ballistic trajectory determined solely by its burnout velocity and the gravitational field. For theater ballistic missiles, the trajectory approximates a parabola. For ICBMs reaching altitudes above 1,000 km, orbital mechanics must be considered.

r(t) = r₀ + v₀·t + ½·g·t²
v(t) = v₀ + g·t
Ballistic Flight Equations: For sub-orbital trajectories (altitude < 200 km): Position: x(t) = x₀ + vx₀·t y(t) = y₀ + vy₀·t z(t) = z₀ + vz₀·t - ½·g·t² Velocity: vz(t) = vz₀ - g·t (vertical component decreases linearly) Maximum Altitude (Apogee): h_max = h₀ + (vz₀²) / (2·g) Range (flat Earth approximation): R = (v₀² · sin(2θ)) / g where θ = launch elevation angle For ICBM trajectories (altitude > 200 km): Must use orbital mechanics (Kepler's laws): r(t)·v(t) = constant (angular momentum conservation) E = ½v² - μ/r = constant (energy conservation) where μ = GM = 3.986×10¹⁴ m³/s² (Earth's gravitational parameter) Earth Rotation Effects: Must account for Coriolis and centrifugal forces in rotating frame: a_Coriolis = -2·ω × v a_centrifugal = -ω × (ω × r) where ω = 7.2921×10⁻⁵ rad/s (Earth's rotation rate) These effects cause eastward-fired missiles to have ~15% greater range than westward-fired missiles at same burnout velocity.

3.2.3 Terminal Phase (Atmospheric Reentry)

As the warhead reenters the atmosphere (typically below 100 km altitude), aerodynamic drag becomes significant and must be included in trajectory predictions. The drag force is proportional to the square of velocity and increases exponentially with decreasing altitude as air density rises.

Atmospheric Drag Model: Drag Force: F_drag = -½ · ρ(h) · v² · C_D · A Drag Acceleration: a_drag = -½ · ρ(h) · v² · C_D · A / m where: ρ(h) = Air density at altitude h (kg/m³) v = Velocity magnitude (m/s) C_D = Drag coefficient (0.2-0.5 for missiles) A = Cross-sectional area (m²) m = Mass (kg) Atmospheric Density Model (Exponential): ρ(h) = ρ₀ · exp(-h / H) where: ρ₀ = 1.225 kg/m³ (sea level density) H = 8,500 m (scale height) Ballistic Coefficient: β = m / (C_D · A) (kg/m²) Higher β → less drag deceleration (harder to defend against) Typical warhead: β = 10,000-30,000 kg/m² Typical decoy: β = 100-1,000 kg/m² (lower mass) Reentry Velocity: SRBM: 2-3 km/s (Mach 6-9) MRBM: 3-5 km/s (Mach 9-15) ICBM: 6-8 km/s (Mach 18-24) Reentry Heating: q = 0.5 · ρ · v³ (watts/m²) At v = 7 km/s, ρ = 0.1 kg/m³: q ≈ 17 MW/m² (ablative heat shield required!)
Flight Phase Duration Dominant Forces Prediction Challenge Update Rate
Boost 30-300 sec Thrust, Gravity Unknown thrust profile 1 Hz
Midcourse 5-25 min Gravity only Earth rotation, perturbations 0.1 Hz
Terminal 30-120 sec Gravity, Drag Atmospheric density variations 10 Hz

3.3 Impact Point Prediction

The ultimate goal of trajectory tracking is to predict where the missile will impact if not intercepted. This information drives threat assessment (which assets are at risk) and engagement planning (which interceptor battery should respond).

3.3.1 Impact Point Calculation

Impact Point Prediction Algorithm: 1. Extrapolate trajectory to altitude = 0: Given current state [x, y, z, vx, vy, vz, ax, ay, az] Solve for time t when z(t) = 0: If ballistic (no drag): 0 = z + vz·t + ½·az·t² t = [-vz ± √(vz² - 2·az·z)] / az (take positive root) If including drag (numerical integration required): Iterate: t_impact = t when z(t) ≈ 0 2. Calculate impact coordinates: x_impact = x + vx·t_impact + ½·ax·t_impact² y_impact = y + vy·t_impact + ½·ay·t_impact² 3. Convert to latitude/longitude: lat = atan2(y_impact, x_impact) lon = atan2(z_impact, √(x_impact² + y_impact²)) 4. Calculate uncertainty ellipse: σ_impact = √(σ_x² + σ_y²) From error covariance matrix P(k|k) Typical Impact Prediction Accuracy: At T+30 seconds into flight: CEP (Circular Error Probable) = 500-1000 m At T+180 seconds (midcourse): CEP = 100-300 m At T-30 seconds before impact: CEP = 50-100 m Accuracy improves as more measurements are processed and flight time increases.

3.4 Track Quality Metrics

Not all track solutions are equally reliable. The missile defense system must assess track quality in real-time to determine when a track is sufficiently accurate to support engagement decisions.

Metric Formula Good Value Poor Value Use
Position Error σ_pos = √(σ_x² + σ_y² + σ_z²) < 50 m > 500 m Impact prediction accuracy
Velocity Error σ_vel = √(σ_vx² + σ_vy² + σ_vz²) < 10 m/s > 100 m/s Intercept geometry planning
Innovation Magnitude ||ν(k)|| = ||Z(k) - H·X̂(k|k-1)|| < 3σ > 5σ Detect maneuvers or sensor errors
Track Age t_track = Current_time - Init_time > 10 sec < 3 sec Confidence in state estimate
Measurement Count N_meas > 20 < 5 Statistical confidence

Key Takeaways

  • Kalman Filter Optimality: The EKF provides mathematically optimal state estimation by combining sensor measurements with physics-based predictions, achieving better accuracy than either alone.
  • Nine-Dimensional State: Complete trajectory tracking requires estimating position (3D), velocity (3D), and acceleration (3D) simultaneously—9 parameters total that must be updated each measurement cycle.
  • Three Distinct Phases: Ballistic missile flight consists of boost (powered), midcourse (ballistic), and terminal (reentry) phases, each requiring different mathematical models due to changing dominant forces.
  • Drag Discrimination: Atmospheric drag during reentry causes lightweight decoys to decelerate much faster than heavy warheads (10× difference in deceleration), enabling discrimination based on trajectory alone.
  • Accuracy Improves with Time: Impact point prediction uncertainty decreases from ~1000m at detection to ~50m at late midcourse as more measurements refine the trajectory estimate.
  • Earth Rotation Matters: For ICBM trajectories, Coriolis effects from Earth's rotation can change the range by 15%, making it essential to compute trajectories in Earth-Centered Inertial frames, not rotating frames.
