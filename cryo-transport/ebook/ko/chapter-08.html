8.1 출발 시설 절차
--------------------------------------------------------------------------------

이송 출발은 성공적인 이송의 기초를 마련합니다. 출발 시설은 대상이 적절하게
준비되고, 듀어가 최적 상태이며, 모든 문서가 완료되었는지 확인해야 합니다.

[CRYO] 출발 전 체크리스트 (T-24시간):

    대상 준비:
      [ ] 대상 ID 확인 (바코드, RFID, 육안)
      [ ] 현재 보관 온도 기록 (-196°C ± 2°C)
      [ ] 듀어 내 위치 확인
      [ ] 손상 또는 이상 육안 검사
      [ ] 가족/권한자 최종 승인

    듀어 준비:
      [ ] 듀어 일련번호 및 인증 확인
      [ ] 진공 무결성 테스트 (압력 센서)
      [ ] LN2 레벨 90% 이상으로 충전
      [ ] 온도 센서 보정 및 테스트 (3개 모두)
      [ ] GPS/통신 모듈 테스트
      [ ] 압력 릴리프 밸브 기능 확인
      [ ] 외부 청소 및 라벨 부착

    차량 준비:
      [ ] 차량 검사 (기계, 전기, 타이어)
      [ ] 고정 시스템 설치 및 테스트
      [ ] 비상 키트 재고 확인
      [ ] 연료 충전 (전체)
      [ ] 비상 연락처 프로그램 (GPS)

    문서 준비:
      [ ] 보관 연속성 문서 초안 작성
      [ ] 이송 계획 및 경로 확인
      [ ] 규제 허가 확인 (수출, 위험물)
      [ ] 보험 정책 검토
      [ ] 비상 연락처 목록 최신화

출발 절차 (T-0):

    단계 1: 대상 이송 (듀어 내)
      시간: T-2시간
      - 이송 팀 브리핑
      - 극저온 PPE 착용
      - 대상을 보관 듀어에서 이송 듀어로 리프팅
      - 이송 듀어 밀봉
      - 온도 안정화 확인 (30분)

    단계 2: 듀어 적재
      시간: T-1시간
      - 듀어를 차량으로 이동 (리프팅 장비)
      - 고정 시스템에 배치
      - 스트랩 및 안정화 부착
      - 흔들림 테스트 (움직임 없음)
      - 센서 최종 테스트

    단계 3: 문서화 및 승인
      시간: T-30분
      - CoC 문서 서명 (출발 관리자)
      - 출발 검사 사진
      - 초기 센서 판독값 기록
      - 최종 브리핑 (이송 팀)

    단계 4: 출발
      시간: T-0
      - 출발 타임스탬프 (UTC)
      - GPS 추적 활성화
      - 실시간 모니터링 시작
      - 출발 시설 통지
      - 도착 시설 ETA 전송

8.2 이송 중 통신 프로토콜
--------------------------------------------------------------------------------

이송 중 지속적인 통신은 이해관계자들에게 정보를 제공하고 문제를 신속하게
해결하기 위해 필수적입니다.

[CRYO] 통신 빈도 및 내용:

    정기 업데이트 (매 2시간):
      수신자:
        - 출발 시설 (관리자)
        - 도착 시설 (준비 팀)
        - 가족 (선택 사항, 요청 시)

      내용:
        - 현재 위치 (GPS 좌표, 주소)
        - 온도 상태 (3개 센서 평균)
        - LN2 레벨 (%)
        - 예상 도착 시간 (ETA)
        - 다음 체크포인트
        - 상태: 정상/우려/긴급

    체크포인트 업데이트 (각 체크포인트):
      내용:
        - 체크포인트 ID 및 시간
        - 온도 및 LN2 레벨
        - 활동 (휴게소, LN2 보충 등)
        - 다음 체크포인트 ETA
        - 사진 (선택 사항)

    사건 업데이트 (즉시):
      트리거:
        - 온도 경고/위험
        - 경로 우회
        - 차량 문제
        - 비상 사태

      내용:
        - 사건 유형 및 심각도
        - 현재 상태 및 위치
        - 취해진 조치
        - 지원 필요 사항
        - 업데이트된 ETA

[i] 통신 채널:

    1차 채널: 모바일 앱
      - 실시간 GPS 지도
      - 자동 센서 데이터
      - 푸시 알림
      - 양방향 메시징

    2차 채널: 이메일
      - 2시간마다 자동 보고서
      - 체크포인트 요약
      - 첨부 파일 (사진, 로그)

    3차 채널: SMS
      - 중요 알림만
      - 경고 및 위험 수준
      - ETA 변경 (>1시간)

    비상 채널: 전화
      - 즉시 위험 (수준 1)
      - 통신 시스템 고장
      - 의사결정 필요

8.3 도착 시설 절차
--------------------------------------------------------------------------------

도착 시설은 대상을 안전하게 수령하고, 무결성을 확인하며, 장기 보관으로 전환할
준비를 해야 합니다.

[CRYO] 도착 전 준비 (T-4시간):

    시설 준비:
      [ ] 수령 구역 청소 및 준비
      [ ] 백업 듀어 사전 냉각 (-196°C)
      [ ] 리프팅 장비 테스트
      [ ] LN2 공급 확인 (충분한 재고)
      [ ] 수령 팀 브리핑

    문서 준비:
      [ ] 수령 체크리스트 검토
      [ ] CoC 문서 템플릿 준비
      [ ] 검사 양식 인쇄
      [ ] 카메라 및 기록 장비 준비

    인력 준비:
      [ ] 수령 관리자 배정
      [ ] 기술 팀 대기
      [ ] 품질 보증 담당관 통지
      [ ] 보안 알림 (접근 허가)

도착 절차 (T-0):

    단계 1: 도착 확인
      시간: T-0
      - 도착 타임스탬프 (UTC)
      - 차량 및 팀 육안 검사
      - 보안 확인 (ID, 인증)
      - 초기 온도 확인 (외부 센서)

    단계 2: 듀어 하역
      시간: T+15분
      - 고정 시스템 해제
      - 리프팅 장비로 듀어 하역
      - 수령 구역으로 이동
      - 안정적인 위치에 배치

    단계 3: 검사 및 평가
      시간: T+30분
      - 듀어 외부 검사:
        * 물리적 손상 (찌그러짐, 균열)
        * 라벨 및 표시 확인
        * 압력 릴리프 밸브 상태
      - 센서 데이터 다운로드:
        * 전체 이송 온도 로그
        * GPS 추적 데이터
        * 사건 로그
      - 내부 온도 확인:
        * 3개 센서 모두 판독
        * -196°C ± 3°C 확인
      - LN2 레벨 확인:
        * >50% 확인 (안전 최소값)

    단계 4: 수령 승인 또는 거부
      시간: T+1시간
      - 검사 결과 검토
      - 승인 기준:
        * 온도 < -190°C (이상적: <-194°C)
        * LN2 레벨 > 50%
        * 물리적 손상 없음
        * 문서 완전성 100%
      - 거부 조건:
        * 온도 > -180°C (경고)
        * 심각한 물리적 손상
        * CoC 문서 불완전
        * 대상 ID 불일치

    단계 5: 대상 이송 (승인 시)
      시간: T+2시간
      - 이송 듀어에서 장기 보관 듀어로
      - 극저온 PPE 및 안전 절차
      - 배치 위치 기록
      - 온도 안정화 확인 (30분)
      - CoC 문서 최종 서명

8.4 품질 보증 및 인증
--------------------------------------------------------------------------------

이송 후 품질 보증은 표준이 충족되었는지 확인하고 지속적인 개선을 위한 데이터를
제공합니다.

[CRYO] 품질 검증 프로세스:

    레벨 1: 자동 검증
      시간: 도착 후 즉시
      시스템:
        - 온도 로그 분석 (범위 이탈 감지)
        - GPS 추적 검증 (경로 준수)
        - 센서 데이터 무결성 (간격 없음)
        - 타임스탬프 일관성
      출력:
        - 자동 품질 점수 (0-100)
        - 플래그된 문제 목록
        - 예비 TQS 계산

    레벨 2: 인간 검토
      시간: 도착 후 4시간
      검토자: 품질 보증 담당관
      활동:
        - CoC 문서 완전성 확인
        - 디지털 서명 검증
        - 사진 문서 검토
        - 사건 보고서 평가 (있는 경우)
        - 인터뷰 이송 팀 (주요 문제)
      출력:
        - 품질 보증 보고서
        - 확인된 문제 및 권장사항
        - 최종 TQS 승인

    레벨 3: 관리 승인
      시간: 도착 후 24시간
      승인자: 시설 관리자
      활동:
        - QA 보고서 검토
        - TQS 수용 (≥0.70 필수)
        - 예외 승인 (필요 시)
        - 시정 조치 지시
        - 최종 인증 서명
      출력:
        - 공식 완료 인증서
        - 아카이브 패키지 (모든 문서)
        - 경험 교훈 요약

이송 품질 점수 (TQS) 세부사항:

    TQS = (0.40 × T) + (0.25 × R) + (0.20 × D) + (0.15 × C)

    T (온도 준수):
      - 100% 시간 -196°C ~ -190°C: 1.00
      - 95-99% 시간 범위 내: 0.90
      - 90-94% 시간 범위 내: 0.80
      - <90% 시간 범위 내: 0.00

    R (경로 준수):
      - 계획된 경로 100%: 1.00
      - 승인된 우회만: 0.90
      - 경미한 지오펜스 위반: 0.80
      - 심각한 위반: 0.00

    D (기간 준수):
      - 계획 시간의 ±10% 내: 1.00
      - 계획 시간의 ±20% 내: 0.90
      - 계획 시간의 ±30% 내: 0.80
      - >30% 차이: 0.50

    C (문서화 품질):
      - 100% 완료, 모든 서명: 1.00
      - 경미한 누락 (<5%): 0.90
      - 중간 누락 (5-10%): 0.80
      - 심각한 누락 (>10%): 0.50

[!] TQS 임계값 및 조치:

    +-------------------+----------+------------------------------------+
    | TQS 범위          | 등급     | 필요 조치                          |
    +-------------------+----------+------------------------------------+
    | 0.95 - 1.00       | A (우수) | 팀 인정, 모범 사례 공유            |
    | 0.85 - 0.94       | B (양호) | 경미한 개선 권장                   |
    | 0.70 - 0.84       | C (허용) | 근본 원인 분석, 재교육             |
    | 0.50 - 0.69       | D (한계) | 공식 검토, 시정 조치 계획          |
    | 0.00 - 0.49       | F (실패) | 조사, 잠재적 인증 정지             |
    +-------------------+----------+------------------------------------+


--------------------------------------------------------------------------------
8.5 지속적 개선 및 피드백
--------------------------------------------------------------------------------

각 이송은 시스템, 절차 및 인력을 개선할 기회입니다.

[CRYO] 이송 후 검토 프로세스:

    즉시 검토 (도착 후 48시간):
      참여자:
        - 이송 팀
        - 출발 시설 관리자
        - 도착 시설 관리자
        - 품질 보증 담당관

      형식: 가상 회의 또는 구조화된 설문

      의제:
        [1] 무엇이 잘 되었는가?
            - 성공 요인
            - 효과적인 절차
            - 우수한 팀워크 순간

        [2] 무엇이 개선될 수 있는가?
            - 과제 및 장애물
            - 거의 놓친 사건
            - 비효율성

        [3] 경험 교훈
            - 예상치 못한 문제
            - 창의적 해결책
            - 미래 권장사항

        [4] 조치 항목
            - 절차 업데이트 (누가, 언제)
            - 추가 교육 (주제, 인력)
            - 장비 개선 (무엇, 언제)

      출력:
        - 검토 회의 록
        - 조치 항목 목록 (책임, 기한)
        - 경험 교훈 문서

    분기별 집계 검토:
      범위: 모든 이송 (지난 3개월)
      분석:
        - TQS 추세 (개선/악화?)
        - 일반적인 문제 패턴
        - 우수 성과자 식별
        - 장비 신뢰성 통계
        - 경로 효율성 비교

      출력:
        - 분기별 품질 보고서
        - 표준 업데이트 권장사항
        - 교육 프로그램 조정
        - 예산 권장사항 (장비, 인력)

[i] 피드백 루프:

    이송 팀 → QA 담당관 → 표준 위원회 → 표준 업데이트
        ↑                                                 ↓
        ← 교육 업데이트 ← 절차 개선 ← 시행 ← 검토 ←

연간 인증 갱신:

    요구사항:
      - 최소 10회 이송 완료 (지난 12개월)
      - 평균 TQS ≥ 0.85
      - 심각한 사건 없음 (수준 1)
      - 지속적 교육 이수 (16시간/년)
      - 장비 인증 최신 (듀어, 센서, 차량)

    갱신 프로세스:
      [1] 자체 평가 제출 (T-60일)
      [2] 문서 검토 (WIA 감사자)
      [3] 현장 감사 (선택, 필요 시)
      [4] 인증 결정 (T-0)
      [5] 인증서 발급 (1년 유효)


--------------------------------------------------------------------------------
장 요약
--------------------------------------------------------------------------------

이 장에서는 극저온 이송의 시작과 끝을 정의하는 시설 인수인계 절차를 다루었습니다.
이러한 절차는 대상 무결성, 규정 준수 및 지속적 품질 개선을 보장하는 데
필수적입니다.

주요 요점:

  [1] 출발 절차는 T-24시간에 시작하여 대상 준비, 듀어 준비, 차량 준비, 문서 준비를
      포함하며, 4단계 출발 프로세스(대상 이송, 듀어 적재, 문서화, 출발)를 따릅니다.

  [2] 이송 중 통신은 정기 업데이트(2시간마다), 체크포인트 업데이트, 사건 업데이트
      (즉시)를 포함하며, 4개 채널(앱, 이메일, SMS, 전화)을 사용합니다.

  [3] 도착 절차는 T-4시간 준비로 시작하여 5단계(도착 확인, 듀어 하역, 검사 및
      평가, 수령 승인/거부, 대상 이송)를 포함하며, 엄격한 승인 기준을 적용합니다.

  [4] 품질 보증은 3레벨 검증(자동, 인간 검토, 관리 승인)을 사용하며, TQS는 온도
      (40%), 경로(25%), 기간(20%), 문서화(15%)로 계산되고, ≥0.70이 허용됩니다.

  [5] 지속적 개선은 즉시 검토(48시간 내), 분기별 집계 검토, 연간 인증 갱신
      (최소 10회 이송, 평균 TQS ≥0.85)을 통해 달성됩니다.
