2.1 듀어 컨테이너 사양 및 선택
--------------------------------------------------------------------------------

모바일 극저온 이송의 핵심은 듀어 컨테이너입니다. 이는 보존된 대상을 액체 질소
(LN2) 온도에서 유지하는 특수 설계된 진공 단열 용기입니다.

[CRYO] 듀어 기본 설계 원리:

듀어는 이중벽 구조로 설계되어 내부 스테인리스 스틸 용기와 외부 쉘 사이에
진공이 형성됩니다. 이 진공 공간은 열 전달을 최소화하며 종종 다층 단열재(MLI)로
추가 강화됩니다.

    +-------------------------------------------------------------------------+
    | 듀어 구조층                                                              |
    +-------------------------------------------------------------------------+
    | [1] 외부 쉘 (스테인리스 스틸 또는 알루미늄)                              |
    | [2] 진공 단열 공간 (10^-5 torr 압력)                                     |
    | [3] 다층 단열재 (MLI) - 알루미늄 코팅 마일러 층                          |
    | [4] 내부 용기 (고급 스테인리스 스틸)                                     |
    | [5] LN2 저장소 (-196°C)                                                  |
    | [6] 대상 보관 구역 (케이지 또는 캐니스터 내)                             |
    +-------------------------------------------------------------------------+

[!] 인증된 이송 듀어 요구사항:

    +----------------------+--------------------+-----------------------------+
    | 사양                 | 요구사항           | 인증 기준                   |
    +----------------------+--------------------+-----------------------------+
    | 진공 수준            | <10^-4 torr       | 연간 진공 검증              |
    | 단열 성능            | <0.5% LN2/day     | 정적 증발률 테스트          |
    | 내부 온도            | -196°C ~ -190°C    | 연속 모니터링               |
    | 충격 저항            | 최대 5G            | 낙하 테스트 인증            |
    | 압력 릴리프          | 자동 밸브          | 안전 밸브 기능 검증         |
    | 재질                 | 의료 등급 SS       | 재질 인증서                 |
    | 보관 시간            | 최소 7-21일        | 크기 및 용도에 따라         |
    | 센서 통합            | 온도/압력/GPS      | 실시간 원격 측정            |
    +----------------------+--------------------+-----------------------------+

듀어 크기 선택 가이드:

대상 유형과 이송 기간에 따라 적절한 듀어 크기를 선택하는 것이 중요합니다.

    [전신 보존] - 200L ~ 500L 듀어
      - 일반적인 모델: MVE 1500, Taylor-Wharton CX100, Chart HEco
      - 보관 시간: 14-21일
      - 무게 (비어있음): 150-280kg
      - 무게 (적재): 300-500kg
      - 내부 직경: 55-75cm
      - 높이: 180-220cm

    [신경 보존] - 50L ~ 200L 듀어
      - 일반적인 모델: MVE XLC, Taylor-Wharton HC, Chart HC
      - 보관 시간: 10-14일
      - 무게 (비어있음): 40-120kg
      - 무게 (적재): 80-200kg
      - 내부 직경: 35-55cm
      - 높이: 80-150cm

    [장기/조직] - 10L ~ 50L 듀어
      - 일반적인 모델: MVE Lab, Taylor-Wharton LS, Chart CryoSystem
      - 보관 시간: 7-10일
      - 무게 (비어있음): 8-35kg
      - 무게 (적재): 20-70kg
      - 내부 직경: 20-35cm
      - 높이: 40-80cm

2.2 온도 센서 및 모니터링 시스템
--------------------------------------------------------------------------------

실시간 온도 모니터링은 이송 무결성의 핵심입니다. 모든 인증된 이송은 다중 센서
시스템과 지속적인 데이터 로깅을 사용해야 합니다.

[CRYO] 센서 배치 요구사항:

최소 3개의 독립적인 온도 센서가 필요합니다:
  - 센서 1: 듀어 상단부 (LN2 증기 공간)
  - 센서 2: 듀어 중간부 (액체 수준)
  - 센서 3: 듀어 하단부 (대상 근처)

각 센서는 다음 사양을 충족해야 합니다:

    +-------------------+------------------------+---------------------------+
    | 센서 사양         | 요구사항               | 인증 기준                 |
    +-------------------+------------------------+---------------------------+
    | 온도 범위         | -210°C ~ +50°C         | 전체 범위 보정            |
    | 정확도            | ±0.5°C                 | NIST 추적 가능            |
    | 해상도            | 0.1°C                  | 디지털 출력               |
    | 샘플링 속도       | 최소 1분마다           | 구성 가능한 간격          |
    | 응답 시간         | <30초                 | 온도 변화 감지            |
    | 센서 유형         | Pt100/Pt1000 RTD       | 또는 K형 열전대           |
    | 데이터 로깅       | 로컬 + 클라우드        | 자동 백업                 |
    | 배터리 수명       | 최소 30일              | 교체 가능                 |
    | 알림 시스템       | SMS/이메일/앱          | 다중 채널                 |
    +-------------------+------------------------+---------------------------+

2.3 컨테이너 고정 및 충격 보호
--------------------------------------------------------------------------------

이송 중 듀어는 심각한 물리적 힘에 노출됩니다: 가속, 감속, 회전, 진동 및
잠재적 충격. 적절한 고정은 필수적입니다.

[CRYO] 충격 및 진동 한계:

    +------------------------+------------------+-----------------------------+
    | 힘 유형                | 최대 허용        | 측정 방법                   |
    +------------------------+------------------+-----------------------------+
    | 정상 운전 중 가속      | <2G              | 3축 가속도계                |
    | 돌발 사건 (급정거)     | <5G              | 피크 기록                   |
    | 지속적인 진동          | <0.5G @ 5-50Hz   | 주파수 분석                 |
    | 회전력 (회전)          | <30°/초          | 자이로스코프                |
    | 낙하 높이 (최대)       | 0m (낙하 없음)   | 절대 금지                   |
    +------------------------+------------------+-----------------------------+

고정 시스템 설계:

모든 이송 차량은 다음을 포함하는 인증된 고정 시스템을 갖춰야 합니다:

    [1] 하단 크래들 플랫폼
        - 듀어 베이스 직경에 맞춤 설계
        - 고무/폼 라이너로 진동 감쇠
        - 차량 바닥에 볼트로 고정
        - 최소 하중 정격: 듀어 무게의 3배

    [2] 측면 지지 스트랩
        - 래칫 장력 조절 시스템
        - 패딩 처리된 접촉 지점
        - 듀어 높이의 33%, 66% 지점에 배치
        - 장력: 단단하지만 과도하지 않음 (변형 없음)

    [3] 상단 안정화 시스템
        - 수평 이동 방지
        - 차량 프레임에 부착
        - 빠른 해제 메커니즘 (응급 상황용)

    [4] 충격 흡수 재료
        - 고밀도 폼 패딩 (모든 접촉 지점)
        - 최소 5cm 두께
        - 극저온 안전 재료 (균열 없음)

[i] 고정 체크리스트:

이송 전 검증:
  [ ] 크래들 플랫폼이 차량에 안전하게 고정됨
  [ ] 듀어가 크래들에 중앙 정렬됨
  [ ] 하단 스트랩이 체결되고 장력이 조정됨
  [ ] 중간 스트랩이 체결되고 장력이 조정됨
  [ ] 상단 안정화 시스템이 부착됨
  [ ] 모든 접촉 지점에 패딩 배치됨
  [ ] 흔들림 테스트 수행 (육안으로 움직임 없음)
  [ ] 가속도계가 설치되고 테스트됨
  [ ] 빠른 해제 메커니즘 접근 가능
  [ ] 사진 문서화 완료


--------------------------------------------------------------------------------
2.4 LN2 보충 시스템 및 절차
--------------------------------------------------------------------------------

장거리 이송의 경우 경로 중간에 LN2를 보충해야 할 수 있습니다. 이는 신중하게
계획하고 실행해야 하는 중요한 절차입니다.

[CRYO] LN2 증발률 계산:

정적 증발률 (시설 보관):
  - 고품질 듀어: 0.3-0.5% LN2/일
  - 표준 듀어: 0.5-1.0% LN2/일

동적 증발률 (이송 중):
  - 지상 이송 (도로): 1.0-2.0% LN2/일
  - 항공 이송 (비행): 1.5-3.0% LN2/일
  - 거친 지형/극한 기후: 최대 4.0% LN2/일

예시 계산:

    듀어 용량: 500L
    초기 충전: 90% (450L)
    이송 기간: 72시간 (3일)
    동적 증발률: 2.0%/일

    일일 손실: 500L × 2.0% = 10L/일
    3일 후 잔여: 450L - (10L × 3) = 420L (84%)

    안전 최소값: 50% (250L)
    여유: 420L - 250L = 170L ✓ 안전함

보충 계획 요구사항:

    +-------------------------+----------------------+-----------------------+
    | 이송 기간               | 보충 필요성          | 계획 권장사항         |
    +-------------------------+----------------------+-----------------------+
    | <48시간                | 일반적으로 불필요    | 비상용 LN2 확보       |
    | 48-72시간               | 경계선               | 중간 지점 계획        |
    | >72시간                | 필수                 | 복수 중간 지점        |
    | >7일                   | 매일 모니터링        | 지역 공급업체 사전배치|
    +-------------------------+----------------------+-----------------------+

[!] LN2 보충 안전 절차:

  단계 1: 준비
    - 환기가 잘 되는 실외 지역 선택
    - 모든 점화원 제거 (차량 엔진 끔)
    - 개인 보호 장비 착용:
      * 극저온 장갑 (팔꿈치까지)
      * 안면 보호대 또는 고글
      * 극저온 앞치마
      * 폐쇄형 신발 (통기성 없음)
    - LN2 공급 듀어의 압력 확인

  단계 2: 충전
    - 이송 듀어 뚜껑을 천천히 열기
    - LN2 이송 호스를 연결 (액체 상 사용)
    - 천천히 충전 시작 (거품/비등 관찰)
    - LN2 레벨을 90%까지 충전 (100% 아님)
    - 충전률: 일반적으로 10-20L/분

  단계 3: 완료
    - 호스를 천천히 분리 (잔여 LN2 배출)
    - 듀어 뚜껑을 단단히 닫되 완전히 밀봉하지 않음
    - 압력 릴리프 밸브 기능 확인
    - 온도 센서 정상 판독 확인 (5분 내)
    - 충전 활동 로그 기록

  단계 4: 문서화
    - 충전 시간 및 위치
    - 추가된 LN2 양 (리터)
    - 충전 전후 온도
    - 수행한 인력 (인증 번호)
    - 사용된 LN2 공급원 (배치 번호)


--------------------------------------------------------------------------------
2.5 비상 백업 컨테이너
--------------------------------------------------------------------------------

모든 이송 계획은 주 듀어가 손상될 경우를 대비한 비상 백업을 포함해야 합니다.
이는 고가치 또는 민감한 이송에 특히 중요합니다.

[CRYO] 백업 전략:

    전략 A: 전용 백업 듀어 (최고 안전성)
      - 두 번째 빈 듀어를 이송 차량에 동행
      - 주 듀어와 동일하거나 더 큰 용량
      - 사전 냉각 (-196°C로)
      - 이송 시간 < 30분 (긴급 시)

    전략 B: 경로 따라 배치된 듀어 (비용 효율적)
      - 경로를 따라 3-4개의 시설과 사전 계약
      - 각 시설에 적합한 듀어 대기
      - 최대 4시간 이내 접근
      - 정기적인 준비 상태 확인

    전략 C: 모바일 백업 팀 (항공 이송)
      - 두 번째 차량이 12-24시간 뒤따름
      - 완전한 중복 장비
      - 주 팀과의 통신 유지

비상 이송 절차:

주 듀어에서 백업 듀어로의 이송 시나리오:

    [긴급 상황 트리거]
      - 주 듀어 진공 손실
      - 제어 불가능한 온도 상승
      - 물리적 손상 (사고, 낙하)
      - LN2 누출 또는 완전 손실

    [즉시 조치 - 첫 5분]
      1. 모든 관련 인력에게 알림
      2. 안전한 위치로 이동 (가능한 경우)
      3. 백업 듀어 준비 시작
      4. 주 듀어의 현재 온도 확인
      5. 응급 이송 도구 세트 배치

    [이송 실행 - 5-30분]
      1. 백업 듀어가 -196°C인지 확인
      2. 양쪽 듀어 뚜껑 열기
      3. 대상을 주 듀어에서 신속하게 리프팅
      4. 즉시 백업 듀어로 배치
      5. 백업 듀어 닫기 및 밀봉
      6. 온도 안정화 확인 (10분)

    [이송 후 - 30분+]
      1. 모든 센서 데이터 확인
      2. 사건 완전히 문서화
      3. 주 듀어 손상 평가
      4. 보험/법적 통지 개시
      5. 원래 일정 조정 또는 계속

[i] 비상 이송 키트 내용물:

  필수 도구:
    - 극저온 리프팅 집게 또는 그래플
    - 극저온 장갑 (여러 쌍)
    - 안면 보호대 (여러 개)
    - LED 헤드램프 (핸즈프리 작업용)
    - 예비 온도 센서
    - 휴대용 LN2 소형 듀어 (10-20L)

  문서화 도구:
    - 방수 사건 보고서 양식
    - 카메라 (타임스탬프 사진용)
    - 음성 녹음기 (실시간 내레이션용)

  통신:
    - 위성 전화 (원격 지역용)
    - 비상 연락처 목록 (방수)
    - 신호 플레어 (극한 응급 상황)


--------------------------------------------------------------------------------
장 요약
--------------------------------------------------------------------------------

이 장에서는 안전한 극저온 이송을 위한 모바일 컨테이너 시스템의 기술적 요구사항과
운영 절차를 다루었습니다. 올바른 듀어 선택, 센서 시스템, 고정 방법 및 비상
절차를 이해하는 것은 성공적인 이송의 기초입니다.

주요 요점:

  [1] 듀어 선택은 대상 유형, 이송 기간, 보관 시간 요구사항을 기반으로 해야 하며,
      전신은 200-500L, 신경은 50-200L, 장기는 10-50L 듀어가 필요합니다.

  [2] 최소 3개의 독립적인 온도 센서(상단, 중간, 하단)가 필요하며, 1분마다 샘플링
      하고 ±0.5°C 정확도로 -196°C 주변의 온도를 모니터링해야 합니다.

  [3] 듀어 고정 시스템은 정상 운전 중 2G, 돌발 사건 시 5G까지의 가속력을 견딜 수
      있어야 하며, 하단 크래들, 측면 스트랩, 상단 안정화를 포함해야 합니다.

  [4] LN2 보충은 72시간 이상의 이송에 필요하며, 동적 증발률은 1.0-4.0%/일이고,
      환기가 잘 되는 실외 지역에서 적절한 PPE를 착용하고 수행해야 합니다.

  [5] 모든 이송 계획은 주 듀어 고장 시 30분 이내에 대상을 이송할 수 있는 비상
      백업 듀어 전략을 포함해야 하며, 완전한 비상 이송 키트를 갖춰야 합니다.
