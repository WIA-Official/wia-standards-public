제7장: 국제 운송 규정
================================================================================--------------------------------------------------------------------------------
7.1 국제 항공 운송 협회 (IATA) 규정
--------------------------------------------------------------------------------

항공 이송은 위험물(DG) 분류 및 포장 요구사항을 포함하는 IATA 위험물 규정(DGR)을
준수해야 합니다.

[CRYO] IATA 위험물 분류:

    극저온 액체 (액체 질소):
      UN 번호: UN1977
      적절한 선적 명칭: "Nitrogen, refrigerated liquid"
      클래스: 2.2 (비인화성, 무독성 가스)
      포장 그룹: 해당 없음 (극저온 액체)
      라벨: "Non-flammable Gas" + "Keep Frozen"

    생물학적 재료 (보존된 대상):
      UN 번호: UN3373 (진단 표본) 또는 면제
      분류: Division 6.2 (감염 물질) 또는 면제
      포장: PI650 (면제 인간 표본)
      라벨: "Biological Substance, Category B"

[!] IATA 포장 요구사항:

    듀어 컨테이너:
      - IATA PI 202 준수 (극저온 액체)
      - 압력 릴리프 밸브 (필수)
      - 이차 격리 (누출 시)
      - 외부 라벨링:
        * UN1977 (LN2)
        * UN3373 (생물학적, 필요 시)
        * "Cryogenic Liquid"
        * "This Way Up" (방향 화살표)
        * 비상 연락처 정보

    문서 요구사항:
      - 위험물 신고서 (DGD)
      - 송하인 적합성 선언
      - 항공화물 운송장 (AWB)
      - 수입/수출 허가 (국가별)
      - 동물 검역 인증서 (해당 시)

    인력 교육:
      - IATA DG 교육 (카테고리 6: 패커)
      - 2년마다 재인증
      - 교육 기록 3년 보관

항공 이송 제한사항:

    +------------------------+-------------------------+-----------------------------+
    | 제한 사항              | 승객 항공기             | 화물 전용 항공기            |
    +------------------------+-------------------------+-----------------------------+
    | LN2 최대량             | 제한적 (일반적으로 금지) | 무제한 (항공사 승인 시)     |
    | 듀어 크기              | 소형만 (<50L)          | 모든 크기 (승인된 경우)     |
    | 포장 요구사항          | PI 202                  | PI 202                      |
    | 사전 승인              | 항공사 + 공항           | 항공사                      |
    | 비상 대응 계획         | 필수                    | 필수                        |
    +------------------------+-------------------------+-----------------------------+

7.2 미국 운송부 (DOT) 규정
--------------------------------------------------------------------------------

미국 내 지상 이송은 DOT 위험물 규정(HMR)을 준수해야 합니다.

[CRYO] DOT 위험물 분류:

    극저온 액체 (액체 질소):
      DOT 클래스: 2.2 (비인화성 가스)
      ID 번호: UN1977
      포장 그룹: 해당 없음
      라벨: "Non-flammable Gas"
      플래카드: "Non-flammable Gas" (1001 lbs 이상)

    생물학적 재료:
      DOT 클래스: Division 6.2 또는 면제
      ID 번호: UN3373 (카테고리 B)
      포장: 49 CFR §173.196
      라벨: "Biological Substance"

DOT 운송 요구사항:

    차량 표시:
      - 플래카드 (4면, 1001 lbs 이상)
      - DOT 번호 (상업 차량)
      - 회사 이름 및 주소
      - 비상 대응 전화번호

    운전자 자격:
      - 상업 운전 면허 (CDL) - 해당 시
      - 위험물 인증 (HME)
      - DOT 신체 검사
      - 운전 기록 확인

    운송 문서:
      - 선적 서류 (Shipping Paper)
        * UN1977, Nitrogen, refrigerated liquid, 2.2
        * 수량 (kg 또는 L)
        * 비상 연락처 번호 (24시간)
      - 비상 대응 가이드북 (ERG)
      - 사고 보고 절차

    휴게 및 운전 시간 제한:
      - 최대 운전: 11시간/일
      - 최대 근무: 14시간/일
      - 필수 휴식: 10시간 연속
      - 주간 제한: 60시간/7일 또는 70시간/8일


--------------------------------------------------------------------------------
7.3 유럽 위험물 도로 운송 협정 (ADR)
--------------------------------------------------------------------------------

유럽 내 이송은 ADR (Accord Dangerous Goods by Road)을 준수해야 합니다.

[CRYO] ADR 분류 및 요구사항:

    극저온 액체:
      UN 번호: UN1977
      클래스: 2
      분류 코드: 3A
      포장 그룹: 해당 없음
      터널 제한 코드: E
      제한 수량: 0 (해당 없음)

    차량 요구사항:
      - ADR 승인 차량
      - 소화기 (최소 2개, 6kg ABC)
      - 안전 장비:
        * 경고 삼각대 (2개)
        * 반사 조끼 (각 승무원)
        * 손전등
        * 보안경 및 장갑
      - 비상 정보:
        * Tremcard (운송 비상 카드)
        * 안전 데이터 시트 (SDS)

    운전자 교육:
      - ADR 운전자 교육 인증
      - 5년마다 갱신
      - 전문화 교육:
        * 클래스 2 (가스)
        * 탱크 차량 (해당 시)

국경 간 이송:

    쉥겐 지역 (자유 이동):
      - 일반적으로 국경 검사 없음
      - ADR 문서 휴대 필수
      - 국가별 규정 준수

    비쉥겐 또는 EU 외부:
      - 세관 신고
      - 수입/수출 허가
      - 동물 검역 인증서 (필요 시)
      - 번역된 문서 (현지 언어)

7.4 아시아-태평양 지역 규정
--------------------------------------------------------------------------------

아시아-태평양 국가들은 다양한 규정을 갖고 있으며, 일반적으로 UN 권고사항을
기반으로 합니다.

[CRYO] 주요 국가별 요구사항:

    일본:
      규제 기관: PMDA (의약품의료기기종합기구)
      요구사항:
        - 수입 허가 (생물학적 재료)
        - 일본어 라벨링
        - 현지 대리인 (필수)
        - 세관 사전 신고
      운송: UN 권고사항 기반 (IATA/DOT 유사)

    중국:
      규제 기관: CAAC (민항국) / 세관
      요구사항:
        - 특별 수입 허가 (생물학적 재료)
        - 중국어 문서 (필수)
        - 현지 파트너 (권장)
        - GPS 데이터 로컬 저장 (법률)
        - 사전 검사 (때때로)
      운송: GB 12268 표준 (위험물)

    호주:
      규제 기관: CASA (민간항공안전청)
      요구사항:
        - 생물안전 수입 허가
        - 동물 검역 검사
        - ADG Code 준수 (호주 위험물 코드)
        - 주 간 이송 허가 (일부)
      운송: ADG Code (UN 기반)

    한국:
      규제 기관: 식품의약품안전처
      요구사항:
        - 생물학적 재료 수입 신고
        - 한국어 라벨링
        - 안전 인증 (특정 품목)
      운송: IMDG/IATA 준수

[!] 공통 아시아-태평양 과제:

    언어 장벽:
      - 현지어 문서 필요
      - 통역사 또는 대리인 권장
      - 번역 인증 (때때로)

    문화적 고려사항:
      - 사망 및 보존에 대한 태도
      - 가족 동의 요구사항
      - 종교적 민감성

    인프라 차이:
      - LN2 공급 가용성 다양
      - 도로 품질 및 표지판
      - 통신 시스템


--------------------------------------------------------------------------------
7.5 국제 이송 계획 및 조정
--------------------------------------------------------------------------------

국제 이송은 광범위한 사전 계획과 다중 당사자 조정을 필요로 합니다.

[CRYO] 국제 이송 타임라인:

    T-90일: 초기 계획
      [ ] 경로 및 모드 결정 (지상/항공/해상)
      [ ] 규제 요구사항 조사 (양국)
      [ ] 현지 파트너 식별
      [ ] 비용 견적 및 예산 승인

    T-60일: 허가 및 승인
      [ ] 수입/수출 허가 신청
      [ ] 항공사/운송사 승인 확보
      [ ] 동물 검역 인증서 (필요 시)
      [ ] 생물안전 허가
      [ ] 보험 정책 확인

    T-30일: 물류 확정
      [ ] 항공편/차량 예약 확인
      [ ] LN2 공급 사전 배치
      [ ] 비상 백업 시설 식별
      [ ] 이송 팀 교육 (국가별 요구사항)
      [ ] 통신 프로토콜 (국제 로밍/위성)

    T-7일: 최종 준비
      [ ] 모든 문서 재검토 (번역 포함)
      [ ] 장비 검사 및 테스트
      [ ] 기상 예보 확인
      [ ] 이해관계자 브리핑
      [ ] 비상 연락처 배포

    T-0: 이송 실행
      [ ] 출발 전 최종 검사
      [ ] 세관/국경 문서 준비
      [ ] 실시간 추적 활성화
      [ ] 정기 상태 업데이트

    T+24h: 이송 후
      [ ] 도착 확인 및 검사
      [ ] 세관 통관
      [ ] 규정 준수 보고서 제출
      [ ] 경험 교훈 검토

국제 조정 체크리스트:

    +-------------------------+----------------------+------------------------+
    | 영역                    | 출발 국가            | 도착 국가              |
    +-------------------------+----------------------+------------------------+
    | 규제 승인               | 수출 허가            | 수입 허가              |
    | 생물안전                | 건강 인증서          | 검역 승인              |
    | 세관                    | 수출 신고            | 수입 관세 및 세금      |
    | 운송                    | ADR/DOT 준수         | 현지 규정 준수         |
    | 시설                    | 출발 시설 인증       | 수신 시설 인증         |
    | 인력                    | 이송 팀 자격         | 현지 지원 팀           |
    +-------------------------+----------------------+------------------------+


--------------------------------------------------------------------------------
장 요약
--------------------------------------------------------------------------------

이 장에서는 극저온 이송에 적용되는 복잡한 국제 운송 규정 환경을 다루었습니다.
각 관할권에는 고유한 요구사항이 있지만, 대부분은 UN 권고사항을 기반으로 하여
어느 정도 조화를 제공합니다.

주요 요점:

  [1] IATA 규정은 항공 이송을 지배하며, LN2를 UN1977(클래스 2.2)로, 생물학적
      재료를 UN3373(필요 시)으로 분류하고, PI 202 포장 및 위험물 신고서가
      필요합니다.

  [2] 미국 DOT 규정은 지상 이송에 플래카드, HME 인증, 선적 서류, 운전 시간 제한
      (11시간/일)을 요구하며, 상업 운송에는 CDL이 필요합니다.

  [3] 유럽 ADR은 UN1977(클래스 2, 터널 코드 E), ADR 승인 차량, Tremcard, 5년
      운전자 인증을 요구하며, 쉥겐 지역에서 국경 간 이동을 단순화합니다.

  [4] 아시아-태평양 국가들은 다양한 요구사항(일본 PMDA, 중국 특별 허가, 호주
      생물안전, 한국 KFDA)을 갖고 있으며, 일반적으로 현지어 문서와 현지 파트너가
      필요합니다.

  [5] 국제 이송 계획은 90일 타임라인(T-90: 초기 계획, T-60: 허가, T-30: 물류,
      T-7: 최종 준비, T-0: 실행, T+24h: 사후)을 따르며, 다중 당사자 조정이
      필요합니다.

복습 질문
--------------------------------------------------------------------------------

Q1: IATA 위험물 규정 하에서 액체 질소(LN2)의 UN 번호, 클래스, 적절한 선적
    명칭은 무엇이며, 필요한 라벨은 무엇입니까?

Q2: 미국 DOT 규정 하에서 상업 극저온 이송 운전자의 자격 요구사항(CDL, HME,
    신체 검사)과 운전 시간 제한은 무엇입니까?

Q3: 유럽 ADR 차량의 필수 안전 장비와 운전자가 휴대해야 하는 비상 정보 문서를
    나열하십시오.

Q4: 아시아-태평양 지역의 4개 주요 국가(일본, 중국, 호주, 한국)의 규제 기관과
    각각의 고유한 요구사항을 비교하십시오.

Q5: 국제 이송 계획 타임라인의 5가지 주요 단계(T-90, T-60, T-30, T-7, T-0)와
    각 단계에서 완료해야 하는 주요 활동을 설명하십시오.

Q6: 국경 간 이송에서 언어 장벽, 문화적 고려사항, 인프라 차이가 제기하는 과제는
    무엇이며, 어떻게 해결할 수 있습니까?
