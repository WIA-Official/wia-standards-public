③ Notes for Reviewers

This simulator is a single-page, dependency-free demonstration of the conformance gates documented in the parent specification. It is not a substitute for the conformance test suite — it is a pedagogical companion that lets reviewers exercise the boundary conditions interactively. All evaluations occur in-browser; no payload leaves the page.

  • Replay window is 300 seconds; nonce reuse triggers a 409 response (Phase 3 §F.6 vector).
  • Oversize payloads are rejected with code E_LIMIT_BYTES (Phase 1 §F.4).
  • Plain bearer tokens fail at L3 because they lack sender constraint (Phase 8 §1.2).
  • Field-mask omissions are non-destructive (Phase 2 §P.2.4).
