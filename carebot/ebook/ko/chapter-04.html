4.1 이동성의 중요성

이동성은 독립성과 삶의 질의 기반입니다. 노인과 장애인에게 이동성 제한은 연쇄적인 부정적 효과를 만듭니다: 신체 활동 감소는 근육 약화로 이어지고, 이는 낙상 위험을 증가시키며, 낙상에 대한 두려움과 활동 감소로 이어집니다. 케어 로봇은 낙상을 예방하면서 활동을 장려하는 안전하고 신뢰할 수 있는 이동 보조를 제공하여 이 악순환을 끊을 수 있습니다.

WIA-CAREBOT은 로봇이 제공할 수 있는 세 가지 수준의 이동 보조를 정의하며, 각각 특정 기술 요구사항과 안전 고려사항을 갖습니다:

수준 설명 로봇 기능
모니터링 수동적 관찰 및 알림 낙상 감지, 보행 분석, 활동 추적
안내 내비게이션 및 보행 동반 경로 안내, 장애물 경고, 속도 조절 지원
물리적 지원 직접적인 신체 보조 보행 지원, 이송 보조, 낙상 예방

4.2 낙상 감지 시스템

낙상은 노인의 부상과 사망의 주요 원인입니다. 신속한 감지와 대응은 결과를 크게 개선할 수 있습니다. WIA-CAREBOT은 거짓 음성(누락된 낙상)을 최소화하면서 허용 가능한 거짓 양성 비율을 유지하는 다중 모달 낙상 감지 시스템을 지정합니다.

낙상 감지를 위한 센서 융합

// 다중 모달 낙상 감지 시스템
class FallDetectionSystem {
    async monitorContinuously(): Promise<void> {
        while (true) {
            const sensorData = await this.collectSensorData();
            const prediction = await this.analyzeFallRisk(sensorData);
            if (prediction.fallDetected) await this.handleFallEvent(prediction);
            else if (prediction.fallRisk > 0.7) await this.issuePreemptiveWarning(prediction);
            await this.sleep(50);  // 20Hz 모니터링
        }
    }

    private async analyzeFallRisk(data: SensorData): Promise<FallPrediction> {
        const fusedFeatures = this.fuseFeatures({
            radar: this.extractRadarFeatures(data.radar),
            visual: this.extractVisualFeatures(data.cameras)
        });
        const prediction = await this.mlModel.predict(fusedFeatures);
        return { fallDetected: prediction.fallProbability > 0.85, fallRisk: prediction.fallProbability };
    }
}

레이더 기반 낙상 감지

밀리미터파 레이더는 카메라 없이 프라이버시를 보호하면서 낙상 감지를 제공합니다. 의복과 침구를 통해 신체 위치와 움직임 패턴을 감지하므로 프라이버시가 가장 중요한 욕실과 침실 모니터링에 이상적입니다.

# ROS 2 레이더 낙상 감지 노드
# carebot_fall_detection/src/radar_fall_detector.py

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import PointCloud2
from carebot_msgs.msg import FallAlert, PersonState

class RadarFallDetector(Node):
    def __init__(self):
        super().__init__('radar_fall_detector')

        # 파라미터
        self.declare_parameter('fall_height_threshold', 0.5)  # 미터
        self.declare_parameter('fall_velocity_threshold', 1.5)  # m/s

        # 구독자
        self.radar_sub = self.create_subscription(
            PointCloud2,
            '/sensors/radar/point_cloud',
            self.radar_callback, 10
        )

        # 발행자
        self.fall_pub = self.create_publisher(FallAlert, '/safety/fall_alert', 10)
        self.state_pub = self.create_publisher(PersonState, '/monitoring/person_state', 10)

    def analyze_fall_pattern(self):
        if len(self.position_history) < 10:
            return False, None

        # 수직 속도 계산
        heights = [p['position'][2] for p in self.position_history[-10:]]
        vertical_velocity = (heights[-1] - heights[0]) / 0.5  # 0.5초 동안

        height_threshold = self.get_parameter('fall_height_threshold').value
        velocity_threshold = self.get_parameter('fall_velocity_threshold').value

        is_fall = (heights[-1] < height_threshold and
                   vertical_velocity < -velocity_threshold)

        return is_fall, {'height': heights[-1], 'velocity': vertical_velocity}

4.3 보행 보조 시스템

보행 보조 로봇은 이동 장애가 있는 사용자에게 물리적 지원과 안내를 제공합니다. 이 시스템은 속도를 조절하고 장애물 경고를 제공하는 단순한 보행 동반자부터 체중을 지탱하고 보행을 안정시키는 능동 지원 시스템까지 다양합니다.

수동적 보행 동반자

// 보행 동반자 행동 컨트롤러
class WalkingCompanion {
    private navigation: NavigationSystem;
    private personTracker: PersonTracker;
    private speechOutput: SpeechSynthesis;
    private gaitAnalyzer: GaitAnalyzer;

    async accompanyWalk(userId: string, destination?: Location): Promise<void> {
        const user = await this.personTracker.getUser(userId);
        this.setState('accompanying');

        while (this.state === 'accompanying') {
            const userPose = await this.personTracker.getCurrentPose(userId);
            const userGait = await this.gaitAnalyzer.analyze(userId);

            // 사용자 옆에 로봇 위치
            const targetPose = this.calculateCompanionPose(userPose, {
                lateralOffset: 0.6,  // 측면 60cm
                longitudinalOffset: -0.2,  // 약간 뒤
                side: user.preferences.companionSide || 'left'
            });

            await this.navigation.moveToward(targetPose, {
                maxSpeed: Math.min(userGait.speed * 1.1, 0.8),
                avoidOvertaking: true
            });

            // 피드백 제공
            await this.provideWalkingFeedback(userGait);

            // 앞쪽 장애물 확인
            const obstacles = await this.detectUpcomingObstacles(userPose);
            if (obstacles.length > 0) {
                await this.warnAboutObstacles(obstacles);
            }

            await this.sleep(100);
        }
    }

    private async provideWalkingFeedback(gait: GaitAnalysis): Promise<void> {
        if (gait.asymmetry > 0.3) {
            this.speechOutput.speak(
                "왼쪽을 조금 편하게 하시는 것 같아요. 잠시 쉬어가실래요?"
            );
        }
    }
}

4.4 보행 분석 및 모니터링

지속적인 보행 분석은 가치 있는 건강 통찰력과 잠재적 문제에 대한 조기 경고를 제공합니다. 보행 패턴의 변화는 통증, 약화, 약물 효과 또는 질병 진행을 나타낼 수 있습니다.

// 보행 분석 시스템
interface GaitMetrics {
    cadence: number;           // 분당 걸음 수
    strideLength: number;      // 보폭 길이 (미터)
    leftRightSymmetry: number; // 1.0 = 완벽한 대칭
    gaitVariability: number;   // 변동 계수
    fallRiskScore: number;     // 0-100, 높을수록 위험
}

class GaitAnalyzer {
    async analyzeGait(frames: VideoFrame[]): Promise<GaitMetrics> {
        const poses = await Promise.all(frames.map(f => this.poseEstimator.estimate(f)));
        const metrics = this.computeMetrics(this.extractFootTrajectories(poses));
        metrics.fallRiskScore = this.calculateFallRisk(metrics);
        return metrics;
    }

    private calculateFallRisk(metrics: GaitMetrics): number {
        let risk = 0;
        if (metrics.strideLength * metrics.cadence / 60 < 0.8) risk += 25;
        if (metrics.gaitVariability > 0.05) risk += 20;
        if (metrics.leftRightSymmetry < 0.9) risk += 15;
        return Math.min(risk, 100);
    }
}

4.5 이송 보조

앉은 자세와 선 자세 사이의 이송, 또는 침대와 휠체어 사이의 이송은 고위험 활동입니다. 케어 로봇은 사용자의 안전과 존엄성을 유지하면서 상당한 도움을 제공할 수 있습니다.

이송 유형:
- 앉기-일어서기: 의자, 침대 또는 화장실에서 일어나기
- 일어서기-앉기: 앉은 자세로 제어된 내려가기
- 측면 이송: 인접한 표면 사이 이동
- 피벗 이송: 지지를 받으며 회전 
// 이송 보조 서버
class TransferAssistServer {
    private forceLimit = 150.0;  // 뉴턴
    private velocityLimit = 0.1;  // m/s

    async executeSitToStand(goalHandle, userWeight): Promise<TransferResult> {
        // 1단계: 준비 - 사용자가 손잡이를 잡음
        await this.announce("손잡이를 단단히 잡아주세요");
        await this.waitForGripConfirmation();

        // 2단계: 앞으로 기울이기
        await this.announce("살짝 앞으로 기울어주세요");
        await this.guideForwardLean();

        // 3단계: 일어서기 보조
        await this.announce("다리로 밀면서 제가 도와드릴게요");

        let currentHeight = 0.45;  // 앉은 높이
        const targetHeight = 0.9;   // 선 높이

        while (currentHeight < targetHeight) {
            const userForce = await this.readHandleForces();
            const supportForce = this.calculateSupport(userForce, userWeight);

            if (Math.abs(supportForce) > this.forceLimit) {
                await this.pauseAndReassess();
                continue;
            }

            await this.applyLiftForce(supportForce);
            currentHeight = await this.getUserHeight();
        }

        // 4단계: 안정화
        await this.stabilizeStanding();

        return { success: true, finalPosition: 'standing' };
    }
}

4.6 재활 지원

재활 로봇은 치료 운동과 부상 또는 질병으로부터의 회복을 지원합니다. 정밀하고 반복 가능한 움직임을 제공하고, 진행 상황을 측정하며, 개별 환자 능력에 적응합니다.

// 재활 운동 시스템
class RehabilitationSystem {
    private exerciseLibrary: ExerciseLibrary;
    private motionTracker: MotionTracker;
    private feedbackSystem: FeedbackSystem;

    async conductSession(userId: string, exercisePlan: ExercisePlan): Promise<SessionResult> {
        const results: ExerciseResult[] = [];

        for (const exercise of exercisePlan.exercises) {
            await this.announceExercise(exercise);
            await this.demonstrateExercise(exercise);

            const exerciseResult = await this.guideExercise(exercise, userId);
            results.push(exerciseResult);

            // 휴식 및 피드백
            await this.provideFeedback(exerciseResult);
            await this.rest(exercise.restDuration);
        }

        return {
            userId,
            completedExercises: results.filter(r => r.completed).length,
            totalExercises: exercisePlan.exercises.length,
            overallScore: this.calculateOverallScore(results),
            recommendations: this.generateRecommendations(results)
        };
    }

    private async guideExercise(exercise: Exercise, userId: string): Promise<ExerciseResult> {
        const motionData = await this.motionTracker.track(userId, exercise.duration);
        const accuracy = this.compareToIdeal(motionData, exercise.idealMotion);

        // 실시간 교정 피드백
        if (accuracy < 0.7) {
            await this.feedbackSystem.correctForm(exercise, motionData);
        }

        return {
            exerciseId: exercise.id,
            completed: accuracy > 0.5,
            accuracy,
            duration: exercise.duration,
            notes: this.generateExerciseNotes(motionData)
        };
    }
}

4.7 낙상 예방 개입

낙상 감지를 넘어 케어 로봇은 불안정성이 감지되면 낙상을 예방하기 위해 능동적으로 개입할 수 있습니다. 이를 위해서는 빠른 대응과 해를 끼치지 않는 개입 방법에 대한 신중한 고려가 필요합니다.

// 사전적 낙상 예방 시스템
class FallPreventionSystem {
    private balanceMonitor: BalanceMonitor;
    private audioSystem: AudioSystem;

    async monitorAndIntervene(): Promise<void> {
        while (this.isActive) {
            const balance = await this.balanceMonitor.assess();

            if (balance.instabilityLevel > 0.8) {
                // 임박한 낙상 - 즉각 개입
                await this.executeImmediateIntervention(balance);
            } else if (balance.instabilityLevel > 0.5) {
                // 우려되는 불안정성 - 음성 경고
                await this.issueVerbalWarning(balance);
            }

            await this.sleep(20);  // 50Hz 모니터링
        }
    }

    private async executeImmediateIntervention(balance: BalanceAssessment): Promise<void> {
        await Promise.all([
            this.audioSystem.playAlert('stability_warning'),
            this.baseController.moveToward(balance.userPosition, { speed: 0.5 })
        ]);

        await this.logInterventionEvent(balance);
    }
}
