7장: 제조 프로세스

7.1 첨단 패키징 제조 개요

첨단 패키징 제조는 전통적인 백엔드 패키징을 넘어 프론트엔드 팹 수준의 공정 정밀도를 요구합니다. TSV 형성, 미세 피치 범핑, 하이브리드 본딩 등은 나노미터 수준의 정밀도와 클린룸 환경이 필수입니다.

프론트엔드와 백엔드의 융합: 첨단 패키징은 반도체 제조의 프론트엔드(웨이퍼 팹)와 백엔드(OSAT)의 경계를 허물고 있습니다. 파운드리들이 패키징까지 수직 통합하는 추세가 강화되고 있습니다.

7.4.1 열압착 본딩 (TCB)

// TCB 프로세스 파라미터
interface TCBProcess {
    // 온도 프로파일
    thermal: {
        preheatTemp: number;     // °C
        bondingTemp: number;     // °C (220-280)
        coolingRate: number;     // °C/s
    };

    // 압력 프로파일
    force: {
        placement: number;       // N
        bonding: number;        // N (higher)
        perBump: number;        // N/bump
    };

    // 시간
    time: {
        bondingTime: number;    // seconds
        dwellTime: number;      // seconds
    };

    // 정확도
    accuracy: {
        placement: number;      // µm (±2)
        coplanarity: number;    // µm
    };
}

7.4.2 하이브리드 본딩

  1. 표면 준비: 원자 수준 평탄화 (Ra < 0.5nm)
  2. 활성화: 플라즈마 처리로 표면 에너지 증가
  3. 정렬: 서브미크론 정확도 정렬
  4. 상온 접합: 산화물-산화물 접합
  5. 어닐링: 금속 확산 (200-400°C)
하이브리드 본딩의 핵심: 표면 청정도와 평탄도가 성공의 열쇠입니다. 파티클 관리와 CMP 공정 최적화가 수율을 결정합니다.

// 복합 수율 계산
function calculateCompositeYield(steps: ProcessStep[]): number {
    return steps.reduce((yield, step) => yield * step.yield, 1.0);
}

// 예시: 4 다이 칩렛 패키지
const chipletPackageYield = calculateCompositeYield([
    { name: 'Die1 KGD', yield: 0.98 },
    { name: 'Die2 KGD', yield: 0.98 },
    { name: 'Die3 KGD', yield: 0.98 },
    { name: 'Die4 KGD', yield: 0.98 },
    { name: 'Assembly', yield: 0.95 },
    { name: 'Test', yield: 0.99 }
]);
// 결과: 약 86%
