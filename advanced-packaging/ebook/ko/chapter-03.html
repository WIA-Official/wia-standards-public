3장: 인터커넥트 기술

3.1 인터커넥트의 중요성

첨단 패키징에서 인터커넥트는 시스템 성능의 핵심 결정 요인입니다. 다이 간 연결의 대역폭, 지연, 전력 효율은 전체 시스템의 성능을 좌우합니다. 미세 피치 인터커넥트 기술의 발전은 첨단 패키징의 핵심 동력입니다.

인터커넥트 성능 지표: 데이터 레이트(Gb/s), 에너지 효율(pJ/bit), 밀도(connections/mm²), 레이턴시(ns)가 핵심 지표입니다. 첨단 인터커넥트는 모든 지표에서 혁신적인 개선을 추구합니다.

// 범프 사양 정의
interface BumpSpec {
    type: 'C4' | 'microbump' | 'Cu_pillar' | 'hybrid';
    pitch: number;          // µm
    diameter: number;       // µm
    height: number;         // µm
    material: {
        bump: 'SnAg' | 'Cu' | 'Au';
        ubm: string[];      // Under Bump Metallization

    };

    // 전기적 특성
    electrical: {
        resistance: number;  // mΩ
        inductance: number;  // pH
        capacitance: number; // fF
    };

    // 기계적 특성
    mechanical: {
        shearStrength: number;    // MPa
        fatigueLife: number;      // cycles
        thermalCycleRange: string;
    };
}

3.3 하이브리드 본딩

하이브리드 본딩은 Cu-Cu 금속 연결과 유전체 접합을 동시에 수행하는 기술입니다:

3.3.1 프로세스 단계

  1. 표면 준비: CMP로 원자 수준 평탄화
  2. 플라즈마 활성화: 표면 에너지 증가
  3. 정렬 및 본딩: 상온 유전체 접합
  4. 어닐링: Cu-Cu 금속 확산 연결

3.3.2 장점

  • 초미세 피치 (< 10µm) 가능
  • 솔더 없는 직접 연결
  • 낮은 저항, 높은 밀도
  • 탁월한 열 전도성
밀도 혁신: 하이브리드 본딩은 마이크로범프 대비 100배 이상의 연결 밀도를 제공합니다. 이는 3D 적층의 새로운 가능성을 열어갑니다.

3.4 UCIe (Universal Chiplet Interconnect Express)

// UCIe 사양
interface UCIeSpec {
    version: '1.0' | '1.1' | '2.0';

    // 물리 계층
    phy: {
        standardPackage: {
            pitch: 110;       // µm (bump)
            lanes: 16;        // minimum
            dataRate: 4;      // Gb/s/lane
        };
        advancedPackage: {
            pitch: 25;        // µm (microbump)
            lanes: 64;        // minimum
            dataRate: 32;     // Gb/s/lane
        };
    };

    // 프로토콜 계층
    protocol: {
        streaming: 'flit-based';
        flitSize: 256;        // bits
        crcType: 'CRC-16';
    };

    // 다이-투-다이 어댑터
    d2dAdapter: {
        protocols: ['PCIe', 'CXL', 'custom'];
        rawBandwidth: number;
        efficiency: 0.85;     // protocol overhead
    };
}

// 대역폭 계산
function calculateBandwidth(config: UCIeSpec): number {
    const phy = config.phy.advancedPackage;
    const rawBW = phy.lanes * phy.dataRate / 8;  // GB/s
    return rawBW * config.d2dAdapter.efficiency;
}

// TSV 설계 규칙
interface TSVDesignRule {
    diameter: number;       // µm
    pitch: number;          // µm
    keepOut: number;        // µm from active devices
    aspectRatio: number;    // depth/diameter

    // 전기적 모델
    model: {
        resistance: number;   // mΩ
        inductance: number;   // pH
        capacitance: number;  // fF
    };

    // 기계적 고려
    mechanical: {
        stressKeepOut: number;  // µm
        cteMatch: boolean;
        annealRequired: boolean;
    };
}

3.6 신호 무결성

  • 크로스토크: 인접 배선 간 간섭 최소화
  • 임피던스 매칭: 반사 손실 방지
  • 전원 무결성: IR 드롭 및 노이즈 관리
  • EMI: 전자기 간섭 차폐

3.7 미래 인터커넥트

  • 광 인터커넥트: 초고속, 저전력 광 연결
  • 무선 인터커넥트: 패키지 내 RF/mmWave 연결
  • 나노와이어: 원자 수준 배선
