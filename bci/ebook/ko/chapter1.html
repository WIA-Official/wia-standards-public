1.1 뇌-컴퓨터 인터페이스(BCI)란?

뇌-컴퓨터 인터페이스(BCI)는 뇌와 외부 장치 간의 직접적인 통신 경로입니다. BCI는 개인이 근육 움직임 없이 뇌 신호만으로 컴퓨터, 의수, 휠체어 및 기타 보조 기술을 제어할 수 있게 합니다.

신호 유형

  • EEG (뇌전도) - 뇌의 전기적 활동 기록
  • EMG (근전도) - 근육의 전기적 활동 기록
  • EOG (안전도) - 안구 운동 기록
  • 운동 상상 (Motor Imagery) - 실제 움직임 없이 정신적 운동 시뮬레이션

WIA BCI 표준의 목적

WIA BCI 표준은 다음을 제공합니다:

  • 통합 데이터 형식 - 모든 신경 신호 유형을 위한 단일 표준
  • 장치 독립적 API - 동일한 코드로 모든 BCI 장치 연결
  • 실시간 프로토콜 - 반응성 애플리케이션을 위한 저지연 스트리밍
  • 접근성 통합 - TTS, 수어, 점자에 대한 기본 지원
  • 설계 단계의 개인정보 보호 - 내장 암호화 및 동의 관리

4단계 표준

1단계: 데이터 형식 - 신경 데이터 저장 및 교환 (제2장)

2단계: API 인터페이스 - 장치 연결 및 데이터 스트리밍 (제3장)

3단계: 통신 프로토콜 - 실시간 저지연 전송 (제4장)

4단계: 생태계 통합 - WIA 출력 시스템 통합 (제5장)

1.2 사용 사례

의료 및 임상

  • ALS/폐쇄 증후군 환자를 위한 의사소통
  • 뇌졸중 재활
  • 간질 모니터링

접근성

  • 휠체어 제어
  • 컴퓨터 액세스
  • 다중 모드 출력 (음성, 수어, 점자)
