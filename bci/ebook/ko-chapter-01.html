1장: 신경 신호 획득

전극 기술 및 신호 기록 기초

1.1 신경 신호 기록의 기초

신경 신호 획득은 모든 뇌-컴퓨터 인터페이스 시스템의 기초를 형성합니다. 기록된 신호의 품질은 전체 BCI 시스템의 기능과 신뢰성을 직접적으로 결정합니다. WIA-BCI 표준은 신호 획득 하드웨어에 대한 포괄적인 요구사항을 수립하여 다양한 구현과 제조업체 간에 일관된 성능을 보장합니다.

신경 신호는 뉴런의 전기화학적 활동에서 발생합니다. 뉴런이 통신할 때 이온 전류가 막을 가로질러 흐르며, 적절하게 배치된 전극으로 감지할 수 있는 전기 전위를 생성합니다. 문제는 근육 활동, 전력선 및 기타 소스의 노이즈가 훨씬 더 클 수 있는 상황에서 이러한 작은 신호(마이크로볼트에서 밀리볼트)를 캡처하는 것입니다.

1.2 전극 기술

전극 기술의 선택은 BCI 시스템의 기능을 근본적으로 결정합니다. 다양한 전극 유형은 신호 품질, 침습성, 수명 및 공간 해상도 간의 다양한 절충안을 제공합니다. WIA-BCI 표준은 각 전극 범주에 대한 요구사항을 명시합니다.

1.2.1 표면 전극 (비침습)

표면 전극 사양 (WIA-BCI-E-001):

전극 유형: 습식 Ag/AgCl
  - 임피던스: 10 Hz에서 < 5 kOhm
  - 직경: 8-10 mm 표준
  - 접촉 매체: 전도성 젤/페이스트
  - 준비: 피부 마모로 < 5 kOhm 달성
  - 수명: 일회용 권장

전극 유형: 건식 능동
  - 임피던스: 10 Hz에서 < 50 kOhm
  - 접촉 핀: 다중 스프링 장착
  - 내장 증폭: 필수
  - 준비 불필요
  - 수명: 일반적으로 1000회 이상 사용

전극 유형: 반건식
  - 임피던스: 10 Hz에서 < 20 kOhm
  - 하이드로젤 접촉 매체
  - 최소 준비
  - 빠른 설정: < 5분
  - 수명: 50-100회 사용

전극 배치 표준

WIA-BCI 표준은 전극 배치를 위해 국제 10-20 시스템을 채택하며, 고밀도 응용을 위한 확장(10-10 및 10-5 시스템)을 포함합니다. 주요 위치는 다음과 같습니다:

  • Fp1, Fp2: 전두극 (전전두엽 피질)
  • F3, F4, Fz: 전두 (운동 계획, 실행 기능)
  • C3, C4, Cz: 중심 (일차 운동/감각 피질)
  • P3, P4, Pz: 두정 (감각 통합)
  • O1, O2, Oz: 후두 (시각 처리)
  • T3, T4: 측두 (청각, 언어)

1.2.2 두개내 전극 (침습)

침습 전극은 훨씬 높은 신호 품질과 공간 해상도를 제공하지만 수술적 이식이 필요합니다. WIA-BCI 표준은 모든 이식 가능한 전극 시스템에 대해 엄격한 생체적합성 및 장기 신뢰성 요구사항을 명시합니다.

전극 유형 위치 채널 신호 유형 해상도
ECoG 그리드 피질 표면 32-256 LFP, ECoG ~1 cm
ECoG 스트립 피질 표면 4-16 LFP, ECoG ~1 cm
유타 어레이 피질내 96-128 스파이크, LFP 400 um
미시간 프로브 피질내 32-256 스파이크, LFP 100-200 um
뉴로픽셀 심부 뇌 384-960 스파이크, LFP 25 um

1.3 신호 증폭 및 조건화

원시 신경 신호는 디지털화 전에 상당한 증폭과 조건화가 필요합니다. 프론트엔드 전자장치는 낮은 노이즈를 유지하고 공통 모드 간섭을 제거하면서 높은 이득을 제공해야 합니다. WIA-BCI 표준은 증폭 시스템에 대한 최소 성능 요구사항을 명시합니다. 

신호 조건화 요구사항 (WIA-BCI-S-001):

증폭기 사양
  이득: 1000-50000x (조절 가능)
  입력 임피던스: > 100 MOhm
  CMRR: 50/60 Hz에서 > 100 dB
  입력 노이즈: < 1 uVrms (0.1-100 Hz)
  대역폭: 0.1-500 Hz (EEG), DC-10 kHz (스파이크)
  DC 오프셋 허용: +/- 300 mV

필터링 단계
  고역 통과: 0.1-1 Hz (구성 가능)
  저역 통과: 100-500 Hz (EEG), 5-10 kHz (스파이크)
  노치 필터: 50/60 Hz (전환 가능)
  앨리어싱 방지: 최소 4차

아날로그-디지털 변환
  해상도: 최소 16-24 비트
  샘플링 속도: 250-1000 Hz (EEG), 20-30 kHz (스파이크)
  입력 범위: +/- 5 mV ~ +/- 500 mV
  채널 수: 8-1024 채널
  동기화: < 1 샘플 지터

1.4 증폭기 회로 설계

WIA-BCI 표준은 신경 증폭기 프론트엔드에 대한 참조 설계를 제공합니다. 이러한 설계는 광범위한 테스트를 통해 검증되었으며 새로운 BCI 구현에 권장됩니다. 계측 증폭기 구성은 전기적으로 노이즈가 많은 환경에서 마이크로볼트 수준의 신호를 기록하는 데 필수적인 우수한 공통 모드 제거를 제공합니다. 

// WIA-BCI 참조 증폭기 구성
// EEG/ECoG 기록용 계측 증폭기

class NeuralAmplifier {
    // 구성 매개변수
    private double gain = 10000;           // 총 이득
    private double highPassCutoff = 0.5;   // Hz
    private double lowPassCutoff = 100;    // Hz
    private double samplingRate = 500;     // Hz
    private double inputImpedance = 1e9;   // 1 GOhm

    // 증폭기 단계
    Stage1: InstrumentationAmplifier {
        gain: 100
        CMRR: 120 dB
        inputNoise: 0.5 uVrms
    }

    Stage2: HighPassFilter {
        type: Butterworth
        order: 2
        cutoff: highPassCutoff
    }

    Stage3: GainStage {
        gain: 100
        bandwidth: 10 kHz
    }

    Stage4: LowPassFilter {
        type: Butterworth
        order: 4
        cutoff: lowPassCutoff
    }

    Stage5: NotchFilter {
        frequency: [50, 60] // 양쪽 전력선 주파수
        Q: 30
        attenuation: 40 dB
    }

    Stage6: ADC {
        resolution: 24 bits
        samplingRate: samplingRate
        inputRange: +/- 5 mV
    }
}

1.5 노이즈 소스 및 완화

노이즈를 이해하고 제어하는 것은 성공적인 신경 신호 획득에 매우 중요합니다. WIA-BCI 표준은 주요 노이즈 소스를 식별하고 인증된 시스템에서 구현해야 하는 완화 전략을 명시합니다.

주요 노이즈 소스

  • 열 노이즈: 전극 임피던스의 존슨-나이퀴스트 노이즈, sqrt(4kTRB)에 비례
  • 전력선 간섭: AC 전원 및 고조파에서 50/60 Hz 픽업
  • EMG 아티팩트: 근육 활동(10-500 Hz), 특히 두피 및 안면 근육
  • EOG 아티팩트: 눈 움직임과 깜박임으로 큰 전위 생성(100+ uV)
  • ECG 아티팩트: 특히 EEG 몽타주에서 심장박동 관련 전위
  • 움직임 아티팩트: 움직임에 의한 전극-피부 인터페이스 변화
  • 증폭기 노이즈: 능동 전자장치의 입력 참조 노이즈
노이즈 예산 분석 (WIA-BCI 요구사항):

소스                | 크기        | 완화 전략
--------------------+-------------+----------------------------------
전극 열             | 0.5-2 uVrms | 저임피던스 전극, 젤
전력선              | 50-500 uV   | 차폐, 노치 필터, CMRR
EMG 오염            | 10-100 uV   | 공간 필터링, ICA, 이완
EOG 아티팩트        | 50-500 uV   | 회귀, ICA, 거부
ECG 아티팩트        | 5-20 uV     | 적응형 필터링, ICA
움직임 아티팩트     | 10-500 uV   | 안전한 장착, 거부
증폭기 노이즈       | 0.3-1 uVrms | 저노이즈 설계, 평균화

총 노이즈 플로어 목표: < 2 uVrms (EEG)
신호 대 잡음비 요구사항: BCI 특징에 대해 > 10 dB

1.6 아티팩트 감지 및 거부

실시간 아티팩트 감지는 신뢰할 수 있는 BCI 작동에 필수적입니다. WIA-BCI 표준은 자동 감지 알고리즘 및 사용자 알림 프로토콜을 포함한 아티팩트 처리에 대한 최소 요구사항을 명시합니다. 시스템은 오염된 데이터 세그먼트를 식별하고 적절하게 처리해야 합니다. 

// WIA-BCI 아티팩트 감지 알고리즘 참조

class ArtifactDetector {
    // 감지 임계값 (구성 가능)
    amplitudeThreshold: 100 uV      // 절대 진폭
    gradientThreshold: 50 uV/ms     // 변화율
    varianceThreshold: 5x baseline  // 분산 비율

    function detectArtifacts(signal, windowSize) {
        artifacts = []

        // 진폭 검사
        for (sample in signal) {
            if (abs(sample) > amplitudeThreshold) {
                artifacts.add({
                    type: "AMPLITUDE",
                    position: sample.index,
                    severity: abs(sample) / amplitudeThreshold
                })
            }
        }

        // 기울기 검사 (근육 아티팩트)
        gradient = diff(signal)
        for (g in gradient) {
            if (abs(g) > gradientThreshold) {
                artifacts.add({
                    type: "GRADIENT",
                    position: g.index,
                    severity: abs(g) / gradientThreshold
                })
            }
        }

        // 분산 검사 (움직임 아티팩트)
        for (window in slidingWindow(signal, windowSize)) {
            localVar = variance(window)
            if (localVar > varianceThreshold * baselineVariance) {
                artifacts.add({
                    type: "VARIANCE",
                    position: window.center,
                    severity: localVar / baselineVariance
                })
            }
        }

        return mergeOverlapping(artifacts)
    }

    function handleArtifact(artifact, mode) {
        switch (mode) {
            case "REJECT":
                return null  // 세그먼트 폐기
            case "INTERPOLATE":
                return linearInterpolate(artifact.segment)
            case "ATTENUATE":
                return artifact.segment * (1 / artifact.severity)
            case "FLAG":
                return {data: artifact.segment, flag: true}
        }
    }
}

현대 BCI는 종종 수십에서 수백 개의 채널이 있는 고밀도 전극 어레이를 사용합니다. WIA-BCI 표준은 전극 간의 공간적 관계가 정확하게 유지되도록 다채널 동기화 요구사항을 명시합니다.

기본 EEG 8-32 순차 ADC 100 us 128 kbps
고밀도 EEG 64-256 병렬 ADC 10 us 12 Mbps
ECoG 어레이 64-256 동기화 클럭 1 us 25 Mbps

1.8 무선 전송 요구사항

많은 BCI 응용 프로그램은 이동성을 지원하고 이식된 시스템의 감염 위험을 줄이기 위해 무선 데이터 전송이 필요합니다. WIA-BCI 표준은 지연, 신뢰성 및 전력 효율성을 포함한 무선 성능 요구사항을 명시합니다. 

무선 BCI 요구사항 (WIA-BCI-W-001):

응용별 지연 요구사항
  통신 BCI: 종단간 < 100 ms
  운동 제어 BCI: 종단간 < 50 ms
  커서 제어: 종단간 < 30 ms
  의수 제어: 종단간 < 20 ms

신뢰성 요구사항
  패킷 손실률: < 0.1%
  비트 오류율: < 10^-9
  링크 가용성: > 99.9%
  우아한 저하: 필수

전력 요구사항 (이식형)
  전송 전력: < 10 mW (SAR 한계)
  평균 전력: 총 < 50 mW
  배터리 수명: > 5년 또는 무선 충전
  무선 충전: Qi 호환 또는 RF

주파수 대역
  MICS (의료 임플란트): 402-405 MHz
  ISM 대역: 2.4-2.5 GHz
  UWB: 3.1-10.6 GHz (저전력)
  맞춤형 의료: 규제 승인 필요

1.9 신호 품질 지표

WIA-BCI 표준은 신호 품질을 평가하기 위한 객관적인 지표를 정의합니다. 이러한 지표는 BCI 작동 중에 지속적으로 모니터링되어야 하며 적절한 경우 사용자와 임상의 모두에게 보고되어야 합니다.

필수 품질 지표

  • 임피던스: 1 kHz에서 전극-조직 임피던스, 채널별 보고
  • SNR: 특징 관련 대역에서 계산된 dB 단위의 신호 대 잡음비
  • 아티팩트 비율: 아티팩트로 인해 거부된 데이터 세그먼트 비율
  • 스펙트럼 품질: 예상 신경 리듬(알파, 베타 등)의 존재
  • 정상성 지수: 시간에 따른 신호 안정성 측정
  • 채널 상관관계: 품질 평가를 위한 채널 간 상관 행렬
// 신호 품질 평가 구현

class SignalQualityMonitor {
    function assessQuality(signal, channel) {
        metrics = {}

        // SNR 계산
        signalPower = bandpower(signal, featureBand)
        noisePower = bandpower(signal, noiseBand)
        metrics.snr = 10 * log10(signalPower / noisePower)

        // 아티팩트 비율 계산
        artifactSamples = countArtifacts(signal)
        metrics.artifactPercent = artifactSamples / signal.length * 100

        // 스펙트럼 품질 검사
        psd = welch(signal)
        metrics.alphaPresent = checkPeak(psd, 8, 13)
        metrics.betaPresent = checkPeak(psd, 13, 30)

        // 정상성
        metrics.stationarity = calculateStationarity(signal)

        // 전체 품질 점수
        metrics.overallScore = weightedAverage([
            {value: metrics.snr, weight: 0.3, normalize: x => x/20},
            {value: 100-metrics.artifactPercent, weight: 0.3, normalize: x => x/100},
            {value: metrics.stationarity, weight: 0.2},
            {value: metrics.alphaPresent + metrics.betaPresent, weight: 0.2}
        ])

        return metrics
    }

    function reportQuality(metrics) {
        if (metrics.overallScore > 0.8) return "우수"
        if (metrics.overallScore > 0.6) return "양호"
        if (metrics.overallScore > 0.4) return "허용"
        if (metrics.overallScore > 0.2) return "불량"
        return "허용 불가"
    }
}

이 장에서는 WIA-BCI 표준에 따른 신경 신호 획득의 기본 요구사항을 다루었습니다. 적절한 신호 획득은 모든 후속 처리와 해석이 의존하는 기초를 형성합니다. 다음 장에서는 신호 처리 알고리즘, 기기 인증 요구사항 및 임상 응용에 대해 설명합니다.
