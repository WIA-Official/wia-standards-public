# WIA Medical Device Accessibility: Phase 1 Research

## 1. Executive Summary

본 문서는 의료기기 접근성 표준화를 위한 기술 조사 결과를 정리합니다.

**조사 일자:** 2025-12
**Version:** 1.0.0

---

## 2. 현재 산업 현황

### 2.1 규제 환경 (2024-2025)

#### U.S. Access Board 최종 규칙 (2024년 7월)

2024년 7월 25일, U.S. Access Board는 의료 진단 장비(MDE)의 접근성에 관한 최종 규칙을 발표했습니다:

- **낮은 이동 높이**: 앙와위, 복와위, 측와위 및 좌위에서 사용되는 MDE에 대해 17인치의 낮은 이동 높이 지정
- **2017년 표준 개정**: 기존 17-19인치 허용 범위에서 17인치로 확정
- **발효일**: 2024년 9월 23일

#### DOJ ADA Title II 규칙 (2024년 8월)

2024년 8월 9일, 법무부는 ADA Title II에 대한 최종 규칙을 발표했습니다:

- **2024년 10월 8일부터**: 주 및 지방 정부 기관이 구매/임대하는 모든 MDE는 접근 가능해야 함
- **2026년 8월 9일까지**:
  - 최소 1개의 접근 가능한 검사 테이블 보유 필수
  - 최소 1개의 접근 가능한 체중계 보유 필수

#### Medical Device Nonvisual Accessibility Act (법안)

Jan Schakowsky 의원이 발의한 법안 (H.R.1328):

- **대상**: Class II 및 Class III 가정용 의료기기
- **요구사항**: FDA가 비시각적 접근성 규정 제정
- **시행**: 법 제정 후 12개월 내 규칙 제안, 24개월 내 최종 규칙 발표
- **효과**: 접근성 표준 미충족 기기 판매 금지

### 2.2 주요 과제

| 과제 | 설명 | 영향받는 사용자 |
|------|------|----------------|
| 시각적 디스플레이 의존 | 대부분 의료기기가 시각적 UI만 제공 | 시각 장애인 |
| 복잡한 인터페이스 | 작은 버튼, 터치스크린 의존 | 운동 장애인, 고령자 |
| 청각 알림만 제공 | 경보음에만 의존하는 알림 | 청각 장애인 |
| 인지적 복잡성 | 복잡한 메뉴, 불명확한 지시 | 인지 장애인 |
| 물리적 접근성 | 높은 이동 높이, 휠체어 비호환 | 이동 장애인 |

## 3. 기존 표준 및 프로토콜

### 3.1 국제 표준

#### IEC 62366-1:2015 (의료기기 사용성 공학)

- **범위**: 의료기기의 사용성 관련 안전 요구사항
- **목적**: 사용 오류 관련 위험 분석, 명세, 개발, 평가
- **개정**: IEC 62366-1:2015/Amd 1:2020
- **연관 표준**: ISO 14971 (위험 관리)
- **인정**: FDA 인정 합의 표준

```
IEC 62366 프로세스:
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  Use        │───▶│  User       │───▶│  Usability  │───▶│  Usability  │
│  Specification│   │  Research   │    │  Validation │    │  File       │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

#### ISO 14971 (의료기기 위험 관리)

- 의료기기 전 생애주기에 걸친 위험 관리 프로세스
- IEC 62366과 밀접하게 연계

#### MDE Standards (U.S. Access Board)

36 CFR Part 1195에 명시된 의료 진단 장비 접근성 표준:

- 검사 테이블
- 체중계
- 진단 영상 장비
- 유방 촬영 장비
- 기타 진단 장비

### 3.2 미국 규제

| 기관 | 표준/지침 | 대상 |
|------|----------|------|
| **FDA** | 21 CFR Part 820 | 품질 시스템 규정 |
| **FDA** | Human Factors Guidance | 사용성 엔지니어링 |
| **Access Board** | MDE Standards | 진단 장비 접근성 |
| **DOJ** | ADA Title II/III | 공공장소 접근성 |
| **HHS** | Section 508 | 전자 정보 기술 |

### 3.3 유럽 규제

| 규제 | 설명 |
|------|------|
| **MDR 2017/745** | EU 의료기기 규정 |
| **DiGAV** | 독일 디지털 건강 앱 규정 (접근성 증명 요구) |
| **EN 301 549** | ICT 제품 및 서비스 접근성 |

## 4. 주요 기업 및 제품

### 4.1 접근성 기능 있는 의료기기

#### 연속 혈당 모니터 (CGM)

| 제품 | 제조사 | 접근성 기능 |
|------|--------|------------|
| Dexcom G7 | Dexcom | 음성 읽기, 햅틱 알림 |
| FreeStyle Libre 3 | Abbott | 스마트폰 앱 통합 |
| Guardian 4 | Medtronic | 음성 안내 |

#### 혈압계

| 제품 | 접근성 기능 |
|------|------------|
| Talking Blood Pressure Monitors | 음성 결과 안내 |
| Large Display Models | 저시력자용 대형 화면 |
| Bluetooth Connected | 스마트폰 앱 연동 |

#### 웨어러블 건강 기기

| 유형 | 예시 | 접근성 고려사항 |
|------|------|----------------|
| 스마트워치 | Apple Watch, Galaxy Watch | VoiceOver/TalkBack 지원 |
| 피트니스 트래커 | Fitbit, Garmin | 제한적 접근성 |
| 의료용 웨어러블 | Cala Trio (떨림 치료) | FDA 승인, 손목 착용 |

### 4.2 보조 의료기기

- **Cala Trio**: 손 떨림 감소를 위한 신경조절 웨어러블
- **전자 BCG 양말**: 원격 환자 모니터링
- **스마트 인슐린 펜**: 자동 용량 기록

### 4.3 접근 가능한 진단 장비

- **접근 가능 검사 테이블**: 17인치 이동 높이
- **휠체어 접근 가능 체중계**: 경사로 또는 플랫폼형
- **높이 조절 영상 장비**: 전동 높이 조절

## 5. 기술 트렌드

### 5.1 디지털 헬스 기술

```
┌─────────────────────────────────────────────────────────────────┐
│                  디지털 헬스 접근성 트렌드                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. 스마트폰 통합                                                 │
│     └── 의료기기 → 스마트폰 앱 → 접근성 기능 활용                   │
│                                                                  │
│  2. 음성 인터페이스                                               │
│     └── 음성 명령, 음성 안내, 대화형 인터페이스                     │
│                                                                  │
│  3. 햅틱 피드백                                                   │
│     └── 진동 알림, 촉각 안내                                      │
│                                                                  │
│  4. AI 기반 적응                                                 │
│     └── 사용자 패턴 학습, 개인화된 인터페이스                       │
│                                                                  │
│  5. 원격 모니터링                                                 │
│     └── 가정용 기기 → 클라우드 → 의료진                           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 포용적 디자인 원칙

1. **유니버설 디자인**: 처음부터 모든 사용자 고려
2. **다중 감각 피드백**: 시각 + 청각 + 촉각
3. **유연한 인터페이스**: 사용자 맞춤 설정
4. **단순성**: 인지 부담 최소화
5. **오류 방지**: 사용 오류 최소화

### 5.3 신흥 기술

| 기술 | 의료 적용 | 접근성 잠재력 |
|------|----------|--------------|
| **AR/VR** | 재활, 치료 | WIA XR 표준 적용 가능 |
| **AI/ML** | 진단, 모니터링 | 적응형 인터페이스 |
| **IoT** | 연결된 건강 | 다중 기기 접근성 |
| **Wearables** | 지속적 모니터링 | 비침습적, 편안한 |
| **Voice AI** | 의료 상담 | 시각 장애인 지원 |

## 6. 건강 형평성과 접근성

### 6.1 격차 분석

```
접근성 격차:

고소득 지역 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 높은 접근성
              ↑
              │ 격차
              ↓
저소득 지역 ━━━━━━━━━ 낮은 접근성

장애 유형별:
시각 장애 ━━━━━━━━━━━━━━ 중간 (음성 기기 증가)
청각 장애 ━━━━━━━━━━━━━━━━━━━━ 높음 (시각적 대안 용이)
운동 장애 ━━━━━━━━━━ 낮음-중간 (물리적 제약)
인지 장애 ━━━━━ 낮음 (복잡성 문제)
```

### 6.2 개선 기회

1. **비용 접근성**: 보험 적용, 보조금 프로그램
2. **교육**: 사용자 및 의료진 훈련
3. **표준화**: 일관된 접근성 요구사항
4. **혁신**: 저비용 접근성 솔루션

## 7. WIA 통합 기회

### 7.1 기존 WIA 표준과의 연계

| WIA 표준 | 의료기기 적용 |
|----------|--------------|
| **Exoskeleton** | 재활 의료기기, 보조 장치 |
| **Bionic Eye** | 시각 장애인용 의료기기 인터페이스 |
| **Voice-Sign** | 청각 장애인용 의료 커뮤니케이션 |
| **XR Accessibility** | VR 재활, 치료 |
| **Smart Wheelchair** | 이동 보조, 의료기기 통합 |
| **Haptic** | 촉각 피드백 표준화 |

### 7.2 제안 통합 영역

```typescript
interface WIAMedicalIntegration {
  // 기존 WIA 프로토콜 활용
  protocols: {
    exoskeleton: "재활기기 연동";
    bionicEye: "시각 인터페이스 대안";
    voiceSign: "의료 커뮤니케이션";
    haptic: "촉각 알림 표준";
  };

  // 의료 특화 확장
  medical: {
    deviceProfile: "기기 접근성 프로필";
    userProfile: "사용자 의료 접근성 프로필";
    alarmSystem: "다중 감각 알람";
    dataExchange: "의료 데이터 접근성";
  };
}
```

## 8. 결론 및 권고사항

### 8.1 핵심 발견

1. **규제 강화**: 2024-2025 미국에서 의료기기 접근성 규제 대폭 강화
2. **표준 존재**: IEC 62366 등 사용성 표준 존재하나 접근성 특화 부족
3. **기술 발전**: 디지털 헬스, AI, 웨어러블 기술이 접근성 개선 기회 제공
4. **격차 존재**: 장애 유형별, 지역별 의료기기 접근성 격차 큼

### 8.2 WIA Medical 표준 권고사항

1. **다중 감각 인터페이스**: 모든 정보에 시각/청각/촉각 대안 필수
2. **FDA 비시각 접근성 선제 대응**: 예정된 FDA 규정에 맞는 표준
3. **WIA 생태계 통합**: 기존 WIA 기기와 의료기기 연동
4. **사용자 프로필 기반**: 개인화된 접근성 설정 지원
5. **원격 모니터링 접근성**: 가정용 의료기기 접근성 표준
