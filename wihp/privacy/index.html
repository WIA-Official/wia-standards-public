Privacy Policy

WIHP Chrome Extension

🛡️ WIHP does NOT collect, store, or transmit any user data. All processing happens locally on your device.

📋 Overview

WIHP (WIA International Hangul Phonology) is a browser extension that converts text from various languages into Korean Hangul pronunciation. This privacy policy explains how we handle user information.

🚫 Data We Do NOT Collect

  • Personal identification information
  • Browsing history
  • Text you convert (processed locally only)
  • IP addresses
  • Cookies or tracking data
  • Any analytics or usage statistics

💾 Data Storage

The only data stored locally on your device is your language preference (EN/KO) for the extension interface. This is stored using Chrome's local storage API and never leaves your device.

🔐 Permissions Explained

  • activeTab: Access selected text on the current tab for conversion
  • contextMenus: Provide "Convert to Hangul" option in right-click menu
  • storage: Save your language preference locally

🌐 Third-Party Services

WIHP does not use any third-party services, analytics, or external APIs. All phonetic conversion algorithms run entirely within your browser.

👶 Children's Privacy

WIHP is safe for users of all ages. Since we do not collect any personal data, there are no special concerns regarding children's privacy.

📝 Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
