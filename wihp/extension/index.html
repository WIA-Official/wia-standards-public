📦 Installation Guide

1

Download & Extract ZIP

Click the button above to download ZIP and extract to a folder.

2

Open Chrome Extensions Page

Type chrome://extensions and press Enter

3

Enable Developer Mode

Toggle Developer mode ON at top right

4

Load Extension

Click Load unpacked → Select extracted folder

5

Done! 🎉

WIHP icon appears at top right of browser. Click to use!
