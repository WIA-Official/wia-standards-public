## 왜 한글인가?

### 세계 문자 체계 비교

| 문자 | 유형 | IPA 매핑 가능성 |
|------|------|----------------|
| 로마자 | 알파벳 | 제한적 (1:多) |
| 아랍 문자 | 아브자드 | 모음 생략 |
| 데바나가리 | 아부기다 | 자음+모음 결합 |
| 한자 | 표의문자 | 불가능 |
| **한글** | **자질 문자** | **완전 (1:1)** |

### 한글의 과학적 설계

```
ㄱ = 혀뿌리가 목구멍을 막는 모양 (연구개음)
ㄴ = 혀가 윗잇몸에 닿는 모양 (치조음)
ㅁ = 입술 모양 (양순음)
ㅅ = 이빨 모양 (치음)
ㅇ = 목구멍 모양 (후음/비음)
```

**세종대왕의 훈민정음 창제 원리가 600년 후, 전 세계 7,000개 언어를 위한 표준이 됩니다.**
