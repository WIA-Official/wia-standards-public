## 홍익인간 (弘益人間)
**널리 인간을 이롭게**

WIA Talk 시스템은 다음 원칙에 기반합니다:

1. **평등**: 모든 언어는 동등한 가치를 가짐
2. **포용**: 멸종위기 언어 포함
3. **접근성**: 청각장애인과 비장애인 모두를 위한 설계
4. **보존**: 언어 다양성 보호

## ALL FIVE ARE EQUAL. NO DEFAULT EXISTS.

5개 요소 (손모양, 위치, 움직임, 방향, 비수지신호)는 모두 동등하며,
어떤 요소도 기본값으로 생략되지 않습니다.
