## 주요 언어 상세

### 56. Монгол хэл (Mongolian) [mn]

#### 모음조화
- 남성모음: а, о, у
- 여성모음: э, ө, ү
- 중성모음: и

#### 예시
| 표현 | 의미 |
|------|------|
| Сайн байна уу | 안녕하세요 |
| Баярлалаа | 감사합니다 |
