| 항목 | 내용 |
|------|------|
| 언어명 | 하우사어 (Hausa / هَوْسَ) |

| 어족 | 아프로아시아어족 > 차드어파 |
| 문자 체계 | 라틴 문자 (보코), 아랍 문자 (아자미) |
