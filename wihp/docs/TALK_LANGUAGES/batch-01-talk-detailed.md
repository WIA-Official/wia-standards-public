| 항목 | 내용 |
|------|------|
| **어족** | 인도유럽어족 > 로망스어파 > 이베로로망스어 |
| **문자 체계** | 라틴 문자 (27자) |
| **화자 수** | 약 1억 2,900만 |
| **주요 지역** | 멕시코, 미국 남서부 |
| **음운 특징** | Seseo (θ→s), Yeísmo (ʎ→j), 나와틀어 영향 |

## 8.2 es와의 주요 차이점

| 특징 | 스페인 (es) | 멕시코 (es-MX) | WIA Talk 처리 |
|------|-------------|---------------|---------------|
| c/z 발음 | /θ/ | /s/ (Seseo) | HS-07-MV-08 (s와 동일) |
| ll 발음 | /ʎ/ | /j/ (Yeísmo) | HS-07-MV-01 (j와 동일) |
| j/g 발음 | /x/ (강함) | /h/ (약함) | MV-08 약하게 |
| s 유지 | 유지 | 유지 | HS-07-MV-08 |

## 8.3 자음 차이

| 자음 | 스페인 IPA | 멕시코 IPA | WIA Talk 매핑 |
|------|-----------|-----------|--------------|
| c (e,i 앞) | /θ/ | /s/ | HS-07-LC-09-MV-08 |
| z | /θ/ | /s/ | HS-07-LC-09-MV-08 |
| ll | /ʎ/ | /j/ | HS-07-LC-09-MV-01 |
| j | /x/ | /h/ | HS-02-LC-07-MV-01 (약하게) |

## 8.4 예시 문장

| 표현 | IPA (멕시코) | 의미 |
|------|-------------|------|
| Hola | /ˈola/ | 안녕하세요 |
| Gracias | /ˈgɾasjas/ | 감사합니다 (Seseo) |
| ¿Qué onda? | /ˈke ˈonda/ | 어떻게 지내? (멕시코 구어) |
| México | /ˈmehiko/ | 멕시코 |
| Chocolate | /tʃokoˈlate/ | 초콜릿 (나와틀어 유래) |

## 8.5 멕시코 스페인어 특이사항

- **Seseo**: /θ/ 음소가 없음, 모두 /s/로 발음
- **Yeísmo**: /ʎ/와 /j/ 합류
- **나와틀어 영향**: chocolate, tomate 등 차용어
