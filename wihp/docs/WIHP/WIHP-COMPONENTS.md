# WIHP 3대 컴포넌트 상세
## Place - Manner - Airstream

**버전**: 1.0
**작성일**: 2025-12-21

---

## 개요

WIHP는 모든 음소를 3대 자질로 분해합니다:

```
음소 = PL (조음 위치) + MN (조음 방법) + AR (기류 특성)

예시:
/p/ = PL01 (양순) + MN01 (파열) + AR01 (평음) → ㅂ
/pʰ/ = PL01 (양순) + MN01 (파열) + AR02 (기식) → ㅍ
/b/ = PL01 (양순) + MN01 (파열) + AR04 (유성) → ㅂ
```

---

## 1. PL: 조음 위치 (Place of Articulation)

소리가 만들어지는 위치

### PL01: 양순음 (Bilabial)
```
위치: 두 입술
IPA: /p, b, m, ɸ, β/
한글: ㅂ, ㅍ, ㅃ, ㅁ
예시: "papa" /papa/ → 파파
```

### PL02: 순치음 (Labiodental)
```
위치: 아랫입술 + 윗니
IPA: /f, v, ɱ/
한글: ㅍ, ㅸ (확장)
예시: "five" /faɪv/ → 파이브
```

### PL03: 치음 (Dental)
```
위치: 혀 + 윗니
IPA: /θ, ð/
한글: ㅆ, ㄷ (근사)
예시: "think" /θɪŋk/ → 씽크
```

### PL04: 치조음 (Alveolar)
```
위치: 혀 + 치조 (윗니 뒤 잇몸)
IPA: /t, d, n, s, z, l, r, ɾ/
한글: ㄷ, ㅌ, ㄸ, ㄴ, ㅅ, ㅆ, ㄹ
예시: "top" /tɒp/ → 탑
```

### PL05: 후치조음 (Postalveolar)
```
위치: 치조 뒤
IPA: /ʃ, ʒ, tʃ, dʒ/
한글: ㅅ(시), ㅈ, ㅊ
예시: "she" /ʃiː/ → 쉬
```

### PL06: 권설음 (Retroflex)
```
위치: 혀 말아올림
IPA: /ʈ, ɖ, ʂ, ʐ, ɳ, ɭ/
한글: ㄷ̤, ㅅ̤ (확장, 아래점)
예시: 힌디어, 타밀어
```

### PL07: 경구개음 (Palatal)
```
위치: 경구개 (입천장 앞)
IPA: /c, ɟ, ɲ, ç, ʝ, j/
한글: ㅈ(확장), ㅣ
예시: "yes" /jes/ → 예스
```

### PL08: 연구개음 (Velar)
```
위치: 연구개 (입천장 뒤)
IPA: /k, g, ŋ, x, ɣ/
한글: ㄱ, ㅋ, ㄲ, ㅇ(종성)
예시: "go" /goʊ/ → 고
```

### PL09: 구개수음 (Uvular)
```
위치: 목젖
IPA: /q, ɢ, ɴ, χ, ʁ/
한글: ㄱ(확장), ㅎ(확장)
예시: 아랍어, 프랑스어 R
```

### PL10: 인두음 (Pharyngeal)
```
위치: 인두
IPA: /ħ, ʕ/
한글: ㅎ(확장)
예시: 아랍어 ح, ع
```

### PL11: 성문음 (Glottal)
```
위치: 성문 (성대)
IPA: /ʔ, h, ɦ/
한글: ㅎ, ㆆ(옛한글)
예시: "uh-oh" /ʔʌʔoʊ/ → 어ㆆ오ㆆ
```

## 2. MN: 조음 방법 (Manner of Articulation)

소리가 만들어지는 방식

### MN01: 파열음 (Plosive/Stop)
```
방법: 공기 흐름 완전 차단 후 방출
IPA: /p, t, k, b, d, g, ʔ/
한글: ㅂ, ㄷ, ㄱ, ㅍ, ㅌ, ㅋ, ㅃ, ㄸ, ㄲ
특징: 가장 기본적인 자음
```

### MN02: 비음 (Nasal)
```
방법: 코로 공기 흐름
IPA: /m, n, ŋ, ɲ, ɴ/
한글: ㅁ, ㄴ, ㅇ(종성), ㆁ(옛한글)
특징: 콧소리
```

### MN03: 전동음 (Trill)
```
방법: 혀 또는 입술 진동
IPA: /r, ʀ, ʙ/
한글: ㄹ(확장, 떨림)
예시: 스페인어 rr, 프랑스어 R
```

### MN04: 탄음 (Tap/Flap)
```
방법: 혀 한 번 튕김
IPA: /ɾ, ɽ/
한글: ㄹ
예시: 한국어 "사이" 의 ㄹ
```

### MN05: 마찰음 (Fricative)
```
방법: 좁은 틈으로 공기 마찰
IPA: /f, v, s, z, ʃ, ʒ, x, h/
한글: ㅅ, ㅆ, ㅎ, ㅍ(근사)
특징: 쉬 소리
```

### MN06: 접근음 (Approximant)
```
방법: 살짝 좁힘 (마찰 없음)
IPA: /w, j, ɹ, l/
한글: ㅜ/ㅗ, ㅣ, ㄹ
예시: "we" /wiː/, "yes" /jes/
```

### MN07: 측면음 (Lateral)
```
방법: 혀 옆으로 공기 흐름
IPA: /l, ɭ, ʎ, ʟ/
한글: ㄹ
예시: "let" /let/
```

### MN08: 파찰음 (Affricate)
```
방법: 파열 + 마찰 결합
IPA: /ts, dz, tʃ, dʒ/
한글: ㅈ, ㅊ, ㅉ
예시: "church" /tʃɜːtʃ/ → 처치
```

## 3. AR: 기류 특성 (Airstream Mechanism)

공기 흐름의 특성

### AR01: 평음 (Plain)
```
특성: 기본 발음
한글: ㄱ, ㄷ, ㅂ, ㅈ
IPA: 무표
```

### AR02: 기식음 (Aspirated)
```
특성: 공기 방출 강함
한글: ㅋ, ㅌ, ㅍ, ㅊ
IPA: /pʰ, tʰ, kʰ, tʃʰ/
표기: +ʰ
```

### AR03: 경음 (Tense/Fortis)
```
특성: 성문 긴장, 강한 폐쇄
한글: ㄲ, ㄸ, ㅃ, ㅆ, ㅉ
IPA: /p͈, t͈, k͈/ (또는 /pp, tt, kk/)
```

### AR04: 유성음 (Voiced)
```
특성: 성대 진동
한글: 표기 없음 (문맥으로 구분)
IPA: /b, d, g, v, z/
확장: 옛한글 ㅸ 활용
```

### AR05: 방출음 (Ejective)
```
특성: 성문 닫고 공기 방출
IPA: /pʼ, tʼ, kʼ, sʼ/
한글: +ʼ 표시 (확장)
예시: 그루지야어, 나바호어
```

### AR06: 내파음 (Implosive)
```
특성: 공기 흡입
IPA: /ɓ, ɗ, ɠ/
한글: +점 (확장)
예시: 스와힐리어, 베트남어
```

### AR07: 클릭음 (Click)
```
특성: 혀로 공기 흡입
IPA: /ǀ, ǃ, ǁ, ǂ, ʘ/
한글: 조합 표기 (확장)
예시: 코사어, 줄루어
```

## 4. 코드 형식

### 표준 형식
```
[PL##]-[MN##]-[AR##]

예시:

/tʃ/ = PL05-MN08-AR01
/ŋ/ = PL08-MN02-AR01
```
