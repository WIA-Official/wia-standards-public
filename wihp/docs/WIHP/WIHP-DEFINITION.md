# WIHP 정의서
## WIA International Hangul Phonology (윕)

**버전**: 1.0
**작성일**: 2025-12-21
**상태**: 초안

---

## 1. 정의

**WIHP (WIA International Hangul Phonology)**는 전 세계 모든 언어의 음소를 한글로 표기하는 국제 표준 체계입니다.

```
WIHP = WIA + International + Hangul + Phonology
     = 세계 + 국제 + 한글 + 음운론
     = 국제 한글 음운론

한글 발음: 윕 (WIHP)
```

---

## 2. 탄생 배경

### 2.1 WIA 생태계

| 연도 | 체계 | 대상 | 원리 |
|------|------|------|------|
| 1888 | IPA | 학자 | 라틴 문자 기반 음성 표기 |
| 2025 | WIA Braille | 시각장애인 | IPA → 점자 |
| 2025 | ISP | 청각장애인 | 수어 → 5대 컴포넌트 |
| 2025 | **WIHP** | **전 인류** | **IPA → 한글** |

### 2.2 해결하는 문제

```
문제 1: IPA는 학자용 (일반인 접근 어려움)
문제 2: 7,000+ 언어, 각각 다른 문자 체계
문제 3: 언어 소멸 위기 (100년 후 50% 소멸 예측)
```

### 2.3 해결책

```
WIHP: 모든 언어 → IPA → 한글

결과:
- 한글 하나 배우면 모든 언어 읽음
- 한글로 모든 언어 발음 보존
- 100년 후 언어 복원 가능
```

## 3. 핵심 원리

### 3.1 세종대왕의 유산

```
1443년 훈민정음:
"나랏말싸미 듕귁에 달아 문자와로 서로 사맛디 아니할쎄"
(우리나라 말이 중국과 달라 문자가 서로 통하지 않으니)

2025년 WIHP:
"온 세상 말이 다 달라 서로 읽지 못하니,
한글로 모든 소리를 적어 누구나 읽게 하노라"
```

### 3.2 한글의 과학적 설계

```
자질 문자 (Featural Alphabet):
├── 자음: 발음 기관 모양을 본뜸
│   ├── ㄱ = 혀뿌리가 목구멍 막는 모양
│   ├── ㄴ = 혀가 윗잇몸에 닿는 모양
│   └── ㅁ = 입술 모양
├── 모음: 천지인 철학
│   ├── ㆍ = 하늘 (천)
│   ├── ㅡ = 땅 (지)
│   └── ㅣ = 사람 (인)
└── 획 추가 = 소리 변화
    ├── ㄱ → ㅋ (기식음)
    └── ㄱ → ㄲ (경음)
```

### 3.3 WIHP 확장 원리

```
한글의 자질 문자 원리를 확장:

기본 자모 (24자) + 옛한글 (5자) + 신규 확장

= 모든 IPA 음소 (135+) 표현 가능
```

## 4. 3대 자질 체계

모든 음소는 3가지 자질로 분해됩니다:

```
음소 = 조음 위치 (PL) + 조음 방법 (MN) + 기류 특성 (AR)
       Place         Manner          Airstream
```

### 4.1 조음 위치 (Place) - 11개

```
PL01 양순음    PL02 순치음    PL03 치음
PL04 치조음    PL05 후치조음  PL06 권설음
PL07 경구개음  PL08 연구개음  PL09 구개수음
PL10 인두음    PL11 성문음
```

### 4.2 조음 방법 (Manner) - 8개

```
MN01 파열음    MN02 비음      MN03 전동음
MN04 탄음      MN05 마찰음    MN06 접근음
MN07 측면음    MN08 파찰음
```

### 4.3 기류 특성 (Airstream) - 7개

```
AR01 평음      AR02 기식음    AR03 경음
AR04 유성음    AR05 방출음    AR06 내파음
AR07 클릭음
```

## 5. 한글 체계

### 5.1 현대 한글 (24자)

```
자음 (14): ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅅ ㅇ ㅈ ㅊ ㅋ ㅌ ㅍ ㅎ
모음 (10): ㅏ ㅓ ㅗ ㅜ ㅡ ㅣ ㅐ ㅔ ㅚ ㅟ
```

### 5.2 옛한글 복원 (5자)

```
ㆍ (아래아)  - 후설 원순 모음 /ɒ/
ㆁ (옛이응)  - 어두 비음 /ŋ/
ㅿ (반시옷)  - 유성 치찰음 /z/
ㆆ (여린히읗) - 성문 파열음 /ʔ/
ㅸ (순경음)  - 순치 마찰음 /β, v/
```

### 5.3 신규 확장 (필요시)

```
권설음: ㄷ + 점 = ㄷ̤
클릭음: 조합 표기
성조:  ˊ(상) ˋ(하) ˉ(중)
```

## 6. 비교

### 6.1 IPA vs WIHP

| 항목 | IPA | WIHP |
|------|-----|------|
| 문자 체계 | 라틴 + 특수기호 | 한글 |
| 학습 난이도 | 높음 | 낮음 |
| 대상 | 언어학자 | 누구나 |
| 예시 | /həˈloʊ/ | 헐로우 |

### 6.2 한국어 외래어 표기법 vs WIHP

| 항목 | 외래어 표기법 | WIHP |
|------|-------------|------|
| 목적 | 한국어 어휘화 | 원음 보존 |
| 범위 | 주요 언어 | 7,000+ 언어 |
| 체계 | 언어별 규칙 | 통일된 IPA 기반 |

## 7. 효과

### 7.1 단기 (1-5년)

```
- 외국어 학습 도구로 활용
- 한글 국제화 촉진
- 언어 교육 혁신
```

### 7.2 중기 (5-20년)

```
- 한글 = 세계 문자 인식
- 제2외국어로 한글 선택 증가
- 언어 다양성 보존 기여
```

### 7.3 장기 (20-100년)

```
- 소멸 언어 발음 보존
- 언어 복원 가능성 확보
- 인류 문화 유산 보존
```
