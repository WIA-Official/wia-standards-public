# Sign Language ISO 639-3 Codes
## Registered Sign Language Codes

### Overview
| Attribute | Detail |
|-----------|--------|
| **Standard** | ISO 639-3 |
| **Authority** | SIL International |
| **Purpose** | Language identification |
| **Sign Languages** | 100+ registered |
| **Updates** | Ongoing additions |

### Introduction
ISO 639-3 is the international standard for language codes, maintained by SIL International. Many sign languages have been assigned three-letter codes, providing official recognition and enabling systematic cataloguing.

### Languages Without Codes
Many sign languages lack ISO codes:
- Village sign languages (most)
- Recently documented languages
- Regional varieties
- Historical extinct languages

### How to Request a Code
| Step | Process |
|------|---------|
| Documentation | Provide linguistic evidence |
| Application | Submit to ISO 639-3/RA |
| Review | SIL evaluates |
| Decision | Accept, modify, or reject |
