# Batch 114: LSV (Lengua de Señas Venezolana / Venezuelan Sign Language)

## 기본 정보
- **언어 코드**: LSV / VSL
- **ISO 639-3**: vsl
- **정식 명칭**: Lengua de Señas Venezolana
- **영어 명칭**: Venezuelan Sign Language
- **사용 국가**: 베네수엘라
- **사용자 수**: 15,000~52,000명 (수어 사용 농인)
- **청각장애인 전체**: 120만 명 (대부분 후천적)
- **선천 농인 추정**: 약 15,000명
- **법적 인정**: 1999년 헌법 인정!
- **언어 상태**: Ethnologue - 안정적(Stable), 제도화(Institutional)

## 역사적 배경

### 기원
- **1930년대**: "베네수엘라 수어" 용어 사용 시작
- **1935**: 최초 농학교 설립 (IVCyS - 맹농연구소)
- **1937**: 학교에서 LSV 사용 시작
- **1939**: IVCyS 공식 설립 (카라카스)

### 초기 발전
- 각 아동이 가정에서 사용하던 "홈사인" 통합
- 소규모 수어 커뮤니티 형성
- 맹인/농인 분리 → 농학교 별도 설립

### José Arquero의 영향
- 농인 협회 시기에 외부 표지 도입
- LSV 추가 변형에 기여
- 많은 베네수엘라 농인이 "LSV 창시자"로 인식

## 헌법적 지위 (1999)

### 제81조
- 농인이 LSV로 소통할 권리 인정

### 제101조
- 농인이 자신의 언어로 정보받을 권리
- 공영/민영 TV에서 보장

### 제9조와의 차이
- 스페인어: 공식 언어 (전국)
- 토착어: 공식 언어 (조상 영토 내)
- **LSV: 공식 언어 지위 없음!**

### 법적 한계
- 헌법에 포함되었으나 "공식 언어"는 아님
- 토착어보다 낮은 지위
- 이중언어-이중문화 교육 정책 부재

## 언어적 특성

### 독자성
- 자체 문법과 어휘
- 다른 국가 수어와 다름
- 스페인 수어 영향 받음

### 표준화
- 국내 이동성 높음 → 상당히 표준화
- 커뮤니티 상호 연결성 강함
- 교실 언어 vs 성인 언어 차이 존재

## 교육 현황

### 국가 이중언어 교육
- LSV + 스페인어 병행
- **문제**: 성인 언어 vs 교실 언어 차이

### 특수교육 틀의 한계
- 이중언어-이중문화 본질 미인식
- 포괄적 교육정책 부재
- 이중언어-이중문화 기본교육과정 없음

### 교육 성과 문제 (CONSORVEN 조사)
| 항목 | 비율 |
|------|------|
| 고교 졸업 | 67% |
| 고급 읽기 수준 | **10%만** |
| 고급 쓰기 수준 | **1%만** |

## 커뮤니티 현황

### 긍정적 요소
- 커뮤니티 성장 중
- 고유 민족언어문화로 인정
- 언어/사회적 권리 투쟁 활발
- 통역사 양성 증가
- 전국/지역 농인 협회 통합

### 과제
- 공식 언어 지위 획득
- 포괄적 이중언어 교육정책
- 읽기/쓰기 교육 개선
- 평등한 고용 기회

## 특이사항

### 헌법 인정 vs 공식 언어
- 1999년 헌법에 포함 (제81조, 제101조)
- 그러나 제9조 "공식 언어" 지위는 없음
- 토착어보다 낮은 법적 지위

### 교실 언어 vs 성인 언어
- 국가 이중언어 교육 프로그램 존재
- 교실에서 배우는 LSV ≠ 성인 커뮤니티 LSV
- 언어 변이 문제

### 정치적 논란 (2018)
- 정치인의 LSV 사용이 진정한 포용인가?
- 농인 커뮤니티의 비판적 시각
