# Batch 104: LSC (Llengua de Signes Catalana / Catalan Sign Language)

## 기본 정보
- **언어 코드**: LSC
- **ISO 639-3**: csc
- **정식 명칭**: Llengua de Signes Catalana
- **영어 명칭**: Catalan Sign Language
- **사용 지역**: 카탈루냐, 아라곤(라 프란하), 메노르카
- **사용자 수**: 약 25,000-28,000명 (농인 6,000명 + 청인 19,000명)
- **법적 인정**: 1994년 처음, 2006년 헌법적 인정, 2010년 LSC 법률
- **언어 계통**: 프랑스 수어(LSF) 가족 (추정)

## 역사적 배경

### 주요 연혁
| 연도 | 사건 |
|------|------|
| ~200년 전 | LSC 고유 언어로 형성 시작 |
| 1900년대 초 | Casal de Sords de Barcelona 설립 |
| 1979 | FESOCA (카탈루냐 농인 연합) 설립 |
| **1994** | 카탈루냐 정부 첫 비구속 결의안 (스페인 최초!) |
| 2004 | IEC (카탈루냐 연구소)가 LSC 지원 시작 |
| **2006** | 카탈루냐 자치법 개정 - 헌법적 인정 (50.6조) |
| **2007** | 스페인 법률 27/2007 - LSE와 LSC 동시 인정 |
| **2010** | 카탈루냐 의회 LSC 법률 만장일치 통과 |
| 2015 | LSC상(賞) 제정 |

### 스페인 최초의 수어 법률!
카탈루냐는 **스페인 자치주 최초**로 수어 법률을 제정한 지역!

## 언어적 특성

### LSE와의 관계
| 항목 | 내용 |
|------|------|
| 상호이해도 | 약 50-80% |
| 관계 | 별개 언어 (언어학적 입증) |
| 구분 기준 | 상호이해도, 어휘, 사회적 태도 |

### 언어 계통
- **프랑스 수어(LSF) 가족** 추정 (Wittmann 1991)
- 초기 전파, 증명 어려움
- 약 200년간 독립적 발전

### 문법적 특징
- **공간 활용**: 중립 공간(neutral space) 사용
- **평면 사용**: 수평면 + 수직면 주로 사용
- **동사 분류**: 단순, 지시, 공간-위치 동사 (ASL과 유사)
- **숫자**: 스페인어/카탈루냐어보다 "투명" (십진수 명시)
- **비인칭문**: 대명사, 역할 전환, 공간 위치로 구분

### 은유 표현
- 이중 매핑(double-mapping) 연구
- 도상성(iconicity)이 인지 이론에 핵심 역할

## 교육 현황

### 이중언어 교육
- 농인 학생: 구어(카탈루냐어/스페인어) 읽기/쓰기 학습
- 이중언어 센터: 자격 있는 농인 교사 배치
- LSC를 L1, L2로 교육

### 자원
- AGILS Accesibilidad: 접근성 서비스
- SILS: LSC/LSE 통역, 강좌
- Casal de Sords de Barcelona: 바르셀로나 농인 센터

### UPF 프로그램
- LSC 통역/번역 학부 과정
- 전문 강좌
- 온라인 종합 과정
- 스페인+카탈루냐 최초 수어 대학 과정!

## 법적 체계

### 카탈루냐 자치법 50.6조 (2006)
> "공공기관은 카탈루냐 수어 사용과 이를 선택한 농인의 평등 조건을 보장한다. 이 언어는 교육, 보호, 존중의 대상이 된다."

### 법률 17/2010 (LSC 법)
- **만장일치** 카탈루냐 의회 통과
- IEC를 표준화 기관으로 지정
- 언어정책국(DGPL) 정책 담당

## 특이사항

### 구어 카탈루냐어와 지역 불일치
- LSC 사용 지역 ≠ 구어 카탈루냐어 사용 지역
- 카탈루냐, 아라곤(라 프란하), 메노르카에서만 사용
- 발렌시아는 LSE 사용

### 이주자 적응
- 카탈루냐로 이주한 농인 → LSC 채택
- "수어는 농인 커뮤니티의 소통 필요에서 발생"

### LSC상 (2015)
- 문화부 제정
- LSC 홍보/확산 공헌자 시상
