# Batch 116: KSL (Kenyan Sign Language / 케냐 수어)

## 기본 정보
- **언어 코드**: KSL
- **스와힐리어**: LAK (Lugha ya Alama ya Kenya)
- **ISO 639-3**: xki
- **정식 명칭**: Kenyan Sign Language
- **사용 국가**: 케냐, 소말리아
- **사용자 수**: 340,000~600,000명+ (100만 명 주장도)
- **법적 인정**: 2010년 헌법 - **제3 공식 언어!**
- **언어 상태**: Ethnologue - 발전 중 (그러나 위협받음)

## 역사적 배경

### 기원
- 1950년대부터 케냐에서 독립적 발전
- 프랑스/미국 수어에서 유래
- 교회가 농학교 설립 주도

## 헌법적 지위

### 2010년 헌법 제7조
- KSL과 점자 언급
- 케냐 토착어와 함께 보호
- **영어, 스와힐리어에 이어 제3 공식 언어!**

### 의회 공식 언어
- KSL = 케냐 의회 공식/업무 언어
- 의회 방송 시 KSL 통역 제공

### KSL 법안 (심의 중)
- 헌법 인정의 실질적 이행 목적
- 교육, 고용, 의료, 사법 접근권 보장
- 통역사/교육자/콘텐츠 제작자 양성
- AI 기반 수어 도구 발전 지원

### 부족한 점
- 농인 전용 프로그램 없음
- 자막 서비스 없음
- 의료 정보 제한적 (폴리오만)
- 에볼라, 결핵, HIV 포스터에 KSL 일러스트

## 특이사항

### 헌법 인정 vs 완전한 이행
- 2010년 헌법에 명시
- 그러나 "공식 인정"은 아직 완료 안 됨
- 2025년 KSL 법안 심의 중

### 수어 이름 변화
- 전통적 KSL 수어 이름 → 초성화된 이름
- 젊은 세대/청인 사용자에서 변화 진행
- 언어 정체성 위협

### 소말리아 수어 기원
- 소말리아 최초 농학교 = 케냐에서 온 교사 3명
- Annalena Tonelli의 와지르 농학교에서 출발
- KSL이 소말리아 수어의 기반

### 종교 기관의 역할
- 여호와의증인: KSL 포함 106개 수어로 자료 제작
- 부분 성경 번역 (총 97개 수어 중 하나)
