## Linguistic Features

### Prosodic Structure
| Level | Description |
|-------|-------------|
| Package | Syllable-like units |
| Sign | Word-like individual signs |
| Phrase | Sentence-like units |
| Utterance | Paragraph-like units |

### Juncture Markers (Unique to PISL)
1. **Paragraph-Final**: Hands crossed over stomach (standing) or lap (sitting)
2. **Sentence-Final**: Hands move toward final position but recoil
3. **Emphatic**: Audible clap or slap

### Writing System
- Petroglyphs
- Pictographs
- Hieroglyphs
- One of few sign languages with historical written form

### Dialects
1. **North Central Plains** (Plains Standard)
2. **Far Northern Plains** (Story-Telling Dialect)

## Functions

### Historical Uses
| Function | Context |
|----------|---------|

| **Trade** | Commercial exchanges |
| **Diplomacy** | Treaties and councils |
| **Ceremony** | Religious and cultural events |
| **Storytelling** | Narrative tradition |
| **Hunting** | Silent communication |

### Current Uses
- Storytelling and oratory
- Ceremonies and rituals
- Daily communication (Deaf members)
- Cultural preservation

## Deaf Community Role

### Historical Evidence
- White Swan (Crow): Famous Deaf PISL master
- Deaf Piikáni (Peigan) signers documented 1897
- Primary language for Deaf Plains people
- Full integration in tribal life

### Current Status
- Deaf elders still use PISL
- Some Deaf children learn PISL before ASL
- ASL increasingly replacing PISL for Deaf youth

## Decline Factors

### Colonial Impact
1. **Boarding Schools**: Sign language forbidden
2. **Residential Schools (Canada)**: Cultural suppression
3. **English Dominance**: Replacement of indigenous languages
4. **ASL Introduction**: Deaf schools use ASL

### Current Threats
1. Elder fluent speakers dying
2. Limited child acquisition
3. ASL dominance in Deaf education
4. No formal PISL instruction
5. Geographic dispersal of signers

## Cultural Significance

### "Hand Talk" Philosophy
- Preferred Indigenous term
- Direct translation from Native languages
- Each nation has own name in their language

### Writing System
- Ancient petroglyphs contain hand signs
- Living tradition of visual expression
- Pre-dates European contact documentation

## WIA Talk Approach

### Priority Actions
1. Partner with tribal nations and researchers
2. Support community-led preservation
3. Document remaining fluent signers
4. Create digital learning resources
5. Connect Deaf Indigenous communities

### Ethical Considerations
- Tribal sovereignty over language
- Community permission required
- Avoid academic extraction
- Support Indigenous-led projects

## Comparison: PISL vs ASL

| Feature | PISL | ASL |
|---------|------|-----|
| Origin | Indigenous North America | French Sign Language |
| Age | Pre-contact (500+ years) | ~200 years |
| Primary Users | Originally hearing + deaf | Primarily deaf |
| Function | Lingua franca + primary | Primary language |
| Prosody | Marked junctures | Flowing signs |
| Status | Endangered | Thriving |
