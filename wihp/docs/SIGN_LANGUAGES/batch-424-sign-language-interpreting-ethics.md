### Introduction
Sign language interpreting ethics guide professional conduct and protect all parties in interpreted interactions. The field has evolved from a helper model to a professional model with established ethical standards and accountability mechanisms.

### Historical Evolution
| Era | Model |
|-----|-------|
| Pre-1960s | Helper/volunteer |
| 1960s-70s | Emerging profession |
| 1980s | Conduit model |
| 1990s | Ally model discussions |
| 2000s+ | Bilingual-bicultural professional |

### Core Ethical Principles

#### Confidentiality
| Aspect | Requirement |
|--------|-------------|
| Information | Keep private |
| Exceptions | Safety, legal requirements |
| After assignment | Still applies |
| Social situations | Don't discuss |

#### Accuracy
| Aspect | Requirement |
|--------|-------------|
| Meaning | Convey equivalent meaning |
| Completeness | Don't omit |
| Additions | Don't add |
| Clarification | Request when needed |

#### Impartiality
| Aspect | Requirement |
|--------|-------------|
| No favoritism | Either party |
| No advice | Unless asked role |
| No opinion | On content |
| Boundaries | Professional distance |

#### Professional Conduct
| Aspect | Requirement |
|--------|-------------|
| Appearance | Appropriate dress |
| Preparation | Research assignment |
| Punctuality | Arrive ready |
| Demeanor | Professional manner |
