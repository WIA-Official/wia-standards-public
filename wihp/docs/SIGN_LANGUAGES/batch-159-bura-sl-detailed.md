# Batch 159: Bura Sign Language - 나이지리아 마을 수어

## 기본 정보
- **언어명**: Bura Sign Language
- **국가**: 나이지리아
- **지역**: Kukurpu 마을, Biu 남동쪽 40km
- **민족**: Bura people
- **발견**: Robert Blench 2003년
- **상태**: 미문서화

## 역사적 배경

### 발견
- 2003년 Robert Blench 짧은 방문
- 비디오테이프 촬영
- 학술적 분석 제한적

### 지역 배경
- 나이지리아 북동부
- 높은 선천성 청각장애 비율
- 외딴 지역

## 언어적 특징

### 독자성
- Nigerian Sign Language와 무관
- 화자들 학교 교육 없음
- 외부 접촉 없이 발생

### 특징
- 이완된 손 모양
- 넓은 수화 공간
- 서아프리카 토착 수어 특성

### 서아프리카 수어와의 유사성
- Adamorobe SL과 유사 어휘
- Nanabin SL과 유사점
- 서아프리카 제스처 문화 공유

### 공통 은유
- 땀 = 일하다
- 잠 = 다음 날

## Mofu-Gudur 제스처와의 관계

### 배경
- Mofu-Gudur: 카메룬 청인 제스처 체계
- Bura SL 인근 지역
- 유사성 연구 필요

## 연구 필요성

### 긴급 과제
1. 현장 재방문
2. 체계적 어휘 수집
3. 문법 분석
4. 화자 수 파악

## 서아프리카 마을 수어 비교

|------|------|--------|------|
| Adamorobe SL | 가나 | 완료 | 위험 |
| Nanabin SL | 가나 | 부분 | 미확인 |
| Bura SL | 나이지리아 | 최소 | 미확인 |
