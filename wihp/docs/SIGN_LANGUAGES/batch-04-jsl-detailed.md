# WIA Talk Sign Language Mapping - JSL (Japanese Sign Language / 日本手話)
## Batch 04: 20 Signs with Full WIA Talk Component Analysis

**Language:** JSL - Japanese Sign Language (日本手話, Nihon Shuwa)
**Region:** Japan
**Estimated Users:** ~60,000 native JSL users (304,000 Deaf/HoH total)
**Language Family:** Japanese Sign Language family (related to KSL, TSL)
**Recognition:** 2011 Basic Act for Persons with Disabilities recognized sign language

---

## JSL Characteristics

### Key Features
- **Mouthing (口話 kōwa)**: Critical for distinguishing signs - more than ASL
- **Non-manual markers (NM表現)**: Face, shoulders, eyebrows essential for grammar
- **Fingerspelling (指文字 yubimoji)**: Based on ASL but for 46 kana characters
- **Air writing (空書 kūsho)**: Tracing kanji in the air for names/places
- **Topic-comment structure**: Similar to spoken Japanese and ASL

### Three Types in Japan
1. **日本手話 (Nihon Shuwa)** - True JSL, natural language
2. **対応手話 (Taiou Shuwa)** - Signed Japanese (follows Japanese grammar)
3. **中間手話 (Chuukan Shuwa)** - Pidgin Signed Japanese

### Historical Notes
- First deaf school: Kyoto, 1878 (founded by Tashiro Furukawa)
- Compulsory education for deaf: 1948
- Imperial advocate: Princess Kiko (trained interpreter)
- ~60% lexical similarity with Taiwanese Sign Language (TSL)
- Influence on Korean Sign Language (KSL) from colonial period

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION A: Basic Greetings (5 signs)

### 1. こんにちは (KONNICHIWA - Hello)

**JSL Concept:** Two people bowing to each other

**Detailed Motion:**
- Both index fingers held upright
- Positioned a few inches apart at chest level
- Bend both fingers toward each other simultaneously
- Represents two people bowing in greeting

**WIA Talk Code:** `HS07-LC09-MV16-OR05-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 07 | Index finger extended (両手) |
| LC | 09 | Neutral space (chest level) |
| MV | 16 | Fingers bend toward each other |
| OR | 05 | Fingers facing each other |
| FE | 02 | Pleasant/neutral smile |

**Cultural Note:** Unlike ASL's salute-like hello, JSL reflects the bowing culture

**Source:** tofugu.com, sltranslator.com, soranews24.com

---

### 2. ありがとう (ARIGATOU - Thank You)

**JSL Concept:** Derived from sumo wrestler's winning gesture

**Detailed Motion:**
- Flat hand touches chin area
- Hand moves forward and slightly down
- Palm facing down
- Similar to a graceful bow gesture

**WIA Talk Code:** `HS02-LC05-MV03-OR04-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat/open hand |
| LC | 05 | Chin area |
| MV | 03 | Forward movement |
| OR | 04 | Palm facing down |
| FE | 02 | Grateful smile |

**Cultural Note:** The motion originates from how sumo wrestlers accept prize money

**Source:** soranews24.com, dinolingo.com, suretalk.mb.softbank.jp

---

### 3. ごめんなさい / すみません (GOMEN NASAI - Sorry)

**JSL Concept:** Expressing contrition with a bow

**Detailed Motion:**
- Thumb and index finger pinch eyebrow area (眉間)
- Hand then stands vertically
- Move hand slightly forward
- Head bows slightly simultaneously

**WIA Talk Code:** `HS05-LC02-MV03-OR05-FE04`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 05 | Pinching gesture (thumb + index) |
| LC | 02 | Eyebrow/forehead area (眉間) |
| MV | 03 | Forward movement |
| OR | 05 | Hand vertical, palm inward |
| FE | 04 | Apologetic expression, head bow |

**Alternative Expression:**
- Fist rubs chest in circular motion (similar to ASL SORRY)
- WIA Code: `HS01-LC07-MV12-OR05-FE04`

**Source:** suretalk.mb.softbank.jp, tendershuwa.com

---

### 4. はい (HAI - Yes)

**JSL Concept:** Nodding fist mimics head nod

**Detailed Motion:**
- Make a fist with dominant hand
- Nod the fist up and down
- Like a head nodding in agreement

**WIA Talk Code:** `HS01-LC09-MV07-OR05-FE03`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 01 | Closed fist |
| LC | 09 | Neutral space |
| MV | 07 | Up and down nodding motion |
| OR | 05 | Fist orientation neutral |
| FE | 03 | Affirmative expression, nod |

**Source:** dinolingo.com, sltranslator.com, tendershuwa.com

---

### 5. いいえ (IIE - No)

**JSL Concept:** Beak-like tapping gesture

**Detailed Motion:**
- Index and middle fingers extended
- Tap these fingers against thumb repeatedly
- Like a bird's beak opening and closing
- Head may shake slightly

**WIA Talk Code:** `HS04-LC09-MV13-OR05-FE03`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 04 | V-shape (index + middle finger) |
| LC | 09 | Neutral space |
| MV | 13 | Repeated tapping against thumb |
| OR | 05 | Palm facing inward |
| FE | 03 | Negative expression, head shake |

**Alternative:** Wave hand side to side (いやいや)
- WIA Code: `HS02-LC09-MV06-OR01-FE03`

**Source:** dinolingo.com, at-living.press

## SECTION C: Basic Phrases (5 signs)

### 16. 愛してる (AISHITERU - I Love You)

**JSL Concept:** Stroking/cherishing gesture (same as "precious")

**Detailed Motion:**
- One hand palm down at chest level
- Other hand strokes over it gently
- Like patting something precious
- Also means: かわいい (cute), 大事 (important), 大切 (precious)

**WIA Talk Code:** `HS02-LC07-MV09-OR04-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands (両手) |
| LC | 07 | Chest area |
| MV | 09 | Stroking/patting motion |
| OR | 04 | Palms down |
| FE | 01 | Loving, tender expression |

**Full Sentence "I love you":**
私 + あなた + 愛 + する
- 私 (I): Point to self
- あなた (You): Palm up, extend toward person
- 愛: Stroking gesture
- する: Part of compound

**Alternative:** ILY handshape (ASL influence) - internationally recognized
- WIA Code: `HS15-LC09-MV00-OR01-FE01`

**Source:** syuwafriends.com, kihonshuwa.com, foriio.com

---

### 17. 名前は？ (NAMAE WA? - What is your name?)

**JSL Concept:** Question marked by non-manual features

**Detailed Motion:**
- "Name" sign: Stamp-like motion at chest
  - Two fingers tap chest area
  - Like a name badge/stamp
- Question: Add NM markers
  - Slight head tilt/shake
  - Eyebrow movement

**WIA Talk Code:** `HS04-LC07-MV15-OR05-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 04 | Two fingers (V or N shape) |
| LC | 07 | Chest area (where name badge goes) |
| MV | 15 | Tapping motion |
| OR | 05 | Fingers point inward |
| FE | 06 | WH-question face (首ふり) |

**Grammar Note:** JSL uses 首ふり (slight head shake) for WH-questions

**Source:** ja.wikipedia.org/日本手話, 1101.com

### 19. 元気ですか？ (GENKI DESU KA? - How are you?)

**JSL Concept:** Showing energy/vitality

**Detailed Motion:**
- Both fists at chest
- Pull outward with energy
- Add question markers (eyebrow raise, head tilt)

**WIA Talk Code:** `HS01-LC07-MV19-OR05-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 01 | Fists (両手) |
| LC | 07 | Chest area |
| MV | 19 | Pull apart with energy |
| OR | 05 | Palms facing body |
| FE | 06 | Question expression |

**YES/NO Question Markers:**
- 顎引き (chin pull back)
- Point to person at end
- Head tilt

**Source:** at-living.press, note.com/ap_yoshihara

### 20. トイレはどこですか？ (TOIRE WA DOKO? - Where is the toilet?)

**JSL Concept:** WC (Water Closet) combined with location question

**Detailed Motion - TOILET:**
- Middle, ring, pinky extended (3 fingers = W)
- Thumb and index form circle (C)
- Combines to represent "WC"
- OR: Hand washing gesture

**WIA Talk Code (WC version):** `HS31-LC09-MV00-OR01-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 31 | WC handshape (W+C combined) |
| LC | 09 | Neutral space |
| MV | 00 | Static (show sign) |
| OR | 01 | Palm forward |
| FE | 06 | Question expression (どこ) |

**WIA Talk Code (Hand washing version):** `HS02-LC09-MV20-OR05-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Open hands |
| LC | 09 | Neutral space (in front of body) |
| MV | 20 | Washing motion |
| OR | 05 | Palms facing each other |
| FE | 06 | Question expression |

**Cultural Note:** The "hand washing" version considered more elegant for women

**Source:** hs84.blog.jp, tendershuwa.com

## JSL Linguistic Features Summary

### Non-Manual Markers (NM表現) - Critical in JSL

| Type | Description | Usage |
|------|-------------|-------|
| 首ふり | Slight head shake | WH-questions (何、誰、どこ) |
| 顎引き | Chin pull back | YES/NO questions |
| 眉上げ | Eyebrow raise | Questions |
| 眉寄せ | Eyebrow furrow | Certain questions |
| うなずき | Nodding | Statement boundaries, affirmation |

### JSL vs KSL Connection

Due to historical Japanese occupation period:
- KSL (Korean) shares signs with JSL
- TSL (Taiwanese) ~60% lexical similarity
- Called "Japanese Sign Language family"
