# Batch 132: Mardin Sign Language (마르딘 수어)

## 기본 정보
- **언어명**: Mardin Sign Language (마르딘 수어)
- **터키어명**: Mardin Isaret Dili
- **현지명**: Dilsizce ("농인 언어"), Eski Isaretler ("옛 수화")
- **약칭**: MarSL
- **ISO 639-3**: dsz
- **국가**: 터키
- **지역**: 원래 Mardin (시리아 국경), 현재 Istanbul/Izmir
- **사용자 수**: ~40명 (청각장애+청인, 확대 가족)
- **언어 유형**: 가족 수어 (Family Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 최소 5세대 전 (100년+ 추정)
- **원인**: Dilsiz 가족 내 유전성 청각장애
- **특징**: 단일 확대 가족 내에서 발생한 최초 문서화 사례
- **가족 이름**: "Dilsiz" = 터키어로 "농인" (문자 그대로 "혀 없는")

### Dilsiz 가족
- 5세대에 걸친 청각장애
- 청각장애인과 청인 모두 수어 사용
- 가족 성이 곧 언어 이름의 유래

### 이주 역사
- **원래 거주지**: Mardin (남동부 터키, 시리아 국경)
- **이주**: 모든 화자가 Istanbul과 Izmir로 이주
- **현재**: Mardin에 화자 없음

### 마을 배경
- **위치**: 터키 남동부
- **문화**: 쿠르드/아랍/터키 혼합 지역
- **종교**: 이슬람

## 언어적 특징

### Turkish Sign Language (TID)와의 관계
- **완전히 다른 언어**: 독자 발생
- **현재 상황**: 대부분 화자 TID 이중언어
- **젊은 세대**: TID만 사용, MarSL 수동적 이해만

### 음운론
- 독자적 수형 체계
- TID와 구별되는 패턴
- 연구 진행 중

### 특수 문법 특징
- **수 체계**: 특이한 수 표현 방식
- **비수지 신호**: 풍부하게 사용
- 상세 분석 진행 중

## 사회언어학적 특징

### 사용 현황
- **주 사용 환경**: 가정, 가족 모임 (결혼식 등)
- **빈도**: 점점 감소
- **아동 습득**: 더 이상 없음

### 화자 분포
| 유형 | 인원 | 비고 |
|------|------|------|
| 유창한 화자 | ~15-20명 | 50-60대 |
| 부분적 화자 | ~20명 | 젊은 세대 |
| 총 화자 | ~40명 | 청각장애+청인 |

### 세대 간 차이
- **노년층**: MarSL 유창, TID도 사용
- **중년층**: MarSL + TID 이중언어
- **청년층**: TID 위주, MarSL 수동적 이해
- **아동**: MarSL 습득 안 함

### 특수한 상황
- 청각장애인과 청인 모두 수어 사용
- 청인 이중언어자가 공동체에서 중요한 역할
- 도시 농인 공동체와의 접촉 경험

## 언어 위기 상태

### 멸종위기 등급
- **UNESCO**: 거의 멸종 (Nearly Extinct)
- **Ethnologue**: 8b (거의 멸종)
- **EGIDS**: 8b (조부모 세대만 사용)

### 위협 요인

#### 1. 세대 간 전달 중단
- 아이들이 MarSL 습득하지 않음
- TID가 젊은 세대의 주 언어
- 유창한 화자 50-60대에 집중

#### 2. 지리적 분산
- 원래 마을에서 모두 이주
- Istanbul과 Izmir로 분산
- 가족 접촉 빈도 감소

#### 3. TID의 우세
- TID의 사회적 위신
- 교육, 미디어에서 TID 사용
- 도시 농인 공동체 통합

### 예상 시나리오
- 현재 유창한 화자 사망 시 언어 소멸
- 2-3세대 내 완전 멸종 예상
- 긴급한 문서화 필요
