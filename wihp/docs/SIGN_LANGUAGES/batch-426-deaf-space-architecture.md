# DeafSpace Architecture
## Designing for Visual Communication

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Architecture for deaf people |

| **Developer** | Hansel Bauman, Gallaudet |
| **Principle** | Visual-centered design |
| **Application** | Buildings, spaces |

### Introduction
DeafSpace is an architectural design approach that considers the unique sensory and visual needs of deaf people. Developed at Gallaudet University, it creates environments optimized for visual communication, spatial awareness, and deaf community life.

### Five Principles of DeafSpace

#### 1. Space and Proximity
| Feature | Purpose |
|---------|---------|
| Wider hallways | Signing while walking |
| Conversation circles | Group visual access |
| Optimal distance | Sign language range |
| Gathering spaces | Visual communication |

#### 2. Sensory Reach
| Feature | Purpose |
|---------|---------|
| Open sightlines | See across spaces |
| Visual connections | Between areas |
| Reduced visual clutter | Clear communication |
| Wide angles | Peripheral awareness |

#### 3. Mobility and Proximity
| Feature | Purpose |
|---------|---------|
| Soft edges | Rounded corners |
| Open paths | Walk and sign |
| Ramp preference | See while moving |
| Space for groups | Walk together signing |

#### 4. Light and Color
| Feature | Purpose |
|---------|---------|
| Soft diffused light | Reduce eye strain |
| Glare reduction | Visual comfort |
| Contrasting colors | Highlight hands |
| Blue/green tones | Optimal for skin contrast |

#### 5. Acoustics
| Feature | Purpose |
|---------|---------|
| Sound dampening | Reduce distraction |
| Vibration | Alert systems |
| Noise reduction | For hard of hearing |
| Quiet HVAC | Less auditory distraction |

### Specific Design Elements

#### Lighting
| Element | Application |
|---------|-------------|
| Natural light | Soft, diffused |
| No backlighting | Faces visible |
| Adjustable | User control |
| Even distribution | No harsh shadows |

#### Color
| Application | Recommendation |
|-------------|----------------|

#### Layout

|---------|--------|
