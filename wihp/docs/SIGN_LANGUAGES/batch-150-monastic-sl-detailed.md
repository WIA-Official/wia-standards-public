# Batch 150: Monastic Sign Languages (수도원 수어)

## 개요

수도원 수어는 침묵 서약을 지키는 수도원에서 발달한 수어입니다.
청각장애와 무관하게 발생한 특수한 형태의 수어입니다.

## 1. Cistercian Sign Language (시토회 수어)

### 기본 정보
- **사용 집단**: 시토회 (트라피스트) 수도사
- **기원**: 12세기 프랑스
- **현재**: 전 세계 시토회 수도원
- **특징**: 침묵 서약 하에 의사소통

### 역사
- **1098년**: 시토회 창설
- **침묵 규칙**: 엄격한 침묵 생활
- **수어 발달**: 필수적 의사소통 수단

### 언어적 특징
- 제한된 어휘 (약 400-500개)
- 수도원 생활 관련 용어 중심
- 종교 용어 풍부
- 완전한 언어보다는 보조 의사소통 체계

### 어휘 범주
- 종교 의식
- 음식, 식사
- 작업 지시
- 시간, 일과
- 인사, 허락 요청

## 2. Benedictine Sign Language (베네딕토회 수어)

### 기본 정보
- **사용 집단**: 베네딕토회 수도사
- **기원**: 6세기 이탈리아
- **현재**: 일부 전통 수도원

### 역사
- **529년**: 성 베네딕토 규칙서
- 침묵의 가치 강조
- 수어 보조적 사용

### 시토회 수어와의 관계
- 시토회는 베네딕토회에서 파생
- 유사한 수어 체계
- 지역별 변형 존재

---

## 3. Trappist Sign Language (트라피스트 수어)

### 기본 정보
- 시토회의 엄격한 분파
- 더 철저한 침묵 서약
- 수어 더 발달

### 특징
- 시토회 수어와 유사
- 일부 수도원별 변형

---

## 수도원 수어의 특성

### 청각장애 수어와의 차이

| 특징 | 수도원 수어 | 청각장애 수어 |
|------|-------------|---------------|
| 사용 목적 | 침묵 서약 | 의사소통 필요 |
| 화자 | 청인 | 청각장애인 + 청인 |
| 어휘 범위 | 제한적 | 완전한 언어 |
| 문법 복잡성 | 단순 | 복잡 |
| 사용 환경 | 수도원 | 일상 전체 |

### 언어학적 지위
- "언어"보다 "의사소통 체계"
- 완전한 문법 체계 부재
- 보조적 수단

### 역사적 가치
- 수어 역사의 일부
- 중세 유럽 문화 반영
- 침묵의 의사소통 연구

## 현대적 상황
