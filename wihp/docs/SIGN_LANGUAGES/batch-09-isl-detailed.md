# WIA Talk Sign Language Mapping - ISL (Indian Sign Language / भारतीय सांकेतिक भाषा)
## Batch 09: 20 Signs with Full WIA Talk Component Analysis

**Language:** ISL - Indian Sign Language (भारतीय सांकेतिक भाषा)
**Region:** India, parts of South Asia
**Estimated Users:** ~5 million deaf persons (50 lakh per 2011 Census); 15+ million signers (Indo-Pakistani SL family)
**Language Family:** Indo-Pakistani Sign Language family
**Recognition:** ISLRTC established 2015; ISL Dictionary 2018; not yet scheduled language

---

## ISL Characteristics

### Historical Background
- 12th century: Early reference in Islamic legal text Hidayah (deaf legal standing)
- Early 20th century: Village sign languages in Naga hills
- 1978: Dr. Madan Vashishta pioneered ISL research
- 1981: First ISL Dictionary published
- 2001: First formal ISL course at AYJNISHD, Mumbai
- 2015: ISLRTC (Indian Sign Language Research and Training Centre) established
- 2018: First official ISL Dictionary launched
- 2020: New Education Policy mentions ISL standardization

### Key Challenges
- **NO legal recognition** as scheduled language (unlike ASL in US)
- **Only ~300-325 certified interpreters** for 50+ lakh deaf population
- **Oralist education dominant** - ISL not used in most deaf schools
- **Regional variations** across India's diverse communities
- **Lack of teaching materials** incorporating sign language

### Indo-Pakistani Sign Language Family
ISL is part of broader IPSL family including:
- Indian Sign Language (ISL)
- Pakistani Sign Language (PSL)
- Nepalese Sign Language (NSL)
- Possibly Bangladeshi varieties

**Note:** Some scholars consider these as varieties of one language (IPSL), others as separate languages.

### Key Features
- **Two-handed signs** common (unlike ASL's one-handed preference)
- **Both hands used for gestures** - more complex than ASL
- **Facial expressions** - critical for grammar and emotion
- **Cultural influence** - reflects India's diversity
- **Namaste gesture** incorporated into formal greetings

### Alphabet
- 26 letters representing English alphabet
- Some signs use both hands
- Numbers 1-9 have specific hand gestures

### Organizations
- **ISLRTC** - Indian Sign Language Research and Training Centre (Government)
- **AIFD** - All India Federation of Deaf
- **NAD** - National Association of the Deaf
- **AYJNIHH** - Ali Yavar Jung National Institute
- **DEF** - Deaf Enabled Foundation

### Notable Figures
- **Dr. Madan Vashishta** - Pioneer of ISL research (1978)
- Author of "Deaf in Delhi" and "Deaf in DC"

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION A: Basic Greetings (5 signs)

### 1. NAMASTE / HELLO (नमस्ते)

**ISL Concept:** Traditional Indian greeting gesture

**Detailed Motion:**
- Both palms pressed together
- Fingers pointing upward
- Hands at chest level
- Slight bow of head (respectful)
- Warm/welcoming facial expression

**WIA Talk Code:** `HS02-LC07-MV00-OR05-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands pressed together (both) |
| LC | 07 | Chest level |
| MV | 00 | Static (or slight bow) |
| OR | 05 | Palms facing each other |
| FE | 02 | Warm, respectful expression |

**Cultural Note:** Namaste is deeply embedded in Indian culture - "The divine in me bows to the divine in you"

**Source:** slideshare.net, howtosayguide.com

---

### 2. DHANYAVAAD / THANK YOU (धन्यवाद)

**ISL Concept:** Gratitude gesture from chin

**Detailed Motion:**
- Method 1 (Formal): Open right hand flat against chin, move forward
- Method 2: Interlocked fingers, palms up, move away from chest
- Method 3 (Informal): Like blowing a kiss from chin outward
- Grateful expression

**WIA Talk Code (Formal):** `HS02-LC05-MV03-OR04-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat open hand |
| LC | 05 | Chin area |
| MV | 03 | Forward movement |
| OR | 04 | Palm down/forward |
| FE | 02 | Grateful expression |

**Note:** Tapping motion (2x) is common variation

**Source:** howtosayguide.com, indiansignlanguage.org

## SECTION B: Numbers 1-10 (संख्या)

### ISL Number System Overview

ISL uses hand gestures for digits 1-9. Numbers can be:
- One-handed or two-handed
- Facial expressions used for larger numbers (10, 100, 1000)
- Both hands commonly used for complex numbers

**Key Difference from ASL:** ISL often uses both hands for gestures, making it more complex.

**Source:** researchgate.net, resna.org

## SECTION C: Basic Phrases (5 signs)

### 16. MAIN TUMSE PYAAR KARTA/KARTI HOON (I Love You - मैं तुमसे प्यार करता/करती हूँ)

**ISL Concept:** Three-part or compound phrase

**Detailed Motion:**
- MAIN (I): Point to self
- PYAAR (Love): Cross arms over heart, hug self
- TUM (You): Point to person
- Loving expression essential

**WIA Talk Code (PYAAR/Love):** `HS02-LC07-MV26-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands crossed over chest |
| LC | 07 | Chest/heart area |
| MV | 26 | Cross arms, hug motion |
| OR | 05 | Arms across body |
| FE | 01 | Loving expression |

**Note:** ILY (🤟) is internationally recognized but may not be traditional ISL

## ISL Linguistic Features Summary

### Grammar
- Visual-manual language with own grammar
- Different from spoken Hindi, English
- Facial expressions critical for grammar
- Uses space and directional signs

### Cultural Integration
- **Namaste** - Traditional greeting incorporated
- Reflects India's diverse regional cultures
- Multiple regional variations exist

### Current Challenges
1. **No scheduled language status** - Unlike ASL in US
2. **Interpreter shortage** - Only ~325 for millions of deaf
3. **Oralist education** - ISL not used in most schools
4. **Standardization ongoing** - ISLRTC working on this
5. **Teaching materials lacking**

### Related Sign Languages
- **Pakistani Sign Language** - Very similar
- **Nepalese Sign Language** - Related
- **Bangladeshi varieties** - Possibly related
- Together form Indo-Pakistani Sign Language (IPSL) family

### Film Reference
- **Black (2005)** - Bollywood film about blind-deaf girl (Helen Keller inspired)
