# Sign Language Idioms
## Idiomatic Expressions in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Idioms in sign languages |

### Introduction
Sign languages have rich idiomatic expressions that cannot be understood from the literal meaning of their component signs. These idioms often reflect deaf culture, visual experience, and the unique properties of sign language itself.

### What Are Sign Language Idioms?
| Feature | Description |
|---------|-------------|
| Definition | Fixed expressions |
| Meaning | Non-compositional |
| Cultural | Deaf community usage |
| Learning | Must memorize |

### Deaf Cultural Idioms

#### PAH (success)
| Expression | Meaning |
|------------|---------|
| Sign | Puffing motion |
| Idiomatic | "Finally! At last!" |
| Usage | Relief, success |

#### MIND-FREEZE
| Expression | Meaning |
|------------|---------|
| Literal | "Mind frozen" |
| Idiomatic | "Brain freeze, blank" |
| Usage | Can't remember |

#### HARD-OF-HEARING (index finger)
| Expression | Meaning |
|------------|---------|
| Gesture | Wiggling between ear/mouth |
| Idiomatic | "Unsure, uncertain" |
