# Batch 119: SASL (South African Sign Language / 남아프리카공화국 수어)

## 기본 정보
- **언어 코드**: SASL
- **아프리칸스어**: Suid-Afrikaanse Gebaretaal
- **ISO 639-3**: sfs
- **정식 명칭**: South African Sign Language
- **사용 국가**: 남아프리카공화국
- **사용자 수**: 600,000~2,000,000명
- **법적 인정**: **2023년 7월 19일 - 제12 공식 언어!**
- **농인 인구**: 약 600,000명 (청력 손실: 140만 명)

## 역사적 배경

### 기원
- **1863년**: 아일랜드 수녀들이 농인 훈련 프로그램 시작
- 다양한 토착 수어 시스템에서 발전
- BSL 영향 (역사적 연결)
- ASL 영향 (일부 농인 사용)

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1863 | 아일랜드 수녀 농인 훈련 시작 |
| 1984 | 교육 매체: 토착어 → 영어 변경 |
| 1995 | SANCD → DeafSA 전환 (정책 대전환) |
| 1996 | 헌법에 "수어" 언급 (보호 대상) |
| 2001 | SASL 국가언어부 설치 |
| 2008 | SASL 정책 이행 회의 |
| 2011 | 84명 통역사 등록 (DeafSA) |
| **2023.01.12** | 법안 국회 제출 |
| **2023.05.02** | 국회 헌법 개정 승인 |
| **2023.07.19** | **대통령 서명 - 제12 공식 언어!** |
| 2024.08 | 최초 청각장애 미스 사우스아프리카 당선 |

## 헌법적 지위

### 2023년 법안 목적
1. SASL 문화적 수용 증진
2. 농인/난청인 권리 실현 (법 앞 평등, 인간 존엄)
3. 포용적/실질적 평등 촉진
4. 장애 기반 불공정 차별 방지/제거 (헌법 제9조)

### 역사적 의의
- 아프리카에서 **4번째** 수어 공식 언어 국가
- (케냐, 짐바브웨, 우간다 다음)
- 25년간의 캠페인 결실!
- 람아포사 대통령: "역사적 순간, 민주주의 역사"

### 대통령 발언
> "이것은 시작일 뿐. 더 많은 작업 필요."
> "통역만이 아니라 농인 삶의 모든 측면에 적용되어야."

## 언어적 특성

### 토착 언어
- 남아프리카 고유 언어
- 독자적 문법 구조와 어휘
- 다른 언어와 독립적

### 변이
- 지역별 변이 존재
- 역사적 교육 정책으로 인한 차이
- 2개 주요 교육기관 → 2개 변종
- 전국 농인포럼에서 상호 이해 가능

### 영향
| 수어 | 영향 |
|------|------|
| BSL | 역사적 연결 |
| ASL | 일부 농인 사용, 대부분 지역 수어에 영향 |

### "다른 수어" 오해
- 남아공 농인들: "각 언어 공동체마다 다른 수어"라고 믿음
- **실제 원인**: "통역사"가 수어 대신 Signed English/Signed Xhosa 사용
- 진짜 수어는 전국에서 상호 이해 가능

### 교육 문제
- 농인 교사 대부분 청인
- SASL 훈련/자격 요구 없음!
- 수어 능력 평가 없이 임용
- 교사가 학생에게서 SASL 배움
- 품질 보증 메커니즘 부재

- 농학교 졸업자 평균: 청인 8세 수준

## 특이사항

### 25년간의 캠페인
- 1996년 헌법: "수어" 언급 (보호 대상)
- 그러나 공식 언어는 아니었음
- 2023년까지 25년간 캠페인

### 넬슨 만델라 비전
> "모든 사람이 조화롭게 평등한 기회를 가지고 사는 민주적이고 자유로운 사회"

### 우분투(Ubuntu) 정신
- "인간성을 다른 사람에게 보여주는 것"
- SASL 공식화 = 우분투 실천

### 여전한 과제
- 정부 기관 통역사 부재
- 병원, 경찰서, 법원 접근성 부족
- 교육법 개정 캠페인 진행 중
- 농인 교사 양성 필요
