# Batch 112: LSPY (Lengua de Señas Paraguaya / Paraguayan Sign Language)

## 기본 정보
- **언어 코드**: LSPY
- **ISO 639-3**: pys
- **정식 명칭**: Lengua de Señas Paraguaya / Lengua de Señas del Paraguay
- **영어 명칭**: Paraguayan Sign Language
- **사용 국가**: 파라과이
- **사용자 수**: 1만~10만 명 추정 (Ethnologue 범위)
- **법적 인정**: 2020년 법률 No. 6530 공식 인정!
- **언어 상태**: Ethnologue 6a 등급 (활발 - Vigorous)
- **언어 계통**: 파라과이-우루과이 수어 어족

## 역사적 배경

### 기원
- 학교 밖에서 자연 발생/발전
- 우루과이 수어와 역사적 연결
- ASL, 인접국 수어와 상호 이해 불가

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1990 | "Habla Conmigo" 교재 출판 (Lichtenberger) |
| 2008 | SIL 연구 - LSPY 변이 조사 |
| 2008 | Letra Paraguay - 성경 번역팀 구성 |
| **2009** | 교육에 LSPY 도입 시작! |
| 2017 | SIM 선교사 파라과이 장기 사역 시작 |
| **2020** | **법률 No. 6530 - LSPY 공식 인정!** |
| 2020 | 지역 학교 교사 통역사 의무화 법률 |
| ~2024 | 신약성경 LSPY 비디오 번역 완료! |

## 언어적 특성

### 언어 계통
- **파라과이-우루과이 수어 어족**
- ASL과 상호 이해 불가
- 인접국 수어와 상호 이해 불가
- 우루과이 수어와 역사적 연결 보고됨

### 이중언어 국가 반영
- 파라과이: 스페인어 + 과라니어 공용
- **언어법 No. 4251**: 양 언어에 적합한 지문자 규정
- 독특한 이중 지문자 체계

### 변이
- 수도 아순시온 중심
- 시우다드 델 에스테 (제2도시)
- 차코 지역 (북부): 농인 거의 없음
- 지역별 변이 존재
