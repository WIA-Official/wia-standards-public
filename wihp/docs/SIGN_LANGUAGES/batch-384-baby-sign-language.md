# Baby Sign Language
## Pre-verbal Communication with Hearing Infants

### Overview
| Attribute | Detail |
|-----------|--------|
| **Type** | Adapted signs for infants |
| **Users** | Hearing parents + hearing babies |
| **Purpose** | Pre-verbal communication |
| **Status** | Popular in many countries |
| **Relation to Deaf SL** | Borrows some signs |

### Introduction
Baby sign language refers to the practice of teaching simplified signs to hearing infants before they can speak. Based on the observation that babies can control their hands before developing speech, this approach allows pre-verbal communication between parents and infants.

This is distinct from sign languages used by the deaf community, though it often borrows signs from established sign languages like ASL.

|-------|---------|

| Step | Practice |
|------|----------|
| Start | 6-8 months old |

### Best Practices
| Practice | Rationale |
|----------|-----------|

| Be patient | Babies respond at own pace |

| Consistent | Use same sign each time |
