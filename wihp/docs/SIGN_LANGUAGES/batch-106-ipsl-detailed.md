# Batch 106: IPSL (Indo-Pakistani Sign Language)

## 기본 정보
- **언어 코드**: IPSL
- **ISO 639-3**: ins (인도), pks (파키스탄), nsp (네팔), 별도 코드 존재
- **정식 명칭**: Indo-Pakistani Sign Language
- **영어 명칭**: Indo-Pakistani Sign Language
- **사용 지역**: 인도, 파키스탄, 네팔, 방글라데시
- **사용자 수**: **최소 1,500만 명** (세계 최대!)
- **Ethnologue 순위**: 세계 149위 언어 (2024)
- **법적 인정**: 제한적 (인도 2016년 법률, 파키스탄 미인정)
- **언어 계통**: 독립적 (아시아/유럽 수어와 무관)

## 언어적 특성

### 언어 계통
- **독립적 언어**: 유럽/아시아 다른 수어와 무관
- **언어학적 통일성**: 문법은 동일, 어휘만 지역 차이
- **방언**: 카라치/라호르(파키스탄), 델리/콜카타/벵갈루루(인도)

### 문법적 특징 (서양 수어와 다름!)
| 특징 | IPSL |
|------|------|
| **어순** | 항상 **술어 말미** (Predicate Final) |
| **품사** | 명확한 품사 구분 없음 (ASL과 다름) |
| **시제** | 시제 굴절 없음! BEFORE/THEN/AFTER로 표현 |
| **의문사 위치** | 문장 **끝**에 위치 |
| **생략** | 광범위, 한 단어 문장 흔함 |
| **문법 관계** | 공간 메커니즘으로만 표시 |

### 공간 사용
- IPSL의 핵심 특징
- 방향 표지(directional signs)로 문법 관계 표시
- 문맥에서 추론

### 숫자 체계
| 숫자 | 표현 |
|------|------|
| 0-5 | 해당 수 손가락 펴기 |
| 6-9 | 아라비아 숫자 모양 특수 손모양 |
| 10 | 두 5-손 또는 1+0 |

### 가족 관계 표현
- **남성 가족**: 남성/남자 + 관계어
- **여성 가족**: 여성/여자 + 관계어

### 지문자 체계
| 유형 | 사용 |
|------|------|
| **BSL 기반** | 대다수 사용 |
| 데바나가리 기반 | 바도다라/구자라트 일부 |
| ASL 영향 | 벵갈루루 일부 (문법은 IPSL) |

### 비수지 요소
- 최소쌍 구분: 뚱뚱한/강한, 추운/어려운 (표정만 다름!)
- 부정: 비수지 요소로 표시

## 특이사항

### 서양 수어와의 차이점
1. 품사 구분 없음 (ASL은 있음)
2. 시제 굴절 없음
3. 의문사 문장 끝 위치
4. 공간 메커니즘이 문법 관계의 유일한 표지

### 정치적 분열
- 언어학적으로는 단일 언어
- 인도-파키스탄 갈등으로 별도 취급
- 연구/교육 분절화 문제

### 벵갈루루 ASL 학교
- ASL 어휘 사용하지만 **문법은 IPSL!**
- 혼합 형태
