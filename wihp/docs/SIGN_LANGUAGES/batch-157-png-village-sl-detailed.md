# Batch 157: Papua New Guinea Village Sign Languages

## 개요
파푸아뉴기니는 800개 이상 구어가 있는 언어 다양성 핫스팟
마을 수어도 다수 존재하나 대부분 미문서화

## 1. Enga Sign Language

### 기본 정보
- **위치**: Enga Province, Upper Lagaip Valley
- **연구**: Adam Kendon 1980년
- **상태**: 최초로 연구된 PNG 마을 수어

### 특징
- 청각장애 가족 중심
- 청인도 사용
- 세계 최초 마을 수어 상세 연구 중 하나

---

## 2. Kailge Sign Language

### 기본 정보
- **위치**: Western Highlands Province
- **화자**: Ku Waru 언어권
- **특징**: 넓은 지역 네트워크

### 언어적 특징
- 공간 사용이 농인 공동체 수어와 유사
- Sinasina SL과 어휘 유사성

---

## 3. Sinasina Sign Language

### 기본 정보
- **위치**: Chimbu Province, Sinasina valley
- **화자**: 약 3 deaf + 50 hearing
- **발견**: Samantha Rarrick 2016년

### 특징
- Kere 공동체 사용
- Kailge SL과 관계 연구 필요

---

## 4. Mount Avejaha Sign Language

### 기본 정보
- **위치**: Oro Province
- **특징**: 많은 청각장애 아동
- **상태**: 다른 PNG 수어와 구별

### 사회적 통합
- 청각장애인 완전 통합
- SIL 연구팀 보고

---

## 5. Nebilyer-Kaugel Sign Network

### 기본 정보
- **위치**: Western Highlands
- **연구**: ANU 박사논문 2020년대

### 특징
- 12명 deaf의 언어 연구
- 마을 수어와 홈사인 사이
- 수어 네트워크 개념 제안

---

## PNG 수어 상황

### 국가 수어
- PNGSL: 2015년 제4 공용어 지정
- 약 30,000명 사용
- 교육/도시 중심

### 마을 수어
- 다수 존재 추정
- 대부분 미문서화
- 긴급 조사 필요
