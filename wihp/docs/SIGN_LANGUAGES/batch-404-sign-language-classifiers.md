### Introduction
Classifiers (also called depicting signs or polycomponential signs) are a distinctive feature of sign languages worldwide. They are handshapes that represent categories of objects and combine with movement and location to create meaning. Classifiers allow signers to show the movement, location, and manipulation of objects in space.

### What Are Classifiers?
| Feature | Description |
|---------|-------------|
| Handshape | Represents category of object |
| Movement | Shows action or path |
| Location | Shows position in space |
