# Sign Language Acquisition
## How Children Learn to Sign

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | First language acquisition in sign |
| **Finding** | Parallel to spoken language |
| **Stages** | Same milestones |
| **Critical period** | Early exposure essential |
| **Populations** | Deaf children, CODAs |

### Introduction
Research on sign language acquisition shows that deaf children learning sign language as a first language go through the same developmental stages as hearing children learning spoken language. This parallel development provides powerful evidence that sign languages are natural human languages.

### Acquisition Stages
| Stage | Age | Description |
|-------|-----|-------------|
| Babbling | 7-10 months | Manual babbling |
| First signs | 8-14 months | First recognizable signs |
| Two-sign | 18-22 months | Combining signs |
| Grammar | 2-3 years | Grammatical development |
| Complex | 3-5 years | Adult-like signing |
