# Batch 136: Algerian Jewish Sign Language (알제리 유대 수어)

## 기본 정보
- **언어명**: Algerian Jewish Sign Language
- **약칭**: AJSL
- **ISO 639-3**: 없음
- **국가**: 이스라엘/프랑스 (원래 알제리)
- **원래 지역**: Ghardaia, 알제리
- **사용자 수**: 소수 (고령층만, 추정 10명 미만)
- **언어 유형**: 가족/공동체 수어
- **상태**: 거의 멸종

## 역사적 배경

### 기원
- **발생 시기**: 5세대 이상, 100년+ 추정
- **장소**: 알제리 Ghardaia - 사하라 북부 오아시스 도시
- **공동체**: Ghardaia 유대인 공동체 내 청각장애 가족

### Ghardaia 배경
- **위치**: 알제리 사하라 사막 북부, Mzab 계곡
- **역사**: 11세기부터 유대인 공동체 존재
- **특징**: 베르베르 무슬림과 유대인 공존
- **고립**: 지리적, 문화적 고립으로 독자적 수어 발달

### 이주 역사
- **1962년**: 알제리 독립 후 유대인 대탈출
- **목적지**: 이스라엘, 프랑스
- **현재**: 화자들 이스라엘과 프랑스에 분산

## 언어적 특징

### Israeli Sign Language와의 관계
- **완전히 다른 언어**: ISL과 어휘, 문법 다름
- **현재 상황**: ISL과 AJSL 이중언어 사용
- **젊은 세대**: ISL만 사용

### 어휘 특징
- Ghardaia 지역 생활 관련
- 유대 전통 관련 용어
- 사하라 오아시스 문화 반영
