# Batch 153: Kajana Sign Language (카자나 수어)

## 기본 정보
- **언어명**: Kajana Sign Language
- **약칭**: KajSL
- **ISO 639-3**: 없음
- **국가**: 수리남
- **지역**: Kajana 마을, Saramaccan 지역
- **사용자 수**: 미확인 (소규모)
- **언어 유형**: 마을 수어
- **계통**: 독립 언어
- **상태**: 미확인 (문서화 필요)

## 역사적 배경

### 기원
- **발생 시기**: 미확인
- **장소**: Kajana, 수리남 내륙 열대우림
- **공동체**: Saramaccan Maroons (마룬)

### Maroon 배경
- **Maroons**: 탈출 노예 후손
- **역사**: 17-18세기 네덜란드 식민지 탈출
- **문화**: 서아프리카 전통 + 현지 발전

### Saramaccan 언어
- Saramaccan Creole 사용 공동체
- 영어/포르투갈어 기반 크리올
- 성조 언어

## 지리적 맥락

### 위치
- 수리남 내륙
- 아마존 열대우림
- 강을 따라 마을 분포

### 접근성
- 도로 없음, 보트로만 접근
- 외부와 제한된 접촉
- 전통 생활 방식 유지

## 언어적 특징

### 독자성 추정
- 외부 수어 영향 없이 발생
- Maroon 공동체 내 발달
- 연구 거의 없음

### 연구 필요
- 체계적 분석 없음
- 기초 정보만 존재
- 긴급 문서화 필요

## 사회언어학적 특징

### 화자 현황
- 정확한 숫자 미확인
- 마을 내 청각장애 가족
- 청인 수화자 존재 추정

### 사용 환경
- 가정 내 의사소통
- 마을 활동
- 농업, 어업 작업

## 연구 및 문서화

### 현황
- 거의 연구되지 않음
- 일부 보고만 존재
- 체계적 조사 없음

### 필요 조치
1. 현장 조사 (보트로 접근)
2. 화자 파악
3. 어휘, 문법 기록
4. Saramaccan 문화 맥락 이해

### 어려움
- 접근성 극히 어려움
- 자원 부족
- 현지 연구자 부족

## 남아메리카 마을 수어 맥락

### 유사 사례
- Providence Island SL (콜롬비아)
