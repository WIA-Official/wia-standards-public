# Batch 127: Ban Khor Sign Language (반 코르 수어)

## 기본 정보
- **언어명**: Ban Khor Sign Language (반 코르 수어)
- **현지명**: ภาษามือบ้านค้อ (파사 무 반 코르)
- **별칭**: Pasa Kidd ("벙어리 언어")
- **ISO 639-3**: bfk
- **국가**: 태국 (이산 지방)
- **지역**: Ban Khor 마을, Nong Khai 주
- **사용자 수**: ~16명 deaf + ~400명 hearing (2009년 기준)
- **언어 유형**: 마을 수어 (Village Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 1930년대
- **원인**: 마을 내 유전성 청각장애 급증
- **발견**: 1997년 James Woodward에 의해 학계에 알려짐
- **연구**: 2000년 Angela M. Nonaka의 본격적 연구 시작

### 마을 배경
- **위치**: 태국 동북부 이산 지역
- **인구**: 약 2,741명 (2009년)
- **주요 산업**: 벼농사
- **특징**: 지리적으로 고립된 시골 마을

### 발전 단계
1. **1930년대 이전**: 청각장애인 없음, 수어 없음
2. **1930년대**: 유전성 청각장애 출현
3. **1940-1960년대**: 가정 수화에서 마을 수어로 발전
4. **1970-1990년대**: 완전한 언어 체계 형성
5. **2000년대 이후**: Thai Sign Language 영향으로 쇠퇴

## 언어적 특징

### 음운론 (Phonology)
- **수형**: 독자적 체계
- **위치**: 얼굴을 가리는 수화 존재 (희귀 특징)
- **움직임**: 특유의 패턴
- **비수지 신호**: 풍부하게 사용

### 특이 수화 예시
| 의미 | 특징 |
|------|------|
| 저기 (THERE) | 팔이 얼굴을 가림 |
| 외국인 (FOREIGN) | 팔이 얼굴을 가림 |

### 문법 구조
- **어순**: 유연함
- **형태론**: 독자 발전
- **공간 문법**: Thai SL과 다름

## 사회언어학적 특징

### 사용 현황 (2009년)
| 유형 | 인원 | 비율 |
|------|------|------|
| 청각장애 화자 | 16명 | 0.6% |
| 청인 유창 화자 | 400명+ | 15% |
| 마을 전체 인구 | 2,741명 | 100% |

### 과거 상황
- 거의 모든 마을 주민이 BKSL 사용
- 청각장애인 완전 사회 통합
- 청인-청각장애인 간 자연스러운 소통

### 현재 상황
- 젊은 청각장애인: 기숙 청각장애 학교 진학
- 학교에서 Thai Sign Language (ThSL) 습득
- ThSL이 1차 언어로 전환

## 언어 위기 상태

### 멸종위기 등급
- **UNESCO**: 심각한 위험 (Severely Endangered)
- **Ethnologue**: 이동 중 (Shifting)
- **EGIDS**: 7 (Shifting)

### 위협 요인

#### 1. 교육 시스템
- 청각장애 학교에서 ThSL 사용
- BKSL 사용 금지/억제
- 교육받은 세대의 언어 전환

#### 2. 사회 변화
- 젊은 세대의 도시 이주
- 외부와의 접촉 증가
- ThSL의 사회적 위신

#### 3. 태도 변화
- 중년 청인도 ThSL 어휘 혼용
- "ThSL이 더 좋다"는 인식 확산
- BKSL에 대한 낮은 평가

### 언어 전이 패턴
- 1세대: BKSL 단일어
- 2세대: BKSL + ThSL 이중어
- 3세대: ThSL 단일어 (BKSL 수동적 이해만)
