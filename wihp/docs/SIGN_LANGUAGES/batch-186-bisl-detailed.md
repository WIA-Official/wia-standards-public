# Batch 186: Bay Islands Sign Language (BISL) - 온두라스 시각-촉각 수어

## 개요
| 항목 | 내용 |
|------|------|
| **언어 코드** | fren1280 (Glottolog) |
| **다른 이름** | French Harbour Sign Language |
| **지역** | 온두라스, Bay Islands (Roatan, Guanaja) |
| **화자 수** | 약 11명 (농맹 4명 + 청시/농시/청맹) |
| **특수성** | 시각-촉각 양식 (Visual and Tactile) |
| **상태** | 심각한 위기 |

## 역사적 배경

### 기원
- 1895년: 알려진 가장 오래된 농맹 화자 출생
- Roatan섬 French Harbour 마을에서 시작
- 인근 Jonesville로 확산
- 이후 Guanaja섬으로 전파

### Usher Syndrome
- 유전적 질환으로 인한 농과 후천적 실명
- French Harbour에서 높은 발생률
- 먼저 농으로 태어나고, 나중에 실명
- 100년 이상 이 지역에 존재

### 마을 정보
- French Harbour: Roatan 남해안 어촌
- Jonesville: 전통적 정착지, 보트로만 접근
- White Islander 인구에 주로 분포

## 사용자 커뮤니티

### 화자 구성
| 유형 | 수 | 특징 |
|------|-----|------|
| 농맹 | 4명 (생존) | 51-73세, 핵심 화자 |
| 농시 | 1명+ | Usher Syndrome 없는 농인 |
| 청시 | 다수 | 청인 가족/이웃 |
| 청맹 | 드묾 | 후천적 실명 청인 |

### 세대 분포
- 최소 3세대에 걸쳐 11명의 Usher Syndrome 화자
- 가장 젊은 농맹: 1965년경 출생
- 새로운 농맹 출생 없음
- 청시 젊은 화자는 존재

### 지리적 분포
- Roatan: 3명 농맹 화자
- Guanaja: 1명 농맹 화자
- White Islander 공동체에 제한

## 언어학적 특성

### 양식 (Modality)
가장 특별한 특성: 시각과 촉각 두 양식 모두 사용

**시각-제스처 양식 (Visual-Gestural)**
- 청시-농시, 청시-청시 대화
- 일반적인 수어와 유사

**촉각-제스처 양식 (Tactile-Gestural)**
- 농맹 화자 참여 대화
- 손을 잡고 수어
- 촉각으로 수어 인식
- 고유한 적응 형태

### 어휘 특성
- 500개 어휘 데이터베이스 (ELDP 프로젝트)
- ID-gloss 체계 개발
- 개인/마을별 변이 기록
- 해양/어업 관련 어휘 풍부

### 언어 접촉
- LESHO (온두라스 국가 수어) 접촉
- ASL (선교사 통해) 영향
- 그러나 독자적 체계 유지

## 위기 요인

### 인구학적
- 가장 젊은 농맹 1965년생
- 새로운 농맹 출생 없음
- 4명 농맹 모두 51-73세
- 청시 화자만으로 언어 유지 어려움

### 사회적 변화
- 전통 어촌 공동체 변화
- 관광업 발달로 인구 유입
- White Islander 공동체 희석
- 젊은 세대 외부 이주

### 언어적
- 농맹 화자 사망 시 촉각 양식 소멸 위험
- LESHO/ASL 접촉 증가
- 청시 화자들의 제한적 사용
