# WIA Talk Sign Language Mapping - HKSL (香港手語 / Hong Kong Sign Language)
## Batch 17: 20 Signs with Full WIA Talk Component Analysis

**Language:** HKSL - 香港手語 (Hoeng1 gong2 sau2 jyu5 / Hong Kong Sign Language)
**Region:** Hong Kong and Macau
**Estimated Users:** ~6,000 primary users (2021 Census); 300 under age 15
**Language Family:** Derived from Southern Chinese Sign Language, now independent
**Recognition:** Not officially recognized; lacks institutional support

---

## HKSL Characteristics

### Historical Background
- **~1949:** Origin traced - ~20 deaf people moved from Shanghai/Nanjing to Hong Kong
- Began tutoring local deaf community
- CSL was initial medium → adapted to local context
- **1935:** First written records of HKSL
- Language developed independently with little external influence
- Recent decades: More borrowings (American manual alphabet adopted with adaptations)
- **2013:** HKSL Browser launched (first comprehensive online database)
- **2024:** HKSLLEX database with 3,700+ words

### Language Family & Relationships
- **Parent:** Southern dialect of Chinese Sign Language (CSL)
- **Now independent** - mutually unintelligible with CSL
- **American manual alphabet** borrowed and adapted
- Influenced by Hong Kong Cantonese culture

### Key Features - UNIQUE CHARACTERISTICS
- **ENTIRELY AMBIDEXTROUS** - unique among sign languages!
- **40-50 basic handshapes**
- **Five parameters:** Handshape, location, movement, orientation, non-manual features
- **Sign derivation methods:**
  1. **Conceptual representation** (abstract): father, mother
  2. **Visual representation** (direct): to separate, thick-skinned
  3. **Chinese character representation**: to introduce, Chinese language
  4. **English term representation** (rare): toilet/WC
- **Iconicity** highly valued - signs visually resemble meaning

### Grammar Features
- **Non-manual markers** obligatory in content questions
- **Brow raise** = question marker
- **Wh-signs never clause-initial**
- **NO/NOT:** B-handshape with waving motion
- **CAN'T:** I-handshape with twisting motion
- **Negation:** Cantonese mouthing pattern (m4-hai6 = 'no')

### Numeral System
- **Cardinal numerals:** By default NO movement, produced in neutral space
- **Four numeral types** when movement added:
  1. Event numerals (how many times)
  2. Transfer numerals (ownership transfer)
  3. Conjunction numerals (all items included)
  4. List numerals

### Organizations & Resources
- **CSLDS** - Centre for Sign Linguistics and Deaf Studies (CUHK)
- **HKSL Browser** - 9,889 entries, 340,578 users, 2.79M views (2025)
- **HKSLLEX** - 3,700+ words (HKU Linguistics)
- **HKSL Corpus** - video archive with ELAN annotations
- **Hong Kong Sign Language Association** (HKSLA)

### Challenges - ENDANGERED STATUS
- **Only 6,000 primary users** (2021)
- **Only 300 users under 15!** - critical for language transmission
- **Deaf schools reduced from 5 to 1** (serving 60-100 pupils)
- **No official recognition**
- Lacks institutional and educational support
- Language barrier → marginalization

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION B: Numbers 0-10 (數字)

### HKSL Number System Overview

From John Benjamins research:
- **Cardinal numerals:** Default NO movement, neutral space
- Codepoint 65 = "zero" handshape (O-shape → character "A" in font)
- Codepoint 67 = "EIGHT" (single-handed sign)
- Movement changes create different numeral types

**Source:** jbe-platform.com, osf.io HKSLLEX

| HS51 | Cross/十 shape | Number 10 |
| HS52 | Three fingers | Number 3 |
| HS53 | HKSL 6 shape | Number 6 |
| HS55 | Bell-ringing | Admiralty |

### Unique HKSL Features
- **Entirely ambidextrous** - unique among sign languages!
- **Cardinal numerals:** NO movement by default
- **Questions require brow raise** (obligatory)
- **NO/NOT:** B-handshape + wave + Cantonese mouthing
- **Chinese character representation** in many signs
- **Iconicity highly valued** - transparency ratings in HKSLLEX

### Sign Derivation Categories
1. Conceptual (abstract)
2. Visual (direct)
3. Chinese character
4. English term (rare)
