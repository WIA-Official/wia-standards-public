# Georgian Sign Language (GESL)
## ქართული ჟესტური ენა (Kartuli Zhest'uri Ena)

### Overview
| Attribute | Detail |
|-----------|--------|
| **Official Name** | Georgian Sign Language |
| **Native Name** | ქართული ჟესტური ენა |
| **Abbreviation** | GESL |
| **ISO 639-3** | gss |
| **Country** | Georgia |
| **Users** | ~2,500 Deaf and Hard of Hearing |
| **Status** | Not officially recognized as national language |
| **Language Family** | Independent (some RSL influence) |

### Historical Development
- **Soviet Era**: Russian manual alphabet used for fingerspelling
- **2011**: Final workshop in GESL by Ministry of Education
- **2012**: New Mkhedruli-based manual alphabet developed
- **2013**: Georgian Dactyl Alphabet research published
- **2014**: First online dictionary (geodeaf.ge) launched
- **2015-present**: Ongoing linguistic research at universities

### Linguistic Features
- Unique grammar separate from spoken Georgian
- Morpho-lexical system for numerals
- Pentagram counting system (finger-based)
- Ordinal numeral suffix system (derivational)
- Numeral incorporation features
- Temporal categories: body=present, front=future, back=past

### Manual Alphabet (2012 Reform)
- Based on Mkhedruli Georgian script
- Replaced Russian-based fingerspelling system
- Developed with Union of the Deaf of Georgia
- Culturally significant adaptation to native script

### Educational Challenges
- Large gap in deaf education
- Many deaf community members lack Georgian literacy
- Written communication avoided even on social media
- Need for bilingual education (GESL + Georgian)
- Agrammatism elimination efforts ongoing
