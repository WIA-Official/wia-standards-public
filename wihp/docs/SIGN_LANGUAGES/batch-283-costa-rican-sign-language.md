# Costa Rican Sign Language (LESCO)
## Lengua de Senas Costarricense

### Overview
| Attribute | Detail |
|-----------|--------|

| **Abbreviation** | LESCO |
| **ISO 639-3** | csr |

| **Users** | Significant deaf community |
| **Status** | Official language (2012) |
| **Language Family** | Mixed origins (ASL influence ~70%) |

### Legal Recognition (2012)
- Declared official language to protect deaf linguistic rights
- Recognized as unique language with own grammar and structure
- Guarantees accessibility for deaf community

### Historical Development
- **Pre-1990s**: Original LESCO developed independently
- **1990s**: Significant ASL influence introduced
- **~70% ASL vocabulary**: Result of linguistic imperialism
- New LESCO (NLESCO) emerged as distinct variety
- Historical relationship between original LESCO and ASL studied

### Research and Documentation
- **LESCO Corpus**: Created for linguistic description
- **2014**: Basic dictionary (1,100 entries) published online
- **2014**: Basic descriptive grammar available at cenarec-lesco.org
- Second corpus-based signed language description in Spanish Latin America
- University of Costa Rica linguistic research ongoing

### Linguistic Features
- Independent grammatical structure
- Influenced by but distinct from ASL
- Original LESCO elements preserved
- Myths about universality and iconicity debunked
- Natural language with full linguistic complexity
