## 가나의 수어들

### 3가지 수어
| 수어 | 설명 |
|------|------|
| **Ghanaian Sign Language** | 표준 수어 (ASL 기반) |
| **Adamorobe Sign Language** | 마을 수어 (200년+) |
| **Nanabin Sign Language** | 가족 수어 |

### AdaSL (아다모로베 수어)
- **위치**: 아크라 북쪽 30-40km, 아쿠아펨 힐스
- **역사**: 200년+ (세계 최고령 마을 수어 중 하나)
- **사용자**: 농인 약 30명 + 청인 1,370명 (2003)
- **특징**: GSL과 완전 독립적으로 발전
- **위기**: GSL 영향으로 멸종 위기

### AdaSL 특이사항
| 항목 | 내용 |
|------|------|
| 유전적 농 | 마을 내 높은 유전적 농 발생률 |
| 공유 수어 | 청인도 함께 사용 |
| 사회 통합 | 농인이 완전히 통합된 사회 |
| 낙인 없음 | 농에 대한 낙인 없음 |
| 1975년 결혼 금지 | 추장이 농인 간 결혼 금지령 |
| 결과 | 농 출생 감소 + AdaSL 위기 |

### AdaSL 언어학적 특징
- **분류기 부재**: 움직임/위치 분류기 없음 (다른 수어와 다름)
- **연속동사구문**: 아칸어(구어)와 유사한 구조
- **고립어**: 다른 수어와 계통 관계 없음

## 특이사항

### GSL vs AdaSL
- GSL: ASL 기반, 학교 도입, 표준화 진행
- AdaSL: 토착 발생, 200년 역사, 멸종 위기

### 아다모로베 마을
- **공유 수어 커뮤니티**: 청인 대다수가 수어 사용
- **Martha's Vineyard와 유사**: 역사적 공유 수어 커뮤니티
- **세대 간 언어 변화**: 젊은 세대 → GSL 전환

### 인권 옹호
- GNAD: GSL 공식 언어 인정 캠페인
- 정신건강 정책 포용성 주장
- 서아프리카 농인 협회 역량 강화 프로젝트
