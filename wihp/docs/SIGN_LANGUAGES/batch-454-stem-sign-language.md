### Introduction
STEM (Science, Technology, Engineering, Mathematics) fields present unique challenges for sign language, as many technical concepts lack established signs. Various projects worldwide work to develop and standardize STEM vocabulary in sign languages.

### Challenges in STEM Signing
| Challenge | Issue |
|-----------|-------|
| No existing signs | New concepts |
| Inconsistency | Multiple signs for same term |
| Fingerspelling | Slow, imprecise |
| Regional variation | Different signs |
| Rapid change | New terms constantly |
