# Batch 128: Al-Sayyid Bedouin Sign Language (알-사이드 베두인 수어)

## 기본 정보
- **언어명**: Al-Sayyid Bedouin Sign Language (알-사이드 베두인 수어)
- **아랍어명**: لغة الإشارة لعشيرة السيد
- **약칭**: ABSL
- **ISO 639-3**: syy
- **국가**: 이스라엘
- **지역**: 네게브 사막, Al-Sayyid 마을
- **사용자 수**: ~150명 deaf + 수천 명 hearing (2005년 기준)
- **언어 유형**: 마을 수어 (Village Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 약 70-80년 전 (1940년대 추정)
- **원인**: 열성 유전자에 의한 선천성 청각장애
- **마을 설립**: 19세기 중반, 이집트 남성과 현지 여성에 의해
- **청각장애 비율**: 마을 인구의 약 4% (미국 평균의 40배)

### 마을 특성
- **인구**: 약 4,000-4,500명 (2020년 기준)
- **민족**: 베두인 아랍인
- **사회 구조**: 확대 가족 중심, 일부다처제
- **특징**: 사촌 간 결혼 관행으로 유전자 확산

### 발전 과정
1. **1세대**: 첫 청각장애 아동 출현, 가정 수화
2. **2세대**: 형제자매 간 언어 공유, 구조 발전
3. **3세대**: 마을 전체로 확산, 문법 복잡화
4. **현재**: 완전한 언어 체계, 학술 연구 대상

## 언어적 특징

### 어순 (Word Order)
- **기본 어순**: SOV (주어-목적어-동사)
- **예**: "John dog saw" (존이 개를 보았다)
- **특이점**: 아랍어(SVO), 히브리어(SVO), ISL과 완전히 다름

### 수식어 위치
- **명사-형용사 순서**: 형용사가 명사 뒤에
- **예**: "dog big" (큰 개)

### 음운론적 특징
- **음소 체계**: 아직 완전히 발달하지 않음
- **이중 분절성**: 제한적 (signs = holistic)
- **최소 대립쌍**: 거의 없음
- **구순 동작 (mouthing)**: 사용하지 않음

### 형태론
- **복합어**: 발달 중
- **굴절**: 제한적
- **파생**: 단순

### 공간 문법
- **지시 체계**: 실제 위치 기반
- **동사 일치**: 발달 중
- **대명사**: 직접 지시 방식

## 사회언어학적 특징

### 언어 사용 환경
- 가정 내 일상 대화
- 결혼식, 장례식 등 행사
- 일상 업무 및 거래
- 스토리텔링

### 화자 분포
| 세대 | 연령대 | 특징 |
|------|--------|------|
| 1세대 | 60+ | 언어 창시자 |
| 2세대 | 40-60 | 언어 발전 기여 |
| 3세대 | 20-40 | 완전한 문법 습득 |
| 4세대 | 0-20 | ISL 영향 받음 |

### 사회적 통합
- 청각장애인에 대한 낙인 없음
- 청인-청각장애인 결혼 일반적
- 모든 사회 활동에 참여
- 별도의 "농인 문화" 없음

## 언어학적 중요성

### 연구 가치
1. **언어 기원 연구**: 언어가 어떻게 탄생하는지 관찰
2. **문법 발생**: 구조가 세대에 따라 어떻게 발전하는지
3. **음운론 발달**: 음소 체계의 점진적 형성
4. **보편 문법**: 인간 언어의 본질적 특성 탐구

### 주요 발견
- 어순이 주변 언어와 무관하게 자생적으로 결정됨
- 문법 복잡성이 세대를 거치며 증가
- 음운 체계가 가장 늦게 발달

## 위협 요인

### 언어 상태
- **EGIDS**: 6a (활발)
- **위기 수준**: 취약 (Vulnerable)

### 주요 위협
1. **Israeli Sign Language (ISL) 확산**
   - 청각장애 학교에서 ISL 교육
   - TV를 통한 ISL 노출
   - 젊은 세대 ISL 선호

2. **사회 변화**
   - 외부와의 결혼 증가
   - 도시로 이주
   - 현대화

3. **Jordanian Sign Language**
   - 요르단 TV 프로그램 시청

## 문화적 특징

### 청각장애 인식
- "장애"가 아닌 "다름"으로 인식
- 청각장애 정체성 정치 없음
- 마을 공동체의 일부

### 의사소통 패턴
- 청인이 청각장애인에게 복잡한 주제 논의
- 가족 내 완전한 언어 전달
- 아이들이 자연스럽게 ABSL 습득
