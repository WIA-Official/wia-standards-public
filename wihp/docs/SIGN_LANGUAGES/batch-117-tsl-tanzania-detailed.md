# Batch 117: TSL/LAT (Tanzanian Sign Language / 탄자니아 수어)

## 기본 정보
- **언어 코드**: TSL / LAT
- **스와힐리어**: LAT (Lugha ya Alama ya Tanzania)
- **ISO 639-3**: tza
- **정식 명칭**: Tanzanian Sign Language
- **사용 국가**: 탄자니아
- **사용자 수**: 1만~100만 명 추정 (Ethnologue 범위)
- **법적 인정**: 1994년 정부 공식 인정
- **언어 상태**: Ethnologue 5등급 (발전 중)

## 역사적 배경

### 특이한 기원: 7개 학교 = 7개 수어!
- **1963년**: 최초 농학교 설립
- 각 농학교에서 **독립적으로** 수어 발전
- 약 7개 서로 다른 탄자니아 수어 존재
- 일부 학교는 연구/사용 금지!

### 주요 연혁

## 언어적 특성

### 복잡한 언어 상황
- **7개 학교 = 7개 독립 수어**
- 공통/유사 표지로 표준화 시도 (1984)
- 그러나 공식 이행 안 됨
- 학교 간 영향 거의 없음

### Tabora 변종
- 구화주의 농학교에서 발전
- 표준 사전과 상당히 다름
- 현재 조사 중

## 교육 현황

### 학교 현황
| 유형 | 수 |
|------|-----|
| 특수 농학교 | 8개 |
| 통합 초등/중등 (농인 유닛) | 약 28개 |

### 교육 방식
- 구화 + 수어 (Total Communication)
- 수어 습득: 대부분 입학 후
- 중등/고등 교육: TSL로 거의 제공 안 됨
- **농아동 취학률: 1% 미만!**

### 문제점
- 통역사/수어 교사 부족
- 교사가 통역사 겸임 (이중 역할)

## 특이사항

### 7개 학교 = 7개 언어
- 세계적으로 드문 상황
- 각 학교가 독립적으로 수어 발전
- 일부 학교: 자체 수어 연구/사용 금지

### 농인 정체성 논쟁
- (문화적) 농인 정체성 인식 정도 불확실
- 수어 사용 vs 농인 문화 정체성 연결 불분명

### 표준화의 정치학
- 언어 표준화 = 권력의 문제
- 어떤 변종을 표준으로?
- 학교 간 이해관계 충돌

### 지역 기반 프로젝트 (CICED-CHAVITA)
- 3개 지역 집중: 음트와라(남), 탕가(북), 도도마(중부)
- 농인 권리 인식 제고
- 수어 앱 개발 계획
