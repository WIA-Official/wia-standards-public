# Batch 113: LSU (Lengua de Señas Uruguaya / Uruguayan Sign Language)

## 기본 정보
- **언어 코드**: LSU
- **ISO 639-3**: ugy
- **정식 명칭**: Lengua de Señas Uruguaya
- **영어 명칭**: Uruguayan Sign Language
- **사용 국가**: 우루과이
- **사용자 수**: 약 28,600명 (도시 지역, 2006)
- **사용 시작**: 1910년~
- **법적 인정**: 2001년 법률 17.378 - 공식 언어!
- **언어 계통**: 파라과이-우루과이 수어 어족

## 역사적 배경

### 기원과 발전
| 연도 | 사건 |
|------|------|
| 1910년대 | 농학교 환경에서 LSU 안정화 |
| 1920년대 | 비공식 커뮤니티 모임에서 확산 |
| **1928** | **최초 농인 주도 공공 협회 설립!** |
| 1980년대 | 몬테비데오, 살토, 파이산두 커뮤니티 존재 확인 |
| 1983~ | LSU 사회적 가시성 지속 증가 |
| **1987** | **이중언어 교육 프로그램 시작!** |
| **1989** | 법률 16.095 - LSU 연구/교육/보급 지원 |
| **2001** | **법률 17.378 - LSU 공식 인정!** |
| **2008** | **법률 18.437 - LSU를 모국어로 인정!** |
| 2022 | LSU-DS 데이터셋 공개 |
| 2025 | iLSU-T 번역 데이터셋 공개 (185시간+) |

### 농학교와 구화주의
- 국가 지원으로 농학교 설립
- 초기 교육자: 구화주의 방식 훈련
- 학생들: 또래 상호작용에서 제스처 → 완전한 언어로 발전
- 구화주의 강조에도 불구하고 LSU가 주요 소통 수단

## 언어적 특성

### 언어 계통
- **파라과이-우루과이 수어 어족**
- 파라과이 수어와 역사적 연결
- 인접국 수어와 상호 이해 불가
- 아르헨티나, 브라질 수어와 유사성 보고

### 지역 변이
| 변이 | 설명 |
|------|------|
| 몬테비데오 | 가장 큰 커뮤니티, 연령별 어휘 차이 |
| 살토 | 두 번째 큰 커뮤니티, 몬테비데오와 유사 |
| 파이산두 | 서부 우루과이 |
| 리베라 | 브라질 접경, 독특한 변종 |
| 말도나도 | 남부 해안, 독특한 변종 |

### 방언 논의
- **몬테비데오 수어** vs **살토 수어** (2개 방언설)
- 젊은 세대: 더 많은 표준화
- 아르헨티나/브라질 수어에서 어휘 차용

### 수어 습득 시기 (1980년대 연구)
| 시기 | 비율 |
|------|------|
| 3세 전 | 9% |
| 3-10세 | 46% |
| 10세 후 | 28% |
| 학교에서 | ~50% |
| 농인 협회/기타 | ~50% |

## 법적 지위

### 법률 체계
| 연도 | 법률 | 내용 |
|------|------|------|
| 1989 | 법률 16.095 | LSU 연구/교육/보급 지원 |
| **2001** | **법률 17.378** | **LSU = 농인 커뮤니티의 자연 언어** |
| **2008** | **법률 18.437** | **LSU = 우루과이 시민의 모국어!** |

### 2001년 법률 17.378 요구사항
- TV 및 농인 관련 행사에 통역 제공
- 공공 접근 가능 정부 건물: 시각 알림 시스템

### 언어 정책 혁신
- 농인을 **소수 언어 집단**으로 취급
- 포르투갈어/이민자 언어와 동등 지위
- 우루과이: 공식 언어 없음 (스페인어 관습적 사용)

## 교육 현황

### 이중언어 교육 (1987~)
- 초등학교 수준 시행
- LSU + 스페인어 병행
- 의사소통 및 인지 발달 촉진

### 교육 정책 문제
- 의료적 장애 정책과 연결
- 포괄적 국가 언어정책 부재
- 갈등과 모순 존재

### 미래 과제
- 대학 통역사 접근성 증가
- 의료 환경 통역사 확대
- 평등한 취업 기회

## 커뮤니티 발전

### 1928년 이후 지속적 발전
- 최초 농인 주도 협회 설립
- 세대 간 언어 전승 시작
- 학교 외부에서 언어 확산

### 긍정적 방향
- 법적 지원 강화
- 사전 편찬 진행
- 기술 개발 활발
- 연구 지속

## 지문자 체계

### 스페인어 알파벳
- A, B, C, D, E, F, G, H, I, J, K, L, **LL**, M, N, **Ñ**, O, P, Q, R, S, T, U, V, W, X, Y, Z
- 스페인어 특수 문자 포함 (LL, Ñ)

## 특이사항

### 모국어 지위 (2008)
- 우루과이 스페인어
- 우루과이 포르투갈어
- **LSU**
- 세 언어 모두 "우루과이 시민의 모국어"!

### 소수 언어 접근법
- 농인 = 의료적 장애 집단 ❌
- 농인 = 소수 언어 집단 ⭕
- 진보적 언어정책 모델

### 연구 활성화
- 1980년대부터 언어학 연구
- 수십 년간 문서화 진행
- 대학 연구진 참여 활발
