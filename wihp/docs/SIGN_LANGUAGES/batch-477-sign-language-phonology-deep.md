### Introduction
Sign language phonology studies the smallest meaningless units of sign languages and how they combine. Just as spoken languages have phonemes, sign languages have parameters that function as phonological units, subject to systematic constraints and patterns.

### The Five Parameters
| Parameter | Description |
|-----------|-------------|
| Handshape | Configuration of fingers |
| Location | Where sign is made |
| Movement | How hand moves |
| Orientation | Direction palm faces |
| Non-manual | Face and body |

|------|-------------|

#### Deletion

|------|-------------|

#### Metathesis

### Phonotactic Constraints

#### Symmetry Condition
| Rule | Description |
|------|-------------|
| Two-handed signs | Same handshape if both move |
| Or | One hand stationary |
| Violation | Very rare |

#### Dominance Condition
| Rule | Description |
|------|-------------|
| Two-handed | Dominant hand more active |
| Non-dominant | Stationary or same movement |
| Asymmetric | Dominant does more |
