### Introduction
Sign language linguistics is the scientific study of the structure and properties of sign languages. This field established that sign languages are fully developed natural languages with complex grammar, not merely gesture systems or manual versions of spoken languages.

The recognition of sign languages as true languages revolutionized both linguistics and deaf education.

### Key Concepts

#### Minimal Pairs
Signs differing in one parameter:
| Example (ASL) | Difference |
|---------------|------------|
| APPLE / ONION | Location |
| MOTHER / FATHER | Location |
| SIT / CHAIR | Movement |

#### Iconicity
| Type | Description |
|------|-------------|
| Transparent | Meaning obvious |
| Translucent | Meaning with explanation |
| Arbitrary | No visual connection |

#### Classifiers
| Type | Use |
|------|-----|
| Entity | Represent nouns |
| Handle | How objects held |
| SASS | Size and shape |
| Body part | Body as classifier |

| Spoken | Signed |
|--------|--------|
| Sequential | More simultaneous |

| Oral | Manual-facial |
