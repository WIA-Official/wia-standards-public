# Sign Language and Aphasia

### Introduction
Aphasia in sign language users provides crucial evidence that sign languages are processed as true languages in the brain. When deaf signers have strokes or brain injuries affecting language areas, they show aphasia patterns parallel to those in spoken language users, demonstrating that language processing is independent of modality.

|---------|--------------|
