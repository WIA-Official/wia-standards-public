# Batch 110: LSEC (Lengua de Señas Ecuatoriana / Ecuadorian Sign Language)

## 기본 정보
- **언어 코드**: LSEC
- **ISO 639-3**: ecs
- **정식 명칭**: Lengua de Señas Ecuatoriana / Lengua de Señas de Ecuador
- **영어 명칭**: Ecuadorian Sign Language
- **사용 국가**: 에콰도르
- **사용자 수**: 수만 명 (정확한 추정 어려움)
- **법적 인정**: 2008년 헌법 인정!
- **언어 상태**: Ethnologue 5등급 (발전 중)

## 역사적 배경

### 기원과 영향
- **오래된 에콰도르 토착 수어** 존재
- **ASL 영향**: 상당한 어휘적 영향
- **스페인 수어 영향**: 일부 변종 영향
- 세 가지가 혼합되어 현재 LSEC 형성

### 언어 계통
- **ASL 하위 분류** (Wikidata)
- 독자적 발전 진행 중
- 지역/연령별 변이 존재

### 관련 수어
| 수어 | 관계 |
|------|------|
| 페루 수어 | 높은 어휘 유사성 |
| 볼리비아 수어 | 높은 어휘 유사성 |
| 콜롬비아 수어 | 높은 어휘 유사성 |
| 칠레/아르헨티나 수어 | 낮은 유사성 |
| ASL | 기원적 영향 |

### 사회언어학적 변이
- **지역**: 주요 변수
- **연령**: 주요 변수

## 법적 상태

- 이중언어 발전 지향
- 구화주의(oralism) 대신 수어 우선

### 교육

- 이중언어 교육 모델 추구

## 연구 현황

### SIL 사회언어학 조사
- LSEC = 에콰도르 주요 수어
- 영향: 토착 수어 + ASL + 스페인 수어
- 변이 요인: 지역, 연령
- 정부 지원 강함

### 기술 연구
- NFA (비결정적 유한 오토마타) 모델링
- LSTM 딥러닝 실시간 인식
- GIF 기반 시각 출력 시스템

## 특이사항

### 공식 인정 vs 실제 상태
- 2008년 헌법 인정
- 그러나 SIL 보고서: 아직 공식 인정 부족
- 실제 이행과 법적 인정 간 차이 존재

### 남미 수어 네트워크
- 페루, 볼리비아, 콜롬비아, 에콰도르 수어
- 상당한 어휘적 유사성
- 모두 ASL 영향 받음
- 칠레, 아르헨티나는 유사성 낮음
