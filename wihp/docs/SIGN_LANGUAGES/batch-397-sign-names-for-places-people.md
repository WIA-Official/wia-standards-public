# Sign Names and Place Signs
## Named Signs in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Signs for proper nouns |
| **Types** | People, places, brands |
| **Creation** | Community-developed |
| **Variation** | Differs by language, region |
| **Culture** | Important deaf cultural element |

### Introduction
Sign languages have their own signs for proper nouns including people's names, place names, country names, and brand names. These are not fingerspelled but are distinct signs created within the deaf community. Name signs for people are particularly important in deaf culture.

### Personal Name Signs

#### What They Are
| Aspect | Description |
|--------|-------------|
| Definition | Sign for individual's name |
| Alternative to | Fingerspelling |
| Given by | Deaf community member |
| When | Upon joining community |
| Permanence | Usually for life |

#### Types of Name Signs
| Type | Description | Example |
|------|-------------|---------|
| Arbitrary | Based on handshape from initial | "M" handshape at chin |
| Descriptive | Based on characteristic | Sign for "curly" + "hair" |
| Combination | Initial + descriptor | "J" + glasses motion |
| Loan | From another sign | Sign similar to word |

#### Rules and Etiquette
| Rule | Explanation |
|------|-------------|
| Given, not chosen | Must be given by deaf |
| By deaf person | Hearing can't assign |
| Unique in community | Avoid duplicates |
| Respectful | Never offensive |
| Permanent | Rarely changed |

|-----------|------------|
