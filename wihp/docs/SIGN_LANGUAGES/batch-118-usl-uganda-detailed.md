# Batch 118: USL (Ugandan Sign Language / 우간다 수어)

## 기본 정보
- **언어 코드**: USL / UgSL
- **ISO 639-3**: ugn
- **정식 명칭**: Ugandan Sign Language
- **사용 국가**: 우간다
- **사용자 수**: 160,000~1,290,000명
- **법적 인정**: **1995년 헌법 - 세계 2번째!**
- **언어 상태**: Ethnologue 5등급 (발전 중)

## 역사적 배경

### 세계 최초급 헌법 인정!
- **1995년**: 우간다 헌법에 USL 공식 인정
- **세계에서 두 번째**로 수어를 헌법에 명시!
- 동아프리카 지역 선구자

## 언어적 기원

### 크리올 가설
- 1988년 수어 교실 허용 전: 각 학교별 홈사인 사용
- 1988년 이후: 여러 학교 수어가 통합
- **USL = 크리올** (여러 로컬 수어의 혼합)

### 영향을 준 수어
| 수어 | 영향 경로 |
|------|----------|
| BSL | 초기 농학교 교육 언어 |
| ASL | 교육 영향 (제한적 평가) |
| 케냐 수어 | 고등교육을 위해 케냐 방문 |

### 관련 수어
- 케냐 수어와 상호 이해 가능
- 탄자니아 수어와 유사성

### 변이
- 국경 지역: 지역 방언 존재
- BSL 영향: Uganda School for the Deaf에서 특히 강함

## 교육 현황

### 학교 현황 (2014)
| 유형 | 수 |
|------|-----|
| 초등 농학교 | 11개 |
| 중등 농학교 | 2개 |
| 통합학교 농인 유닛 | 약 40개 |
| 유닛 있는 지역 | 100개 지역 중 40개 미만 |

### 취학률
- **농아동 취학률: 2% 미만!**
- 대부분 교육 기회 없음
- 일반 교육 시스템에 남는 경우도 있음

### 교육 방식
- 이중언어 교육 + Total Communication
- 유아원~대학/직업학교까지
- 모든 농아동에게 접근 가능하지 않음
- 영어 글쓰기 학습: 약 40%
- **SEE (Signed Exact English) 증가 추세**

### 우간다의 언어 상황
- 44개 이상 구어 존재
- 농학교: 영어 글쓰기 + USL 수어 교육
- Signed English와의 간섭 현상 (Diane Brentari)

### 도시/농촌 격차
- USL 지식: **주로 도시** 집중
- 농촌 농인: 교육 접근성 열악
- 그러나 USL = 농인 커뮤니티 **핵심 정체성**
