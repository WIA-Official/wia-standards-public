# Batch 107: EriSL (Eritrean Sign Language)

## 기본 정보
- **언어 코드**: EriSL
- **ISO 639-3**: (미등록)
- **정식 명칭**: Eritrean Sign Language
- **영어 명칭**: Eritrean Sign Language
- **사용 국가**: 에리트레아
- **사용자 수**: 약 15,000명
- **언어 계통**: 스웨덴 수어 가족 (핀란드 수어 영향)

## 역사적 배경

### 기원
- **1955**: 스웨덴/핀란드 기독교 선교사들이 수어 도입
- 스웨덴 수어 + 핀란드 수어 혼합
- 지역 에리트레아 표지 추가
- 수단 영향 (ASL 기반)

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1955 | 스웨덴/핀란드 선교사 수어 도입 |
| 1970 | 전쟁으로 선교사들 철수, 에리트레아인 교사 계속 |
| **1998** | EriNAD (에리트레아 농인 협회) 설립 |
| **2005** | "탈선교화" (demissionization) 시작 |
| 2006 | Bata School for the Deaf 설립 (Noreen Corey) |

### 식민지 영향 없음
- 이탈리아, 영국, 에티오피아 식민 수어 영향 없음
- 스웨덴/핀란드 수어가 기반

## 언어적 특성

### 언어 계통
- **스웨덴 수어 가족**
- 핀란드 수어와 **70% 동일** (Moges 2011)
- 수단 수어 영향 (ASL 기반)

### 탈선교화 (Demissionization)
- **2005년부터** EriNAD 주도
- 스웨덴/핀란드 표지 → 에리트레아 고유 표지로 대체
- 목표: 더 독립적이고 "토착적" 언어 창출

### 지역 변이
- 지역 간 변이 존재
- 학교 내 사용 vs 농촌 사용 차이
