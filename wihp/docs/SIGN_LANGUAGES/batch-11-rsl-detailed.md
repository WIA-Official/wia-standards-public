## RSL Characteristics

### Historical Background
- 1806: First Russian deaf school founded near St. Petersburg
- Early teachers were from **France and Austria** → LSF family connection
- 1969: Galina Zaitseva wrote PhD thesis on RSL spatial relationships
- 1992: Galina Zaitseva standardized term "Russkii Zhestovyi Yazyk" (Русский Жестовый Язык)
- First RSL Dictionary created (descriptive)
- 2020s: Bukva dataset created for RSL dactyl recognition (3,757 videos)

### Key Figures
- **Galina Zaitseva** - Pioneer of RSL research
- **Victor Fleury** - Authored "Glukhonemye" (Deafmutes, 1835), first RSL dictionary
- **Ivan Arnold** - Deaf artist, opened Moscow school for Deaf (1860)

### RSL vs Signed Russian (Important Distinction!)
| Feature | RSL (Русский Жестовый Язык) | Signed Russian (Калькирующая речь) |
|---------|----------------------------|-----------------------------------|
| Origin | Natural, evolved in deaf community | Artificial, follows Russian grammar |
| Word Order | SVO/SOV flexible | Strictly follows Russian |
| Grammar | Own unique grammar | Russian grammatical features |
| Status | Lower social status | Higher status (literacy association) |
| Use in Schools | Rarely | Near-exclusive |

**Note:** Many deaf RSL users see their own language as "jargon without grammar" due to societal attitudes.

### Key Features
- **French Sign Language family** - Due to French/Austrian teachers
- **33 letters in dactyl** (one for each Russian alphabet letter)
- **8 dynamic letters** in dactyl: д, ё, з, й, ц, щ, ъ, ь
- **25 static letters** in dactyl
- **SVO and SOV word order** (Subject usually before predicate 95%)
- **Object before verb** 74-81% of time
- **Regional variations** - 70-80% lexical similarity between regions
- **Bilingualism common** - RSL + Russian written/spoken

### Dactylology (Fingerspelling)
- Called "дактилология" or "дактиль" in Russian
- 33 signs for 33 Russian letters
- Some signs copy the letter shape (л, м)
- Used for proper nouns, technical terms
- Parents of deaf children often only know dactyl

### Grammar Features
- **Topicalization:** Lowered eyebrows + head nod, or raised eyebrows
- **No determiners:** Pointing signs act as demonstratives
- **Doubling:** Lexical signs doubled for emphasis
- **Classifier predicates:** Handshape = object, movement = motion

### Organizations
- **VOG** - All-Russian Society of the Deaf (Всероссийское общество глухих)
- **G.L. Zaitseva Centre** - Education of Deaf and Sign Language
- **Surdophone** - RSL translation system

### Resources
- **Spreadthesign** - RSL video dictionary (14,000+ words)
- **Surdoserver** - RSL video dictionary
- **Bukva dataset** - 3,757 videos for dactyl recognition

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION A: Basic Greetings (5 signs)

### 1. ПРИВЕТ (Privet / Hello)

**RSL Concept:** Informal greeting

**Detailed Motion:**
- Wave hand in greeting
- Or: Flat hand from forehead moves outward
- Friendly facial expression
- Often accompanied by smile and eye contact

**WIA Talk Code:** `HS02-LC09-MV06-OR01-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat open hand |
| LC | 09 | Neutral space (or near forehead) |
| MV | 06 | Wave side to side |
| OR | 01 | Palm forward |
| FE | 02 | Friendly smile |

**Russian Spoken:** Привет (Privet) - informal, Здравствуйте (Zdravstvuyte) - formal

**Source:** Wikipedia (RSL), TikTok RSL community

---

### 2. СПАСИБО (Spasibo / Thank You)

**RSL Concept:** Gratitude gesture

**Detailed Motion:**
- Method 1: Flat hand from chin moves forward and down
- Method 2: Hand touches chin, arcs outward
- May include slight bow
- Grateful expression

**WIA Talk Code:** `HS02-LC05-MV03-OR04-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat open hand |
| LC | 05 | Chin area |
| MV | 03 | Forward and down movement |
| OR | 04 | Palm initially down/forward |
| FE | 02 | Grateful expression |

**Russian Spoken:** Спасибо (Spasibo), Большое спасибо (Bol'shoye spasibo - Thanks a lot)

**Cultural Note:** Russians value sincerity in gratitude; often paired with handshake, hug, or nod

**Source:** polyglottistlanguageacademy.com, mondly.com

## SECTION B: Numbers 1-10 (Числа)

### RSL Number System Overview

RSL numbers are similar to many European sign languages:
- One-handed system
- Palm typically facing viewer
- Numbers 1-5 use finger counting
- Numbers 6-10 have specific configurations

**Note:** RSL dactyl recognition research (Bukva dataset) includes numbers 0-9

## SECTION C: Basic Phrases (5 signs)

### 16. Я ТЕБЯ ЛЮБЛЮ (Ya Tebya Lyublyu / I Love You)

**RSL Concept:** Love expression

**Detailed Motion:**
- Я (I): Point to self
- ЛЮБЛЮ (Love): Cross arms over heart, hug self
- ТЕБЯ (You): Point to person
- Loving expression essential

**WIA Talk Code (ЛЮБЛЮ/Love):** `HS02-LC07-MV26-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands crossed over chest |
| LC | 07 | Chest/heart area |
| MV | 26 | Cross arms, hug motion |
| OR | 05 | Arms across body |
| FE | 01 | Loving expression |

**Cultural Note:** Russians express love more deliberately and meaningfully than quick English "love you"

### 19. КАК ДЕЛА? (Kak Dela? / How are you?)

**RSL Concept:** Wellness inquiry

**Detailed Motion:**
- КАК (How): Question gesture
- ДЕЛА (Affairs): Directional sign
- Question expression
- **Important:** Russians ask genuinely, not as empty greeting

**WIA Talk Code:** `HS06-LC09-MV03-OR01-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 06 | Thumbs up or open hand |
| LC | 09 | Neutral space |
| MV | 03 | Forward inquiry motion |
| OR | 01 | Palm/thumb forward |
| FE | 06 | Question expression |

**Cultural Note:** Unlike English, Russians ask "Как дела?" genuinely interested in the answer, not as reflexive greeting

**Response Options:**
- Хорошо, спасибо! (Khorosho, spasibo!) - Good, thanks!
- Спасибо, прекрасно! (Spasibo, prekrasno!) - Thank you, excellent!
- Нормально (Normal'no) - Normal/OK

## RSL Linguistic Features Summary

### Grammar
- **Word Order:** SVO and SOV both used
- **Subject before predicate:** 95% of time
- **Object before verb:** 74-81% of time
- **Topicalization:** Lowered eyebrows + head nod, or raised eyebrows
- **No determiners:** Pointing signs act as demonstratives
- **Doubling:** Lexical signs repeated for emphasis

### Dactylology (Fingerspelling)
- 33 letters for 33 Russian alphabet letters
- 25 static + 8 dynamic letters
- Used for proper nouns, technical terms
- Many signs "copy" the letter shape

### Regional Variation
- 70-80% lexical similarity between regions
- Comparable to Polish/Estonian sign language variation
- No standardized form across Russia

### Cultural Features
- **Cheek-kissing:** Common greeting among close friends (3 kisses on alternating cheeks)
- **Handshakes:** Common, especially among men
- **Genuine questions:** "Как дела?" is genuine inquiry, not empty greeting
- **Sincerity valued:** Russians appreciate meaningful gratitude

### Related Sign Languages
- **French Sign Language family** (via French/Austrian teachers)
- **Israeli deaf community** has RSL speaker base
- Possibly used in Ukraine, Belarus, Tajikistan

### Current Challenges
1. **Lacks state support** despite legal recognition
2. **Signed Russian dominates schools** - RSL seen as "jargon"
3. **Shortage of skilled interpreters**
4. **Most deaf born to hearing families** - limited native acquisition
5. **Low social status** compared to Signed Russian
