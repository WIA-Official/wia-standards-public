### Introduction
Deaf education has been shaped by centuries of debate over whether deaf children should be taught through sign language or oral methods (speech and lipreading). This history has profound implications for sign language vitality, deaf identity, and educational outcomes.

Today, growing evidence supports bilingual-bicultural approaches that use sign language as the primary language of instruction.

### Educational Philosophies

#### Oralism
| Aspect | Description |
|--------|-------------|
| Method | Speech, lipreading, audition |
| Sign | Prohibited or discouraged |
| Goal | Integration with hearing |
| Peak era | 1880-1970s |
| Current | Still practiced some places |

#### Manualism
| Aspect | Description |
|--------|-------------|
| Method | Sign language primary |
| Speech | Optional or not emphasized |
| Goal | Language access |
| Era | 1760-1880, post-1960s |
| Current | Growing support |

#### Total Communication
| Aspect | Description |
|--------|-------------|
| Method | All communication modes |
| Simultaneous | Sign + speech together |
| Era | 1970s-1990s peak |
| Criticism | Neither language fully |
| Current | Declining |

#### Bilingual-Bicultural (Bi-Bi)
| Aspect | Description |
|--------|-------------|
| Method | SL first, written language second |
| Principle | Two complete languages |
| Culture | Both deaf and hearing |
| Evidence | Growing research support |
| Current | Increasingly adopted |

### Milan Congress 1880
Critical turning point:
| Aspect | Detail |
|--------|--------|
| Location | Milan, Italy |
| Decision | Oral method "superior" |
| Voting | Mostly hearing educators |
| Deaf exclusion | Few deaf attendees |
| Impact | Sign banned in schools |
| Duration | 100+ years of impact |
| Reversal | ICED 2010 rejects |
