### Introduction
Sign languages, like spoken languages, create new vocabulary through compounding—combining two or more existing signs to create a new meaning. Compound signs often undergo phonological changes that distinguish them from the sequence of their parts.

### What Is Compounding?
| Feature | Description |
|---------|-------------|
| Definition | Two+ signs combine |
| New meaning | Different from parts |
| Single unit | Functions as one sign |
| Productive | Creates vocabulary |

### Lexicalization
| Stage | Description |
|-------|-------------|
| Phrase | Two separate signs |

### Productive vs. Lexicalized
| Type | Feature |
|------|---------|
| Productive | Can create new |
| Lexicalized | Fixed in dictionary |
| Both | Exist in language |

### Compound vs. Phrase
| Feature | Compound | Phrase |
|---------|----------|--------|
| Duration | Shorter | Longer |
| Hold | Reduced | Present |
| Meaning | New, single | Compositional |
| Modifiable | Usually not | Can be |
