# Batch 105: LSQ (Langue des Signes Québécoise / Quebec Sign Language)

## 기본 정보
- **언어 코드**: LSQ
- **ISO 639-3**: fcs
- **정식 명칭**: Langue des Signes Québécoise
- **영어 명칭**: Quebec Sign Language
- **사용 지역**: 퀘벡, 온타리오, 뉴브런즈윅 (프랑스어권 캐나다)
- **사용자 수**: 수만 명 (정확한 추정 어려움)
- **법적 인정**: 2019년 캐나다 접근성법 연방 인정, 2007년 온타리오 주 인정
- **언어 계통**: 프랑스 수어(LSF) 가족

## 역사적 배경

### 기원
- **1831**: 프랑스에서 수어 교육자들 도착
- **1836**: 생아쟁트(Saint-Hyacinthe) 학교 설립
- **1848**: 몬트리올 학교 설립
- **~1850**: LSQ 형성 시작
- 이전에는 조직화된 농인 수어 기록 없음

### 명칭 역사
- 1980년대 이전: "langue des signes canadiens français"
- **1980년대**: Raymond Dewar(농인 활동가)가 "LSQ" 명칭 확립
- ASL과 LSF로부터의 독립성 강조

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1831 | 프랑스 Clercs de Saint-Viateur 교육자 도착 |
| 1848 | Sœurs de la Providence 몬트리올 학교 |
| 1880 | 밀라노 회의 후 구화주의 확산 |
| 1972 | 프랑스어권 학교에서 수어 금지 (마지막까지) |
| 1980년대 | "LSQ" 명칭 공식화, 해석 서비스 시작 |
| **1986** | 제1회 청각장애 정상회의 - LSQ 인정 요구 시작 |
| **1988** | UQAM Groupe de recherche sur la LSQ 설립 |
| 2002 | 법안 104 이후 권고안 거부 |
| **2007** | 온타리오 법안 213 - 교육/입법/사법 분야 인정 |
| 2013 | 법안 14 논의 (통과 실패) |
| **2019** | 캐나다 접근성법 - 연방 차원 인정! |

## 언어적 특성

### 언어 계통
- **프랑스 수어(LSF) 가족**
- LSF + ASL 혼합에서 발전
- ASL과 다른 독립적 언어!

### 차용 관계
| 출처 | 차용 내용 |
|------|----------|
| **LSF** | "how", "why" 등 기본 표현 |
| **ASL** | 지문자, 가족 관련 어휘 |
| **프랑스어** | 입모양(mouthing), 초성화 |
| **국제 수어** | 지명 (CHINA, JAPAN 등) |

### 문법적 특징
- **손모양**: 116개
- **지문자**: ASL과 동일 (한 손 체계)
- **숫자**: ASL과 동일 (한 손 체계)
- **조음 영역**: 신체, 신체 앞 공간, 지문자 영역(어깨 높이)

### 입모양(Mouthing)
- 가장 빈번한 차용 형태
- 동사, 형용사, 명사에 사용
- 의미 명확화, 용어 형성 역할
- 때로 수지 표현 없이 단독 사용

## 연구 및 기관

### 주요 기관
| 기관 | 역할 |
|------|------|
| **UQAM Groupe de recherche sur la LSQ** | 핵심 연구기관 (1988~) |
| CAD-ASC | 캐나다 농인 협회, ASL/LSQ 인정 캠페인 |
| OPHQ | 퀘벡 장애인 사무소, 해석 서비스 |
| RESO | LSQ-프랑스어 어린이 사전 개발 |

### 연구 현황
- **Colette Dubuisson & Marie Nadeau (1993)**: LSQ가 완전한 언어임을 입증
- **Anne-Marie Parisot** 등: LSQ 문법 연구
- **UQAM**: LSQ 언어학 연구 센터

## 법적 상태

### 연방 (캐나다)
- **2019 접근성법**: ASL, LSQ, 원주민 수어 인정
- 공식 언어 지위(영어/프랑스어)와 동등하지는 않음
- 연방 정보/서비스 접근성 요구

### 온타리오 (2007)
- **법안 213**: 교육, 입법, 사법 분야에서 인정
- "캐나다에서 LSQ를 인정한 유일한 지역" (2007 당시)

### 퀘벡
- **아직 공식 인정 없음!**
- 프랑스어 헌장(Charter of the French Language) 개정 요구 지속
- 2002, 2013년 권고안 모두 거부됨

## 교육 현황

### 교육 유형
| 유형 | 설명 |
|------|------|
| 일반 학급 | 통역사 유무에 따라 통합 |
| 특수 학급 | 일반 학교 내 |
| 특수 학교 | 총의사소통(Total Communication) |

### 이중언어-이중모드 교육
- École Gadbois 등에서 채택
- LSQ로 교육, LSQ 과목으로 학습
- 몬트리올, 오타와 농학교에서 LSQ-프랑스어 이중언어 프로그램

### 고등교육
- 1980년대부터 해석 서비스로 대학 접근 가능
- 1984년 OPHQ 정책 "À part … égale"

## 특이사항

### Sourde 정체성
- **Sourd** (대문자 S): LSQ 사용, 농문화 정체성, 장애가 아닌 정체성
- **sourditude**: 농인의 자기 발견과 긍정의 여정

### 역사적 억압
- 2차 대전 후~1972년: 구화주의 강제
- 수어 사용 금지, 잔존 청력만 활용 강요
- 청각주의(audism)로 인한 낮은 문해율

### 관련 수어
| 수어 | 관계 |
|------|------|
| ASL | 영어권 캐나다, 지문자/숫자 동일 |
| LSF | 조상 언어, 어휘 차용 |
| 원주민 수어 | 함께 연방 인정 |
