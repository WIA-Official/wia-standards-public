# Batch 142: Marthas Vineyard Sign Language (마서스 비니어드 수어)

## 기본 정보
- **언어명**: Marthas Vineyard Sign Language
- **약칭**: MVSL
- **ISO 639-3**: mre
- **국가**: 미국
- **지역**: Massachusetts, Marthas Vineyard 섬
- **사용자 수**: 0 (멸종)
- **언어 유형**: 마을 수어
- **계통**: 독립 언어
- **상태**: 멸종 (1952년 마지막 화자 사망)

## 역사적 배경

### 기원
- **발생 시기**: 17세기 초
- **장소**: 영국 Kent 지역에서 이주민과 함께
- **특징**: 미국 농인 공동체 역사의 중요한 일부

### 역사적 발전
1. **1690년대**: 최초 기록
2. **18-19세기**: 전성기, 섬 전체 사용
3. **1800년대**: Thomas Hopkins Gallaudet 방문
4. **1817년**: ASL 형성에 영향
5. **1952년**: 마지막 원어민 Katie West 사망

### 청각장애 비율
- **Chilmark 타운**: 인구의 약 1/25 (4%)
- **일반 미국**: 약 1/5,700
- **원인**: 유전적 청각장애 (열성 유전자)

### 사회적 통합
- "Everyone here spoke sign language"
- 청인과 청각장애인 구분 거의 없음
- 완전한 사회 통합의 모범 사례

## 언어적 특징

### ASL과의 관계
- **ASL 형성에 기여**: 1817년 American School for the Deaf 설립 시
- **Laurent Clerc + MVSL 화자**: ASL 어휘 일부 MVSL 기원
- **그러나 별도 언어**: MVSL은 ASL과 구별

### 기원
- **영국 수어 영향**: Kent 지역 수어와 연관 추정
- **독자 발전**: 섬 환경에서 200년 이상 발전

### 어휘 특징
- 섬 생활 관련
- 어업, 농업 관련
- 지역 지명

### 문법
- 연구 제한적 (멸종 전 체계적 기록 부족)
- 인터뷰 기록 일부 존재

## 사회언어학적 특징

### 언어 사용 패턴
- 청인도 수어 유창
- 교회, 가정, 직장 모두 사용
- "말하기 귀찮을 때" 수어 사용

### 지역별 분포
- **Chilmark**: 청각장애 비율 가장 높음
- **West Tisbury**: 높은 비율
- **Edgartown, Vineyard Haven**: 낮은 비율

### 인터뷰 기록 (Nora Ellen Groce)
- 1980년대 노인 인터뷰
- "청각장애인? 그런 구분 없었어"
- "모두가 수어 했으니까"

## 언어 멸종 과정

### 멸종 원인
1. **외부 결혼 증가**: 유전자 풀 확대
2. **이주**: 젊은 세대 섬 떠남
3. **ASL 교육**: 청각장애 학교 ASL 사용
4. **청각장애 출생 감소**: 20세기 초 급감

### 주요 시점
- **1850년대 이후**: 외부 결혼 증가
- **1900년대 초**: 청각장애 출생 급감
- **1952년**: 마지막 원어민 사망
