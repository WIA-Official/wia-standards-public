# Batch 166: Japan Village Sign Languages - 일본 마을 수어

## 개요
일본에는 JSL 외에 독자적인 마을 수어들이 존재
섬 지역, 가족 단위 발생

## 1. Amami Oshima Sign Language

### 기본 정보
- **일본명**: 奄美大島手話 / シマ手話
- **별칭**: Koniya Sign Language
- **ISO 639-3**: jks
- **위치**: Amami Islands, Koniya 지역
- **유형**: 마을/가족 수어

### 특징
- 우성 유전 청각장애
- 지형으로 인한 가족별 고립
- 극단적 어휘 다양성
- 단일 언어라기보다 언어 복합체

### 연구
- 1999년 13 deaf + 8 hearing 연구
- 25개 기본 수화 패턴 확인
- 홈사인 특성

## 2. Miyakubo Sign Language

### 기본 정보
- **일본명**: 宮窪手話
- **별칭**: Ehime-Oshima Sign Language
- **위치**: Ehime-Oshima Island, 남서 일본
- **화자**: 3가족, 29 deaf + 22 hearing

### 역사
- 최소 3세대 이상
- 2000년대 이전 역사 불명

### 특징
- 공유 수어 공동체
- JSL과 별개 언어

- JSL: 1878년 이후 발달, 국가 수어
- 마을 수어들: 독자 발생, 별개 계통

### 위협
