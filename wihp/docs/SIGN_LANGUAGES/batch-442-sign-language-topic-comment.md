# Sign Language Topic-Comment Structure
## Information Structure in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|

### Introduction
Many sign languages, including ASL, use topic-comment structure as a primary way of organizing information. The topic (what the sentence is about) is established first, often marked by raised eyebrows, and then the comment (what is said about the topic) follows.

### What Is Topic-Comment?
| Component | Function |
|-----------|----------|
| Topic | What we're talking about |
| Comment | What we say about it |
| Order | Topic first |
| Marker | Non-manual features |

### Topic Markers (Non-manual)
| Feature | Description |
|---------|-------------|
| Eyebrows | Raised |
| Head | Slight nod/tilt |
| Pause | Brief hold |
| Body | Slight lean |
| Eyes | Widened |

### Topic-Comment Examples (ASL)
| Topic | Comment | Meaning |
|-------|---------|---------|
| THAT BOOK | I READ FINISH | "That book, I finished reading it" |
| MY SISTER | SHE DOCTOR | "My sister, she's a doctor" |
| YESTERDAY | I SICK | "Yesterday, I was sick" |

### Topicalization
| Process | Description |
|---------|-------------|
| Move to front | Topic position |
| Add markers | Non-manual features |
| Establish | What's discussed |
| Comment | Follow with statement |

### Topic vs. Subject

|-------------|------------|

### Contrast with English
| Feature | English | ASL |
|---------|---------|-----|
| Structure | SVO dominant | Topic-comment common |
| Marking | Word order | Non-manual + position |
| Frequency | Less common | Very common |
