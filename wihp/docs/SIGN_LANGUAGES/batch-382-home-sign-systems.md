# Home Sign Systems
## Spontaneous Family Communication with Deaf Children

### Overview
| Attribute | Detail |
|-----------|--------|
| **Type** | Idiolect/Family sign systems |
| **Users** | Deaf children + hearing families |
| **Context** | No sign language access |
| **Status** | Ubiquitous globally |
| **Research | Significant linguistic interest |

### Introduction
Home sign systems (also called homesign or kitchen sign) are gestural communication systems that spontaneously develop when a deaf child has no access to a conventional sign language. These systems emerge naturally in families where deaf children are raised by hearing parents who don't know a sign language.

Home sign represents crucial evidence for the human language instinct and provides insight into language origins and development.

### How Home Sign Develops
| Stage | Development |
|-------|-------------|
| Early | Pointing, reaching, showing |
| Toddler | First consistent signs emerge |
| Childhood | Vocabulary expands |
| Later | Grammatical structure appears |
| Long-term | Can become complex system |

### Key Characteristics
| Feature | Description |
|---------|-------------|
| Iconicity | High visual similarity |
| Consistency | Individual uses same signs |
| Invention | Not taught, created by child |
| Shared | Family learns child's signs |
| Limited input | No conventional language model |

| Property | Evidence |
|----------|----------|

| Word order | Emerges systematically |

### Research Significance
Home sign is important because it shows:
- Language can emerge without input
- Children create grammatical structure
- Innate language capacity
- Resilience of language drive
- Basis for creole development

### Nicaraguan Sign Language Connection
Home sign became full language:
| Context | Development |
|---------|-------------|
| 1970s-80s | Deaf schools opened Nicaragua |
| Home signers | Brought individual systems |
| Contact | Children combined home signs |
| Result | Nicaraguan Sign Language emerged |
| Significance | Language creation documented |

### What Home Sign Lacks
| Feature | Absent/Limited |
|---------|----------------|
| Community use | Usually just family |
| Full grammar | Incomplete |
| Morphological complexity | Limited |
| Register variation | Minimal |
| Literature/culture | No |
| Standardization | Each family unique |

### Outcomes for Home Signers
| Outcome | Frequency |
|---------|-----------|
| Later SL acquisition | Common if access gained |
| Continued home sign | If isolated |
| Language delay | Without intervention |
| Cognitive effects | Can impact development |

### Home Sign vs. Full Sign Language
| Aspect | Home Sign | Full SL |
|--------|-----------|---------|
| Source | Individual creation | Community transmission |
| Complexity | Limited | Full |
| Users | Usually family | Community |
| Standardization | None | Conventional |
| Input | Minimal | Rich |
