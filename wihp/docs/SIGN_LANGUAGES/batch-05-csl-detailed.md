# WIA Talk Sign Language Mapping - CSL (Chinese Sign Language / 中国手语)
## Batch 05: 20 Signs with Full WIA Talk Component Analysis

**Language:** CSL - Chinese Sign Language (中国手语, Zhōngguó Shǒuyǔ)
**Region:** Mainland China
**Estimated Users:** ~20 million Deaf and hard-of-hearing
**Dialects:** Northern CSL (Beijing/ASL influence), Southern CSL (Shanghai/FSL influence)
**Recognition:** 2018 National Common Sign Language Standard published

---

## CSL Characteristics

### Key Features
- **Character-based iconicity**: Many signs visually represent Chinese characters
- **One-handed finger counting**: Numbers 1-10 expressible with single hand
- **Pinyin fingerspelling**: 30-character manual alphabet (adopted 1963)
- **Cultural encoding**: Signs reflect Chinese kinship/social concepts
- **Thumb/pinky polarity**: Thumb = positive, Pinky = negative (shared with BSL)

### Two Main Dialects
1. **Northern CSL (北方手语)** - Beijing-centered
   - Influenced by American Sign Language (ASL)
   - More character-based puns
   - Originates from Chefoo School (1887)
   
2. **Southern CSL (南方手语)** - Shanghai-centered
   - Influenced by French Sign Language (FSL)
   - Spread to Hong Kong, Malaysia
   - Originates from Shanghai School (1897)

### Historical Timeline
- Tang Dynasty (618-959 CE): First written reference to 手语 (shǒuyǔ)
- 1887: First deaf school by American missionary Mills (Chefoo)
- 1897: French Catholic school in Shanghai
- 1959: Chinese Manual Phonetic system published
- 1963: Official Fingerspelling Scheme adopted
- 1990: Term "中国手语" officially designated
- 2018: National Common Sign Language Standard
- 2019: 5,000-word National List published

### Not Related To
- Taiwanese Sign Language (TSL) - Japanese family
- Hong Kong Sign Language (HKSL) - derived from Southern CSL but now separate
- Tibetan Sign Language - isolate

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION A: Basic Greetings (5 signs)

### 1. 你好 (NǏ HǍO - Hello)

**CSL Concept:** Compound of "you" + "good"

**Detailed Motion:**
1. 你 (YOU): Index finger points toward person
   - Or: Open palm gesture toward person (more polite)
2. 好 (GOOD): Thumbs up gesture
   - Thumb extended upward

**WIA Talk Code:** `HS07-LC09-MV03-OR01-FE02` + `HS06-LC09-MV00-OR01-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 07→06 | Index finger → Thumb up |
| LC | 09 | Neutral space |
| MV | 03→00 | Point forward → Static thumbs up |
| OR | 01 | Palm/thumb toward viewer |
| FE | 02 | Friendly smile |

**Source:** chinadaily.com.cn, bilibili.com, douyin.com

### 2. 谢谢 (XIÈ XIÈ - Thank You)

**CSL Concept:** Both thumbs bow down in gratitude

**Detailed Motion:**
- Both hands form thumbs up
- Both thumbs bend downward at the joint twice
- Like thumbs bowing in thanks

**WIA Talk Code:** `HS06-LC09-MV18-OR01-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 06 | Thumbs up (両手) |
| LC | 09 | Neutral space (chest level) |
| MV | 18 | Thumbs bend down at knuckle (2x) |
| OR | 01 | Palms toward each other, thumbs up |
| FE | 02 | Grateful expression |

**Cultural Note:** The bowing thumbs represent bowing in gratitude

**Source:** bilibili.com/手护有你, douyin.com, zhihu.com

### 3. 对不起 (DUÌ BÙ QǏ - Sorry)

**CSL Concept:** Two methods exist

**Method 1 - Finger snap:**
- Both hands: thumb and middle finger touching
- Left hand below, right hand above
- Touch together and snap fingers apart
- Repeat twice

**Method 2 - Forehead + chest:**
- Right hand, all five fingers together
- Touch forehead
- Then pinky touches chest twice

**WIA Talk Code (Method 1):** `HS05-LC09-MV21-OR05-FE04`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 05 | Pinch gesture (thumb + middle) |
| LC | 09 | Neutral space |
| MV | 21 | Touch and snap apart (2x) |
| OR | 05 | Palms facing each other |
| FE | 04 | Apologetic expression |

**WIA Talk Code (Method 2):** `HS03-LC02-MV03-OR05-FE04` + `HS32-LC07-MV15-OR05-FE04`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 03→32 | Open 5 fingers → Pinky extended |
| LC | 02→07 | Forehead → Chest |
| MV | 03→15 | Forward → Tap twice |
| OR | 05 | Palm facing body |
| FE | 04 | Sorry/regretful expression |

**Source:** bilibili.com/video/BV13sZnY8ErS, douyin.com

### 5. 不是 / 不 (BÙ SHÌ / BÙ - No)

**CSL Concept:** Wave off or negative gesture

**Detailed Motion:**
- Open hand waves side to side (like waving off)
- Or: Index finger wagged side to side
- Pinky extended for strong negation

**CSL Feature:** Pinky = negative meaning (shared with BSL)

**WIA Talk Code:** `HS02-LC09-MV06-OR01-FE03`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Open hand (or HS32 pinky for emphasis) |
| LC | 09 | Neutral space |
| MV | 06 | Wave side to side |
| OR | 01 | Palm facing forward |
| FE | 03 | Negative expression, head shake |

**Source:** en.wikipedia.org/Chinese_Sign_Language

## SECTION B: Numbers 1-10 (Chinese Hand Counting)

### Overview: 手势数字 (Shǒushì Shùzì)

Chinese number gestures are unique:
- 1-5: Similar to Western counting
- 6-10: Special one-hand symbolic gestures
- Represent Chinese character shapes
- Used for commerce, bargaining, noisy environments
- Fully integrated into CSL

**Regional Variations:** Taiwan differs slightly for 7-9

**Source:** en.wikipedia.org/Chinese_number_gestures, baodaotalk.com

## SECTION C: Basic Phrases (5 signs)

### 16. 我爱你 (WǑ ÀI NǏ - I Love You)

**CSL Concept:** Three-part compound: I + Love + You

**Detailed Motion:**
1. 我 (I): Index finger points to self
2. 爱 (LOVE): One hand gently strokes the thumb of the other hand
   - Represents tender, cherishing feeling
3. 你 (YOU): Point toward person

**WIA Talk Code:** `HS07-LC07-MV00-OR05-FE01` + `HS02-LC09-MV09-OR05-FE01` + `HS07-LC09-MV03-OR01-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 07→02→07 | Point → Flat hand → Point |
| LC | 07→09→09 | Chest → Neutral → Neutral |
| MV | 00→09→03 | Static → Stroke → Point out |
| OR | 05→05→01 | Inward → Inward → Forward |
| FE | 01 | Loving expression throughout |

**Source:** zhihu.com, bilibili.com/爱情表白

## CSL Linguistic Features Summary

### Grammar Patterns
- **SOV Order:** Subject + Object + Verb (我+水果+买 = I buy fruit)
- **Time First:** 昨天+我+学校+上课 (Yesterday I school attend)
- **Spatial Reference:** Direction indicates who→whom (我→你 = I give you)
- **Question Markers:** Eyebrow movement + expression (not hand sign 吗)

### Cultural Encoding Examples
- No generic "brother" - only "older brother" (哥哥) and "younger brother" (弟弟)
- Numbers carry cultural meaning:
  - 8 (八) = prosperity (sounds like 发)
  - 9 (九) = longevity (sounds like 久)
  - 4 (四) = sometimes avoided (sounds like 死, death)

### Thumb/Pinky Polarity (Shared with BSL)
- **Thumb** = positive meaning (好, 是)
- **Pinky** = negative meaning (不, 不知道)
