| Attribute | Detail |
|-----------|--------|
| **Topic** | Fingerspelling in sign languages |

| **Types** | One-handed, two-handed |
| **Use** | Names, new words, emphasis |
| **Part of** | Most sign languages |

### Introduction
Fingerspelling is a system of representing letters of a written language using hand configurations. It serves as a bridge between sign language and written language, used for proper nouns, technical terms, and words without established signs.
