# Batch 185: Trinidad and Tobago Sign Language (TTSL)

## 개요
| 항목 | 내용 |
|------|------|
| **언어 코드** | lst (Ethnologue) |
| **현지명** | Trinidad and Tobago Sign Language / Trinbago Sign Language (TSL) |
| **지역** | 트리니다드토바고 |
| **화자 수** | 수백 명 (정확한 수 미상) |
| **기원** | 1943년 Cascade School for the Deaf 설립 |
| **상태** | 취약 (Vulnerable) - ASL과의 경쟁 |

## 역사적 배경

### 농교육 이전 (1943년 이전)
- 특수 농교육 부재
- 농인들 전국에 분산
- 개인별 홈사인 사용

### Cascade School for the Deaf 설립 (1943)
- 1943: Edward Street, Port of Spain에 첫 농학교 설립
- 이후 Cascade 현 위치로 이전
- 기숙사 제공 (먼 거리 학생용)
- 전국 농아동 최초 집결

### 구화주의 시대 (1943-1974)
- 구화법 교육: 발음, 독순술 중심
- 수어 금지: 교실에서 수어 사용 처벌
- 숨겨진 발전: 기숙사에서 자유롭게 수어 사용
- 결과: 학생들이 자체적으로 새 언어 창조

### Total Communication 도입 (1974)
- 1974: 구화주의에서 Total Communication으로 전환
- ASL 도입: 미국 수어 교육 시작

## 사용자 커뮤니티

### 세대 구분
| 세대 | 출생 | 특징 |
|------|------|------|
| 노년층 | 1975년 이전 | 순수 TTSL 사용 |
| 중년층 | 1975-1990 | TTSL + ASL 혼합 |
| 청년층 | 1990년 이후 | 주로 ASL, TTSL 제한적 |

### 사용 영역
- 공식/교육: ASL 우세
- 농협회: TTSL + ASL 혼합
- 비공식/가정: TTSL 우세 (노년층)

### 주요 기관
- Cascade School for the Deaf: 발생지
- Deaf Organization of Trinidad and Tobago (DOTT)
- Deaf Empowerment Organization of Trinidad and Tobago

## 언어학적 특성

### 발달 증거
- 수화 공간 축소: 시간이 지나며 수화가 더 작은 공간에서 이루어짐
- 효율성 증가: 언어 성숙의 일반적 패턴
- 세대 간 차이 관찰 가능

### 문법적 특성
- 독자적 문법 구조
- ASL과 구별되는 어순
- Trinidad English Creole 영향
- 독특한 시제 표현 체계

### 영향 요인들
1. BSL 영향: 초기 교육자 일부
2. ASL 수지 알파벳: 철자법 차용
3. ASL: 1974년 이후 대규모 영향
4. Trinidad English Creole: 지역 문화 반영

## 위기 요인

### 언어적 위협
- ASL 위세: 교육, 국제 소통에서 ASL 선호
- 세대 단절: 젊은 세대 TTSL 미습득
- 코드 혼합: 순수 TTSL 사용 감소

### 사회적 위협
- 농학교 등록 감소
- 인공와우 보급 확대
- 주류 통합 교육 증가

### 지역사회 입장
- DOTT: TTSL은 소중한 유산, 보존 필요
- 일부: ASL로 국제 소통 가능하므로 TTSL 사라져도 무방
- 농공동체 자체 보존 의지 존재
