# Batch 140: Inuit Sign Language (이누이트 수어)

## 기본 정보
- **언어명**: Inuit Sign Language
- **이누이트어명**: Uukturausingit (이누크티투트)
- **약칭**: IUR
- **ISO 639-3**: iks
- **국가**: 캐나다
- **지역**: 누나부트, 북극권
- **사용자 수**: 47-50명 (2020년 추정)
- **언어 유형**: 원주민 수어
- **계통**: 독립 언어

## 역사적 배경

### 기원
- **발생 시기**: 여러 세대 전 (정확한 시기 미확인)
- **장소**: 캐나다 북극권 이누이트 공동체
- **특징**: 이누이트 문화와 함께 발전한 독자 수어

### 역사적 맥락
- 캐나다 청각장애 학교 이전부터 존재
- 고립된 북극 공동체에서 자연 발생
- ASL 도입 이전 세대 사용

### 발견 및 연구
- **2000년대**: 학계에서 인식 시작
- **2008년**: MacDougall 연구 시작
- **2019년**: Schuit 등 본격 연구

## 언어적 특징

### ASL과의 관계
- **완전히 다른 언어**: ASL과 무관하게 발생
- **현재**: 젊은 세대는 ASL 사용
- **이중언어**: 일부 IUR + ASL 사용

### 특이한 특징

#### 장갑 사용
- 북극 기후로 인해 장갑 착용 상태에서 사용
- 수형 단순화
- 손가락 개별 움직임 제한

#### 공간 사용
- 작은 수화 공간
- 추운 날씨에 팔 움직임 제한
- 실내 소통 최적화

### 음운론
- 제한된 수형 목록
- 장갑 착용 고려한 형태
- 손바닥 방향 활용

### 어휘 특징
- 북극 생활 관련 풍부
- 사냥 관련 용어
- 이누이트 문화 반영

### 독특한 어휘 예시
- 물개 사냥 관련
- 이글루/카약 관련
- 오로라 관련
- 북극 동물 (북극곰, 순록 등)

## 사회언어학적 특징

### 화자 현황
| 유형 | 인원 | 비고 |
|------|------|------|
| 유창한 화자 | ~20명 | 고령층 |
| 부분적 화자 | ~30명 | 중년층 |
| 젊은 세대 | 소수 | ASL 위주 |

### 지역 분포
- 누나부트 (Nunavut)
- 누나빅 (Nunavik, 퀘벡 북부)
- 북서준주 일부

### 언어 전이
- 노년층: IUR 유창
- 중년층: IUR + ASL
- 청년층: ASL 위주, IUR 제한적

## 언어 위기 상태

### 멸종위기 등급
- **UNESCO**: 심각한 위험
- **상태**: 긴급한 문서화 필요

### 위협 요인
1. **ASL 교육 확산**: 청각장애 학교에서 ASL 사용
2. **화자 고령화**: 유창한 화자 대부분 노년층
3. **지리적 분산**: 북극 공동체 고립
4. **세대 간 단절**: 젊은 세대 IUR 미습득

### 특수 상황
- 이누이트 구어도 위협 받음
- 이중 언어 위기 상황
- 원주민 언어 권리 문제

- 학교 교육 도입 논의

## 문화적 맥락

### 이누이트 문화
- 사냥 중심 전통 생활
- 공동체 강조
- 구전 전통

### 청각장애 인식
- 공동체의 일원으로 통합
- 사냥, 가정 활동 참여
- 별도의 농인 문화 없음
