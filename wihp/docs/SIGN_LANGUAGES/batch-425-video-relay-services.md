# Video Relay Services (VRS)
## Telecommunications for the Deaf

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Video-based phone relay |
| **Function** | Deaf-hearing phone calls |
| **Technology** | Video + interpreting |
| **Funding** | Government (USA) |
| **Access** | Free for users |

### Introduction
Video Relay Service (VRS) enables deaf and hard of hearing individuals to make phone calls through a sign language interpreter via video connection. This technology has revolutionized telecommunications access for deaf people.

### How VRS Works
| Step | Process |
|------|---------|
| 1 | Deaf user connects via video |
| 2 | Interpreter appears on screen |
| 3 | User signs to interpreter |
| 4 | Interpreter calls hearing party |
| 5 | Interpreter voices for deaf user |
| 6 | Interpreter signs hearing responses |
