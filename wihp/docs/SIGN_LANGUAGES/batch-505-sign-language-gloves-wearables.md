### Introduction
Sign language gloves and wearable devices aim to translate sign language to text or speech. While receiving media attention and investment, these technologies face significant technical limitations and criticism from the deaf community.

### Technology Types

#### Sensor Gloves
| Component | Function |
|-----------|----------|
| Flex sensors | Finger position |
| Accelerometers | Movement |
| Gyroscopes | Orientation |
| Bluetooth | Data transmission |
