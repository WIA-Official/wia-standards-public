# Batch 146: Henniker Sign Language (헤니커 수어)

## 기본 정보
- **언어명**: Henniker Sign Language
- **약칭**: HenikSL
- **ISO 639-3**: 없음
- **국가**: 미국
- **지역**: Henniker, New Hampshire
- **사용자 수**: 0 (멸종)
- **상태**: 멸종 (19세기)

## 역사적 배경

### 기원
- **발생 시기**: 18세기 후반-19세기 초
- **장소**: New Hampshire, Henniker 마을
- **원인**: 유전성 청각장애

### MVSL과의 유사성
- Marthas Vineyard와 유사한 패턴
- 마을 내 높은 청각장애 비율
- 청인과 청각장애인 모두 수어 사용
- ASL 도입 전 독자적 수어

### 역사적 맥락
- New England 농촌 공동체
- 18-19세기 미국 수어 역사의 일부
- 문서화 없이 멸종

## 언어적 특징

### 정보 부족
- 체계적 기록 없음
- 언어학적 분석 불가능
- 역사적 언급만 존재

### 추정
- 독자 발생 마을 수어
- MVSL, ASL과 다름
- 지역 제스처 기반

## 멸종 과정

### 원인
1. ASL 확산
2. 외부 결혼 증가
3. 청각장애 출생 감소
4. 마을 근대화

### 시기
- 19세기 후반-20세기 초 추정
- 정확한 멸종 시점 미확인

## 연구 현황

### 기록 부재
- 체계적 문서화 없음
- 역사적 언급만
- 복원 불가능

### 교훈
- 문서화의 중요성
- 미국 내 다른 마을 수어 존재 가능성
- MVSL만이 유일한 사례가 아님
