# Deaf Education Methodologies
## Historical and Contemporary Approaches

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Methods of educating deaf students |
| **Debate** | Oral vs. manual, 300+ years |
| **Current** | Shift toward bilingual-bicultural |
| **Key event** | Milan Congress 1880 |
| **Framework** | Language access as right |

### Introduction
The history of deaf education has been marked by debates between oral and manual methods. These debates have profound implications for deaf children's language development, academic achievement, and cultural identity. Understanding this history helps inform current best practices.

### Main Methodologies

#### Oralism
| Aspect | Description |
|--------|-------------|
| Philosophy | Deaf should learn to speak |
| Methods | Speech, lipreading |
| Sign | Prohibited or discouraged |
| Goal | Integration with hearing |
| Peak | Post-Milan to 1970s |

#### Manualism
| Aspect | Description |
|--------|-------------|
| Philosophy | Sign language primary |
| Methods | Sign-based instruction |
| Speech | Not emphasized |
| Goal | Language access first |
| Era | Pre-Milan, revival post-1960s |

#### Total Communication
| Aspect | Description |
|--------|-------------|
| Philosophy | Use all methods |
| Methods | Sign, speech, fingerspelling |
| Era | 1970s-1990s popular |
| Criticism | Often became SimCom |
| Problem | Neither language fully |

#### Bilingual-Bicultural (Bi-Bi)
| Aspect | Description |
|--------|-------------|
| Philosophy | Two languages, two cultures |
| Languages | Sign language + written |
| Culture | Deaf and hearing |
| Goal | Full bilingualism |
| Current | Increasingly supported |

### Milan Congress 1880
| Aspect | Detail |
|--------|--------|
| Event | International educator conference |
| Location | Milan, Italy |
| Decision | Oral method declared superior |
| Impact | Sign banned in schools |
| Duration | 100+ years of oralism |
| Legacy | Language deprivation |

### ICED 2010 Vancouver
| Aspect | Detail |
|--------|--------|
| Event | International Congress |
| Action | Rejected Milan resolutions |
| Statement | Sign languages are full languages |
| Significance | Symbolic rejection |
| Impact | Supports bilingual approach |

### Oral Method Details
| Component | Method |
|-----------|--------|
| Auditory training | Use residual hearing |
| Speech therapy | Articulation |
| Lipreading | Visual speech reception |
| Hearing aids | Amplification |
| Cochlear implants | Modern addition |

### Problems with Oralism
| Problem | Impact |
|---------|--------|
| Language deprivation | Delays development |
| Limited access | Much speech not understood |
| Identity | Deaf identity suppressed |
| Outcomes | Variable, often poor |
| Time | Much time on speech, less academics |

### Manually Coded English Systems
| System | Description |
|--------|-------------|
| SEE (Signed Exact English) | English on hands |
| PSE/Contact Sign | ASL-English mix |
| Cued Speech | Visual phonemes |
| Problem | Not natural language |

### Bilingual Education Benefits
| Benefit | Evidence |
|---------|----------|
| Early language | Strong foundation |
| Cognitive | Bilingual advantages |
| Literacy | Transfer to reading |
| Identity | Healthy deaf identity |
| Research | Increasingly supported |

| Advantage | Challenge |
|-----------|-----------|
| Local school | Often only deaf child |
| Family nearby | Interpreter dependency |

| Inclusive intent | Language access issues |

### Teacher Preparation
| Need | Current Status |
|------|----------------|
| Sign language fluency | Often inadequate |
| Deaf educators | Underrepresented |

### Language Deprivation
| Cause | Consequence |
|-------|-------------|
| No early language | Delayed development |
| Oral-only failure | Cognitive impacts |
| Late sign exposure | May not fully acquire |
| Prevention | Early sign access |

### Research Evidence
