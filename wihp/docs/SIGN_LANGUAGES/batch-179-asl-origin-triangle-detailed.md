# Batch 179: ASL Origin Triangle - ASL 기원 3대 수어

## 개요
ASL 형성에 기여한 3가지 미국 마을 수어
New England "삼각형" 지역

## 1. Martha Vineyard Sign Language

### 기본 정보
- Batch 142에서 상세 문서화
- 가장 유명한 ASL 기원 수어
- 1952년 멸종

---

## 2. Henniker Sign Language

### 기본 정보
- Batch 146에서 문서화
- New Hampshire주
- 우성 유전 청각장애

### 특징
- MVSL과 다른 발생
- 가족 중심 전달

---

## 3. Sandy River Valley Sign Language

### 기본 정보
- **위치**: Maine주, Sandy River Valley
- **시기**: 19세기
- **상태**: 멸종

### 역사
- Martha Vineyard 이주민이 세운 30개 마을
- MVSL 전달 또는 새 언어 발생
- ASL 형성에 기여

### 연구
- Lane, Pillard, French 2000년 연구
- MVSL과의 관계 불확실

---

## ASL 기원 연구

### 3대 기원
1. Old French Sign Language (LSF)
2. MVSL + Henniker + Sandy River Valley
3. 학교 내 자연 발생

### 언어 접촉
- Hartford 농인학교 설립 (1817년)
- LSF + 마을 수어들 접촉
- 크리올화 과정
