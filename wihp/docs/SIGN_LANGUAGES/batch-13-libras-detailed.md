# WIA Talk Sign Language Mapping - Libras (Língua Brasileira de Sinais / Brazilian Sign Language)
## Batch 13: 20 Signs with Full WIA Talk Component Analysis

**Language:** Libras - Língua Brasileira de Sinais (Brazilian Sign Language)
**Region:** Brazil (all states)
**Estimated Users:** 5-10 million deaf/hard-of-hearing; ~2.7 million with severe hearing loss
**Language Family:** French Sign Language (LSF) family - considered creole of 18th century LSF
**Recognition:** Official language since April 24, 2002 (Law No. 10.436)

---

## Libras Characteristics

### Historical Background
- 1855: Ernest Huet (deaf French teacher) arrived in Brazil
- Huet supported by Emperor Dom Pedro II
- Founded first public school for deaf in Rio de Janeiro
- Huet brought LSF methodology from Paris
- 1880: Milan Congress → oralism imposed, sign language banned
- 1911: Oralism enforced at INES (Instituto Nacional de Educação de Surdos)
- 1977: FENEIDA founded (National Federation)
- 1987: FENEIDA dissolved, new advocacy organization formed
- 2002: **Law No. 10.436 recognized Libras as official language** (April 24)
- 2005: Decree No. 5.626 regulated Libras implementation
- 2010: Translator/Interpreter profession regulated
- **April 24:** National Day of Brazilian Sign Language

### Ernest Huet - Founder
- French noble family
- Multilingual (Portuguese, German, French, Spanish)
- Became deaf at age 12 from measles
- Studied at National Institute for the Deaf in Paris
- 1855: Moved to Brazil with wife
- Pivotal role in Libras development

### Key Features
- **5 phonological parameters** (like ASL):
  1. Handshape (configuração de mão) - 61-111 distinct shapes documented
  2. Location (ponto de articulação) - 35 body points identified
  3. Movement (movimento)
  4. Palm orientation (orientação)
  5. Non-manual markers (expressão facial e corporal)
- **Independent of Portuguese** - Own grammar and syntax
- **4,000+ signs** in lexicon
- **Two-handed signs** common
- **Regional variations:** Signs differ between Rio, São Paulo, Salvador
- **Spatial-visual language** - Uses 3D space

### Organizations
- **INES** - Instituto Nacional de Educação de Surdos
- **FENEIS** - Federação Nacional de Educação e Integração dos Surdos
- **Hand Talk** - Popular translation app (Hugo and Maya avatars)

### Cultural Features
- **Sinal de batismo:** Name signs given through baptism ceremony
- **TV visibility:** Three broadcasts with Libras interpreters simultaneously!
- **Strong community advocacy:** Grassroots movements preserved language during oralism
- Deaf cultural events: poetry, storytelling, artistic performances in sign

### Writing Systems
- **SignWriting:** 600+ symbols for transcription
- **SER-Libras (2022):** Developed at Federal University of Santa Catarina
- **Escrita SEL:** Sistema de Escrita para Línguas de Sinais

### Resources
- **Hand Talk App** - Award-winning translator (UN best social app)
- **Acesso Brasil** - Online visual dictionary
- **Novo Deit-Libras** - Encyclopedia dictionary (Capovilla et al.)

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION C: Basic Phrases (4 signs)

### 17. EU TE AMO (I Love You)

**Libras Concept:** Love expression

**Detailed Motion:**
- EU (I): Point to self
- AMAR (Love): Cross arms over heart, hug motion
- VOCÊ (You): Point to person
- Loving expression essential

**WIA Talk Code (AMAR):** `HS02-LC07-MV26-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands crossed over chest |
| LC | 07 | Chest/heart area |
| MV | 26 | Cross arms, hug motion |
| OR | 05 | Arms across body |
| FE | 01 | Loving expression |

---

### 18. QUAL SEU NOME? (What is your name?)

**Libras Concept:** Name question

**Detailed Motion:**
- NOME: Tap chest or specific sign
- QUAL/SEU: Question gesture
- Question expression

**WIA Talk Code:** `HS04-LC07-MV15-OR05-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 04 | Two fingers |
| LC | 07 | Chest area |
| MV | 15 | Tapping motion |
| OR | 05 | Fingers toward body |
| FE | 06 | Question expression |

**Note:** In Libras, verbs are written in infinitive and in ALL CAPS: VOCÊ GOSTAR CARRO? = Você gosta do carro?

## Summary: Libras WIA Talk Code Inventory

### Hand Shapes Used (HS)
| Code | Shape | Signs Using |
|------|-------|-------------|
| HS01 | Fist | Yes |
| HS02 | Flat hand | Hello, Thank you, Sorry, I love you |
| HS03 | 5 fingers spread | Number 10 |
| HS04 | V-shape | Number 2, Name |
| HS06 | Thumb | Numbers 6, How are you |
| HS07 | Index finger | Numbers 1, No |
| HS10 | Four fingers | Number 4 |
| HS45-47 | Libras 7-9 shapes | Numbers 7-9 |
| HS48 | O-shape | Number 0 |
| HS49 | Index+middle+ring | Number 3 |
| HS50 | Bent-knuckle V | Number 5 (unique!) |

### Unique Libras Features
- **Number 5:** Bent knuckles on extended fingers
- **Number 0:** O-shape (hole)
- **Regional variations:** Signs differ by city
- **Verb writing:** Infinitive form in ALL CAPS

### Related Sign Languages
- **LSF** - French Sign Language (ancestor)
- **ASL** - American Sign Language (cousin, both from LSF)
- Part of Western Sign Language family
