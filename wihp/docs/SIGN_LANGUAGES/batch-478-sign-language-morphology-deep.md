### Introduction
Sign language morphology is remarkably rich, with unique features arising from the visual-spatial modality. Unlike spoken languages that primarily use sequential affixation, sign languages exploit simultaneity, allowing multiple morphemes to be expressed at the same time.

### Types of Morphological Processes

#### Simultaneous
| Process | Description |
|---------|-------------|
| Multiple at once | Several morphemes together |
| Spatial | Use of 3D space |
| Non-manual | Facial, body |

### Verb Morphology

#### Agreement Verbs
| Feature | Description |
|---------|-------------|
| Subject agreement | Starting point |
| Object agreement | Ending point |
| Spatial | Use of loci |
| Examples | GIVE, TELL, ASK |

#### Plain Verbs
| Feature | Description |
|---------|-------------|
| No agreement | Fixed form |
| Examples | KNOW, LIKE, UNDERSTAND |
| Separate pronouns | Needed for reference |

#### Spatial Verbs
| Feature | Description |
|---------|-------------|
| Location | Actual place reference |
| Movement | Path through space |
| Examples | PUT, MOVE, WALK |

| Relationship | Pattern |
|--------------|---------|
| Related | Same base |
