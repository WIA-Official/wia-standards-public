# Batch 111: LSP (Lengua de Señas Peruana / Peruvian Sign Language)

## 기본 정보
- **언어 코드**: LSP
- **ISO 639-3**: prl
- **정식 명칭**: Lengua de Señas Peruana
- **영어 명칭**: Peruvian Sign Language
- **사용 국가**: 페루
- **사용자 수**: 10,000명+ (센서스) ~ 532,000명 (청각장애인 전체)
- **법적 인정**: 2010년 공식 인정!
- **언어 상태**: Ethnologue - 안정적(Stable), 제도화(Institutional)

## 역사적 배경

### 기원
- 페루 농인 커뮤니티에서 자연 발생
- 지역 문화와 소통 필요에서 유기적 발전
- 스페인어와는 별개의 독립 언어

### Inmaculada 수어 (전신)
- **1939년**: CEBE La Inmaculada de Barranco 농학교 개교
- **~1960년**: LSP가 국가 표준으로 확립되기 전까지 사용
- ASL 영향: LSP의 절반 수준
- 지문자 체계: LSP와 상당히 다름
- 현재: 리마 고령 세대 농인이 사용

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1939 | Inmaculada 농학교 개교 |
| ~1960 | LSP가 국가 표준으로 확립 |
| - | ANSP (페루 농인 협회) 설립 |
| **2010** | **법적 공식 인정!** |
| 2019 | LSP 알파벳 인식 연구 (Lazo) |
| 2020 | 지문자 인식 연구 (Mejía Gamarra, Berrú-Novoa, Nureña-Jara) |
| 2022 | PeruSIL 데이터셋 프레임워크 발표 (LREC) |
| 2024 | LSP→케추아어 번역 LSTM 모델 연구 |

## 언어적 특성

### 독립 언어
- 스페인어의 수어 버전이 **아님**
- 독자적 문법, 어휘, 문화적 뉘앙스
- 공간 문법 및 표정으로 의미 전달

### 언어 계통
- ASL 영향 있음
- 스페인 LSE 영향 일부
- 남미 수어 네트워크 (페루, 볼리비아, 에콰도르, 콜롬비아)

### 변이
| 변이 유형 | 설명 |
|----------|------|
| **지역** | 리마 변종이 가장 권위 있고 이해하기 쉬움 |
| **세대** | 젊은 세대 vs 고령 세대 차이 |
| **종교** | 종교 집단별 변이 존재 |

### ASL의 위협
- ASL이 LSP를 파괴할 수 있다는 우려
- 사전 제작 등으로 LSP 보호 노력
- 커뮤니티가 더 많은 LSP 자료 희망

## 교육 현황

### 정부 통합 교육 실패
- 농인 학생을 일반 교육에 통합 시도
- 실제 성공 없음
- 농인 학생 교육 수준 낮음

### 대안: 사립학교 및 사회 모임
- 농인 사회 모임 → LSP 강화
- 사립 농학교 → LSP 유지

### 수어 습득 시기
- 대부분 농인: 9세 이후에야 수어 학습
- 조기 습득 부족 문제

## 케추아어와의 연결

### 2024년 연구
- **목표**: LSP → 케추아어 번역
- **기술**: LSTM 딥러닝
- **의의**: 페루 토착어와 수어 연결

### 페루 청각장애 현황
- 약 532,000명 영구 청력 손실
- 다양한 소통 방식: 음성, 제스처, 수어
- 수어 학습 소수 - 시간/노력 필요

## 특이사항

### 언어 생존력
- LSP는 사멸하지 않을 것
- 페루 농인들이 적극적으로 보호/사용
- 사전 제작, 자료 개발 진행 중

### 리마 변종
- 가장 권위 있고 이해하기 쉬움
- 자원과 기회 집중

### 세대 간 전승
- 가족 내 청각장애 대물림 사례 다수
- 수어가 세대 간 전승
- 일부 가정: 청력 손실 정도에 따라 수어보다 구어 사용
