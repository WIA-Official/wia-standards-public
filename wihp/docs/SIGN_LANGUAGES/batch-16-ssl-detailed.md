# WIA Talk Sign Language Mapping - SSL (Svenskt Teckenspråk / Swedish Sign Language)
## Batch 16: 20 Signs with Full WIA Talk Component Analysis

**Language:** SSL/STS - Svenskt Teckenspråk (Swedish Sign Language)
**Region:** Sweden
**Estimated Users:** 10,000-13,000 native; ~30,000 total speakers
**Language Family:** Independent origin - influenced Finnish and Portuguese Sign Languages!
**Recognition:** Official national language since 1981; Language Law 2009

---

## SSL Characteristics

### Historical Background
- **1800:** SSL first came into use - did NOT stem from any other language!
- **1809:** First deaf school **Manillaskolan** founded by **Pär Aron Borg** in Stockholm
- Borg created the original Swedish manual alphabet
- **1916:** First illustrated dictionary by O. Österberg
- **1979:** Sweden recognized deaf people as BILINGUAL - first in world!
- **1981:** Swedish Parliament recognized SSL as national language
- **2009:** SSL included in Swedish Language Law (Språklag)
- **2024:** Svenskt Teckenspråkslexikon has 24,500 signs (21,734 published)

### Pioneer Status
- **Sweden was FIRST country in world** to legally recognize sign language (1981)
- **First to recognize deaf people as bilingual** (1979)
- Policy: "Deaf have to be fluent in visual/gestural language AND Swedish"
- Model for other countries' recognition efforts

### Language Family & Relationships
- **Independent origin** - self-created language!
- **SSL → Finnish Sign Language (FSL)** (influenced)
- **SSL → Portuguese Sign Language** (influenced)
- **Eastern Nordic group:** SSL, FSL, Finland-Swedish Sign Language (FinSSL)
- **Western Nordic group:** Danish (DTS), Norwegian (NTS), Icelandic (ÍTM)
- Nordic sign languages have shared sign for "Norden/Nordic"

### Key Features
- **Spatial syntax** - uses 3D space
- **Iconicity** - signs visually depict referents
- **Simultaneous morphology** - multiple grammatical elements at once
- **Manual parameters:**
  1. Handshape (handform)
  2. Location (placering)
  3. Movement (rörelse)
  4. Orientation (orientering)
- **Non-manual markers** essential
- **Independent of spoken Swedish** - own grammar and lexicon

### Organizations & Resources
- **SDR** - Sveriges Dövas Riksförbund (Swedish National Association of the Deaf)
- **Svenskt Teckenspråkslexikon** - 24,500 signs (Stockholm University)
  - 4 million visits per year!
  - Website, app, mobile platforms
- **STS-korpus** - Swedish Sign Language Corpus
- **ISOF** - Institutet för språk och folkminnen (language cultivation)
- **Manillaskolan** - historic deaf school

### Cultural Features
- **Fika** - beloved Swedish coffee tradition has its own sign!
- **Bilingual-bicultural education** policy
- SSL as language of instruction, written Swedish as second language
- Strong Deaf cultural identity

### Related Languages
- **Finland-Swedish Sign Language (FinSSL)** - moribund, ~300 speakers
- Founded 1846 by Carl Oscar Malm (deaf himself)
- Borgå school closed 1993 - language endangered

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |
