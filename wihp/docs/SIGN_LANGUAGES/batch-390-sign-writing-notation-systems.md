### Introduction
Sign language writing systems are methods for representing sign languages in written form. Unlike spoken languages which have long writing traditions, sign language writing is relatively new and faces unique challenges in representing three-dimensional, simultaneous movement in two-dimensional text.

Several systems have been developed, each with different approaches, applications, and levels of adoption.

### Why Writing Sign Languages is Difficult
| Challenge | Explanation |
|-----------|-------------|
| 3D to 2D | Space compressed |
| Simultaneity | Multiple features at once |
| Movement | Hard to represent |
| Non-manual | Face, body included |
| No tradition | No historical system |
