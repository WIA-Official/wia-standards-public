# Batch 151: Australian Aboriginal Sign Languages (호주 원주민 수어)

## 개요

호주 원주민들은 수천 년 전부터 다양한 수어를 사용해왔습니다.
이들은 청각장애와 무관하게 발달한 "대체 수어" (alternate sign languages)입니다.

## 역사적 배경

### 기원
- **발생 시기**: 수천 년 전 (정확한 시기 미확인)
- **목적**: 애도 기간 침묵, 사냥, 의식, 원거리 의사소통
- **특징**: 구어 대체/보조 수단

### 사용 맥락
1. **애도 침묵 (Mourning Silence)**: 사망 시 여성들 장기간 침묵
2. **사냥**: 동물 놀라지 않게 무음 소통
3. **의례**: 신성한 침묵 기간
4. **원거리**: 소리가 닿지 않는 거리에서 소통
5. **비밀 유지**: 타 부족에게 내용 숨김

## 주요 수어들

### 1. Warlpiri Sign Language

#### 기본 정보
- **민족**: Warlpiri 사람들
- **지역**: Northern Territory, 중앙 호주
- **사용자**: 수천 명 (구어 화자 중)
- **연구**: 가장 잘 문서화된 호주 원주민 수어

#### 언어적 특징
- 수백 개 어휘
- Warlpiri 구어 문법 따름
- 완전한 대화 가능
- 복잡한 추상 개념 표현 가능

#### 애도 침묵
- 여성: 사망 시 1-2년 침묵
- 이 기간 수어만 사용
- 남성도 특정 상황에서 사용

### 2. Yolngu Sign Language

#### 기본 정보
- **민족**: Yolngu 사람들
- **지역**: Arnhem Land, Northern Territory
- **특징**: Yolngu Matha와 함께 사용

### 3. Central Australian Aboriginal Sign Languages

#### 개요
- 여러 부족 수어 총칭
- Pitjantjatjara, Arrernte 등
- 각 부족별 변형

### 4. 기타 지역 수어
- Western Desert 수어
- Cape York 수어
- 다수 미문서화

## 언어적 특징

### 대체 수어의 특성
| 특징 | 호주 원주민 수어 | 청각장애 수어 |
|------|------------------|---------------|
| 사용자 | 청인 (주로) | 청각장애인 |
| 목적 | 침묵 필요 시 | 상시 의사소통 |
| 구어 관계 | 구어 문법 따름 | 독립 문법 |
| 동시 사용 | 구어와 함께 가능 | 독립적 |

### 어휘
- 일상 용어
- 친족 관계 (매우 발달)
- 동물, 식물
- 드림타임 관련

### 문법
- 구어 어순 따름
- 구어 형태론 반영
- 일부 독자적 요소

## 현재 상황

### 사용 현황
- 전통 공동체: 여전히 활발
- 도시 원주민: 감소
- 젊은 세대: 감소 추세

### 위협 요인
1. 전통 문화 약화
2. 영어 확산
3. 도시화
4. 세대 간 전달 감소

### 보존 노력
- 원주민 공동체 주도 기록
- 대학 연구 프로젝트
- 교육 자료 개발

## Auslan과의 관계

### 구별
- **Auslan**: 청각장애인 수어, 영국 수어 계통
- **원주민 수어**: 청인 대체 수어, 독자 발생

### 청각장애 원주민
- Auslan + 원주민 수어 이중언어 가능
- 문화적 정체성 복합
