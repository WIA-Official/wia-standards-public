# African Sign Language Overview

### Overview
| Attribute | Detail |
|-----------|--------|
| **Region** | Sub-Saharan Africa (54 countries) |
| **Estimated Languages** | 50+ distinct sign languages |
| **Signers** | Millions (exact numbers unknown) |
| **Documentation** | Limited for most |
| **Status** | Many endangered or undocumented |

### Introduction
Sub-Saharan Africa is home to dozens of distinct sign languages, reflecting the continent's linguistic diversity. However, African sign languages remain among the world's least documented, with many still awaiting basic linguistic description.

This overview surveys the sign language situation across the region, highlighting documented languages, village sign languages, and the challenges facing African deaf communities.
