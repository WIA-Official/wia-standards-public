# Batch 144: Hawaii Sign Language (하와이 수어)

## 기본 정보
- **언어명**: Hawaii Sign Language / Hawaii Pidgin Sign Language
- **약칭**: HSL / HPSL
- **ISO 639-3**: hps
- **국가**: 미국 (하와이)
- **사용자 수**: 소수 (고령층, 추정 수십 명)
- **언어 유형**: 피진 수어 / 지역 수어
- **계통**: 독자 발전 + ASL 영향
- **상태**: 거의 멸종

## 역사적 배경

### 기원
- **발생 시기**: 19세기 후반
- **장소**: 하와이 농장 (설탕, 파인애플)
- **배경**: 다문화 노동자 공동체

### 역사적 맥락
- **이민 노동자**: 일본, 중국, 한국, 필리핀, 포르투갈 등
- **언어 다양성**: 다양한 구어 배경
- **피진 영어**: Hawaii Creole English 발달
- **수어**: 청각장애 노동자 간 자연 발생

### 발전 과정
1. **1800년대 후반**: 농장에서 자연 발생
2. **1900년대 초**: 하와이 청각장애 공동체 형성
3. **1900년대 중반**: ASL 영향 증가
4. **현재**: HSL 거의 사라짐, ASL 대체

### ASL 도입
- **1914년**: 하와이 청각장애 학교 ASL 도입
- **이후**: 점진적으로 HSL 대체
- **현재**: 젊은 세대는 ASL만 사용

## 언어적 특징

### 혼합적 특성
- **토착 요소**: 하와이 제스처
- **아시아 영향**: 일본, 중국 수어 요소
- **ASL 영향**: 점점 증가

### 음운론
- 독자적 수형 일부
- ASL과 다른 패턴
- 연구 제한적

### 어휘 특징
- 하와이 문화 관련
- 농장 생활 용어
- 태평양 도서 관련

### 독특한 어휘 예시
- 하와이 음식 (포이, 칼루아 피그 등)
- 훌라 관련
- 서핑 관련
- 화산, 해변 관련

## 사회언어학적 특징

### 화자 현황
- 대부분 70대 이상
- 현재 ASL도 사용
- 정확한 숫자 미확인

### 언어 사용
- 동년배 간 회상 대화
- 일상: ASL 사용
- 점점 잊혀짐

### 지역 분포
- 오아후 섬 중심
- 마우이, 빅 아일랜드 일부
- 호놀룰루 농인 공동체

## 언어 위기 상태

### 멸종위기 등급
- **상태**: 거의 멸종
- **EGIDS**: 8b

### 위협 요인
1. ASL로 완전 대체
2. 화자 고령화
3. 체계적 문서화 부재
4. 세대 간 전달 중단
