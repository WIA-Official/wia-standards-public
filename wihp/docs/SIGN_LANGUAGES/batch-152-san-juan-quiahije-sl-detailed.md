# Batch 152: San Juan Quiahije Sign Language (산 후안 키아히헤 수어)

## 기본 정보
- **언어명**: San Juan Quiahije Chatino Sign Language
- **약칭**: SJQCSL
- **ISO 639-3**: 없음
- **국가**: 멕시코
- **지역**: San Juan Quiahije, Oaxaca 주
- **사용자 수**: 소규모 (미확인)
- **언어 유형**: 마을 수어
- **계통**: 독립 언어
- **상태**: 위험

## 역사적 배경

### 기원
- **발생 시기**: 여러 세대 전
- **장소**: San Juan Quiahije, Oaxaca 주 산악 지역
- **원인**: 마을 내 유전성 청각장애

### 지리적 배경
- **위치**: 멕시코 남부, Oaxaca 주
- **민족**: Chatino 원주민
- **언어**: Chatino어 (성조 언어) 사용 지역

### 문화적 맥락
- Chatino 원주민 공동체
- 전통 농업 사회
- Mesoamerican 문화권

## 언어적 특징

### 독자성
- **Mexican Sign Language (LSM)와 무관**: 독자 발생
- **Yucatec Maya SL과 유사한 패턴**: 멕시코 내 또 다른 마을 수어

### 연구 필요
- 체계적 언어학적 분석 없음
- 기초 정보만 존재

### Chatino어와의 관계
- Chatino어: 복잡한 성조 체계
- 수어로의 성조 전환 연구 가치
- 언어 접촉 연구 대상

## 사회언어학적 특징
