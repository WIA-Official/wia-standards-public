# Batch 145: Sri Lankan Village Sign Languages (스리랑카 마을 수어들)

## 개요

스리랑카에는 국가 수어 (SLSL) 외에 여러 마을 수어가 존재합니다.
이 문서는 알려진 스리랑카 마을 수어들을 기록합니다.

## 1. Wellaweediya Sign Language

### 기본 정보
- **위치**: Wellaweediya 마을, 스리랑카
- **사용자 수**: 미확인 (소규모)
- **상태**: 연구 필요

### 배경
- 마을 내 유전성 청각장애
- Sri Lankan Sign Language와 다른 독자 수어
- 제한적 정보만 존재

### 개요
- 타밀 지역 마을 수어 존재 가능성

## 스리랑카 수어 생태계

### 국가 수어
- **Sri Lankan Sign Language (SLSL)**
- 학교 기반
- 전국적 사용

### 마을 수어
- 존재 확인되나 미문서화
- 긴급 조사 필요
- SLSL 영향으로 쇠퇴 가능성

### 연구 현황
- 체계적 조사 없음
- 개별 보고만 존재
- 현지 연구자 필요

## 연구 필요성

### 긴급 과제
1. 마을 수어 존재 확인 조사
2. 화자 수, 위치 파악
3. 기초 어휘 수집
4. SLSL과의 관계 분석

### 권장 조치
- 현지 대학 협력
- 농인 단체 협력
- 체계적 현장 조사
