# Batch 148: Rennellese Sign Language (렌넬 수어)

## 기본 정보
- **언어명**: Rennellese Sign Language
- **약칭**: RSL (Rennell)
- **ISO 639-3**: rsi
- **국가**: 솔로몬 제도
- **지역**: Rennell Island (Mugaba)
- **사용자 수**: 불명 (소규모)
- **언어 유형**: 마을 수어
- **계통**: 독립 언어
- **상태**: 위험

## 역사적 배경

### 기원
- **발생 시기**: 알 수 없음 (여러 세대 전)
- **장소**: Rennell Island, 솔로몬 제도
- **원인**: 유전성 청각장애

### 지리적 배경
- **위치**: 남태평양, 솔로몬 제도 남부
- **거리**: 수도 Honiara에서 남쪽 200km
- **특징**: 폴리네시아 영향권, 격리된 섬

### 인구
- Rennell Island 전체 인구: 약 3,000명
- 청각장애 비율: 연구 필요
- 수어 화자: 소규모

## 언어적 특징

### 독자성
- 완전히 독자적으로 발생
- 다른 수어와 관계 없음
- 폴리네시아 제스처 영향 가능

### 연구 부족
- 체계적 언어학적 분석 없음
- 기초 정보만 존재
- 긴급 문서화 필요

### 어휘
- 섬 생활 관련 추정
- 어업, 농업 용어
- 전통 문화 관련

## 사회언어학적 특징

### 화자 현황
- 정확한 숫자 미확인
- 청각장애 공동체 존재
- 청인 수화자 존재 추정

### 사회적 맥락
- 폴리네시아 문화
- 소규모 섬 공동체
- 전통 생활 방식 유지

## 위협 요인

### 언어 상태
- **상태**: 위험
- 체계적 평가 필요

### 잠재적 위협
1. 소규모 화자 집단
2. 현대화, 외부 접촉
3. 문서화 부재
4. 교육 정책

## 연구 및 문서화

### 현황
- 기초 정보만 존재
- Ethnologue 등록
- 체계적 연구 없음

### 필요 조치
1. 현장 조사
2. 화자 파악
3. 어휘, 문법 기록
4. 비디오 코퍼스

## 태평양 도서 수어

### 맥락
- 태평양 도서 지역 수어 연구 부족
- 여러 섬에 미문서화 수어 존재 가능
- 솔로몬 제도, 바누아투 등

### 연구 필요성
- 태평양 수어 조사 프로젝트 필요
- 섬별 독자 수어 가능성
