# Batch 175: Historical/Extinct Sign Languages

## 개요
역사 속에서 멸종되었거나 쇠퇴한 수어들
언어 발생과 변천 연구에 중요

## 1. Old Kentish Sign Language

### 기본 정보
- **위치**: England, Kent 지역
- **시기**: 17세기
- **상태**: 멸종

### 역사적 기록
- Samuel Pepys 일기 (1666년)
- 런던 대화재에 대해 수어로 대화하는 장면 기록
- Sir George Downing의 청각장애 하인

### BSL에 미친 영향
- BSL 발달에 기여 추정
- MVSL에도 영향 (Kent 이민)

## 2. Old French Sign Language

### 기본 정보
- **위치**: 파리, 프랑스
- **시기**: 18세기
- **상태**: 현대 LSF로 진화

### 역사적 중요성
- Abbe de Epee 이전 파리 청인 공동체 수어
- 200명 농인 사용
- 체계적 교육 전 자연 발생

### 영향
- 현대 LSF의 기원
- ASL에 영향
- 세계 수어 계통 기원

## 3. Old Bangkok Sign Language

### 기본 정보
- **위치**: Bangkok, 태국
- **상태**: 거의 멸종 (65세 이상만)
- **화자**: 약 19명 추정

### 특징
- Thai SL 도입 전 수어
- Chiangmai SL과 65% 동족어
- 베트남/라오스 수어와 관련
