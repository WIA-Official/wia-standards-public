### Introduction
Like spoken languages, sign languages vary according to social context, with formal and informal registers differing in vocabulary, grammar, and style. Understanding register is essential for appropriate communication in different settings.

### What Is Register?
| Aspect | Description |
|--------|-------------|
| Definition | Language variation by context |
| Factors | Setting, audience, purpose |
| Range | Formal to casual |
| Choice | Speaker/signer selects |

### Formal Register
| Feature | Description |
|---------|-------------|
