# Jamaican Country Sign Language (JCSL)
## Konchri Sain / Country Sign

### Overview
| Attribute | Detail |
|-----------|--------|
| **Native Name** | Konchri Sain (KS) |
| **Other Names** | Country Sign, Jamaican Country Sign Language |
| **ISO 639-3** | jcs |
| **Country** | Jamaica |
| **Location** | St. Elizabeth Parish (southwestern Jamaica) |
| **Language Type** | Village sign language |
| **Status** | Moribund (8a on Ethnologue scale) |

### Demographics
| Attribute | Detail |
|-----------|--------|
| **Speakers** | ~40 remaining (2025 estimate) |
| **Jamaica deaf pop** | ~7,500 total |
| **Deaf children** | ~2,500 |
| **Usage pattern** | Primarily elderly monolingual deaf |

### Language Context
- Indigenous village sign language
- Used by both deaf AND hearing residents
- Historically used by all persons in the area
- Now only used with monolingual KS speakers
- JSL (Jamaican Sign Language) dominant elsewhere

### Endangerment Timeline
- **Pre-1975**: All persons in area used KS
- **1975**: American Mennonite missionaries introduced formal education
- **1975+**: Signed English and ASL introduced
- **1985**: KS used primarily by elderly
- **2005**: Maranatha School for the Deaf established (uses JSL)
- **2007**: Projected extinction in 20-30 years
- **2009**: Adult KS signers still present
- **2011**: Sociolinguistic survey documented status

### Threats to Language
1. **Formal education**: Schools use JSL/ASL, discourage KS
2. **ASL prestige**: American missionaries brought higher-status language
3. **Signed English**: School-imposed system
4. **JSL dominance**: 75% ASL-based language
5. **Institutional suppression**: KS actively discouraged in schools
6. **Generational shift**: Youth use JSL

### Community Attitudes
- Country Sign users see their language as "beautiful"
- Positive attitudes toward unique language and community
- Pride in distinctive heritage
- Transmission continues informally where possible

### Comparison with JSL

| Feature | Konchri Sain (KS) | Jamaican Sign Language (JSL) |
|---------|-------------------|------------------------------|
| Origin | Indigenous, village | ASL-based (75%) |
| Users | ~40 elderly | ~7,500+ |
| Status | Moribund | National language |
| Location | St. Elizabeth Parish | All Jamaica |
| Education | Suppressed | Used in schools |
