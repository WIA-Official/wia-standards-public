# Batch 141: Plains Indian Sign Language (평원 인디언 수어)

## 기본 정보
- **언어명**: Plains Indian Sign Language
- **별칭**: Hand Talk, Plains Sign Talk
- **약칭**: PISL
- **ISO 639-3**: psd
- **국가**: 미국, 캐나다
- **지역**: 북미 대평원 (Great Plains)
- **사용자 수**: 매우 적음 (고령층, 추정 수십 명)
- **언어 유형**: 원주민 수어 / 공용어 (lingua franca)
- **계통**: 독자 발전
- **상태**: 거의 멸종

## 역사적 배경

### 기원
- **발생 시기**: 수백 년 전 (유럽인 도착 이전)
- **장소**: 북미 대평원
- **목적**: 다른 언어를 사용하는 부족 간 의사소통

### 역사적 역할
- **공용어**: 30개 이상 부족 간 의사소통
- **사용 범위**: 록키산맥 ~ 미시시피강
- **전성기**: 19세기, 수만 명 사용

### 사용 부족들
- Crow, Cheyenne, Arapaho
- Kiowa, Comanche, Apache
- Sioux (Lakota, Dakota)
- Blackfoot, Assiniboine
- 기타 30+ 부족

### 쇠퇴 원인
1. **인디언 보호구역 정책**: 부족 간 접촉 감소
2. **기숙학교 정책**: 원주민 언어 금지
3. **영어 교육**: 공용어 필요성 감소
4. **인구 감소**: 전염병, 전쟁

## 언어적 특징

### 유형적 특징
- **보조 수어**: 구어 보조 수단으로도 사용
- **청각장애와 무관**: 주로 청인 간 사용
- **청각장애인도 사용**: 공동체 내 청각장애인 포함

### 음운론
- 수형 체계 독자적
- ASL과 다름
- 도상성 높음

### 어순
- **SOV**: 주어-목적어-동사
- 유연성 있음

### 어휘 특징
- 대평원 생활 관련
- 사냥, 전쟁 관련 풍부
- 부족명, 지명 표현
- 자연 현상 관련

### 독특한 어휘 예시
- 버팔로 사냥 관련
- 전쟁 관련 (쿠 데타 등)
- 계절, 날씨 관련
- 부족 이름 수화

## 사회언어학적 특징

### 역사적 화자 수
- **19세기 전성기**: 수만 명
- **20세기 초**: 수천 명
- **현재**: 수십 명 (추정)

### 현재 화자
- 대부분 70대 이상
- 주로 의례적 사용
- 일상 사용 거의 없음

### 사용 환경
- 전통 의례
- 스토리텔링
- 부족 행사
- 학술 기록

## 언어 위기 상태

### 멸종위기 등급
- **UNESCO**: 거의 멸종
- **Ethnologue**: 8b (거의 멸종)

### 위협 요인
1. 화자 고령화
2. 세대 간 전달 중단
3. 일상 사용 기회 없음
4. ASL 대체

### 복원 노력
- 일부 부족에서 교육 시도
- 학술 기록 프로젝트
- 비디오 아카이브

## 연구 및 문서화

### 역사적 연구
- **1880년**: Garrick Mallery 최초 체계적 기록
- **1930년대**: 영화 기록
- **1960년대**: 학술 연구

### 주요 문헌
1. Mallery, G. (1881). "Sign Language Among North American Indians"
2. Clark, W.P. (1885). "The Indian Sign Language"
3. Tomkins, W. (1926). "Universal Indian Sign Language"
4. Davis, J. (2010). "Hand Talk: Sign Language Among American Indian Nations"

### 현대 문서화
- University of Manitoba 프로젝트
- Smithsonian Institution 아카이브
- 부족별 보존 프로그램

### 아카이브
- Library of Congress
- National Anthropological Archives
- 각 부족 문화센터
