# WIA Talk Sign Language Mapping - LGP (Língua Gestual Portuguesa / Portuguese Sign Language)
## Batch 23: 20 Signs with Full WIA Talk Component Analysis

**Language:** LGP - Língua Gestual Portuguesa (Portuguese Sign Language)
**Region:** Portugal (mainland, Azores, Madeira), Cape Verde influence
**Estimated Users:** 30,000-60,000 (150,000 deaf individuals total)
**Language Family:** Influenced by Swedish Sign Language
**Recognition:** Constitutional recognition September 20, 1997

---

## LGP Characteristics

### Historical Background
- **15th century:** Deaf seen as "possessed by demons" (religious stigma)
- **1823:** First school for deaf opened in Lisbon - formal LGP development began
- **1880:** Milan Conference - oralism wave began
- **1893:** Milan Congress prohibited use of LGP in schools
- **1958:** Associação Portuguesa de Surdos founded (first deaf organization)
- **1960:** William Stokoe pioneered LGP grammar research
- **September 20, 1997:** Constitutional recognition!
- **2004:** Attempt to make LGP co-official language FAILED
- **November 15:** National Day of Portuguese Sign Language

### Swedish Sign Language Influence
- **Pär Aron Borg** (Swedish educator) established school in Lisbon
- Significant influence from Swedish Sign Language
- Same Pär Aron Borg who influenced Swedish deaf education!
- LGP is **basis for Cape Verdean Sign Language**

### Key Features
- Visual-spatial grammar independent of spoken Portuguese
- Own syntax and grammar - NOT signed Portuguese
- Handshape, location, movement, non-manual markers
- Distinct from spoken Portuguese entirely

### Organizations & Resources
- **Associação Portuguesa de Surdos** (1958) - first organization
- **FPAS** (Federação Portuguesa das Associações de Surdos) - leading organization
- **European Union of the Deaf** - supports interpretation
- **~100 professional interpreters** nationwide
- SpreadTheSign - LGP dictionary

### Regional Coverage
- Mainland Portugal
- Azores (autonomous region)
- Madeira (autonomous region)
- Influenced Cape Verdean Sign Language

### Challenges
- 2004 attempt for co-official status failed
- Only ~100 interpreters for 60,000 users
- Many deaf students must pay for own interpreters
- Implementation varies by region

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |
