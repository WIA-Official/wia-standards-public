# WIA Talk Sign Language Mapping - LSF (French Sign Language / Langue des Signes Française)
## Batch 06: 20 Signs with Full WIA Talk Component Analysis

**Language:** LSF - Langue des Signes Française (French Sign Language)
**Region:** France, French-speaking Switzerland
**Estimated Users:** 100,000 native signers
**Language Family:** French Sign Language family (ancestor of many sign languages)
**Recognition:** 2005 fully recognized as language in France

---

## LSF Characteristics

### Historical Significance - "Mother of Sign Languages"
LSF is one of the oldest documented sign languages, with profound influence on:
- **ASL** (American Sign Language) - via Laurent Clerc
- **LSQ** (Quebec Sign Language)
- **NGT** (Dutch Sign Language)
- **VGT** (Flemish Sign Language)
- **LSFB** (Belgian-French Sign Language)
- **ISL** (Irish Sign Language)
- **LSB/Libras** (Brazilian Sign Language)
- **RSL** (Russian Sign Language)

### Key Historical Events
- 1755: Abbé Charles-Michel de l'Épée founded first free deaf school in Paris
- 1760s: l'Épée documented "Old French Sign Language" from deaf twin sisters
- 1817: Laurent Clerc brought LSF to America → became ASL
- 1880: Milan Conference banned sign language in education (dark period)
- 1991: Fabius Law authorized LSF in deaf education
- 2005: LSF officially recognized as a language

### Key Features
- **Facial expressions**: Crucial for grammar (eyebrows for questions)
- **Spatial grammar**: Uses 3D space for time and relationships
- **Word order**: Time + Place + Subject + Object + Verb
- **One-handed manual alphabet**: Similar to ASL but with variations
- **Regional variations**: Exist within France

### Manual Alphabet Differences from ASL
- G: Oriented upward (vs sideways in ASL)
- H: Like ASL '8'/horns handshape
- J: Swiveling Y (vs swiveling I in ASL)
- M, N: Oriented sideways
- X: Two fingers (like flexed V)
- T: Like F but thumb inside

## SECTION A: Basic Greetings (5 signs)

### 1. BONJOUR (Hello/Good day)

**LSF Concept:** Greeting gesture from forehead

**Detailed Motion:**
- Start with hand near forehead, palm facing out
- Move hand outward and down in an arc
- End with palm facing up
- Friendly facial expression

**WIA Talk Code:** `HS02-LC02-MV23-OR01-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat/open hand |
| LC | 02 | Forehead area |
| MV | 23 | Arc motion outward and down |
| OR | 01→02 | Palm out → Palm up at end |
| FE | 02 | Friendly smile |

**Alternative (Informal SALUT):**
- Simple wave, hand up, side to side
- WIA Code: `HS02-LC09-MV06-OR01-FE02`

**Source:** lecon.ai, studysmarter.co.uk

**LSF Concept:** Hand moves forward from chin

- Grateful expression

## SECTION B: Numbers 1-10 (Chiffres LSF)

### LSF Number System Overview

LSF numbers 1-9 use one hand with palm facing the viewer.
Numbers 6-10 differ from ASL - they use finger-folding pattern.

**Key Difference from Other Sign Languages:**
- 6-9: Thumb folds + one finger folds progressively
- 10: Closed fist

**Source:** clic-campus.fr, walter-learning.com, 2-as.org

## SECTION C: Basic Phrases (5 signs)

### 16. JE T'AIME (I Love You)

**LSF Concept:** Two-action romantic expression

**Detailed Motion:**
- Action 1: Bring fingertips together on each hand, then put hands together
  - Like hands are "kissing"
- Action 2: Place flat right hand on belly
  - Run it up the front of body
  - At heart height, push hand out toward person

**WIA Talk Code (Action 1):** `HS34-LC09-MV24-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 34 | Fingertips bunched (both hands) |
| LC | 09 | Neutral space |
| MV | 24 | Hands come together (kissing) |
| OR | 05 | Fingertips toward each other |
| FE | 01 | Loving expression |

**WIA Talk Code (Action 2):** `HS02-LC08-MV08-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hand |
| LC | 08 | Belly → Chest → Outward |
| MV | 08 | Upward then push out |
| OR | 05→01 | Palm on body → Palm out |
| FE | 01 | Loving expression |

**Note:** ILY (🤟) handshape is NOT standard LSF but internationally recognized

**Source:** signstation.org, sltranslator.com

## LSF Linguistic Features Summary

### Grammar Structure
- **Word Order:** Time + Place + Subject + Object + Verb
- **Spatial grammar:** 3D space used for relationships and time
- **Facial markers:** Essential for questions and emotions
  - Raised eyebrows = question
  - Nodding = affirmation
  - Head shake = negation

### LSF vs ASL Comparison
Despite historical connection, LSF and ASL have diverged:
- Different number systems (especially 6-9)
- Different manual alphabet letters (H, J, M, N, T, X)
- Different vocabulary for many words
- Similar grammatical concepts (spatial reference, facial markers)

### Regional Variations
- Variations exist within France
- Belgium-French Sign Language (LSFB) is related but distinct
- Quebec Sign Language (LSQ) diverged significantly
