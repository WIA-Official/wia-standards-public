# Batch 184: Kajana Sign Language - 수리남 마룬족 마을 수어

## 🏞️ 개요
| 항목 | 내용 |
|------|------|
| **언어 코드** | kaja1257 (Glottolog) |
| **현지명** | Kajana Gebarentaal (네덜란드어) |
| **지역** | 수리남, Sipaliwini District, Kajana 마을 |
| **화자 수** | 극소수 (단 3가족) |
| **위치** | Gaan-lio (Gran Rio) 강변 |
| **민족** | Saramaka 마룬족 |
| **상태** | 심각한 위기 |

## 📖 역사적 배경

### Saramaka 마룬족
- **기원**: 17-18세기 네덜란드 식민지 농장에서 탈출한 노예
- **자칭**: Kadosi-nengre (Cardoso 농장 출신)
- **언어**: Saramaccan (크레올어)
- **위치**: Suriname River 상류

### 마을 정보: Kajana
- **위치**: 3°54′2″N 55°34′21″W
- **인구**: 약 200명
- **인프라**:
  - Cayana Airstrip (비행장)
  - Radio Thijs (지역 라디오)
  - 초등학교
  - 의료 센터 (Medische Zending)
  - 유치원

### 수어 발생
- 유전적 농으로 인한 높은 청각장애 비율
- 마룬 공동체 특유의 밀착된 사회구조
- 가족 내 세대 간 전승

## 👥 사용자 커뮤니티

### 극소규모 공동체
- **단 3가족**만 사용
- 가장 작은 마을 수어 중 하나
- 청인 가족 구성원도 사용

### Saramaka 문화적 맥락
- 모계제 사회 (matrilineal)
- 확대 가족 구조
- 전통 마을 100-200명 규모
- 강변 생활 (수상 교통, 어업)

## 🌐 지역적 맥락

### 수리남의 언어 다양성
- **공용어**: 네덜란드어
- **링구아 프랑카**: Sranan Tongo (영어 기반 크레올)
- **마룬 언어들**:
  - Saramaccan (30% 영어 + 20% 포르투갈어 + 50% 아프리카어)
  - Ndyuka, Paramaka, Aluku
  - Matawai
- **원주민 언어**: Lokono, Akawaio 등

### Saramaccan의 특수성
- 세계에서 가장 아프리카 어휘 비율 높은 크레올
- 서아프리카/중앙아프리카 기원의 50% 어휘
- 영어 30%, 포르투갈어 20%
- 18세기 초 분리 후 독자 발전

## 🔤 언어학적 특성

### 알려진 정보 제한
- 체계적 언어학적 기술 부족
- Richard Cokart 2010, van den Bogaerde 2005 일부 언급
- 마을 수어 일반 특성 적용 가능

### 예상되는 특성 (마을 수어 유형)
- 높은 도상성
- 지역 제스처 기반
- 가족 내 변이
- 제한된 어휘 (일상생활 중심)
- 청인 제스처 영향

- 고립된 공동체
- 청인-농인 공유
- ASL/LSC 등 대형 수어 위협

## ⚠️ 위기 요인

### 인구학적
- 단 3가족만 사용
- 젊은 세대 전승 불확실
- 외부 이주 (특히 French Guiana로)

### 사회경제적 변화
- 1990년대 수리남 내전 영향
- French Guiana로의 이주 증가
- 금광 개발로 인한 외부인 유입
- 전통 생활방식 변화

### 언어적 압력
- Sranan Tongo 영향력 확대
- 교육에서 네덜란드어/ASL 사용
- 농교육 기관 부재
