### Introduction
Number systems in sign languages vary considerably worldwide, reflecting cultural differences in counting systems and visual-manual adaptations. Understanding number signs is essential for practical communication in any sign language.

### ASL Number System

|--------|----------|
| 1-5 | Extended fingers |
