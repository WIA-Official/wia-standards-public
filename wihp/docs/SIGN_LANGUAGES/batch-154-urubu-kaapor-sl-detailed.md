# Batch 154: Urubu-Kaapor Sign Language

## 기본 정보
- **언어명**: Urubu-Kaapor Sign Language
- **현지명**: Kaapor Sign Language
- **약칭**: UKSL
- **ISO 639-3**: 없음
- **국가**: 브라질
- **지역**: Maranhao 주, 아마존
- **사용자 수**: 수십 명 deaf + 청인 화자
- **언어 유형**: 마을 수어 / 원주민 수어
- **상태**: 취약

## 역사적 배경

### 기원
- 발생 시기: 여러 세대 전
- 장소: Urubu-Kaapor 원주민 영토
- 원인: 공동체 내 유전성 청각장애

### Urubu-Kaapor 민족
- 인구: 약 1,500명
- 언어: Kaapor - 투피 계통
- 위치: Maranhao 주, 아마존 열대우림

## 언어적 특징

### 독자성
- Libras와 무관: 완전히 독자 발생
- 외부 영향 없음: 격리된 원주민 영토
- 완전한 언어: 일상 의사소통 전체 가능

### 특이한 특징
- 청인도 광범위 사용: 마을 전체가 수어 사용
- 이중 모달리티: Kaapor 구어 + 수어 함께 사용
- 도상성 높음: 열대우림 환경 반영

### 어휘 특징
- 열대우림 동식물 풍부
- 사냥, 채집 관련
- Kaapor 문화 관련

## 사회언어학적 특징

### 화자 현황
- 청각장애 화자: 수십 명
- 청인 유창 화자: 수백 명

### 사회적 통합
- 청각장애인 완전 통합
- 차별 없음
- Marthas Vineyard와 유사한 패턴
