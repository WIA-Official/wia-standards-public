# WIA Talk Sign Language Mapping - Auslan (Australian Sign Language)
## Batch 07: 20 Signs with Full WIA Talk Component Analysis

**Language:** Auslan - Australian Sign Language
**Region:** Australia (nationwide)
**Estimated Users:** 20,000+ native signers
**Language Family:** BANZSL family (British-Australian-New Zealand Sign Language)
**Recognition:** 1987 officially recognized as community language by Australian government

---

## Auslan Characteristics

### Historical Background
- Developed from **British Sign Language (BSL)** and **Irish Sign Language (ISL)**
- Brought by Deaf immigrants and teachers in the 1800s
- Evolved into distinct language with unique regional dialects
- Shares ~80% lexical similarity with BSL and NZSL
- Part of BANZSL (British-Australian-New Zealand Sign Language) family

### Key Features
- **Two-handed fingerspelling** (like BSL, unlike ASL's one-handed)
- **66 handshapes total** (38 major + 28 variants)
- **3D spatial language** - uses space for grammar and context
- **Facial expressions** - critical grammatical markers
- **Size and Shape Specifiers (SASS)** - descriptive classifiers

### Regional Dialects
- **Northern dialect:** Sydney, Queensland
- **Southern dialect:** Victoria, South Australia, Western Australia
- Dialects are mutually intelligible
- Example: "hungry" in Melbourne = "sexy" in Sydney!

### Grammar Structure
- Different from English word order
- **W-words go LAST** (not first like English)
  - English: "Where do you live?"
  - Auslan: "YOU LIVE WHERE?"
- **Bracing:** W-word can go first AND last (e.g., "WHO WASH-DISHES WHO?")
- Articles ("a", "the", "it") are not used
- Time reference usually at start

### Dominant vs Blackboard Hand
- **Dominant hand:** Your writing hand - used for movement
- **Blackboard hand:** Non-dominant - used as support
- This distinction is critical in two-handed alphabet

## WIA Talk 5-Component System Reference

| Code | Component | Description |
|------|-----------|-------------|
| HS | Hand Shape | Configuration of fingers and palm |
| LC | Location | Where the sign is performed |
| MV | Movement | Direction and type of motion |
| OR | Orientation | Which way palm/fingers face |
| FE | Facial Expression | Required non-manual markers |

## SECTION A: Basic Greetings (5 signs)

### 1. HELLO

**Auslan Concept:** Simple wave greeting

**Detailed Motion:**
- Wave flat hand several times
- Friendly facial expression
- Similar to universal wave gesture
- Alternative: Small salute-like gesture

**WIA Talk Code:** `HS02-LC09-MV06-OR01-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat/open hand |
| LC | 09 | Neutral space |
| MV | 06 | Wave side to side |
| OR | 01 | Palm forward |
| FE | 02 | Friendly smile |

**Source:** signplanet.net, superprof.com

---

### 2. THANK YOU

**Auslan Concept:** Hand moves from chin outward

**Detailed Motion:**
- Flat hand, fingers upwards
- Palm toward you initially
- Start at chin
- Move forward in a slight arc
- Grateful expression

**WIA Talk Code:** `HS02-LC05-MV23-OR05-FE02`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hand (fingers up) |
| LC | 05 | Chin area |
| MV | 23 | Arc motion forward |
| OR | 05→01 | Palm toward you → Palm forward |
| FE | 02 | Grateful expression |

**Note:** Very similar to BSL "thank you"

**Source:** auslan.org.au, signplanet.net

---

### 3. SORRY

**Auslan Concept:** Clawed hand shakes across mouth

**Detailed Motion:**
- Clawed hand (fingers spread and curved)
- Palm facing you
- Shake across mouth twice
- Apologetic expression

**WIA Talk Code:** `HS08-LC04-MV06-OR05-FE04`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 08 | Clawed hand (spread, curved fingers) |
| LC | 04 | Mouth area |
| MV | 06 | Shake side to side (2x) |
| OR | 05 | Palm toward face |
| FE | 04 | Apologetic expression |

**Note:** Similar to "dangerous" but fingers more spread for "sorry"

**Source:** signplanet.net, fixiefoo.typepad.com

### 4. YES

**Detailed Motion:**

### 5. NO

**Auslan Concept:** Two fingers tap thumb

**Detailed Motion:**
- Index and middle finger extended
- Tap against thumb
- Like bird beak closing
- Negative head shake

**WIA Talk Code:** `HS04-LC09-MV13-OR05-FE03`

**Note:** Similar to JSL "no"

## SECTION B: Numbers 1-10

### Auslan Number System Overview

Auslan uses **one-handed** number signs for 1-10.
Numbers are similar to BSL system.
For **age**: Numbers are signed starting from the nose.
For **time**: Numbers are signed from the pointer finger of opposite hand.

**Source:** helloasphyxia.wordpress.com, deafnav.com.au

## SECTION C: Basic Phrases (5 signs)

### 16. I LOVE YOU

**Auslan Concept:** Similar to BSL - not ASL's ILY handshape

**Detailed Motion:**
- Cross arms over chest
- Like hugging yourself
- Bounce arms off chest twice (like heartbeat)
- Alternative: Point to self + sign LOVE + point to person

**WIA Talk Code:** `HS02-LC07-MV26-OR05-FE01`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands (arms crossed) |
| LC | 07 | Chest area |
| MV | 26 | Cross arms, bounce 2x |
| OR | 05 | Arms across body |
| FE | 01 | Loving expression |

**Note:** ASL's ILY (🤟) handshape is understood but not traditional Auslan

**Source:** superprof.com

---

### 17. WHAT IS YOUR NAME? (NAME YOU WHAT?)

**Auslan Concept:** Name sign + question marker

**Detailed Motion:**
- Sign NAME: Two fingers tap chest (like name badge)
- Point to person (YOU)
- Sign WHAT: Index finger shakes side to side
- Question face (raised eyebrows)

**WIA Talk Code:** `HS04-LC07-MV15-OR05-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 04 | Two fingers |
| LC | 07 | Chest area |
| MV | 15 | Tap |
| OR | 05 | Fingers toward body |
| FE | 06 | Question expression |

**Grammar Note:** In Auslan, the question word (WHAT) goes LAST

**Source:** fixiefoo.typepad.com

### 20. WHERE IS THE TOILET? (TOILET WHERE?)

**Auslan Concept:** Toilet sign + location question

**Detailed Motion:**
- TOILET: Various signs exist
  - Method 1: T-handshape shakes
  - Method 2: Tapping motion
- WHERE: Both hands flat, move in and out (2x)
  - Question expression essential

**WIA Talk Code (WHERE):** `HS02-LC09-MV27-OR01-FE06`

| Component | Code | Description |
|-----------|------|-------------|
| HS | 02 | Flat hands (both) |
| LC | 09 | Neutral space |
| MV | 27 | In and out motion (2x) |
| OR | 01 | Palms facing each other |
| FE | 06 | Strong question expression |

**Grammar Note:** Sign "TOILET WHERE?" not "Where is the toilet?"

## Auslan Linguistic Features Summary

### BANZSL Family
Auslan is part of the **BANZSL** family:
- **B**ritish Sign Language (BSL)
- **A**ustralian Sign Language (Auslan)
- **N**ew **Z**ealand Sign Language (NZSL)

These three share approximately 80% lexical similarity due to common historical origins.

### Key Grammar Rules
1. **W-words go LAST** (unlike English)
   - "What is your name?" → "NAME YOU WHAT?"
   - "Where do you live?" → "YOU LIVE WHERE?"

2. **Bracing:** W-word can appear first AND last
   - "WHO WASH-DISHES WHO?"

3. **No articles:** Delete "a", "the", "it", "be", "are"

4. **Pronouns:** Not gender-specific, mostly pointing

5. **Spatial grammar:** Set up locations in space, then point to them

### Types of Signs in Auslan
1. **Lexicalised signs:** Standard dictionary signs
2. **Depicting signs:** Size/shape specifiers (SASS)
3. **Fingerspelling:** Two-handed alphabet
4. **Pointing:** For pronouns, locations, reference
5. **Enactment:** Taking on character (storytelling)

### Non-Manual Features
- **Facial expression:** Essential for grammar
- **Eye contact:** Critical in Deaf culture
- **Body language:** Part of the message
- **Mouthing:** Sometimes accompanies signs
