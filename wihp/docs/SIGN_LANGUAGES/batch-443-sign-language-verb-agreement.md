# Sign Language Verb Agreement
## Directional Verbs and Spatial Agreement

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Verb agreement in sign languages |
| **Feature** | Directional movement |
| **Function** | Mark subject/object |
| **Unique** | Spatial agreement |
| **Type** | Agreement verbs |

### Introduction
Sign languages have a unique system of verb agreement where many verbs physically move between locations in space that represent the subject and object. This spatial agreement system marks grammatical relationships through the direction of verb movement.

### Types of Verbs

#### Plain Verbs
| Type | Description |
|------|-------------|
| Definition | No agreement marking |
| Examples | KNOW, LIKE, THINK |
| Movement | Doesn't change direction |
| Citation | One form only |

#### Agreement Verbs
| Type | Description |
|------|-------------|
| Definition | Show agreement |
| Examples | GIVE, TELL, ASK |
| Movement | Direction meaningful |
| Marking | Subject and/or object |

#### Spatial Verbs
| Type | Description |
|------|-------------|
| Definition | Location-based |
| Examples | PUT, MOVE |
| Movement | Actual locations |
| Function | Place reference |

### How Agreement Works
| Element | Description |
|---------|-------------|
| Subject | Starting point |
| Object | Ending point |
| Direction | Shows relationship |
| Modification | Verb movement changes |

### Agreement Verb Examples
| Verb | Movement |
|------|----------|
| GIVE | From giver to receiver |
| ASK | From asker to asked |
| TELL | From teller to told |
| HELP | From helper to helped |
| SHOW | From shower to shown |

### I-GIVE-YOU vs. YOU-GIVE-ME
| Direction | Meaning |
|-----------|---------|
| 1st → 2nd | I give to you |
| 2nd → 1st | You give to me |
| 1st → 3rd | I give to him/her |
| 3rd → 1st | He/she gives to me |

### Establishing Reference Points
| Step | Process |

### Subject Agreement
| Marking | Description |
|---------|-------------|
| Starting point | Subject location |
| Hand orientation | May mark subject |
| Some verbs | Subject only |

### Object Agreement
| Marking | Description |
|---------|-------------|
| Ending point | Object location |
| Direction | Toward object |
| Some verbs | Object only |

### Double Agreement
| Type | Description |
|------|-------------|
| Both marked | Subject and object |
| Full agreement | Complete spatial info |
| Example | GIVE, TELL |

### Number Agreement
| Number | Marking |
|--------|---------|
| Singular | One endpoint |
| Dual | Two endpoints |
| Multiple | Arc movement |
| Exhaustive | Each individual |

### Reciprocal Agreement
| Form | Meaning |
|------|---------|
| Alternating | "Each other" |
| Example | We-LOOK-AT-each-other |
| Movement | Back and forth |
