# Batch 108: EthSL (Ethiopian Sign Language)

## 기본 정보
- **언어 코드**: EthSL
- **ISO 639-3**: eth
- **정식 명칭**: Ethiopian Sign Language / የኢትዮጵያ የእጅ ቋንቋ
- **영어 명칭**: Ethiopian Sign Language
- **사용 국가**: 에티오피아
- **사용자 수**: 약 **100만 명** 이상
- **법적 인정**: 교육용 언어로 사용 (초등~대학)
- **언어 계통**: 혼합 (ASL, 스웨덴 수어 영향 + 토착 발전)

## 역사적 배경

### 기원
- **341-346 AD**: 기독교 도입, 전통 교회 교육 시작 (농인 교육 없음)
- **1956**: 초등 농교육 시작
- **1961**: 최초 농학교 설립 (피아자-아디스아바바)
- **1971**: 다양한 농학교에서 수어 사용 시작

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1956 | 초등 농교육 시작 |
| **1961** | 미국 선교사 Carl Thompson & Larency Darton 최초 농학교 설립 |
| 1961~ | ASL + 스웨덴 수어 → 에티오피아 맥락에 적응 |
| | Sheba 암하라 지문자 개발 |
| 1971 | 다양한 학교에서 수어 사용 |
| 현재 | 아디스아바바 대학에서 고등교육 수어 사용 |

### 두 가지 기원
1. **에리트레아 Karen School**: 스웨덴 수어
2. **에티오피아 미국 선교사 학교**: ASL 기반

→ 두 영향이 합쳐져 에티오피아 고유 표지 추가

## 언어적 특성

### 언어 구성
- **시각-운동-공간적** 언어
- 손모양, 방향, 위치, 특정 신체 부위 움직임
- 수지 + 체계적 비수지 신호

### 독립적 언어
- 에티오피아 구어와 **독립적**
- 자연어의 모든 기준 충족
- 200개 이상의 세계 수어와 동등한 의사소통 잠재력

### 지역 변이
- 에티오피아 80개 이상 토착어 존재
- 지역 변이가 EthSL 어휘에 반영
- **수렴 현상**: 사회언어적 요인으로 하나의 주요 변종으로 통합

### 암하라어 접촉
- 암하라어, 기타 에티오피아 언어와 접촉 불가피
- 그러나 EthSL은 독립적 언어

### Sheba 암하라 지문자
- 최초 학생들이 교사가 되어 개발
- 암하라 문자 기반 지문자 체계
