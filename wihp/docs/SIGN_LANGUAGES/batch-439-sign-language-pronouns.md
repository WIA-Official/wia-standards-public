# Sign Language Pronouns
## Pronominal Systems in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Pronouns in sign languages |
| **Method** | Pointing, spatial reference |
| **Feature** | Directional incorporation |

| **Universal** | All sign languages have |

### Introduction
Pronouns in sign languages work differently from spoken languages, relying heavily on pointing and spatial reference. Sign languages establish referents in space and then point to those locations to refer back to them, creating an efficient pronominal system.

### Basic Pronouns
| Pronoun | ASL Form |
|---------|----------|
| I/me | Point to self |
| You | Point to addressee |
| He/she/it | Point to location |
| We | Various forms |
| You (plural) | Point to group |
| They | Point to group |
