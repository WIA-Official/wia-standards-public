### Introduction
Sign language creoles are sign languages that have emerged from contact situations, most notably when deaf individuals with different home sign systems come together. The most famous example is Nicaraguan Sign Language (ISN/LSN), which emerged in the 1970s-80s when deaf children were brought together in schools for the first time.

Creole sign languages provide unique evidence for understanding how languages are created and how innate language capacity works.

### How NSL Emerged
| Stage | Development |
|-------|-------------|
| 1977 | First deaf school opens Managua |
| Pre-school | Children used home signs |
| Contact | Home signs mixed on playground |
| First cohort | Pidgin-like communication |
| Second cohort | Children grammaticized it |
| Result | Full natural language |

### Generations of NSL
| Generation | Characteristics |
|------------|-----------------|
| First cohort (1970s-80s) | Less grammatical complexity |
| Second cohort (1980s-90s) | Added grammar, spatial verbs |
| Third cohort (1990s+) | Native-like acquisition |
| Current | Fully developed language |
