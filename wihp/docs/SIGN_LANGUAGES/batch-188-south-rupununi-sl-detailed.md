# Batch 188: South Rupununi Sign Language - 가이아나 원주민 마을 수어

## 개요
| 항목 | 내용 |
|------|------|
| **언어 코드** | 미지정 |
| **지역** | 가이아나 남부, Rupununi 사바나 |
| **민족** | Wapishana (와피샤나) 원주민 |
| **범위** | 최소 7개 마을 |
| **경계** | 가이아나-브라질 국경 지역 |
| **상태** | 미문서화 / 위기 |

## 지리적 배경

### 위치
- Lethem 마을 남쪽
- Rupununi 사바나 지대
- 가이아나-브라질 국경 지역
- 열대 사바나 기후

### Wapishana 마을들
- 최소 7개 마을에서 높은 선천적 농 비율
- 정확한 마을 명칭 미공개 (프라이버시)
- 가이아나/브라질 양측에 분포

## 민족적 배경

### Wapishana 민족
- 아라왁 (Arawakan) 어족
- 가이아나/브라질 국경 지역 거주
- 전통적 농업/목축 생활
- 강한 공동체 결속

### 언어적 맥락
가이아나의 언어 다양성:
- 공용어: 영어
- 크레올: Guyanese Creole
- 원주민 언어: Lokono, Akawaio, Wapishana 등
- 이민자 언어: Bhojpuri (Guyanese Hindustani)

## 수어 발생

### 유전적 농
- 선천적 농의 높은 비율
- 마을 내 유전자 집중
- 여러 세대에 걸친 농인 존재

### 공유 수어 특성
- 농인과 청인 모두 사용
- 가족/마을 내 소통 도구
- 일상생활에 통합

## 문서화 상황

### 현재 상태
- 체계적 문서화 없음
- 초기 현지 조사만 수행
- 언어학적 기술 부재

### ELDP 조사 (초기)
연구팀: Ben Braithwaite, Lily Kwok, Rehana Omardeen
(University of the West Indies, St. Augustine)

**조사 내용**:
- 문화 특화 자료로 핵심 어휘 유도
- 지역 간 어휘 변이 조사
- 자연스러운 대화 녹화

**대화 유형 기록**:
1. 농인-농인 대화
2. 청인-농인 대화
3. 농인-연구자 대화

**성과**:
- 언어 구조 초기 스케치
- 공동체의 추가 협력 연구 희망 확인

## 사회적 기능

### 일상 통합
- 농업 활동 참여
- 고용 참여
- 문화 활동 참여
- 농인들이 일반적으로 잘 통합됨

### 문제점
- 일부 외진 지역 농아동은 접근 부재
- 결정적 초기 연령에 수어 노출 없음
- 홈사인만 사용하는 경우

## 위협 요인

### 외부 수어 접촉
- Brazilian Sign Language (Libras) 접촉 가능성
- American Sign Language (ASL) 접촉
- 특수 교육 프로그램 도입 시

### 공동체 입장
- 기존 수어가 문화적으로 적합
- 외국 체계 도입보다 자체 수어 선호
- 농교육에서 자체 수어 사용 희망

### 인권적 관점
- 언어 문서화와 농인 인권 불가분
- 의사소통 접근권 보장 필요
- 농아동 조기 언어 노출 필요

## 가이아나 수어 상황

### Georgetown (수도)
- ASL이 주로 사용됨
- 젊은 세대 ASL 습득
- 농가족 출신은 다른 수어 사용 가능
- 국가 토착 수어 미확립

### GSL (Guyanese Sign Language)
- 존재 주장 있음
- 그러나 체계적 연구 부재
- ASL과의 관계 불명확

### 농공동체 기관
- Deaf Association of Guyana (DAG): 전국 조직
- Georgetown 중심 활동
- 수어 연구/훈련 지원

## 학술적 가치

### 원주민 마을 수어
- 남미 원주민 마을 수어 사례
- Arawakan 문화권 수어
- 국경 횡단 수어 (가이아나-브라질)

### 언어 발생 연구
- 고립 공동체 수어 발달

- 제스처에서 수어로의 발달
- 외부 수어 접촉 영향
