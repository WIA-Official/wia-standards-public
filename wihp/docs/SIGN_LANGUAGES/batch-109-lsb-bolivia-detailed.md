# Batch 109: LSB (Lengua de Señas Boliviana / Bolivian Sign Language)

## 기본 정보
- **언어 코드**: LSB
- **ISO 639-3**: bvl
- **정식 명칭**: Lengua de Señas Boliviana
- **영어 명칭**: Bolivian Sign Language
- **사용 국가**: 볼리비아
- **사용자 수**: 약 23,000명 (추정)
- **법적 인정**: 2009년 헌법에서 토착어로 인정
- **언어 계통**: ASL 방언/파생어

## 역사적 배경

### 기원
- **1973**: ASL이 볼리비아에 도입
- 선교사 Eleanor & Lloyd Powlison이 미국에서 전래
- 기존 토착 수어가 ASL로 대체됨
- 이후 지역 토착 표지 통합

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1973 | ASL 볼리비아 도입 (미국 선교사) |
| 1988 | 9개 농인 기관, 사용자 350-400명 추정 |
| 1992 | 첫 LSB 책 출판 (교육부) - 90% ASL 표지 |
| 1990s-2000s | 연구로 토착 표지 발굴/추가 |
| 2002 | 약 25개 농학교 |
| **2008** | 세계 농인의 날 시위 - 약 1,000명 동원 |
| **2009** | 헌법에서 토착어로 인정! |
| 2010 | 교육부 사전 출판 (500개 표지) |
| 2019 | Ipelec 추가 언어 인정 계획 발표 |
| 2025 | 가상 통역사 개발 (GPT-4o 기반) |

### 사용자 수 변화
| 연도 | 사용자 수 |
|------|----------|
| 1988 | 350-400명 |
| 1988 | 46,800명 농인 (전체) |
| 2008 | 약 1,000명 (시위 동원) |
| 현재 | 약 23,000명 (SIL 추정) |

## 언어적 특성

### 언어 계통
- **ASL 방언/파생어**로 분류
- 90% 이상 ASL 기반 (1992년 기준)
- 지역 토착 표지 통합 중
- 페루, 에콰도르, 콜롬비아 수어와 유사

### 볼리비아 표지
- 🦙 **라마가 머리 돌리는 모습** = 볼리비아!
- 토착 문화 반영 표지 개발 중

### 이중언어 교육
- LSB (주요 언어) + 스페인어 (보조 언어)
- Riberalta에서 시작, 전국 확산
