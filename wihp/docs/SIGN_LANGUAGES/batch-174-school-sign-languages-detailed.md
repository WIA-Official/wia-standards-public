# Batch 174: School Sign Languages - 학교 수어

## 개요
농인학교에서 자연 발생한 수어들
교육 언어와 별개로 학생들 사이에서 발달

## 1. Nicaraguan Sign Language

### 기본 정보
- **스페인어명**: Idioma de Senas de Nicaragua (ISN)
- **위치**: 니카라과
- **발생**: 1970년대-1980년대

### 특징
- 언어 발생 연구의 대표 사례
- 홈사인 → 피진 → 크리올
- 세대별 문법 복잡성 증가

### 연구 가치
- 언어 기원 연구
- 문법 발생 연구
- 크리올화 연구

## 2. Penang Sign Language

### 기본 정보
- **위치**: 말레이시아 페낭
- **발생**: 농인학교 학생들

### 특징
- 학교 수어
- Malaysian SL과 구별
- 연구 제한적

---

## 3. Tanzanian School Sign Languages

### 기본 정보
- **위치**: 탄자니아
- **유형**: 각 학교별 수어

### 특징
- 학교마다 다른 수어
- 통합 노력 중
- Tanzanian SL 표준화

---

## 4. Sri Lankan School Sign Languages

### 기본 정보
- **위치**: 스리랑카
- **유형**: 학교별 수어

### 특징
- 각 농인학교별 다른 수어
- 학교 간 상호 이해 제한
- 국가 표준화 필요

## 학교 수어 발생 조건

### 조건
1. 구어 교육 학교
2. 수어 교육 언어로 미사용
3. 다양한 지역 학생 집합
4. 기숙 환경

### 발생 과정
1. 홈사인 유입
2. 피진화
3. 크리올화 (세대 거치며)
