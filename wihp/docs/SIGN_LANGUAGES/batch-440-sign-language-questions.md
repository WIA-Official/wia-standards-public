# Sign Language Questions
## Interrogative Structures in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Questions in sign languages |
| **Feature** | Non-manual markers essential |
| **Types** | Yes/no, WH, rhetorical |
| **Universal** | All SLs have question forms |
| **Grammar** | Distinct from statements |

### Introduction
Questions in sign languages are marked primarily through non-manual features, particularly facial expressions and head position. Understanding question formation is essential for sign language fluency, as the grammatical marking distinguishes questions from statements.

### Non-manual Question Markers

#### Yes/No Questions
| Feature | Description |
|---------|-------------|
| Eyebrows | Raised |
| Eyes | Widened |
| Head | Tilted forward |
| Body | May lean forward |
| Hold | At end of question |

#### WH-Questions
| Feature | Description |
|---------|-------------|
| Eyebrows | Furrowed/lowered |
| Eyes | Squinted slightly |
| Head | Tilted |
| Body | May lean |
| Hold | At end of question |

### Yes/No Questions
| Structure | Description |
|-----------|-------------|
| Statement + NMM | Add non-manual markers |
| Word order | Usually unchanged |
| Essential | Facial expression |
| Answer | YES or NO |

### Yes/No Examples
| Statement | Question |
|-----------|----------|
| YOU HUNGRY | YOU HUNGRY (+ raised eyebrows) |
| SHE HOME | SHE HOME? (+ NMM) |
| Difference | Non-manual markers |

### WH-Question Words
| Word | Use |
|------|-----|
| WHO | Person identity |
| WHAT | Thing identity |
| WHEN | Time reference |
| WHERE | Location |
| WHY | Reason |
| HOW | Method |
| WHICH | Selection |
| HOW-MANY | Quantity |

### WH-Word Position
| Position | Description |
|----------|-------------|
| Final | Common in ASL |
| Initial | Also possible |
| Both | Doubling for emphasis |
| Language | Varies by SL |

### WH-Question Examples
| Question | Structure |
|----------|-----------|
| What's your name? | NAME YOU WHAT? |
| Where do you live? | YOU LIVE WHERE? |
| When did you arrive? | YOU ARRIVE WHEN? |

### Rhetorical Questions
| Feature | Description |
|---------|-------------|
| Purpose | Self-answering |
| Function | Topic introduction |
| Structure | Question + answer |
| NMM | Distinct markers |

### Rhetorical Example
| Structure | Meaning |
|----------|---------|
| WHY I LATE? | "Why was I late?" |
| CAR BREAK-DOWN | "My car broke down" |
| Function | Explains situation |

### Tag Questions
| Structure | Description |
|-----------|-------------|
| Statement + tag | Confirmation seeking |
| Example | YOU GO, RIGHT? |
| Function | Check understanding |

### Negative Questions
| Structure | Description |
|-----------|-------------|
| NEG + question | Negative inquiry |
| Example | YOU NOT GO? |
| Markers | Combined features |

### Question Scope
| Feature | Description |
|---------|-------------|
| NMM span | Over questioned portion |
| Duration | Held throughout |
| Marking | Clear boundaries |

### Embedded Questions
| Type | Description |
|------|-------------|
| Indirect | "I asked what..." |
| Structure | Complex sentences |
| Markers | Modified |

### Responses to Questions
| Question Type | Typical Response |
|---------------|------------------|
| Yes/No | YES, NO, or elaboration |

| Rhetorical | Speaker answers |
