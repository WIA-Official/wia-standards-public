# Batch 131: Central Taurus Sign Language (중앙 타우루스 수어)

## 기본 정보
- **언어명**: Central Taurus Sign Language (중앙 타우루스 수어)
- **터키어명**: Orta Toroslar Isaret Dili
- **약칭**: CTSL
- **ISO 639-3**: 없음 (미등록)
- **국가**: 터키
- **지역**: 중앙 타우루스 산맥, 남부 터키 (3개 마을)
- **사용자 수**: ~25명 deaf + ~80명 hearing (2021년 기준)
- **언어 유형**: 마을 수어 (Village Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 약 50년 전 (1970년대 추정)
- **원인**: 근친 결혼으로 인한 열성 유전성 청각장애
- **발견**: Rabia Ergin에 의해 학계에 알려짐

### 발견 스토리
- Rabia Ergin: Tufts University 인지과학 박사과정 학생
- 어머니의 고향 마을에서 어린 시절 친척들과 수화 사용
- 2013년 연구 세미나에서 언급하여 학계 주목
- Ray Jackendoff (Tufts 인지과학센터장) 등 "Wow!" 반응

### 마을 배경
- **위치**: 남부 터키, 중앙 타우루스 산맥
- **인구**: 약 450명 (주요 마을)
- **특징**: 지리적으로 고립된 산악 지역
- **청각장애 비율**: 약 4.6% (일반 인구의 9배)

### 세대 구분
1. **CTSL-1 (1세대)**: 49-61세, 가족 내 첫 청각장애인, 언어 창시자 (9명)
2. **CTSL-2 (2세대)**: 42-54세, 1세대의 동생들, 언어 입력 더 많음 (8명)
3. **CTSL-3 (3세대)**: 24-30세, 1-2세대 자녀, 완전한 언어 습득 (4명)
4. **CTSL-4 (4세대)**: 어린이, 아직 문서화되지 않음

## 언어적 특징

### Turkish Sign Language (TID)와의 관계
- **완전히 다른 언어**: TID 원어민이 확인
- **외부 영향 없음**: 지리적, 사회적 고립으로 독자 발전
- **TID 영향**: 일부 화자 TID 접촉, 혼합 사용 시작

### 음운론 발달
- **수형 목록**: NSL (니카라과 수어)보다 느리게 발달
- **수형 복잡성**: NSL보다 낮음
- **해석**: 작은 공동체 규모와 사회 구조의 영향

### 어휘 특징
- **물체 표현 전략**:
  - 물체 기반 도상성 (object-based iconicity)
  - 행위 기반 도상성 (action-based iconicity)
  - 복합 전략

### 동사 일치 (Agreement)
- 1세대부터 동사 일치 존재
- 2세대에서 이중 일치 (double agreement) 체계적 사용
- 언어 진화의 빠른 속도 시사

### 문법 구조
- **분류사 (Classifiers)**: 발달 중
- **논항 구조**: 세대 간 복잡화
- **공간 문법**: 독자 발전

## 사회언어학적 특징

### 사용 현황
| 세대 | 인원 | 언어 사용 |
|------|------|-----------|
| 1세대 | 9명 | CTSL 단일어 (일부) |
| 2세대 | 8명 | CTSL 단일어 |
| 3세대 | 4명 | CTSL + 일부 TID |
| 청인 | ~80명 | CTSL 다양한 유창성 |

### 가족 구조
- 청각장애인 출생 또는 결혼으로 연결
- 확대 가족 내 전파
- 3개 인접 마을로 확산

### 사회적 특징
- 청각장애인 공식 교육 없음
- TID 접촉 기회 없었음 (과거)
- 최근 일부 화자 TID 다양한 수준 습득

## 언어학적 중요성

### 연구 가치
1. **언어 창발 연구**: 언어가 무에서 어떻게 발생하는가
2. **세대 간 변화**: 구조적 복잡성 증가 관찰
3. **소규모 공동체 언어**: 공동체 크기와 언어 구조 관계
4. **음운론 발달**: 수형 체계 형성 과정

### NSL과의 비교
| 특징 | CTSL | NSL |
|------|------|-----|
| 나이 | ~50년 | ~50년 |
| 공동체 크기 | ~25명 deaf | 수백 명 deaf |
| 수형 목록 | 느린 발달 | 빠른 발달 |
| 수형 복잡성 | 낮음 | 높음 |
| 사회 구조 | 동질적, 소규모 | 이질적, 대규모 |

### 결론
- 공동체 상호작용과 크기가 음운 체계 발달에 영향
- 작은 공동체는 더 느린 언어 변화 속도

## 위협 요인

### 언어 상태
- **EGIDS**: 미등록 (연구 중)
- **위기 수준**: 취약 (Vulnerable)

### 주요 위협
1. **Turkish Sign Language (TID) 확산**
   - 젊은 세대 TID 접촉 증가
   - 교육, 미디어 통한 TID 노출
   
2. **이주**
   - 젊은 세대 도시 이주
   - 마을 인구 감소

3. **세대 전환**
   - 청각장애 여성 타 마을로 결혼
   - 수어 공동체 분산
