# Batch 101: ÖGS (Österreichische Gebärdensprache / Austrian Sign Language)

## 기본 정보
- **언어 코드**: ÖGS
- **ISO 639-3**: asq
- **정식 명칭**: Österreichische Gebärdensprache
- **영어 명칭**: Austrian Sign Language
- **사용 국가**: 오스트리아
- **사용자 수**: 약 10,000명
- **법적 인정**: 2005년 9월 1일 헌법 인정
- **언어 계통**: 프랑스 수어(LSF) 가족 (일부 학자 분류)

## 역사적 배경

### 주요 연혁
| 연도 | 사건 |
|------|------|
| 1779 | 비엔나 최초 농학교 설립 |
| 1865 | 오스트리아 농인 협회 설립 |
| 1949 | 제1회 국제 동계 데프림픽 개최 (젤펠트) |
| 1990년대 | ÖGS 언어학 연구 시작 |
| 2002 | 연방 장애인평등법 (해석 서비스 권리) |
| 2005 | **헌법 인정** - §8(3) 독립 언어로 인정 |

### 헌법 조항 (2005)
> "Die Österreichische Gebärdensprache ist als eigenständige Sprache anerkannt."
> (오스트리아 수어는 독립적인 언어로 인정된다)

## 언어적 특성

### 언어 계통
- **프랑스 수어 가족** (Wittmann 1991 분류)
- 독일 수어(DGS), 스위스 수어와 문법 공유
- 헝가리 수어(HSL)와 역사적 연관 (비엔나 농학교)
- 어휘는 DGS, 스위스 SL과 다름

### 문법적 특징
- **어순**: SOV (주어-목적어-동사) 선호
- **주어 선호**: 문장 초반 모호한 논항 → 주어로 해석
- **일치 표지**: 두 가지 유형 (agrm-bc, agrm-mf)
- **비수지 표지**: 몸/어깨/머리 이동으로 주어 위치 표시

### 지문자 체계
| 구분 | 개수 | 설명 |
|------|------|------|
| 기본 알파벳 | 26개 | 독일어 알파벳 |
| 움라우트 | 3개 | Ä, Ö, Ü |
| 특수 문자 | 1개 | "sch" (삼중자) |
| **총계** | **30개** | |
