# Sign Language Dictionaries
## Lexicography and Documentation

### Overview
| Attribute | Detail |
|-----------|--------|
| **Field** | Sign language lexicography |
| **Challenge** | Representing visual language |
| **Formats** | Print, video, digital |
| **Purpose** | Reference, learning, preservation |
| **Evolution** | From print to multimedia |

### Introduction
Sign language dictionaries face unique challenges in representing three-dimensional, moving signs in a fixed format. The field has evolved from early print dictionaries with drawings or photos to modern multimedia databases with video recordings.

Lexicography is also crucial for sign language documentation and preservation.

### Historical Development
