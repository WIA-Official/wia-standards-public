# Batch 149: Jhankot-Mohalla Sign Language / Jumla Sign Language (줌라 수어)

## 기본 정보
- **언어명**: Jhankot-Mohalla Sign Language / Jumla Sign Language
- **약칭**: JMSL
- **ISO 639-3**: jhs
- **국가**: 네팔
- **지역**: Jumla District, 서부 네팔 히말라야
- **사용자 수**: 약 20명 deaf (2009년 추정)
- **언어 유형**: 마을 수어
- **계통**: 독립 언어
- **상태**: 심각한 위험

## 역사적 배경

### 기원
- **발생 시기**: 여러 세대 전
- **장소**: Jhankot, Mohalla 마을, Jumla
- **원인**: 유전성 청각장애

### 지리적 배경
- **위치**: 네팔 서부, 히말라야 산악 지역
- **고도**: 해발 2,500m 이상
- **접근성**: 매우 어려움, 도보 여러 날
- **고립**: 지리적 고립으로 독자 수어 발달

### 인구 배경
- Jhankot, Mohalla: 소규모 마을
- 청각장애 비율: 높음 (유전적 요인)
- 근친 결혼 관습

## 언어적 특징

### 독자성
- **Nepali Sign Language와 무관**: 독자 발생
- **Indian Sign Language 영향 없음**: 격리된 발달
- **완전한 언어 체계**: 복잡한 문법 구조

### 음운론
- 독자적 수형 체계
- 연구 필요

### 어휘 특징
- 히말라야 산악 생활 관련
- 농업 (계단식 밭)
- 목축 (야크, 양)
- 계절, 날씨 관련

### 문법
- 연구 진행 중
- 기초 분석만 존재

## 사회언어학적 특징

### 화자 현황
| 유형 | 인원 | 비고 |
|------|------|------|
| 청각장애 화자 | ~20명 | 2009년 기준 |
| 청인 수화자 | 수십 명 | 가족 |

### 사회적 통합
- 마을 공동체 통합
- 농업, 목축 활동 참여
- 가족 내 수어 전달

### 교육 상황
- 공식 청각장애 교육 없음
- Nepali SL 접촉 거의 없음
- 마을 내 자연 습득

## 언어 위기 상태

### 멸종위기 등급
- **상태**: 심각한 위험
- **EGIDS**: 6b-7

### 위협 요인
1. **도로 개통**: 외부 접촉 증가
2. **Nepali SL 영향**: 교육 확대 시
3. **인구 이동**: 젊은 세대 도시 이주
4. **소규모 화자**: 20명 내외

### 특수 상황
- 지리적 고립이 역설적으로 보존 요인
- 도로 개통으로 변화 예상
- 긴급 문서화 필요

## 연구 및 문서화

- **2009년**: 첫 체계적 보고
- **연구자**: 네팔 언어학자, 국제 연구팀
- **상태**: 기초 조사 완료, 심층 연구 필요

### 문서화 필요

- 자원 부족
- 현지 연구자 부족
