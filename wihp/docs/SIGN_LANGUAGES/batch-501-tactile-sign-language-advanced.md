### Introduction
Tactile sign language represents a critical adaptation of visual sign languages for individuals who are deaf-blind. This document explores advanced methods, variations, and the specialized skills required for tactile communication.

### Tactile Methods

#### Hand-over-Hand
| Method | Description |
|--------|-------------|
| Receiver's hands | On signer's hands |
| Feel movement | Shape, motion |
| Most common | Standard method |
| Bilateral | Both hands used |

#### Hand-under-Hand
| Method | Description |
|--------|-------------|
| Signer's hands | On receiver's hands |
| Less common | Alternative |
| Preference | Individual choice |

#### Tracking
| Method | Description |
|--------|-------------|
| Light touch | On wrist/forearm |
| Less intrusive | More independence |
| Some vision | With residual vision |
