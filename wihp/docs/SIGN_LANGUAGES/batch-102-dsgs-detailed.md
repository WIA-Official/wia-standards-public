# Batch 102: DSGS (Deutschschweizerische Gebärdensprache / Swiss German Sign Language)

## 기본 정보
- **언어 코드**: DSGS
- **ISO 639-3**: sgg
- **정식 명칭**: Deutschschweizerische Gebärdensprache
- **영어 명칭**: Swiss German Sign Language
- **사용 국가**: 스위스 (독일어권), 리히텐슈타인
- **사용자 수**: 약 7,500-10,000명
- **법적 인정**: 헌법적 인정 없음 (연방 차원)
- **언어 계통**: 연구 중 (DGS 파생 가능성, LSF 차용 경향)

## 스위스의 3대 수어

| 수어 | 지역 | 언어권 |
|------|------|--------|
| **DSGS** | 독일어권 칸톤 | 독일어 |
| LSF-CH | 로만드 (서부) | 프랑스어 |
| LIS-CH | 티치노 (남부) | 이탈리아어 |

**특징**: 스위스 농인은 2-3개 스위스 수어 + 외국 수어도 구사!

## 언어적 특성

### 언어 계통 (연구 중)
- **DGS (독일 수어)** 파생 가능성 연구 진행 중
- 실제로는 **LSF-SR (프랑스 수어 스위스 방언)**에서 더 많이 차용
- 독립적 발전 경로

### 지문자 체계
| 구분 | 설명 |
|------|------|
| 기본 알파벳 | DGS/ASL과 유사 |
| Ä | A + 엄지 움직임 |
| Ö | O → C 열림/닫힘 |
| Ü | U + 손가락 굽힘 |
| CH | 전용 표시 |

### 방언
- **5개 지역 방언** 존재
- 세대별 변이
- 데이터베이스에 모든 변이 기록

### 특이사항
- **지문자 사용 빈도 낮음**: BSL보다도 적음
- **입모양(Mouthing)** 의존도 높음
- 최근 젊은 세대에서 지문자 사용 증가

## 연구 프로젝트

### SMILE 프로젝트 (2016-2020+)
- **목적**: SLR 기반 DSGS 평가 시스템
- **데이터셋**: 100개 어휘, 30명 서명자
- **기술**: Microsoft Kinect v2 + GoPro Hero 4
- **피드백**: 손모양, 위치, 움직임 자동 평가

### FZG Basel 어휘 데이터베이스
- **규모**: 대규모 DSGS 어휘 수집
- **형식**: HamNoSys + SignWriting 표기
- **내용**: 5개 방언, 세대별 변이, 의미/문법 정보

### Multi-parallel Corpus
- **특징**: 6방향 병렬 데이터
- **언어**: DSGS + LSF-CH + LIS-CH + 독일어 + 프랑스어 + 이탈리아어
- **출처**: TV 프로그램 수어 번역

## 위기 요인

### 인공와우 영향
- 2006년 기준 **80%** 농아 영아에 인공와우 시술
- 13개월 영아부터 조기 시술
- 의료진: 수어 사용 권장하지 않음
- 농학교 학생 수 **지속 감소**

### 통합 교육
- 대다수 인공와우 아동 → 청인 학급 통합
- 수어 지원 없는 통합 교육

## 리히텐슈타인

- 리히텐슈타인 농인도 DSGS 사용
- 스위스 독일어권 방언과 밀접한 관련

## 특이사항

### 다중 수어 사용
- 스위스 농인: 복수의 스위스 수어 구사
- 외국 수어도 습득 (DGS, LSF, LIS, ASL)
- 스위스 농인 스포츠 협회 등 전국 조직에서 교류

### SignWriting 도서
- 2권의 SignWriting 책 출판
- DSGS 이야기 완전 표기
