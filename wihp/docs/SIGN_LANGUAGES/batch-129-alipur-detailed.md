# Batch 129: Alipur Sign Language (알리푸르 수어)

## 기본 정보
- **언어명**: Alipur Sign Language (알리푸르 수어)
- **약칭**: APSL
- **ISO 639-3**: 없음 (미등록)
- **국가**: 인도 (카르나타카 주)
- **지역**: Alipur 마을, 방갈로르 북쪽 80km
- **사용자 수**: 150-250명 deaf + 10,000명+ hearing (2015년 기준)
- **언어 유형**: 마을 수어 (Village Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 10세대 이상 전 (정확한 시기 미확인)
- **원인**: 근친 결혼 (consanguineous marriage)으로 인한 유전성 청각장애
- **마을 특성**: 시아파 무슬림 엔클레이브

### 마을 배경
- **위치**: 카르나타카 주, 방갈로르에서 북쪽 80km
- **인구**: 약 20,000-26,000명 (2015년 기준)
- **종교**: 시아파 이슬람 (이란 계통)
- **역사**: Bijapur의 Adil Shahi 왕조 몰락 후 이주

### 사회적 특성
- **별명**: "미니 이란" (Mini Iran in Karnataka)
- **특징**: 경찰서, 주류 판매점, 영화관 없음
- **통치**: Anjum-e-Jaffria (신탁 기관)이 비공식 자치

### 청각장애 비율
- **마을 내**: 약 0.75% (인도 평균 0.41%의 2배)
- **가족 관계**: 거의 모든 가족이 청각장애 친척과 연결

## 언어적 특징

### 독립성
- **Indian Sign Language (ISL)와의 관계**: 완전히 다른 언어
- **ASL 영향**: 최근 젊은 세대에서 유입 시작
- **지문자**: 전통적으로 없음 (2007년 이전)

### 수 체계 (특이점)
- **복잡한 수 표현**: 덧셈과 뺄셈 조합 사용
- **구순 동작**: 큰 숫자 표현에 사용
- **예**: 숫자를 조합하여 복잡한 수 표현

### 음운론
- 독자적 수형 체계
- ISL과 다른 위치 사용
- 비수지 신호 풍부

### 문법 구조
- **어순**: 연구 진행 중
- **공간 문법**: 독자 발전
- **형태론**: ISL과 구별됨

## 사회언어학적 특징

### 사용 환경
- **가정**: 일상 대화
- **직장**: 보석 세공 (마을 주요 산업)
- **시장**: 거래 의사소통
- **종교**: 모스크 예배는 통역 없음

### 화자 분포
| 유형 | 인원 | 비고 |
|------|------|------|
| 청각장애 화자 | 150-250명 | 단일어 사용자 대부분 |
| 청인 유창 화자 | 수백 명 | 가족 구성원 |
| 청인 기본 화자 | 수천 명 | 기본 의사소통 |

### 교육 상황
- **전통**: 청각장애인 문해력 없음
- **2008년 이후**: 청각장애 학교 프로젝트 시작
- **현재**: APSL을 교육 매체로 사용한 문해 교육

### 미디어
- **Ali Channel**: 마을 자체 TV 채널
- APSL 어휘 방송 프로그램 운영
- 지역 사회 APSL 확산에 기여

## 언어 위기 상태

### 현재 상태
- **EGIDS**: 6a/6b (활발/위협)
- **위기 수준**: 취약 (Vulnerable)

### 위협 요인

#### 1. ISL/ASL 영향
- 젊은 세대 방갈로르 방문 증가
- 무료 버스 패스로 이동성 증가
- 도시에서 ASL/ISL 접촉

#### 2. 세대 간 차이
- 젊은 세대: ASL 어휘 혼용
- 노년층: 순수 APSL 사용
- 소통 어려움 발생 시작

#### 3. 사회 변화
- 외부 결혼 시도
- 도시 이주 증가
- 현대화 압력

### 노년층 불만
"왜 아이들이 이상한 방갈로르 수화를 쓰는가?"

## 문화적 특징

### 결혼 관습
- **제한**: 청각장애인 간 결혼 권장하지 않음
- **이유**: 청각장애 확산 방지 목적
- **실제**: 청인-청각장애인 결혼 일반적

### 보석 세공 산업
- 마을 주요 산업
- 청각장애인도 동등하게 참여
- APSL로 업무 의사소통
