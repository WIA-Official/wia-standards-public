# Sign Language Negation
## Expressing Negative Concepts in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|

### Introduction
Negation in sign languages involves both manual signs and non-manual markers, particularly the side-to-side headshake. The combination of these elements creates a rich system for expressing negative concepts, denial, and absence.

### Non-manual Negation
| Feature | Description |
|---------|-------------|
| Headshake | Side to side |
| Brow furrow | Often accompanies |
| Nose wrinkle | Some negative forms |
| Mouth | Pursed or frown |
| Duration | Spans negated element |
