# Sign Language vs. Gesture
## Distinguishing Language from Non-linguistic Communication

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Sign language vs. gesture distinction |
| **Misconception** | Sign is "just gesture" |
| **Reality** | Fundamentally different systems |
| **Importance** | Validates sign as language |
| **Research** | Extensive evidence |

### Introduction
A persistent misconception is that sign languages are merely elaborate gesture systems. Research has definitively shown that sign languages and gesture are fundamentally different: sign languages are full natural languages with complex grammar, while gestures are non-linguistic communication that accompanies speech.

Understanding this distinction is crucial for recognizing the linguistic status of sign languages.

### Key Differences
| Aspect | Sign Language | Gesture |
|--------|---------------|---------|
| Structure | Complex grammar | No grammar |
| Arbitrary | Many arbitrary signs | Mostly iconic |
| Systematic | Rule-governed | Variable |
| Acquisition | Natural acquisition | Learned informally |
| Brain | Language areas | Different processing |
| Community | Shared by deaf community | Individual variation |

### Linguistic Properties of Sign Language
| Property | Sign Language Has | Gesture Lacks |
|----------|-------------------|---------------|
| Phonology | Yes (parameters) | No |
| Morphology | Yes (inflection) | No |
| Syntax | Yes (word order) | No |
| Productivity | Unlimited | Limited |
| Duality | Yes | No |

### Emblems (Conventional Gestures)
| Emblem | Meaning | Culture |
|--------|---------|---------|
| Thumbs up | Good/OK | Western |
| Head nod | Yes | Many |
| Wave | Hello/goodbye | Widespread |
| Shrug | Don't know | Western |

### Emblems Are Not Sign Language
| Difference | Explanation |
|------------|-------------|
| Number | Few emblems (~60) vs. thousands of signs |
| Grammar | No emblem grammar |
| Variation | Culture-specific |
| Combination | Can't form sentences |

### Brain Evidence
| Finding | Implication |
|---------|-------------|
| Sign: left hemisphere | Language processing |
| Gesture: both/right | Not language areas |
| Aphasia | Sign impaired, gesture spared |
| Different systems | Neurologically distinct |

### Aphasia Evidence
| Condition | Effect on Sign | Effect on Gesture |
|-----------|----------------|-------------------|
| Broca's aphasia | Sign production impaired | Gesture preserved |
| Wernicke's aphasia | Sign comprehension impaired | Gesture preserved |
| Implication | Sign is language, gesture is not |

### Acquisition Differences
| Aspect | Sign Language | Gesture |
|--------|---------------|---------|
| Age | Developmental stages | Gradual |
| Milestones | Babbling, first signs | Less structured |
| Grammar | Emerges naturally | No grammar emerges |
| Input | From signing community | Observation |
