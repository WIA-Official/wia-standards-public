# Batch 168: Brazil Indigenous Sign Languages - 브라질 원주민 수어

## 개요
브라질에 Libras 외 다수 원주민/마을 수어 존재
대부분 미문서화 또는 초기 연구 단계

## 1. Kaapor Sign Language

### 기본 정보
- Batch 154에서 상세 문서화
- 가장 잘 연구된 브라질 원주민 수어
- Maranhao 주, 아마존

---

## 2. Terena Sign Language

### 기본 정보
- **위치**: 남부 브라질
- **민족**: Terena 원주민
- **유형**: 마을 수어

### 특징
- 학교에서는 Libras 사용
- 가정에서 Terena SL 사용
- 코드 스위칭

### 연구
- 음운론 비교 연구 진행 중
- Libras와의 비교 분석

---

## 3. Marajo Sign Language

### 기본 정보
- **위치**: Marajo Island, Para 주
- **상태**: 불확실

### 특징
- 마을 수어인지 홈사인인지 불명확
- 여러 가족 홈사인 가능성
- 체계적 연구 필요

### 연구
- 2008-2013년 Soure 마을 조사
- 언어 지위 논쟁

---

## 4. Maxakali Sign Language

### 기본 정보
- **위치**: Minas Gerais 주
- **민족**: Maxakali 원주민
- **상태**: 매우 젊은 언어

### 특징
- 최근 발생
- 홈사인에 가까움
- University of Oslo 연구

---
