# Batch 183: Providence Island Sign Language (PISL) - 콜롬비아 카리브해 섬 수어

## 🏝️ 개요
| 항목 | 내용 |
|------|------|
| **언어 코드** | pisl (비공식) |
| **지역** | 콜롬비아 Providence Island (San Andrés, Providencia y Santa Catalina 제도) |
| **화자 수** | 농인 13명 + 청인 수백 명 (공유 수어) |
| **위치** | 서카리브해, 니카라과 동쪽, 콜롬비아령 |
| **발생 시기** | 19세기 후반-20세기 초반 (약 120년) |
| **상태** | 심각한 위기 (Critically Endangered) |

## 📖 역사적 배경

### 발견과 연구
- **1970s-80s**: William Washabaugh와 동료들 최초 체계적 연구
- **1978**: "Providence Island Sign: A Context-Dependent Language" 발표
- **2019**: Rehana Omardeen (트리니다드토바고 출신) ELDP 프로젝트로 재조사
- **특이점**: 세계 최초로 문서화된 마을 수어 중 하나

### 발생 조건
- **유전적 농**: 19세기 시작, 상염색체 열성 유전
- **고립된 섬**: 약 39km² (15평방마일)
- **고비율 농**: 1970년대 주민 140명당 1명 농인
- **근친혼**: 지리적 고립으로 인한 유전자 집중

## 👥 사용자 커뮤니티

### 인구 변화
| 시기 | 농인 수 | 총 인구 | 비율 |
|------|---------|---------|------|
| 1970년대 | 약 50명 | 약 3,000명 | 1:60 |
| 1980년대 | 약 20명 | 약 4,000명 | 1:200 |
| 2019년 | 13명 | 약 5,000명 | 1:385 |

### 공유 수어 특성
- 농인뿐 아니라 청인도 광범위하게 사용
- **특이성**: 농인들 사이 직접 대화 거의 없음
- 대부분의 대화가 농인-청인 사이에서 발생
- 청인이 중개자 역할
- 서부/동부 마을 간 변이 존재

### 사회적 위치
- Washabaugh 연구: 농인이 지적 능력 열등하다는 인식 존재
- 복잡한 아이디어 공유 제한
- 사회적으로 주변화된 위치
- 그러나 완전한 배제는 아님

## 🔤 언어학적 특성

### "Context-Dependent Language" (문맥 의존 언어)
PISL의 가장 주목할 특성:
- 발화가 물리적/사회적 문맥 없이는 모호함
- 문맥에서 분리되면 해석 불가능
- 어휘 항목 자체가 문맥에 의해 동기 부여
- 체계 내부 원칙만으로는 명확한 문장 대조 불가

### 제한된 손모양 체계
- 다른 수어에 비해 손모양 수가 적음
- 높은 도상성 (iconicity)
- 관습화 정도가 낮음

### 비수지 표지 사용
- 얼굴 표정, 입 모양 광범위하게 사용
- 문법적 기능 담당

### 어휘 특성
- 높은 어휘 변이
- 개인별/마을별 다른 수화
- 제한된 직접 농인-농인 상호작용으로 표준화 부족

### 공간 참조
- **절대적 공간 참조** (absolute spatial reference) 사용
- 다른 수어에서 드문 특성
- 섬의 지리와 연결된 고정 방향 체계

## 🌐 지역적 연결 가능성

### 카리브해 수어 네트워크 가설
- 자메이카 Country Sign과 어휘 비교 연구 제안
- Kajana SL (수리남)과의 관계 조사 필요
- 유전적/역사적 연결 가능성
- 카리브해 농인 집단 간 이동 역사

### 콜롬비아 수어와의 관계
- Colombian Sign Language (LSC)와 별개
- 섬이 언어적/문화적/민족적으로 본토와 구별됨
- 영어 기반 크레올어 사용 지역

## ⚠️ 위기 요인

### 인구학적 압력
- 농인 수 감소 (50명 → 13명)
- 총 인구 증가로 비율 급감
- 젊은 농인 부재
- 나이든 세대에 집중

### 언어적 압력
- Colombian Sign Language (LSC) 영향력 증가
- 교육에서 LSC 사용
- 젊은 세대 PISL 습득 기회 감소

### 교육적 압력
- 공식 농교육 없음
- 농아동의 본토 학교 진학
- PISL 세대 간 전승 단절

## 📚 문서화 프로젝트

### ELDP 프로젝트 (2019)
**연구자**: Rehana Omardeen
**기간**: 2019년 1-4월
**성과**:
- 12시간 비디오 녹화
- 농인 화자들의 자연스러운 대화
- 500개 어휘 데이터베이스
- ID-gloss 체계 개발
- ELAN 주석 1시간+

### 연구 의의
- 최초 마을 수어 연구 사례
- 언어 유형론적 가치
- 수어 발생 연구에 기여
- 공유 수어 (shared sign language) 개념 확립

## 🔬 학술적 가치

### 언어 발생 연구
- 자연 발생 수어의 구조 발달 관찰
- 청인 제스처에서 수어로의 전이
- 최소한의 농인-농인 상호작용에서의 언어

### 유형론적 시사점
- 마을 수어의 공통 특성 식별
- 언어 연령과 구조의 관계
- 공동체 규모와 언어 복잡성

## 📖 WIA Talk 표기법 접근

### 기록 전략
1. **문맥 의존성 처리**: 사용 상황 상세 기술
2. **변이 포용**: 개인별/마을별 변이 모두 수용
3. **비수지 요소 강조**: 얼굴, 입, 몸 동작 상세 기록
4. **공간 참조 체계**: 섬 지리와 연결된 방향 체계 문서화

### 문화적 민감성
- 농인 화자 직접 참여
- 청인 매개 상황 인식
- 사회적 맥락 기록
- 소규모 커뮤니티 존중
