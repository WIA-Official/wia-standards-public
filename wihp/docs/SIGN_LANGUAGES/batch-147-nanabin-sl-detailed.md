# Batch 147: Nanabin Sign Language (나나빈 수어)

## 기본 정보
- **언어명**: Nanabin Sign Language
- **약칭**: NanaSL
- **ISO 639-3**: 없음
- **국가**: 가나
- **지역**: Nanabin 마을
- **사용자 수**: 소규모 (3세대 가족)
- **언어 유형**: 가족/마을 수어
- **상태**: 미확인 (문서화 필요)

## 역사적 배경

### 기원
- 가나 Nanabin 마을
- 3세대에 걸친 청각장애 가족
- Adamorobe Sign Language와 별개

### 발견
- Adamorobe SL 연구 중 발견
- Victoria Nyst 연구팀
- AdaSL 코퍼스에 일부 기록

## 언어적 특징

### AdaSL과의 관계
- 별개의 수어
- 지역적 인접성
- 상호 이해 정도 미확인

### GSL과의 관계
- Ghanaian Sign Language와 다름
- 독자 발생

### 연구 필요
- 체계적 분석 없음
- 어휘, 문법 미확인

## 사회언어학적 특징

### 화자 현황
- 단일 확대 가족 중심
- 3세대 사용
- 정확한 숫자 미확인

### 사용 환경
- 가정 내 의사소통
- 마을 내 제한적 사용

## 위협 요인

### 잠재적 위협
1. GSL 영향
2. 소규모 화자 집단
3. 문서화 부재
4. 세대 간 전달 불확실

## 연구 현황

### 문서화
- AdaSL 코퍼스에 일부 포함
- 별도 체계적 연구 없음

### 필요 조치
1. 화자 확인 및 인터뷰
2. 어휘 수집
3. AdaSL과의 비교 분석
4. 독립 언어 지위 확인
