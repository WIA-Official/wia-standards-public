# Jamaican Sign Language (JSL)
## Jamaican Sign Language

### Overview
| Attribute | Detail |
|-----------|--------|
| **Official Name** | Jamaican Sign Language |
| **Native Name** | JSL |
| **ISO 639-3** | jls |
| **Country** | Jamaica |
| **Users** | ~7,500 (up to 200,000 estimated) |
| **Status** | Developing (5) |
| **Language Family** | ASL-related with BSL influence |

### Historical Development
| Year | Event |
|------|-------|
| 1939 | British man began teaching BSL to Jamaican deaf |
| 1958 | Caribbean Christian Centre for the Deaf (CCCD) founded |
| 1975 | Maranatha School for the Deaf (Virginia Mennonites) |
| 1990s | Jamaican Christian School for the Deaf (JCSD) |
| Present | Schools transitioning to JSL from ASL |

### Language Influences
- **British Sign Language**: Early influence (1939+)
- **American Sign Language**: North American missionaries
- **Signed English**: Used in some schools
- ~75% overlap with ASL reported by JAD

### Linguistic Features
- Handshape, orientation, location, movement
- Non-manual cues (facial expressions)
- WGH sign for sentence punctuation
- Own grammar distinct from English
- Classifiers instead of prepositions

### Regional Variation
- Kingston and Mandeville: Most prestigious, easiest to understand
- St. Elizabeth and Port Antonio: Most difficult to understand
- Variation by age, education, social networks

### Education System
- **JAD** (Jamaica Association for the Deaf) schools
- St. Christophers School for the Deaf
- Caribbean Christian Centre for the Deaf
- Maranatha School for the Deaf
- Jamaican Christian School for the Deaf
- Some schools use Total Communication philosophy

### Related Sign Languages
- **Jamaican Country Sign Language (Konchri Sain)**
  - Indigenous village sign language
  - St. Elizabeth parish, Top Hill origin
  - ~40 speakers remaining
  - Highly endangered
  - JSL used as L2 by Konchri Sain users

### Organizations
- Jamaica Association for the Deaf (JAD)
- Caribbean Christian Centre for the Deaf (CCCD)
- Deaf Bible Church (Top Hill)
- Jehovahs Witnesses deaf outreach

### Cultural Features
- Signs for Jamaican foods, places, concepts
- Reflects Jamaican culture and expressions
- Unique characteristics distinct from ASL
- Comparison with Jamaican Patois grammar
