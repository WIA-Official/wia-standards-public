# Batch 172: PNG Additional Sign Languages - PNG 추가 마을 수어

## 개요
PNG에는 알려진 것보다 훨씬 많은 마을 수어 존재
홈사인과 마을 수어 사이의 회색 지대

## 1. Mehek Sign Language

### 기본 정보
- **위치**: 북서부 PNG
- **민족**: Mehek 사람들
- **화자**: 약 100명 (추정)

### 특징
- 홈사인 또는 초기 마을 수어
- 2명 deaf (별개 공동체)
- 표준화 없음

### 언어적 특징
- 30개 일관된 수화
- 높은 신체 모방성
- 전신 수화 포함
- 반복 다양한 형태

### 연구
- Adam Hatfield 2016년 박사논문
- Glottolog 등재

## 2. Wanib Sign Language

### 기본 정보
- **위치**: Mehek SL 서쪽
- **구어**: Heyo어 공동체

### 특징
- Mehek SL과 다름
- 각각 구어 문법 반영
- 연구 필요
