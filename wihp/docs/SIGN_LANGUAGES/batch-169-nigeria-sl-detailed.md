# Batch 169: Nigeria Sign Languages - 나이지리아 수어 복합체

## 개요
나이지리아는 531개 구어 + 다수 수어 존재
ASL 기반 Nigerian SL 외 토착 수어들

## 1. Hausa Sign Language

### 기본 정보
- **하우사명**: Maganar Hannu / Harshen bebaye
- **위치**: Kano, 북부 나이지리아
- **유형**: 도시 공동체 수어

### 특징
- 독자 발생 토착 수어
- 청인 하우사 제스처와 연결
- Nigerian SL/ASL과 구별

### 연구
- Constanze Schmaling 2000년 박사논문
- 상세한 음운론, 형태론, 어휘 분석
- 어휘집 시리즈 제작 중

### 상태
- 활발 (Bickford 척도)
- 일부 학교에서 사용
- 성인 교육에서 사용
