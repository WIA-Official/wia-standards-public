# Sign Language Aspect
## Aspectual Marking in Sign Languages

### Overview
| Attribute | Detail |
|-----------|--------|
| **Topic** | Aspect in sign languages |
| **Function** | How action unfolds |
| **Method** | Movement modification |
| **Types** | Habitual, continuous, etc. |
| **Difference** | From tense |

### Introduction
Aspect in sign languages refers to how an action or state is viewed in terms of its internal structure—whether it's ongoing, completed, habitual, or momentary. Unlike tense (which locates events in time), aspect describes the nature of the action itself, expressed through modifications to verb movement.

### Aspect vs. Tense
| Concept | Function |
|---------|----------|
| Tense | When (past, present, future) |
| Aspect | How action unfolds |
| Example | "Ate" vs. "was eating" |
| Sign | Tense = time signs, Aspect = movement |

### Types of Aspect

#### Habitual Aspect
| Feature | Description |
|---------|-------------|
| Meaning | Regular, repeated action |
| Movement | Repeated, rhythmic |
| Example | "I exercise regularly" |
| Marking | Multiple movements |

#### Continuous Aspect
| Feature | Description |
|---------|-------------|
| Meaning | Ongoing action |
| Movement | Smooth, elongated |
| Example | "I am studying" |
| Marking | Extended duration |

#### Completive/Perfective
| Feature | Description |
|---------|-------------|
| Meaning | Completed action |
| Sign | FINISH often used |
| Movement | Sharp, definite |
| Example | "I have finished" |

#### Inceptive
| Feature | Description |
|---------|-------------|
| Meaning | Beginning of action |
| Sign | START, BEGIN |
| Focus | Starting point |
| Example | "Starting to learn" |

### Movement Modifications

#### Repeated Movement
| Type | Meaning |
|------|---------|
| Regular repetition | Habitual |
| Fast repetition | Intense/frequent |
| Slow repetition | Extended habit |

#### Elongated Movement
| Type | Meaning |
|------|---------|
| Slow, stretched | Continuous |
| Smooth arc | Duration |
| Held | Extended state |

#### Reduplication
| Pattern | Meaning |
|---------|---------|
| Verb-verb | Iterative |
| Movement copy | Repeated action |
| Various | Different aspects |

### Specific Aspect Markers

#### FINISH
| Function | Use |
|----------|-----|
| Completion | Action done |
| Perfect | Already happened |
| Position | After verb |
| Common | Very frequent |

#### CONTINUE/STILL
| Function | Use |
|----------|-----|
| Ongoing | Action continues |
| Duration | Extended time |
| Persistence | Still happening |

#### TEND-TO
| Function | Use |
|----------|-----|
| Habitual | Regular tendency |
| Disposition | Characteristic |
| Pattern | Repeated behavior |

### Aspect in Different Verb Types
| Verb Type | Aspect Marking |
|-----------|----------------|
| Plain verbs | Movement + NMM |
| Agreement verbs | Direction + movement |
| Classifier verbs | Movement modification |

### Iterative Aspect
| Feature | Description |
|---------|-------------|
| Meaning | Repeated instances |
| Movement | Multiple actions |
| Example | "Read books one after another" |
| Distinction | From habitual |

### Durative Aspect
| Feature | Description |
|---------|-------------|
| Meaning | Extended duration |
| Movement | Held, elongated |
| Example | "Stared for a long time" |
| Emphasis | Length of time |

### Punctual vs. Durative
| Type | Description |
|------|-------------|
| Punctual | Momentary, quick |
| Durative | Extended time |
| Movement | Sharp vs. prolonged |
| Contrast | Different meanings |
