### Introduction
Deaf-blind individuals use various adaptations of sign languages depending on their degree of vision and hearing loss. These range from close-range visual signing for those with tunnel vision to fully tactile signing for those without usable vision or hearing.

### Spectrum of Deaf-Blindness
| Type | Communication Method |
|------|----------------------|
| Usher Type 1 | Tactile signing |
| Usher Type 2 | Close visual, then tactile |
| Usher Type 3 | Progressive adaptation |
| Congenital deaf-blind | Multiple methods |
| Acquired | Varies by onset |

### Visual Field Adaptations

#### Tunnel Vision (Retinitis Pigmentosa)
| Adaptation | Description |
|------------|-------------|
| Smaller space | Sign in restricted area |
| Close range | Signer closer |
| Good lighting | Essential for seeing |
| Tracking | Signer holds position |

#### Central Vision Loss
| Adaptation | Description |
|------------|-------------|
| Peripheral | Sign off-center |
| Contrast | High contrast needed |
| Lighting | Specific requirements |
