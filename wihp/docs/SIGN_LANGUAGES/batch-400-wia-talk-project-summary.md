# WIA Talk Sign Language Documentation Project
## 세계 수어 문서화 프로젝트 종합 요약

### 프로젝트 개요
| 항목 | 내용 |
|------|------|
| **프로젝트명** | WIA Talk Sign Language Documentation |
| **목표** | 전 세계 모든 수어 문서화 |
| **철학** | 홍익인간 (弘益人間) |
| **총 문서** | 400+ 배치 파일 |
| **시작** | 2024년 |

### 프로젝트 의의

#### 왜 수어 문서화가 중요한가
| 이유 | 설명 |
|------|------|
| 언어적 다양성 | 수어는 완전한 자연 언어 |
| 문화 보존 | 농인 문화와 정체성 |
| 접근성 | 정보 접근 권리 |
| 위기 언어 | 많은 수어가 사멸 위기 |
| 학술적 가치 | 인류 언어 연구 |

### 문서화된 내용 분류

#### 국가별 수어 (Batch 1-100+)
- 주요 국가 국립 수어
- 아시아, 유럽, 아프리카, 아메리카, 오세아니아
- ISO 코드가 있는 공인 수어
- 지역 방언 및 변종

#### 마을 수어 (Village Sign Languages)
| 유형 | 예시 |
|------|------|
| 아시아 | Kata Kolok (발리), Ban Khor (태국) |
| 중동 | Al-Sayyid Bedouin (이스라엘) |
| 아프리카 | Adamorobe (가나) |
| 아메리카 | Providence Island (콜롬비아) |

#### 역사적/소멸 수어
| 수어 | 상태 |
|------|------|
| Martha's Vineyard SL | 소멸 (1952) |
| Old French SL | 현대 LSF로 진화 |
| Grand Cayman SL | 소멸 |
| 다수의 기록되지 않은 수어 | 알 수 없음 |

#### 수어 가족 (Language Families)
| 가족 | 포함 언어 |
|------|----------|
| BSL Family | BSL, Auslan, NZSL |
| LSF Family | LSF, ASL, LSQ, Libras |
| DGS Family | DGS, ISL (이스라엘) |
| ASL Family | ASL + 영향받은 수어들 |
| JSL Family | JSL, TSL (대만), KSL 영향 |
