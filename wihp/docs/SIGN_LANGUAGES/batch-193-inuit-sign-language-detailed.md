- Trade signs between Inuit language groups

## Unique Features

### Origins from Hunting Signs
- Silent communication during hunts (seal, caribou)
- Distance communication in open tundra
- Winter igloo close-quarters communication
- Proto-signs became foundation for deaf language

### Linguistic Characteristics
1. **Limited Non-Manual Simultaneity**: Unlike many sign languages
2. **Manual Simultaneity**: Present in IUR
3. **Verb Agreement**: All types (plain, agreement, spatial)
4. **Subject Unspecified**: Not subject-drop, but unspecified
5. **Plural Morphology**: 4 different methods
6. **Body-Anchored Signs**: Same form for singular/plural

### Cultural Values Reflected
- **Tunnganarniq**: Being welcoming and inclusive
- Community-wide signing by hearing relatives
- No stigma attached to deafness
- Deaf fully integrated in hunting, art, work

## Sociolinguistic Situation

### Current Status
| User Type | Language Use |
|-----------|--------------|
| Unschooled Deaf Adults | IUR primary |
| School-Educated Deaf | ASL dominant |
| Deaf Children Today | ASL (via interpreters) |
| Hearing Family Members | IUR (declining) |

### Threat Factors
1. **No IUR Schools**: Children sent south for ASL education
2. **ASL Interpreters**: Used in local schools
3. **No Formal IUR Training**: No interpreter programs
4. **Geographic Dispersion**: Isolated communities
5. **Young Generation**: No IUR acquisition

## Preservation Challenges

### Critical Issues
1. **Two-Generation Extinction Risk**: No child acquisition
2. **ASL Dominance**: Educational and service default
3. **No Interpreter Training**: Arctic College program needed
4. **Geographic Barriers**: Communities 1000s km apart
5. **Limited Funding**: Research and revitalization

### Revitalization Efforts
- CDRTI/NDS collaboration
- Video documentation
- Vocabulary materials
- Life story recordings
- Legislative Assembly interpretation
