### Introduction
Sign language corpus projects are large-scale efforts to document natural signing through video recordings. These corpora serve as resources for linguistic research, dictionary development, language technology, and historical preservation. Major projects exist for many sign languages worldwide.

### What Is a Sign Language Corpus?
| Component | Description |
|-----------|-------------|
| Video recordings | Native signers |
| Annotations | Linguistic coding |
| Metadata | Signer information |
| Searchable | Database access |
| Preserved | Long-term storage |
