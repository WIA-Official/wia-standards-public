# Cuban Sign Language (LSC / LSCub)
## Lengua de Senas Cubana

### Overview
| Attribute | Detail |
|-----------|--------|
| **Official Name** | Cuban Sign Language |
| **Native Name** | Lengua de Senas Cubana (LSC) |
| **ISO 639-3** | csf |
| **Country** | Cuba |
| **Users** | ~19,000 (109,000 deaf population) |
| **Status** | Official language (2024) |
| **Language Family** | French SL, Spanish, indigenous influences |

### Legal Recognition (2024)
- **September 17, 2024**: Council of State approved Decree Law
- **February 2025**: Published in Official Gazette
- President Miguel Diaz-Canel confirmed formalization
- First official recognition as natural language
- Prime Minister announced intent December 2023

### Historical Timeline
| Year | Event |
|------|-------|
| Late 1800s | First deaf school founded in Havana |
| Early 1900s | Oralist teaching methods dominated |
| Pre-1977 | Alfabeto manual antiguo developed informally |
| 1977 | Manual alphabet allowed in schools |
| 1978 | ANSOC (National Deaf Association) founded |
| 1994 | LSC introduced as pedagogical alternative |
| 2000 | Cochlear implants available |
| 2004 | Bilingual education model (Modelo Cubano) |
| 2024 | Official legal recognition |

### Linguistic Features
- Four distinctive elements:
  - Placement
  - Orientation  
  - Movement
  - Hand configuration
- Own grammatical and syntactical rules
- Distinct from Cuban Spanish
- Dactilema used for fingerspelling

### Manual Alphabet History
- **Alfabeto manual antiguo**: Old system, two-hands + face
- Used in schools from 1980s
- No longer widely used
- **Dactilema**: Current fingerspelling system

### Educational Models
1. **Total Communication**
2. **Bimodal**
3. **Bilingual** (LSC + Spanish literacy)

### Bilingual Education
- **2004**: Modelo Cubano de Educacion Bilingue launched
- LSC as first language
- Spanish (written) as second language
- Started in three schools, expanding nationally

### Healthcare System
- Early infant screening for hearing loss
- High-risk infant identification
- Hearing aids: 35 Cuban pesos (government subsidized)
- Cochlear implants since 2000
- ~250 children implanted since 2000
- Priority for Deaf-blind individuals

### Organizations
- **ANSOC** (Asociacion Nacional de Sordos de Cuba)
  - Founded 1978
  - Fights for equal rights
  - Facilitates bilingual education
- **ANSOC News** (Noticiero ANSOC): Only LSC news program
- **Centro de Superacion y Desarrollo del Sordo**
  - Located in San Miguel del Padron
  - Supports bilingualism advancement

### Cultural Significance
- Strong Deaf culture and community identity
- Shared history and experiences
- Solidarity among community members
- Wary of outsiders in Deaf spaces
- Cultural identity values promoted by law

### Decree Law Provisions
- Guarantees non-discriminatory information access
- Promotes bilingual education
- Full social inclusion
- Deaf cultural identity values
- Interpreter training requirements
- Literature and project development support

### Population Statistics
- ~52,660 people with hearing disability (Minsap 2022)
- ~109,000 deaf people total
- ~19,000 LSC users
