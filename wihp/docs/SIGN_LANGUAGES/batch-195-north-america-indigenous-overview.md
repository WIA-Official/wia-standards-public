# Batch 195: North American Indigenous Sign Languages - Overview

## Classification Summary

### Major Categories

#### 1. Plains Sign Language Complex (Hand Talk)
Regional lingua franca used by 37+ nations across Great Plains

| Regional Variety | Primary Users | Status |
|-----------------|---------------|--------|
| Plains Sign Talk | Crow, Cheyenne, Blackfoot | Endangered |
| Northeast Hand Talk | Oneida and others | Nearly Extinct |
| Great Basin SL | Ute and others | Unknown |
| Southwest Hand Talk | Navajo, Hopi, Apache | Declining |

#### 2. Arctic Sign Language
| Language | Region | Users | Status |
|----------|--------|-------|--------|
| Inuit Sign Language (IUR) | Nunavut, Canada | ~40 | Severely Endangered |

#### 3. Village/Family Sign Languages
| Language | Location | Status |
|----------|----------|--------|
| Keresan Sign Language | Zia Pueblo, NM | Endangered |
| Navajo Family Sign | Navajo Nation | Unknown |

#### 4. Mesoamerican Sign Languages
| Language | Location | Status |
|----------|----------|--------|
| Yucatec Maya SL | Yucatan, Mexico | Developing |
| Highland Maya SL | Guatemala | Endangered |
