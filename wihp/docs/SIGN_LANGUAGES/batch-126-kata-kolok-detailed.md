# Batch 126: Kata Kolok (발리 마을 수어)

## 기본 정보
- **언어명**: Kata Kolok (카타 콜록)
- **영문명**: Kata Kolok / Benkala Sign Language / Balinese Sign Language
- **ISO 639-3**: 없음 (미등록)
- **국가**: 인도네시아 (발리 북부)
- **지역**: Bengkala 마을 및 인근 마을 (Buleleng 군)
- **사용자 수**: 약 50명 deaf + 1,500명 이상 hearing (2011년 기준)
- **언어 유형**: 마을 수어 (Village Sign Language)
- **계통**: 독립 언어 (언어 격리)

## 역사적 배경

### 기원
- **발생 시기**: 5-7세대 전 (약 100-150년 전 추정)
- **원인**: MYO15A 유전자의 열성 돌연변이로 인한 유전성 청각장애
- **특징**: 마을 인구의 약 2.2%가 청각장애 (일반 인구 대비 50배 높음)

### 마을 전설
현지 전설에 따르면, 두 명의 마법사가 마을에서 싸우다가 서로를 저주하여 청각장애가 되었다고 함. "Bengkala"는 "숨을 곳"이라는 의미.

### 발전 과정
1. **1세대**: 청각장애 아동 출현, 가정 내 수화 발생
2. **2-3세대**: 마을 전체로 확산, 문법 구조 형성
3. **4-5세대**: 완전한 언어 체계로 발전
4. **현재**: 청각인과 청인 모두 사용하는 공유 수어

## 언어적 특징

### 음운론 (Phonology)
- **수형 (Handshape)**: 3가지 범주
  - 기본형 (Basic): 5, Curved-5, IX, B
  - 규칙형 (Regular): 중간 빈도
  - 제한형 (Limited): 단일 수화에만 사용
- **최소 대립쌍**: 거의 식별 불가 (음소 지위 불명확)
- **구순 동작**: 사용하지 않음 (mouthing 없음)
- **지문자**: 없음 (fingerspelling 없음)

### 공간 문법
- **절대 좌표계**: 실제 마을 위치를 기준으로 한 방향 체계 사용
- **추상적 공간 사용**: 제한적 (ASL과 다름)
- **지시 (Pointing)**: 실제 세계 위치를 가리킴

### 문법 구조
- **어순**: 유연함, 맥락 의존적
- **대명사**: 직접 지시 방식
- **존재/소유 표현**: 실제 위치 지시로 표현
- **부정**: 비수지 신호 (머리 흔들기) 사용

## 사회언어학적 특징

### 사용 환경
- **가정**: 일상 대화
- **직장**: 농업, 수도관 수리 등
- **종교**: 힌두 의례 참여 (청각장애 신에 대한 믿음)
- **교육**: 마을 초등학교에서 사용
- **예술**: Janger Kolok (청각장애 춤)

### 화자 분포
| 유형 | 인원 | 비고 |
|------|------|------|
| 청각장애 화자 | ~50명 | 원어민 |
| 청인 유창 화자 | ~200명 | 가족, 친구 |
| 청인 부분 화자 | ~1,300명 | 기본 의사소통 |

### 사회적 지위
- 청각장애인 완전 통합
- 결혼, 직업, 사회 활동에서 차별 없음
- 청각장애 아동이 청인 아동과 함께 교육
- 마을 고유 정체성의 상징

## 위협 요인

### 언어 위기 상태
- **상태**: 취약 (Vulnerable)
- **EGIDS**: 6a (활발) → 점차 약화 중

### 주요 위협
1. **BISINDO 영향**: 인도네시아 국가 수어의 확산
2. **교육 정책**: 청각장애 학교에서 BISINDO 사용
3. **결혼 패턴 변화**: 외부 결혼 증가로 청각장애 출생 감소
4. **도시화**: 젊은 세대의 도시 이주
5. **관광**: 외부 영향 증가

## 문화적 특징

### Janger Kolok (청각장애 춤)
- 1990년대 개발
- 16명의 청각장애 무용수 (16-72세)
- 월 3회 공연
- 발리 문화의 상징

### 종교적 역할
- 힌두 의례에서 특별한 역할
- 무덤 파기, 수도관 관리 등 전통적 임무
- 청각장애 신에 대한 공유 믿음
