## 1. Plains Indian Sign Language (PISL) / Hand Talk

### Basic Information
| Item | Content |
|------|---------|
| **Preferred Name** | Hand Talk |
| **Also Known As** | Plains Sign Talk, Plains Sign Language, First Nation Sign Language |
| **ISO 639-3** | psd |
| **Region** | Central Canada, Central/Western US, Northern Mexico |
| **Status** | ENDANGERED |

### Historical Scope
In 1885: **110,000+ "sign-talking Indians"** including:
- Blackfoot
- Cheyenne
- Sioux
- Kiowa
- Arapaho

### Historical Use
- **Lingua franca** across nations
- International relations
- Trade
- Diplomacy
- Story-telling
- Ceremonies

### Current Use
- Elders and deaf citizens
- Story-telling and oratory
- Various ceremonies
- Daily use by deaf
- Some deaf learn ASL later

### Regional Varieties (McKay-Cody Classification)

**Melanie R. McKay-Cody** (Cherokee Deaf woman, researcher):

| Regional Language | Area/Nations |
|-------------------|--------------|
| Northeast Hand Talk | Including Oneida Sign Language |
| Plains Sign Talk | Great Plains core |
| Great Basin SL | Ute and others |
| Southwest Hand Talk | Navajo, Hopi, Apache, Pueblo |
| West Coast | Chumash |
| (Possible relation) | Inuit Sign Language |

### Linguistic Features

#### Prosodic Structure
Unique prosodic units:
- Syllable-like packages/sub-packages
- Word-like individual signs
- Sentence-like phrases
- Paragraph-like utterances

#### Juncture System
Three types of junctures mark boundaries:
1. **Paragraph-final**: Hands crossed/folded over lower stomach (standing) or lap (sitting)
2. **Sentence-final**: Hands move toward final position but recoil
3. **Emphatic variant**: Strong clap or slap

### Historical Documentation
- Cabeza de Vaca (1527): First European record
- Coronado (1541): Early documentation
- 1930 Conference: Films by General Hugh L. Scott

### Connection to ASL?
**Jeffrey E. Davis** hypothesizes:
- Contact with Martha's Vineyard Sign Language
- May have influenced ASL development
- Possible partial ancestor of ASL

### Decline Causes
- European colonization
- American boarding schools
- Canadian residential schools
- Language suppression policies

### Revival Efforts
- Growing interest in 21st century
- Preservation work increasing
- Visibility improving
- Community-led revitalization

## 2. Keresan Sign Language (KPISL)

### Basic Information
| Item | Content |
|------|---------|
| **Also Known As** | Keresan Pueblo Indian Sign Language, Keresign |
| **Type** | Village Sign Language |
| **Location** | Keresan Pueblo (specific pueblo not publicly identified) |
| **Status** | Active |

### Community Profile
- Relatively high incidence of congenital deafness
- Both deaf and hearing use the language
- Integrated into Pueblo community

### Relationship to Hand Talk
- UNRELATED to Plains Hand Talk
- Developed independently
- Separate linguistic system

## Cultural Significance

### Not Just Communication
Hand Talk serves:
- Spiritual ceremonies
- Historical preservation
- Inter-tribal diplomacy
- Cultural identity
- Deaf community connection

### Church Latin Analogy
Historical formal register compared to Church Latin:
- Ceremonial use
- Cross-community intelligibility
- Prestige functions
