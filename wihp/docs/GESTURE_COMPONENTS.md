# 🧩 WIA Talk - 제스처 구성 요소 분해

**홍익인간 (弘益人間) - Benefit All Humanity**

---

## 📊 개요 (Overview)

WIA Talk의 93개 제스처는 **5가지 핵심 구성 요소**의 조합으로 이루어져 있습니다.

이는 수화 언어학(Sign Language Linguistics)의 **5개 파라미터 이론**을 기반으로 합니다:

1. **손 모양 (Handshape)**
2. **위치 (Location)**
3. **움직임 (Movement)**
4. **방향 (Orientation)**
5. **비수지 신호 (Non-manual Signals)**

---

## 🖐️ 1. 손 모양 (Handshape)

### 정의
손가락과 손바닥이 형성하는 형태입니다. WIA Talk의 가장 기본적인 구성 요소입니다.

### 손 모양 특징

#### 단순 손 모양 (Simple): 8종
- 주먹, 손바닥, 집게, V자, 삼지창, 4개, 검지, 원형

#### 복합 손 모양 (Complex): 10종
- 포용, 합장, 연결, 양손 펼침, 악수, 십자, X자, 평행선, 무한대, 나선

#### 지문 보호 원칙
✅ 모든 손 모양은 **손등 기반** 또는 **측면 기반**으로 설계
✅ 손바닥 지문이 카메라에 직접 노출되지 않음
✅ 프라이버시 보호와 자연스러움의 균형

#### 공간 영역 (Space Region): LC-09 ~ LC-12
- 움직임이 강조된 제스처
- 3D 공간 활용

## 🔄 3. 움직임 (Movement)

### 정의
제스처 실행 시 손의 궤적과 동작 패턴입니다.

### 기본 움직임 패턴 (15종)

| 코드 | 움직임 | 설명 | 속도 | 사용 제스처 예시 |
|------|--------|------|------|------------------|
| **MV-01** | 상승 (Up) | 아래→위 | 중간 (1-2초) | G-BM-01 (상승) |
| **MV-02** | 하강 (Down) | 위→아래 | 느림 (2-3초) | G-BM-02 (하강) |
| **MV-03** | 전진 (Forward) | 몸→앞 | 빠름 (0.5-1초) | G-BM-03 (전진) |
| **MV-04** | 후퇴 (Backward) | 앞→몸 | 느림 (1.5-2초) | G-BM-04 (후퇴) |
| **MV-05** | 확산 (Diverge) | 중심→바깥 | 중간 (1-2초) | G-BM-05 (확산) |
| **MV-06** | 수렴 (Converge) | 바깥→중심 | 중간 (1-2초) | G-BM-06 (수렴) |
| **MV-07** | 원형 (Circle) | 원 그리기 | 일정 (1초/회) | G-BM-07 (순환), G-NS-08 (8) |
| **MV-08** | 파동 (Wave) | 물결 모양 | 부드럽게 (2-3초) | G-BM-08 (파동) |
| **MV-09** | 급변 (Sharp) | 빠르고 날카롭게 | 매우 빠름 (<0.3초) | G-BM-09 (급변) |
| **MV-10** | 정지 (Still) | 움직임 없음 | - | G-BM-10 (평온) |
| **MV-11** | 좌우 (Horizontal) | 수평 이동 | 중간 (1-2초) | G-BH-06 (강조) |
| **MV-12** | 회전 (Rotate) | 손목 회전 | 중간 (1-2초) | G-DE-03 (시공간회전) |
| **MV-13** | 나선 (Spiral) | 나선형 이동 | 중간 (2-3초) | G-DE-03 (시공간회전) |
| **MV-14** | 반복 (Repeat) | 동작 반복 | 가변적 | G-BM-07 (순환) |
| **MV-15** | 페이드 (Fade) | 서서히 사라짐 | 느림 (3-4초) | G-DE-05 (투명모드) |

### 움직임 특징

#### 직선 움직임 (Linear): 6종
- 상승, 하강, 전진, 후퇴, 확산, 수렴

#### 곡선 움직임 (Curved): 5종
- 원형, 파동, 회전, 나선, 페이드

#### 특수 움직임 (Special): 4종
- 급변, 정지, 좌우, 반복

#### 속도 범위
- **매우 빠름**: < 0.5초 (긴급, 강조)
- **빠름**: 0.5-1초 (의지, 결단)
- **중간**: 1-2초 (일반적 표현)
- **느림**: 2-3초 (신중, 안정)
- **매우 느림**: > 3초 (명상, 평화)

## 🧭 4. 방향 (Orientation)

### 정의
손바닥과 손등이 향하는 방향입니다.

### 6방향 체계

| 코드 | 방향 | 설명 | 사용 제스처 예시 |
|------|------|------|------------------|
| **OR-01** | 전방 (Forward) | 손바닥→앞 | G-BH-04 (정중함) |
| **OR-02** | 후방 (Backward) | 손등→앞, 손바닥→뒤 | G-BH-03 (선택), G-BH-06 (강조) |
| **OR-03** | 상방 (Upward) | 손바닥→위 | G-BH-08 (포용), G-NS-06 (6) |
| **OR-04** | 하방 (Downward) | 손바닥→아래 | G-OS-02 (빼기) |
| **OR-05** | 측방-좌 (Left Side) | 손바닥→왼쪽 | G-BH-07 (화합) |
| **OR-06** | 측방-우 (Right Side) | 손바닥→오른쪽 | G-BH-07 (화합) |

### 방향 특징

#### 지문 보호 우선순위
1. **OR-02 (후방)**: 가장 안전 - 손등이 카메라를 향함 ✅
2. **OR-03 (상방)**: 안전 - 손바닥이 위를 향함 ✅
3. **OR-05/06 (측방)**: 안전 - 손바닥이 옆을 향함 ✅
4. **OR-04 (하방)**: 주의 - 손바닥이 아래를 향하지만 카메라 각도 고려
5. **OR-01 (전방)**: 사용 최소화 - 손바닥이 정면을 향함 ⚠️

#### 자연스러움
- 인간의 자연스러운 손 방향을 최대한 존중
- 억지스러운 방향 전환 최소화
- 편안한 손목 각도 유지

## 🔗 구성 요소 조합 시스템

### 제스처 = [손모양] + [위치] + [움직임] + [방향] + [비수지]

### 조합 예시

#### 예시 1: G-BH-01 (완성)
```
[손모양] HS-01 (주먹)
[위치]   LC-07 (가슴)
[움직임] MV-10 (정지)
[방향]   OR-02 (후방 - 손등 앞)
[비수지] NM-FE-15 (완성) 또는 NM-FE-07 (결의)

코드: HS01-LC07-MV10-OR02-FE15
```

#### 예시 2: G-BM-01 (상승)
```
[손모양] HS-02 (손바닥) 또는 기타
[위치]   LC-09 (중립 공간)
[움직임] MV-01 (상승)
[방향]   OR-03 (상방)
[비수지] NM-FE-11 (희망)

코드: HS02-LC09-MV01-OR03-FE11
```

#### 예시 3: G-NS-02 (숫자 2)

#### 예시 4: G-OS-01 (더하기)
