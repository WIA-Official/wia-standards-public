## 제1장: 서문 - 왜 WIA Braille인가

### 1.1 시각장애인의 현실

전 세계 **2억 8천만 명**의 시각장애인이 있습니다.
그들에게 문자는 **점자(Braille)**를 통해서만 접근 가능합니다.

그러나 현실은:

**첫째, 학습의 어려움**
- 기존 점자 학습: **3-5년** 소요
- 높은 중도 포기율
- 표준화 부재로 국가마다 다른 체계

**둘째, 접근성 한계**
- 점자 도서 부족 (전체 도서의 1% 미만)
- 점자 교육자 부족
- 디지털 시대에 뒤처진 기술

**셋째, 소통 단절**
- 청각장애인과 시각장애인 간 소통 불가
- 일반인과의 소통 장벽
- 고립된 언어 생태계

### 1.2 WIA Braille의 탄생

WIA Braille은 이 모든 한계를 넘어서기 위해 창제되었습니다.

**핵심 혁신:**

1. **IPA 기반 표준화**: 국제음성기호(IPA)를 매개로 전 세계 언어 통합
2. **WIA Talk 연동**: 시각장애인과 청각장애인의 완전한 소통
3. **학습 시간 단축**: 3-5년 → **1개월**
4. **5대 동등 표현**: 한글 = IPA = 점자 = 제스처 = 음성

### 1.3 5대 동등 표현 원칙

```
┌─────────────────────────────────────────────────────┐
│                                                      │
│    한글     IPA      점자     제스처     음성        │
│     ↓       ↓        ↓        ↓        ↓           │
│    ㅎ   →   h    →   ⠓   →   🤚   →   /h/         │
│     ↑       ↑        ↑        ↑        ↑           │
│                                                      │
│            모든 표현은 동등하다                       │
│            어느 것도 기본이 아니다                    │
│                                                      │
└─────────────────────────────────────────────────────┘
```

## 제2장: 루이 브라유 정신의 계승

### 2.1 루이 브라유 (1809-1852)

![placeholder_louis_braille](images/louis_braille.png)
<!-- 루이 브라유 초상화 스케치 -->

**루이 브라유(Louis Braille)**는 15세에 6점 점자 시스템을 발명했습니다.

| 항목 | 내용 |
|------|------|
| 출생 | 1809년 1월 4일, 프랑스 쿠프브레 |
| 실명 | 3세 (사고로 인한 실명) |
| 발명 | 15세 (1824년) |
| 완성 | 20세 (1829년) |
| 사망 | 1852년 1월 6일 (43세) |

### 2.2 6점 점자의 천재성

브라유가 발명한 6점 시스템은 **수학적 완벽성**을 가집니다:

```
점자 셀 구조:
┌───┬───┐
│ 1 │ 4 │
├───┼───┤
│ 2 │ 5 │
├───┼───┤
│ 3 │ 6 │
└───┴───┘

가능한 조합: 2⁶ = 64가지
```

**64가지 조합**으로:
- 알파벳 26자
- 숫자 10개
- 특수문자
- 악보 기호
- 수학 기호

모두 표현 가능합니다.

### 2.3 200년 후의 계승

| 구분 | 루이 브라유 (1824) | WIA Braille (2025) |
|------|-------------------|-------------------|
| 목적 | 시각장애인 문해력 | 전 인류 통합 소통 |
| 기반 | 프랑스어 알파벳 | IPA (국제음성기호) |
| 범위 | 서양 언어 | 전 세계 모든 언어 |
| 연동 | 없음 | WIA Talk 완전 연동 |
| 학습 | 수년 | 1개월 |

## 제3장: 5대 동등 표현 철학

### 3.1 철학의 핵심

> **"ALL FIVE ARE EQUAL. NO DEFAULT EXISTS."**
> (다섯 표현 모두 동등하다. 기본은 존재하지 않는다.)

이것이 WIA 시스템의 근본 철학입니다.

### 3.2 5대 표현 체계

| 번호 | 표현 | 설명 | 사용자 |
|------|------|------|--------|
| 1 | **한글** | 문자 표현 | 일반인 |
| 2 | **IPA** | 음성학적 표준 | 언어학자, AI |
| 3 | **점자** | 촉각 표현 | 시각장애인 |
| 4 | **제스처** | 시각 표현 | 청각장애인 |
| 5 | **음성** | 청각 표현 | 일반인 |

### 3.3 동등성의 의미

```
전통적 관점:
한글 (기본) → 점자 (변환) → 음성 (출력)

WIA 관점:
한글 ═══ IPA ═══ 점자 ═══ 제스처 ═══ 음성
      ↑        ↑        ↑         ↑
      모든 방향으로 동등하게 변환 가능
```

어떤 표현에서 시작하든, 어떤 표현으로든 변환 가능합니다.
**시작점도 없고, 끝점도 없습니다.**

### 3.4 실제 예시: "안녕"

| 표현 | 값 |
|------|-----|
| 한글 | 안녕 |
| IPA | /an.njʌŋ/ |
| 점자 | ⠁⠈⠝⠈⠝⠬⠖⠈⠿ |
| 제스처 | 👋 + 🤝 |
| 음성 | [안녕] (음파) |

다섯 가지 모두 **"안녕"**이라는 **동일한 의미**를 담고 있습니다.

## 제4장: 점자의 기본 구조

### 4.1 점자 셀 (Braille Cell)

모든 점자는 **6개의 점**으로 이루어진 셀로 구성됩니다.

```
표준 점자 셀:

    ●(1)  ●(4)
    ●(2)  ●(5)
    ●(3)  ●(6)

점 번호:
- 왼쪽 열: 1, 2, 3 (위에서 아래)
- 오른쪽 열: 4, 5, 6 (위에서 아래)
```

### 4.2 점자 표기법

| 유니코드 | 점 구성 | 의미 |
|----------|---------|------|
| ⠀ | (없음) | 빈 칸 |
| ⠁ | 1 | 점 1만 |
| ⠃ | 1,2 | 점 1,2 |
| ⠇ | 1,2,3 | 점 1,2,3 |
| ⠿ | 1,2,3,4,5,6 | 모든 점 |

### 4.3 WIA Braille의 확장

WIA Braille은 기존 6점 시스템을 **IPA 기반으로 재정의**합니다.

```
기존 점자:        WIA Braille:
A = ⠁            /a/ = ⠁
B = ⠃            /p/ = ⠏
C = ⠉            /tɕ/ = ⠵
```

**음성학적 정확성**을 기반으로 전 세계 모든 언어를 표현합니다.

## 제7장: 종성 자음 (27개)

### 7.1 종성의 특징

한글 종성(받침)은 초성과 다른 발음 규칙을 가집니다.

**핵심 원리: 7종성 법칙**

27개 종성 표기가 있지만, 실제 발음은 **7가지**로 줄어듭니다:

```
ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ
```

### 7.2 종성 점자 표기

종성은 **앞에 표지**를 붙여 구분합니다:
- ⠁ (점 1): 불파음 표지
- ⠈ (점 4): 비음/유음 표지

| 한글 | IPA | 점자 | 대표음 | 설명 |

| (없음) | - | - | - | 받침 없음 |
| ㄱ | k̚ | ⠁⠨ | ㄱ | 연구개 불파음 |
| ㄲ | k͈̚ | ⠁⠨⠨ | ㄱ | 연구개 불파음 |
| ㄳ | k̚ | ⠁⠨ | ㄱ | (ㄱ으로 발음) |
| ㄴ | n | ⠈⠝ | ㄴ | 치경 비음 |
| ㄵ | n | ⠈⠝ | ㄴ | (ㄴ으로 발음) |

| ㄾ | l | ⠈⠗ | ㄹ | (ㄹ으로 발음) |
| ㄿ | p̚ | ⠁⠏ | ㅂ | (ㅂ으로 발음) |
| ㅀ | l | ⠈⠗ | ㄹ | (ㄹ으로 발음) |
| ㅁ | m | ⠈⠍ | ㅁ | 양순 비음 |

```
┌─────────────────────────────────────────────────────┐
│                    7종성 법칙                        │
├───────┬─────────────────────────────────────────────┤
│  ㄱ   │  ㄱ, ㄲ, ㅋ, ㄳ, ㄺ                          │
│  ㄴ   │  ㄴ, ㄵ, ㄶ                                  │
│  ㄷ   │  ㄷ, ㅅ, ㅆ, ㅈ, ㅊ, ㅌ, ㅎ                   │
│  ㄹ   │  ㄹ, ㄽ, ㄾ, ㅀ                              │
│  ㅁ   │  ㅁ, ㄻ                                      │
│  ㅂ   │  ㅂ, ㅍ, ㄼ, ㄿ, ㅄ                          │
│  ㅇ   │  ㅇ                                          │
└───────┴─────────────────────────────────────────────┘
```

### 7.4 불파음 vs 비음

**불파음 (⠁ 표지)**: 공기 방출 없이 조음 위치에서 막힘
- /k̚/: 연구개에서 막힘 (각, 국)
- /t̚/: 잇몸에서 막힘 (닫, 옷)
- /p̚/: 입술에서 막힘 (밥, 입)

**비음/유음 (⠈ 표지)**: 코나 혀 옆으로 공기 흐름
- /n/: 잇몸 + 코 (안, 산)
- /m/: 입술 + 코 (감, 점)
- /ŋ/: 연구개 + 코 (강, 정)
- /l/: 잇몸 + 혀 옆 (달, 물)

## 제8장: 이중모음 (11개)

### 8.1 이중모음 체계

이중모음은 **반모음 + 단모음**의 결합입니다.

| 한글 | IPA | 점자 | 구성 | 예시 |
|------|-----|------|------|------|
| ㅑ | ja | ⠬⠁ | j + a | 야구 |
| ㅕ | jʌ | ⠬⠖ | j + ʌ | 여우 |
| ㅛ | jo | ⠬⠕ | j + o | 요리 |
| ㅠ | ju | ⠬⠥ | j + u | 유리 |
| ㅖ | jɛ | ⠬⠑ | j + ɛ | 예절 |
| ㅘ | wa | ⠺⠁ | w + a | 과일 |
| ㅝ | wʌ | ⠺⠖ | w + ʌ | 원숭이 |
| ㅙ | wɛ | ⠺⠌ | w + æ | 왜가리 |
| ㅞ | we | ⠺⠑ | w + e | 웨딩 |
| ㅟ | wi | ⠺⠊ | w + i | 위험 |
| ㅢ | ɰi | ⠳⠊ | ɰ + i | 의자 |

### 8.2 반모음

| 반모음 | IPA | 점자 | 설명 |
|--------|-----|------|------|
| j | j | ⠬ | 경구개 접근음 ("이" 계열) |
| w | w | ⠺ | 양순 연구개 접근음 ("우" 계열) |
| ɰ | ɰ | ⠳ | 연구개 접근음 ("으" 계열) |

### 8.3 이중모음 조합 규칙

```
j (⠬) 계열:        w (⠺) 계열:        ɰ (⠳) 계열:
j + a = ja (ㅑ)    w + a = wa (ㅘ)    ɰ + i = ɰi (ㅢ)
j + ʌ = jʌ (ㅕ)    w + ʌ = wʌ (ㅝ)
j + o = jo (ㅛ)    w + æ = wɛ (ㅙ)
j + u = ju (ㅠ)    w + e = we (ㅞ)
j + ɛ = jɛ (ㅖ)    w + i = wi (ㅟ)
```

## 제11장: IPA → 점자 변환

### 11.1 변환 알고리즘

```
IPA 입력
    ↓
모음/자음 분류
    ↓
초성/중성/종성 판별
    ↓
해당 점자 매핑
    ↓
점자 출력
```

### 11.2 점자 조합 규칙

**1. 초성 + 중성** (받침 없음)
```
가: ㄱ + ㅏ = ⠨ + ⠁ = ⠨⠁
```

**2. 초성 + 중성 + 종성** (받침 있음)
```
간: ㄱ + ㅏ + ㄴ = ⠨ + ⠁ + ⠈⠝ = ⠨⠁⠈⠝
```

**3. 이중모음 포함**
```
과: ㄱ + ㅘ = ⠨ + ⠺⠁ = ⠨⠺⠁
```

## 제12장: 음운 규칙과 연음

### 12.1 연음 법칙

받침이 다음 음절 초성으로 연결되는 현상

```
읽어요 → 일거요
/ilk.ʌ.jo/ → /il.gʌ.jo/
```

### 12.2 비음화

파열음이 비음 앞에서 비음으로 변함

```
국물 → 궁물
/kuk.mul/ → /kuŋ.mul/
```

### 12.3 경음화

평음이 경음으로 변하는 현상

```
학교 → 학꾜
/hak.kjo/ → /hak.k͈jo/
```

### 12.4 WIA Braille의 처리

WIA Braille은 **표기 기준**과 **발음 기준** 두 가지 모드를 지원합니다:

| 모드 | "읽어요" 표기 | 용도 |
|------|--------------|------|
| 표기 기준 | ⠁⠈⠗⠨⠖⠬⠕ | 학습, 문서 |
| 발음 기준 | ⠁⠈⠗⠨⠖⠬⠕ | 음성 합성, TTS |

## 제13장: 점자-제스처 매핑

### 13.1 통합 원리

WIA Braille과 WIA Talk는 **6점 체계**를 공유합니다.

```
점자 6점              손 6영역
┌───┬───┐            ┌───────────────┐
│ 1 │ 4 │            │  엄지   새끼  │
├───┼───┤     ═══    ├───────────────┤
│ 2 │ 5 │            │  검지   약지  │
├───┼───┤            ├───────────────┤
│ 3 │ 6 │            │  중지   중지  │
└───┴───┘            └───────────────┘
```

### 13.2 점자 → 제스처 변환

각 점자 패턴은 손가락 조합으로 변환됩니다:

| 점자 | 점 구성 | 손가락 | 의미 |
|------|---------|--------|------|
| ⠁ | 1 | 엄지 | 모음 /a/ |
| ⠃ | 1,2 | 엄지+검지 | - |
| ⠇ | 1,2,3 | 엄지+검지+중지 | - |
| ⠿ | 1,2,3,4,5,6 | 전체 | ㅇ (종성) |

### 13.3 실시간 양방향 변환

```
시각장애인 (점자)              청각장애인 (제스처)
      ↓                              ↓
   점자 입력                      제스처 입력
      ↓                              ↓
    IPA 변환      ←═══════════→    IPA 변환
      ↓                              ↓
   제스처 출력                     점자 출력
      ↓                              ↓
청각장애인이 봄                시각장애인이 읽음
```

## 제14장: 양손 시스템

### 14.1 기본 구조

| 손 | 역할 | 사용자 |
|----|------|--------|
| **왼손** | 점자 입력/출력 | 시각장애인 |
| **오른손** | 제스처 입력/출력 | 청각장애인 |

### 14.2 왼손 점자 영역

```
왼손 (점자 담당):

      손등 방향
         ↑
   [새끼][약지][중지][검지][엄지]
     6     5     4     3    2    1
         (점자 점 번호 매핑)
```

| 손가락 | 점자 점 | 영역 |
|--------|---------|------|
| 엄지 | 1 | 좌상 |
| 검지 | 2 | 좌중 |
| 중지 | 3 | 좌하 |
| 약지 | 4 | 우상 |
| 새끼 | 5 | 우중 |
| 손바닥 | 6 | 우하 |

### 14.3 오른손 제스처 영역

```
오른손 (제스처 담당):

      손등 방향 (보안!)
           ↑
   [엄지][검지][중지][약지][새끼]
     HS    MV    LC    OR    FE
         (컴포넌트 코드 매핑)
```

| 손가락 | 컴포넌트 | 의미 |
|--------|----------|------|
| 엄지 | HS | 손모양 |
| 검지 | MV | 움직임 |
| 중지 | LC | 위치 |
| 약지 | OR | 방향 |
| 새끼 | FE | 표정 |

### 14.4 통합 사용 예시

**"안녕" 표현:**

| 단계 | 왼손 (점자) | 오른손 (제스처) |
|------|-------------|-----------------|
| 1 | ⠁⠈⠝ (안) | 👋 (인사) |
| 2 | ⠈⠝⠬⠖⠈⠿ (녕) | 🤝 (만남) |

시각장애인은 왼손으로 점자를 읽고, 청각장애인은 오른손 제스처를 봅니다.
**동시에 완전한 소통!**

## 제15장: 실시간 변환

### 15.1 변환 파이프라인

```
입력 (5가지 중 택1)
        ↓
    ┌───────────────────────────────────┐
    │           WIA 변환 엔진            │
    │  ┌─────┐  ┌─────┐  ┌─────┐       │

    │  └─────┘  └─────┘  └─────┘       │
    │      ╚═══════╦═══════╝           │
    │  ┌─────┐     ║     ┌─────┐       │
    │  │제스처│═════╬═════│음성 │       │
    │  └─────┘           └─────┘       │
    └───────────────────────────────────┘
        ↓
```

| IPA → 점자 | < 1ms | 즉시 |
| 점자 → 제스처 | < 5ms | 매핑 |
| 제스처 → 음성 | < 50ms | TTS |
| 전체 경로 | < 100ms | 실시간 |

**URL**: https://wia.live/bidirectional/

- 제스처 입력 → 텍스트/점자/음성 출력

### 16.2 FEEL (1일) - 촉감 익히기

**목표**: 6점 점자의 물리적 감각 익히기

**학습 내용:**
- 점자 셀 구조 이해
- 6개 점 위치 암기
- 손끝 민감도 훈련

**연습:**
```
⠀ (빈칸) - 아무 점 없음
⠿ (전체) - 모든 점 있음
⠁ (점1) - 좌상단 점
⠈ (점4) - 우상단 점
```

### 16.3 FIND (1주) - 기본 자모

**목표**: 한글 자모 40개 매핑

**1일차**: 모음 10개
```
ㅏ(⠁) ㅓ(⠖) ㅗ(⠕) ㅜ(⠥) ㅡ(⠳)
ㅣ(⠊) ㅐ(⠌) ㅔ(⠑) ㅚ(⠪) ㅟ(⠬)
```

**2-3일차**: 기본 자음 9개
```
ㄱ(⠨) ㄴ(⠝) ㄷ(⠞) ㄹ(⠗) ㅁ(⠍)
ㅂ(⠏) ㅅ(⠎) ㅇ(⠿) ㅎ(⠓)
```

**4-5일차**: 격음/경음 10개
```
ㅋ(⠅) ㅌ(⠇) ㅍ(⠋) ㅊ(⠟)
ㄲ(⠨⠨) ㄸ(⠞⠞) ㅃ(⠏⠏) ㅆ(⠎⠎) ㅉ(⠵⠵)
ㅈ(⠵)
```

**6-7일차**: 이중모음 11개
```
ㅑ(⠬⠁) ㅕ(⠬⠖) ㅛ(⠬⠕) ㅠ(⠬⠥)

ㅖ(⠬⠑) ㅟ(⠺⠊) ㅢ(⠳⠊)
```

**목표**: 초성+중성+종성 조합

### 16.5 FORM (1주) - 문장 구성

**목표**: 완전한 문장 읽고 쓰기

**연습 문장:**
```
안녕하세요.
⠁⠈⠝⠝⠬⠖⠈⠿⠓⠁⠎⠑⠬⠕⠲

감사합니다.
⠨⠁⠈⠍⠎⠁⠓⠁⠈⠍⠝⠊⠞⠁⠲

⠏⠁⠈⠝⠨⠁⠈⠏⠎⠳⠈⠍⠝⠊⠞⠁⠲
```

### 16.6 FREE (1주) - 자유 소통

**목표**: WIA Talk 연동 + 실시간 소통

**실습:**
- WIA Live 플랫폼 사용
