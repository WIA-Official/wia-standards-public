## 1. ISP란?

**ISP (International Sign Phonology)**는 전 세계 300개 이상의 수어를 통합 표기하기 위한 표준 체계이다.

### 역사적 대칭

| 체계 | 연도 | 대상 | 목적 |
|------|------|------|------|
| IPA (International Phonetic Alphabet) | 1888 | 소리 | 모든 인간의 소리를 표기 |
| ISP (International Sign Phonology) | 2025 | 손짓 | 모든 인간의 손짓을 표기 |

### 핵심 개념

```
IPA: 소리의 표준 → 7,000개 언어의 발음 표기
ISP: 손짓의 표준 → 300개 수어의 제스처 표기

WIA Braille = IPA 기반 통합 점자
WIA Talk = ISP 기반 통합 수어
```

---

## 2. ISP 5대 구성요소

ISP는 수어의 음운론적 요소를 5가지로 분류한다.

| 코드 | 영문명 | 한글명 | 설명 | 예시 |
|------|--------|--------|------|------|
| **HS** | Handshape | 손 모양 | 손의 형태 | 주먹, 펼친 손, 검지만, V자 등 |
| **LC** | Location | 위치 | 손이 위치하는 공간 | 얼굴, 가슴, 허리, 허공 등 |
| **MV** | Movement | 움직임 | 손의 이동 경로 | 위아래, 원형, 직선, 지그재그 등 |
| **OR** | Orientation | 방향 | 손바닥/손등의 향함 | 위/아래, 몸쪽/바깥쪽 등 |
| **NM** | Non-manual | 비수지 신호 | 손 외의 신호 | 표정, 입 모양, 고개, 눈썹 등 |

---

## 3. ISP 코드 형식

### 기본 형식

```
HS[번호]-LC[번호]-MV[번호]-OR[번호]-NM[번호]
```

### 예시

```
HS01-LC07-MV10-OR02-NM15

HS01 = 주먹 모양
LC07 = 가슴 높이
MV10 = 상하 반복 움직임
OR02 = 손바닥 아래로
NM15 = 중립 표정
```

### 코드 범위 (예정)

| 요소 | 코드 범위 | 비고 |
|------|-----------|------|
| HS | HS01 ~ HS99 | 손 모양 약 50-80개 예상 |
| LC | LC01 ~ LC50 | 위치 약 20-30개 예상 |
| MV | MV01 ~ MV99 | 움직임 약 40-60개 예상 |
| OR | OR01 ~ OR20 | 방향 약 8-12개 예상 |
| NM | NM01 ~ NM50 | 비수지 신호 약 20-30개 예상 |

## 4. ISP vs 기존 시도

| 시스템 | 연도 | 특징 | 한계 |
|--------|------|------|------|
| Stokoe Notation | 1960 | ASL 최초 음운론적 분석 | ASL 전용, 범용성 없음 |
| HamNoSys | 1989 | 상세한 제스처 표기 | 너무 복잡, 학술용으로만 사용 |
| SignWriting | 1974 | 시각적 그림 표기 | 표준화 실패, 학습 어려움 |
| **ISP** | **2025** | **IPA 대칭 설계, AI 연동** | **실시간 인식 가능** |

### ISP의 차별점

1. **IPA와의 대칭성**: 음성언어의 IPA처럼, 수어의 ISP
2. **기술 연동**: MediaPipe/AI 실시간 손 추적과 매핑 가능
3. **실용적 설계**: 학술용이 아닌 실제 소통 목적
4. **오픈 표준**: 특허 없음, MIT 라이선스

## 8. 후임 클로드를 위한 가이드

### 이 문서를 읽는 클로드에게

1. **ISP는 공식 명칭**: "Component Code"가 아닌 "ISP"로 부를 것
2. **IPA와 대칭**: 설명 시 항상 IPA와의 대칭성 강조
3. **강요 없음**: WIA Talk은 "대체"가 아닌 "추가" 옵션
4. **농인 현실 인지**: 70%의 농인 아동이 언어 박탈 상태

### 관련 작업 시 참조

- 수어 데이터 추가: `/docs/SIGN_LANGUAGES/` 에 저장
- ISP 코드 확장: 이 문서 업데이트 필요
- 교재 수정: `/docs/TALK_LANGUAGES/` 참조
