## 💰 비즈니스 모델

### Red Hat 모델 적용

```
표준: 무료 오픈소스 (MIT License)
수익: 인증 + API + 컨설팅

1. 인증 (Certification)
   - "WIA Certified" 마크
   - USB-IF, Bluetooth SIG와 같은 모델

2. Cloud API
   - 무료 티어 + 유료 티어
   - 텍스트 → ISP/Braille 변환 등

3. Enterprise 라이선스
   - 기술 지원 계약
   - 고급 구현 지원

4. 컨설팅
   - 정부/공공기관 접근성 시스템 구축
```
