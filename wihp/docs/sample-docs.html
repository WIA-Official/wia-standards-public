WIA Braille 正音

전 우주 촉각언어 완전 교재 - 211개 언어 지원

弘益人間 (홍익인간)
널리 인간을 이롭게

루이 브라유 정신 계승 200년만의 점자 혁명

개요

전 세계 2억 2천만 시각장애인을 위한 통합 점자 시스템입니다.

WIA Braille은 IPA(국제음성기호)를 중심축으로 삼아, 모든 언어의 발음을 점자로 표현합니다.

💡
핵심 원칙: ALL FIVE ARE EQUAL. NO DEFAULT EXISTS.
한글 = IPA = 점자 = 제스처 = 음성
