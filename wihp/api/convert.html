WIHP API

IPA to Hangul Phonology Converter API

Endpoint

GET /api/convert?ipa={ipa}&lang={lang}

Parameters

Parameter Type Description Example
ipa string IPA transcription /həˈloʊ/
lang string ISO 639-1 language code en, ko, ja
word string Word (optional, for lookup) hello

Response

{ "success": true, "input": { "ipa": "/həˈloʊ/", "language": "en" }, "output": { "hangul": "헐로우", "braille": "⠓⠑⠇⠇⠕", "wihp_code": ["PL11-MN05-AR01", "VL02-VH02-VR01", ...] } }

Example Requests

# cURL curl "https://wihp.wiatalk.com/api/convert?ipa=/həˈloʊ/&lang=en" # Python import requests r = requests.get("https://wihp.wiatalk.com/api/convert", params={"ipa": "/həˈloʊ/", "lang": "en"}) print(r.json()["output"]["hangul"]) # 헐로우 # JavaScript fetch("/api/convert?ipa=/həˈloʊ/&lang=en") .then(r => r.json()) .then(d => console.log(d.output.hangul)); // 헐로우
