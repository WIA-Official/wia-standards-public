## 2. 특수 음운 현상

### 2.1 Ы 소실
우크라이나어와 달리 ы 없음:
```
우크라이나어: сын [sɪn]
루신어: син [sɪn] "아들"
```

### 2.2 І와 И 구별
두 모음 모두 사용:
```
і [i] - 경모음
и [ɪ] - 연모음
```

### 2.3 Ґ와 Г 구별
```
ґ [ɡ] - 경구개음
г [ɦ] - 유성 성문 마찰음
```

### 2.4 방언 다양성
```
동부 (우크라이나): ся [sʲa] "자신"
서부 (슬로바키아): ше [ʃe] "자신"
남부 (세르비아): са [sa] "자신"
```
