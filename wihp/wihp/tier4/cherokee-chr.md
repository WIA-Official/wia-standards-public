## 2. 특수 음운 현상

### 2.1 음절 문자 체계
체로키 음절 문자는 85개의 글자로 구성되며, 각 글자가 자음+모음 조합을 나타냅니다:
```
Ꭰ (a), Ꭱ (e), Ꭲ (i), Ꭳ (o), Ꭴ (u), Ꭵ (v)
Ꭶ (ga), Ꭷ (ka), Ꭸ (ge), Ꭹ (gi), Ꭺ (go), Ꭻ (gu), Ꭼ (gv)
Ꭽ (ha), Ꭾ (he), Ꭿ (hi), Ꮀ (ho), Ꮁ (hu), Ꮂ (hv)
```

### 2.2 성조 체계
체로키어는 고저 성조 언어입니다:
```
고성조: ́ (고음)
저성조: ̀ (저음)
하강성조: ̂ (고→저)
```

### 2.3 순음 부재
/p/, /b/, /f/, /v/ 음소가 없음 - 북아메리카 원주민 언어 중 특이한 특징

### 2.4 음절 구조
주로 CV (자음+모음) 구조, CVC는 제한적

## 9. 문법 특징

### 9.1 다인칭 체계
체로키어는 주어와 목적어를 동사에 모두 표시하는 복잡한 인칭 체계를 가집니다.

### 9.2 애니메이트/이나니메이트 구별
생물과 무생물에 따라 문법이 달라집니다.

### 9.3 방향성 접두사
동작의 방향을 동사에 표시:
```
ᎠᏆᏙᎵᎭ (atsvaloli'ha) - 밖으로 가다
ᎤᎳᏫᏛᎦ (ulawisvga) - 안으로 가다
```
