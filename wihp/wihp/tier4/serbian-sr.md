## 3. 특수 음운 현상

### 3.1 에카브 vs 이에카브 (Ekavian vs Ijekavian)
세르비아 표준어는 에카브 방언을 사용합니다:
```
고대 슬라브어 ě → e (세르비아)
                → ije/je (크로아티아)

예:
세르비아: млеко [mleko] (우유)
크로아티아: mlijeko [mlijeko]

세르비아: бело [belo] (하얀)
크로아티아: bijelo [bijelo]
```

### 3.2 성조 체계 (Pitch Accent)
세르비아어는 4가지 성조를 구분합니다:
| 기호 | 명칭 | 설명 |
|------|------|------|
| ˋ | 단하강 | 짧고 내리는 |
| ˊ | 단상승 | 짧고 올라가는 |
| ˆ | 장하강 | 길고 내리는 |
| ˜ | 장상승 | 길고 올라가는 |

### 3.3 무성화 (Final Devoicing)
어말 유성 자음은 무성화됩니다:
```
град [ɡrat] - 도시
хлеб [xlep] - 빵
```

## 10. 문법 참고 (Grammar Notes)

### 10.1 격 체계 (7격)
세르비아어는 7개의 격을 사용합니다:
1. **Nominativ** (주격): Ко? Шта?
2. **Genitiv** (속격): Кога? Чега?
3. **Dativ** (여격): Коме? Чему?
4. **Akuzativ** (대격): Кога? Шта?
5. **Vokativ** (호격): 부르기
6. **Instrumental** (구격): Киме? Чиме?
7. **Lokativ** (처소격): О коме? О чему?

### 10.2 성 (Gender)
- **Мушки род** (남성): студент, град
- **Женски род** (여성): жена, школа
- **Средњи род** (중성): дете, море

### 10.3 미래형
세르비아어는 да+현재형을 선호합니다:
```
Ја ћу да радим (나는 일할 것이다)
vs 크로아티아: Ja ću raditi
```
