## 2. 특수 음운 현상

### 2.1 T/K 교체 (T-style vs K-style)
격식체(T)와 비격식체(K) 구별:
```
격식: talofa (안녕)
비격식: kalofa (안녕)
```

### 2.2 존칭 체계
일반어와 존경어 구분:
```
일반: 'ai (먹다)
존경: taumafa (드시다)
```
