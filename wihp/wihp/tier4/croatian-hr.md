## 2. 특수 음운 현상

### 2.1 성조 체계 (Pitch Accent)
크로아티아어는 4가지 성조를 구분합니다:
| 기호 | 명칭 | 설명 | 예시 |
|------|------|------|------|
| ˋ | 단하강 | 짧고 내리는 | sȁm [sâm] 혼자 |
| ˊ | 단상승 | 짧고 올라가는 | sàm [sǎm] ~이다 |
| ˆ | 장하강 | 길고 내리는 | grȃd [ɡrâːd] 도시 |
| ˜ | 장상승 | 길고 올라가는 | gráce [ɡrǎːtsɛ] 은혜들 |

### 2.2 이에카브 방언 (Ijekavian)
크로아티아 표준어는 이에카브 방언을 사용합니다:
```
고대 슬라브어 ě → ie (장) / je (단)
예: mleko → mlijeko (우유)
    belo → bijelo (하얀)
```

### 2.3 ć vs č, đ vs dž
- ć [tɕ] / đ [dʑ]: 연구개음 (더 부드러움)
- č [tʃ] / dž [dʒ]: 경구개음 (더 강함)

## 9. 문법 참고 (Grammar Notes)

### 9.1 격 체계 (7격)
크로아티아어는 7개의 격을 사용합니다:
1. **Nominativ** (주격): Tko? Što?
2. **Genitiv** (속격): Koga? Čega?
3. **Dativ** (여격): Komu? Čemu?
4. **Akuzativ** (대격): Koga? Što?
5. **Vokativ** (호격): 부르기
6. **Lokativ** (처소격): O komu? O čemu?
7. **Instrumental** (구격): Kime? Čime?

### 9.2 성 (Gender)
- **Muški rod** (남성): student, grad
- **Ženski rod** (여성): žena, škola
- **Srednji rod** (중성): dijete, more

### 9.3 세르비아어와의 차이
| 특징 | 크로아티아어 | 세르비아어 |
|------|--------------|------------|
| 문자 | 라틴 | 키릴/라틴 |
| 반사음 | ijekavian | ekavian (표준) |
| 미래형 | ću+infinitive | ću+da+present |
| 어휘 | 독자 발전 | 독자 발전 |
