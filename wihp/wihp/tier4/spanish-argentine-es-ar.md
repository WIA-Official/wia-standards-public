## 개요

아르헨티나 스페인어(리오플라텐세 방언)는 voseo, ʃeísmo/žeísmo, 이탈리아어 영향, 독특한 억양이 특징입니다. 부에노스아이레스를 중심으로 한 리오 델 라플라타 지역의 방언입니다.

## 아르헨티나 스페인어 특징

### 주요 음운 특징

| 특징 | 설명 | 예시 |
|------|------|------|
| **ʃeísmo/žeísmo** | ll, y = /ʃ/ 또는 /ʒ/ | calle → /ˈkaʃe/ 카셰 |
| **세세오 (Seseo)** | z, c(e,i) = /s/ | zapato → /saˈpato/ 사파토 |
| **voseo** | tú → vos | vos tenés (너는 가진다) |
| **이탈리아 억양** | 상승 억양 | ¿Qué hacés?↗ |
| **어말 -s 약화** | 일부 계층 | -s → -h 또는 ∅ |

### 스페인 vs 아르헨티나 차이

| 특징 | 스페인 (ES) | 아르헨티나 (AR) | WIHP |
|------|-----------|----------------|------|
| ll, y | /ʎ/ → /ʝ/ | /ʃ/ 또는 /ʒ/ | 셰/줘 |
| z, c(e,i) | /θ/ 씨 | /s/ 스 | 사 |
| tú | tú eres | vos sos | 보스 소스 |
| vosotros | vosotros sois | ustedes son | 우스테데스 손 |
| 억양 | 평탄 | 이탈리아식 상승 | - |

## 자음 매핑 (아르헨티나 변이)

### ʃeísmo / žeísmo (가장 특징적)

| IPA | 철자 | WIHP 코드 | 한글 | 아르헨티나 예시 |
|-----|------|-----------|------|----------------|
| /ʃ/ | ll, y | PL05-MN05-AR01 | 쉬/ㅅ | **ll**amar → 샤마르 |
| /ʃ/ | - | - | - | **y**o → 쇼 |
| /ʒ/ | ll, y (유성) | PL05-MN05-AR04 | ㅈ/ㅿ | ca**ll**e → 카줘 |
| /ʒ/ | - | - | - | pla**y**a → 플라쟈 |

**발음 변이**:
- 부에노스아이레스 (젊은층): /ʒ/ (유성) → yo /ʒo/ 줘
- 부에노스아이레스 (전통): /ʃ/ (무성) → yo /ʃo/ 쇼
- 지방: /ʝ/ (표준 스페인어식)

### 세세오 (Seseo)

| IPA | 철자 | WIHP 코드 | 한글 | 예시 |
|-----|------|-----------|------|------|
| /s/ | s, c(e,i), z | PL04-MN05-AR01 | ㅅ | **z**apato → 사파토 |
| /s/ | - | - | - | **c**ine → 시네 |

### 어말 -s 약화 (일부 계층/지역)

| 환경 | 변화 | 예시 | WIHP |
|------|------|------|------|
| -s (어말) | → [h] 또는 ∅ | vos → [boh] | 보 |
| -s (어말) | → [s] (표준층) | los libros | 로스 리브로스 |
