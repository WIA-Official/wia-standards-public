## 2. 특수 음운 현상

### 2.1 성조 체계
쇼나어는 고저 성조를 가집니다:
```
banga (고고) - 칼
bàngà (저저) - 호박
```

### 2.2 명사 부류 체계
21개의 명사 부류:
```
mu-nhu (사람) → va-nhu (사람들)
chi-garo (의자) → zvi-garo (의자들)
```
