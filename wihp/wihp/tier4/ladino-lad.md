## 2. 특수 음운 현상

### 2.1 중세 카스티야어 특징 보존
현대 스페인어가 상실한 음운 유지:
```
hijo [ˈfixo] (현대 스페인어 [ˈixo])
fazer [faˈzeɾ] "하다" (현대 스페인어 hacer)
```

### 2.2 유성 치경 마찰음 보존
/z/ 발음 유지:
```
caza [ˈkaza] "집" (현대 스페인어 casa)
dezir [deˈziɾ] "말하다" (현대 스페인어 decir)
```

### 2.3 히브리어 차용어
종교·문화 용어:
```
shabbat [ʃaˈbat] "안식일"
kasher [kaˈʃeɾ] "코셔"
mazzal [maˈzal] "행운"
```

### 2.4 터키어 영향
오스만 제국 거주로 인한 차용어:
```
chelebi [tʃeˈlebi] "신사"
bazar [baˈzaɾ] "시장"
```
