## 2. 특수 음운 현상

### 2.1 아랍어-로맨스어 혼합
몰타어는 아랍어 기반에 이탈리아어/시칠리아어 어휘를 포함:
```
아랍어 유래: ilma (물), kelb (개), xemx (태양)
이탈리아어 유래: skola (학교), kamra (방), televizjoni (TV)
```

### 2.2 정관사 동화
정관사 'il-'이 다음 자음에 동화:
```
il-kelb [ilkelp] (그 개)
ix-xemx [iʃʃemʃ] (그 태양)
```
