## 2. 특수 음운 현상

### 2.1 쌍수 (Dual Number)
슬로베니아어는 슬라브어 중 유일하게 쌍수를 완전히 보존합니다:
```
단수: en človek (한 사람)
쌍수: dva človeka (두 사람)
복수: trije ljudje (세 사람 이상)
```

### 2.2 성절 /r/ (Syllabic R)
```
슬로베니아어에서 /r/은 모음처럼 음절을 형성:
prst [pr̩st] - 손가락
vrh [vr̩x] - 정상
trg [tr̩ɡ] - 광장
```

### 2.3 모음 교체 (Vowel Alternation)
열린/닫힌 e, o의 교체:
```
lep [lep] (아름다운 - 남성 단수)
lepa [lɛpa] (아름다운 - 여성 단수)
```
