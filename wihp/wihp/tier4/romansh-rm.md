## 2. 특수 음운 현상

### 2.1 로망슈어의 5개 방언
```
Sursilvan: 서부 방언 (가장 보수적)
Sutsilvan: 중부 방언
Surmiran: 중앙 방언
Puter: 상부 엥가딘
Vallader: 하부 엥가딘

Rumantsch Grischun: 1982년 표준화된 통합 문어
```

### 2.2 구개음화
라틴어 /k/, /ɡ/ + 전설모음 → 구개음:
```
Latin: caelum → tschiel [tʃiəl] 하늘
Latin: gentem → glieud [ɡliət] 사람들
```

### 2.3 어두 자음군 보존
라틴어 자음군이 로망슈어에 잘 보존됨:
```
Latin: clavis → clav [klav] 열쇠
Latin: flamma → flomma [ˈflɔma] 불꽃
```
