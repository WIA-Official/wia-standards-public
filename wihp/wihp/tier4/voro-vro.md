# WIHP: Võro (Võro keel) - vro

**ISO 코드**: vro
**언어명**: Võro / Võro keel / 버로어
**어족**: Uralic > Finnic > South Estonian
**화자 수**: 약 70,000명 (에스토니아 남부)
**난이도**: ★★★★★ (모음 조화, 15격, 장단 구분)
**상태**: 지역 소수언어 (UNESCO: Definitely Endangered)
**작성일**: 2025-12-22
**버전**: 1.0 (A-grade)

---

## 개요

Võro어(버로어)는 에스토니아 남동부 Võrumaa(버루마) 지역에서 사용되는 남부 에스토니아어군의 언어입니다. 표준 에스토니아어와는 별개의 언어로 간주되며, 2024년 에스토니아 정부로부터 공식 지역 언어로 인정받았습니다. 약 70,000명의 화자가 있으며, 활발한 언어 부흥 운동이 진행 중입니다. 독자적인 문자 표기법과 문법을 가지고 있으며, 표준 에스토니아어보다 고대 핀란드어의 특징을 더 많이 보존하고 있습니다.

## 2. 특수 음운 현상

### 2.1 모음 조화 (Vowel Harmony)
Võro어는 표준 에스토니아어보다 강한 모음 조화를 유지합니다:
- 전설 모음: ä, ö, ü
- 후설 모음: a, o, u
- 중립: e, i, õ

### 2.2 자음 등급 교체 (Consonant Gradation)
```
강형 → 약형
kk → k: jakkʼ → jakiʼ (재킷)
pp → p: kapp → kapiʼ (옷장)
tt → t: kattʼ → katiʼ (지붕)
k → ∅: jalg → jallaʼ (다리)
```

### 2.3 성문 폐쇄음 (Glottal Stop)
Võro어의 특징적 현상으로 단어 말미에 성문 폐쇄음 [ʔ]를 표기합니다:
- kodʼ [kotʔ] - 집
- nimʼ [nimʔ] - 이름
- 철자로 ' (apostrophe) 사용
