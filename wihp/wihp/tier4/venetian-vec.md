## 2. 특수 음운 현상

### 2.1 어말 자음 보존
다른 Gallo-Italic 언어와 달리 어말 -o를 더 잘 보존:
```
롬바르드어: bell [bɛl]
베네치아어: belo [ˈbɛlo]

롬바르드어: gat [ɡat]
베네치아어: gato [ˈɡato]
```

### 2.2 구개음화
라틴어 pl-, cl-, fl- → 베네치아어 pj-, cj-, fj-:
```
라틴어 plenum → 베네치아어 pien [pjɛŋ]
라틴어 clavem → 베네치아어 ciave [ˈtʃave]
라틴어 florem → 베네치아어 fior [fjor]
```

### 2.3 조사 체계
베네치아어는 독특한 조사 체계를 가집니다:
```
el = 남성 단수 정관사
la = 여성 단수 정관사
i = 남성 복수 정관사
łe = 여성 복수 정관사
```

### 2.4 주어 대명사 필수
이탈리아어와 달리 주어 대명사 필수:
```
베네치아어: Mi parlo (나는 말한다)
이탈리아어: Parlo (말한다)
```

### 2.5 ł 철자
베네치아어 특유의 철자 ł [ł] (연음 L):
```
łe = [le] (여성 복수 정관사)
beło = [ˈbɛło] (아름다운)
```
