## 2. 특수 음운 현상

### 2.1 음절 문자 체계
이누크티투트는 음절 문자를 사용하며, 각 글자가 자음+모음 조합을 나타냅니다:
```
ᐱ (pi), ᐳ (pu), ᐸ (pa)
ᑎ (ti), ᑐ (tu), ᑕ (ta)
ᑭ (ki), ᑯ (ku), ᑲ (ka)
```

### 2.2 후설 자음 (Uvular)
```
q [q] - 무성 후음 파열음
r [ʁ] - 유성 후음 마찰음
```

### 2.3 무성 측면 마찰음
```
ł [ɬ] - 웨일즈어 'll'과 유사
```

### 2.4 포합어 특성
동사에 많은 접사가 결합하여 하나의 긴 단어를 형성:
```
ᖃᐅᔨᒪᔭᐅᖕᓂᑦᑐᖅ (qaujimajaunngittuq)
"알려지지 않았다"
qaujima-jau-nngit-tuq
알다-수동-부정-3인칭단수
```

## 9. 문법 특징

### 9.1 능격-절대격 언어
이누크티투트는 능격-절대격 언어로, 타동사의 주어(능격)와 자동사의 주어(절대격)가 다르게 표시됩니다.

### 9.2 포합어
동사에 많은 형태소가 결합하여 문장 전체를 표현:
```
ᖃᓪᓗᓈᑐᑦ ᐅᖃᓚᐅᑎᐊᕐᒪᖓ
Qallunaatuut uqallautiarmanga
"영어로 말하게 해 주세요"
```

### 9.3 쌍수(Dual)
단수, 쌍수(2개), 복수(3개 이상) 구별:
```
ᐃᓄᒃ (inuk) - 1명
ᐃᓅᒃ (inuuk) - 2명
ᐃᓄᐃᑦ (inuit) - 3명 이상
```
