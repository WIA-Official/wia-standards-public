## 2. 특수 음운 현상

### 2.1 프랑스어와의 관계
아이티 크레올은 프랑스어에서 파생되었지만 독립된 언어입니다:
```
프랑스어: Je suis content → 크레올: Mwen kontan
프랑스어: L'eau → 크레올: Dlo
```

### 2.2 TMA 마커 (시제-상-서법)
동사 활용 대신 입자를 사용합니다:
```
Mwen manje - 나는 먹는다 (현재/습관)
Mwen te manje - 나는 먹었다 (과거)
Mwen ap manje - 나는 먹고 있다 (진행)
Mwen pral manje - 나는 먹을 것이다 (미래)
```

### 2.3 정관사 후치
정관사는 명사 뒤에 옵니다:
```
liv (책) → liv la (그 책)
kay (집) → kay la (그 집)
```

## 9. 문법 참고 (Grammar Notes)

### 9.1 TMA 마커 시스템
| 마커 | 의미 | 예시 |
|------|------|------|

| ap | 진행 | Mwen ap manje (나는 먹고 있다) |
| pral | 미래 | Mwen pral manje (나는 먹을 것이다) |
| ta | 조건 | Mwen ta manje (나는 먹을 텐데) |
