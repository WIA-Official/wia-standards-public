## 2. 특수 음운 현상

### 2.1 음절 구조
(C)V 구조만 허용:
```
fa-le (집)
To-nga (통가)
```

### 2.2 존칭 체계
왕/귀족을 위한 별도 어휘:
```
일반: kai (먹다)
왕실: taumafa (드시다)
일반: ʻalu (가다)
왕실: meʻa (가시다)
```
