## 2. 특수 음운 현상

### 2.1 F → H 변화
라틴어 F가 H로 (스페인어와 유사):
```
FACERE → hazel [aˈθel] "하다"
FERRUM → hierru [ˈhjeru] "철"
```

### 2.2 -L 보존
어말 L 유지 (레온어 특징):
```
árbol [ˈaɾbol] "나무" (스페인어 árbol)
papel [paˈpel] "종이"
```

### 2.3 CH의 다양한 발음
지역에 따라 /tʃ/ 또는 /ʃ/:
```
muchu [ˈmutʃu] ~ [ˈmuʃu] "많은"
```
