## 2. 특수 음운 현상

### 2.1 H 발음
프랑스어와 달리 H 발음:
```
hôte [ɔt] "높은" (프랑스어는 무음)
hauter [ɔtɛʀ] "높이"
```

### 2.2 이중자음 간소화
라틴어 이중자음 유지:
```
bonne [bɔn] "좋은" (프랑스어처럼)
terre [tɛʀ] "땅"
```

### 2.3 비음화
비음모음 광범위 사용:
```
bén [bɛ̃] "잘"
djin [dʒɛ̃] "사람들"
walon [walɔ̃] "왈롱"
```
