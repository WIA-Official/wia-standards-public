## 2. 특수 음운 현상

### 2.1 삼중 대립
케추아어는 세 가지 조음 방식을 구분합니다:
- 일반음 (plain): p, t, ch, k, q
- 기식음 (aspirated): ph, th, chh, kh, qh
- 방출음 (ejective): p', t', ch', k', q'

```
tanta [tanta] - 빵
thanta [tʰanta] - 낡은
t'anta [t'anta] - 모으다
```

### 2.2 모음 저하 (Vowel Lowering)
/q/ 인접시 모음이 낮아집니다:
```
qillqa [qeʎqa] - 쓰기 (i→e)
qucha [qotʃa] - 호수 (u→o)
```

### 2.3 교착어 형태론
접미사를 통해 의미를 확장합니다:
```
wasi (집) → wasiy (나의 집) → wasiyki (너의 집)
→ wasiyku (우리의 집) → wasikunapi (집들에서)
```
