## 2. 특수 음운 현상

### 2.1 완벽한 정서법
에스페란토는 "한 글자 = 한 소리" 원칙을 따릅니다:
```
모든 글자는 항상 같은 소리
예외 없음
강세는 항상 끝에서 두 번째 음절
```

### 2.2 규칙적인 문법
품사를 어미로 구분합니다:
```
-o: 명사 (domo = 집)
-a: 형용사 (bela = 아름다운)
-e: 부사 (rapide = 빠르게)
-i: 동사 원형 (lerni = 배우다)
```

### 2.3 동사 시제
```
-as: 현재 (mi lernas = 나는 배운다)
-is: 과거 (mi lernis = 나는 배웠다)
-os: 미래 (mi lernos = 나는 배울 것이다)
-us: 조건 (mi lernus = 나는 배우겠다)
-u: 명령 (lernu! = 배워라!)
```

## 7. 문법 (Grammar)

### 7.1 접두사/접미사
| 접사 | 의미 | 예시 |
|------|------|------|
| mal- | 반대 | bona → malbona (좋은 → 나쁜) |
| -ej- | 장소 | lerni → lernejo (배우다 → 학교) |
| -ul- | 사람 | juna → junulo (젊은 → 젊은이) |
| -in- | 여성 | patro → patrino (아버지 → 어머니) |
| -et- | 작은 | domo → dometo (집 → 작은집) |
| -eg- | 큰 | domo → domego (집 → 큰집) |
