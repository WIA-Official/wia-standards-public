## 2. 특수 음운 현상

### 2.1 аканне (Akanne)
비강세 о는 [a]로 발음됩니다:
```
молоко [malako] - 우유
вода [vada] - 물
```

### 2.2 дзеканне와 цеканне
т와 д가 연음화되면 ц와 дз로 변합니다:
```
дзень [dzʲenʲ] - 날 (러시아어: день)
цётка [tsʲotka] - 이모 (러시아어: тётка)
```

### 2.3 경/연 자음 대립
벨라루스어는 경자음과 연자음을 구분합니다:
```
быць [bɨtsʲ] - 있다
бiць [bʲitsʲ] - 치다
```
