## 2. 특수 음운 현상

### 2.1 중자음 (Gemination)
코르시카어는 의미 구분에 중요한 중자음을 보유:
```
pala [ˈpala] "삽" vs. palla [ˈpalːa] "공"
casu [ˈkazu] "경우" vs. cassu [ˈkasːu] "치즈"
```

### 2.2 어말 모음 보존
이탈리아어와 달리 어말 모음이 더 안정적:
```
Italian: cane [ˈkane]
Corsican: cane [ˈkane] (같음)
```

### 2.3 방언 구분
```
북부 코르시카어 (Cismuntincu): 이탈리아 토스카나어에 가까움
남부 코르시카어 (Pumuntincu): 사르디니아어 영향
```
