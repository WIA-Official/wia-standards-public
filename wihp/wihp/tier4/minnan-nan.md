# WIHP 매핑: Min Nan / Hokkien (閩南語/민난어)

**ISO 코드**: nan
**언어명**: Min Nan / Hokkien / 閩南語 / Bân-lâm-gú / 민난어
**어족**: Sino-Tibetan > Sinitic > Min > Min Nan
**화자 수**: 50M+ (L1: 48M)
**주요 지역**: 푸젠성(福建), 대만, 광둥 동부, 하이난, 동남아
**작성일**: 2025-12-22
**버전**: 1.0 (A등급)
**품질등급**: A

---

## 개요

민난어(閩南語)는 중국 남부 푸젠성 남부와 대만에서 주로 사용되는 중국어 방언입니다. 호키엔어(Hokkien), 대만어로도 불립니다. 7-8개의 성조를 가진 성조 언어이며, POJ(白話字, Pe̍h-ōe-jī) 로마자 표기법이 널리 사용됩니다. 입성(入聲)을 유지하고 있으며, 만다린과는 상호이해가 거의 불가능합니다.

---

## 특이 음운 현상

### 1. 변조 (Tone Sandhi)

민난어의 가장 특징적인 현상으로, 연속된 음절에서 앞 음절의 성조가 변합니다.

| 기본조 | 변조 후 | 예시 |
|--------|---------|------|
| 1 (44) → 7 (33) | 三 saⁿ (44) + 個 ê (24) → sā ê (33-24) |
| 2 (53) → 1 (44) | 美 bí (53) + 國 kok (32) → bi kok (44-32) |
| 3 (21) → 2 (53) | 四 sì (21) + 個 ê (24) → sí ê (53-24) |
| 5 (24) → 7 (33) | 時 sî (24) + 間 kan (44) → si kan (33-44) |
| 7 (33) → 3 (21) | 是 sī (33) + 無 bô (24) → sì bô (21-24) |

### 2. 비음화 (Nasalization)

많은 모음이 비음화됩니다:
- saⁿ (三), chheⁿ (青), iⁿ (圓)

### 3. 입성 보존

고대 중국어의 입성이 완전히 보존되어 있습니다:
- -p: cha̍p (十), sa̍p (濕)
- -t: chhut (出), pe̍h (白)
- -k: pak (北), pe̍k (壁)
- -h: o̍h (學), ta̍h (踏)
