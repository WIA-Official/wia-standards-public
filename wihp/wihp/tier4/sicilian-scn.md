## 2. 특수 음운 현상

### 2.1 권설음 (Retroflex)
시칠리아어의 가장 특징적인 음운 현상:
```
라틴어 -ll- → 시칠리아어 /ɖɖ/
  bellum → beddu [ˈbɛɖɖu]

라틴어 -str- → 시칠리아어 /ʂʈɽ/
  maestro → mastru [ˈmaʂʈɽu]
```

### 2.2 자음 중복 (Gemination)
강세 음절 뒤의 자음은 항상 중복:
```
casa [ˈkasa] → casuzza [kaˈsuttsa] (작은 집)
beddu [ˈbɛɖɖu] (아름다운)
gattu [ˈɡattu] (고양이)
```

### 2.3 모음 약화
비강세 e, o는 i, u로 약화:
```
이탈리아어 verde → 시칠리아어 virdi (초록)
이탈리아어 posto → 시칠리아어 pustu (장소)
```

### 2.4 아랍어 차용어
아랍 지배의 영향:
```
zabbara (알로에) ← 아랍어 ṣabīr
gebbia (물 저장소) ← 아랍어 jābiya
```
