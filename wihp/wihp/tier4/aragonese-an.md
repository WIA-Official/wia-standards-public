## 2. 특수 음운 현상

### 2.1 라틴어 F- 보존
스페인어와 달리 어두 /f/ 유지:
```
라틴어 FARINA → 아라곤어 farina (스페인어 harina "밀가루")
라틴어 FERRUM → 아라곤어 fierro (스페인어 hierro "철")
```

### 2.2 -ll- 보존
라틴어 이중 L 유지:
```
CASTELLUM → castiello "성"
CABALLUS → caballo "말"
```

### 2.3 CH의 발음
카탈루냐어 영향으로 /ʒ/:
```
choven [ʒoβen] "젊은"
mucho [muʒo] "많은"
```
