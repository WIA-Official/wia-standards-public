## 2. 특수 음운 현상

### 2.1 초자음 변이 (Kemmyskans)
4가지 주요 변이:
```
연음화 (Medhel): tyr → dyr [diːr] (땅)
경음화 (Kales): tyr → tyr [tiːr] (변화 없음)
유기음화 (Hwisk): tyr → thyr [θiːr]
혼합변이 (Kemmysk): tyr → zyr [ziːr]
```

### 2.2 SVO/VSO 어순
현대 콘월어는 두 어순 모두 허용:
```
SVO: My a wel ki.
VSO: Gweles a wrama ki.
둘 다: "나는 개를 본다"
```

### 2.3 부활 방언 차이
3가지 표준 철자법:
```
- Kernewek Kemmyn (통합 콘월어, 가장 일반적)
- Kernowek Standard (표준 콘월어)
- Unified Cornish Revised (수정 통합)
```
