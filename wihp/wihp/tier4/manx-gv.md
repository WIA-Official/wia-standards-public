## 2. 특수 음운 현상

### 2.1 초자음 변이 (Consonant Mutation)
아일랜드어/스코틀랜드 게일어보다 단순화됨:
```
Lenition: moddey → voddey [ˈvɔðʲə] (나의 개)
h-prothesis: eeast → h-eeast (그녀의 물고기)
Eclipsis: (드물게 사용)
```

### 2.2 VSO 어순
동사-주어-목적어:
```
Honnick mee yn moddey.
보았다 나 그 개
"나는 개를 보았다"
```

### 2.3 영어식 철자법
다른 게일어와 달리 영어 발음 규칙 사용:
```
sh = /ʃ/ (스코틀랜드 게일어 s(i)와 동일)
çh = /x/ (스코틀랜드 게일어 ch와 동일)
```
