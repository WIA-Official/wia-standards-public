## 2. 특수 음운 현상

### 2.1 영어 차용어
홍콩의 독특한 영어 차용어 발음:
```
bus → 巴士 [baa¹ si⁶²]
taxi → 的士 [dik¹ si⁶²]
file → 檔 [fail¹]
```

### 2.2 게으른 음 (懶音)
젊은 세대의 발음 특징:
```
ng- [ŋ] → ∅
我 ngo⁵ → o⁵
n- [n] → l- [l]
你 nei⁵ → lei⁵
```

### 2.3 종합어미
홍콩 특유의 어기사:
```
啦 laa¹ (강조)
咯 lo¹ (설명)
喎 wo³ (놀람)
呀嘛 aa¹ maa³ (당연함)
```
