## 2. 특수 음운 현상

### 2.1 F- 보존
라틴어 어두 F 유지 (포르투갈어와 동일):
```
FACERE → fazer [fɐˈzeɾ] "하다"
FERRUM → fierro [ˈfjɛru] "철"
```

### 2.2 PL-, CL-, FL- 보존
라틴어 자음군 유지:
```
PLORARE → plorar [pluˈɾaɾ] "울다"
CLAVE → chlabe [ˈklaβɨ] "열쇠"
```

### 2.3 L의 연음화
어말 L이 /ɫ/ (dark L):
```
sol [sɔɫ] "태양"
mal [maɫ] "나쁜"
```
