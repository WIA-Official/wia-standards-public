## 개요

카렐리아어(Karjalan kieli)는 러시아 카렐리아 공화국과 핀란드 동부 지역에서 사용되는 핀란드어군 언어입니다. 약 60,000-80,000명의 화자가 있으며, 핀란드어와 매우 가까운 관계에 있습니다. 일부 언어학자들은 카렐리아어를 핀란드어의 방언으로 보기도 하나, 현재는 **독립된 언어**로 인정받고 있습니다.

카렐리아어는 **3개의 주요 방언군**(Proper Karelian, Olonets Karelian, Ludic Karelian)으로 나뉘며, 각 방언 간 차이가 상당합니다. 카렐리아는 핀란드 서사시 **Kalevala**의 발상지로, 풍부한 구전 문학 전통(룬노래, runo)을 보유하고 있습니다.

소련 시대 탄압과 핀란드어/러시아어와의 경쟁으로 화자가 급감하고 있으나, 2000년대 이후 언어 부흥 운동이 활발히 진행 중입니다.

## 2. 특수 음운 현상

### 2.1 모음 조화 (Vowel Harmony)
카렐리아어는 핀란드어와 유사한 강한 모음 조화를 가집니다:

- **전설 모음**: ä, ö, y
- **후설 모음**: a, o, u
- **중립**: e, i

**예시**:
```
talossa (집에서) - 후설
kielešä (언어에서) - 전설
```

### 2.2 자음 등급 교체 (Consonant Gradation)
카렐리아어는 복잡한 자음 등급 교체를 가집니다:

```
강형 → 약형
kk → k: kukko → kukon (수탉)
pp → p: kappali → kappalien (조각)
tt → t: katto → katon (지붕)
k → ∅: jalka → jalan (다리)
p → v: tapa → tavan (습관)
t → d: katu → kadun (거리)
mp → mm: lampi → lammen (연못)
nt → nn: ranta → rannan (해변)
nk → ng: kenkä → kengän (신발)
```

### 2.3 카렐리아어 특유 현상

**1) -h 복수 표지**
카렐리아어는 복수를 만들 때 -h를 자주 사용:
```
talo (집) → taloh (집들)
kala (물고기) → kalah (물고기들)
```

**2) 이중 격 표지**
```
talošša (집들에서)
= talo + h (복수) + ssa (내격)
```

**3) 어두 자음군 보존**
러시아어 차용어에서 어두 자음군을 그대로 유지:
```
prua (바지) - 러시아어 брюки
klass (학급) - 러시아어 класс
```

### 8.3 Kalevala 전통

카렐리아는 핀란드 국민 서사시 **Kalevala**의 발상지입니다. Elias Lönnrot은 1830-40년대 카렐리아를 여행하며 룬노래(runot)를 채집하여 Kalevala를 편찬했습니다.

**유명한 Kalevala 구절 (카렐리아어 버전)**:
```
Mieleni minun tekevi,          내 마음이 이끌고,
aivoni ajattelevi              내 생각이 명하네
lähteäni laulamaha,            노래 부르러 가자,
sää'töömähä suuta.             입을 열어 말하자.
```

**Kantele (칸텔레)**:
카렐리아 전통 5-40현 발현악기. Kalevala의 영웅 Väinämöinen이 연주했다고 전해집니다.

## 10. 방언

카렐리아어는 크게 **3개 주요 방언군**으로 나뉩니다:

### 10.1 Varsinaiskarjala (Proper Karelian, 북부 카렐리아어)
- **지역**: 러시아 카렐리아 공화국 북부, 핀란드 동부
- **화자**: 약 30,000명
- **특징**: 가장 보수적, 고대 핀란드어 특징 보존
- **철자**: 라틴 문자 (핀란드 기반)

### 10.2 Livvin karjala (Olonets Karelian, 올로네츠 카렐리아어)
- **지역**: 러시아 카렐리아 공화국 남부 (올로네츠 지역)
- **화자**: 약 20,000명
- **특징**: 베프스어 영향, 독특한 음운 변화
- **철자**: 라틴 문자 또는 키릴 문자

### 10.3 Lyydin karjala (Ludic Karelian, 류디크 카렐리아어)
- **지역**: 러시아 카렐리아 공화국 중부
- **화자**: 약 3,000명
- **특징**: 매우 독특, 일부 학자는 별개 언어로 분류
- **상태**: 가장 위급한 방언

**상호 이해도**: 방언 간 상호 이해가 어려운 경우가 많음. 표준 카렐리아어 개발 시도 중.

## 12. 역사 및 언어 보존

### 12.1 역사적 배경

**고대-중세**:
- 카렐리아인은 라도가 호수 주변에 정착
- 노브고로드 공국 영향권
- 독특한 룬노래 전통 발달

**스웨덴-러시아 시대 (17-19세기)**:
- 1617: Stolbovo 조약 - 카렐리아 분할
- 서부 → 스웨덴 (후에 핀란드)
- 동부 → 러시아
- 대규모 인구 이동 (Old Finland)

**20세기 초**:
- 1918: 핀란드 독립, 카렐리아인 일부 핀란드로
- 1920: 소련, 카렐리아 노동 코뮌 설립
- 1923: 카렐리아 자치 소비에트 사회주의 공화국

**소련 시대 (1920-1991)**:
- 1930s: 카렐리아어 학교, 신문, 출판
- 1938: 스탈린 숙청, 카렐리아어 탄압 시작
- 1939-1944: 겨울 전쟁, 계속 전쟁
- 1944 이후: 러시아화 정책 강화
- 카렐리아어 교육 중단

**결과**: 화자 급감 (1926: 248,000 → 2010: 25,000)

### 12.2 현재 상황 (2024)

**화자 통계**:
- 러시아: 약 40,000-50,000명
- 핀란드: 약 10,000-20,000명
- 기타 (스웨덴, 에스토니아): 약 5,000명
- **총계**: 약 60,000-80,000명
- **그러나**: 유창한 화자는 약 20,000명 미만
- 젊은 화자 (30세 이하): 약 2,000명

**위기 요인**:
1. 러시아어/핀란드어로의 언어 교체
2. 세대 간 전승 약화
3. 교육 시스템 부재
4. 경제적 압박 (도시 이주)
5. 방언 간 분열

### 12.3 언어 부흥 운동

**러시아 카렐리아 공화국**:
- **2004**: 카렐리아어 공식 언어 지위 (핀란드어, 베프스어와 함께)
- **Karjalan Kielen Seura** (카렐리아어 협회)
- 카렐리아어 교실 (페트로자보츠크 등)
- TV/라디오 카렐리아어 프로그램
- 이중언어 표지판

**핀란드**:
- **Karjalan Kielen Seura** (핀란드 지부)
- University of Eastern Finland - 카렐리아어 강좌
- 여름 언어 캠프
- 카렐리아 문화 페스티벌

**디지털 프로젝트**:
- 카렐리아어 Wikipedia (소규모)
- 온라인 사전 (Karjalan kielellä)
- YouTube 언어 강좌
- 스마트폰 앱

**출판**:
- 신문 "Oma Mua" (우리 땅)
- 문학 잡지 "Carelia"
- 카렐리아어 교과서

### 12.4 성과와 도전

**성과**:
- 젊은 세대 관심 증가
- 교육 자료 개발
- 문화적 자긍심 회복
- 국제 인정

**도전**:
- 방언 표준화 문제
- 재정 부족
- 러시아 정치 상황
- 세대 간 단절

## 13. 핀란드어와의 관계

카렐리아어와 핀란드어는 **상호 이해 가능**한 수준의 유사성을 가지고 있습니다.

| 의미 | 카렐리아어 | 핀란드어 | 차이점 |
|------|-----------|---------|--------|
| 나 | mie | minä | 음운 변화 |
| 집 | talo/koti | talo/koti | 동일 |
| 물 | vesi | vesi | 동일 |
| 언어 | kieli | kieli | 동일 |
| 사람들 | rahvah | rahvas | 유사 |
| 말하다 | pagista/paissa | puhua | 다름 |
| 일하다 | ruatua | tehdä työtä | 다름 |

**주요 차이**:
1. 음운: 카렐리아어는 -h 복수, 이중모음 단순화
2. 어휘: 러시아어 차용어 많음
3. 문법: 약간 다른 격 체계

**논쟁**: 카렐리아어가 핀란드어의 방언인가, 독립된 언어인가?
- **핀란드**: 대부분 별개 언어로 인정
- **러시아**: 공식적으로 별개 언어
- **언어학**: 의견 분분
