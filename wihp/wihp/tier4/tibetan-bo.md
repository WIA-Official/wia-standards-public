## 2. 특수 음운 현상

### 2.1 성조 체계 (2-4개, 방언별)
라싸 방언은 2개 성조:
```
高 (고음): ཀ་ [ká] (기둥)
低 (저음): ཀ་ [kà] (눈)
```

암도 방언은 4개 성조:
```
高평 (55), 上승 (35), 低평 (11), 降하 (31)
```

### 2.2 티벳 문자 구조
음절 단위, 최대 7개 요소:
```
접두사 + 상자 + 주자음 + 하자 + 모음 + 후자 + 재후자
སྒྲིགས་ = ས (접두) + ག (주자음) + ྲ (하자) + ི (모음) + ག (후자) + ས (재후자)
발음: [triɡ]
```

### 2.3 묵음 규칙
철자와 발음의 큰 차이:
```
སྒོ་ (s-g-o) → [ɡo] (문)
བསྟན་ (b-s-t-n) → [ten] (가르침)
དགོངས་པ་ (d-g-ng-s-p) → [ɡoŋpa] (마음)
```

### 2.4 접두사 효과
접두사로 유성음화/무성음화:
```
ག → [ɡa] (하지만) དགའ་ → [ɡaː] (기쁨)
བ → [pa] (소) སྦ་ → [pa] (가죽)
```

### 2.5 후자음 효과
후자음은 성조에 영향:
```
ཀ་ (ka, 무후자) → 고음
ཀག་ (kag, 유후자) → 저음
```
