## 2. 특수 음운 현상

### 2.1 성조 체계 (2개)
종카어는 고저 성조를 가집니다:
```
高 (고음): ཁ་ [kʰá] (입)
低 (저음): ཁ་ [kʰà] (눈)
```

### 2.2 티벳 문자 구조
음절 단위로 쓰며, 최대 7개 요소:
```
prefix + superscript + 주자음 + subscript + vowel + suffix + post-suffix
དགྱིས = ད (prefix) + ག (main) + ྱ (subscript) + ི (vowel) + ས (suffix)
발음: [ɟiː]
```

### 2.3 묵음 자음
철자에는 있지만 발음 안 됨:
```
སྒོ་ (철자: s-g-o) → [ɡo] (문)
བརྒྱད་ (철자: b-r-g-y-d) → [ɟeː] (8)
```

### 2.4 경음화
특정 접두사로 무성음화:
```
སྦ་ → [p] (가죽)
སྒ་ → [k] (안장)
```
