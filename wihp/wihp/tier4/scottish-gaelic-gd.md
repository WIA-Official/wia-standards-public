## 2. 특수 음운 현상

### 2.1 초자음 변이 (Initial Consonant Mutation)
3가지 주요 변이:
```
Lenition (연음화): cat → chat [xat̪]
h-prothesis: àite → h-àite [haːtʲə]
Nasalization: cat → gcat (드문 편)
```

### 2.2 VSO 어순
동사-주어-목적어:
```
Chunnaic mi an cù.
보았다 나 그 개
"나는 개를 보았다"
```

### 2.3 Broad/Slender 규칙
"Caol ri caol, leathann ri leathann"
(슬렌더는 슬렌더와, 브로드는 브로드와)
