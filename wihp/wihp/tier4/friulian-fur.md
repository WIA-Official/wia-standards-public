## 2. 특수 음운 현상

### 2.1 구개음화 (Palatalization)
프리울리어의 가장 특징적인 현상 - 라틴어 cl-, gl-, pl- → cj-, gj-:
```
라틴어 clavem → 프리울리어 clâf [tʃaf]
라틴어 glaciem → 프리울리어 glaç [dʒats]
라틴어 plenum → 프리울리어 plen [plɛŋ]
```

### 2.2 모음 약화
비강세 위치에서 모음 약화:
```
강세: biel [bjɛl] (아름다운)
비강세: bielece [bjeˈlɛtse] (아름다움)
```

### 2.3 자음 중복
일부 자음은 중복 가능:
```
gnoç [ɲɔtʃ] (뇨키, 단수)
gnoccs [ɲɔtʃs] (뇨키, 복수)
```

### 2.4 관사 체계
프리울리어 관사:
```
il = 남성 단수 (자음 앞)
l' = 남성 단수 (모음 앞)
la = 여성 단수 (자음 앞)
i = 남성 복수
lis = 여성 복수
```

### 2.5 주어 대명사 복잡성
주어 대명사 + 조동사 대명사:
```
o soi = 나는 ~이다 (jo + o)
tu sês = 너는 ~이다 (tu + ê)
al è = 그는 ~이다 (lui + al)
```
