## 2. 특수 음운 현상

### 2.1 중설모음 /ɨ/
루마니아어와 공유하는 특징:
```
pãne [ˈpɨne] "빵"
cãrne [ˈkɨrne] "고기"
```

### 2.2 최종 -a → -ã
여성형 어미:
```
bunã [ˈbunə] "좋은 (여성)"
casã [ˈkasə] "집"
```

### 2.3 발칸 언어동맹 특징
정관사 후치:
```
om "사람" → omu "그 사람"
casã "집" → casa "그 집"
```

### 2.4 그리스어 차용어
많은 그리스어 영향:
```
cafenea [kafeˈne̯a] "카페" (그리스어 kafeneío)
hora [ˈhora] "마을" (그리스어 chóra)
```
