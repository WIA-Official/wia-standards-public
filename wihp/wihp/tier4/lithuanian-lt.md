## 2. 특수 음운 현상

### 2.1 성조 (Pitch Accent)
리투아니아어는 복잡한 성조 체계를 가집니다:
| 성조 | 명칭 | 설명 |
|------|------|------|
| ˜ | 상승조 (Acute) | 올라가는 성조 |
| ˋ | 하강조 (Circumflex) | 내려가는 성조 |

```
예:
súdyti [suːdʲɪtʲɪ] - 심판하다 (상승조)
sùdyti [sʊdʲɪtʲɪ] - 임명하다 (하강조)
```

### 2.2 연자음 (Palatalization)
i, y, ė, e, į 앞에서 대부분의 자음이 연음화됩니다:
```
diena [dʲiɛna] - 날
kietas [kʲiɛtas] - 딱딱한
```

### 2.3 인도유럽어 보수성
리투아니아어는 산스크리트어와 유사한 고대 특징을 보존:
```
산스크리트: devas (신)
리투아니아: dievas [dʲiɛvas] (신)
```
