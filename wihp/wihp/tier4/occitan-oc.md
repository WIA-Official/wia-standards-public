## 2. 특수 음운 현상

### 2.1 6개 주요 방언
```
Languedocian (랑그도크): 중앙 방언, 표준어 기반
Provençal (프로방스): 동부 방언
Gascon (가스코뉴): 서부 (바스크어 기질)
Limousin (리무쟁): 북부
Auvergnat (오베르뉴): 북동부
Vivaro-Alpine (비바로알프스): 남동부
```

### 2.2 트루바두르 전통
중세 오크어는 유럽 궁정 문학의 언어:
```
"Tant m'abellis l'amoros pensamen"
(사랑의 생각이 나를 기쁘게 한다)
- 베르나르 드 벤타도른
```

### 2.3 프랑스어와의 차이
```
Occitan: òc [ɔk] "예"
French: oui [wi] "예"
→ 언어명의 유래
```
