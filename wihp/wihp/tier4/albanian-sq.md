## 2. 특수 음운 현상

### 2.1 정관사 후치
알바니아어는 정관사가 명사 뒤에 붙습니다:
```
libër (책) → libri (그 책)
vajzë (소녀) → vajza (그 소녀)
```

### 2.2 두 개의 r 음
```
r [ɾ] - 탄설음 (가벼운 r)
rr [r] - 전동음 (굴리는 r)
```

### 2.3 두 개의 l 음
```
l [l] - 일반 l
ll [ɫ] - 연구개화 l (어두운 l)
```
