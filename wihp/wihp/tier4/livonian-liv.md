## 개요

리보니아어(Līvõ kēļ)는 라트비아 북서부 해안 지역(Līvõmō)에서 사용되던 핀란드어군 언어로, **현존하는 가장 위급한 우랄어족 언어** 중 하나입니다. 2024년 기준 모어 화자는 약 10명 미만이며, L2 화자 포함 약 30명이 사용합니다. 2013년 마지막 원어민 화자가 사망했으나, 언어 부흥 운동을 통해 새로운 학습자들이 생겨나고 있습니다.

리보니아어는 에스토니아어와 가까우나, **성조(broken tone/stretched tone)** 구분, 독특한 모음 조화, 라트비아어 차용어 등 고유한 특징을 가지고 있습니다. 12개의 격을 가진 교착어이며, 풍부한 구전 문학 전통(dainas)을 보유하고 있습니다.

## 2. 특수 음운 현상

### 2.1 성조 (Tone/Stød)
리보니아어는 핀란드어군 중 **유일하게 성조를 가진 언어**입니다:

**1) Broken Tone (Stød, 단절조)** - 목소리가 끊어지는 느낌
```
sǭda [søː²dɑ] (전쟁) - 성조 표시: ²
```

**2) Stretched Tone (연장조)** - 부드럽게 이어지는 느낌
```
sōda [soː¹dɑ] (편지) - 성조 표시: ¹
```

성조는 의미 구분에 중요:
- `sūr` [suːr¹] (큰) vs `sǖr` [syːr²] (회색)

### 2.2 모음 조화
리보니아어는 부분적 모음 조화:
- 전설: ä, ö, ü, ȯ
- 후설: a, o, u
- 중립: e, i, õ

### 2.3 자음 등급 교체
```
강형 → 약형
kk → k: jokk → joga (강)
pp → p: kupp → kuobjõ (컵의)
tt → t: mett → mettõ (숲의)
```

### 8.3 민요 (Dainas) 전통

리보니아어는 풍부한 구전 시가(dainas) 전통을 가지고 있습니다:

```
Līvõmō, Līvõmō,     리보니아여, 리보니아여,
minā kuodimō        나의 고향이여
```

## 10. 언어 부흥 및 보존 현황

### 10.1 역사적 배경
- **고대**: 발트해 연안 전역에 걸쳐 사용
- **13세기**: 독일/덴마크 정복 후 쇠퇴 시작
- **20세기**: 소련 시대 급격한 감소
- **2013**: 마지막 원어민 Grizelda Kristiņa 사망 (103세)
- **2024**: 약 30명의 L2 화자 활동 중

### 10.2 현재 부흥 노력
- **Līvõ Kultūr sidām** (리보니아 문화센터, 마질리)
- **Līvõd Institūt** (라트비아 대학)
- **언어 캠프**: 매년 여름 청소년 대상
- **디지털화**: 온라인 사전, Līvõ kēļ app
- **미디어**: 월간 신문 "Līvli"
- **교육**: 라트비아 학교 일부에서 선택 과목

### 10.3 UNESCO 지정
- **Status**: Critically Endangered
- **2011**: 라트비아 정부 공식 인정
- **보호 프로그램**: EU 소수언어 보호 프레임워크
