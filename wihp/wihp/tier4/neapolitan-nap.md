## 2. 특수 음운 현상

### 2.1 Schwa (슈와) /ə/
나폴리어의 가장 독특한 특징 - 비강세 e, o → ə:
```
이탈리아어 bene → 나폴리어 buono [ˈbwɔnə]
이탈리아어 sole → 나폴리어 sole [ˈsolə]
이탈리아어 nome → 나폴리어 nomme [ˈnommə]
```

### 2.2 자음 중복 (Gemination)
나폴리어는 자음 중복이 매우 발달:
```
gatto → [ˈɡattə] (고양이)
mamma → [ˈmammə] (엄마)
pizza → [ˈpittsə] (피자)
```

### 2.3 구개음화
라틴어 -ct- → 나폴리어 -tt-:
```
라틴어 noctem → 나폴리어 notte [ˈnottə]
라틴어 factum → 나폴리어 fatto [ˈfattə]
```

### 2.4 메타포니 (Metaphony)
어말 -i, -u가 강세 모음에 영향:
```
ɛ → e / ___ C(C)i
ɔ → o / ___ C(C)u
피에데 (발, 단수) → 피에디 (발, 복수)
```
