# WIHP: Mapuzugun (Mapudungun) - arn

**어족**: Language isolate (Araucanian family)
**문자**: Latin alphabet (Mapuche Unified Alphabet - Alfabeto Mapuche Unificado)
**화자 수**: 약 200,000명 (칠레, 아르헨티나)
**난이도**: ★★★★☆ (교착어, 풍부한 증거성 체계, 특수 음운)

## 2. 특수 음운 현상

### 2.1 치간음 (Interdental)
마푸둥운은 d와 z를 치간음으로 발음:
```
d [ð] - 유성 치간 마찰음 (영어 'this')
z [θ] - 무성 치간 마찰음 (영어 'think')
```

### 2.2 권설음
```
tr [ʈ] - 권설 파열음
ll [ɭ] - 권설 측면음
```

### 2.3 6모음 체계
에스파냐어(5모음)와 달리 /ɨ/ (ü) 포함

### 2.4 어두 자음군
마푸둥운은 복잡한 어두 자음군 허용:
```
trewa [ʈɾewa] - 개
ptra [pt̪ɾa] - 어디든
```

## 9. 문법 특징

### 9.1 교착어
마푸둥운은 동사에 많은 접미사가 붙는 교착어:
```
küme-nie-fuy-nge-fu-ymi
좋다-가지다-완료-증거-가정-2인칭단수
"당신이 좋은 것을 가졌을 것입니다"
```

### 9.2 증거성(Evidentiality)
화자가 정보를 어떻게 얻었는지 문법적으로 표시:
```
-rke (직접 목격)
-fuy (간접 정보)
-ngey (추정)
```

### 9.3 어순
기본 어순은 SVO이지만 유연함.

### 9.4 인칭 표시
동사에 주어와 목적어를 모두 표시하는 복잡한 체계.
