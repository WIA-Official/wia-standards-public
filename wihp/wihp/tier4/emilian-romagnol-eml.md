## 2. 특수 음운 현상

### 2.1 비음모음 발달
북부 이탈리아어의 특징:
```
bene → bēn [bɛ̃ː] "잘"
mano → mān [mɛ̃ː] "손"
vino → vīn [vĩː] "와인"
```

### 2.2 자음 약화
어말 자음 탈락:
```
grande → grand [ɡrɛ̃ːd] "큰"
forte → fōrt [fɔːrt] "강한"
```

### 2.3 PL-, CL-, FL- 유지
라틴어 자음군 보존:
```
PLUS → pl** [pju] "더"
CLAMARE → ciamēr [tʃaˈmeːr] "부르다"
```

### 2.4 에밀리아 vs 로마냐 차이
```
에밀리아: pōrc [pɔːrtʃ] "돼지"
로마냐: pōrz [pɔːrts] "돼지"
```
