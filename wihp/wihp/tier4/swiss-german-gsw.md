## 2. 특수 음운 현상

### 2.1 K-음 강화
스위스 독일어의 가장 두드러진 특징:
```
표준 독일어 /k/ → 스위스 독일어 /x/
Kind → Chind [xint] 아이
kaufen → chaufe [ˈxaufə] 사다
```

### 2.2 축약형
구어에서 많은 축약:
```
isch es → isches [ˈiʃəs] "그것은 ~이다"
hän ich → häni [ˈhæni] "나는 ~을 가졌다"
```

### 2.3 방언 다양성
취리히, 베른, 바젤 등 지역별 차이 큼:
```
취리히: Grüezi [ˈɡryətsi]
베른: Grüessech [ˈɡryəsəx]
바젤: Grüss Gott [ɡryːs ɡɔt]
```
