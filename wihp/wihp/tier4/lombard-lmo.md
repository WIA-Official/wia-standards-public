## 2. 특수 음운 현상

### 2.1 어말 자음 탈락
롤바르드어의 가장 특징적인 현상 - 많은 어말 자음 탈락:
```
라틴어 tempus → 롬바르드어 temp [tɛmp]
라틴어 paneum → 롬바르드어 pan [paŋ]
라틴어 bellum → 롬바르드어 bell [bɛl]
```

### 2.2 전설 원순 모음
프랑스어/독일어와 유사:
```
이탈리아어 ora → 롬바르드어 öra [øra]
이탈리아어 burro → 롬바르드어 beur [bœr]
```

### 2.3 자음군 단순화
복잡한 자음군은 단순화:
```
라틴어 scriptum → 롬바르드어 scritt [skrit]
라틴어 septem → 롬바르드어 sett [sɛt]
```

### 2.4 주어 대명사 필수
이탈리아어와 달리 주어 대명사 필수:
```
롬바르드어: Mi parli (나는 말한다)
```
