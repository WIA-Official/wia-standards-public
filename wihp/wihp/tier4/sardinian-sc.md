## 2. 특수 음운 현상

### 2.1 라틴어 /k/, /ɡ/ 보존
다른 로망스어와 달리 구개음화 없음:
```
Latin: caelum → kelu [ˈkelu] 하늘 (cf. It. cielo)
Latin: gentem → gente [ˈɡɛnte] 사람들 (cf. It. gente [ˈdʒɛnte])
```

### 2.2 정관사 체계
독특한 정관사 시스템:
```
남성 단수: su [su]
여성 단수: sa [sa]
남성 복수: sos [sos]
여성 복수: sas [sas]
```

### 2.3 중자음 (Gemination)
의미 구분에 핵심적:
```
pala [ˈpala] "삽" vs. palla [ˈpalːa] "공"
note [ˈnoːte] "밤" vs. notte [ˈnɔtːe] "밤"
```

### 2.4 방언 구분
```
Logudorese (북부): 가장 보수적, 표준어 기반
Campidanese (남부): 더 혁신적
Sassarese (북서): 코르시카어 영향
Gallurese (북동): 코르시카어 유형
```
