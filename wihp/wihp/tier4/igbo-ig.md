## 2. 특수 음운 현상

### 2.1 성조 체계
이보어는 3개의 성조를 가집니다:
```
àkwà (저저) - 천
ákwà (고고) - 울음
àkwá (저고) - 다리
```

### 2.2 모음 조화
+ATR/-ATR 모음 조화:
```
+ATR: i, e, o, u
-ATR: ị, ọ, ụ, a
```
