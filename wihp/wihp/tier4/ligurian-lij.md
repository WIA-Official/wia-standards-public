## 2. 특수 음운 현상

### 2.1 비음 모음 (Nasal Vowels)
프랑스어와 매우 유사한 비음 모음 체계:
```
pan [pɑ̃] (빵)
vin [vĩ] (와인)
bon [bɔ̃] (좋은)
eun [œ̃] (하나)
```

### 2.2 전설 원순 모음
프랑스어 영향:
```
이탈리아어 il → 리구리아어 eu [œ]
이탈리아어 fiore → 리구리아어 sciô [ʃø]
```

### 2.3 이중 자음 보존
이탈리아어와 유사하게 이중 자음 보존:
```
bello [ˈbɛllu] (아름다운)
gatto [ˈɡattu] (고양이)
mamma [ˈmamma] (엄마)
```

### 2.4 관사 체계
리구리아어 특유의 관사:
```
o = 남성 단수 (이탈리아어 il)
a = 여성 단수 (이탈리아어 la)
i = 남성/여성 복수 (이탈리아어 i/le)
```

### 2.5 주어 대명사 필수
이탈리아어와 달리 주어 대명사 필수:
```
리구리아어: Me parlo (나는 말한다)
이탈리아어: Parlo (말한다)
```
