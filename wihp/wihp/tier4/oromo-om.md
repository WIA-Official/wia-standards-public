## 2. 특수 음운 현상

### 2.1 장단 모음 대립
장단 모음으로 의미 구별:
```
lafa [lafa] - (단) 땅 (한 조각)
laafa [laːfa] - (장) 땅 (일반)
```

### 2.2 SOV 어순
주어-목적어-동사 순서:
```
Inni kitaaba dubbifte.
그 책 읽었다
"그는 책을 읽었다"
```
