## 2. 특수 음운 현상

### 2.1 성조 악센트 (Grave Accent)
파피아멘투는 grave accent (`)를 사용하여 개모음을 표시합니다:
```
è = /ɛ/ (개모음): pèrt [pɛrt] 잃다
e = /e/ (폐모음): de [de] ~의
ò = /ɔ/ (개모음): kòp [kɔp] 컵
o = /o/ (폐모음): bom [bom] 좋은
```

### 2.2 크레올 형성 과정
파피아멘투는 포르투갈어, 스페인어, 네덜란드어, 서아프리카 언어들의 혼합으로 형성되었습니다:
- 어휘: 주로 포르투갈어/스페인어 (70-80%)
- 문법: 단순화된 크레올 구조
- 영향: 네덜란드어, 아라와크어, 서아프리카 언어

### 2.3 단순 문법 구조
```
주어 + 동사 + 목적어 (SVO)
Mi ta kome pan (나는 빵을 먹는다)
주어: Mi (나)
동사마커: ta (현재진행)
동사: kome (먹다)
목적어: pan (빵)
```

## 9. 문법 노트 (Grammar Notes)

### 9.1 시제 마커 (Tense Markers)
```
현재: ta + 동사
Mi ta kome (나는 먹는다)

과거: a + 동사
Mi a kome (나는 먹었다)

미래: lo + 동사
Mi lo kome (나는 먹을 것이다)
```

### 9.2 부정 (Negation)
```
no + 동사마커 + 동사
Mi no ta kome (나는 먹지 않는다)
Mi no a kome (나는 먹지 않았다)
```

### 9.3 소유격
```
di + 소유자
kas di mi (나의 집)
buki di Maria (마리아의 책)
```
