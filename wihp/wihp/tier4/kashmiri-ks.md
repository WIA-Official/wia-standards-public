## 2. 특수 음운 현상

### 2.1 성조 체계
카슈미르어는 고저 악센트를 가집니다:
```
کۄر [kóːr] (고음) - 딸
کور [kòːr] (저음) - 장님
```

### 2.2 권설음 (Retroflexes)
혀를 말아서 발음:
```
ٹ [ʈ], ڈ [ɖ], ݨ [ɳ], ڑ [ɽ]
```

### 2.3 유기음 (Aspirated)
숨을 내쉬며 발음:
```
پھ [pʰ], تھ [tʰ], کھ [kʰ], چھ [t͡ʃʰ]
```

### 2.4 비음화
모음 뒤 비음 마커:
```
ہٕنٛد [hɨ̃nd] - 인도
گٔنٛڑ [ɡɨ̃ɽ] - 매듭
```
