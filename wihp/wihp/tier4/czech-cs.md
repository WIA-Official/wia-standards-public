## 2. 특수 음운 현상

### 2.1 ř 음소 (Unique Czech Sound)
- 세계에서 체코어에만 있는 독특한 음소
- /r/와 /ʒ/의 동시 조음
- IPA: [r̝] 또는 [r̝̊]

```
řeka [r̝ɛka] - 강
Dvořák [dvɔr̝aːk] - 드보르작 (작곡가)
tři [tr̝ɪ] - 셋
```

### 2.2 자음군 (Consonant Clusters)
체코어는 복잡한 자음군을 허용합니다:
```
strč prst skrz krk - "목구멍으로 손가락을 찔러라"
[str̩tʃ pr̩st skr̩s kr̩k]
(모음 없이 음절 형성 가능)
```

### 2.3 성절 자음 (Syllabic Consonants)
/r/과 /l/이 모음 역할을 할 수 있습니다:
```
vlk [vl̩k] - 늑대
krk [kr̩k] - 목
prst [pr̩st] - 손가락
```

## 9. 문법 참고 (Grammar Notes)

### 9.1 격 체계 (7격)
체코어는 7개의 격을 사용합니다:
1. **Nominativ** (주격): Kdo? Co?
2. **Genitiv** (속격): Koho? Čeho?
3. **Dativ** (여격): Komu? Čemu?
4. **Akuzativ** (대격): Koho? Co?
5. **Vokativ** (호격): 부르기
6. **Lokál** (처소격): O kom? O čem?
7. **Instrumentál** (구격): Kým? Čím?

### 9.2 성 (Gender)
- **Mužský rod** (남성): student, hrad
- **Ženský rod** (여성): žena, škola
- **Střední rod** (중성): město, moře

### 9.3 어순
기본 어순은 SVO이지만 유동적입니다:
- Já čtu knihu. (나는 책을 읽는다)
- Knihu čtu já. (책을 읽는 건 나다 - 강조)
