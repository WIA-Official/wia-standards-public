# WIHP: Vaďďa tšeeli (Votic) - vot

**ISO 코드**: vot
**언어명**: Votic / Vaďďa tšeeli / Vadja keel / 보티크어
**어족**: Uralic > Finnic > Votic
**화자 수**: 약 4-10명 (2024년 기준, 모두 고령)
**난이도**: ★★★★★★ (복잡한 자음 체계, 4개 길이 등급, 고대 특징)
**상태**: Critically Endangered, Nearly Extinct (UNESCO)
**작성일**: 2025-12-22
**버전**: 1.0 (A-grade)

---

## 개요

보티크어(Vaďďa tšeeli)는 러시아 레닌그라드주(상트페테르부르크 인근) Luuga강과 Neva강 사이 지역에서 사용되던 핀란드어군 언어입니다. 2024년 기준 **모어 화자 4-10명**으로 **거의 멸종 상태**에 있으며, 모두 70세 이상 고령자입니다.

보티크어는 에스토니아어, 핀란드어와 가까우나 **4단계 길이 구분**, **복잡한 자음 체계**, **이중 모음 조화** 등 독특한 특징을 가지고 있습니다. 19세기까지 약 5,000명이 사용했으나 러시아화 정책으로 급격히 감소했습니다. 현재 언어학자들이 마지막 화자들과 협력하여 긴급 기록 작업을 진행 중입니다.

## 2. 특수 음운 현상

### 2.1 4단계 길이 등급 (Quantity)
보티크어는 **세계에서 유일하게 4단계 길이를 구분**하는 언어입니다:

```
Q1 (초단): lina [linɑ] - 아마
Q2 (단): linna [linːɑ] - 도시로
Q3 (장): linna [linːːɑ] - 도시의
Q4 (초장): linna [linːːːɑ] - 도시에! (강조)
```

이는 에스토니아어의 3단계보다 더 세밀한 구분입니다.

### 2.2 모음 조화 (Vowel Harmony)
보티크어는 이중 모음 조화를 가집니다:
- 전설 모음: ä, ö, ü
- 후설 모음: a, o, u
- 중립: e, i, õ

접미사도 조화:
- talossa (집에서) - 후설
- tšeelessä (언어에서) - 전설

### 2.3 자음 등급 교체 (Gradation)
```
강형 → 약형
kk → k: kukko → kukon (수탉)
pp → p: kappale → kappalee (조각)
tt → t: katto → katton (지붕)
k → ∅: jalka → jalla (다리)
p → v: tapa → tavan (습관)
t → d: katu → kadun (거리)
```

### 2.4 경구개화 (Palatalization)
/ď/ (ď)는 보티크어 특유의 경구개 파열음:
```
vaďďa [vɑdːʲɑ] - 보티크족
maďďo [mɑdːʲo] - 뱀
```

## 10. 언어 보존 및 긴급 기록

### 10.1 역사적 배경
- **중세**: Luuga강-Neva강 유역에 약 20,000명
- **19세기**: 약 5,000명 (Sjögren 조사)
- **1926**: 소련 인구조사 705명
- **1989**: 62명
- **2010**: 68명 신고 (대부분 비유창)
- **2024**: 모어 화자 4-10명 (모두 70세 이상)

### 10.2 현재 위기 상황
**마지막 화자들**:
- Zinaida Chernyakova (1930년생) - Jõgõperä 마을
- Yelena Maksimova (1940년생) - Luuga 지역
- 기타 4-6명 (대부분 부분적 유창성)

**멸종 위기 요인**:
1. 러시아화 정책 (소련 시대)
2. 도시 이주
3. 러시아어와의 언어 교체
4. 세대 간 전승 단절
5. 경제적 압박

### 10.3 긴급 기록 프로젝트
**주요 프로젝트**:
- **ELDIA** (European Language Diversity for All, 2010-2013)
- **Endangered Languages Archive** (SOAS, London)
- **Institute for Linguistic Studies** (러시아과학원, 상트페테르부르크)
- **Helsinki University** 보티크어 디지털 아카이브

**기록 내용**:
- 구술 역사 (200+ 시간)
- 민요 및 시가
- 일상 대화
- 전통 지식
- 비디오/오디오 아카이브

### 10.4 부흥 시도
2020년대부터 소규모 부흥 운동:
- 언어 교실 (상트페테르부르크)
- 온라인 학습 자료
- 젊은 활동가 그룹 (10-15명)
- 소셜 미디어 캠페인

**그러나**: 원어민 화자 고령화로 시간이 매우 촉박한 상황
