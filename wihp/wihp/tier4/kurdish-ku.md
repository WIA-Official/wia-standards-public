## 2. 특수 음운 현상

### 2.1 인두음 (Pharyngeal Consonants)
쿠르드어의 특징적인 목구멍 소리:
```
ḧ [ħ] - 무성 인두 마찰음
' [ʕ] - 유성 인두 마찰음
```

### 2.2 구개수음 (Uvular Consonants)
목젖 부분에서 발음:
```
q [q] - 무성 구개수 파열음
x [x] - 무성 연구개/구개수 마찰음
ẍ [ɣ] - 유성 연구개/구개수 마찰음
```

### 2.3 모음 조화
장모음과 단모음 구분이 의미 차이를 만듭니다:
```
pir [pir] - 노인
pîr [piːr] - 성자
```

## 10. 발음 주의사항

### 10.1 인두음 발음
- **ḧ [ħ]**: 목구멍을 좁혀 공기를 내보내며 발음 (아랍어 ح와 유사)
- **' [ʕ]**: 목구멍을 조이며 발음 (아랍어 ع와 유사)

### 10.2 구개수음 발음
- **q [q]**: k보다 더 뒤쪽(목젖 부분)에서 발음
- **x [x]**: 독일어 'ach'의 ch와 유사
- **ẍ [ɣ]**: 프랑스어 'r'과 유사한 유성음

### 10.3 장단모음 구별
