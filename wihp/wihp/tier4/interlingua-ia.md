## 2. 설계 철학 (Design Philosophy)

### 2.1 자연주의적 접근
인터링구아는 "국제 단어"를 사용합니다:
```
영어, 프랑스어, 이탈리아어, 스페인어, 포르투갈어에서
공통된 어근을 찾아 사용
→ 학습 없이도 이해 가능한 언어
```

### 2.2 문법적 단순성
```
- 동사 변화 없음 (io es, tu es, ille es)
- 명사 복수형: -s 추가
- 형용사는 명사 앞이나 뒤 가능
- 어순: 주어-동사-목적어 (SVO)
```

### 2.3 IALA 원칙 (International Auxiliary Language Association)
```
1. 최대 이해성 (Maximum intelligibility)
2. 자연성 (Naturalistic vocabulary)
3. 국제성 (International recognition)
```

## 8. 문법 (Grammar)

### 8.1 동사 시제
인터링구아는 단순한 시제 체계를 사용합니다:

| 시제 | 형태 | 예시 (parlar = 말하다) |
|------|------|-------------------------|
| 현재 | 원형 | io parla (나는 말한다) |
| 과거 | -va | io parlava (나는 말했다) |
| 미래 | va + 동사 | io va parlar (나는 말할 것이다) |
| 조건 | area + 동사 | io area parlar (나는 말하겠다) |

### 8.2 명사와 형용사
```
명사 복수: -s 추가
  libro → libros (책 → 책들)

형용사는 불변:
  un bon libro (좋은 책)
  duo bon libros (두 개의 좋은 책들)
```

### 8.3 어순
```
기본 어순: SVO (주어-동사-목적어)
  Io lege le libro.
  (나는 읽는다 그 책을)

의문문: 어순 유지 + 억양
  Tu lege le libro? (너는 그 책을 읽니?)
```

## 9. 철학적 개념 (Philosophical Concepts)

### 9.1 국제성 (Internationality)
| 인터링구아 | 의미 |
|------------|------|
| International comprension | 국제적 이해 |
| Vocabulario natural | 자연스러운 어휘 |
| Simplicitate | 단순성 |
| Claritate | 명확성 |
| Neutralitate cultural | 문화적 중립성 |

### 9.2 IALA 핵심 원칙
```
1. Maxime intelligibilitate
   (최대 이해성 - 학습 없이도 이해 가능)

2. Naturalismo
   (자연주의 - 인공적이지 않은 어휘)

3. Internationalitate
   (국제성 - 광범위하게 알려진 단어 사용)

4. Facilitate
   (용이성 - 쉬운 문법과 발음)
```

### 9.3 과학적 접근
```
인터링구아는 언어학적 연구를 바탕으로:
- 로망스어 공통 어근 추출
- 국제 과학 용어 통합
- 통계적 분석으로 단어 선정
```
