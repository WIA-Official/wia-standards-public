## 2. 특수 음운 현상

### 2.1 중성 명사 (Neutro de materia)
아스투리아어의 독특한 문법 특징:
```
남성: el lleche "우유 한 잔"
중성: lo lleche "우유(물질)"

남성: el vinu "포도주 병"
중성: lo vinu "포도주(일반)"
```

### 2.2 디프통화 (Diphthongization)
라틴어 /ɛ/, /ɔ/ → ie, ue:
```
Latin: terra → tierra [ˈtjera] 땅
Latin: porta → puerta [ˈpweɾta] 문
```

### 2.3 어두 f- 보존
카스티야어와 달리 라틴어 f- 유지:
```
Asturian: fíu [ˈfiu] vs. Spanish: hijo [ˈixo] "아들"
Asturian: fueya vs. Spanish: hoja "잎"
```

### 2.4 방언 구분
```
Occidental (서부): 갈리시아어 영향
Central (중부): 표준어 기반
Oriental (동부): 카스티야어 영향
```
