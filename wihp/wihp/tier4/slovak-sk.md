## 2. 특수 음운 현상

### 2.1 리듬 법칙 (Rhythmic Law)
슬로바키아어 고유의 규칙: 두 개의 장음절이 연속할 수 없습니다.
```
krásny [kraːsnɪ] (아름다운) - 올바름
*krásný - 불가능 (두 번째 장음 단축)
```

### 2.2 성절 자음 (Syllabic Consonants)
/r/과 /l/이 모음처럼 음절을 형성할 수 있습니다:
```
vlk [vl̩k] - 늑대
prst [pr̩st] - 손가락
krk [kr̩k] - 목
```

### 2.3 연자음 (Soft Consonants)
ď, ť, ň, ľ는 구개음화된 연자음입니다:
```
ďakujem [ɟakujɛm] - 감사합니다
ťava [cava] - 낙타
```

## 9. 문법 참고 (Grammar Notes)

### 9.1 격 체계 (6격)
체코어와 달리 슬로바키아어는 6개의 격을 사용합니다:
1. **Nominatív** (주격): Kto? Čo?
2. **Genitív** (속격): Koho? Čoho?
3. **Datív** (여격): Komu? Čomu?
4. **Akuzatív** (대격): Koho? Čo?
5. **Lokál** (처소격): O kom? O čom?
6. **Inštrumentál** (구격): Kým? Čím?

### 9.2 성 (Gender)
- **Mužský rod** (남성): student, hrad
- **Ženský rod** (여성): žena, škola
- **Stredný rod** (중성): mesto, more

### 9.3 체코어와의 차이
| 특징 | 체코어 | 슬로바키아어 |
|------|--------|--------------|
| 격 수 | 7격 (호격 포함) | 6격 |
| ř 음소 | 있음 [r̝] | 없음 |
| 리듬 법칙 | 없음 | 있음 |
| ô 모음 | 없음 | 있음 [uo] |
