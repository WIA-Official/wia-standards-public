## 개요

페루 스페인어는 케추아어(Quechua)의 강력한 영향, 안데스 vs 해안 지역의 뚜렷한 차이, 그리고 보수적인 발음이 특징입니다. 리마 방언과 안데스 방언은 상당히 다릅니다.

## 페루 스페인어 특징

### 주요 음운 특징

| 특징 | 설명 | 예시 |
|------|------|------|
| **세세오 (Seseo)** | z, c(e,i) = /s/ | zapato → /saˈpato/ |
| **예이스모 (Yeísmo)** | ll = y = /ʝ/ | calle = caye → 카예 |
| **케추아 차용어** | 고유 음운 | papa, choclo |
| **안데스: 모음 축약** | e/i, o/u 혼동 | piru (pero) |
| **해안: -s 약화** | 리마에서 흔함 | lo(h) libros |

### 지역별 차이

| 지역 | 특징 | 예시 |
|------|------|------|
| **Lima (리마)** | -s 약화, 빠름 | lo(h) niño(h) |
| **Andes (안데스)** | 케추아 영향, 모음 혼동 | e/i, o/u |
| **Costa (해안)** | 리마와 유사 | -s 약화 |
| **Selva (정글)** | 천천히, 명료 | 명확한 발음 |

### 안데스 vs 해안

| 특징 | 안데스 (Cusco) | 해안 (Lima) | WIHP |
|------|---------------|------------|------|
| -s 발음 | 유지 /s/ | 약화 /h/ | ㅅ/- |
| e/i, o/u | 혼동 (케추아) | 구분 | - |
| 발화 속도 | 느림 | 빠름 | - |
| ll/y | /ʎ/ 일부 유지 | /ʝ/ | 이/야 |

## 케추아어 영향 (Quechua Influence)

페루 스페인어의 가장 큰 특징은 케추아어의 영향입니다.

### 음운적 영향 (안데스 지역)

| 특징 | 케추아 영향 | 예시 | 표준 | WIHP |
|------|-----------|------|------|------|
| e/i 혼동 | 3모음 체계 | piru | pero | 피루 |
| o/u 혼동 | - | dicir | decir | 디시르 |
| 억양 | 평탄 | - | - | - |

### 문법적 영향

| 특징 | 케추아 영향 | 예시 |
|------|-----------|------|
| 과거분사 | -endo → -ando | comiendo → comindo |
| 진행형 과용 | - | está siendo (is being) |
| 전치사 탈락 | - | voy Lima (voy a Lima) |
