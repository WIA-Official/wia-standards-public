## 2. 특수 음운 현상

### 2.1 성조 체계 (Tone System)
은듀카는 두 가지 성조를 가지고 있습니다:
- 고조 (H): 높은 음
- 저조 (L): 낮은 음

```
성조 예시:
wáka [wáka] (H-L) = 걷다
waká [waká] (L-H) = 걸음
wata [wata] (L-L) = 물
```

철자에서 성조는 보통 표시되지 않지만, 의미 구별에 중요합니다.

### 2.2 마룬 크레올 특성
은듀카는 17-18세기 탈출 노예들이 내륙에서 발달시킨 언어입니다:
- 어휘: 영어 (50%), 아프리카 언어 (30%), 포르투갈어/네덜란드어 (20%)
- 문법: 강한 서아프리카 구조 유지
- 성조: 아칸어, 기쿨루어 등 아프리카 언어의 영향
- 고립 발달: 유럽 영향에서 상대적으로 자유로운 발달

### 2.3 자음 특성
```
영어 r → 은듀카 l
river → liba [liba] 강
river → 은듀카에서 l로 변화
```

### 2.4 모음 길이 구별
모음 길이가 의미를 구별합니다:
```
so [so] = 그래서
soo [soː] = 부드러운
```

## 9. 문법 노트 (Grammar Notes)

### 9.1 TMA 시스템 (Tense-Mood-Aspect)
```
현재/습관: e + 동사
Mi e waka (나는 걷는다)

과거: ben + 동사
Mi ben waka (나는 걸었다)

미래: o / sa + 동사
Mi o waka (나는 걸을 것이다)
Mi sa waka (나는 걸을 것이다)

완료: kaba
Mi kaba nyannyan (나는 먹었다/식사를 마쳤다)
```

### 9.2 부정 (Negation)
```
no + 동사구
Mi no e waka (나는 걷지 않는다)
Mi no ben waka (나는 걷지 않았다)
```

### 9.3 소유격
```
fu + 소유자
A hoo fu mi (나의 집)
A buku fu en (그의/그녀의 책)
```

### 9.4 복수 표시
```
den = 복수 (명사 뒤)
A piki den (아이들)
Mi mati den (나의 친구들)
```

### 9.5 연속동사 구문
은듀카는 연속동사를 많이 사용합니다:
```
Mi teke buku go (나는 책을 가져가다)
직역: 나는 잡다 책 가다
```

### 9.6 의문문
```
의문사 + 문장
Pee yu e go? (어디 가세요?)
Omeni a de? (얼마입니까?)
Fa a musu de? (어떻게 해야 하나요?)
```

## 10. 아프리카 언어 영향 (African Language Influence)

### 10.1 아칸어 (Akan) 영향
많은 단어가 가나의 아칸어에서 유래:
```
Ndyuka → Akan
akuba (여성 이름) → akua (수요일에 태어난 여성)
kwasiba (남성 이름) → kwasi (일요일에 태어난 남성)
```

### 10.2 기쿨루어 (Kikongo) 영향
콩고 지역 언어의 흔적:
```
Ndyuka → Kikongo
nzambi (신) → nzambi (신)
```

### 10.3 아프리카식 명명 체계
요일별 이름 시스템 (아칸어 유래):
- 월요일: Kwadyo (남), Adjuba (여)
- 화요일: Kwabena (남), Abena (여)
- 수요일: Kwaku (남), Akuba (여)
- 목요일: Yaw (남), Amba (여)
- 금요일: Kofi (남), Afia (여)
- 토요일: Kwame (남), Ama (여)
- 일요일: Kwasiba (남), Akosua (여)
