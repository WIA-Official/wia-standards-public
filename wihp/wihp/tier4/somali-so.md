## 2. 특수 음운 현상

### 2.1 성조 체계
소말리어는 고저 성조로 의미를 구별:
```
ínan [ínan] 고성조 - 소년
inán [inán] 고성조 - 소녀
```

### 2.2 초점 표지
주어/목적어에 따라 동사 접미사 변화:
```
Wiilku waa seexday. 소년은 잤다.
Wiilka baan arkay. 나는 소년을 보았다.
```
