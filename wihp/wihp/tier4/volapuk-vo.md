## 2. 설계 철학 (Design Philosophy)

### 2.1 슐라이어의 비전 (Schleyer's Vision)
```
1879년 독일 가톨릭 신부 요한 마르틴 슐라이어가 창안:
"하나님께서 계시해주신 보편 언어"
- 세계 평화와 형제애를 위한 언어
- 모든 인류가 소통할 수 있는 도구
```

### 2.2 단어 형성 원리
```
유럽 언어(주로 영어)에서 어근을 가져와 변형:
  World → Vol → Volapük (vol-a-pük = 세계의 언어)
  Speak → Pük
  English → Linglänapük

변형 규칙:
- R을 회피 (발음의 어려움 때문)
- 단음절 어근 선호
- 모든 단어는 자음으로 끝남
```

### 2.3 복잡한 문법 체계
```
명사 격 변화: 4개
  주격 (-): dom (집이)
  속격 (-a): doma (집의)
  여격 (-e): dome (집에게)
  대격 (-i): domi (집을)

동사 시제: 16개 (!)
  현재, 과거, 미래, 완료...

사람 표시 어미:
  -ob (나는), -ol (너는), -om (그는)...
```

## 8. 문법 (Grammar)

### 8.1 격 변화 체계 (Case System)
```
명사는 4개의 격으로 변화합니다:

주격 (Nominative): -
  Dom binon gletik. (집은 크다)

속격 (Genitive): -a
  Doma fat. (집의 아버지)

여격 (Dative): -e
  Givob dome buki. (나는 집에 책을 준다)

대격 (Accusative): -i
  Logob domi. (나는 집을 본다)
```

### 8.2 동사 시제와 인칭 변화
```
기본 형태: 동사 어근 + 시제 + 인칭

현재형:
  binob (나는 ~이다)
  binol (너는 ~이다)
  binom (그는 ~이다)
  binobs (우리는 ~이다)
  binols (너희는 ~이다)
  binoms (그들은 ~이다)

과거형: ä- 접두사
  äbinob (나는 ~였다)

미래형: o- 접두사
  obinob (나는 ~일 것이다)

완료형: e- 접두사
  ebinob (나는 ~였었다)
```

### 8.3 복수형
```
명사 복수: -s 추가
  dom → doms (집 → 집들)

격 변화는 복수형에도 적용:
  domas (집들의)
  domis (집들을)
```

### 8.4 어순
```
기본 어순: SOV 또는 SVO 모두 가능
  Logob domi. (나는 집을 본다)
  Domi logob. (집을 나는 본다)

격 변화가 역할을 표시하므로 어순 자유로움
```

```
   - le- 접두사 = 과거/오래된
```

```
- 언어 구축의 실험 모델 제공
- "universal language" 개념을 대중화

현재: 소수의 열성적인 사용자가 유지
```

## 11. 볼라퓌크 기도문 (Lord's Prayer in Volapük)

```
O Fat obas, kel binol in süls,
(오 우리의 아버지, 하늘에 계신)

paisaludomöz nem ola!
(당신의 이름이 거룩히 여겨지기를!)

Kömomöd monargän ola!
(당신의 왕국이 임하소서!)

Jenomöz vil olik,
(당신의 뜻이 이루어지기를,)

äs in sül, i su tal!
(하늘에서와 같이, 땅에서도!)
```
