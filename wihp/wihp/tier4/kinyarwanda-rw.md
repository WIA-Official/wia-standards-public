## 2. 특수 음운 현상

### 2.1 성조 체계
키냐르완다어는 고저 성조를 가집니다:
```
gútera [고-저-저] - 심다
gutéra [저-고-저] - 던지다
```

### 2.2 명사 부류
16개의 명사 부류:
```
umu-ntu (사람) → aba-ntu (사람들)
iki-ntu (물건) → ibi-ntu (물건들)
```
