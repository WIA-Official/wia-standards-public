## 2. 특수 음운 현상

### 2.1 자음 변이 (Initial Consonant Mutation)
3가지 유형:
```
원형: pen (머리)
연변이: ben (fy mhen - 내 머리)
비음변이: mhen (fy mhen - 나의 머리)
기음변이: phen (ei phen - 그녀의 머리)
```

### 2.2 VSO 어순
동사-주어-목적어:
```
Rydw i'n bwyta afal.
있다 나 먹다 사과
"나는 사과를 먹는다"
```
