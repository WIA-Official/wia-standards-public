## 2. 특수 음운 현상

### 2.1 측면 파찰음 (Lateral Affricate)
나우아틀어의 가장 특징적인 음소:
```
tl [t͡ɬ] - 무성 치경 측면 파찰음
웨일즈어와 나우아틀어에서만 흔히 발견
```

### 2.2 성문 파열음
```
h [ʔ] - 성문 파열음 (철자상 'h'로 표기)
예: huēhueh [weːweʔ] "노인"
```

### 2.3 순치화 연구개음
```
cu/uc [kʷ] - 순치화 연구개음
예: cuāuhtli [kʷaːwt͡ɬi] "독수리"
```

### 2.4 모음 장단 대립
```
atl [at͡ɬ] - 물 (단모음)
ātl [aːt͡ɬ] - 물 (장모음)
```

### 2.5 음절 구조
주로 CV, CVC 구조, 복잡한 자음군 허용:
```
tlālli [t͡ɬalːi] - 땅
xōchitl [ʃoːt͡ʃit͡ɬ] - 꽃
```

## 9. 문법 특징

### 9.1 포합어 (Polysynthetic)
나우아틀어는 동사에 많은 접사가 결합하여 복잡한 의미를 표현:
```
ni-mitz-tla-tlazohtla-tīz
나는-당신을-어떤것을-사랑하다-미래
"나는 당신을 사랑할 것입니다"
```

### 9.2 절대격 접미사
명사는 반드시 절대격 접미사(-tl, -tli, -li, -in)를 가집니다:
```
calli - 집
xōchitl - 꽃
āmatl - 종이
```

### 9.3 경어 체계
존칭 접미사 -tzin을 사용:
```
cihuātl - 여자
cihuātzin - 여성분 (존칭)
```

기본 어순은 VOS이지만 유연함.
