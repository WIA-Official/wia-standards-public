## 2. 특수 음운 현상

### 2.1 비음 조화 (Nasal Harmony)
과라니어에서 비음성은 형태소 전체로 퍼집니다:
```
he'i (말하다/구강)
hẽ'ĩ (달콤한/비강)
```

### 2.2 성문 폐쇄음 (Glottal Stop)
puso (배꼽 중앙)를 표시합니다:
```
ka'a [kaʔa] - 숲/마테
ka'aguy [kaʔaɦuɨ] - 숲
```

### 2.3 인칭 접두사
동사에 인칭 접두사가 붙습니다:
```
-guata (걷다)
aguata - 내가 걷다
reguata - 네가 걷다
oguata - 그가 걷다
```
