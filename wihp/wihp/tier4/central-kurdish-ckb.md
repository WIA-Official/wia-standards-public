## 2. 특수 음운 현상

### 2.1 아랍 문자 체계
소라니 쿠르드어는 수정된 페르시아-아랍 문자를 사용:
```
추가 문자: پ چ ژ گ ڤ ڵ ڕ ێ ۆ
```

### 2.2 인두음 (Pharyngeal Consonants)
목구멍에서 발음되는 독특한 소리:
```
ح [ħ] - 무성 인두 마찰음
ع [ʕ] - 유성 인두 마찰음
```

### 2.3 특수 자음
```
ڵ [ɫ] - 연구개화된 l (dark L)
ڕ [ɹ] - 치경 접근음 (영어 r과 유사)
```

### 2.4 이즈피 (Ezafe) 연결
명사와 형용사를 연결하는 문법 표지:
```
کتێبی باش [ktêbî bash] - 좋은 책
(کتێب book + ی ezafe + باش good)
```

## 10. 발음 주의사항

### 10.1 아랍 문자 읽기
소라니는 오른쪽에서 왼쪽으로 씁니다:
```
سڵاو (sillaw) = ل + ا + و + س
```

### 10.2 인두음 발음
- **ح [ħ]**: 목구멍을 좁혀 공기를 내보내며 발음
- **ع [ʕ]**: 목구멍을 조이며 발음

### 10.3 특수 자음
- **ڵ [ɫ]**: 영어 'all'의 l과 유사한 어두운 L
- **ڕ [ɹ]**: 영어 'red'의 r과 유사한 접근음

### 10.4 모음 표시
아랍 문자에서 단모음은 생략되는 경우가 많으며, 장모음은 반드시 표시:
```
کتێب [ktêb] = 책
(ک + ت + ێ + ب)
```

## 11. 문법 특징

### 11.1 이즈피 (Ezafe)
명사 연결 표지 (-î/-ê):
```
کتێبی من = my book (kitêbî min)
مناڵی باش = good child (minallî bash)
```

### 11.2 정관사
앞에 붙는 정관사 ئەو:
```
کتێب = 책 (kitêb)
ئەو کتێبە = 그 책 (ew kitêbe)
```

### 11.3 동사 시제
과거형 접두사 (de-):
```
دەچم = 나는 간다 (dechim)
چووم = 나는 갔다 (chûm)
```
