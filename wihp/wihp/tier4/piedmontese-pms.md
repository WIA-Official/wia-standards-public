## 2. 특수 음운 현상

### 2.1 비음 모음 (Nasal Vowels)
프랑스어와 유사한 비음 모음 체계:
```
pan [pɑ̃] (빵)
vin [vĩ] (와인)
bon [bɔ̃] (좋은)
eun [œ̃] (하나)
```

### 2.2 전설 원순 모음
프랑스어/독일어 영향:
```
이탈리아어 ore → 피에몬테어 eure [œr]
이탈리아어 può → 피에몬테어 peu [pø]
```

### 2.3 어말 자음 탈락
많은 어말 자음이 탈락:
```
라틴어 tempus → 피에몬테어 temp [tɛmp]
라틴어 bellum → 피에몬테어 bel [bɛl]
라틴어 cantus → 피에몬테어 cant [kɑ̃t]
```

### 2.4 주어 대명사 필수
이탈리아어와 달리 주어 대명사 필수:
```
피에몬테어: Mi i parlo (나는 말한다)
이탈리아어: Parlo (말한다)
```

### 2.5 대명사 중복 (Clitic Doubling)
주어 대명사 + 보조 대명사:
```
Mi i son (나는 ~이다)
Ti it ses (너는 ~이다)
Chiel a l'é (그는 ~이다)
```
