## 2. 특수 음운 현상

### 2.1 고유 자음 ѕ [d͡z]
마케도니아어만의 독특한 자음:
```
ѕвезда [d͡zvezda] - 별
ѕид [d͡zid] - 벽
```

### 2.2 연구개음 ќ [c]와 ѓ [ɟ]
불가리아어에 없는 마케도니아어 고유 음소:
```
ќе [ce] - ~할 것이다
ѓубре [ɟubre] - 쓰레기
```

### 2.3 정관사 후치
불가리아어처럼 정관사가 명사 뒤에 붙습니다:
```
книга (책) → книгата (그 책)
човек (사람) → човекот (그 사람)
```
