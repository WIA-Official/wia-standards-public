## 2. 특수 음운 현상

### 2.1 3계열 대립 (Tripartite System)
아이마라어는 파열음에서 평음-유기음-방출음의 3계열 대립을 보입니다:
```
p [p] - t [t] - ch [tʃ] - k [k] - q [q] (평음)
ph [pʰ] - th [tʰ] - chh [tʃʰ] - kh [kʰ] - qh [qʰ] (유기음)
p' [p'] - t' [t'] - ch' [tʃ'] - k' [k'] - q' [q'] (방출음/ejective)
```

### 2.2 방출음 (Ejectives)
성문을 닫은 상태에서 공기를 압축하여 발음:
```
p' [p'] - 양순 방출음
t' [t'] - 치경 방출음
ch' [tʃ'] - 경구개 방출음
k' [k'] - 연구개 방출음
q' [q'] - 후음 방출음
```

### 2.3 후음 (Uvular)
```
q [q] - 무성 후음 파열음
qh [qʰ] - 유기 후음 파열음
q' [q'] - 방출 후음 파열음
x [χ] - 무성 후음 마찰음
```

### 2.4 음절 구조
아이마라어는 (C)(C)V(C)(C) 구조를 허용하여 복잡한 자음군 가능.

## 9. 문법 특징

### 9.1 교착어
아이마라어는 풍부한 접미사 체계를 가진 교착어:
```
yati-ña-nak-tañ-wa
알다-명사화-복수-의무-확인
"알아야 합니다"
```

### 9.2 증거성 (Evidentiality)
화자가 정보를 어떻게 얻었는지 문법적으로 표시:
```
-wa (직접 경험)
-tayna (간접 보고)
-chi (추정)
-pacha (불확실)
```

### 9.3 4인칭
아이마라어는 독특하게 4인칭(화자와 청자를 제외한 타인)을 가집니다.

### 9.4 어순
기본 어순은 SOV (주어-목적어-동사).

### 9.5 포용-배제 구별
"우리"를 청자 포함(inclusive)과 배제(exclusive)로 구별.
