## 2. 특수 음운 현상

### 2.1 영어 기반 어휘
스라난 통고의 대부분 어휘는 영어에서 유래했지만 발음이 변형되었습니다:
```
English → Sranan
walk → waka [waka]
speak → taki [taki]
book → buku [buku]
house → oso [oso] (h 약화)
```

### 2.2 성조 체계
스라난 통고는 고저 성조를 가지고 있지만 철자에는 표시되지 않습니다:
```
ábi [ábi] (고-저) = 우리
abi [abi] (저-저) = 가지다
```

### 2.3 크레올 형성
스라난은 17세기 수리남 플랜테이션에서 형성:
- 어휘: 영어 (60%), 네덜란드어 (10%), 포르투갈어 (5%), 아프리카 언어 (25%)
- 문법: 서아프리카 언어 구조
- TMA 체계: 아프리카식 시제-상-양태 마커

## 9. 문법 노트 (Grammar Notes)

### 9.1 TMA 시스템 (Tense-Mood-Aspect)
```
현재/습관: e + 동사
Mi e wroko (나는 일한다)

과거: ben + 동사
Mi ben wroko (나는 일했다)

미래: o + 동사
Mi o wroko (나는 일할 것이다)

완료: kaba
Mi kaba wroko (나는 일을 끝냈다)
```

### 9.2 부정 (Negation)
```
no + 동사구
Mi no e wroko (나는 일하지 않는다)
Mi no ben wroko (나는 일하지 않았다)
```

### 9.3 소유격
```
fu + 소유자
A oso fu mi (나의 집)
A buku fu en (그의/그녀의 책)
```

### 9.4 복수 표시
```
(nen)den = 복수
Mi brada den (나의 형제들)
A pikin den (아이들)
```

### 9.5 의문문
```
의문사 + 문장
Pe yu e go? (어디 가세요?)
Omeni a de? (얼마입니까?)
```

## 10. 발음 규칙 (Pronunciation Rules)

### 10.1 자음군 단순화
영어 자음군이 단순화됩니다:
```
English → Sranan
strong → tranga
street → strati
black → blaka
```

### 10.2 어말 자음
영어 어말 자음이 모음과 함께 발음됩니다:
```
English → Sranan
walk → waka
talk → taki
work → wroko
```
