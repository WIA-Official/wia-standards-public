## 2. 특수 음운 현상

### 2.1 성조 체계
링갈라는 2개의 성조를 가집니다:
```
mokó (고성조) - 팔
mokò (저성조) - 쓴맛
```

### 2.2 반투어 명사 부류
명사 접두사로 부류를 구분:
```
mo-to (단수) → ba-to (복수) - 사람
li-nei (단수) → ma-nei (복수) - 길
```
