## 2. 특수 음운 현상

### 2.1 격변화 체계 (4격)
페로어는 아이슬란드어처럼 4격을 유지:
```
주격 (Nominative): Ein bátur (배 한 척)
대격 (Accusative): Eg síggi bátin (나는 배를 본다)
여격 (Dative): í bátinum (배 안에서)
속격 (Genitive): bátsins (배의)
```

### 2.2 V-2 어순 (Verb-Second)
주절에서 동사는 항상 두 번째 위치:
```
Í dag eti eg fisk.
오늘 먹다 나 생선
"오늘 나는 생선을 먹는다"
```
