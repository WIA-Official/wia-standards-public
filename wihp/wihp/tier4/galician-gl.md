## 2. 특수 음운 현상

### 2.1 Seseo와 Gheada
일부 지역 방언 특징:
```
Seseo: /θ/ → /s/ (스페인어식 구분 없음)
Gheada: /g/ → /h/ (갈리시아 특유)
  gato → [ˈhato] (일부 방언)
```

### 2.2 격변화 없음
스페인어와 달리 주어 대명사 자주 사용:
```
Eu falo galego.
나 말한다 갈리시아어
"나는 갈리시아어를 말한다"
```
