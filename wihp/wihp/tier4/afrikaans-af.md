## 2. 특수 음운 현상

### 2.1 g = [x]
아프리칸스에서 g는 연구개 마찰음입니다:
```
goed [xut] - 좋은
dag [dax] - 날
```

### 2.2 이중 부정
아프리칸스는 이중 부정을 사용합니다:
```
Ek praat nie Afrikaans nie
(나는 아프리칸스어를 말하지 않는다)
"nie...nie" 구조
```

### 2.3 간소화된 문법
네덜란드어에 비해 문법이 단순화되었습니다:
- 동사 활용 최소화
- 격변화 거의 없음
- 성(gender) 구분 없음
