## 2. 특수 음운 현상

### 2.1 성조 체계
줄루어는 고저 성조로 의미를 구별:
```
úkúbóna [고고고고] - 보는 것 (동명사)
ukubôna [저저고저] - 보다 (동사 원형)
```

### 2.2 명사 부류
15개의 명사 부류 체계:
```
umu-ntu → aba-ntu (사람 → 사람들)
in-ja → izin-ja (개 → 개들)
```
