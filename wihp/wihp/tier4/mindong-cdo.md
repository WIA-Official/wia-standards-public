## 개요

민동어(閩東語)는 중국 푸젠성 동부 지역에서 사용되는 중국어 방언으로, 푸저우어(福州話)가 대표적입니다. 7개의 성조를 가진 성조 언어이며, 복잡한 성조 연쇄(tone sandhi) 규칙이 특징입니다. 입성을 보존하고 있으며, Bàng-uâ-cê (平話字) 로마자 표기법이 사용됩니다.

### 성조 연쇄 (Tone Sandhi)

민동어의 매우 복잡한 특징으로, 거의 모든 성조가 변합니다:
- 두 음절 단어에서 첫 음절 성조가 변화
- 예: 福州 Hók-ciŭ → [hoʔ²³ tsiu⁴⁴]

## 특이 음운 현상

### 1. 복잡한 성조 연쇄 (Tone Sandhi)

민동어의 가장 특징적인 현상으로, 거의 모든 성조가 복잡하게 변합니다:

**2음절 단어의 성조 변화** (7가지 패턴):
- 1성 (55) → 44: 詩人 sĭ nè̤ng /si⁴⁴ nøyŋ²⁴²/
- 2성 (53) → 21: 死人 sī nè̤ng /si²¹ nøyŋ²⁤²/
- 3성 (33) → 55: 四人 sé nè̤ng /se⁵⁵ nøyŋ²⁴²/
- 4성 (213) → 53: 時候 sì hâiu /si⁵³ hau²¹³/
- 5성 (242) → 44: 市區 sê kṳ̆ /se⁴⁴ ky⁵⁵/
- 6성 (5) → 23: 識字 sĭk cê /siʔ²³ tse²¹³/
- 7성 (23) → 55: 食飯 sĭk buông /siʔ⁵⁵ puoŋ²¹³/

### 2. 입성 보존

민동어는 입성을 보존하되, -p/-t/-k가 모두 -k 또는 -h(성문폐쇄음)로 변화:
- 十 sĭk /siʔ⁵/
- 北 báe̤k /paiʔ⁵/
- 六 lĕ̤k /løyʔ⁵/

### 3. 모음 축약 현상

민동어는 복잡한 모음 체계를 가지며, 많은 이중모음이 있습니다:
- /uo/ → /uɔ/: 杯 buŏi
- /ai/ → /ɔi/: 開 khuŏi
- /au/ → /ou/: 歐 ĕu

### 4. 특수 모음 /øy/

민동어 특유의 모음 /ø/와 이중모음:
- 人 nè̤ng /nøyŋ²⁴²/
- 先 siĕng /siøyŋ⁵⁵/

### 5. "儂" (nè̤ng) 용법

민동어 특유의 "사람" 표현:
- 儂 nè̤ng "사람"
- 哪底儂? "어디 사람?" (어디 출신?)
