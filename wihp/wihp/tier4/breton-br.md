## 2. 특수 음운 현상

### 2.1 초자음 변이 (Kemmadurioù)
9가지 변이 (켈트어 중 가장 복잡):
```
연음화 (Dre vlotaat): tad → dad [dad]
경음화 (Dre galetaat): tad → zad [zad]
혼합변이 (Dre gemmeskat): tad → c'had [xad]
유기음화 등...
```

### 2.2 SVO 어순
현대 브르타뉴어는 주어-동사-목적어:
```
Me a lenn ul levr.
나 읽다 한 책
"나는 책을 읽는다"
```

### 2.3 전치사 융합
전치사와 관사 융합:
```
da + an = d'an
e + ar = er
```
