**어족**: Constructed > Minimalist Philosophical Language
**문자**: Latin alphabet (14 letters) + sitelen pona (hieroglyphic)
**화자 수**: 약 10,000명 (수천 명 유창자)

## 2. 설계 철학 (Design Philosophy)

### 2.1 도교적 단순성 (Taoist Simplicity)
```
2001년 소냐 랭(Sonja Lang)이 창안:

"pona" = 좋은, 단순한
"toki" = 말, 언어
→ toki pona = 좋은 언어, 단순한 언어

핵심 철학:
- 단순함은 좋음
- 적은 것이 더 많음
- 현재에 집중
- 긍정적 사고
```

### 2.2 120개의 단어 (nimi ale)
```
모든 표현을 120개 단어로:

기본 단어: 120개 (pu 기준)
추가 단어: ~20개 (커뮤니티)

복잡한 개념은 조합으로:
  komputa → ilo sona (지식 도구)
  telefon → ilo toki (말하기 도구)
  자동차 → tomo tawa (움직이는 구조물)
```

### 2.3 맥락 의존성
```
토키포나는 고도로 맥락 의존적:

tomo = 집, 건물, 방, 실내 공간
soweli = 동물, 짐승, 포유류

의미는 맥락에서 결정:
  tomo telo = 물의 집 = 욕실
  tomo moku = 음식의 집 = 식당
```

## 8. 문법 (Grammar)

### 8.1 기본 문장 구조
```
주어 + li + 술어:
  mi li pona = 나는 좋다

특별 규칙:
  "mi"와 "sina"는 "li" 생략:
  mi pona (나는 좋다)
  sina pona (너는 좋다)
```

### 8.2 e로 목적어 표시
```
주어 + 동사 + e + 목적어:
  mi moku e kili
  (나는 과일을 먹는다)

  ona li pana e mani
  (그는 돈을 준다)
```

### 8.3 pi로 소유 표시
```
pi = "의" (복잡한 소유)

tomo mi = 나의 집
tomo pi jan pona = 친구의 집
  (pi 없으면: tomo jan pona = 좋은-사람 집)

규칙: pi는 2개 이상 단어 그룹화
```

### 8.4 la로 맥락 설정
```
맥락 + la + 문장:

  tenpo ni la mi moku
  (이 시간에, 나는 먹는다 = 지금 먹는다)

  sina kama la mi pona
  (네가 오면, 나는 좋다)
```

### 8.5 전치사 (Prepositions)
```
lon = 에, ~에 있다
tawa = ~로, ~를 향해
tan = ~에서, ~로부터, ~때문에
kepeken = ~로 (도구)
sama = ~처럼

예:
  mi lon tomo (나는 집에 있다)
  mi tawa tomo (나는 집으로 간다)
```

### 8.6 질문 형성
```
seme = 무엇? 어떤?

ni li seme? (이것은 무엇?)
sina moku e seme? (너는 무엇을 먹니?)

ala 질문 (예/아니오):
  sina pona ala pona? (너 괜찮아 안 괜찮아?)
  대답: pona (좋아) / ala (아니)
```

## 9. 철학적 개념 (Philosophical Concepts)

### 9.1 나스름 (nasin pona - 좋은 길)
| 토키포나 | 의미 |
|----------|------|
| pona | 좋음, 단순함, 고침 |
| ike | 나쁨, 복잡함, 깨짐 |
| suli | 중요함, 큼 |
| lili | 작음, 적음 |
| wawa | 강함, 에너지 |
| lape | 쉼, 잠 |
| pilin | 느낌, 생각, 감정 |

### 9.2 도교적 가치
```
토키포나는 도교 철학을 반영:

1. 무위 (wu wei) - 억지로 하지 않음
   "pona li kama" (좋음이 온다)

2. 소박함 (simplicity)
   120개 단어로 모든 것 표현

3. 현재에 집중
   복잡한 시제 없음
   tenpo ni (이 시간 = 지금)

4. 긍정성
   ike를 pona ala (좋음-아님)으로 표현 가능
```

### 9.3 언어와 사고
```
토키포나를 배우면:

- 단순하게 생각하기
- 본질에 집중
- 긍정적 표현 연습
- 맥락 이해 능력 향상

예:
"자동차" → tomo tawa (움직이는 구조물)
→ 자동차의 본질: 움직이는 공간
```

### 9.4 니미 (nimi - 단어들)
```
토키포나 120개 핵심 단어 (pu 기준):

명사/동사: 대부분이 양쪽으로 사용
  moku = 음식 / 먹다
  toki = 말, 언어 / 말하다
  pona = 좋음 / 고치다

품사는 위치로 결정:
  pona li pona (좋음은 좋다)
  mi pona e ni (나는 이것을 고친다)
```

### 10.2 sitelen pona (상형문자)
```
토키포나는 고유 문자 체계가 있음:

각 단어가 하나의 기호로:
  mi = 사람 형상
  tomo = 집 모양
  suno = 태양 원형

조합:
  tomo tawa = 집+이동 = 자동차 기호
```

### 10.3 유명한 문장과 속담
```
toki pona li toki pona!
(토키포나는 좋은 언어다!)

jan li sona ala e ijo ale
(사람은 모든 것을 알지 못한다)

tenpo ale la sina ken kama sona
(모든 시간에, 너는 배울 수 있다)

pona li lon insa
(좋음은 안에 있다 - 내면의 평화)

mi sona e ni: mi sona ala e ale
(나는 안다: 나는 모든 것을 알지 못한다)
```

### 10.4 토키포나 노래와 시
```
유명한 토키포나 노래:
- "toki pona la" by jan Misali
- "telo suli" (큰 물 = 바다)
- "soweli lili" (작은 동물)

토키포나 하이쿠:
  suno li kama
  lape li pini pona
  tenpo sin li lon

  (태양이 온다
   잠이 잘 끝났다
   새로운 시간이 있다)
```

## 12. 실전 복잡한 문장

### 12.1 긴 문장 구조
```
tenpo pini la mi en jan pona mi li tawa tomo moku li moku e moku pona
(과거에, 나와 내 친구는 식당에 가서 좋은 음식을 먹었다)

분석:
  tenpo pini la = 과거 시간에
  mi en jan pona mi = 나와 내 친구
  li tawa tomo moku = 식당에 갔다
  li moku e moku pona = 좋은 음식을 먹었다
```

### 12.2 철학적 표현
```
nasin pona pi toki pona li ni:
o sona lili. o pilin mute. o pona.
(토키포나의 좋은 길은 이것이다:
적게 알고, 많이 느끼고, 좋게 되어라)
```
