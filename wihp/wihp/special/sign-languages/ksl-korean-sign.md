# Korean Sign Language (한국수어/KSL) - WIHP Mapping

## Language Information
- **Language**: Korean Sign Language (한국수어)
- **ISO 639-3**: kvk
- **Region**: South Korea
- **Users**: ~400,000 (Deaf community + interpreters)
- **Recognition**: Official language (한국수어법, 2016)
- **WIHP Category**: Special/Sign Language

## Notation System

KSL은 음성언어가 아니므로 IPA 대신 **WIHP 수어 코드** 체계를 사용합니다.

### WIHP Sign Code Format
```
HS##-LO##-MV##-OR##-NM##
```

## Cultural Notes

1. **농문화**: 한국 농인 공동체의 고유한 문화와 정체성
2. **수어법**: 2016년 제정, 한국수어를 공용어로 인정
3. **지문자**: 음성언어 고유명사 등을 표현할 때 사용
4. **NMM**: 비수지신호(표정, 입모양)가 문법적 기능 수행
