# American Sign Language (ASL) - WIHP Mapping

## Language Information
- **Language**: American Sign Language (ASL)
- **ISO 639-3**: ase
- **Region**: United States, Canada (English-speaking)
- **Users**: ~500,000 - 2 million (Deaf + hearing signers)
- **Family**: French Sign Language family
- **WIHP Category**: Special/Sign Language

## Notation System

ASL uses **WIHP Sign Code** system instead of IPA.

### WIHP Sign Code Format
```
HS##-LO##-MV##-OR##-NM##
```

## Cultural Notes

1. **Deaf Culture**: ASL is central to American Deaf culture and identity
2. **Facial Grammar**: Non-manual markers (eyebrows, mouth) are grammatical
3. **Visual Language**: Relies on visual-spatial grammar, not English
4. **Fingerspelling**: Used for English words, names, technical terms
5. **Regional Variation**: Some signs vary by region (like accents)
