# Japanese Sign Language (日本手話/JSL) - WIHP Mapping

## Language Information
- **Language**: Japanese Sign Language (日本手話)
- **ISO 639-3**: jsl
- **Region**: Japan
- **Users**: ~320,000 (Deaf community)
- **Family**: Japanese Sign Language family
- **WIHP Category**: Special/Sign Language

## Notation System

JSL uses **WIHP Sign Code** system.

### WIHP Sign Code Format
```
HS##-LO##-MV##-OR##-NM##
```

## Japanese Manual Alphabet (指文字)

## JSL Unique Features

### Mouth Patterns (口形)
JSL uses specific mouth patterns with signs:
| 口形 | 의미 | 사용 |
|------|------|------|
| パ (pa) | 갑자기, 확실히 | 강조 시 |
| フ (fu) | 부드럽게 | 약한 동작 |
| プー (pu-) | 부풀다 | 증가 표현 |

### Non-Manual Markers
| NMM | 의미 | 문법 기능 |
|-----|------|----------|
| 눈썹 올림 | 의문 (Y/N) | 예/아니오 질문 |
| 눈썹 내림 | 의문 (WH) | 무엇/누구/어디 질문 |
| 고개 끄덕 | 긍정 | 확인, 동의 |
| 고개 젓기 | 부정 | 거부, 반대 |

## Cultural Notes

1. **ろう文化 (Deaf Culture)**: Strong Deaf community identity in Japan
2. **手話言語条例**: Sign language recognition ordinances in many prefectures
3. **指文字**: Japanese fingerspelling follows kana system
4. **Simultaneous Communication**: Some use JSL + spoken Japanese together
5. **Regional Variation**: Different signs exist between regions
