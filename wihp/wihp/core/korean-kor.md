## 1. 개요 및 철학적 의의

### 1.1 왜 한국어가 WIHP의 기준점인가?

```
WIHP의 핵심 원리:
    모든 언어 → IPA → 한글

한국어의 특수성:
    한글 → IPA → 한글 = 원본 (왕복 검증)

논리적 완결성:
    다른 모든 언어의 WIHP 매핑이 올바르다면,
    한국어의 WIHP 매핑은 원래의 한글로 돌아와야 한다.
```

### 1.2 세종대왕의 설계 원리

```
훈민정음 (1443년):
├── 자음: 발음 기관의 모양을 본뜸
│   ├── ㄱ = 혀뿌리가 목구멍을 막는 모양
│   ├── ㄴ = 혀가 윗잇몸에 닿는 모양
│   ├── ㅁ = 입술 모양
│   ├── ㅅ = 이의 모양
│   └── ㅇ = 목구멍 모양
├── 모음: 천지인 삼재 철학
│   ├── ㆍ (아래아) = 하늘 (天)
│   ├── ㅡ = 땅 (地)
│   └── ㅣ = 사람 (人)
└── 획 추가 = 소리 강화
    ├── ㄱ → ㅋ (기식음 추가)
    └── ㄱ → ㄲ (긴장음 추가)
```

## 5. 음운 규칙 (Phonological Rules)

### 5.1 비음화 (鼻音化, Nasalization)

파열음 받침 + 비음 → 비음 + 비음

| 규칙 | 예시 | 표기 | 발음 | IPA |

| ㄱ + ㄴ/ㅁ → ㅇ + ㄴ/ㅁ | 국물 | guk-mul | 궁물 | /kuŋmul/ |
| ㄷ + ㄴ/ㅁ → ㄴ + ㄴ/ㅁ | 믿는 | mid-neun | 민는 | /minːɯn/ |
| ㅂ + ㄴ/ㅁ → ㅁ + ㄴ/ㅁ | 밥물 | bab-mul | 밤물 | /pammul/ |
| ㄱ + ㄹ → ㅇ + ㄴ | 독립 | dok-lib | 동닙 | /toŋnip/ |

### 5.2 유음화 (流音化, Lateralization)

### 10.1 주요 방언별 특징

| 방언 | 특징 | 예시 |
|------|------|------|
| 서울말 (표준어) | 기준 | 뭐해? /mwʌ.hɛ/ |
| 경상도 | 성조 보존, /ㅓ/≈/ㅡ/ | 뭐하노? (고저 악센트) |
| 전라도 | /ㅓ/≈/ㅗ/ 중화 | 뭐해라우? |
| 충청도 | 느린 억양 | 뭐해유~ |
| 제주도 | 아래아 보존 | ᄒᆞ영 |

## 11. WIHP 검증 요약

### 11.1 왕복 검증 (Round-Trip)

```
한글 입력 → IPA 변환 → WIHP 코드 → 한글 출력

예시:
"안녕하세요" → /an.njʌŋ.ha.se.jo/
    → PL04-MN02...(생략)... → "안녕하세요"

결과: 원본과 동일 ✅
```

### 11.2 검증 기준

| 항목 | 기준 | 결과 |
|------|------|------|
| 자음 19개 | 모두 매핑 | ✅ |
| 모음 10+개 | 모두 매핑 | ✅ |
| 받침 7종성 | 모두 처리 | ✅ |
| 음운 규칙 7개 | 모두 반영 | ✅ |
| 왕복 검증 | 원본 복원 | ✅ |

### 11.3 한계 및 주의사항

1. **동음이의어**: IPA가 같으면 구분 불가 (배[果實] vs 배[船] vs 배[腹])
2. **띄어쓰기**: 음운 경계에 영향 (웃어 vs 우서)
3. **외래어**: 원어 발음 vs 한국어화 발음 차이
4. **방언**: 표준어 기준, 방언은 별도 처리 필요
