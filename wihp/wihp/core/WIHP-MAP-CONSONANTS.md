## 개요

이 문서는 전 세계 언어의 자음 음소를 한글로 매핑하는 공식 표준입니다.

```
모든 자음 = 조음 위치 (PL) + 조음 방법 (MN) + 기류 특성 (AR) → 한글
```

## 참고 사항

### 한글 표기 원칙

1. **현대 한글 우선**: 가능하면 현대 한글 24자로 표기
2. **옛한글 사용**: 현대 한글로 불가능할 때만 사용
   - ㆆ (여린히읗): 성문 파열음
   - ㅸ (순경음): 순치 마찰음
   - ㅿ (반시옷): 유성 치찰음
   - ㆁ (옛이응): 어두 연구개 비음
3. **확장 표기**: 점(̣), 첨자(ʸ, ˤ) 등으로 구분

### 유성음 처리

- 한국어는 유성/무성 구분이 없음
- 어두: 평음 사용 (/b/ → ㅂ)
- 어중: 문맥에 따라 자연스럽게 발음
