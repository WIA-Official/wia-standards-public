## 개요

이 문서는 전 세계 언어의 모음 음소를 한글로 매핑하는 공식 표준입니다.

```
모음 분류 = 혀 높이 (Height) + 혀 위치 (Backness) + 입술 모양 (Roundedness)
```

## 3. 비모음 (Nasal Vowels)

비모음은 모음 뒤에 ㄴ/ㅇ을 추가하거나 틸데(~)로 표시합니다.

| IPA | 한글 | 설명 | 예시 언어 |
|-----|------|------|----------|
| /ã/ | ㅏ̃/앙 | 비음화 a | 프랑스어 |
| /ɛ̃/ | ㅔ̃/엥 | 비음화 ɛ | 프랑스어 |
| /ɔ̃/ | ㅗ̃/옹 | 비음화 ɔ | 프랑스어 |
| /œ̃/ | ㅚ̃/욍 | 비음화 œ | 프랑스어 |

### 프랑스어 비모음 예시

| 단어 | IPA | WIHP 한글 |
|------|-----|----------|
| an | /ɑ̃/ | 앙 |
| in | /ɛ̃/ | 앵 |
| on | /ɔ̃/ | 옹 |
| un | /œ̃/ | 엉 |

## 4. 장모음 (Long Vowels)

장모음은 기본 모음과 동일하게 표기하되, 필요시 모음을 두 번 쓰거나 장음 기호(:)를 사용합니다.

| IPA | 한글 | 설명 | 예시 |
|-----|------|------|------|
| /iː/ | ㅣ(ㅣ) | 장 이 | see /siː/ → 시 |
| /uː/ | ㅜ(ㅜ) | 장 우 | too /tuː/ → 투 |
| /ɑː/ | ㅏ(ㅏ) | 장 아 | car /kɑː/ → 카 |
| /ɔː/ | ㅗ(ㅗ) | 장 오 | law /lɔː/ → 로 |
| /ɜː/ | ㅓ(ㅓ) | 장 어 | bird /bɜːd/ → 버드 |

## 5. 성조 모음 (Tone)

성조 언어의 경우 모음에 성조 기호를 추가합니다.

| 성조 | 기호 | 한글 표기 | 예시 언어 |
|------|------|----------|----------|
| 고조 (High) | ˊ / 1 | ㅏˊ | 중국어, 베트남어 |
| 중조 (Mid) | ˉ / 2 | ㅏˉ | 중국어 |
| 저조 (Low) | ˋ / 3 | ㅏˋ | 베트남어 |
| 상승조 (Rising) | ˇ / 4 | ㅏˇ | 중국어 |
| 하강조 (Falling) | ˆ / 5 | ㅏˆ | 베트남어 |

### 중국어 성조 예시

| 한자 | 병음 | 성조 | WIHP |
|------|------|------|------|
| 妈 | mā | 1성 (고평) | 마ˉ |
| 麻 | má | 2성 (상승) | 마ˊ |
| 马 | mǎ | 3성 (하강상승) | 마ˇ |
| 骂 | mà | 4성 (하강) | 마ˋ |
| 吗 | ma | 경성 | 마 |
