# WIHP: Oromo (Afaan Oromoo) - orm

## Language Overview
- **Native Name:** Afaan Oromoo
- **ISO 639-3:** orm
- **Speaker Population:** ~40 million
- **Language Family:** Afroasiatic → Cushitic → East Cushitic
- **Writing System:** Latin (Qubee) since 1991
- **Phonological Type:** Agglutinative, tonal

## Phonological Features
- **Gemination:** Consonant length is phonemic
- **Pharyngeal Consonants:** Common Cushitic feature
- **Long Vowels:** Phonemic length distinction
- **Tone/Pitch:** Affects meaning

## Phonological Rules

### Gemination
Consonant length is phonemic:
- buna /buˈna/ "coffee" vs. bunna /bunˈna/ different meaning

### Ejectives
Oromo has ejective consonants typical of Ethiopian languages.

### Vowel Length
Vowel length distinguishes meaning:
- hara /haˈra/ vs. haaraa /haːˈraː/ "new"

### Qubee Alphabet
Modern Oromo uses the Qubee (Latin-based) alphabet since 1991.

---

## WIHP Implementation Notes

1. **Ejectives:** Mark with AR03 airstream code
2. **Pharyngeal:** ħ maps to ㅎ
3. **Gemination:** Write single consonant in Hangul
4. **Long Vowels:** Not distinguished in Hangul
5. **Glottal Stop:** Use ㅇ initial position
