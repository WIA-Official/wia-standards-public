# Samoan (Gagana Sāmoa) - WIHP A-Grade Mapping

## Language Information
- **Language**: Samoan (Gagana Sāmoa)
- **ISO 639-3**: smo
- **Region**: Samoa, American Samoa, New Zealand
- **Speakers**: ~510,000
- **Script**: Latin
- **WIHP Tier**: 3

## Phonological Features
- Polynesian language family
- Simple phoneme inventory (14 consonants, 5 vowels)
- Vowel length distinction
- Glottal stop is phonemic (ʻ)
- Syllable structure: (C)V(V)
- No consonant clusters
- Stress usually on penultimate mora
- Two registers: formal (tautala lelei) and informal (tautala leaga)

## Standard 50 Sentences

### Basic Communication (1-10)
1. **Talofa** /talofa/ → 탈로파 "Hello"
2. **O ā mai oe?** /o aː mai oe/ → 오 아아 마이 오에 "How are you?"
3. **Manuia** /manuia/ → 마누이아 "I am fine/good"
4. **Faʻafetai** /faʔafetai/ → 파아페타이 "Thank you"
5. **Faʻamolemole** /faʔamolemole/ → 파아몰레몰레 "Please"
6. **Faʻamalie atu** /faʔamalie atu/ → 파아말리에 아투 "Sorry"
7. **Ioe** /ioe/ → 이오에 "Yes"
8. **Leai** /leai/ → 레아이 "No"
9. **Tofa soifua** /tofa soifua/ → 토파 소이푸아 "Goodbye"
10. **Taeao manuia** /taeao manuia/ → 타에아오 마누이아 "Good morning"

### Daily Life (11-20)
11. **O loʻu igoa o Sione** /o loʔu iŋoa o sione/ → 오 로우 이응오아 오 시오네 "My name is Sione"
12. **O aʻu o le Sāmoa** /o aʔu o le saːmoa/ → 오 아우 오 레 사아모아 "I am Samoan"
13. **O le ā lenei?** /o le aː lenei/ → 오 레 아아 레네이 "What is this?"
14. **Faʻamolemole le vai** /faʔamolemole le vai/ → 파아몰레몰레 레 바이 "Water please"
15. **Ua ou fia ʻai** /ua ou fia ʔai/ → 우아 오우 피아 아이 "I am hungry"
16. **E manaia le aso** /e manaia le aso/ → 에 마나이아 레 아소 "The day is beautiful"
17. **Ou te alu e moe** /ou te alu e moe/ → 오우 테 알루 에 모에 "I am going to sleep"
18. **O le ā le taimi?** /o le aː le taimi/ → 오 레 아아 레 타이미 "What time is it?"
19. **Fesoasoani mai** /fesoasoani mai/ → 페소아소아니 마이 "Help me"
20. **Ua ou malamalama** /ua ou malamalama/ → 우아 오우 말라말라마 "I understand"

### Questions and Directions (21-30)
21. **O fea le maketi?** /o fea le maketi/ → 오 페아 레 마케티 "Where is the market?"
22. **E fia le tau?** /e fia le tau/ → 에 피아 레 타우 "How much is it?"
23. **O fea mai oe?** /o fea mai oe/ → 오 페아 마이 오에 "Where are you from?"
24. **E mamao Apia?** /e mamao apia/ → 에 마마오 아피아 "Is Apia far?"
25. **Alu i le taumatau** /alu i le taumatau/ → 알루 이 레 타우마타우 "Go right"
26. **Alu i le tauagavale** /alu i le tauaŋavale/ → 알루 이 레 타우앙아발레 "Go left"
27. **O fea le falemai?** /o fea le falemai/ → 오 페아 레 팔레마이 "Where is the hospital?"
28. **E moni lenei?** /e moni lenei/ → 에 모니 레네이 "Is this true?"
29. **E te iloa le gagana Sāmoa?** /e te iloa le ŋaŋana saːmoa/ → 에 테 일로아 레 앙아나 사아모아 "Do you know Samoan?"
30. **Ua ou lē iloa le auala** /ua ou leː iloa le auala/ → 우아 오우 레에 일로아 레 아우알라 "I am lost"

### Samoan Culture (31-40)
31. **Faʻa Sāmoa** /faʔa saːmoa/ → 파아 사아모아 "The Samoan way"
32. **Siva Sāmoa** /siva saːmoa/ → 시바 사아모아 "Samoan dance"
33. **Fale Sāmoa** /fale saːmoa/ → 팔레 사아모아 "Samoan house"
34. **Umu Sāmoa** /umu saːmoa/ → 우무 사아모아 "Samoan earth oven"
35. **ʻAva e inu** /ʔava e inu/ → 아바 에 이누 "Drink kava"
36. **Ola le niu** /ola le niu/ → 올라 레 니우 "The coconut palm"
37. **Pe'a ma malu** /peʔa ma malu/ → 페아 마 말루 "Traditional tattoos"
38. **Lāvalava e ofu** /laːvalava e ofu/ → 라아발라바 에 오푸 "Wear the lavalava"
39. **Matai o le nu'u** /matai o le nuʔu/ → 마타이 오 레 누우 "Chief of the village"
40. **ʻIeʻie toga** /ʔieʔie toŋa/ → 이에이에 통아 "Fine mat"

### Advanced Expressions (41-50)
41. **O se tusi lelei lenei** /o se tusi lelei lenei/ → 오 세 투시 레레이 레네이 "This is a good book"
42. **E te manaʻo i se ti?** /e te manaʔo i se ti/ → 에 테 마나오 이 세 티 "Would you like tea?"
43. **E matagofie Sāmoa** /e mataŋofie saːmoa/ → 에 마탕오피에 사아모아 "Samoa is beautiful"
44. **E maualuga le aganuʻu Sāmoa** /e maualuŋa le aŋanuʔu saːmoa/ → 에 마우알룽아 레 앙아누우 사아모아 "Samoan culture is rich"
45. **O le Vasa Pasefika** /o le vasa pasefika/ → 오 레 바사 파세피카 "The Pacific Ocean"
46. **O le lagi matagofie** /o le laŋi mataŋofie/ → 오 레 랑이 마탕오피에 "The sky is beautiful"
47. **Teuteu o le malaga** /teuteu o le malaŋa/ → 테우테우 오 레 말랑아 "Preparation for the journey"
48. **O le muʻa o anamua** /o le muʔa o anamua/ → 오 레 무아 오 아나무아 "Ancient times"
49. **Ou te alofa iā Sāmoa** /ou te alofa iaː saːmoa/ → 오우 테 알로파 이아아 사아모아 "I love Samoa"
50. **Ola Sāmoa** /ola saːmoa/ → 올라 사아모아 "Long live Samoa"

## Phonological Rules
1. **Simple syllable structure**: (C)V, no consonant clusters
2. **Vowel length**: Contrastive (marked with macron)
3. **Glottal stop**: Phonemic, written as ʻ
4. **Register variation**: Formal (tautala lelei) vs informal (tautala leaga)
5. **Velar nasal**: Written as g, pronounced /ŋ/
6. **Stress**: Penultimate mora
7. **No final consonants**: All syllables end in vowels
8. **Reduplication**: Common morphological process
