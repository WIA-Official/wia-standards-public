# WIHP: Zulu (isiZulu) - zul

## Language Overview
- **Native Name:** isiZulu
- **ISO 639-3:** zul
- **Speaker Population:** ~12 million
- **Language Family:** Niger-Congo → Bantu → Nguni
- **Writing System:** Latin alphabet
- **Phonological Type:** Agglutinative with click consonants

## Phonological Features
- **Click Consonants:** Three click types (c, q, x)
- **Tone System:** High and low tones
- **Noun Classes:** 15+ grammatical classes
- **Prenasalization:** Common consonant feature

## Phonological Rules

### Click Consonants
Zulu has three click types with various modifications:
- Dental click (c): tongue tip against upper teeth
- Postalveolar click (q): tongue blade against alveolar ridge
- Lateral click (x): tongue sides release air laterally

### Tone System
Two-tone system (High and Low) with tonal morphology.

### Prenasalization
Many consonants occur with preceding nasal:
- mb, nd, ng, nj, etc.

### Implosives
Voiced implosives /ɓ/ written as 'b' are common.

---

## WIHP Implementation Notes

1. **Click Consonants:** Use dedicated WIHP place codes (PL10-12)
2. **Ejectives:** Mark with AR03 airstream code
3. **Implosives:** Distinguish from plain voiced stops
4. **Tones:** Not marked in standard WIHP Hangul
5. **Prenasalization:** Write nasal + consonant sequence
