# WIHP: Shona (chiShona) - sna

## Language Overview
- **Native Name:** chiShona
- **ISO 639-3:** sna
- **Speaker Population:** ~15 million
- **Language Family:** Niger-Congo → Bantu → Shona
- **Writing System:** Latin alphabet
- **Phonological Type:** Agglutinative, tonal

## Phonological Features
- **Tone System:** Two-level (High and Low)
- **Whistled Sibilants:** Unique feature
- **Noun Classes:** 21 grammatical classes
- **Vowel System:** 5-vowel system

## Phonological Rules

### Whistled Sibilants
Unique to Shona and related languages:
- sv /s̪͡ʍ/ and zv /z̪͡ʍ/ produced with lip rounding and whistling

### Tone
Two-level tone system (High and Low) with grammatical functions.

### Labiodental Affricates
Shona has unusual pf /p͡f/ and bv /b͡v/ affricates.

### Implosives
Implosive stops /ɓ/ and /ɗ/ contrast with plain voiced stops.

---

## WIHP Implementation Notes

1. **Whistled Sibilants:** Special AR codes (07, 08) for sv/zv
2. **Implosives:** Mark with AR06 airstream code
3. **Labiodental Affricates:** pf, bv are distinct phonemes
4. **Tone:** Not marked in standard WIHP Hangul
5. **Prenasalization:** Write nasal + consonant sequence
