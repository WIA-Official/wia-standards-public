# WIHP: Georgian (ქართული) - kat

## Language Overview
- **Native Name:** ქართული ენა (Kartuli ena)
- **ISO 639-3:** kat
- **Speaker Population:** ~4 million
- **Language Family:** Kartvelian (South Caucasian)
- **Writing System:** Georgian script (Mkhedruli)
- **Phonological Type:** Agglutinative, ejective consonants

## Phonological Features
- **Ejective Consonants:** p', t', k', ts', tʃ'
- **Harmonic Clusters:** Complex consonant sequences
- **No Grammatical Gender:** Unique among regional languages
- **Unique Script:** One of 14 unique writing systems

## Phonological Rules

### Ejective Consonants
Five ejective stops/affricates: პ /p'/, ტ /t'/, კ /k'/, წ /ts'/, ჭ /tʃ'/, ყ /q'/

### Consonant Clusters
Georgian allows complex initial and final clusters:
- მწვრთნელი /mts'vrtʰneli/ "trainer"

### Harmonic Clusters
Consonants agree in voicing within clusters.

### No Gender
Georgian has no grammatical gender.

## WIHP Implementation Notes

1. **Ejectives:** Use AR03 code (ㅃ, ㄸ, ㄲ, etc.)
2. **Aspirated:** Use AR01 code (ㅍ, ㅌ, ㅋ)
3. **Uvular Ejective:** ყ→ㅋ
4. **Velar Fricative:** ხ→ㅎ, ღ→ㄱ
5. **Complex Clusters:** Write sequentially
