# WIHP: Hawaiian ( ĽŇĆlelo Hawai Ľi) - haw

## Language Overview
- **Native Name:**  ĽŇĆlelo Hawai Ľi
- **ISO 639-3:** haw
- **Speaker Population:** ~24,000 fluent
- **Language Family:** Austronesian ‚Üí Polynesian ‚Üí Eastern Polynesian
- **Writing System:** Latin with  Ľokina ( Ľ) and kahakŇć (macron)
- **Phonological Type:** (C)V syllable structure, very simple phonology

## Phonological Features
- **8 Consonants:** One of the smallest consonant inventories
- **Glottal Stop:** Phonemic ( Ľokina)
- **Vowel Length:** Phonemic (kahakŇć)
- **5 Vowels:** With long counterparts
