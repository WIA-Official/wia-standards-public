# WIHP: Tigrinya (ትግርኛ) - tir

## Language Overview
- **Native Name:** ትግርኛ (Tigrinya)
- **ISO 639-3:** tir
- **Speaker Population:** ~9 million
- **Language Family:** Afroasiatic → Semitic → Ethiopic
- **Writing System:** Ge'ez script (Fidäl)
- **Phonological Type:** Consonantal root system, ejectives

## Phonological Features
- **Ejectives:** Common feature (pʼ, tʼ, kʼ, sʼ, tʃʼ)
- **Gemination:** Phonemic consonant length
- **Pharyngeals:** ħ and ʕ present
- **7-Vowel System:** With orders in Ge'ez

## Phonological Rules

### Ejectives
Tigrinya has ejective stops and affricates (pʼ, tʼ, kʼ, tsʼ, tʃʼ) - a key feature.

### Gemination
Consonant length is phonemic and written with doubled letters in romanization.

### Ge'ez Script Orders
Each consonant has 7 orders (vowel modifications):
- ህ ha, ሁ hu, ሂ hi, ሃ ha, ሄ he, ህ hə, ሆ ho

### Pharyngeals
ħ and ʕ are distinctive pharyngeal consonants.
