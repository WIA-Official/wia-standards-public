# WIHP: Catalan (Català) - cat

## Language Overview
- **Native Name:** Català
- **ISO 639-3:** cat
- **Speaker Population:** ~10 million
- **Language Family:** Indo-European → Romance → Occitano-Romance
- **Writing System:** Latin alphabet
- **Phonological Type:** Romance with distinctive features

## Phonological Features
- **Vowel Reduction:** Unstressed vowels reduce
- **Final Consonants:** Devoicing common
- **Gemination:** In some dialects
- **Two L Sounds:** l and l·l (geminate)

## Phonological Rules

### Vowel Reduction
Unstressed vowels reduce in Central Catalan:
- a, e → [ə]
- o → [u]

### Final Devoicing
Voiced consonants devoice word-finally.

### Geminate L
l·l represents geminate /lː/.

---

## WIHP Implementation Notes

1. **Schwa:** Unstressed a/e → ㅓ
2. **Palatal Lateral:** ll → ㄹ+이
3. **Voiced Postalveolar:** j/g → ㅈ
4. **Final Devoicing:** Mark voiceless
5. **Stress:** Generally penultimate
