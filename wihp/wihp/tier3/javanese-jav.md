# WIHP: Javanese (Basa Jawa) - jav

## Language Overview
- **Native Name:** Basa Jawa / ꦧꦱꦗꦮ
- **ISO 639-3:** jav
- **Speaker Population:** ~82 million
- **Language Family:** Austronesian → Malayo-Polynesian → Javanese
- **Writing System:** Latin, Javanese script (Aksara Jawa)
- **Phonological Type:** Speech levels (ngoko, madya, krama)

## Phonological Features
- **Speech Levels:** Three politeness registers
- **Vowel System:** 8 vowels with schwa
- **Retroflex Consonants:** ḍ and ṭ
- **No Tones:** Unlike many Southeast Asian languages

## Standard Sentences (50 Sentences)

### Basic Communication (1-10) - Ngoko (Informal)
1. **Piye kabare?** /pijɛ kaˈbarɛ/
   - 피예 카바레
   - "How are you?" (informal)

2. **Apik-apik wae.** /apiʔ apiʔ wae/
   - 아픽 아픽 와에
   - "I'm fine."

3. **Matur nuwun.** /matur nuˈwun/
   - 마투르 누운
   - "Thank you."

4. **Iya.** /ija/
   - 이야
   - "Yes."

5. **Ora.** /ɔra/
   - 오라
   - "No."

6. **Ngapura.** /ŋapura/
   - 응아푸라
   - "Sorry."

7. **Tulung.** /tuluŋ/
   - 툴룽
   - "Please."

8. **Aku ora ngerti.** /aku ɔra ŋərti/
   - 아쿠 오라 응으르티
   - "I don't understand."

9. **Kowe iso ngomong Inggris?** /kɔwɛ isɔ ŋɔmɔŋ iŋgris/
   - 코웨 이소 응오몽 잉그리스
   - "Do you speak English?"

10. **Sugeng enjing.** /sugəŋ ənjiŋ/
    - 수겅 은징
    - "Good morning."

## Phonological Rules

### Retroflex Consonants
Javanese has retroflex ṭ /ʈ/ and ḍ /ɖ/ distinct from dental t and d.

### Schwa (Pepet)
The schwa /ə/ is very common and written as 'e' (vs. é /e/ and è /ɛ/).

### Glottal Stop
Common word-finally and sometimes medially.

### Speech Levels
Vocabulary changes based on politeness level, not phonology.

---

## WIHP Implementation Notes

1. **Retroflex Stops:** ṭ→ㅌ, ḍ→ㄷ (same as dental in Hangul)
2. **Schwa:** Map to ㅓ
3. **Palatal Stops:** c→ㅊ, j→ㅈ
4. **Glottal Stop:** Mark with ㅇ final
5. **Speech Levels:** Same phonology, different vocabulary
