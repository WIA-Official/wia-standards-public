## 각 언어별 포함 내용

각 A등급 매핑 파일은 다음 정보를 포함합니다:

1. **기본 정보**
   - ISO 코드, 어족, 화자 수, 주요 지역

2. **자음 매핑**
   - IPA → WIHP 코드 → 한글
   - 조음 위치/방법/기류 분류

3. **모음 매핑**
   - 단모음, 이중모음, 비모음
   - 장단음 구분

4. **50 표준 문장**
   - 기본 인사 (1-10)
   - 자기소개 (11-20)
   - 일상 표현 (21-30)
   - 질문 (31-40)
   - 긴급/필수 (41-50)

5. **어휘 세트**
   - 숫자 0-20, 100, 1000
   - 요일 7개
   - 색깔 10개
   - 가족 10개

6. **규칙 및 특징**
   - 철자-발음 대응
   - 음운 변화 규칙
   - 방언 차이
   - 문화 특화 표현

## 언어별 특징 요약

### 신규 언어 (16-20)

#### 태국어 (Thai)
- **5성조 체계**: 중평, 저조, 하강, 고조, 상승
- **3중 대립**: 무기음/유기음/유성음 (p/pʰ/b)
- **21개 초성, 6개 종성**: 종성 중화
- **장단 모음 대립**: 9개 × 2 = 18개

#### 폴란드어 (Polish)
- **3중 치찰음**: 치경 /s z/ vs 후설 /ʂ ʐ/ vs 구개 /ɕ ʑ/
- **비모음 ą, ę**: 위치에 따른 실현 변이
- **복잡한 자음군**: szcz, chrz, prz, strz
- **끝에서 두 번째 강세**

#### 네덜란드어 (Dutch)
- **13개 모음 + 이중모음**: 풍부한 모음 체계
- **연구개 마찰음**: /x/ (ch) vs /ɣ/ (g)
- **어말 무성화**: dag /dɑx/
- **Gezellig 문화**: 번역 불가능한 개념

#### 인도네시아어 (Indonesian)
- **단순한 음운 체계**: 6모음, 23자음
- **규칙적 철자**: 거의 완벽한 철자-발음 일치
- **성조 없음, 굴절 거의 없음**
- **어말 k → 성문폐쇄음 /ʔ/**

#### 그리스어 (Greek)
- **5개 단순 모음**: /a e i o u/
- **구개음화**: /k ɡ x ɣ/ → /c ɟ ç ʝ/ (e, i 앞)
- **θ, δ 음**: 치간 마찰음
- **3500년 문헌 역사**

### 기존 언어 (1-15)

#### 영어 (English)
- 대모음추이로 인한 철자-발음 불일치
- 강세 박자 언어 (stress-timed)
- 다양한 방언 (GA, RP, Australian 등)

#### 중국어 (Mandarin)
- 4성 + 경성 성조 체계
- 권설음 (zh, ch, sh, r)
- 성조 변화 규칙

#### 힌디어 (Hindi)
- 권설음 vs 치조음 대립
- 4중 파열음 체계
- 슈와 삭제

#### 스페인어 (Spanish)
- 5개 순수 모음
- 철자-발음 일관성
- 전동음 rr vs 탄음 r

#### 프랑스어 (French)
- 비모음 4개
- 전설 원순 모음
- 연음 (liaison)

#### 아랍어 (Arabic)
- 인두음, 구개수음
- 인두화 자음 (emphatics)
- 태양/달 문자

#### 벵골어 (Bengali) ★A등급
- 29개 자음, 4중 대립
- 내재 모음 /ɔ/
- 권설음 체계

#### 포르투갈어 (Portuguese) ★A등급
- 비모음과 비이중모음
- BR/PT 방언 차이
- 구개음화 (t/d + i)

#### 러시아어 (Russian)
- 경자음/연자음 대립
- 강세에 따른 모음 약화
- 어말 무성화

#### 일본어 (Japanese)
- CV 음절 구조
- 모라 기반 리듬
- 피치 악센트

#### 독일어 (German)
- ich-Laut /ç/ vs ach-Laut /x/
- 전설 원순 모음 (ü, ö)
- 어말 무성화

#### 한국어 (Korean)
- **WIHP 기준 언어**
- 자질 문자 (featural alphabet)
- 평음/기식음/경음 3중 대립

#### 베트남어 (Vietnamese)
- 6성조 체계
- 북부/남부 방언 차이
- 단음절 구조

#### 터키어 (Turkish)
- 8모음 대칭 체계
- 모음 조화 (vowel harmony)
- 교착어적 형태론

#### 이탈리아어 (Italian) ★A등급
- 7개 모음 (개/폐 e, o 구분)
- 쌍자음 (geminate consonants)
- 철자-발음 일관성
