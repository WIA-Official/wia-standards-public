# WIHP 매핑: Japanese (日本語/일본어)

**ISO 코드**: jpn
**언어명**: Japanese / 日本語 / 일본어
**어족**: Japonic
**화자 수**: 125M+ (L1: 125M)
**작성일**: 2025-12-21
**버전**: 2.0 (A-grade)

---

## 개요

일본어는 CV(자음+모음) 음절 구조가 특징적이며, 히라가나/가타카나 문자 체계와 1:1 대응됩니다. 모라(拍) 기반 언어로, 피치 악센트를 사용합니다.

## 철자-발음 규칙

### 기본 규칙

1. **CV 구조**: 일본어는 자음+모음 구조가 기본 (ん 제외)
2. **모라 단위**: 각 가나는 동일한 길이 (1모라)
3. **장음**: ー 또는 모음 연속으로 표기 (2모라)
4. **촉음**: っ로 표기, 다음 자음을 겹침 (1모라)

### 히라가나/가타카나 특수 표기

| 표기 | 발음 | 규칙 |
|------|------|------|
| は (조사) | wa | 주제 조사로 사용 시 |
| へ (조사) | e | 방향 조사로 사용 시 |
| を (조사) | o | 목적격 조사 |
| ぢ/づ | ji/zu | じ/ず와 동일 발음 |

## 음운 규칙

### 렌다쿠 (連濁)

복합어에서 두 번째 요소의 첫 자음이 유성음화:
- 山 + 川 → 山川 (やまがわ yamakawa → yamagawa)
- 人 + 々 → 人々 (ひとびと hitohito → hitobito)

### 음편 (音便)

활용에서 발생하는 음운 변화:
- イ音便: 書きて → 書いて (kakite → kaite)
- 促音便: 持ちて → 持って (mochite → motte)
- 撥音便: 読みて → 読んで (yomite → yonde)

### 모음 무성화

고모음 /i/, /ɯ/가 무성 자음 사이에서 무성화:
- すき /sɯki/ → [sɯ̥ki]
- きつね /kitsɯne/ → [ki̥tsɯ̥ne]
