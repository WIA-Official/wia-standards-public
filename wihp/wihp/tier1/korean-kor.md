## 개요

한국어는 한글이라는 자질 문자(featural alphabet)를 사용합니다. WIHP의 기준 언어이자 목표 표기 체계입니다. 한글의 과학적 설계 원리가 WIHP 전체의 기반이 됩니다.

---

## 한글의 과학적 원리

### 자음: 발음 기관 상형

```
ㄱ = 혀뿌리가 목구멍 막는 모양
ㄴ = 혀가 윗잇몸에 닿는 모양
ㅁ = 입술 모양
ㅅ = 이의 모양
ㅇ = 목구멍 모양
```

### 획 추가 = 소리 변화

```
ㄱ → ㅋ (기식음화) → ㄲ (경음화)
ㄴ → ㄷ (파열음화) → ㅌ (기식음화)
ㅁ → ㅂ (파열음화) → ㅍ (기식음화)
ㅅ → ㅈ (파찰음화) → ㅊ (기식음화)
ㅇ → ㅎ (기식음화)
```

### 모음: 천지인 철학

```
ㆍ = 하늘 (천/天) - 둥근 점
ㅡ = 땅 (지/地) - 평평한 선
ㅣ = 사람 (인/人) - 서 있는 선
```

| ㅈ → [dʑ] | 유성음 사이 | 아저씨 /adʑʌɕ͈i/ |

### 경음화

| 규칙 | 조건 | 예시 |

| 규칙 | 변화 | 예시 |

## WIHP에서의 한국어 위상

한국어와 한글은 WIHP 시스템의 핵심입니다:

1. **기준 언어**: 모든 IPA 음소의 한글 대응 기준
2. **자질 문자**: 음운 자질이 문자에 반영됨
3. **과학적 설계**: 조음 위치/방법이 자모 형태에 반영
4. **확장 가능**: 옛한글로 추가 음소 표현 가능

```
세종대왕 (1443): 한글 → 한국어 보편적 읽기
WIHP (2025): 한글 → 전 인류 언어 보편적 읽기

600년의 완성.
홍익인간 (弘益人間)
```
