## 개요

중국어(보통화/普通话)는 세계에서 가장 많은 원어민 화자를 보유한 언어입니다. 성조 언어로, 4개의 성조와 경성이 있습니다.

## 성조 변화 규칙

### 3성 연속 규칙

| 조건 | 변화 | 예시 |
|------|------|------|
| 3성 + 3성 | 첫 3성 → 2성 | 你好 nǐ hǎo → ní hǎo |
| 3성 + 1/2/4성 | 반3성 (저하강만) | 你们 nǐmen → nǐ↓men |

### 一(yī) 변조

| 조건 | 변화 | 예시 |
|------|------|------|
| 단독/숫자 | 1성 | 一 yī |
| + 1,2,3성 | 4성 | 一天 yì tiān |
| + 4성 | 2성 | 一个 yí gè |

### 不(bù) 변조

| 조건 | 변화 | 예시 |
|------|------|------|
| 기본 | 4성 | 不 bù |
| + 4성 | 2성 | 不是 bú shì |

## 철자-발음 대응 규칙

### ü 표기

| 규칙 | 예시 |
|------|------|
| j, q, x 뒤 → u로 표기 | ju, qu, xu (실제: jü, qü, xü) |
| l, n 뒤 → ü로 표기 | lü, nü |

### w, y 규칙

| 규칙 | 예시 |
|------|------|
| u로 시작 → w 추가 | u → wu, ua → wa |
| i로 시작 → y 추가 | i → yi, ia → ya |
| ü로 시작 → yu | ü → yu |

### 성모 없는 운모

| 운모 | 표기 |
|------|------|
| i | yi |
| u | wu |
| ü | yu |
| ie | ye |
| iu | you |
| ue | yue |
