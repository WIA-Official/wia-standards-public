# WIHP: Hungarian (Magyar) - hun

## Language Overview
- **Native Name:** Magyar
- **ISO 639-3:** hun
- **Speaker Population:** ~13 million
- **Language Family:** Uralic → Ugric
- **Writing System:** Latin alphabet (extended)
- **Phonological Type:** Agglutinative with vowel harmony

## Phonological Features
- **Vowel Harmony:** Front/back distinction
- **Length Distinction:** Both vowels and consonants
- **Palatalization:** Systematic
- **Stress:** Fixed initial stress

## Phonological Rules

### Vowel Harmony
Hungarian has strict vowel harmony dividing vowels into:
- **Back vowels:** a, á, o, ó, u, ú
- **Front vowels:** e, é, i, í, ö, ő, ü, ű

Suffixes must match the vowel class of the stem:
- házban (in the house) - back vowel stem
- kertben (in the garden) - front vowel stem

### Consonant Length
Consonant length is phonemic:
- hal /hɒl/ "fish" vs. hall /hɒll/ "hall"
- megy /mɛɟ/ "goes" vs. meggy /mɛɟɟ/ "sour cherry"

### Palatalization
Consonants before 'y' digraphs are palatalized:
- ty /c/, gy /ɟ/, ny /ɲ/, ly /j/

### Voicing Assimilation
Obstruents assimilate in voicing:
- képtár → /keːptaːr/ (assimilation in clusters)
- község → /køzʃeːg/ (regressive assimilation)

## WIHP Implementation Notes

1. **Vowel Length:** Use standard Hangul; length distinction not marked in WIHP
2. **Front Rounded Vowels:** ㅚ for ö/ő, ㅟ for ü/ű
3. **Palatal Consonants:** ㅊ for ty, ㅈ for gy, ㄴ+이 for ny
4. **Affricates:** Map carefully - c→ㅊ (ts), cs→ㅊ (tʃ)
5. **Initial Stress:** Hangul syllable structure accommodates this well
