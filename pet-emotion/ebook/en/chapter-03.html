3.1 Multi-Modal Approach Overview

No single data source can reliably capture the full emotional state of a pet. The WIA-PET-EMOTION standard requires integration of multiple modalities for accurate emotion detection.

Data Source Hierarchy

Modality Weight Primary Use
Visual 35% Body language, expressions
Physiological 30% Internal state validation
Audio 25% Vocalization analysis
Environmental 10% Context factors

3.3 Facial Expression Analysis

Dog Facial Action Units

DOG FACIAL CODING SYSTEM
========================

AU101 - Inner brow raiser
  - Associated with: Sadness, pleading
  - Detection: Medial brow position

AU102 - Outer brow raiser
  - Associated with: Alertness, surprise
  - Detection: Lateral brow position

AU201 - Ear forward
  - Associated with: Attention, excitement
  - Detection: Ear pinna angle

AU202 - Ear flattener
  - Associated with: Fear, submission
  - Detection: Ear against head

AU301 - Lip corner pull
  - Associated with: Relaxation (not smile)
  - Detection: Commissure position

AU401 - Eye widener
  - Associated with: Fear, alertness
  - Detection: Sclera visibility

Feature State Emotion

3.4 Tail Analysis

Dog Tail Positions

TAIL POSITION MATRIX
====================

Position: HIGH (above spine level)
  + Fast wag  = Excited, dominant
  + Slow wag  = Curious, assessing
  + Still     = Alert, confident

Position: NEUTRAL (spine level)
  + Fast wag  = Happy, friendly
  + Slow wag  = Relaxed attention
  + Still     = Calm, neutral

Position: LOW (below spine)
  + Slow wag  = Uncertain, submissive
  + Still     = Nervous, uncomfortable

Position: TUCKED (between legs)
  + Any       = Fear, extreme submission

Wag Frequency Analysis

Frequency (Hz) Amplitude Interpretation
Fast (>3) Wide Very excited, happy
Fast (>3) Narrow Nervous excitement
Medium (1-3) Wide Friendly, interested
Medium (1-3) Narrow Uncertain interest
Slow (<1) Any Cautious, assessing

3.5 Audio Detection System

Vocalization Classification

DOG VOCALIZATION TYPES
======================

BARK:
  - Alert bark: Sharp, repetitive, mid-pitch
  - Playful bark: Higher pitch, variable pattern
  - Demand bark: Persistent, directed
  - Fear bark: High pitch, rapid

WHINE:
  - Anxiety: Continuous, high pitch
  - Attention-seeking: Intermittent
  - Pain: Sharp onset, sustained

GROWL:
  - Warning: Low, sustained
  - Play: Variable pitch, starts/stops
  - Possessive: Intensifies with approach

HOWL:
  - Communication: Long, modulated
  - Separation: Sustained, mournful
  - Response: Triggered by sounds

Audio Feature Extraction

Feature Description Emotion Correlation
Fundamental Frequency Base pitch of vocalization Higher = arousal/stress
Frequency Range Variation in pitch Wider = excitement
Duration Length of vocalization Context-dependent
Harmonics Overtone structure Richer = positive states
Energy Volume/intensity Higher = urgency

3.6 Physiological Detection

Heart Rate Monitoring

HEART RATE EMOTION CORRELATION
==============================

Species: Dog (normal resting 60-140 bpm)

Elevation (+20% from baseline):
  - Excitement
  - Fear
  - Anxiety
  - Physical activity

Reduction (-10% from baseline):
  - Deep relaxation
  - Sleep approach
  - Contentment

High Variability (HRV):
  - Calm, adaptable state
  - Emotional flexibility
  - Good stress recovery

Low Variability:
  - Sustained stress
  - Fatigue
  - Illness possible

Wearable Device Requirements

Metric Minimum Spec Recommended
Heart Rate Accuracy +/- 5 bpm +/- 2 bpm
HRV Resolution 10 ms 1 ms
Sampling Rate 1 Hz 10 Hz
Activity Detection 3-axis accelerometer 6-axis IMU
Battery Life 24 hours 7 days

3.7 Environmental Context

Context Factors

ENVIRONMENTAL MODIFIERS
=======================

Location:
  - Home (baseline environment)
  - Outdoor (heightened alertness normal)
  - Veterinary (stress expected)
  - New location (curiosity/anxiety)

Time of Day:
  - Morning (activity increase)
  - Midday (rest period common)
  - Evening (feeding/play time)
  - Night (sleep/calm)

Weather:
  - Storm approaching (anxiety trigger)
  - Hot weather (reduced activity)
  - Cold weather (seeking warmth)

Social Context:
  - Owner present (baseline)
  - Owner absent (separation context)
  - Strangers present (alertness)
  - Other animals (social dynamics)

3.9 Conflict Resolution

Signal Disagreement Handling

Scenario Resolution
Visual vs Audio conflict Weight by confidence; flag for review
Physio contradicts behavioral Prioritize physiological (internal state)
All sources low confidence Return UNKNOWN with data quality flag
Single source much higher confidence Use highest confidence source
Two sources agree, one disagrees Majority vote with confidence adjustment
