제6장: 웰빙 평가

"홍익인간: 진정한 돌봄은 고통의 순간을 넘어 지속적인 감정적 번영을 가꾸는 것입니다."

6.1 웰빙 모델 개요

웰빙 평가는 시간에 따른 감정 데이터를 종합하여 동물의 삶의 질에 대한 전체적인 관점을 제공합니다. 순간적인 감정 읽기와 달리, 웰빙은 패턴과 추세를 반영합니다.

웰빙 피라미드

웰빙 피라미드
============

         /\
        /  \       번영
       / FL \      (자아실현)
      /------\
     /        \    감정 균형
    /   EB     \   (안정적 긍정 기분)
   /------------\
  /              \  기본 편안함
 /      BC        \ (고통 없음)
/------------------\

FL = 번영 (번창, 기쁨, 참여)
EB = 감정 균형 (안정, 적응 가능)
BC = 기본 편안함 (안전, 배부름, 건강)

6.2 웰빙 점수 계산

웰빙 점수 알고리즘
=================

입력: 30일 감정 히스토리

1. 긍정 감정 비율 계산
   긍정_수 = count(HAPPY, CALM, CONTENT, PLAYFUL, AFFECTIONATE)
   부정_수 = count(SAD, ANXIOUS, FEARFUL, FRUSTRATED, AGGRESSIVE)
   중립_수 = count(BORED, TIRED, ALERT)

   긍정_비율 = 긍정_수 / 총_수

2. 감정 안정성 계산
   상태_변화 = count(감정_전환)
   예상_변화 = 기준선_변화율 * 일수
   안정성 = 1 - abs(상태_변화 - 예상_변화) / 예상_변화

3. 고통 빈도 계산
   고통_에피소드 = count(강도 >= 4 AND 부정_감정)
   고통_점수 = 1 - (고통_에피소드 / 최대_허용)

4. 참여 수준 계산
   참여 = average(PLAYFUL, CURIOUS, EXCITED 강도)

5. 최종 점수 계산
   웰빙 = weighted_average(
       긍정_비율 * 30,
       안정성 * 25,
       고통_점수 * 25,
       참여 * 20
   )

출력: WellbeingScore (0-100)

6.3 웰빙 범주

점수 범위 범주 설명 조치
90-100 우수 번창, 번영 현재 케어 유지
75-89 양호 건강한 감정 상태 사소한 개선 가능
60-74 보통 일부 우려 존재 케어 루틴 검토
40-59 불량 중요한 문제 개입 권장
0-39 위험 심각한 고통 즉시 주의 필요

6.4 추세 분석

추세 감지

추세 분석 알고리즘
=================

기간: 30일 롤링 윈도우

1. 일일 웰빙 점수
   일일_점수 = [각 일의 점수]

2. 선형 회귀
   기울기, 절편 = linear_fit(일일_점수)

3. 추세 분류
   if 기울기 > 0.5:
       추세 = "IMPROVING"
   elif 기울기 < -0.5:
       추세 = "DECLINING"
   else:
       추세 = "STABLE"

4. 변화율
   비율 = 기울기 * 30  # 월간 변화

5. 예측
   30일_예측 = 현재_점수 + 비율
   90일_예측 = 현재_점수 + (비율 * 3)

추세 해석

추세 비율 해석
IMPROVING >5점/월 케어 변경이 잘 작동
IMPROVING 1-5점/월 점진적 긍정 변화
STABLE -1 ~ 1 일관된 감정 상태
DECLINING -1 ~ -5 면밀히 모니터링
DECLINING <-5점/월 긴급 주의 필요

6.5 기여 요인

요인 분석
=========

추적 요인:
  - 운동 빈도와 지속시간
  - 사회적 상호작용 시간
  - 급식 규칙성
  - 수면 품질
  - 환경 변화
  - 날씨 패턴
  - 보호자 존재/부재
  - 건강 이벤트

상관관계 분석:
  각 요인에 대해:
    상관관계 = pearson(요인_값, 웰빙_점수)
    if |상관관계| > 0.3:
        중요_요인_표시(요인)

영향 분류:
  POSITIVE: 상관관계 > 0.3 (웰빙 개선)
  NEUTRAL: 상관관계 -0.3 ~ 0.3
  NEGATIVE: 상관관계 < -0.3 (웰빙 해침)

일반적 긍정 요인

  • [OK] 에너지 수준에 맞는 규칙적 운동
  • [OK] 일관된 일상 루틴
  • [OK] 긍정적 사회적 상호작용
  • [OK] 정신적 자극과 놀이
  • [OK] 보호자 존재와 관심

일반적 부정 요인

  • [!] 갑작스러운 루틴 변화
  • [!] 장기간 보호자 부재
  • [!] 운동 부족
  • [!] 환경적 스트레스 요인 (소음, 갈등)
  • [!] 건강 문제

6.6 감정 인사이트

인사이트 생성

인사이트 유형
============

1. 패턴 인사이트
   "맥스는 천둥 시 불안이 증가합니다"
   "아침 산책과 행복 에피소드가 상관관계"

2. 비교 인사이트
   "웰빙이 품종 평균보다 15% 높음"
   "활동 수준이 나이 대비 평균 이하"

3. 권장 인사이트
   "오후 놀이 세션 추가 고려"
   "분리 시간이 편안한 지속시간 초과"

4. 경고 인사이트
   "이번 주 불안 빈도 증가"
   "수면 패턴이 불규칙해짐"

인사이트 구조

interface EmotionInsight {
  insightId: string
  profileId: string
  timestamp: DateTime

  category: "pattern" | "comparison" | "recommendation" | "warning"
  priority: "low" | "medium" | "high"

  title: string
  description: string

  supportingData: {
    metric: string
    currentValue: number
    comparisonValue: number
    period: string
  }

  actionable: boolean
  suggestedAction: string
}

6.7 비교 분석

비교 유형 목적 데이터 소스
자가 과거 개인 진행 추적 반려동물 자체 히스토리
품종 평균 종 전형 비교 집계된 익명 데이터
연령 그룹 생애 단계 비교 연령 매칭 코호트
가정 내 반려동물 다중 반려동물 비교 같은 보호자의 반려동물

6.8 웰빙 보고서

주간 웰빙 보고서
===============

+-----------------------------------+
|  웰빙 요약: 맥스                  |
|  2024년 1월 15-21일 주            |
+-----------------------------------+

전체 점수: 78/100 [양호]
추세: IMPROVING (지난주 대비 +3)

감정 분포:
  HAPPY:       25% ||||||||
  CALM:        30% ||||||||||
  PLAYFUL:     15% |||||
  ANXIOUS:      5% ||
  OTHER:       25% ||||||||

주요 관찰:
  [OK] 불안 에피소드 40% 감소
  [OK] 놀이 시간 45분/일로 증가
  [i] 수면 약간 방해됨 (새 이웃 소음)

권장사항:
  1. 현재 운동 루틴 계속
  2. 수면 공간에 백색 소음 고려

  - 아침 산책 (상관관계: 0.72)
  - 보호자 재택근무일 (상관관계: 0.58)

우려사항:
  - 천둥 불안 여전히 존재
  - 짧은 분리 불안 관찰됨

6.9 개입 추적

개입 유형

개입 범주
=========

환경:
  - 조용한 공간 조성
  - 운동 구역 개선
  - 편안함 용품 추가

행동:
  - 훈련 세션
  - 탈감작 프로토콜
  - 풍부화 활동

의료:
  - 수의사 상담
  - 약물 시작/조정
  - 보충제 추가

사회:
  - 탁아소 등록
  - 반려동물 동반자 도입
  - 보호자 일정 조정

개입 성공 측정

지표 목표 변화 기간
웰빙 점수 +10점 30일
부정 감정 빈도 -50% 14일
고통 에피소드 지속시간 -30% 14일
긍정 감정 비율 +15% 30일

6.10 장기 웰빙

생애 단계 고려사항
=================

강아지/새끼고양이 (0-1세):
  - 더 높은 기준선 각성
  - 빠른 감정 발달
  - 사회화 기간 중요
  - 나이에 맞게 기대 조정

성체 (1-7세):
  - 안정적 감정 패턴
  - 성격 완전히 발달
  - 최적 기준선 기간
  - 변화 모니터링

노령 (7세 이상):
  - 불안 증가 가능
  - 활동 수준 감소
  - 인지 변화 가능
  - 정상 범위 조정

장 요약

  • [OK] 웰빙 점수는 시간에 따른 감정 데이터 종합
  • [OK] 우수(90+)에서 위험(<40)까지 범주화
  • [OK] 추세 분석이 개선 또는 쇠퇴 추적
  • [OK] 요인 분석이 웰빙에 대한 영향 식별
  • [OK] 인사이트가 실행 가능한 권장사항 제공
  • [OK] 개입 효과성 추적 가능
