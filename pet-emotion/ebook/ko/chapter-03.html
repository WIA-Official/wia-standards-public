제3장: 다중 모달 감지 시스템

"홍익인간: 말, 어조, 행동을 통해 친구를 이해하듯, 여러 표현 채널을 통해 동반자를 이해합니다."

3.1 다중 모달 접근법 개요

단일 데이터 소스만으로는 반려동물의 전체 감정 상태를 신뢰성 있게 포착할 수 없습니다. WIA-PET-EMOTION 표준은 정확한 감정 감지를 위해 여러 모달리티의 통합을 요구합니다.

데이터 소스 계층

모달리티 가중치 주요 용도
시각 35% 신체 언어, 표정
생리 30% 내부 상태 검증
청각 25% 발성 분석
환경 10% 맥락 요인

신체 자세 분석

자세 감지 파이프라인
==================

1. 프레임 캡처 (최소 30 fps)
      |

      |

   - 3초 롤링 평균
   - 급격한 변화 감지

11-14 다리 활동, 준비
15-17 꼬리 포인트 직접적 감정 신호

3.3 표정 분석

개 얼굴 행동 단위

개 얼굴 코딩 시스템
=================

AU101 - 안쪽 눈썹 올림
  - 관련: 슬픔, 간청
  - 감지: 중앙 눈썹 위치

AU102 - 바깥 눈썹 올림
  - 관련: 경계, 놀람
  - 감지: 측면 눈썹 위치

AU201 - 귀 앞으로
  - 관련: 주의, 흥분
  - 감지: 귓바퀴 각도

AU202 - 귀 납작
  - 관련: 두려움, 복종
  - 감지: 귀가 머리에 붙음

AU301 - 입꼬리 당김
  - 관련: 이완 (미소 아님)
  - 감지: 입꼬리 위치

AU401 - 눈 크게 뜸
  - 관련: 두려움, 경계
  - 감지: 흰자 노출

고양이 얼굴 특징

특징 상태 감정
동공 확장 두려움, 흥분, 어두움
동공 수축 공격성, 밝은 빛
귀 앞으로 호기심, 경계
귀 납작 두려움, 공격성
수염 앞으로 관심, 사냥
수염 뒤로 두려움, 방어

3.4 꼬리 분석

개 꼬리 위치

꼬리 위치 매트릭스
=================

위치: 높음 (척추 위)
  + 빠른 흔들기 = 흥분, 지배적
  + 느린 흔들기 = 호기심, 평가
  + 정지       = 경계, 자신감

위치: 중립 (척추 수준)
  + 빠른 흔들기 = 행복, 친근
  + 느린 흔들기 = 이완된 주의
  + 정지       = 평온, 중립

위치: 낮음 (척추 아래)
  + 느린 흔들기 = 불확실, 복종
  + 정지       = 긴장, 불편

위치: 다리 사이 숨김
  + 모두       = 두려움, 극도의 복종

흔들기 빈도 분석

빈도 (Hz) 진폭 해석
빠름 (>3) 넓음 매우 흥분, 행복
빠름 (>3) 좁음 긴장된 흥분
보통 (1-3) 넓음 친근, 관심
보통 (1-3) 좁음 불확실한 관심
느림 (<1) 모두 조심, 평가

3.5 청각 감지 시스템

발성 분류

개 발성 유형
============

짖음:
  - 경계 짖음: 날카롭고, 반복적, 중간 음
  - 장난 짖음: 높은 음, 다양한 패턴
  - 요구 짖음: 지속적, 방향성
  - 두려움 짖음: 높은 음, 빠름

끙끙거림:
  - 불안: 연속적, 높은 음
  - 관심 끌기: 간헐적
  - 통증: 급격한 시작, 지속

으르렁:
  - 경고: 낮고, 지속
  - 놀이: 다양한 음, 시작/멈춤
  - 소유욕: 접근 시 강화

울부짖음:
  - 소통: 길고, 변조됨
  - 분리: 지속적, 슬픔
  - 반응: 소리에 의해 유발

오디오 특징 추출

특징 설명 감정 상관관계
기본 주파수 발성의 기본 음 높음 = 각성/스트레스
주파수 범위 음 변화 넓음 = 흥분
지속시간 발성 길이 맥락 의존
배음 상음 구조 풍부함 = 긍정 상태
에너지 볼륨/강도 높음 = 긴급

3.6 생리 감지

심박수 모니터링

심박수 감정 상관관계
==================

종: 개 (정상 휴식 60-140 bpm)

상승 (기준선 +20%):
  - 흥분
  - 두려움
  - 불안
  - 신체 활동

감소 (기준선 -10%):
  - 깊은 이완
  - 수면 접근
  - 만족

높은 변이도 (HRV):
  - 평온, 적응 가능 상태
  - 감정 유연성
  - 좋은 스트레스 회복

낮은 변이도:
  - 지속적 스트레스
  - 피로
  - 질병 가능

웨어러블 기기 요구사항

지표 최소 사양 권장
심박수 정확도 +/- 5 bpm +/- 2 bpm
HRV 해상도 10 ms 1 ms
샘플링 속도 1 Hz 10 Hz
활동 감지 3축 가속도계 6축 IMU
배터리 수명 24시간 7일

3.7 환경 맥락

맥락 요인

환경 수정자
==========

위치:
  - 집 (기준 환경)
  - 야외 (높은 경계 정상)
  - 동물병원 (스트레스 예상)
  - 새 장소 (호기심/불안)

시간대:
  - 아침 (활동 증가)
  - 낮 (휴식 시간 일반)
  - 저녁 (식사/놀이 시간)
  - 밤 (수면/평온)

날씨:
  - 폭풍 접근 (불안 트리거)
  - 더운 날씨 (활동 감소)
  - 추운 날씨 (따뜻함 찾음)

사회적 맥락:
  - 보호자 있음 (기준)
  - 보호자 없음 (분리 맥락)
  - 낯선 사람 있음 (경계)
  - 다른 동물 (사회적 역학)

3.9 충돌 해결

신호 불일치 처리

시나리오 해결 방법
시각 vs 청각 충돌 신뢰도로 가중치; 검토 플래그
생리가 행동과 모순 생리 우선 (내부 상태)
모든 소스 낮은 신뢰도 UNKNOWN 반환, 데이터 품질 플래그
단일 소스 훨씬 높은 신뢰도 가장 높은 신뢰도 소스 사용
두 소스 일치, 하나 불일치 신뢰도 조정과 함께 다수결

3.10 실시간 처리 요구사항

지연 시간 요구사항
=================

시각 처리:
  - 프레임 캡처에서 감정까지: < 100ms
  - 키포인트 감지: < 30ms
  - 표정 분류: < 50ms

청각 처리:
  - 발성 감지: < 50ms
  - 분류: < 100ms
  - 전체: < 150ms

생리:
  - 센서에서 읽기까지: < 1000ms
  - 추세 감지: < 5000ms

융합:
  - 모든 입력에서 출력까지: < 200ms
  - 총 시스템 지연: < 500ms

  • [OK] 얼굴 행동 단위가 종별 표정 코딩 제공
  • [OK] 꼬리 위치와 움직임은 핵심 감정 지표
  • [OK] 청각 분석은 발성에서 감정 내용 추출
  • [OK] 충돌 해결이 있는 센서 융합이 정확성 보장
