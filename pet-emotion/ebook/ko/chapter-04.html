4.2 시각 분석 알고리즘

신체 자세 분류

자세 분석 알고리즘
=================

입력: 키포인트 좌표 (17개 포인트)

1. 척추 곡률 계산
   곡률 = angle(어깨, 중간_척추, 엉덩이)

2. 긴장 지표 측정
   머리_높이 = normalize(머리.y - 어깨.y)
   꼬리_위치 = normalize(꼬리_기저.y - 척추.y)
   다리_벌림 = distance(앞다리) + distance(뒷다리)

3. 자세 분류
   if 꼬리_숨김 AND 머리_낮음:
       자세 = "두려운_웅크림"
   elif 척추_위로_곡선 AND 다리_벌림:
       자세 = "공격적_자세"
   elif 누움 AND 이완된_근육:
       자세 = "편안한_휴식"
   elif 서있음 AND 꼬리_높음:
       자세 = "경계_자신감"

4. 긴장 점수 계산 (0-1)
   긴장 = weighted_average(
       근육_경직,
       움직임_떨림,
       자세_경직
   )

출력: PostureResult(자세, 긴장, 신뢰도)

4.3 표정 알고리즘

표정 특징 추출

얼굴 분석 단계
=============

1. 얼굴 감지
   - 얼굴 바운딩 박스 위치
   - 얼굴 영역 추출

2. 랜드마크 감지
   - 눈 (각 4포인트)
   - 귀 (각 3포인트)
   - 코 (1포인트)
   - 입꼬리 (2포인트)

3. 특징 계산
   눈_열림 = 눈_높이 / 눈_너비
   귀_각도 = angle(귀_끝, 귀_기저, 머리_중심)
   눈썹_위치 = 눈썹.y - 눈_중심.y

4. 행동 단위 감지
   AU_점수 = {}
   for AU in [101, 102, 201, 202, 301, 401]:
       AU_점수[AU] = classify_AU(랜드마크, AU)

5. 표정 분류
   표정 = map_AU_to_expression(AU_점수)

표정 감정 매핑

표정 패턴 주요 감정 신뢰도 증가
이완된 얼굴, 부드러운 눈 CALM / CONTENT +15%
큰 눈, 귀 앞으로 ALERT / CURIOUS +10%
귀 뒤로, 흰자 보임 FEARFUL / ANXIOUS +20%
코 주름, 이빨 노출 AGGRESSIVE +25%
올라간 눈썹, 부드러운 입 HAPPY / AFFECTIONATE +10%

4.4 청각 분석 알고리즘

발성 처리

청각 분석 파이프라인
==================

1. 오디오 전처리
   - 샘플레이트: 44.1kHz
   - 노이즈 게이트: -40dB 임계값
   - 밴드패스 필터: 50Hz - 8kHz

2. 발성 감지
   에너지 = compute_energy(프레임)
   if 에너지 > 임계값:
       발성_시작_표시()
   elif 에너지 < 해제_임계값:
       발성_종료_표시()

3. 특징 추출
   각 발성 세그먼트에 대해:
   - 기본 주파수 (F0)
   - 포만트 (F1, F2, F3)
   - MFCC 계수 (13개)
   - 지속시간 및 에너지 윤곽

4. 분류
   특징 = extract_features(세그먼트)
   발성_유형 = classifier.predict(특징)

   유형: 짖음, 끙끙, 으르렁, 울부짖음, 골골, 야옹, 하악

5. 감정 추론
   감정_점수 = vocalization_to_emotion(
       유형, 음높이, 지속시간, 에너지
   )

발성-감정 매핑

발성 감정 규칙
=============

짖음 분석:
  높은 음 + 빠름 = 흥분 또는 두려움
  낮은 음 + 단발 = 경계
  중간 음 + 리듬적 = 장난
  거친 + 연속 = 공격적

끙끙거림 분석:
  지속 + 높음 = 불안
  간헐 + 상승 = 관심 끌기
  급격한 시작 = 통증 지표

으르렁 분석:
  지속 + 낮음 = 경고 (공격적)
  다양한 음 + 장난 맥락 = 놀이 으르렁
  몸 경직과 함께 = 진짜 위협

골골 분석 (고양이):
  안정 + 이완된 몸 = 만족
  더 큼 + 접촉 추구 = 애정
  질병 징후와 함께 = 자기 위안

4.5 생리 분석 알고리즘

심박수 분석

심박수 감정 알고리즘
==================

입력: HR 시계열, 기준선 값

1. 현재 지표 계산
   현재_hr = average(최근_30초)
   현재_hrv = rmssd(rr_간격)

2. 기준선 비교
   hr_편차 = (현재_hr - 기준선_hr) / 기준선_hr
   hrv_비율 = 현재_hrv / 기준선_hrv

3. 각성 상태 분류
   if hr_편차 > 0.3:
       각성 = HIGH
   elif hr_편차 > 0.1:
       각성 = MODERATE
   elif hr_편차 < -0.1:
       각성 = LOW
   else:
       각성 = NEUTRAL

4. 스트레스 평가
   if hrv_비율 < 0.7 AND hr_편차 > 0.2:
       스트레스_수준 = HIGH
   elif hrv_비율 < 0.85:
       스트레스_수준 = MODERATE
   else:
       스트레스_수준 = LOW

출력: PhysioState(각성, 스트레스_수준, 신뢰도)

4.6 다중 모달 융합 알고리즘

가중 융합

감정 융합 알고리즘
=================

입력:
  시각_점수: Dict[EmotionType, float]
  청각_점수: Dict[EmotionType, float]
  생리_점수: Dict[EmotionType, float]
  신뢰도: Dict[소스, float]

1. 신뢰도 가중치 정규화
   총_신뢰도 = sum(신뢰도.values())
   가중치 = {소스: 신뢰/총_신뢰도 for 소스, 신뢰 in 신뢰도}

2. 기본 모달리티 가중치 적용
   모달리티_가중치 = {
       'visual': 0.35,
       'audio': 0.25,
       'physio': 0.30,
       'environment': 0.10
   }

3. 결합 가중치 계산
   for 소스 in 소스들:
       결합_가중치[소스] = (
           모달리티_가중치[소스] *
           가중치[소스]
       )
   # 재정규화
   결합_가중치 = normalize(결합_가중치)

4. 감정 점수 융합
   융합_점수 = {}
   for 감정 in EmotionType:
       융합_점수[감정] = (
           결합_가중치['visual'] * 시각_점수[감정] +
           결합_가중치['audio'] * 청각_점수[감정] +
           결합_가중치['physio'] * 생리_점수[감정]
       )

5. 주요 감정 결정
   주요 = max(융합_점수, key=융합_점수.get)
   신뢰도 = 융합_점수[주요]

출력: EmotionResult(주요, 강도, 신뢰도)

4.7 시간적 통합

감정 스무딩

시간적 스무딩 알고리즘
====================

목적: 노이즈 감소, 안정성 보장

매개변수:
  윈도우_크기 = 5초
  최소_지속시간 = 2초
  변화_임계값 = 0.3

1. 감정 히스토리 버퍼 유지
   히스토리 = RingBuffer(용량=윈도우_크기 * fps)

2. 새 읽기마다
   히스토리.append(현재_감정)

3. 안정성 계산
   감정_카운트 = Counter(히스토리)
   지배_감정 = 감정_카운트.most_common(1)[0]
   안정성 = 지배_감정[1] / len(히스토리)

4. 스무딩 적용
   if 안정성 > 0.6:
       출력 = 지배_감정[0]
   elif 현재 != 이전 AND 현재.신뢰도 > 변화_임계값:
       출력 = 현재  # 전환 허용
   else:
       출력 = 이전  # 안정성 유지

5. 상태 지속시간 추적
   if 출력 == 이전_출력:
       지속시간 += 델타_시간
   else:
       지속시간 = 0
       상태_변화_이벤트_트리거()

출력: SmoothedEmotion(감정, 지속시간, 안정성)

4.8 신뢰도 계산

신뢰도 점수 알고리즘
==================

신뢰도에 기여하는 요인:

1. 데이터 품질 점수 (0-25점)
   - 센서 읽기 정상 범위 내
   - 누락 데이터 없음
   - 신호 대 잡음비 허용

2. 소스 일치 (0-25점)
   - 모든 소스 같은 감정 제안: +25
   - 다수 일치: +15
   - 분할된 결정: +5
   - 모순: +0

3. 과거 일관성 (0-25점)
   - 최근 패턴과 일치: +25
   - 합리적 전환: +15
   - 예상치 못한 변화: +5

4. 기준선 편차 (0-25점)
   - 기준선에서 명확한 편차: +25
   - 보통 편차: +15
   - 정상 변동 내: +10

최종 신뢰도 = sum(점수)

신뢰도 수준:
  85-100: HIGH
  65-84:  MODERATE
  50-64:  LOW
  0-49:   INSUFFICIENT
