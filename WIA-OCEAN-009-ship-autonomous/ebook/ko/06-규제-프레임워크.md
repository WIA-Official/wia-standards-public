# 6장: 규제 및 법적 프레임워크

## 미지의 법적 해역 항해

자율운항선박은 선박에 인간 선원이 승선한다는 가정 하에 구축된 수세기의 해사법에 도전합니다. 국제 기구, 기국 및 선급 협회는 안전하고 법적으로 준수하는 자율 해운을 가능하게 하는 새로운 규제 프레임워크를 개발하고 있습니다.

### 국제해사기구 (IMO)

유엔 전문 기관인 IMO는 국제 해운 규정을 관장합니다.

**IMO MASS (해상 자율운항선박) 프레임워크:**

2021년 6월 해사안전위원회(MSC)가 채택한 프레임워크는 4단계의 자율성을 정의합니다:

```typescript
enum MASSAutonomyLevel {
  DEGREE_ONE = 1,   // 자동화된 프로세스 및 의사결정 지원이 있는 선박
  DEGREE_TWO = 2,   // 선원이 승선한 원격조종 선박
  DEGREE_THREE = 3, // 선원이 미승선한 원격조종 선박
  DEGREE_FOUR = 4   // 완전 자율운항선박
}

interface MASSVessel {
  imoNumber: number;
  autonomyDegree: MASSAutonomyLevel;

  // 준수 요구사항은 단계에 따라 다름
  complianceRequirements: {
    safety: SafetyRequirements;
    security: SecurityRequirements;
    environmental: EnvironmentalRequirements;
    liability: LiabilityRequirements;
  };

  // 운항 제한
  operationalLimits: {
    geographicRestrictions?: GeoFence[];
    weatherLimits?: WeatherLimits;
    trafficDensityLimits?: TrafficDensityRestriction;
    timeRestrictions?: TimeRestriction;
  };

  // 인증
  certificates: {
    classification: ClassificationCertificate;
    flagState: FlagStateApproval;
    portState: PortStateAcceptance[];
  };
}
```

**MASS에 영향을 받는 주요 IMO 협약:**

**SOLAS (해상인명안전협약):**
- 5장: 항행의 안전
  - 규정 10: 선박 항로
  - 규정 19: 선박용 항행 시스템 탑재 요구사항
  - **MASS 영향:** 자율 항행 시스템은 기존 유인 항행과 동등한 안전성을 제공해야 함

**COLREGS (충돌 예방 규칙):**
- 규칙 5: 경계
- 규칙 8: 충돌을 피하기 위한 동작
- **MASS 영향:** 자율 시스템은 모든 충돌 회피 규칙 준수를 입증해야 함

**STCW (선원의 훈련, 자격증명 및 당직 근무 기준):**
- **MASS 영향:** 다음에 대한 새로운 인증 필요:
  - 원격 운영자
  - 육상 제어 센터 인력
  - 자율 시스템 엔지니어
  - AI 안전 감사자

**MARPOL (해양 오염 방지):**
- **MASS 영향:** 자율 시스템은 선원 감독 없이 오염을 모니터링하고 방지해야 함
