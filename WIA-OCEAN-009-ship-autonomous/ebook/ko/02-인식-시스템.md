# 2장: 인식 시스템 - 자율운항선박의 눈과 귀

## 환경 인식 구축

자율운항선박은 인간 항해사보다 더 높은 신뢰성으로 환경을 인식해야 합니다. 인간 항해사가 눈, 귀, 직관을 사용하는 반면, 자율 선박은 주변 상황에 대한 포괄적인 이해를 만들기 위해 함께 작동하는 여러 센서 시스템에 의존합니다.

### 센서 모음

현대 자율운항선박은 6가지 주요 센서 범주를 통합합니다:

#### 해양 레이더 시스템

레이더(무선 탐지 및 거리 측정)는 1940년대 이후 신뢰성이 입증되어 해양 인식의 중추로 남아 있습니다.

**X-밴드 레이더 (9-10 GHz):**
- **주 용도:** 고해상도 목표물 탐지
- **범위:** 최대 48해리
- **정확도:** 근거리에서 10-20미터
- **업데이트 속도:** 회전당 2-3초
- **기상:** 비/안개의 영향을 받음 (높은 감쇠)
- **목표물 탐지:** 소형 보트, 부표, 해안선 세부사항

**S-밴드 레이더 (2-4 GHz):**
- **주 용도:** 장거리 감시
- **범위:** 96+ 해리
- **정확도:** 50-100미터
- **업데이트 속도:** 회전당 2-4초
- **기상:** 더 나은 비/안개 투과
- **목표물 탐지:** 대형 선박, 기상 시스템, 육지

**현대 고체 상태 레이더:**
- 마그네트론 없음 (예열이 필요한 기존 구성 요소)
- 즉시 켜짐 기능
- 50,000+ 시간 수명 (마그네트론은 3,000시간)
- 낮은 전력 소비
- 여러 동시 빔
- 더 나은 클러터 억제

**한국 레이더 기술:**

삼성탈레스와 LIG넥스원은 선박용 레이더 시스템을 개발합니다:
- 고해상도 해양 감시 레이더
- 충돌 회피를 위한 ARPA 시스템
- 한국 해군 및 상선에 공급

**레이더 처리:**

```typescript
interface RadarSystem {
```

#### 라이다 (Light Detection and Ranging)

라이다는 레이저 펄스를 사용하여 선박 주변의 고해상도 3D 맵을 제공합니다.

**해양 라이다 사양:**
- **범위:** 해양 응용에 일반적으로 200-500미터
- **정확도:** ±2-5센티미터
- **업데이트 속도:** 10-20 Hz (전체 360° 스캔)
- **파장:** 905nm 또는 1550nm (적외선)
- **초당 포인트:** 300,000 - 1,000,000
- **기상 성능:** 짙은 안개/비에서 감소

**레이더 대비 장점:**
- 훨씬 높은 해상도 (센티미터 대 미터)
- 장애물의 정확한 3D 형상
- 인근 물체까지의 정확한 거리
- 작은 물체 감지 (잔해, 소형 보트, 부표)
- 항만/항구 운영에 적합

**라이다 데이터 처리:**

**장애물 감지 알고리즘:**
```typescript
class LidarObstacleDetector {
  private readonly MIN_OBJECT_POINTS = 50;
  private readonly MAX_OBJECT_DISTANCE = 300;  // 미터

  detectObstacles(pointCloud: PointCloud): Obstacle[] {
    // 1. 지면 제거 (해수면)
    const seaLevel = this.estimateSeaLevel(pointCloud);
    const filteredPoints = pointCloud.points.filter(
      p => p.z > seaLevel + 0.5  // 수면 위 50cm
    );

    // 2. 포인트를 물체로 클러스터링
    const clusters = this.dbscanClustering(filteredPoints, {
      epsilon: 0.5,      // 포인트 간 최대 거리 50cm
      minPoints: this.MIN_OBJECT_POINTS
    });

    // 3. 각 클러스터 분류
    const obstacles = clusters.map(cluster => {
      const bbox = this.calculateBoundingBox(cluster);
      const type = this.classifyCluster(cluster, bbox);

      return {
        type,
        position: this.calculateCentroid(cluster),
        boundingBox: bbox,
        distance: this.calculateDistance(cluster),
        threat: this.assessThreat(bbox, type)
      };
    });

    return obstacles.filter(o => o.distance < this.MAX_OBJECT_DISTANCE);
  }

  private classifyCluster(points: LidarPoint[], bbox: BoundingBox3D): string {
    const volume = bbox.length * bbox.width * bbox.height;
    const aspectRatio = bbox.length / bbox.width;

    // 선박: 크고 길쭉함
    if (volume > 1000 && aspectRatio > 2.0) {
      return "vessel";
    }

    // 부표: 작고 거의 구형
    if (volume < 10 && aspectRatio < 2.0 && bbox.height > 1.0) {
      return "buoy";
    }

    // 부두/선착장: 매우 길고 낮음
    if (bbox.length > 50 && bbox.height < 5) {
      return "pier";
    }

    return "unknown";
  }
}
```

#### 광학 카메라 시스템

카메라는 시각적 확인을 제공하고 AI 기반 물체 인식을 가능하게 합니다.

**카메라 어레이 구성:**
- **360° 커버리지:** 선박 주변에 6-10개 카메라
- **해상도:** 카메라당 최소 4K (3840×2160)
- **프레임 속도:** 표준 30 fps, 중요 영역은 60 fps
- **시야각:** 카메라당 80-120°, 20% 중첩
- **다이나믹 레인지:** 태양 눈부심과 그림자를 처리하기 위한 HDR

**카메라 유형:**

**가시광선 (RGB) 카메라:**
- 주간 1차 인식
- 깃발 식별, 조명 색상을 위한 색상 정보
- 항해등 패턴 읽기 (빨강/녹색/흰색)
- 다른 선박의 AIS 트랜스폰더 디스플레이 캡처

**적외선 (열화상) 카메라:**
- 능동 조명 없이 야간 투시
- 선박, 사람의 열 신호 감지
- 가벼운 안개 투과
- 범위: 선박 크기 목표물에 대해 5-10 km

**저조도 카메라:**
- 별빛 감도 (0.001 lux)
- 주간 및 야간 카메라 간의 간격 연결
- 저조도에서 색상 정보

**컴퓨터 비전 처리:**

**AI 물체 감지:**
```typescript
class MarineObjectDetector {
  private model: NeuralNetwork;  // YOLO v8 또는 유사

  async detectShips(frame: CameraFrame): Promise<ShipDetection[]> {
    // 신경망 추론 실행
    const detections = await this.model.detect(frame.image);

    // 선박 감지 필터링
    const ships = detections
      .filter(d => d.class === "ship" || d.class === "vessel")
      .filter(d => d.confidence > 0.7);

    // 2차 모델을 사용하여 선박 유형 분류
    const classified = await Promise.all(
      ships.map(async ship => {
        const cropped = this.cropImage(frame.image, ship.boundingBox);
        const vesselType = await this.classifyVesselType(cropped);

        return {
          ...ship,
          vesselType,
          estimatedLength: this.estimateLengthFromImage(ship, frame.cameraParams),
          navigationStatus: this.analyzeNavigationStatus(cropped)
        };
      })
    );

    return classified;
  }
}
```

#### AIS (자동 식별 시스템)

AIS는 VHF 라디오를 통해 선박 식별, 위치, 침로 및 속도를 방송합니다.

**AIS Class A (대형 선박):**
- 이동 시 2-10초마다 업데이트
- 범위: 20-40해리
- 300 GT 이상 선박에 의무
- 송신 전력: 12.5W

**AIS Class B (소형 선박):**
- 30초마다 업데이트
- 범위: 5-10해리
- 낮은 송신 전력: 2W
- 레저 보트, 어선에서 사용

**AIS 데이터 구조:**
```typescript
interface AISMessage {
```

#### GPS/GNSS 항법

전 세계 항법 위성 시스템을 사용한 정밀 위치 측정.

**다중 위성군 GNSS:**
- **GPS (미국):** 31개 위성, 전 세계 커버리지
- **GLONASS (러시아):** 24개 위성
- **Galileo (EU):** 26개 위성
- **BeiDou (중국):** 35개 위성

**위치 정확도:**
- 표준 GPS: 5-10미터
- 이중 주파수 GPS: 1-2미터
- SBAS (WAAS/EGNOS): 1-3미터
- RTK (실시간 동적 측위): 1-5센티미터

**자율 항법 요구사항:**

### 센서 융합

개별 센서에는 한계가 있습니다. 센서 융합은 모든 소스의 데이터를 결합하여 강력한 환경 인식을 만듭니다.

```typescript
  ownShip: {
    position: GNSSPosition;
```
