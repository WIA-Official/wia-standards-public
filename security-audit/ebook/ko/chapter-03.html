제3장: 규정 준수 프레임워크

주요 표준 및 요구 사항

3.1 규정 준수 개요

WIA-SEC-017은 다음 주요 규정 준수 프레임워크를 지원하도록 설계되었습니다:

프레임워크적용 대상주요 초점
SOC 2 Type IISaaS 제공업체서비스 조직 통제
ISO/IEC 27001모든 조직정보 보안 관리
GDPREU 개인정보 처리데이터 보호
HIPAA의료 기관건강 정보 보호
PCI-DSS결제 처리카드 데이터 보안

3.2 SOC 2 Type II

SOC 2(Service Organization Control 2)는 클라우드 서비스 제공업체를 위한 감사 표준입니다. Type II 보고서는 일정 기간 동안 통제의 효과성을 평가합니다.

신뢰 서비스 기준

기준설명감사 요구 사항
보안무단 액세스 방지액세스 로깅, 침입 탐지
가용성시스템 가용성가동 시간 모니터링
처리 무결성완전하고 정확한 처리트랜잭션 로깅
기밀성기밀 정보 보호액세스 제어 로깅
개인정보개인정보 보호동의 및 액세스 추적

주요 통제 항목

  • CC6.1: 보안 이벤트 탐지를 위한 시스템 모니터링
  • CC6.2: 구성 및 소프트웨어 변경 로깅
  • CC6.3: 사용자 액세스 및 활동 모니터링
  • CC7.1: 보안 이벤트 및 이상 탐지
  • CC7.2: 보안 사고 대응

3.3 ISO/IEC 27001

ISO 27001은 정보 보안 관리 시스템(ISMS)의 국제 표준입니다.

Annex A 로깅 통제

통제설명
A.12.4.1이벤트 로깅: 사용자 활동, 예외, 오류, 보안 이벤트 기록
A.12.4.2로그 정보 보호: 변조 및 무단 액세스 방지
A.12.4.3관리자/운영자 로그: 별도 유지 관리
A.12.4.4시계 동기화: 정확한 타임스탬프 보장

WIA-SEC-017 매핑

  • A.12.4.1 → 포괄적 이벤트 스키마
  • A.12.4.2 → 해시 체인 및 WORM 저장소
  • A.12.4.3 → 역할 기반 이벤트 분리
  • A.12.4.4 → NTP 동기화 요구 사항

3.4 GDPR

일반 데이터 보호 규정(EU)은 개인정보 처리에 대한 엄격한 요구 사항을 규정합니다.

제30조: 처리 활동 기록

조직은 다음을 포함한 처리 활동 기록을 유지해야 합니다:

  • 처리 목적
  • 데이터 주체 및 개인정보 범주
  • 개인정보 수신자
  • 제3국 전송 여부
  • 보존 기간
  • 보안 조치

⚠ 제33조: 침해 통지

개인정보 침해 인지 후 72시간 이내에 감독 기관에 통지해야 합니다. 이를 위해 감사 시스템은 거의 실시간으로 침해를 탐지해야 합니다.

제17조: 삭제권 (잊힐 권리)

감사 로그와 삭제권 간의 균형:

  • 감사 로그는 법적 의무로 보존 가능
  • 개인정보는 익명화/가명화 처리
  • 보존 기간 후 자동 삭제 구현

3.5 HIPAA

건강 보험 이동성 및 책임에 관한 법률은 미국 의료 기관에 적용됩니다.

감사 통제 요구 사항

조항요구 사항
§164.312(b)ePHI 포함 시스템의 활동 기록 및 검사 메커니즘
§164.308(a)(1)(ii)(D)감사 로그의 정기적 검토
§164.312(c)(1)ePHI 무결성 보호 메커니즘
§164.312(e)(1)전송 보안 보호

보존 요구 사항

HIPAA는 감사 로그를 최소 6년간 보존하도록 요구합니다. WIA-SEC-017의 계층화된 저장소는 이 장기 보존을 비용 효율적으로 지원합니다.

3.6 PCI-DSS

지불 카드 산업 데이터 보안 표준의 요구 사항 10은 감사 로깅에 전적으로 초점을 맞춥니다.

요구 사항설명WIA-SEC-017 구현
10.1모든 액세스를 개별 사용자와 연결actor 필드 필수
10.2자동화된 감사 추적 구현SDK 자동 로깅
10.3각 이벤트에 특정 데이터 요소 기록표준화된 스키마
10.4모든 중요 시스템 시계 동기화NTP 요구 사항
10.5감사 추적 변경 방지해시 체인, WORM
10.6매일 로그 검토자동화된 알림
10.7최소 1년간 보존계층화된 저장소

3.7 규정 준수 매핑

WIA-SEC-017은 자동 규정 준수 매핑을 제공합니다:

규정 준수 대시보드

  • 각 프레임워크별 준수 상태 실시간 표시
  • 누락된 통제 항목 자동 식별
  • 감사 증거 자동 수집
  • 규정 준수 보고서 생성

3.8 장 요약

핵심 내용

  • SOC 2: 서비스 조직을 위한 5가지 신뢰 서비스 기준
  • ISO 27001: Annex A 로깅 통제 (A.12.4.1-4)
  • GDPR: 72시간 침해 통지, 처리 활동 기록
  • HIPAA: ePHI 액세스 로깅, 6년 보존
  • PCI-DSS: 요구 사항 10, 1년 보존
  • WIA-SEC-017은 모든 프레임워크에 대한 매핑 제공
