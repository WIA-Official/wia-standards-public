제7장: Phase 4 - 통합

弘益人間 · 널리 인간을 이롭게 하라

개요

Phase 4 Integration은 WIA-ART-011을 실제 세계와 연결하는 마지막 퍼즐 조각입니다. 데이터(Phase 1), API(Phase 2), 프로토콜(Phase 3)을 정의했다면, 이제 이들을 기존 클라우드 인프라, 인기 VR 플랫폼, NFT 마켓플레이스, 그리고 다른 WIA 표준들과 통합합니다.

Phase 4의 목표는 "한 번 만들고, 어디서나 배포"입니다. 박물관이 WIA 형식으로 전시를 제작하면, 클릭 한 번으로 Spatial.io, Mozilla Hubs, Decentraland, 자체 웹사이트, NFT 마켓플레이스에 동시에 배포할 수 있어야 합니다.

생성된 Decentraland 코드 예시:

// game.ts
import { createGallery } from '@wia/dcl-adapter';

// WIA manifest 로드
const manifest = await loadWIAManifest('exhibition.wex');

// 갤러리 생성
const gallery = createGallery(manifest, {
  parcel: new Vector3(-52, 0, 48),
  autoLayout: true
});

// 작품 클릭 이벤트
gallery.artworks.forEach(artwork => {
  artwork.onClick = () => {
    showArtworkDialog(artwork);
    
    // NFT인 경우 OpenSea 링크
    if (artwork.nft) {
      addButton('View on OpenSea', () => {
        openExternalURL(artwork.nft.openseaUrl);
      });
    }
  };
});

WIA 생태계 통합

WIA-SOCIAL 연동

WIA-SOCIAL 표준과 통합하여 소셜 기능 추가: 

// WIA-SOCIAL SDK 통합
import { WIASocial } from '@wia/social';

const social = new WIASocial({
  apiKey: 'YOUR_API_KEY',
  exhibition: gallery.exhibition
});

// 친구 초대
social.inviteFriends(['user-456', 'user-789'], {
  message: '이 인상주의 전시 함께 보실래요?',
  meetingPoint: 'Gallery Entrance'
});

// 공개 피드에 게시
social.post({
  type: 'artwork.favorite',
  artworkId: 'artwork-001',
  comment: '모네의 이 작품은 정말 경이롭습니다!',
  screenshot: await gallery.captureScreenshot()
});

// 댓글 스레드
social.on('comment', (comment) => {
  gallery.showNotification(`${comment.author}: ${comment.text}`);
});

// 좋아요/반응
social.on('reaction', (reaction) => {
  gallery.showEmojiEffect(reaction.emoji, reaction.position);
});

WIA-INTENT 연동

WIA-INTENT로 AI 기반 개인화 큐레이션: 

// WIA-INTENT AI 큐레이션
import { WIAIntent } from '@wia/intent';

const intent = new WIAIntent({
  apiKey: 'YOUR_API_KEY'
});

// 사용자 선호도 학습
intent.on('user.view.artwork', (artwork, duration) => {
  intent.recordInteraction({
    type: 'view',
    artworkId: artwork.id,
    duration: duration,
    tags: artwork.tags
  });
});

// 개인화 추천
const recommendations = await intent.recommend({
  userId: 'user-123',
  context: 'current-exhibition',
  limit: 5
});

gallery.highlightArtworks(recommendations, {
  color: '#FFD700',
  label: '당신을 위한 추천'
});

// 자연어 검색
const query = "19세기 프랑스 풍경화";
const results = await intent.search(query, {
  semantic: true,  // 의미 기반 검색
  filters: {
    exhibition: gallery.exhibition.id
  }
});

사례 연구: 국립중앙박물관의 다중 플랫폼 배포

2024년, 한국 국립중앙박물관은 "조선 왕조 500년" 특별 전시를 WIA-ART-011로 제작하고, 5개 플랫폼에 동시 배포했습니다. 이는 한국 최초의 대규모 다중 플랫폼 가상 전시였습니다.

제작 과정:

  • 150점의 유물을 Photogrammetry로 3D 스캔 (3개월)
  • GLTF 모델 최적화 및 PBR 텍스처 작업 (2개월)
  • 한국어, 영어, 중국어, 일본어 메타데이터 작성 (1개월)
  • .wex 패키지 생성 및 검증 (2주)
  • 총 제작 기간: 6.5개월, 비용: 약 3억 원

배포 플랫폼:

  1. 자체 웹사이트 (AWS): CloudFront CDN, 월 10만 방문자
  2. Spatial.io: VR 헤드셋 사용자, 월 2만 방문자
  3. Mozilla Hubs: 교육 기관용, 월 5천 방문자
  4. Decentraland: NFT 컬렉터 대상, 월 1천 방문자
  5. 모바일 앱 (Flutter): iOS/Android, 월 8만 다운로드

결과:

  • 총 방문자: 6개월간 120만 명 (물리적 전시 50만 명의 2.4배)
  • 국제 사용자: 65% (한국 35%, 미국 20%, 중국 15%, 기타 30%)
  • 평균 체류 시간: 18분 (물리적 전시 12분보다 50% 증가)
  • 배포 시간: 5개 플랫폼에 총 3일 (WIA 덕분에 자동화)
  • 유지보수 비용: 월 $250 (5개 플랫폼 통합, 물리적 전시 대비 95% 절감)
  • NFT 수익: 한정판 디지털 복제본 판매로 1.5억 원 추가 수익

교훈:

  • 다중 플랫폼 배포가 접근성과 도달 범위를 극대화
  • WIA 표준화로 플랫폼별 재작업 없이 일관된 경험 제공
  • NFT 통합이 새로운 수익 모델 창출
  • 국제화가 글로벌 관람객 유치에 결정적

핵심 요점

  • 클라우드 배포: AWS, Google Cloud, Azure에 원클릭 배포하며, 월 $28-35로 10,000명 방문자를 처리할 수 있습니다.
  • VR 플랫폼: Spatial.io, Mozilla Hubs, Decentraland에 자동 변환 및 배포하며, 각 플랫폼의 고유 기능을 활용합니다.
  • NFT 통합: OpenSea에 자동 등록하고, WIA 메타데이터를 NFT 표준으로 매핑하여 새로운 수익 모델을 창출합니다.
  • WIA 생태계: WIA-SOCIAL로 소셜 기능, WIA-INTENT로 AI 큐레이션을 추가하여 사용자 경험을 향상시킵니다.
  • 다중 플랫폼 전략: 국립중앙박물관 사례에서 5개 플랫폼 배포로 물리적 전시 대비 2.4배 방문자, 65% 국제 사용자를 달성했습니다.
  • 컨테이너화: Docker와 Kubernetes로 확장 가능한 배포를 지원하며, 자동 스케일링과 고가용성을 보장합니다.
  • 비용 효율성: 다중 플랫폼 통합 유지보수 비용이 월 $250로 물리적 전시 대비 95% 절감됩니다.
  • 자동화: 배포 시간이 5개 플랫폼에 총 3일로 단축되어, 수작업 대비 90% 시간 절약을 실현합니다.

복습 질문

  1. AWS, Google Cloud, Azure의 WIA 배포 아키텍처를 비교하십시오. 각 클라우드의 스토리지, CDN, 데이터베이스, WebSocket, 인증 서비스와 월 비용 ($28-35)을 설명하십시오.
  2. Spatial.io, Mozilla Hubs, Decentraland의 WIA 어댑터가 어떻게 작동하는지 설명하십시오. 각 플랫폼의 배포 방식, 사용자 기반, 비용 구조를 비교하고, GLTF 변환 과정을 논의하십시오.
  3. WIA 전시를 NFT로 민팅하고 OpenSea에 등록하는 과정을 설명하십시오. WIA metadata.json이 OpenSea 메타데이터로 어떻게 매핑되며, IPFS는 어떤 역할을 하는지 코드 예시와 함께 제시하십시오.
  4. WIA-SOCIAL과 WIA-INTENT 통합을 설명하십시오. 소셜 기능(친구 초대, 피드 게시, 댓글)과 AI 큐레이션(선호도 학습, 개인화 추천, 자연어 검색)이 사용자 경험을 어떻게 향상시키는지 논의하십시오.
  5. 국립중앙박물관 사례 연구를 분석하십시오. 5개 플랫폼 배포 전략, 총 방문자 120만 명, 국제 사용자 65%, NFT 수익 1.5억 원 등의 성과를 설명하고, 다중 플랫폼 접근의 장점을 논의하십시오.
  6. Docker 컨테이너화와 Kubernetes 배포의 이점을 설명하십시오. WIA 전시를 컨테이너로 패키징하는 과정, HorizontalPodAutoscaler를 활용한 자동 스케일링, 고가용성 보장 방법을 제시하십시오.
