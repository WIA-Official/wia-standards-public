테스트 및 검사

읽기 시간: 18분 최종 업데이트: 2026년 1월 WIA-SEMI-PKG-001

6.1 첨단 패키징 테스트의 중요성

첨단 패키징에서 테스트와 검사는 품질 보증의 핵심입니다. 2.5D/3D 패키지, 칩렛 아키텍처에서는 한번 조립되면 개별 다이를 교체할 수 없으므로, 조립 전 다이의 품질 보증(KGD: Known Good Die)이 필수적입니다. 불량 다이 하나가 전체 고가의 패키지를 무용지물로 만들 수 있습니다.

테스트 비용은 총 패키징 비용의 30-40%에 달할 수 있으며, 이 비용을 줄이면서 품질을 유지하는 것이 산업의 주요 과제입니다. 테스트 커버리지와 비용/시간 간의 균형이 중요합니다.

테스트 단계

단계 시점 목적 방법
웨이퍼 소트 웨이퍼 상태 기본 기능 검증 프로브 테스트
KGD 테스트 다이싱 후 양품 다이 선별 상세 프로브 테스트
중간 테스트 조립 중 조립 품질 확인 인터커넥트 테스트
최종 테스트 패키지 완성 후 전체 기능 검증 ATE 테스트
시스템 테스트 보드 실장 후 실사용 환경 검증 기능/스트레스 테스트

6.2 KGD (Known Good Die) 테스트

KGD 테스트는 첨단 패키징의 경제성을 결정하는 핵심 요소입니다. 멀티다이 패키지에서 단 하나의 불량 다이가 전체 패키지를 불량으로 만들기 때문에, 조립 전 다이의 품질 보증이 필수적입니다.

KGD 테스트 요구사항

  • 높은 테스트 커버리지: 최종 테스트와 동등한 수준의 결함 검출
  • 파라메트릭 테스트: DC 특성, AC 타이밍, 전류 소모 측정
  • 고온/저온 테스트: 온도 범위 전체에서 기능 검증
  • 번인(Burn-In): 조기 고장 유발을 위한 가속 스트레스
  • BIST: 내장 자가 테스트 기능 활용

KGD 테스트 방법

방법 장점 단점
웨이퍼 레벨 프로브 일괄 테스트 가능, 효율적 미세 피치 프로빙 어려움
임시 본딩 테스트 풀 테스트 가능 추가 공정, 비용 증가
BIST 기반 외부 장비 최소화 테스트 로직 오버헤드
인터포저 테스트 실제 환경에 근접 복잡한 테스트 구조

HBM의 KGD 테스트

HBM 스택은 여러 다이가 적층되므로 각 다이의 KGD 테스트가 특히 중요합니다. 베이스 다이 및 각 코어 다이를 개별 테스트한 후 적층합니다. HBM3E의 경우 12개 다이를 적층하므로, 개별 다이 수율이 99%여도 스택 수율은 0.99^12 ≈ 88.6%로 떨어집니다. 따라서 각 다이의 수율을 최대화하는 것이 경제적으로 매우 중요합니다.

6.3 프로브 테스트 기술

프로브 테스트는 웨이퍼 또는 다이 상태에서 칩에 직접 접촉하여 전기적 특성을 측정하는 기술입니다. 첨단 패키징의 미세 피치 범프에 대응하여 프로브 기술도 계속 발전하고 있습니다.

프로브 카드 유형

유형 피치 범위 특징 응용
캔틸레버 프로브 > 100μm 단순 구조, 저비용 레거시 제품
버티컬 프로브 50-100μm 좁은 피치 대응, 균일한 접촉 로직, 메모리
MEMS 프로브 30-50μm 초미세 피치, 정밀 제어 첨단 로직, HBM
멤브레인 프로브 40-80μm 풀 에어리어 프로빙 고밀도 메모리

프로브 테스트 과제

  • 미세 피치: 40μm 이하 피치에서 안정적 접촉 확보
  • 패드 손상: 프로브 마크가 범핑에 미치는 영향 최소화
  • 평탄도: 대면적 웨이퍼에서 모든 패드 동시 접촉
  • 온도 제어: 프로브 중 열 팽창/수축 관리
  • 처리량: 테스트 시간 vs 커버리지 균형

6.4 광학 검사 (AOI)

자동 광학 검사(AOI: Automated Optical Inspection)는 패키지의 외관 결함을 비접촉으로 검출하는 기술입니다. 와이어 본딩, 범프, RDL 패턴 등의 결함을 고속으로 검사합니다.

AOI 검사 항목

  • 와이어 본딩: 와이어 위치, 루프 높이, 볼 크기, 단선/단락
  • 범프: 범프 위치, 크기, 형상, 결함(bridging, missing)
  • RDL: 패턴 결함, 단선, 단락, 폭 편차
  • 다이: 다이 위치, 틸트, 크랙, 치핑
  • 패키지: 보이드, 이물질, 표면 결함

3D AOI

첨단 패키징에서는 2D 이미지만으로는 검사가 어려운 경우가 많습니다. 3D AOI는 높이 정보를 추가하여 범프 코플래너리티, 와이어 루프 형상 등을 정밀하게 측정합니다.

  • 레이저 삼각측량: 높은 해상도, 빠른 스캔
  • 백색광 간섭계: 나노미터 수준 정밀도
  • 공초점 현미경: 미세 구조 분석
  • 위상 편이: 반사 표면 측정

6.5 X-ray 검사

X-ray 검사는 패키지 내부의 숨겨진 결함을 비파괴적으로 검출하는 기술입니다. 플립칩 범프, 솔더 조인트, TSV, 언더필 보이드 등 외부에서 보이지 않는 결함을 검사할 수 있습니다.

X-ray 검사 유형

유형 특징 응용
2D X-ray 빠른 스캔, 전체 검사 와이어 본딩, 범프 정렬
경사 X-ray 다중 각도 이미지 솔더 조인트 검사
CT (Computed Tomography) 3D 재구성, 단면 이미지 TSV, 3D 스택 분석
라미노그래피 선택적 깊이 이미지 다층 구조 검사

X-ray 검사 대상

  • 솔더 조인트: 보이드, 브릿지, 결손, 균열
  • TSV: 충전 상태, 보이드, 얼라인먼트
  • 언더필: 보이드, 미충전, 박리
  • 와이어: 새깅, 넥킹, 스윕
  • BGA: 볼 크기, 위치, 결함

인라인 vs 오프라인 X-ray

인라인 X-ray는 생산 라인에 통합되어 전수 검사에 사용됩니다. 처리량이 중요하므로 2D 또는 간단한 경사 X-ray가 주로 사용됩니다. 오프라인 X-ray는 불량 분석이나 상세 검사에 사용되며, CT 같은 고해상도 3D 분석이 가능합니다.

6.6 음향 검사 (SAM)

스캐닝 음향 현미경(SAM: Scanning Acoustic Microscopy)은 초음파를 이용하여 내부 결함을 검출하는 기술입니다. 특히 박리(delamination), 균열, 보이드 검출에 효과적입니다.

SAM 원리

고주파 초음파(수십 MHz ~ 수백 MHz)를 샘플에 조사하고 반사파를 분석합니다. 물질 계면에서 음향 임피던스 차이에 의해 반사가 발생하며, 박리나 보이드와 같은 결함은 강한 반사 신호를 생성합니다.

SAM 검사 모드

  • 반사 모드 (C-SAM): 특정 깊이의 계면 이미지, 박리 검출
  • 투과 모드 (T-SAM): 내부 결함 전체 검출, 보이드 확인
  • 시간 분해 모드: 깊이별 분석, 다층 구조 검사

SAM 응용

검사 대상 결함 유형 주파수
다이 어태치 박리, 보이드 15-50 MHz
언더필 미충전, 보이드 50-100 MHz
몰드 컴파운드 박리, 균열 15-50 MHz
범프/조인트 균열, 보이드 100-200 MHz

6.7 전기적 테스트 (ATE)

자동 테스트 장비(ATE: Automatic Test Equipment)를 이용한 전기적 테스트는 패키지의 기능과 성능을 검증하는 최종 단계입니다. DC, AC, 고속 디지털, RF 등 다양한 테스트가 수행됩니다.

테스트 유형

테스트 내용 측정 항목
DC 파라메트릭 정적 전기 특성 연속성, 누설전류, Voh/Vol
AC 타이밍 동적 성능 셋업/홀드, 전파지연, Fmax
기능 테스트 논리 기능 입출력 패턴 검증
IDDQ 정지 전류 내부 결함 검출
스캔 테스트 내부 노드 DFT 기반 결함 검출
메모리 테스트 메모리 셀 다양한 패턴/알고리즘

고속 테스트 과제

첨단 패키지의 고속 인터페이스(PCIe Gen5, DDR5, UCIe 등) 테스트에는 특수한 과제가 있습니다.

  • 신호 무결성: 테스트 보드/소켓의 SI 최적화
  • 고주파 측정: 수십 GHz 신호 측정 능력
  • 로드보드 설계: 손실 최소화, 임피던스 정합
  • 테스트 시간: 복잡한 프로토콜 테스트의 시간 최적화

6.8 신뢰성 테스트

신뢰성 테스트는 패키지가 사용 수명 동안 정상 작동함을 검증합니다. 가속 스트레스 테스트를 통해 실제 사용 환경보다 빠르게 고장을 유발하고, 이를 기반으로 수명을 예측합니다.

주요 신뢰성 테스트

테스트 조건 검출 대상 표준
온도 사이클 (TC) -55/+125°C, 1000cyc 솔더 피로, 박리 JESD22-A104
고온고습 (HAST) 130°C, 85%RH, 96h 부식, 수분 침투 JESD22-A110
고온 보관 (HTS) 150°C, 1000h 금속간 성장, 확산 JESD22-A103
전력 사이클 (PC) 온/오프 반복 접합부 피로 JESD22-A122
전기이동 (EM) 고온, 고전류 배선/범프 열화 JESD61

자동차 신뢰성 등급

자동차 반도체는 AEC-Q100 규격에 따라 더욱 엄격한 신뢰성 요구사항을 충족해야 합니다.

  • Grade 0: -40°C ~ +150°C (엔진/변속기)
  • Grade 1: -40°C ~ +125°C (파워트레인)
  • Grade 2: -40°C ~ +105°C (일반 자동차)
  • Grade 3: -40°C ~ +85°C (인포테인먼트)

첨단 패키징 신뢰성의 과제

2.5D/3D 패키징에서는 새로운 신뢰성 과제가 있습니다. TSV 스트레스, 하이브리드 본딩 계면, 칩렛 간 열팽창 차이 등이 새로운 고장 모드를 야기할 수 있습니다. 이러한 새로운 구조에 대한 신뢰성 모델과 테스트 방법론이 계속 개발되고 있습니다.

요약

첨단 패키징의 테스트와 검사는 품질과 경제성을 결정하는 핵심 요소입니다. KGD 테스트, 광학/X-ray/SAM 검사, ATE 테스트, 신뢰성 테스트가 종합적으로 수행되어 패키지 품질을 보증합니다.

핵심 요점

  • KGD: 멀티다이 패키지 경제성의 핵심, 조립 전 다이 품질 보증
  • 프로브 테스트: 미세 피치 대응 프로브 카드 기술 발전
  • AOI: 외관 결함 고속 검출, 3D AOI로 높이 측정
  • X-ray: 내부 결함 비파괴 검사, CT로 3D 분석
  • SAM: 박리, 보이드 검출에 효과적
  • 신뢰성 테스트: TC, HAST, HTS 등 가속 수명 테스트
