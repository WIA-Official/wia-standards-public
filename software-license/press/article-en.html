WIA-COMP-016: Automated License Compliance

WIA-COMP-016 provides SPDX standard-based automated license management solutions:

License Categories

Category Licenses Features Restrictions
Permissive MIT, Apache-2.0, BSD Minimal restrictions, commercial use allowed Almost none
Weak Copyleft LGPL, MPL Can be proprietary when used as library Must disclose modifications
Strong Copyleft GPL-2.0, GPL-3.0, AGPL Derivative works must use same license Cannot use in proprietary software
Public Domain CC0, Unlicense Waive all rights None

License Compatibility Rules

Combination Compatibility Description
MIT → Apache-2.0 ✓ Compatible Permissive licenses compatible
MIT → GPL-3.0 ✓ Compatible Permissive compatible with Copyleft
Apache-2.0 → GPL-2.0 ✗ Incompatible Patent clause conflict
GPL-2.0 → GPL-3.0 ✗ Incompatible Explicit permission needed
LGPL → GPL ✓ Compatible Weak → Strong possible

About WIA (World Certification Industry Association)

WIA is an international standards organization that designs and distributes future industry standards. We develop practical, practitioner-oriented standards in cutting-edge technology fields including artificial intelligence, blockchain, and software engineering.

All WIA standards are based on high-performance core engines written in Rust, and provide SDKs for various languages including TypeScript, Python, and Go. WIA releases all standards under the MIT License for free, following the '弘益人間 (Benefit All Humanity)' spirit.
