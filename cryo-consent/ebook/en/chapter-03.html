3.1 PRESERVATION CONSENT TYPES
--------------------------------------------------------------------------------

[ID] CONSENT TYPE OPTIONS:

    FULL CONSENT:
    - All preservation options enabled
    - Standard revival conditions
    - Research participation allowed
    - Most flexible for future decisions

    CONDITIONAL CONSENT:
    - Preservation only under specific conditions
    - Custom revival requirements
    - May limit options available

    LIMITED CONSENT:
    - Restricted revival scenarios
    - Specific exclusions defined
    - Narrower scope of authority

    RESEARCH ONLY:
    - No intention of revival
    - Scientific use authorized
    - Donation to cryonics research

[ID] BODY DISPOSITION OPTIONS:

    +------------------------------------------+
    | Option         | Description             |
    +------------------------------------------+
    | WHOLE_BODY     | Complete body preserved |
    | NEURO_ONLY     | Head/brain only         |
    +------------------------------------------+

    [i] Neuro-only option assumes future technology can
        regenerate or provide a new body.

3.2 ACKNOWLEDGED RISKS
--------------------------------------------------------------------------------

[!] Subject must acknowledge understanding of all risks.

[ID] RISK CATEGORIES:

    TECHNICAL RISKS:
    - Equipment failure during preservation
    - Power outages affecting storage
    - Facility damage (natural disaster, etc.)
    - Transportation incidents

    BIOLOGICAL RISKS:
    - Ice crystal formation (cellular damage)
    - Incomplete vitrification
    - Tissue degradation over time
    - Unknown long-term effects

    ORGANIZATIONAL RISKS:
    - Facility bankruptcy
    - Company dissolution
    - Management changes
    - Service discontinuation

    LEGAL RISKS:
    - Law changes prohibiting cryopreservation
    - Jurisdictional conflicts
    - Family legal challenges
    - Estate disputes

    FINANCIAL RISKS:
    - Funding depletion
    - Economic collapse
    - Currency devaluation
    - Trust mismanagement

    UNKNOWN RISKS:
    - Future technologies may not develop
    - Revival may never be possible
    - Unforeseen complications

[ID] RISK ACKNOWLEDGMENT:

    AcknowledgedRisk {
      riskId:       "RISK-TECH-001"
      category:     "TECHNICAL"
      description:  "Equipment failure may cause irreversible damage"
      acknowledged: true
      acknowledgedAt: "2025-03-01T10:00:00Z"
    }


--------------------------------------------------------------------------------
3.3 RESEARCH CONSENT
--------------------------------------------------------------------------------

[ID] RESEARCH CONSENT OPTIONS:

    ResearchConsent {
      authorized: true
      scope:      "PRESERVATION_RESEARCH"

      allowedResearch: {
        tissuesampling:       true   // Small tissue samples
        imagingStudies:       true   // Non-invasive imaging
        geneticResearch:      false  // DNA analysis
        procedureObservation: true   // Learning from case
      }

      dataSharing: {
        anonymizedData:   true   // De-identified data
        identifiableData: false  // Keep identity private
        commercialUse:    false  // No commercial applications
      }

      exclusions: [
        "No genetic modification research",
        "No cloning attempts"
      ]
    }

[ID] RESEARCH SCOPE LEVELS:

    NONE                 -> No research participation
    PRESERVATION_RESEARCH -> Improve preservation techniques
    REVIVAL_RESEARCH     -> Develop revival methods
    GENERAL_MEDICAL      -> Broader medical research
    UNRESTRICTED         -> Any legitimate research

3.4 ALTERNATIVE DIRECTIVES
--------------------------------------------------------------------------------

[ID] IF REVIVAL BECOMES IMPOSSIBLE:

    ImpossibleRevivalDirective {
      waitDuration:  "P100Y"  // Wait 100 years
      afterWaiting:  "RESEARCH_DONATION"

      researchDonation: {
        authorized:   true
        institutions: ["WIA Research Institute", "MIT Cryonics Lab"]
        restrictions: ["No commercial use", "Credit subject"]
      }

      termination: null  // Not applicable if donating
    }

[ID] IF FACILITY FAILS:

    FacilityFailureDirective {
      transferPriority: [
        "Alcor Life Extension Foundation",
        "Cryonics Institute",
        "WIA Partner Network"
      ]

      ifNoTransferPossible: "TERMINATE"

      insuranceClaim: {
        authorized:    true
        beneficiaries: ["Estate of John Smith"]
      }
    }

[ID] IF FUNDING FAILS:

    FundingFailureDirective {
      gracePeriod: "P5Y"  // 5 years to restore funding

      fundingRecoveryAttempts: {
        familyAppeal:         true
        crowdfunding:         true
        charitableFunds:      true
        governmentAssistance: false
      }

      ifFundingNotRestored: "REDUCED_SERVICE"
    }

[ID] IF LAW CHANGES:

    LegalChangeDirective {
      ifPreservationBanned: "TRANSFER_JURISDICTION"
      preferredJurisdictions: ["Switzerland", "Costa Rica"]

      ifRevivalBanned: "WAIT"
      waitDuration: "P50Y"

      legalDefenseFund: {
        authorized:    true
        maximumAmount: { amount: 100000, currency: "USD" }
      }
    }
