5.1 SPOUSE CONSENT
--------------------------------------------------------------------------------

Spouse involvement is recommended for married subjects.

[ID] SPOUSE AGREEMENT RECORD:

    SpouseAgreement {
      spouseId:       "SPOUSE-2025-001"
      spouseName:     "Jane Smith"
      marriageStatus: "MARRIED"

      consentGiven:   true
      consentScope: {
        preservationApproved:       true
        financialArrangementApproved: true
        revivalConditionsApproved:  true
        objections:                 []
        negotiatedTerms:            []
      }
    }

[ID] RIGHTS AFTER PRESERVATION:

    rightsAfterPreservation: {
      remarriageAllowed:    true   // Spouse may remarry
      inheritanceRights:    "FULL" // Full inheritance rights
      decisionMakingRole:   "PRIMARY" // Primary guardian
      revivalVoteWeight:    2      // Weight in revival decisions
    }

[ID] JOINT PRESERVATION:

    jointPreservation: {
      requested:  true
      conditions: [
        "Same facility",
        "Revival together if possible",
        "Notify other if one revived first"
      ]
    }

[!] SPOUSE OBJECTIONS:
    If spouse objects, mediation is recommended before proceeding.
    Document all objections and resolutions.

    ChildAgreement {

      dateOfBirth: "2000-01-15"
      isMinor:     false

      notification: {
        notified:         true
        notificationDate: "2025-02-15"
        method:           "In-person family meeting"
      }

      acknowledgment: {
        acknowledged:     true
        acknowledgedDate: "2025-02-20"
        concerns:         ["Financial impact on family"]
      }

      guardianRole: {
        designated:       true
        responsibilities: ["CONSENT_DECISIONS", "FAMILY_COMMUNICATION"]
        voteWeight:       1
      }
    }

[ID] MINOR CHILDREN:

    For minor children:
    - Notification appropriate to age
    - Legal guardian signature on behalf
    - Re-consent when child reaches majority
    - Consider child's future role as guardian

[ID] NOTIFICATION REQUIREMENTS:

    +----------------------------------------------------------+
    | Relationship | Notification | Guardian Role | Signature  |
    +----------------------------------------------------------+
    | Spouse       | Required     | Recommended   | Recommended|
    | Adult Child  | Required     | Optional      | Optional   |
    | Minor Child  | Age-Approp.  | Future        | Guardian   |
    | Parent       | Recommended  | Optional      | Optional   |
    | Sibling      | Optional     | Optional      | Optional   |
    +----------------------------------------------------------+


--------------------------------------------------------------------------------
5.3 GUARDIAN DESIGNATIONS
--------------------------------------------------------------------------------

[ID] GUARDIAN TYPES:

    PERSON       -> Individual trusted by subject
    ORGANIZATION -> Legal entity (law firm, trust company)
    SMART_CONTRACT -> Autonomous on-chain guardian (advanced)

[ID] GUARDIAN RECORD:

    GuardianDesignation {
      guardianId:    "GUARD-2025-001"
      guardianType:  "PERSON"

      personalInfo: {
        name:         "Jane Smith"
        relationship: "Spouse"
        contactInfo:  "[ENCRYPTED]"
      }

      responsibilities: [
        "CONSENT_DECISIONS",
        "FINANCIAL_OVERSIGHT",
        "FACILITY_LIAISON",
        "REVIVAL_DECISIONS",
        "FAMILY_COMMUNICATION"
      ]

      voteWeight:   2

      succession: {
        primarySuccessor:   "John Smith Jr."
        secondarySuccessor: "Sarah Smith"
        successionTrigger:  "Death, incapacity, or resignation"
      }

      acceptanceRecord: {
        accepted:     true
        acceptedDate: "2025-02-25"
        signature:    "[DIGITAL SIGNATURE]"
      }
    }

[ID] RESPONSIBILITY TYPES:

    CONSENT_DECISIONS    -> Amend or withdraw consent
    FINANCIAL_OVERSIGHT  -> Monitor preservation funding
    FACILITY_LIAISON     -> Communicate with facility
    REVIVAL_DECISIONS    -> Approve/deny revival
    FAMILY_COMMUNICATION -> Keep family informed
    LEGAL_REPRESENTATION -> Handle legal matters

[ID] GUARDIAN NETWORK CONFIGURATION:

    Recommended: 3-5 guardians with mixed types

    Example:
    +------------------------------------------+
    | Guardian        | Type   | Weight | Role |
    +------------------------------------------+
    | Spouse          | PERSON | 2      | All  |
    | Adult Child     | PERSON | 1      | All  |
    | Law Firm        | ORG    | 1      | Legal|
    | Trust Company   | ORG    | 1      | Fin  |
    +------------------------------------------+
    Total Weight: 5
    Threshold: 3 (majority)

5.4 CONFLICT RESOLUTION
--------------------------------------------------------------------------------

[ID] CONFLICT RESOLUTION PLAN:

    ConflictResolutionPlan {
      primaryMethod: "MEDIATION"

      mediator: "WIA Mediation Services"
      arbitrationProvider: "American Arbitration Association"
      governingLaw: "State of California, USA"
      venue: "San Francisco, CA"

      escalationPath: [
        { step: 1, method: "Family Discussion",  timeout: "30 days" },
        { step: 2, method: "Formal Mediation",   timeout: "60 days" },
        { step: 3, method: "Binding Arbitration", timeout: "90 days" },
        { step: 4, method: "Court Proceedings",  timeout: "None" }
      ]

      deadlockResolution: "Arbitrator decides by majority of evidence"
    }

[ID] COMMON CONFLICTS:

    1. Revival timing disagreement
       Resolution: Medical board opinion, then guardian vote

    2. Funding allocation disputes
       Resolution: Trust terms govern, then arbitration

    3. Facility transfer disagreement
       Resolution: Mediation, consider subject's stated preferences

    4. Amendment approval deadlock
       Resolution: Wait 6 months, re-vote, then arbitration

[i] PREVENTION:
    Clear communication and regular family meetings
    reduce conflict likelihood significantly.
