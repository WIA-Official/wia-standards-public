7.1 AMENDMENT PROCESS
--------------------------------------------------------------------------------

Consent documents can be amended after activation, subject to approval.

[ID] AMENDMENT RECORD:

    Amendment {
      amendmentId:      "AMD-CONSENT-2025-001-001"
      amendmentNumber:  1

      effectiveDate:    "2025-06-15"
      createdAt:        "2025-06-01T10:00:00Z"

      amendmentType:    "REVIVAL_CONDITIONS"
      description:      "Extend minimum preservation duration to 75 years"

      changes: [
        {
          section:       "revival_conditions"
          field:         "minimum_duration"
          previousValue: "P50Y"
          newValue:      "P75Y"
          justification: "Technology projections suggest longer wait optimal"
        }
      ]

      initiator:        "John Smith Jr."  // Guardian
      reason:           "Updated medical technology forecasts"

      approvals: []
      requiredApprovals: 3

      status:           "PENDING"
      previousVersion:  "1.0.0"
    }

    +------------------+
    | INITIATE         |

7.2 CRITICAL VS NON-CRITICAL CHANGES
--------------------------------------------------------------------------------

[ID] CRITICAL CHANGES (Require 2/3 majority):

    - Revival conditions modification
    - Withdrawal authorization
    - Guardian designation changes
    - Research consent changes
    - Alternative directive changes
    - Financial arrangement changes

[ID] NON-CRITICAL CHANGES (Simple majority):

    - Contact information updates
    - Minor clarifications
    - Facility preferences
    - Administrative details

[ID] APPROVAL THRESHOLDS:

    +----------------------------------------------------------+
    | Change Type      | Threshold | Waiting Period | Notes    |
    +----------------------------------------------------------+
    | Critical         | 2/3       | 30 days        | Legal rev|
    | Non-Critical     | 1/2       | 7 days         | Standard |
    | Emergency        | 2/3       | None           | Urgent   |
    | Complete Withdraw| Unanimous | 30 days        | All guard|
    +----------------------------------------------------------+

[ID] AMENDMENT APPROVAL:

    AmendmentApproval {
      approverId:   "GUARD-001"
      approverRole: "GUARDIAN"
      decision:     "APPROVE"  // APPROVE | REJECT | ABSTAIN
      timestamp:    "2025-06-05T14:30:00Z"
      notes:        "Agree with updated technology projections"
      signature:    "[DIGITAL SIGNATURE]"
    }

7.3 WITHDRAWAL TYPES
--------------------------------------------------------------------------------

[ID] PRE-PRESERVATION WITHDRAWAL:

    Before preservation begins, subject can freely withdraw.

    PrePreservationWithdrawal {
      withdrawalType: "PRE_PRESERVATION"
      initiator:      "John Smith"  // Subject
      initiatorType:  "SUBJECT"

      requirements: {
        subjectSignature:     true
        guardianApproval:     false  // Not required
        waitingPeriod:        "None"
        notarization:         false
      }

      consequences: {
        documentStatus:       "WITHDRAWN"
        refundAvailable:      true
        recordsDestroyed:     true  // Optional
      }
    }

[ID] DURING-PRESERVATION WITHDRAWAL:

    After preservation, only guardians can initiate withdrawal.

    DuringPreservationWithdrawal {
      withdrawalType: "DURING_PRESERVATION"
      initiator:      "Jane Smith"
      initiatorType:  "GUARDIAN"

      requirements: {
        guardianApproval:     true
        approvalThreshold:    "2/3 majority"
        waitingPeriod:        "30 days"
        legalReview:          true
        notarization:         true
      }

      consequences: {
        documentStatus:       "WITHDRAWN"
        preservationTerminated: true
        remainsDisposition:   "Per Alternative Directives"
      }
    }

[ID] REVIVAL REFUSAL:

    Subject retains right to refuse revival after conditions are met.

    RevivalRefusal {
      withdrawalType: "REVIVAL_REFUSAL"
      initiator:      "John Smith"  // Revived subject
      initiatorType:  "SUBJECT"

      requirements: {
        subjectConsent:       true
        capacityVerification: true  // Subject must be mentally capable
        counselingCompleted:  true
        waitingPeriod:        "48 hours"
      }

      consequences: {
        revivalAborted:       true
        refundToBeneficiaries: true
        finalDisposition:     "Per Subject's choice"
      }
    }

7.4 REFUND POLICIES
--------------------------------------------------------------------------------

[ID] REFUND POLICY STRUCTURE:

    RefundPolicy {
      prePreservationWithdrawal: {
        fullRefund:          false
        refundPercentage:    80
        deductions: [
          "Administrative fee: $2,500",
          "Medical examination costs",
          "Legal document preparation"
        ]
        processingTime:      "30 days"
      }

      duringPreservation: {
        refundAvailable:     true
        conditions: [
          "Prorated based on time preserved",
          "Deduct facility costs incurred",
          "Minimum 12 months of preservation fees"
        ]
        calculation:         "See formula below"
      }

      revivalRefusal: {
        refundAvailable:     true
        beneficiaries: [
          { name: "Jane Smith",     share: 50 },
          { name: "John Smith Jr.", share: 25 },
          { name: "Sarah Smith",    share: 25 }
        ]
      }
    }

[ID] DURING-PRESERVATION REFUND FORMULA:

    Base Amount:          Total prepaid amount
    Less: Time consumed   (Years preserved x Annual rate)
    Less: Administrative  $5,000 flat fee
    Less: Legal costs     Actual costs incurred
    Less: Termination fee 10% of remaining balance
    --------------------------------------------------
    Equals: Refund amount

    Example:
    Total paid:           $200,000
    Years preserved:      10 years
    Annual rate:          $3,000
    Time consumed:        $30,000
    Administrative:       $5,000
    Legal costs:          $2,500
    Remaining before fee: $162,500
    Termination fee:      $16,250
    --------------------------------------------------
    Refund amount:        $146,250

[!] REFUND DELAYS:
    Refunds during preservation may take 90-180 days
    to process due to legal and administrative requirements.
