2.1 SUBJECT IDENTIFICATION
--------------------------------------------------------------------------------

Accurate subject identification is essential for long-term consent validity.

[ID] SUBJECT INFORMATION RECORD:

    SubjectInformation {
      subjectId:      "SUBJECT-2025-001"
      identityRef:    "CRYOID-2025-001"  // Link to CRYO-IDENTITY

      personalInfo: {
        fullLegalName:  "John William Smith"
        dateOfBirth:    "1970-03-15"
        placeOfBirth:   "Seoul, South Korea"
        nationality:    ["South Korea", "USA"]
        currentAddress: "[ENCRYPTED]"
        contactInfo:    "[ENCRYPTED]"
      }
    }

[ID] IDENTITY VERIFICATION:

    Before consent can be finalized:
    1. Government-issued ID verification
    2. Cross-reference with CRYO-IDENTITY record
    3. Biometric capture (photo, fingerprint)
    4. Witness attestation of identity


--------------------------------------------------------------------------------
2.2 MENTAL CAPACITY ASSESSMENT
--------------------------------------------------------------------------------

[!] CRITICAL: Mental capacity assessment is REQUIRED before consent is valid.

[ID] ASSESSMENT REQUIREMENTS:

    The assessment must be conducted by a qualified professional:
    - Licensed psychiatrist or psychologist
    - Medical doctor with relevant training
    - Court-appointed assessor (if required)

[ID] UNDERSTANDING VERIFICATION:

    Subject must demonstrate understanding of:

    +---------------------------------------------------------------+
    | Area                        | Required | Subject Confirms     |
    +---------------------------------------------------------------+
    | Preservation process        | YES      | [  ]                 |
    | Known risks and limitations | YES      | [  ]                 |
    | No guarantee of revival     | YES      | [  ]                 |
    | Financial implications      | YES      | [  ]                 |
    | Impact on family            | YES      | [  ]                 |
    +---------------------------------------------------------------+

[ID] DECISION-MAKING CAPACITY:

    DecisionCapacity {
      voluntary:               true   // Decision freely made
      noCoercion:              true   // No pressure from others
      understoodAlternatives:  true   // Aware of other options
    }

[ID] CAPACITY DETERMINATION:

    CAPABLE       -> Subject has full mental capacity
    QUESTIONABLE  -> Additional review recommended
    INCAPABLE     -> Legal representative required


--------------------------------------------------------------------------------
2.3 LEGAL CAPACITY
--------------------------------------------------------------------------------

[ID] LEGAL CAPACITY ASSESSMENT:

    LegalCapacity {
      assessmentDate:  "2025-03-01"
      assessedBy:      "Dr. Park Min-soo, Psychiatrist"
      capacity:        "FULL"

      limitations:     []  // Any restrictions on decision-making
      courtOrders:     []  // Relevant court orders
    }

[ID] SPECIAL SITUATIONS:

    MINORS:
    - Generally cannot consent to cryopreservation
    - Parent/guardian consent with court approval
    - Subject must consent upon reaching majority

    DIMINISHED CAPACITY:
    - Legal representative must co-sign
    - Court approval may be required
    - Additional safeguards apply

    EMERGENCY SITUATIONS:
    - Pre-authorized emergency consent
    - Guardian decision-making
    - Medical power of attorney
