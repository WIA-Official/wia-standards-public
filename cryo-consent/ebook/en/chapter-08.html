8.1 CONDITION EVALUATION
--------------------------------------------------------------------------------

Periodic evaluation determines if revival conditions have been met.

[ID] EVALUATION SCHEDULE:

    Automatic Evaluation:
    - Annually: Basic status check
    - Major events: Medical breakthroughs, technology milestones
    - Guardian request: Any guardian can request evaluation

[ID] EVALUATION PROCESS:

    | - Social factors |

    +------------------+
    | EVALUATE EACH    |
    | CONDITION        |
    +--------+---------+
             |
             v
    +------------------+

    | MET / NOT_MET /  |
    | PENDING          |
    +--------+---------+
             |

    | APPLY LOGIC      |
    | ALL / ANY /      |

    +------------------+

8.2 REVIVAL DECISION
--------------------------------------------------------------------------------

[ID] REVIVAL DECISION RECORD:

    RevivalDecision {
      documentId:   "CONSENT-2025-001"
      decision:     "APPROVE"

      conditionsMet: [
        "COND-MED-001",   // Cancer curable
        "COND-TECH-001",  // Revival technology
        "COND-TIME-001"   // 50 years elapsed
      ]

      conditionsNotMet:   []
      conditionsPending:  []

      overallScore:   1.0
      recommendation: "All conditions met. Revival approved."

      evaluatedAt:    "2075-06-15T10:30:00Z"
    }

[ID] DECISION OUTCOMES:

    APPROVE:
    - All required conditions met
    - Proceed to revival preparation
    - Notify all guardians and family
    - Begin facility preparations

    DENY:
    - Required conditions not met
    - Continue preservation
    - Schedule next evaluation
    - Document reasons

    PENDING:
    - Some conditions evaluating
    - Awaiting additional evidence
    - Re-evaluate in specified timeframe

[ID] APPROVAL WORKFLOW:

8.4 REVIVAL REFUSAL RIGHTS
--------------------------------------------------------------------------------

[!] SUBJECT'S ULTIMATE RIGHT:

    Even after revival conditions are met and process begins,
    the subject retains the right to refuse revival.

[ID] REFUSAL WINDOW:

    RefusalWindow {
      windowStart:    "Upon consciousness restoration"
      windowEnd:      "48 hours after capacity confirmed"

      requirements: {
        mentalCapacity: "Must be assessed as capable"
        counseling:     "Mandatory counseling session"
        waitingPeriod:  "24 hours after counseling"
        witnessedDecision: true
      }
    }

[ID] REFUSAL PROCESS:

    1. CAPACITY ASSESSMENT
       - Psychiatrist evaluates subject
       - Confirms understanding of situation
       - Documents decision-making capacity

    2. COUNSELING SESSION
       - Explain current date and circumstances
       - Discuss what has changed
       - Review available options
       - Address questions and concerns

    3. WAITING PERIOD
       - 24 hours minimum
       - Subject can change mind
       - Additional counseling available

    4. WITNESSED DECISION
       - Subject states decision clearly
       - Two witnesses present
       - Decision recorded

    5. EXECUTION
       - If refusing: Process per Alternative Directives
       - If accepting: Continue revival process

[ID] REFUSAL CONSEQUENCES:

    RefusalConsequences {
      ifRefused: {
        revivalAborted:     true
        subjectStatus:      "Per Alternative Directives"
        assetDistribution:  "Per consent document"
        familyNotification: "Immediate"
        recordRetention:    "Permanent"
      }

      options: {
        returnToPreservation: "If medically feasible"
        traditionalDeath:     "Natural conclusion"
        organDonation:        "If previously authorized"
        researchDonation:     "If previously authorized"
      }
    }
