6.1 서명 요구사항
--------------------------------------------------------------------------------

[ID] 필수 서명:

    +----------------------------------------------------------+
    | 서명자              | 필수       | 시점                  |
    +----------------------------------------------------------+
    | 대상자              | 예         | 활성화 전             |
    | 증인 1              | 예         | 대상자와 같은 세션    |
    | 증인 2              | 예         | 대상자와 같은 세션    |
    | 공증인              | 예         | 모든 서명 후          |
    | 배우자              | 권장       | 활성화 전             |
    | 법률 자문           | 선택       | 복잡한 경우           |
    | 의료 자문           | 선택       | 건강 관련 조항        |
    +----------------------------------------------------------+

[ID] 서명 기록:

    SignatureRecord {
      signerId:     "SIGNER-001"
      signerName:   "홍길동"
      signerRole:   "SUBJECT"

      signatureType: "DIGITAL"  // WET | DIGITAL | BIOMETRIC
      signatureData: "[암호화된 서명 데이터]"

      timestamp:     "2025-03-01T10:30:00Z"
      location:      "서울, 대한민국"
      ipAddress:     "203.0.113.50"

      verificationStatus: "VERIFIED"
      verificationMethod: "PKI 인증서"
      verificationDate:   "2025-03-01T10:31:00Z"
    }

[ID] 서명자 역할:

    SUBJECT             -> 보존되는 사람
    SPOUSE              -> 대상자의 배우자
    CHILD               -> 대상자의 자녀 (성인)
    GUARDIAN            -> 지정된 후견인
    WITNESS             -> 독립적인 증인
    NOTARY              -> 면허를 가진 공증인
    LEGAL_COUNSEL       -> 문서를 검토하는 변호사
    MEDICAL_ADVISOR     -> 의료 측면을 검토하는 의사
    FACILITY_REP        -> 보존 시설 대표


--------------------------------------------------------------------------------
6.2 증인 요구사항
--------------------------------------------------------------------------------

[!] 증인 자격:

    증인은 반드시:
    - 18세 이상이어야 함
    - 건전한 정신을 가져야 함
    - 대상자와 혈연 또는 혼인 관계가 아니어야 함
    - 후견인으로 지정되지 않아야 함
    - 보존 시설 직원이 아니어야 함
    - 대상자의 서명 시 참석해야 함

[ID] 증인 인증:

    WitnessCertification {
      witnessId:      "WITNESS-001"
      witnessName:    "김철수"

      certification: """
        본인 김철수는 다음을 인증합니다:
        1. 홍길동이 이 문서에 서명하는 것을 목격했습니다
        2. 대상자는 건전한 정신인 것으로 보였습니다
        3. 강압이나 부당한 영향이 관찰되지 않았습니다
        4. 대상자는 문서를 이해한다고 인정했습니다
        5. 본인은 대상자와 친족 관계가 아닙니다
        6. 본인은 이 문제에 재정적 이해관계가 없습니다
      """

      signature:      "[서명]"
      signedAt:       "2025-03-01T10:35:00Z"
      location:       "서울, 대한민국"
    }

[ID] 2인 증인 규칙:

    두 증인 모두:
    - 동시에 참석해야 함
    - 대상자가 서명하거나 서명을 인정하는 것을 목격해야 함
    - 대상자 직후에 서명해야 함
    - 확인을 위한 연락처 정보를 제공해야 함


--------------------------------------------------------------------------------
6.3 공증 절차
--------------------------------------------------------------------------------

[ID] 공증 기록:

    NotarizationRecord {
      notaryId:           "NOTARY-KR-2025-001"
      notaryName:         "박수진"
      notaryLicense:      "KR-NOTARY-12345"
      jurisdiction:       "대한민국"

      notarizationDate:   "2025-03-01T11:00:00Z"
      notarizationLocation: "서울 공증사무소"
      sealNumber:         "SEAL-2025-0301-001"

      expirationDate:     "2030-03-01"
      verificationUrl:    "https://verify.notary.kr/SEAL-2025-0301-001"
    }

[ID] 공증 단계:

    1. 신원 확인
       - 공증인이 대상자 신원 확인
       - 정부 발급 신분증 필요
       - 사진 비교

    2. 능력 확인
       - 공증인이 대상자가 능력있어 보임을 확인
       - 강압 징후 관찰되지 않음
       - 대상자가 문서를 인정

    3. 증인 확인
       - 공증인이 증인 신원 확인
       - 공증 시 증인 참석

    4. 문서 검토
       - 공증인이 문서가 완전함을 확인
       - 모든 필수 서명 있음
       - 변경이나 불규칙 없음

    5. 공증
       - 공증인이 인장 부착
       - 공증인이 서명하고 날짜 기입
       - 인증서 첨부

[ID] 국제 공증:

    관할권 간 유효성을 위해:
    - 아포스티유 인증 (헤이그 협약 국가)
    - 대사관/영사관 인증 (비헤이그)
    - 다중 관할권 공증 권장

6.4 디지털 서명
--------------------------------------------------------------------------------

[ID] 디지털 서명 표준:

    지원 표준:
    - PKCS #7 / CMS (암호화 메시지 구문)
    - PAdES (PDF 고급 전자 서명)
    - XAdES (XML 고급 전자 서명)
    - 블록체인 앵커링 서명

[ID] 디지털 서명 요구사항:

    DigitalSignatureRequirements {
      algorithm:        "RSA-2048" 또는 "ECDSA-P256" 또는 "Ed25519"
      hashFunction:     "SHA-256" 이상
      timestampAuthority: "WIA 타임스탬프 서비스"

      certificateRequirements: {
        issuer:          "공인 인증 기관"
        keyUsage:        ["digitalSignature", "nonRepudiation"]
        validityPeriod:  "서명 시점에 유효해야 함"
      }
    }
