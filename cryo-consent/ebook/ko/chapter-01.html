1.1 동의 표준이 중요한 이유
--------------------------------------------------------------------------------

냉동보존은 다른 어떤 의료 결정과도 다릅니다.
수십 년 또는 수백 년 후에야 실행될 수 있는 선택을 하는 것입니다.

[ID] 고유한 도전 과제:

    시간 규모        -> 오늘의 결정이 50-500년 후 부활에 영향
    불확실성         -> 기술이 성공할 수도 있고 아닐 수도 있음
    비가역성         -> 보존 후에는 옵션이 제한됨
    가족 영향        -> 배우자, 자녀, 미래 세대에 영향
    법적 모호성      -> 법률이 시간에 따라 극적으로 변할 수 있음

[!] 적절한 동의 없이:
    - 부활 결정이 논쟁의 대상이 됨
    - 가족 분쟁이 부활을 막을 수 있음
    - 법적 이의제기가 절차를 지연하거나 차단
    - 대상자의 의사가 무시될 수 있음


--------------------------------------------------------------------------------
1.2 핵심 원칙
--------------------------------------------------------------------------------

[ID] 원칙 1: 정보에 기반한 이해

    대상자는 진정으로 이해해야 합니다:
    - 냉동보존이 무엇을 포함하는지
    - 알려진 모든 위험과 제한
    - 부활이 보장되지 않는다는 것
    - 재정적 의미
    - 가족 관계에 대한 영향

[ID] 원칙 2: 자발적 결정

    결정은 반드시:
    - 강압 없이 이루어져야 함
    - 부당한 영향에서 자유로워야 함
    - 압박 하에 이루어지지 않아야 함
    - 재고할 수 있어야 함

[ID] 원칙 3: 문서화된 의사

    동의는 명확히 명시해야 합니다:
    - 부활 조건
    - 결정을 내릴 수 있는 사람
    - 대안 지시
    - 철회 조건

[ID] 원칙 4: 철회 가능성

    대상자는 다음 권리를 유지합니다:
    - 보존 전 동의 철회
    - 수정을 통한 조건 변경
    - 조건 충족 후에도 부활 거부
    - 후견인에게 결정 위임


--------------------------------------------------------------------------------
1.3 문서 생애주기
--------------------------------------------------------------------------------

[ID] 동의 문서 상태:

    DRAFT (초안)
      |
      v
    PENDING_SIGNATURES (서명 대기)
      |
      v
    PENDING_NOTARIZATION (공증 대기)
      |
      v
    ACTIVE (활성) ---------> SUSPENDED (문제 발생시 일시 중단)
      |                          |
      |                          v
      |                      ACTIVE (해결 시)
      |
      v
    EXECUTED (보존 시작)
      |
      +-------------> WITHDRAWN (대상자가 철회 시)
      |
      v
    COMPLETED (부활 완료 또는 종료)

[ID] 상태 정의:

    DRAFT                -> 초기 작성, 아직 서명되지 않음
    PENDING_SIGNATURES   -> 필요한 서명 대기 중
    PENDING_NOTARIZATION -> 서명됨, 공증 대기 중
    ACTIVE               -> 완전히 실행됨, 법적 구속력 있음
    SUSPENDED            -> 일시적으로 보류
    EXECUTED             -> 보존이 시작됨
    WITHDRAWN            -> 대상자 또는 후견인이 철회
    COMPLETED            -> 부활 완료 또는 보존 종료


--------------------------------------------------------------------------------
1.4 법적 프레임워크
--------------------------------------------------------------------------------

[ID] 동의 요구사항:

    +----------------------------------------------------------+
    | 요구사항                | 상태       | 비고              |
    +----------------------------------------------------------+
    | 대상자 정신적 능력      | 필수       | 평가 문서 필요    |
    | 대상자 서명             | 필수       | 디지털 또는 직필  |
    | 2명의 증인              | 필수       | 비가족            |
    | 공증                    | 필수       | 활성화 전         |
    | 배우자 동의             | 권장       | 기혼인 경우       |
    | 법률 자문 검토          | 권장       | 복잡한 경우       |
    +----------------------------------------------------------+

[ID] 관할권 고려사항:

    - 동의 유효성은 관할권에 따라 다름
    - 일부 국가는 냉동보존을 인정하지 않을 수 있음
    - 다중 관할권 보호 권장
    - 미래 법률 변경 고려

[i] 법률 자문:
    동의 문서를 최종화하기 전에 법률 상담을 강력히 권장합니다.
    특히 다음의 경우:
    - 국제 대상자
    - 복잡한 가족 상황
    - 상당한 자산
    - 연구 동의 조항
