--------------------------------------------------------------------------------
2.1 대상자 식별
--------------------------------------------------------------------------------

정확한 대상자 식별은 장기 동의 유효성에 필수적입니다.

[ID] 대상자 정보 기록:

    SubjectInformation {
      subjectId:      "SUBJECT-2025-001"
      identityRef:    "CRYOID-2025-001"  // CRYO-IDENTITY 연결

      personalInfo: {
        fullLegalName:  "홍길동"
        dateOfBirth:    "1970-03-15"
        placeOfBirth:   "서울특별시, 대한민국"
        nationality:    ["대한민국", "미국"]
        currentAddress: "[암호화됨]"
        contactInfo:    "[암호화됨]"
      }
    }

[ID] 신원 확인:

    동의가 최종화되기 전:
    1. 정부 발급 신분증 확인
    2. CRYO-IDENTITY 기록과 교차 참조
    3. 생체 정보 캡처 (사진, 지문)
    4. 증인의 신원 증명


--------------------------------------------------------------------------------
2.2 정신적 능력 평가
--------------------------------------------------------------------------------

[!] 중요: 동의가 유효하려면 정신적 능력 평가가 필수입니다.

[ID] 평가 요구사항:

    평가는 자격을 갖춘 전문가가 수행해야 합니다:
    - 면허를 가진 정신과 의사 또는 심리학자
    - 관련 교육을 받은 의사
    - 법원이 지정한 평가자 (필요시)

[ID] 이해 확인:

    대상자는 다음에 대한 이해를 보여야 합니다:

    +---------------------------------------------------------------+
    | 영역                        | 필수     | 대상자 확인          |
    +---------------------------------------------------------------+
    | 보존 과정                   | 예       | [  ]                 |
    | 알려진 위험과 제한          | 예       | [  ]                 |
    | 부활 보장 없음              | 예       | [  ]                 |
    | 재정적 의미                 | 예       | [  ]                 |
    | 가족에 대한 영향            | 예       | [  ]                 |
    +---------------------------------------------------------------+

[ID] 의사결정 능력:

    DecisionCapacity {
      voluntary:               true   // 결정이 자유롭게 이루어짐
      noCoercion:              true   // 타인의 압박 없음
      understoodAlternatives:  true   // 다른 옵션 인지
    }

[ID] 능력 판정:

    CAPABLE (능력있음)       -> 대상자가 완전한 정신적 능력 보유
    QUESTIONABLE (의문)     -> 추가 검토 권장
    INCAPABLE (능력없음)    -> 법적 대리인 필요


--------------------------------------------------------------------------------
2.3 법적 능력
--------------------------------------------------------------------------------

[ID] 법적 능력 평가:

    LegalCapacity {
      assessmentDate:  "2025-03-01"
      assessedBy:      "박민수 정신과 전문의"
      capacity:        "FULL"

      limitations:     []  // 의사결정 제한 사항
      courtOrders:     []  // 관련 법원 명령
    }

[ID] 특수 상황:

    미성년자:
    - 일반적으로 냉동보존에 동의할 수 없음
    - 법원 승인과 함께 부모/후견인 동의
    - 성년이 되면 대상자가 동의해야 함

    제한된 능력:
    - 법적 대리인이 공동 서명해야 함
    - 법원 승인이 필요할 수 있음
    - 추가 보호 조치 적용

    응급 상황:
    - 사전 승인된 응급 동의
    - 후견인 의사결정
    - 의료 위임장
