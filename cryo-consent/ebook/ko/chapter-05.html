--------------------------------------------------------------------------------
5.1 배우자 동의
--------------------------------------------------------------------------------

기혼 대상자의 경우 배우자 참여가 권장됩니다.

[ID] 배우자 합의 기록:

    SpouseAgreement {
      spouseId:       "SPOUSE-2025-001"
      spouseName:     "홍길순"
      marriageStatus: "MARRIED"

      consentGiven:   true
      consentScope: {
        preservationApproved:       true
        financialArrangementApproved: true
        revivalConditionsApproved:  true
        objections:                 []
        negotiatedTerms:            []
      }
    }

[ID] 보존 후 권리:

    rightsAfterPreservation: {
      remarriageAllowed:    true   // 배우자 재혼 가능
      inheritanceRights:    "FULL" // 완전한 상속권
      decisionMakingRole:   "PRIMARY" // 주요 후견인
      revivalVoteWeight:    2      // 부활 결정에서의 가중치
    }

[ID] 공동 보존:

    jointPreservation: {
      requested:  true
      conditions: [
        "같은 시설",
        "가능하면 함께 부활",
        "한 쪽이 먼저 부활하면 다른 쪽에 통지"
      ]
    }

[!] 배우자 이의:
    배우자가 이의를 제기하면 진행 전에 중재가 권장됩니다.
    모든 이의와 해결 사항을 문서화하세요.

    ChildAgreement {

      }

      acknowledgment: {
        acknowledged:     true
        acknowledgedDate: "2025-02-20"
        concerns:         ["가족에 대한 재정적 영향"]
      }

      guardianRole: {
        designated:       true
        responsibilities: ["CONSENT_DECISIONS", "FAMILY_COMMUNICATION"]
        voteWeight:       1
      }
    }

[ID] 미성년 자녀:

    미성년 자녀의 경우:
    - 나이에 적합한 통지
    - 법정 후견인이 대신 서명
    - 자녀가 성년이 되면 재동의
    - 자녀의 미래 후견인 역할 고려

[ID] 통지 요구사항:

    +----------------------------------------------------------+
    | 관계          | 통지       | 후견인 역할  | 서명         |
    +----------------------------------------------------------+
    | 배우자        | 필수       | 권장         | 권장         |
    | 성인 자녀     | 필수       | 선택         | 선택         |
    | 미성년 자녀   | 나이적합   | 미래         | 후견인       |
    | 부모          | 권장       | 선택         | 선택         |
    | 형제자매      | 선택       | 선택         | 선택         |
    +----------------------------------------------------------+


--------------------------------------------------------------------------------
5.3 후견인 지정
--------------------------------------------------------------------------------

[ID] 후견인 유형:

    PERSON       -> 대상자가 신뢰하는 개인
    ORGANIZATION -> 법인 (로펌, 신탁 회사)
    SMART_CONTRACT -> 자율적 온체인 후견인 (고급)

[ID] 후견인 기록:

    GuardianDesignation {
      guardianId:    "GUARD-2025-001"
      guardianType:  "PERSON"

      personalInfo: {
        name:         "홍길순"
        relationship: "배우자"
        contactInfo:  "[암호화됨]"
      }

      responsibilities: [
        "CONSENT_DECISIONS",
        "FINANCIAL_OVERSIGHT",
        "FACILITY_LIAISON",
        "REVIVAL_DECISIONS",
        "FAMILY_COMMUNICATION"
      ]

      voteWeight:   2

      succession: {
        primarySuccessor:   "홍명동"
        secondarySuccessor: "홍명희"
        successionTrigger:  "사망, 무능력, 또는 사임"
      }
    }

[ID] 책임 유형:

    CONSENT_DECISIONS    -> 동의 수정 또는 철회
    FINANCIAL_OVERSIGHT  -> 보존 자금 모니터링
    FACILITY_LIAISON     -> 시설과 소통
    REVIVAL_DECISIONS    -> 부활 승인/거부
    FAMILY_COMMUNICATION -> 가족에게 정보 제공
    LEGAL_REPRESENTATION -> 법적 문제 처리

[ID] 후견인 네트워크 구성:

    권장: 혼합 유형의 3-5명 후견인

    예시:
    +------------------------------------------+
    | 후견인          | 유형   | 가중치 | 역할 |
    +------------------------------------------+
    | 배우자          | PERSON | 2      | 전체 |
    | 성인 자녀       | PERSON | 1      | 전체 |
    | 로펌            | ORG    | 1      | 법적 |
    | 신탁 회사       | ORG    | 1      | 재정 |
    +------------------------------------------+
    총 가중치: 5
    결정을 위한 임계값: 3 (과반수)


--------------------------------------------------------------------------------
5.4 분쟁 해결
--------------------------------------------------------------------------------

[ID] 분쟁 해결 계획:

    ConflictResolutionPlan {
      primaryMethod: "MEDIATION"

      mediator: "WIA 중재 서비스"
      arbitrationProvider: "대한상사중재원"
      governingLaw: "대한민국"
      venue: "서울"

      escalationPath: [
        { step: 1, method: "가족 토론",     timeout: "30일" },
        { step: 2, method: "공식 중재",     timeout: "60일" },
        { step: 3, method: "구속력 있는 중재", timeout: "90일" },
        { step: 4, method: "법원 소송",     timeout: "없음" }
      ]

      deadlockResolution: "중재자가 증거 다수에 의해 결정"
    }

[ID] 일반적인 분쟁:

    1. 부활 시기 불일치
       해결: 의료 위원회 의견, 그 다음 후견인 투표

    2. 자금 배분 분쟁
       해결: 신탁 조건이 지배, 그 다음 중재

    3. 시설 이전 불일치
       해결: 중재, 대상자의 명시된 선호 고려

[i] 예방:
    명확한 의사소통과 정기적인 가족 회의가
    분쟁 가능성을 크게 줄입니다.
