3.1 보존 동의 유형
--------------------------------------------------------------------------------

[ID] 동의 유형 옵션:

    전체 동의 (FULL CONSENT):
    - 모든 보존 옵션 활성화
    - 표준 부활 조건
    - 연구 참여 허용
    - 미래 결정에 가장 유연함

    조건부 동의 (CONDITIONAL CONSENT):
    - 특정 조건 하에서만 보존
    - 맞춤 부활 요구사항
    - 사용 가능한 옵션 제한 가능

    제한 동의 (LIMITED CONSENT):
    - 제한된 부활 시나리오
    - 특정 제외 사항 정의
    - 권한 범위 좁음

    연구 전용 (RESEARCH ONLY):
    - 부활 의도 없음
    - 과학적 사용 승인
    - 냉동보존 연구에 기증

[ID] 신체 처리 옵션:

    +------------------------------------------+
    | 옵션           | 설명                    |
    +------------------------------------------+
    | WHOLE_BODY     | 전신 보존               |
    | NEURO_ONLY     | 두부/뇌만               |
    +------------------------------------------+

    [i] 두부만 옵션은 미래 기술이 새로운 신체를
        재생하거나 제공할 수 있다고 가정합니다.


--------------------------------------------------------------------------------
3.2 인정된 위험
--------------------------------------------------------------------------------

[!] 대상자는 모든 위험에 대한 이해를 인정해야 합니다.

[ID] 위험 카테고리:

    기술적 위험:
    - 보존 중 장비 고장
    - 저장에 영향을 미치는 정전
    - 시설 손상 (자연재해 등)
    - 운송 사고

    생물학적 위험:
    - 얼음 결정 형성 (세포 손상)
    - 불완전한 유리화
    - 시간에 따른 조직 열화
    - 알려지지 않은 장기 효과

    조직적 위험:
    - 시설 파산
    - 회사 해산
    - 경영진 변경
    - 서비스 중단

    법적 위험:
    - 냉동보존을 금지하는 법률 변경
    - 관할권 충돌
    - 가족의 법적 이의제기
    - 유산 분쟁

    재정적 위험:
    - 자금 고갈
    - 경제 붕괴
    - 통화 가치 하락
    - 신탁 잘못된 관리

    미지의 위험:
    - 미래 기술이 개발되지 않을 수 있음
    - 부활이 불가능할 수 있음
    - 예측하지 못한 합병증


--------------------------------------------------------------------------------
3.3 연구 동의
--------------------------------------------------------------------------------

[ID] 연구 동의 옵션:

    ResearchConsent {
      authorized: true
      scope:      "PRESERVATION_RESEARCH"

      allowedResearch: {
        tissuesampling:       true   // 소량 조직 샘플
        imagingStudies:       true   // 비침습적 영상
        geneticResearch:      false  // DNA 분석
        procedureObservation: true   // 케이스에서 학습
      }

      dataSharing: {
        anonymizedData:   true   // 비식별화 데이터
        identifiableData: false  // 신원 비공개 유지
        commercialUse:    false  // 상업적 응용 없음
      }

      exclusions: [
        "유전자 변형 연구 불허",
        "복제 시도 불허"
      ]
    }

[ID] 연구 범위 수준:

    NONE                 -> 연구 참여 없음
    PRESERVATION_RESEARCH -> 보존 기술 개선
    REVIVAL_RESEARCH     -> 부활 방법 개발
    GENERAL_MEDICAL      -> 더 넓은 의학 연구
    UNRESTRICTED         -> 모든 합법적 연구


--------------------------------------------------------------------------------
3.4 대안 지시
--------------------------------------------------------------------------------

[ID] 부활이 불가능해지면:

    ImpossibleRevivalDirective {
      waitDuration:  "P100Y"  // 100년 대기
      afterWaiting:  "RESEARCH_DONATION"

      researchDonation: {
        authorized:   true
        institutions: ["WIA 연구소", "MIT 냉동보존 연구소"]
        restrictions: ["상업적 사용 불허", "대상자 크레딧"]
      }
    }

[ID] 시설이 실패하면:

    FacilityFailureDirective {
      transferPriority: [
        "Alcor 생명연장재단",
        "냉동보존연구소",
        "WIA 파트너 네트워크"
      ]

      ifNoTransferPossible: "TERMINATE"

      insuranceClaim: {
        authorized:    true
        beneficiaries: ["홍길동 유산"]
      }
    }

[ID] 자금이 부족하면:

    FundingFailureDirective {
      gracePeriod: "P5Y"  // 자금 복구까지 5년

      fundingRecoveryAttempts: {
        familyAppeal:         true
        crowdfunding:         true
        charitableFunds:      true
        governmentAssistance: false
      }

      ifFundingNotRestored: "REDUCED_SERVICE"
    }

[ID] 법률이 변경되면:

    LegalChangeDirective {
      ifPreservationBanned: "TRANSFER_JURISDICTION"
      preferredJurisdictions: ["스위스", "코스타리카"]

      ifRevivalBanned: "WAIT"
      waitDuration: "P50Y"

      legalDefenseFund: {
        authorized:    true
        maximumAmount: { amount: 100000, currency: "USD" }
      }
    }
