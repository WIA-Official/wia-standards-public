# WIA-SEMI-004 Benchmarking Guide

## Standard Benchmark Suite

This guide provides detailed instructions for benchmarking AI chips according to WIA-SEMI-004 standards.

### Benchmark Models
- ResNet-50: Image classification baseline
- BERT-Base: NLP transformer model
- YOLOv5: Object detection
- Stable Diffusion: Generative AI

### Metrics
- Throughput (inferences/sec)
- Latency (p50, p95, p99 percentiles)
- Power consumption (watts)
- TOPS/W efficiency

### Reporting
Results must be published in JSON format with full system configuration.
