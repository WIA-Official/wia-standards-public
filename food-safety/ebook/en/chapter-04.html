Chapter 4: Phase 1 - Data Format Specification

弘益人間 (Hongik Ingan)

Standardized data formats are the foundation of interoperability. Phase 1 defines JSON schemas for products, safety monitoring, recalls, and traceability—enabling seamless data exchange across the global food supply chain.

4.1 Product Data Structure

{
  "productId": "FS-2025-STR-001",
  "productName": "Fresh Strawberries",
  "category": "fresh-produce",
  "batchNumber": "FARM-20250115-001",
  "productionDate": "2025-01-15T06:00:00Z",
  "expiryDate": "2025-01-22T23:59:59Z",
  "manufacturer": {
    "id": "MFR-12345",
    "name": "Sunshine Farms",
    "did": "did:wia:manufacturer:sunshine-farms"
  },
  "allergens": {
    "contains": [],
    "free": ["Gluten", "Dairy", "Nuts"]
  },
  "certifications": ["HACCP", "Organic-USDA"]
}

4.2 Safety Monitoring Data

{
  "monitoringId": "MON-2025-001",
  "productId": "FS-2025-STR-001",
  "timestamp": "2025-01-15T10:30:00Z",
  "temperature": {
    "current": 4.2,
    "criticalLimit": 7.0,
    "status": "safe"
  },
  "contamination_tests": [
    {
      "testType": "microbial",
      "parameter": "E.coli",
      "result": "negative",
      "status": "pass"
    }
  ],
  "haccp_ccp": [
    {
      "ccpNumber": 1,
      "measuredValue": 3.8,
      "status": "within-limit"
    }
  ]
}

4.3 Recall Data Structure

{
  "recallId": "RECALL-2025-001",
  "timestamp": "2025-01-15T10:00:00Z",
  "severity": "class-I",
  "reason": "Salmonella contamination detected",
  "affectedProducts": [
    {
      "productId": "FS-2025-STR-001",
      "batchNumbers": ["FARM-20250115-001"],
      "quantity": 5000
    }
  ],
  "regulatoryAgencies": [
    {
      "agency": "FDA",
      "notificationDate": "2025-01-15T10:05:00Z"
    }
  ]
}

4.4 Validation Rules

RuleValidationExample
Expiry Date expiryDate > productionDate Jan 22 > Jan 15 ✓
Temperature current ≤ criticalLimit 4.2°C ≤ 7.0°C ✓
Batch Number ^[A-Z0-9]{4}-[0-9]{8}-[0-9]{3}$ FARM-20250115-001 ✓

Summary

Phase 1 defines standardized JSON schemas for all food safety data, ensuring consistency and interoperability across systems. These formats support products, safety monitoring, recalls, and full traceability chains.
