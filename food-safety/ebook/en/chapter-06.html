6.2 MQTT for IoT Sensors

Topic Structure:
wia/food-safety/{facility_id}/{zone}/{sensor_type}/{sensor_id}

Example:
wia/food-safety/FAC-001/cold-storage/temperature/TEMP-001

Payload:
{
  "temperature": 4.2,
  "status": "normal",
  "timestamp": "2025-01-15T10:00:00Z"
}

6.4 WebSocket Real-Time Alerts

const ws = new WebSocket('wss://api.wia.org/food-safety/stream');

ws.onmessage = (event) => {
  const alert = JSON.parse(event.data);
  if (alert.severity === 'critical') {
    triggerEmergencyResponse(alert);
  }
};

Summary

Phase 3 enables real-time communication through MQTT (IoT), blockchain (traceability), WebSockets (alerts), and regulatory APIs (compliance). These protocols ensure instant response to safety threats.
