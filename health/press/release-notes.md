# WIA Health — Release Notes (1.0)

**Status:** Stable
**Release Date:** 2026-04-26
**Sponsor:** WIA Standards working group
**Maintainers:** WIA Standards Editorial Team

## Overview

WIA Health defines a cross-vendor data and protocol layer covering data
format, API interface, protocol, and integration concerns. The standard
references only ALLOW sources per the WIA citation policy.

## What's in this release

- PHASE-1 Data Format — schemas in JSON Schema 2020-12
- PHASE-2 API Interface — REST API in OpenAPI 3.1
- PHASE-3 Protocol — message types and trust model (Sigstore DSSE + Rekor)
- PHASE-4 Integration — deployment topologies and evidence packaging

## Conformance

Implementations follow the three-tier conformance model documented in
the README: Surface (self-attested), Verified (annual audit), Anchored
(continuous evidence package).
