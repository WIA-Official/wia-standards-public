Chapter 6: Online Obituary Systems

Publishing and Syndication Standards

6.1 Online Obituary Overview

Online obituary systems modernize traditional death notices through digital publishing, syndication, and search optimization. This chapter establishes standards for obituary creation, multi-platform distribution, SEO optimization, memorial website integration, and analytics tracking.

The WIA Online Obituary standard addresses content management systems, newspaper syndication APIs, social media sharing, search engine indexing, and permanent archival. These systems enable families to widely announce a death while creating lasting digital tributes.

Key Features

  • Template Engine: Professional obituary templates with customization options
  • Multi-platform Publishing: Simultaneous distribution to newspapers, websites, social media
  • SEO Optimization: Search-friendly markup and indexing for discoverability
  • Analytics: Track views, shares, and condolence engagement
  • Permanent Archive: Long-term preservation with persistent URLs
  • Syndication: RSS feeds and API access for aggregators

6.5 Implementation Examples

Example 1: Obituary Publisher (JavaScript)

class ObituaryPublisher {
  async createObituary(deceasedInfo) {
    const obituary = {
      deceased_name: deceasedInfo.fullName,
      date_of_birth: deceasedInfo.dob,
      date_of_death: deceasedInfo.dod,
      age: this.calculateAge(deceasedInfo.dob, deceasedInfo.dod),
      location: deceasedInfo.location,
      content: {
        headline: `${deceasedInfo.fullName}, ${this.calculateAge(deceasedInfo.dob, deceasedInfo.dod)}`,
        body_text: deceasedInfo.biography,
        survived_by: deceasedInfo.survivors,
        life_summary: deceasedInfo.summary
      },
      seo: {
        meta_description: `Obituary for ${deceasedInfo.fullName}, ${deceasedInfo.dod}`,
        keywords: [deceasedInfo.fullName, deceasedInfo.location.city, 'obituary'],
        schema_markup: this.generateSchemaMarkup(deceasedInfo)
      }
    };

    const response = await fetch('/api/v1/obituary/create', {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify(obituary)
    });

    return await response.json();
  }

  generateSchemaMarkup(info) {
    return {
      "@context": "https://schema.org",
      "@type": "Person",
      "name": info.fullName,
      "birthDate": info.dob,
      "deathDate": info.dod,
      "description": info.biography
    };
  }

  calculateAge(dob, dod) {
    const birth = new Date(dob);
    const death = new Date(dod);
    return Math.floor((death - birth) / (365.25 * 24 * 60 * 60 * 1000));
  }
}

Example 2: Search and Discovery (Python)

import requests

class ObituarySearch:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = 'https://api.obituary.funeral-services.org'

    def search(self, query, filters=None):
        params = {'q': query}
        if filters:
            params.update(filters)

        response = requests.get(
            f'{self.base_url}/api/v1/obituary/search',
            headers={'Authorization': f'Bearer {self.api_key}'},
            params=params
        )
        return response.json()

    def get_recent(self, limit=10, location=None):
        params = {'limit': limit}
        if location:
            params['location'] = location

        response = requests.get(
            f'{self.base_url}/api/v1/obituary/recent',
            headers={'Authorization': f'Bearer {self.api_key}'},
            params=params
        )
        return response.json()

# Usage
search = ObituarySearch('api-key')
results = search.search('John Smith', {'city': 'Springfield', 'year': 2024})

Example 3: Multi-platform Distribution (TypeScript)

class ObituaryDistributor {
  async publishToChannels(obituaryId: string, channels: string[]) {
    const results = await Promise.all(
      channels.map(channel => this.publishToChannel(obituaryId, channel))
    );
    return results;
  }

  private async publishToChannel(obituaryId: string, channel: string) {
    const response = await fetch(
      `/api/v1/obituary/${obituaryId}/publish`,
      {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({channel})
      }
    );
    return await response.json();
  }
}
