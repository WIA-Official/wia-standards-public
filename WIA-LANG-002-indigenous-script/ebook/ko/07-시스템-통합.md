# 제7장: 시스템 통합

## 플랫폼별 원주민 문자 지원

이 장에서는 다양한 운영체제, 브라우저, 애플리케이션에 원주민 문자를 통합하는 방법을 다룹니다.

## 운영체제 통합

### Windows

**폰트 설치**:
```powershell
# 시스템 폰트 디렉토리에 복사
Copy-Item "NotoSansOlChiki.ttf" "$env:WINDIR\Fonts\"

# 레지스트리 등록
New-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Fonts" `
    -Name "Noto Sans Ol Chiki (TrueType)" `
    -Value "NotoSansOlChiki.ttf"
```

**키보드 설치**:
```
1. Microsoft Keyboard Layout Creator (MSKLC) 사용
2. .klc 파일에서 설치 패키지 생성
3. setup.exe 실행하여 설치
4. 설정 > 언어에서 활성화
```

### macOS

**폰트 설치**:
```bash
# 사용자 폰트
cp NotoSansOlChiki.ttf ~/Library/Fonts/

# 시스템 폰트 (관리자)
sudo cp NotoSansOlChiki.ttf /Library/Fonts/
```

**키보드 번들 구조**:
```
OlChiki.bundle/
├── Contents/
│   ├── Info.plist
│   ├── Resources/
│   │   └── OlChiki.keylayout
│   └── version.plist
```

```css

  font-display: swap;
  unicode-range: U+1C50-1C7F;
}

  font-family: 'Ol Chiki', 'Noto Sans Ol Chiki', sans-serif;
}

/* RTL 지원 (해당 시) */

  direction: rtl;
  text-align: right;
}
```

### JavaScript 가상 키보드

```javascript
class VirtualKeyboard {
  constructor(scriptId) {
    this.layout = null;
  }

  async load(layoutUrl) {
    const response = await fetch(layoutUrl);
    this.layout = await response.json();
    this.render();
  }

  render() {
    const container = document.getElementById('keyboard');
    this.layout.layers.default.forEach((row, i) => {
      const rowEl = document.createElement('div');
      rowEl.className = 'keyboard-row';
      row.forEach(key => {
        const keyEl = document.createElement('button');
        keyEl.textContent = key;
        keyEl.onclick = () => this.onKey(key);
        rowEl.appendChild(keyEl);
      });
      container.appendChild(rowEl);
    });
  }

  onKey(key) {
    const input = document.activeElement;
    if (input.tagName === 'INPUT' || input.tagName === 'TEXTAREA') {
      input.value += key;
    }
  }
}
```

```kotlin
// assets/fonts/OlChiki.ttf 로드

textView.typeface = typeface
```

**커스텀 키보드 (InputMethodService)**:
```kotlin
class OlChikiKeyboard : InputMethodService() {
    override fun onCreateInputView(): View {
        val keyboardView = layoutInflater.inflate(
            R.layout.keyboard_olchiki, null
        ) as KeyboardView
        return keyboardView
    }

    override fun onKey(primaryCode: Int, keyCodes: IntArray?) {
        currentInputConnection.commitText(
            String(Character.toChars(primaryCode)), 1
        )
    }
}
```

### 워드프로세서

**Microsoft Word 매크로**:
```vba
Sub InsertOlChiki()
    Selection.Font.Name = "Noto Sans Ol Chiki"
    Selection.TypeText Text:=ChrW(&H1C5A)
End Sub
```

**LibreOffice 확장**:
```python
# OlChiki toolbar extension
def insert_character(character):
    doc = XSCRIPTCONTEXT.getDocument()
    text = doc.Text
    cursor = text.createTextCursor()
    cursor.gotoEnd(False)
    text.insertString(cursor, character, False)
```

### HarfBuzz 셰이핑

```c
// HarfBuzz를 사용한 텍스트 셰이핑
hb_buffer_t *buf = hb_buffer_create();
hb_buffer_add_utf8(buf, text, -1, 0, -1);
hb_buffer_set_direction(buf, HB_DIRECTION_LTR);
hb_buffer_set_script(buf, HB_SCRIPT_OL_CHIKI);

hb_font_t *font = hb_font_create(hb_face);
hb_shape(font, buf, NULL, 0);

// 결과 글리프 가져오기
unsigned int glyph_count;
hb_glyph_info_t *glyph_info =
    hb_buffer_get_glyph_infos(buf, &glyph_count);
```
