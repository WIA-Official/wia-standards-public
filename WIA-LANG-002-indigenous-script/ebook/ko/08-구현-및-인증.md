## 샘플 구현

### 문자 정의

```json
{
  "script": {
    "code": "Olck",
    "name": "Ol Chiki"
  },
  "characters": [
    {
      "codePoint": "U+1C5A",
      "name": "OL CHIKI LETTER LA",
      "glyph": "ᱚ",
      "category": "Lo",
      "phonetic": "/ɔ/"
    },
    {
      "codePoint": "U+1C5B",
      "name": "OL CHIKI LETTER AT",
      "glyph": "ᱛ",
      "category": "Lo",
      "phonetic": "/at/"
    }
  ]
}
```

### 폰트 개발

```python
from fontTools.fontBuilder import FontBuilder

# 새 폰트 생성
fb = FontBuilder(1000, isTTF=True)
fb.setupGlyphOrder([".notdef", "ola", "olat"])
fb.setupCharacterMap({
    0x1C5A: "ola",
    0x1C5B: "olat"
})

# 글리프 정의
pen = fb.setupGlyf({})
# ... 글리프 경로 정의

fb.save("OlChiki.ttf")
```

### 키보드 생성

```json
{
  "keyboard": {
    "id": "olck-phonetic",
    "name": "Ol Chiki Phonetic",
    "rules": [
      {"input": "a", "output": "ᱚ"},
      {"input": "t", "output": "ᱛ"},
      {"input": "g", "output": "ᱜ"},
      {"input": "ng", "output": "ᱝ"}
    ]
  }
}
```

## 테스트

### 폰트 검증

```bash
# 폰트 유효성 검사
wia-script validate-font OlChiki.ttf

# 출력:
# ✓ Required tables present
# ✓ Character coverage complete
# ✓ OpenType features valid
# ✓ Metrics consistent
```

### 키보드 테스트

```javascript
describe('Ol Chiki Keyboard', () => {
  it('should convert phonetic input', () => {
    const ime = new OlChikiIME();
    expect(ime.process('a')).toBe('ᱚ');
    expect(ime.process('ng')).toBe('ᱝ');
  });

  it('should handle sequences', () => {
    const ime = new OlChikiIME();
    expect(ime.process('santali')).toBe('ᱥᱟᱱᱛᱟᱞᱤ');
  });
});
```

### 렌더링 테스트

```python
def test_rendering_all_characters():
    font = load_font("OlChiki.ttf")

    for cp in range(0x1C50, 0x1C80):
        char = chr(cp)
        glyph = font.get_glyph(cp)
        assert glyph is not None, f"Missing glyph for U+{cp:04X}"
        assert glyph.width > 0, f"Zero width for U+{cp:04X}"
```

## WIA 인증 프로그램

### 인증 등급

| 등급 | 요구사항 |
|------|----------|
| **Bronze** | 기본 폰트 + 키보드 1개 |
| **Silver** | + 웹폰트 + 웹 키보드 |
| **Gold** | + 다중 플랫폼 + 테스트 |
| **Platinum** | + 커뮤니티 승인 + 운영 |

### 인증 절차

```
1. 신청서 제출
   └── 프로젝트 정보, 커뮤니티 정보

2. 자가 평가
   └── 체크리스트 작성

3. 기술 검토
   ├── 폰트 검증
   ├── 키보드 테스트
   └── 렌더링 확인

4. 커뮤니티 확인
   └── 대표 승인

5. 인증서 발급
```

### 인증 신청

```bash
# 자가 진단
wia-script certification check --level silver

# 결과:
# WIA-LANG-002 Certification Check
# ================================
# Level: Silver
#
# Encoding:     ✓ 4/4 passed
# Fonts:        ✓ 5/5 passed
# Keyboards:    ✓ 3/3 passed
# Rendering:    ✓ 3/3 passed
#
# Score: 100/100
# Status: READY FOR CERTIFICATION

# 신청
wia-script certification submit --level silver
```
