Key Features

주요 기능

🔤

Unicode Integration

유니코드 통합

Complete Unicode support for indigenous scripts including Cherokee, Vai, N'Ko, Ogham, Runic, and 140+ other writing systems.

체로키, 바이, 은코, 오검, 룬 문자 및 140개 이상의 문자 시스템을 포함한 토착 문자에 대한 완전한 유니코드 지원.

👁️

Advanced OCR

고급 OCR

AI-powered optical character recognition for historical manuscripts, stone inscriptions, and hand-written documents in any script.

역사적 필사본, 석문, 모든 문자의 손글씨 문서를 위한 AI 기반 광학 문자 인식.

⌨️

Virtual Keyboards

가상 키보드

Generate custom keyboard layouts for any indigenous script, supporting web, mobile, and desktop platforms.

웹, 모바일, 데스크톱 플랫폼을 지원하는 모든 토착 문자에 대한 맞춤형 키보드 레이아웃 생성.

🖋️

Font Development

폰트 개발

Create, modify, and distribute fonts for indigenous scripts with automatic glyph generation and stylistic variations.

자동 글리프 생성 및 양식적 변형을 통한 토착 문자용 폰트 생성, 수정 및 배포.

📖

Text Rendering

텍스트 렌더링

Advanced text layout engine supporting right-to-left, vertical writing, and complex ligature systems.

오른쪽에서 왼쪽, 세로 쓰기, 복잡한 합자 시스템을 지원하는 고급 텍스트 레이아웃 엔진.

Paleographic Analysis

고문서학적 분석

Tools for analyzing historical script evolution, identifying writing styles, and dating manuscripts based on paleographic features.

역사적 문자 진화 분석, 서체 식별, 고문서학적 특징에 기반한 필사본 연대 측정 도구.
