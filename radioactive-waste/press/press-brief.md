# WIA-ENE-026 — Radioactive Waste Management (Press Brief)

**WIA Standards releases canonical Radioactive Waste Management envelope shape**

The WIA Standards Committee has published the WIA-ENE-026 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for radioactive waste management hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
