## 1. Overview

This specification defines clinical protocols for implementing the Neuroplasticity Reversal approach to chronic pain management, focusing on reversing maladaptive neural changes through multimodal intervention.

## 2. Assessment Protocol

### 2.1 Comprehensive Pain Assessment

```
PROTOCOL: WIA-CP-ASSESS-001
Name: Multidimensional Chronic Pain Assessment
Duration: 2-3 hours (can be split across visits)

┌─────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: Pain History & Characterization (30-45 min)                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ REQUIRED:                                                                   │
│ □ Pain onset, duration, progression                                        │
│ □ Location(s) with body diagram                                            │
│ □ Quality descriptors (burning, shooting, aching, etc.)                    │
│ □ Intensity scales (NRS, VAS)                                              │
│ □ Temporal pattern (constant, intermittent, diurnal variation)             │
│ □ Aggravating and relieving factors                                        │
│                                                                             │
│ PAIN PHENOTYPING:                                                           │
│ □ Nociceptive features: localized, proportional to tissue                  │
│ □ Neuropathic features: painDETECT, DN4 questionnaire                      │
│ □ Nociplastic features: widespread, disproportionate, CSI                  │
│                                                                             │
│ TREATMENT HISTORY:                                                          │
│ □ Prior medications and responses                                           │
│ □ Interventional procedures                                                 │
│ □ Physical therapy history                                                  │
│ □ Alternative treatments tried                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: Central Sensitization Evaluation (30-45 min)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│ SCREENING:                                                                  │
│ □ Central Sensitization Inventory (CSI)                                    │
│   • Score ≥40: High probability of central sensitization                   │
│   • Score ≥60: Severe central sensitization                                │
│                                                                             │
│ QUANTITATIVE SENSORY TESTING (QST):                                        │
│ □ Mechanical detection threshold (MDT)                                     │
│ □ Mechanical pain threshold (MPT)                                          │
│ □ Pressure pain threshold (PPT) - local and remote                         │
│ □ Temporal summation (wind-up)                                              │
│ □ Conditioned pain modulation (CPM)                                         │
│                                                                             │
│ INTERPRETATION:                                                             │
│ • Enhanced temporal summation → Facilitated pain processing                │
│ • Reduced CPM → Impaired descending inhibition                             │
│ • Widespread low PPT → Generalized sensitization                           │
│ • Allodynia/hyperalgesia → Altered processing                              │
│                                                                             │
│ CALCULATE NEUROPLASTICITY INDEX:                                           │
│ NPI = (CSI×0.3) + (TS_ratio×20) + (100-CPM)×0.3 + (Spread×0.2)            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: Psychosocial Assessment (30-45 min)                                │
├─────────────────────────────────────────────────────────────────────────────┤
│ PAIN COGNITIONS:                                                            │
│ □ Pain Catastrophizing Scale (PCS)                                         │
│   • Score ≥30: Clinically significant catastrophizing                      │
│                                                                             │
│ FEAR-AVOIDANCE:                                                             │
│ □ Tampa Scale of Kinesiophobia (TSK)                                       │
│   • Score ≥37: High fear of movement                                       │
│ □ Fear-Avoidance Beliefs Questionnaire (FABQ)                              │
│                                                                             │
│ MOOD & ANXIETY:                                                             │
│ □ PHQ-9 (Depression)                                                        │
│ □ GAD-7 (Anxiety)                                                           │
│                                                                             │
│ SLEEP:                                                                      │
│ □ Insomnia Severity Index (ISI)                                            │
│ □ Pain-sleep interference                                                   │
│                                                                             │
│ SELF-EFFICACY:                                                              │
│ □ Pain Self-Efficacy Questionnaire (PSEQ)                                  │
│                                                                             │
│ QUALITY OF LIFE:                                                            │
│ □ Brief Pain Inventory (BPI) - Interference subscale                       │
│ □ SF-36 or EQ-5D                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ STEP 4: Opioid Risk Assessment (if applicable)                             │
├─────────────────────────────────────────────────────────────────────────────┤
│ □ Current opioid use (MME calculation)                                     │
│ □ Opioid Risk Tool (ORT)                                                   │
│ □ Duration of opioid use                                                    │
│ □ Signs of opioid-induced hyperalgesia                                     │
│ □ Urine drug screen (if indicated)                                         │
│ □ Prescription Drug Monitoring Program check                               │
│                                                                             │
│ TAPERING READINESS:                                                         │
│ □ Patient motivation                                                        │
│ □ Support system                                                            │
│ □ Alternative pain management in place                                      │
│ □ Mental health stability                                                   │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2.2 Neuroplasticity Reversal Index (NRI)

```
PROTOCOL: WIA-CP-NRI-001
Name: Neuroplasticity Reversal Index Calculation

┌─────────────────────────────────────────────────────────────────────────────┐
│                     NRI CALCULATION                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  NRI = 100 - [(CS×0.35) + (PSY×0.25) + (STRUCT×0.25) + (DUR×0.15)]         │
│                                                                             │
│  Where:                                                                     │
│  • CS = Central Sensitization score (0-100, from CSI + QST)                │
│  • PSY = Psychosocial score (0-100, from PCS + TSK + PHQ + GAD)            │
│  • STRUCT = Structural changes (0-100, from neuroimaging if available)     │
│  • DUR = Duration factor (0-100, based on chronicity)                      │
│                                                                             │
│  Interpretation:                                                            │
│  80-100: Excellent reversal potential (early intervention window)          │
│  60-79:  Good reversal potential (standard protocols)                      │
│  40-59:  Moderate potential (intensive intervention needed)                │
│  20-39:  Limited potential (prolonged multimodal approach)                 │
│  0-19:   Challenging (focus on function over cure)                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 3. Treatment Protocols

### 3.1 Non-Invasive Neuromodulation Protocol

```
PROTOCOL: WIA-CP-NEUROMOD-001
Name: Non-Invasive Brain Stimulation for Chronic Pain
Indication: Central sensitization with NRI 40-80

┌─────────────────────────────────────────────────────────────────────────────┐
│ MODALITY SELECTION                                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 1. rTMS (repetitive Transcranial Magnetic Stimulation)                     │
│    Target: Primary Motor Cortex (M1) or Dorsolateral PFC (DLPFC)           │
│    Indication: Neuropathic pain, fibromyalgia, CRPS                         │
│    ┌─────────────────────────────────────────────────────────────┐         │
│    │ Parameters:                                                 │         │
│    │ • Frequency: 10 Hz (high frequency)                         │         │
│    │ • Intensity: 80-120% motor threshold                        │         │
│    │ • Pulses: 1500-3000 per session                             │         │
│    │ • Sessions: 10-20 (daily for 2-4 weeks)                     │         │
│    │ • Maintenance: Weekly for 1-3 months                        │         │
│    └─────────────────────────────────────────────────────────────┘         │
│                                                                             │
│ 2. tDCS (transcranial Direct Current Stimulation)                          │
│    Target: Motor cortex (anodal) or DLPFC                                   │
│    Indication: Fibromyalgia, low back pain, post-stroke pain               │
│    ┌─────────────────────────────────────────────────────────────┐         │
│    │ Parameters:                                                 │         │
│    │ • Current: 2 mA                                             │         │
│    │ • Duration: 20-30 minutes                                   │         │
│    │ • Sessions: 10-20 (daily or every other day)                │         │
│    │ • Home-based option available (after training)              │         │
│    └─────────────────────────────────────────────────────────────┘         │
│                                                                             │
│ 3. Focused Ultrasound (emerging)                                           │
│    Target: Thalamus, ACC, insula                                           │
│    Indication: Refractory pain, research protocols                         │
│                                                                             │
│ 4. TENS (Transcutaneous Electrical Nerve Stimulation)                      │
│    Target: Peripheral nerves / spinal gate                                 │
│    Indication: Localized pain, adjunct therapy                             │
│    ┌─────────────────────────────────────────────────────────────┐         │
│    │ Parameters:                                                 │         │
│    │ • High frequency: 80-100 Hz (sensory modulation)            │         │
│    │ • Low frequency: 2-4 Hz (endorphin release)                 │         │
│    │ • Duration: 30-60 minutes, multiple times daily             │         │
│    └─────────────────────────────────────────────────────────────┘         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ CONTRAINDICATIONS                                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│ Absolute for rTMS:                                                          │
│ □ Metallic implants in head (except dental)                                │
│ □ Cochlear implants                                                         │
│ □ History of seizures (relative for some protocols)                         │
│ □ Cardiac pacemaker/ICD near stimulation site                              │
│                                                                             │
│ Relative:                                                                   │
│ □ Pregnancy (first trimester)                                              │
│ □ Severe cardiac disease                                                    │
│ □ Active substance use disorder                                             │
│ □ Severe psychiatric instability                                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ MONITORING & RESPONSE                                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│ Before each session:                                                        │
│ □ NRS pain score                                                            │
│ □ Adverse event screening                                                   │
│ □ Mood assessment (brief)                                                   │
│                                                                             │
│ Weekly:                                                                     │
│ □ BPI interference                                                          │
│ □ Global impression of change                                               │
│                                                                             │
│ End of treatment course:                                                    │
│ □ Full reassessment (NRI recalculation)                                    │
│ □ QST repeat (if baseline abnormal)                                        │
│                                                                             │
│ Response criteria:                                                          │
│ • Responder: ≥30% pain reduction                                           │
│ • Good responder: ≥50% pain reduction                                      │
│ • Excellent responder: ≥70% pain reduction + function improvement          │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3.3 Exercise Prescription Protocol

```
PROTOCOL: WIA-CP-EXERCISE-001
Name: Graded Exercise Therapy for Chronic Pain
Goal: Restore function, reverse deconditioning, reduce central sensitization

┌─────────────────────────────────────────────────────────────────────────────┐
│ PHASE 1: BASELINE & GRADUAL INTRODUCTION (Weeks 1-4)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ ASSESSMENT:                                                                 │
│ □ Current activity level (accelerometer or diary)                          │
│ □ Fear-avoidance beliefs                                                    │
│ □ Specific feared activities                                                │
│ □ Physical capacity testing (6MWT, sit-to-stand)                           │
│                                                                             │
│ PRESCRIPTION:                                                               │
│ □ Start at 50-70% of current tolerance                                     │
│ □ Focus on time-based, not pain-based                                      │
│ □ Low-intensity aerobic: Walking, swimming, cycling                        │
│ □ Duration: 10-15 minutes, 3x/week                                         │
│ □ Intensity: RPE 2-3/10 (light)                                            │
│                                                                             │
│ KEY PRINCIPLES:                                                             │
│ • Do NOT stop when pain increases (unless safety concern)                  │
│ • Consistency over intensity                                                │
│ • Exercise through mild pain is SAFE                                        │
│ • Pain ≠ harm (reinforce pain neuroscience)                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ PHASE 2: PROGRESSIVE LOADING (Weeks 5-12)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ PROGRESSION:                                                                │
│ □ Increase duration by 5-10% per week                                      │
│ □ Target: 30 minutes moderate intensity, 5x/week                           │
│ □ Add resistance training 2x/week                                          │
│                                                                             │
│ AEROBIC:                                                                    │
│ □ Duration: 20-30 minutes                                                   │
│ □ Intensity: RPE 4-6/10 (moderate)                                         │
│ □ Types: Walking, swimming, cycling, elliptical                            │
│                                                                             │
│ RESISTANCE:                                                                 │
│ □ Major muscle groups                                                       │

│ □ Bodyweight or light resistance                                            │
│ □ Focus on movement quality                                                 │
│                                                                             │
│ FLEXIBILITY & MOBILITY:                                                     │
│ □ Daily stretching routine                                                  │
│ □ Yoga or tai chi 1-2x/week                                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │

│                                                                             │
```
