Chapter 3: Tribute Systems

WIA Digital Memorial Standard - Interactive Condolences & Memory Sharing

3.1 Tribute System Overview

Tribute systems enable family, friends, and community members to express condolences, share memories, and celebrate the life of the deceased. These interactive features create a collaborative memorial that grows over time with contributions from multiple participants.

A comprehensive tribute system includes condolence messages, memory sharing, story collaboration, and virtual ceremonies, all while maintaining appropriate moderation and emotional sensitivity.

Core Features

  • Guestbook for condolence messages
  • Memory wall for shared stories and anecdotes
  • Photo/video tribute submissions
  • Virtual candle lighting ceremonies
  • Memorial donation integration
  • Social media sharing capabilities
  • Moderation and content filtering
  • Notification system for family members
  • Multi-language tribute support
  • Anonymous tribute options

Submit a new tribute to a memorial

Get all tributes for a memorial (paginated)

Get specific tribute details

Update tribute (author only, within 24 hours)

Delete tribute (author or memorial admin)

Like/appreciate a tribute

POST /api/v1/tributes/{tributeId}/reply

Reply to a tribute

POST /api/v1/tributes/{tributeId}/flag

Flag tribute for review

PATCH /api/v1/tributes/{tributeId}/moderate

Approve/reject tribute (moderator only)

GET /api/v1/memorials/{memorialId}/tributes/pending

Get pending tributes awaiting moderation

POST /api/v1/tributes/candle

Light a virtual candle

GET /api/v1/memorials/{memorialId}/candles

Get candle lighting statistics

POST /api/v1/tributes/batch-approve

Batch approve multiple tributes

GET /api/v1/tributes/search

Search tributes by keyword or author

Example 1: Tribute Submission Form

React - Tribute Form Component
import React, { useState } from 'react';
import axios from 'axios';

interface TributeFormData {
  type: 'condolence' | 'memory' | 'photo' | 'candle';
  authorName: string;
  email: string;
  relationship: string;
  title?: string;
  text: string;
  isAnonymous: boolean;
  emotion?: string;
}

const TributeForm: React.FC<{ memorialId: string }> = ({ memorialId }) => {
  const [formData, setFormData] = useState<TributeFormData>({
    type: 'condolence',
    authorName: '',
    email: '',
    relationship: 'friend',
    text: '',
    isAnonymous: false
  });

  const [submitting, setSubmitting] = useState(false);
  const [submitted, setSubmitted] = useState(false);

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    setSubmitting(true);

    try {
      const response = await axios.post(
        'https://api.wia-memorial.org/v1/tributes',
        {
          memorialId,
          type: formData.type,
          author: {
            name: formData.authorName,
            email: formData.email,
            relationship: formData.relationship,
            isAnonymous: formData.isAnonymous
          },
          content: {
            title: formData.title,
            text: formData.text,
            emotion: formData.emotion
          }
        },
        {
          headers: {
            'Content-Type': 'application/json'
          }
        }
      );

      if (response.data.moderation?.status === 'pending') {
        alert('Your tribute has been submitted and is awaiting approval.');
      } else {
        alert('Your tribute has been published successfully.');
      }

      setSubmitted(true);

    } catch (error) {
      console.error('Error submitting tribute:', error);
      alert('Failed to submit tribute. Please try again.');
    } finally {
      setSubmitting(false);
    }
  };

  if (submitted) {
    return (
      <div className="tribute-success">
        <h3>Thank you for your tribute</h3>
        <p>Your message has been received and will be displayed on the memorial.</p>
      </div>
    );
  }

  return (
    <form className="tribute-form" onSubmit={handleSubmit}>
      <h3>Leave a Tribute</h3>

      <div className="form-group">
        <label>Tribute Type*</label>
        <select
          value={formData.type}
          onChange={(e) => setFormData({...formData, type: e.target.value as any})}
          required
        >
          <option value="condolence">Condolence Message</option>
          <option value="memory">Share a Memory</option>
          <option value="photo">Photo Tribute</option>
          <option value="candle">Light a Candle</option>
        </select>
      </div>

      <div className="form-row">
        <div className="form-group">
          <label>Your Name*</label>
          <input
            type="text"
            value={formData.authorName}
            onChange={(e) => setFormData({...formData, authorName: e.target.value})}
            required
            disabled={formData.isAnonymous}
          />
        </div>

        <div className="form-group">
          <label>Email*</label>
          <input
            type="email"
            value={formData.email}
            onChange={(e) => setFormData({...formData, email: e.target.value})}
            required
          />
        </div>
      </div>

      <div className="form-group">
        <label>Relationship*</label>
        <select
          value={formData.relationship}
          onChange={(e) => setFormData({...formData, relationship: e.target.value})}
          required
        >
          <option value="family">Family</option>
          <option value="friend">Friend</option>
          <option value="colleague">Colleague</option>
          <option value="neighbor">Neighbor</option>
          <option value="acquaintance">Acquaintance</option>
          <option value="other">Other</option>
        </select>
      </div>

      {formData.type === 'memory' && (
        <div className="form-group">
          <label>Memory Title</label>
          <input
            type="text"
            value={formData.title || ''}
            onChange={(e) => setFormData({...formData, title: e.target.value})}
            placeholder="A meaningful moment..."
          />
        </div>
      )}

      <div className="form-group">
        <label>Your Message*</label>
        <textarea
          value={formData.text}
          onChange={(e) => setFormData({...formData, text: e.target.value})}
          rows={6}
          required
          maxLength={formData.type === 'memory' ? 5000 : 500}
          placeholder="Share your thoughts, memories, or condolences..."
        />
        <small>
          {formData.text.length} / {formData.type === 'memory' ? 5000 : 500} characters
        </small>
      </div>

      <div className="form-group">
        <label className="checkbox-label">
          <input
            type="checkbox"
            checked={formData.isAnonymous}
            onChange={(e) => setFormData({...formData, isAnonymous: e.target.checked})}
          />
          Post anonymously
        </label>
      </div>

      <button type="submit" disabled={submitting}>
        {submitting ? 'Submitting...' : 'Submit Tribute'}
      </button>
    </form>
  );
};

export default TributeForm;

Example 2: Tribute Moderation System

Python - AI-Powered Content Moderation
import openai
import re
from typing import Dict, List, Tuple
from dataclasses import dataclass

@dataclass
class ModerationResult:
    is_appropriate: bool
    score: float
    flags: List[str]
    suggested_action: str
    reason: str

class TributeModerationService:
    def __init__(self, api_key: str):
        self.openai_client = openai.OpenAI(api_key=api_key)
        self.profanity_list = self.load_profanity_list()

    def load_profanity_list(self) -> List[str]:
        """Load profanity filter list"""
        # In production, load from database or file
        return ['badword1', 'badword2', 'offensive_term']

    def moderate_tribute(self, tribute_text: str, author_info: Dict) -> ModerationResult:
        """Comprehensive tribute moderation"""

        # Basic checks
        if len(tribute_text.strip()) < 10:
            return ModerationResult(
                is_appropriate=False,
                score=0.0,
                flags=['too_short'],
                suggested_action='reject',
                reason='Tribute text is too short'
            )

        # Profanity check
        profanity_score = self.check_profanity(tribute_text)

        # Spam detection
        spam_score = self.detect_spam(tribute_text)

        # AI content analysis
        ai_result = self.ai_content_analysis(tribute_text)

        # Combine scores
        flags = []
        if profanity_score > 0.3:
            flags.append('profanity')
        if spam_score > 0.5:
            flags.append('spam')
        if ai_result['toxicity'] > 0.7:
            flags.append('toxic')
        if ai_result['inappropriate'] > 0.6:
            flags.append('inappropriate')

        # Calculate final score (0-1, higher is better)
        final_score = 1.0 - max(
            profanity_score,
            spam_score,
            ai_result['toxicity'],
            ai_result['inappropriate']
        )

        # Determine action
        if final_score >= 0.8:
            action = 'approve'
            is_appropriate = True
            reason = 'Content passed all moderation checks'
        elif final_score >= 0.5:
            action = 'review'
            is_appropriate = False
            reason = 'Content requires human review'
        else:
            action = 'reject'
            is_appropriate = False
            reason = f'Content flagged: {", ".join(flags)}'

        return ModerationResult(
            is_appropriate=is_appropriate,
            score=final_score,
            flags=flags,
            suggested_action=action,
            reason=reason
        )

    def check_profanity(self, text: str) -> float:
        """Check for profanity and inappropriate language"""
        text_lower = text.lower()
        matches = sum(1 for word in self.profanity_list if word in text_lower)
        return min(matches * 0.3, 1.0)

    def detect_spam(self, text: str) -> float:
        """Detect spam patterns"""
        spam_indicators = 0

        # Check for excessive URLs
        urls = re.findall(r'http[s]?://(?:[a-zA-Z]|[0-9]|[$-_@.&+]|[!*\\(\\),]|(?:%[0-9a-fA-F][0-9a-fA-F]))+', text)
        if len(urls) > 2:
            spam_indicators += 1

        # Check for excessive capitalization
        capital_ratio = sum(1 for c in text if c.isupper()) / max(len(text), 1)
        if capital_ratio > 0.5:
            spam_indicators += 1

        # Check for repeated characters
        if re.search(r'(.)\1{4,}', text):
            spam_indicators += 1

        # Check for excessive emoji
        emoji_count = len(re.findall(r'[\U0001F300-\U0001F9FF]', text))
        if emoji_count > 10:
            spam_indicators += 1

        return min(spam_indicators * 0.25, 1.0)

    def ai_content_analysis(self, text: str) -> Dict[str, float]:
        """Use AI to analyze content appropriateness"""

        try:
            response = self.openai_client.moderations.create(input=text)
            result = response.results[0]

            return {
                'toxicity': result.category_scores.get('hate', 0.0),
                'inappropriate': result.category_scores.get('harassment', 0.0),
                'violence': result.category_scores.get('violence', 0.0),
                'sexual': result.category_scores.get('sexual', 0.0)
            }
        except Exception as e:
            print(f"AI moderation error: {e}")
            return {
                'toxicity': 0.0,
                'inappropriate': 0.0,
                'violence': 0.0,
                'sexual': 0.0
            }

    def batch_moderate(self, tributes: List[Dict]) -> List[ModerationResult]:
        """Moderate multiple tributes"""
        return [
            self.moderate_tribute(t['content']['text'], t['author'])
            for t in tributes
        ]

Example 3: Virtual Candle Lighting

JavaScript - Interactive Candle Feature
class VirtualCandleSystem {
  constructor(memorialId) {
    this.memorialId = memorialId;
    this.apiBase = 'https://api.wia-memorial.org/v1';
    this.candleCanvas = document.getElementById('candle-canvas');
    this.ctx = this.candleCanvas.getContext('2d');
    this.candles = [];
    this.init();
  }

  async init() {
    await this.loadCandles();
    this.render();
    this.startAnimation();
  }

  async loadCandles() {
    try {
      const response = await fetch(
        `${this.apiBase}/memorials/${this.memorialId}/candles`
      );
      const data = await response.json();
      this.candles = data.candles || [];
    } catch (error) {
      console.error('Error loading candles:', error);
    }
  }

  async lightCandle(authorName, message) {
    try {
      const response = await fetch(`${this.apiBase}/tributes/candle`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          memorialId: this.memorialId,
          author: {
            name: authorName
          },
          content: {
            text: message,
            emotion: 'remembrance'
          },
          type: 'candle'
        })
      });

      const result = await response.json();

      // Add candle to display
      this.candles.push({
        id: result.tributeId,
        author: authorName,
        message: message,
        lit: Date.now(),
        x: Math.random() * this.candleCanvas.width,
        y: Math.random() * this.candleCanvas.height,
        flicker: 0
      });

      this.render();
      return result;

    } catch (error) {
      console.error('Error lighting candle:', error);
      throw error;
    }
  }

  render() {
    this.ctx.clearRect(0, 0, this.candleCanvas.width, this.candleCanvas.height);

    this.candles.forEach((candle, index) => {
      this.drawCandle(candle, index);
    });

    // Display total count
    this.ctx.fillStyle = '#64748B';
    this.ctx.font = '24px Arial';
    this.ctx.fillText(
      `${this.candles.length} candles lit`,
      20,
      40
    );
  }

  drawCandle(candle, index) {
    const { x, y, flicker } = candle;

    // Draw candle body
    this.ctx.fillStyle = '#F3F4F6';
    this.ctx.fillRect(x - 5, y + 20, 10, 30);

    // Draw flame
    const flameHeight = 20 + Math.sin(flicker) * 3;
    const flameWidth = 8 + Math.sin(flicker * 1.5) * 2;

    const gradient = this.ctx.createRadialGradient(x, y, 0, x, y, flameWidth);
    gradient.addColorStop(0, '#FFF');
    gradient.addColorStop(0.5, '#FFD700');
    gradient.addColorStop(1, '#FF8C00');

    this.ctx.fillStyle = gradient;
    this.ctx.beginPath();
    this.ctx.ellipse(x, y, flameWidth, flameHeight, 0, 0, Math.PI * 2);
    this.ctx.fill();

    // Draw glow
    this.ctx.shadowBlur = 20;
    this.ctx.shadowColor = 'rgba(255, 200, 0, 0.6)';
    this.ctx.fillStyle = 'rgba(255, 200, 0, 0.3)';
    this.ctx.beginPath();
    this.ctx.arc(x, y, 25, 0, Math.PI * 2);
    this.ctx.fill();
    this.ctx.shadowBlur = 0;
  }

  startAnimation() {
    const animate = () => {
      this.candles.forEach(candle => {
        candle.flicker += 0.1;
      });
      this.render();
      requestAnimationFrame(animate);
    };
    animate();
  }

  getCandleStats() {
    const now = Date.now();
    const last24h = this.candles.filter(c => now - c.lit < 24 * 60 * 60 * 1000);
    const last7d = this.candles.filter(c => now - c.lit < 7 * 24 * 60 * 60 * 1000);

    return {
      total: this.candles.length,
      last24Hours: last24h.length,
      last7Days: last7d.length,
      averagePerDay: last7d.length / 7
    };
  }
}

To achieve WIA Digital Memorial Standard certification for tribute systems:

  • Support minimum 4 tribute types (condolence, memory, photo, video)
  • Implement content moderation system with AI filtering
  • Provide manual moderation interface for administrators
  • Support anonymous tribute submissions
  • Implement tribute reply/comment functionality
  • Provide tribute like/appreciation feature
  • Support virtual candle lighting with visualization
  • Implement real-time notification system for new tributes
  • Provide tribute search and filtering capabilities
  • Support multiple moderation levels (open, filtered, strict)
  • Implement spam detection and prevention
  • Provide profanity filtering with customizable word lists
  • Support tribute editing within 24 hours of submission
  • Implement tribute flagging system for inappropriate content
  • Provide batch moderation tools for administrators
  • Support tribute scheduling for future publication
  • Implement tribute analytics (views, engagement)
  • Provide email notifications to memorial managers
  • Support multi-language tributes with translation
  • Implement tribute export functionality (PDF, CSV)
  • Provide tribute wall display with customizable layouts
  • Support tribute sharing on social media
  • Implement tribute pinning for featured messages
