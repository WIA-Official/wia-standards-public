제7장: 국제 조화

"홍익인간: 복지는 국경을 알지 못합니다; 우리의 표준은 어디에서든 동물에게 봉사해야 합니다."

7.1 글로벌 복지 현황

주요 복지 프레임워크
====================

다섯 가지 자유 (FIVE FREEDOMS) (글로벌)
  기원: 영국 농장동물복지위원회, 1979
  채택: 전 세계 기준 표준
  범위: 모든 동물 유형
  상태: WIA 표준의 기초

세계동물보건기구 (WOAH)
  이전명: OIE
  회원: 182개국
  범위: 육상 및 수생 동물
  표준: 국제 무역 초점

유럽연합
  프레임워크: EU 동물복지 전략
  범위: 27개 회원국
  표준: 법적 구속력 있는 규정
  초점: 농장 및 반려동물

미국
  연방: 동물복지법 (AWA)
  범위: 제한적 (연구, 판매상, 전시자)
  주: 상당히 다양함
  초점: 최소 기준

세계동물보호협회
  유형: NGO
  범위: 글로벌 옹호
  표준: 동물보호지수
  초점: 법적 및 정책 개혁


지역별 변동
===========

높은 복지 기준:
  - 북유럽 (스웨덴, 노르웨이, 덴마크)
  - 영국
  - 스위스
  - 호주, 뉴질랜드
  - 캐나다 일부

발전 중인 기준:
  - 한국 (빠르게 개선 중)
  - 일본 (인식 증가)
  - 남미 일부
  - 도시 인도

가변적 기준:
  - 미국 (주에 따라)
  - 중국 (도시 vs 농촌)
  - 동남아시아
  - 아프리카 일부

7.2 표준 매핑

WIA와 지역 표준 매핑
====================

목적:
  - 동등성 식별
  - 격차 인식
  - 상호 인정 활성화
  - 국제 무역 촉진

매핑 과정:
  1. 지역 표준 요건 식별
  2. WIA 요건과 비교
  3. 동등성 수준 평가
  4. 격차 문서화
  5. 연결 요건 정의


동등성 수준
===========

동등:
  - 지역 요건이 WIA 충족 또는 초과
  - 추가 요건 불필요
  - 직접 인정 가능

실질적 동등:
  - 지역 요건이 대부분 WIA 충족
  - 경미한 격차 식별
  - 연결 요건 정의

부분적 동등:
  - 상당한 격차 존재
  - 보완 평가 필요
  - 추가 인증 필요

비동등:
  - 근본적 차이
  - 전체 WIA 평가 필요
  - 인정 불가
지역 표준 동등성 격차
EU EU 동물복지 지침 실질적 경미한 문서
영국 동물복지법 2006 실질적 평가 방법론
미국 AWA + 주법 부분적 범위, 일관성
호주 동물학대방지법 실질적 운송 기준
캐나다 NFACC 코드 실질적 집행 메커니즘
일본 동물애호관리법 부분적 범위, 평가

7.3 상호 인정

상호 인정 협정
==============

정의:
  WIA와 지역 기관 간 상대방의
  인증을 동등하게 인정하는 협정

혜택:
  - 중복 평가 감소
  - 기관 비용 절감
  - 빠른 인증 과정
  - 조화된 복지 표준

MRA 요건:
  - 실질적 동등성 매핑된 표준
  - 비교 가능한 감사 품질
  - 정보 공유 협정
  - 분쟁 해결 메커니즘


MRA 과정
========

단계 1: 탐색적 논의
  - 기관 간 초기 연락
  - 관심 및 타당성 평가
  - 자원 약속

단계 2: 기술 매핑
  - 상세 표준 비교
  - 격차 분석
  - 연결 요건 식별

단계 3: 시스템 평가
  - 인증 기관 검토
  - 감사원 역량 평가
  - 품질 시스템 평가

단계 4: 파일럿 단계
  - 제한된 상호 인정
  - 모니터링 및 평가
  - 필요에 따라 조정

단계 5: 전체 협정
  - 공식 MRA 서명
  - 공개 발표
  - 구현 안내

단계 6: 지속적 관리
  - 연간 검토 회의
  - 성과 모니터링
  - 협정 업데이트

7.4 국경 간 인증

국경 간 인증 과정
=================

시나리오: A국에서 인증받은 기관이 B국에서 운영

상호 인정이 있는 경우:
========================

단계 1: 원천 인증 확인
  - 유효한 WIA 인증 확인
  - 등급 확인
  - 정지 없음 확인

단계 2: MRA 상태 확인
  - 대상국과 MRA 존재 확인
  - 연결 요건 식별

단계 3: 이전 신청
  - 이전 신청서 제출
  - 행정 수수료 납부
  - 문서 제공

단계 4: 이전 인증서 발행
  - 이전 인증서 생성
  - 동일 등급 인정
  - 동일 만료일

소요 시간: 5-10 영업일


상호 인정이 없는 경우:
========================

단계 1: 대상 표준 식별
  - 현지 요건 조사
  - 인증 기관 식별
  - 과정 이해

단계 2: 격차 평가
  - 현재 관행과 대상 비교
  - 추가 요건 식별
  - 보완 평가 준비

단계 3: 인증 신청
  - 현지 기관에 전체 신청
  - WIA 인증 증거 제공
  - 범위 축소 감사 가능

단계 4: 보완 평가
  - 격차 영역 초점
  - 가능한 경우 기존 증거 사용
  - 추가 요건 완료

단계 5: 현지 인증
  - 별도 인증서 발행
  - 현지 등급 할당
  - 현지 갱신 주기

소요 시간: 30-90일


국제 이전 인증서
================

인증서 내용:
  - 원천 인증 세부사항
  - 원천 인증 기관
  - 이전 목적지
  - 동등성 진술
  - 유효 기간
  - 확인 코드

확인:
  - 온라인 확인 포털
  - QR 코드 확인
  - 직접 문의 옵션

7.5 지역별 변형

허용되는 지역별 변형
====================

핵심 요건 (변형 불가):
  - 다섯 가지 자유 프레임워크
  - 4개 인증 등급
  - 감사 및 준수 과정
  - 민원 메커니즘
  - 공개 투명성

변형 가능 요소:
  - 종별 기준 (현지 종)
  - 기후 관련 요건
  - 언어 및 문서화
  - 현지 법적 통합
  - 지역 기관 유형


지역 적용 예시
==============

온도 기준:
  북유럽 지역:
    - 더 높은 내한성 예상
    - 겨울 야외 접근: 품종별
    - 보호소 난방 초점

  열대 지역:
    - 냉방 요건 강조
    - 그늘과 환기 중요
    - 열 스트레스 프로토콜

종 범위:
  동아시아:
    - 특정 기준: 진돗개, 시바이누
    - 이국 반려동물 기준 (더 일반적인 종)

  호주:
    - 토착종 상호작용 고려
    - 특정 운송 요건 (거리)

법적 통합:
  EU 회원국:
    - EU 규정 참조
    - 국가별 이행 변동
    - 국경 간 이동 규칙

  미국:
    - 연방 AWA 참조
    - 주별 요건
    - 지방 조례 고려


지역 변형 요청
==============

과정:
  1. 변형 필요성 식별
  2. 정당성 포함 제안 제출
  3. 기술 위원회 검토
  4. 이해관계자 협의
  5. 승인 또는 거부
  6. 승인 시 공표

승인 기준:
  - 복지 결과 저하 없음
  - 과학적 정당성
  - 문화적 적절성
  - 실제 구현 가능
  - 무역 장벽 미생성
