제2장: 다섯 가지 자유 평가

"홍익인간: 다섯 가지 자유는 우리가 돌보는 모든 생명체의 기본권을 명시합니다."

2.1 다섯 가지 자유

기원과 발전
===========

다섯 가지 자유는 1965년 브램벨 보고서에서 파생되어
1979년 영국 농장동물복지위원회에서 개발되었습니다.
이후 동물복지 평가를 위한 글로벌 표준이 되었습니다.

다섯 가지 자유:
  1. 굶주림과 갈증으로부터의 자유
  2. 불편함으로부터의 자유
  3. 고통, 부상, 질병으로부터의 자유
  4. 정상적 행동 표현의 자유
  5. 공포와 고통으로부터의 자유

반려동물을 위한 WIA 적용:
  - 종별 요건
  - 행동 풍부화 강조
  - 사회적 필요 인식
  - 정서적 웰빙 포함

2.2 굶주림과 갈증으로부터의 자유

정의
====
건강과 활력을 유지하기 위한 신선한 물과
적절한 식이에 대한 접근.

평가 지표
=========

물 가용성 (10점)
  10: 24시간 신선한 물, 다중 급수원
   8: 항상 신선한 물 이용 가능
   6: 대부분의 시간 물 이용 가능
   4: 정해진 시간에만 물 제공
   2: 불규칙한 물 접근
   0: 신뢰할 수 있는 물 접근 없음

식이 적절성 (10점)
  10: 종과 연령에 맞는 고품질
   8: 적절한 상업용 사료
   6: 충분한 영양
   4: 기본 영양 충족
   2: 영양 결핍 가능성
   0: 부적절한 영양

급식 일정 (10점)
  10: 종/연령에 최적화된 일정
   8: 일관되고 적절한 시간
   6: 규칙적인 급식
   4: 일관성 없는 시간
   2: 불규칙한 급식
   0: 급식 일정 없음

체중 상태 모니터링 (10점)
  10: 정기 평가, 기록, 행동 계획
   8: 기록 포함 정기 모니터링
   6: 주기적 모니터링
   4: 가끔 모니터링
   2: 드문 모니터링
   0: 모니터링 없음

특수 식이 요구 (10점)
  10: 완전 수용, 의료용 식이 가능
   8: 좋은 요구 수용
   6: 기본 수용
   4: 제한된 수용
   2: 부족한 수용
   0: 수용 없음

최대 점수: 50점

2.7 점수 산정 방법론

점수 계산
=========

총 가능 점수: 250점 (자유당 50점)

전체 점수 = 모든 지표 합계 / 250 * 100%

등급 기준:
  PLATINUM: >= 95% (238점 이상)
  GOLD:     >= 85% (213점 이상)
  SILVER:   >= 75% (188점 이상)
  BRONZE:   >= 60% (150점 이상)
  탈락:     < 60%  (150점 미만)


개별 자유 요건
==============

인증을 위한 자유별 최소 점수:
  PLATINUM: >= 45/50 (90%) 각 자유
  GOLD:     >= 40/50 (80%) 각 자유
  SILVER:   >= 35/50 (70%) 각 자유
  BRONZE:   >= 25/50 (50%) 각 자유

참고: 전체 점수와 개별 최소 점수 모두 충족 필요


필수 지표
=========

일부 지표는 필수 - 기준 미달 시 자동 탈락:

  - 물 가용성: 최소 4점
  - 수의 접근성: 최소 4점
  - 부상 예방: 최소 4점
  - 온도 관리: 최소 4점

필수 지표에서 2점 이하 = 탈락


평가 문서화
===========

필요 증거:
  - 시설 사진
  - 정책 및 절차 문서
  - 교육 기록
  - 건강 기록
  - 급식 로그
  - 풍부화 일정

감사원 메모:
  - 구체적인 관찰
  - 방문 시 동물 행동
  - 관찰된 직원 상호작용
  - 강점 영역
  - 개선 영역
등급 전체 점수 자유별 최소 필수 지표
PLATINUM >= 95% >= 90% 모두 >= 8
GOLD >= 85% >= 80% 모두 >= 6
SILVER >= 75% >= 70% 모두 >= 4
BRONZE >= 60% >= 50% 모두 >= 4
