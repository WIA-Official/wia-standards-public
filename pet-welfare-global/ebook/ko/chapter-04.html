4.2 감사 과정

사전 감사 단계
==============

30일 전:
  - 감사팀 배정
  - 통보 발송 (예정된 감사의 경우)
  - 문서 요청 발행

14일 전:
  - 문서 수령 및 검토
  - 이전 발견사항 검토
  - 감사 계획 초안

7일 전:
  - 최종 감사 계획 발송
  - 물류 확정
  - 질문 처리


현장 감사 단계
==============

개회 회의 (30분):
  - 소개
  - 감사 범위 및 목적
  - 일정 확인
  - 안전 브리핑
  - 질의 처리

시설 투어 (1-2시간):
  - 모든 동물 사육 구역
  - 지원 구역 (사료 준비, 보관)
  - 수의/의료 구역
  - 야외 구역
  - 비상 장비

문서 검토 (1-2시간):
  - 정책 및 절차
  - 교육 기록
  - 건강 기록
  - 급식 로그
  - 사고 보고서

동물 관찰 (1-2시간):
  - 체중 상태 평가
  - 행동 관찰
  - 취급자와의 상호작용
  - 환경 적정성
  - 무작위 샘플링

직원 면담 (30-60분):
  - 지식 확인
  - 실무 이해
  - 우려 식별
  - 교육 적정성

다섯 가지 자유 평가 (1-2시간):
  - 체계적 점수 산정
  - 증거 수집
  - 사진 문서화
  - 격차 식별

폐회 회의 (30-60분):
  - 예비 발견사항
  - 관찰 논의
  - 강점 강조
  - 부적합 설명
  - 다음 단계 안내


사후 감사 단계
==============

5일 이내:
  - 초안 보고서 기관에 전달

10일 이내:
  - 최종 보고서 발행
  - 부적합 문서화
  - 시정조치 일정 설정

30일 이내:
  - 인증 결정
  - 인증서 발행 (승인 시)
  - 공개 레지스트리 업데이트

4.3 발견사항 분류

부적합 분류
===========

중대 (CRITICAL)
  정의:
    동물 복지에 즉각적 위협
    법률 위반
    사기 행위

  예시:
    - 물 접근 없는 동물
    - 학대 관찰됨
    - 건강 기록 위조
    - 부상 유발하는 과밀

  대응:
    - 인증 불가
    - 기존 인증 정지
    - 즉시 시정조치 필요
    - 당국 연루 가능


주요 (MAJOR)
  정의:
    요건에서 상당한 이탈
    통제의 시스템적 실패
    동물 복지에 위험

  예시:
    - 부적절한 수의 관리
    - 상당한 공간 부족
    - 교육 프로그램 없음
    - 같은 영역에서 다수의 경미한 문제

  대응:
    - 30일 시정조치 필요
    - 후속 감사 필요 가능
    - 인증 결정 연기


경미 (MINOR)
  정의:
    요건에서 개별적 이탈
    동물 복지에 낮은 위험
    쉽게 교정 가능

  예시:
    - 불완전한 기록
    - 경미한 공간 부족
    - 단일 장비 문제
    - 문서화되지 않은 절차

  대응:
    - 90일 시정조치
    - 자가 교정 선언
    - 다음 감시 시 확인


관찰 (OBSERVATION)
  정의:
    개선 기회
    요건 위반 아님
    모범 사례 권고

  예시:
    - 향상된 풍부화 옵션
    - 더 나은 문서 형식
    - 프로세스 효율성
    - 직원 개발 아이디어

  대응:
    - 의무 조치 없음
    - 개선 고려
    - 향후 요건이 될 수 있음

4.4 시정 조치

시정조치 과정
=============

단계 1: 근본 원인 분석
  - 왜 부적합이 발생했나?
  - 어떤 시스템이 실패했나?
  - 개별적인가 시스템적인가?
  - 어떤 조건이 기여했나?

단계 2: 즉시 교정
  - 즉각적인 문제 중지
  - 지금 동물 복지 보호
  - 문제 억제
  - 취한 조치 문서화

단계 3: 시정조치 계획
  - 근본 원인 해결
  - 재발 방지
  - 구체적 조치 정의
  - 책임 할당
  - 현실적 일정 설정

단계 4: 이행
  - 계획 실행
  - 진행 문서화
  - 증거 수집
  - 장애물 해결

단계 5: 확인
  - 효과 자가 확인
  - 증거 수집
  - 인증 기관에 제출
  - 필요시 후속 감사 수락


시정조치 일정
=============

중대:
  즉시: 문제 억제
  24시간: 조치 계획 제출
  7일: 중대 사항 해결
  30일: 완전 해결

주요:
  7일: 조치 계획 제출
  30일: 해결 기한
  45일: 후속 확인

경미:
  14일: 조치 계획 제출
  90일: 해결 기한
  다음 감사: 확인
발견사항 유형 계획 기한 해결 기한 확인
중대 24시간 7-30일 후속 감사
주요 7일 30일 후속 감사
경미 14일 90일 다음 감시
관찰 해당 없음 해당 없음 선택적 검토

4.5 인증 결정

결정 기준
=========

인증 승인:
  - 다섯 가지 자유 점수가 등급 기준 충족
  - 모든 개별 자유 최소점 충족
  - 중대 부적합 없음
  - 주요 부적합: 조치 계획과 함께 최대 2개
  - 경미 부적합: 계획과 함께 제한 없음

조건부 승인:
  - 점수가 기준 충족
  - 주요 부적합 해결 중
  - 후속 감사 예정
  - 시간 제한 인증 가능

결정 연기:
  - 증거 불충분
  - 주요 부적합 해결 필요
  - 후속 감사 필요
  - 90일 이내 결정

인증 거부:
  - 중대 부적합 발견
  - 점수 기준 미달
  - 개별 자유 최소 미달
  - 사기 또는 기만 식별
  - 법적 미준수


결정 위원회
===========

구성:
  - 기술 전문가 (복지)
  - 수석 감사원 (감사에서 제외된)
  - 품질 관리자
  - 외부 자문 (필요시)

절차:
  1. 감사 보고서 검토
  2. 발견사항 논의
  3. 기관 응답 고려
  4. 결정 도출
  5. 근거 문서화

일정:
  - 표준: 감사 후 30일 이내
  - 복잡한 경우: 60일 이내

4.6 이의제기 절차

이의제기 권리
=============

이의 가능한 결정:
  - 인증 거부
  - 인증 정지
  - 인증 취소
  - 등급 하향
  - 부적합 분류

이의 불가:
  - 감사 일정
  - 감사원 배정
  - 관찰
  - 비용 결정


이의제기 과정
=============

단계 1: 이의 통보 (결정 후 30일)
  - 서면 이의 제출
  - 이의 근거 명시
  - 지지 증거 제공
  - 이의 비용 납부 (승소 시 환급)

단계 2: 이의 검토 (14일)
  - 완전성 점검
  - 유효성 평가
  - 필요시 추가 정보 요청

단계 3: 이의 패널 (30일)
  - 독립 패널 구성
  - 원본 자료 검토
  - 이의 신청자 제출 고려
  - 청문회 실시 (요청 시)

단계 4: 이의 결정 (패널 후 14일)
  - 서면 결정 발행
  - 사유 제공
  - 결정은 최종

결과:
  - 이의 인용: 원 결정 번복
  - 이의 부분 인용: 수정된 결정
  - 이의 기각: 원 결정 유지


이의 패널 구성
==============

최소 3인:
  - 독립 의장
  - 복지 전문가 (외부)
  - 업계 대표 (이해충돌 없음)

배제:
  - 원 결정 관여자
  - 이해충돌 있는 자
  - 이의 신청자와 같은 조직 소속
