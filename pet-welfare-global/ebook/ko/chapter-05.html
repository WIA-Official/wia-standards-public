제5장: 민원 및 조사

"홍익인간: 동물복지를 위해 제기된 모든 목소리는 경청될 자격이 있습니다."

5.1 민원 채널

이용 가능한 채널
================

온라인 포털
  - 24시간 이용 가능
  - 익명 옵션
  - 파일 업로드 지원
  - 추적 번호 제공
  - 상태 업데이트 가능

전화 핫라인
  - 운영 시간: 현지 시간 오전 8시-오후 8시
  - 시간 외 음성 메일
  - 통역 서비스 가능
  - 통화 참조 번호 제공

이메일
  - 전용 민원 주소
  - 24시간 이내 확인
  - 첨부 파일 보안
  - 기록 유지

우편
  - 우편 주소 제공
  - 기밀 취급
  - 물리적 증거용
  - 우편 또는 전화로 응답

직접 방문
  - 인증 기관 사무실
  - 예약 필요
  - 즉시 문서화
  - 지원 가능


접근성
======

언어:
  - 항상 현지어
  - 영어 보조
  - 지역 필요에 따라 추가

형식:
  - 웹: 화면 판독기 호환
  - 전화: 청각 장애 옵션
  - 우편: 큰 글씨 가능

지원:
  - 양식 작성 도움
  - 과정 안내
  - 법적 대리 불필요

5.2 민원 분류

유형 설명 우선순위
PHYSICAL_ABUSE 동물에 대한 고의적 해 긴급
NEGLECT 기본 돌봄 제공 실패 긴급
LIVING_CONDITIONS 부적절한 사육/환경 높음
INADEQUATE_CARE 불충분한 건강/급식 관리 높음
ILLEGAL_BREEDING 무면허 또는 금지된 관행 높음
TRANSPORT_VIOLATIONS 동물 운송 우려 중간
DOCUMENTATION_FRAUD 위조된 기록 또는 인증서 높음
STAFF_CONDUCT 부적절한 취급/행동 중간
OTHER_WELFARE 기타 복지 우려 평가됨 
우선순위별 응답 시간
====================

긴급:
  초기 검토: 4시간
  조사 시작: 24시간
  필요시 현장: 48시간

높음:
  초기 검토: 24시간
  조사 시작: 3일
  필요시 현장: 7일

중간:
  초기 검토: 3일
  조사 시작: 7일
  필요시 현장: 14일

낮음:
  초기 검토: 7일
  조사 시작: 14일
  해결 목표: 60일

5.3 조사 과정

조사 단계
=========

단계 1: 초기 평가
=================
기간: 1-3일

조치:
  - 민원 등록 및 분류
  - 우선순위 할당
  - 제보자 연락 (익명이 아닌 경우)
  - 기관 인증 상태 확인
  - 이전 민원 검토
  - 초기 증거 평가

결과:
  - 전체 조사 진행
  - 추가 정보 요청
  - 적절한 당국에 회부
  - 종결 (근거 불충분)


단계 2: 증거 수집
=================
기간: 3-14일

조치:
  - 모든 가용 증거 수집
  - 제보자 면담 (상세)
  - 증인 면담
  - 문서 확보
  - 과거 기록 검토
  - 기관 연락 준비

증거 유형:
  - 사진/영상
  - 문서/기록
  - 증인 진술
  - 전문가 의견
  - 과거 데이터


단계 3: 기관 응답
=================
기간: 7-14일

조치:
  - 기관에 민원 통보
  - 세부사항 공유 (제보자 보호)
  - 응답 요청
  - 문서 요청
  - 자체 평가 요청

기관 권리:
  - 민원의 성격 알 권리
  - 응답 및 증거 제공
  - 관련 증인 식별
  - 조사 일정 요청


단계 4: 현장 조사
=================
기간: 1-3일 (필요시)

발생 조건:
  - 증거가 심각한 위반 시사
  - 상충되는 정보
  - 제보자 신뢰성 확립
  - 기관 응답 부적절

조치:
  - 예고 없는 또는 단기 통보 방문
  - 전체 시설 점검
  - 동물복지 평가
  - 직원 면담
  - 문서 확인
  - 증거 수집


단계 5: 분석 및 결정
===================
기간: 7-14일

조치:
  - 모든 증거 평가
  - 기관 응답 고려
  - 기준 준수 평가
  - 전문가 자문 (필요시)
  - 결정 도출
  - 보고서 준비

결정 옵션:
  - 민원 입증 (조치 필요)
  - 민원 부분 입증
  - 민원 미입증
  - 증거 불충분
  - 당국에 회부

5.4 내부고발자 보호

보호 원칙
=========

기밀성
  - 전 과정에서 신원 보호
  - 알 필요가 있는 사람만 접근
  - 기관과 공유하지 않음
  - 안전한 데이터 저장

보복 금지
  - 보복 금지
  - 법적 보호 적용
  - 보복 모니터링
  - 보복 발생 시 지원

선의 보호
  - 선의의 민원 보호
  - 최종 미입증이라도 보호
  - 정직한 믿음 충분
  - 절차 준수가 핵심


직원 내부고발자
===============

특별 보호:
  - 고용 보호
  - 기밀 채널
  - 익명 보고 옵션
  - 법적 지원 정보

고용주 의무:
  - 보고로 인한 해고 금지
  - 강등 또는 불이익 금지
  - 적대적 근무 환경 금지
  - 보고된 보복 조사


익명 보고
=========

가능:
  - 온라인 포털 (익명 옵션)
  - 익명 제보 전화
  - 제3자 보고 서비스

제한:
  - 직접 업데이트 수신 불가
  - 참조 번호로 상태 확인
  - 연락 없이 일부 조사 제한
  - 편하면 연락처 제공 권장

보호:
  - IP 주소 기록 안 함
  - 등록 불필요
  - 신원 없이 증거 수락

5.5 결과 및 제재

가능한 결과
===========

민원이 입증된 경우:

수준 1: 경고
  - 공식 서면 경고
  - 시정조치 필요
  - 해결 일정
  - 인증 파일에 기록

수준 2: 강화된 감시
  - 감사 빈도 증가
  - 예고 없는 점검
  - 진행 모니터링
  - 기간: 6-12개월

수준 3: 인증 조건
  - 특정 개선 필요
  - 준수 기한
  - 후속 확인
  - 미충족 시 정지 가능

수준 4: 등급 하향
  - 인증 수준 감소
  - 공개 통보
  - 상향에 재평가 필요
  - 상향 전 최소 기간

수준 5: 정지
  - 인증 정지
  - 인증 상태 주장 불가
  - 공개 통보
  - 시정조치 필요
  - 복원 감사 필요

수준 6: 취소
  - 인증 취소
  - 공개 통보
  - 재신청 대기 기간
  - 심각한 경우 영구 가능


민원이 미입증된 경우:

  - 기관에 종결 통보
  - 제보자에 통보 (익명이 아닌 경우)
  - 기관에 조치 없음
  - 패턴 분석을 위해 파일 보관


당국에 회부된 경우:

  - 법 집행 기관 통보
  - 동물보호 기관 통보
  - 증거 적절히 공유
  - 결과 대기 중 인증 조치

5.6 공개 투명성

공개 정보
=========

항상 공개:
  - 기관 인증 상태
  - 인증 등급
  - 정지 또는 취소
  - 제재 요약 (최종 후)

공개 레지스트리 항목:
  기관: [이름]
  상태: [활성/정지/취소]
  등급: [BRONZE/SILVER/GOLD/PLATINUM]
  마지막 감사: [날짜]
  제재: [예/아니오 - 예인 경우 세부사항]


제재 레지스트리
===============

게시 정보:
  - 기관명 및 위치
  - 위반 성격 (요약)
  - 적용된 제재
  - 제재 날짜
  - 현재 상태

보관:
  - 경고: 1년
  - 강화된 감시: 2년
  - 조건: 2년
  - 하향: 3년
  - 정지: 5년
  - 취소: 영구


제보자 소통
===========

진행 업데이트:
  - 확인: 24-48시간 이내
  - 초기 평가: 7일 이내
  - 조사 업데이트: 매월
  - 최종 결과: 결정 후 7일 이내

공유 정보:
  - 민원 접수 및 분류됨
  - 조사 진행 중
  - 조사 결과
  - 취해진 조치 (요약)

공유 안 함:
  - 상세한 기관 응답
  - 내부 심의
  - 다른 제보자 정보
  - 기밀 증거
