AI 의사결정의 위험 식별 및 완화

AI 위험 평가의 중요성

AI 의사결정 시스템은 다양한 위험을 수반합니다. 편향된 결정, 개인정보 침해, 안전 문제, 규정 위반 등이 발생할 수 있습니다. 체계적인 위험 평가는 이러한 위험을 식별하고, 우선순위를 정하며, 완화 조치를 구현하는 데 필수적입니다.

위험 평가의 목표:

  • 위험 식별: AI 시스템의 잠재적 위험 발견
  • 위험 측정: 각 위험의 가능성과 영향 평가
  • 위험 우선순위: 가장 중요한 위험에 집중
  • 위험 완화: 효과적인 완화 전략 구현
  • 지속적 모니터링: 새로운 위험 발견 및 대응

위험 평가 프레임워크

체계적인 위험 평가를 위한 프레임워크:

// AI 위험 평가 프레임워크
class AIRiskAssessment {
  // 위험 평가 수행
  async assess(model: AIModel, context: DeploymentContext): Promise {
    // 1. 위험 식별
    const risks = await this.identifyRisks(model, context);

    // 2. 위험 분석
    const analyzed = await this.analyzeRisks(risks);

    // 3. 위험 평가 (가능성 x 영향)
    const evaluated = await this.evaluateRisks(analyzed);

    // 4. 위험 우선순위 결정
    const prioritized = this.prioritizeRisks(evaluated);

    // 5. 완화 전략 수립
    const mitigations = await this.developMitigations(prioritized);

    // 6. 보고서 생성
    return this.generateReport({
      model: model,
      context: context,
      risks: prioritized,
      mitigations: mitigations,
      overallRiskScore: this.calculateOverallRisk(prioritized),
      recommendations: this.generateRecommendations(prioritized, mitigations)
    });
  }

  // 위험 식별
  private async identifyRisks(
    model: AIModel,
    context: DeploymentContext
  ): Promise {
    const risks: Risk[] = [];

    // 편향 위험
    risks.push(...await this.identifyBiasRisks(model));

    // 프라이버시 위험
    risks.push(...await this.identifyPrivacyRisks(model, context));

    // 안전 위험
    risks.push(...await this.identifySafetyRisks(model, context));

    // 성능 위험
    risks.push(...await this.identifyPerformanceRisks(model));

    // 규정 준수 위험
    risks.push(...await this.identifyComplianceRisks(model, context));

    // 윤리적 위험
    risks.push(...await this.identifyEthicalRisks(model, context));

    return risks;
  }

  // 위험 분석
  private async analyzeRisks(risks: Risk[]): Promise {
    return Promise.all(
      risks.map(async risk => ({
        ...risk,
        likelihood: await this.assessLikelihood(risk),
        impact: await this.assessImpact(risk),
        detectability: await this.assessDetectability(risk),
        currentControls: await this.identifyCurrentControls(risk)
      }))
    );
  }

  // 위험 평가 (점수 계산)
  private async evaluateRisks(
    risks: AnalyzedRisk[]
  ): Promise {
    return risks.map(risk => ({
      ...risk,
      riskScore: this.calculateRiskScore(risk),
      riskLevel: this.determineRiskLevel(risk),
      residualRisk: this.calculateResidualRisk(risk)
    }));
  }

  // 위험 점수 계산
  private calculateRiskScore(risk: AnalyzedRisk): number {
    // 위험 점수 = 가능성 x 영향 x (1 / 감지성)
    const likelihood = this.scoreToNumber(risk.likelihood);
    const impact = this.scoreToNumber(risk.impact);
    const detectability = this.scoreToNumber(risk.detectability);

    return likelihood * impact * (1 / detectability);
  }

  // 위험 수준 결정
  private determineRiskLevel(risk: AnalyzedRisk): RiskLevel {
    const score = this.calculateRiskScore(risk);

    if (score >= 80) return 'CRITICAL';
    if (score >= 60) return 'HIGH';
    if (score >= 40) return 'MEDIUM';
    if (score >= 20) return 'LOW';
    return 'MINIMAL';
  }

  // 전체 위험 점수 계산
  private calculateOverallRisk(risks: EvaluatedRisk[]): number {
    if (risks.length === 0) return 0;

    // 가중 평균 (높은 위험에 더 큰 가중치)
    const weights = risks.map(r => {
      switch (r.riskLevel) {
        case 'CRITICAL': return 5;
        case 'HIGH': return 3;
        case 'MEDIUM': return 2;
        case 'LOW': return 1;
        default: return 0.5;
      }
    });

    const weightedSum = risks.reduce(
      (sum, risk, i) => sum + risk.riskScore * weights[i],
      0
    );
    const totalWeight = weights.reduce((sum, w) => sum + w, 0);

    return weightedSum / totalWeight;
  }
}

편향 감지 및 평가

AI 의사결정에서 편향을 감지하고 측정하는 방법:

// 편향 감지 시스템
class BiasDetector {
  // 편향 분석 수행
  async detectBias(
    decisions: AuditRecord[],
    protectedAttributes: string[]
  ): Promise {
    const biasMetrics: BiasMetric[] = [];

    // 각 보호 속성에 대해 편향 검사
    for (const attribute of protectedAttributes) {
      // 1. 통계적 패리티
      const statisticalParity = await this.checkStatisticalParity(
        decisions,
        attribute
      );

      // 2. 동등한 기회
      const equalOpportunity = await this.checkEqualOpportunity(
        decisions,
        attribute
      );

      // 3. 예측 패리티
      const predictiveParity = await this.checkPredictiveParity(
        decisions,
        attribute
      );

      // 4. 개별 공정성
      const individualFairness = await this.checkIndividualFairness(
        decisions,
        attribute
      );

      biasMetrics.push({
        attribute: attribute,
        statisticalParity: statisticalParity,
        equalOpportunity: equalOpportunity,
        predictiveParity: predictiveParity,
        individualFairness: individualFairness,
        overallBiasScore: this.calculateBiasScore([
          statisticalParity,
          equalOpportunity,
          predictiveParity,
          individualFairness
        ])
      });
    }

    return {
      metrics: biasMetrics,
      overallBias: this.aggregateBiasScores(biasMetrics),
      recommendations: this.generateBiasRecommendations(biasMetrics),
      timestamp: new Date().toISOString()
    };
  }

  // 통계적 패리티 검사
  private async checkStatisticalParity(
    decisions: AuditRecord[],
    attribute: string
  ): Promise {
    // 그룹별 긍정 결정 비율 계산
    const groups = this.groupByAttribute(decisions, attribute);
    const rates: Record = {};

    for (const [group, records] of Object.entries(groups)) {
      const positive = records.filter(r => this.isPositiveDecision(r)).length;
      rates[group] = positive / records.length;
    }

    // 최대 차이 계산
    const rateValues = Object.values(rates);
    const maxRate = Math.max(...rateValues);
    const minRate = Math.min(...rateValues);
    const disparity = maxRate - minRate;

    return {
      name: 'Statistical Parity',
      value: 1 - disparity, // 1에 가까울수록 공정
      threshold: 0.8,
      passed: disparity < 0.2, // 20% 임계값
      details: {
        rates: rates,
        disparity: disparity
      }
    };
  }

  // 동등한 기회 검사
  private async checkEqualOpportunity(
    decisions: AuditRecord[],
    attribute: string
  ): Promise {
    // 실제 긍정인 경우의 True Positive Rate 비교
    const groups = this.groupByAttribute(decisions, attribute);
    const tprByGroup: Record = {};

    for (const [group, records] of Object.entries(groups)) {
      // 실제 긍정인 경우만 필터
      const actualPositive = records.filter(r =>
        this.getGroundTruth(r) === true
      );

      if (actualPositive.length > 0) {
        const truePositives = actualPositive.filter(r =>
          this.isPositiveDecision(r)
        ).length;

        tprByGroup[group] = truePositives / actualPositive.length;
      }
    }

    // TPR 차이 계산
    const tprValues = Object.values(tprByGroup);
    const disparity = Math.max(...tprValues) - Math.min(...tprValues);

    return {
      name: 'Equal Opportunity',
      value: 1 - disparity,
      threshold: 0.8,
      passed: disparity < 0.2,
      details: {
        truePositiveRates: tprByGroup,
        disparity: disparity
      }
    };
  }

  // 예측 패리티 검사
  private async checkPredictiveParity(
    decisions: AuditRecord[],
    attribute: string
  ): Promise {
    // Positive Predictive Value (정밀도) 비교
    const groups = this.groupByAttribute(decisions, attribute);
    const ppvByGroup: Record = {};

    for (const [group, records] of Object.entries(groups)) {
      // 긍정 예측만 필터
      const predicted = records.filter(r => this.isPositiveDecision(r));

      if (predicted.length > 0) {
        const truePositives = predicted.filter(r =>
          this.getGroundTruth(r) === true
        ).length;

        ppvByGroup[group] = truePositives / predicted.length;
      }
    }

    const ppvValues = Object.values(ppvByGroup);
    const disparity = Math.max(...ppvValues) - Math.min(...ppvValues);

    return {
      name: 'Predictive Parity',
      value: 1 - disparity,
      threshold: 0.8,
      passed: disparity < 0.2,
      details: {
        positivePredictiveValues: ppvByGroup,
        disparity: disparity
      }
    };
  }

  // 개별 공정성 검사
  private async checkIndividualFairness(
    decisions: AuditRecord[],
    attribute: string
  ): Promise {
    // 유사한 개인에 대한 유사한 결정
    const similarityScores: number[] = [];

    for (let i = 0; i < decisions.length; i++) {
      for (let j = i + 1; j < decisions.length; j++) {
        // 보호 속성을 제외한 특징의 유사도
        const similarity = this.calculateSimilarity(
          decisions[i],
          decisions[j],
          [attribute]
        );

        if (similarity > 0.9) { // 매우 유사한 경우
          // 결정의 일관성 확인
          const outcomeConsistency = this.compareOutcomes(
            decisions[i],
            decisions[j]
          );

          similarityScores.push(outcomeConsistency);
        }
      }
    }

    const avgConsistency = similarityScores.length > 0
      ? similarityScores.reduce((a, b) => a + b) / similarityScores.length
      : 1;

    return {
      name: 'Individual Fairness',
      value: avgConsistency,
      threshold: 0.9,
      passed: avgConsistency >= 0.9,
      details: {
        comparisonsMade: similarityScores.length,
        averageConsistency: avgConsistency
      }
    };
  }
}

이상 징후 감지

비정상적인 의사결정 패턴을 감지하는 시스템:

// 이상 징후 감지 엔진
class AnomalyDetector {
  private baseline: BaselineProfile;
  private threshold: number = 2.5; // 표준편차 임계값

  // 이상 징후 감지
  async detect(decisions: AuditRecord[]): Promise {
    const anomalies: Anomaly[] = [];

    // 1. 통계적 이상 징후
    anomalies.push(...await this.detectStatisticalAnomalies(decisions));

    // 2. 패턴 이상 징후
    anomalies.push(...await this.detectPatternAnomalies(decisions));

    // 3. 시계열 이상 징후
    anomalies.push(...await this.detectTimeSeriesAnomalies(decisions));

    // 4. 클러스터 기반 이상 징후
    anomalies.push(...await this.detectClusterAnomalies(decisions));

    return this.rankAnomalies(anomalies);
  }

  // 통계적 이상 징후 감지
  private async detectStatisticalAnomalies(
    decisions: AuditRecord[]
  ): Promise {
    const anomalies: Anomaly[] = [];

    // 신뢰도 점수 분포 확인
    const confidences = decisions.map(d => d.output.confidence);
    const mean = this.mean(confidences);
    const stdDev = this.standardDeviation(confidences);

    for (const decision of decisions) {
      const zScore = (decision.output.confidence - mean) / stdDev;

      if (Math.abs(zScore) > this.threshold) {
        anomalies.push({
          type: 'STATISTICAL',
          severity: Math.abs(zScore) > 3 ? 'HIGH' : 'MEDIUM',
          decision: decision.id,
          description: `비정상적인 신뢰도: ${decision.output.confidence}`,
          score: Math.abs(zScore),
          timestamp: decision.timestamp
        });
      }
    }

    return anomalies;
  }

  // 패턴 이상 징후 감지
  private async detectPatternAnomalies(
    decisions: AuditRecord[]
  ): Promise {
    const anomalies: Anomaly[] = [];

    // 동일한 결정의 과도한 반복
    const decisionCounts = new Map();

    for (const decision of decisions) {
      const key = decision.output.decision;
      decisionCounts.set(key, (decisionCounts.get(key) || 0) + 1);
    }

    // 예상 분포와 비교
    for (const [decision, count] of decisionCounts) {
      const expectedCount = decisions.length * this.baseline.decisionRates[decision];
      const deviation = Math.abs(count - expectedCount) / expectedCount;

      if (deviation > 0.5) { // 50% 이상 편차
        anomalies.push({
          type: 'PATTERN',
          severity: deviation > 1 ? 'HIGH' : 'MEDIUM',
          description: `비정상적인 결정 패턴: ${decision}이(가) ${count}회 (예상: ${expectedCount}회)`,
          score: deviation,
          timestamp: new Date().toISOString()
        });
      }
    }

    return anomalies;
  }

  // 시계열 이상 징후 감지
  private async detectTimeSeriesAnomalies(
    decisions: AuditRecord[]
  ): Promise {
    const anomalies: Anomaly[] = [];

    // 시간대별 결정 속도
    const hourlyRates = this.groupByHour(decisions);

    for (const [hour, rate] of Object.entries(hourlyRates)) {
      const expectedRate = this.baseline.hourlyRates[hour];

      if (rate > expectedRate * 2) { // 2배 이상
        anomalies.push({
          type: 'TIME_SERIES',
          severity: 'MEDIUM',
          description: `비정상적으로 높은 결정 속도: ${hour}시에 ${rate}회`,
          score: rate / expectedRate,
          timestamp: new Date().toISOString()
        });
      }
    }

    return anomalies;
  }

  // 순위 매기기
  private rankAnomalies(anomalies: Anomaly[]): Anomaly[] {
    return anomalies.sort((a, b) => {
      // 심각도 우선
      const severityOrder = { HIGH: 3, MEDIUM: 2, LOW: 1 };
      const severityDiff = severityOrder[b.severity] - severityOrder[a.severity];

      if (severityDiff !== 0) return severityDiff;

      // 점수 우선
      return b.score - a.score;
    });
  }
}

위험 모니터링 대시보드

실시간 위험 모니터링을 위한 주요 지표:

핵심 위험 지표 (KRI)
  • 편향 점수: 보호 속성에 대한 공정성 메트릭
  • 이상 징후 빈도: 시간당 감지된 이상 징후 수
  • 성능 드리프트: 기준선 대비 성능 변화
  • 규정 준수율: 규제 요구사항 충족 비율
  • 에러율: 오결정 또는 시스템 오류 비율
  • 사용자 이의제기: 결정에 대한 이의제기 건수

지속적 위험 관리

위험 관리는 일회성 활동이 아니라 지속적인 프로세스입니다:

// 지속적 위험 관리 시스템
class ContinuousRiskManagement {
  private riskAssessment: AIRiskAssessment;
  private biasDetector: BiasDetector;
  private anomalyDetector: AnomalyDetector;

  // 정기 위험 평가 (일일/주간)
  async scheduledAssessment(): Promise {
    // 최근 결정 조회
    const recentDecisions = await this.getRecentDecisions();

    // 위험 평가
    const riskReport = await this.riskAssessment.assess(
      this.model,
      this.context
    );

    // 편향 감지
    const biasReport = await this.biasDetector.detectBias(
      recentDecisions,
      this.protectedAttributes
    );

    // 이상 징후 감지
    const anomalies = await this.anomalyDetector.detect(recentDecisions);

    // 결과 집계 및 알림
    await this.processResults({
      risk: riskReport,
      bias: biasReport,
      anomalies: anomalies
    });
  }

  // 실시간 위험 모니터링
  async monitorDecision(decision: AuditRecord): Promise {
    // 즉각적인 위험 검사
    const risks = await this.quickRiskCheck(decision);

    // 높은 위험 감지 시 경고
    if (risks.some(r => r.level === 'CRITICAL' || r.level === 'HIGH')) {
      await this.alerting.sendAlert({
        type: 'HIGH_RISK_DECISION',
        decision: decision.id,
        risks: risks,
        timestamp: new Date().toISOString()
      });
    }
  }
}

챕터 요약

  • AI 위험 평가는 편향, 프라이버시, 안전, 성능, 규정 준수 등 다양한 위험을 다룹니다
  • 체계적인 위험 평가 프레임워크는 식별, 분석, 평가, 우선순위 결정 및 완화를 포함합니다
  • 편향 감지는 통계적 패리티, 동등한 기회, 예측 패리티, 개별 공정성을 평가합니다
  • 이상 징후 감지는 통계적, 패턴, 시계열 및 클러스터 기반 접근 방식을 사용합니다
  • 각 위험 유형에는 특정 완화 전략이 있으며, 편향 제거, 차등 프라이버시, 설명 가능 AI 등이 포함됩니다
  • 실시간 위험 모니터링은 핵심 위험 지표(KRI)를 추적하고 즉각적인 경고를 제공합니다
  • 지속적 위험 관리는 정기 평가와 실시간 모니터링을 결합합니다
  • 위험 평가는 안전하고 공정하며 책임 있는 AI를 보장하여 弘益人間의 원칙을 실현합니다
