Key Features and Benefits

  • Decentralized Identifiers (DIDs): W3C DID standard compliant, self-sovereign identity
  • Verifiable Credentials: Issue, hold, and verify credentials without centralized authority
  • Multi-Factor Authentication: Support for biometric, hardware keys (YubiKey), OTP, passkeys (WebAuthn)
  • Zero-Knowledge Proofs: Prove attributes without revealing underlying data
  • Cross-Platform: Works on web, mobile (iOS/Android), IoT devices
  • Blockchain Integration: DID anchoring on Ethereum, Polygon, Hyperledger Indy
  • Privacy-Preserving: Minimal disclosure, selective attribute sharing, GDPR compliant
  • Interoperable: Compatible with OAuth 2.0, OpenID Connect, SAML 2.0 legacy systems
  • Revocation Registry: Real-time credential revocation checking
  • Identity Wallets: Secure storage on mobile devices with biometric protection
  • Enterprise Ready: SSO integration, LDAP/Active Directory sync, role-based access control
  • WIA Ecosystem: Integrates with Universal Consent for privacy-first identity management

About This Standard

Universal Identity (WIA-CORE-001) is part of the WIA (World Certification Industry Association) Core Standards suite, designed to ensure interoperability, security, and compliance across all implementations worldwide. This standard has been developed through collaborative efforts of industry experts, deployed in production by leading organizations, and continuously refined based on real-world feedback.

弘益人間 (Benefit All Humanity) - This standard aims to create universal frameworks that benefit all stakeholders in the global digital ecosystem, promoting innovation while ensuring privacy, security, and regulatory compliance.

Compliance Levels

Level Requirements Capabilities
Bronze Basic DID support, simple credential issuance, single authentication factor Self-sovereign identity, basic verification, manual revocation
Silver Multi-factor authentication, verifiable credentials, blockchain anchoring, identity wallets Advanced authentication, automated workflows, mobile wallet support, OAuth/OIDC integration
Gold Zero-knowledge proofs, full GDPR compliance, enterprise SSO, biometric authentication, automated revocation Privacy-preserving identity, enterprise-grade security, complete interoperability, regulatory compliance

Technology Stack

DID Methods: did:web, did:key, did:ethr, did:ion, did:sov

Credential Formats: JSON-LD, JWT, Verifiable Credentials Data Model 1.1

Blockchain: Ethereum, Polygon, Hyperledger Indy, Hyperledger Fabric

Authentication: WebAuthn, FIDO2, OAuth 2.0, OpenID Connect, SAML 2.0

Wallets: Mobile SDK (iOS/Android), Browser Extension, Hardware Wallets

Cryptography: Ed25519, secp256k1, RSA-2048, BLS signatures

Standards: W3C DID Core, VC Data Model, DIF Presentation Exchange
