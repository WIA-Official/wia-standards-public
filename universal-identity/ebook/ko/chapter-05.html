Chapter Summary

Key Takeaways:

  1. 자격증명 스키마는 검증 가능한 자격증명의 구조, 데이터 타입, 검증 규칙을 정의하여 자격증명 클레임에 대한 공유 이해를 통해 발급자와 검증자 간 상호운용성을 가능하게 합니다 - 스키마는 탈중앙 생태계 전반에 걸쳐 일관된 자격증명 해석을 보장하는 계약 역할을 합니다.
  2. JSON 스키마는 유니버설 아이덴티티의 자격증명 스키마를 위한 기반을 제공하며, 데이터 타입, 필수 필드, 값 제약, 중첩 구조에 대한 검증 규칙을 제공하고 사용자 정의 속성과 조건부 검증 논리를 통한 확장성을 지원합니다.
  3. 블록체인 플랫폼의 스키마 레지스트리(하이퍼레저 인디 스키마 객체, 이더리움 스키마 스마트 계약)는 발급자가 자격증명을 생성할 때 참조하고 검증자가 클레임을 처리하기 전에 자격증명 구조를 검증하는 데 사용하는 불변의 버전 관리된 스키마 정의를 생성합니다.
  4. 표준 자격증명 타입(교육 학위, 전문 면허, 정부 신분증, 고용 확인)은 Schema.org 어휘 및 OpenID Connect 클레임과 같은 범용 스키마의 이점을 누리며, 통합 복잡성을 줄이고 국경 간 자격증명 인정을 가능하게 합니다.
  5. 프라이버시 보호 스키마 설계에는 클레임 세분성 결정(단일 주소 대 거리/도시/국가 필드), 민감한 데이터 처리(PII 해싱, 오프체인 저장), 전체 자격증명을 공개하지 않고 원자적 클레임 프레젠테이션을 허용하는 스키마 분해를 통한 선택적 공개 지원이 포함됩니다.
  6. 스키마 버전 관리 및 진화는 신중한 거버넌스가 필요합니다 - 하위 호환 가능한 추가(새로운 선택적 필드)는 점진적인 생태계 업데이트를 가능하게 하지만, 호환성을 깨는 변경(필드 제거, 타입 변경)은 검증 안정성을 유지하기 위해 새로운 스키마 버전과 자격증명 재발급이 필요합니다.
  7. 유니버설 동의와의 통합은 스키마가 동의 의미론을 정의하도록 요구합니다 - 어떤 클레임이 명시적 동의를 필요로 하는지, 동의 만료 기간, 목적 제한 - 프라이버시 요구사항을 자격증명 구조에 직접 임베드하고 자동화된 GDPR 준수 확인을 가능하게 합니다.

Review Questions

  1. JSON 스키마 검증이 발급자와 검증자가 독립적으로 운영되는 탈중앙 시스템에서 자격증명 무결성을 어떻게 보장하는지 설명하십시오. 필수 필드, 데이터 타입, 검증 제약을 포함하고 국제 자격증명 형식을 지원하는 대학 학위 자격증명을 위한 JSON 스키마를 설계하십시오.
  2. 하이퍼레저 인디의 온레저 스키마 객체와 이더리움 기반 스키마 레지스트리(IPFS 스키마를 참조하는 스마트 계약)를 비교하십시오. 비용, 불변성, 검색 가능성, 스키마 진화 기능 측면에서 어떤 트레이드오프가 있습니까?
  3. 자격증명 스키마와 영지식 증명 시스템 간의 관계를 분석하십시오. 스키마가 선택적 공개와 술어 증명을 지원하려면 어떻게 구조화되어야 합니까? 생년월일을 공개하지 않고 "나이 > 21"을 증명할 수 있는 스키마를 설계하십시오.
  4. 다중 이해관계자 자격증명 생태계에서 스키마 거버넌스 과제를 설명하십시오. 표준 기관, 정부, 민간 발급자가 스키마 정의에 대해 어떻게 협력해야 합니까? 스키마 거버넌스 프레임워크를 제공하는 유니버설 아이덴티티의 역할을 평가하십시오.
  5. 10년 유효 기간을 가진 정부 신분증 자격증명을 위한 스키마 버전 관리 전략을 설계하십시오. 하위 호환성, 더 이상 사용되지 않는 필드 처리, 검증자 전환 기간, 시민 재발급 요구사항을 최소화하면서 스키마에 대한 긴급 보안 업데이트를 다루십시오.
  6. 자격증명 스키마가 유니버설 동의 요구사항과 어떻게 통합되는지 평가하십시오. 데이터 카테고리(진단, 약물), 처리 목적(치료, 연구), 보유 정책을 포함한 동의 메타데이터를 임베드하는 건강 기록 스키마를 설계하여 GDPR 제9조 준수를 보장하십시오.
  7. 세밀한 대 거친 스키마 설계의 프라이버시 영향을 논의하십시오. 원자적 필드(거리, 도시, 국가)를 가진 주소 스키마와 단일 주소 문자열을 비교하십시오. 세분성이 선택적 공개, 국제화, 연령 제한 서비스를 위한 프라이버시 보호 검증에 어떻게 영향을 미칩니까?
