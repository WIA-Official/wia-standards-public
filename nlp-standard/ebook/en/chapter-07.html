Text Generation and Summarization

7.1 Introduction to Text Generation

Text generation involves creating coherent, contextually relevant text automatically. From machine translation to chatbots, story generation to code completion, generative NLP models have transformed how we interact with computers and access information.

Modern text generation leverages neural language models trained on vast text corpora. These models learn probabilistic distributions over text sequences, enabling them to generate fluent, contextually appropriate continuations.

Text Generation Applications

  • Machine Translation: Translating text between languages
  • Summarization: Generating concise summaries of documents
  • Dialogue Systems: Chatbots and virtual assistants
  • Content Creation: Article writing, poetry, creative stories
  • Code Generation: Generating programming code from descriptions
  • Data-to-Text: Converting structured data to narratives

7.2 Language Modeling Basics

Language models assign probabilities to sequences of words, enabling prediction of the next word given previous context. The quality of generated text depends on how well the model captures linguistic patterns and world knowledge.

Autoregressive Generation

from transformers import GPT2LMHeadModel, GPT2Tokenizer
import torch

# Load pre-trained GPT-2
model = GPT2LMHeadModel.from_pretrained('gpt2')
tokenizer = GPT2Tokenizer.from_pretrained('gpt2')

# Generate text
prompt = "Natural language processing enables computers to"
inputs = tokenizer(prompt, return_tensors='pt')

# Generate with different strategies
outputs = model.generate(
    inputs['input_ids'],
    max_length=50,
    num_return_sequences=1,
    temperature=0.8,
    top_k=50,
    top_p=0.95,
    do_sample=True
)

generated_text = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(generated_text)

7.3 Decoding Strategies

Different decoding strategies balance between determinism and creativity, fluency and diversity. The choice of strategy significantly impacts generation quality.

Common Decoding Methods

Strategy Description Use Case
Greedy Search Always pick highest probability token Deterministic, factual generation
Beam Search Maintain k best sequences Translation, summarization
Top-k Sampling Sample from k most likely tokens Creative text, dialogue
Top-p (Nucleus) Sample from smallest set with cumulative p Balanced fluency and diversity
Temperature Sampling Scale probabilities by temperature Control randomness

7.4 Sequence-to-Sequence Models

Seq2Seq models transform input sequences into output sequences of potentially different lengths. The encoder-decoder architecture with attention mechanisms powers machine translation, summarization, and other generation tasks. 

from transformers import T5ForConditionalGeneration, T5Tokenizer

# T5 is a unified text-to-text model
model = T5ForConditionalGeneration.from_pretrained('t5-small')
tokenizer = T5Tokenizer.from_pretrained('t5-small')

# T5 uses task prefixes
def generate_with_t5(task, text):
    input_text = f"{task}: {text}"
    inputs = tokenizer(input_text, return_tensors='pt', max_length=512, truncation=True)

    outputs = model.generate(
        inputs['input_ids'],
        max_length=150,
        num_beams=4,
        early_stopping=True
    )

    return tokenizer.decode(outputs[0], skip_special_tokens=True)

# Summarization
article = "Natural language processing has revolutionized how computers understand human language. Modern NLP systems can translate languages, answer questions, and generate human-like text. These capabilities are powered by deep learning and large-scale pre-training."
summary = generate_with_t5("summarize", article)
print(f"Summary: {summary}")

# Translation
text_en = "Hello, how are you?"
text_de = generate_with_t5("translate English to German", text_en)
print(f"German: {text_de}")

7.5 Text Summarization

Summarization condenses long documents into shorter versions while preserving key information. Extractive summarization selects important sentences, while abstractive summarization generates new text.

Extractive vs Abstractive Summarization

Extractive Summarization

Selects and combines existing sentences from the source document. Simpler and more factually accurate, but can be choppy and lack coherence.

Abstractive Summarization

Generates new sentences that capture the essence of the source. More fluent and coherent, but risk of factual errors and hallucinations. 

# Extractive summarization with sumy
from sumy.parsers.plaintext import PlaintextParser
from sumy.nlp.tokenizers import Tokenizer
from sumy.summarizers.lsa import LsaSummarizer

text = """
Natural language processing (NLP) has transformed how computers interact with human language.
Modern NLP systems can perform diverse tasks including translation, question answering, and text generation.
Deep learning models like transformers have achieved remarkable performance across NLP benchmarks.
These models are pre-trained on massive text corpora and fine-tuned for specific tasks.
Applications of NLP span healthcare, finance, customer service, and education.
The field continues to evolve with new architectures and training techniques.
"""

parser = PlaintextParser.from_string(text, Tokenizer("english"))
summarizer = LsaSummarizer()
summary = summarizer(parser.document, sentences_count=2)

print("Extractive Summary:")
for sentence in summary:
    print(sentence)
# Abstractive summarization with transformers
from transformers import pipeline

summarizer = pipeline("summarization", model="facebook/bart-large-cnn")

article = """
The WIA-AI-023 NLP standard provides comprehensive guidelines for natural language processing systems.
It defines data formats, API interfaces, and integration protocols to ensure interoperability.
The standard covers tokenization, entity recognition, sentiment analysis, and text generation.
By following WIA-AI-023, organizations can build NLP systems that work seamlessly across platforms.
The standard promotes ethical AI development and supports multilingual applications.
It includes specifications for model evaluation, deployment, and monitoring.
"""

summary = summarizer(article, max_length=50, min_length=25, do_sample=False)
print(f"Abstractive Summary: {summary[0]['summary_text']}")

7.6 Controlled Text Generation

Controlling generation attributes like sentiment, style, topic, or format enables more useful and safe applications. Techniques include prompt engineering, fine-tuning, and constrained decoding.

Prompt Engineering

# Controlling generation through prompts
from transformers import pipeline

generator = pipeline('text-generation', model='gpt2')

# Control sentiment
positive_prompt = "The review of the product was overwhelmingly positive:"
positive_text = generator(positive_prompt, max_length=50)[0]['generated_text']
print(f"Positive: {positive_text}\n")

# Control format
list_prompt = "Here are 5 tips for better NLP:\n1."
list_text = generator(list_prompt, max_length=100)[0]['generated_text']
print(f"List: {list_text}")

7.7 Evaluation of Generated Text

Evaluating generated text is challenging because there are often many valid outputs. Metrics combine automatic scores with human evaluation.

Automatic Metrics

Metric Task Description
BLEU Translation N-gram overlap with references
ROUGE Summarization Recall-oriented n-gram overlap
METEOR Translation Alignment with synonyms
BERTScore General Semantic similarity using BERT
Perplexity Language Model Model uncertainty 
# Evaluate with ROUGE
from rouge import Rouge

reference = "Natural language processing enables computers to understand human language"
hypothesis = "NLP allows machines to comprehend text and speech"

rouge = Rouge()
scores = rouge.get_scores(hypothesis, reference)[0]

print("ROUGE Scores:")
print(f"ROUGE-1: {scores['rouge-1']['f']:.4f}")
print(f"ROUGE-2: {scores['rouge-2']['f']:.4f}")
print(f"ROUGE-L: {scores['rouge-l']['f']:.4f}")

7.8 Few-Shot and Zero-Shot Generation

Large language models can perform tasks with minimal examples (few-shot) or no task-specific training (zero-shot) through in-context learning, demonstrating remarkable generalization capabilities. 

# Few-shot learning with GPT
prompt = """
Translate English to French:
English: Hello
French: Bonjour

English: Goodbye
French: Au revoir

English: Thank you
French: Merci

English: How are you?
French:"""

# Model learns translation pattern from examples
response = generator(prompt, max_length=len(prompt.split()) + 5)[0]['generated_text']
print(response)

7.9 Safety and Ethics in Generation

Generated text can contain biases, toxicity, or misinformation. Responsible deployment requires filtering, human oversight, and continuous monitoring.

Safety Measures

  • Content Filtering: Block toxic or inappropriate outputs
  • Fact Checking: Verify factual claims in generated text
  • Human in the Loop: Review critical applications
  • Attribution: Clearly mark AI-generated content
  • Bias Mitigation: Monitor and reduce demographic biases
