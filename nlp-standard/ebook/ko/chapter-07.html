7장

텍스트 생성과 요약

7.1 텍스트 생성 소개

텍스트 생성은 일관되고 문맥적으로 관련된 텍스트를 자동으로 생성하는 것을 포함합니다. 기계 번역에서 챗봇, 스토리 생성에서 코드 완성까지, 생성적 NLP 모델은 우리가 컴퓨터와 상호작용하고 정보에 접근하는 방식을 변화시켰습니다.

현대 텍스트 생성은 방대한 텍스트 말뭉치에 대해 학습된 신경 언어 모델을 활용합니다. 이러한 모델은 텍스트 시퀀스에 대한 확률적 분포를 학습하여 유창하고 문맥적으로 적절한 연속을 생성할 수 있게 합니다.

텍스트 생성 애플리케이션

  • 기계 번역: 언어 간 텍스트 번역
  • 요약: 문서의 간결한 요약 생성
  • 대화 시스템: 챗봇 및 가상 비서
  • 콘텐츠 생성: 기사 작성, 시, 창의적 스토리
  • 코드 생성: 설명으로부터 프로그래밍 코드 생성
  • 데이터-투-텍스트: 구조화된 데이터를 내러티브로 변환

7.2 언어 모델링 기초

언어 모델은 단어 시퀀스에 확률을 할당하여 이전 맥락이 주어졌을 때 다음 단어를 예측할 수 있게 합니다. 생성된 텍스트의 품질은 모델이 언어적 패턴과 세계 지식을 얼마나 잘 포착하는지에 달려 있습니다.

자기회귀 생성

from transformers import GPT2LMHeadModel, GPT2Tokenizer
import torch

# 사전 학습된 GPT-2 로드
model = GPT2LMHeadModel.from_pretrained('gpt2')
tokenizer = GPT2Tokenizer.from_pretrained('gpt2')

# 텍스트 생성
prompt = "자연어 처리는 컴퓨터가"
inputs = tokenizer(prompt, return_tensors='pt')

# 다양한 전략으로 생성
outputs = model.generate(
    inputs['input_ids'],
    max_length=50,
    num_return_sequences=1,
    temperature=0.8,
    top_k=50,
    top_p=0.95,
    do_sample=True
)

generated_text = tokenizer.decode(outputs[0], skip_special_tokens=True)
print(generated_text)

7.3 디코딩 전략

다양한 디코딩 전략은 결정론과 창의성, 유창성과 다양성 사이의 균형을 맞춥니다. 전략 선택은 생성 품질에 상당한 영향을 미칩니다.

일반적인 디코딩 방법

전략 설명 사용 사례
탐욕 탐색 항상 가장 높은 확률 토큰 선택 결정론적, 사실적 생성
빔 탐색 k개 최상의 시퀀스 유지 번역, 요약
Top-k 샘플링 k개의 가장 가능성 높은 토큰에서 샘플링 창의적 텍스트, 대화
Top-p (Nucleus) 누적 p를 가진 가장 작은 집합에서 샘플링 유창성과 다양성의 균형
온도 샘플링 온도로 확률 스케일링 무작위성 제어

7.4 시퀀스-투-시퀀스 모델

Seq2Seq 모델은 입력 시퀀스를 잠재적으로 다른 길이의 출력 시퀀스로 변환합니다. 어텐션 메커니즘을 가진 인코더-디코더 아키텍처는 기계 번역, 요약 및 기타 생성 작업을 강화합니다. 

from transformers import T5ForConditionalGeneration, T5Tokenizer

# T5는 통합 텍스트-투-텍스트 모델
model = T5ForConditionalGeneration.from_pretrained('t5-small')
tokenizer = T5Tokenizer.from_pretrained('t5-small')

# T5는 작업 접두사 사용
def generate_with_t5(task, text):
    input_text = f"{task}: {text}"
    inputs = tokenizer(input_text, return_tensors='pt', max_length=512, truncation=True)

    outputs = model.generate(
        inputs['input_ids'],
        max_length=150,
        num_beams=4,
        early_stopping=True
    )

    return tokenizer.decode(outputs[0], skip_special_tokens=True)

# 요약
article = "자연어 처리는 컴퓨터가 인간 언어를 이해하는 방식을 혁신했습니다. 현대 NLP 시스템은 언어를 번역하고, 질문에 답하며, 인간과 같은 텍스트를 생성할 수 있습니다. 이러한 기능은 딥러닝과 대규모 사전 학습으로 구동됩니다."
summary = generate_with_t5("요약", article)
print(f"요약: {summary}")

# 번역
text_ko = "안녕하세요, 어떻게 지내세요?"
text_en = generate_with_t5("한국어를 영어로 번역", text_ko)
print(f"영어: {text_en}")

7.5 텍스트 요약

요약은 핵심 정보를 보존하면서 긴 문서를 더 짧은 버전으로 압축합니다. 추출적 요약은 중요한 문장을 선택하고, 요약적 요약은 새로운 텍스트를 생성합니다.

추출적 대 요약적 요약

추출적 요약

소스 문서에서 기존 문장을 선택하고 결합합니다. 더 간단하고 사실적으로 정확하지만 다소 불연속적이고 일관성이 부족할 수 있습니다.

요약적 요약

소스의 본질을 포착하는 새로운 문장을 생성합니다. 더 유창하고 일관성 있지만 사실 오류와 환각의 위험이 있습니다. 

# sumy를 사용한 추출적 요약
from sumy.parsers.plaintext import PlaintextParser
from sumy.nlp.tokenizers import Tokenizer
from sumy.summarizers.lsa import LsaSummarizer

text = """
자연어 처리(NLP)는 컴퓨터가 인간 언어와 상호작용하는 방식을 변화시켰습니다.
현대 NLP 시스템은 번역, 질의응답 및 텍스트 생성을 포함한 다양한 작업을 수행할 수 있습니다.
트랜스포머와 같은 딥러닝 모델은 NLP 벤치마크에서 놀라운 성능을 달성했습니다.
이러한 모델은 대규모 텍스트 말뭉치에 대해 사전 학습되고 특정 작업에 대해 미세 조정됩니다.
NLP의 응용은 의료, 금융, 고객 서비스 및 교육에 걸쳐 있습니다.
이 분야는 새로운 아키텍처와 학습 기법으로 계속 발전하고 있습니다.
"""

parser = PlaintextParser.from_string(text, Tokenizer("korean"))
summarizer = LsaSummarizer()
summary = summarizer(parser.document, sentences_count=2)

print("추출적 요약:")
for sentence in summary:
    print(sentence)
# 트랜스포머를 사용한 요약적 요약
from transformers import pipeline

summarizer = pipeline("summarization", model="facebook/bart-large-cnn")

article = """
WIA-AI-023 NLP 표준은 자연어 처리 시스템에 대한 포괄적인 가이드라인을 제공합니다.
데이터 형식, API 인터페이스 및 상호운용성을 보장하는 통합 프로토콜을 정의합니다.
표준은 토큰화, 개체 인식, 감정 분석 및 텍스트 생성을 다룹니다.
WIA-AI-023을 따름으로써 조직은 플랫폼 간에 원활하게 작동하는 NLP 시스템을 구축할 수 있습니다.
표준은 윤리적 AI 개발을 촉진하고 다국어 애플리케이션을 지원합니다.
모델 평가, 배포 및 모니터링에 대한 사양을 포함합니다.
"""

summary = summarizer(article, max_length=50, min_length=25, do_sample=False)
print(f"요약적 요약: {summary[0]['summary_text']}")

7.6 제어된 텍스트 생성

감정, 스타일, 주제 또는 형식과 같은 생성 속성을 제어하면 더 유용하고 안전한 애플리케이션을 가능하게 합니다. 기법에는 프롬프트 엔지니어링, 미세 조정 및 제약 디코딩이 포함됩니다.

프롬프트 엔지니어링

# 프롬프트를 통한 생성 제어
from transformers import pipeline

generator = pipeline('text-generation', model='gpt2')

# 감정 제어
positive_prompt = "제품에 대한 리뷰는 압도적으로 긍정적이었습니다:"
positive_text = generator(positive_prompt, max_length=50)[0]['generated_text']
print(f"긍정: {positive_text}\n")

# 형식 제어
list_prompt = "더 나은 NLP를 위한 5가지 팁:\n1."
list_text = generator(list_prompt, max_length=100)[0]['generated_text']
print(f"리스트: {list_text}")

7.7 생성된 텍스트 평가

생성된 텍스트를 평가하는 것은 도전적입니다. 왜냐하면 종종 많은 유효한 출력이 있기 때문입니다. 메트릭은 자동 점수와 인간 평가를 결합합니다.

자동 메트릭

메트릭 작업 설명
BLEU 번역 참조와의 N-그램 겹침
ROUGE 요약 재현율 지향 N-그램 겹침
METEOR 번역 동의어를 포함한 정렬
BERTScore 일반 BERT를 사용한 의미적 유사도
복잡도 언어 모델 모델 불확실성 
# ROUGE로 평가
from rouge import Rouge

reference = "자연어 처리는 컴퓨터가 인간 언어를 이해할 수 있게 합니다"
hypothesis = "NLP는 기계가 텍스트와 음성을 이해할 수 있게 합니다"

rouge = Rouge()
scores = rouge.get_scores(hypothesis, reference)[0]

print("ROUGE 점수:")
print(f"ROUGE-1: {scores['rouge-1']['f']:.4f}")
print(f"ROUGE-2: {scores['rouge-2']['f']:.4f}")
print(f"ROUGE-L: {scores['rouge-l']['f']:.4f}")

7.8 퓨샷 및 제로샷 생성

대형 언어 모델은 맥락 내 학습을 통해 최소한의 예제(퓨샷) 또는 작업별 학습 없이(제로샷) 작업을 수행할 수 있어 놀라운 일반화 능력을 보여줍니다. 

# GPT를 사용한 퓨샷 학습
prompt = """
영어를 프랑스어로 번역:
영어: Hello
프랑스어: Bonjour

영어: Goodbye
프랑스어: Au revoir

영어: Thank you
프랑스어: Merci

영어: How are you?
프랑스어:"""

# 모델이 예제로부터 번역 패턴 학습
response = generator(prompt, max_length=len(prompt.split()) + 5)[0]['generated_text']
print(response)

7.9 생성에서의 안전과 윤리

생성된 텍스트는 편향, 유해성 또는 잘못된 정보를 포함할 수 있습니다. 책임있는 배포에는 필터링, 인간 감독 및 지속적인 모니터링이 필요합니다.

안전 조치

  • 콘텐츠 필터링: 유해하거나 부적절한 출력 차단
  • 사실 확인: 생성된 텍스트의 사실적 주장 검증
  • 인간 참여: 중요한 애플리케이션 검토
  • 귀속: AI 생성 콘텐츠를 명확히 표시
  • 편향 완화: 인구통계적 편향 모니터링 및 감소
