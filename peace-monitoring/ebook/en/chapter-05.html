5.1 RESTful API Architecture

WIA-UNI-015 Phase 2 defines a comprehensive RESTful API that enables programmatic access to peace monitoring data. The API follows modern best practices including versioning, pagination, filtering, and standardized error handling.

5.2 Authentication & Authorization

All API requests require Bearer token authentication. Tokens are obtained through OAuth 2.0 client credentials flow.

Token Acquisition

POST /auth/token
Content-Type: application/json

{
  "organizationId": "UN_COMMAND",
  "credentials": {
    "clientId": "unc-observer-001",
    "clientSecret": "***",
    "scope": ["read:monitoring", "write:reports"]
  }
}

Authorization Scopes

ScopePermissionsTypical Users
read:monitoringView monitoring events and sensor dataObservers, analysts
write:reportsSubmit monitoring events and reportsAuthorized reporters
write:inventorySubmit arms inventory declarationsGovernment officials
create:verificationRequest verification inspectionsIAEA, observers
admin:allFull access including system configurationSystem administrators

5.4 WebSocket Real-Time Streaming

For time-sensitive data, WIA-UNI-015 provides WebSocket connections for real-time event streaming.

Connection Example

const ws = new WebSocket('wss://api.wia.org/peace-monitoring/v1/stream');

ws.onopen = () => {
  // Authenticate
  ws.send(JSON.stringify({
    type: 'authenticate',
    token: 'Bearer {token}'
  }));

  // Subscribe to channels
  ws.send(JSON.stringify({
    type: 'subscribe',
    channels: ['dmz-sensors', 'critical-alerts']
  }));
};

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('Alert:', message);
};

Available Channels

  • dmz-sensors: Real-time sensor readings from DMZ monitoring network
  • critical-alerts: High-priority incidents requiring immediate attention
  • troop-movements: Declared military movements
  • verification-updates: Status changes on verification requests
  • cbm-activities: Confidence-building measure notifications

5.7 SDK Implementation

Official SDKs abstract API complexity and provide idiomatic interfaces for each language.

TypeScript/JavaScript SDK Example

import { PeaceMonitoringClient } from '@wia/peace-monitoring';

const client = new PeaceMonitoringClient({
  apiKey: process.env.WIA_API_KEY,
  environment: 'production'
});

// Get monitoring events
const events = await client.getMonitoringEvents({
  zone: 'DMZ',
  startDate: '2025-12-20T00:00:00Z',
  limit: 50
});

// Submit sensor data
await client.submitSensorData({
  sensorId: 'DMZ-SEISMIC-045',
  sensorType: 'SEISMIC',
  location: {latitude: 38.234, longitude: 127.345},
  readings: [{
    value: 2.4,
    unit: 'RICHTER',
    threshold: 2.0
  }]
});

// Real-time streaming
client.connectStream({
  channels: ['critical-alerts'],
  onMessage: (msg) => console.log('Alert:', msg),
  onError: (err) => console.error(err)
});
