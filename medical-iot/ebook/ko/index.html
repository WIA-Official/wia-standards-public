표준 소개

WIA-MEDICAL-IOT 표준은 안전하고 상호 운용 가능한 의료 사물 인터넷(IoMT) 시스템을 구현하기 위한 포괄적인 프레임워크를 제공합니다. 이 가이드는 연결된 의료 기기, 원격 환자 모니터링, 데이터 전송 프로토콜, 보안 프레임워크, 의료 정보 시스템과의 통합을 다룹니다.

弘益人間 (홍익인간) - 널리 인간을 이롭게 하라

의료 사물 인터넷은 병원 벽을 넘어 헬스케어를 확장하여 지속적인 모니터링, 조기 개입, 개인화된 치료를 가능하게 합니다. WIA-MEDICAL-IOT 표준은 이러한 연결된 기기가 안전하고 신뢰성 있게 상호 운용되어 어디서나 환자의 건강 결과를 개선하는 것을 목표로 합니다.

WIA 4단계 아키텍처

구현 프레임워크

WIA-MEDICAL-IOT 표준은 WIA 4단계 아키텍처 패턴을 따릅니다:

  • 1단계 - 데이터 계층: 기기 데이터 모델, 관측 리소스, 환자 인구 통계
  • 2단계 - API 계층: FHIR Device, Observation, 기기 관리용 REST 엔드포인트
  • 3단계 - 프로토콜 계층: Bluetooth PHD, IEEE 11073, MQTT, CoAP 메시징
  • 4단계 - 통합 계층: EHR 통합, 원격 모니터링 플랫폼, 분석
