심해 탐사선과 ROV

유인 잠수정부터 무인 로봇까지: 심해를 탐험하는 다양한 플랫폼과 그들의 독특한 능력, 장단점, 그리고 응용 분야.

탐사 플랫폼의 진화

심해 탐사는 세 가지 주요 플랫폼 유형으로 수행됩니다: 유인 잠수정(HOV: Human Occupied Vehicle), 원격 조종 무인 잠수정(ROV: Remotely Operated Vehicle), 그리고 자율 무인 잠수정(AUV: Autonomous Underwater Vehicle). 각 플랫폼은 고유한 강점과 제한 사항을 가지고 있으며, WIA-OCEAN-001 표준은 모든 유형에 대한 통합된 프레임워크를 제공합니다.

유인 잠수정 (HOV)

유인 잠수정은 과학자들이 직접 심해 환경을 관찰하고 의사 결정을 실시간으로 내릴 수 있게 합니다. 인간의 직관과 적응력은 예상치 못한 발견에 대응하는 데 있어 여전히 무인 시스템을 능가합니다.

원격 조종 무인 잠수정 (ROV)

ROV는 제대로 테더(케이블)를 통해 수면 선박과 연결되어 전력과 통신을 제공받습니다. 이는 장시간 작업과 무거운 도구 사용을 가능하게 하지만, 기동 범위가 테더 길이로 제한됩니다.

ROV 클래스

ROV는 크기와 능력에 따라 여러 클래스로 분류됩니다:

  • 관찰급(Observation Class): 소형, 경량, 주로 검사 및 모니터링
  • 작업급(Work Class): 대형, 강력한 매니퓰레이터, 복잡한 작업 수행
  • 검사급(Inspection Class): 중간 크기, 파이프라인 및 구조물 검사

한국의 해미래 ROV

KIOST가 개발한 해미래는 6,000미터 수심까지 작동 가능한 작업급 ROV입니다. 7기능 매니퓰레이터, 고해상도 카메라, 그리고 첨단 센서 패키지를 탑재하여 해저 자원 탐사, 생물학적 샘플링, 그리고 지질학적 조사를 수행합니다. 해미래는 한국이 심해 탐사 기술의 주요 국가임을 입증합니다.

자율 무인 잠수정 (AUV)

AUV는 테더 없이 독립적으로 작동하며, 사전 프로그래밍된 임무를 수행하거나 AI 기반 의사 결정을 사용합니다. 이는 넓은 지역 매핑, 장거리 조사, 그리고 사람이 접근할 수 없는 환경 탐사에 이상적입니다.

AUV의 장점

  • 테더 없이 자유로운 기동
  • 광범위한 지역 조사 가능
  • 낮은 운영 비용
  • 위험한 환경에서 안전한 작업

추진 및 기동 시스템

심해 탐사선은 정밀한 위치 제어가 필요합니다. 대부분은 스러스터(추진기) 배열을 사용하여 6자유도(DOF) 제어를 달성합니다: 전후, 좌우, 상하 이동 및 롤, 피치, 요 회전.

스러스터 배치

일반적인 작업급 ROV는 6-8개의 스러스터를 사용합니다. 수평 스러스터는 전진/후진 및 좌우 이동을 제어하고, 수직 스러스터는 상승/하강과 피치/롤 제어를 담당합니다.

TypeScript 구현: ROV 제어 시스템

interface Thruster {
  id: string;
  position: { x: number; y: number; z: number };
  orientation: { roll: number; pitch: number; yaw: number };
  maxThrust: number; // 뉴턴
  currentPower: number; // 0-100%
}

interface ROVState {
  position: { x: number; y: number; z: number };
  orientation: { roll: number; pitch: number; yaw: number };
  velocity: { vx: number; vy: number; vz: number };
  depth: number;
}

class ROVController {
  private thrusters: Thruster[];
  private state: ROVState;
  private targetDepth: number;

  constructor() {
    this.thrusters = this.initializeThrusters();
    this.state = this.getInitialState();
  }

  initializeThrusters(): Thruster[] {
    return [
      { id: 'FWD_PORT', position: {x: -0.5, y: 1, z: 0},
        orientation: {roll: 0, pitch: 0, yaw: 0},
        maxThrust: 500, currentPower: 0 },
      { id: 'FWD_STBD', position: {x: 0.5, y: 1, z: 0},
        orientation: {roll: 0, pitch: 0, yaw: 0},
        maxThrust: 500, currentPower: 0 },
      { id: 'VERT_FWD', position: {x: 0, y: 0.5, z: 0},
        orientation: {roll: 0, pitch: 90, yaw: 0},
        maxThrust: 300, currentPower: 0 },
      { id: 'VERT_AFT', position: {x: 0, y: -0.5, z: 0},
        orientation: {roll: 0, pitch: 90, yaw: 0},
        maxThrust: 300, currentPower: 0 }
      // ... 추가 스러스터
    ];
  }

  getInitialState(): ROVState {
    return {
      position: {x: 0, y: 0, z: 0},
      orientation: {roll: 0, pitch: 0, yaw: 0},
      velocity: {vx: 0, vy: 0, vz: 0},
      depth: 0
    };
  }

  descend(targetDepth: number): void {
    this.targetDepth = targetDepth;
    const depthError = targetDepth - this.state.depth;

    // 수직 스러스터 제어
    const vertThrusters = this.thrusters.filter(t =>
      t.id.startsWith('VERT'));

    vertThrusters.forEach(t => {
      t.currentPower = Math.min(100, depthError * 10);
    });
  }

  hover(): void {
    // 중성 부력 유지를 위한 스러스터 조정
    const vertThrusters = this.thrusters.filter(t =>
      t.id.startsWith('VERT'));

    vertThrusters.forEach(t => {
      t.currentPower = 50; // 안정화를 위한 기준 전력
    });
  }

  moveForward(speed: number): void {
    const fwdThrusters = this.thrusters.filter(t =>
      t.id.startsWith('FWD'));

    fwdThrusters.forEach(t => {
      t.currentPower = speed;
    });
  }
}

로봇 매니퓰레이터

작업급 ROV의 핵심은 정밀한 매니퓰레이터입니다. 이는 샘플 채취, 도구 사용, 그리고 복잡한 조립 작업을 수행합니다. 현대 매니퓰레이터는 7자유도 이상을 제공하여 인간 팔의 유연성을 모방합니다.

이사부호: 한국의 심해 연구선

KIOST의 이사부호는 5,900톤급 심해 연구선으로 해미래 ROV를 운용합니다. 동적 위치 제어 시스템을 갖춰 정밀한 위치 유지가 가능하며, 첨단 과학 장비와 실험실을 탑재하고 있습니다. 이사부호는 한국의 해양 탐사 능력의 상징입니다.

ROV 운용 안전

ROV 테더는 해저 장애물에 걸리거나 손상될 수 있습니다. WIA-OCEAN-001은 테더 관리 시스템, 비상 절단 장치, 그리고 회수 프로토콜을 요구합니다. 모든 임무는 상세한 위험 평가를 거쳐야 합니다.

요약: 핵심 사항

  • 추진 시스템: 6-8개 스러스터로 6DOF 제어
