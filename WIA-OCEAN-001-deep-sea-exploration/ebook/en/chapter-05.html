Communication and Navigation

Finding position and transmitting data through a medium that blocks radio, scatters light, and challenges every terrestrial communication paradigm.

The Underwater Communication Challenge

Electromagnetic waves—the foundation of terrestrial wireless communication—attenuate catastrophically in seawater. Conductivity creates short-circuit currents that dissipate radio energy within meters. Even extremely low frequency (ELF) radio requires megawatt transmitters and kilometer-scale antennas to achieve ranges of hundreds of kilometers at data rates measured in bits per minute.

This electromagnetic blackout fundamentally reshapes underwater operations. There is no underwater GPS. There is no high-bandwidth wireless. The ocean enforces physical laws that demand acoustic communication, dead-reckoning navigation, and creative solutions to problems solved trivially on land.

Acoustic Communication

Acoustic modems transmit data through the water column using sound waves, typically in the 10-30 kHz band balancing range (lower frequency) and bandwidth (higher frequency). The underwater acoustic channel presents unique challenges: severe multipath, time-varying Doppler shift, and ambient noise.

Modulation Schemes

Early underwater modems used simple frequency-shift keying (FSK), achieving 100-300 bits per second. Modern systems employ OFDM and phase-coherent methods like PSK. The WHOI Micro-Modem achieves 5,400 bits/second using PSK at 6 km range, or 80 bps at 20+ km range.

Channel Characteristics

The underwater acoustic channel defies terrestrial intuition. Multipath propagation creates multiple signal copies arriving at different times. In shallow water, dozens of arrivals can span tens of milliseconds, causing severe inter-symbol interference. Doppler shift from platform motion further complicates reception.

Acoustic Positioning Systems

With GPS unavailable underwater, acoustic ranging provides positioning. By measuring acoustic travel time between transceivers and applying trilateration, vehicles determine position relative to known reference beacons.

Ultra-Short Baseline (USBL)

USBL systems mount a compact transceiver array on the surface vessel hull. The array interrogates a transponder on the underwater vehicle, measuring range from travel time and bearing from phase differences. Sonardyne AvTrak 6 achieves 0.5-2% of slant range accuracy. At 3,000 meters, this translates to 15-60 meter error.

Long Baseline (LBL)

LBL deploys multiple transponder beacons on the seafloor surrounding the work area. The vehicle interrogates beacons, measuring range to each. Trilateration provides position accuracy of 0.5-5 meters over areas up to 100 km²—far superior to USBL.

Dead-Reckoning Navigation

Dead reckoning integrates sensor measurements—velocity from DVL, heading from compass, depth from pressure—to estimate position relative to a known starting point. Without external updates, errors accumulate. High-quality systems achieve 0.1% distance traveled error.

Inertial Navigation Systems (INS)

INS measures acceleration and angular rate using accelerometers and gyroscopes. Integrating acceleration yields velocity and position. The challenge: integration amplifies sensor errors. Accelerometer bias of 1 mg produces position error growing quadratically—1 meter after 45 seconds, 100 meters after 7 minutes.

DVL-Aided Navigation

Doppler velocity logs provide velocity measurements accurate to ±0.2 cm/s when operating above seafloor. Integrating DVL velocity produces position estimates with 0.1-0.5% distance traveled accuracy—dramatically better than pure INS.

Optimal navigation fuses INS and DVL using Kalman filtering. The INS provides high-rate attitude; DVL corrects velocity drift; periodic USBL updates correct position drift. Result: sub-meter accuracy over hours-long missions.

TypeScript Implementation: Navigation Filter

interface NavigationState {
  position: { x: number; y: number; z: number };
  velocity: { x: number; y: number; z: number };
  attitude: { roll: number; pitch: number; yaw: number };
  timestamp: Date;
}

interface DVLMeasurement {
  velocity: { x: number; y: number; z: number };
  altitude: number;
  valid: boolean;
  timestamp: Date;
}

class DeadReckoningNavigator {
  private state: NavigationState;
  private positionUncertainty: number = 0;

  updateDVL(measurement: DVLMeasurement): void {
    if (!measurement.valid) return;
    const dt = 1.0; // seconds
    this.state.position.x += measurement.velocity.x * dt;
    this.state.position.y += measurement.velocity.y * dt;
    this.state.position.z += measurement.velocity.z * dt;
    this.positionUncertainty += Math.sqrt(
      measurement.velocity.x ** 2 + measurement.velocity.y ** 2
    ) * dt * 0.001;
  }

  getCurrentState() {
    return { ...this.state, uncertainty: this.positionUncertainty };
  }
}

Terrain-Relative Navigation

When DVL loses bottom-lock or operating mid-water, terrain-relative navigation (TRN) matches measured seafloor topography from multibeam sonar against known bathymetric maps, providing position updates without external beacons.

Bathymetric SLAM

Simultaneous localization and mapping (SLAM) builds maps while using those maps for localization. Particle filters represent position uncertainty as thousands of hypothetical trajectories. Particles matching actual observations receive higher weights, converging toward the true trajectory.

Optical Communication

While acoustic modems dominate long-range communication, optical links provide high bandwidth over short distances. Blue-green laser light (450-550 nm) achieves 10-100 Mbps at ranges up to 100 meters—sufficient for vehicle-to-vehicle file transfer or downloading AUV data without recovery.

Navigation Failures and Contingencies

Lost Communication: ROVs losing tether signal must execute pre-programmed emergency procedures: halt, ascend, activate beacon.

Position Uncertainty: AUVs exceeding uncertainty thresholds (typically 50-100m) should abort and surface for GPS fix.

Acoustic Interference: Multiple systems can interfere. Frequency coordination prevents destructive interference.

Korean Navigation Infrastructure

KIOST maintains extensive bathymetric databases for Korean waters, enabling terrain-relative navigation in the East Sea, Yellow Sea, and Southern Ocean. The institute develops acoustic positioning systems optimized for shallow, high-noise environments typical of Korean coastal waters. Korean telecom companies explore underwater optical communication for coastal monitoring networks.

  • Terrain Navigation: SLAM enables GPS-denied operation
