3.1 The Detection-to-Catalog Pipeline

Transforming a faint signal or subtle anomaly into a verified registry entry involves a multi-stage process designed to eliminate false positives while capturing genuine discoveries. The WIA-CONTACT-010 standard defines this pipeline to ensure consistency across all contributing observatories and research institutions.

3.1.1 Five Stages from Signal to Civilization

┌─────────────────────────────────────────────────────────┐ │ DETECTION-TO-CATALOG PIPELINE │ ├─────────────────────────────────────────────────────────┤ │ │ │ Stage 1: INITIAL DETECTION │ │ └─ Automated systems flag anomalies │ │ └─ Human review of candidates │ │ └─ Output: Candidate technosignature (unconfirmed) │ │ │ │ ▼ │ │ │ │ Stage 2: VERIFICATION │ │ └─ Independent observation by different facilities │ │ └─ Rule out instrumental artifacts │ │ └─ Statistical significance assessment │ │ └─ Output: Confirmed technosignature (95% confidence) │ │ │ │ ▼ │ │ │ │ Stage 3: CHARACTERIZATION │ │ └─ Detailed signal analysis │ │ └─ Source localization │ │ └─ Multi-wavelength observations │ │ └─ Output: Characterized source with properties │ │ │ │ ▼ │ │ │ │ Stage 4: ATTRIBUTION │ │ └─ Technology vs. natural phenomenon determination │ │ └─ Civilization capability estimation │ │ └─ Threat/opportunity assessment │ │ └─ Output: Civilization profile (provisional) │ │ │ │ ▼ │ │ │ │ Stage 5: CATALOG ENTRY │ │ └─ Peer review and consensus │ │ └─ Provenance documentation │ │ └─ Registry integration │ │ └─ Output: Official catalog entry with unique ID │ │ │ └─────────────────────────────────────────────────────────┘

3.2 Detection Methods and Technosignatures

The registry categorizes technosignatures by detection method, as each approach has different strengths, limitations, and false-positive profiles.

3.2.1 Radio Technosignatures

The classic SETI approach: searching for narrow-band radio signals or structured patterns inconsistent with natural emission:

3.2.3 Atmospheric Biosignatures and Industrial Pollutants

While primarily indicating biology rather than technology, atmospheric anomalies can suggest industrial civilizations:

Technosignature: Industrial Atmospheric Pollution

Molecules like chlorofluorocarbons (CFCs), sulfur hexafluoride (SF6), or nitrogen trifluoride (NF3) have no known natural sources and persist in atmospheres for decades to centuries. Detection via high-resolution spectroscopy during planetary transits.

Challenge: Requires extremely large telescope apertures and long observation times. Earth's industrial atmosphere would be barely detectable at 10 parsecs with current technology.

Catalog Entry Requirements: 5-sigma detection of multiple industrial gases, ruling out exotic photochemistry through atmospheric modeling.

3.5.2 The "WOW Signal" Problem: Transient Detections

Case Study: One-Time Signals

The famous 1977 "Wow! signal" was a strong, narrowband radio detection that has never been repeated despite extensive follow-up observations. How should such cases be cataloged?

WIA-CONTACT-010 Policy:

  • Single-detection events cataloged as "historical candidates" with confidence capped at 0.60
  • Remain in registry indefinitely for cross-reference against future detections
  • Clearly marked as unconfirmed to prevent overinterpretation
  • Automated alerts if any future signal detected from same location

Philosophical Rationale: A single anomaly might represent a genuine but brief transmission, a civilization that ceased broadcasting, or a one-time natural phenomenon. Preserving these records serves future researchers while maintaining scientific caution.

3.7.2 Handling Contradictory Evidence

When new observations contradict previous conclusions, the registry employs a structured reconciliation process:

Contradiction Resolution Protocol:

  1. Flag entry for expert review when confidence drops > 0.2
  2. Convene ad-hoc committee of original observers + new observers
  3. Reanalyze all raw data with current best methods
  4. If reconciliation successful: Update entry with explanation
  5. If irreconcilable: Split into multiple candidate entries, each with lower confidence
  6. If proven false positive: Move to "archived detections" with full explanation
