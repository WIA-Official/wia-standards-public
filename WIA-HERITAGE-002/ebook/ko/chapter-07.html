07.2 블록체인 기반 출처 추적

문화재 불법 거래는 연간 60억 달러 규모의 암시장을 형성하고 있습니다. 약탈된 유물이 합법적인 경로를 거친 것처럼 위조된 서류와 함께 경매에 올라오는 경우가 빈번합니다. 블록체인의 불변성과 투명성은 이러한 문제를 근본적으로 해결합니다.

스마트 컨트랙트 구조

WIA-HERITAGE-002는 ERC-721 NFT 표준을 확장한 Heritage Token을 정의합니다. 각 유물은 고유한 토큰으로 발행되며, 발굴부터 현재까지의 모든 이력이 온체인에 기록됩니다:

// Solidity 스마트 컨트랙트 (Ethereum)
pragma solidity ^0.8.0;

contract HeritageToken is ERC721 {
    struct Artifact {
        string wia_id;           // WIA-HERITAGE-002 ID
        string ipfs_metadata;    // IPFS에 저장된 메타데이터
        uint256 discovery_date;  // 발굴 날짜 (timestamp)
        string site;             // 발굴지
        address discoverer;      // 최초 등록자
        bool is_authentic;       // 인증 여부
        string[] custody_chain;  // 보관 이력
    }

    mapping(uint256 => Artifact) public artifacts;
    mapping(string => uint256) public wiaIdToToken;

    event ArtifactRegistered(uint256 tokenId, string wia_id);
    event CustodyTransferred(uint256 tokenId, address from, address to);
    event AuthenticationUpdated(uint256 tokenId, bool status);

    function registerArtifact(
        string memory _wia_id,
        string memory _ipfs_metadata,
        string memory _site
    ) public returns (uint256) {
        uint256 tokenId = totalSupply() + 1;

        artifacts[tokenId] = Artifact({
            wia_id: _wia_id,
            ipfs_metadata: _ipfs_metadata,
            discovery_date: block.timestamp,
            site: _site,
            discoverer: msg.sender,
            is_authentic: false,
            custody_chain: new string[](0)
        });

        wiaIdToToken[_wia_id] = tokenId;
        _mint(msg.sender, tokenId);

        emit ArtifactRegistered(tokenId, _wia_id);
        return tokenId;
    }

    function transferCustody(uint256 _tokenId, address _to) public {
        require(ownerOf(_tokenId) == msg.sender, "Not the owner");

        artifacts[_tokenId].custody_chain.push(
            string(abi.encodePacked(
                block.timestamp, ":", msg.sender, "->", _to
            ))
        );

        _transfer(msg.sender, _to, _tokenId);
        emit CustodyTransferred(_tokenId, msg.sender, _to);
    }
}

07.3 AI 기반 유물 자동 분류

전문 고고학자가 유물을 분류하고 연대를 추정하는 데는 수일에서 수주가 걸립니다. 딥러닝 모델은 수십만 개의 유물 이미지를 학습하여 몇 초 만에 시대, 문화권, 용도를 90% 이상의 정확도로 예측할 수 있습니다.

컴퓨터 비전 파이프라인

WIA-HERITAGE-002는 ResNet-152 기반 분류 모델을 표준으로 채택합니다. 이 모델은 ImageNet에서 사전 학습된 후, 고고학 데이터셋으로 전이 학습(Transfer Learning)됩니다:

멀티모달 학습

이미지뿐만 아니라 3D 스캔, X-ray, 화학 조성 데이터를 결합하면 정확도가 크게 향상됩니다:

class MultiModalClassifier(nn.Module):
    def __init__(self):
        super().__init__()
        self.image_encoder = ResNet152()
        self.point_cloud_encoder = PointNet()
        self.spectral_encoder = MLP([512, 256, 128])

        self.fusion = nn.Linear(128 + 256 + 128, 1247)

    def forward(self, image, point_cloud, spectral_data):
        img_features = self.image_encoder(image)
        pc_features = self.point_cloud_encoder(point_cloud)
        spec_features = self.spectral_encoder(spectral_data)

        combined = torch.cat([img_features, pc_features, spec_features], dim=1)
        output = self.fusion(combined)
        return output

이 멀티모달 접근법은 단일 이미지 분류 대비 정확도를 평균 7.2% 향상시킵니다.

07.4 GAN을 이용한 유물 디지털 복원

대부분의 고고학 유물은 시간의 흐름에 따라 파손되거나 일부가 유실됩니다. 전통적으로 복원 전문가가 수작업으로 복원하지만, 이는 시간과 비용이 많이 듭니다. Generative Adversarial Networks(GAN)는 파손된 부분을 자동으로 추정하여 원형에 가까운 디지털 복원을 수행합니다.

Pix2Pix 기반 2D 복원

평면 유물(벽화, 문서 등)의 복원에는 Pix2Pix GAN을 사용합니다:

import tensorflow as tf
from tensorflow.keras import layers

def build_generator():
    # U-Net 아키텍처
    inputs = layers.Input(shape=[256, 256, 3])

    # 인코더
    down1 = downsample(64)(inputs)
    down2 = downsample(128)(down1)
    down3 = downsample(256)(down2)
    down4 = downsample(512)(down3)

    # 디코더
    up1 = upsample(512)(down4, down3)
    up2 = upsample(256)(up1, down2)
    up3 = upsample(128)(up2, down1)
    up4 = upsample(64)(up3, inputs)

    outputs = layers.Conv2D(3, 4, padding='same', activation='tanh')(up4)

    return tf.keras.Model(inputs=inputs, outputs=outputs)

# 학습 데이터: (파손된 이미지, 완전한 이미지) 쌍
generator = build_generator()
discriminator = build_discriminator()

# 손실 함수
def generator_loss(disc_generated_output, gen_output, target):
    gan_loss = loss_object(tf.ones_like(disc_generated_output), disc_generated_output)
    l1_loss = tf.reduce_mean(tf.abs(target - gen_output))
    total_loss = gan_loss + (100 * l1_loss)  # L1 손실 가중치
    return total_loss

# 복원 실행
damaged_image = load_image('damaged_artifact.jpg')
restored_image = generator(damaged_image, training=False)
save_image(restored_image, 'restored_artifact.jpg')

PointNet++를 이용한 3D 복원

3D 유물의 경우 포인트 클라우드 데이터를 직접 처리하는 PointNet++ 기반 모델을 사용합니다:

import torch
from pointnet2_ops import pointnet2_utils

class PointCloudCompletion(nn.Module):
    def __init__(self):
        super().__init__()
        # 인코더: 부분 포인트 클라우드 → 잠재 벡터
        self.encoder = PointNet2Encoder(input_dim=3, output_dim=512)
        # 디코더: 잠재 벡터 → 완전한 포인트 클라우드
        self.decoder = PointNet2Decoder(latent_dim=512, output_points=16384)

    def forward(self, partial_pc):
        # partial_pc: [B, N, 3] (예: N=2048 포인트)
        latent = self.encoder(partial_pc)
        # latent: [B, 512]
        complete_pc = self.decoder(latent)
        # complete_pc: [B, 16384, 3]
        return complete_pc

# 사용 예제
model = PointCloudCompletion().cuda()
partial_points = load_partial_scan('broken_vase.ply')  # [1, 2048, 3]
complete_points = model(partial_points)
save_point_cloud(complete_points, 'restored_vase.ply')

07.5 WebXR 가상 박물관

물리적 박물관은 공간, 보안, 접근성 제약이 있습니다. 가상 박물관은 이러한 한계를 초월하여 전 세계 누구나 집에서 VR 헤드셋이나 웹 브라우저로 문화유산을 체험할 수 있게 합니다.

WebXR API를 이용한 구현

WIA-HERITAGE-002는 WebXR Device API를 사용하여 VR, AR, 일반 웹 브라우저 모두에서 작동하는 크로스 플랫폼 가상 전시를 구현합니다:

import * as THREE from 'three';
import { VRButton } from 'three/examples/jsm/webxr/VRButton.js';
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader.js';

// 장면 설정
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer({ antialias: true });

renderer.setSize(window.innerWidth, window.innerHeight);
renderer.xr.enabled = true;
document.body.appendChild(renderer.domElement);
document.body.appendChild(VRButton.createButton(renderer));

// 가상 전시관 환경
const loader = new GLTFLoader();
loader.load('virtual_museum_hall.glb', (gltf) => {
    scene.add(gltf.scene);
});

// 유물 로드
async function loadArtifacts() {
    const response = await fetch('https://api.heritage.wia.org/v1/collections/silla-gold');
    const artifacts = await response.json();

    artifacts.data.forEach((artifact, index) => {
        loader.load(artifact.model_url, (gltf) => {
            const model = gltf.scene;
            model.position.set(index * 2, 1.5, -5);

            // 설명 패널 추가
            const infoPanel = createInfoPanel(artifact);
            infoPanel.position.set(index * 2, 2.5, -4.8);
            scene.add(infoPanel);

            scene.add(model);
        });
    });
}

// VR 컨트롤러 상호작용
const controller = renderer.xr.getController(0);
controller.addEventListener('selectstart', onSelectStart);
controller.addEventListener('selectend', onSelectEnd);

function onSelectStart(event) {
    // 레이캐스팅으로 유물 선택
    const raycaster = new THREE.Raycaster();
    raycaster.setFromXRController(controller);
    const intersects = raycaster.intersectObjects(scene.children, true);

    if (intersects.length > 0) {
        const selectedArtifact = intersects[0].object;
        // 상세 정보 표시
        showDetailView(selectedArtifact.userData.wia_id);
    }
}

// 애니메이션 루프
renderer.setAnimationLoop(() => {
    renderer.render(scene, camera);
});

loadArtifacts();

            const pose = hit.getPose(referenceSpace);

});

  • 색각 이상 모드: 색맹/색약 사용자를 위한 필터

07.6 지식 그래프와 시맨틱 검색

고고학 데이터는 단순한 개별 유물의 집합이 아니라, 복잡한 관계망으로 연결된 지식 체계입니다. 지식 그래프(Knowledge Graph)는 유물, 발굴지, 문화권, 시대, 인물 간의 관계를 명시적으로 모델링하여 통찰력 있는 질문에 답할 수 있게 합니다.

Neo4j 그래프 데이터베이스

WIA-HERITAGE-002는 Neo4j를 사용하여 유물 간 관계를 저장합니다:

// Cypher 쿼리 언어로 데이터 모델링
CREATE (artifact:Artifact {
    wia_id: 'artifact-12345',
    name: '금관총 금관',
    date_range: '5th-6th century CE'
})

CREATE (site:Site {
    name: 'Geumgwanchong Tomb',
    location: point({latitude: 35.8328, longitude: 129.2097})
})

CREATE (period:Period {
    name: 'Silla Dynasty',
    start_year: 57,
    end_year: 935
})

CREATE (material:Material {name: 'Gold'})
CREATE (material2:Material {name: 'Jade'})

// 관계 정의
CREATE (artifact)-[:DISCOVERED_AT]->(site)
CREATE (artifact)-[:BELONGS_TO_PERIOD]->(period)
CREATE (artifact)-[:MADE_OF]->(material)
CREATE (artifact)-[:MADE_OF]->(material2)
CREATE (site)-[:IN_PERIOD]->(period)

복잡한 쿼리 예제

지식 그래프를 사용하면 SQL로는 어려운 질문에 쉽게 답할 수 있습니다:

// "신라 시대에 금으로 만든 유물이 발견된 모든 발굴지를 찾고,
// 각 발굴지에서 발견된 유물 수를 내림차순으로 정렬"
MATCH (period:Period {name: 'Silla Dynasty'})
      <-[:BELONGS_TO_PERIOD]-(artifact:Artifact)
      -[:MADE_OF]->(material:Material {name: 'Gold'})
MATCH (artifact)-[:DISCOVERED_AT]->(site:Site)
RETURN site.name AS 발굴지,
       count(artifact) AS 유물수,
       collect(artifact.name)[0..5] AS 예시유물
ORDER BY 유물수 DESC

// "이 유물과 비슷한 시기, 지역, 재질의 유물 추천"
MATCH (target:Artifact {wia_id: 'artifact-12345'})
      -[:BELONGS_TO_PERIOD]->(period:Period)
MATCH (target)-[:DISCOVERED_AT]->(site:Site)
MATCH (similar:Artifact)-[:BELONGS_TO_PERIOD]->(period)
MATCH (similar)-[:DISCOVERED_AT]->(nearSite:Site)
WHERE distance(site.location, nearSite.location) < 50000  // 50km 이내
  AND similar.wia_id <> target.wia_id
RETURN similar.name, similar.wia_id
LIMIT 10

07.7 고대 문자 자동 인식 및 번역

수천 개의 고대 비문과 문서가 해독되지 않은 채로 박물관 수장고에 보관되어 있습니다. 딥러닝 기반 OCR과 기계 번역은 이러한 자료를 대량으로 디지털화하고 번역하여 연구자와 대중에게 공개합니다.

Transformer 기반 고대 문자 인식

WIA-HERITAGE-002는 TrOCR(Transformer-based OCR) 모델을 사용합니다:

from transformers import TrOCRProcessor, VisionEncoderDecoderModel
from PIL import Image

# 사전 학습된 모델 로드 (한자, 한글 고문서에 특화)
processor = TrOCRProcessor.from_pretrained('wia/heritage-ocr-hanja')
model = VisionEncoderDecoderModel.from_pretrained('wia/heritage-ocr-hanja')

# 고대 문서 이미지 처리
image = Image.open('joseon_document.jpg')
pixel_values = processor(image, return_tensors='pt').pixel_values

# OCR 수행
generated_ids = model.generate(pixel_values)
generated_text = processor.batch_decode(generated_ids, skip_special_tokens=True)[0]

print("인식된 텍스트:", generated_text)
# 출력: "景福宮重建都監儀軌"

다국어 번역

인식된 텍스트는 자동으로 현대 언어로 번역됩니다:

from transformers import MarianMTModel, MarianTokenizer

# 한문 → 한국어 번역
model_name = 'wia/classical-chinese-to-korean'
tokenizer = MarianTokenizer.from_pretrained(model_name)
model = MarianMTModel.from_pretrained(model_name)

text = "景福宮重建都監儀軌"
translated = model.generate(**tokenizer(text, return_tensors="pt"))
result = tokenizer.decode(translated[0], skip_special_tokens=True)

print("번역:", result)
# 출력: "경복궁 중건 도감 의궤"

# 추가 번역 (한국어 → 영어, 중국어, 일본어 등 100개 언어)
translate_to_multiple_languages(result, target_langs=['en', 'zh', 'ja', 'es'])
