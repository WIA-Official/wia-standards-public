# DALL-E Image Prompts for WIA-COMP-015 (Open Source Standard)

## Image 1: Main Hero - Global Open Source Collaboration
**Style**: Modern, collaborative, interconnected
**Color Scheme**: Dark navy background (#0f172a), blue accents (#3B82F6), vibrant global network colors
**Mood**: Inclusive, collaborative, empowering

**Prompt**:
```
A stunning 3D visualization of global open source collaboration on a dark navy (#0f172a) background. In the center, a large glowing blue (#3B82F6) globe with interconnected nodes representing developers worldwide. From each node, streams of code flow between continents forming a beautiful network. Around the globe, floating license badges (MIT, Apache, GPL) glow softly. Holographic code contribution graphs show pull requests merging, issues being resolved, and stars increasing. Diversity of developers represented by avatars from different countries collaborating on GitHub-like interfaces. Text overlays show "10M+ Contributors", "OSI Approved", "Global Community". Fork symbols and merge icons animate throughout. Hyper-realistic, inclusive, celebratory atmosphere with bright blue and green collaboration highlights.
```

## Image 2: Problem Visualization - Closed Source Isolation
**Style**: Dark, isolated, restricted
**Color Scheme**: Muted grays, dark tones, red restriction symbols
**Mood**: Limiting, siloed, expensive

**Prompt**:
```
A stark visualization of closed-source software problems. Center shows a developer sitting alone in a dark, isolated office surrounded by locked file cabinets with chain locks and "PROPRIETARY" stamps. Computer screens display "Access Denied", "License Expired", expensive price tags ($50,000+ licenses), and legal restriction warnings. Behind protective barriers, valuable code sits unused while outside developers pound on glass walls wanting to contribute. A ticking license meter shows mounting costs. Lawyers with contracts hover threateningly. In the background, innovation being blocked by license incompatibility diagrams showing red X marks. Community collaboration graphs trending to zero. Separate silos prevent knowledge sharing. Muted grays with red restriction symbols and warning signs everywhere. Documentary style showing corporate bureaucracy stifling creativity.
```

## Image 3: Solution/Future - Open Source Abundance
**Style**: Bright, abundant, thriving
**Color Scheme**: Bright blues, greens (growth), vibrant colors, optimistic
**Mood**: Empowering, abundant, innovative

**Prompt**:
```
A beautiful vision of open source abundance in bright daylight. Center shows a vibrant digital plaza where diverse developers from every continent collaborate joyfully. Large floating screens display thriving open source projects with thousands of stars, active forks, and pull requests flowing smoothly. License badges (MIT, Apache, GPL) appear as friendly, welcoming gates rather than barriers. In the foreground, a developer freely downloads, modifies, and contributes back to projects - showing the complete open source cycle. Background displays innovation graphs soaring upward, community size growing exponentially, and global software quality improving. Knowledge flows freely between developers represented by glowing streams of shared wisdom. Companies and individuals both contribute and benefit equally. Gardens of code growing organically with community care. Everything bright, open, accessible. Celebration of collaboration with developers high-fiving across continents. Bright blues, vibrant greens, and rainbow diversity colors. Apple-style optimistic aesthetic celebrating human connection through code.
```

---

**Design Notes**:
- Image 1: Main hero - shows WIA-COMP-015 enabling global collaboration
- Image 2: Problem statement - emphasizes restrictions and isolation of closed source
- Image 3: Future vision - demonstrates abundance and innovation of open source culture
- Blue (#3B82F6) is the primary brand color for COMP category
- Emphasize community, collaboration, and freedom
- Show diversity and global inclusivity
