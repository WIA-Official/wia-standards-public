Chapter 3: API Endpoints

3.1 Project Management APIs

RESTful APIs for managing biotechnology research projects.

Project Endpoints

POST /api/v1/projects

Create a new biotech project

Request Body
{
  "project_info": {
    "name": "CRISPR Gene Therapy Development",
    "description": "Development of CRISPR-based therapy for genetic disorder X",
    "category": "gene_therapy"
  },
  "organization": {
    "name": "Research Institute",
    "department": "Molecular Biology",
    "principal_investigator": "Dr. Kim",
    "contact_email": "contact@example.org"
  },
  "ethics": {
    "irb_approval": "IRB-2025-0001",
    "biosafety_level": "BSL-2",
    "gmo_approval": "GMO-2025-001"
  }
}
Response (201 Created)
{
  "project_id": "proj-20251215-001",
  "status": "created",
  "created_at": "2025-12-15T10:00:00Z"
}
GET /api/v1/projects/{project_id}

Retrieve project details

PUT /api/v1/projects/{project_id}

Update project metadata

DELETE /api/v1/projects/{project_id}

Archive project (soft delete)

3.2 Sequence Upload APIs

Upload and manage gene sequences with FASTA/GenBank compatibility.

POST /api/v1/projects/{project_id}/sequences

Upload a new sequence

Request Body (Multipart)
{
  "sequence_info": {
    "name": "Target Gene ABC",
    "organism": "Homo sapiens",
    "gene_symbol": "ABC1"
  },
  "sequence_type": "dna",
  "fasta_file": "@sequence.fasta"
}
Response (201 Created)
{
  "sequence_id": "seq-001",
  "length_bp": 2500,
  "gc_content": 0.52,
  "checksum": "sha256:abc123...",
  "status": "uploaded"
}
GET /api/v1/sequences/{sequence_id}

Retrieve sequence metadata

GET /api/v1/sequences/{sequence_id}/fasta

Download FASTA file

POST /api/v1/sequences/{sequence_id}/annotations

Add feature annotations

3.3 CRISPR Experiment Tracking

APIs for managing CRISPR experiments from gRNA design to results analysis.

POST /api/v1/projects/{project_id}/crispr-experiments

Create a new CRISPR experiment

Request Body
{
  "experiment_info": {
    "name": "Gene ABC Knockout Study",
    "experimenter": "Dr. Lee"
  },
  "target": {
    "gene_symbol": "ABC1",
    "cell_line": "HEK293T",
    "sequence_ref": "seq-001"
  },
  "editing_system": {
    "type": "crispr_cas9",
    "cas_variant": "SpCas9",
    "delivery_method": "lipofection"
  },
  "guide_rnas": [
    {
      "name": "gRNA-ABC-1",
      "sequence": "ATCGATCGATCGATCGATCG",
      "target_start": 500,
      "pam": "TGG"
    }
  ],
  "editing_type": "knockout"
}
Response (201 Created)
{
  "experiment_id": "crispr-exp-001",
  "status": "created",
  "grna_count": 1,
  "created_at": "2025-12-15T11:00:00Z"
}
POST /api/v1/crispr-experiments/{experiment_id}/results

Upload experiment results (NGS data)

Request Body
{
  "analysis_date": "2025-12-16",
  "analysis_method": "NGS",
  "editing_efficiency": {
    "overall_efficiency": 0.78,
    "indel_frequency": 0.75
  },
  "fastq_files": [
    "@sample-001.fastq.gz",
    "@sample-002.fastq.gz"
  ]
}
GET /api/v1/crispr-experiments/{experiment_id}/off-targets

Get predicted off-target sites

3.4 Protein Structure APIs

AlphaFold integration and structure prediction management.

POST /api/v1/projects/{project_id}/structures

Upload protein structure prediction

Request Body (Multipart)
{
  "structure_info": {
    "name": "ABC1 Protein",
    "uniprot_id": "P12345"
  },
  "prediction": {
    "method": "alphafold",
    "version": "AlphaFold 2.3"
  },
  "pdb_file": "@structure.pdb",
  "confidence_file": "@confidence.json"
}
Response (201 Created)
{
  "structure_id": "struct-001",
  "mean_plddt": 85.5,
  "confident_residues_pct": 0.78,
  "status": "uploaded"
}
GET /api/v1/structures/{structure_id}/pdb

Download PDB file

GET /api/v1/structures/{structure_id}/confidence

Get confidence metrics (pLDDT, PAE)

POST /api/v1/structures/predict

Request AlphaFold prediction (async)

3.5 Synthetic Biology Part APIs

BioBrick and SBOL part management.

POST /api/v1/projects/{project_id}/parts

Register a new BioBrick part

Request Body
{
  "part_info": {
    "name": "Strong Constitutive Promoter",
    "part_type": "promoter"
  },
  "registry": {
    "source": "iGEM Registry",
    "registry_id": "BBa_J23100"
  },
  "sequence": {
    "dna": "TTGACGGCTAGCTCAGTCCTAGGTACAGTGCTAGC"
  },
  "assembly": {
    "standard": "biobrick"
  }
}
POST /api/v1/projects/{project_id}/assemblies

Design multi-part assembly

Request Body
{
  "assembly_info": {
    "name": "GFP Expression Cassette"
  },
  "assembly_method": "golden_gate",
  "parts": [
    {"position": 1, "part_ref": "part-001", "type": "promoter"},
    {"position": 2, "part_ref": "part-002", "type": "rbs"},
    {"position": 3, "part_ref": "part-003", "type": "cds"},
    {"position": 4, "part_ref": "part-004", "type": "terminator"}
  ]
}
GET /api/v1/parts/{part_id}/sbol

Export part as SBOL format
