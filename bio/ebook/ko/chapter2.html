2.1 유전자 서열 형식 (Gene Sequence Format)

WIA 표준은 완전한 FASTA 호환성을 갖춘 생물학적 서열 데이터를 위한 포괄적인 JSON 스키마를 정의합니다.

서열 스키마 구조

JSON - sequence.json
{
  "$schema": "https://wia.live/schemas/bio/sequence.schema.json",
  "sequence_id": "seq-001",
  "sequence_info": {
    "name": "대상 유전자 ABC",
    "description": "질병 X와 관련된 인간 유전자 ABC",
    "organism": "Homo sapiens",
    "taxonomy_id": 9606,
    "gene_symbol": "ABC1"
  },
  "sequence_type": "dna",
  "length_bp": 2500,
  "gc_content": 0.52,
  "annotations": [
    {
      "feature": "exon",
      "start": 1,
      "end": 500,
      "name": "엑손 1"
    }
  ]
}

서열 유형

유형 코드 설명
DNAdna데옥시리보핵산
RNArna리보핵산
mRNAmrna메신저 RNA
단백질protein아미노산 서열
합성synthetic합성 구성

2.2 CRISPR 실험 데이터 (CRISPR Experiment Data)

가이드 RNA (gRNA) 설계부터 결과 분석까지 CRISPR 유전자 편집 실험을 위한 완전한 스키마입니다.

CRISPR 실험 스키마

JSON - crispr-experiment.json
{
  "$schema": "https://wia.live/schemas/bio/crispr-experiment.schema.json",
  "experiment_id": "crispr-exp-001",
  "experiment_info": {
    "name": "유전자 ABC 녹아웃 연구",
    "experimenter": "이 박사"
  },
  "target": {
    "gene_symbol": "ABC1",
    "cell_line": "HEK293T"
  },
  "editing_system": {
    "type": "crispr_cas9",
    "cas_variant": "SpCas9",
    "delivery_method": "lipofection"
  },
  "guide_rnas": [
    {
      "grna_id": "grna-001",
      "name": "gRNA-ABC-1",
      "sequence": "ATCGATCGATCGATCGATCG",
      "pam": "TGG",
      "on_target_score": 0.88
    }
  ],
  "editing_type": "knockout"
}

편집 유형

유형 코드 설명
녹아웃knockoutNHEJ를 통한 유전자 파괴
녹인knockinHDR을 통한 유전자 삽입
염기 편집base_edit단일 염기 변환
프라임 편집prime_editpegRNA를 사용한 정밀 편집
CRISPRicrispri전사 억제
CRISPRacrispra전사 활성화

2.3 단백질 구조 데이터 (Protein Structure Data)

신뢰도 메트릭 (pLDDT, PAE)이 포함된 AlphaFold 호환 단백질 구조 스키마입니다.

구조 스키마

JSON - structure.json
{
  "$schema": "https://wia.live/schemas/bio/structure.schema.json",
  "structure_id": "struct-001",
  "structure_info": {
    "name": "ABC1 단백질",
    "description": "인간 ABC1 단백질의 3D 구조"
  },
  "prediction": {
    "method": "alphafold",
    "version": "AlphaFold 2.3"
  },
  "quality": {
    "mean_plddt": 85.5,
    "confident_residues_pct": 0.78
  }
}

예측 방법

방법 코드 설명
AlphaFoldalphafoldDeepMind AI 예측
AlphaFold3alphafold3복합체 예측
RoseTTAFoldrosettafoldBaker Lab 예측
ESMFoldesmfoldMeta AI 예측
실험적experimentalX-ray, NMR, Cryo-EM

2.4 합성생물학 부품 (Synthetic Biology Parts)

모듈식 DNA 조립을 위한 BioBrick 및 SBOL 호환 부품 정의입니다.

부품 스키마

JSON - part.json
{
  "$schema": "https://wia.live/schemas/bio/part.schema.json",
  "part_id": "part-001",
  "part_info": {
    "name": "강력한 구성적 프로모터",
    "part_type": "promoter"
  },
  "registry": {
    "source": "iGEM Registry",
    "registry_id": "BBa_J23100"
  },
  "sequence": {
    "dna": "TTGACGGCTAGCTCAGTCCTAGGTACAGTGCTAGC"
  },
  "assembly": {
    "standard": "biobrick"
  }
}

부품 유형

유형 코드 설명
