6.4 접근 제어 및 권한 부여

역할 기반 액세스 제어 (RBAC)

Python - RBAC 구현
from enum import Enum

class Role(Enum):
    PI = "principal_investigator"
    RESEARCHER = "researcher"
    TECHNICIAN = "technician"
    VIEWER = "viewer"

class Permission(Enum):
    READ = "read"
    WRITE = "write"
    DELETE = "delete"
    SHARE = "share"
    ADMIN = "admin"

ROLE_PERMISSIONS = {
    Role.PI: {Permission.READ, Permission.WRITE, Permission.DELETE, Permission.SHARE, Permission.ADMIN},
    Role.RESEARCHER: {Permission.READ, Permission.WRITE, Permission.SHARE},
    Role.TECHNICIAN: {Permission.READ, Permission.WRITE},
    Role.VIEWER: {Permission.READ}
}

6.5 준수 체크리스트

  • 저장 시 데이터 암호화 (AES-256) 및 전송 중 (TLS 1.3)
  • 인간 유전 데이터의 비식별화
  • 인간 피험자 연구를 위한 IRB 승인
  • GMO 작업을 위한 생물안전성 위원회 승인
  • 역할 기반 권한을 갖춘 접근 제어
  • 포괄적인 감사 로깅 (5년 보존)
  • 문서화된 사고 대응 계획
  • 정기적인 보안 평가
  • 데이터 보존 및 삭제 정책
  • 제3자 데이터 공유 계약
