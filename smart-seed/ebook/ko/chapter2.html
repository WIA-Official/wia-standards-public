🔐 품종 등록 및 지적재산권 보호

품종보호제도, UPOV 통합, 블록체인 기반 육종가권 보호 방법을 학습합니다.

2.1 품종보호권 제도

품종보호권은 육종가에게 신품종에 대한 20-25년간의 독점권을 부여합니다. 종자산업법에 따라 국립종자원에서 품종보호 출원을 심사합니다.

  • 구별성: 기존 품종과 명확히 구별됨
  • 균일성: 주요 특성이 충분히 균일함
  • 안정성: 증식 과정에서 특성이 불변함

2.2 국립종자원 연계

WIA 시스템은 국립종자원의 품종 등록 시스템과 연계하여 다음을 지원합니다:

  • 품종보호 출원 자동 제출
  • 재배 시험 데이터 디지털 기록
  • 품종 명칭 중복 검사
  • 육성자권 블록체인 등록

2.3 블록체인 기반 IP 보호

모든 품종 등록은 블록체인에 암호화 증명과 함께 기록되어 소급 날짜 조작을 방지하고 육종가권의 글로벌 인정을 보장합니다.

2.4 품종 등록 절차

WIA 플랫폼을 통한 품종 등록 단계:

  1. 품종 정보 입력 (학명, 특성, 육성자)
  2. 재배 시험 데이터 제출
  3. 국립종자원 출원 자동 전송
  4. 블록체인 타임스탬프 기록
  5. UPOV PLUTO 데이터베이스 동기화

요약

  • 품종보호권은 육종가에게 20-25년 독점권 부여
  • 구별성, 균일성, 안정성(DUS) 기준 충족 필요
  • 국립종자원 시스템과 자동 연계
  • 블록체인으로 불변의 육성자권 기록 보장
  • UPOV PLUTO와 글로벌 IP 보호 통합
