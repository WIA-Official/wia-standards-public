공역 통합 및 분류

UTM과 기존 항공 시스템의 조화로운 통합

기존 공역 분류 시스템

무인 교통 관리를 이해하려면 먼저 기존 공역 구조를 이해해야 합니다. 전 세계의 항공 당국은 공역을 클래스로 나누어 각각 고유한 규칙, 통신 요구 사항 및 진입 권한을 가지고 있습니다. 미국에서 FAA는 클래스 A부터 G까지 공역을 정의하며, UTM은 주로 클래스 G(비관제 공역)와 일부 통제 공역의 저고도 부분에서 작동합니다.

공역 클래스 고도 범위 ATC 서비스 드론 작업 승인 요구 사항
클래스 A 18,000' MSL 이상 모든 항공기 제어 일반적으로 해당 없음 매우 제한적, 특수 허가
클래스 B 지상부터 10,000' MSL 모든 항공기 제어 LAANC 또는 파트 107 면제 사전 ATC 승인 필수
클래스 C 지상부터 4,000' AGL IFR/VFR 분리 LAANC 승인 ATC 통신 필수
클래스 D 지상부터 2,500' AGL VFR 조언 LAANC 승인 타워와 통신
클래스 E 다양함 (700'/1,200' AGL) IFR 제어 허용됨 (제한적) 일반적으로 승인 불필요
클래스 G 지상부터 14,500' MSL ATC 서비스 없음 허용됨 (400' AGL 미만) UTM 조정만

클래스 G 공역: UTM의 주요 영역

클래스 G 공역은 비관제 공역으로 ATC 서비스가 제공되지 않는 곳입니다. 미국에서 대부분의 저고도 공역(공항에서 멀리 떨어진)은 클래스 G입니다. 이것이 UTM이 초점을 맞추는 곳입니다—수천 개의 드론이 인간 관제사의 직접적인 감독 없이 안전하게 작동할 수 있도록 하는 자율 조정 시스템을 제공합니다.

클래스 G에서 Part 107에 따라 운영되는 드론은 다음을 준수해야 합니다:

  • 400피트 AGL 미만 유지
  • 시각선(VLOS) 내에서 유지 (면제가 없는 한)
  • 사람들 위로 비행 규칙 준수
  • 유인 항공기에 양보
  • 원격 ID 브로드캐스트 (2023년 9월부터)
弘益人間 (홍익인간) - 널리 인간을 이롭게 하라

UTM이 클래스 G 공역에 초점을 맞추는 것은 공평성의 문제입니다. 클래스 G는 누구나 접근할 수 있는 공역입니다—대형 항공사, 군대 또는 부유한 기업만이 아닙니다. 클래스 G에서 안전하고 확장 가능한 드론 작업을 가능하게 함으로써 UTM은 소규모 기업, 연구원, 공공 안전 기관 및 커뮤니티가 드론 기술의 혜택을 받을 수 있도록 합니다.

통제 공역에서의 UTM

클래스 B, C, D 공역(주로 공항 주변)에서 작동하는 드론은 ATC 승인이 필요합니다. 전통적으로 이것은 느리고 수동적인 프로세스였습니다—운영자는 양식을 제출하고 몇 주를 기다렸습니다. LAANC(저고도 인증 및 알림 기능)는 이를 혁신했습니다.

UAS 시설 지도 (USFM)

UAS 시설 지도는 UTM 공역 관리의 기초입니다. USFM은 공항 주변의 허용 가능한 최대 드론 고도를 정의하는 3D 그리드입니다. 공항의 모든 활주로에 대해 USFM은 지리적 위치에 따라 0피트(작업 없음)에서 400피트(전체 작업)까지의 고도 제한을 지정합니다.

USFM 그리드 구조

USFM은 공항을 중심으로 한 여러 동심 구역으로 구성됩니다:

  • 표면 구역(0'): 공항 바로 주변, 드론 작업 금지
  • 100' 구역: 공항에서 약간 떨어진 곳, 최대 100피트 허용
  • 200' 구역: 추가로 떨어진 곳, 최대 200피트
  • 300' 구역: 최대 300피트
  • 400' 구역: 공항 영향 밖, 전체 Part 107 고도

그리드는 활주로 방향, 접근 경로 및 공항 교통 패턴을 기반으로 합니다. 활주로 끝의 접근 경로는 일반적으로 측면 영역보다 더 제한적입니다.

USFM과 LAANC 통합

USS는 비행 계획을 평가할 때 FIMS에서 USFM 데이터를 쿼리합니다. 요청된 작업이 해당 위치의 USFM 고도 한도 내에 있으면 즉시 승인될 수 있습니다(LAANC를 통해). 한도를 초과하는 경우 인간 ATC 검토가 필요하며 처리하는 데 더 오래 걸립니다.

공항으로부터의 거리 USFM 최대 고도 일반적인 승인 시간 검토 유형
0 - 0.5 마일 0 피트 (금지) 즉시 거부 자동
0.5 - 2 마일 0 - 100 피트 즉시 (LAANC) 자동
2 - 4 마일 100 - 200 피트 즉시 (LAANC) 자동
4 - 6 마일 200 - 300 피트 즉시 (LAANC) 자동
6+ 마일 400 피트 즉시 (LAANC) 자동

동적 공역 관리

공역은 정적이지 않습니다—제약은 끊임없이 변경됩니다. UTM은 실시간으로 적응해야 합니다.

임시 비행 제한 (TFR)

TFR은 특정 시간 동안 특정 영역의 공역을 폐쇄합니다. 일반적인 이유는 다음과 같습니다:

  • VIP 이동: 대통령, 외국 지도자가 여행할 때 최대 30마일의 제한
  • 스포츠 이벤트: 대형 경기장 주변 3마일, 최대 3,000피트
  • 재해 대응: 산불, 허리케인, 홍수—비상 항공기만
  • 보안 작업: 법 집행, 군사 훈련
  • 우주 발사: 로켓 발사장 주변 넓은 영역

FIMS는 FAA NOTAM 시스템에서 TFR 업데이트를 받고 모든 USS에 배포합니다. USS는 다음을 수행해야 합니다:

  • 진행 중인 비행 계획에 대해 TFR을 확인합니다
  • 충돌이 발견되면 운영자에게 경고합니다
  • 작업이 영향을 받으면 조기 착륙 또는 우회를 요청합니다
  • 활성 TFR 내의 새 비행 계획을 거부합니다

동적 날씨 제약

날씨 조건은 빠르게 변할 수 있으며 특히 작은 드론에 위험을 초래합니다:

  • 풍속: 대부분의 소비자 드론은 25+ mph 바람에서 어려움을 겪습니다
  • 가시성: Part 107은 3 statute miles 가시성이 필요합니다
  • 구름 간격: 구름에서 500피트 아래 유지
  • 강수량: 많은 드론이 방수가 아닙니다
  • 온도: 극한의 추위 또는 더위가 배터리 성능에 영향을 미칩니다
  • 번개: 뇌우 반경 10마일 내에서 작업 금지

USS는 날씨 SDSP와 통합하여 실시간 조건을 얻고 운영자에게 경고합니다. 일부 고급 USS는 안전하지 않은 조건이 예상될 때 자동으로 비행 계획을 거부합니다.

특별 사용 공역 (SUA)

SUA는 군사 훈련, 포격 범위, 위험한 활동을 위해 예약된 영역입니다. 일부 SUA는 항상 활성화되어 있고 다른 일부는 예약 시간에만 활성화됩니다. FIMS는 USS에 SUA 상태를 제공합니다:

  • 제한 구역: 일반적으로 완전히 금지됨 (군사 기지)
  • 금지 구역: 국가 보안상 금지됨 (백악관)
  • 경고 구역: 위험한 활동, 조심스럽게 진입
  • 군사 작전 구역 (MOA): 군사 훈련, 조정 권장
  • 경보 구역: 비정상적인 항공 활동, 주의

지오펜싱 구현

지오펜싱은 드론이 제한된 공역에 진입하는 것을 방지하는 기술적 수단입니다. 지오펜싱의 두 가지 접근 방식이 있습니다:

제조업체 지오펜싱

DJI, Parrot, Autel과 같은 주요 드론 제조업체는 펌웨어에 지오펜싱을 구현합니다. 드론에는 제한 구역 데이터베이스(공항, TFR, 국립 공원)가 포함되어 있으며 이륙하거나 진입하는 것을 방지합니다. 운영자는 승인 토큰을 제공하여 잠금을 해제할 수 있습니다(LAANC를 통해).

USS 지오펜싱

USS는 비행 계획 승인 중에 지오펜싱 경계를 시행합니다. 드론이 승인된 볼륨을 벗어나거나 제한 구역에 진입하는 것으로 감지되면 USS는 다음을 수행합니다:

  • 운영자에게 즉각적인 경고 보내기
  • 다른 USS에 위반 알림
  • FIMS에 보고
  • 심각한 경우 비상 착륙 요청

고도 계층화 및 분리 전략

수천 개의 드론이 동일한 도시 공역을 공유할 때 효율적인 분리 전략이 중요합니다. UTM은 여러 접근 방식을 사용합니다:

고도 계층화

드론 작업을 고도 밴드로 분리하면 충돌 위험이 줄어듭니다:

  • 0-50 피트: 검사, 사진, 농업 스프레이
  • 50-150 피트: 건물 검사, 부동산 사진
  • 150-250 피트: 인프라 검사, 감시
  • 250-350 피트: 장거리 배송, 측량
  • 350-400 피트: 고속 통과, 긴급 작업

USS는 고도 계층화를 비행 계획 승인에 통합할 수 있습니다—혼잡한 영역에서 운영자에게 덜 사용되는 고도를 제안합니다.

통로 및 고속도로

일부 UTM 개념은 특정 경로를 따라 트래픽을 채널링하는 "드론 고속도로"를 제안합니다:

  • 동서 통로: 200피트, 동쪽으로 이동하는 트래픽 전용
  • 남북 통로: 250피트, 북쪽으로 이동하는 트래픽
  • 배송 경로: 물류 허브에서 배송 영역으로 전용 레인
  • 긴급 통로: 경찰, 소방, EMS 드론을 위해 예약됨

통로는 트래픽을 예측 가능하게 만들고, 충돌을 줄이고, 효율성을 향상시킵니다. 그러나 유연성을 제한하고 혼잡한 통로를 만들 수 있습니다.

시간 분리

공간적으로 분리할 수 없을 때 시간적으로 분리할 수 있습니다:

  • 순차적 시간 슬롯: 드론 A는 10:00-10:15, 드론 B는 10:15-10:30
  • 버퍼 시간: 작업 사이 5-10분 간격으로 청소
  • 우선 순위 스케줄링: 긴급 작업이 우선, 상업적 작업이 대기

4D 볼륨 예약

가장 정교한 접근 방식은 3D 공간에 시간을 추가한 4D 볼륨입니다. 운영자가 비행 계획을 제출할 때 정확한 위도/경도 경계, 고도 범위 및 시간 창을 지정합니다. USS는 이 4D 볼륨이 다른 승인된 작업과 겹치지 않는지 확인합니다.

예: 드론 A는 (37.7749° N, 122.4194° W, 100-150 ft, 10:00-10:30)을 예약합니다. 드론 B는 동일한 위치에서 (37.7749° N, 122.4194° W, 100-150 ft, 10:35-11:00)을 예약할 수 있지만 10:00-10:30은 예약할 수 없습니다.

유인 항공과의 조정

UTM은 공백에서 작동하지 않습니다—저고도 공역은 헬리콥터, 경비행기, 농업 항공기 및 기타 유인 항공기와 공유됩니다. 안전한 공존을 보장하는 것이 중요합니다.

유인 항공기 우선권

FAA 규칙은 명확합니다: 드론은 항상 유인 항공기에 양보해야 합니다. 이것은 UTM 설계에 통합되어 있습니다:

  • 유인 항공기가 감지되면 드론은 하강하거나 착륙해야 합니다
  • 헬리콥터 경로는 드론이 없는 구역으로 지정됩니다
  • 의료 비행(LifeFlight)에는 비상 공역 클리어런스가 있습니다
  • 공항 교통 패턴은 절대 드론 작업에 사용되지 않습니다

ADS-B 통합

자동 종속 감시-방송(ADS-B)은 항공기가 위치를 브로드캐스트하는 기술입니다. 많은 유인 항공기(특히 2020년 이후)에는 ADS-B Out이 장착되어 있습니다. USS는 ADS-B 데이터를 통합하여 다음을 수행할 수 있습니다:

  • 실시간 유인 항공기 위치 추적
  • 유인 항공기가 드론 작업 영역에 접근할 때 운영자에게 경고
  • 유인 트래픽이 나타날 때 자동으로 드론 작업 일시 중지
  • 유인 항공기 경로 기록을 기반으로 안전한 작업 창 제안

조종사 인식 도구

유인 항공기 조종사도 드론 활동을 인식해야 합니다. UTM은 다음을 통해 이를 가능하게 합니다:

  • NOTAM: 주요 드론 작업에 대한 조종사 공지
  • 조종사 앱: ForeFlight, Garmin Pilot은 드론 활동 오버레이를 표시합니다
  • ATC 알림: 관제사는 통제 공역의 드론을 인식합니다
  • 시각적 관찰자: 대규모 작업에는 하늘을 스캔하는 시각적 관찰자가 필요합니다

혼합 작업 시나리오

일부 환경에서는 유인 및 무인 항공기가 정기적으로 상호 작용합니다:

  • 농업: 작물 살포 드론 + 작물 살포 비행기
  • 긴급 대응: 화재 드론 + 화재 진압 헬리콥터
  • 인프라 검사: 교량 드론 + 검사 비행기
  • 도시 작업: 배송 드론 + 경찰 헬리콥터 + 뉴스 헬리콥터

이러한 시나리오에서 UTM은 다음을 통해 조정을 용이하게 합니다:

  • 유인 및 무인 작업 모두에 대한 미리 조정된 비행 계획
  • 전용 무선 주파수로 실시간 통신
  • 명확한 고도 분리(예: 유인 500피트 이상, 무인 400피트 미만)
  • 비상 프로토콜(예: 헬리콥터가 접근하면 모든 드론 착륙)

핵심 요점

  • UTM은 주로 클래스 G(비관제) 공역에서 작동하지만 LAANC를 통해 통제 공역(클래스 B, C, D)의 승인을 자동화하여 몇 주에서 몇 초로 단축합니다
  • UAS 시설 지도(USFM)는 공항 주변의 허용 가능한 최대 드론 고도를 정의하는 3D 그리드로, 안전한 자동 승인을 가능하게 합니다

  • 고도 계층화, 통로 시스템 및 4D 볼륨 예약은 고밀도 도시 환경에서 수천 개의 동시 드론 작업을 가능하게 하는 전략적 분리 방법을 제공합니다
