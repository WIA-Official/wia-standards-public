# 챕터 10 · 데이터셋 준비 7단계 — 수집 · 라이선스 · 전처리

## 학습 목표
- AI 학습용 데이터셋 수집의 7단계 프로세스를 체계적으로 실행할 수 있다
- 라이선스 위험 없이 공개 이미지를 합법적으로 사용하는 기준을 세울 수 있다
- 이미지-마스크 페어 데이터를 자동으로 생성하는 스크립트를 작성할 수 있다

---

## 이론

### 왜 데이터셋이 모델보다 중요한가

ML 실무에서 흔한 착각은 "좋은 모델이 좋은 결과를 낸다"는 것이다. 현실은 다르다:

> **"Garbage in, garbage out"** — 학습 데이터 품질이 결과 품질의 80%를 결정한다.

SDXL을 고문서 복원에 파인튜닝한다면 필요한 것은:
1. **원본 고문서 이미지** (손상 없는 이미지 or 부분 손상)
2. **손상 마스크** (복원할 영역 표시)
3. **타깃 이미지** (복원된 결과물)

인페인팅 학습은 `(원본, 마스크, 타깃)` 트리플을 학습한다.

### 7단계 데이터셋 파이프라인

```
Step 1: 소스 식별 및 접근
Step 2: 라이선스 검증
Step 3: 대량 다운로드
Step 4: 중복 제거
Step 5: 품질 필터링
Step 6: 전처리 및 정규화
Step 7: 마스크 생성 및 페어 구성
```

### Step 1 — 한국 공개 이미지 소스

**국립중앙도서관 디지털 컬렉션**
- URL: `https://www.nl.go.kr/NL/contents/search.do`
- API: `https://www.nl.go.kr/seoji/contents/S80100000000.do?schM=api`
- 저작권 소멸 자료: 출판 후 70년 경과 (국내법 기준)
- 고해상도 이미지 제공: 300~600 DPI

**국가기록원 공개 자료**
- URL: `https://www.archives.go.kr`
- OpenAPI: `https://www.data.go.kr/dataset/15049046/openapi.do`
- 1945년 이전 공공 기록물: 저작권 없음

**AI-Hub 한국 이미지 데이터셋**
- URL: `https://www.aihub.or.kr`
- 한국 문화유산 고해상도 이미지 (승인 필요, 연구 목적 무료)
- 조선시대 회화, 문헌 자료 포함

**Wikimedia Commons**
- 한국 관련 Public Domain 이미지
- API: `https://commons.wikimedia.org/w/api.php`
- 필터: `Category:Old_Korean_documents`

**국립민속박물관 e뮤지엄**
- `https://www.emuseum.go.kr`
- API 제공, 공공누리 1~4유형 라이선스

### Step 2 — 라이선스 검증 원칙

| 라이선스 | 상업 이용 | 수정 | 재배포 |
|---|---|---|---|
| Public Domain / CC0 | ✅ | ✅ | ✅ |
| CC BY (1~4) | ✅ (출처 표기) | ✅ | ✅ |
| 공공누리 1유형 | ✅ | ✅ | ✅ |
| 공공누리 2유형 | ✅ | ❌ | ✅ |
| CC BY-NC | ❌ | ✅ | ✅ |
| 저작권 있음 | ❌ | ❌ | ❌ |

**고문서 복원 AI 학습용**: CC BY, CC0, 공공누리 1유형, 저작권 소멸 자료만 사용.

### Step 5 — 품질 필터링 기준

- 최소 해상도: 512×512 이상 (SDXL 기준 1024×1024 권장)
- 포맷: PNG 또는 고품질 JPEG (압축률 90 이상)
- 블러 점수: Laplacian variance > 100 (흔들린 이미지 제거)
- 중복 제거: pHash (perceptual hash) 해밍 거리 10 이하 제거

### Step 7 — 합성 마스크 생성 전략

실제 손상 마스크를 1:1 매칭하기 어려우므로 **합성 마스크**를 생성한다:

1. **Random Rectangle**: 무작위 사각형 영역 마스킹
2. **Brush Stroke**: 불규칙한 붓터치 형태
3. **Morphological Erosion**: 실제 얼룩과 유사한 형태
4. **Otsu Binarization**: 손상된 실제 이미지의 밝기 임계값으로 마스크 추출

```python
# 고문서 복원 데이터셋 준비 파이프라인
# 실행 환경: Python 3.10+
# pip install requests Pillow numpy opencv-python imagehash tqdm

import os
import hashlib
import requests
import numpy as np
import cv2
from PIL import Image, ImageFilter, ImageDraw
import imagehash
from pathlib import Path
from tqdm import tqdm
from typing import Optional


# ──────────────────────────────────────────────
# Step 3: 이미지 다운로드 (Wikimedia Commons 예시)
# ──────────────────────────────────────────────
def download_wikimedia_category(
    category: str = "Category:Old_Korean_documents",
    output_dir: str = "./raw_images",
    max_images: int = 200
) -> list:
    """Wikimedia Commons API로 카테고리 이미지 다운로드"""
    os.makedirs(output_dir, exist_ok=True)
    
    api_url = "https://commons.wikimedia.org/w/api.php"
    params = {
        "action": "query",
        "list": "categorymembers",
        "cmtitle": category,
        "cmtype": "file",
        "cmlimit": 50,
        "format": "json",
    }
    
    downloaded = []
    cmcontinue = None
    
    while len(downloaded) < max_images:
        if cmcontinue:
            params["cmcontinue"] = cmcontinue
        
        resp = requests.get(api_url, params=params, timeout=30)
        data = resp.json()
        
        members = data.get("query", {}).get("categorymembers", [])
        if not members:
            break
        
        for m in members:
            if len(downloaded) >= max_images:
                break
            title = m["title"]
            # 이미지 URL 가져오기
            info_params = {
                "action": "query", "titles": title,
                "prop": "imageinfo", "iiprop": "url",
                "format": "json",
            }
            info_resp = requests.get(api_url, params=info_params, timeout=30)
            info_data = info_resp.json()
            pages = info_data.get("query", {}).get("pages", {})
            
            for page_id, page in pages.items():
                if "imageinfo" not in page:
                    continue
                img_url = page["imageinfo"][0]["url"]
                
                # 파일 다운로드
                fname = title.replace("File:", "").replace(" ", "_")
                fpath = os.path.join(output_dir, fname)
                
                if not os.path.exists(fpath):
                    try:
                        img_resp = requests.get(img_url, timeout=60)
                        with open(fpath, "wb") as f:
                            f.write(img_resp.content)
                        downloaded.append(fpath)
                        print(f"다운로드: {fname}")
                    except Exception as e:
                        print(f"실패: {fname} — {e}")
        
        cmcontinue = data.get("continue", {}).get("cmcontinue")
        if not cmcontinue:
            break
    
    print(f"총 {len(downloaded)}개 다운로드 완료")
    return downloaded


# ──────────────────────────────────────────────
# Step 4: 중복 제거 (pHash)
# ──────────────────────────────────────────────
def deduplicate_images(image_dir: str, hamming_threshold: int = 10) -> list:
    """
    pHash 기반 중복 제거
    해밍 거리 < threshold인 이미지를 중복으로 판정
    """
    image_files = list(Path(image_dir).glob("*.{jpg,jpeg,png,JPG,PNG}"))
    # glob 패턴이 bash 방식이 아니므로 직접 확장자별로
    image_files = [
        p for p in Path(image_dir).iterdir()
        if p.suffix.lower() in [".jpg", ".jpeg", ".png"]
    ]
    
    hashes = {}
    duplicates = set()
    
    for img_path in tqdm(image_files, desc="중복 제거"):
        try:
            img = Image.open(img_path).convert("RGB")
            h = imagehash.phash(img)
            
            is_dup = False
            for existing_hash, existing_path in hashes.items():
                if abs(h - imagehash.hex_to_hash(existing_hash)) < hamming_threshold:
                    duplicates.add(str(img_path))
                    is_dup = True
                    break
            
            if not is_dup:
                hashes[str(h)] = str(img_path)
        except Exception:
            duplicates.add(str(img_path))
    
    unique_files = [str(p) for p in image_files if str(p) not in duplicates]
    print(f"원본: {len(image_files)}개 → 중복 제거 후: {len(unique_files)}개")
    return unique_files


# ──────────────────────────────────────────────
# Step 5: 품질 필터링
# ──────────────────────────────────────────────
def filter_quality(image_paths: list, min_size: int = 512, blur_threshold: float = 100.0) -> list:
    """
    해상도 + 블러 점수로 저품질 이미지 제거
    Laplacian variance로 블러 감지: 낮을수록 흐린 이미지
    """
    passed = []
    for path in tqdm(image_paths, desc="품질 필터"):
        try:
            img = Image.open(path).convert("RGB")
            w, h = img.size
            
            # 최소 해상도 체크
            if w < min_size or h < min_size:
                continue
            
            # 블러 점수 계산
            gray = np.array(img.convert("L"))
            lap_var = cv2.Laplacian(gray, cv2.CV_64F).var()
            if lap_var < blur_threshold:
                continue
            
            passed.append(path)
        except Exception:
            continue
    
    print(f"품질 필터: {len(image_paths)}개 → {len(passed)}개")
    return passed


# ──────────────────────────────────────────────
# Step 6: 전처리 — SDXL 입력 포맷으로 변환
# ──────────────────────────────────────────────
def preprocess_for_sdxl(
    image_path: str,
    output_path: str,
    target_size: int = 1024,
) -> bool:
    """
    1. 정사각형 크롭 (중앙)
    2. 1024×1024 리사이즈
    3. PNG로 저장 (무손실)
    """
    try:
        img = Image.open(image_path).convert("RGB")
        w, h = img.size
        
        # 정사각형 중앙 크롭
        min_dim = min(w, h)
        left   = (w - min_dim) // 2
        top    = (h - min_dim) // 2
        img = img.crop((left, top, left + min_dim, top + min_dim))
        
        # 리사이즈 (LANCZOS = 고품질)
        img = img.resize((target_size, target_size), Image.LANCZOS)
        
        img.save(output_path, "PNG")
        return True
    except Exception as e:
        print(f"전처리 실패 {image_path}: {e}")
        return False


# ──────────────────────────────────────────────
# Step 7: 합성 손상 마스크 생성
# ──────────────────────────────────────────────
def generate_damage_mask(
    size: int = 1024,
    mask_type: str = "mixed",
    damage_ratio: float = 0.15,  # 전체 면적의 15% 손상
) -> Image.Image:
    """
    합성 손상 마스크 생성
    흰색(255) = 복원 대상 영역
    검정(0)   = 보존 영역
    """
    mask = Image.new("L", (size, size), 0)
    draw = ImageDraw.Draw(mask)
    
    n_patches = int(size * size * damage_ratio / (30 * 30))  # 평균 패치 30×30 기준
    
    for _ in range(max(1, n_patches)):
        if mask_type == "rectangle":
            # 사각형 패치
            w_p = np.random.randint(20, size // 6)
            h_p = np.random.randint(20, size // 6)
            x = np.random.randint(0, size - w_p)
            y = np.random.randint(0, size - h_p)
            draw.rectangle([x, y, x + w_p, y + h_p], fill=255)
        
        elif mask_type == "brush":
            # 붓터치 형태 (불규칙 곡선)
            n_points = np.random.randint(5, 15)
            points = [(np.random.randint(0, size), np.random.randint(0, size))
                      for _ in range(n_points)]
            width = np.random.randint(5, 40)
            draw.line(points, fill=255, width=width)
        
        else:  # mixed
            if np.random.rand() > 0.5:
                w_p = np.random.randint(15, size // 8)
                h_p = np.random.randint(15, size // 8)
                x = np.random.randint(0, size - w_p)
                y = np.random.randint(0, size - h_p)
                draw.rectangle([x, y, x + w_p, y + h_p], fill=255)
            else:
                n_points = np.random.randint(4, 12)
                points = [(np.random.randint(0, size), np.random.randint(0, size))
                          for _ in range(n_points)]
                draw.line(points, fill=255, width=np.random.randint(8, 35))
    
    # 약간 블러 처리 (경계 부드럽게)
    mask = mask.filter(ImageFilter.GaussianBlur(radius=2))
    # 다시 이진화
    mask = mask.point(lambda p: 255 if p > 30 else 0)
    
    return mask


def build_inpainting_dataset(
    clean_images_dir: str,
    output_dir: str,
    n_masks_per_image: int = 3
):
    """
    깨끗한 이미지 → (원본, 마스크, 타깃) 트리플 생성
    - 원본: 합성 손상 이미지 (clean × inverted_mask)
    - 마스크: 손상 영역
    - 타깃: 깨끗한 원본 이미지
    """
    os.makedirs(f"{output_dir}/images", exist_ok=True)
    os.makedirs(f"{output_dir}/masks", exist_ok=True)
    os.makedirs(f"{output_dir}/targets", exist_ok=True)
    
    image_files = [p for p in Path(clean_images_dir).iterdir()
                   if p.suffix.lower() in [".jpg", ".jpeg", ".png"]]
    
    idx = 0
    for img_path in tqdm(image_files, desc="데이터셋 생성"):
        clean = Image.open(img_path).convert("RGB")
        
        for i in range(n_masks_per_image):
            mask = generate_damage_mask(size=clean.width, mask_type="mixed")
            
            # 손상 이미지 생성: 마스크 영역을 흰색(종이 색)으로 채움
            damaged = clean.copy()
            white_patch = Image.new("RGB", clean.size, (240, 230, 210))  # 종이색
            damaged = Image.composite(white_patch, damaged, mask)
            
            stem = img_path.stem
            damaged.save(f"{output_dir}/images/{stem}_{i:02d}.png")
            mask.save(f"{output_dir}/masks/{stem}_{i:02d}.png")
            clean.save(f"{output_dir}/targets/{stem}_{i:02d}.png")
            idx += 1
    
    print(f"데이터셋 생성 완료: {idx}개 트리플")
    print(f"  images/ : 손상된 입력 이미지")
    print(f"  masks/  : 손상 마스크 (흰색=복원영역)")
    print(f"  targets/: 복원 목표 이미지")


if __name__ == "__main__":
    # 빠른 테스트: 합성 마스크 생성 확인
    mask = generate_damage_mask(size=512, mask_type="mixed", damage_ratio=0.15)
    mask.save("test_mask.png")
    print(f"테스트 마스크 저장: test_mask.png")
    print(f"마스크 흰색 비율: {np.array(mask).sum() / (255 * 512 * 512) * 100:.1f}%")
```

## 함정 카드 — 자주 실수하는 5가지

1. **"크롤링으로 아무 이미지나 가져다 쓴다"** → AI 학습 목적 데이터 수집은 저작권법의 규제를 받는다. 한국 저작권법 제35조의5(텍스트·데이터 마이닝 면책)는 연구 목적에 한해 허용하지만, 상업적 서비스에는 적용되지 않는다. 항상 라이선스 확인 후 사용.

2. **"이미지를 JPEG로 저장한다"** → JPEG는 손실 압축이라 마스크 경계에 블러가 생긴다. 마스크와 타깃 이미지는 반드시 **PNG로 저장**. 학습 이미지도 PNG 권장.

3. **"마스크 비율을 너무 크게 설정한다"** → 손상 비율 50% 이상은 모델이 복원 방향을 잡기 어렵다. 실제 고문서 손상률 5~30%를 기준으로 마스크 생성. 학습 시작은 `damage_ratio=0.10`으로 보수적으로.

4. **"다운로드 후 파일 검증을 안 한다"** → 다운로드 실패, 부분 저장, 손상 JPEG는 학습 중 에러를 유발한다. 모든 이미지를 `PIL.Image.open().verify()` 또는 `cv2.imread() is not None`으로 사전 검증.

5. **"데이터셋 통계를 확인하지 않는다"** → 학습 전 필수: ①평균 이미지 크기 ②밝기 분포 ③마스크 평균 비율. 분포가 극단적으로 치우치면 배치 정규화가 불안정해진다. `torchvision.datasets.ImageFolder` + 루프로 간단히 계산.

## 연습문제

**문제 1.** 국립중앙도서관 공개 자료 API를 사용하여 "조선시대" 키워드로 이미지 50개를 다운로드하는 코드를 작성하라. 단, 저작권 소멸 확인을 위해 출판연도 필드를 함께 수집해야 한다.

> **해설:** `https://www.nl.go.kr/seoji/contents/S80100000000.do?schM=api&kwd=조선시대&pageNo=1&pageSize=50&drm_yn=Y`로 GET 요청. `drm_yn=Y` 파라미터가 저작권 소멸(DRM 없음) 필터. 반환된 JSON의 `PUBLISH_PREDATE` 필드로 출판연도 확인. `requests.get()` + `json()` 파싱 후 `FILE_PATH` 필드로 원본 파일 다운로드.

**문제 2.** pHash 기반 중복 제거에서 해밍 거리(Hamming distance) 임계값을 10으로 설정했다. 이 값을 0으로 낮추거나 20으로 높이면 어떤 결과가 나오는가?

> **해설:** 임계값 0 → 완전 동일한 이미지만 제거 (변형 중복 못 잡음). 임계값 20 → 유사한 이미지도 제거 (귀중한 다양성 손실). 실무에서는 10~12를 기준으로 삼고, 필요 시 샘플 100개 육안 검수로 보정. 고문서처럼 비슷한 시대 유물이 많을 때는 7~8로 낮춰 다양성 보존.

**문제 3.** `generate_damage_mask()` 함수를 수정하여 Otsu 이진화 기반 마스크를 추가하라. 실제 손상된 고문서 이미지에서 자동으로 손상 영역을 감지하는 방식이다.

> **해설:** `gray = np.array(img.convert("L"))` → `_, mask_np = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)` → `Image.fromarray(mask_np)`. Otsu threshold가 손상(밝은 영역)과 텍스트(어두운 영역)를 자동 분리. 단, 고문서의 경우 적응형 임계값(Adaptive Threshold)이 더 정확할 수 있다: `cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)`.
