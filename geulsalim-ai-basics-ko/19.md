## 이론

### 기술이 먼저가 아니다 — 윤리가 기반이다

나는 수백만 장의 이미지를 처리하는 서비스를 운영하면서 깨달았다: **기술 역량과 윤리 기준이 함께 성장해야 한다**. 강력한 AI 도구를 만들수록 그 책임도 커진다.

고문서 복원 AI는 두 가지 위험성을 내포한다:
1. **저작권 침해**: 아직 저작권이 살아있는 문서 복원
2. **역사 위조**: 복원이라는 명목으로 내용 변조

이 두 가지는 기술적 실수가 아니라 **설계 단계에서 차단**해야 한다.

### WIA Geulsalim AI 윤리 헌장

**홍익인간(弘益人間) 원칙 기반**:
> "널리 인간을 이롭게 하라" — 단군 조선 건국 이념

AI 복원 기술이 인류의 문화유산을 보존하는 데 기여해야 한다는 원칙.

**5대 윤리 원칙**:

1. **보존(Preservation)**: 복원은 보존이지 창작이 아니다. 원본에 충실.
2. **투명성(Transparency)**: 복원된 이미지에는 반드시 AI 처리 사실 명시.
3. **감사가능성(Auditability)**: 모든 복원 과정 기록, 전문가 검수 루프 포함.
4. **접근성(Accessibility)**: 복원된 문화유산 데이터를 연구자와 공공이 사용할 수 있도록.
5. **무해성(Non-maleficence)**: 악용 방지 메커니즘을 설계에 내장.

```python
# 윤리 차단 레이어 구현
# pip install Pillow numpy opencv-python-headless

import hashlib
import json
import logging
import numpy as np
from datetime import datetime
from pathlib import Path
from PIL import Image
from typing import Optional, Tuple


# ──────────────────────────────────────────────
# 1. 입력 이미지 사전 검증
# ──────────────────────────────────────────────
class DocumentSafetyChecker:
    """
    복원 요청 전 안전성 검사
    - 마스크 비율 제한
    - 이미지 크기 검증
    - 해시 기반 중복 탐지
    """
    
    MAX_MASK_RATIO  = 0.40   # 전체 면적의 40% 이상 마스킹 거부
    MIN_RESOLUTION  = 256
    MAX_RESOLUTION  = 4096
    
    def __init__(self, blocklist_path: Optional[str] = None):
        self.blocklist = set()
        if blocklist_path and Path(blocklist_path).exists():
            with open(blocklist_path) as f:
                self.blocklist = set(json.load(f))
    
    def check_image(self, image: Image.Image) -> Tuple[bool, str]:
        """
        이미지 안전성 검사
        Returns: (통과여부, 메시지)
        """
        w, h = image.size
        
        # 해상도 검사
        if w < self.MIN_RESOLUTION or h < self.MIN_RESOLUTION:
            return False, f"해상도 너무 낮음: {w}×{h} (최소 {self.MIN_RESOLUTION}×{self.MIN_RESOLUTION})"
        
        if w > self.MAX_RESOLUTION or h > self.MAX_RESOLUTION:
            return False, f"해상도 너무 높음: {w}×{h} (최대 {self.MAX_RESOLUTION}×{self.MAX_RESOLUTION})"
        
        # 이미지 해시 확인 (blocklist)
        img_hash = self._image_hash(image)
        if img_hash in self.blocklist:
            return False, "차단된 이미지"
        
        return True, "통과"
    
    def check_mask(self, mask: Image.Image) -> Tuple[bool, str]:
        """
        마스크 안전성 검사
        - 40% 이상 마스킹은 내용 변조로 간주
        """
        mask_arr = np.array(mask.convert("L"))
        white_ratio = (mask_arr > 128).sum() / mask_arr.size
        
        if white_ratio > self.MAX_MASK_RATIO:
            return False, f"마스크 비율 초과: {white_ratio*100:.1f}% (최대 {self.MAX_MASK_RATIO*100:.0f}%)"
        
        if white_ratio < 0.001:
            return False, "복원 영역이 너무 작음 (최소 0.1% 필요)"
        
        return True, f"마스크 비율: {white_ratio*100:.1f}% — 통과"
    
    @staticmethod
    def _image_hash(image: Image.Image) -> str:
        """perceptual hash는 아니고 픽셀 해시"""
        img_bytes = image.tobytes()
        return hashlib.sha256(img_bytes[:10000]).hexdigest()[:16]


# ──────────────────────────────────────────────
# 2. 감사 로그 — 모든 복원 기록
# ──────────────────────────────────────────────
class RestorationAuditLogger:
    """
    복원 작업 감사 로그
    - 입력 이미지 해시 (원본 파일 없이도 추적 가능)
    - 파라미터 기록
    - 타임스탬프 + IP
    - WORM(Write Once Read Many) 방식: 삭제 불가
    """
    
    def __init__(self, log_dir: str = "./audit_logs"):
        self.log_dir = Path(log_dir)
        self.log_dir.mkdir(parents=True, exist_ok=True)
        self.logger = logging.getLogger("audit")
        
        # 파일 핸들러 (일별 파일)
        log_file = self.log_dir / f"audit_{datetime.now().strftime('%Y%m%d')}.jsonl"
        fh = logging.FileHandler(log_file, mode="a", encoding="utf-8")
        fh.setFormatter(logging.Formatter("%(message)s"))
        self.logger.addHandler(fh)
        self.logger.setLevel(logging.INFO)
    
    def log_restoration(
        self,
        input_image: Image.Image,
        mask: Image.Image,
        params: dict,
        result_image: Optional[Image.Image],
        client_ip: str = "unknown",
        success: bool = True,
        error: Optional[str] = None,
    ):
        """복원 작업 1건 기록"""
        entry = {
            "timestamp":     datetime.utcnow().isoformat() + "Z",
            "client_ip":     client_ip,
            "input_hash":    hashlib.sha256(input_image.tobytes()[:50000]).hexdigest(),
            "input_size":    f"{input_image.width}×{input_image.height}",
            "mask_ratio":    round(
                (np.array(mask.convert("L")) > 128).mean(), 4
            ),
            "params":        params,
            "success":       success,
            "result_hash":   (
                hashlib.sha256(result_image.tobytes()[:50000]).hexdigest()
                if result_image else None
            ),
            "error":         error,
        }
        self.logger.info(json.dumps(entry, ensure_ascii=False))
        return entry["input_hash"]


# ──────────────────────────────────────────────
# 3. 출력 워터마킹 — AI 생성 표시
# ──────────────────────────────────────────────
def embed_invisible_watermark(
    image: Image.Image,
    payload: str = "WIA-GEULSALIM-AI-RESTORED",
) -> Image.Image:
    """
    간단한 LSB(Least Significant Bit) 스테가노그래피
    이미지 픽셀의 최하위 비트에 정보 은닉
    육안으로 보이지 않지만 추출 가능
    
    실제 프로덕션에서는 더 강건한 기법 사용:
    - dwtDct (주파수 도메인 워터마킹)
    - invisible-watermark 라이브러리
    """
    img_arr = np.array(image.copy())
    
    payload_bytes = payload.encode("utf-8")
    payload_len = len(payload_bytes)
    
    # 길이 먼저 4바이트로 인코딩
    header = payload_len.to_bytes(4, "big")
    data = header + payload_bytes
    
    bits = []
    for byte in data:
        for i in range(8):
            bits.append((byte >> (7 - i)) & 1)
    
    flat = img_arr.flatten()
    if len(bits) > len(flat):
        raise ValueError("페이로드가 이미지 크기 초과")
    
    for i, bit in enumerate(bits):
        flat[i] = (flat[i] & 0xFE) | bit
    
    watermarked = flat.reshape(img_arr.shape).astype(np.uint8)
    return Image.fromarray(watermarked)


def extract_watermark(image: Image.Image) -> Optional[str]:
    """워터마크 추출 (진위 확인용)"""
    try:
        flat = np.array(image).flatten()
        
        # 헤더 4바이트 읽기
        header_bits = [flat[i] & 1 for i in range(32)]
        length = int("".join(map(str, header_bits)), 2)
        
        if length <= 0 or length > 1000:
            return None
        
        # 페이로드 읽기
        payload_bits = [flat[32 + i] & 1 for i in range(length * 8)]
        payload_bytes = bytes([
            int("".join(map(str, payload_bits[i*8:(i+1)*8])), 2)
            for i in range(length)
        ])
        
        return payload_bytes.decode("utf-8")
    except Exception:
        return None


# ──────────────────────────────────────────────
# 4. 라이선스 메타데이터 삽입
# ──────────────────────────────────────────────
def attach_restoration_metadata(
    image: Image.Image,
    source_url: str,
    license_type: str,
    restoration_date: str,
    operator: str = "WIA Geulsalim AI",
) -> Image.Image:
    """
    EXIF 또는 PNG iTXt 청크에 복원 메타데이터 삽입
    나중에 이미지 출처 추적 가능
    """
    from PIL import PngImagePlugin
    
    meta = PngImagePlugin.PngInfo()
    meta.add_itxt("AI-Restoration", "true")
    meta.add_itxt("Source-URL", source_url)
    meta.add_itxt("License", license_type)
    meta.add_itxt("Restoration-Date", restoration_date)
    meta.add_itxt("Operator", operator)
    meta.add_itxt("Ethics-Standard", "WIA-Geulsalim-홍익인간-v1.0")
    
    import io
    buf = io.BytesIO()
    image.save(buf, format="PNG", pnginfo=meta)
    buf.seek(0)
    
    return Image.open(buf)


# ──────────────────────────────────────────────
# 5. 통합 복원 파이프라인 + 윤리 레이어
# ──────────────────────────────────────────────
class EthicalRestorationPipeline:
    """
    기술 + 윤리 통합 파이프라인
    모든 복원 요청은 이 클래스를 통해야 한다
    """
    
    def __init__(self, pipe, audit_log_dir: str = "./audit_logs"):
        self.pipe = pipe
        self.safety = DocumentSafetyChecker()
        self.audit  = RestorationAuditLogger(audit_log_dir)
    
    def restore(
        self,
        input_image: Image.Image,
        mask: Image.Image,
        params: dict,
        source_info: dict,  # {"url": ..., "license": ..., "year": ...}
        client_ip: str = "unknown",
    ) -> Tuple[Optional[Image.Image], dict]:
        """
        윤리 검사 → 복원 → 워터마킹 → 감사 로그
        """
        report = {"passed": False, "reason": "", "hash": ""}
        
        # 1. 안전성 검사
        ok, msg = self.safety.check_image(input_image)
        if not ok:
            self.audit.log_restoration(input_image, mask, params, None,
                                        client_ip, False, msg)
            report["reason"] = msg
            return None, report
        
        ok, msg = self.safety.check_mask(mask)
        if not ok:
            self.audit.log_restoration(input_image, mask, params, None,
                                        client_ip, False, msg)
            report["reason"] = msg
            return None, report
        
        # 2. 복원 실행
        if self.pipe is None:
            report["reason"] = "모델 미로드"
            return None, report
        
        result = self.pipe(
            prompt=params.get("prompt", "historical document"),
            image=input_image,
            mask_image=mask,
            num_inference_steps=params.get("num_steps", 20),
        ).images[0]
        
        # 3. 워터마킹
        result = embed_invisible_watermark(result, "WIA-GEULSALIM-AI-RESTORED")
        
        # 4. 메타데이터 삽입
        result = attach_restoration_metadata(
            result,
            source_url=source_info.get("url", ""),
            license_type=source_info.get("license", "unknown"),
            restoration_date=datetime.now().strftime("%Y-%m-%d"),
        )
        
        # 5. 감사 로그
        img_hash = self.audit.log_restoration(
            input_image, mask, params, result, client_ip, True
        )
        
        report.update({"passed": True, "hash": img_hash, "reason": "통과"})
        return result, report


if __name__ == "__main__":
    # 안전성 검사 시뮬레이션
    checker = DocumentSafetyChecker()
    
    # 테스트 이미지
    test_img = Image.fromarray(np.random.randint(0, 200, (1024, 1024, 3), dtype=np.uint8))
    test_mask = Image.fromarray(np.random.randint(0, 255, (1024, 1024), dtype=np.uint8))
    
    ok, msg = checker.check_image(test_img)
    print(f"이미지 검사: {ok} — {msg}")
    
    ok, msg = checker.check_mask(test_mask)
    print(f"마스크 검사: {ok} — {msg}")
    
    # 워터마킹 테스트
    watermarked = embed_invisible_watermark(test_img, "WIA-TEST-2026")
    extracted = extract_watermark(watermarked)
    print(f"워터마크 삽입: WIA-TEST-2026")
    print(f"워터마크 추출: {extracted}")
    print(f"일치: {extracted == 'WIA-TEST-2026'}")
```

## 함정 카드 — 자주 실수하는 5가지

1. **"공공기관 자료는 무조건 무료라고 생각한다"** → 국립중앙도서관도 자료에 따라 공공누리 2유형(변경 금지)이 있고, 일부는 저작권이 살아있다. 국가기록원 OpenAPI 사용 시 반드시 각 자료의 `drm_yn` 필드 확인. 무결성 없는 라이선스 관리는 나중에 서비스 전체를 위협한다.

2. **"복원과 생성을 혼동한다"** → 복원(Restoration)은 원본에 최대한 충실해야 한다. 프롬프트로 "더 예쁘게" 또는 "현대적으로" 유도하는 것은 복원이 아니라 창작 조작. `strength=0.6~0.85`를 엄수하고 텍스트 프롬프트는 "clear, preserve original" 방향으로만.

3. **"감사 로그를 나중에 추가하겠다고 미룬다"** → 감사 로그는 서비스 첫날부터 있어야 한다. 사건이 터진 후에 로그를 넣으면 이미 늦다. 최소 입력 이미지 해시 + 타임스탬프만이라도 첫날부터 기록.

4. **"딥페이크 차단은 마지막에 하겠다고 생각한다"** → 딥페이크 방지 메커니즘은 서비스 설계 초기부터 내장해야 한다. 출시 후 추가하면 기존 API 구조 전체를 바꿔야 하는 경우가 많다. 처음부터 `EthicalRestorationPipeline` 패턴으로 설계.

5. **"워터마킹이 완벽하다고 믿는다"** → LSB 스테가노그래피는 이미지 압축(JPEG 저장, 리사이즈)에 취약하다. 복원 이미지를 PNG로만 제공하거나, 더 강건한 DCT 도메인 워터마킹 (`invisible-watermark` 라이브러리) 사용 권장. 기술적 워터마킹은 보조 수단일 뿐, 법적·정책적 조치와 함께 사용.

## 연습문제

**문제 1.** 고문서 복원 서비스에서 저작권 소멸 자료와 저작권 있는 자료를 자동으로 구분하는 파이프라인을 설계하라. 어떤 데이터소스와 API를 활용할 수 있는가?

> **해설:** ① 국립중앙도서관 API: `drm_yn=Y` 파라미터로 DRM 없는 공개 자료만 필터. ② SEOJI (서지정보유통): 출판연도 + 저자 사망연도 조회 → 70년 경과 확인. ③ 국가기록원 OpenAPI: `open_yn=Y` 필드 확인. ④ 로컬 캐시: 한 번 확인한 이미지의 라이선스 정보를 Redis/SQLite에 저장하여 API 반복 호출 방지. ⑤ 불확실한 자료는 `license=unknown`으로 표시하고 수동 검수 큐에 추가.

**문제 2.** EU AI Act(2024)가 고문서 복원 AI 서비스에 어떤 의무를 부과하는가? 주요 조항 3가지를 설명하라.

> **해설:** ① **Article 52 (투명성 의무)**: AI가 생성하거나 수정한 콘텐츠는 그 사실을 명시해야 함 → 복원 이미지에 "AI 처리 사실" 워터마크 또는 메타데이터 필수. ② **Article 10 (데이터 거버넌스)**: 학습 데이터의 출처, 라이선스, 개인정보 포함 여부 문서화. ③ **Annex III (고위험 AI 목록)**: 문화유산 관련 AI가 특정 조건에서 고위험으로 분류될 수 있으며, 이 경우 적합성 평가(conformity assessment) 필요. 현실적으로 역사 문서 복원은 중간 위험으로 분류될 가능성 높음.

**문제 3.** 누군가 복원된 고문서 이미지를 위조에 사용했다고 주장할 경우, 감사 로그와 워터마킹을 통해 어떻게 서비스의 무결성을 증명하겠는가?

> **해설:** ① 감사 로그에서 해당 이미지 해시로 원본 요청 검색 → 요청 시간, IP, 입력 이미지 해시 확인 ② `extract_watermark(suspicious_image)` → "WIA-GEULSALIM-AI-RESTORED" 추출되면 서비스 통해 복원된 이미지 ③ PNG 메타데이터 검사: `exiftool suspicious.png` → 복원 날짜, 운영자 정보 ④ 원본 고문서 소유 기관(국립중앙도서관 등)에 원본과 비교 요청 → 내용 변조 여부 전문가 감정. 이 전체 과정이 처음부터 설계돼 있어야 신속한 대응 가능.
