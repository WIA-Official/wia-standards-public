상세한 기술 요구사항과 데이터 형식을 정의합니다.

제4장: 구현 가이드

단계별 구현 방법과 참조 아키텍처를 제공합니다.

제5장: API 및 SDK

프로그래밍 인터페이스와 개발 도구를 설명합니다.

제6장: 보안 고려사항
