### 2.1.4 규제 분절로 인한 과제

| 과제 | 영향 | 예시 |
|-----|-----|-----|
| 규정 준수 복잡성 | 다국적 기업은 각 시장에서 다른 요구사항에 직면 | PC 제조업체는 27개 EU 회원국과 25개 미국 주에 준수 |
| 시장 왜곡 | 제품이 규제가 적은 시장으로 흐름 | 중고 전자기기가 아프리카, 동남아로 수출 |
| 보고 부담 | 다양한 형식, 일정, 정의 | 범주 정의가 WEEE와 미국 주법 간 다름 |
| 시행 격차 | 일부 관할권에서 약한 시행 | "중고품" 또는 "기부품"으로 라벨링된 국경간 선적 |
| 순환경제 장벽 | 다른 품질/인증 표준 | 한 시장에서 허용되는 재활용 플라스틱이 다른 시장에서는 거부 |

### 2.2.3 일반적인 밀수 방법

|-----|-----|---------|
| 오라벨링 | "중고품", "기부품", "부품"으로 신고 | 기능성 확인을 위한 검사 필요 |

### 2.3.3 공식 재활용 산업에 대한 영향

| 영향 | 설명 |
|-----|-----|
| 가격 경쟁 | 비공식 운영자가 전자폐기물 수거에 더 높은 가격 제시 |
| 재료 전용 | 고가치 품목이 공식 시설 도달 전에 전용 |
| 품질 저하 | 조악한 처리가 재료 스트림 오염 |
| 시장 왜곡 | 공식 재활용업체가 가격 경쟁 불가 |
| 투자 억제 | 불확실성이 시설 투자 저해 |
| 선별적 수거 | 비공식이 가치있는 품목 가져가고, 공식은 잔류물 받음 |

## 2.6 복습 문제

### 문제 1
EU WEEE 지침과 한국 EPR 제도의 생산자 책임 메커니즘을 비교하시오. 각 접근법의 장단점은 무엇인가?

### 문제 2
"중고 전자기기" 컨테이너가 EU에서 서아프리카로 운송됩니다. 밀수업자들이 바젤 협약 통제를 우회하는 세 가지 방법과 시행이 어려운 이유를 설명하시오.

설명된 경제학을 바탕으로 공식 재활용과 비공식 재활용 간의 톤당 대략적인 비용 차이를 계산하시오. 공식 재활용업체가 단순히 비공식 가격에 맞출 수 없는 이유는?

### 문제 4
스마트폰 제조업체가 수명 종료 재활용 가능성을 개선하고자 합니다. 그들이 할 수 있는 다섯 가지 설계 변경과 각각의 재활용 이점을 설명하시오.

## 2.7 핵심 요약

| 과제 영역 | 핵심 이슈 | 표준화 기회 |
|---------|---------|----------|
| 규제 분절 | 다른 정의, 목표, 메커니즘 | 공통 데이터 형식, 상호 인정 |
| 국경간 이동 | 오라벨링, 시행 격차 | 관리 연속성 프로토콜 |
| 비공식 부문 | 경제적 이점, 외부화된 비용 | 투명성을 통한 공정한 경쟁 |
| 기술 장벽 | 폐기 대비 설계, 식별 격차 | 제품 여권, 재료 데이터베이스 |
| 데이터 격차 | 추적성 없음, 상호운용성 실패 | 보편적 ID, 공통 API |
| 시장 과제 | 변동성, 품질 불확실성 | 재료 인증 표준 |

### 핵심 통계
- **34%** EU 수출 컨테이너 오라벨링
- **10-50배** 비공식 부문 노동 비용 우위
- **25개** 미국 주에 전자폐기물법, 25개 없음
- **<20%** 전자폐기물 글로벌 공식 추적

### 다음 장 미리보기

제3장에서는 WIA 전자폐기물 관리 표준의 아키텍처를 소개하며, 4단계 접근법(데이터 형식, API 인터페이스, 프로토콜, 시스템 통합)이 이 장에서 식별된 각 과제를 어떻게 해결하는지 보여줍니다.
