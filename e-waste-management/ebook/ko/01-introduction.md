## 1.1 전자폐기물 위기

### 1.1.1 전자폐기물이란?

전자폐기물(e-waste)은 수명이 다했거나 소유자가 더 이상 원하지 않는 모든 폐전기전자제품(EEE)을 포함합니다:

```
전자폐기물 범주:
├── 대형 가전제품
│   ├── 냉장고 및 냉동고
│   ├── 세탁기
│   ├── 에어컨
│   └── 식기세척기
├── 소형 가전제품
│   ├── 청소기
│   ├── 토스터 및 커피메이커
│   ├── 헤어드라이어
│   └── 전기면도기
├── IT 및 통신 장비
│   ├── 데스크톱 컴퓨터
│   ├── 노트북 컴퓨터
│   ├── 스마트폰 및 태블릿
│   ├── 프린터 및 스캐너
│   └── 네트워크 장비
├── 소비자 가전
│   ├── 텔레비전
│   ├── 오디오 장비
│   ├── 카메라
│   └── 게임 콘솔
├── 조명 장비
│   ├── 형광등
│   ├── LED 조명
│   └── 고휘도 방전 램프
└── 배터리 및 축전지
    ├── 납축전지
    ├── 리튬이온 배터리
    ├── 니켈카드뮴 배터리
    └── 알카라인 배터리
```

### 1.1.2 글로벌 전자폐기물 통계

글로벌 전자폐기물 과제의 규모는 엄청나며 빠르게 성장하고 있습니다:

| 지표 | 2019년 | 2022년 | 2025년 (예상) | 2030년 (예상) |
|-----|-------|-------|-------------|-------------|
| 글로벌 발생량 (Mt) | 53.6 | 62.0 | 74.7 | 82.0 |
| 1인당 (kg) | 7.3 | 7.8 | 9.0 | 9.5 |
| 공식 수거율 | 17.4% | 22.3% | 30% | 40% |
| 재료 가치 ($B) | 57 | 67 | 80 | 95 |

## 1.2 전자폐기물의 재료 구성

### 1.2.1 가치있는 재료

전자폐기물은 1차 광석보다 높은 농도의 가치있는 재료를 포함합니다:

```
재료 가치 비교 (톤당):
┌────────────────────────────────────────────────────────────────────┐
│ 재료       │ 전자폐기물 함량    │ 1차 광석 품위      │ 가치    │
├────────────┼──────────────────┼──────────────────┼────────┤
│ 금         │ 200-350 g/t (PCB)│ 1-5 g/t          │ 높음   │
│ 은         │ 500-2000 g/t     │ 100-300 g/t      │ 중간   │
│ 팔라듐     │ 50-100 g/t       │ 2-10 g/t         │ 높음   │
│ 구리       │ 10-20% (PCB)     │ 0.5-1%           │ 높음   │
│ 알루미늄   │ 5-15%            │ 20-25% (보크사이트)│ 중간   │
│ 철/강철    │ 20-30%           │ 30-65%           │ 낮음   │
│ 플라스틱   │ 15-25%           │ N/A              │ 낮음   │
│ 유리       │ 5-15%            │ N/A              │ 낮음   │
└────────────────────────────────────────────────────────────────────┘
```

### 1.2.2 핵심 및 희토류 원소

현대 전자기기는 기술에 필수적이지만 회수가 어려운 원소를 포함합니다:

| 원소 | 주요 용도 | 일반적 함량 | 회수 과제 |
|-----|---------|-----------|---------|
| 리튬 | 배터리 | 리튬이온 배터리 3-7% | 처리 중 화재 위험 |
| 코발트 | 배터리 양극재 | 배터리 5-20% | 비용이 많이 드는 추출 |
| 네오디뮴 | 자석, 스피커 | 자석 10-30% | 복잡한 분리 |
| 인듐 | 디스플레이, 터치스크린 | 0.01-0.1% | 매우 낮은 농도 |
| 탄탈럼 | 커패시터 | 커패시터 0.5-5% | 작은 부품 크기 |
| 갈륨 | LED, 반도체 | 미량 | 극히 어려움 |

## 1.3 환경 및 건강 영향

### 1.3.1 환경 오염

부적절한 전자폐기물 처리는 심각한 환경 오염을 유발합니다:

```
환경 영향 경로:
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  ┌──────────────┐                                                   │
│  │   매립       │ → 침출 → 지하수 오염                              │
│  │   처리       │ → 중금속 토양 침투                                │
│  └──────────────┘                                                   │
│         │                                                           │
│         ▼                                                           │
│  ┌──────────────┐                                                   │
│  │   비공식     │ → 노천 소각 → 대기 오염 (다이옥신, 퓨란)          │
│  │   재활용     │ → 산 침출 → 수질 오염                            │
│  └──────────────┘ → 잔류물 투기 → 토양 오염                        │
│         │                                                           │
│         ▼                                                           │
│  ┌──────────────┐                                                   │
│  │   생태계     │ → 먹이사슬 생물축적                               │
│  │   영향       │ → 생물다양성 손실                                 │
│  └──────────────┘ → 장기 토양 악화                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**사례 연구: 중국 구이위**
- 위치: 광동성, 역사적으로 세계 최대 전자폐기물 처리 지역
- 노출 인구: 약 15만 명의 작업자
- 발견 사항:
  - 어린이 혈중 납 농도 대조군보다 54% 높음
  - 토양 납 농도 비오염 지역보다 371배 높음
  - 퇴적물 다이옥신 농도 세계 최고 수준
  - 음용수 사용 불가, 30km 떨어진 곳에서 트럭으로 운반

### 1.3.2 비공식 재활용의 건강 영향

| 활동 | 노출 경로 | 건강 영향 | 영향 그룹 |
|-----|---------|---------|---------|
| 케이블 소각 | 연기 흡입 | 호흡기 질환, 염소성 여드름 | 작업자, 인근 지역사회 |
| 산 침출 | 피부 접촉, 증기 | 화학적 화상, 폐 손상 | 작업자 |
| CRT 파손 | 분진 흡입 | 납 중독 | 작업자, 어린이 |
| 토너 취급 | 분진 흡입 | 호흡기 문제 | 작업자 |
| 배터리 분해 | 접촉, 증기 | 납/산 노출 | 작업자 |
| 일반 노출 | 환경 오염 | 발달 문제 | 지역사회 어린이 |

## 1.5 표준화의 필요성

### 1.5.1 표준화가 필요한 이유

현재의 분절된 환경은 심각한 문제를 만듭니다:

```
표준화가 필요한 문제:
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  1. 추적성 격차                                                     │
│     └─ 기기가 비공식 채널로 사라짐                                  │
│     └─ 실제 재활용 vs. 투기에 대한 가시성 없음                      │
│     └─ 생산자 책임을 검증할 수 없음                                 │
│                                                                     │
│  2. 상호운용성 실패                                                 │
│     └─ 서로 다른 추적 시스템이 통신하지 않음                        │
│     └─ 국가 보고를 위한 데이터 집계 불가                            │
│     └─ 다국적 운영이 통합 장벽에 직면                               │
│                                                                     │
│  3. 규정 준수 복잡성                                                │
│     └─ 각 관할권마다 다른 요구사항                                  │
│     └─ 규정 준수 증명에 여러 감사 필요                              │
│     └─ 소규모 재활용업체가 인증을 감당할 수 없음                    │
│                                                                     │
│  4. 시장 왜곡                                                       │
│     └─ 비공식 운영자가 공식 재활용업체보다 저렴                     │
│     └─ 환경 기준에서 바닥으로의 경쟁                                │
│     └─ 품질 재료 시장이 발전할 수 없음                              │
│                                                                     │
│  5. 순환경제 장벽                                                   │
│     └─ 2차 재료에 품질 인증 부족                                    │
│     └─ 재료 구성 데이터가 보존되지 않음                             │
│     └─ 재사용/리퍼비시 경로 불명확                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.5.2 WIA 표준의 이점

| 이해관계자 | 이점 |
|----------|-----|
| **생산자** | 간소화된 EPR 규정 준수, 표준화된 보고, 감소된 감사 부담 |
| **수거업체** | 상호운용 가능한 추적, 자동화된 계량 검증, 디지털 관리 연속성 |
| **재활용업체** | 재료 구성 데이터, 품질 인증, 시장 접근 |
| **규제기관** | 실시간 모니터링, 자동화된 보고, 시행 가시성 |
| **소비자** | 투명한 재활용 증명, 쉬운 반품 채널, 환경 보증 |
| **환경** | 비공식 부문으로의 누출 감소, 높은 회수율, 추적 가능한 유해물질 |

## 1.6 복습 문제

### 문제 1
전자폐기물에서 발견되는 세 가지 주요 가치있는 재료 범주는 무엇이며, "도시 광업"이 1차 광업보다 잠재적으로 더 효율적인 이유는?

### 문제 2
비공식 전자폐기물 재활용에서의 환경 오염 경로를 설명하고, 영향받는 지역사회에서 문서화된 건강 영향 두 가지를 제시하시오.

### 문제 3
글로벌 전자폐기물의 50-60%가 "불명"한 운명을 가지는 이유와 더 나은 추적을 방해하는 장벽이 무엇인지 설명하시오.

### 문제 4
한국의 EPR 제도에서 생산자의 재활용 의무 이행 방법과 미이행 시 제재를 설명하시오.

### 문제 5
현재 전자폐기물 관리 환경에서 표준화가 필요한 다섯 가지 문제를 식별하고, 각각에 대해 보편적 표준이 어떻게 해결할 수 있는지 설명하시오.

## 1.7 핵심 요약

| 주제 | 핵심 포인트 |
|-----|-----------|
| 전자폐기물 규모 | 연간 62+ Mt 발생, 연간 2-3% 성장 |
| 재료 가치 | 1차 광석보다 높은 농도의 귀금속 포함 |
| 유해 물질 | 납, 수은, 카드뮴, 브롬화 난연제는 특별 취급 필요 |
| 건강 영향 | 비공식 재활용은 문서화된 신경독성, 호흡기 질환, 암 유발 |
| 수거 격차 | 17-22%만 공식 수거; 대부분 운명 불명 |
| 표준화 필요 | 추적성, 상호운용성, 규정 준수 검증 모두 공통 표준 필요 |

### 기억해야 할 핵심 통계
- **53.6 Mt** (2019년) → **82 Mt** (2030년): 예상 글로벌 전자폐기물 성장
- **$670억**: 글로벌 전자폐기물 내 연간 재료 가치 (2022년)
- **200-350 g/t**: PCB의 금 함량 (광석 1-5 g/t 대비)
- **17.4%**: 글로벌 공식 수거율
- **75%**: 한국의 EPR 수거율 (법정 의무량 기준)
