# WIA Physics Standard Ebook - Complete Documentation

## Overview
Complete ebook for WIA Physics Standard covering physics simulation and data standards.

- **Total Files:** 18 HTML files
- **Languages:** English (en/) and Korean (ko/)
- **Chapters:** 8 comprehensive chapters + 1 index per language

## Viewing the Ebook

### Open in Browser
```bash
# English version
open /home/user/wia-standards/physics/ebook/en/index.html

# Korean version
open /home/user/wia-standards/physics/ebook/ko/index.html
```

### Local Web Server (recommended)
```bash
cd /home/user/wia-standards/physics/ebook
python3 -m http.server 8000

# Then navigate to:
# http://localhost:8000/en/index.html
# http://localhost:8000/ko/index.html
```
