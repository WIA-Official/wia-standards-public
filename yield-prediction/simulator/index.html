Data Quality Standards

• GPS coordinates required (±10m accuracy)
• Weather data: hourly or daily resolution
• Soil tests: minimum 3-year intervals
• Yield measurement: calibrated equipment

Model Update Triggers

• New harvest data available
• Prediction error > 15%
• Extreme weather events
• Major input changes (new cultivar)

Validation Metrics

• RMSE (Root Mean Square Error)
• MAE (Mean Absolute Error)
• R² Score (>0.85 target)
• Cross-validation (5-fold minimum)
