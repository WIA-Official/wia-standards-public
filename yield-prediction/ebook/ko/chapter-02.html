제2장

데이터 수집 및 관리

2.1 역사적 수확량 데이터

모든 수확량 예측 시스템의 기초는 고품질의 역사적 수확량 데이터입니다. 여기에는 여러 재배 시즌에 걸쳐 측정된 실제 수확량이 포함됩니다.

필수 데이터 항목

  • 작물 종류 및 품종 (예: 쌀-신동진벼)
  • 지리적 위치 (GPS 좌표)
  • 농장/포장 면적
  • 파종 및 수확 날짜
  • 실제 수확량 (kg/ha 또는 kg/10a)
  • 투입 자재 데이터 (비료, 관개, 농약)
모범 사례: 강력한 모델 훈련을 위해 최소 5-10년의 역사적 데이터를 수집하십시오. 일반적으로 더 많은 데이터가 더 나은 예측으로 이어집니다.

2.2 기상 데이터 통합

기상은 작물 수확량에 영향을 미치는 가장 중요한 단일 요인입니다. 포괄적인 기상 데이터 수집이 중요합니다.

주요 기상 변수

  • 기온 (일 최저, 최고, 평균)
  • 강수량/강우량
  • 일조 시간 또는 일사량
  • 습도
  • 풍속
  • 증발산량

데이터 소스

기상 데이터는 다음에서 얻을 수 있습니다:

  • 국가 기상청 (한국의 경우 기상청 KMA)
  • 상업용 기상 API
  • 농장 내 기상 관측소
  • 위성 기반 기상 추정

2.3 토양 데이터 수집

토양 특성은 수확량 잠재력에 큰 영향을 미칩니다. 주요 토양 매개변수는 다음과 같습니다:

  • 토양 유형 및 질감 (사질토, 양토, 점토)
  • pH 수준 (5.5-7.0이 대부분의 작물에 최적)
  • 유기물 함량
  • 영양소 수준 (질소 N, 인 P, 칼륨 K)
  • 보수력
  • 배수 특성

2.4 한국 농업 데이터 표준

한국에서는 수확량 데이터가 통계청(KOSIS)의 특정 표준을 따릅니다:

한국 단위

  • 재배면적: 경작 면적 (헥타르 또는 10a)
  • 단수: 단위 면적당 수확량 (kg/10a)
  • 생산량: 총 생산량 (톤)

단위 변환

한국 농업은 전통적으로 10a (1,000 m²)를 면적 단위로 사용합니다. 변환하려면:

  • 1 헥타르(ha) = 10 × 10a
  • 1 kg/10a = 10 kg/ha

예: 쌀 수확량 520 kg/10a = 5,200 kg/ha

2.5 데이터 품질 표준

WIA-AGRI-008 표준은 품질 요구사항을 정의합니다:

필수 필드

  • 농장/포장 고유 식별자
  • 작물 유형
  • GPS 좌표 (±10m 정확도)
  • 수확 연도
  • 단위가 있는 실제 수확량
  • 데이터 수집 타임스탬프

검증 규칙

  • 작물 유형에 대한 현실적인 범위 내의 수확량 값
  • 수확 날짜 이전의 파종 날짜
  • 유효한 지리적 좌표
  • 중복 레코드 없음
  • 레코드당 누락 데이터 < 10%

2.6 데이터 저장 형식

표준화된 형식은 상호 운용성을 보장합니다. WIA-AGRI-008은 JSON 및 CSV 형식을 지원합니다.

한국 쌀 데이터 예시:
농장ID: KR-충남-당진-2025-001
작물: 쌀 (신동진벼)
수확량: 520 kg/10a (5,200 kg/ha)
위치: 충청남도 당진시 (36.8934°N, 126.6469°E)
