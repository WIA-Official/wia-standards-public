4.2 행동 분석

분석 유형 기준선 이상 지표
사용자 행동 (UEBA) 로그인 시간, 접근 패턴 비업무 시간 접근, 비정상 위치
엔터티 행동 장치 활동, 데이터 흐름 새 목적지, 데이터 볼륨 급증
프로세스 행동 부모-자식 관계 비정상 스폰, 인젝션
네트워크 행동 트래픽 패턴, 프로토콜 비콘, 터널링, 유출

4.3 MITRE ATT&CK 프레임워크

위협의 공통 언어

MITRE ATT&CK는 실제 관찰을 기반으로 적대자 전술과 기법의 전 세계적으로 접근 가능한 지식 기반을 제공합니다. WIA-SEC-015는 모든 탐지 규칙을 ATT&CK 기법에 매핑합니다.

전술 설명 예시 기법
초기 접근 네트워크 진입 피싱, 유효한 계정
실행 악성 코드 실행 PowerShell, 명령줄
지속성 접근 유지 레지스트리 실행 키, 예약 작업
권한 상승 더 높은 권한 획득 익스플로잇, 토큰 조작
방어 회피 탐지 회피 난독화, 로그 삭제
자격 증명 접근 자격 증명 탈취 키로깅, 자격 증명 덤핑
횡적 이동 네트워크 내 이동 RDP, SMB, Pass-the-Hash
데이터 유출 데이터 탈취 암호화 채널, C2

4.5 위협 헌팅

4.5.1 헌팅 방법론

  • 가설 주도: 위협 행위자 TTP로 시작
  • 정보 주도: 알려진 IOC 헌팅
  • 이상 주도: 통계적 이상값 조사
  • 엔터티 주도: 의심스러운 엔터티 심층 분석

4.5.2 헌팅 프로세스

단계 활동 산출물
가설 정보 기반 헌팅 가설 개발 헌팅 계획, 쿼리
조사 데이터 검색, 패턴 분석 발견사항, 아티팩트
검증 가설 확인 또는 반박 검증된 발견사항
대응 필요시 에스컬레이션, 봉쇄 사고, IOC
개선 탐지 규칙 생성 새 탐지
