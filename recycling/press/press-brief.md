# WIA-ENE-023 — Recycling (Press Brief)

**WIA Standards releases canonical Recycling envelope shape**

The WIA Standards Committee has published the WIA-ENE-023 standard covering the canonical envelope shape, API surface, protocol exchanges, and ecosystem integration for recycling hosts. The standard composes with the wider WIA Standards family for cross-standard audit transport, identity, and federation.
