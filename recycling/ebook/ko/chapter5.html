AI 분류 시스템의 필요성

전통적인 재활용 시설에서 수동 분류는 여전히 주요 병목 현상입니다. 작업자들은 빠르게 이동하는 컨베이어 벨트에서 다양한 재활용 가능 물질을 식별하고 분리해야 합니다. 이 과정은 피로, 부상 위험, 그리고 인간의 인지적 한계로 인해 상당한 오류율을 보입니다. AI 기반 분류 시스템은 이러한 문제를 근본적으로 해결합니다.

AI 분류 시스템은 고속 카메라, 근적외선 센서, 그리고 딥러닝 알고리즘을 결합하여 인간의 능력을 훨씬 초월하는 속도와 정확도로 재활용 가능 물질을 식별합니다. 이러한 시스템은 밀리초 단위로 수백 가지 재료 유형을 구별할 수 있으며, 피로 없이 연속 작동이 가능합니다.

왜 AI인가?

재활용 물류의 복잡성은 기존 자동화 방식으로는 해결하기 어렵습니다. 재활용품은 형태, 색상, 크기, 오염 상태가 매우 다양하며, 새로운 포장재와 제품이 지속적으로 시장에 출시됩니다. 규칙 기반 시스템은 이러한 변동성에 적응하기 어렵지만, AI 시스템은 새로운 데이터로부터 학습하고 점진적으로 개선됩니다.

AI 분류의 핵심 장점

  • 적응성: 새로운 포장재와 제품에 대해 재학습 가능
  • 일관성: 피로, 기분, 주의력 변화에 영향받지 않음
  • 속도: 초당 수십 개의 항목을 처리
  • 정밀성: 미세한 재료 차이까지 감지
  • 데이터 생성: 분류 과정에서 귀중한 분석 데이터 생성

컴퓨터 비전 아키텍처

현대 AI 분류 시스템의 핵심은 컴퓨터 비전입니다. 이 시스템은 여러 계층의 기술이 통합되어 작동합니다.

시스템 아키텍처 계층

1. 센서 계층 (Sensor Layer)

고해상도 RGB 카메라, 근적외선(NIR) 분광계, 깊이 센서(LiDAR), 금속 탐지기가 원시 데이터를 수집합니다.

2. 전처리 계층 (Preprocessing Layer)

이미지 정규화, 노이즈 제거, 조명 보정, 분할(segmentation)을 통해 원시 데이터를 정제합니다.

3. 특징 추출 계층 (Feature Extraction Layer)

CNN(합성곱 신경망)이 시각적 특징을 추출하고, 분광 데이터가 재료 특성을 분석합니다.

4. 분류 계층 (Classification Layer)

멀티모달 퓨전을 통해 모든 데이터 소스를 결합하고 최종 분류 결정을 내립니다.

5. 액추에이터 계층 (Actuator Layer)

고속 로봇 암 또는 공기 분사기가 분류된 항목을 적절한 스트림으로 분리합니다.

다중 센서 융합

단일 센서 시스템은 특정 상황에서 한계를 보입니다. 예를 들어, RGB 카메라는 시각적으로 유사한 PET와 PP 플라스틱을 구별하기 어렵고, NIR 센서만으로는 검은색 플라스틱을 탐지할 수 없습니다. 다중 센서 융합은 각 센서의 강점을 결합하여 이러한 한계를 극복합니다.

// 다중 센서 융합 분류기 구현 import tensorflow as tf from tensorflow.keras import layers, models class MultiSensorFusionClassifier: """ RGB, NIR, 깊이 데이터를 융합하는 분류기 """ def __init__(self, num_classes=50): self.num_classes = num_classes self.model = self._build_model() def _build_rgb_branch(self, input_shape=(224, 224, 3)): # RGB 이미지 처리 브랜치 inputs = layers.Input(shape=input_shape, name='rgb_input') # EfficientNet-B0 백본 base = tf.keras.applications.EfficientNetB0( include_top=False, weights='imagenet', input_tensor=inputs ) x = layers.GlobalAveragePooling2D()(base.output) x = layers.Dense(512, activation='relu')(x) x = layers.Dropout(0.3)(x) return models.Model(inputs, x, name='rgb_branch') def _build_nir_branch(self, input_shape=(256,)): # NIR 분광 데이터 처리 브랜치 inputs = layers.Input(shape=input_shape, name='nir_input') x = layers.Dense(512, activation='relu')(inputs) x = layers.BatchNormalization()(x) x = layers.Dense(256, activation='relu')(x) x = layers.Dropout(0.3)(x) x = layers.Dense(128, activation='relu')(x) return models.Model(inputs, x, name='nir_branch') def _build_depth_branch(self, input_shape=(112, 112, 1)): # 깊이 맵 처리 브랜치 inputs = layers.Input(shape=input_shape, name='depth_input') x = layers.Conv2D(32, 3, activation='relu', padding='same')(inputs) x = layers.MaxPooling2D()(x) x = layers.Conv2D(64, 3, activation='relu', padding='same')(x) x = layers.MaxPooling2D()(x) x = layers.Conv2D(128, 3, activation='relu', padding='same')(x) x = layers.GlobalAveragePooling2D()(x) x = layers.Dense(64, activation='relu')(x) return models.Model(inputs, x, name='depth_branch') def _build_model(self): # 모든 브랜치 결합 rgb_branch = self._build_rgb_branch() nir_branch = self._build_nir_branch() depth_branch = self._build_depth_branch() # 특징 융합 combined = layers.Concatenate()([ rgb_branch.output, nir_branch.output, depth_branch.output ]) # 분류 헤드 x = layers.Dense(512, activation='relu')(combined) x = layers.BatchNormalization()(x) x = layers.Dropout(0.4)(x) x = layers.Dense(256, activation='relu')(x) outputs = layers.Dense(self.num_classes, activation='softmax')(x) model = models.Model( inputs=[ rgb_branch.input, nir_branch.input, depth_branch.input ], outputs=outputs ) return model def predict(self, rgb_image, nir_spectrum, depth_map): # 실시간 추론 predictions = self.model.predict([ rgb_image[np.newaxis, ...], nir_spectrum[np.newaxis, ...], depth_map[np.newaxis, ...] ]) material_id = np.argmax(predictions[0]) confidence = predictions[0][material_id] return { 'material_id': material_id, 'confidence': float(confidence), 'processing_time_ms': self._get_inference_time() }

커스텀 하이브리드 모델

WIA-RECYCLING 표준에서 권장하는 하이브리드 모델은 EfficientNet의 효율성과 Transformer의 전역 컨텍스트 이해 능력을 결합합니다. 이 아키텍처는 지역적 특징(텍스처, 색상)과 전역적 특징(형태, 문맥)을 동시에 활용합니다.

// 하이브리드 CNN-Transformer 모델 class HybridRecyclingClassifier(nn.Module): """ CNN과 Transformer를 결합한 하이브리드 분류기 로컬 및 글로벌 특징을 동시에 캡처 """ def __init__(self, num_classes=50, embed_dim=512): super().__init__() # CNN 백본 (로컬 특징 추출) self.cnn_backbone = timm.create_model( 'efficientnet_b2', pretrained=True, num_classes=0 # 헤드 제거 ) # CNN 출력을 Transformer 차원으로 프로젝션 self.cnn_proj = nn.Linear(1408, embed_dim) # Transformer 인코더 (글로벌 컨텍스트) encoder_layer = nn.TransformerEncoderLayer( d_model=embed_dim, nhead=8, dim_feedforward=2048, dropout=0.1, activation='gelu', batch_first=True ) self.transformer = nn.TransformerEncoder( encoder_layer, num_layers=4 ) # 위치 인코딩 self.pos_embed = nn.Parameter( torch.randn(1, 49, embed_dim) * 0.02 ) # 분류 헤드 self.classifier = nn.Sequential( nn.LayerNorm(embed_dim), nn.Dropout(0.3), nn.Linear(embed_dim, num_classes) ) # 재료 속성 보조 헤드 self.material_head = nn.Sequential( nn.Linear(embed_dim, 128), nn.ReLU(), nn.Linear(128, 5) # 밀도, 광택, 투명도, 유연성, 재활용성 ) def forward(self, x): # CNN 특징 추출 cnn_features = self.cnn_backbone.forward_features(x) # 공간 차원 평탄화: (B, C, H, W) -> (B, H*W, C) B, C, H, W = cnn_features.shape cnn_features = cnn_features.flatten(2).transpose(1, 2) # 프로젝션 및 위치 인코딩 x = self.cnn_proj(cnn_features) x = x + self.pos_embed # Transformer 인코딩 x = self.transformer(x) # 글로벌 평균 풀링 x = x.mean(dim=1) # 분류 및 재료 속성 예측 class_logits = self.classifier(x) material_props = self.material_head(x) return class_logits, material_props

근적외선(NIR) 분광 분석

시각적으로 유사한 플라스틱을 구별하는 데 있어 NIR 분광법은 필수적입니다. NIR 센서는 각 재료의 고유한 분자 진동 패턴을 감지하여 화학적 조성을 식별합니다.

NIR 스펙트럼 해석

// NIR 스펙트럼 분석기 구현 import numpy as np from scipy.signal import savgol_filter from sklearn.preprocessing import StandardScaler class NIRSpectrumAnalyzer: """ NIR 스펙트럼 전처리 및 분석 """ # 알려진 재료의 특징적 피크 위치 (nm) MATERIAL_PEAKS = { 'PET': [1660, 2140, 2270], 'HDPE': [1720, 1760, 2310], 'LDPE': [1725, 1765, 2305], 'PP': [1195, 1720, 2270], 'PS': [1685, 2165, 2450], 'PVC': [1720, 2275, 2390], 'PAPER': [1450, 1940, 2100], 'CARDBOARD': [1450, 1935, 2095], } def __init__(self, wavelengths): """ wavelengths: 센서의 파장 범위 (nm) """ self.wavelengths = wavelengths self.scaler = StandardScaler() def preprocess_spectrum(self, spectrum): """ 원시 스펙트럼 데이터 전처리 1. Savitzky-Golay 평활화 2. 기준선 보정 3. SNV (표준 정규 변수) 정규화 4. 1차 미분 """ # 평활화 (노이즈 제거) smoothed = savgol_filter(spectrum, window_length=11, polyorder=3) # 기준선 보정 (다항식 피팅) baseline = self._fit_baseline(smoothed) corrected = smoothed - baseline # SNV 정규화 snv_normalized = (corrected - np.mean(corrected)) / np.std(corrected) # 1차 미분 (피크 강조) derivative = savgol_filter( snv_normalized, window_length=11, polyorder=3, deriv=1 ) return derivative def _fit_baseline(self, spectrum): """다항식 기준선 피팅""" x = np.arange(len(spectrum)) coeffs = np.polyfit(x, spectrum, deg=2) baseline = np.polyval(coeffs, x) return baseline def identify_peaks(self, processed_spectrum, threshold=0.1): """특징적 피크 식별""" peaks = [] for i in range(1, len(processed_spectrum) - 1): if (processed_spectrum[i] > processed_spectrum[i-1] and processed_spectrum[i] > processed_spectrum[i+1] and abs(processed_spectrum[i]) > threshold): peak_wavelength = self.wavelengths[i] peak_intensity = processed_spectrum[i] peaks.append((peak_wavelength, peak_intensity)) return sorted(peaks, key=lambda x: abs(x[1]), reverse=True) def match_material(self, peaks, top_n=3): """피크 매칭을 통한 재료 식별""" scores = {} peak_wavelengths = [p[0] for p in peaks[:top_n]] for material, ref_peaks in self.MATERIAL_PEAKS.items(): score = 0 for peak_wl in peak_wavelengths: for ref_wl in ref_peaks: # 허용 오차 범위 내 매칭 if abs(peak_wl - ref_wl) < 20: # ±20nm score += 1 - abs(peak_wl - ref_wl) / 20 scores[material] = score / len(ref_peaks) # 점수로 정렬 sorted_materials = sorted( scores.items(), key=lambda x: x[1], reverse=True ) return sorted_materials

검은색 플라스틱 문제

검은색 플라스틱은 NIR 분광법의 주요 과제입니다. 카본 블랙 안료가 NIR 광을 흡수하여 스펙트럼 신호를 크게 감쇠시킵니다. 이 문제를 해결하기 위한 여러 접근법이 있습니다:

중적외선(MIR) 센서

2.5-25μm 파장 범위에서 작동하며, 카본 블랙의 영향을 덜 받습니다. 하지만 비용이 높고 처리 속도가 느립니다.

레이저 유도 파괴 분광법(LIBS)

레이저로 물질을 기화시켜 원소 조성을 분석합니다. 모든 색상의 플라스틱에 효과적입니다.

형광 표지자

제조 단계에서 플라스틱에 형광 마커를 추가하여 분류를 용이하게 합니다. 신규 제품에만 적용 가능합니다.

밀도 기반 분류

물이나 염용액에서의 부력 차이를 이용합니다. 정확도는 낮지만 보조 수단으로 활용됩니다.

로봇 분류 시스템

AI 시스템이 물질을 식별한 후, 물리적으로 분리하는 것은 로봇 시스템의 역할입니다. 현대 재활용 시설에서는 고속 델타 로봇과 공압 분사기가 주로 사용됩니다.

델타 로봇 아키텍처

델타 로봇은 병렬 운동학적 구조로 인해 높은 속도와 정밀도를 제공합니다. 재활용 분류에서는 초당 2-4회의 픽 앤 플레이스 작업을 수행할 수 있습니다.

델타 로봇 사양 요구 사항

  • 작업 공간: 최소 800mm 직경
  • 페이로드: 0.5-2kg (대부분의 재활용품 처리)
  • 정확도: ±0.5mm 반복 정밀도
  • 사이클 시간: < 500ms (픽-이동-플레이스)
  • 내구성: IP65 이상 (먼지/습기 보호)

실시간 AI 추론을 위해 GPU 가속은 필수입니다. NVIDIA Jetson 시리즈나 Intel Movidius VPU와 같은 엣지 AI 가속기가 일반적으로 사용됩니다.

// TensorRT 최적화된 추론 엔진 import tensorrt as trt import pycuda.driver as cuda import pycuda.autoinit import numpy as np class TensorRTInferenceEngine: """ TensorRT를 사용한 최적화된 추론 엔진 딥러닝 모델의 실시간 추론 가속 """ def __init__(self, engine_path: str): # TensorRT 로거 self.logger = trt.Logger(trt.Logger.WARNING) # 엔진 로드 with open(engine_path, 'rb') as f: engine_data = f.read() runtime = trt.Runtime(self.logger) self.engine = runtime.deserialize_cuda_engine(engine_data) self.context = self.engine.create_execution_context() # CUDA 스트림 생성 self.stream = cuda.Stream() # 메모리 할당 self._allocate_buffers() def _allocate_buffers(self): """입출력 버퍼 할당""" self.inputs = [] self.outputs = [] self.bindings = [] for binding in self.engine: shape = self.engine.get_binding_shape(binding) dtype = trt.nptype(self.engine.get_binding_dtype(binding)) size = trt.volume(shape) # 호스트 및 디바이스 메모리 host_mem = cuda.pagelocked_empty(size, dtype) device_mem = cuda.mem_alloc(host_mem.nbytes) self.bindings.append(int(device_mem)) if self.engine.binding_is_input(binding): self.inputs.append({ 'host': host_mem, 'device': device_mem, 'shape': shape }) else: self.outputs.append({ 'host': host_mem, 'device': device_mem, 'shape': shape }) def infer(self, input_data: np.ndarray) -> np.ndarray: """ 비동기 추론 실행 Args: input_data: 전처리된 입력 이미지 (NCHW 형식) Returns: 분류 확률 배열 """ # 입력 데이터 복사 np.copyto(self.inputs[0]['host'], input_data.ravel()) # 호스트 -> 디바이스 전송 cuda.memcpy_htod_async( self.inputs[0]['device'], self.inputs[0]['host'], self.stream ) # 추론 실행 self.context.execute_async_v2( bindings=self.bindings, stream_handle=self.stream.handle ) # 디바이스 -> 호스트 전송 cuda.memcpy_dtoh_async( self.outputs[0]['host'], self.outputs[0]['device'], self.stream ) # 스트림 동기화 self.stream.synchronize() # 출력 reshape output = self.outputs[0]['host'].reshape( self.outputs[0]['shape'] ) return output # 사용 예시 if __name__ == '__main__': engine = TensorRTInferenceEngine('recycling_classifier.trt') # 카메라에서 프레임 획득 frame = camera.capture() # 전처리 preprocessed = preprocess_image(frame) # 추론 (약 8-15ms) predictions = engine.infer(preprocessed) # 결과 해석 material_class = np.argmax(predictions) confidence = predictions[material_class]

  • 지역 특성: 지역별로 다른 제품 유형과 브랜드 포함
