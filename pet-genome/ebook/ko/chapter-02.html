2장: 유전체 프로필 및 데이터 표준

"표준화된 데이터 형식은 반려동물 유전체학에서 글로벌 상호운용성을 가능하게 합니다."

2.1 개요

이 장은 WIA-PET-GENOME 표준 전반에 걸쳐 사용되는 핵심 데이터 구조를 정의합니다. 일관된 데이터 형식은 전 세계 실험실, 수의 시스템, 등록 기관 간의 상호운용성을 가능하게 합니다.

핵심 데이터 엔티티

  • PetGenomeProfile - 반려동물 유전 정보의 마스터 레코드
  • GeneticMarker - 개별 유전 마커 및 변이
  • SequencingJob - DNA 시퀀싱 워크플로우 추적
  • GeneticFingerprint - 고유 신원 서명
  • HealthPrediction - 질병 위험 평가

2.2 프로필 식별자 형식

프로필 ID 구조

형식: PGP-YYYY-NNNNNN

구성:
  PGP    = Pet Genome Profile 접두사
  YYYY   = 등록 연도
  NNNNNN = 6자리 순차 번호

예시: PGP-2025-001234

길이고정 15자
대소문자대문자만 사용

2.8 품질 지표

프로필 품질 점수

품질 점수 = (커버리지_점수 * 0.3) +
           (콜_레이트 * 0.3) +
           (정확도_점수 * 0.4)

구성:
  커버리지_점수 = min(실제_커버리지 / 목표_커버리지, 1.0) * 100
  콜_레이트 = (성공_판정_마커 / 전체_마커) * 100
  정확도_점수 = 복제 일치율 기반

품질 임계값

수준점수 범위조치
우수95-100전체 분석 가능
양호85-94전체 분석 가능
허용70-84대부분 분석 가능
경계50-69제한된 분석, 재검사 권장
실패0-49재채취 필요

요약

이 장에서 정의한 데이터 구조는 WIA-PET-GENOME 표준의 기반을 형성합니다. 이러한 형식의 일관된 사용은 검사나 치료가 어디서 이루어지든 전 세계 시스템 간에 유전 데이터를 교환할 수 있게 하여, 반려동물에 대한 포괄적인 케어를 가능하게 합니다.
