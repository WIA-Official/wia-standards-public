API Protocol API 프로토콜

RESTful API endpoints for wetland monitoring and data management.

Available Endpoints 사용 가능한 엔드포인트

GET /api/v1/wetlands

List all registered wetlands 등록된 모든 습지 목록 조회

GET /api/v1/wetlands/{id}

Get detailed wetland information 상세 습지 정보 조회

POST /api/v1/wetlands

Register a new wetland 새 습지 등록

POST /api/v1/wetlands/{id}/monitor

Submit monitoring data 모니터링 데이터 제출

GET /api/v1/wetlands/{id}/health

Get ecosystem health metrics 생태계 건강 지표 조회

GET /api/v1/wetlands/{id}/species

Get biodiversity data 생물다양성 데이터 조회

PUT /api/v1/wetlands/{id}/restoration

Update restoration plan 복원 계획 업데이트
