Optical Communication Methods

Light piercing the blue depths: leveraging blue-green lasers and LED arrays to achieve megabit data rates over short underwater distances where bandwidth trumps range.

The Promise of Optical Communication

While acoustic systems dominate long-range underwater communication, optical methods offer orders of magnitude higher bandwidth for shorter distances. Blue-green light penetrates seawater better than other wavelengths, enabling data rates from 1 Mbps to over 100 Mbps—sufficient for real-time video streaming from ROVs, rapid AUV data offload during docking, and high-resolution sensor networks.

The fundamental advantage of optical communication stems from the vastly higher carrier frequency compared to acoustics: blue light at 470 nm corresponds to 640 THz, versus 25 kHz for typical acoustic carriers—a factor of 25 billion. This enormous bandwidth enables sophisticated modulation schemes and multi-gigahertz data rates, limited only by scattering and absorption in water.

Optical Propagation in Seawater

Light propagation underwater is governed by absorption (conversion to heat) and scattering (direction change). Both processes are wavelength-dependent, creating an optical "window" in the blue-green spectrum (450-550 nm) where attenuation is minimized.

Beer-Lambert Law

The intensity I of light at distance r from a source with initial intensity I₀ follows: I(r) = I₀ × e^(-c × r), where c is the attenuation coefficient in m⁻¹. In clear ocean water, c ≈ 0.05-0.15 m⁻¹; in turbid coastal water, c can exceed 1.0 m⁻¹.

Wavelength Selection

Different water types have different optimal wavelengths. Clear ocean water transmits blue light (450-480 nm) best, while coastal waters with higher chlorophyll and dissolved organic matter favor green light (510-540 nm). WIA-OCEAN-003 specifies three wavelength bands optimized for different environments.

Optical Sources: Lasers vs. LEDs

Two classes of optical sources dominate underwater communication: semiconductor lasers (laser diodes) and light-emitting diodes (LEDs). Each offers distinct advantages.

Laser Diode Systems

Blue-green laser diodes provide high optical power (10-1000 mW), narrow beam divergence (1-10 mrad), and fast modulation (up to 1 GHz). The narrow beam enables long range and spatial reuse but requires accurate pointing. Laser safety regulations limit power to prevent eye damage and require beam expansion or scanning to meet safety class limits.

LED-Based Systems

High-power LEDs (500 nm-530 nm) offer lower cost, simpler drive electronics, eye-safe operation without beam expansion, and wide field-of-view that eliminates pointing requirements. However, their broader emission spectrum and larger divergence angle (20-60°) limit range compared to lasers.

Optical Modulation Schemes

Optical communication systems use intensity modulation with direct detection (IM/DD), varying transmitted light intensity to encode data.

On-Off Keying (OOK)

The simplest scheme: transmit light for '1', no light for '0'. OOK achieves data rates up to the modulation bandwidth of the source (tens of MHz for LEDs, hundreds of MHz for lasers). Synchronization and clock recovery are straightforward, but OOK is susceptible to DC drift and baseline wander.

Pulse Position Modulation (PPM)

PPM encodes data in the timing of short pulses within symbol periods. L-PPM uses one of L time slots per symbol, encoding log₂(L) bits. PPM offers better power efficiency than OOK—crucial for battery-powered AUVs—but requires tighter synchronization and lower data rates for the same bandwidth.

Subcarrier Modulation

Modulating data onto an RF subcarrier (1-100 MHz) that then intensity-modulates the optical source enables use of mature RF modulation schemes (OFDM, QAM) and simplifies receiver electronics. The tradeoff is reduced bandwidth efficiency and increased complexity.

interface OpticalModemConfig {
  wavelength: number;           // nm (465, 520, or 530)
  opticalPower: number;         // mW
  modulationBandwidth: number;  // Hz
  modulationType: 'OOK' | 'PPM' | 'Subcarrier';
  ppmOrder?: number;            // For PPM: 4, 8, 16, etc.
}

class LEDOpticalModem {
  private config: OpticalModemConfig;
  private photodiode: PhotodiodeReceiver;

  constructor(config: OpticalModemConfig) {
    this.config = config;
    this.photodiode = new PhotodiodeReceiver(config.wavelength);
  }

  transmitOOK(dataBits: number[], bitRate: number): void {
    const bitPeriod = 1000000 / bitRate; // microseconds

    for (let i = 0; i < dataBits.length; i++) {
      if (dataBits[i] === 1) {
        this.ledOn(this.config.opticalPower);
      } else {
        this.ledOff();
      }
      this.delay(bitPeriod);
    }
  }

  transmitPPM(dataBits: number[], symbolRate: number): void {
    const order = this.config.ppmOrder || 4;
    const bitsPerSymbol = Math.log2(order);
    const slotDuration = 1000000 / (symbolRate * order); // μs

    for (let i = 0; i < dataBits.length; i += bitsPerSymbol) {
      const symbolBits = dataBits.slice(i, i + bitsPerSymbol);
      const pulsePosition = this.bitsToInt(symbolBits);

      // Transmit pulse in correct slot
      for (let slot = 0; slot < order; slot++) {
        if (slot === pulsePosition) {
          this.ledPulse(this.config.opticalPower, slotDuration);
        } else {
          this.ledOff();
          this.delay(slotDuration);
        }
      }
    }
  }

  receiveOOK(duration: number, bitRate: number): number[] {
    const samples = this.photodiode.sample(duration, bitRate * 4);
    const bits: number[] = [];
    const samplesPerBit = 4;

    for (let i = 0; i < samples.length; i += samplesPerBit) {
      const bitSamples = samples.slice(i, i + samplesPerBit);
      const avgIntensity = bitSamples.reduce((a, b) => a + b) / samplesPerBit;
      bits.push(avgIntensity > 0.5 ? 1 : 0);
    }

    return bits;
  }

  receivePPM(duration: number, symbolRate: number): number[] {
    const order = this.config.ppmOrder || 4;
    const bitsPerSymbol = Math.log2(order);
    const samplesPerSlot = 8;
    const samples = this.photodiode.sample(duration, symbolRate * order * samplesPerSlot);

    const bits: number[] = [];
    const samplesPerSymbol = order * samplesPerSlot;

    for (let i = 0; i < samples.length; i += samplesPerSymbol) {
      const symbolSamples = samples.slice(i, i + samplesPerSymbol);

      // Find which slot has the pulse
      let maxSlot = 0;
      let maxEnergy = 0;

      for (let slot = 0; slot < order; slot++) {
        const slotStart = slot * samplesPerSlot;
        const slotSamples = symbolSamples.slice(slotStart, slotStart + samplesPerSlot);
        const energy = slotSamples.reduce((sum, s) => sum + s * s, 0);

        if (energy > maxEnergy) {
          maxEnergy = energy;
          maxSlot = slot;
        }
      }

      // Convert slot position to bits
      const symbolBits = this.intToBits(maxSlot, bitsPerSymbol);
      bits.push(...symbolBits);
    }

    return bits;
  }

  private ledOff(): void {
    // Turn LED off
  }

  private ledPulse(power: number, durationUs: number): void {
    this.ledOn(power);
    this.delay(durationUs);
    this.ledOff();
  }

  private delay(microseconds: number): void {
    // Precise timing delay
  }

  private bitsToInt(bits: number[]): number {
    return bits.reduce((val, bit, idx) => val + bit * Math.pow(2, bits.length - 1 - idx), 0);
  }

  private intToBits(value: number, numBits: number): number[] {
    const bits: number[] = [];
    for (let i = numBits - 1; i >= 0; i--) {
      bits.push((value >> i) & 1);
    }
    return bits;
  }

Beam Pointing and Tracking

Narrow laser beams require accurate pointing to establish and maintain links. Two approaches dominate:

Mechanical Beam Steering

Gimbal-mounted lasers with servo control can scan for targets and track moving platforms. Inertial measurement units (IMUs) provide fast initial pointing, refined by optical tracking of beacons or retroreflectors. Tracking accuracy of ±1 mrad is achievable with modern servos and control algorithms.

Beam Expansion

Expanding the laser beam from 1 mrad to 10-20 mrad relaxes pointing requirements at the cost of reduced range and increased power consumption. Beam expanders use refractive or reflective optics to increase divergence while maintaining beam quality.

Eye Safety Requirements

Underwater optical systems must comply with laser safety standards (IEC 60825). WIA-OCEAN-003 mandates:
• LED systems: inherently eye-safe due to large emission area
• Laser systems <1 mW: Class 1, eye-safe without engineering controls
• Laser systems 1-500 mW: Class 3B, beam expansion or enclosure required
• Laser systems >500 mW: Class 4, enclosed optical path mandatory
Human-accessible areas require automatic shutdown when covers are removed.

KIOST Optical Communication Research

KIOST has pioneered cost-effective LED-based optical communication for AUV swarms. Their "Blue Cluster" system uses arrays of 470 nm LEDs achieving 15 Mbps at 30 meters range—sufficient for coordinated multi-AUV operations and rapid mission data sharing during underwater rendezvous.

Summary: Key Takeaways

  • Optical Advantage: Megabit to gigabit data rates, low power, but limited to short/medium range
  • Blue-Green Window: 450-550 nm wavelengths minimize absorption and scattering
  • Laser vs. LED: Lasers offer range/bandwidth; LEDs offer simplicity and eye safety
  • Modulation: OOK for simplicity, PPM for power efficiency, subcarrier for advanced modulation
  • Beam Pointing: Narrow laser beams require tracking; beam expansion trades range for pointing tolerance
  • WIA-OCEAN-003: Three classes (short LED, medium LED, long laser) for different applications
  • Safety Critical: Laser safety standards prevent eye injury
  • Korean Innovation: KIOST Blue Cluster enables AUV swarm coordination
