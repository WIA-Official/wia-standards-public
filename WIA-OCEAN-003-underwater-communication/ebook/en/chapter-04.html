RF and Electromagnetic Waves

Fighting the ocean's conductivity: where very low frequencies and magnetic induction enable specialized electromagnetic communication through water, sediment, and even rock.

The Electromagnetic Challenge

Seawater's electrical conductivity—approximately 4 S/m—causes electromagnetic waves to attenuate rapidly. The skin depth δ, where field strength drops to 37%, follows: δ = √(2 / (ωμσ)), where ω is angular frequency, μ is permeability, and σ is conductivity. At 1 MHz, δ ≈ 0.25 meters; practical RF communication is limited to centimeters.

However, very low frequency (VLF, 3-30 kHz) and extremely low frequency (ELF, 3-3000 Hz) electromagnetic waves can penetrate tens to hundreds of meters. WIA-OCEAN-003 defines specialized EM communication for applications where acoustic and optical methods fail: through-rock communication for cave exploration, sub-seabed sensors, and ice-penetrating links.

VLF and ELF Communication

VLF systems (3-30 kHz) achieve 10-100 meter range with data rates of 10-100 bps. Applications include submarine communication, seafloor penetration, and through-ice links in polar regions. ELF systems (<3 kHz) can penetrate hundreds of meters but provide only single-digit bit rates.

Magnetic Induction Communication

Magnetic induction (MI) uses near-field coupling between coil antennas, creating magnetic fields that penetrate seawater effectively. Unlike far-field EM propagation, MI signal strength follows 1/r³ dependence, limiting range to 1-50 meters but providing immunity to multipath and enabling communication through conductive barriers.

TypeScript Implementation: MI Transceiver

interface MIParameters {
  frequency: number;          // Hz (typically 1-100 kHz)
  coilRadius: number;         // meters
  coilTurns: number;          // number of wire turns
  transmitCurrent: number;    // Amperes
  receiverSensitivity: number; // V/T (volts per Tesla)
}

class MagneticInductionModem {
  private params: MIParameters;

  constructor(params: MIParameters) {
    this.params = params;
  }

  calculateMagneticField(distance: number): number {
    // Near-field magnetic field at distance r
    const mu0 = 4 * Math.PI * 1e-7; // H/m
    const R = this.params.coilRadius;
    const N = this.params.coilTurns;
    const I = this.params.transmitCurrent;

    // Simplified: on-axis field of circular coil
    const B = (mu0 * N * I * R * R) / (2 * Math.pow(R * R + distance * distance, 1.5));
    return B; // Tesla
  }

  calculateRange(minSNR: number, noisePower: number): number {
    // Find maximum range for given SNR requirement
    for (let r = 0.1; r < 100; r += 0.1) {
      const B = this.calculateMagneticField(r);
      const receivedVoltage = B * this.params.receiverSensitivity;
      const signalPower = receivedVoltage * receivedVoltage;
      const snr = 10 * Math.log10(signalPower / noisePower);

      if (snr < minSNR) {
        return r - 0.1;
      }
    }
    return 100;
  }

  modulateOOK(dataBits: number[], bitRate: number): number[] {
    // On-off keying: full current for '1', zero for '0'
    const samplesPerBit = 10;
    const samples: number[] = [];

    for (const bit of dataBits) {
      const current = bit === 1 ? this.params.transmitCurrent : 0;
      for (let i = 0; i < samplesPerBit; i++) {
        samples.push(current);
      }
    }
    return samples;
  }

  demodulateOOK(receivedSignal: number[], bitRate: number): number[] {
    const samplesPerBit = 10;
    const bits: number[] = [];

    for (let i = 0; i < receivedSignal.length; i += samplesPerBit) {
      const bitSamples = receivedSignal.slice(i, i + samplesPerBit);
      const avgSignal = bitSamples.reduce((a, b) => a + b) / bitSamples.length;
      bits.push(avgSignal > this.params.receiverSensitivity * 0.5 ? 1 : 0);
    }

    return bits;
  }
}

Surface Communication: WiFi and Cellular

At the water surface or in near-surface (<10 cm depth) environments, standard RF technologies become viable. WIA-OCEAN-003 defines interfaces for AUVs that surface for communication, deploying WiFi (2.4/5 GHz) or cellular (LTE, 5G) antennas above water while the vehicle hull remains submerged. Data rates reach Mbps to Gbps, enabling rapid mission data upload and software updates.

Antenna Design Challenges

Transitioning between underwater and air environments creates unique challenges:
• Impedance Mismatch: Seawater (377 Ω) vs. air (377 Ω) requires careful matching
• Corrosion: Saltwater exposure requires corrosion-resistant materials
• Wave Action: Surface antennas must survive mechanical stress
• Deployable Designs: Retractable/extendable antennas reduce drag when submerged

KIOST EM Research

KIOST has developed magnetic induction systems for communication through sea ice and sediment in Arctic and Antarctic research. Their MI transceivers enable data collection from sensors buried in seafloor sediment without cable penetrations, reducing environmental impact and improving reliability.

Summary: Key Takeaways

  • Severe Attenuation: Seawater conductivity kills high-frequency RF; only VLF/ELF propagate significantly
  • VLF/ELF Trade-off: Lower frequency = greater range but lower data rate and larger antennas
  • Magnetic Induction: Near-field MI penetrates barriers; limited range but robust
  • Surface RF: WiFi/cellular viable when antenna breaks surface; Mbps to Gbps rates
  • Specialized Applications: Through-rock, through-ice, sub-seabed where other methods fail
  • Korean Innovation: KIOST MI systems enable Arctic/Antarctic research
