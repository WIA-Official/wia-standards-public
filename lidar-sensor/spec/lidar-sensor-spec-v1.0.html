WIA-SEMI-014: LiDAR Sensor Standard Specification v1.0

Document Version: 1.0
Release Date: 2025-01-15
Status: Active Standard
Organization: WIA (World Industry Association)

1. Scope and Purpose

This specification defines standards for automotive LiDAR (Light Detection and Ranging) sensors used in Advanced Driver Assistance Systems (ADAS) and autonomous vehicles. It covers performance requirements, safety standards, interface protocols, environmental specifications, and testing methodologies for LiDAR systems operating in the 905nm and 1550nm wavelength bands.

1.1 Application Domains

  • Automotive ADAS (SAE Level 1-2)
  • Conditional Automation (SAE Level 3)
  • High/Full Automation (SAE Level 4-5)
  • Commercial Vehicles and Trucking
  • Robotaxi and Mobility Services

3. Safety and Regulatory Compliance

3.1 Laser Safety

Requirement: All automotive LiDAR systems SHALL comply with IEC 60825-1:2014 laser safety standard.

Wavelength Required Classification Maximum Power
905nm (Near-IR) Class 1 or Class 1M Per IEC 60825-1 Table 6
1550nm (SWIR) Class 1 Per IEC 60825-1 Table 7

3.2 Functional Safety (ISO 26262)

LiDAR systems for ADAS and autonomous driving SHALL meet ISO 26262 functional safety requirements:

  • ASIL-B: Minimum for driver assistance features (ACC, AEB)
  • ASIL-C: Recommended for conditional automation (Level 3)
  • ASIL-D: Required for high/full automation (Level 4-5) safety-critical functions

3.3 Automotive Qualification (AEC-Q100)

Electronic components SHALL be qualified per AEC-Q100 automotive IC standard:

  • Temperature cycling: 1000 cycles, -40°C to +150°C junction temp
  • High Temperature Operating Life (HTOL): 1000 hours @ 150°C
  • Environmental stress testing per JEDEC standards

5.2 Data Protocols

  • Ethernet (UDP): Real-time point cloud streaming, 10-100 Mbps bandwidth
  • Ethernet (TCP): Configuration, firmware updates, diagnostics
  • ROS/ROS2: sensor_msgs/PointCloud2 message format (recommended)
  • PCL Format: Point Cloud Library binary/ASCII format support

6. Testing and Validation

6.1 Acceptance Testing

All production units SHALL undergo the following tests:

  1. Range Calibration: Verify accuracy against calibrated targets at 10m, 50m, 100m, 200m
  2. Angular Calibration: Verify FOV and angular resolution meet specifications
  3. Intensity Calibration: Normalize intensity values across reflectivity range (10%-90%)
  4. Functional Test: Verify all interfaces, protocols, and modes operate correctly
  5. Environmental Pre-screening: Temperature soak test at -40°C and +85°C

6.2 Type Approval Testing

Representative samples SHALL undergo comprehensive validation:

  • Full environmental qualification per AEC-Q100
  • EMC testing per ISO 11452 (immunity) and CISPR 25 (emissions)
  • Laser safety certification per IEC 60825-1
  • Functional safety assessment per ISO 26262
  • Weather performance validation per standards listed in Section 4.2

7. Documentation Requirements

LiDAR manufacturers SHALL provide:

  1. Datasheet: Complete performance specifications, dimensions, interfaces
  2. Integration Guide: Mechanical mounting, electrical connections, calibration procedures
  3. Software API Documentation: Point cloud format, configuration parameters, diagnostic codes
  4. Safety Manual: Laser safety procedures, installation guidelines, end-of-life disposal
  5. Compliance Certificates: IEC 60825-1, ISO 26262, AEC-Q100, IP rating certificates
  6. Reliability Data: MTTF (Mean Time To Failure), failure modes, warranty terms
