  • 확장성: 소규모 프로젝트부터 엔터프라이즈 시스템까지 지원

주요 구성 요소

1. 데이터 형식 표준

WIA-ART-006-art-authentication는 다양한 파일 형식을 지원하며, 각 형식은 특정 사용 사례에 최적화되어 있습니다. 올바른 형식 선택은 워크플로우 효율성과 품질에 직접적인 영향을 미칩니다.

2. API 통합

WIA-ART-006-art-authentication SDK는 TypeScript/JavaScript로 작성되었으며, 모든 주요 플랫폼과 프레임워크를 지원합니다. 다음은 기본 사용 예제입니다: 

import { ART006artauthenticationSDK } from '@wia/wia-art-006-art-authentication';

// SDK 초기화
const sdk = new ART006artauthenticationSDK({
  apiKey: process.env.WIA_API_KEY,
  environment: 'production'
});

// 콘텐츠 생성
async function createContent() {
  try {
    const metadata = {
      title: "인증 저작권 예제",
      creator: "WIA Developer",
      license: "CC BY-SA 4.0",
      timestamp: new Date().toISOString()
    };
    
    const result = await sdk.create(metadata);
    console.log('생성 성공:', result.id);
    
    // 검증
    const validation = await sdk.validate(result);
    if (validation.isValid) {
      console.log('검증 통과');
      return result;
    } else {
      console.error('검증 실패:', validation.errors);
    }
  } catch (error) {
    console.error('오류:', error.message);
    throw error;
  }
}

createContent().then(console.log).catch(console.error);

COA (Certificate of Authenticity) 법적 효력

진품 증명서(COA)는 예술품 진위를 보증하는 법적 문서로, 작가 또는 권한 있는 기관이 발급합니다.

유효한 COA 필수 요소

  • 작품 정보: 제목, 제작 연도, 재료, 크기
  • 작가 정보: 이름, 서명, 날짜
  • 발급 기관: 갤러리, 경매사, 작가
  • 고유 번호: 시리얼 넘버 또는 등록 번호
  • 사진: 작품 전체 및 세부 사진

유럽 추급권(Droit de Suite)은 미술품 재판매 시 작가에게 판매가 일정 비율 지급. NFT 스마트 컨트랙트로 자동 구현 가능.

    }

한국 저작권법과 예술품 인증

한국 저작권법 제2조는 저작물을 "인간의 사상 또는 감정을 표현한 창작물"로 정의하며, 예술 작품은 당연히 저작권 보호를 받습니다.

구분 저작권 소유권

디지털 저작권 관리 (DRM) 통합

class ArtworkDRM {
    constructor(nftContract) {
        this.nftContract = nftContract;
        this.licenses = new Map();
    }

    async grantLicense(tokenId, licensee, terms) {
        /**
         * NFT 소유자가 제3자에게 사용 라이선스 부여
         * - 전시 라이선스
         * - 복제 라이선스
         * - 상업적 사용 라이선스
         */
        const owner = await this.nftContract.ownerOf(tokenId);
        const license = {
            licensor: owner,
            licensee: licensee,
            tokenId: tokenId,
            type: terms.type,  // 'exhibition', 'reproduction', 'commercial'
            duration: {
                start: terms.start,
                end: terms.end
            },
            territory: terms.territory,  // 'worldwide', 'korea', 'asia'
            exclusivity: terms.exclusive,
            royalty: {
                percentage: terms.royaltyPercentage,
                recipient: owner
            },
            restrictions: terms.restrictions,
            issued: new Date().toISOString()
        };

        this.licenses.set(`${tokenId}-${licensee}`, license);

        // 블록체인에 라이선스 기록
        await this.recordLicenseOnChain(license);

        return license;
    }

    async verifyLicense(tokenId, user, usageType) {
        const licenseKey = `${tokenId}-${user}`;
        const license = this.licenses.get(licenseKey);

        if (!license) {
            return { authorized: false, reason: 'No license found' };
        }

        const now = new Date();
        const start = new Date(license.duration.start);
        const end = new Date(license.duration.end);

        if (now < start || now > end) {
            return { authorized: false, reason: 'License expired or not yet valid' };
        }

        if (license.type !== usageType && license.type !== 'all-rights') {
            return { authorized: false, reason: 'Usage type not authorized' };
        }

        return { 
            authorized: true, 
            license: license,
            royalty_due: this.calculateRoyalty(license)
        };
    }

    calculateRoyalty(license) {
        // 라이선스 사용에 따른 로열티 계산
        return {
            percentage: license.royalty.percentage,
            recipient: license.royalty.recipient,
            payment_method: 'automatic_smart_contract'
        };
    }
}

// 사용 예제
const drm = new ArtworkDRM(artNFTContract);

// 전시 라이선스 부여
const exhibitionLicense = await drm.grantLicense(
    tokenId,
    'museum@example.com',
    {
        type: 'exhibition',
        start: '2025-06-01',
        end: '2025-08-31',
        territory: 'korea',
        exclusive: false,
        royaltyPercentage: 5,
        restrictions: ['no-reproduction', 'no-commercial-use']
    }
);

// 라이선스 검증
const verification = await drm.verifyLicense(
    tokenId,
    'museum@example.com',
    'exhibition'
);

if (verification.authorized) {
    console.log('전시 승인됨');
    console.log(`로열티: ${verification.royalty_due.percentage}%`);
} else {
    console.log(`거부: ${verification.reason}`);
}

CC 라이선스와 NFT

Creative Commons 라이선스를 NFT에 적용하여 개방형 예술 생태계 구축.

CC 라이선스 종류

  • CC BY: 저작자 표시 (출처만 밝히면 자유 사용)
  • CC BY-SA: 저작자 표시 + 동일 조건 변경 허락
  • CC BY-NC: 저작자 표시 + 비영리
  • CC BY-NC-SA: 저작자 표시 + 비영리 + 동일 조건
  • CC BY-ND: 저작자 표시 + 변경 금지
  • CC BY-NC-ND: 저작자 표시 + 비영리 + 변경 금지
