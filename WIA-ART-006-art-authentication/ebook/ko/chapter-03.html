개요

ISCC, C2PA 콘텐츠 인증 표준

이 장에서는 인증 데이터 형식의 기술적 세부사항, 실무 구현 방법, 그리고 WIA-ART-006-art-authentication 표준과의 통합 방안을 상세히 다룹니다. 예술 인증 분야는 빠르게 발전하고 있으며, 표준화를 통해 상호운용성과 혁신을 촉진할 수 있습니다.

기술적 배경

인증 데이터 형식는 현대 디지털 창작 환경의 핵심 요소입니다. 최신 기술 동향을 이해하고 효과적으로 활용하기 위해서는 기본 원리부터 고급 응용까지 체계적으로 학습해야 합니다.

핵심 특징

  • 표준화: WIA-ART-006-art-authentication 표준을 준수하여 플랫폼 간 상호운용성 보장
  • 확장성: 소규모 프로젝트부터 엔터프라이즈 시스템까지 지원
  • 개방성: 오픈 소스 도구와 API를 통한 생태계 구축
  • 미래 지향성: 신흥 기술과의 통합을 고려한 설계

주요 구성 요소

1. 데이터 형식 표준

WIA-ART-006-art-authentication는 다양한 파일 형식을 지원하며, 각 형식은 특정 사용 사례에 최적화되어 있습니다. 올바른 형식 선택은 워크플로우 효율성과 품질에 직접적인 영향을 미칩니다.

형식 용도 장점 단점
형식 A 프로덕션 환경 고품질, 표준 준수 파일 크기가 큼
형식 B 웹 배포 압축률 우수 일부 기능 제한
형식 C 아카이빙 무손실, 메타데이터 풍부 처리 속도 느림

ISCC (International Standard Content Code)

1. ISCC 개요 및 구조

ISCC는 ISO 24138로 표준화된 콘텐츠 식별 코드로, 텍스트, 이미지, 비디오, 오디오 등 모든 유형의 디지털 콘텐츠에 고유 식별자를 부여합니다. 예술 작품 인증에서 ISCC는 작품의 내용 기반 지문(content fingerprint)을 생성하여 유사 작품 검색 및 위작 탐지에 활용됩니다.

ISCC 코드 구성

  • Meta-Code: 제목, 작가 등 메타데이터 기반 (64비트)
  • Content-Code-Text: 텍스트 콘텐츠 유사성 (64비트)
  • Content-Code-Image: 이미지 perceptual hash (64비트)
  • Data-Code: 원본 파일 암호화 해시 (64비트)
  • Instance-Code: 특정 파일 버전 식별 (64비트)

예제: ISCC:KACYPXW445FTYNJ3-CEBZQXY
Artworks can have up to 320 bits of combined codes

2. 예술 작품용 ISCC 생성 구현

from iscc_sdk import iscc_from_object
import hashlib
from PIL import Image
import numpy as np

class ArtworkISCC:
    def __init__(self):
        self.version = 'v1.0'

    def generate_iscc(self, artwork_file, metadata):
        """
        예술 작품의 ISCC 코드 생성
        - Meta-Code: 작품명, 작가명
        - Content-Code-Image: 이미지 perceptual hash
        - Data-Code: 파일 SHA-256
        - Instance-Code: 정확한 파일 버전
        """
        # 1. Meta-Code 생성 (메타데이터 기반)
        meta_string = f"{metadata['title']}-{metadata['artist']}"
        meta_code = self.generate_meta_code(meta_string)

        # 2. Content-Code-Image 생성 (perceptual hash)
        image = Image.open(artwork_file)
        content_code = self.generate_image_content_code(image)

        # 3. Data-Code 생성 (파일 해시)
        with open(artwork_file, 'rb') as f:
            file_data = f.read()
        data_code = self.generate_data_code(file_data)

        # 4. Instance-Code 생성 (정확한 파일 인스턴스)
        instance_code = self.generate_instance_code(file_data)

        # ISCC 코드 조합
        iscc_code = f"ISCC:{meta_code}-{content_code}-{data_code}-{instance_code}"

        return {
            'iscc': iscc_code,
            'meta_code': meta_code,
            'content_code': content_code,
            'data_code': data_code,
            'instance_code': instance_code,
            'metadata': metadata
        }

    def generate_meta_code(self, meta_string):
        """메타데이터 기반 코드 (정규화 + 해시)"""
        # 소문자 변환, 공백 제거, 유니코드 정규화
        normalized = meta_string.lower().replace(' ', '')
        hash_bytes = hashlib.sha256(normalized.encode('utf-8')).digest()
        # 첫 8바이트를 base32 인코딩
        return self.base32_encode(hash_bytes[:8])

    def generate_image_content_code(self, image):
        """이미지 perceptual hash (DCT 기반)"""
        # 그레이스케일 변환 및 리사이징
        img_gray = image.convert('L')
        img_resized = img_gray.resize((32, 32), Image.LANCZOS)

        # DCT 계산 (간소화 버전)
        pixels = np.array(img_resized, dtype=float)
        dct = self.simple_dct(pixels)

        # 저주파 8x8 영역 추출
        dct_low = dct[:8, :8].flatten()
        median = np.median(dct_low)

        # 이진화
        hash_bits = (dct_low > median).astype(int)
        hash_bytes = np.packbits(hash_bits).tobytes()

        return self.base32_encode(hash_bytes)

    def generate_data_code(self, file_data):
        """파일 데이터 해시"""
        hash_bytes = hashlib.sha256(file_data).digest()
        return self.base32_encode(hash_bytes[:8])

    def generate_instance_code(self, file_data):
        """파일 인스턴스 코드 (BLAKE3)"""
        import blake3
        hash_bytes = blake3.blake3(file_data).digest()
        return self.base32_encode(hash_bytes[:8])

    @staticmethod
    def simple_dct(matrix):
        """간단한 2D DCT 구현"""
        from scipy.fftpack import dct
        return dct(dct(matrix.T, norm='ortho').T, norm='ortho')

    @staticmethod
    def base32_encode(data):
        """Base32 인코딩 (ISCC 표준)"""
        import base64
        return base64.b32encode(data).decode('utf-8').rstrip('=')

    def compare_iscc(self, iscc1, iscc2):
        """두 ISCC 코드의 유사도 계산"""
        # Meta-Code 비교 (완전 일치)
        meta_match = iscc1['meta_code'] == iscc2['meta_code']

        # Content-Code 해밍 거리
        content_similarity = self.hamming_similarity(
            iscc1['content_code'],
            iscc2['content_code']
        )

        # Data-Code 일치 (동일 파일)
        data_match = iscc1['data_code'] == iscc2['data_code']

        return {
            'meta_match': meta_match,
            'content_similarity': content_similarity,
            'data_match': data_match,
            'is_similar': content_similarity > 0.85,
            'is_identical': data_match
        }

    @staticmethod
    def hamming_similarity(code1, code2):
        """두 코드 간 해밍 유사도 (0-1)"""
        if len(code1) != len(code2):
            return 0.0

        matching = sum(c1 == c2 for c1, c2 in zip(code1, code2))
        return matching / len(code1)


# 사용 예제
iscc_gen = ArtworkISCC()

artwork_metadata = {
    'title': '추상 풍경',
    'artist': '박예술',
    'year': 2025
}

# ISCC 생성
iscc_result = iscc_gen.generate_iscc('artwork.jpg', artwork_metadata)
print(f"ISCC 코드: {iscc_result['iscc']}")

# 유사 작품 비교
suspected_copy_iscc = iscc_gen.generate_iscc('suspected_copy.jpg', artwork_metadata)
comparison = iscc_gen.compare_iscc(iscc_result, suspected_copy_iscc)

if comparison['is_identical']:
    print("동일한 파일입니다 (복사본)")
elif comparison['is_similar']:
    print(f"유사 작품 (유사도: {comparison['content_similarity']*100:.1f}%)")
else:
    print("다른 작품입니다")

C2PA Manifest 상세 구조

1. C2PA 데이터 형식 명세

C2PA (Coalition for Content Provenance and Authenticity) Manifest는 디지털 콘텐츠의 출처, 편집 이력, 권리 정보를 JUMBF (JPEG Universal Metadata Box Format) 형식으로 이미지에 삽입합니다.

Manifest 구성 요소 설명 필수 여부
Claim 콘텐츠에 대한 주장 (누가, 언제, 무엇을) 필수
Assertions 구체적 정보 (액션, 해시, 메타데이터) 필수
Signature 암호학적 서명 (EdDSA, RSA) 필수
Credentials 서명자 신원 증명 (X.509, W3C VC) 선택
Ingredient Claims 사용된 소스 콘텐츠 정보 선택

1. ISCC ↔ C2PA ↔ NFT 통합

class UnifiedArtAuthentication:
    """ISCC, C2PA, NFT를 통합한 예술품 인증"""

    def __init__(self):
        self.iscc_generator = ArtworkISCC()
        self.c2pa_manifest = C2PAManifest()
        self.nft_registry = NFTRegistry()

    async def create_comprehensive_certificate(self, artwork_file, metadata):
        """
        통합 인증서 생성
        - ISCC: 콘텐츠 기반 식별
        - C2PA: 편집 이력 및 메타데이터
        - NFT: 블록체인 소유권 증명
        """
        # 1. ISCC 생성
        iscc_data = self.iscc_generator.generate_iscc(artwork_file, metadata)

        # 2. NFT 민팅
        nft_result = await self.nft_registry.mint_artwork({
            'title': metadata['title'],
            'artist': metadata['artist'],
            'iscc': iscc_data['iscc'],
            'image_url': f"ipfs://{upload_to_ipfs(artwork_file)}"
        })

        # 3. C2PA Manifest 생성 (ISCC + NFT 정보 포함)
        c2pa_manifest = self.c2pa_manifest.create({
            'artwork': metadata,
            'iscc': iscc_data['iscc'],
            'nft': {
                'contract': nft_result['contract'],
                'token_id': nft_result['token_id'],
                'blockchain': 'Polygon',
                'tx_hash': nft_result['tx_hash']
            },
            'creator_signature': await self.sign_with_did(metadata['artist_did'])
        })

        # 4. C2PA Manifest를 이미지에 삽입
        authenticated_image = self.c2pa_manifest.embed(
            artwork_file,
            c2pa_manifest
        )

        return {
            'iscc': iscc_data['iscc'],
            'nft_token_id': nft_result['token_id'],
            'nft_contract': nft_result['contract'],
            'c2pa_manifest_url': f"jumbf://c2pa/manifest/{iscc_data['iscc']}",
            'authenticated_file': authenticated_image,
            'verification_url': f"https://wia-art.org/verify/{iscc_data['iscc']}"
        }

    async def verify_artwork(self, image_file):
        """
        통합 인증서 검증
        - C2PA 서명 검증
        - ISCC 일치 확인
        - NFT 소유권 조회
        """
        # C2PA Manifest 추출
        manifest = self.c2pa_manifest.read(image_file)

        if not manifest:
            return {'valid': False, 'reason': 'No C2PA manifest'}

        # 서명 검증
        signature_valid = await self.c2pa_manifest.verify_signature(manifest)

        # ISCC 재계산 및 비교
        current_iscc = self.iscc_generator.generate_iscc(
            image_file,
            manifest['metadata']
        )
        iscc_match = current_iscc['iscc'] == manifest['iscc']

        # NFT 소유권 확인
        nft_info = await self.nft_registry.get_token_info(
            manifest['nft']['contract'],
            manifest['nft']['token_id']
        )

        return {
            'valid': signature_valid and iscc_match,
            'signature_valid': signature_valid,
            'iscc_match': iscc_match,
            'current_owner': nft_info['owner'],
            'creator': nft_info['creator'],
            'creation_date': nft_info['mint_date'],
            'provenance_events': manifest['provenance'],
            'verification_timestamp': datetime.now().isoformat()
        }
