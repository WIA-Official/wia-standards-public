Chapter 3: Decision Making and Control

3.2 Path Planning Algorithms

Path planning algorithms generate feasible routes through the environment while respecting vehicle kinematic constraints, traffic rules, and safety margins.

Planning Approaches

  • Graph-Based Planning: A*, Dijkstra, D* operate on discretized representations
  • Sampling-Based Planning: RRT and PRM explore continuous configuration spaces
  • Optimization-Based Planning: Model Predictive Control formulates planning as constrained optimization
// Model Predictive Control Path Planning
class MPCPlanner {
    Trajectory plan(State current, Goal goal, Obstacles obstacles) {
        problem = OptimizationProblem();
        N = 50;  // Planning horizon
        dt = 0.1;  // Time step

        states = problem.addVariables("x", N+1, state_dim);
        controls = problem.addVariables("u", N, control_dim);

        // Objective: minimize distance to goal + control effort
        cost = 0;
        for (k = 0; k < N; k++) {
            cost += (states[k] - goal).norm() * Q;
            cost += controls[k].norm() * R;
        }
        problem.minimize(cost);

        // Dynamics and obstacle constraints
        for (k = 0; k < N; k++) {
            problem.addConstraint(
                states[k+1] == vehicleDynamics(states[k], controls[k], dt)
            );
        }
        return Trajectory(problem.solve());
    }
}

3.3 Behavior Prediction

Predicting future behavior of other road users is essential for safe autonomous driving. WIA-AUTO requires prediction systems to generate multi-modal probability distributions over future trajectories.

Prediction Requirements

  • Prediction horizon: minimum 5 seconds for highway, 3 seconds for urban
  • Update rate: minimum 10 Hz
  • Multi-modal output: at least 3 trajectory hypotheses with probabilities
  • Uncertainty quantification: confidence bounds on predicted positions

3.7 Minimal Risk Condition

When the autonomous driving system can no longer operate safely, it must transition to a minimal risk condition:

  • Controlled Stop: Gradual deceleration to standstill in current lane
  • Roadside Pullover: Navigate to shoulder or safe parking area
  • Maintain Lane: Continue at reduced speed while alerting driver
  • Emergency Stop: Maximum braking when collision is imminent
