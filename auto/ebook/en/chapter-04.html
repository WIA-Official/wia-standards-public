4.2 Communication Technologies

DSRC (Dedicated Short-Range Communications)

  • Frequency: 5.850-5.925 GHz (75 MHz bandwidth)
  • Range: Up to 1000m (typical 300-500m)
  • Latency: < 2ms for direct communication
  • Data Rate: 6-27 Mbps

C-V2X (Cellular Vehicle-to-Everything)

  • LTE-V2X (PC5): Direct communication, 5.9 GHz band
  • LTE-V2X (Uu): Network-based communication
  • 5G NR-V2X: Enhanced reliability, lower latency

4.3 Message Types and Protocols

Basic Safety Message Structure
==============================
BSM {
    header {
        messageId: 20 (BSM)
        generationTime: timestamp_ms
    }
    coreData {
        tempId: 4-byte random ID
        position {
            latitude: degrees * 1e7
            longitude: degrees * 1e7
            elevation: meters * 10
        }
        speed: m/s * 50
        heading: degrees * 80
        acceleration {
            longitudinal: m/s^2 * 100
            lateral: m/s^2 * 100
        }
        brakes: brake system status
        size { width, length }
    }
}

4.5 V2I Applications

Vehicle-to-Infrastructure communication connects vehicles with roadside equipment and traffic management systems for traffic signal optimization, road hazard notification, and parking guidance.

Security Requirements

  • Message Authentication: All messages signed using ECDSA
  • Certificate Management: Short-term pseudonym certificates
  • Misbehavior Detection: Systems to identify malicious nodes
  • Privacy Protection: Temporary IDs prevent tracking
