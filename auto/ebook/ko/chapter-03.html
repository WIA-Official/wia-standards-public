3.2 경로 계획 알고리즘

경로 계획 알고리즘은 차량 운동학적 제약, 교통 규칙 및 안전 마진을 준수하면서 환경을 통한 실행 가능한 경로를 생성합니다.

계획 접근법

  • 그래프 기반 계획: A*, Dijkstra, D*는 이산화된 표현에서 작동
  • 샘플링 기반 계획: RRT와 PRM은 연속 구성 공간 탐색
  • 최적화 기반 계획: 모델 예측 제어는 계획을 제약 최적화로 공식화
// 모델 예측 제어 경로 계획
class MPCPlanner {
    Trajectory plan(State current, Goal goal, Obstacles obstacles) {
        problem = OptimizationProblem();
        N = 50;  // 계획 지평
        dt = 0.1;  // 시간 스텝

        states = problem.addVariables("x", N+1, state_dim);
        controls = problem.addVariables("u", N, control_dim);

        // 목표: 목표까지 거리 + 제어 노력 최소화
        cost = 0;
        for (k = 0; k < N; k++) {
            cost += (states[k] - goal).norm() * Q;
            cost += controls[k].norm() * R;
        }
        problem.minimize(cost);

        // 동역학 및 장애물 제약
        for (k = 0; k < N; k++) {
            problem.addConstraint(
                states[k+1] == vehicleDynamics(states[k], controls[k], dt)
            );
        }
        return Trajectory(problem.solve());
    }
}

3.3 행동 예측

다른 도로 사용자의 미래 행동을 예측하는 것은 안전한 자율주행에 필수적입니다. WIA-AUTO는 예측 시스템이 미래 궤적에 대한 다중 모달 확률 분포를 생성하도록 요구합니다.

예측 요구사항

  • 예측 지평: 고속도로 최소 5초, 도시 3초
  • 업데이트 속도: 최소 10 Hz
  • 다중 모달 출력: 확률이 있는 최소 3개 궤적 가설
  • 불확실성 정량화: 예측 위치에 대한 신뢰 구간

3.7 최소 위험 조건

자율주행 시스템이 더 이상 안전하게 작동할 수 없을 때, 최소 위험 조건으로 전환해야 합니다:

  • 제어된 정지: 현재 차선에서 점진적 감속하여 정지
  • 갓길 정차: 갓길 또는 안전한 주차 구역으로 이동
  • 차선 유지: 감속하면서 운전자에게 경고
  • 긴급 정지: 충돌이 임박했을 때 최대 제동
