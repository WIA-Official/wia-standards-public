제4장: V2X 통신 표준

V2X(Vehicle-to-Everything) 통신은 자율주행차가 탑재 센서를 넘어 인지를 확장하여 다른 차량, 인프라, 보행자 및 네트워크 서비스와 정보를 공유할 수 있게 합니다.

4.2 통신 기술

DSRC (전용 단거리 통신)

  • 주파수: 5.850-5.925 GHz (75 MHz 대역폭)
  • 범위: 최대 1000m (일반적으로 300-500m)
  • 지연시간: 직접 통신 2ms 미만
  • 데이터 속도: 6-27 Mbps

C-V2X (셀룰러 V2X)

  • LTE-V2X (PC5): 직접 통신, 5.9 GHz 대역
  • LTE-V2X (Uu): 네트워크 기반 통신
  • 5G NR-V2X: 향상된 신뢰성, 낮은 지연시간

    header {
        messageId: 20 (BSM)
        generationTime: timestamp_ms

    coreData {
        tempId: 4바이트 랜덤 ID
        position {
            latitude: degrees * 1e7
            longitude: degrees * 1e7
            elevation: meters * 10
        }
        speed: m/s * 50
        heading: degrees * 80

            longitudinal: m/s^2 * 100
            lateral: m/s^2 * 100
        }

        size { width, length }
    }
}

4.5 V2I 애플리케이션

V2I(Vehicle-to-Infrastructure) 통신은 차량을 노변 장비 및 교통 관리 시스템과 연결하여 신호 최적화, 도로 위험 알림 및 주차 안내를 제공합니다.

V2I 메시지 흐름: 교차로 접근
============================
차량                          RSU/신호 제어기
   │                                      │
   │◄────── MAP (교차로 기하) ────────────│
   │                                      │
   │◄────── SPaT (신호 타이밍) ───────────│
   │                                      │
   │─────── BSM (위치, 속도) ─────────────►│
   │                                      │
   │◄────── 신호 우선순위 응답 ───────────│
   │                                      │
   │─────── 교차로 통과 ─────────────────►│
