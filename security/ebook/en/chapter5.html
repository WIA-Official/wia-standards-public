Chapter 5: Zero Trust Architecture

5.1 Introduction to Zero Trust

Zero Trust is a security paradigm that eliminates implicit trust and requires continuous verification of every user, device, and network flow. The core principle is "never trust, always verify" - no user or system is trusted by default, regardless of whether they are inside or outside the network perimeter.

Core Principle: Traditional perimeter-based security assumes everything inside the network is trustworthy. Zero Trust assumes breach and verifies each request as though it originates from an untrusted network, regardless of source.

Zero Trust Principles

Principle Description Implementation Technologies
Verify Explicitly Always authenticate and authorize based on all available data points Multi-factor authentication, Device posture, Location MFA, MDM, Context-aware access
Least Privilege Limit user access with Just-In-Time and Just-Enough-Access RBAC, Time-bound access, Risk-based policies PAM, IAM, PIM
Assume Breach Minimize blast radius and segment access Micro-segmentation, Encryption, Analytics SASE, ZTNA, EDR
Device Trust Verify device health before granting access Device compliance, Posture assessment MDM, UEM, NAC
Continuous Monitoring Real-time threat detection and response Behavioral analytics, Anomaly detection SIEM, UEBA, SOAR

5.2 Zero Trust Maturity Model

Organizations progress through different maturity levels when implementing Zero Trust architecture.

Zero Trust Maturity Levels

Level Stage Characteristics Key Capabilities
Level 1 Traditional Perimeter-based security, Static policies Firewall, VPN, Basic authentication
Level 2 Initial Some identity awareness, Manual processes MFA, Basic RBAC, Network segmentation
Level 3 Advanced Context-aware access, Automated response ZTNA, Micro-segmentation, Continuous auth
Level 4 Optimal Fully automated, AI-driven, Adaptive ML-based policies, Self-healing, Zero standing privileges

Example 1: Zero Trust Policy Engine (Python)

Python - Zero Trust Policy Evaluation Engine
from typing import Dict, List, Optional
from datetime import datetime, time
import ipaddress
from enum import Enum

class AccessDecision(Enum):
    ALLOW = "allow"
    DENY = "deny"
    STEP_UP_AUTH = "step_up_auth"
    MFA_REQUIRED = "mfa_required"

class ZeroTrustPolicyEngine:
    """
    WIA Zero Trust Policy Evaluation Engine
    Evaluates access requests against Zero Trust policies
    """

    def __init__(self):
        self.policies = []
        self.device_trust_service = DeviceTrustService()
        self.risk_scorer = RiskScorer()

    def evaluate_access(self, access_request: Dict) -> Dict:
        """
        Evaluate access request against all applicable policies

        Returns decision with reasoning and required actions
        """
        # Extract request context
        user_id = access_request['userId']
        resource = access_request['resource']
        device_id = access_request.get('deviceId')
        ip_address = access_request.get('ipAddress')

        # Find applicable policies (sorted by priority)
        applicable_policies = self._get_applicable_policies(
            user_id, resource
        )

        if not applicable_policies:
            return {
                "decision": AccessDecision.DENY.value,
                "reason": "No applicable policy found",
                "requiresAction": []
            }

        # Evaluate each condition
        evaluation_results = []

        for policy in applicable_policies:
            result = self._evaluate_policy(policy, access_request)
            evaluation_results.append(result)

            # First matching policy wins (priority-based)
            if result['matched']:
                return self._build_decision(policy, result)

        # No policy matched - default deny
        return {
            "decision": AccessDecision.DENY.value,
            "reason": "No policy matched request conditions",
            "requiresAction": []
        }

    def _evaluate_policy(self, policy: Dict, request: Dict) -> Dict:
        """Evaluate all conditions in a policy"""
        conditions = policy.get('conditions', {})
        results = {
            'matched': True,
            'failedConditions': []
        }

        # Device compliance check
        if 'deviceCompliance' in conditions:
            device_compliant = self._check_device_compliance(
                request.get('deviceId'),
                conditions['deviceCompliance']
            )
            if not device_compliant:
                results['matched'] = False
                results['failedConditions'].append('device_compliance')

        # Network conditions check
        if 'networkConditions' in conditions:
            network_allowed = self._check_network_conditions(
                request.get('ipAddress'),
                conditions['networkConditions']
            )
            if not network_allowed:
                results['matched'] = False
                results['failedConditions'].append('network_conditions')

        # Contextual factors (time, location, risk)
        if 'contextualFactors' in conditions:
            context_valid = self._check_contextual_factors(
                request,
                conditions['contextualFactors']
            )
            if not context_valid:
                results['matched'] = False
                results['failedConditions'].append('contextual_factors')

        # Authentication strength
        if 'authenticationStrength' in conditions:
            auth_sufficient = self._check_authentication_strength(
                request,
                conditions['authenticationStrength']
            )
            if not auth_sufficient:
                results['matched'] = False
                results['failedConditions'].append('authentication_strength')

        return results

    def _check_device_compliance(self, device_id: str,
                                 requirements: Dict) -> bool:
        """Verify device meets compliance requirements"""
        if not device_id:
            return not requirements.get('required', False)

        device_info = self.device_trust_service.get_device_info(device_id)

        # Check encryption
        if requirements.get('encryptionRequired'):
            if not device_info.get('encrypted'):
                return False

        # Check antivirus
        if requirements.get('antivirusRequired'):
            if not device_info.get('antivirusEnabled'):
                return False

        # Check OS version
        if 'os' in requirements:
            if device_info.get('os') not in requirements['os']:
                return False

        return True

    def _check_network_conditions(self, ip_address: str,
                                  conditions: Dict) -> bool:
        """Validate network-based conditions"""
        if not ip_address:
            return False

        # Check allowed IP ranges
        if 'allowedIpRanges' in conditions:
            allowed = False
            for ip_range in conditions['allowedIpRanges']:
                network = ipaddress.ip_network(ip_range)
                if ipaddress.ip_address(ip_address) in network:
                    allowed = True
                    break
            if not allowed:
                return False

        # Check blocked countries (would use GeoIP lookup)
        if 'blockedCountries' in conditions:
            country = self._get_country_from_ip(ip_address)
            if country in conditions['blockedCountries']:
                return False

        return True

    def _check_contextual_factors(self, request: Dict,
                                  factors: Dict) -> bool:
        """Evaluate contextual access factors"""
        now = datetime.utcnow()

        # Time of day check
        if 'timeOfDay' in factors:
            current_time = now.time()
            start = datetime.strptime(factors['timeOfDay']['start'],
                                     '%H:%M').time()
            end = datetime.strptime(factors['timeOfDay']['end'],
                                   '%H:%M').time()

            if not (start <= current_time <= end):
                return False

        # Day of week check
        if 'daysOfWeek' in factors:
            if now.weekday() not in factors['daysOfWeek']:
                return False

        # Risk score evaluation
        if 'riskScore' in factors:
            user_risk = self.risk_scorer.calculate_risk(request)
            threshold = factors['riskScore']['threshold']

            if user_risk > threshold:
                return False

        return True

    def _check_authentication_strength(self, request: Dict,
                                       requirements: Dict) -> bool:
        """Validate authentication strength requirements"""
        # Check MFA
        if requirements.get('mfaRequired'):
            if not request.get('mfaValidated'):
                return False

        # Check session age
        if 'reauthenticationInterval' in requirements:
            session_age = request.get('sessionAge', 0)
            if session_age > requirements['reauthenticationInterval']:
                return False

        return True

    def _build_decision(self, policy: Dict, result: Dict) -> Dict:
        """Build access decision from policy evaluation"""
        actions = policy.get('actions', {})

        if result['matched'] and actions.get('allow'):
            return {
                "decision": AccessDecision.ALLOW.value,
                "policyId": policy['policyId'],
                "grantDuration": actions.get('grantDuration', 3600),
                "requiresAction": []
            }
        elif actions.get('requireStepUp'):
            return {
                "decision": AccessDecision.STEP_UP_AUTH.value,
                "policyId": policy['policyId'],
                "reason": "Additional authentication required",
                "requiresAction": ["step_up_authentication"]
            }
        else:
            return {
                "decision": AccessDecision.DENY.value,
                "policyId": policy['policyId'],
                "reason": f"Failed conditions: {result['failedConditions']}",
                "requiresAction": []
            }

    def _get_applicable_policies(self, user_id: str,
                                resource: str) -> List[Dict]:
        """Get policies applicable to user and resource"""
        applicable = []

        for policy in self.policies:
            # Check if user matches
            subjects = policy.get('subjects', {})
            if user_id in subjects.get('users', []):
                # Check if resource matches
                resources = policy.get('resources', {})
                if resource in resources.get('applications', []):
                    applicable.append(policy)

        # Sort by priority
        return sorted(applicable, key=lambda p: p.get('priority', 999))

    def _get_country_from_ip(self, ip_address: str) -> str:
        """Get country code from IP address (simplified)"""
        # In production, use MaxMind GeoIP2 or similar
        return "US"


class DeviceTrustService:
    """Service for device trust verification"""

    def get_device_info(self, device_id: str) -> Dict:
        """Retrieve device compliance information"""
        # In production, integrate with MDM/UEM solution
        return {
            "deviceId": device_id,
            "encrypted": True,
            "antivirusEnabled": True,
            "os": "Windows",
            "lastCheckin": datetime.utcnow().isoformat()
        }


class RiskScorer:
    """Calculate real-time risk scores"""

    def calculate_risk(self, request: Dict) -> float:
        """Calculate risk score (0-100)"""
        risk_score = 0.0

        # Location risk
        if request.get('newLocation'):
            risk_score += 20.0

        # Device risk
        if not request.get('knownDevice'):
            risk_score += 30.0

        # Time-based risk
        hour = datetime.utcnow().hour
        if hour < 6 or hour > 22:
            risk_score += 10.0

        return min(risk_score, 100.0)

package zerotrust

import (
    "fmt"
    "net"

type MicroSegmentRule struct {
    RuleID      string
    SourceZone  string
    DestZone    string
    Protocol    string
    Port        int
    Action      string // "allow" or "deny"
    Priority    int
}

type MicroSegmentationEngine struct {
    rules       []MicroSegmentRule
    zones       map[string]*SecurityZone
}

type SecurityZone struct {
    ZoneID      string
    Name        string
    Networks    []string
    TrustLevel  int // 0-100
}

func NewMicroSegmentationEngine() *MicroSegmentationEngine {
    return &MicroSegmentationEngine{
        rules: make([]MicroSegmentRule, 0),
        zones: make(map[string]*SecurityZone),
    }
}

// AddSecurityZone defines a new security zone
func (mse *MicroSegmentationEngine) AddSecurityZone(zone *SecurityZone) {
    mse.zones[zone.ZoneID] = zone
}

// AddRule creates a new micro-segmentation rule
func (mse *MicroSegmentationEngine) AddRule(rule MicroSegmentRule) {
    mse.rules = append(mse.rules, rule)
    mse.sortRulesByPriority()
}

// EvaluateTraffic determines if traffic should be allowed
func (mse *MicroSegmentationEngine) EvaluateTraffic(
    sourceIP, destIP string,
    protocol string,
    port int,
) (bool, string) {

    // Determine source and destination zones
    sourceZone := mse.getZoneForIP(sourceIP)
    destZone := mse.getZoneForIP(destIP)

    if sourceZone == "" || destZone == "" {
        return false, "Unknown zone"
    }

    // Evaluate rules in priority order
    for _, rule := range mse.rules {
        if rule.matches(sourceZone, destZone, protocol, port) {
            allowed := rule.Action == "allow"
            return allowed, fmt.Sprintf("Matched rule: %s", rule.RuleID)
        }
    }

    // Default deny
    return false, "No matching rule - default deny"
}

func (rule *MicroSegmentRule) matches(
    sourceZone, destZone, protocol string,
    port int,
) bool {
    return rule.SourceZone == sourceZone &&
           rule.DestZone == destZone &&
           rule.Protocol == protocol &&
           rule.Port == port
}

func (mse *MicroSegmentationEngine) getZoneForIP(ipAddr string) string {
    ip := net.ParseIP(ipAddr)

            _, network, _ := net.ParseCIDR(networkCIDR)
            if network.Contains(ip) {

            }
        }
    }

interface JITAccessRequest {
    userId: string;
    resource: string;
    role: string;
    duration: number; // seconds
    justification: string;
}

interface JITAccessGrant {
    grantId: string;
    userId: string;
    resource: string;
    role: string;
    grantedAt: Date;
    expiresAt: Date;
    status: 'active' | 'expired' | 'revoked';
}

class JustInTimeAccessManager {
    private grants: Map = new Map();
    private maxDuration: number = 3600; // 1 hour max

    async requestAccess(request: JITAccessRequest): Promise {
        // Validate request
        if (request.duration > this.maxDuration) {
            throw new Error(`Duration exceeds maximum of ${this.maxDuration}s`);
        }

        // Check if user is eligible for role
        const eligible = await this.checkEligibility(
            request.userId,
            request.resource,
            request.role
        );

        if (!eligible) {
            throw new Error('User not eligible for requested role');
        }

        // Require approval for high-privilege roles
        if (this.isHighPrivilegeRole(request.role)) {
            await this.requestApproval(request);
        }

        // Create time-bound grant
        const grant: JITAccessGrant = {
            grantId: this.generateGrantId(),
            userId: request.userId,
            resource: request.resource,
            role: request.role,
            grantedAt: new Date(),
            expiresAt: new Date(Date.now() + request.duration * 1000),
            status: 'active'
        };

        this.grants.set(grant.grantId, grant);

        // Schedule automatic revocation
        this.scheduleRevocation(grant.grantId, request.duration);

        // Audit log
        await this.logJITAccess(grant, request.justification);

        return grant;
    }

    async validateAccess(userId: string, resource: string, role: string): Promise {
        // Find active grants for user
        for (const grant of this.grants.values()) {
            if (grant.userId === userId &&
                grant.resource === resource &&
                grant.role === role &&
                grant.status === 'active' &&
                grant.expiresAt > new Date()) {
                return true;
            }
        }

        return false;
    }

    private scheduleRevocation(grantId: string, duration: number): void {
        setTimeout(() => {
            this.revokeAccess(grantId);
        }, duration * 1000);
    }

    private async revokeAccess(grantId: string): Promise {
        const grant = this.grants.get(grantId);
        if (grant) {
            grant.status = 'expired';
            await this.logRevocation(grant);
        }
    }

5.6 WIA Zero Trust Architecture Certification

Organizations must implement comprehensive Zero Trust controls to achieve WIA certification:

Certification Requirements

  • Zero Trust strategy documented and approved by executive leadership
  • Identity-centric security model implemented organization-wide
  • Continuous verification of user and device trust before granting access
  • Least privilege access enforced with regular access reviews
  • Micro-segmentation implemented for critical applications and data
  • Device trust verification integrated into access decisions
  • Network location no longer grants implicit trust
  • Multi-factor authentication required for all users and applications
  • Just-In-Time (JIT) access implemented for privileged operations
  • Zero standing privileges for administrative accounts
  • Continuous monitoring and analytics for anomaly detection
  • Risk-based adaptive access policies deployed
  • End-to-end encryption for all sensitive data in transit
  • Application-level authorization enforced independent of network
  • API security with OAuth 2.0 and API gateways
  • Service mesh or equivalent for service-to-service authentication
  • Software-Defined Perimeter (SDP) or ZTNA solution deployed
  • User and Entity Behavior Analytics (UEBA) implemented
  • Automated response to security anomalies and policy violations
  • Regular Zero Trust maturity assessments conducted
  • Zero Trust architecture integrated with incident response
  • Cloud workload protection aligned with Zero Trust principles
  • Third-party and partner access governed by Zero Trust policies
  • Zero Trust training provided to IT and security teams
Best Practice: Zero Trust is a journey, not a destination. Start with high-value assets and expand incrementally across the organization.
