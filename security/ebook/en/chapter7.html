Chapter 7: Compliance & Governance

7.1 Introduction to Security Compliance

Security compliance ensures that organizations meet legal, regulatory, and industry requirements for protecting information and systems. The WIA Security Standard aligns with major compliance frameworks to help organizations achieve and maintain certification across multiple standards simultaneously.

Core Principle: Compliance is not a one-time achievement but a continuous process of assessment, improvement, and validation. Organizations must maintain evidence of ongoing compliance to satisfy regulatory and business requirements.

Major Compliance Frameworks

Framework Focus Industry Geographic Scope Audit Frequency
SOC 2 Service organization controls SaaS, Cloud providers Global (US-based) Annual
ISO 27001 Information security management All industries Global Annual surveillance, 3-year recertification
PCI DSS Payment card security Payment processing Global Annual (Quarterly scans)
HIPAA Healthcare data protection Healthcare United States Ongoing compliance
GDPR Data privacy and protection All industries EU/EEA + global reach Continuous compliance
NIST CSF Cybersecurity framework Critical infrastructure United States Continuous assessment

7.2 SOC 2 Trust Service Criteria

SOC 2 evaluates controls based on five Trust Service Criteria (TSC). Security is mandatory, while the other four are optional based on business needs.

SOC 2 Trust Service Criteria Mapping

Criteria Focus Area Key Controls WIA Alignment
Security (CC) Information and systems protection Access control, Encryption, Monitoring Chapters 1-6
Availability (A) System availability and performance Redundancy, Backup, Disaster recovery Chapter 6
Processing Integrity (PI) System processing accuracy Validation, Error handling, Monitoring Chapter 1
Confidentiality (C) Confidential information protection Encryption, Classification, DLP Chapter 3
Privacy (P) Personal information handling Consent, Data subject rights, Retention Chapter 7

Example 1: Compliance Control Tracker (Python)

Python - Compliance Control Management
from typing import List, Dict, Optional
from datetime import datetime, timedelta
from enum import Enum

class ImplementationStatus(Enum):
    IMPLEMENTED = "implemented"
    PARTIALLY_IMPLEMENTED = "partially_implemented"
    PLANNED = "planned"
    NOT_APPLICABLE = "not_applicable"

class TestingStatus(Enum):
    PASSED = "passed"
    FAILED = "failed"
    NOT_TESTED = "not_tested"
    IN_PROGRESS = "in_progress"

class ComplianceControlTracker:
    """
    WIA Compliance Control Management System
    Tracks implementation and testing of compliance controls
    """

    def __init__(self):
        self.controls = {}
        self.evidence_store = EvidenceStore()
        self.testing_engine = ControlTestingEngine()

    def add_control(self, control_data: Dict) -> str:
        """Add new compliance control"""
        control_id = control_data.get('controlId') or self._generate_control_id()

        control = {
            "controlId": control_id,
            "framework": control_data["framework"],
            "controlNumber": control_data["controlNumber"],
            "title": control_data["title"],
            "description": control_data["description"],
            "category": control_data.get("category", ""),
            "implementationStatus": ImplementationStatus.PLANNED.value,
            "testingStatus": TestingStatus.NOT_TESTED.value,
            "lastTested": None,
            "owner": control_data.get("owner", ""),
            "evidence": [],
            "findings": []
        }

        self.controls[control_id] = control
        return control_id

    def update_implementation_status(self, control_id: str,
                                    status: ImplementationStatus,
                                    notes: str = "") -> bool:
        """Update control implementation status"""
        if control_id not in self.controls:
            return False

        control = self.controls[control_id]
        control["implementationStatus"] = status.value

        # Add audit trail
        self._add_audit_log(
            control_id,
            "implementation_status_updated",
            f"Status changed to {status.value}",
            notes
        )

        return True

    def add_evidence(self, control_id: str, evidence_data: Dict) -> Optional[str]:
        """Add evidence for compliance control"""
        if control_id not in self.controls:
            return None

        evidence_id = self._generate_evidence_id()
        evidence = {
            "evidenceId": evidence_id,
            "type": evidence_data["type"],
            "description": evidence_data["description"],
            "location": evidence_data["location"],
            "collectedDate": datetime.utcnow().isoformat(),
            "collectedBy": evidence_data.get("collectedBy", "")
        }

        # Store evidence file
        if "file" in evidence_data:
            storage_location = self.evidence_store.upload(
                evidence_id,
                evidence_data["file"]
            )
            evidence["location"] = storage_location

        self.controls[control_id]["evidence"].append(evidence)

        return evidence_id

    def test_control(self, control_id: str, test_procedure: str) -> Dict:
        """Execute control testing"""
        if control_id not in self.controls:
            return {"success": False, "error": "Control not found"}

        control = self.controls[control_id]

        # Execute automated testing if available
        test_result = self.testing_engine.execute_test(
            control_id,
            test_procedure
        )

        # Update control testing status
        control["testingStatus"] = test_result["status"]
        control["lastTested"] = datetime.utcnow().isoformat()

        # Record findings if test failed
        if test_result["status"] == TestingStatus.FAILED.value:
            self._add_finding(
                control_id,
                test_result["findings"]
            )

        return test_result

    def _add_finding(self, control_id: str, finding_data: Dict):
        """Add audit finding for control"""
        finding = {
            "findingId": self._generate_finding_id(),
            "severity": finding_data.get("severity", "medium"),
            "description": finding_data["description"],
            "remediation": finding_data.get("remediation", ""),
            "identifiedDate": datetime.utcnow().isoformat(),
            "dueDate": (datetime.utcnow() + timedelta(days=30)).isoformat(),
            "status": "open"
        }

        self.controls[control_id]["findings"].append(finding)

    def get_compliance_status(self, framework: str) -> Dict:
        """Get compliance status summary for framework"""
        framework_controls = [
            c for c in self.controls.values()
            if c["framework"] == framework
        ]

        total = len(framework_controls)
        if total == 0:
            return {"framework": framework, "status": "not_started"}

        # Count statuses
        implemented = sum(
            1 for c in framework_controls
            if c["implementationStatus"] == ImplementationStatus.IMPLEMENTED.value
        )

        tested_passed = sum(
            1 for c in framework_controls
            if c["testingStatus"] == TestingStatus.PASSED.value
        )

        open_findings = sum(
            len([f for f in c["findings"] if f["status"] == "open"])
            for c in framework_controls
        )

        return {
            "framework": framework,
            "totalControls": total,
            "implemented": implemented,
            "implementationPercentage": (implemented / total * 100),
            "testedPassed": tested_passed,
            "testingPercentage": (tested_passed / total * 100),
            "openFindings": open_findings,
            "status": self._determine_status(implemented, total, open_findings)
        }

    def generate_audit_report(self, framework: str,
                             audit_period: Dict) -> Dict:
        """Generate compliance audit report"""
        status = self.get_compliance_status(framework)
        controls = [
            c for c in self.controls.values()
            if c["framework"] == framework
        ]

        return {
            "framework": framework,
            "auditPeriod": audit_period,
            "generatedAt": datetime.utcnow().isoformat(),
            "summary": status,
            "controls": controls,
            "findings": self._get_all_findings(framework),
            "recommendations": self._generate_recommendations(controls)
        }

    def perform_gap_analysis(self, framework: str,
                            required_controls: List[str]) -> Dict:
        """Identify gaps in compliance implementation"""
        implemented_controls = set(
            c["controlNumber"]
            for c in self.controls.values()
            if c["framework"] == framework
        )

        missing_controls = set(required_controls) - implemented_controls

        return {
            "framework": framework,
            "requiredControls": len(required_controls),
            "implementedControls": len(implemented_controls),
            "missingControls": list(missing_controls),
            "gapPercentage": (len(missing_controls) / len(required_controls) * 100)
        }

    def _determine_status(self, implemented: int, total: int,
                         open_findings: int) -> str:
        """Determine overall compliance status"""
        percentage = implemented / total * 100

        if percentage == 100 and open_findings == 0:
            return "compliant"
        elif percentage >= 80 and open_findings < 5:
            return "in_progress"
        else:
            return "non_compliant"

    def _get_all_findings(self, framework: str) -> List[Dict]:
        """Get all findings for framework"""
        findings = []
        for control in self.controls.values():
            if control["framework"] == framework:
                findings.extend(control["findings"])
        return findings

    def _generate_recommendations(self, controls: List[Dict]) -> List[str]:
        """Generate improvement recommendations"""
        recommendations = []

        # Check for untested controls
        untested = [c for c in controls if c["testingStatus"] == "not_tested"]
        if untested:
            recommendations.append(
                f"Test {len(untested)} controls that have not been tested"
            )

        # Check for old test results
        stale = [
            c for c in controls
            if c["lastTested"] and
            datetime.fromisoformat(c["lastTested"]) <
            datetime.utcnow() - timedelta(days=90)
        ]
        if stale:
            recommendations.append(
                f"Retest {len(stale)} controls with test results older than 90 days"
            )

        return recommendations

    def _generate_control_id(self) -> str:
        return f"CTRL-{datetime.utcnow().timestamp()}"

    def _generate_evidence_id(self) -> str:
        return f"EVD-{datetime.utcnow().timestamp()}"

    def _generate_finding_id(self) -> str:
        return f"FIND-{datetime.utcnow().timestamp()}"

    def _add_audit_log(self, control_id: str, action: str,
                      description: str, notes: str):
        """Add entry to audit log"""
        print(f"[AUDIT] Control {control_id}: {action} - {description}")

interface SOC2Control {
    controlId: string;
    category: 'CC' | 'A' | 'PI' | 'C' | 'P';
    criteriaNumber: string;
    description: string;
    testingProcedure: string;
    testingFrequency: string;
    evidenceRequired: string[];
}

class SOC2ReportGenerator {
    private controls: Map = new Map();
    private testResults: Map = new Map();

    async generateType2Report(auditPeriod: { start: Date; end: Date }): Promise {
        const report = {
            reportType: 'SOC 2 Type II',
            auditPeriod: {
                start: auditPeriod.start.toISOString(),
                end: auditPeriod.end.toISOString()
            },
            organizationInfo: await this.getOrganizationInfo(),
            systemDescription: await this.getSystemDescription(),
            trustServiceCriteria: this.evaluateTrustServiceCriteria(),
            controlsTested: await this.summarizeControlTesting(auditPeriod),
            exceptions: await this.identifyExceptions(),
            managementAssertion: this.generateManagementAssertion()
        };

        return report;
    }

    evaluateTrustServiceCriteria(): any {
        const criteria = {
            security: this.evaluateSecurityControls(),
            availability: this.evaluateAvailabilityControls(),
            processingIntegrity: this.evaluateProcessingIntegrityControls(),
            confidentiality: this.evaluateConfidentialityControls(),
            privacy: this.evaluatePrivacyControls()
        };

        return criteria;
    }

    private evaluateSecurityControls(): any {
        const securityControls = Array.from(this.controls.values())
            .filter(c => c.category === 'CC');

        return {
            totalControls: securityControls.length,
            controlsImplemented: securityControls.filter(
                c => this.isControlImplemented(c.controlId)
            ).length,
            controlsTested: securityControls.filter(
                c => this.isControlTested(c.controlId)
            ).length,
            exceptionsIdentified: this.getExceptionsForCategory('CC').length
        };
    }

    private evaluateAvailabilityControls(): any {
        // Similar implementation for Availability criteria
        return {};
    }

    private evaluateProcessingIntegrityControls(): any {
        // Similar implementation for Processing Integrity
        return {};
    }

    private evaluateConfidentialityControls(): any {
        // Similar implementation for Confidentiality
        return {};
    }

    private evaluatePrivacyControls(): any {
        // Similar implementation for Privacy
        return {};
    }

    private async summarizeControlTesting(period: any): Promise {
        const summary = {
            totalTests: 0,
            testsPerformed: 0,
            testsPassed: 0,
            testsFailed: 0,
            samplesSelected: 0,
            exceptionRate: 0
        };

        for (const [controlId, results] of this.testResults) {
            const periodResults = results.filter(r =>
                new Date(r.testDate) >= period.start &&
                new Date(r.testDate) <= period.end
            );

            summary.totalTests += periodResults.length;
            summary.testsPassed += periodResults.filter(r => r.result === 'passed').length;
            summary.testsFailed += periodResults.filter(r => r.result === 'failed').length;
        }

        summary.testsPerformed = summary.testsPassed + summary.testsFailed;
        summary.exceptionRate = summary.testsFailed / summary.testsPerformed * 100;

        return summary;
    }

    private async identifyExceptions(): Promise {
        const exceptions = [];

        for (const [controlId, results] of this.testResults) {
            const failures = results.filter(r => r.result === 'failed');

            for (const failure of failures) {
                exceptions.push({
                    controlId,
                    testDate: failure.testDate,
                    description: failure.description,
                    rootCause: failure.rootCause,
                    remediation: failure.remediation,
                    status: failure.remediationStatus
                });
            }
        }

        return exceptions;
    }

    private generateManagementAssertion(): string {
        return `Management of [Organization Name] is responsible for designing, implementing, ` +
               `operating, and maintaining effective controls within the system to provide ` +
               `reasonable assurance that the trust services criteria relevant to security, ` +
               `availability, and confidentiality were achieved based on the criteria in ` +
               `TSP section 100, 2017 Trust Services Criteria for Security, Availability, ` +
               `Processing Integrity, Confidentiality, and Privacy.`;
    }

    private isControlImplemented(controlId: string): boolean {
        // Check implementation status
        return true;
    }

    private isControlTested(controlId: string): boolean {
        return this.testResults.has(controlId);
    }

    private getExceptionsForCategory(category: string): any[] {
        return [];
    }

    private async getOrganizationInfo(): Promise {
        return {
            name: 'Organization Name',
            address: 'Organization Address',
            description: 'System description'
        };
    }

    private async getSystemDescription(): Promise {

            infrastructure: 'Cloud-based infrastructure',
            software: 'Application stack',
            people: 'Organizational structure',
            procedures: 'Operational procedures',
            data: 'Data flows and classification'
        };
    }
}

Example 3: Automated Compliance Scanner (Bash)

Bash - Security Configuration Scanner
#!/bin/bash
# WIA Compliance Scanner - Automated Security Configuration Validation

set -euo pipefail

REPORT_FILE="/var/log/compliance/scan-$(date +%Y%m%d-%H%M%S).log"
FAILED_CHECKS=0
PASSED_CHECKS=0

log_check() {
    local status="$1"
    local control="$2"
    local message="$3"

    echo "[$status] $control: $message" | tee -a "$REPORT_FILE"

    if [ "$status" = "PASS" ]; then
        ((PASSED_CHECKS++))
    else
        ((FAILED_CHECKS++))
    fi
}

check_ssh_config() {
    echo "=== Checking SSH Configuration ==="

    # Check if root login is disabled
    if grep -q "^PermitRootLogin no" /etc/ssh/sshd_config; then
        log_check "PASS" "CC6.1" "Root login disabled"
    else
        log_check "FAIL" "CC6.1" "Root login is enabled"
    fi

    # Check if password authentication is disabled
    if grep -q "^PasswordAuthentication no" /etc/ssh/sshd_config; then
        log_check "PASS" "CC6.1" "Password authentication disabled"
    else
        log_check "FAIL" "CC6.1" "Password authentication is enabled"
    fi
}

check_firewall() {
    echo "=== Checking Firewall Configuration ==="

    if systemctl is-active --quiet ufw; then
        log_check "PASS" "CC6.6" "Firewall is active"
    else
        log_check "FAIL" "CC6.6" "Firewall is not active"
    fi
}

check_encryption() {
    echo "=== Checking Encryption ==="

    # Check if disk encryption is enabled
    if lsblk -o NAME,FSTYPE | grep -q "crypto_LUKS"; then
        log_check "PASS" "CC6.7" "Disk encryption enabled"
    else
        log_check "FAIL" "CC6.7" "Disk encryption not detected"
    fi
}

check_logging() {
    echo "=== Checking Logging Configuration ==="

    if systemctl is-active --quiet rsyslog; then
        log_check "PASS" "CC7.2" "Logging service is active"
    else
        log_check "FAIL" "CC7.2" "Logging service is not active"
    fi
}

check_updates() {
    echo "=== Checking System Updates ==="

    UPDATES=$(apt list --upgradable 2>/dev/null | grep -c "upgradable" || true)

    if [ "$UPDATES" -eq 0 ]; then
        log_check "PASS" "CC8.1" "System is up to date"
    else
        log_check "FAIL" "CC8.1" "$UPDATES updates available"
    fi
}

# Run all checks
check_ssh_config
check_firewall
check_encryption
check_logging
check_updates

# Generate summary
echo ""
echo "=== Compliance Scan Summary ===" | tee -a "$REPORT_FILE"
echo "Passed: $PASSED_CHECKS" | tee -a "$REPORT_FILE"
echo "Failed: $FAILED_CHECKS" | tee -a "$REPORT_FILE"
echo "Report: $REPORT_FILE"

if [ "$FAILED_CHECKS" -gt 0 ]; then
    exit 1
fi

7.6 WIA Compliance & Governance Certification

Organizations must demonstrate comprehensive compliance and governance capabilities to achieve WIA certification:

Certification Requirements

  • Compliance framework(s) selected and documented (SOC 2, ISO 27001, or equivalent)
  • Gap analysis performed against chosen compliance framework(s)
  • Compliance program established with executive sponsorship
  • Dedicated compliance officer or team assigned
  • All required policies documented and approved
  • Policy review process established (annual minimum)
  • Risk assessment methodology defined and documented
  • Enterprise risk assessment performed annually
  • Risk register maintained and regularly updated
  • Compliance controls mapped to framework requirements
  • Control implementation status tracked and reported
  • Evidence collection process automated where possible
  • Internal audit program established with annual audits
  • External audit completed for applicable frameworks
  • Audit findings tracked with remediation plans
  • Security awareness training program with annual completion
  • Role-based security training for privileged users
  • Vendor risk management program established
  • Third-party security assessments performed
  • Business continuity and disaster recovery plans tested annually
  • Data protection impact assessments (DPIA) for high-risk processing
  • Privacy program established for personal data handling
