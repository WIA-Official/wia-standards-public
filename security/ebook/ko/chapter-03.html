챕터 03

물리적 보안 시스템

3.1 물리적 보안 개요

물리적 보안은 시설, 장비 및 자원을 물리적 위협과 무단 접근으로부터 보호하는 보안 조치입니다. 사이버보안과 함께 물리적 보안은 조직의 전체 보안 태세의 필수적인 부분입니다.

현대 보안 환경에서 물리적 보안과 사이버보안은 점점 더 통합되고 있습니다. 물리적 접근 제어 시스템은 IT 네트워크와 연결되고, 감시 시스템은 AI 분석을 활용하며, 사물인터넷(IoT) 장치가 물리적 보안 인프라의 일부가 되었습니다.

물리적 보안의 세 가지 기둥

접근 제어: 누가 시설에 들어올 수 있는지 제어하고 모니터링

감시: 활동을 모니터링하고 기록하여 사고 탐지 및 조사 지원

보안 테스트: 물리적 침입 및 공격에 대한 보안 조치의 효과성 검증

3.1.1 물리적 보안 목표

  • 무단 접근 방지 및 탐지
  • 직원, 방문자 및 자산 보호
  • 도난, 파괴 및 사보타지 방지
  • 환경적 위협으로부터 보호
  • 비즈니스 연속성 지원
  • 규제 준수 달성

3.2 접근 제어 시스템

접근 제어 시스템은 누가 특정 영역에 들어갈 수 있는지를 관리합니다. 현대 시스템은 다양한 인증 방식과 기술을 결합하여 다계층 보안을 제공합니다.

3.2.1 인증 요소

소유 기반 (Something You Have)

  • 접근 카드 (근접/스마트)
  • 키 FOB
  • 모바일 자격 증명
  • 토큰

지식 기반 (Something You Know)

  • PIN 코드
  • 비밀번호
  • 패턴
  • 보안 질문

생체 기반 (Something You Are)

  • 지문
  • 안면 인식
  • 홍채 스캔
  • 손바닥 정맥

3.2.2 접근 제어 기술

기술 작동 원리 보안 수준 일반적 용도
근접 카드 125kHz RFID 신호 낮음-중간 일반 사무실
스마트 카드 13.56MHz, 암호화 높음 보안 시설
생체 인식 생체 특성 분석 매우 높음 고보안 구역
모바일 자격 증명 BLE/NFC 통신 중간-높음 현대적 시설
다중 요소 두 가지 이상 결합 매우 높음 중요 인프라

3.3 감시 시스템

비디오 감시 시스템은 물리적 보안의 핵심 구성 요소로, 실시간 모니터링, 사고 탐지 및 사후 조사를 지원합니다.

3.3.1 CCTV 시스템 구성요소

카메라 유형

  • 고정 카메라 - 특정 영역 감시
  • PTZ 카메라 - 팬/틸트/줌 기능
  • 돔 카메라 - 실내용, 내구성
  • IR 카메라 - 저조도/야간 감시
  • 열화상 카메라 - 열 감지
  • 360도 카메라 - 전방위 커버리지

녹화 및 저장

  • NVR (네트워크 비디오 레코더)
  • DVR (디지털 비디오 레코더)
  • 클라우드 스토리지
  • 하이브리드 저장소
  • 엣지 스토리지
  • RAID 구성

3.3.2 AI 기반 비디오 분석

현대 감시 시스템은 인공지능을 활용하여 자동화된 위협 탐지 및 분석 기능을 제공합니다.

AI 비디오 분석 기능: ==================== 동작 감지: • 움직임 감지 및 추적 • 이상 행동 탐지 • 배회 감지 • 침입 감지 얼굴 인식: • 실시간 얼굴 식별 • 관심인물 알림 • 접근 제어 통합 • 감정/행동 분석 객체 인식: • 사람/차량/물체 분류 • 번호판 인식 • 포기된 물체 탐지 • 군중 밀집도 분석 이벤트 분석: • 넘어짐 감지 • 싸움/폭력 감지 • 무기 감지 • 연기/화재 감지

3.3.3 감시 시스템 고려사항

개인정보 및 규정 준수

감시 시스템 구현 시 개인정보 보호법과 규정을 준수해야 합니다:

  • 감시 구역에 대한 명확한 고지
  • 녹화 데이터 보관 기간 정책
  • 데이터 접근 권한 제한
  • 직원 프라이버시 고려
  • GDPR, 개인정보보호법 등 규정 준수

3.4 환경 제어

환경 제어는 화재, 홍수, 온도 변화 및 기타 환경적 위협으로부터 시설과 장비를 보호합니다.

3.4.1 화재 감지 및 진압

감지 시스템

  • 연기 감지기
  • 열 감지기
  • 화염 감지기
  • 가스 감지기
  • 복합 감지기

진압 시스템

  • 스프링클러 시스템
  • 가스 소화 시스템 (FM-200, Novec)
  • 분말 소화 시스템
  • 워터미스트 시스템
  • 휴대용 소화기

3.4.2 데이터 센터 환경 제어

요소 권장 범위 모니터링 주기
온도 18-27°C (64-80°F) 실시간
습도 40-60% RH 실시간
공기질 ISO 14644-1 Class 8 지속적
기압 양압 유지 지속적

3.4.3 전력 보호

전력 인프라 구성

  • UPS (무정전 전원 장치): 순간 정전 및 전력 품질 문제 해결
  • 발전기: 장시간 정전 시 백업 전원
  • 자동 전환 스위치 (ATS): 전원 소스 간 자동 전환
  • PDU (전원 분배 장치): 랙 레벨 전원 분배 및 모니터링
  • 서지 보호기: 전압 급등으로부터 장비 보호

3.5 물리적-논리적 보안 통합

물리적 보안과 사이버보안의 통합은 현대 보안 운영의 핵심입니다. 물리적 접근 제어와 IT 시스템의 연계를 통해 보다 강력한 보안 태세를 구축할 수 있습니다.

3.5.1 통합 이점

보안 강화

  • 물리적 위치 기반 논리적 접근 제어
  • 배지 및 로그인 상관관계 분석
  • 내부자 위협 탐지 향상
  • 포괄적인 감사 추적

운영 효율성

  • 통합 신원 관리
  • 자동화된 온보딩/오프보딩
  • 단일 대시보드 모니터링
  • 통합 사고 대응

3.5.2 통합 사용 사례

물리적-논리적 보안 통합 시나리오: ================================ 시나리오 1: 배지-로그인 상관관계 - 직원이 건물에 배지 인 없이 네트워크 로그인 시도 - SIEM이 이상 탐지 알림 생성 - SOC가 잠재적 자격 증명 공유/도난 조사 시나리오 2: 퇴근 후 접근 제어 - 직원이 업무 시간 외에 민감 구역 접근 - 물리적 접근 시스템이 IT 시스템에 알림 - 해당 사용자의 네트워크 접근 제한 또는 강화된 모니터링 활성화 시나리오 3: 자동화된 오프보딩 - HR 시스템에서 직원 퇴사 처리 - 물리적 배지 자동 비활성화 - 모든 IT 시스템 접근 권한 자동 철회 - 반납 자산 목록 생성 시나리오 4: 응급 상황 대응 - 화재 경보 활성화 - 비상 출구 잠금 해제 - 엘리베이터 운행 중단 - 비상 대응팀에 자동 알림 - 건물 내 인원 명단 생성

3.6 보안 구역 설계

효과적인 물리적 보안을 위해서는 시설을 보안 수준에 따라 구역화하고 각 구역에 적절한 통제를 적용해야 합니다.

3.6.1 보안 구역 분류

구역 설명 접근 제어 예시
공용 구역 일반인 접근 가능 최소 로비, 리셉션
제한 구역 직원 및 승인된 방문자 배지/PIN 일반 사무실
보안 구역 특정 역할의 직원만 배지+PIN 또는 생체 서버룸, 재무
고보안 구역 엄격히 제한된 접근 다중 요소 + 감시 데이터 센터, 금고

3.6.2 계층형 방어

양파 껍질 모델

시설 보안은 양파 껍질처럼 여러 겹의 보안 계층으로 구성되어야 합니다:

  1. 외곽 경계: 울타리, 조명, 외부 CCTV
  2. 건물 외부: 출입문 제어, 경비원, 방문자 관리
  3. 건물 내부: 구역별 접근 제어, 내부 CCTV
  4. 보안 구역: 강화된 접근 통제, 에어락
  5. 핵심 자산: 금고, 케이지, 락커

챕터 요약

이 챕터에서는 물리적 보안 시스템의 핵심 요소를 다루었습니다:

  • 접근 제어: 다양한 인증 방식과 기술을 활용한 물리적 접근 관리
  • 감시 시스템: CCTV와 AI 기반 비디오 분석을 통한 모니터링
  • 환경 제어: 화재, 전력, 온도 등 환경적 위협으로부터 보호
  • 통합 보안: 물리적-논리적 보안의 연계를 통한 강화된 보안
  • 구역 설계: 계층형 방어를 통한 체계적인 시설 보호
