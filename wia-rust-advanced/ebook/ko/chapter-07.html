🎯 WebAssembly란?

WebAssembly (WASM)는 웹 브라우저에서 실행되는 저수준 바이너리 포맷입니다. JavaScript보다 빠른 성능을 제공하며, Rust를 포함한 다양한 언어로 작성할 수 있습니다.

💡 왜 Rust + WASM인가?
  • 성능: 네이티브에 가까운 속도
  • 안전성: Rust의 메모리 안전성 보장
  • 크기: 작은 바이너리 크기
  • 에코시스템: wasm-bindgen, wasm-pack 등 훌륭한 도구

🔧 DOM 조작 (web-sys)

web-sys 크레이트를 사용하면 DOM API에 접근할 수 있습니다.

Cargo.toml에 의존성 추가

[dependencies]
wasm-bindgen = "0.2"
web-sys = { version = "0.3", features = [
    "Document",
    "Element",
    "HtmlElement",
    "Node",
    "Window",
] }

DOM 요소 생성 및 조작

use wasm_bindgen::prelude::*;
use web_sys::{Document, Element, HtmlElement, Window};

#[wasm_bindgen(start)]
pub fn main() -> Result<(), JsValue> {
    let window = web_sys::window().expect("no window");
    let document = window.document().expect("no document");

    // div 요소 생성
    let div = document.create_element("div")?;
    div.set_text_content(Some("Hello from Rust!"));
    div.set_class_name("greeting");

    // body에 추가
    let body = document.body().expect("no body");
    body.append_child(&div)?;

    Ok(())
}

이벤트 리스너

use wasm_bindgen::prelude::*;
use wasm_bindgen::JsCast;
use web_sys::{EventTarget, HtmlElement};

#[wasm_bindgen]
pub fn setup_button() -> Result<(), JsValue> {
    let window = web_sys::window().unwrap();
    let document = window.document().unwrap();

    let button = document
        .get_element_by_id("myButton")
        .unwrap()
        .dyn_into::<HtmlElement>()?;

    let closure = Closure::wrap(Box::new(move |_event: web_sys::Event| {
        web_sys::console::log_1(&"Button clicked!".into());
    }) as Box<dyn FnMut(_)>);

    button.add_event_listener_with_callback(
        "click",
        closure.as_ref().unchecked_ref()
    )?;

    closure.forget(); // 메모리 누수 방지 (주의!)

    Ok(())
}

🎨 Canvas 그리기

[dependencies.web-sys]
version = "0.3"
features = [
    "CanvasRenderingContext2d",
    "HtmlCanvasElement",
]

use wasm_bindgen::prelude::*;
use web_sys::{CanvasRenderingContext2d, HtmlCanvasElement};

#[wasm_bindgen]
pub fn draw() -> Result<(), JsValue> {
    let document = web_sys::window()
        .unwrap()
        .document()
        .unwrap();

    let canvas = document
        .get_element_by_id("canvas")
        .unwrap()
        .dyn_into::<HtmlCanvasElement>()?;

    let context = canvas
        .get_context("2d")?
        .unwrap()
        .dyn_into::<CanvasRenderingContext2d>()?;

    // 사각형 그리기
    context.set_fill_style(&"red".into());
    context.fill_rect(10.0, 10.0, 100.0, 100.0);

    // 원 그리기
    context.begin_path();
    context.arc(200.0, 200.0, 50.0, 0.0, std::f64::consts::PI * 2.0)?;
    context.set_fill_style(&"blue".into());
    context.fill();

    Ok(())
}

⚡ Yew 프레임워크

Yew는 Rust로 작성하는 모던 웹 프레임워크입니다. React와 유사한 컴포넌트 기반 아키텍처를 제공합니다.

프로젝트 설정

[dependencies]
yew = { version = "0.21", features = ["csr"] }

간단한 카운터 앱

use yew::prelude::*;

#[function_component]
fn App() -> Html {
    let counter = use_state(|| 0);

    let increment = {
        let counter = counter.clone();
        Callback::from(move |_| counter.set(*counter + 1))
    };

    let decrement = {
        let counter = counter.clone();
        Callback::from(move |_| counter.set(*counter - 1))
    };

    html! {
        <div class="container">
            <h1>{"Counter App"}</h1>
            <p>{"Count: "}{*counter}</p>
            <button onclick={increment}>{"+"}</button>
            <button onclick={decrement}>{"-"}</button>
        </div>
    }
}

fn main() {
    yew::Renderer::<App>::new().render();
}

컴포넌트와 Props

#[derive(Properties, PartialEq)]
pub struct ButtonProps {
    pub label: String,
    pub onclick: Callback<MouseEvent>,
}

#[function_component]
fn CustomButton(props: &ButtonProps) -> Html {
    html! {
        <button onclick={&props.onclick}>
            {&props.label}
        </button>
    }
}

메모리 공유

use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn process_data(data: &[u8]) -> Vec<u8> {
    // JavaScript와 메모리 공유
    data.iter().map(|&x| x * 2).collect()
}
