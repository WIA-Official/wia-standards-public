🦀 Rust 고급 프로그래밍

WIA-RUST-ADVANCED: 마스터 레벨로 가는 여정

이 책에 대하여

WIA-RUST-ADVANCED는 Rust 프로그래밍의 고급 개념을 마스터하기 위한 종합 가이드입니다. Unsafe 코드부터 운영체제 개발, WebAssembly까지 Rust의 모든 고급 기능을 다룹니다.

이 과정은 10개의 레벨(Level 0-9)로 구성되어 있으며, 각 레벨은 체계적으로 난이도가 증가하여 여러분을 Rust 마스터로 이끌어 줄 것입니다.

📚 체계적 학습

기초부터 고급까지 단계적으로 학습할 수 있는 커리큘럼

💻 실전 코드

즉시 실행 가능한 예제 코드와 프로젝트

🎯 명확한 목표

각 레벨마다 명확한 학습 목표와 성취 기준

🌟 실무 중심

실제 프로젝트에 바로 적용할 수 있는 실용적 지식

📖 목차

01
Unsafe 마스터
Rust의 안전장치를 벗어나는 방법과 책임있는 Unsafe 코드 작성법을 배웁니다. 원시 포인터, Unsafe 함수, 정적 변수 등을 다룹니다.
Level 0: Unsafe Mastery
02
FFI 완전정복
C/C++ 라이브러리와의 상호운용, 외부 함수 호출, 데이터 구조 매핑 등 Foreign Function Interface의 모든 것을 마스터합니다.
Level 1: FFI Expert
03
매크로 마스터
선언적 매크로와 절차적 매크로를 깊이있게 학습합니다. 커스텀 derive, 속성 매크로, 함수형 매크로를 만들어봅니다.
Level 2: Macro Wizard
04
컴파일러 & 임베디드
Rust 컴파일러 내부 구조, MIR, LLVM IR을 이해하고, 임베디드 시스템과 bare-metal 프로그래밍을 학습합니다.
Level 3-4: Compiler & Embedded
05
운영체제 개발
Rust로 운영체제를 만들어봅니다. 부트로더, 인터럽트 처리, 메모리 관리, 멀티태스킹까지 OS의 핵심을 구현합니다.
Level 5: OS Developer
06
메모리 할당자 & SIMD
커스텀 메모리 할당자를 구현하고, SIMD를 활용한 초고성능 연산을 학습합니다. 성능 최적화의 정점에 도달합니다.
Level 6-7: Performance Master
07
WebAssembly
Rust를 WebAssembly로 컴파일하여 웹에서 실행합니다. wasm-bindgen, wasm-pack을 활용한 실전 웹 개발을 배웁니다.
Level 8: WASM Expert
08
마스터 프로젝트 FINAL
모든 학습 내용을 통합한 종합 프로젝트를 수행합니다. 고성능 웹 서버, 시스템 모니터링 도구 등을 구축하며 진정한 마스터가 됩니다.
Level 9: Rust Master
