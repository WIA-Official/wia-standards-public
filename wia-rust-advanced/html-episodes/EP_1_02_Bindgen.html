bindgen이란?What is bindgen?

🤖 bindgen - C → Rust🤖 bindgen - C → Rust

C 헤더 파일을 읽어서 Rust FFI 바인딩 자동 생성

✅ 수천 개의 함수도 자동 변환
✅ 구조체, 열거형, 매크로 지원
✅ 빌드 스크립트 통합
✅ 타입 안전성 보장

Reads C header files and auto-generates Rust FFI bindings

✅ Auto-converts thousands of functions
✅ Supports structs, enums, macros
✅ Integrates with build scripts
✅ Ensures type safety

bindgen 사용법Using bindgen

// build.rs
use bindgen;
fn main() {
let bindings = bindgen::Builder::default()
.header("wrapper.h")
.generate()
.expect("Unable to generate bindings");
bindings.write_to_file("src/bindings.rs")
.expect("Couldn't write bindings!");
}

📝 Cargo.toml

[build-dependencies]
bindgen = "0.69"

cbindgen이란?What is cbindgen?

🔄 cbindgen - Rust → C🔄 cbindgen - Rust → C

Rust 코드로부터 C/C++ 헤더 파일 생성

✅ Rust 라이브러리를 C에서 사용 가능
✅ 구조체, 함수, 상수 자동 변환
✅ C와 C++ 모두 지원
✅ 문서 주석도 변환

Generates C/C++ header files from Rust code

✅ Makes Rust libraries usable from C
✅ Auto-converts structs, functions, constants
✅ Supports both C and C++
✅ Converts doc comments too

cbindgen 사용법Using cbindgen

// lib.rs
#[no_mangle]
pub extern "C" fn add(a: i32, b: i32) -> i32 {
a + b
}
// Generated header (mylib.h)
int32_t add(int32_t a, int32_t b);

🚀 빌드 명령어🚀 Build Command

cbindgen --lang c --output mylib.h

고급 설정Advanced Configuration

// bindgen customization
bindgen::Builder::default()
.header("wrapper.h")
.allowlist_function("my_.*")
.blocklist_type("internal_.*")
.derive_default(true)
.generate()
# cbindgen.toml
language = "C"
style = "both"
documentation = true
