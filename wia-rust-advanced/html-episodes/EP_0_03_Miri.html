Miri란? What is Miri?

🔍 Miri = MIR Interpreter 🔍 Miri = MIR Interpreter

Rust의 중간 표현(MIR)을 인터프리트하며 실행하는 도구

✅ Undefined Behavior 탐지
✅ 메모리 안전성 위반 검출
✅ 동시성 버그 발견
✅ unsafe 코드 검증

Tool that interprets and executes Rust's Mid-level IR (MIR)

✅ Detects Undefined Behavior
✅ Finds memory safety violations
✅ Discovers concurrency bugs
✅ Validates unsafe code

💡 핵심 아이디어 💡 Key Idea

일반적인 실행은 UB를 놓칠 수 있지만,
Miri는 모든 메모리 접근을 추적하여 UB를 즉시 탐지합니다.

Normal execution may miss UB,
but Miri tracks all memory accesses to detect UB immediately.

Miri 설치 Installing Miri

$ rustup +nightly component add miri
info: downloading component 'miri'
info: installing component 'miri'
✓ Miri installed successfully
$ cargo +nightly miri --version
miri 0.1.0 (nightly-2024-01-15)

📝 참고사항 📝 Notes

• Miri는 nightly 채널에서만 사용 가능
• 프로젝트에서 `cargo +nightly miri test` 실행
• 일부 외부 함수(syscall 등)는 실행 불가

• Miri is only available on nightly channel
• Run `cargo +nightly miri test` in your project
• Some external functions (syscalls, etc.) cannot run

UB 탐지 예제 UB Detection Example

fn main() {
unsafe {
let ptr = 0x1234 as *const i32;
println!("{}", *ptr); // UB!
}
}
$ cargo +nightly miri run
error: Undefined Behavior: dereferencing pointer failed:
0x1234 is not a valid pointer
--> src/main.rs:4:27
|
4 | println!("{}", *ptr);
| ^^^^ invalid pointer

Use-After-Free 탐지 Use-After-Free Detection

fn main() {
unsafe {
let ptr = Box::into_raw(Box::new(42));
drop(Box::from_raw(ptr));
println!("{}", *ptr); // Use after free!
}
}
$ cargo +nightly miri run
error: Undefined Behavior: pointer to alloc was dereferenced
after this allocation got freed
--> src/main.rs:5:27
|
5 | println!("{}", *ptr);
| ^^^^ use-after-free
