syn 크레이트 개요 syn Crate Overview

use syn::{
DeriveInput, // struct/enum 정의
Data, // 데이터 타입
Fields, // 필드 목록
Type, // 타입 정보
Attribute, // 어트리뷰트
Generics, // 제네릭 파라미터
};

⚡ 핵심 개념 ⚡ Key Concept

syn은 Rust 문법을 파싱하는 라이브러리
TokenStream → AST 변환
"full" 피처로 모든 Rust 문법 지원

syn is a Rust syntax parser
TokenStream → AST conversion
"full" feature supports all Rust syntax

quote 크레이트 개요 quote Crate Overview

use quote::{quote!, format_ident!};
// 코드 생성
let tokens = quote! {
impl #name {
pub fn new() -> Self {
#name { }
}
}
};

⚡ 핵심 개념 ⚡ Key Concept

quote는 코드 생성 라이브러리
#변수로 값 삽입
준인용(quasi-quotation) 지원

quote is a code generation library
#var for value interpolation
Supports quasi-quotation

보간법(Interpolation) Interpolation

#var #var

  • 단일 값 삽입
  • Single value
  • quote! { #name }

#(#var)* #(#var)*

  • 반복 삽입
  • Repetition
  • quote! { #(#fields)* }

#(#var),* #(#var),*

  • 쉼표로 구분
  • Comma separated
  • quote! { #(#args),* }

format_ident! format_ident!

  • 식별자 생성
  • Identifier creation
  • format_ident!("set_{}", name)

실전 예제 Real-world Example

let fields = match &input.data {
Data::Struct(s) => &s.fields,
_ => panic!("Only structs"),
};
let setters = fields.iter().map(|f| {
let name = &f.ident;
let ty = &f.ty;
let setter = format_ident!("set_{}", name.as_ref().unwrap());
quote! {
pub fn #setter(&mut self, val: #ty) -> &mut Self {
self.#name = val;
self
}
}
});
quote! { impl #name { #(#setters)* } }
