절차적 매크로의 3가지 유형 Three Types of Proc Macros

🎯 Derive 🎯 Derive

  • #[derive(MyTrait)]
  • 구조체/열거형에 적용
  • Applied to structs/enums
  • 트레이트 자동 구현
  • Auto trait impl

🏷️ Attribute 🏷️ Attribute

  • #[my_attr]
  • 모든 아이템에 적용
  • Applied to any item
  • 코드 변환/추가
  • Transform/add code

📦 Function-like 📦 Function-like

  • my_macro!(...)
  • 함수처럼 호출
  • Called like function
  • DSL 구현
  • DSL implementation

[lib]
proc-macro = true

proc-macro = true is required!

Derive 매크로 작성 Writing Derive Macros

use proc_macro::TokenStream;
use quote::quote!;
use syn::{parse_macro_input, DeriveInput};
#[proc_macro_derive(Builder)]
pub fn derive_builder(input: TokenStream) -> TokenStream {
let input = parse_macro_input!(input as DeriveInput);
let name = &input.ident;
let expanded = quote! {
impl #name {
pub fn builder() -> Builder { ... }
}
};
TokenStream::from(expanded)
}

Attribute 매크로 작성 Writing Attribute Macros

#[proc_macro_attribute]
pub fn route(
attr: TokenStream,
item: TokenStream
) -> TokenStream {
// attr = "GET", "/api/users"
// item = fn handler() { ... }
let input = parse_macro_input!(item as ItemFn);
let name = &input.sig.ident;
let expanded = quote! {
#input
inventory::submit!(
Route::new(#attr, #name)
);
};
TokenStream::from(expanded)
}

Function-like 매크로 Function-like Macros

#[proc_macro]
pub fn sql(input: TokenStream) -> TokenStream {
// sql! { SELECT * FROM users WHERE id = $1 }
let query = input.to_string();
// SQL 파싱 및 검증
let parsed = parse_sql(&query);
let expanded = quote! {
Query {
sql: #query,
params: vec![],
}
};
TokenStream::from(expanded)
}
