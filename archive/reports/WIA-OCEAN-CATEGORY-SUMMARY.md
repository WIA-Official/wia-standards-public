## 🚀 Quick Start

Each standard can be installed and used independently:

```bash
# Navigate to any standard
cd /home/user/wia-standards/standards/WIA-OCEAN-001-deep-sea-exploration

# Run installation
./install.sh

# Use CLI
./cli/deep-sea-exploration.sh status

# View documentation
open index.html

# Run simulator
open simulator/index.html

# Read ebook
open ebook/en/index.html  # English
open ebook/ko/index.html  # Korean
```
