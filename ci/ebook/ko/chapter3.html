3장: API 엔드포인트

3.1 REST API 개요

WIA-CI REST API는 인공와우 프로세서에 대한 포괄적인 제어를 제공하여 장치 구성, 오디오 스트리밍 및 실시간 모니터링을 가능하게 합니다.

기본 URL

https://api.wia.ci/v1

모든 엔드포인트는 Bearer 토큰을 통한 인증이 필요합니다.

3.2 장치 구성 API

장치 상태 조회

GET /processors/{processorId}/status

응답:
{
  "status": "online" | "offline" | "streaming",
  "battery": 85,              // 퍼센트
  "rfLink": 95,               // 링크 품질 %
  "lastSeen": "2025-12-20T10:30:00Z",
  "currentProgram": "music",
  "octaveEnhancement": true
}

설정 업데이트

PUT /processors/{processorId}/config

요청:
{
  "processor": {
    "volume": 85
  },
  "wiaExtension": {
    "octaveEnhancement": true,
    "musicMode": "on"
  }
}

응답:
{
  "status": "success",
  "appliedAt": "2025-12-20T10:35:00Z",
  "config": { /* 업데이트된 설정 */ }
}

3.3 옥타브 향상 API

옥타브 향상 활성화/비활성화

PUT /processors/{processorId}/wia/octave

요청:
{
  "enabled": true,
  "musicMode": "auto",
  "tfsEncoding": true,
  "harmonicPriority": true
}

응답:
{
  "status": "success",
  "octaveEnhancement": {
    "enabled": true,
    "activeAlgorithms": ["F0_estimation", "harmonic_analysis", "TFS_encoding"]
  }
}

3.4 전극 매핑 API

전극 임피던스 조회

GET /processors/{processorId}/electrodes/impedances

응답:
{
  "measured": "2025-12-20T10:30:00Z",
  "impedances": [
    { "electrode": 1, "value": 8500, "unit": "ohm", "status": "normal" },
    { "electrode": 2, "value": 9200, "unit": "ohm", "status": "normal" },
    // ... 22개 전극
  ],
  "summary": {
    "average": 8800,
    "min": 7200,
    "max": 10500,
    "outliers": []
  }
}

3.6 WebSocket API

실시간 스트림 연결

// WebSocket 연결
ws://api.wia.ci/v1/stream/{processorId}

// 메시지 유형
{
  "type": "audio" | "envelope" | "stimulation" | "octave" | "status",
  "timestamp": 1234567890,
  "data": { /* 페이로드 */ }
}

// 예: 옥타브 감지 메시지
{
  "type": "octave",
  "timestamp": 1234567890,
  "data": {
    "f0": 261.63,
    "octaveNumber": 4,
    "noteName": "C",
    "confidence": 0.92,
    "isMusic": true
  }
}
