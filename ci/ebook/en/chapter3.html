Chapter 3: API Endpoints

3.1 REST API Overview

The WIA-CI REST API provides comprehensive control over cochlear implant processors, enabling device configuration, audio streaming, and real-time monitoring.

Base URL

https://api.wia.ci/v1

All endpoints require authentication via Bearer token.

3.2 Device Configuration API

Get Device Status

GET /processors/{processorId}/status

Response:
{
  "status": "online" | "offline" | "streaming",
  "battery": 85,              // Percentage
  "rfLink": 95,               // Link quality %
  "lastSeen": "2025-12-20T10:30:00Z",
  "currentProgram": "music",
  "octaveEnhancement": true
}

Get Configuration

GET /processors/{processorId}/config

Response:
{
  "version": "1.0.0",
  "processor": {
    "sensitivity": 75,
    "volume": 80,
    "noiseReduction": "medium",
    "directionalMic": true
  },
  "mapping": {
    "strategy": "WIA-OCTAVE",
    "maxima": 10,
    "stimulationRate": 900,
    "pulseWidth": 25
  },
  "wiaExtension": {
    "octaveEnhancement": true,
    "musicMode": "auto",
    "tfsEncoding": true,
    "harmonicPriority": true
  }
}

Update Configuration

PUT /processors/{processorId}/config

Request:
{
  "processor": {
    "volume": 85
  },
  "wiaExtension": {
    "octaveEnhancement": true,
    "musicMode": "on"
  }
}

Response:
{
  "status": "success",
  "appliedAt": "2025-12-20T10:35:00Z",
  "config": { /* updated config */ }
}

3.3 Audio Streaming API

Start Audio Stream

POST /processors/{processorId}/stream/start

Request:
{
  "sampleRate": 16000,
  "frameSize": 256,
  "codec": "opus",
  "bitrateKbps": 32
}

Response:
{
  "streamId": "stream_abc123",
  "status": "started",
  "websocketUrl": "wss://stream.wia.ci/v1/stream_abc123"
}

Stop Audio Stream

POST /processors/{processorId}/stream/stop

Response:
{
  "status": "stopped",
  "totalFrames": 15420,
  "duration": "2m 34s"
}

3.4 Electrode Mapping API

Get Electrode Impedances

GET /processors/{processorId}/electrodes/impedances

Response:
{
  "measured": "2025-12-20T10:30:00Z",
  "impedances": [
    { "electrode": 1, "value": 8500, "unit": "ohm", "status": "normal" },
    { "electrode": 2, "value": 9200, "unit": "ohm", "status": "normal" },
    // ... 22 electrodes
  ],
  "summary": {
    "average": 8800,
    "min": 7200,
    "max": 10500,
    "outliers": []
  }
}

Update Electrode Map

PUT /processors/{processorId}/electrodes/map

Request:
{
  "electrodes": [
    {
      "electrode": 1,
      "enabled": true,
      "threshold": 150,      // T-level (μA)
      "comfort": 180,        // C-level (μA)
      "frequencyShift": 0    // Hz
    },
    // ... for all 22 electrodes
  ]
}

Response:
{
  "status": "success",
  "validated": true,
  "appliedAt": "2025-12-20T10:40:00Z"
}

3.5 WIA Octave Enhancement API

Enable/Disable Octave Enhancement

PUT /processors/{processorId}/wia/octave

Request:
{
  "enabled": true,
  "musicMode": "auto",      // "auto" | "on" | "off"
  "tfsEncoding": true,
  "harmonicPriority": true
}

Response:
{
  "status": "success",
  "octaveEnhancement": {
    "enabled": true,
    "activeAlgorithms": ["F0_estimation", "harmonic_analysis", "TFS_encoding"]
  }
}

Get Octave Performance Stats

GET /processors/{processorId}/wia/octave/stats

Response:
{
  "period": "last_30_days",
  "octaveRecognitionRate": 0.87,
  "musicDetectionAccuracy": 0.92,
  "harmonicPreservation": 0.85,
  "usage": {
    "totalHours": 120,
    "musicHours": 45,
    "speechHours": 75
  }
}

3.6 Telemetry API

Get Telemetry Data

GET /processors/{processorId}/telemetry

Response:
{
  "timestamp": "2025-12-20T10:30:00Z",
  "implantStatus": {
    "voltage": 3400,         // mV
    "current": 850,          // μA
    "temperature": 37.2,     // °C
    "compliance": true
  },
  "electrodeStatus": {
    "activeCount": 22,
    "shortCircuits": 0,
    "openCircuits": 0
  }
}

Get Neural Response Telemetry (NRT)

POST /processors/{processorId}/telemetry/nrt

Request:
{
  "electrode": 11,
  "amplitude": 200,         // μA
  "pulseWidth": 25          // μs
}

Response:
{
  "electrode": 11,
  "amplitude": 15.2,        // μV (response)
  "latency": 420,           // μs
  "waveform": [/* sampled response */],
  "quality": "good"
}

3.8 Session Management API

Start Session

POST /processors/{processorId}/session/start

Request:
{
  "config": {
    "mode": "music",
    "octaveEnhancement": true
  }
}

Response:
{
  "sessionId": "session_xyz789",
  "status": "connected",
  "implantInfo": {
    "manufacturer": "Cochlear",
    "model": "CI622",
    "serialNumber": "CI-2025-001234",
    "electrodeCount": 22
  }
}

End Session

POST /processors/{processorId}/session/end

Request:
{
  "sessionId": "session_xyz789",
  "reason": "user"
}

Response:
{
  "status": "disconnected",
  "duration": "1h 23m",
  "stats": {
    "framesSent": 298440,
    "framesDropped": 12
  }
}

3.9 WebSocket API

Real-time Stream Connection

// Connect to WebSocket
ws://api.wia.ci/v1/stream/{processorId}

// Message types
{
  "type": "audio" | "envelope" | "stimulation" | "octave" | "status",
  "timestamp": 1234567890,
  "data": { /* payload */ }
}

// Example: Octave detection message
{
  "type": "octave",
  "timestamp": 1234567890,
  "data": {
    "f0": 261.63,
    "octaveNumber": 4,
    "noteName": "C",
    "confidence": 0.92,
    "isMusic": true
  }
}

3.10 Batch Operations API

Batch Configuration Update

POST /processors/batch/config

Request:
{
  "processors": ["proc_001", "proc_002", "proc_003"],
  "config": {
    "wiaExtension": {
      "octaveEnhancement": true
    }
  }
}

Response:
{
  "status": "success",
  "updated": 3,
  "failed": 0,
  "results": [
    { "processorId": "proc_001", "status": "success" },
    { "processorId": "proc_002", "status": "success" },
    { "processorId": "proc_003", "status": "success" }
  ]
}

Rate Limits

Endpoint Type Rate Limit Burst
Configuration 60 req/min 10
Telemetry 120 req/min 20
Streaming Unlimited N/A
Analytics 30 req/min 5

3.12 Error Responses

Standard Error Format

{
  "error": {
    "code": "INVALID_CONFIG",
    "message": "T-level must be less than C-level",
    "details": {
      "electrode": 5,
      "tLevel": 200,
      "cLevel": 180
    },
    "timestamp": "2025-12-20T10:30:00Z"
  }
}

Common Error Codes

Code HTTP Status Description
PROCESSOR_NOT_FOUND 404 Processor ID does not exist
INVALID_CONFIG 400 Configuration validation failed
PROCESSOR_OFFLINE 503 Processor is not connected
RATE_LIMIT_EXCEEDED 429 Too many requests
UNAUTHORIZED 401 Invalid or missing auth token
