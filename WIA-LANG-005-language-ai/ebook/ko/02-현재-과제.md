## 데이터 수집의 현실적 어려움

### 디지털 데이터의 절대적 부족

저자원 언어가 직면한 가장 근본적인 문제는 **디지털 데이터의 부족**입니다. 현대 AI 모델은 방대한 양의 데이터를 필요로 하지만, 대부분의 위기 언어는 이를 충족할 수 없습니다.

#### 데이터 요구량 vs 실제 가용 데이터

| 작업 | 전통적 요구량 | 고자원 언어 가용량 | 저자원 언어 가용량 | 격차 |
|-----|------------|-----------------|-----------------|------|
| 기계 번역 | 수백만 문장 쌍 | 수억 문장 | 수천~수만 문장 | 1:10,000 |
| 음성 인식 | 1,000+ 시간 | 10,000+ 시간 | 1~10 시간 | 1:1,000 |
| 언어 모델 | 10GB+ 텍스트 | 100GB+ 텍스트 | 1~100MB 텍스트 | 1:1,000 |
| TTS | 20+ 시간 고품질 음성 | 100+ 시간 | 10분~1시간 | 1:1,200 |
| 개체명 인식 | 100,000+ 주석 | 수백만 주석 | 0~1,000 주석 | 1:10,000+ |


**해결책: 언어 균형 샘플링**
```python
def balanced_sampling(language_datasets, temperature=5):
    """온도 기반 샘플링으로 저자원 언어 비중 증가"""
    sampling_probs = {}
    for lang, dataset in language_datasets.items():
        size = len(dataset)
        # 온도로 크기 조정
        sampling_probs[lang] = size ** (1 / temperature)

    # 정규화
    total = sum(sampling_probs.values())
    for lang in sampling_probs:
        sampling_probs[lang] /= total

    return sampling_probs

# 예:
# 영어: 1억 문장 -> (100M)^(1/5) = 158
# 위기어: 1천 문장 -> (1000)^(1/5) = 3.98
# 비율: 158:4 (훨씬 균형잡힘!)
```

```python
            "연구비 확보"
        ]

        self.community_needs = [
            "언어 교육 도구",
            "세대 간 전승",
            "문화적 자긍심",
            "경제적 이익"
        ]

    def outcome(self):
        if self.researcher_goals != self.community_needs:
            # 목표 불일치
            trust = "파괴됨"
            data_access = "거부됨"
            sustainability = "없음"

            # 연구자는 떠나고, 공동체는 배신감
            return "실패"
```

## 핵심 요약

**주요 내용:**

1. **데이터 희소성**: 저자원 언어는 고자원 언어 대비 1:1,000 ~ 1:10,000 수준의 데이터만 가용하며, 품질도 열악합니다.

2. **기술적 장벽**: 형태론적 복잡성, 음운론적 다양성, 문자 체계 미지원, 평가 벤치마크 부재가 주요 과제입니다.

3. **모델링 과제**: 소규모 데이터에서의 과적합, 다국어 모델의 언어 간 간섭, 제로샷/퓨샷 학습의 한계를 극복해야 합니다.

4. **인프라 불평등**: 컴퓨팅 자원 부족과 전문가 부족이 저자원 언어 AI 개발을 제약합니다.

5. **사회문화적 도전**: 언어 공동체와의 신뢰 구축, 언어 태도 개선, 동기 부여가 기술만큼 중요합니다.

6. **윤리적 고려**: 기술 식민주의 방지, 데이터 주권 보장, 살아있는 언어 유지가 핵심입니다.
