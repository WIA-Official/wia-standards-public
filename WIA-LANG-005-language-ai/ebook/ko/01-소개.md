# 제1장: 언어 AI 기술의 이해

## 인공지능과 언어의 만남

인공지능(AI) 기술은 21세기의 가장 혁명적인 발전 중 하나이며, 특히 자연어 처리(NLP) 분야는 인간과 기계 간 소통의 새로운 지평을 열었습니다. 그러나 이러한 기술의 혜택은 주로 영어, 중국어, 스페인어와 같은 고자원 언어에 집중되어 있으며, 세계 7,000개 언어 중 대부분은 AI 기술의 사각지대에 놓여 있습니다.

## 언어 AI의 현주소

### 고자원 언어 vs 저자원 언어

현재 언어 AI 기술은 심각한 불균형을 보이고 있습니다:

| 언어 범주 | 언어 수 | AI 모델 지원 | 훈련 데이터 | 예시 |
|---------|--------|------------|-----------|------|
| 초고자원 | 약 10개 | 매우 풍부 | 수백 GB ~ TB | 영어, 중국어, 스페인어 |
| 고자원 | 약 50개 | 풍부 | 수십 GB | 프랑스어, 독일어, 일본어 |
| 중자원 | 약 200개 | 제한적 | 수 GB | 태국어, 터키어, 베트남어 |
| 저자원 | 약 1,000개 | 매우 제한적 | 수백 MB | 아이누어, 나바호어, 체로키어 |
| 극저자원 | 약 5,740개 | 거의 없음 | 수 MB 이하 | 대부분의 원주민 언어 |

### GPT와 대규모 언어 모델의 시대

**GPT-4, Claude, Gemini**와 같은 대규모 언어 모델(LLM)의 등장으로 AI는 인간 수준의 언어 이해와 생성 능력을 보여주고 있습니다. 그러나:

- GPT-3는 약 500억 개의 토큰으로 훈련되었지만, 그 중 92%가 영어
- 나머지 8%도 주로 유럽 주요 언어
- 위기 언어는 사실상 완전히 배제

```
대규모 언어 모델 훈련 데이터 구성
┌────────────────────────────────────┐
│ 영어: 92%  ████████████████████████│
│ 중국어: 3% ███                     │
│ 기타 주요 언어: 4.5% ████          │
│ 위기 언어: 0.5% ▌                  │
└────────────────────────────────────┘
```

## 저자원 언어 NLP의 도전

### 데이터 희소성 문제

저자원 언어가 직면한 가장 큰 문제는 **데이터 부족**입니다:

**텍스트 데이터 부족**:
- 디지털화된 문헌의 부재
- 온라인 콘텐츠의 극소량
- 표준화된 철자법의 부재
- 문자 체계의 디지털 미지원

**음성 데이터 부족**:
- 녹음된 음성 자료의 부족
- 전사(transcription)된 자료의 부재
- 다양한 화자 데이터 부족
- 녹음 품질 문제

**주석(Annotation) 부족**:
- 품사 태깅 데이터 없음
- 구문 분석 트리 부재
- 의미 주석 부재
- 전문가 부족으로 주석 작업 어려움

### 기술적 장벽

| 과제 영역 | 구체적 문제 | 영향 |
|----------|-----------|------|
| 토큰화 | 형태소 복잡성, 띄어쓰기 불명확 | 전처리 실패 |
| 임베딩 | 어휘 부족으로 벡터 표현 불가 | 의미 파악 불가 |
| 모델 학습 | 훈련 데이터 부족 | 성능 저하 |
| 평가 | 벤치마크 부재 | 진척도 측정 불가 |
| 배포 | 인프라 부족 | 실사용 어려움 |

## 전이 학습: 게임 체인저

### 전이 학습이란?

전이 학습(Transfer Learning)은 한 작업에서 학습한 지식을 다른 작업에 적용하는 기법입니다. 저자원 언어 NLP에서 전이 학습은 혁명적입니다:

```python
# 전이 학습의 기본 개념
class TransferLearning:
    def __init__(self):
        # 1단계: 고자원 언어로 사전 훈련
        self.pretrained_model = train_on_high_resource_languages(
            languages=['en', 'zh', 'es', 'fr', 'de'],
            data_size='100GB'
        )

    def adapt_to_low_resource(self, target_language, small_data):
        # 2단계: 저자원 언어로 미세 조정
        adapted_model = self.pretrained_model.fine_tune(
            language=target_language,
            data=small_data,  # 단 몇 MB만으로도 가능!
            epochs=10
        )
        return adapted_model
```

### 다국어 모델의 힘

**mBERT, XLM-R, mT5**와 같은 다국어 사전 훈련 모델은:

- 100개 이상의 언어에서 동시 학습
- 언어 간 공통 패턴 학습
- 제로샷(Zero-shot) 또는 퓨샷(Few-shot) 학습 가능
- 극저자원 언어에도 적용 가능

```
다국어 모델의 지식 전이
┌──────────────────────────────────────┐
│   고자원 언어 (Source)                 │
│   ├── 영어: 문법, 의미, 문맥           │
│   ├── 중국어: 형태소, 어순             │
│   └── 스페인어: 어미 변화              │
│              ↓ 전이                    │
│   저자원 언어 (Target)                 │
│   └── 아이누어: 소량 데이터로도        │
│       고품질 모델 학습 가능             │
└──────────────────────────────────────┘
```

### 2. 음성 인식 (ASR)

**과제**:
- 훈련 데이터 부족 (수백 시간 필요 → 수 시간만 가능)
- 방언 및 화자 변이
- 배경 소음 처리

**혁신 기술**:
- **Self-supervised Learning**: Wav2Vec 2.0, HuBERT
- **Cross-lingual Transfer**: 다국어 음성 모델
- **Data Augmentation**: 음성 합성으로 데이터 증강

```python
# Wav2Vec 2.0 기반 저자원 ASR
from transformers import Wav2Vec2ForCTC, Wav2Vec2Processor

# 사전 훈련된 다국어 모델 로드
processor = Wav2Vec2Processor.from_pretrained("facebook/wav2vec2-xls-r-300m")
model = Wav2Vec2ForCTC.from_pretrained("facebook/wav2vec2-xls-r-300m")

# 소량의 타겟 언어 데이터로 미세 조정
# 단 1시간의 전사된 음성으로도 가능!
model.fine_tune(
    language="ainu",
    audio_data=ainu_audio,  # 1-10 hours
    transcriptions=ainu_text
)
```

### 3. 텍스트 음성 변환 (TTS)

**목표**: 자연스러운 음성 합성

**기술 스택**:
- **Tacotron 2**: 텍스트 → 멜 스펙트로그램
- **WaveGlow/HiFi-GAN**: 멜 스펙트로그램 → 음성
- **Multi-speaker Models**: 화자 적응

**저자원 언어 적용**:
```yaml
tts_pipeline:
  stage1_text_to_mel:
    model: Tacotron2
    pretrain_languages: [en, es, fr]  # 다국어 사전 훈련
    finetune_language: target_endangered_language
    required_audio: 30_minutes  # 전통적 방법의 1/10

  stage2_mel_to_audio:
    model: HiFi-GAN
    universal_vocoder: true  # 언어 독립적
```

### 4. 언어 모델링과 텍스트 생성

**응용**:
- 자동 완성
- 문법 교정
- 텍스트 생성
- 질의응답

**GPT 스타일 모델 적용**:
```python
# 저자원 언어를 위한 소형 GPT
class SmallGPTForEndangered:
    def __init__(self):
        self.model = GPT2LMHeadModel(
            vocab_size=5000,      # 소규모 어휘
            n_positions=512,      # 짧은 문맥
            n_layer=6,            # 경량 아키텍처
            n_head=8
        )

    def train_with_limited_data(self, corpus):
        # 데이터 증강 기법 활용
        augmented_corpus = [
            self.back_translation(text),
            self.paraphrase(text),
            self.context_substitution(text)
        ] for text in corpus

        self.model.train(augmented_corpus)
```
