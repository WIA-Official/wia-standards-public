5.1 RESTful API 설계

WIA-ROB-014는 OpenAPI 3.0 스펙을 기반으로 RESTful API를 정의합니다.

5.1.1 API 엔드포인트 구조

리소스 엔드포인트 메서드 설명
Pets /v1/pets GET 모든 반려동물 목록 조회
/v1/pets POST 새 반려동물 등록
/v1/pets/{petId} GET 특정 반려동물 조회
/v1/pets/{petId} PATCH 반려동물 정보 수정
/v1/pets/{petId} DELETE 반려동물 삭제
Activities /v1/pets/{petId}/activities GET 활동 기록 조회
/v1/pets/{petId}/activities POST 활동 기록 추가
/v1/pets/{petId}/activities/{activityId} GET 특정 활동 조회
Health /v1/pets/{petId}/health GET 건강 데이터 조회
/v1/pets/{petId}/health POST 건강 측정 기록
Devices /v1/devices GET 등록된 디바이스 목록
/v1/devices/{deviceId} GET 디바이스 정보 조회
/v1/devices/{deviceId}/control POST 디바이스 제어

5.7.2 표준 에러 코드

HTTP 상태 에러 코드 설명 재시도
400 INVALID_REQUEST 요청 데이터 유효성 오류 수정 후 재시도
401 UNAUTHORIZED 인증 실패 재인증 후 재시도
403 FORBIDDEN 권한 없음 재시도 불가
404 RESOURCE_NOT_FOUND 리소스 없음 재시도 불가
409 CONFLICT 데이터 충돌 확인 후 재시도
429 RATE_LIMIT_EXCEEDED 요청 한도 초과 대기 후 재시도
500 INTERNAL_ERROR 서버 내부 오류 자동 재시도
503 SERVICE_UNAVAILABLE 서비스 일시 중단 대기 후 재시도

5.10 SDK 및 클라이언트 라이브러리

개발자를 위한 공식 SDK를 제공합니다.

5.10.1 TypeScript SDK 예제

import { WIARob014Client } from '@wia/rob-014-sdk';

const client = new WIARob014Client({
  apiKey: process.env.WIA_API_KEY,
  region: 'us-east-1'
});

// 반려동물 조회
const pet = await client.pets.get('pet-a1b2c3d4e5f67890a1b2c3d4e5f67890');
console.log(`반려동물 이름: ${pet.name}`);

// 급식 실행
const feedResult = await client.devices.control('device-feeder-001', {
  action: 'dispense',
  parameters: {
    amount: { value: 150, unit: 'grams' },
    petId: pet.petId
  }
});

// 건강 요약 조회
const healthSummary = await client.health.getSummary(pet.petId, {
  period: '7d'
});
console.log(`건강 점수: ${healthSummary.overallHealth.score}`);
