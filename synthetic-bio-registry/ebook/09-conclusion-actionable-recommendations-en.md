### For Researchers and Users

**Best Practices for Using Registries:**

**1. Start with High-Quality Parts**
```
✓ Filter by quality tier (prefer Tier 2+)
✓ Check for multiple successful uses
✓ Review characterization data carefully
✓ Read experience pages
✓ Start with well-known parts for critical applications
✓ Use established collections (Anderson promoters, etc.)
```

**2. Contribute Back to the Community**
```
✓ Submit new designs to registry
✓ Share characterization data from your experiments
✓ Document successes and failures
✓ Provide detailed protocols
✓ Respond to questions from other users
✓ Participate in community standards development
```

**3. Cite Registry Parts Properly**
```
In publications:
"We used promoter BBa_J23100 from the iGEM Registry
(http://parts.igem.org/Part:BBa_J23100)."

Include:
- Part ID
- Registry source
- URL
- Access date
- Version (if applicable)

This enables reproducibility and credits contributors.
```

**4. Verify Before Critical Use**
```
For important applications:
☐ Re-sequence parts to confirm sequence
☐ Characterize in your specific context
☐ Include appropriate controls
☐ Test multiple biological replicates
☐ Don't assume part will work identically in your system
```

**5. Engage with Registry Development**
```
☐ Provide feedback on usability
☐ Request needed features
☐ Participate in surveys and studies
☐ Join advisory committees if invited
☐ Advocate for sustained funding
```

## Appendices

### Appendix A: Glossary of Terms

**BioBrick:** DNA sequence with standardized prefix and suffix for modular assembly

**Chassis:** Host organism engineered to contain synthetic genetic circuits

**Characterization:** Experimental determination of part behavior and properties

**SBOL:** Synthetic Biology Open Language, data format for genetic designs

**Quality Tier:** Classification of parts by characterization completeness (0-4)

**OpenMTA:** Open Material Transfer Agreement for sharing biological materials

**Interoperability:** Ability of different systems to exchange and use information

**RBS:** Ribosome Binding Site, controls translation of mRNA to protein

**CDS:** Coding Sequence, DNA encoding a protein

**Registry API:** Application Programming Interface for programmatic registry access
