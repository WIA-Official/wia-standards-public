# 2장: BioBricks 및 표준화된 부품

## 개요

BioBricks는 복잡한 유전자 시스템을 구축하기 위해 쉽게 결합할 수 있는 표준화된 생물학적 부품의 기본 개념을 나타냅니다. 이 장에서는 BioBricks 표준, 조립 방법, 부품 계층 구조, 설계 원칙 및 이 혁신적인 생물학적 엔지니어링 접근 방식을 중심으로 구축된 커뮤니티를 탐구합니다.

## 2.1 BioBricks 표준

### 기원 및 철학

BioBricks 개념은 2000년대 초반 MIT에서의 대화, 특히 컴퓨터 과학자에서 생물학자로 전환한 Tom Knight의 작업을 통해 등장했습니다. Knight는 생물학이 컴퓨터 혁명을 가능하게 한 것과 동일한 종류의 표준화가 필요하다는 것을 인식했습니다.

**핵심 통찰**: 집적 회로가 표준화된 트랜지스터로 구축되고 표준 라이브러리에서 소프트웨어가 구축되는 것처럼 생물학적 시스템은 표준화된 유전자 부품에서 엔지니어링될 수 있습니다.

**창립 원칙**:
1. **표준화**: 부품은 일관된 인터페이스를 가져야 함
2. **모듈성**: 부품은 상호교환 및 결합 가능해야 함
3. **추상화**: 사용자는 내부 복잡성을 이해할 필요 없음
4. **개방성**: 부품 및 설계는 자유롭게 공유되어야 함
5. **측정**: 부품은 표준 메트릭으로 특성화되어야 함

### 원본 BioBricks 표준 (RFC 10)

BioBricks Foundation은 원본 표준을 정의하는 RFC 10 (Request for Comments 10)을 발표했습니다:

**주요 특징**:
- 표준 접두사 및 접미사 서열
- 4개의 제한 효소 부위: EcoRI, XbaI, SpeI, PstI
- 3-항생제 조립 방법 (3A 조립)
- 부품 서열 내에 제한 부위 없음
- 멱등 조립 (흉터 서열은 향후 조립과 호환)

**표준 부품 구조**:
```
[Prefix]---[Part Sequence]---[Suffix]

Prefix:  5'- GAATTC GCGGCCGC TCTAGA G -3'
             EcoRI  NotI     XbaI

Suffix:  5'- T ACTAGT GCGGCCGC CTGCAG -3'
             SpeI   NotI     PstI
```

**조립 메커니즘**:
두 BioBricks가 결합될 때:
1. 상류 부품을 EcoRI 및 SpeI로 소화
2. 하류 부품을 XbaI 및 PstI로 소화
3. EcoRI 및 PstI로 절단된 벡터에 함께 결찰
4. 결과는 사이에 흉터 서열이 있는 두 부품을 모두 포함

**흉터 서열**:
```
XbaI + SpeI 결찰 생성:
5'- T ACTAGA G -3'
```

이 흉터는:
- XbaI 또는 SpeI에 의해 인식되지 않음
- 추가 조립을 방해하지 않음
- 부품 사이에 8개의 뉴클레오티드 추가
- 코딩 서열에 영향을 줄 수 있음 (설계 시 고려해야 함)

### 표준화가 중요한 이유

**표준 없이**:
- 각 부품에는 맞춤형 클로닝 전략 필요
- 다른 출처의 부품을 결합하기 어려움
- 각 조합에 대한 호환성 확인 필요
- 시간 소모적인 제한 매핑
- 높은 오류율

**BioBricks와 함께**:
- 확인 없이 두 부품 결합 가능
- 자동화된 조립 계획
- 모든 출처의 부품 혼합
- 예측 가능한 결과
- 빠른 프로토타이핑

**USB 표준과의 유사성**:
USB가 컴퓨터 주변 장치를 표준화한 것처럼:
- 모든 USB 장치가 모든 USB 포트에서 작동
- 내부 프로토콜을 이해할 필요 없음
- 호환 가능한 장치의 생태계 가능
- 혁신을 극적으로 가속화

BioBricks는 생물학에 대해 동일한 것을 목표로 합니다.

## 2.2 조립 방법

### 3-항생제 조립 (3A 조립)

정식 BioBricks 조립 방법:

**구성 요소**:
- 항생제 저항성 A가 있는 플라스미드의 상류 BioBrick
- 저항성 B가 있는 플라스미드의 하류 BioBrick
- 저항성 C가 있는 목적지 벡터
- E. coli 적격 세포

**프로토콜**:
1. **부품 소화**:
   - 상류: EcoRI + SpeI (1-2시간)
   - 하류: XbaI + PstI (1-2시간)
   - 벡터: EcoRI + PstI (1-2시간)

2. **겔 정제**:
   - 아가로스 겔에서 실행
   - 올바른 밴드 추출
   - DNA 정제 (30-60분)

3. **결찰**:
   - 정제된 3개 단편 혼합
   - T4 DNA 리가아제 추가
   - 16°C에서 밤새 또는 실온에서 2시간 배양

4. **형질전환**:
   - 적격 세포에 결찰 추가
   - 열충격 또는 전기천공
   - 성장 배지에서 회복 (1시간)

5. **선택**:
   - 항생제 C가 있는 배지에 플레이팅
   - 밤새 배양
   - 조립된 구조물이 있는 세포만 성장

**총 시간**: 2-3일
**성공률**: 최적화된 프로토콜로 70-90%

**장점**:
- 양성 선택 (조립된 구조물만 저항성 C 보유)
- 모든 BioBricks 결합 가능
- 특수 장비 필요 없음
- 잘 확립된 프로토콜

**단점**:
- 다른 저항성을 가진 3개의 다른 플라스미드 필요
- 시간 소모적인 겔 정제
- 여러 단계로 오류 가능성 증가
- 흉터 서열이 기능에 영향을 줄 수 있음

### 2-항생제 조립 (2A 조립)

2개의 저항성만 사용하는 단순화된 버전:

**방법**:
3A와 유사하지만 목적지 벡터가 하나의 입력과 동일한 저항성 보유

**장점**:
- 필요한 플라스미드 변형 수 감소
- 약간 더 간단함

**단점**:
- 더 높은 배경 (일부 절단되지 않은 벡터 생존)
- 신중한 최적화 필요

### Gibson 조립 대안

BioBricks와 구체적으로 관련은 없지만 Gibson 조립이 인기를 얻었습니다:

**메커니즘**:
- 겹치는 서열 사용 (15-40 bp)
- 동시에 소화, 어닐링 및 결찰
- 단일 튜브, 등온 반응 (50°C, 15-60분)

**장점**:
- 매우 빠름 (1시간 반응)
- 제한 부위 필요 없음
- 동시에 여러 단편 조립 가능
- 높은 효율

**단점**:
- 표준화되지 않음 (각 조립에 대한 겹침 설계 필요)
- 더 비싼 시약
- 반복 서열 문제 가능

**BioBricks + Gibson 하이브리드**:
일부 팀은 표준화를 위해 BioBricks를 사용하지만 실제 조립에는 Gibson을 사용:
- BioBricks 원칙으로 부품 설계
- PCR을 사용하여 Gibson 겹침 추가
- Gibson 방법으로 조립
- 두 접근 방식의 장점

### Golden Gate 조립

Type IIS 제한 효소를 사용하는 또 다른 인기 있는 대안:

**메커니즘**:
- Type IIS 효소 (BsaI, BsmBI)는 인식 부위 외부에서 절단
- 맞춤형 오버행 생성
- 순서대로 조립 설계 가능
- 단일 튜브, 1단계 반응

**예제**:
```
Part 1: [BsaI]--ATCG--[Gene]--GCTA--[BsaI]
Part 2: [BsaI]--GCTA--[Gene]--TTAG--[BsaI]
Part 3: [BsaI]--TTAG--[Gene]--ATCG--[BsaI]

BsaI 소화 후:
      ATCG--[Gene1]--GCTA
                     GCTA--[Gene2]--TTAG
                                    TTAG--[Gene3]--ATCG

결찰하여 형성: [Gene1]--[Gene2]--[Gene3]
```

**장점**:
- 흉터 없는 조립
- 하나의 반응으로 여러 단편
- 방향성 조립
- 매우 높은 효율 (>95%)

**단점**:
- 내부 BsaI/BsmBI 부위 제거 필요
- 더 많은 사전 설계 필요
- BioBricks보다 유연성 낮음

**MoClo (Modular Cloning)**:
표준화된 Golden Gate 시스템:
- 계층적 조립 (Level 0, 1, 2...)
- 정의된 부품 유형 및 위치
- 식물 합성생물학에서 널리 채택

## 2.3 부품 유형 및 계층 구조

### 기본 부품 유형

BioBricks는 생물학적 기능별로 분류됩니다:

#### 프로모터
전사 개시 제어:

**구성 프로모터**:
- 항상 활성
- 다양한 강도 (약함, 중간, 강함)
- 예: Anderson 라이브러리의 J23100 시리즈
- 일정한 발현이 필요할 때 사용

**유도 가능한 프로모터**:
- 특정 신호에 의해 활성화
- 예:
  - lac 프로모터 (IPTG 유도)
  - ara 프로모터 (아라비노스 유도)
  - tet 프로모터 (aTc 유도)
  - T7 프로모터 (T7 중합효소 필요)
- 시간적 제어 허용

**특성화 메트릭**:
- 기본 발현 수준
- 최대 유도 수준
- 유도 배수 변화
- 반응 시간
- 용량-반응 곡선

#### RBS (리보솜 결합 부위)
번역 개시 제어:

**특성**:
- 짧은 서열 (10-30 bp)
- 일반적으로 Shine-Dalgarno 서열 포함 (AGGAGG)
- 시작 코돈 상류 ~5-10 bp에 위치

**강도 범위**:
- 약한 RBS: 0.01-0.1x (미세 조정, 독성 유전자)
- 중간 RBS: 0.1-1x (일반적 사용)
- 강한 RBS: 1-10x (높은 발현)

**설계 도구**:
- Salis Lab RBS Calculator
- 서열에서 번역 속도 예측
- 원하는 강도의 합리적 설계 허용

**예제 RBS**:
```
강함: 5'- AAAAAAAAAAAAAAGAGGAGAAA -3'
중간: 5'- AAAGAGGAGAAA -3'
약함:   5'- AGAGAGAAA -3'
        시작 코돈 → ATG
```

#### 코딩 서열 (CDS)
단백질을 암호화하는 유전자:

**카테고리**:
- **리포터**: 형광 단백질 (GFP, RFP), 발광
- **선택 마커**: 항생제 저항성 유전자
- **효소**: 대사, 합성 경로
- **조절 단백질**: 전사 인자, 재조합효소
- **구조 단백질**: 스캐폴드, 앵커

**일반적인 리포터**:
- **GFP** (녹색 형광 단백질): 표준 리포터, 빠른 성숙
- **RFP** (빨간색): GFP와 최소 스펙트럼 겹침
- **CFP/YFP** (시안/노란색): 추가 색상
- **mCherry**: 단량체 빨간색, 안정적
- **LuxAB**: 발광, 여기 필요 없음

**고려사항**:
- 숙주 생물체에 대한 코돈 최적화
- 내부 제한 부위 제거
- 2차 구조 방지
- 융합 태그 (His, FLAG 등)

#### 종결자
전사 중지 및 mRNA 방출:

**유형**:
- **고유 (rho-독립적)**: 헤어핀 구조 + poly-U
- **Rho-의존적**: Rho 단백질 필요

**강도**:
- read-through 백분율로 측정
- 강한 종결자: <5% read-through
- 약한 종결자: 10-50% read-through

**일반적인 BioBrick 종결자**:
- **B0015** (이중 종결자): 매우 강함, 널리 사용
- **B0010, B0012**: 개별 종결자
- **B0014**: 중간 강도

**강한 종결자의 구조**:
```
5'- [Coding Sequence]---[GC-rich stem]---[Loop]---[Stem complement]---TTTTTT -3'
                         헤어핀 형성 → 중합효소 일시 정지 → 떨어짐
```

#### 복합 부품
기본 부품의 조합:

**일반적인 조합**:
- **전사 단위**: 프로모터 + RBS + CDS + 종결자
- **리포터 장치**: 프로모터 + RBS + GFP + 종결자
- **발현 카세트**: 종결자가 있는 여러 유전자
- **조절 장치**: 센서 + 반응

**예 - 유도 가능한 GFP 발현**:
```
[Plac]--[RBS]--[GFP]--[Terminator]

IPTG 없이: 낮음/GFP 없음
IPTG와 함께: 높은 GFP 발현
```

### 부품 계층 구조 수준

BioBricks는 계층적 수준으로 구성됩니다:

**레벨 0: 기본 부품**
- 단일 생물학적 기능
- 더 이상 세분화할 수 없음
- 예: 단일 프로모터, 하나의 RBS, 하나의 CDS

**레벨 1: 복합 부품**
- 기본 부품의 조합
- 기능적 장치
- 예: 발현 카세트, 리포터 구조물

**레벨 2: 장치**
- 여러 복합 부품
- 특정 작업 수행
- 예: 바이오센서, 토글 스위치

**레벨 3: 시스템**
- 함께 작동하는 여러 장치
- 복잡한 동작
- 예: 대사 경로, 유전 회로

**추상화 이점**:
- 적절한 수준에서 작업 가능
- 하위 수준을 이해할 필요 없음
- 모듈식 설계 가능
- 팀 협업 촉진

## 2.4 설계 원칙

### 모듈성 및 조합 가능성

**모듈식 설계**:
각 부품은 다음을 수행해야 합니다:
- 잘 정의된 기능 보유
- 맥락과 독립적으로 작동 (가능한 한)
- 표준 인터페이스 보유
- 대안과 교환 가능

**조합 가능성**:
부품은 예측 가능하게 결합되어야 합니다:
- 원치 않는 상호작용 없음
- 가산 또는 곱셈 효과
- 여러 부품으로 확장 가능

**과제**:
생물학은 매우 맥락 의존적입니다:
- 프로모터 강도는 숙주, 배지, 온도에 따라 다름
- RBS 효과는 주변 서열에 따라 다름
- 유전자 발현은 성장 속도에 영향
- 자원 경쟁 (중합효소, 리보솜, 대사산물)

**조합 가능성 향상 전략**:
- 맥락 효과를 줄이는 절연체 서열
- 균형 잡힌 발현 수준
- 별도의 조절 네트워크
- 직교 시스템 (T7, T3 중합효소)

### 직교성

**정의**: 부품이 서로 또는 숙주 프로세스를 방해하지 않음

**예**:
- **직교 프로모터**: T7 프로모터는 T7 중합효소에 의해서만 인식
- **직교 리보솜**: 수정된 리보솜 + 일치하는 RBS
- **직교 종결자**: 숙주 유전자와의 상호작용 최소화

**가치**:
- 예측 가능한 동작
- 누화 감소
- 복잡한 회로 가능
- 신뢰성 증가

### 추상화 계층 구조

**원칙**: 하위 수준에서 복잡성 숨기기

**DNA 수준**:
- 개별 뉴클레오티드
- 제한 부위
- 2차 구조

**부품 수준**:
- 기능적 요소 (프로모터, RBS)
- 특성화된 동작
- 표준 인터페이스

**장치 수준**:
- 다중 부품 구조물
- 특정 기능 (감지, 작동)
- 성능 메트릭

**시스템 수준**:
- 여러 장치
- 창발적 동작

### 특성화 표준

**프로모터 특성화**:

- 조건: 배지, 온도, 성장 단계

- 맥락 의존성

**완전한 특성화**:
- 여러 조건
- 관련 있는 경우 여러 숙주
- 시간 경과 측정
- 집단 대 단일 세포 데이터

**보고 표준**:
- 프로토콜 문서
- 원시 데이터 가용성
- 통계 분석
- 대조 실험

## 2.6 RFC 표준

### Request for Comments (RFC) 시스템

인터넷 엔지니어링에서 차용:

**목적**:
- 기술 표준 문서화
- 커뮤니티 피드백 요청
- 표준의 진화 추적
- 역사적 기록 유지

**프로세스**:
1. 저자가 RFC 초안 작성
2. 커뮤니티 검토 및 의견
3. 피드백에 따른 수정
4. 최종 또는 업데이트된 버전 발표

**주요 BioBricks RFC**:

**RFC 10** - 원본 BioBricks 표준:
- EcoRI-XbaI-SpeI-PstI 조립
- 접두사/접미사 정의
- iGEM Registry의 기초

**RFC 12** - 단백질 도메인 벽돌:
- 단백질 부품 조립 표준
- 읽기 프레임 유지
- 융합 단백질 가능

**RFC 21** - BglBricks:
- RFC 10의 대안
- BglII 및 BamHI 사용
- 다른 흉터 서열

**RFC 23** - 실버 표준:
- 단순화된 표준
- 부품 사이의 단일 절단 부위
- 구현 더 쉬움

**RFC 25** - 폴리펩티드 조립:
- 다중 도메인 단백질 조립
- 펩티드 링커
- 흉터 없는 조립

**RFC 30** - SBOL Visual:
- 유전 회로에 대한 그래픽 표기법
- 부품에 대한 표준 글리프
- 명확한 커뮤니케이션 가능

### 표준의 진화

**긴장**: 원본 BioBrick 표준에는 제한 사항이 있습니다:
- 흉터 서열이 일부 조립을 방해
- 박테리아 부품으로 제한
- 제한 부위가 문제가 될 수 있음

**대안 표준**:
- Golden Gate (Type IIS 기반)
- Gibson 조립
- SEVA (Standard European Vector Architecture)
- MoClo (Modular Cloning)

**수렴**:
- 다른 요구에 대한 다른 표준
- 상호운용성 노력
- 범용 교환 형식으로서의 SBOL
- 실용적 다원주의

## 2.7 실용 예제

### 예제 1: 간단한 리포터 구축

**목표**: lac 프로모터 제어 하에 GFP 발현

**필요한 부품**:
- BBa_R0010: lac 프로모터
- BBa_B0034: 강한 RBS
- BBa_E0040: GFP 코딩 서열
- BBa_B0015: 이중 종결자
- pSB1C3: 표준 BioBricks 플라스미드

**조립**:
1. R0010 + B0034 결합 → Part A (프로모터 + RBS)
2. E0040 + B0015 결합 → Part B (GFP + 종결자)
3. Part A + Part B 결합 → 최종 구조물
4. E. coli로 형질전환

**예상 동작**:
- IPTG 없이: 낮음/형광 없음 (억제됨)
- IPTG와 함께: 녹색 형광 (유도됨)

**특성화**:
- 다양한 IPTG 농도에서 형광 측정
- 용량-반응 곡선 플롯
- 최대 발현, EC50 결정

### 예제 2: 다중 유전자 경로

**목표**: 리코펜 생합성 경로 발현 (빨간색 색소 생성)

**필요한 부품**:
- 프로모터 (구성적, 예: J23100)
- 각 유전자에 대한 RBS
- crtE, crtB, crtI 유전자 (리코펜 경로)
- 각 유전자에 대한 종결자

**조립 전략**:
1. 3개의 발현 카세트 생성:
   - [Promoter]--[RBS]--[crtE]--[Terminator]
   - [Promoter]--[RBS]--[crtB]--[Terminator]
   - [Promoter]--[RBS]--[crtI]--[Terminator]

2. 카세트를 순차적으로 또는 다중 유전자 조립 사용하여 결합

3. E. coli로 형질전환

**예상 결과**:
- 콜로니가 리코펜 생산으로 인해 빨간색/주황색으로 변함
- 가시적 색소 축적

**응용**:
- 대사 공학 교육
- 가치 있는 화합물 생산
- 다중 유전자 조립 시연

### 예제 3: 유전자 토글 스위치

**목표**: 쌍안정 스위치 구축 (메모리 요소)

**설계** (Gardner et al., Nature 2000):
- 유전자 A가 유전자 B를 억제
- 유전자 B가 유전자 A를 억제
- 외부 신호가 상태를 전환할 수 있음

**BioBricks 구현**:
- TetR 억제자 + pTet 프로모터
- LacI 억제자 + pLac 프로모터
- 유도자: aTc (Tet용), IPTG (Lac용)

**동작**:
- 상태 1에서 시작 (A 높음, B 낮음)
- IPTG 추가: 상태 2로 전환 (A 낮음, B 높음)
- IPTG 제거: 상태 2 유지 (쌍안정성)
- aTc 추가: 상태 1로 다시 전환

**교육적 가치**:
- 유전자 메모리 시연
- 간단한 부품에서 복잡한 동작
- 합성생물학의 기초

## 2.8 제한 사항 및 진화

### 알려진 제한 사항

**맥락 의존성**:
- 부품이 다른 맥락에서 항상 동일하게 작동하지 않음
- 숙주 균주 효과
- 복제 수 변동
- 대사 부담

**흉터 서열**:
- XbaI-SpeI 결찰에서 8bp 흉터
- 코딩 서열을 방해할 수 있음
- 요소 사이의 간격에 영향을 줄 수 있음
- 모든 응용 프로그램에 이상적이지 않음

**제한 부위**:
- 내부 서열에서 EcoRI, XbaI, SpeI, PstI 제거 필요
- 일부 자연 서열 제한
- 침묵 돌연변이가 기능에 영향을 줄 수 있음

**측정 표준**:
- 절대 단위 어려움
- 실험실 간 변동
- 기기 차이
- 참조 표준 필요

### 현대적 개선

**더 나은 조립 방법**:
- 흉터 없는 조립을 위한 Golden Gate
- 빠른 프로토타이핑을 위한 Gibson
- 대형 구조물을 위한 효모 조립
- CRISPR 기반 염색체 통합

**개선된 특성화**:
- 집단 데이터를 위한 유세포 분석
- 단일 세포 동역학을 위한 현미경
- 전사체학을 위한 RNA-seq
- 단백질 수준을 위한 프로테오믹스

**계산 도구**:
- 자동화된 설계 (Cello, Clotho)
- 시뮬레이션 (iBioSim, TinkerCell)
- 최적화 (유전 알고리즘)
- 예측을 위한 머신러닝

**참조 표준**:
- 형광 표준 (비드, 플레이트)
- 절대 단위 (세포당 분자)
- 보정 프로토콜
- 실험실 간 연구

### 표준화된 부품의 미래

**트렌드**:
- E. coli를 넘어 다양한 숙주로 이동
- 포유류 합성생물학
- 식물 엔지니어링
- 무세포 시스템

**새로운 표준**:
- 단백질 기반 로직 (DNA뿐만 아니라)
- RNA 장치
- 대사산물 센서 및 작동기
- 후성유전학적 제어 요소

**자동화와의 통합**:
- 로봇 조립
- 고처리량 스크리닝
- 폐쇄 루프 최적화
- AI 설계 부품

**비전**:
- 모든 기능을 다루는 포괄적인 라이브러리
- 예측 가능하고 신뢰할 수 있는 동작
- 진정한 플러그 앤 플레이 생물학
- 생물공학의 민주화

## 결론

BioBricks 및 표준화된 부품은 생물학적 엔지니어링의 근본적인 변화를 나타냅니다:
- 표준화는 체계적인 설계를 가능하게 함
- 모듈성은 개발을 가속화함
- 개방 공유는 전체 커뮤니티에 이익
- 새로운 기술과 함께 계속 진화

**주요 통찰**:
- 표준은 생물학을 위한 공통 언어 제공
- 여러 조립 방법이 다른 요구를 충족
- 부품 특성화가 신뢰성에 중요
- 커뮤니티 및 개방성이 혁신을 주도
- 과제는 남아 있지만 진전은 계속됨

**장 요약**:
- BioBricks는 일관된 인터페이스를 가진 표준화된 생물학적 부품
- 원본 표준 (RFC 10)은 4개의 제한 부위와 3A 조립 사용
- 대안 방법에는 Gibson 조립 및 Golden Gate 포함
- 부품은 계층적으로 구성: 기본 → 복합 → 장치 → 시스템

- 제한 사항에도 불구하고 BioBricks는 합성생물학을 변화시킴

**핵심 요점**:
1. 표준화는 생물학 엔지니어링의 기본
2. BioBricks는 모듈식, 조합적 설계를 가능하게 함
3. 다른 요구를 위한 여러 조립 방법 사용 가능
4. 적절한 특성화는 부품 유용성에 필수
5. 개방 표준 및 공유가 혁신을 가속화
