**버전:** 1.0

**철학:** 弘益人間 (홍익인간)

## 6.1 개요

생물학적 부품 레지스트리 시스템의 이 측면은 전체 생태계의 성공에 매우 중요합니다. WIA-SYNTHETIC-BIO-REGISTRY 표준은 국제 모범 사례를 기반으로 하며 20,000개 이상의 부품이 있는 레지스트리의 요구 사항을 충족하도록 설계되었습니다.

### 주요 과제

현대 레지스트리가 직면한 과제는 다음과 같습니다:
- 규모 확장 및 복잡성 관리
- 다양한 기여자에 걸친 품질 보장
- 국제 표준과의 상호 운용성 유지
- 지속 가능한 자금 모델 보장
- 안전 및 보안 요구 사항 균형
- 커뮤니티 참여 및 기여 촉진

### 핵심 원칙

효과적인 구현에는 다음이 필요합니다:
✓ 강력한 기술 인프라
✓ 명확한 거버넌스 구조
✓ 포괄적인 문서화
✓ 활성 커뮤니티 참여
✓ 지속적인 개선 프로세스
✓ 국제 협력

## 6.2 글로벌 레지스트리 네트워크 아키텍처

### 핵심 레지스트리 시스템

**중앙 집중식 레지스트리 (iGEM Registry - parts.igem.org):**
- 위치: MIT (미국)
- 부품 수: 20,000개 이상
- 데이터베이스: PostgreSQL + Elasticsearch
- API: RESTful API v2.0
- 접근성: 완전 공개

**AddGene 플라스미드 저장소:**
- 위치: 미국 (Harvard 기반)
- 플라스미드: 50,000개 이상 보유
- 물리적 배포: 6개 대륙의 14개 배포 센터
- 비용: 부품 당 $65-$75 (대학), $125-$150 (기업)
- 추적: 블록체인 기반 출처 기록

**JBEI (Joint BioEnergy Institute) 레지스트리:**
- 위치: Lawrence Berkeley National Lab (미국)
- 전문 분야: 에너지 및 바이오연료 관련 부품
- 부품 수: 3,000개 이상
- 특징: 고급 특성화 데이터

**Peking University 중국 생물부품 레지스트리:**
- 위치: 중국
- 부품 수: 1,500개 (포함 중국 고유 부품)
- 특징: 중국어 인터페이스, 아시아 중심 부품
- 네트워크: 동아시아 다른 대학과 연계

## 6.3 상호 운용성 및 표준화

### 글로벌 부품 식별 시스템 (Global Part Identifier, GPI)

**형식:**
```
[registry_code]:[collection]:[part_id]:[version]
```

**예시:**
```
IGEM:StandardCollection:J23100:1
AddGene:CRISPR:12345:2
BJCELL:Metabolic:P001:1
WIA:BioParts:PROM001:1
```

### 데이터 교환 프로토콜

**SBOL (Synthetic Biology Open Language) 표준:**
- XML/RDF 기반 표현
- 부품과 설계 정보 교환
- 버전 3.0 (2022년 발표)
- 다양한 소프트웨어 도구 지원

**OBO (Open Biomedical Ontologies) 포맷:**
- 개방형 생물 의학 온톨로지 교환 형식
- 여러 레지스트리 간 용어 표준화

**RDF/XML 형식:**
- 의미론적 웹 기술
- 자동 추론 및 통합 가능
- 기계 판독 가능

### 국제 협력 체계

**WIA-SYNTHETIC-BIO-REGISTRY 글로벌 얼라이언스:**

주요 회원 기관:
1. iGEM Foundation (미국) - 중앙 조정 역할
2. AddGene (미국) - 물리적 배포 담당
3. JBEI (미국) - 기술 표준 개발
4. BioBricks Foundation (국제) - IP 관리
5. 주요 대학 레지스트리들 (전 세계)
6. WIA (World Certification Industry Association) - 표준화

## 6.4 글로벌 부품 가용성 및 물류

### 부품 배포 채널

**온라인 디지털 접근:**
```
방법: 무료 다운로드 (FASTA, GenBank, SBOL)
비용: 무료
시간: 즉시 (5분 이내)
속도: 거의 무제한
제약: 디지털 형태만 (물리적 DNA 없음)
```

**물리적 샘플 배포 (AddGene 기반):**
```
플라스미드 형태: 글리세롤 스톡 또는 냉동 (-80°C)
배송: 냉동 택배 (5-7일 국내, 10-14일 국제)
비용: $65-150 (부픔당)
배송지: 학계 및 비영리(할인), 기업(정가)
검증: 배송 전 서열 확인
추적: 온라인 추적 가능
```

**현지 배포 센터 (대기 시간 단축):**
```
미국 (4개 센터):
- 배송 시간: 2-4일
- 비용: 기본과 동일
- 관리: AddGene

유럽 (3개 센터):
- 배송 시간: 1-3일

- 온도: -80°C 유지

아시아 (3개 센터):
- 배송 시간: 1-5일
- 비용: 지역별 약간 다름
- 확장 중
```

## 6.6 생물안전 및 규제 준수

### 글로벌 생물안전 표준

**부품별 생물안전 분류:**
```
Biosafety Level 1 (BSL-1):
- 대부분의 표준 부품
- E. coli 기반 부품
- 병원성 없음
- 특별 규제 없음
- 예: Anderson 프로모터, GFP

Biosafety Level 2 (BSL-2):
- 조건부 병원성
- 독소 관련 부품
- 항생제 내성 마커 포함
- 특별 취급 필요
- 예: 독소-항독소 회로

Biosafety Level 3 (BSL-3):
- 병원성 미생물 관련
- 제한된 배포
- 승인된 기관만 사용
- 규제 기관 보고 필요
- 예: 병원성 인자 부품

제한 물질:
- 선택 에이전트 (Select Agents)
- 이중 용도 연구 물질
- 제한된 국가에만 배포
- 미국: Select Agent Regulation
```

### 국제 규제 준수

**미국 (FDA/CDC):**
- Select Agent 등록 필수
- FBI 신원 조회 필요
- 정기 감시 (분기별)
- 위반 시 벌금 및 투옥

**유럽 (90/219/EEC, 98/81/EC):**
- GMO 규제 지침 준수
- 위험 평가 필수
- 국가별 통지 필요
- 모니터링 및 보고

**일본:**
- 법무성 승인 필수
- 격리 시설 요구
- 정기 감시
- 환경 영향 평가

**중국:**
- 과학원 (CAS) 승인 필수
- 국방부 조정 가능
- 제한적 배포
- 엄격한 통제
