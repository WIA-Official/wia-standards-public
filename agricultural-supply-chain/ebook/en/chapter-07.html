7.2 WMS integration

Manhattan Active Warehouse Management, Blue Yonder Luminate WM, and SAP EWM all accept inbound notices via REST or messaging. The WIA-AGRI-014 connector emits an InboundShipment when a carrier crosses the dock geofence:

POST /wms/inbound
{
  "shipmentId":"SHIP-2026-0421","sscc":"356123450000000017",
  "expectedArrival":"2026-04-23T17:00:00+09:00",
  "warehouseId":"WH-INC-002","storageZone":"REFRIGERATED_2_8C",
  "putaway":{"strategy":"FEFO",
             "shelfLifeAfterPackingDays":18,
             "packedOn":"2026-04-22"}
}

FEFO (First-Expired-First-Out) putaway is wired to the FHIR ProductShelfLife from §4.5; nothing is duplicated, the date flows straight through.

7.3 POS and consumer-facing integration

Retailer POS (NCR Commerce, Toshiba TCx, Walmart Retail Link) records receipt of WIA-traceable lots and surfaces a QR code on the receipt or shelf-edge label:

const consumer = {
  productName: shipment.productInfo.productName,
  farm: shipment.origin.farmName,
  harvestDate: shipment.productInfo.harvestDate,
  verifyUrl: `https://verify.wia-supply-chain.org/p/${shipment.productInfo.gtin}/${shipment.productInfo.lotNumber}`,
  blockchain: { verified:true, txId: shipment.blockchain.transactionId }
};
const qr = await QRCode.toDataURL(JSON.stringify(consumer));

Consumer scan resolves to a verifiable provenance page powered by WIA-VC.

WIA-CARBON

Each cold-chain ingest emits a side-record into carbon.cold_chain using GHG Protocol Scope 3 Category 4 factors. ISO 14083:2023 rules govern the calculation; the resulting tCO2e attaches to the shipment for retailer ESG dashboards.

7.6 Outbreak trace-back: 22 minutes, not 22 days

FDA's 2018 romaine outbreak took three weeks to trace. With WIA-AGRI-014 the trace runs as a single EPCIS query plus an Iceberg time-travel scan:

-- 1. Find every shipment where the lot was an input.
SELECT shipment_id, customer_glnFROM agri.epcis.events
WHERE event_type = 'TransformationEvent'
  AND ARRAY_CONTAINS(input_epc_list, 'urn:epc:id:sgtin:880123.4567896.JJHBG2026W17');

-- 2. Walk forward through AggregationEvents until retail.
-- 3. Cold-chain temperature window.
SELECT * FROM agri.cold_chain.readings
FOR VERSION AS OF 7841029384712340987
WHERE shipment_id IN (SELECT shipment_id FROM step1);

Iceberg time-travel guarantees the query is reproducible months later for litigation defense.
