8장: 구현 · 인증 · ROI

장 개요: 8개월 도입 로드맵, 세 가지 모범 사례, 한국 현장 케이스 스터디, 인증 3등급(Bronze/Silver/Gold), 농가·도매·체인별 ROI 분석을 한자리에 정렬한다.

8.1 8개월 도입 로드맵

1~2개월차 — Phase 1 데이터

  • 품목별 GTIN/SSCC/GLN 부여 — GS1 Korea 회원 등록 또는 LEI 매핑.
  • JSON Schema 2020‑12 검증 자동화(Ajv 8) 파이프라인.
  • Iceberg v2 테이블 + 파티션 변환 적용.
  • FHIR R5 ProductShelfLife/Provenance 프로파일 발행.

3~4개월차 — Phase 2 API

  • OpenAPI 3.1 + EPCIS 2.0 두 façade 서버 가동.
  • OAuth 2.1 + PKCE, mTLS 발급, RFC 9421 서명 키 회전.
  • EUDR DDS·MAFRA 어댑터 활성화.
  • RFC 9457 problem+json 일관 적용.

5~6개월차 — Phase 3 프로토콜

  • MQTT 5 브로커(EMQX 5 / HiveMQ 4) 배포 + ACL 설계.
  • BLE 로거 OSCORE 키 프로비저닝, 펌웨어 OTA.
  • LoRaWAN 게이트웨이 설치(KR920) + ChirpStack 연동.
  • Hyperledger Fabric 다중 조직 endorsement, WIA Chain PoA 노드 합류.

7~8개월차 — Phase 4 통합

  • SAP/Oracle/NetSuite/iCUBE 어댑터.
  • WMS FEFO putaway, POS QR.
  • FSMA Rule 204·EUDR·MFDS·MAFRA 자동 보고.
  • WIA‑VC 발급 + WIA‑CARBON 대시보드 연동.

8.3 한국 현장 케이스 스터디

케이스 1 — 제주 유기 협동조합 (50개 농가)

  • 과제: 친환경 인증 보유 농가가 가락시장 경매에서 일반 농산물과 같은 라인에 섞여 가격 프리미엄을 못 받음.
  • 해결: WIA‑VC + 패키지 QR + EPCIS 출하 이벤트 자동 발행.
  • 결과: 평균 35% 가격 프리미엄, 일본 수출 200% 증가, 12개월간 원산지 분쟁 0건.

케이스 2 — 서울 신선마켓 체인 (매장 87개)

  • 과제: 콜드체인 실패로 12% 부패율, 매장 클레임 다발.
  • 해결: MQTT 5 + BLE 로거 + 실시간 CAP 1.2 경보 + 예측 분석.
  • 결과: 부패율 3%, 연 24억 원 절감, 고객만족 98%.

케이스 3 — 국내 유제품 가공사 (산하 농가 5만 호)

  • 과제: FSMA Rule 204 발효 → 미국 수출 재개 위해 24시간 추적 의무 충족.
  • 해결: SAP S/4HANA + WIA‑AGRI‑014 통합, EPCIS 이벤트 자동 발행.
  • 결과: 수출 재개, 회수 추적 2.2초, 회피한 잠재 회수 비용 ≒ 150억 원.
