Basic Usage

Initialize the Client

import { InterKoreanExchange } from '@wia/inter-korean-data-exchange';

// Initialize with trust anchors
const exchange = new InterKoreanExchange({
  trustAnchors: [
    'rok-gov',      // Republic of Korea government
    'dprk-gov',     // DPRK government
    'un-observer',  // UN observer
    'icrc'          // International Committee of the Red Cross
  ],
  encryption: 'military-grade',
  auditLevel: 'transparent',
  environment: 'production'
});

// Authenticate user
await exchange.authenticate({
  region: 'south',
  userId: 'user-12345',
  credentials: userCredentials
});

Send a Message

// Send humanitarian message
const message = await exchange.sendMessage({
  type: 'family-reunification',
  from: {
    region: 'south',
    userId: 'sender-id',
    name: 'Kim Min-ji'
  },
  to: {
    region: 'north',
    userId: 'recipient-id',
    name: 'Kim Chol-su'
  },
  content: {
    message: 'Dear brother, I have been searching for you...',
    attachments: [
      { type: 'photo', file: './family-photo.jpg' },
      { type: 'document', file: './family-tree.pdf' }
    ]
  },
  priority: 'high',
  humanitarian: true
});

console.log('Message sent:', message.id);
console.log('Verified by:', message.verifications);

Receive Messages

// Listen for incoming messages
exchange.on('message', async (message) => {
  console.log('New message from:', message.from.name);

  // Decrypt content
  const decrypted = await exchange.decrypt(message.content);

  // Display to user
  displayMessage({
    from: message.from,
    content: decrypted,
    timestamp: message.timestamp
  });

  // Send read receipt
  await exchange.sendReceipt(message.id, 'delivered');
});

Security Best Practices

Essential Security Guidelines:
  • Key Management: Never store private keys in code or version control
  • Certificate Pinning: Validate trust anchor certificates on every connection
  • Secure Storage: Use platform keychain/keystore for sensitive data
  • Regular Updates: Keep SDK updated to receive security patches
  • Audit Logging: Log all security-relevant events

Secure Key Generation

// Generate secure key pair
const keyPair = await exchange.generateKeyPair({
  algorithm: 'ECDH-P256',
  storage: 'hardware-backed' // Use hardware security if available
});

// Store private key securely
await exchange.secureStorage.store('user-private-key', keyPair.privateKey, {
  requireBiometric: true,
  requirePasscode: true
});

Message Encryption

// Encrypt with perfect forward secrecy
const encrypted = await exchange.encrypt(message, {
  recipient: recipientPublicKey,
  algorithm: 'AES-256-GCM',
  pfs: true,  // Perfect Forward Secrecy
  authenticate: true
});

// Verify encryption
console.log('Encrypted:', encrypted.ciphertext);
console.log('Ephemeral key:', encrypted.ephemeralKey);
console.log('Authentication tag:', encrypted.authTag);

Integration Patterns

Red Cross Integration

// Integrate with Red Cross verification system
const redCrossClient = new RedCrossIntegration({
  apiKey: process.env.RED_CROSS_API_KEY,
  region: 'korea'
});

// Verify humanitarian status
const verification = await redCrossClient.verify({
  userId: 'user-id',
  category: 'separated-family',
  supportingDocuments: ['./registration-form.pdf']
});

if (verification.approved) {
  // Grant humanitarian priority
  await exchange.grantHumanitarianStatus(userId, {
    category: 'family-reunification',
    verifiedBy: 'icrc',
    expiresAt: new Date('2026-12-31')
  });
}

Error Handling

try {
  await exchange.sendMessage(message);
} catch (error) {
  if (error instanceof EncryptionError) {
    console.error('Encryption failed:', error.message);
    // Retry with different encryption method
  } else if (error instanceof VerificationError) {
    console.error('Trust anchor verification failed:', error.trustAnchor);
    // Alert user to wait and retry
  } else if (error instanceof NetworkError) {
    console.error('Network error:', error.message);
    // Queue message for retry
    await exchange.queueForRetry(message);
  } else {
    console.error('Unknown error:', error);
    // Log to monitoring system
    await logError(error);
  }
}

Performance Optimization

  • Message Batching: Batch multiple small messages for efficient transmission
  • Compression: Enable gzip compression for large messages
  • Caching: Cache trust anchor certificates and public keys
  • Connection Pooling: Reuse connections to trust anchors
  • Async Processing: Use async/await for non-blocking operations
